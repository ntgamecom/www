# 装备与锻造 / Items & Crafting

暗黑4拥有极为丰富的装备和锻造系统，理解这些系统是构筑强力流派的基石 / Diablo 4 has a deep gear and crafting system — understanding it is essential for building powerful characters.

---

## 装备品质 / Gear Quality

| 品质 / Quality | 颜色 / Color | 说明 / Description |
|------|:----:|------|
| **普通 / Common** | 灰色 / Gray | 基础装备，无词缀 / Base gear, no affixes |
| **魔法 / Magic** | 蓝色 / Blue | 1-2条词缀 / 1-2 affixes |
| **稀有 / Rare** | 黄色 / Yellow | 3-5条词缀——后期升级底材 / 3-5 affixes — crafting base for endgame |
| **神圣 / Sacred** | 浅色 / Lighter | 世界难度3掉落，属性比普通高约30% / WT3 drop, ~30% higher stats |
| **远古 / Ancestral** | **橙色底 / Orange BG** | 世界难度4掉落，属性最高。终局装备的终极形态 / WT4 drop, highest stats. Ultimate endgame gear |
| **独特 / Unique** | **棕色/深橙 / Brown/Orange** | 固定词缀+独特效果。流派核心构筑 / Fixed affixes + unique effect. Core build enablers |
| **超级独特 / Uber Unique** | **红色/金边 / Red/Gold** | 极其稀有，游戏中最强的装备 / Extremely rare, the most powerful items in the game |

## 传奇威能 / Legendary Aspects

威能是暗黑4装备系统的灵魂——它们将稀有装备转化为传奇装备 / Aspects are the soul of D4's itemization — they turn rare items into legendary items.

### 威能来源 / Aspect Sources

| 来源 / Source | 说明 / Description |
|------|------|
| **地下城解锁 / Dungeon Unlock** | 首次完成特定地下城解锁对应的威能，可在无限使用 / Completing a specific dungeon unlocks its aspect permanently in your Codex |
| **提取 / Extraction** | 从传奇装备中提取威能（消耗该装备），可刻印到其他装备一次 / Extract aspect from a legendary (destroys the item) — apply to another item once |
| **掉落 / Drop** | 传奇装备自带随机威能 / Legendary items come with random aspects |

### 威能刻印 / Aspect Imprinting

在**秘术师**处将威能刻印到稀有装备上，将稀有升级为传奇 / Visit the Occultist to imprint an aspect onto a rare item, upgrading it to legendary.

| 刻印方式 / Method | 效果 / Result | 注意 / Note |
|------|------|------|
| **法典威能 / Codex Aspect** | 最低roll值，但可无限重复使用 / Minimum roll, but infinitely reusable | 适合过渡装备 / Good for transitional gear |
| **提取威能 / Extracted Aspect** | 保留原装备的roll值 / Keeps original item's roll values | **只能使用一次** / **One-time use only** |

### 威能类型 / Aspect Types

| 类型 / Type | 说明 / Description |
|------|------|
| **攻击威能 / Offensive** | 增加伤害输出 / Increase damage output |
| **防御威能 / Defensive** | 增加生存能力 / Increase survivability |
| **资源威能 / Resource** | 改善能量/怒气/灵力管理 / Improve resource management |
| **移动威能 / Mobility** | 增加移速和机动性 / Increase speed and mobility |
| **通用威能 / Utility** | 混合功能 / Hybrid functions |

## 回火系统 / Tempering

**解锁 / Unlock**: 世界难度3+ / World Tier 3+

回火是给装备附加额外词缀的系统——在**铁匠**处使用回火手册操作 / Tempering adds extra affixes to gear — use Tempering Manuals at the Blacksmith.

### 回火流程 / Tempering Process

1. 获取回火手册（通过地狱狂潮、梦魇地下城、世界BOSS等）/ Obtain Tempering Manuals (from Helltide, NMDs, World Bosses, etc.)
2. 在铁匠处选择"回火" / Select "Temper" at the Blacksmith
3. 选择手册 → 随机添加2条该手册池中的词缀 / Choose a manual → adds 2 random affixes from that pool
4. 每条装备**最多回火2次**，不可撤销——选择要慎重 / **Max 2 tempers per item**, irreversible — choose carefully

### 回火词缀池 / Tempering Affix Pools

| 手册类型 / Manual Type | 包含词缀示例 / Example Affixes |
|------|------|
| **武器攻击 / Weapon Attacks** | 对近距离/远距离伤害、对精英伤害、核心技能伤害 / Close/distant dmg, elite dmg, core skill dmg |
| **武器属性 / Weapon Stats** | 暴击伤害、易伤伤害、全属性伤害 / Crit dmg, Vulnerable dmg, all dmg |
| **防具生存 / Armor Defense** | 全减伤、近远减伤、护甲%、生命 / All-resist, close/distant DR, armor %, life |
| **防具效用 / Armor Utility** | 冷却缩减、资源生成、幸运一击 / Cooldown reduction, resource gen, lucky hit |
| **首饰 / Jewelry** | 元素抗性、控制时长、伤害减免 / Elemental resist, CC duration, dmg reduction |

## 精铸系统 / Masterworking

**解锁 / Unlock**: 世界难度4 + 完成秘域传送门第1层 / WT4 + complete Pit Tier 1

精铸是终局装备的终极强化系统，可提升装备所有词缀的数值 / Masterworking is the ultimate gear upgrade system — it boosts all affix values on an item.

### 精铸等级 / Masterworking Levels

| 等级 / Level | 提升 / Improvement | 材料 / Materials |
|:----:|------|------|
| **1-4** | 每级所有词缀+5% / +5% all affixes per level | 精铸石 + 金币 / Obolite + Gold |
| **5** | **暴击**——随机一条词缀额外+25% / **Crit** — random affix +25% | 遗忘之魂 + 精铸石 + 金币 / Forgotten Souls + Obolite + Gold |
| **6-8** | 所有词缀+5% / +5% all affixes | 精铸石 + 金币 / Obolite + Gold |
| **9** | **第二次暴击**——另一条词缀额外+25% / **2nd Crit** — another affix +25% | 遗忘之魂 + 精铸石 + 金币 / Forgotten Souls + Obolite + Gold |
| **10-11** | 所有词缀+5% / +5% all affixes | 精铸石 + 金币 / Obolite + Gold |
| **12** | **第三次暴击**——另一条词缀额外+25% / **3rd Crit** — another affix +25% | 遗忘之魂 + 精铸石 + 金币 / Forgotten Souls + Obolite + Gold |

**材料来源 / Material Sources**:
- **精铸石 / Obolite**: 秘域传送门、炼狱大军 / The Pit, Infernal Hordes
- **遗忘之魂 / Forgotten Souls**: 地狱狂潮神秘宝箱、炼狱大军 / Helltide Mystery Chests, Infernal Hordes
- **金币 / Gold**: 卖装备、世界事件 / Selling gear, world events

> **精铸策略**：先用低等级精铸石升到8级，确认前两次暴击是否加到理想词缀上。如果不是，重置重来 / **Strategy**: Masterwork to level 8 first, check if the first 2 crits hit desired affixes — if not, reset and retry

## 符文之语 / Runewords

**解锁 / Unlock**: 资料片"憎恨之躯" / Vessel of Hatred expansion

符文之语是资料片新增的系统：在装备上镶嵌两个符文（仪式符文+触发符文）组合获得额外效果 / System added in Vessel of Hatred: socket 2 runes (Ritual + Trigger) into gear for bonus effects.

### 符文类型 / Rune Types

| 类型 / Type | 说明 / Description |
|------|------|
| **仪式符文 / Ritual Rune** | 触发条件——执行特定操作获得奉献值 / Trigger condition — perform an action to generate Tribute |
| **触发符文 / Trigger Rune** | 消耗奉献值触发效果 / Consume Tribute to trigger an effect |

### 常用符文组合 / Popular Runeword Combos

| 组合 / Combo | 效果 / Effect | 适用职业 / Best For |
|------|------|------|
| **莫尼+泰克 / Moni + Tec** | 施放非基础技能后获得暴击加成 / Gain crit bonus after casting non-basic skill | 所有职业 / All classes |
| **巴奥+克雷 / Bac + Cle** | 移动距离累积后触发圣光弹 / Move distance triggers holy bolts | 游侠/魂灵师 / Rogue/Spiritborn |
| **艾奥+奈尔 / Ioe + Nal** | 消耗资源减少冷却 / Spending resources reduces cooldowns | 高消耗流派 / High-spend builds |
| **瑟尔+伊格 / Thul + Igni** | 施放终极技能触发火球新星 / Cast ultimate triggers fireball nova | 法师 / Sorcerer |

## 佣兵系统 / Mercenary System

**解锁 / Unlock**: 资料片"憎恨之躯" / Vessel of Hatred

佣兵是陪伴你战斗的AI伙伴，提供额外输出和辅助 / Mercenaries are AI companions that provide additional damage and utility.

### 可选佣兵 / Available Mercenaries

| 佣兵 / Mercenary | 类型 / Type | 核心技能 / Core Skill |
|------|------|------|
| **拉希尔 / Raheir** | 坦克 / Tank | 盾牌冲撞 + 群体嘲讽 / Shield bash + group taunt |
| **瓦莉雅娜 / Varyana** | 输出 / DPS | 双持狂暴双斧 / Dual-wield berserker axes |
| **索科 / Sokor** | 辅助 / Support | 治疗 + 增益Buff / Healing + Buffs |
| **亚尔丹 / Aldkin** | 法师 / Mage | 元素法术 + 控场 / Elemental spells + CC |

### 佣兵升级 / Mercenary Progression

| 机制 / Mechanic | 说明 / Description |
|------|------|
| **等级 / Level** | 共同战斗升级，解锁新技能 / Level up by fighting together, unlock new skills |
| **技能树 / Skill Tree** | 每个佣兵有独立的技能树 / Each mercenary has an independent skill tree |
| **装备 / Gear** | 佣兵可以装备武器和护符 / Mercenaries can equip weapons and amulets |
| **忠诚度 / Rapport** | 一起战斗提升忠诚度，解锁额外奖励 / Fight together to increase rapport, unlock bonus rewards |

## 宝石系统 / Gem System

宝石可以镶嵌到插槽装备（头盔、胸甲、裤子、武器）/ Gems socket into gear with slots (helm, chest, pants, weapons).

### 宝石效果 / Gem Effects

| 宝石 / Gem | 武器效果 / Weapon | 护甲效果 / Armor | 饰品效果 / Jewelry |
|------|------|------|------|
| **红宝石 / Ruby** | +持续伤害 / +DoT | +生命 / +Life | +全抗 / +All Resist |
| **蓝宝石 / Sapphire** | +暴击伤害 / +Crit Dmg | +强韧减伤 / +Fortify DR | +冰冷抗性 / +Cold Resist |
| **绿宝石 / Emerald** | +易伤伤害 / +Vuln Dmg | +闪避 / +Dodge | +毒抗 / +Poison Resist |
| **黄宝石 / Topaz** | +基本技能伤害 / +Basic Dmg | +控制减伤 / +CC DR | +闪电抗性 / +Lightning Resist |
| **紫宝石 / Amethyst** | +暗影伤害 / +Shadow Dmg | +持续减伤 / +DoT DR | +暗影抗性 / +Shadow Resist |
| **钻石 / Diamond** | +终极技能伤害 / +Ult Dmg | +屏障产生 / +Barrier Gen | +火抗 / +Fire Resist |
| **骷髅 / Skull** | +生命值恢复 / +Life on Kill | +护甲值 / +Armor | 无 / None |

### 宝石等级 / Gem Tiers

粗糙 → 普通 → 优质 → 完美 → 皇家 → **传说**（最高级）/ Crude → Chipped → Standard → Perfect → Royal → **Legendary** (highest)

> **赛季宝石提示**：某些赛季会引入赛季专属宝石，有赛季角色限定效果 / **Note**: Some seasons introduce seasonal-specific gems with exclusive effects
