## 传奇威能 / Legendary Aspects

威能是暗黑4装备系统的灵魂——它们将稀有装备转化为传奇装备 / Aspects are the soul of D4's itemization — they turn rare items into legendary items.

### 威能来源 / Aspect Sources

| 来源 / Source | 说明 / Description |
|------|------|
| **地下城解锁 / Dungeon Unlock** | 首次完成特定地下城解锁对应的威能，可在无限使用 / Completing a specific dungeon unlocks its aspect permanently in your Codex |
| **提取 / Extraction** | 从传奇装备中提取威能（消耗该装备），可刻印到其他装备一次 / Extract aspect from a legendary (destroys the item) — apply to another item once |
| **掉落 / Drop** | 传奇装备自带随机威能 / Legendary items come with random aspects |

### 威能刻印 / Aspect Imprinting

在**秘术师**处将威能刻印到稀有装备上，将稀有升级为传奇 / Visit the Occultist to imprint an aspect onto a rare item, upgrading it to legendary.

| 刻印方式 / Method | 效果 / Result | 注意 / Note |
|------|------|------|
| **法典威能 / Codex Aspect** | 最低roll值，但可无限重复使用 / Minimum roll, but infinitely reusable | 适合过渡装备 / Good for transitional gear |
| **提取威能 / Extracted Aspect** | 保留原装备的roll值 / Keeps original item's roll values | **只能使用一次** / **One-time use only** |

### 威能类型 / Aspect Types

| 类型 / Type | 说明 / Description |
|------|------|
| **攻击威能 / Offensive** | 增加伤害输出 / Increase damage output |
| **防御威能 / Defensive** | 增加生存能力 / Increase survivability |
| **资源威能 / Resource** | 改善能量/怒气/灵力管理 / Improve resource management |
| **移动威能 / Mobility** | 增加移速和机动性 / Increase speed and mobility |
| **通用威能 / Utility** | 混合功能 / Hybrid functions |

### 威能配置策略 / Aspect Setup Strategy

- **双手武器**: 放唯一的攻击威能（数值翻倍）/ **2H weapon**: single offensive aspect (value doubled)
- **护符**: 放核心攻击威能（数值+50%）/ **Amulet**: core offensive aspect (+50% value)
- **单手武器+副手**: 可以放2个不同的攻击威能 / **1H + Off-hand**: 2 different offensive aspects
- **防具**: 防御+资源+通用威能均衡搭配 / **Armor**: balance defensive, resource, and utility aspects
- **戒指**: 剩余的攻击威能 / **Rings**: remaining offensive aspects

## 回火系统 / Tempering

**解锁 / Unlock**: 世界难度3+ / World Tier 3+

回火是给装备附加额外词缀的系统——在**铁匠**处使用回火手册操作 / Tempering adds extra affixes to gear — use Tempering Manuals at the Blacksmith.

### 回火流程 / Tempering Process

1. 获取回火手册（通过地狱狂潮、梦魇地下城、世界BOSS等）/ Obtain Tempering Manuals (from Helltide, NMDs, World Bosses, etc.)
2. 在铁匠处选择"回火" / Select "Temper" at the Blacksmith
3. 选择手册 → 随机添加2条该手册池中的词缀 / Choose a manual → adds 2 random affixes from that pool
4. 每条装备**最多回火2次**，不可撤销——选择要慎重 / **Max 2 tempers per item**, irreversible — choose carefully

| **防具生存 / Armor Defense** | 全减伤、近远减伤、护甲%、生命 / All-resist, close/distant DR, armor %, life |
| **防具效用 / Armor Utility** | 冷却缩减、资源生成、幸运一击 / Cooldown reduction, resource gen, lucky hit |
| **首饰 / Jewelry** | 元素抗性、控制时长、伤害减免 / Elemental resist, CC duration, dmg reduction |

## 精铸系统 / Masterworking

**解锁 / Unlock**: 世界难度4 + 完成秘域传送门第1层 / WT4 + complete Pit Tier 1

精铸是终局装备的终极强化系统，可提升装备所有词缀的数值 / Masterworking is the ultimate gear upgrade system — it boosts all affix values on an item.

### 精铸等级 / Masterworking Levels

| 等级 / Level | 提升 / Improvement | 材料 / Materials |
|:----:|------|------|
| **1-4** | 每级所有词缀+5% / +5% all affixes per level | 精铸石 + 金币 / Obolite + Gold |
| **5** | **暴击**——随机一条词缀额外+25% / **Crit** — random affix +25% | 遗忘之魂 + 精铸石 + 金币 / Forgotten Souls + Obolite + Gold |
| **6-8** | 所有词缀+5% / +5% all affixes | 精铸石 + 金币 / Obolite + Gold |
| **9** | **第二次暴击**——另一条词缀额外+25% / **2nd Crit** — another affix +25% | 遗忘之魂 + 精铸石 + 金币 / Forgotten Souls + Obolite + Gold |
| **10-11** | 所有词缀+5% / +5% all affixes | 精铸石 + 金币 / Obolite + Gold |
| **12** | **第三次暴击**——另一条词缀额外+25% / **3rd Crit** — another affix +25% | 遗忘之魂 + 精铸石 + 金币 / Forgotten Souls + Obolite + Gold |

**材料来源 / Material Sources**:
- **精铸石 / Obolite**: 秘域传送门、炼狱大军 / The Pit, Infernal Hordes
- **遗忘之魂 / Forgotten Souls**: 地狱狂潮神秘宝箱、炼狱大军 / Helltide Mystery Chests, Infernal Hordes
- **金币 / Gold**: 卖装备、世界事件 / Selling gear, world events

### 精铸策略 / Masterworking Strategy

1. **前期只升到4级**: 材料有限时优先多件装备+4 / **Early: only go to +4**: prioritize getting all gear to 4
2. **8级停手检查**: 升到8级看前两次暴击是否加在理想词缀上 / **Pause at 8**: check if first 2 crits hit desired affixes
3. **如果不理想**: 使用重置功能（消耗精铸石）回到0级重来 / **If bad**: reset (costs Obolite) and retry
4. **如果理想**: 继续升到12级，赌第三次暴击 / **If good**: push to 12, gamble on 3rd crit
5. **优先精铸部位**: 武器 > 护符 > 手套 > 戒指 > 防具 / **Priority order**: Weapon > Amulet > Gloves > Rings > Armor

> **精铸策略**：先用低等级精铸石升到8级，确认前两次暴击是否加到理想词缀上。如果不是，重置重来 / **Strategy**: Masterwork to level 8 first, check if the first 2 crits hit desired affixes — if not, reset and retry

### 第8赛季精铸优化 / S8 Masterworking Improvements

第8赛季降低了精铸重置成本，不再消耗遗忘之魂，只需要精铸石和金币 / S8 reduced reset costs — no more Forgotten Souls needed for reset, only Obolite + Gold

## 装备进阶策略 / Advanced Gearing Strategy

### 完美装备流程 / Perfect Gear Progression

1. **刷到好的远古底材** / **Farm a good Ancestral base**: 5条词缀中至少3条正确 / At least 3/5 affixes correct
2. **附魔优化** / **Enchant**: 秘术师处重Roll一条错误词缀 / Reroll 1 bad affix at Occultist
3. **刻印威能** / **Imprint Aspect**: 刻印最好的版本 / Imprint your best roll aspect
4. **回火** / **Temper**: 添加2条回火词缀（不可撤销，谨慎选择）/ Add 2 temper affixes (irreversible, choose carefully)
5. **打孔** / **Socket**: 使用散光棱镜添加插槽 / Add socket with Scattered Prism
6. **精铸** / **Masterwork**: 升到12级 + 正确暴击 / Masterwork to 12 with correct crits
7. **符文之语** / **Runewords**: 镶嵌合适的符文组合 / Socket the right rune combo

| 炼狱大军 / Infernal Hordes | 1-3个 / 1-3 |
