## 技能树结构 / Skill Tree Structure

暗黑4的技能树分为七层——从基础技能到终极被动。每层需要在上层投入至少1点才能解锁。每个技能可以投入最多5点(部分技能有3级上限)。**总技能点数**: 58点(50级+8点来自声望奖励) / Diablo 4's skill tree has seven tiers — from basic skills to ultimate passives. Each tier requires at least 1 point in the previous tier to unlock. Each skill can have up to 5 points (some have 3-point cap). **Total skill points**: 58 (50 levels + 8 from reputation rewards).

|---|---|---|---|

| 2 | 核心技能 / Core Skills | 消耗资源的主要输出 / Resource-spending main DPS | 5-6个 / 5-6 skills |
| 3 | 防御技能 / Defensive Skills | 保命和减伤 / Survival and damage reduction | 3-4个 / 3-4 skills |
| 4 | 辅助技能 / Utility Skills | 位移和控制 / Mobility and control | 3-4个 / 3-4 skills |
| 5 | 精通技能 / Mastery Skills (Sorcerer) / 武器精通 / Weapon Mastery (Barbarian) | 特殊流派核心 / Specialized core | 3-4个 / 3-4 skills |

| 7 | 终极被动 / Ultimate Passives | 终局被动选择 / Endgame passive choice | 3-5个 / 3-5 passives |

## 各职业核心技能分析 / Class Core Skill Analysis

### 野蛮人核心技能 / Barbarian Core Skills

**双手挥击**: 消耗25怒气——两把武器同时攻击。Build核心——配合"狂乱"被动和"怒意冲动"传奇节点。推荐5点满级。**旋风斩**: 持续消耗怒气——旋转攻击周围所有敌人。终局Build的核心。需要配合"旋风斩延长"传奇词缀。**先祖之锤**: 消耗怒气——巨大锤子砸地。高爆发技能，配合"地震"传奇节点和"焦土"被动。**双重横扫**: 消耗怒气——前方扇形范围攻击——适合清图。第8赛季中双手挥击获得额外范围加成 / **Double Swing**: 25 Fury — dual weapon attack. Build core — syncs with "Frenzy" passive and "Unbridled Rage" legendary node. Max 5 points. **Whirlwind**: Continuous Fury drain — spin attack all nearby. Endgame core — needs "Whirlwind extension" affix. **Hammer of the Ancients**: Fury cost — giant hammer slam. High burst with "Earthquake" legendary node. **Double Sweep**: Fury cost — frontal cone AoE — great for clear. In S8, Double Swing gets extra area bonus.

### 法师核心技能 / Sorcerer Core Skills

**寒冰弹片**: 消耗法力——发射多枚冰弹。冰系Build的主力技能。配合**寒冰裂片附魔**实现自动攻击。**火焰弹**: 消耗法力——范围火焰伤害。火系核心技能——配合"燃尽"被动和"火焰精通"。**闪电链**: 消耗法力——闪电在敌人间跳跃。电系Build——配合"电击"和"充能"被动。**暴风雪**: 持续法力消耗——大范围冰风暴——冰系控场主力。第8赛季中寒冰弹片的投射物数量从5增加到7 / **Ice Shards**: Mana cost — multiple ice projectiles. Main skill for Cold builds. Syncs with **Ice Shards Enchantment**. **Fireball**: Mana cost — AoE fire damage. Fire core — syncs with "Combustion" passive. **Chain Lightning**: Mana cost — lightning jumps between enemies. Lightning build with "Electrocution" passive. **Blizzard**: Channeled mana — large ice storm — cold CC main. In S8, Ice Shards projectile count increased from 5 to 7.

### 游侠核心技能 / Rogue Core Skills

**穿透射击**: 能量消耗——发射穿透箭矢。远程Build的核心。配合"精准射击"关键被动。**乱舞**: 能量消耗——快速双持攻击。近战游侠核心。配合"连击点数"机制——满层使用乱舞造成巨额伤害。**快速射击**: 能量消耗——连射多箭。站桩输出最强技能。配合"暗影灌注"和"毒素灌注"。**死亡陷阱**: 放置陷阱——延迟后爆炸——暗影游侠的核心爆发技能 / **Penetrating Shot**: Energy cost — piercing arrow. Ranged build core. Syncs with "Precision" key passive. **Twisting Blades**: Energy cost — rapid dual-wield. Melee Rogue core. Syncs with "Combo Points" — max stacks for massive damage. **Rapid Fire**: Energy cost — rapid arrow volley. Highest stationary DPS. Syncs with "Shadow Imbuement" and "Poison Imbuement". **Death Trap**: Place trap — delayed explosion — Shadow Rogue's core burst.

### 死灵法师核心技能 / Necromancer Core Skills

**骨矛**: 精魂消耗——发射骨矛穿透敌人。最主流的死灵Build。配合"骷髅战士"和"骨质碎片"。**邪爆**: 消耗尸体——引爆尸体造成范围伤害。尸爆流核心。配合"亡者之书"的尸爆强化。**血潮**: 精魂消耗——波浪形鲜血攻击。鲜血Build的核心。配合"血之新星"和"压制"。**枯萎**: 精魂消耗——暗影持续伤害区域——暗影死灵的核心技能 / **Bone Spear**: Essence cost — bone spear that pierces. Most popular Necro build. Syncs with "Skeletal Warriors" and "Splintering". **Corpse Explosion**: Consumes corpse — AoE damage. Corpse Explosion core. Syncs with "Book of the Dead" CE upgrade. **Blood Surge**: Essence cost — wave of blood attack. Blood build core. Syncs with "Blood Nova" and "Overpower". **Blight**: Essence cost — shadow DoT zone — Shadow Necro core skill.

## 巅峰盘系统 / Paragon Board System

50级后进入巅峰盘——**巅峰点数**每级获得4点。巅峰盘由多个面板组成，通过**雕文**和**传奇节点**提供终局战力。第8赛季新增了额外的巅峰面板和雕文插槽 / After level 50, enter Paragon Board — **Paragon Points** 4 per level. Multiple boards connected by **Glyphs** and **Legendary Nodes** for endgame power. S8 adds additional paragon boards and glyph sockets.

### 雕文系统 / Glyph System

雕文插入巅峰盘上的插槽——根据周围已激活的稀有节点数量获得效果。雕文通过**梦魇地下城**升级——最多15级。Lv15雕文扩大影响范围，Lv21获得额外效果。**推荐雕文**: 利用(Exploit)——对受伤敌人伤害+12%。愤怒(Wrath)——核心技能伤害+12%。控制(Control)——对控制的敌人伤害+12%。领地(Territorial)——对近距敌人伤害+10%, 减伤+10% / Glyphs are socketed into board slots — grant effects based on activated rare nodes in radius. Level up via **Nightmare Dungeons** — max level 15. Lv15 expands radius, Lv21 grants bonus effect. **Recommended Glyphs**: Exploit — +12% to injured. Wrath — core skill +12%. Control — +12% to controlled. Territorial — melee damage +10%, DR +10%.

### 巅峰盘加点策略 / Paragon Board Strategy

**优先开第二个面板**: 50级后先开第二个面板而不是在第一个面板上浪费点数——第二个面板的传奇节点更早解锁。**雕文升级优先**: 先升到15级扩大范围——再升到21级解锁额外效果。**稀有节点路径**: 连最关键稀有节点的最短路径规划——不要浪费点数在无用的小节点上 / **Priority second board**: After 50, unlock second board instead of wasting points on first board — earlier access to second board's legendary node. **Glyph leveling priority**: Level to 15 for radius expansion first — then to 21 for bonus effect. **Rare node pathing**: Plan shortest path connecting key rare nodes — don't waste points on useless small nodes.

## 技能加点策略 / Skill Point Allocation Strategy

**1-50级升级期**: 优先投入一个核心技能满5点→对应的基础技能1点→终极技能1点→关键被动1点。剩余点投到防御被动和生存被动。**终局优化**: 洗点一次重新分配——投入所有点到一个主力Build。使用**技能重置卷轴**或在秘术师处洗点。巅峰盘按最终Build模板规划。**第8赛季新的技能系统**: 赛季祝福可以提供额外技能点数——最高+5点——让Build更加灵活 / **Leveling 1-50**: Max one core skill (5 pts) → basic skill (1 pt) → ultimate (1 pt) → key passive (1 pt). Remaining points into defensive and survival passives. **Endgame optimization**: One respec — allocate all points into one main build. Use **Scroll of Respec** or reset at Occultist. Paragon board follows final build template. **S8 new system**: Seasonal blessings grant extra skill points — up to +5 — making builds more flexible.

## 第8赛季技能改动 / Season 8 Skill Changes

第8赛季对多个职业的技能进行了调整。**野蛮人**: 双手挥击范围扩大30%, 旋风的怒气消耗降低。**法师**: 冰霜技能冷却时间缩短, 火焰弹的燃烧伤害叠加层数上限提高。**游侠**: 灌注技能持续时间延长2秒, 穿透射击伤害提升15%。**死灵法师**: 召唤物的AI改善, 骨矛的精魂消耗降低。**德鲁伊**: 变形技能的伤害加成提升, 大地技能的范围扩大。**魂灵师**: 大幅重做——四灵系统更直观, 伤害倍率调整 / S8 adjusts skills across classes. **Barbarian**: Double Swing range +30%, Whirlwind Fury cost reduced. **Sorcerer**: Frost skill cooldowns reduced, Fireball burn stack cap increased. **Rogue**: Imbuement duration +2 sec, Penetrating Shot damage +15%. **Necromancer**: Minion AI improved, Bone Spear essence cost reduced. **Druid**: Shapeshift damage bonus increased, Earth skill area expanded. **Spiritborn**: Major rework — spirit system more intuitive, damage multipliers adjusted.
