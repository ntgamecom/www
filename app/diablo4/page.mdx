# 暗黑破坏神4 攻略 / Diablo 4 Guide

暗黑破坏神4（Diablo IV）是暴雪娱乐开发的暗黑破坏神系列最新正统续作，是一款开放世界动作角色扮演游戏（ARPG）。你将扮演庇护之地的流浪者，击退地狱恶魔，探索五个风格迥异的区域，挑战烈焰地狱的 ultimate 力量 / Diablo IV is the latest installment in Blizzard Entertainment's legendary ARPG series. Set in the open world of Sanctuary, you fight demons from the Burning Hells across five distinct regions and challenge the ultimate evil.

## 为何选择暗黑4？/ Why Diablo 4?

暗黑4适合以下玩家 / Diablo 4 is great for:

- **ARPG老玩家**：如果你喜欢暗黑2/3、流放之路、最后纪元——这里有你熟悉的核心循环 / ARPG veterans who enjoy the classic loot-and-slash loop
- **新玩家**：相比同类游戏，暗黑4的入门曲线更加平滑，战役模式引导清晰 / Newcomers: smoother learning curve than most ARPGs, with excellent campaign guidance
- **社交玩家**：开放世界中的世界BOSS、军团事件、交易系统让社交成为核心体验 / Social players: World Bosses, Legion Events, and trading make multiplayer a core part of the experience
- **刷宝爱好者**：强大的终局系统提供了数百小时的刷刷刷内容 / Loot enthusiasts: robust endgame systems provide hundreds of hours of farming content
- **剧情党**：暗黑4的过场动画和故事叙事达到了系列新高度 / Story lovers: D4's cinematics and narrative storytelling reach new series heights

## 游戏核心系统 / Core Game Systems

### 世界与探索 / World & Exploration

暗黑4采用真正的开放世界设计，5个主要区域无缝连接 / D4 features a true open world with 5 seamlessly connected regions:

| 区域 / Region | 主题 / Theme | 关键地点 / Key Location |
|------|------|------|
| **破碎群峰 / Fractured Peaks** | 雪山峡谷 / Snowy mountains | 涅瓦克、圣光大教堂 / Nevesk, Cathedral of Light |
| **斯科斯格伦 / Scosglen** | 森林高地 / Forested highlands | 图尔·杜拉、塞利格 / Tur Dulra, Cerrigar |
| **干燥平原 / Dry Steppes** | 沙漠戈壁 / Arid desert | 凯·巴杜、寂默平原 / Ked Bardu, Fields of Desecration |
| **凯基斯坦 / Kehjistan** | 沙漠绿洲 / Desert oasis | 卡尔蒂姆、吉·库尔 / Caldeum, Gea Kul |
| **哈维泽 / Hawezar** | 沼泽湿地 / Toxic swamps | 扎宾泽特、低语之树 / Zarbinzet, Tree of Whispers |
| **纳汉图 / Nahantu** | 丛林天险 / Dense jungle | 资料片区域 / Expansion zone |

### 战斗系统 / Combat System

暗黑4的战斗注重节奏感和打击感 / D4's combat emphasizes rhythm and impact:
- **基础技能**产生资源，**核心技能**消耗资源造成伤害 / **Basic skills** generate resources, **Core skills** consume them for damage
- **终极技能**冷却长但效果强大 / **Ultimate skills** have long cooldowns but massive effects
- **闪避**有次数限制，需要合理管理 / **Evade** has limited charges — manage them wisely
- **连击奖励**连续击杀提供经验加成 / **Kill streak** bonuses grant XP multipliers
