# 流派攻略 / Builds Guide

第8赛季带来了全新的赛季机制和平衡调整，本指南涵盖6大职业的Meta流派，包括技能循环、装备威能要求和属性优先级 / Season 8 brings new seasonal mechanics and balance changes. This guide covers meta builds for all 6 classes, including skill rotations, gear/aspect requirements, and stat priorities.

---

## 1. 野蛮人 / Barbarian

### 猛击蛮 / Bash Barbarian (S-Tier)

**强度 / Power**: ★★★★★ — 第8赛季最强野蛮人流派 / Top Barbarian build for Season 8

**技能循环 / Skill Rotation**:
1. 使用**猛击**叠加伤害层数 / Use **Bash** to stack damage
2. 触发**战略**后使用**先祖之锤**爆发 / Trigger **Tactical** then unleash **Hammer of the Ancients**
3. **战吼**和**挑战怒吼**保持增伤和减伤 / Maintain **War Cry** and **Challenging Shout** for buffs
4. 怒气充足时循环输出 / Cycle damage when Fury is full

**装备威能 / Gear & Aspects**:

| 部位 / Slot | 威能 / Aspect | 说明 / Description |
|------|------|------|
| **双手武器1** | 无尽怒火 / Aspect of Limitless Rage | 怒气上限+60，核心技能伤害按怒气比例提升 / +60 Fury cap, core skill dmg scales with Fury |
| **双手武器2** | 狂暴撕裂 / Aspect of Berserk Ripping | 狂暴时造成额外流血伤害 / Extra Bleed damage while Berserking |
| **护符** | 无厌狂怒 / Aspect of Anemia | 对流血敌人幸运一击有几率眩晕 / Lucky hit vs Bleeding enemies has chance to stun |
| **戒指** | 回火怒气 / Aspect of Echoing Fury | 战吼产生怒气 / War Cry generates Fury |
| **头盔** | 永恒怒火 / Aspect of Eternal Fury | 基础技能产生更多怒气 / Basic skills generate more Fury |

**属性优先级 / Stat Priority**:
- 力量 > 暴击伤害 > 暴击几率 > 冷却缩减 > 全属性伤害 / Strength > Crit Damage > Crit Chance > Cooldown Reduction > All Damage

**巅峰盘关键雕文 / Key Paragon Glyphs**: 利用(Exploit), 控制(Control), 怒不可遏(Territorial), 无暇(Flawless)

### 旋风蛮 / Whirlwind Barbarian (A-Tier)

**技能循环 / Skill Rotation**:
1. **猛击**或**剥皮**产生怒气 / Generate Fury with **Bash** or **Flay**
2. **战吼 + 集结呐喊** 开启增伤 / Pop **War Cry + Rallying Cry** for buffs
3. 按住**旋风斩**持续旋转输出 / Hold **Whirlwind** for sustained damage
4. 面对精英时用**踢击**打断并造成易伤 / Use **Kick** to interrupt and apply Vulnerable on elites

**关键装备 / Key Items**: 风之力(Windforce) — 旋风斩攻速提升 / Whirlwind attack speed boost

**属性优先级 / Stat Priority**: 力量 > 攻速 > 暴击几率 > 移动速度 > 资源生成 / Strength > Attack Speed > Crit Chance > Move Speed > Resource Gen

---

### 先祖锤蛮 / Hammer of the Ancients Barbarian (S-Tier)

**技能循环 / Skill Rotation**: 基础技能产怒 → 等待狂暴触发 → 释放先祖之锤秒杀精英群 / Basic skill for Fury -> wait for Berserking -> one-shot elite packs with Hammer

**核心独特 / Core Unique**: 猩红戒指(Ring of Red Furor) — 使用基础技能后核心技能暴击 + 伤害提升 / Core skill crit + dmg after using basic skill

**属性优先级 / Stat Priority**: 力量 > 暴击伤害 > 暴击几率 > 先祖之锤等级 > 全属性 / Strength > Crit Dmg > Crit Chance > HotA ranks > All Stats

## 2. 法师 / Sorcerer

### 冰封球法 / Frozen Orb Sorcerer (S-Tier)

**技能循环 / Skill Rotation**:
1. **冰霜新星**冻结敌人起手 / Open with **Frost Nova** to freeze
2. 释放**冰封球**穿透敌人 / Cast **Frozen Orb** through enemies
3. **传送术**调整位置 / Use **Teleport** to reposition
4. 冰封球飞行途中使用**寒冰护甲**自保 / Use **Ice Armor** while Orb travels

**装备威能 / Gear & Aspects**:

| 部位 / Slot | 威能 / Aspect | 说明 / Description |
|------|------|------|
| **双手武器** | 冰心 / Iceheart Brais | 冻死敌人触发冰霜新星，连锁清场 / Frozen kill triggers Frost Nova chain |
| **护符** | 雪幕 / Aspect of the Snowshield | 寒冰护甲提供额外减伤 / Ice Armor gives extra damage reduction |
| **戒指** | 冰冷之触 / Aspect of Frozen Touch | 幸运一击冰冻敌人 / Lucky hit freezes enemies |
| **手套** | 风暴涌动 / Aspect of the Storm Swell | 对冻结敌人暴击率提升 / Crit chance vs Frozen enemies |

**属性优先级 / Stat Priority**: 智力 > 暴击伤害 > 暴击几率 > 冷却缩减 > 法力减耗 / Intelligence > Crit Damage > Crit Chance > CDR > Mana Cost Reduction

**附魔推荐 / Enchantments**: 冰霜新星(Frost Nova) + 火球(Fireball)

**技能循环 / Skill Rotation**:

**属性优先级 / Stat Priority**: 智力 > 持续伤害 > 暴击几率 > 资源生成 > 火焰技能等级 / Int > DoT > Crit Chance > Resource Gen > Fire Skill Ranks

## 3. 德鲁伊 / Druid

2. **山崩**作为核心输出 / **Landslide** as core damage
3. **飓风**和**藤蔓**维持控场 / Maintain **Hurricane** and **Creeping Vine** for CC
4. **石爆术**在灵力充足时高爆发 / **Boulder** for high burst when Spirit is full

**装备威能 / Gear & Aspects**:

| 部位 / Slot | 威能 / Aspect | 说明 / Description |
|------|------|------|
| **护符** | 大地之环 / Aspect of the Unsatiated | 山崩额外触发一次 / Landslide triggers an extra hit |
| **双手武器** | 震荡波 / Aspect of Shockwave | 大地尖刺波宽度提升 / Earth Spike wave width increased |
| **戒指** | 磨石 / Aspect of the Changing Boulder | 石爆术触发山崩 / Boulder triggers Landslide |
| **头盔** | 北极星 / Vasily's Prayer (Unique) | 大地技能获得熊人标签 / Earth skills gain Werebear tag |

**属性优先级 / Stat Priority**: 感知 > 暴击伤害 > 暴击几率 > 核心技能伤害 > 资源生成 / Willpower > Crit Dmg > Crit Chance > Core Skill Dmg > Resource Gen
