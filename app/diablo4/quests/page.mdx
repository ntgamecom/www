## 主线剧情流程 / Main Storyline

暗黑4的主线分为六个章节，从破碎群峰开始，逐步推进至烈焰地狱。全程约15-25小时。第8赛季新增了一条短剧情线——讲述魂灵师职业的起源故事 / Diablo 4's main story spans six acts, starting from Fractured Peaks and progressing to Hell. Duration ~15-25 hours. S8 adds a short story chain — telling the Spiritborn class origin story.

### 第一章至第六章 / Acts 1-6

| 章节 / Act | 区域 / Zone | 主线时长 / Duration | 推荐等级 / Level Range | 关键Boss / Key Boss |
|---|---|---|---|---|
| 第一章 / Prologue + Act 1 | 破碎群峰 / Fractured Peaks | 8-12小时 | 1-25 | 阿斯塔洛 / Astaroth |
| 第二章 / Act 2 | 干燥平原 / Dry Steppes | 6-8小时 | 25-35 | 安达尔 / Andariel (回忆关) |
| 第三章 / Act 3 | 哈维泽 / Hawezar | 8-10小时 | 35-45 | 墨菲斯托之影 / Mephisto Echo |
| 第四章 / Act 4 | 凯基斯坦 / Kehjistan | 4-5小时 | 45-50 | 莉莉丝之影 / Lilith Echo |
| 第五章 / Act 5 | 诅咒之地 / Scosglen (南) | 6-8小时 | 50-55 | 多恩 / Donan (剧情) |
| 第六章 / Act 6 | 烈焰地狱 / Hell | 4-6小时 | 55-60 | 莉莉丝 / Lilith (最终Boss) |

**主线快速通关建议**: 第一章+第二章全清(经验和装等积累)→第三章跳过支线直达主线→第四章解锁圣光大教堂(世界3)→第五章刷到55级→第六章通关解锁世界4。**第8赛季**中新增了跳过战役选项——创建赛季角色时可以直接跳过 / **Fast clear**: Act 1+2 full clear (XP + gear) → Act 3 main quest only → Act 4 unlock Cathedral of Light (WT3) → Act 5 grind to 55 → Act 6 completion unlocks WT4. **S8**: Skip Campaign option added for seasonal characters.

## 要塞任务 / Stronghold Quests

要塞是可重置的开放世界挑战区域。每个区域有1-2个要塞——清理后解锁传送点、商人、和专有任务线。要塞也是获取**声望**的重要来源——每个要塞给予100点区域声望 / Strongholds are resetable open-world challenge zones. Each zone has 1-2 strongholds — clearing them unlocks waypoints, merchants, and exclusive quest lines. Strongholds also grant significant **reputation** — 100 zone reputation each.

### 全要塞列表 / All Strongholds

| 要塞 / Stronghold | 区域 / Zone | 推荐等级 / Level | 奖励 / Reward |
|---|---|---|---|
| 马诺克要塞 / Malnok | 破碎群峰 / Fractured Peaks | 20-25 | 解锁新传送点 / Waypoint + vendor |
| 科尔萨德拉 / Kor Dragan | 破碎群峰 / Fractured Peaks | 25-30 | 传奇装备 + 迷宫入口 / Legendary + dungeon access |
| 迷途要塞 / Onyx Watchtower | 干燥平原 / Dry Steppes | 30-35 | 解锁商人 + 工艺材料 / Vendor + crafting mats |
| 阿尔卡努斯要塞 / Alcanus Fortress | 干燥平原 / Dry Steppes | 35-40 | 解锁地下城入口 / Dungeon entrance |
| 庇护所要塞 / Ruins of Rathma | 哈维泽 / Hawezar | 40-45 | 解锁死灵法师隐藏任务 / Necro hidden quest |
| 育巢 / The Nest | 哈维泽 / Hawezar | 45-50 | 解锁梦魇雕塑者 / Nightmare Sculptor |
| 诅咒之城 / Omath's Redoubt | 凯基斯坦 / Kehjistan | 45-50 | 结束传送点 + 成就 / Waypoint + achievement |
| 血红教堂 / Moordaine Lodge | 诅咒之地 / Scosglen | 35-40 | 药水升级材料 / Potion upgrade materials |

**最优路线**: 序章结束后立刻去马诺克要塞——等级20左右可以清理。装备提升后再去科尔萨德拉。全程按区域顺序清理要塞可以获得最大化经验。**第8赛季提示**: 要塞现在每周重置一次——可以重复获得奖励 / **Optimal route**: After prologue, go Malnok immediately (clear at ~20). Then Kor Dragan with better gear. Clear strongholds by zone order for max XP. **S8 tip**: Strongholds now reset weekly — repeatable rewards.

## 地下城目标 / Dungeon Objectives

暗黑4的地下城分为普通地下城、梦魇地下城和特定Boss地下城。每个地下城有特定的目标机制。梦魇地下城是终局的核心内容——通过消耗**梦魇符印**进入——符印有等级和词缀 / Diablo 4's dungeons are split into normal, Nightmare, and boss-specific. Each dungeon has specific objective mechanics. Nightmare Dungeons are the core endgame content — enter via **Nightmare Sigils** — sigils have levels and affixes.

### 地下城目标类型 / Dungeon Objective Types

| 目标类型 / Objective Type | 描述 / Description | 例子 / Example |
|---|---|---|
| 击杀全部精英 / Kill All Elites | 消灭地下城内的所有精英敌人 / Eliminate all elite enemies | 各类地下城 / Various |
| 收集目标物品 / Collect Objective Items | 从指定敌人掉落或场景中找到物品 / Find items from specific enemies or environment | 科尔萨德拉 / Kor Dragan |
| 摧毁目标物 / Destroy Target | 破坏特定结构或机关 / Destroy specific structures or mechanisms | 沉没神庙 / Sunken Temple |
| 护送目标 / Escort Target | 保护NPC通过危险区域 / Guide NPC through danger | 利刃之山 / Bladewardens |
| 击杀Boss / Kill Boss | 击败区域Boss / Defeat zone boss | 所有NMD / All NMDs |

**最快捷清理方法**: 直接冲主线目标，不恋战清理小怪。精英怪如果挡住路顺便击杀。组队可以分头行动加速——一人负责一边。**第8赛季新增梦魇词缀**: 包括"吸血"(敌人攻击恢复生命)、"护盾"(敌人周期性获得护盾)、"暴怒"(敌人伤害随时间增加) / **Fastest clear**: Rush main objective, skip trash mobs. Kill elites that block path. Party can split up. **S8 new NM affixes**: "Vampiric" (enemies heal on hit), "Shielded" (periodic shields), "Enrage" (increasing damage over time).

## 赛季旅程 / Seasonal Journey

每个赛季有独立的赛季旅程——分为七个章节。完成赛季旅程获得赛季奖励、赛季祝福和赛季称号。**第8赛季旅程**围绕魂灵师和符文系统展开 / Each season has an independent Seasonal Journey — seven chapters. Completing the journey grants seasonal rewards, blessings, and titles. **S8 journey** centers on Spiritborn and rune systems.

### 赛季旅程章节 / Seasonal Journey Chapters

| 章节 / Chapter | 完成条件 / Requirements | 主要奖励 / Primary Reward |
|---|---|---|
| 第一章 / Chapter 1 | 完成5个任务——任意地下城+收集材料 / 5 tasks — any dungeon + gather mats | 赛季祝福I / Blessing I |
| 第二章 / Chapter 2 | 完成10个任务——包括传奇装备收集 / 10 tasks — includes Legendary gear | 赛季祝福II + 造型奖励 |
| 第三章 / Chapter 3 | 完成15个任务——世界Boss+梦魇地下城 / 15 tasks — World Boss + NMD | 赛季祝福III + 赛季称号 |
| 第四章 / Chapter 4 | 完成20个任务——攻克要塞 / 20 tasks — conquer stronghold | 赛季宠物 + 赛季祝福IV |
| 第五章 / Chapter 5 | 完成25个任务——终局活动 / 25 tasks — endgame activities | 赛季坐骑 + 造型盔甲 |
| 第六章 / Chapter 6 | 完成30个任务——高难度挑战 / 30 tasks — high difficulty challenge | 赛季终极奖励 / Ultimate reward |
| 终章 / Final Chapter | 击败赛季终局Boss / Defeat season endboss | 赛季成就 + 飞升点数 |

## 世界事件 / World Events

开放世界中的定时事件——参与获得经验、装备和赛季进度。世界事件是快速升级和获取装备的最佳方式之一 / Timed events in the open world — participate for XP, gear, and season progress. World events are among the best ways to level up and gear up quickly.

### 事件类型 / Event Types

**世界Boss**: 每6小时刷新一次。目前有五个世界Boss——阿莎瓦、贪婪、疫灾、瓦尔申的怒火、以及赛季Boss。世界Boss掉落高装等传奇和独特装备。推荐必打——每周前三次奖励提升。**地狱潮**: 每2小时15分钟在指定区域触发。持续60分钟——区域出现地狱裂隙和精英怪物群。收集**畸变余烬**使用宝箱兑换装备。**军团事件**: 定时刷新的群体PVE事件——玩家组队对抗敌人浪潮。完成后获得大量经验和装备 / **World Boss**: Spawn every 6 hours. Five current bosses. Drop high IP Legendary and Unique. Must-do — first three kills per week have bonus rewards. **Helltide**: Triggers every 2h15min in designated zone. Lasts 60 min. Collect **Aberrant Cinders** for chest exchanges. **Legion Event**: Timed group PVE event — huge XP and gear.

### 低语之树 / Tree of Whispers

25级后解锁——完成地图上的标记任务(击杀精英、收集物品、完成事件)获取**恐怖低语**。集满10个可以兑换宝箱——开出装备、材料和梦魇符印。**第8赛季**: 低语之树新增了赛季专属奖励路线——完成赛季任务额外获得低语进度 / Unlocks at 25 — complete marked tasks (kill elites, collect items, complete events) for **Whispers**. At 10 whispers, redeem a cache — gear, mats, Nightmare Sigils. **S8**: New seasonal reward track at Tree of Whispers — seasonal tasks grant bonus whisper progress.

## 终局活动坐骑任务 / Mount Quest

完成**第三章的"坐骑：信仰的飞跃"**任务解锁坐骑战斗和在坐骑上的战斗能力。任务位置在凯基斯坦的"基瓦沙德"接取。完成后获得全账号共享的坐骑能力——包括坐骑冲锋、惊吓敌人、和加速冲刺。**第8赛季新增坐骑能力**: 坐骑可以在空中短暂滑翔——跨越地形障碍 / Complete **Act 3 "Mount: Leap of Faith"** quest to unlock mounted combat. Quest starts at "Ked Bardu" in Kehjistan. Unlocks account-wide mount abilities — including charge, enemy scare, and speed burst. **S8 new mount ability**: brief aerial glide — crossing terrain obstacles.

## 终局任务循环 / Endgame Quest Loop

通关后的终局循环: 刷地狱狂潮获得材料→打低语之树获得宝箱→挑战梦魇地下城提升雕文等级→收集终局Boss召唤材料→挑战终局Boss获得独特装备→精铸毕业装备。**第8赛季新增循环**: 符文收集→符文合成→赛季Boss挑战→获得赛季独特装备 / Endgame loop after campaign: Helltide for mats → Tree of Whispers for caches → Nightmare Dungeons for glyph levels → collect boss summon mats → challenge endgame bosses for uniques → masterwork gear. **S8 new loop**: Rune collection → rune crafting → seasonal boss challenge → seasonal unique gear.
