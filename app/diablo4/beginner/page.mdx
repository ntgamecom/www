# 新手入门 / Beginner Guide

暗黑破坏神4是一款庞大的游戏，但入门并不困难。本章带你从创建角色到打通战役，再到开始终局内容 / Diablo 4 is a massive game, but getting started is straightforward. This guide takes you from character creation through the campaign and into endgame.

---

## 第一步：选择职业 / Step 1: Choose Your Class

暗黑4有6个职业，每个都有独特的玩法和风格 / D4 has 6 classes, each with a unique playstyle:

| 职业 / Class | 玩法风格 / Playstyle | 难度 / Difficulty | 推荐新手？/ New Player? |
|------|------|:----:|:----:|
| **野蛮人 / Barbarian** | 近战猛男，4把武器、战吼和跺脚 / Melee powerhouse with 4 weapons, war cries, and stomps | ★☆☆☆☆ | **强烈推荐 / Highly Recommended** |
| **法师 / Sorcerer** | 远程法爷，火冰雷三系法术 / Ranged caster with Fire/Ice/Lightning magic | ★★☆☆☆ | 推荐 / Recommended |
| **德鲁伊 / Druid** | 变形大师，熊人/狼人/风暴/大地多形态 / Shapeshifter between Werebear/Werewolf/Storm/Earth | ★★★☆☆ | 中等 / Medium |
| **游侠 / Rogue** | 敏捷刺客，远程弓弩+近战双持切换 / Agile assassin with bow+melee weapon swap | ★★★☆☆ | 中等 / Medium |
| **死灵法师 / Necromancer** | 召唤大师，骷髅大军+骨/血/暗影法术 / Summoner with skeleton army + Bone/Blood/Shadow magic | ★★☆☆☆ | 推荐 / Recommended |
| **魂灵师 / Spiritborn** | (资料片职业) 四灵之力，极致速度和爆发力 / (Expansion class) Four spirit powers, extreme speed and burst | ★★★★☆ | 不推荐（需购买资料片）/ Not recommended (needs expansion) |

> **新手建议**：野蛮人或死灵法师是最友好的入门职业。野蛮人简单粗暴，死灵法师有骷髅吸引火力 / **Tip**: Barbarian or Necromancer are the most beginner-friendly. Barbarian is straightforward; Necromancer has skeletons to tank for you.

## 第二步：战役流程 / Step 2: Campaign Walkthrough

### 主线概述 / Campaign Overview

战役分为**序幕 + 8幕**，约15-25小时完成 / The campaign has a Prologue + 8 Acts, roughly 15-25 hours:

| 章节 / Act | 地区 / Region | 主要目标 / Main Goal |
|------|------|------|
| **序章 / Prologue** | 破碎群峰 / Fractured Peaks | 逃出涅瓦克 / Escape Nevesk |
| **第一幕 / Act 1** | 破碎群峰 / Fractured Peaks | 寻找伊纳瑞斯之光 / Seek the Light of Inarius |
| **第二幕 / Act 2** | 斯科斯格伦 / Scosglen | 德鲁伊的秘密 / The Druid's Secret |
| **第三幕 / Act 3** | 干燥平原 / Dry Steppes | 铁狼佣兵团 / The Iron Wolves |
| **第四幕 / Act 4** | 凯基斯坦 / Kehjistan | 憎恨之王墨菲斯托 / Mephisto, Lord of Hatred |
| **第五幕 / Act 5** | 哈维泽 / Hawezar | 赫拉迪姆的传承 / The Horadric Legacy |
| **第六幕 / Act 6** | 地狱 / Hell | 莉莉丝最终决战 / Final Battle with Lilith |
| **第七幕 / Act 7** | 纳汉图 / Nahantu | (资料片) 寻找纳汉图的秘密 / (Expansion) Secrets of Nahantu |
| **第八幕 / Act 8** | 丛林深处 / Jungle Depths | (资料片) 墨菲斯托的阴谋 / (Expansion) Mephisto's Conspiracy |

### 战役技巧 / Campaign Tips

| 技巧 / Tip | 说明 / Description |
|------|------|
| **跟随主线** / Follow Main Quests | 优先做主线任务，支线可以通关后再补 / Prioritize main quests — do side quests after campaign |
| **世界难度保持普通** / Stay on Normal | 战役期间保持"普通"难度，通关后再提升 / Stay on Normal during the campaign — increase after beating it |
| **随时传送** / Use Waypoints | 激活所有传送点，方便快速移动 / Activate all waypoints for fast travel |
| **拾取所有装备** / Pick Up Everything | 前期装备更新快，每件都捡起来对比 / Gear upgrades come fast early on — pick up and compare everything |
| **药水升级** / Upgrade Potions | 记得在药剂师处升级药水容量 / Upgrade potion capacity at the Alchemist |
| **技能重置免费** / Free Respec | 前期重置技能树是免费的——大胆尝试不同技能 / Early respecs are free — experiment with different skills |

## 第三步：快速升级指南 / Step 3: Fast Leveling

### 1-60级升级路线 / Leveling Path (1-60)

| 等级 / Level | 最佳方式 / Best Method |
|:------:|------|
| **1-15** | 序章主线 / Prologue campaign |
| **15-30** | 第一、二幕主线 + 沿途完成要塞 / Act 1-2 campaign + Strongholds along the way |
| **30-45** | 第三、四幕主线 + 世界事件 / Act 3-4 campaign + World Events |
| **45-50** | 第五、六幕主线 / Act 5-6 campaign |
| **50-60** | 通关战役后解锁世界难度3 → 刷梦魇地下城和地狱狂潮 / Unlock World Tier 3 after campaign → run Nightmare Dungeons + Helltide |

### 升级提速技巧 / Leveling Speed Tips

- **头部插红宝石**：头部插槽放入红宝石获得额外经验百分比 / **Ruby in Helm**: socket a Ruby for bonus XP percentage
- **使用经验药水**：药剂师处可以制作经验加成药水 / **Use XP Potions**: craft XP boost potions at the Alchemist
- **参加世界事件**：军团事件和世界BOSS给大量经验 / **Join World Events**: Legion Events and World Bosses give massive XP
- **尽量不死亡**：连续击杀奖励在死亡后重置 / **Avoid Dying**: kill streak bonuses reset on death
- **赛季祝福**：使用赛季币购买经验加成（赛季角色）/ **Season Blessings**: use season currency for XP bonuses (seasonal characters)

## 第四步：通关后做什么 / Step 4: What After Campaign?

| 步骤 / Step | 操作 / Action | 解锁内容 / Unlocks |
|:------:|------|------|
| **1** | 在凯基斯坦的** Cathedral of Light **完成圣光大教堂顶石地下城 / Complete Cathedral of Light capstone dungeon in Kehjistan | **世界难度3：苦痛 / World Tier 3: Nightmare** |
| **2** | 在世界难度3刷地狱狂潮和梦魇地下城，收集装备 / Farm Helltide and Nightmare Dungeons on WT3 for gear | 神圣装备掉落 / Sacred gear drops |
| **3** | 在干燥平原的** Fallen Temple **完成堕落神殿顶石地下城 / Complete Fallen Temple capstone dungeon in Dry Steppes | **世界难度4：折磨 / World Tier 4: Torment** |
| **4** | 在世界难度4解锁全部终局内容——见"终局玩法"章节 / Unlock all endgame content on WT4 — see Endgame chapter | 远古装备掉落 + 终局Boss / Ancestral gear + Endgame Bosses |
