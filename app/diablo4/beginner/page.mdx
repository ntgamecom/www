# 新手入门 / Beginner Guide

暗黑破坏神4是一款庞大的游戏，但入门并不困难。本章带你从创建角色到打通战役，再到开始终局内容 / Diablo 4 is a massive game, but getting started is straightforward. This guide takes you from character creation through the campaign and into endgame.

---

## 第一步：选择职业 / Step 1: Choose Your Class

暗黑4有6个职业，每个都有独特的玩法和风格 / D4 has 6 classes, each with a unique playstyle:

| 职业 / Class | 玩法风格 / Playstyle | 难度 / Difficulty | 推荐新手？/ New Player? |
|------|------|:----:|:----:|
| **野蛮人 / Barbarian** | 近战猛男，4把武器、战吼和跺脚 / Melee powerhouse with 4 weapons, war cries, and stomps | ★☆☆☆☆ | **强烈推荐 / Highly Recommended** |
| **法师 / Sorcerer** | 远程法爷，火冰雷三系法术 / Ranged caster with Fire/Ice/Lightning magic | ★★☆☆☆ | 推荐 / Recommended |
| **德鲁伊 / Druid** | 变形大师，熊人/狼人/风暴/大地多形态 / Shapeshifter between Werebear/Werewolf/Storm/Earth | ★★★☆☆ | 中等 / Medium |
| **游侠 / Rogue** | 敏捷刺客，远程弓弩+近战双持切换 / Agile assassin with bow+melee weapon swap | ★★★☆☆ | 中等 / Medium |
| **死灵法师 / Necromancer** | 召唤大师，骷髅大军+骨/血/暗影法术 / Summoner with skeleton army + Bone/Blood/Shadow magic | ★★☆☆☆ | 推荐 / Recommended |
| **魂灵师 / Spiritborn** | (资料片职业) 四灵之力，极致速度和爆发力 / (Expansion class) Four spirit powers, extreme speed and burst | ★★★★☆ | 不推荐(需购买资料片) / Not recommended (needs expansion) |

> **新手建议**: 野蛮人或死灵法师是最友好的入门职业。野蛮人简单粗暴，死灵法师有骷髅吸引火力 / **Tip**: Barbarian or Necromancer are the most beginner-friendly. Barbarian is straightforward; Necromancer has skeletons to tank for you.

### 职业选择考虑因素 / Class Choice Considerations

- **野蛮人**：4把武器提供更多威能槽位，生存能力极强 / 4 weapon slots = more aspects, excellent survivability
- **法师**：远程输出安全，AOE能力强，但身板脆 / Safe ranged DPS, strong AoE, but squishy
- **德鲁伊**：后期成长性最高，但前期较慢 / Best late-game scaling, but slow early game
- **游侠**：操作要求最高，但伤害上限极高 / Highest skill ceiling, but extreme damage potential
- **死灵法师**：召唤流让战役变得轻松 / Minion build makes campaign a breeze
- **魂灵师**：如果你有资料片，这是第8赛季最强的职业之一 / If you own the expansion, this is one of S8's strongest classes

## 第二步：战役流程 / Step 2: Campaign Walkthrough

### 主线概述 / Campaign Overview

战役分为 **序幕 + 8幕**，约15-25小时完成 / The campaign has a Prologue + 8 Acts, roughly 15-25 hours:

| 章节 / Act | 地区 / Region | 主要目标 / Main Goal |
|------|------|------|
| **序章 / Prologue** | 破碎群峰 / Fractured Peaks | 逃出涅瓦克 / Escape Nevesk |
| **第一幕 / Act 1** | 破碎群峰 / Fractured Peaks | 寻找伊纳瑞斯之光 / Seek the Light of Inarius |
| **第二幕 / Act 2** | 斯科斯格伦 / Scosglen | 德鲁伊的秘密 / The Druid's Secret |
| **第三幕 / Act 3** | 干燥平原 / Dry Steppes | 铁狼佣兵团 / The Iron Wolves |
| **第四幕 / Act 4** | 凯基斯坦 / Kehjistan | 憎恨之王墨菲斯托 / Mephisto, Lord of Hatred |
| **第五幕 / Act 5** | 哈维泽 / Hawezar | 赫拉迪姆的传承 / The Horadric Legacy |
| **第六幕 / Act 6** | 地狱 / Hell | 莉莉丝最终决战 / Final Battle with Lilith |
| **第七幕 / Act 7** | 纳汉图 / Nahantu | (资料片) 寻找纳汉图的秘密 / (Expansion) Secrets of Nahantu |
| **第八幕 / Act 8** | 丛林深处 / Jungle Depths | (资料片) 墨菲斯托的阴谋 / (Expansion) Mephisto's Conspiracy |

### 战役技巧 / Campaign Tips

| 技巧 / Tip | 说明 / Description |
|------|------|
| **跟随主线** / Follow Main Quests | 优先做主线任务，支线可以通关后再补 / Prioritize main quests — do side quests after campaign |
| **世界难度保持普通** / Stay on Normal | 战役期间保持"普通"难度，通关后再提升 / Stay on Normal during the campaign — increase after beating it |
| **随时传送** / Use Waypoints | 激活所有传送点，方便快速移动 / Activate all waypoints for fast travel |
| **拾取所有装备** / Pick Up Everything | 前期装备更新快，每件都捡起来对比 / Gear upgrades come fast early on — pick up and compare everything |
| **药水升级** / Upgrade Potions | 记得在药剂师处升级药水容量 / Upgrade potion capacity at the Alchemist |
| **技能重置免费** / Free Respec | 前期重置技能树是免费的——大胆尝试不同技能 / Early respecs are free — experiment with different skills |
| **完成要塞** / Clear Strongholds | 要塞给大量经验和装备，尽快清理 / Strongholds give massive XP and gear — clear them early |
| **马匹解锁** / Unlock Mount | 完成第四幕"骑乘作战"任务解锁坐骑 / Complete Act 4 "Mount" quest to unlock your horse |

### 资料片战役提示 / Expansion Campaign Tips

如果你购买了憎恨之躯资料片，可以在通关原始战役后或在第一角色通关后直接跳过 / If you own Vessel of Hatred, you can skip the base campaign after your first character:
- **创建新角色时**：选择"跳过战役"直接开始赛季内容 / **New character**: select "Skip Campaign" to jump straight into seasonal content
- **资料片新区域**：纳汉图是独立地图，资料片战役约8-12小时 / **New zone**: Nahantu is a separate map; expansion campaign is ~8-12 hours
- **魂灵师职业**：资料片战役中直接体验魂灵师 / **Spiritborn**: play the new class directly during the expansion campaign

## 第三步：快速升级指南 / Step 3: Fast Leveling

| 等级 / Level | 最佳方式 / Best Method | 预计时间 / Est. Time |

| 技巧 / Tip | 说明 / Description |
|------|------|
| **赛季祝福优先** | 赛季币优先购买经验加成（最多+20%）/ Priority: season XP blessings (up to +20%) |

### 升级提速技巧 / Leveling Speed Tips

- **头部插红宝石**: 头部插槽放入红宝石获得额外经验百分比 / **Ruby in Helm**: socket a Ruby for bonus XP percentage
- **使用经验药水**: 药剂师处可以制作经验加成药水 / **Use XP Potions**: craft XP boost potions at the Alchemist
- **参加世界事件**: 军团事件和世界BOSS给大量经验 / **Join World Events**: Legion Events and World Bosses give massive XP
- **尽量不死亡**: 连续击杀奖励在死亡后重置 / **Avoid Dying**: kill streak bonuses reset on death
- **赛季祝福**: 使用赛季币购买经验加成(赛季角色) / **Season Blessings**: use season currency for XP bonuses (seasonal characters)
- **梦魇地下城**: 50级后刷梦魇地下城是最高效的升级方式 / **NM Dungeons**: post-50, NMDs are the fastest XP source

### 升级时技能加点建议 / Skill Leveling Advice

- **1-25级**: 主加一个核心输出技能，其他点被动 / Focus on one core damage skill, put rest into passives
- **25-40级**: 完善终极技能和相关被动 / Complete ultimate skill and related passives
- **40-50级**: 补强防御类被动，开始规划终局流派 / Fill defensive passives, start planning endgame build
- **50-60级**: 重置技能树并按照终局流派加点 / Respec to final endgame build

## 第四步：装备入门 / Step 4: Gearing Basics

### 装备属性优先级 / Stat Priority by Class

| 职业 / Class | 主属性 / Main Stat | 次属性 / Secondary |
|:-----:|:-----:|:-----:|
| **野蛮人 / Barbarian** | 力量(Strength) | 暴击几率、暴击伤害、冷却缩减 / Crit chance, Crit dmg, CDR |
| **法师 / Sorcerer** | 智力(Intelligence) | 暴击几率、爆伤、法力减耗 / Crit chance, Crit dmg, Mana cost reduction |
| **德鲁伊 / Druid** | 感知(Willpower) | 暴击几率、爆伤、资源生成 / Crit chance, Crit dmg, Resource gen |
| **游侠 / Rogue** | 敏捷(Dexterity) | 暴击几率、爆伤、幸运一击 / Crit chance, Crit dmg, Lucky hit |
| **死灵法师 / Necromancer** | 智力(Intelligence) | 暴击几率、爆伤、召唤伤害 / Crit chance, Crit dmg, Minion dmg |
| **魂灵师 / Spiritborn** | 敏捷(Dexterity) | 暴击几率、爆伤、资源生成 / Crit chance, Crit dmg, Resource gen |

**装备更替节奏**: 1-50级不需要考虑配装——装备白字高就换。50级以后开始关注词缀(暴击、爆伤、主属性)。装备附魔(Enchant)功能在 mystic 处可以从50级开始用——重roll一个不满意的词缀 / **Gear upgrade rhythm**: Level 1-50: equip highest item power. Post-50: start caring about affixes. Enchanting at the Occultist (reroll 1 affix) becomes available around 50 — use to fix one bad roll per item.

### 如何判断装备好坏 / How to Evaluate Gear

- **物品强度(Item Power)**是关键指标——越高越好，直接影响白字伤害 / **Item Power** is key — higher is better, directly affects base damage
- **词缀数量**: 3条词缀好于2条，优先选词缀多的 / **Affix count**: 3 affixes > 2, prioritize items with more affixes
- **词缀搭配**: 确保词缀之间有协同效应（如暴击率+暴击伤害） / **Affix synergy**: ensure affixes work together (e.g., crit chance + crit dmg)
- **插槽**: 有插槽的装备价值更高，可以镶嵌宝石 / **Sockets**: socketed gear is more valuable — gems provide significant bonuses

## 升级与赛季系统 / Paragon & Season System

**巅峰系统(Paragon)**: 50级后解锁——每升一级给4点巅峰点数。在巅峰面板上镶嵌雕文解锁强力加成。雕文需要通过噩梦地下城升级(最高21级，后通过秘域传送门升至50级) / **Paragon**: Unlocks at 50 — 4 points per level. Socket Glyphs on Paragon boards for powerful bonuses. Glyphs level up in Nightmare Dungeons (max 21, then via The Pit to 50).

**赛季系统**: 每3个月一个新赛季——角色从头开始(新赛季专属机制)。赛季角色通关后变为永恒角色 / **Seasons**: Every 3 months — fresh start (season-exclusive mechanics). Season characters roll to Eternal after season ends.

**第8赛季特殊说明**: 第8赛季引入了全新的赛季机制，强调符文和威能的自由组合。赛季角色可以从赛季任务线中获得独特的赛季装备，赛季结束后部分装备会保留 / **S8 Note**: Season 8 introduces a new mechanic focused on free rune and aspect combination. Seasonal characters earn exclusive seasonal gear from the seasonal questline; some items carry over after the season ends.
