## 地狱狂潮 / Helltide

**解锁 / Unlock**: 世界难度3+ / World Tier 3+

地狱狂潮是地图上特定区域的恶魔入侵事件，持续60分钟，结束后切换区域 / Periodic demon invasions covering a specific zone, lasts 60 minutes, then rotates.

### 玩法要点 / How It Works

| 要点 / Tip | 说明 / Description |
|------|------|
| **区域标记 / Zone Marking** | 地图上红色区域即是狂潮区域 / Red zone on map = active Helltide |
| **余烬 / Aberrant Cinders** | 击杀狂潮怪物获得余烬，退出区域后消失 / Kill Helltide mobs for Cinders — they expire when you leave the zone |
| **地狱之箱 / Tortured Gifts** | 用余烬打开的宝箱——有装备、材料、武器、饰品、神秘四类 / Open chests with Cinders — gear, mats, weapons, jewelry, and mystery types |
| **神秘宝箱 / Mystery Chest** | 地图上隐藏位置出现，需要175+余烬。掉率最好的宝箱 / Hidden spawns, costs 175+ Cinders. Best loot per cost |
| **活钢 / Living Steel** | 击败狂潮指挥官获得，用于召唤督瑞尔 / Drops from Helltide Commanders, used to summon Duriel |

### 效率刷法 / Efficient Farming

1. 准备阶段：清空背包，准备传送卷轴 / Prepare: empty inventory, bring scroll of escape
2. 核心策略：找一个**神秘宝箱位置**附近的小怪密集区域反复击杀 / Find a Mystery Chest location and farm dense mob packs nearby
3. 攒够175余烬后开神秘宝箱 → 继续刷 → 收集活钢 / Save 175 Cinders → open Mystery Chest → continue farming → collect Living Steel
4. 狂潮结束后前往**低语之树**提交任务 / After Helltide ends, turn in at Tree of Whispers

## 梦魇地下城 / Nightmare Dungeons

**解锁 / Unlock**: 使用梦魇徽记打开任意普通地下城 / Use a Nightmare Sigil to convert any normal dungeon

梦魇地下城是暗黑4的核心终局玩法之一，随机词缀增加挑战难度和奖励 / One of D4's core endgame activities, with random affixes adding challenge and rewards.

### 梦魇词缀 / Nightmare Affixes

| 词缀 / Affix | 效果 / Effect | 难度评估 / Difficulty |
|------|------|:----:|
| **闪电脉冲 / Lightning Pulse** | 怪物死亡释放闪电脉冲 / Monsters release lightning on death | ★☆☆☆☆ |
| **火山 / Volcanic** | 地面随机喷发火焰 / Random flame eruptions on ground | ★★☆☆☆ |
| **凌寒 / Cold Enchanted** | 精英脚下生成冰霜 / Elites leave frost patches | ★★★☆☆ |
| **死亡冲击 / Death Pulse** | 怪物死亡后延迟造成范围伤害 / Delayed AoE on monster death | ★★★☆☆ |
| **风暴涌动 / Storm Banisher** | 周期性吸附玩家到一处 / Periodically pulls players to a point | ★★★★☆ |
| **瞬移 / Teleporter** | 精英瞬移至玩家身边 / Elites teleport to players | ★★☆☆☆ |

### 雕文系统 / Glyph System

雕文是巅峰盘的核心组件，通过梦魇地下城升级 / Glyphs are key Paragon Board components, upgraded via Nightmare Dungeons.

| 等级范围 / Level Range | 经验来源 / XP Source |
|:----:|------|
| **1-15** | 梦魇地下城（难度3）/ NM Dungeons (Tier 1-30) |
| **15-21** | 梦魇地下城（难度45+）/ NM Dungeons (Tier 45+) |
| **21-46** | 秘域传送门（难度65+） / The Pit (Tier 65+) |
| **46-50** | 秘域传送门（难度100+） / The Pit (Tier 100+) |

**优先升级的雕文 / Priority Glyphs**:
- 根据你的流派选择对应的雕文 / Choose based on your build
- 通用强力雕文：**利用**（对易伤敌人+额外伤害）、**控制**（对被控敌人+额外伤害）/ Universal: Exploit (+dmg vs Vulnerable), Control (+dmg vs CC'd)
- 稀有雕文在半径范围内满足属性要求后可获得额外倍率 / Rare glyphs get bonus multipliers when stat requirements are met within radius

## 秘域传送门 / The Pit

**解锁 / Unlock**: 世界难度4下完成任何梦魇地下城46层 / Complete any Tier 46 Nightmare Dungeon on WT4

秘域传送门是高难度的分层挑战，共有200层 / The Pit is a tiered challenge dungeon with 200 levels.

| 层数 / Tier | 怪物等级 / Monster Level | 推荐用途 / Recommended Use |
|:-----:|:----:|------|
| **1-65** | 100-140 | 雕文升到21级 / Glyph leveling to 21 |
| **65-100** | 140-175 | 精铸材料收集 / Masterworking mat farming |
| **100+** | 175-200 | 高等级雕文升级 / High-level glyph upgrading |
| **150+** | 200+ | 终极挑战/冲层 / Ultimate challenge/pushing ranks |

**奖励 / Rewards**: 精铸石（Obolite）+ 遗忘之魂（Forgotten Souls）+ 雕文经验 / Masterworking materials

## 炼狱大军 / Infernal Hordes

**解锁 / Unlock**: 世界难度4 / World Tier 4

炼狱大军是第5赛季新增的肉鸽模式，在地狱之门中面对一波波敌人 / Wave-based roguelike mode added in Season 5, facing waves of enemies in the Hell Portal.

| 特性 / Feature | 说明 / Description |
|------|------|
| **波次 / Waves** | 选择难度→进行6/8/10波战斗 / Choose difficulty → fight 6/8/10 waves |
| **地狱赠礼 / Infernal Offers** | 每波之间选择增强/buff，也会增加难度 / Between waves, choose buffs that also increase difficulty |
| **BOSS波 / Boss Wave** | 最终波是BOSS战 / Final wave is a boss fight |
| **恶魔玫瑰 / Fiend Rose** | 主要货币，用于地狱狂潮和锻造 / Main currency used in Helltide and crafting |

**最佳刷法 / Best Strategy**: 选择"额外精英"和"移速降低"等负面较小的地狱赠礼，优先选"精英+伤害"增益 / Choose minor negatives (extra elites, slow) — prioritize damage buffs

## 世界BOSS / World Bosses

每天刷新数次的世界级BOSS（Avarice, Ashava, Wandering Death），队伍越大人越多奖励越好 / World bosses spawn several times daily, scale with player count.

| BOSS | 位置 / Location | 特殊技能 / Special Skill |
|------|------|------|
| **阿煞巴 / Ashava** | 破碎群峰 / Fractured Peaks | 毒爪横扫 + AOE毒雾 / Poison claw sweep + AoE poison cloud |
| **贪婪 / Avarice** | 凯基斯坦 / Kehjistan | 吸金（拉人）+ 金币弹幕 / Gold magnet (pull) + coin barrage |
| **流浪死亡 / Wandering Death** | 哈维泽 / Hawezar | 骨牢 + 死亡射线 / Bone prison + death ray |

**刷新时间 / Spawn Times**: 每隔3.5小时刷新一次（具体可在地图上查看）/ Spawns every 3.5 hours (check map for exact times)

## 黑暗堡垒 / Dark Citadel

**解锁 / Unlock**: 资料片憎恨之躯 + 世界难度4 / Vessel of Hatred + WT4

黑暗堡垒是暗黑4首个突袭式终局副本，需要组队完成（2-4人）/ D4's first raid-style endgame dungeon, requires 2-4 players.

| 翅膀 / Wing | 说明 / Description | Boss数 / Bosses |
|------|------|:----:|
| **暗影之翼 / Wing of Shadow** | 潜行+机关解谜 / Stealth + puzzle mechanics | 2 |
| **毁灭之翼 / Wing of Ruin** | 战斗+生存挑战 / Combat + survival challenges | 2 |
| **最终之翼 / Final Wing** | 每周解锁其中一翼 / Rotates weekly | 1 |

**每周奖励 / Weekly Rewards**: 独特装备、幻化外观、每周成就 / Uniques, transmogs, weekly cache
