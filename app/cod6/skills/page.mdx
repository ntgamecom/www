## Perk系统概述 / Perk System Overview

现代战争2有三个Perk槽位——Perk 1、Perk 2、Perk 3。每个槽位下有多个Perk可选。Perk的选择决定了你的玩法风格 / MW2 has 3 Perk slots — Perk 1, Perk 2, Perk 3. Each slot has multiple options. Your Perk choice defines your playstyle.

**解锁方式 / Unlock Method**：Perk通过升级解锁。满级70级解锁全部Perk和Pro版本 / Perks unlock via leveling. Max level 70 unlocks all Perks and Pro versions.

### Pro版本系统 / Pro Version System

每个Perk都有Pro版本——需要完成特定挑战才能解锁。Pro版本提供额外加成 / Each Perk has a Pro version — unlocked via specific challenges. Pro versions give additional bonuses.

### 执法者 / Sleight of Hand

**全游戏最通用的Perk 1**。换弹速度翻倍——在任何武器上都有用 / **Most versatile Perk 1**. 2x reload speed — useful on every weapon.

**Pro挑战 / Pro Challenge**：使用该Perk时完成200次击杀并赢得10场比赛 / Get 200 kills and win 10 matches with this Perk equipped.

**Pro效果 / Pro Effect**：瞄准速度提高30%——对霰弹枪和狙击枪的提升巨大 / ADS speed +30% — huge for shotguns and snipers.

### 马拉松 / Marathon

**冲锋流玩家的首选**。无限冲刺让你可以快速穿越战场，绕后偷袭 / **Rusher's first choice**. Unlimited sprint lets you cross the map fast and flank.

**Pro挑战 / Pro Challenge**：冲刺跑26英里（马拉松距离）/ Sprint 26 miles (marathon distance).

**推荐组合 / Recommended Combo**：马拉松 + 轻装(Perk 2) + 忍者(Perk 3) = 完全无声快速移动 / Marathon + Lightweight(Perk 2) + Ninja(Perk 3) = silent fast movement.

### 威力猛冲 / Stopping Power

**全游戏最重要的Perk**。伤害提升40%——AR/SMG/LMG全都质变 / **Most important Perk in the game**. 40% damage increase — ARs/SMGs/LMGs all become lethal.

**为什么必带 / Why Mandatory**：
- 没有Stopping Power：SCAR-H需要打中身体2枪——有时3枪 / Without SP: SCAR-H takes 2-3 hits to body
- 有Stopping Power：SCAR-H打任何距离都是2枪死身体 / With SP: SCAR-H is always 2 hits to kill
- TTK从0.15秒缩短到0.10秒——生死之差 / TTK drops from 0.15s to 0.10s — life and death

**Pro挑战 / Pro Challenge**：使用该Perk时完成150次爆头击杀 / Get 150 headshot kills with this Perk.

**替代方案 / Alternatives**：只有狙击枪可以不依赖Stopping Power——Intervention自带一枪死的威力 / Only snipers can skip SP — Intervention one-shots naturally.

### 硬汉 / Hardline

**连杀流的核心Perk**。4杀就能出鹞式（原本7杀）/ **Core for killstreak builds**. Get Harrier at 4 kills (normally 7).

**Pro挑战 / Pro Challenge**：使用该Perk时完成50次连杀奖励击杀 / Get 50 killstreak kills with this Perk.

**推荐连杀链 / Recommended Streak**：精确空袭 (3杀) > 掠食者导弹 (4杀) > 鹞式战斗机 (6杀)

## Perk组合策略 / Perk Synergy Strategies

### 全隐形配置 / Invisible Build

冷血(Perk 2) + 忍者(Perk 3) + 消音器 = 完全从雷达上消失 / Cold-Blooded + Ninja + Silencer = complete radar invisibility.

适合模式 / Best for：搜索与摧毁 / Search & Destroy——对方完全无法判断你的位置 / Enemy cannot predict your position.

### 极端火力配置 / Extreme Firepower

危险近战(Perk 2) + 榴弹发射器 + RPG = 一枪清场 / Danger Close + GL + RPG = one-shot squad wipe.

适合模式 / Best for：地面战争 / Ground War——人群密集时一发榴弹杀3-4人 / Tight clusters — one GL kills 3-4.

### 速度配置 / Speed Demon

马拉松(Perk 1) + 轻装(Perk 2) + 战术刀 + 冲锋枪 = 全场最快 / Marathon + Lightweight + Tactical Knife + SMG = fastest build.

适合模式 / Best for：抢旗 / CTF——拿旗后跑回基地无人能追 / Grab flag and outrun everyone.

## 常见误解 / Common Misconceptions

- 冷血让你完全免疫所有连杀奖励？不对——你还是会被鹞式的扫射杀死，只是不会在屏幕上被标记 / Cold-Blooded makes you immune to all killstreaks? No — Harrier bullets still kill you, just no red box on screen.

- Stopping Power对所有武器效果一样？不对——对狙击枪的提升最小（本来就是一枪死），对SMG的提升最大 / SP affects all weapons equally? No — snipers benefit least (already one-shot), SMGs benefit most.

- 马拉松+轻装让你跑得更快？不对——这两个Perk的效果不叠加。轻装加移速，马拉松只给无限冲刺 / Marathon + Lightweight stack for speed? No — they don't stack. Lightweight gives speed, Marathon gives infinite sprint only.
