# 多人对战 / Multiplayer Guide

## 总览 / Overview

MW2的多人模式是COD系列的里程碑——定义了现代FPS对战的标准 / MW2 MP is a COD milestone — defined the modern FPS multiplayer standard.

**核心特色 / Core Features**:
- 16张首发地图 / 16 base maps
- 自定义Class系统（5个槽位）/ 5 custom class slots
- 连杀奖励 / Killstreaks (3/5/7/9/11/15/25 kills)
- 10次转生 / 10 Prestiges (Lv70 cap)
- 大量挑战/称号/emblems / Challenges, titles, emblems

---

## 模式介绍 / Game Modes

| 模式 / Mode | 说明 / Description |
|---|---|
| 团队死斗 / Team Deathmatch (TDM) | 6v6——先到7500分 / First to 7500 points |
| 统治 / Domination | 3个占领点——持续得分 / 3 capture points — score over time |
| 爆破 / Search & Destroy (S&D) | 攻方装弹/守方防爆——单命回合制 / Attackers plant, defenders defuse — single life, rounds |
| 夺旗 / Capture the Flag (CTF) | 抢夺敌方旗帜送回己方 / Capture enemy flag to your base |
| 确认击杀 / Kill Confirmed (KC) | 拾取击杀掉落的狗牌确认分数 / Collect dog tags from kills to score |
| 自由混战 / Free-for-All (FFA) | 8人各自为战——先到30杀 / 8 players, first to 30 kills |
| 据点 / Headquarters | 争夺据点——据点在防守时得分 / Capture HQ, score while defending |
| 地面战争 / Ground War | 9v9大型战斗——地图为Domination/HQ / Large-scale 9v9 |
| 3连杀 / 3rd Person | 第三人称视角——适用于多个模式 / 3rd person camera for select modes |

**核弹Nuke攻略 / Nuke Strategy**:
1. 前提：你需要自己配一个含核弹的Class——25连杀 / Requirement: equip nuke in class — 25-kill streak
2. 推荐配装：ACR消音+冷血+忍者保持隐形 / Recommended: ACR silenced + Cold-Blooded + Ninja
3. 不推荐刻意追求核弹——25连杀在公开比赛中极难 / Not recommended to chase nukes — extremely difficult in public matches
4. 最容易出核弹的模式：地面战争 (9v9, 人多容易叠) / Easiest mode: Ground War (9v9, more targets)

## Perk搭配推荐 / Perk Builds

### 冲锋流 / Rush Build

```
Perk 1: 马拉松 / Marathon —— 无限冲刺 / Unlimited sprint
Perk 2: 轻型 / Lightweight —— 移动速度加快 / Faster movement
Perk 3: 军需官 / Commando —— 近战距离翻倍 / Extended melee range
武器: UMP45消音 / UMP45 silenced 或 AA-12
装备: 震撼弹 + 战术小刀 / Stun + Tactical Knife
```

### 架枪流 / Anchor Build

```
Perk 1: 秃鹫 / Bling —— 两个配件 / 2 attachments
Perk 2: 威力猛冲 / Stopping Power —— 40%伤害加成 / 40% more damage
Perk 3: 稳瞄 / Steady Aim —— 盲射精度提升 / Hip-fire accuracy up
武器: ACR + 红点 + 握把 / ACR + red dot + grip
装备: 阔剑地雷 + 闪光弹 / Claymore + Flashbang
```

### 隐形流 / Stealth Build

```
Perk 1: 秃鹫 / 或 马拉松 / Bling or Marathon
Perk 2: 冷血 / Cold-Blooded —— 不被UAV/热成像发现 / Invisible to UAV/thermal
Perk 3: 忍者 / Ninja —— 无声移动 / Silent movement
武器: SCAR-H消音 / SCAR-H silenced
装备: 破片手雷 + 烟雾弹 / Frag + Smoke
连杀: UAV → 隐形轰炸 → EMP
```

### 狙击流 / Sniper Build

```
Perk 1: 秃鹫 / Bling —— 两个狙击配件 / 2 sniper attachments
Perk 2: 威力猛冲 / Stopping Power —— 一枪必杀保证 / Guaranteed one-shot kill
Perk 3: 稳瞄 / Steady Aim —— 切枪快 / Faster weapon swap
武器: Intervention + 热成像 + 心跳感应 / Intervention + thermal + heartbeat
     副手: SPAS-12 / Secondary: SPAS-12
装备: 阔剑地雷 + 烟雾弹 / Claymore + Smoke
```
