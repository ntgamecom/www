# 特别行动 / Special Ops

## 总览 / Overview

特别行动是双人合作模式——23个挑战分布在三个难度等级 / Co-op mode — 23 challenges across 3 tiers.

| 等级 / Tier | 数量 / Count | 单人难度 / Solo Difficulty | 推荐 / Recommendation |
|---|---|---|---|
| Alpha | 15 | ★★ | 可单人完成 / Soloable |
| Bravo | 6 | ★★★★ | 强烈推荐双人 / Co-op strongly recommended |
| Charlie | 2 | ★★★★★ | 必须双人 / Co-op required |

---

## Alpha级别 (15个)

### 1. 雪地狙击 / Sniper Fi

**类型 / Type**: 潜行狙击 / Stealth Sniper
**难度 / Difficulty**: ★★

**攻略 / Strategy**:
1. 使用提供的.50cal狙击枪——一枪一个 / Use the .50cal — one shot one kill
2. 从远到近依次狙杀 / Snipe from farthest to nearest
3. 不要被发现——被发现了警报一响任务失败 / Detection = instant fail
4. 双人时一人一边分工 / Split left/right in co-op

---

### 2. 清理屋顶 / Clear Roofs

**类型 / Type**: 清理房间 / Room Clear
**难度 / Difficulty**: ★★

**攻略 / Strategy**:
1. 用消音武器击杀屋顶守卫 / Use suppressed weapons for roof guards
2. 进入建筑后逐层清理 / Clear building floor by floor
3. 注意楼梯口的敌人 / Watch stairwells
4. 最后屋顶直升机撤离 / Extract via rooftop chopper

---

### 3. 疏散 / Evasion

**类型 / Type**: 生存逃亡 / Survival Escape
**难度 / Difficulty**: ★★

**攻略 / Strategy**:
1. 初始位置有大量敌人——快速击杀突围 / Kill quickly from starting position
2. 沿着小巷向撤离点推进 / Advance along alleys to extraction
3. 不要停留超过5秒——敌人会无限刷 / Don't stay longer than 5 seconds — infinite spawns
4. 到达撤离点后防守30秒 / Defend extraction point for 30 seconds

---

### 4. 营救 / Rescue

**类型 / Type**: 人质营救 / Hostage Rescue
**难度 / Difficulty**: ★★★

**攻略 / Strategy**:
1. 潜行到建筑门口——一击必杀门口卫兵 / Stealth to building entrance
2. 进入后快速控制房间——人质在二楼 / Secure the room — hostage on 2nd floor
3. 带人质撤离——人质跑得慢注意掩护 / Escort hostage out — cover them
4. 人质不能死——死了任务失败 / Hostage dies = mission fail

---

### 5. 营地防御 / Estate Defense

**类型 / Type**: 阵地防守 / Area Defense
**难度 / Difficulty**: ★★

**最好的单人刷经验关** / **Best solo XP farm**

**攻略 / Strategy**:
1. 站在二楼窗户旁——看得到所有刷新点 / Stand by 2nd floor window — full spawn vision
2. 推荐武器：SCAR-H + 榴弹发射器 / SCAR-H + grenade launcher
3. 敌人从三个方向进攻：正面、左翼、右翼 / Enemies from 3 directions: front, left, right
4. 最后阶段大量敌人涌出——用榴弹和手雷 / Final wave: lots of enemies — use grenades and launcher
5. 双人时一人守二楼一人守一楼 / Co-op: one upstairs, one downstairs

---

### 6. 撞车 / Crash

**类型 / Type**: 生存 / Survival
**难度 / Difficulty**: ★★☆

**攻略 / Strategy**:
1. 利用车辆残骸做掩体 / Use wrecked cars for cover
2. 优先击杀RPG兵 / Priority: RPG soldiers
3. 从侧面绕后远比正面硬冲有效 / Flanking is far more effective than pushing head-on
4. 清理完机场后到指定位置撤离 / Clear airport then extract

---

### 7. 天空之王 / King of the Sky

**类型 / Type**: AC-130炮艇 / AC-130 Gunship
**难度 / Difficulty**: ★★

**攻略 / Strategy**:
1. 这是固定炮艇射击关——不用动 / Fixed gunship position — no movement
2. 25mm炮清小兵 / 25mm for infantry
3. 40mm打车辆 / 40mm for vehicles
4. 105mm打建筑和装甲 / 105mm for buildings and armor
5. 优先打RPG兵和防空炮 / Priority: RPG soldiers and anti-air

---

### 8. 隐蔽行动 / Covert Ops

**类型 / Type**: 潜行 / Stealth
**难度 / Difficulty**: ★★★

**攻略 / Strategy**:
1. 全程不能触发警报 / No alarms triggered
2. 用消音武器逐屋清理 / Suppressed weapons, clear room by room
3. 摄像头打掉——否则会触发警报 / Destroy cameras — they trigger alarms
4. 从外围开始逐步深入 / Work from perimeter inward

---

### 9. 大停电 / Blackout

**类型 / Type**: 夜视战斗 / Night Vision Combat
**难度 / Difficulty**: ★★☆

**攻略 / Strategy**:
1. 全程使用夜视仪 / Night vision throughout
2. 消音武器推荐 / Suppressed weapons recommended
3. 电力设施破坏后敌人会恐慌——更容易击杀 / Enemies panic after power out — easier kills
4. 最后在发电机处防守 / Defend the generator at the end

---

### 10. 安全第一 / Safety First

**类型 / Type**: VIP保护 / VIP Protection
**难度 / Difficulty**: ★★★

**攻略 / Strategy**:
1. VIP很脆——不要让他冲前面 / VIP is fragile — keep him behind you
2. 清理房间后再让VIP进入 / Clear rooms before VIP enters
3. 室外段——烟雾弹掩护VIP过街 / Outdoors: smoke grenade cover for VIP
4. VIP到撤离点即完成任务 / Reach extraction with VIP alive

## Bravo级别 (6个)

### 16. 武装号召 / Armored Assault

**类型 / Type**: 阵地防守 / Heavy Defense
**难度 / Difficulty**: ★★★★

**攻略 / Strategy**:
1. 一人带轻机枪架左路，一人带RPG守右路 / One LMG left, one RPG right
2. 装甲车必须用RPG/RPG炸毁 / RPGs for armored vehicles
3. 弹药箱在地图中央——定期补充 / Ammo crate in center — restock regularly
4. 最后两波最疯狂——预留所有重火力 / Final 2 waves are insane — save heavy weapons
5. **关键**：换弹间隙用手枪——不浪费换弹时间 / Swap to pistol during LMG reload

### 20. 云端俱乐部 / Mile High Club

**类型 / Type**: 限时突袭 / Timed Assault
**难度 / Difficulty**: ★★★★★

**全球最难挑战**——1分30秒完成 / **Hardest challenge in the game** — 1:30 time limit.

**攻略 / Strategy**: 参见战役关卡"Mile High Club"攻略——完全相同的流程 / Same as campaign mission "Mile High Club"

## Charlie级别 (2个)

### 22. 断翼 / Wingless

**类型 / Type**: 生存突击 / Survival Assault
**难度 / Difficulty**: ★★★★★

**攻略 / Strategy**:
1. 在飞机残骸处防守——四面八方都有敌人 / Defend at plane wreckage — enemies from all sides
2. 分好方向——一人守一边 / Split directions — one per side
3. 推荐武器：RPD + AA-12（双持）/ RPD + AA-12 (akimbo)
4. 最后一波有3个重甲兵——霰弹枪集火 / Final wave: 3 juggernauts — shotgun focus fire

### 23. 自毁 / Self Destruct

**类型 / Type**: 时间竞速 / Time Trial
**难度 / Difficulty**: ★★★★★

**攻略 / Strategy**:
1. **限时2分钟**——分头行动 / 2-minute limit — split up
2. 一人去控制室解除安保 / One person to control room for security disable
3. 另一人直奔核心放置炸弹 / Other person to core with bomb
4. 碰头后一起撤离——15秒撤离窗口 / Meet and extract — 15-second extraction window
5. 必须语音沟通——没有时间打字 / Voice comms required — no time to type
