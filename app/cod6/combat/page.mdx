## 战斗基础 / Combat Basics

现代战争2的战斗节奏极快——TTK（击杀所需时间）极短，0.2秒内决定生死。掌握核心机制是将老兵难度变成普通难度的关键 / MW2 has the fastest TTK in the series — 0.2 seconds decides life or death. Master the core mechanics to turn Veteran into Normal.

### 生命值与回复 / Health & Regen

| 机制 / Mechanic | 说明 / Description |
|---|---|
| 生命值 / Health | 无血条——屏幕越红受伤越重 / No HP bar — screen redness indicates damage |
| 回复机制 / Regen | 脱离战斗5秒后自动回复至满血 / Auto-regen to full after 5 seconds out of combat |
| 秒杀阈值 / One-Shot | 狙击枪和霰弹枪近距离可一枪毙命 / Snipers and shotguns can one-shot at close range |
| 爆头倍率 / Headshot | 爆头伤害x1.4——手枪爆头也能秒杀 / Headshot does 1.4x — pistol headshots can kill |

### 移动机制 / Movement Mechanics

**核心移动技巧 / Core Movement Tips**:

- **滑铲 / Slide**：冲刺+蹲下——短距离快速平移，穿越掩体间隙的最佳方式 / Sprint + crouch — fast short-distance traversal
- **跳跃射击 / Jump Shot**：跳起的同时射击——可越过矮墙开枪，但精准度下降 / Jump while firing — shoot over low cover with reduced accuracy
- **快跑切手枪 / Sprint Pistol**：冲刺时切手枪移速最快——转点必备 / Pistol while sprinting = max movespeed — essential for repositioning
- **蹲走 / Crouch Walk**：无声移动，不显示在小地图上 / Silent movement — no minimap ping
- **架枪 / Mount**：瞄准状态下靠近掩体边缘可按G架枪——减少后座 / ADS near cover edge — press G to mount — reduces recoil

## 老兵难度通关技巧 / Veteran Difficulty Tips

### 生存法则 / Survival Rules

1. **不要站在开阔地超过2秒** / Never stand in the open for more than 2 seconds
2. **烟雾弹是你的最佳朋友**——遮挡AI的透视瞄准 / Smoke grenades are your best friend — block AI aimbot vision
3. **闪身射击 / Peek Shooting**：从掩体侧面快速探身射击后缩回——对AI极其有效 / Quick peek from cover — extremely effective against AI
4. **手雷反弹投掷 / Bank Grenade**：利用墙角/门框反弹手雷——不露头就能杀人 / Bounce grenades off walls — kill without exposing yourself
5. **AI触发机制 / Trigger AI**：敌人的刷新是触发式的——知道自己触发了多少波敌人很重要 / Enemy spawns are trigger-based — know how many waves you triggered

### 特定关卡老兵攻略 / Mission-Specific Veteran Tips

**攀岩者 / Cliffhanger**：全程潜行——被发现就重来。沿着左侧悬崖走，逐个暗杀哨兵 / Full stealth — restart on detection. Follow left cliff, assassinate guards one by one.

**不许说俄语 / No Russian**：你不需要开枪——跟着马卡洛夫走即可通关。如果你开枪，FBI会记住你的选择 / You don't need to fire — just follow Makarov. If you shoot, the FBI remembers your choice.

**白宫防守 / White House Defense**：这是全游戏最难的老兵关卡——敌人无限刷，必须快速推进到指定位置。使用烟雾弹封住街道，沿着右侧走 / The hardest Veteran mission — infinite enemy spawns, must push fast. Smoke the street, follow right wall.

**古拉格 / Gulag**：监狱地下搏斗——用SPAS-12霰弹枪打。第一发没死就补刀 / Underground prison fight — use SPAS-12 shotgun. Follow up if first shot doesn't kill.

**结束战争 / Endgame**：最后的快艇追逐战——优先打RPG兵和直升机。QTE按键不能按错 / Final speedboat chase — prioritize RPG soldiers and helicopters. Don't miss QTE prompts.

## 专业战术 / Advanced Tactics

### 快速装弹取消 / Reload Cancel

MW2中换弹完成的一瞬间按瞄准——可以取消换弹动画的后摇，节省0.5秒 / Tap ADS the moment the mag clicks in — cancels the reload animation tail, saving ~0.5s.

### 双武器配置 / Class Building Principles

**职业类型 / Class Types**：
- **突击 / Assault**：SCAR-H + 消音 + Stopping Power——全距离通用 / SCAR-H + Silencer + Stopping Power — all-range
- **狙击 / Sniper**：Intervention + 快手 + 稳瞄——远程压制 / Intervention + Sleight of Hand + Steady Aim — long range
- **冲锋 / Rusher**：UMP45 + 马拉松 + 威力猛冲——激进打法 / UMP45 + Marathon + Stopping Power — aggressive
- **支援 / Support**：RPD + 握把 + 快手——火力压制 / RPD + Grip + Sleight of Hand — suppressive fire

### 小地图意识 / Minimap Awareness

- 开枪不消音会在小地图上显示红点 / Unsilenced fire shows red dots on minimap
- UAV激活时全队看到敌人位置 / UAV active = all enemies visible on minimap
- 掠食者导弹使用期间小地图失效 / Predator missile use = minimap disabled

## 枪械后座控制 / Recoil Control

MW2的枪械后座是固定的——掌握压枪节奏就能让子弹全部落在准星上 / MW2 recoil patterns are fixed — master the pull timing to keep shots on target.

**压枪技巧 / Pull Techniques**：
- SCAR-H：前5发垂直上跳——轻微向下压即可 / First 5 rounds vert jump — slight downward pull
- ACR：几乎无后座——适合新手 / Almost zero recoil — best for beginners
- UMP45：垂直后座大——需要持续向下压 / Large vertical recoil — constant downward pull
- Vector：水平散布——不是垂直而是左右跳 / Horizontal spread — not vertical but left-right

**配件对后座的影响 / Attachment Recoil Effects**：
- 握把 / Grip：减少垂直后座20% / Reduces vertical recoil by 20%
- 消音器 / Silencer：减少伤害衰减/增加后座5% / Reduces damage drop-off / +5% recoil
- 榴弹发射器 / GL：增加重量——减少后座但降低开镜速度 / Added weight — less recoil but slower ADS
