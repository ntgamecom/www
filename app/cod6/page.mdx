| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **老兵难度分段打 / Veteran in Segments** | 老兵难度下某些检查点极难——建议每次过一个检查点就休息 / Some checkpoints are brutal on Veteran — pace yourself |
| **2** | **闪光弹是神 / Flashbangs Are Life** | 老兵难度下闪光弹可以让你安全通过绝大多数死亡区域 / Flashbangs clear most death zones on Veteran |

## 难易度说明 / Difficulty Guide

| 难度 / Difficulty | 说明 / Description |
|---|---|
| 简单 / Recruit | 敌人反应慢、伤害低——适合剧情向玩家 / Slow enemies, low damage — for story |
| 普通 / Regular | 标准体验 / Standard experience |
| 困难 / Hardened | 敌人伤害和AI提升——推荐首次游玩 / Buffed enemies — recommended first playthrough |
| 老兵 / Veteran | 一击必杀——部分关卡极其考验耐心和技巧 / Near-one-hit kills — some missions are brutal tests of patience |

**老兵难度核心技巧 / Veteran Core Tips**:
1. **不要露头超过2秒**——敌人精准度极高 / Don't peek for more than 2 seconds — enemies are deadly accurate
2. **闪光弹开路**——每个转角先扔闪光 / Flash every corner
3. **刷敌机制**——COD的敌人会无限刷到玩家推进到某个位置为止 / Enemies spawn infinitely until you reach a trigger point — keep moving forward
4. **霰弹枪+盾牌**——某些狭窄关卡用防暴盾+手枪组合通过 / Riot shield + pistol for tight corridors
5. **烟雾弹掩护**——开阔地烟雾弹掩护推进 / Smoke grenades for open area advances
