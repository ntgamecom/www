# 新手入门 / Beginner Guide

## 剧情背景 / Story Background

现代战争2的时间设定在MW1结束后的5年。马卡洛夫策划了机场恐怖袭击，美俄全面开战——141特遣队必须在世界毁灭前阻止战争 / Set 5 years after MW1. Makarov orchestrates an airport massacre, triggering all-out war between US and Russia — Task Force 141 must stop the conflict before the world burns.

**主要角色 / Main Characters**:
- **肥皂 / Soap MacTavish** — 141特遣队队长 / TF141 Captain
- **普莱斯 / Price** — 141特遣队传奇老兵 / Legendary TF141 soldier
- **幽灵 / Ghost** — 141特遣队突击手 / TF141 assault
- **谢菲尔德 / Shepherd** — 美军总指挥官 / US Army Commander
- **马卡洛夫 / Makarov** — 恐怖分子头目 / Terrorist mastermind
- **艾伦 / Allen** — CIA卧底 / CIA operative (玩家控制 / Player-controlled)

### 老兵难度操作技巧 / Veteran Control Tips

1. **冲刺+滑铲**：冲刺中按蹲下可以做短距离滑铲动作——快速穿越开阔地 / Sprint + crouch = short slide — cross open areas fast
2. **盲射优于瞄准**：近距离遭遇战直接开枪比开镜更快——老兵难度下这0.5秒就是生死 / Hip-fire is faster than ADS — that 0.5s saves your life on Veteran
3. **换弹癌要治**：不要每次都换弹——子弹够用就别换，被换弹打断的瞬间够敌人杀你10次 / Don't obsessively reload — dying mid-reload is the #1 COD mistake
4. **快跑切手枪**：冲刺时切换到手枪移动速度最快 / Sprint with pistol out for maximum movespeed

## 特别行动模式指引 / Spec Ops Guide

**解锁条件 / Unlock**: 完成战役前两关即可进入 / Available after first 2 campaign missions.

**难度分级 / Tier System**:
- **Alpha (15个)**: 简单——单人可完成 / Easy — soloable
- **Bravo (6个)**: 中等——推荐双人 / Medium — co-op recommended
- **Charlie (2个)**: 极难——必须双人+完美配合 / Very hard — needs co-op + perfect coordination

**核心提示 / Core Tips**:
1. Alpha难度的所有任务都可以单人刷——积累经验解锁武器 / All Alpha missions are soloable — grind XP for weapon unlocks
2. "营地防御"和"雪地狙击"是单刷最快的经验来源 / "Estate Defense" and "Sniper Fi" are fastest XP farms
3. Charlie难度的"武装号召"需要至少一人带**轻机枪+全技能** / "Armored Assault" needs at least one LMG + all perks
