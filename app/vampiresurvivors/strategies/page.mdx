# 套路策略 / Strategies Guide — 吸血鬼幸存者 / Vampire Survivors

{/* 构筑类型、武器协同、阿卡纳系统、诅咒管理与进阶玩法 / Build archetypes, weapon synergies, Arcanas system, curse management, and advanced gameplay */}

当新手期过后，你会发现吸血鬼幸存者的策略深度令人难以置信。不同的武器组合、阿卡纳选择、诅咒值管理，都会带来完全不同的游戏体验。本页面将为进阶玩家提供全面的策略分析。

After the beginner phase, you will discover the incredible strategic depth of Vampire Survivors. Different weapon combinations, Arcana choices, and curse value management create completely different gameplay experiences. This page provides comprehensive strategy analysis for advanced players.

---

## 构筑类型 / Build Archetypes

### 1. 范围毁灭型 / Area Destruction Build

| 组件 / Component | 推荐选择 / Recommended |
|:---|:---|
| 核心武器 / Core | 圣水 + 斧头 + 闪电戒指 |
| 进化武器 / Evolved | La Borra + Death Spiral + Thunder Loop |
| 关键道具 / Key Items | 烛台 Candelabrador, 复制器 Duplicator, 菠菜 Spinach |
| 阿卡纳 / Arcana | XX - Silent Old Sanctuary |
| 强度 / Power | S 级，适合所有关卡 |

策略：以大范围持续伤害为核心，走位要求低，清怪效率极高。圣水之池和死亡旋风的组合几乎覆盖整个屏幕。

Strategy: Core is large area持续 damage, low positioning requirements, extremely high clearing效率. La Borra and Death Spiral combo covers almost the entire screen.

### 2. 弹幕环绕型 / Orbital Barrage Build

| 组件 / Component | 推荐选择 / Recommended |
|:---|:---|
| 核心武器 / Core | 国王圣经 + 十字架 + 魔法杖 |
| 进化武器 / Evolved | Unholy Vespers + Heaven Sword + Holy Wand |
| 关键道具 / Key Items | 空白之书 Empty Tome, 烛台 Candelabrador |
| 阿卡纳 / Arcana | I - Gemini (双倍环绕) |
| 强度 / Power | A 级，适合近战防御 |

策略：利用环绕武器的击退效果在角色周围形成防护圈，同时十字架和魔法杖提供远程输出。对近战敌人有极强压制力。

Strategy: Use orbital weapons' knockback to create a protective ring, while Cross and Magic Wand provide ranged damage. Extremely effective against melee enemies.

### 3. 投射物海啸型 / Projectile Tsunami Build

| 组件 / Component | 推荐选择 / Recommended |
|:---|:---|
| 核心武器 / Core | 飞刀 + 火炬 + 符文追踪器 |
| 进化武器 / Evolved | Thousand Edge + Hellfire + NO FUTURE |
| 关键道具 / Key Items | 复制器 Duplicator, 护腕 Bracer, 菠菜 Spinach |
| 阿卡纳 / Arcana | II - Twilight Requiem |
| 强度 / Power | A 级，高爆发但需要走位 |

策略：大量投射物覆盖全屏，配合高弹道速度瞬间清场。需要一定的走位技巧来保证安全。

Strategy: Massive projectiles覆盖 full screen, with high projectile speed for instant clearing. Requires some positioning skill for safety.

### 4. 诅咒挑战型 / Curse Challenge Build

| 组件 / Component | 推荐选择 / Recommended |
|:---|:---|
| 核心武器 / Core | 五芒星 + 时钟柳叶刀 + 月桂叶 |
| 进化武器 / Evolved | Gorgeous Moon + Infinite Corridor + Crimson Shroud |
| 关键道具 / Key Items | 托罗纳魔盒 Torrona's Box, 王冠 Crown |
| 阿卡纳 / Arcana | XV - Disco of Gold |
| 强度 / Power | S 级，高手向，高回报 |

策略：使用高诅咒值（通过托罗纳魔盒和诅咒角色）来大幅增加经验和金币收益。需要极强的生存和反应能力。

Strategy: Use high curse values (via Torrona's Box and curse characters) to massively increase XP and gold rewards. Requires strong survival and reaction abilities.

## 武器协同 / Weapon Synergies

### 顶级协同组合 / Top Synergy Combos

| 组合 / Combo | 协同效果 / Synergy Effect |
|:---|:---|
| La Borra + Unholy Vespers | 地面毒池 + 环绕防护，近身远程兼备 |
| Death Spiral + Thunder Loop | 大范围击退 + 全屏雷击，完美覆盖 |
| Hellfire + Heaven Sword | 弹射火球 + 穿透飞剑，密集敌群瞬间瓦解 |
| Infinite Corridor + Crimson Shroud | 秒杀 + 无敌，击败死神的标准配置 |
| Holy Wand + Bloody Tear | 高频率自动攻击 + 生命偷取，续航之王 |

### 反协同 / Anti-Synergy (避免的组合)

- 五芒星 + 任何需要经验宝石的构筑（未进化前） / Pentagram + any build needing XP gems (before evolution)
- 时钟柳叶刀 + 低冷却构筑（会频繁冻结，影响输出节奏） / Clock Lancet + low cooldown builds (frequent freezing disrupts rhythm)
- 斧头 + 极近战构筑（范围重叠降低效率） / Axe + extremely close-range builds (area overlap reduces efficiency)

## 阿卡纳系统 / Arcana System

### 阿卡纳解锁 / Arcana Unlock

阿卡纳需要先获得遗物 Randomazzo（在 Dairy Plant），然后可以通过关卡中的棺材解锁具体的阿卡纳。每局最多可携带 3 个阿卡纳。

Arcanas require first obtaining the Randomazzo relic (in Dairy Plant), then unlocking specific Arcanas via coffins in stages. Maximum 3 Arcanas per run.

| II - Twilight Requiem | 弹射+1 / Bounce +1 | 投射物型 / Projectile |
| III - Tragic Princess | 击退方向向自己 / Knockback toward self | 近战型 / Melee |
| IV - Awake | 复活 +2 / Revive +2 | 生存型 / Survival |
| V - Chaos | 武器随机进化 / Random weapon evolution | 娱乐型 / Fun |
| VI - Sarabande of Healing | 治疗时造成伤害 / Heal deals damage | 吸血型 / Life steal |
| VII - Iron Blue Will | 冻结时造成伤害 / Freeze deals damage | 控制型 / Control |
| VIII - Mad Groove | 磁铁吸引物品 / Magnet attracts items | 通用 / Universal |
| IX - Divine Bloodline | 护甲增加伤害 / Armor increases damage | 护甲型 / Armor |

- 开局选择对未来构筑最关键的阿卡纳 / Choose the most critical Arcana for your future build at start
- 11 分钟宝箱选择第二个阿卡纳 / Choose second Arcana from 11-minute chest
- 21 分钟宝箱选择第三个阿卡纳 / Choose third Arcana from 21-minute chest
- 阿卡纳选择应该补足构筑的短板 / Arcana choice should complement your build's weaknesses

## 诅咒管理 / Curse Management

### 诅咒值来源 / Curse Sources

| 来源 / Source | 诅咒值 / Curse Amount |
|:---|:---|
| Torrona's Box Lv5 | +40% |
| 诅咒角色被动 / Curse character passive | 每级 +1% |
| 阿卡纳 Wicked Season | +? 随时间增加 |
| Hyper 模式 / Hyper mode | 关卡固定加成 |

### 诅咒管理原则 / Curse Management Principles

- 练级时适当增加诅咒，获得更多敌人和经验 / Add some curse when grinding for more enemies and XP
- 挑战高难关卡时控制诅咒在可接受范围 / Control curse within acceptable range for hard stages
- 诅咒 = 更多敌人 = 更难生存 = 更多经验 / Curse = more enemies = harder survival = more XP
- 新手建议诅咒值不超过 +20% / Beginners recommended curse no more than +20%
- 进阶玩家可以管理 +40% 到 +60% 诅咒值 / Advanced players can manage +40% to +60% curse

## 无尽模式 / Endless Mode

### 解锁条件 / Unlock Condition

通关 Cappella Magna 后解锁无尽模式选项。在关卡选择界面可以切换。

Endless mode is unlocked after clearing Cappella Magna. Toggle it in the stage selection screen.

### 无尽模式策略 / Endless Mode Strategy

- 优先选择无限成长的角色（Pugnala, Sigma） / Prioritize infinite-scaling characters (Pugnala, Sigma)
- 构筑必须包含至少一个治疗来源 / Build must include at least one healing source
- 圣水之池 La Borra 是无尽模式的 MVP / La Borra is the MVP of endless mode
- 阿卡纳选择 XIII - Wicked Season 可以随时间无限成长 / Arcana XIII - Wicked Season provides infinite scaling over time
- 建议在 30 分钟后使用时钟柳叶刀控制场面 / Use Clock Lancet to control crowds after 30 minutes
- 深红裹尸布 Crimson Shroud 在无尽模式后期至关重要 / Crimson Shroud is crucial in late endless mode

## Hyper 模式策略 / Hyper Mode Strategy

| 阶段 / Phase | 时间 / Time | 策略 / Strategy |
|:---|:---|:---|
| 前期 / Early | 0-11 min | 快速升级，确定核心武器方向 |
| 中期 / Mid | 11-21 min | 完成主要进化，开始补充辅助武器 |
| 后期 / Late | 21-30 min | 纯生存模式，围绕安全区域移动 |
| 终局 / Endgame | 30 min | 面对死神，使用无限回廊或深红裹尸布 |

### Hyper 模式注意事项 / Hyper Mode Tips

- 敌人移动速度 +50%，走位节奏加快 / Enemy speed +50%, positioning rhythm accelerates
- 经验获取 +100%，升级更快但危险更大 / XP +100%, faster leveling but more danger
- 建议至少有 2 个超武后再挑战 / Recommended to have at least 2 ultra weapons before challenging
- 阿卡纳 IV - Awake 在 Hyper 模式中价值大增 / Arcana IV - Awake is much more valuable in Hyper mode
