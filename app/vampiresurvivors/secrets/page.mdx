# 隐藏内容 / Secrets Guide — 吸血鬼幸存者 / Vampire Survivors

{/* 隐藏角色、秘密关卡、Sigma、游戏修改器、代码输入完整指南 / Complete guide to secret characters, hidden stages, Sigma, game modifiers, and code inputs */}

吸血鬼幸存者以其丰富的隐藏内容著称。从隐藏角色到秘密关卡，从游戏修改器到彩蛋代码，这些内容为游戏增加了大量的可探索性。本页面将揭露所有已知的秘密。

Vampire Survivors is famous for its rich hidden content. From secret characters to hidden stages, from game modifiers to easter egg codes, these contents add massive explorability to the game. This page reveals all known secrets.

## 隐藏角色 / Secret Characters

### Exdash Exiviiq

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 解锁方式 / Unlock | 主菜单输入代码 x-x1viiq / Enter code x-x1viiq at main menu |
| 初始武器 / Starting Weapon | 无（但超幸运会很快掉落武器） / None (但超 luck drops weapons quickly) |
| 被动加成 / Passive | 所有属性 -50%，幸运 +100% / All stats -50%, Luck +100% |
| 特点 / Trait | 最弱的角色，也是最强的角色（看脸） |

Exdash 是一个恶搞角色。所有属性大幅降低，但幸运极高。游戏中掉落武器和宝箱的概率大幅提高。非极致运气不可玩。

Exdash is a joke character. All stats are significantly reduced but luck is extremely high. Weapon and chest drop rates are greatly increased. Not playable without extreme luck.

#### 获取方法 / How to Unlock

在主菜单画面快速输入：上上下下左右左右 ESC ENTER（不同版本有差异）。或者直接输入代码 x-x1viiq。

At the main menu, quickly input: up up down down left right left right ESC ENTER (varies by version). Or directly enter code x-x1viiq.

### Toastie

| 解锁方式 / Unlock | 主菜单输入代码 tramezzini / Enter code tramezzini at main menu |

| 特点 / Trait | 玻璃大炮，满级时伤害极高 |

Toastie 是另一个彩蛋角色。只有 1 点生命，碰一下就死，但每升一级伤害增加 1%。满级（约 200 级）时拥有 200% 以上额外伤害。

Toastie is another easter egg character. Only 1 HP, dies in one hit, but gains +1% damage per level. At max level (about 200), has over 200% bonus damage.

#### 使用策略 / Usage Strategy

- 必须使用月桂叶 Laurel 或深红裹尸布 Crimson Shroud / Must use Laurel or Crimson Shroud

- 推荐使用阿卡纳 IV - Awake 增加容错 / Recommended: Arcana IV - Awake for more room for error
- 前期极度困难，后期极度强大 / Extremely difficult early, extremely powerful late

## 隐藏关卡 / Hidden Stages

### 圣殿 / Holy Forbidden

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 解锁方式 / Unlock | 使用 Exdash 在 Mad Forest 中找到 / Find with Exdash in Mad Forest |
| 地图特点 / Features | 黑暗地图，视野极窄，敌人极强 |
| 推荐策略 / Strategy | 使用高生存构筑，阿卡纳推荐 IV - Awake |

Holy Forbidden 是一个隐藏的挑战关卡。视野被限制在一个小范围内，敌人从黑暗中不断涌出。需要极高的反应和走位能力。

Holy Forbidden is a hidden challenge stage. Vision is limited to a small radius, with enemies constantly emerging from darkness. Requires extremely high reaction and positioning skills.

### 超圣殿 / Hyper Holy Forbidden

完成 Holy Forbidden 后解锁，难度倍增。所有敌人的速度、血量和数量都大幅提升。只有最顶尖的玩家才能完成。

Unlocked after completing Holy Forbidden, with doubled difficulty. All enemy speed, HP, and数量 are greatly increased. Only the best players can complete it.

### 博纳别墅 / Bone Zone

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 解锁方式 / Unlock | 通关 Holy Forbidden / Clear Holy Forbidden |
| 地图特点 / Features | 全部敌人都是骷髅，密度极高 |
| 推荐策略 / Strategy | Mortaccio 最佳角色，AoE 构筑 |

Bone Zone 的所有敌人都是骷髅，密度极高。但 Mortaccio 的被动对骷髅有额外效果。

All enemies in Bone Zone are skeletons at extreme density. Mortaccio's passive has bonus effects against skeletons.

## 特殊掉落 / Special Drops

### 紫色宝箱 / Purple Chest

紫色宝箱会提供武器进化。当你的武器达到 8 级且有对应道具时，打开紫色宝箱必定触发进化。

Purple chests provide weapon evolution. When a weapon is level 8 with its corresponding item, opening a purple chest guarantees evolution.

### 金色宝箱 / Golden Chest

金色宝箱提供大量金币和经验宝石。出现在特定关卡位置或由精英敌人掉落。

Golden chests provide大量 gold and XP gems. Appear at specific stage locations or dropped by elite enemies.

### 超武宝箱 / Super Weapon Chest

极其稀有的宝箱，可能同时进化多个武器。在大后期关卡中出现的概率较高。

Extremely rare chest that may evolve multiple weapons simultaneously. Higher probability in late-game stages.

- 漂浮的骷髅 / Floating Skull: 在 Dairy Plant 中有一个特定的漂浮骷髅，碰到后触发隐藏效果
- 南瓜头 / Pumpkin Head: 在 Mad Forest 中杀死特定的南瓜头敌人获得额外奖励
