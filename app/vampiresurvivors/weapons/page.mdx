# 武器图鉴 / Weapons Guide — 吸血鬼幸存者 / Vampire Survivors

{/* 全武器数据、进化路线、强度排行详解 / Complete weapon data, evolution paths, and tier list analysis */}

武器是吸血鬼幸存者的核心。游戏中有超过 30 种基础武器和进化武器（超武）。每个武器都有独特的攻击模式、伤害计算和搭配需求。本页面将全面解析所有武器。

Weapons are the core of Vampire Survivors. The game has over 30 base weapons and evolved forms (Ultra Weapons). Each weapon has unique attack patterns, damage calculations, and synergy requirements. This page covers all weapons comprehensively.

---

## 武器系统说明 / Weapon System Overview

- 最多携带 6 个武器 / Max 6 weapons per run
- 武器等级上限为 8 级 / Weapon max level is 8
- 8 级时 + 对应被动道具 = 进化条件 / Level 8 + matching passive = evolution condition
- 进化后的超武不占用额外槽位 / Evolved ultra weapons do not take extra slots
- 宝箱有概率提供进化 / Chests have a chance to provide evolution

## 初始武器 / Starting Weapons

### 鞭子 / Whip

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 类型 / Type | 近战扇形 / Melee arc |
| 基础伤害 / Base Damage | 10 |
| 攻击间隔 / Interval | 0.5 秒 |
| 范围 / Area | 中等 / Medium |
| 进化条件 / Evolution | Whip + Hollow Heart = Bloody Tear |
| 超武效果 / Ultra Effect | 附带生命偷取 / Life steal |

鞭子是 Antonio 的初始武器，进化后获得吸血效果，对生存有极大帮助。前中期清怪效率一般，但吸血在后期非常宝贵。

Whip is Antonio's starting weapon. Evolution grants life steal, greatly helping survival. Early-mid clearing is average, but life steal is invaluable late-game.

### 魔法杖 / Magic Wand

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 类型 / Type | 自动追踪 / Homing |
| 基础伤害 / Base Damage | 8 |
| 攻击间隔 / Interval | 0.4 秒 |
| 范围 / Area | 全屏 / Full screen |
| 进化条件 / Evolution | Magic Wand + Empty Tome = Holy Wand |
| 超武效果 / Ultra Effect | 攻击频率大幅提升 / Greatly increased attack rate |

魔法杖是 Imelda 的初始武器，进化后变为神圣法杖，攻击间隔极短，适合持续稳定输出。

Magic Wand is Imelda's starting weapon. Evolves into Holy Wand with extremely short attack intervals, suitable for consistent stable damage.

### 飞刀 / Knife

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 类型 / Type | 直线穿透 / Linear pierce |
| 基础伤害 / Base Damage | 12 |
| 攻击间隔 / Interval | 0.3 秒 |
| 弹道速度 / Speed | 快速 / Fast |
| 进化条件 / Evolution | Knife + Bracer = Thousand Edge |
| 超武效果 / Ultra Effect | 同时发射多枚飞刀 / Fires multiple knives simultaneously |

千刃形态可同时发射大量飞刀，配合弹道速度加成效果惊人。

Thousand Edge fires many knives simultaneously. With projectile speed bonuses, the effect is astonishing.

### 斧头 / Axe

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 类型 / Type | 弧形投掷 / Arc throw |
| 基础伤害 / Base Damage | 15 |
| 攻击间隔 / Interval | 0.8 秒 |
| 范围 / Area | 大 / Large |
| 进化条件 / Evolution | Axe + Candelabrador = Death Spiral |
| 超武效果 / Ultra Effect | 范围超大，带击退 / Huge area with knockback |

死亡旋风是游戏中最好的范围武器之一，大范围加击退效果让生存能力大幅提升。

Death Spiral is one of the best area weapons in the game. Large area plus knockback greatly improves survivability.

### 圣水 / Holy Water

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 类型 / Type | 地面区域 / Ground AoE |
| 基础伤害 / Base Damage | 10 |

| 进化条件 / Evolution | Holy Water + Attractorb = La Borra |
| 超武效果 / Ultra Effect | 留下持续伤害区域，可移动 / Leaves persistent damage zone, movable |

圣水之池 La Borra 是新手神器。留下的绿色毒池可造成大量持续伤害，进化后可跟随角色移动。

La Borra is a beginner's best friend. The green poison pools deal massive持续 damage and can follow the character after evolution.

Thunder Loop hits multiple enemies per strike, excellent for clearing dense crowds.

Unholy Vespers creates a protective ring around the character, extremely effective against melee enemies.

## 特殊武器 / Special Weapons
