# 道具分析 / Items Guide — 吸血鬼幸存者 / Vampire Survivors

{/* 全被动道具效果、与武器的配合、优先级系统详解 / All passive item effects, weapon pairings, and priority system analysis */}

被动道具是构筑的重要组成部分。它们为武器提供进化条件，同时自身也提供强大的属性加成。合理选择道具是通关的关键。本页面详细分析每个道具的效果和使用策略。

Passive items are essential components of any build. They provide evolution conditions for weapons and grant powerful stat bonuses. Choosing items wisely is key to clearing stages. This page analyzes every item and its usage strategy.

---

## 道具系统说明 / Item System Overview

- 最多携带 6 个被动道具 / Max 6 passive items per run
- 道具等级上限为 5 级 / Item max level is 5
- 道具可以用于武器进化 / Items are used for weapon evolution
- 道具效果大多数线性增长 / Most items scale linearly
- 宝箱中的道具不占用槽位（紫色宝箱） / Chest items do not take slots (purple chests)

---

## 进化相关道具 / Evolution-Related Items

### 菠菜 / Spinach

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 / Per Level | 伤害 +10% / Might +10% |
| 满级效果 / Max Level | 伤害 +50% / Might +50% |
| 进化武器 / Evolves | 火炬 / Fire Wand → Hellfire |
| 优先级 / Priority | ★★★★★ |

菠菜是最通用的伤害道具。任何构筑都能从中受益。满级 +50% 伤害是直接的全武器加成，是所有道具中最值得优先升级的之一。

Spinach is the most versatile damage item. Every build benefits from it. Max level +50% Might is a direct all-weapon bonus, making it one of the highest priority items.

### 空心 / Hollow Heart

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 / Per Level | 最大生命 +20% / Max HP +20% |
| 满级效果 / Max Level | 最大生命 +100% / Max HP +100% |
| 进化武器 / Evolves | 鞭子 / Whip → Bloody Tear |
| 优先级 / Priority | ★★★☆☆ |

空心提供大量生命值加成，是生存型构筑的核心。进化鞭子获得吸血效果后，生存能力进一步提升。

Hollow Heart provides massive HP bonus, core to survival builds. Evolving Whip into Bloody Tear further improves survivability with life steal.

### 空白之书 / Empty Tome

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 / Per Level | 冷却 -6% / Cooldown -6% |
| 满级效果 / Max Level | 冷却 -30% / Cooldown -30% |
| 进化武器 / Evolves | 魔法杖 / Magic Wand → Holy Wand |
| 优先级 / Priority | ★★★★☆ |

冷却缩减对几乎所有武器都有正面效果。30% CDR 意味着武器攻击频率提高约 43%，是最高收益的道具之一。

Cooldown reduction benefits almost all weapons. 30% CDR means approximately 43% more attacks, one of the highest value items.

### 护腕 / Bracer

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 / Per Level | 弹道速度 +12% / Projectile Speed +12% |
| 满级效果 / Max Level | 弹道速度 +60% / Projectile Speed +60% |
| 进化武器 / Evolves | 飞刀 / Knife → Thousand Edge |
| 优先级 / Priority | ★★★☆☆ |

弹道速度对某些武器至关重要（如飞刀、十字架），但对近战武器和自动追踪武器效果有限。构筑特定时需要。

Projectile speed is crucial for some weapons (Knife, Cross) but limited for melee and homing weapons. Needed for specific builds.

### 烛台 / Candelabrador

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 / Per Level | 攻击范围 +10% / Area +10% |
| 满级效果 / Max Level | 攻击范围 +50% / Area +50% |
| 进化武器 / Evolves | 斧头 / Axe → Death Spiral |
| 优先级 / Priority | ★★★★☆ |

范围加成对许多武器是极其强力的属性。更大的范围意味着更多的敌人在攻击范围内。与斧头、国王圣经、五芒星配合极佳。

Area bonus is extremely powerful for many weapons. Larger range means more enemies in attack radius. Excellent with Axe, King Bible, Pentagram.

### 吸引器 / Attractorb

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 / Per Level | 拾取范围 +25% / Pickup Range +25% |
| 满级效果 / Max Level | 拾取范围 +125% / Pickup Range +125% |
| 进化武器 / Evolves | 圣水 / Holy Water → La Borra |
| 优先级 / Priority | ★★★☆☆ |

拾取范围提高后，可以更安全地收集经验宝石，不必冒险冲入敌群。进化圣水后获得游戏中最强的超武之一。

Increased pickup range lets you safely collect XP gems without risking enemy crowds. Evolving Holy Water gives one of the best ultra weapons.

### 复制器 / Duplicator

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 / Per Level | 投射物 +1 / Projectile +1 |
| 满级效果 / Max Level | 投射物 +5 / Projectile +5 |
| 进化武器 / Evolves | 闪电戒指 / Lightning Ring → Thunder Loop |
| 优先级 / Priority | ★★★★☆ |

投射物数量是游戏中最强大的属性之一。更多投射物意味着更多伤害、更大覆盖面。对几乎所有发射类武器都是顶级属性。

Projectile count is one of the most powerful stats. More projectiles means more damage, more coverage. Top-tier for almost all firing weapons.

### 护甲 / Armor

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 / Per Level | 护甲 +1 / Armor +1 |
| 满级效果 / Max Level | 护甲 +5 / Armor +5 |
| 进化武器 / Evolves | 国王圣经 / King Bible → Unholy Vespers |
| 优先级 / Priority | ★★☆☆☆ |

护甲本身的收益递减，但在高难度关卡中仍然有用。主要价值在于进化国王圣经。

Armor has diminishing returns but is still useful in high-difficulty stages. Its main value is evolving King Bible.

### 四叶草 / Clover

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 / Per Level | 幸运 +20% / Luck +20% |
| 满级效果 / Max Level | 幸运 +100% / Luck +100% |
| 进化武器 / Evolves | 十字架 / Cross → Heaven Sword |
| 优先级 / Priority | ★★★☆☆ |

幸运影响宝箱掉落率、敌人掉落物质量和很多隐藏机制。虽然效果难以量化，但对整体游戏体验有显著改善。

Luck affects chest drop rates, enemy drop quality, and many hidden mechanics. While hard to quantify, it significantly improves overall gameplay.

### 王冠 / Crown

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 / Per Level | 经验获取 +12% / XP Gain +12% |
| 满级效果 / Max Level | 经验获取 +60% / XP Gain +60% |
| 进化武器 / Evolves | 五芒星 / Pentagram → Gorgeous Moon |
| 优先级 / Priority | ★★★★☆ |

经验获取直接提高升级速度，更快获得武器和道具。进化五芒星后解决了清屏副作用，是顶级组合。

XP gain directly increases leveling speed, getting weapons and items faster. Evolving Pentagram solves its screen-clear downside, a top-tier combo.

## 特殊道具 / Special Items

### 托罗纳魔盒 / Torrona's Box

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 每级效果 | 所有属性 +4%，诅咒 +8% |
| 满级效果 | 所有属性 +20%，诅咒 +40% |
| 解锁条件 | 通关 Cappella Magna 后解锁 |

托罗纳魔盒不能进化任何武器，但提供全方位的属性加成。代价是增加诅咒（敌人更多更强）。在已经有完整构筑后可以考虑。

Torrona's Box does not evolve any weapon but provides全方位 stat bonuses at the cost of increased curse. Consider it after establishing a complete build.

## 道具优先级系统 / Item Priority System

### 按构筑类型 / By Build Type

| 构筑类型 / Build | 高优先级道具 / High Priority |
|:---|:---|
| 范围伤害 / AoE Damage | 菠菜、烛台、复制器 / Spinach, Candelabrador, Duplicator |
| 生存 / Survival | 空心、护甲、月桂叶 / Hollow Heart, Armor, Laurel |
| 投射物 / Projectile | 复制器、护腕、菠菜 / Duplicator, Bracer, Spinach |
| 冷却 / Cooldown | 空白之书、菠菜 / Empty Tome, Spinach |
| 经验速刷 / XP Farm | 王冠、吸引器、菠菜 / Crown, Attractorb, Spinach |

### 通用优先级 / Universal Priority

1. 菠菜 / Spinach — 任何构筑都需要伤害
2. 空白之书 / Empty Tome — 冷却缩减通用性强
3. 复制器 / Duplicator — 投射物数量收益高
4. 烛台 / Candelabrador — 范围加成强势
5. 王冠 / Crown — 经验收益加速成长
6. 空心 / Hollow Heart — 生存保障
7. 吸引器 / Attractorb — 进化圣水需要
8. 四叶草 / Clover — 整体质量提升
9. 护腕 / Bracer — 特定构筑有用
10. 护甲 / Armor — 优先级最低但特定场合可用
