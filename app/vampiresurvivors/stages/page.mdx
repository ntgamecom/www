# 关卡攻略 / Stages Guide — 吸血鬼幸存者 / Vampire Survivors

{/* 全关卡地图、特色机制、推荐顺序与通关策略 / All stage maps, special mechanics, recommended order, and completion strategies */}

每个关卡都有独特的地图布局、敌人组合和精英分布。了解关卡特性是制定有效策略的关键。本页面详细介绍所有关卡的特点和通关要点。

Each stage has unique map layouts, enemy compositions, and elite distributions. Understanding stage characteristics is key to developing effective strategies. This page details all stages and their completion要点.

---

## 关卡选择原则 / Stage Selection Principles

| 关卡 / Stage | 解锁条件 / Unlock | 推荐等级 / Recommended |
|:---|:---|:---|
| Mad Forest | 初始解锁 / Default | 任何等级 / Any |
| Inlaid Library | Mad Forest 通关 / Clear Mad Forest | 新手完成 / After beginner |
| Dairy Plant | Library 通关 / Clear Library | 进阶 / Intermediate |
| Gallo Tower | Dairy Plant 通关 / Clear Dairy Plant | 进阶 / Advanced |

### 推荐通关顺序 / Recommended Clear Order

```
Mad Forest → Inlaid Library → Dairy Plant → Gallo Tower → Cappella Magna
```

建议在每个关卡先通关 Normal 模式，解锁 Hyper 模式后再挑战。Hyper 模式有更多敌人和经验，适合刷级。

It is recommended to clear Normal mode first in each stage, then challenge Hyper mode. Hyper mode has more enemies and XP, suitable for grinding.

## Mad Forest / 疯狂森林

### 关卡特性 / Stage Features

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 解锁条件 / Unlock | 初始解锁 / Default |
| 地图大小 / Map Size | 中等 / Medium |
| 特色敌人 / Special Enemies | 狼人、蝙蝠 / Werewolves, Bats |
| 精英 Boss / Stage Boss | Werewolf |
| 遗物 / Relic | 无 / None |

### 地图布局 / Map Layout

疯狂森林是一个开阔的森林地图，中间有小片空地，四周树木密集。建议沿地图边缘移动，利用树木作为天然屏障阻挡部分敌人。

Mad Forest is an open forest map with small clearings in the middle and dense trees around the edges. Recommended to move along edges, using trees as natural barriers.

### 推荐策略 / Recommended Strategy

- 新手首选地图，敌人强度低 / Best first stage, low enemy强度
- 推荐武器：圣水、闪电戒指 / Recommended: Holy Water, Lightning Ring
- 11 分钟前完成核心构筑 / Complete core build before 11 minutes
- 精英敌人生成较少，适合熟悉机制 / Few elite spawns, good for learning mechanics

## Inlaid Library / 镶嵌图书馆

### 关卡特性 / Stage Features

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 解锁条件 / Unlock | 通关 Mad Forest / Clear Mad Forest |
| 地图大小 / Map Size | 狭长 / Long narrow |
| 特色敌人 / Special Enemies | 书籍、鬼魂 / Books, Ghosts |
| 精英 Boss / Stage Boss | Giant Book |
| 遗物 / Relic | 空白之书 / Empty Tome |

### 地图布局 / Map Layout

图书馆是狭长的室内地图，上下左右都有书架作为边界。地图狭窄意味着敌人会更集中，AoE 武器效果拔群。

The Library is a long narrow indoor map with bookshelves as boundaries. The narrow layout means enemies are more concentrated, making AoE weapons highly effective.

### 推荐策略 / Recommended Strategy

- 非常适合刷级，窄地形 AoE 收益高 / Great for grinding, narrow terrain favors AoE
- 推荐武器：国王圣经、斧头 / Recommended: King Bible, Axe
- 注意书架间容易卡位，保持移动 / Watch for getting stuck between shelves, keep moving
- 地图左下和右上角落有隐藏物品 / Hidden items in bottom-left and top-right corners

## Dairy Plant / 奶牛场

### 关卡特性 / Stage Features

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 解锁条件 / Unlock | 通关图书馆 / Clear Library |
| 地图大小 / Map Size | 大 / Large |
| 特色敌人 / Special Enemies | 奶牛、僵尸 / Cows, Zombies |
| 精英 Boss / Stage Boss | Giant Cow |
| 遗物 / Relic | 托尔纳的魔咒 / Randomazzo |

### 地图布局 / Map Layout

奶牛场地形开阔，有多个建筑物和围栏。地图分为多个区域，每个区域有不同类型的敌人。建筑物内有宝箱。

Dairy Plant has open terrain with multiple buildings and fences. The map is divided into zones with different enemy types. Buildings contain chests.

### 推荐策略 / Recommended Strategy

- 推荐武器：十字架、火炬（穿墙效果好） / Recommended: Cross, Fire Wand (good through walls)
- 利用建筑物躲避追踪 / Use buildings to dodge pursuers
- 精英敌人变多，需要更强的生存能力 / More elites, need better survivability
- 隐藏区域可以获得额外奖励 / Hidden areas provide bonus rewards

## Gallo Tower / 加洛塔

### 关卡特性 / Stage Features

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 解锁条件 / Unlock | 通关奶牛场 / Clear Dairy Plant |
| 地图大小 / Map Size | 垂直高塔 / Vertical tower |
| 特色敌人 / Special Enemies | 盔甲兵、法师 / Armored soldiers, Mages |
| 精英 Boss / Stage Boss | Giant Armor |
| 遗物 / Relic | 水晶球 / Glass Vizard |

### 地图布局 / Map Layout

加洛塔是一个垂直地图，中间有螺旋楼梯。地图狭长但垂直方向延伸，怪物从上下两个方向出现。

Gallo Tower is a vertical map with spiral staircases in the middle. The map is narrow but extends vertically, with enemies coming from both up and down.

### 推荐策略 / Recommended Strategy

- 推荐武器：五芒星、闪电戒指（全屏效果出色） / Recommended: Pentagram, Lightning Ring
- 难度明显提升，注意走位 / Difficulty明显 increases, focus on positioning
- 地图下方有一个秘密房间 / Secret room at the bottom of the map
- 建议 21 分钟前至少有两个超武 / Aim for at least 2 ultra weapons before 21 minutes

## Cappella Magna / 大教堂

### 关卡特性 / Stage Features

| 属性 / Stat | 数值 / Value |
|:---|:---|
| 解锁条件 / Unlock | 通关加洛塔 / Clear Gallo Tower |
| 地图大小 / Map Size | 中等 / Medium |
| 特色敌人 / Special Enemies | 天使、恶魔 / Angels, Demons |
| 精英 Boss / Stage Boss | Death |
| 遗物 / Relic | 魔镜 / Torrona's Box |

### 地图布局 / Map Layout

大教堂是最终关卡，地图呈十字形布局。中心区域有大量精英敌人。死神在 30 分钟准时出现。

Cappella Magna is the final stage, with a十字-shaped layout. The central area has many elite enemies. Death appears precisely at 30 minutes.

### 推荐策略 / Recommended Strategy

- 必须准备击败死神的武器 / Must prepare weapons to defeat Death
- 核心组合：无限回廊 + 深红裹尸布 / Core combo: Infinite Corridor + Crimson Shroud
- 推荐使用 Sigma 或高等级角色 / Recommended: Sigma or high-level characters
- 保持移动，不要被逼入角落 / Keep moving, avoid being cornered

## Hyper 模式 / Hyper Mode

| 关卡 / Stage | 解锁条件 / Hyper Unlock | 特点 / Features |
|:---|:---|:---|
| Mad Forest Hyper | 通关 Normal / Clear Normal | 敌人 +50%，经验 +50% |
| Library Hyper | 通关 Normal / Clear Normal | 敌人 +75%，经验 +75% |
| Dairy Plant Hyper | 通关 Normal / Clear Normal | 敌人 +100%，经验 +100% |
| Gallo Tower Hyper | 通关 Normal / Clear Normal | 敌人 +125%，经验 +125% |
| Cappella Magna Hyper | 通关 Normal / Clear Normal | 敌人 +150%，经验 +150% |

### Hyper 模式攻略 / Hyper Mode Strategy

Hyper 模式下敌人更多、移动更快、伤害更高，但经验和金币收益也大幅提升。建议在拥有至少 3 个超武后再挑战 Hyper 模式。

Hyper mode has more enemies, faster movement, and higher damage, but also greatly increased XP and gold rewards. Recommended to challenge Hyper mode with at least 3 ultra weapons.
