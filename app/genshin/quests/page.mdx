## 深境螺旋 / Spiral Abyss

**全游戏最难的内容**——每月1日和16日刷新 / The hardest content — resets on 1st and 16th each month.

| 层数 / Floor | 难度 / Difficulty | 推荐等级 / Lv | 原石 / Primos |
|---|---|---|---|
| 9-10 | 低 / Low | 60-70 | 300 |
| 11 | 中 / Medium | 70-80 | 300 |
| **12** | **极高 / Extreme** | **80-90** | **300** |

**满星12层核心要求 / Core Requirements for Floor 12**:
1. 8个角色——需两队 / 8 characters across 2 teams
2. 主C 80-90级——天赋8-9 / Main DPS 80-90, talents 8-9
3. 圣遗物合理主词条——不需要完美副词条 / Correct main stats — no need for perfect substats
4. 了解怪物配置和走位——比装备更重要 / Knowing enemy patterns and positioning > gear

### 12层技巧 / Floor 12 Tips

| 要点 / Tip | 说明 / Description |
|---|---|
| **了解怪物阵容 / Know Enemy Lineup** | 每期深渊上半/下半怪物不同——提前查好阵容再配队 / Each reset has different enemies — check lineup before team building |
| **走位聚怪 / Positioning & Grouping** | 吸引怪物聚在一起再爆发——比单杀快很多 / Lure enemies together before bursting — much faster than 1-by-1 |
| **能量管理 / Energy Management** | 进下一间前确保全队能量满——用E技能产球 / Before next chamber, ensure full energy — use skills to generate particles |
| **优先处理威胁怪 / Priority Targets** | 先打深渊使徒、咏者这种开盾的怪——打断或集火 / Focus shield enemies (Heralds, Lecters) first — interrupt or burst |
| **盾反克制 / Shield Counter** | 对应元素破盾：水盾用冰/草——火盾用水——冰盾用火 / Elemental counters: Hydro shield → Cryo/Dendro, Pyro → Hydro, Cryo → Pyro |
| **重开不丢人 / Reset is OK** | 如果第一间打得不好——直接重开调整——12层不限重开次数 /  If first chamber goes poorly — reset and adjust — unlimited retries |

### 深渊月份轮换 / Abyss Rotation Tips

- 每月1日和16日刷新 / Resets on 1st and 16th
- **地脉异常**每期不同——例如「全队治疗加成+」「下落攻击增伤+」/ **Ley Line Disorder** changes each reset — e.g. "Healing Bonus +", "Plunge DMG +"
- 针对地脉异常配队——能大幅降低难度 / Build teams around the disorder — massively reduces difficulty
- 建议月初/月中各打一次——有充分时间凹满星 / Clear early each reset — plenty of time to 36-star
