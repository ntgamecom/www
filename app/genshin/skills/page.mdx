## 技能系统概述 / Skill System Overview

原神中每个角色拥有三个核心技能——**普通攻击(AA)**、**元素战技(E技能)**、**元素爆发(Q技能)**。此外还有**固有天赋**(被动技能——角色突破后解锁)和**命之座**(重复获得角色解锁的强化效果)。升级每个技能需要不同的材料——包括天赋书、周常Boss材料和智识之冠 / Each character has 3 core skills — **Normal Attack (AA)**, **Elemental Skill (E)**, **Elemental Burst (Q)**. Plus **Passive Talents** (unlocked after ascension) and **Constellations** (duplicate character unlocks). Each skill requires different materials — talent books, weekly boss materials, and Crown of Insight.

|---|---|---|---|---|
| 站场主C / On-field DPS | 1——核心输出 / Core DPS | 2 | 2 | 胡桃、神里绫华、那维莱特等 / Hu Tao, Ayaka, Neuvillette |
| 速切副C / Quick-swap Sub-DPS | 3——很少用 / Rarely used | 1 | 2 | 夜兰、行秋、菲谢尔等 / Yelan, Xingqiu, Fischl |
| 纯辅助 / Pure Support | 3——不升 / Don't level | 2 | 1 | 班尼特、万叶、钟离等 / Bennett, Kazuha, Zhongli |

| 物理主C / Physical DPS | 1——核心输出 / Core DPS | 2 | 3 | 优菈、雷泽等 / Eula, Razor |

## 热门配队技能联动 / Popular Team Skill Synergy

### 胡桃蒸发队 / Hu Tao Vaporize

**队伍**: 胡桃(主C) + 行秋(挂水) + 钟离(盾) + 夜兰/莫娜(增伤) / **Team**: Hu Tao (DPS) + Xingqiu (hydro) + Zhongli (shield) + Yelan/Mona (buffer)。**技能循环**: 钟离长E(盾)→行秋EQE(挂水)→夜兰EQ(增伤)→胡桃E(开E附魔火→重击输出)。胡桃E期间治疗不能超过50%HP——否则失去火伤加成。行秋雨帘剑和胡桃重击完美配合——每次重击触发蒸发 / **Rotation**: Zhongli hold E → Xingqiu EQE → Yelan EQ → Hu Tao E (infuse pyro → charged attack spam). Hu Tao must stay under 50% HP during E — or lose pyro bonus. Xingqiu rain swords + Hu Tao charged attacks = perfect vaporize.

### 那维莱特直伤队 / Neuvillette Hypercarry

**队伍**: 那维莱特(主C) + 芙宁娜(增伤) + 万叶(扩散减抗) + 白术(奶+盾) / **Team**: Neuvillette (DPS) + Furina (buffer) + Kazuha (VV shred) + Baizhu (heal+shield)。**技能循环**: 芙宁娜EQ(召唤众水的歌者)→万叶EQ(扩散减抗)→白术EQ(盾+奶)→那维莱特重击(蓄力喷水——持续约3秒)。那维莱特的重击是全游戏最高倍率的站场输出技能——配合芙宁娜提供的高额增伤——一轮重击约10万伤害 / **Rotation**: Furina EQ → Kazuha EQ (VV shred) → Baizhu EQ → Neuvillette charged attack (charge beam ~3s). Neuvillette's CA is the highest multiplier on-field skill in game — with Furina's massive dmg bonus — ~100k per CA.

### 艾尔海森激化队 / Alhaitham Quickbloom

**队伍**: 艾尔海森(主C) + 纳西妲(副C+挂草) + 久岐忍(雷奶) + 行秋/夜兰(水) / **Team**: Alhaitham (DPS) + Nahida (sub-DPS+dendro) + Kuki (electro+heal) + Xingqiu/Yelan (hydro)。**技能循环**: 纳西妲EQ(扫码挂草)→行秋EQ(挂水)→久岐忍E(雷环)→艾尔海森E(突进附魔草)→下落攻击+重击输出。这个队伍同时触发绽放(草+水→种子)+超绽放(种子+雷→追踪弹)+蔓激化(草+雷→额外草伤)——三重反应叠加——伤害极高 / **Rotation**: Nahida EQ → Xingqiu EQ → Kuki E → Alhaitham E → plunge + CA spam. This team triggers Bloom (dendro+hydro=seeds) + Hyperbloom (seeds+electro=homing) + Spread (dendro+electro=bonus dmg) — triple reaction stack — massive DPS.
