# 原神攻略 / Genshin Impact Guide

《原神》是miHoYo/HoYoverse开发的开放世界动作RPG——旅行者穿越提瓦特大陆，寻找失散的亲人，探索七神的国度。从蒙德到璃月、从稻妻到须弥、从枫丹到纳塔——每一个国家都有独特的地貌、文化和秘密。本攻略涵盖新手入门指引、提瓦特全区域探索、角色培养与队伍搭配以及武器圣遗物全攻略。

Genshin Impact is miHoYo/HoYoverse's open-world action RPG — the Traveler crosses Teyvat searching for their lost sibling, exploring the seven nations. From Mondstadt to Liyue, Inazuma to Sumeru, Fontaine to Natlan — each nation has unique landscapes, culture, and secrets. This guide covers beginner tips, full Teyvat exploration, character builds and team comps, and complete weapons & artifacts guide.

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **别浪费原石抽常驻池 / Don't Waste Primos on Standard** | 原石只抽**限定角色卡池(纠缠之缘)**——常驻池用蓝球(相遇之缘)抽 / Only use primogems on **limited character banner (Intertwined Fate)** — standard banner uses Acquaint Fate |
| **2** | **冒险等级45前别刷圣遗物 / Don't Farm Artifacts Before AR45** | 45级前刷圣遗物副本不掉金——45级后必掉1金 / Before AR45, domains don't drop gold artifacts — AR45+ guarantees 1 gold |
| **3** | **每日委托必须做 / Do Daily Commissions** | 每天60原石——全游戏最稳定的原石来源 / 60 primos/day — most consistent primogem source |
| **4** | **树脂不要囤太久 / Don't Hoard Resin** | 原粹树脂上限160——快满时用掉打地脉或周本 / Resin cap is 160 — use it on ley lines or weekly bosses |
| **5** | **蒙德和璃月的神瞳要收集 / Collect Anemo/Geo Oculi** | 神像升级奖励体力上限+原石——全游戏最值的收集 / Statue upgrades reward stamina + primos — best value collectible |
| **6** | **每个角色拉到20级送蓝球 / Ascend Characters to 20** | 每个角色第一次突破送1个相遇之缘(蓝球) / First ascension = 1 Acquaint Fate each |
| **7** | **传送点先开完 / Unlock All Teleports** | 每到一个新国家先把七天神像+传送点开了 / Prioritize unlocking statues + teleport points first |
| **8** | **大月卡(珍珠纪行)性价比高 / Battle Pass is Good Value** | 68元大月卡=大量资源(树脂+天赋书+武器材料)/ BP = lots of resources |

## 冒险等级 / Adventure Rank Guide

| 等级 / AR | 解锁内容 / Unlock | 建议 / Tip |
|---|---|---|
| 1-10 | 蒙德主线 / Mondstadt main | 跟随主线——不要分心 |
| 10-20 | 璃月主线 + 每日委托 / Liyue main + Dailies | **每天一定要做委托** |
| 20-25 | 突破任务 / Ascension Quest | 不突破经验溢出但等级锁住——尽快打突破 |
| 25-35 | 稻妻主线 / Inazuma main | 可以开始刷天赋材料 / Start farming talent books |
| 35-40 | 圣遗物副本(紫)/ Domains (purple) | 可以打一些紫色圣遗物过渡 |
| **45** | **圣遗物副本(金)/ Domains (gold)** | **开始刷金色圣遗物**——所有树脂投入到圣遗物本 / Start gold artifact farming — all resin here |
| 45-50 | 须弥+枫丹 / Sumeru + Fontaine | 提升角色等级+天赋+武器 / Level characters + talents + weapons |
| 50-55 | 深渊/ Spiral Abyss | 尝试打深渊——每月获取原石 |
| 55+ | 周本+圣遗物/ Weekly bosses | 养新角色——追求圣遗物副词条 |
