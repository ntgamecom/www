|---|---|---|---|
| **原粹树脂 / Original Resin** | 160 | 1个/8分钟(180/天)/ 1 per 8 min (180/day) | 地脉花/秘境/周本 / Ley lines, domains, weekly bosses |

## 祈愿系统详解 / Gacha System Deep Dive

### 角色卡池机制 / Character Banner Mechanics

| 机制 / Mechanic | 说明 / Explanation |
|---|---|
| **5星基础概率 / Base 5★ Rate** | 0.6%——73抽开始软保底(概率逐渐提升) / 0.6% — soft pity starts at 73 (rate increases gradually) |
| **硬保底 / Hard Pity** | 90抽必定出5星 / 90 pulls guarantees a 5-star |
| **小保底 / 50/50** | 第一次出5星有50%概率是当期UP——如果歪了(出常驻5星)下次必出UP / First 5-star has 50% to be the featured — if you lose (get a standard 5-star), next is guaranteed featured |
| **大保底 / Guarantee** | 如果小保底歪了——下一个5星**必定是当期UP** / If you lose 50/50 — next 5-star **is guaranteed featured** |
| **保底继承 / Pity Carries Over** | 抽数跨卡池继承——不会清零 / Pull count carries across banners — never resets |
| **4星保底 / 4★ Pity** | 10抽必定出4星——每2个4星中有1个是当期UP / 10 pulls guarantees a 4-star — every other 4-star is a featured one |

### 武器卡池机制 / Weapon Banner Mechanics

| 机制 / Mechanic | 说明 / Explanation |
|---|---|
| **保底 / Pity** | 80抽必定出5星——65抽开始软保底 / 80 pulls guarantees a 5-star — soft pity starts at 65 |
| **定轨 / Epitomized Path** | 选择一把UP武器——如果出另一把UP——获得1点命定值——2点满后下一把必出定轨武器 / Choose one featured weapon — if you get the other, gain 1 Fate Point — at 2, next is guaranteed chosen weapon |
| **注意 / Warning** | **定轨不继承**——下个卡池命定值清零 / **Fate Points DO NOT carry over** — resets with new banner |

### 原石规划建议 / Primogem Planning

| 目标 / Goal | 所需原石 / Primos Needed | 说明 / Notes |
|---|---|---|
| 抽一个UP角色 / One featured character | ~14400 (90抽) | 大保底预算 / Budget for guarantee |
| 抽一个UP角色+专武 / Character + weapon | ~28800 (180抽+80抽) | 全大保底预算 / Full guarantee budget |
| 月卡党每月获取 / Welkin monthly | ~7500-8000 | 每日委托+月卡+活动 / Dailies + welkin + events |
| 零氪每月获取 / F2P monthly | ~4500-5000 | 日常+活动 / Dailies + events |

**抽卡建议 / Pulling Advice**:
- **只抽你喜欢或需要的角色**——不要人抽我也抽 / **Pull only for characters you want or need** — don't FOMO
- **不要抽武器池**除非你准备好歪的代价 / **Don't pull weapon banner** unless you're ready for the cost
- **月卡(空月祝福)**是最划算的微氪——30元=300结晶+2700原石 / **Welkin Moon** is best value — ¥30 = 300 Genesis + 2700 primos

## 新手阶段规划 / New Player Roadmap

| 阶段 / Phase | AR范围 | 目标 / Goals | 时间 / Duration |
|---|---|---|---|
| **开局 / Start** | 1-10 | 做完蒙德主线——开传送点 / Complete Mondstadt main — unlock waypoints | 2-4小时 |
| **前期 / Early** | 10-25 | 做完璃月主线——每天做委托——清体力 / Complete Liyue main — dailies — spend resin | 1-2周 |
| **中期 / Mid** | 25-35 | 稻妻主线——刷天赋材料——准备45级 / Inazuma main — farm talent books — prep for AR45 | 2-4周 |
| **转折点 / Turning** | **45** | **开始刷金色圣遗物**——把存的脆弱树脂全用掉 / **Start gold artifact farming** — dump all Fragile Resin | 1-2天 |
| **后期 / Late** | 45-55 | 须弥+枫丹主线——养2-3队角色——打深渊 / Sumeru+Fontaine main — build 2-3 teams — clear abyss | 1-3月 |
| **长草期 / Endgame** | 55+ | 追求圣遗物副词条——全角色收集——爱玩啥玩啥 / Substats min-max — collect all chars — whatever you enjoy | 无限 |

**新手节奏提醒 / Rhythm Tips**:
- 不要急着冲等级——慢慢体验剧情和探索 / Don't rush levels — enjoy the story and exploration
- **每天15-30分钟**清日常+体力就够了 / **15-30 min/day** for dailies + resin is enough
- 周末找时间做**传说任务**和**世界任务**——这些是原石的大头 / Weekends do **story quests** and **world quests** — major primo sources
