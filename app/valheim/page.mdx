# 英灵神殿 Valheim
## 维京生存沙盒游戏指南 Viking Survival Sandbox Guide

{/* 游戏概览 / Game Overview */}

**Valheim（英灵神殿）** 是一款由 Iron Gate Studio 开发的维京题材生存沙盒游戏。在第十世界（The Tenth World）中，玩家扮演一名战死的维京战士，被奥丁派遣前往这片蛮荒之地，击败古老的原始生物来取悦众神。游戏于 2021 年 2 月开启抢先体验，凭借其独特的维京美学、程序化生成的广阔世界以及深度的建造与战斗系统，迅速成为全球现象级作品。

Valheim 融合了探索、建造、战斗和生存元素，支持单人及多人合作游玩。其核心循环是：探索新区域 → 收集资源 → 制作装备 → 击败首领 → 解锁更高级区域。每个生物群系都代表一个独特的挑战阶段，玩家需要逐步提升自己的装备和技能。

## 核心特色 Core Features

{/* 探索收集 / Exploration & Gathering */}

**探索与收集 Exploration & Gathering**
探索程序化生成的广阔世界，从茂密的森林到危险的沼泽，从高耸的山脉到神秘的迷雾之地。每个生物群系都有独特的资源、生物和挑战。玩家需要收集木材、石头、矿石和各种材料来制作工具、武器和建筑。

**建造系统 Building System**
利用先进的建筑物理引擎，玩家可以自由建造从简陋小屋到宏伟堡垒的任何建筑。支持梁柱结构、不同材料等级、家具摆放和装饰。建造一个舒适的基地不仅能提供保护，还能激活各种增益效果。

**战斗与首领 Combat & Bosses**
游戏包含多种武器类型（剑、斧、锤、矛、弓、匕首等）和盾牌格挡系统。玩家需要在不同地形中与各种敌人战斗，最终挑战六个强大的首领，每个首领都需要特定的策略和准备。

**航海系统 Sailing System**
建造维京长船（木筏、维京船、长船），在广阔的海洋中航行。航海是探索新大陆和运送资源的唯一方式，但也伴随着风暴和危险的海怪。

**烹饪与食物 Cooking & Food**
食物系统是 Valheim 的核心机制之一。不同的食物提供生命值（HP）和耐力（Stamina）加成。玩家需要狩猎、采集、耕种和烹饪来获取更好的食物，以应对更危险的挑战。

**多人合作 Multiplayer Cooperation**
支持最多 10 人同服务器游玩。团队合作可以显著提高效率，分工协作：有人负责建造基地、有人外出探索、有人专职战斗。死亡后队友可以帮忙捡回物品。

## 快速入门 Quick Start

1. **建立第一个基地** — 在青青草原（Meadows）选择平坦区域，搭建简易庇护所
2. **制作基础工具** — 石斧、木锤、工作台
3. **收集食物** — 采集浆果、狩猎野猪和鹿
4. **挑战第一个首领** — 献祭两只鹿角召唤 Eikthyr
5. **解锁新区域** — 击败 Eikthyr 后获得硬鹿角，制作鹿角镐进入黑暗森林
