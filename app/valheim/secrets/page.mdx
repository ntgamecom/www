### 商人出现规律 / Trader Spawn Pattern

| 规律 / Pattern | 说明 / Description |
|---|---|
| 生成位置 / Spawn | 黑暗森林中随机5个可能点位之一 / Random among 5 possible BF spots |
| 距离 / Distance | 通常离起始岛不远 / Usually near starting island |
| 出现条件 / Condition | 不需要任何前置——直接去找即可 / No prerequisite — go find him |
| 不可杀 / Invincible | 可以攻击他——但他不会死 / Can attack — never dies |

**找到商人的技巧 / Finding the Trader**:
1. 探索黑暗森林——沿着海岸线走 / Explore Black Forest — follow coastline
2. 注意地图上的袋状图标 / Watch for bag icon on minimap
3. 在首次登陆的岛屿中央寻找——大概率在岛中央 / Center of starting island
4. 如果找不到——可以用在线地图种子查询器 / Use online seed map viewer
