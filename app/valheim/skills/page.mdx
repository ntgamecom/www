## 死亡惩罚 / Death Penalty

| 损失 / Loss | 数值 / Amount |
|---|---|
| 全部技能经验 / All Skill XP | 当前等级经验的5% / 5% of current level XP |
| 无技能降级 / No Level Drop | 技能等级不会降级——只是经验条后退 / Level doesn't drop — XP bar moves back |
| 高等级损失 / High Level Loss | 75级以上的技能损失更严重 / Skills above 75 lose more XP |
| 物品丢失 / Item Loss | 身上所有物品掉落——需要跑尸 / All items drop — corpse run required |

**减少死亡损失 / Reducing Death Penalty**:

| 方法 / Method | 效果 / Effect |
|---|---|
| 休息buff / Rested Buff | 技能经验获取+50%——更快恢复死前水平 / +50% XP gain |
| 食用高经验食物 / XP Foods | 某些食物提升技能经验获取率 / Some foods boost XP rate |
| 快速跑尸 / Quick Corpse Run | 死后尽快捡回尸体——减少损失的心理阴影 / Quick retrieval |
| 备份装备 / Backup Gear | 在基地放一套备用装备 / Keep backup gear in base |
