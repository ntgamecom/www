英灵神殿的建筑系统是其核心魅力之一——结构力学模拟让建造既有挑战又有成就感 / Valheim's building system is one of its core appeals — structural integrity simulation makes building challenging and rewarding.

---

## 结构力学 / Structural Integrity

### 基础原理 / Core Principles

英灵神殿的建筑有结构力学系统——每块材料距离支撑点越远——支撑力越弱 / Valheim buildings have a structural integrity system — each piece loses support strength the further it is from a ground connection.

| 支撑概念 / Concept | 说明 / Description |
|---|---|
| 地面连接 / Ground Connection | 直接插在地面上的结构——最稳固 / Structures directly on ground — most stable |
| 结构颜色 / Color Indicator | 蓝色(完美)→绿色→黄色→橙色→红色(即将倒塌) / Blue(good)→Green→Yellow→Orange→Red(collapse) |
| 最大悬空 / Max Overhang | 每种材料有最大悬空距离——超出会垮塌 / Each material has max unsupported distance |
| 支撑传递 / Support Transfer | 支撑力从底部向上传递——连接的越多越强 / Support transfers upward — more connections = stronger |
| 水平跨度 / Horizontal Span | 屋顶和桥梁需要适当支撑——中间可能需要支柱 / Roofs and bridges need mid supports |

| 强制叠加 / Forced Stack | 先放一块——上面再放另一块时按Ctrl / Ctrl while placing | 同一位置双倍厚度 / Double thickness |
| 半块偏移 / Half Offset | 在第一块一半位置放置第二块 / Half-snap placement | 砖缝式排列 / Brick pattern |
| 隐藏支撑 / Hidden Support | 用铁木在里面做支撑架——外面包石材 / Iron frame inside | 全石头建筑外观 / Full stone look |
