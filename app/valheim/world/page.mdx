# 世界探索 / World Exploration
## 地图机制、传送门与航海探险 Map Mechanics, Portals, and Sailing Adventures

{/* 探索概览 / Exploration Overview */}

Valheim 的世界是程序化生成的广阔开放世界，每个存档（seed）都会生成独一无二的地形。世界由多个大型岛屿组成，玩家需要穿越海洋、翻山越岭，逐渐揭开十世界的全部面貌。

---

## 世界生成 / World Generation

{/* 地图生成机制 / Map Generation */}

每次创建新世界时，游戏会使用一个种子（Seed）代码来生成地形。不同的种子会导致完全不同的世界布局。

| 生成要素 | 说明 |
|---|---|
| 世界种子 | 可以是任意字符串，不同种子产生不同地图 |
| 岛屿分布 | 大型主岛 + 多个小型岛屿 |
| 生物群系位置 | 大致遵循中心到边缘的分布规律 |
| 水域比例 | 约 60-70% 为海洋 |
| 地形高度 | 从海岸到山脊逐渐升高 |
| 地下城分布 | 每个区域都有随机生成的地下城 |

### 已知种子示例 Known Seed Examples

| 种子名称 | 特点 |
|---|---|
| `hXklYpPVDg` | 适合单人开局，出生岛资源丰富 |
| `Gallows` | 主岛巨大，连接多个关键区域 |
| `valheim` | 经典的均衡地图布局 |
| `hammer` | 开局岛上有多个商人位置 |

## 地图与标记 / Map & Markers

{/* 地图系统 / Map System */}

玩家按 M 键打开地图。地图最初为迷雾状态，随着探索逐渐显现。地图会记录玩家的足迹、标记和重要发现。

### 地图标记类型 Map Marker Types

| 标记图标 | 用途 |
|---|---|
| House 房子 | 标记基地和庇护所 |
| Hammer 锤子 | 标记工作台或锻造台位置 |
| Sword 剑 | 标记危险区域或战斗地点 |
| Apple 苹果 | 标记食物资源点 |
| Star 星星 | 标记重要地点 |
| Portal 传送门 | 标记传送门位置 |
| Campfire 篝火 | 标记临时休息点 |
| Skull 骷髅 | 标记死亡地点 |
| Ship 船 | 标记码头或造船地点 |
| Dot 圆点 | 自定义标记 |

### 探索记录技巧

1. **即时标记** — 发现任何有价值的资源点立即在地图标记
2. **颜色编码** — 使用不同颜色区分资源类型（绿色食物、蓝色水源、红色危险）
3. **命名规范** — 给标记添加文字说明，如 「铜矿区 - 西北」
4. **传送门标注** — 在传送门附近放置命名标记，便于快速定位
5. **未探索边界** — 沿着海岸线走，逐步推展迷雾边界

## 传送门 / Portals

{/* 传送门系统 / Portal System */}

传送门是 Valheim 中最便捷的快速旅行方式。通过在两个地点各建造一个传送门并设置相同的名称，即可实现即时传送。

### 传送门建造 Portal Construction

| 材料 | 数量 |
|---|---|
| 优质木材 Fine Wood | 10 |
| 灰矮人之心 Greydwarf Eye | 2 |
| 树脂 Resin | 2 |

### 传送门规则 Portal Rules

| 规则项 | 说明 |
|---|---|
| 传送门配对 | 相同名称的传送门互相关联 |
| 可携带物品 | 除矿石和金属锭外的所有物品 |
| 限制物品 | 铜、锡、青铜、铁、银、黑金、黑铁、甲壳 |
| 多人使用 | 所有玩家都可以使用任何传送门 |
| 名称修改 | 修改名称后需要等待几秒重新连接 |
| 传送门上限 | 每个世界最多 64 个传送门 |

### 传送门网络策略 Portal Network Strategy

1. **中央枢纽** — 在主基地建造一个中央传送门大厅，集中连接所有分支传送门
2. **命名规范** — 使用简单一致的命名规则，如 `mine1`、`base_swamp`、`troll_cave`
3. **资源点直连** — 在重要资源点建造传送门，便于快速运输补给
4. **探索传送门** — 探索新区域时随身携带传送门材料，找到安全地点后立即建造
5. **矿石运输** — 由于矿石不能传送，需要在矿点附近的临时基地炼制

**重要：** 矿石和金属锭不能通过传送门。这意味着玩家需要规划好金属的运输路线，通常通过海运或徒步运回主基地。一些替代方案：
- 在矿区附近建立熔炼基地
- 使用船只通过海路运输大量金属
- 在高难度下需要保护运输路线不被敌人袭击

## 航海 / Sailing

{/* 航海系统 / Sailing System */}

航海是 Valheim 中长距离探索和货物运输的主要方式。掌握航海技巧对游戏进程至关重要。

### 风向与航行 Wind and Sailing

| 风向 | 对船速的影响 | 操作建议 |
|---|---|---|
| 顺风 | 全速前进 | 直线航行 |
| 侧风 | 中等速度 | 调整航向角度 |
| 逆风 | 极慢 | Z 字形航行调整 |
| 无风 | 静止 | 等待风力变化 |

**航行技巧：**
- 航行前检查天气预报，避免暴风雨
- 靠近海岸线航行，便于遇到危险时快速靠岸
- 船上可以放置工作台，方便在岛屿上建立临时据点
- 夜晚航行视野极差，建议在日落前靠岸
- 长船（Longship）航行速度最快，载货量最大

### 海上危险 Sea Dangers

| 危险种类 | 描述 | 应对方法 |
|---|---|---|
| 海怪 Serpent | 巨大的海蛇，攻击船只 | 使用弓箭射杀或快速远离 |
| 暴风雨 Storm | 巨浪影响航行和视野 | 靠岸等待风暴结束 |
| 暗礁 Shallow | 船只会搁浅 | 使用长杆探测水深 |
| 寒冷 Cold | 海上温度低 | 携带抗寒装备或蜜酒 |
| 迷雾 Fog | 降低能见度 | 使用 Wisp 灯笼辅助 |

### 码头建设 Dock Construction

在海岸边建造码头可以方便船只停靠和货物装卸。建议：
1. 在较深的水域建造延伸栈桥
2. 码头旁设置仓库和工作台
3. 建立防御工事防止敌人靠近
4. 码头上放置传送门连接主基地

## 探索策略 / Exploration Strategy

{/* 探索建议 / Exploration Tips */}

### 分阶段探索 Phase-based Exploration

| 阶段 | 探索范围 | 交通方式 | 重点关注 |
|---|---|---|---|
| 初期 | 出生岛 | 徒步 | 建立地图基础、标记资源 |
| 青铜时代 | 临近岛屿 | 维京船 Karve | 寻找沼泽、收集铁 |
| 铁器时代 | 中型岛屿 | 长船 Longship | 寻找雪山、商人位置 |
| 银器时代 | 大型岛屿 | 长船 | 寻找平原、修建传送门网 |
| 后期 | 全球 | 全地图 | 迷雾之地、最终挑战 |

### 探索准备清单 Exploration Checklist

出发前确认以下事项：
1. **装备检查** — 武器、防具、工具是否完好
2. **食物储备** — 至少携带 3 种食物，足够 20 分钟续航
3. **蜜酒携带** — 根据目标区域准备相应抗性蜜酒
4. **传送门材料** — 携带 10 优质木材 + 2 灰矮人之心 + 2 树脂
5. **备用装备** — 一套备用武器和工具（放置在船上或安全屋）
6. **床铺** — 在出发基地设置重生点
7. **地图标记** — 确认目标方向的已探索边界

### 团队探索分工 Team Exploration Roles

| 角色 | 职责 | 推荐装备 |
|---|---|---|
| 领航员 Navigator | 掌舵、读取地图、导航 | 望远镜、地图开放 |
| 战士 Fighter | 清理敌人、保护队伍 | 重甲、盾牌、高 HP 食物 |
| 采集者 Gatherer | 收集资源、标记矿产 | 镐、斧、高耐力食物 |
| 建造者 Builder | 建立临时基地和传送门 | 锤子、工作台、大量木材 |

## 世界尽头 / The Edge of the World

在 Valheim 的世界尽头，地图会显示为无限海洋。如果你持续向一个方向航行，最终会遇到一道巨大的瀑布落向虚空。这是世界的边界，无法通行。

世界尽头有一种特殊的氛围感——海洋逐渐变得黑暗，天空呈现诡异的色彩。如果你看到这些迹象，说明你已经接近了十世界的边缘。回头吧，那里没有值得探索的东西。
