# 战斗系统 / Combat System

## 格挡、招架与闪避 / Block, Parry & Dodge

### 防御机制对比 / Defense Comparison

| 动作 / Action | 操作 / Input | 耐力消耗 / Stamina | 效果 / Effect |
|---|---|---|---|
| **普通格挡 / Block** | 右键按住 / Hold RMB | 取决于伤害 / Per dmg | 减少伤害——取决于盾牌格挡值 / Dmg reduction |
| **招架 / Parry** | 攻击瞬间按右键 / RMB at hit | 中等 / Medium | 完美格挡——敌人硬直——大量反击伤害 / Stagger enemy |
| **翻滚闪避 / Dodge Roll** | 空格+方向 / Space+Dir | 高 / High | 无敌帧——闪避一切攻击 / I-frames — evade everything |
| **后退闪避 / Backstep** | 方向下+空格 / Back+Space | 中等 / Medium | 向后跳跃——比翻滚少耗耐 / Back jump — less stamina |

### 盾牌类型 / Shield Types

| 盾牌 / Shield | 类型 / Type | 格挡值 / Block | 招架倍率 / Parry Bonus | 移动速度 / Move Speed |
|---|---|---|---|---|
| **木盾 / Wood Shield** | 小盾 / Buckler | 20 | 1.5x | 正常 / Normal |
| **青铜盾 / Bronze Shield** | 小盾 / Buckler | 30 | 2x | 正常 / Normal |
| **铁盾 / Banded Shield** | 小盾 / Buckler | 50 | 2x | 正常 / Normal |
| **银盾 / Silver Shield** | 小盾 / Buckler | 70 | 2.5x | 正常 / Normal |
| **黑金盾 / Blackmetal Shield** | 小盾 / Buckler | 90 | 2.5x | 正常 / Normal |
| **铁塔盾 / Iron Tower Shield** | 塔盾 / Tower | 80 | 1x | -20% / -20% |
| **黑金塔盾 / BM Tower Shield** | 塔盾 / Tower | 120 | 1x | -20% / -20% |
| **骨盾 / Bone Shield** | 小盾 / Buckler | 15 | 1.5x | 正常 / Normal |

**小盾 vs 塔盾 / Buckler vs Tower**:
- 小盾可以招架(完美格挡)——塔盾不行 / Bucklers can parry — towers cannot
- 塔盾格挡值更高但不消耗耐力时防御极高 / Tower shields have higher block but no parry
- 塔盾减少移动速度——小盾不影响 / Tower slows movement — buckler doesn't

## 耐力管理 / Stamina Management

| 动作 / Action | 消耗 / Cost | 建议 / Advice |
|---|---|---|
| 普通攻击 / Normal Attack | 10-15 | 不要空挥——浪费 / Don't swing in air |
| 重击 / Power Attack | 30-40 | 击倒后用——打硬直敌人 / Use after stagger |
| 格挡攻击 / Block Attack | 15-25 | 格挡后反击用 / Counter after block |
| 翻滚 / Roll | 25-30 | 仅关键闪避时用 / Only critical dodges |
| 持盾跑 / Run with Shield | 持续消耗 / Continuous | 非战斗时不要一直跑 / Don't run constantly |

- 早期: 覆盆子+蓝莓+烤鹿肉 / Raspberry + Blueberry + Deer Meat
- 中期: 胡萝卜汤+烤肉串 / Carrot Soup + Deer Stew
- 后期: 鱼肉卷+血旺 / Fish Wraps + Blood Pudding

## 战斗环境利用 / Combat Environment

| 地形 / Terrain | 优势 / Advantage |

| 狭窄通道 / Narrow Path | 避免被包围——一个一个打 / Avoid surround |

| 建筑内 / Inside Building | 限制大型敌人行动 / Restrict large enemies |
