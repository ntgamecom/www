# Minigames Guide — All 25+ Minigames Complete

{/* ===== 小游戏概述 ===== */}

## 小游戏系统概述 / Minigame System Overview

《双人成行》隐藏着 **超过 25 个可玩的双人小游戏**。这些小游戏分散在各个章节的角落中，找到并完成它们是获得奖杯/成就"百科全书/Encyclopedia"的必要条件。小游戏种类繁多，从射击比赛到音乐节奏游戏应有尽有。

*It Takes Two* hides **over 25 playable two-player minigames** throughout the game. Finding and playing them all is required for the "Encyclopedia" trophy/achievement. Minigames range from shooting contests to rhythm games and everything in between.

## 第一章 — 棚屋小游戏 / Shed Minigames

### 1. 打地鼠 / Whack-A-Mole

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 工具箱后方 / Behind the toolbox |
| 类型 / Type | 反应速度 / Reaction Speed |
| 获胜技巧 / Tip | 盯着洞口，预判击打时机 / Watch the holes, anticipate timing |

两人各站一边，用锤子打击从洞里冒出的地鼠。击中得一分，打到炸弹扣分。先达到目标分数者获胜。

Both players stand on either side, hitting moles with hammers. Hits score points, hitting bombs deducts. First to target score wins.

### 2. 投篮比赛 / Hoops

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 棚屋上层 / Shed upper level |
| 类型 / Type | 瞄准 / Aiming |
| 获胜技巧 / Tip | 掌握抛物线弧度 / Master the parabolic arc |

用钉子枪或投掷物将球投入篮筐。三分球有额外奖励。投篮角度比力度更重要。

Use the nail gun or thrown objects to sink baskets. Three-pointers give bonus points. Angle matters more than power.

### 3. 坦克大战 / Tank Mayhem

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 棚屋角落 / Shed corner |
| 类型 / Type | 竞技射击 / Competitive Shooting |
| 获胜技巧 / Tip | 利用掩体，不要站桩 / Use cover, don't stand still |

控制小型玩具坦克互相射击。地图中有弹药箱和护盾道具。灵活移动比站着瞄准更有效。

Control toy tanks and shoot each other. Maps have ammo boxes and shield pickups. Moving is more effective than standing still to aim.

## 第二章 — 大树小游戏 / Tree Minigames

### 4. 拔河比赛 / Tug of War

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 树洞内部 / Inside tree hollow |
| 类型 / Type | 力量对抗 / Strength Contest |
| 获胜技巧 / Tip | 有节奏地拉，不要乱按 / Pull rhythmically, don't mash |

双方交替拉扯绳子中央的标记。按照节奏拉动比疯狂连按更有效。注意绳子中间的红线——到达你的区域即获胜。

Both sides alternately pull the rope marker. Rhythmic pulling beats button mashing. Watch the red line in the center — reaching your zone wins.

### 5. 虫虫赛跑 / Bug Race

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 树枝分叉处 / Branch fork |
| 类型 / Type | 竞速 / Racing |
| 获胜技巧 / Tip | 抄近道，利用弹跳蘑菇 / Take shortcuts, use bounce mushrooms |

控制甲虫在树枝赛道上竞速。赛道上散布着加速道具和障碍物。熟悉赛道布局是获胜的关键。

Control beetles racing on branch tracks. Speed boosts and obstacles dot the course. Knowing the track layout is key to winning.

### 6. 苹果射击 / Apple Shoot

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 树冠高处 / High canopy |
| 类型 / Type | 瞄准 / Target Shooting |
| 获胜技巧 / Tip | 预判苹果摆动 / Predict apple swinging |

站在不同树枝上射击摆动的苹果。苹果摆动幅度会逐渐加大。击中金色苹果获得额外分数。

Shoot swinging apples from different branches. Apple swing amplitude increases over time. Hit gold apples for bonus points.

## 第三章 — 罗斯房间小游戏 / Rose's Room Minigames

### 7. 弹珠台 / Pinball

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 书桌下方 / Under the desk |
| 类型 / Type | 弹球 / Pinball |
| 获胜技巧 / Tip | 控制挡板时机，不要乱动 / Time your flippers, don't flap randomly |

经典弹珠台玩法。一人控制左挡板，一人控制右挡板。目标是获得最高分而非直接对抗。撞击特定目标可以激活加分模式。

Classic pinball. One controls the left flipper, the other the right. Goal is highest score, not direct competition. Hit specific targets for bonus modes.

### 8. 音乐指挥 / Music Maestro

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 音乐盒旁边 / Next to the music box |
| 类型 / Type | 节奏 / Rhythm |
| 获胜技巧 / Tip | 听节奏不看屏幕 / Listen to the rhythm, don't watch the screen |

根据音乐节奏按下对应的按键。准确度越高得分越多。连续完美命中会触发连击倍率。

Press corresponding buttons to the music rhythm. Accuracy determines score. Perfect streaks trigger combo multipliers.

### 9. 飞机大战 / Plane Battle

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 天花板吊扇下方 / Below ceiling fan |
| 类型 / Type | 空战 / Air Combat |
| 获胜技巧 / Tip | 抢占高度优势 / Gain altitude advantage |

控制纸飞机在空中进行狗斗。翻筋斗和俯冲可以闪避攻击。从上方攻击有命中加成。

Control paper planes in aerial dogfights. Loops and dives dodge attacks. Attacking from above has a命中加成.

## 第四章 — 雪景球小游戏 / Snowglobe Minigames

### 10. 冰壶挑战 / Curling Challenge

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 冰湖区域 / Frozen lake area |
| 类型 / Type | 策略 / Strategy |
| 获胜技巧 / Tip | 力道控制在 60-70% / Control power at 60-70% |

在冰面上进行冰壶比赛。目标是让冰壶停在最靠近中心的位置。可以用钩爪或身体撞击对手的冰壶来改变其位置。

Curling on ice. Goal is stopping your stone closest to the center. Use grappling hooks or body checks to move opponent stones.

### 11. 雪球大战 / Snowball Fight

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 城堡庭院 / Castle courtyard |
| 类型 / Type | 投掷战斗 / Throwing Combat |
| 获胜技巧 / Tip | 滚大雪球再投掷 / Roll bigger snowballs before throwing |

从地面收集雪，滚成雪球后投掷。雪球越大伤害越高，但投掷速度也越慢。躲在雪堡后面可以防御。

Gather snow from the ground, roll into snowballs, and throw. Bigger snowballs deal more damage but throw slower. Hide behind snow forts for defense.

### 12. 冰下钓鱼 / Ice Fishing

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 冰冻瀑布底部 / At the frozen waterfall base |
| 类型 / Type | 反应 / Reaction |
| 获胜技巧 / Tip | 看到浮标下沉再拉钩 / Wait for the bobber to sink before hooking |

在冰洞中钓鱼比赛。鱼出现时需要在恰当时机收线。不同颜色的鱼分值不同。白色鱼最稀有，得分最高。

Fishing contest in ice holes. Reel in at the right moment when fish appear. Different colored fish have different point values. White fish are rarest and score highest.

## 第五章 — 花园小游戏 / Garden Minigames

站在花朵上向飘落的花瓣射击。不同颜色的花瓣分值不同。红色花瓣在空中停留时间最短但分数最高。

Stand on flowers and shoot falling petals. Different colors have different point values. Red petals have the shortest air time but score highest.

骑乘蜜蜂在花丛中竞速。赛道中有花粉加速带和障碍物。控制蜜蜂上下飞行以避开障碍。

Ride bees racing through flowers. Pollen speed zones and obstacles dot the course. Control bee altitude to avoid obstacles.

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 位置 / Location | 向日葵下方 / Below sunflowers |

Fire seeds with a slingshot at targets. Distant targets score higher. Wind affects trajectory — watch the wind indicator.

## 第六章 — 阁楼小游戏 / Attic Minigames

### 16. 音乐记忆 / Memory Melody

| 属性 / Attribute | 详情 / Detail |

| 类型 / Type | 记忆 / Memory |
| 获胜技巧 / Tip | 默念节奏模式 / Silently repeat the rhythm pattern |

一段音乐模式会播放两次，玩家需要准确复现。模式长度逐渐增加。连续正确会触发加速模式。

A music pattern plays twice — players must replicate it. Pattern length increases. Consecutive correct answers trigger speed mode.

### 17. 齿轮迷宫 / Gear Maze

|---|---|

Control small gears through a maze to reach the exit. Rotating blades and moving walls block the path. Touching obstacles resets to the start.

## 奖杯成就 / Trophy & Achievement

**百科全书 / Encyclopedia** — 找到并游玩所有 25+ 小游戏。这是全收集奖杯中最费时的一个，建议通关后使用章节选择功能逐一检查遗漏。

**Encyclopedia** — Find and play all 25+ minigames. This is the most time-consuming collection trophy. Use Chapter Select after beating the game to check for missed ones.

### 小提示 / Pro Tips

- 每个小游戏只需要找到并**开始一局**即可计数，不需要获胜
- 如果漏掉某个小游戏，使用章节选择回到对应章节
- 有些小游戏需要特定能力才能进入（如需要钩爪）
- 在章节选择界面可以看到该章节已找到的小游戏数量
- Each minigame only needs to be **started once** to count — winning is not required
- Use Chapter Select to return to missed chapters
- Some minigames require specific abilities to access (e.g., Grappling Hook)
- Chapter Select shows how many minigames you've found in each chapter
