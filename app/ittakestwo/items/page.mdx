# Items Guide — All Tools, Gadgets & Collectibles

{/* ===== 物品概述 ===== */}

## 物品系统概述 / Item System Overview

《双人成行》中的物品分为三类：**章节专属能力工具**、**消耗品** 和 **收集品**。每个章节都会提供一套全新的工具，替换上一章的装备。收集品则包括帽子、小游戏等，分散在世界各处。

Items in *It Takes Two* fall into three categories: **chapter-specific ability tools**, **consumables**, and **collectibles**. Each chapter provides a brand-new toolset replacing the previous chapter's equipment. Collectibles include hats, minigames, and more scattered throughout the world.

## 第一章：棚屋工具 / Shed Tools

### Cody — 钉子发射器 / Nail Gun

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 远程工具 / Ranged Tool |
| 弹药 / Ammo | 自动回复 / Auto-regenerating |
| 伤害 / Damage | 中等 / Medium |
| 主要用途 / Main Use | 创造攀爬点、远程攻击 |

钉子发射器可以射出铁钉到木制表面。钉子可以作为 May 的立足点。游戏中后期钉子也可以用来触发远处的机关。弹药随时间自动回复，无需手动补充。

The Nail Gun fires iron nails into wooden surfaces. Nails serve as footholds for May. Later in the chapter, nails can also trigger distant mechanisms. Ammo auto-regenerates over time with no manual reload needed.

### May — 锤子 / Hammer

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 近战工具 / Melee Tool |
| 攻击范围 / Range | 中等 / Medium |
| 伤害 / Damage | 高 / High |
| 特殊功能 / Special | 地面冲击波、按压机关 |

锤子是 May 的主力武器和解谜工具。蓄力攻击可以击碎更坚固的障碍物。锤子的冲击波可以一次击中多个敌人。同时锤子可以压住需要持续施力的机关按钮。

The Hammer is May's primary weapon and puzzle tool. Charged attacks break tougher obstacles. The shockwave hits multiple enemies at once. The hammer can also hold down buttons requiring sustained pressure.

## 第二章：大树工具 / Tree Tools

### Cody — 树液发射器 / Sap Shooter

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 辅助工具 / Utility Tool |
| 弹药 / Ammo | 自动回复 / Auto-regenerating |
| 弹性 / Bounce | 高弹性 / High Elasticity |
| 主要用途 / Main Use | 创造弹跳表面、减速敌人 |

树液发射器射出的树液会在表面上形成弹性膜。弹跳高度取决于树液球的大小（由蓄力时间决定）。树液也可以用于暂时黏住敌人使其减速。

The Sap Shooter fires sap that forms an elastic membrane on surfaces. Bounce height depends on sap ball size (determined by charge time). Sap can also temporarily slow down enemies.

### May — 树液靴 / Sap Boots

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 移动工具 / Mobility Tool |
| 弹跳力 / Bounce Power | 蓄力可调 / Charge-adjustable |
| 踩踏伤害 / Stomp Damage | 中等 / Medium |
| 主要用途 / Main Use | 弹跳至高处、踩踏攻击 |

树液靴让 May 在树液表面弹跳。下落时踩踏敌人可以造成范围伤害。在树液上连续弹跳比单次弹跳可获得更多高度。

Sap Boots let May bounce on sap surfaces. Stomping enemies from above deals area damage. Consecutive bounces on sap grant more height than a single bounce.

## 第三章：罗斯房间工具 / Rose's Room Tools

### Cody — 火柴发射器 / Matchstick Shooter

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 元素工具 / Element Tool |
| 弹药 / Ammo | 自动回复 / Auto-regenerating |
| 火焰 / Fire | 点燃易燃物、融化冰面 |
| 主要用途 / Main Use | 点火、爆炸触发 |

火柴发射器射出燃烧的火柴点燃目标。火焰可以烧毁藤蔓、点燃火药桶、融化冰雪。与 May 的水结合产生蒸汽可以推动涡轮机关。

The Matchstick Shooter fires lit matches to ignite targets. Fire burns vines, ignites powder kegs, and melts ice. Combined with May's water, steam powers turbine mechanisms.

### May — 水枪 / Water Gun

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 元素工具 / Element Tool |
| 弹药 / Ammo | 无限 / Infinite |
| 水流 / Water Flow | 持续喷射 / Continuous spray |
| 主要用途 / Main Use | 灭火、清洁、推动水轮 |

水枪可以持续喷射水流，没有弹药限制。水流可以浇灭火墙、清洁脏污表面、推动水轮和水车。在导电场景中，水流可以传导电流。

The Water Gun fires a continuous stream with no ammo limit. Water extinguishes fire walls, cleans dirty surfaces, pushes water wheels, and conducts electricity in electrified scenes.

## 第四章：雪景球工具 / Snowglobe Tools

### Cody — 时间停止器 / Time Stopper

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 时间操控 / Time Manipulation |
| 冷却 / Cooldown | 约 8 秒 / Approximately 8 seconds |
| 范围 / Range | 中型区域 / Medium area |
| 持续时间 / Duration | 约 5 秒 / Approximately 5 seconds |

时间停止器可以暂停时间最长约 5 秒。被暂停的敌人和机关在时间恢复后会继续之前的动作。对快速旋转的齿轮和风扇使用效果最佳。

The Time Stopper pauses time for up to 5 seconds. Paused enemies and mechanisms resume their actions when time resumes. Most effective on快速旋转 gears and fans.

### May — 钩爪 / Grappling Hook

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 移动工具 / Mobility Tool |
| 射程 / Range | 长 / Long |
| 拉力 / Pull Force | 可拉动物体与人 |
| 主要用途 / Main Use | 摆荡、拉拽、快速移动 |

钩爪可以钩住金色锚点，将 May 拉向目标。也可以钩住可移动物体或小型敌人。摆荡时可以蓄力增加摆动幅度。

The Grappling Hook latches onto gold anchor points, pulling May toward them. It can also grab movable objects and small enemies. Charge while swinging to increase momentum.

## 第五章：花园工具 / Garden Tools

### Cody — 重力反转器 / Gravity Flipper

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 空间操控 / Spatial Manipulation |
| 冷却 / Cooldown | 约 6 秒 / Approximately 6 seconds |
| 范围 / Range | 固定区域 / Fixed zones |
| 效果持续时间 / Effect Duration | 持续生效 / Continuous |

重力反转器只能在特定的紫色区域内使用。反转后玩家可以站在天花板上行走。重力变化会影响敌人和场景中的物理物品。

The Gravity Flipper only works within specific purple zones. After flipping, players walk on ceilings. Gravity changes affect enemies and physics objects in the scene.

### May — 滑翔伞 / Glider

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 移动工具 / Mobility Tool |
| 持续时间 / Duration | 约 5 秒 / Approximately 5 seconds |
| 速度 / Speed | 缓慢下降 / Slow descent |
| 主要用途 / Main Use | 跨越距离、空中转向 |

滑翔伞在跳跃后按跳跃键激活。可以利用上升气流重新获得高度。滑翔时可以用方向键微调落点。接触障碍物会取消滑翔状态。

The Glider activates when pressing jump after a leap. Use updrafts to regain altitude. Fine-tune landing with directional keys. Contact with obstacles cancels gliding.

## 第六章：阁楼工具 / Attic Tools

### Cody — 分身装置 / Doppelganger Device

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 复制工具 / Clone Tool |
| 持续时间 / Duration | 约 10 秒 / Approximately 10 seconds |
| 冷却 / Cooldown | 约 15 秒 / Approximately 15 seconds |
| 主要用途 / Main Use | 同时操作多个机关 |

分身会完全复制 Cody 的动作。分身可以踩机关、搬运物体、吸引敌人。分身被攻击会提前消失。使用分身时本体可以做其他操作。

The clone perfectly mirrors Cody's movements. It can hold switches, carry objects, and distract enemies. The clone disappears early if attacked. Use it to perform simultaneous operations.

### May — 变身腰带 / Transformation Belt

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 类型 / Type | 形态变化 / Form Change |
| 形态数量 / Forms | 2 + 基础 / 2 + Base |
| 持续时间 / Duration | 各约 8 秒 / Approximately 8 seconds each |
| 冷却 / Cooldown | 各约 12 秒 / Approximately 12 seconds each |

**坦克形态 (Tank)**：矮小沉重，移动慢但可以压碎脆弱地板和障碍物。免疫某些击退效果。

**潜行形态 (Stealth)**：隐身且无声，可以绕过视线侦测类敌人。攻击会解除隐身。

**Tank Form**: Short and heavy, slow movement but crushes fragile floors and obstacles. Immune to some knockback effects.

**Stealth Form**: Invisible and silent, bypasses vision-based enemies. Attacking breaks stealth.

## 收集品 / Collectibles

### 帽子 / Hats

游戏中共有 **超过 30 顶帽子** 可供收集。帽子被放在发光的宝箱或隐藏区域中。收集后可以在主菜单装备。帽子只是装饰品，不影响游戏玩法。

There are **over 30 hats** to collect throughout the game. Hats are found in glowing chests or hidden areas. Equip them from the main menu. Hats are purely cosmetic and don't affect gameplay.

### 小游戏 / Minigames

详见《小游戏合集》章节。小游戏通过找到特定的互动点解锁，每个小游戏都是一场双人竞技。

See the Minigames chapter for details. Minigames are unlocked by finding specific interaction points, each being a two-player competition.

### 彩蛋 / Easter Eggs

详见《隐藏要素》章节。包含对《逃出生天》(A Way Out) 等游戏的致敬。

See the Secrets chapter for details. Includes references to *A Way Out* and other games.

## 收集品攻略提示 / Collectible Tips

1. **章节选择** / Chapter Select — 通关后使用章节选择功能补收集
2. **绕路探索** / Explore off the main path — 大部分收集品不在主线必经路线上
3. **注意发光物体** / Watch for glowing objects — 宝箱通常发出柔和的金光
4. **双人配合** / Both players needed — 许多收集品需要两人合作才能拿到
5. **不要着急推进** / Take your time — 每章结束后无法回到该章的收集区域

## 工具进阶使用 / Advanced Tool Usage

### 钉子枪进阶 / Nail Gun Advanced

- 钉子可以作为临时平台悬浮在空中
- 钉子可以击中远处的开关
- 在 May 弹跳时在落点射钉子可以延长她到达的高度

- Nails can serve as mid-air temporary platforms
- Nails can hit distant switches
- Shoot a nail at May's landing point mid-bounce to extend her reach

### 锤子进阶 / Hammer Advanced

- 蓄力锤击可以产生冲击波穿过窄缝
- 在旋转平台上使用锤子可以改变旋转方向
- 锤子的重击可以打断某些 BOSS 的技能

- Charged hammer strikes send shockwaves through narrow gaps
- Hammer strikes on rotating platforms can reverse rotation direction
- Heavy hammer hits can interrupt certain boss abilities

### 树液 + 时间停止 组合 / Sap + Time Stop Combo

这是游戏中最强大的组合技之一：Cody 用时间暂停冻结敌人，同时在地上铺设树液，May 在时间恢复后利用树液弹跳踩踏敌人。

One of the most powerful combos: Cody freezes enemies with Time Stop while laying sap on the ground. May bounces on the sap to stomp enemies when time resumes.
