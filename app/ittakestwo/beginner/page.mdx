# Beginner's Guide — Getting Started

{/* ===== 游戏概述 ===== */}

## 游戏概述 / Game Overview

《双人成行》是一款 **强制双人合作** 的动作冒险游戏。你无法单人游玩——必须与另一位玩家一起，无论是本地分屏还是在线联机。游戏节奏飞快，每章都会引入全新的机制和能力，让玩家始终保持新鲜感。

*It Takes Two* is a **mandatory co-op** action-adventure game. You cannot play alone — you must play with another player, either via local split-screen or online. The game moves at a rapid pace, introducing new mechanics and abilities in every chapter to keep players constantly engaged.

{/* ===== 如何开始 ===== */}

## 如何开始 / How to Start

### 本地分屏 / Local Split-Screen

1. 确保你有两个控制器连接至同一台设备
2. 启动游戏后，选择 **本地合作 (Local Co-op)**
3. P1 使用主账号，P2 使用访客账号
4. 主玩家在游戏中推进进度，访客玩家的进度也会保存

1. Make sure two controllers are connected to the same device
2. Launch the game and select **Local Co-op**
3. P1 uses the main account, P2 uses a guest account
4. The host player saves progress; the guest also retains progress

### 在线联机 / Online Co-op

1. 一位玩家购买完整版，另一位下载 **免费友票版 (Friend's Pass)**
2. 主玩家在游戏主菜单选择 **在线合作 (Online Co-op)** → **邀请好友**
3. 友票玩家接受邀请后自动加入
4. 只有主玩家的存档会记录进度

1. One player buys the full game; the other downloads the **Free Friend's Pass**
2. The host selects **Online Co-op** → **Invite Friend** from the main menu
3. The Friend's Pass player accepts and joins automatically
4. Only the host's save file tracks progress

## 新手合作技巧 / Co-op Tips for Beginners

### 沟通是第一位的 / Communication Comes First

《双人成行》中几乎每个谜题都需要 **两个人同步行动**。比如一个人按住机关，另一个人通过；两个人同时站在压力板上开门等。建议使用语音聊天保持实时沟通。

Almost every puzzle in *It Takes Two* requires **synchronized actions**. For example, one player holds a switch while the other passes through; both stand on pressure plates to open doors. Voice chat is highly recommended for real-time coordination.

### 视角合作 / Camera Cooperation

- 在平台跳跃时，建议控制视角的玩家将镜头拉远，方便队友看到落脚点
- 不要两个人同时乱转视角——容易导致晕眩和误操作
- 在战斗时，尽量让两名敌人在屏幕范围内

- During platforming, the player controlling the camera should zoom out so their partner can see landing spots
- Don't both spin the camera randomly — it causes disorientation and mistakes
- During combat, try to keep both enemies visible on screen

### 探索分工 / Explore Separately

许多场景有两条分支路径，建议两人分别探索，然后共享发现。这样能更快找到隐藏的小游戏和收集品。

Many areas have branching paths. Have each player explore a different route, then share what they found. This speeds up finding hidden minigames and collectibles.

### 死掉也不用怕 / Don't Worry About Dying

游戏有 **无限重试** 机制。死亡后会在最近的检查点立即复活，没有任何惩罚。放心大胆地去尝试吧！

The game has **unlimited retries**. You respawn immediately at the nearest checkpoint with no penalty. Feel free to experiment!

### 关于分屏 / About Split-Screen

- 分屏方向可以设置为 **水平** 或 **垂直**，在设置中调整
- 垂直分屏更适合宽屏显示器
- 水平分屏在标准 16:9 屏幕上视野更广
- 可以根据个人喜好随时切换

- Split-screen can be set to **horizontal** or **vertical** in the settings
- Vertical split works better on ultrawide monitors
- Horizontal split gives more vertical视野 on standard 16:9 screens
- You can switch at any time based on preference

## 常见问题 / FAQ

**Q: 我可以和不同平台的玩家一起玩吗？**
A: 不可以。PC 版 (Steam/Origin 可以互通)，主机版只能同平台。PS 不能和 Xbox 联机。

**Q: Can I play cross-platform?**
A: No. PC (Steam/Origin can play together), consoles are platform-locked. PS cannot play with Xbox.

**Q: 友票版需要 EA Play 订阅吗？**
A: 不需要。友票版完全免费，无需任何订阅。

**Q: Does the Friend's Pass require EA Play subscription?**
A: No. The Friend's Pass is completely free with no subscription required.

**Q: 存档位置在哪里？**
A: Steam 版存档在 `%USERPROFILE%/Documents/My Games/It Takes Two/`。

**Q: Where is the save file located?**
A: Steam version saves are in `%USERPROFILE%/Documents/My Games/It Takes Two/`.

## 章节选择 / Chapter Select

完成游戏后，你可以从主菜单的 **章节选择** 回到任意关卡补收集品。进度是每名玩家独立保存的。章节选择会保留你当前所有的能力和装备。

After completing the game, you can use **Chapter Select** from the main menu to revisit any chapter for missed collectibles. Progress is saved per player. Chapter Select retains all your current abilities and equipment.
