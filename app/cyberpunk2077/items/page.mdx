## 快速破解获取 / Quickhack Acquisition

### 购买 / Buy

| 等级 / Tier | 价格 / Price | 需求等级 / Level | 来源 / Source |
|---|---|---|---|
| 普通 / Common | 500-1000 | — | 网络义体医生 / Any netrunner ripperdoc |
| 稀有 / Uncommon | 2000-5000 | 智力 10+ | 黑客专用商人 / Netrunner vendors |
| 史诗 / Rare | 8000-15000 | 智力 14+ | 太平洋区/海伍德黑客商人 / Pacifica/Heywood |
| 传说 / Legendary | 20000-42000 | 智力 20 | 自动贩卖机"来生"/ 巫毒帮区域 / Afterlife vending machine, Voodoo Boys area |

### 制作 / Craft

1. 收集**快速破解制作规范**（Quickhack Crafting Specs）/ Collect Quickhack Crafting Specs
2. 需要对应等级的**快速破解技能** / Requires corresponding level in Breach Protocol
3. 传说快速破解需要 **智力 20** 和专长"遗忘"（Forget-Me-Not）/ Legendary quickhacks need Int 20 + "Forget-Me-Not" perk

### 偷取 / Loot

- 击杀黑客敌人有几率掉落 / Killing netrunner enemies has a chance to drop
- 完成NCPD击杀合约 / Complete NCPD scanner hustles
- 抄袭（Tier 5 专长）允许在击杀时复制敌方的快速破解 / "Copy-Paste" perk allows looting quickhacks on kill
