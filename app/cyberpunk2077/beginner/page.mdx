# 新手入门 / Beginner Guide

## 出身选择 / Lifepath Selection

| 出身 / Lifepath | 背景 / Background | 特色 / Unique Features |
|---|---|---|
| 流浪者 / Nomad | 出生在恶土游牧家族，擅长车辆和野外生存 / Born in the Badlands nomad clans, skilled with vehicles and survival | 专属开头章节（恶土线）、额外对话选项×12、获得武器"分则能成" / Exclusive opening (Badlands), 12+ unique dialogue options, gets "Divided We Stand" |
| 街头小子 / Street Kid | 在夜之城街头长大，熟悉黑市和帮派文化 / Grew up on Night City streets, knows the underworld | 专属开头章节（沃森线）、额外对话选项×15、黑市折扣 / Exclusive opening (Watson), 15+ unique dialogue options, black market discounts |
| 公司员工 / Corpo | 荒坂公司前员工，精通企业政治和黑客技术 / Former Arasaka employee, expert in corporate politics and netrunning | 专属开头章节（公司线）、额外对话选项×15、初始获得高级服装 / Exclusive opening (Corpo), 15+ unique dialogue options, starts with high-end clothing |

**推荐选择 / Recommendation**：街头小子的对话选项在主线任务中最为实用。公司员工线在涉及荒坂的任务中有优势。流浪者线对话最少。想要接近后续DLC《往日之影》剧情，流浪者背景有额外内容 / Street Kid offers the most practical dialogue options. Corpo has advantages in Arasaka-related quests. Nomad has additional content related to Phantom Liberty DLC.

## 属性系统 / Attribute System

游戏共有 **5 大属性**，各含若干专长技能。每个属性最高 **20 点**（含初始 + DLC解锁），50级总计可分配 **71 属性点** / 5 attributes, each with perk trees. Max 20 per attribute, 71 total points at level 50.

| 属性 / Attribute | 效果 / Effect | 核心流派 / Key Builds |
|---|---|---|
| 肉体 / Body | 近战伤害、生命值、耐力、碾压能力 / Melee dmg, HP, stamina, body checks | 莽夫流（霰弹枪+大锤） / Berserker (shotgun + hammer) |
| 反应 / Reflexes | 手枪/刀剑/步枪伤害、暴击率、闪避 / Pistol/blade/rifle dmg, crit, evasion | 时停流 / Sandevistan |
| 技术能力 / Technical Ability | 护甲、装备制作/升级、技术武器伤害 / Armor, crafting/upgrading, tech weapon dmg | 技术武器流 / Tech weapons |
| 智力 / Intelligence | 黑客伤害、RAM上限、快速破解品质 / Hack dmg, RAM cap, quickhack quality | 黑客流 / Netrunner |
| 镇定 / Cool | 潜行伤害、暴击伤害、冷血堆叠 / Stealth dmg, crit dmg, cold blood stacks | 潜行流 / Stealth + Cold Blood |

**加点优先级 / Leveling Priority**：
- **前期 (1-20)**：专注 2-3 个主属性，不要均衡加点 / Focus on 2-3 core attributes
- **中期 (20-40)**：主属性点满到 20，辅属性到 15-20 / Max main stat to 20, secondary to 15-20
- **后期 (40-50)**：补足剩余属性到至少 9 或更高 / Fill remaining to 9+

## 核心机制 / Core Mechanics

### 快速破解 / Quickhacks

黑客流的核心玩法，消耗 RAM 对敌人施加效果 / Core netrunner mechanic, consumes RAM to affect enemies.

| 快速破解 / Quickhack | 效果 / Effect | RAM | 推荐用途 / Usage |
|---|---|---|---|
| 义体故障 / Cyberware Malfunction | 禁用义体 / Disable cyberware | 6 | 针对 Boss |
| 短路 / Short Circuit | 高额电击伤害 + 暴击触发 / High shock dmg + crit proc | 8 | 通用输出 |
| 系统崩溃 / System Collapse | 即死（对普通敌人） / Instant kill (normal enemies) | 12 | 精英秒杀 |
| 传染 / Contagion | 病毒扩散 AoE / Viral spread AoE | 10 | 群体清场 |
| 内存撕裂/ Memory Wipe | 重置敌情状态 / Reset alert state | 6 | 潜行重置 |
| 赛博精神病 / Cyberpsychosis | 敌人攻击友军 / Enemy attacks allies | 14 | 混乱控场 |
| 祈祷 / Synapse Burnout | 极高单体伤害 / Very high single target | 16 | Boss 终结 |

**RAM 恢复 / RAM Recovery**：基础 0.1 RAM/s；义体"生物传导器"增加至 0.4 RAM/s；专长"过度clock"以 HP 换取临时 RAM / Base 0.1/s; Biosynergy cyberware bumps to 0.4/s; Overclock trades HP for RAM.

### 操作系统 / Operating System

操作系统决定你的核心战斗能力，**每人只能装备一个** / Determines your core combat ability, **one slot only**:

| 类型 / Type | 效果 / Effect | 适合流派 / Best For |
|---|---|---|
| 网络接入仓 / Netrunner Deck | 使用快速破解 / Enables quickhacks | 黑客流 |
| 斯安威斯坦 / Sandevistan | 时间减慢，提升暴击和闪避 / Slows time, boosts crit & evasion | 时停近战/手枪流 |
| 贝塞克 / Berserker | 狂暴模式，近战伤害大幅提升 / Berserk mode, huge melee boost | 肉体近战流 |

**强烈建议**：在主线"全食"（The Heist）任务前确定操作系统方向，因为之后会获得传说级接入仓 / Decide your OS direction before "The Heist" for optimal legendary reward.

## 战斗基础 / Combat Basics

### 武器类型 / Weapon Types

| 类型 / Type | 特点 / Traits | 弹药 / Ammo |
|---|---|---|
| 动能武器 / Power Weapons | 子弹可弹射穿墙 / Bullets ricochet off surfaces | 通用 |
| 技术武器 / Tech Weapons | 可充能穿透掩体 / Charge to pierce cover | 通用 |
| 智能武器 / Smart Weapons | 子弹自动追踪敌人 / Homing bullets | 特殊 |

### 伤害类型 / Damage Types

| 类型 / Type | 效果 / Effect |
|---|---|
| 物理 / Physical | 标准伤害 / Standard damage |
| 热能 / Thermal | 燃烧 DoT / Burn DoT |
| 化学 / Chemical | 中毒 DoT / Poison DoT |
| 电磁 / EMP | 对义体额外伤害 / Bonus vs cyberware |
| 电击 / Electrical | 额外对机器人伤害 / Bonus vs robots |

### 战斗技巧 / Combat Tips

1. **掩体射击**：技术武器充能可穿透掩体击杀敌人 / Tech weapons: charge to shoot through walls
2. **扫描敌人**：按 TAB 扫描敌人，可以看到弱点和掉落 / Scan enemies to see weaknesses and loot
3. **破墙击杀**：动能武器弹射可在敌人躲藏时击杀 / Power weapon ricochet kills enemies behind cover
4. **潜行击杀**：镇定属性高时，潜行一击必杀绝大多数敌人 / High Cool = stealth instant kill most enemies
5. **远程骇入**：即使不走黑客流，至少点 3 点智力，可骇入摄像头和炮塔 / Even 3 Int lets you hack cameras and turrets

## 发展方向 / Progression Path

```
1-10级 — 沃森区自由探索，完成街头合约和NCPD任务 / Watson free roam, gigs & NCPD
  ↓
  主线"取货"（The Pickup）— 决定是否救布里克 / The Pickup — decide Brick's fate
  ↓
10-15级 — 完成"全食"（The Heist）前后任务 / Complete missions around The Heist
  ↓
15-25级 — 解锁其他区域，做支线和合约 / Unlock other districts, side jobs & gigs
  ↓
  主线"电波"（Transmission）— 找艾芙琳 / Transmission — find Evelyn
  ↓
25-35级 — 完成罗格/帕南/克里/朱迪支线 / Complete Rogue/Panam/Kerry/Judy sidequests
  ↓
  主线"虎口拔牙"（Play It Safe）— 攻入荒坂工业园 / Play It Safe — assault Arasaka industrial park
  ↓
35-45级 — 完成剩余支线和DLC内容 / Complete remaining side content & DLC
  ↓
  不可回头点（Point of No Return）— 见华子 / Meet Hanako at Embers
  ↓
45-50级 — 准备最终战，选择结局路线 / Prepare for final fight, choose ending path
```

**等级上限 / Level Cap**: 50级（1.6版本为60级，含DLC《往日之影》可达 60+ with Phantom Liberty）
