## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|:----:|
| **1** | **确定操作系统方向 / Choose OS Early** | 全食任务前决定玩黑客（网络接入仓）、时停（斯安威斯坦）还是狂暴（贝塞克）/ Decide Netrunner/Sandevistan/Berserker before "The Heist" |
| **2** | **不要均衡加点 / Don't Spread Attributes** | 前期专注1-2个主属性到20再考虑其他——样样通=样样松 / Focus 1-2 core attributes to 20 first — jack of all trades = master of none |
| **3** | **清光NCPD合约 / Clear NCPD Scanner Hustles** | 每个区域的蓝色击杀合约不仅是经验来源，还会掉传说装备制作规范 / Best XP source + legendary crafting spec drops |
| **4** | **找维克多升级义体 / Visit Viktor for Cyberware** | 沃森区义体医生维克多——早期就能安装强力义体 / Watson ripperdoc Viktor — strong early cyberware available |
| **5** | **保留不朽武器 / Keep Iconic Weapons** | 不朽武器有独有特效且可以升级到传说品质——不要拆解或卖掉 / Iconic weapons have unique effects and can upgrade to legendary — don't dismantle or sell |
| **6** | **注意不可回头点 / Watch Point of No Return** | 主线"夜曲"（Nocturne OP55N1）会触发最终战——做完全部支线再见华子 / "Nocturne OP55N1" triggers endgame — complete all side content before meeting Hanako |
| **7** | **善用扫描 / Use Scan Often** | TAB键扫描——标记敌人、发现弱点、偷取武器、骇入设备 / Tab to scan — mark enemies, find weaknesses, hack devices |
| **8** | **做帕南/朱迪/克里/罗格支线 / Do All Character Sidequests** | 这些支线不仅剧情精彩，还直接影响结局选择 / Affects available endings — essential for story and ending paths |
| **9** | **购买所有制作规范 / Buy All Crafting Specs** | 武器商的制作规范买齐后可以在家做传说武器 / Buy all specs from vendors to craft legendary weapons at home |
| **10** | **DLC《往日之影》建议30级+ / Phantom Liberty Recommended 30+** | DLC难度较高——建议主线推进到一定程度后游玩 / DLC is harder — recommended after mid-game progress |
