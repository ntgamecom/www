## 战斗系统概述 / Combat Overview

**中文：** 赛博朋克2077的战斗系统融合了第一人称射击、黑客入侵和近战格斗三种核心玩法。玩家可以根据自己的属性加点和操作系统选择，打造完全不同的战斗风格。2.0版本对战斗系统进行了彻底重做，引入了新的义体容量系统、重置了专长树，并大幅改进了敌人的AI行为。战斗的核心是灵活运用武器、操作系统、快速破解和义体的组合。

**English:** Cyberpunk 2077's combat system blends FPS shooting, netrunning, and melee combat. Players can craft entirely different playstyles based on attribute allocation and OS choice. Patch 2.0 completely overhauled combat with a new cyberware capacity system, reworked perk trees, and improved enemy AI. Combat mastery comes from combining weapons, OS abilities, quickhacks, and cyberware synergies.

## 武器分类详解 / Weapon Categories

**中文：** 游戏中有三类武器，各有独特的弹道机制。动能武器是传统实弹武器，子弹可在表面弹射；技术武器可充能穿透掩体；智能武器子弹自动追踪敌人。每种武器都有多个变体，包括普通、稀有、史诗和传说品质。不朽武器是独特的传说武器，每把都有专属特效和名称。

**English:** Three weapon classes with unique ballistic mechanics. Power weapons feature ricochet rounds off surfaces. Tech weapons charge to penetrate cover. Smart weapons fire homing bullets. Each class has variants from Common to Legendary quality. Iconic weapons are unique legendaries with special effects and names.

**动能武器 / Power Weapons**:

- 弹射系统：子弹可在硬表面弹射一次，解锁专长后可弹射多次 / Ricochet rounds bounce off hard surfaces once (multiple bounces with perks)
- 弹道协同：射速稳定，弹道可控 / Stable fire rate, predictable recoil
- 推荐专长：弹道反射（Ref 9）、弹射杀机（Ref 15） / Perks: Ricochet (Ref 9), Bounce Back (Ref 15)
- 代表武器：同志的铁锤（不朽左轮）、分则能成（不朽冲锋枪） / Iconic examples: Comrade's Hammer, Divided We Stand

**技术武器 / Tech Weapons**:

- 充能机制：按住射击键充能，释放时造成高额伤害并可穿透掩体 / Hold to charge, release for high damage through walls
- 穿墙击杀：充能满后可以穿透绝大多数掩体 / Full charge penetrates most cover
- 推荐专长：疯狂物理学（Tech 9）、大修（Tech 18） / Perks: Chitinous (Tech 9), Overdrive (Tech 18)
- 代表武器：零五（不朽狙击）、阿修罗（技术步枪） / Iconic examples: Breakthrough, Ashura

**智能武器 / Smart Weapons**:

- 追踪系统：子弹自动追踪目标，盲射也有高命中率 / Homing bullets, high accuracy even from hip-fire
- 目标锁定：需要智能连接义体，锁定后发射 / Requires Smart Link cyberware to lock on
- 推荐专长：智能子弹（Int 9）、全面追踪（Int 15） / Perks: Smart Bullets (Int 9), Total Tracking (Int 15)
- 代表武器：应龙（不朽冲锋枪）、八星铳（不朽手枪） / Iconic examples: Yinglong, Ba Xing Chong

## 操作系统 / Operating System

**中文：** 操作系统是战斗风格的核心决定因素，装备后不可更改（直到后期可更换为传说级）。三种操作系统分别对应三种完全不同的玩法。

**English:** The OS is the core of your combat identity. Once equipped, it can only be replaced later with legendary versions. Three OS types define three distinct playstyles.

**斯安威斯坦 / Sandevistan**:

- 激活后时间减慢 50%-85%，持续 8-16 秒 / Slows time by 50%-85% for 8-16 seconds
- 时停期间暴击率大幅提升 / Crit rate massively increased during slow-mo
- 适合流派：近战武士刀流、手枪爆头流、时停散弹流 / Best for: Katana melee, pistol headshots, shotgun bursts
- 传奇版本：阿波克拉芙·西尼斯福（-85% 时停，+500% 暴击伤害） / Legendary: Apogee Sandevistan (-85% slow, +500% crit dmg)
- 冷却时间：基础 60 秒，可通过专长和义体减少 / Base cooldown 60s, reducible via perks and cyberware

**网络接入仓 / Netrunner Deck**:

- 解锁所有快速破解能力 / Unlocks all quickhack abilities
- RAM 容量决定可使用的快速破解数量和强度 / RAM capacity determines hack volume and potency
- 适合流派：纯黑客流、黑客+手枪混合流 / Best for: Pure netrunner, hybrid netrunner+pistol
- 传奇版本：网络行者（+8 RAM，+4 额外RAM，快速破解传播 +2 目标） / Legendary: Netwatcher (+8 base RAM, +4 extra, hacks spread +2 targets)
- 核心机制：利用"传播"机制让快速破解在敌人之间扩散 / Core mechanic: Quickhack propagation between enemies

**贝塞克 / Berserker**:

- 激活后进入狂暴状态，近战伤害 +200%，生命恢复加速 / Berserk mode: +200% melee dmg, HP regen
- 免疫击倒和眩晕 / Immune to knockdown and stun
- 适合流派：纯近战莽夫流、猩猩臂拳击流 / Best for: Full melee brawler, Gorilla Arms boxer
- 传奇版本：巨熊（狂暴持续时间 +50%，击杀延长狂暴） / Legendary: Grizzly (50% longer berserk, kills extend duration)
- 注意：狂暴状态下无法使用武器，只能徒手或近战武器 / Note: Berserk disables guns, melee only

## 快速破解系统 / Quickhack System

**中文：** 快速破解是黑客流的战斗核心。使用网络接入仓时，可以通过按 TAB 扫描敌人后直接施加快速破解。每个快速破解消耗不同量的 RAM，效果从伤害、控制到即死不等。

**English:** Quickhacks are the netrunner's combat toolkit. Scan enemies with TAB and apply hacks directly. Each hack costs RAM and produces effects from damage to control to instant death.

| 快速破解 / Quickhack | 效果 / Effect | RAM消耗 / Cost | 解锁等级 / Level |
|---|---|---|---|
| 过载 / Overclock | 以HP为代价获得额外RAM / Trade HP for RAM | 被动 / Passive | 智力9 / Int 9 |
| 短路 / Short Circuit | 电击伤害+暴击触发 / Shock dmg + crit proc | 6 | 智力4 / Int 4 |
| 义体故障 / Cyberware Malfunction | 禁用敌人义体，大幅削弱精英 / Disable cyberware | 8 | 智力7 / Int 7 |
| 传染 / Contagion | 病毒扩散AoE，造成化学伤害 / Viral AoE poison | 10 | 智力11 / Int 11 |
| 系统崩溃 / System Collapse | 对普通敌人即死 / Instant kill normals | 14 | 智力15 / Int 15 |
| 赛博精神病 / Cyberpsychosis | 让敌人攻击友军 / Enemy attacks allies | 16 | 智力18 / Int 18 |
| 记忆擦除 / Memory Wipe | 重置敌人警戒状态 / Reset alert status | 8 | 智力12 / Int 12 |

**English:** Quickhack strategies vary by scenario. For stealth approaches, Memory Wipe and Reboot Optics allow silent infiltration. For open combat, Contagion spreads through crowds while Short Circuit melts single targets. Synapse Burnout is the ultimate single-target nuke. Cyberware Malfunction is particularly effective against bosses and mechs.

## 伤害类型与抗性 / Damage Types & Resistances

**中文：** 游戏中存在六种伤害类型，不同敌人对不同伤害有抗性或弱点。了解伤害克制关系可以大幅提升战斗效率。

**English:** Six damage types exist in the game. Enemies have varying resistances and weaknesses. Understanding the damage wheel significantly boosts combat effectiveness.

| 类型 / Type | 效果 / Effect | 克制对象 / Effective Against |
|---|---|---|
| 物理 / Physical | 标准弹药伤害 / Standard ballistic | 人类敌人 / Human enemies |
| 热能 / Thermal | 燃烧 DoT，持续伤害 / Burn DoT | 肉体敌人、有机生命 / Organic enemies |
| 化学 / Chemical | 中毒 DoT / Poison DoT | 生物、动物 / Biological enemies |
| 电磁 / EMP | 对电子设备额外伤害 / Bonus vs electronics | 机器人、无人机 / Robots, drones |
| 电击 / Electrical | 对机器人和义体额外伤害 / Bonus vs cyberware | 赛博改造敌人 / Cyber-augmented enemies |
| 腐蚀 / Corrosive | 对装甲目标特效 / Effective vs armor | 重甲敌人 / Heavily armored enemies |

## 实战技巧 / Combat Tips

**中文：** 在不同战斗场景下，灵活运用以下技巧可以显著提升生存能力和击杀效率。

**English:** Apply these combat tips across different scenarios to greatly improve survivability and efficiency.

**潜行暗杀 / Stealth Kills**:

1. 高镇定属性下，不触发警报可一击必杀绝大多数敌人 / High Cool enables one-hit stealth kills on most enemies
2. 消音武器和快速破解"记忆擦除"是潜行的最佳搭档 / Suppressed weapons + Memory Wipe are the stealth combo
3. 摄像头是潜行最大威胁，优先骇入关闭或摧毁 / Cameras are the biggest threat — disable or destroy them first
4. 掩体暗杀：在掩体后按F可拉过敌人暗杀 / Behind-cover assassinations with F key

**正面交火 / Open Combat**:

1. 利用滑铲（冲刺+蹲下）进行快速位移 / Slide (sprint + crouch) for rapid repositioning
2. 技术武器充能穿墙击杀躲在掩体后的敌人 / Tech weapons: charge and shoot through walls
3. 动能武器弹射可以击杀墙角后的敌人 / Power weapons: ricochet around corners
4. 使用手雷和诱饵弹打乱敌人阵型 / Grenades and decoy grenades disrupt enemy formations
5. 战斗中实时切换武器应对不同距离的敌人 / Swap weapons mid-fight for different engagement ranges

**载具战斗 / Vehicle Combat**:

1. 驾驶时可以使用手枪冲锋枪射击（单手武器） / Drive-by shooting with pistols and SMGs
2. 技术武器在车内可以充能射击 / Tech weapons can charge in vehicles
3. 载具可以撞击敌人造成碾压伤害 / Vehicles can ram enemies for crushing damage
4. 部分委托需要用车载武器 / Some gigs require vehicle-mounted weapons

**义体配合 / Cyberware Synergy**:

1. 强化腿（二段跳/蓄力跳）极大增强机动性 / Reinforced legs (double jump/charge jump) boost mobility
2. 强化皮肤提供额外护甲和元素抗性 / Subdermal armor adds protection and elemental resistance
3. 光学迷彩提供短距离隐身能力 / Optical camo grants short-term invisibility
4. 弹射系统提供额外闪避能力 / Projectile launch system gives extra dodge

## 战斗Build思路 / Build Philosophy

**中文：** 每个Build应该围绕一个核心战斗机制展开，不要贪多。以下是一些经过验证的强力Build框架。

**English:** Every build should revolve around one core combat mechanic. Here are proven build frameworks.

**时停武士刀流 / Sandy Katana Build**:
- 核心：反应20 + 镇定20，斯安威斯坦 + 觉/武士刀 / Core: 20 Reflexes + 20 Cool, Sandevistan + Jinichi-Maru/Katana
- 打法：开时停 → 绕后 → 一刀爆头秒杀 / Play: Activate Sandy → flank → one-shot headshot kill
- 关键专长：暴风骤雨、行云流水、不屈不挠 / Key perks: Hurricane, Flash of Steel, Air Dash

**纯粹黑客流 / Pure Netrunner Build**:
- 核心：智力20 + 镇定20，网络接入仓 / Core: 20 Int + 20 Cool, Netrunner Deck
- 打法：扫描 → 传染扩散 → 短路清理残血 / Play: Scan → Contagion spread → Short Circuit cleanup
- 关键专长：过度clock、快速传播、遗忘 / Key perks: Overclock, Spread, Forget-Me-Not

**技术散弹流 / Tech Shotgun Build**:
- 核心：技术能力20 + 肉体20，贝塞克或斯安威斯坦 / Core: 20 Tech + 20 Body, Berserker or Sandy
- 打法：充能散弹秒杀，近战补刀 / Play: Charge shotgun kill, melee cleanup
- 关键专长：疯狂物理学、大修、角斗士 / Key perks: Chitinous, Overdrive, Gladiator

**潜行狙击流 / Stealth Sniper Build**:
- 核心：镇定20 + 技术能力20 + 反应15 / Core: 20 Cool + 20 Tech + 15 Reflexes
- 打法：超远距离一枪一个，不触发警报 / Play: One-shot from distance, never trigger combat
- 关键专长：忍者、刺客、冷血、鬼魅无形 / Key perks: Ninja, Assassin, Cold Blood, Phantom

## 等级压制与难度 / Level Scaling & Difficulty

**中文：** 2.0版本引入了区域等级动态缩放机制。敌人等级会跟随玩家等级，但装备和义体的品质仍取决于玩家的探索和任务选择。困难模式下敌人AI更激进，攻击频率更高，且会使用快速破解反制玩家。

**English:** Patch 2.0 introduced dynamic level scaling. Enemy levels match the player, but gear quality still depends on exploration. On Very Hard difficulty, enemies are more aggressive, attack more frequently, and counter-hack the player.

| 难度 / Difficulty | 敌人伤害 / Enemy Dmg | 敌人生命 / Enemy HP | AI行为 / AI Behavior |
|---|---|---|---|
| 简单 / Easy | 60% | 70% | 反应迟钝 / Slow reactions |
| 普通 / Normal | 100% | 100% | 标准 / Standard |
| 困难 / Hard | 150% | 130% | 频繁使用闪避和掩体 / Frequent dodge+cover |
| 非常困难 / Very Hard | 200% | 170% | 包抄+黑客反制+手雷 / Flank + counter-hack + grenades |
