# 流派构建 / Build Guides

## 总览 / Overview

游戏五大属性对应多种流派。推荐 **20/20/20/9/3** 或 **20/20/18/12/3** 等配点方案实现强度最大化 / Each of the 5 attributes corresponds to viable builds. Endgame spreads typically 20/20/20/9/3 or similar.

---

## 🔥 黑客流 / Netrunner Build

利用快速破解远程控场和击杀，无需开枪即可清场 / Use quickhacks to control and kill without firing a shot.

**配点 / Attributes (50级)**:
```
智力 / Intelligence: 20
镇定 / Cool: 20
技术能力 / Tech: 18
肉体 / Body: 9
反应 / Reflexes: 3
```

**核心专长 / Core Perks**:
- **遗忘 (Int 20)**：快速破解永不遗忘 / Quickhacks never forgotten
- **过度clock (Int 9)**：HP换RAM，绝境反杀 / Trade HP for RAM
- **快速传播 (Int 14)**：快速破解可连锁传播 / Contagion spread
- **连杀/协同杀敌 (Int 11)**：击杀回复 RAM / Kill grants RAM refund
- **抄袭 (Int 15)**：击杀可拾取新快速破解 / Kill to loot quickhacks
- **冷血 (Cool 14)**：击杀叠层增伤 / Kill stacks for damage
- **忍者 (Cool 7)**：潜行伤害加成 / Stealth damage

**操作系统 / OS**: 传奇网络接入仓 — 最好的是 **荒坂绝密（Tetratronic Rippler）** / Legendary Netrunner Deck — best: Tetratronic Rippler

**推荐武器 / Weapons**:
- 智能手枪《分则能成》（Divided We Stand）— 配合传染 / Smart pistol with Contagion
- 技术冲锋枪《电锯》（Skippy）— 可切换慈悲/暴力模式 / Tech SMG with pacifist/killer modes
- 单分子线（Monomolecular Wire）— 配合黑客连招 / Combo with quickhacks

**战术 / Tactics**:
1. 先用"义体故障"削弱精英/Boss / Start with Cyberware Malfunction on elites
2. "短路"作为主力输出快速破解 / Short Circuit as main DPS quickhack
3. "传染" + "过热"触发化学爆炸连招 / Contagion + Overheat for chemical explosion
4. 被围攻时开"过度clock"用HP硬顶 / Overclock when overwhelmed

**强度评估 / Tier**: S — 清场能力最强，但面对机器人和Boss稍弱 / Best crowd clear, weaker vs robots and bosses

## ⚡ 时停武士刀流 / Sandevistan Katana Build

时间减慢 + 暴击爆伤 = 战斗中最快的男人 / Slow time + crit = fastest killer in combat.

**配点 / Attributes (50级)**:
```
反应 / Reflexes: 20
镇定 / Cool: 20
技术能力 / Tech: 18
肉体 / Body: 9
智力 / Intelligence: 3
```

**核心专长 / Core Perks**:
- **行云流水 (Ref 20)**：刀剑终结技连杀 / Blade finishers chain
- **暴风骤雨 (Ref 15)**：击杀重置时停冷却 / Kills reset Sandevistan cooldown
- **不屈不挠 (Ref 20)**：空中冲刺闪避减伤 / Air dash with damage reduction
- **刺客 (Cool 11)**：对满血敌人暴击率+100% / 100% crit vs full HP enemies
- **冷血 (Cool 14)**：击杀叠暴击爆伤 / Kill stacking for crit
- **暗影使徒 (Cool 20)**：潜行+冷血联动 / Stealth + Cold Blood synergy

**操作系统 / OS**: 斯安威斯坦 — 最好的是 **千里冰封（QianT Sandevistan "Warp Dancer"）** / 最佳属性：时停+30%，暴击+35%，冷却15秒 / Best Sandevistan: +30% time slow, +35% crit, 15s cooldown

**推荐武器 / Weapons**:
- 不朽武士刀《觉》（Satori）— 绀碧大厦楼顶获得 / Legendary katana from Konpeki Plaza rooftop
- 不朽武士刀《尽忠》（Byakko）— 完成"黑拳"支线 / Beat on the Brat reward
- 不朽小刀《痣》（Stinger）— 投掷流血流 / Throwing knife with bleed
- 动能手枪《应龙》（Yinglong）— 副武器/适用于远程 / Backup ranged weapon

**战术 / Tactics**:
1. 开时停前先用扫描标记所有敌人 / Scan-mark all enemies before activating Sandy
2. 时停期间优先击杀精英/威胁大的目标 / Prioritize elites during slowdown
3. 利用"暴风骤雨"在时停结束前击杀重置冷却 / Reset Sandevistan before duration ends
4. 空中冲刺躲避子弹同时接近敌人 / Air dash to dodge and close distance

**强度评估 / Tier**: S — 爆发力最高，操作上限高，成型后近乎无敌 / Highest burst, high skill ceiling, nearly unstoppable when complete

## 🔫 技术喷子流 / Tech Shotgun Build

用技术武器充能穿透掩体，一枪一片 / Charge tech weapons to pierce cover, devastate groups.

**配点 / Attributes (50级)**:
```
技术能力 / Tech: 20
反应 / Reflexes: 20
肉体 / Body: 20
镇定 / Cool: 8
智力 / Intelligence: 3
```

**核心专长 / Core Perks**:
- **大修 (Tech 18)**：技术武器充能速度+100% / Tech weapon charge speed doubled
- **疯狂物理学 (Tech 9)**：暴击可眩晕敌人 / Crits stun enemies
- **充能射击 (Tech 12)**：充能穿透掩体无限敌人 / Charge pierces infinite enemies
- **百战不殆 (Ref 11)**：换弹速度+60% / Reload speed +60%
- **角斗士 (Body 9)**：近战蓄力击倒 / Charged melee knockdown
- **横冲直撞 (Body 20)**：击杀回血+减伤 / Kill gives HP + damage reduction
- **不屈不挠 (Ref 20)**：空中冲刺 / Air dash

**操作系统 / OS**: 斯安威斯坦（推荐）或贝塞克 / Sandevistan (recommended) or Berserker

**推荐武器 / Weapons**:
- 不朽技术霰弹枪《电锯》（Ba Xing Chong）— 全自动充能霰弹 / Full-auto charged shotgun
- 不朽技术霰弹枪《主权》（Sovereign）— 双管高爆发 / Double-barrel high burst
- 技术精准步枪《突破》（Breakthrough）— 远程补充 / Ranged supplement
- 技术左轮《毒龙钻》（Drill— Charged Tech Revolver

**战术 / Tactics**:
1. 开战前先充能 — 技术武器在充能状态下伤害翻倍 / Pre-charge tech weapons for 2x damage
2. 利用掩体充能后探头射击 / Charge behind cover, peek and fire
3. 霰弹枪配合"横冲直撞"实现持续续航 / Shotgun + "You Can't Kill" for sustain
4. 充能射击可以穿透墙壁和多名敌人 / Charge shot pierces walls and multiple enemies

**强度评估 / Tier**: A+ — 适合喜欢正面刚的玩家，操作简单但效率极高 / Great for aggressive play, simple but highly effective

## 🗡️ 潜行暗杀流 / Stealth Assassin Build

消音武器 + 爆头伤害 = 一枪一个不被发现 / Silencer + headshots = one-shot kills without detection.

**配点 / Attributes (50级)**:
```
镇定 / Cool: 20
反应 / Reflexes: 20
技术能力 / Tech: 18
肉体 / Body: 9
智力 / Intelligence: 3
```

**核心专长 / Core Perks**:
- **鬼魅无形 (Cool 20)**：完全隐身（移动时也有效） / Near-invisible even while moving
- **刺客 (Cool 11)**：满血敌人+100%暴击率 / 100% crit vs full HP
- **冷血 (Cool 14)**：潜行击杀叠层 / Stealth kill stacking
- **潜行者 (Cool 15)**：闪避不取消潜行 / Dodging doesn't break stealth
- **行云流水 (Ref 20)**：刀剑连杀 / Blade kills chain
- **破隐一击 (Cool 9)**：从匿踪中攻击+15%伤害 / +15% from stealth

**操作系统 / OS**: 斯安威斯坦（潜行时停秒杀流）或 网络接入仓（可选） / Sandevistan (for stealth time-freeze kills) or Netrunner Deck (optional)

**推荐武器 / Weapons**:
- 不朽消音手枪《死神》（Death and Taxes）— 不用换弹 / No reload needed
- 不朽左轮《同志的铁锤》（Comrade's Hammer）— 一发射击 / Single shot nuke
- 不朽狙击枪《突破》（Breakthrough）— 穿墙击杀 / Wall-penetrating sniper
- 不朽匕首《痣》（Stinger）— 投掷必杀 / Throwing knife

**战术 / Tactics**:
1. 先部署 ping（黑客扫描）标记所有敌人，然后逐个消灭 / Ping all enemies first, then eliminate one by one
2. 消音武器 + 爆头 + 镇定满 = 绝大多数敌人一击必杀 / Silenced weapons + headshot + max Cool = one-shot
3. 被发现时开时停紧急击杀附近敌人 / Use Sandevistan for emergency kills if detected
4. 利用高潜行移动速度绕后 / Use high stealth move speed to flank

**强度评估 / Tier**: A+ — 前期较弱，后期成型后可以无警报通关据点 / Weak early, endgame can clear camps undetected

## 💥 肉体莽夫流 / Body Berserker Build

不用枪，纯靠拳头/大锤/猩猩臂打穿夜之城 / No guns, just fists/hammers/gorilla arms.

**配点 / Attributes (50级)**:
```
肉体 / Body: 20
技术能力 / Tech: 20
反应 / Reflexes: 18
镇定 / Cool: 9
智力 / Intelligence: 3
```

**核心专长 / Core Perks**:
- **横冲直撞 (Body 20)**：击杀回血+减伤 / Kill = HP + damage reduction
- **角斗士 (Body 9)**：蓄力击倒重甲敌人 / Charged knockdown for armored enemies
- **热血沸腾 (Body 11)**：生命越低伤害越高 / Low HP = more damage
- **大猩猩抓取 (Body 15)**：抓取敌人投掷 / Grab and throw enemies
- **大修 (Tech 18)**：降低义体散热需求 / Reduced cyberware cooldown
- **不屈不挠 (Ref 20)**：空中冲刺近身 / Air dash to close distance

**操作系统 / OS**: 贝塞克 — 最佳的是 **荒坂贝塞克（Biodyne Berserker MKI）** / Best Berserker: Biodyne MKI

**推荐义体武器 / Cyberware Weapons**:
- **大猩猩臂 / Gorilla Arms** — 拳击流首选，可击穿墙壁 / Best for brawler, can punch through walls
- **单分子线 / Monowire** — 群体范围攻击 / AoE sweep attack
- **螳螂刀 / Mantis Blades** — 高攻速近战 / High attack speed melee

**战术 / Tactics**:
1. 开贝塞克进入狂暴，免疫控制 + 高额减伤 / Activate Berserker for CC immunity + damage reduction
2. 蓄力重击击倒敌人后追击 / Charged heavy knockdown then ground pound
3. 投掷敌人造成 AoE 伤害 + 眩晕 / Throw enemies for AoE + stun
4. 利用"热血沸腾"在低血量时打出巨额伤害 / Use "Rush" passive at low HP for massive damage bonus

**强度评估 / Tier**: A — 娱乐性强，前期比较吃力，后期装甲成型后极硬 / Fun, struggling early, nearly immortal late game
