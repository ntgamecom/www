## 任务系统概述 / Quest System Overview

**中文：** 赛博朋克2077的任务分为主线、支线（Side Jobs）、委托（Gigs）和NCPD合约（Scanner Hustles）四种类型。主线和部分重要支线直接影响剧情走向和结局选择。委托和NCPD合约是经验值和装备的主要来源。DLC《往日之影》增加了狗镇区域的独立任务线。

**English:** Quests in Cyberpunk 2077 are divided into Main Jobs, Side Jobs, Gigs, and NCPD Scanner Hustles. Main Jobs and key Side Jobs affect story outcomes and ending availability. Gigs and NCPD Hustles are the primary source of XP and gear. The Phantom Liberty DLC adds an independent questline in Dogtown.

| 任务类型 / Type | 图标 / Icon | 数量 / Count | 重要性 / Importance |
|---|---|---|---|
| 主线 / Main Jobs | 黄色感叹号 / Yellow ! | 25+ | 推进剧情，影响结局 / Advance story, affect endings |
| 支线 / Side Jobs | 黄色问号 / Yellow ? | 40+ | 角色故事，装备奖励 / Character stories, gear rewards |
| 委托 / Gigs | 蓝色标记 / Blue marker | 80+ | 经验经验来源 / Best XP source |
| NCPD合约 / Scanner Hustles | 蓝色盾牌 / Blue shield | 100+ | 装备制作规范掉落 / Crafting spec drops |

{/* ===== 主线流程 / Main Story Flow ===== */}

## 主线流程 / Main Story Flow

**中文：** 主线任务分为三个主要章节。第一章在沃森区，第二章解锁全城，第三章导向结局。建议在第二章尽量完成支线内容。

**English:** The main story is divided into three acts. Act 1 is locked to Watson, Act 2 opens the entire city, Act 3 leads to endgame. Complete side content during Act 2.

**第一章 / Act 1**:

1. **人生如梦 / The Rescue** — 和杰克去救桑德拉·多赛特 / Rescue Sandra Dorsett with Jackie
2. **取货 / The Pickup** — 在漩涡帮地盘交易军用科技芯片 / Deal with Maelstrom over the Militech chip
3. **情报 / Information** — 找黑客大佬解码芯片 / Find T-Bug to decode the chip
4. **全食 / The Heist** — 和杰克潜入荒坂大厦夺取RELIC芯片 / Infiltrate Arasaka tower for the Relic
5. **爱的代价 / The Price of Freedom** — 杰克牺牲，V被爆头 / Jackie dies, V gets shot

**注意 / Note**: 完成"全食"之前一定要确定操作系统方向，任务中会赠送传说级操作系统 / Decide OS direction before The Heist — you get a free legendary OS during the mission.

**第二章 / Act 2**:

1. **黑梦 / Playing for Time** — 找维克多处理RELIC问题 / Visit Viktor for Relic treatment
2. **不由自主 / Automatic Love** — 找朱迪去云顶调查 / Find Judy, investigate Clouds
3. **电波 / Transmission** — 找到艾芙琳·帕克 / Locate Evelyn Parker
4. **虎口拔牙 / Play It Safe** — 攻入荒坂工业园 / Assault Arasaka Industrial Park
5. **夜曲 / Nocturne OP55N1** — 见华子触发最终选择 / Meet Hanako at Embers

**第三章 / Act 3**: 根据你的选择进入不同结局路线 / Different ending paths based on choices.

## 重要支线 / Key Side Jobs

**中文：** 以下支线直接影响结局解锁和最强装备获取，建议务必完成。

**English:** These side jobs affect ending unlocks and best-in-slot gear. Complete them all.

**帕南 / Panam (流浪者)**:
- 任务链："沙漠之怒"→"寻宝"→"冲破风暴"→"反目成仇"→"在最黑暗的夜晚"
- 解锁结局：太阳结局（和阿德卡多流浪者一起离开夜之城） / Unlocks The Sun ending
- 奖励：传奇狙击步枪"守望"、"帕南的狙击枪"、阿德卡多载具 / Legendary sniper Overwatch, Panam's rifle, Aldecaldo vehicles

**朱迪 / Judy (Mox帮黑客)**:
- 任务链："幻灭"→"镜中幻影"→"漂浮在城市上空"→"金字塔之歌"
- 解锁结局：V的死亡结局相关 / Affects certain ending variations
- 奖励：传奇技术手枪"八星铳" / Legendary tech pistol Ba Xing Chong

**克里 / Kerry (摇滚歌手)**:
- 任务链："重生"→"第二次冲突"→"我最亲爱的"→"反抗！反抗！"
- 解锁结局：太阳结局相关 / Affects The Sun ending
- 奖励：传奇手枪"骄傲" / Legendary pistol Pride

**罗格 / Rogue (来生酒吧女王)**:
- 任务链："插管"→"废物"→"敲开天堂之门"→"直到死亡把我们分开"→"魔王"
- 解锁结局：传奇结局（罗格线攻击荒坂塔） / Unlocks legendary Rogue ending
- 奖励：传奇动能步枪"守望先锋" / Legendary rifle Overwatch

**中文：** 赛博朋克2077有五个基础结局（加上DLC的第六个结局）。结局选择在最终任务"夜曲"中决定。

**English:** Cyberpunk 2077 has 5 base endings (plus a 6th from the DLC). The choice is made during the final mission "Nocturne OP55N1".

## DLC往日之影任务 / Phantom Liberty Questline

**中文：** 《往日之影》是赛博朋克2077的大型谍战DLC，发生在狗镇区域。任务线大约15-20小时，包含全新的角色和剧情分支。

**English:** Phantom Liberty is the major spy-thriller DLC set in Dogtown. The questline is roughly 15-20 hours with new characters and branching narrative.

**重要任务流程 / Key Quest Flow**:

1. **狗咬狗 / Dog Eats Dog** — 进入狗镇，遇到百灵鸟 / Enter Dogtown, meet Songbird
2. **黑洞佣兵 / The Black Steel** — 在体育馆救出百灵鸟 / Rescue Songbird from the stadium
3. **直到大厦崩塌 / The Killing Moon** — 决定百灵鸟或里德的命运 / Decide Songbird or Reed's fate
4. **比远方更远 / Somewhat Damaged** — 追踪百灵鸟到月球设施 / Track Songbird to the lunar facility
5. **王冠 / The Tower** — DLC最终选择 / DLC final choice

**关键选择 / Key Choices**: 在"直到大厦崩塌"任务中，选择帮助百灵鸟逃跑或通知里德，将决定结局路线。两个选择都有独特奖励和后续任务。

## 委托与合约攻略 / Gigs & NCPD Tips

**中文：** 委託是游戏中最稳定的经验和金钱来源。每个区域的委托做完后可以解锁该区域的传说装备制作规范。NCPD合约虽然重复性高，但掉落的制作规范是自制传说装备的关键。

**English:** Gigs are the most reliable XP and money source. Completing all gigs in a district unlocks legendary crafting specs. NCPD Scanner Hustles are repetitive but drop essential crafting specs for legendary gear.

**委托小贴士 / Gig Tips**:
- 每个区域做完30个委托可获得该区域专属奖励 / Every 30 gigs in a district = district-specific reward
- 委托可以潜行完成，不影响奖励 / Gigs can be stealthed without penalty
- 委托失败可以回退到上一个存档点 / Failed gigs can be reloaded from last checkpoint
- 选择对话选项有时可以额外获得金钱或装备 / Dialogue choices sometimes yield extra rewards

**NCPD合约小贴士 / NCPD Tips**:
- 合约敌人等级跟随玩家，但掉落品质固定 / Enemy levels scale, but drop quality is fixed
- 每个合约必定掉落1-2件装备和制作材料 / Each hustle drops 1-2 gear pieces + crafting materials
- 蓝色合约是武士刀和手枪制作规范的主要来源 / Blue hustles are the best source of katana/pistol specs

## 时间敏感任务 / Time-Sensitive Quests

**中文：** 部分任务有不可逆的时间节点，错过将永久无法完成。

**English:** Some quests have irreversible time gates. Miss them and they are permanently locked.

| 任务 / Quest | 截止点 / Deadline | 错过后果 / Consequence |
|---|---|---|
| 向战乱说再见 / Goodbye to the War | 完成"夜曲"前 / Before Nocturne | 永久失败 / Permanently fails |
| 战士 / Soldier | 完成"虎口拔牙"前 / Before Play It Safe | 永久失败 / Permanently fails |
| 家族事宜 / Family Business | 完成"反抗！反抗！"前 / Before Rebel Rebel | 永久失败 / Permanently fails |
| 拉面订单 / Ramen Order | 完成"电波"前 / Before Transmission | 永久失败 / Permanently fails |

**建议 / Advice**: 主线推进到"虎口拔牙"后，做一个全支线清扫，然后再推进到"夜曲" / After "Play It Safe", clear all side content, then proceed to "Nocturne".
