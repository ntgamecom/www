# 夜之城探索 / Night City World Exploration

## 区域总览 / District Overview

夜之城共分 **7 大区域**，各有不同的氛围、帮派势力和任务密度 / 7 major districts, each with unique atmosphere, gang presence, and quest density.

| 区域 / District | 帮派势力 / Gangs | 探索等级 / Level | 特色 / Highlights |
|---|---|---|---|
| 沃森 / Watson | 虎爪众 / Tigers | 1-15 | 开局区域、绀碧大厦、歌舞伎集市 / Starting area, Konpeki Plaza, Kabuki market |
| 威斯布鲁克 / Westbrook | 虎爪众/漩涡帮 / Tigers/Maelstrom | 15-25 | 日本街、北橡区豪宅 / Japantown, North Oak mansions |
| 市政中心 / City Center | 荒坂/军用科技 / Arasaka/Militech | 20-35 | 公司广场、巨型摩天楼 / Corporate Plaza, megabuildings |
| 海伍德 / Heywood | 瓦伦蒂诺帮 / Valentinos | 10-25 | 全食品工厂、谷地区 / All Foods plant, The Glen |
| 圣多明戈 / Santo Domingo | 漩涡帮/六街帮 / Maelstrom/6th Street | 10-20 | 工业区、荒坂工业园 / Industrial zone, Arasaka industrial park |
| 太平洋区 / Pacifica | 动物帮 / Animals | 25-35 | 废弃主题公园、巫毒帮地盘 / Abandoned theme park, Voodoo Boys turf |
| 恶土 / Badlands | 流浪者 / Nomads | 5-40 | 开放沙漠、隐藏洞穴和营地 / Open desert, hidden caves & camps |

## 沃森区 / Watson

夜之城的 **初始区域**，也是前半段主线的主要舞台 / Starting district, main stage for early main quests.

### 重要地点 / Key Locations

- **绀碧大厦 / Konpeki Plaza**：荒坂豪华酒店，主线"全食"任务地点 / Arasaka luxury hotel, site of "The Heist"
- **歌舞伎集市 / Kabuki Market**：黑市商人、义体医生聚集地 / Black market vendors, ripperdocs
- **小唐人街 / Little China**：早期合约密集区域 / Dense gig area early game
- **北区 / Northside**：漩涡帮据点，主线"取货"地点 / Maelstrom turf, "The Pickup" location

### 沃森区合约 / Watson Gigs (15 个)

| 合约 / Gig | 委托人 / Client | 难度 / Difficulty |
|---|---|---|
| 冷战 / Cold Storage | 罗格 / Rogue | 简单 |
| 数据掠夺者 / Data Miner | 法老 / Fahrad | 简单 |
| 恶癖 / Vices | 神父 / Padre | 中等 |
| 家族猎杀 / Family Heist | 神父 / Padre | 中等 |

## 威斯布鲁克 / Westbrook

**日本街 / Japantown**：虎爪众核心地盘，夜生活集中地 / Tiger Claws core territory, nightlife hub

- **来生酒吧 / Afterlife**：传奇佣兵聚集地，罗格和银手的据点 / Legendary merc bar, Rogue & Silverhand's haunt
- **樱花市场 / Cherry Blossom Market**：多条支线任务的触发点 / Multiple side quest triggers
- **北橡区 / North Oak**：裘德·史密斯的豪宅所在地 / Kerry's mansion location

**隐藏内容**：日本街某公寓楼顶有传说武器"金刚"（Ba Xing Chong）制作规格 / Japantown rooftop - legendary "Ba Xing Chong" crafting spec

---

## 市政中心 / City Center

夜之城核心区，荒坂大厦所在地 / Core district, Arasaka Tower location

- **公司广场 / Corporate Plaza**：最终决战地点之一 / One of final battle locations
- **荒坂大厦 / Arasaka Tower**：结局任务"恶魔"入口 / Ending "The Devil" entry point
- **巨型建筑 H10 公寓 / Megabuilding H10**：V 的初始公寓 / V's starting apartment

**隐藏内容**：公司广场地下停车场有一个传说击杀/传奇敌人 / Corporate Plaza underground parking garage has a legendary enemy drop

---

## 海伍德 / Heywood

夜之城最大区域，瓦伦蒂诺帮控制 / Largest district, Valentinos territory

- **全食品工厂 / All Foods Plant**：主线任务地点，内有传奇武器"前奏"（Chaos）/ Main quest location, contains legendary "Chaos"
- **谷地区 / The Glen**：多段支线任务发生地 / Multiple side quests
- **美景巷 / Vista del Ray**：NCPD 击杀任务密集区 / Dense NCPD scanner hustle area
- **学院街 / College Street**：传说物品商人所在地 / Legendary item vendor

**隐蔽传点 / Fast Travel Hack**：完成海伍德 NCPD 任务可解锁多个隐藏传送点 / Completing Heywood NCPD hustle unlocks hidden fast travel points

---

## 圣多明戈 / Santo Domingo

工业区 + 住宅区 / Industrial + residential

- **荒坂工业园 / Arasaka Industrial Park**：主线"虎口拔牙"任务地点 / "Play It Safe" main quest location
- **科罗纳多农场 / Rancho Coronado**：六街帮控制区域 / 6th Street gang territory
- **河谷 / Arroyo**：帕南的营地最初在此 / Panam's initial camp location

**隐藏内容**：圣多明戈下水道有一把不朽单手棍/一场战斗 / Santo Domingo sewers contain an iconic weapon

---

## 太平洋区 / Pacifica

夜之城最危险的区域，尚未完工就被废弃 / Most dangerous district, unfinished and abandoned

- **大帝国购物中心 / The Grand Imperial Mall**：巫毒帮据点 / Voodoo Boys base
- **主题公园 / Playground**：主线"分裂立场"（I Walk the Line）地点 / "I Walk the Line" main quest
- **海岸观景点 / Coastview**：动物帮核心地盘 / Animals core territory
- **蝙蝠酒店 / Hotel Bat**：小规模战斗区域包 / Small battle area with loot

**重要提示**：太平洋区没有义体医生/武器商等常规设施 / Pacifica lacks normal amenities like ripperdocs and weapon vendors

---

## 恶土 / Badlands

夜之城外面的广阔荒漠 / Vast desert outside Night City

- **垃圾场 / The Junkyard**：帕南相关任务的场景 / Panam quest related location
- **加油站 / Gas Station**：多个随机事件的刷新点 / Multiple random encounter spawns
- **地下洞穴 / Underground Caves**：传说武器制作规范所在地 / Legendary weapon crafting spec locations
- **游牧民营地 / Nomad Camp**：帕南部落营地 / Panam's clan camp

**收集要点**：恶土散布着 5 处传说武器制作规格，建议用摩托车快速搜索 / 5 legendary weapon specs scattered across the Badlands; use motorcycle for fast search

## 支线任务推荐顺序 / Side Quest Recommended Order

### 帕南线 / Panam's Path → 结局"星星" / Ending "The Sun"
1. **亡命之徒 / Riders on the Storm** — 25级后触发
2. **天外奇蹟 / With a Little Help from My Friends** — 30级后
3. **战火 / Queen of the Highway** — 全系列任务终点

### 朱迪线 / Judy's Path → 结局"太阳"影响 / Influences "The Sun"
1. **空间之海 / Both Sides, Now** — 25级后
2. **身份危机 / Ex-Factor** — 自动推进
3. **金字塔之歌 / Pyramid Song** — 朱迪支线终点，需要恋爱选项

### 克里线 / Kerry's Path → 结局"太阳"分支 / "The Sun" branch
1. **二次冲突 / Second Conflict** — 30级后
2. **反抗之歌 / I Don't Wanna Hear It** — 35级后
3. **摇滚巨星 / A Cool Metal Fire** — 最终任务

### 罗格线 / Rogue's Path → 结局"太阳" / Ending "The Sun"
1. **插管 / Chippin' In** — 强尼银手相关任务链
2. **梦殒 / Blistering Love** — 与罗格的浪漫选项
