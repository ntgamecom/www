## 技能系统概述 / Skill System Overview

**中文：** 赛博朋克2077的技能系统基于五大属性（肉体、反应、技术能力、智力、镇定），每个属性下有2-3条专长分支，共计约200个专长。2.0版本完全重做了专长系统，移除了等级专长点限制，改为每升一级获得专长点，同时专长不再需要前置专长（改为属性等级解锁）。这让Build自由度大幅提升。

**English:** Cyberpunk 2077's skill system is built on 5 attributes (Body, Reflexes, Technical Ability, Intelligence, Cool), each with 2-3 perk branches totaling ~200 perks. Patch 2.0 completely revamped the system: perks now unlock based on attribute level rather than perk prerequisites, and each level grants one perk point. This dramatically increases build freedom.

| 属性 / Attribute | 上限 / Cap | 分支 / Branches | 专长数 / Perks | 核心作用 / Core Role |
|---|---|---|---|---|
| 肉体 / Body | 20 | 2 | 40+ | 近战、生命、耐力、碾压 / Melee, HP, Stamina |
| 反应 / Reflexes | 20 | 2 | 40+ | 射击、暴击、闪避、时停 / Guns, Crit, Dodge, Sandy |
| 技术能力 / Tech | 20 | 2 | 40+ | 护甲、制作、技术武器 / Armor, Crafting, Tech Weapons |
| 智力 / Intelligence | 20 | 3 | 60+ | 黑客、快速破解、智能武器 / Hacking, Quickhacks, Smart Weapons |
| 镇定 / Cool | 20 | 2 | 40+ | 潜行、冷血、暴击伤害 / Stealth, Cold Blood, Crit Dmg |

## 肉体专长树 / Body Perk Tree

**中文：** 肉体属性决定你的身体素质、近战能力和生命值。适合喜欢正面硬刚的玩家。两条分支分别是"角斗士"（近战武器）和"莽夫"（徒手/霰弹枪）。

**English:** Body governs physical prowess, melee capability, and HP. Two branches: Gladiator (melee weapons) and Bruiser (brawling/shotguns).

## 专长组合推荐 / Perk Combos

**中文：** 正确的专长组合可以让战斗力倍增。以下是一些经过验证的高效组合。

**English:** The right perk combinations can double your combat effectiveness. Here are proven efficient combos.

**黑客连环套 / Netrunner Loop**:
- 过度clock (Int 9) + 快速传播 (Int 14) + 遗忘 (Int 20) + 传染 (Int 11)
- 机制：用传染起手扩散DoT，过度clock补RAM，遗忘让击杀不触发冷却，实现无限循环 / Start with Contagion, Overclock for RAM, Forget-Me-Not resets cooldown on kill

**时停爆发套 / Sandy Burst Combo**:
- 百战不殆 (Ref 7) + 暴风骤雨 (Ref 11) + 行云流水 (Ref 15) + 终极舞者 (Ref 20)
- 机制：时停中刀剑连击必暴击，击杀后攻速叠满，闪避不消耗耐力 / Sandy + blade combo always crits, kill grants attack speed, free dodges

**坦克生存套 / Tank Survival Combo**:
- 疯狂物理学 (Tech 9) + 金刚不坏 (Body 15) + 横冲直撞 (Body 20) + 铁甲 (Tech 20)
- 机制：护甲+15%，全减伤10%，残血无敌2秒，受伤叠护甲 / Armor +15%, 10% all reduction, 2s invuln at low HP, armor on hit

## 专长点分配策略 / Perk Point Strategy

**中文：** 50级共可获得49个专长点（除去初始1级）。以下各流派的专长点分配比例。

**English:** 49 perk points total at level 50 (excluding level 1). Perk point ratios for each build.

| 流派 / Build | 肉体 / Body | 反应 / Reflex | 技术 / Tech | 智力 / Int | 镇定 / Cool |
|---|---|---|---|---|---|
| 时停武士刀 / Sandy Katana | 4 | 20 | 15 | 3 | 18 |
| 纯粹黑客 / Pure Netrunner | 3 | 4 | 18 | 20 | 15 |
| 技术散弹 / Tech Shotgun | 18 | 8 | 20 | 3 | 11 |
| 潜行狙击 / Stealth Sniper | 3 | 15 | 18 | 3 | 20 |
| 狂暴肉搏 / Berserker Brawler | 20 | 4 | 18 | 3 | 15 |

**专长点获取 / Perk Point Sources**:
- 升级获得49点 (1-50级) / Leveling: 49 points (level 1-50)
- 完成任务获得额外专长点 / Quest rewards for bonus perk points
- DLC往日之影增加额外等级和专长点 / Phantom Liberty adds extra levels and perk points

## 技能重置 / Skill Reset

**中文：** 2.0版本中，专长点可以随时免费重置，但属性点不可重置。这意味着你可以随意尝试不同的专长组合，但需要谨慎分配属性点。规划好前期的属性加点方向至关重要。

**English:** In patch 2.0, perk points can be reset for free at any time, but attribute points cannot be reset. This means you can freely experiment with different perk builds but must carefully plan attribute allocation. Planning your attribute direction early is critical.
