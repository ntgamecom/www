## 马洛斯 / Royce (漩涡帮头目)

**位置 / Location**: 全食品工厂 / All Foods plant
**等级建议 / Level**: 8-12

**打法 / Strategy**:
1. **最简单方法**：在对话中给漩涡帮赊账的付费方式选用"军用科技芯片"（需肉体或智力选项），可直接杀死马洛斯 / Easiest: during conversation, use the Militech chip (requires Body/Int check) to instantly kill him
2. **正面战斗**：马洛斯使用重型机枪，找掩体拉远距离射击 / Uses heavy MG, use cover at range
3. **弱点击破**：暴露的通风管线可以引爆造成重伤 / Exposed vent pipes can be shot to explode
4. **黑客打法**：用"义体故障"大幅降低他的战斗效率 / Cyberware Malfunction cripples him

**掉落 / Loot**: 混沌（Chaos）— 不朽动能手枪，弹道弹射+追踪 / Legendary power pistol with ricochet tracking

## 卡车杀手 / Nash (恶土战斗)

**位置 / Location**: 恶土货车停放区 / Badlands truck depot
**等级建议 / Level**: 12-16

**打法 / Strategy**:
1. **对话选项**：可以劝说他离开（需镇定 8+）/ Can be persuaded to leave (Cool 8+)
2. **战斗**：他有远程狙击 + 近战霰弹枪 / Long-range sniper + close-range shotgun
3. **推荐方式**：用黑客快速破解"短路"起手 / Start with Short Circuit quickhack
4. **走位要点**：保持中距离，他的狙击需要蓄力 / Maintain mid-range, his sniper needs charge time

**掉落 / Loot**: 不朽车辆 / Iconic vehicle

---

## 桥本翔太 / Oda (荒坂忍者)

**位置 / Location**: 云顶酒店, 威斯布鲁克 / Clouds, Westbrook
**等级建议 / Level**: 20-25

**重要提示 / Important**: 这是游戏中**最难而且最容易翻车的Boss战之一** / One of the hardest and most punishing boss fights in the game. 建议打之前存档 / Save before starting.

**Boss特性 / Boss Traits**:
- 极其灵活，频繁瞬移 / Highly mobile, frequent teleportation
- 隐身状态下免疫伤害 / Invisible and immune to damage
- 近战连击伤害极高 / Melee combo deals massive damage
- 远程投掷苦无 / Throws kunai at range

**打法 / Strategy**:

| 阶段 / Phase | 行为 / Behavior | 应对 / Counter |
|---|---|---|
| 第一阶段 (HP 100%-50%) | 主动近战连击 + 瞬移绕后 / Aggressive melee + teleport flank | 拉开距离射/骇入，不要近战 / Keep distance, shoot/hack, don't melee |
| 第二阶段 (HP 50%-0%) | 隐身次数增加 + 召唤无人机 / More invisibility + drone support | 用扫描（TAB）追踪位置，无人机优先打掉 / Use scanning to track, kill drones first |

**核心战术 / Core Tactics**:
1. **开时停**：时停流可以在他瞬移前打出大量伤害 / Sandy users: burst before his teleport
2. **黑客削血**："短路"和"义体故障"非常好用 / Short Circuit + Cyberware Malfunction are very effective
3. **场地利用**：中间的大柱子可以卡位阻挡苦无投掷 / Center pillar blocks kunai throws
4. **不要贪刀**：他的反击非常快 / His counterattack is very fast, don't overcommit

## 马奇科 / Sasquatch (动物帮老大)

**位置 / Location**: 动物帮健身房 / Animals gym
**等级建议 / Level**: 25-30

**Boss特性 / Boss Traits**:
- 使用大锤，攻击范围大、伤害高 / Uses a sledgehammer, wide AoE, high damage
- 召唤动物帮小弟 / Summons Animals minions
- 部分攻击可破坏场景 / Some attacks destroy cover

**打法 / Strategy**:
1. **速杀**：冲过去近战，她有硬直时间 / Rush her, she has recovery frames after swings
2. **风筝**：在场地里绕圈打游击 / Kite around the arena
3. **连线引爆**：她身后有连接线缆，射击可引爆造成大硬直 / Shoot the cables behind her for a massive stun
4. **黑客打法**："赛博精神病"可以让她的小弟去攻击她 / Cyberpsychosis makes her minions attack her

## 普拉西德 / Placide (巫毒帮黑客)

**位置 / Location**: 巫毒帮基地 / Voodoo Boys base, Pacifica
**等级建议 / Level**: 28-32

**特殊机制 / Special Mechanic**: 普拉西德会使用黑客技能对抗你 / Placide uses quickhacks against you

**打法 / Strategy**:
1. **先清小弟**：房间里的巫毒帮黑客优先击杀 / Kill the Voodoo Boys netrunners first
2. **反黑客**：你的智力越高、抗性越高，他的黑客效果越弱 / Higher Int = less effect from his hacks
3. **近身压制**：普拉西德近战较弱，拉近距离即可 / Placide is weak at close range, push in
4. **躲避自爆无人机**：他释放的自爆无人机可射击打掉 / Shoot down his detonator drones

**重要选择**：你可以选择杀了普拉西德或者放过他（影响小部分后续剧情）/ Can kill or spare him (minor story impact)

## 亚当·重锤 / Adam Smasher

**位置 / Location**: 荒坂大厦 / Arasaka Tower
**等级建议 / Level**: 40-50

**这是最终Boss / Final Boss**。重锤是夜之城最恐怖的杀戮机器，拥有极高的生命值、装甲和全屏攻击能力 / Smasher is Night City's most feared killing machine with massive HP, armor, and AoE attacks.

**Boss阶段 / Boss Phases**:

| 阶段 / Phase | HP范围 / Range | 行为 / Behavior |
|---|---|---|
| 一阶段 | 100%-70% | 重机枪压制 + 火箭弹 / Heavy MG suppression + rockets |
| 二阶段 | 70%-30% | 开启斯安威斯坦 + 近战冲刺 / Activates Sandevistan + charge melee |
| 三阶段 | 30%-0% | 狂轰滥炸模式，全屏AoE / All-out assault, full screen AoE |

**核心战术 / Core Tactics**:

**黑客流打法 / Netrunner Strategy**:
- "短路"对重锤极其有效（造成最大生命值百分比伤害）/ Short Circuit deals % max HP damage
- "义体故障"可以关闭他的斯安威斯坦和火箭弹 / Cyberware Malfunction disables his Sandy and rockets
- "系统崩溃"对重锤无效（仅限杂兵）/ System Collapse doesn't work on him

**近战打法 / Melee Strategy**:
- 开时停绕后攻击，不要正面硬刚 / Sandevistan + flank, don't trade face-to-face
- 重锤的近战攻击前摇较长，可以闪避 / His melee has long wind-up, easy to dodge
- 打几下就拉开，小心他的 AoE 踩地 / Hit-and-run, watch for his stomp AoE

**通用技巧 / General Tips**:
1. **跑动是生命**：永远不要停在原地超过2秒 / Never stop moving for more than 2 seconds
2. **破坏柱子**：战场里的柱子可以打碎，但也会减少你的掩体 / Arena pillars are destructible
3. **携带回复道具**：至少带 10 个以上治疗物品 / Bring 10+ healing items
4. **配件"荷尔蒙阻断器"**：减少Boss对你的减速效果 / Biomonitor cyberware reduces his slow effects

## DLC Boss: 奇美拉 / Chimera

**位置 / Location**: 狗镇地下研究设施 / Dogtown underground research facility
**等级建议 / Level**: 35-45

**特殊机制 / Mechanic**: 这是一种四足机甲，拥有多个武器系统 / A four-legged mech with multiple weapon systems

**弱点 / Weakness**:
- **核心暴露**：在特定攻击后核心会暴露 / Core exposes after certain attacks
- **腿部关节**：破坏腿部降低机动性 / Leg joints: destroy to reduce mobility
- **电磁伤害**：EMP 和电击伤害翻倍 / EMP/Electrical damage x2

**打法 / Strategy**:
1. 优先破坏腿部，限制其走位 / Prioritize legs to limit movement
2. 等它发射导弹后核心过热暴露时全力输出 / After missile barrage, core overheats — full DPS
3. 使用电击类武器和快速破解（电磁脉冲）/ Use EMP weapons and quickhacks
4. 场景中的能源罐可以射击引发爆炸 / Shoot energy canisters in the arena for explosions
