# 自定义与收集指南 / Customization and Collectibles

铁拳8提供了丰富的自定义系统，让你可以改变角色的外观、武器特效、战斗风格，甚至个人资料卡片。本章节将全面介绍自定义系统的各个方面，帮助你打造独一无二的角色形象。

Tekken 8 offers an extensive customization system that lets you change your character's appearance, weapon effects, battle style, and even personal profile card. This chapter covers every aspect of the customization system, helping you create a unique character image.

## 自定义系统概述 / Customization Overview

铁拳8的自定义系统分为以下几个主要类别：

Tekken 8's customization system is divided into the following main categories:

| 类别 / Category | 描述 / Description | 解锁方式 / Unlock Method |
|---|---|---|
| 服装 / Outfits | 替换角色全身服装 / Full outfit replacements | 游戏内购买 / In-game purchase |
| 配饰 / Accessories | 头饰、眼镜、面罩等 / Hats, glasses, masks | 商店/Fight Pass / Shop or Fight Pass |
| 武器 / Weapons | 替换武器外观 / Weapon skin replacements | 商店/Fight Pass / Shop or Fight Pass |
| 特效 / Effects | 命中特效、KO特效等 / Hit effects, KO effects | Fight Pass / 特殊活动 |
| 玩家卡片 / Player Card | 名片背景、称号、边框 / Card background, title, border | 游戏内收集 / In-game collection |
| 语音 / Voice | 特殊语音替换 / Special voice replacements | 商店/Fight Pass / Shop or Fight Pass |

## 服装自定义 / Outfit Customization

### 基础服装 / Base Outfits

每个角色从游戏开始时拥有两套基础服装。第一套是角色的默认外观，第二套需要在角色章节或街机模式中解锁。此外，通过游戏内货币（铁拳金币）可以在Tekken Shop中购买额外的服装。

Every character starts with two base outfits. The first is the character's default appearance, while the second must be unlocked through character episodes or arcade mode. Additionally, extra costumes can be purchased in the Tekken Shop using the in-game currency (Tekken Coins).

### 服装部件 / Costume Pieces

服装自定义允许你替换以下部件：

Outfit customization allows you to replace the following components:

- 上衣 / Top（衬衫、夹克、背心 / Shirt, jacket, vest）
- 下装 / Bottom（裤子、短裤 / Pants, shorts）
- 鞋子 / Shoes（靴子、运动鞋 / Boots, sneakers）
- 手套 / Gloves（半指、全指 / Half-finger, full-finger）
- 配件 / Attachments（披风、腰带、肩甲 / Cape, belt, shoulder armor）

### 稀有服装 / Rare Costumes

部分服装只能通过特定方式获得：

Some costumes can only be obtained through specific methods:

**Fight Pass专属**：每个赛季的Fight Pass都包含一套独占服装，只有购买付费通行证才能获得。

**活动限定**：在特殊活动期间（如铁拳系列周年庆、节日活动等）登录游戏可以获得限定时装。

**段位奖励**：达到特定段位可以获得专属段位服装奖励。例如，达到龙黄（Ruler）段位可以获得金色主题服装。

**预购特典**：预购铁拳8的玩家获得了首发特典服装。

**Fight Pass Exclusive**: Each season's Fight Pass includes exclusive outfits only available through the paid pass.

**Event Limited**: Login during special events (such as Tekken series anniversary, holiday events, etc.) to receive limited-time costumes.

**Rank Rewards**: Reaching specific ranks unlocks exclusive rank costumes. For example, reaching Ruler rank rewards a gold-themed costume.

**Pre-order Bonus**: Players who pre-ordered Tekken 8 received launch bonus costumes.

## 配饰系统 / Accessory System

### 头部配饰 / Head Accessories

头部配饰种类繁多，包括：

Head accessories include a wide variety of options:

- 帽子 / Hats（棒球帽、礼帽、安全帽 / Baseball cap, top hat, hard hat）
- 眼镜 / Glasses（墨镜、护目镜、单眼镜 / Sunglasses, goggles, monocle）
- 面具 / Masks（面罩、骷髅面具、恶魔面具 / Face mask, skull mask, demon mask）
- 头饰 / Headpieces（头带、皇冠、角 / Headband, crown, horns）

### 面部和身体配饰 / Face and Body Accessories

- 耳环 / Earrings
- 项圈 / Chokers
- 纹身贴纸 / Tattoo stickers
- 翅膀 / Wings
- 光环 / Halos
- 背饰 / Back accessories（背包、旗帜、武器架 / Backpack, flag, weapon rack）

### 配饰颜色自定义 / Accessory Color Customization

大多数配饰支持颜色自定义。你可以通过调色板选择配饰的主色、副色和装饰色。某些稀有配饰还拥有特殊的金属质感或发光效果。

Most accessories support color customization. You can use the color palette to select primary, secondary, and accent colors for accessories. Some rare accessories also feature special metallic textures or glow effects.

## Tekken Shop / 铁拳商店

Tekken Shop是游戏内的主要购买渠道，使用铁拳金币（Tekken Coins）进行交易。

The Tekken Shop is the main in-game purchasing channel, using Tekken Coins for transactions.

### 货币系统 / Currency System

| 货币类型 / Currency Type | 获取方式 / Acquisition | 用途 / Usage |
|---|---|---|
| 铁拳金币 / Tekken Coins | 游戏内赚取（对战、任务）/ Earned in-game (matches, quests) | 购买大部分自定义道具 / Most customization items |
| 战斗点数 / Fight Points | 参与在线对战 / Playing online matches | 解锁特殊称号和边框 / Special titles and frames |
| 通行证点数 / Pass Points | 完成Fight Pass任务 / Completing Fight Pass quests | 解锁Fight Pass奖励 / Fight Pass rewards |

### 商店商品类型 / Shop Item Types

Tekken Shop每日刷新商品，每周有新的轮换商品。商品类型包括：

The Tekken Shop refreshes daily with new rotating items weekly. Item types include:

- 服装套装 / Outfit Sets（全角色通用或角色限定 / Universal or character-specific）
- 配饰包 / Accessory Packs（包含5到10个配饰 / 5 to 10 accessories per pack）
- 武器皮肤 / Weapon Skins（单角色武器外观 / Single character weapon skins）
- 特效包 / Effect Packs（命中特效、KO特效、入场特效 / Hit, KO, and entrance effects）
- 语音包 / Voice Packs（特殊语音替换 / Special voice replacements）
- 玩家卡片素材 / Player Card Materials（背景、边框、图标 / Backgrounds, borders, icons）

## Fight Pass / 战斗通行证

Fight Pass是铁拳8的赛季制奖励系统。每个赛季持续约2到3个月。

The Fight Pass is Tekken 8's seasonal reward system. Each season lasts approximately 2 to 3 months.

### 免费路线 / Free Track

免费路线包含以下奖励：

The free track includes the following rewards:

- 铁拳金币（少量）/ Tekken Coins (small amounts)
- 普通配饰 / Common accessories
- 玩家卡片背景 / Player card backgrounds
- 称号 / Titles
- 基本特效 / Basic effects

### 付费路线 / Premium Track

购买付费通行证（约$9.99 / 等价地区定价）可以获得：

Purchasing the premium pass (approximately $9.99) grants access to:

- 赛季专属服装 / Season-exclusive outfits
- 稀有武器皮肤 / Rare weapon skins
- 传说级配饰 / Legendary accessories
- 特殊KO特效 / Special KO effects
- 大量铁拳金币 / Large amounts of Tekken Coins
- 限时称号和边框 / Limited-time titles and borders

### 通行证挑战 / Pass Challenges

Fight Pass通过每日和每周挑战获得经验值。主要挑战类型包括：

The Fight Pass gains experience through daily and weekly challenges. Main challenge types include:

**每日挑战 / Daily Challenges**（每天刷新 / Refresh daily）：
- 进行3场对战 / Play 3 matches
- 使用特定角色获胜1次 / Win 1 match with a specific character
- 发动3次Rage Art / Activate 3 Rage Arts
- 完成1次Heat Burst / Perform 1 Heat Burst

**每周挑战 / Weekly Challenges**（每周刷新 / Refresh weekly）：
- 进行15场排位赛 / Play 15 ranked matches
- 使用特定角色完成10场对战 / Play 10 matches with a specific character
- 在练习模式中练习30分钟 / Practice for 30 minutes in practice mode
- 完成全部每日挑战 / Complete all daily challenges

**赛季挑战 / Season Challenges**（赛季内有效 / Valid throughout the season）：
- 达到特定段位 / Reach a specific rank
- 收集一定数量的自定义道具 / Collect a certain number of customization items
- 完成街机模式 / Complete arcade mode

## 收集要素 / Collectible Elements

### 角色特定收集 / Character-Specific Collectibles

每个角色都有专属的收集元素，包括：

Each character has专属collectible elements, including:

**角色配色 / Color Variations**：每个角色拥有约10种预设配色方案，通过街机模式和角色章节解锁。

**特殊道具 / Special Items**：通过完成角色的特定挑战解锁专属配饰。

**角色剧情道具 / Story Items**：完成角色的章节剧情可以获得该角色的剧情相关服装或配饰。

**Color Variations**: Each character has approximately 10 preset color schemes, unlocked through arcade mode and character episodes.

**Special Items**: Unlock exclusive accessories by completing character-specific challenges.

**Story Items**: Complete character episodes to obtain story-related costumes or accessories.

### 通用收集 / Universal Collectibles

**标题集 / Title Collection**：超过200个可解锁称号，涵盖各种类型和主题。

**配音集 / Voice Collection**：特殊入场语音、KO语音和挑衅语音。

**特效集 / Effect Collection**：包括Rage Art特效、Heat Smash特效、KO特效和入场特效。

**Title Collection**: Over 200 unlockable titles covering various types and themes.

**Voice Collection**: Special entrance voices, KO voices, and taunt voices.

**Effect Collection**: Including Rage Art effects, Heat Smash effects, KO effects, and entrance effects.

### 收集效率建议 / Collection Efficiency Tips

**优先Fight Pass**：Fight Pass提供的道具数量和种类最多，性价比最高。

**关注每日商店**：每日刷新中可能出现稀有道具，铁拳金币足够时直接购买。

**完成每周挑战**：每周挑战提供大量通行证点数，加速Fight Pass升级。

**利用角色章节**：通关角色章节是解锁角色特定道具的最稳定方式。

**参与限时活动**：限时活动的道具通常不会在常规商店中再次上架。

**Prioritize Fight Pass**: The Fight Pass offers the most道具数量和种类with the best value.

**Check daily shop**: Rare items may appear in daily refreshes. Purchase directly if you have enough Tekken Coins.

**Complete weekly challenges**: Weekly challenges provide大量pass points, accelerating Fight Pass progression.

**Utilize character episodes**: Completing character episodes is the most reliable way to unlock character-specific items.

**Participate in limited events**: Items from limited events usually do not return to the regular shop.
