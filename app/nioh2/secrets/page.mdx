## 奈落狱 / The Underworld

奈落狱是仁王2的终局迷宫——108层挑战 / The Underworld is Nioh 2's endgame dungeon — 108 floors of challenge.

| 层数 / Floors | 内容 / Content |
|---|---|
| **1-5** | 基础——单个Boss / Basic — single boss |
| **6-10** | 双Boss组合 / Dual boss combos |
| **11-20** | 精英敌人关 / Elite enemy gauntlets |
| **21-50** | 混合关卡 —— 精英 + Boss / Mixed — elites + bosses |
| **51-80** | 高强度挑战 / High intensity |
| **81-108** | 终极挑战——最强Boss组合 / Ultimate challenge — strongest boss combos |

## 深层 / The Depths

通过奈落狱108层后解锁深层——共30层，是真正的终局试炼 / Unlocked after floor 108 of Underworld — 30 floors of true endgame trial.

| 方面 / Aspect | 说明 / Description |
|---|---|
| **层数 / Floors** | 1-30——每层比上一层的敌人更强 / 1-30 — each floor harder than the last |
| **第10层 / Floor 10** | 只出现神珍+品质装备 / Divine+ quality gear only |
| **第20层 / Floor 20** | 所有敌人都是Boss级别 / All enemies are boss-level |
| **第30层 / Floor 30** | 最终挑战——最强Boss连战 / Ultimate challenge — strongest boss rush |
| **通关奖励 / Clear Reward** | 特殊称号 + 最高品质装备 / Special title + max quality gear |
