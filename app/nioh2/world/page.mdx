# 世界探索 / World Exploration

仁王2的关卡是任务制的——从大地图选择关卡进入。每关有木灵、温泉、隐藏道具、置物袋和大量可探索区域。本章提供关键关卡攻略和全探索技巧 / Nioh 2 uses a mission-based structure — select from the world map. Each mission has Kodama, hot springs, hidden items, kodama bowls, and extensive exploration areas. This chapter covers key mission walkthroughs and exploration mastery.

## 关卡详解 / Mission Walkthroughs

### 第一章: 暗影之都 / Chapter 1: Dark Realm

全游戏第一关——同时包含最基础的教程和最经典的新人杀手马头鬼。**推荐探索路线**: 初始神社→左路过桥→击败第一个精英敌人(夜叉兵)→拿到"守护灵召唤符"→右路进入村庄→第二个神社→马头鬼Boss房 / The game's first mission — tutorial zone + newbie-killer Mezuki. **Recommended path**: first shrine → cross left bridge → kill first elite (yokai soldier) → get Guardian Spirit Talisman → right path to village → second shrine → Mezuki boss room.

**隐藏收集**: 开场后不要直走——回头跳下断裂的木桥下方，有一个木灵藏在洞穴里。第二个神社前右侧瀑布后面有温泉 / **Hidden collectibles**: After starting, don't go straight — jump down below the broken bridge for a Kodama in a cave. Hot spring behind the waterfall before second shrine.

### 第二章: 妖怪之海 / Chapter 2: Sea of Yokai

以水底探索为主题——氧气条是此关的核心管理机制。**攻略要点**: 下水前先清掉岸上的弓箭手。水底有3个洞穴入口——最左侧通往隐藏木灵，中间的通往主线进度，右侧的通往温泉。水底的宝箱怪(模仿木箱)特别多——打箱子前先用武器试探 / Underwater-themed — oxygen management is key. **Tips**: Clear archers on shore before diving. 3 underwater cave entrances — left = hidden Kodama, center = main path, right = hot spring. Many mimic chests underwater — poke chests before opening.

**Boss夜叉**: 弱火——推荐带"火符"或"封气符"。她的投技(抓取)伤害极高——看到红光就后闪 / **Boss Yasha**: Weak to fire — bring Fire Talismans or elemental shots. Her grab deals massive damage — dodge back on red aura.

### 第三章: 白刃之塔 / Chapter 3: The Blade's Tower

3层结构的高塔关卡——每一层都有多个分支路线。**探索建议**: 从左侧楼梯上到二层→二层右侧阳台有木灵→三层前先清掉天台上的弓箭手→Boss战在塔顶。塔内有大量陷进(落石、地刺)——注意地板上的警示标记 / 3-floor tower — each floor has branches. **Tips**: Left stairs to 2F → 2F right balcony has Kodama → clear roof archers before 3F → boss at top. Many traps (falling rocks, floor spikes) — watch for warning markers on floor.

**Boss镰鼬**: 速度型Boss——最大输出窗口是它冲刺后的硬直。推荐闪避流，不要尝试格挡它的连段 / **Boss Kamaitachi**: Speed-type boss — biggest window is post-charge stagger. Dodge build recommended, don't try to block its combos.

## 探索技巧 / Exploration Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **神社利用 / Use Shrines** | 神社 = 篝火——存档、升级、奉献装备、购买道具。每关必先找到第一个神社再深入探索 / Shrine = bonfire — save, level, offer gear, buy items. Always find first shrine before deep exploration |
| **木灵探测器 / Kodama Detector** | 装备上"感应木灵"词缀的物品(或守护灵"白野槌")——地图上可看到木灵位置 / Equip "Sense Kodama" affix (or "Isanagami" spirit) — see Kodama on map |
| **隐藏墙 / Hidden Walls** | 部分墙壁可以攻击或互动打开——留意墙上有裂缝或颜色不同的区域 / Some walls breakable/interactive — look for cracks or color differences |
| **可破坏地形 / Destructible Terrain** | 木栅栏、木箱、部分墙壁可用攻击打破——很多藏着道具 / Wooden fences, crates, walls break with attacks — many hide items |
| **远程引怪 / Ranged Pull** | 用远程武器或阴阳术引单个敌人——不要被群殴 / Use ranged weapons or Onmyo to pull single enemies individually |
| **神社解BUFF / Shrine Cleanse** | 如果中了"盲目"或"麻痹"——回神社参拜即可消除 / If blinded or paralyzed — pray at shrine to cleanse |
| **置物袋 / Kodama Bowl** | 每关有一个置物袋——可以存放物品给联机队友。单机模式=多一个获得装备的途径 / One per mission — leave items for co-op partners. Solo = extra gear source |
| **捷径优先 / Shortcuts First** | 每关都有可打开的捷径(门、梯子、桥)——优先解锁这些捷径再打Boss，减少跑尸距离 / Each mission has unlockable shortcuts — prioritize them before boss to reduce corpse run distance |

## 木灵全收集 / All Kodama Locations

每个主线关卡有 3-9 只木灵。提交木灵给神社的大木灵获得永久加成——全收集后药草掉落率、仙药上限、精华获得量全部最大化 / Each main mission has 3-9 Kodama. Submit them at the Great Kodama shrine for permanent bonuses — full collection maximizes herb drops, elixir capacity, and Amrita gain.

| 关卡 / Mission | 木灵数 / Kodama | 关键位置 / Key Location |
|:----:|:----:|------|
| 暗影之都 / Dark Realm | 3 | 一个在Boss房前右转水道 / One in water channel right before boss |
| 妖怪之海 / Sea of Yokai | 5 | 两个在水下洞穴——注意氧气 / Two in underwater caves — watch oxygen |
| 白刃之塔 / The Blade's Tower | 7 | 三个在隐藏房间——打破木墙 / Three in hidden rooms — break wooden walls |
| 鬼火之村 / Phantom Village | 5 | 一个在屋顶跳跃路径末端 / One at end of rooftop jump path |
| 双龙之城 / Twin Dragons' Keep | 9 | 全游戏最多——仔细探索每个角落 / Most in game — search every corner |

## 暗之世界系统 / Dark Realm System

仁王2的特色机制——部分区域和Boss战中会进入"暗之世界"(黑雾笼罩)。在暗之世界中，妖怪精力恢复加快、新生常世频率提高、敌人攻击力上升。**应对**: 击杀暗之世界的红色敌人(精华精华)可以驱散暗之世界。推荐携带"结界符"(自动消除常世)和"护身符"(提高容错率) / A signature Nioh 2 mechanic — some areas and boss phases enter the Dark Realm (black fog). In Dark Realm, yokai recover Ki faster, Yokai Realms spawn more frequently, and enemy attack power increases. **Counter**: Kill red-glowing enemies (Amrita fiends) to dispel. Bring Barrier Talisman (auto-cleanse Yokai Realms) and Protection Talisman (damage buffer).
