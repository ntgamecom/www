# 世界探索 / World Exploration

仁王2的关卡是任务制的——从大地图选择关卡进入。每关有木灵、温泉、隐藏道具。本章提供关键关卡攻略 / Nioh 2 uses a mission-based structure — select from the world map. Each mission has Kodama, hot springs, and hidden items. This chapter covers key mission walkthroughs.

## 木灵全收集 / All Kodama Locations

每个主线关卡有 3-9 只木灵。提交木灵给神社的大木灵获得永久加成 / Each main mission has 3-9 Kodama. Submit them at the Great Kodama shrine for permanent bonuses.

| 关卡 / Mission | 木灵数 / Kodama | 关键位置 / Key Location |
|:----:|:----:|------|
| 暗影之都 / Dark Realm | 3 | 一个在Boss房前右转水道 / One in water channel right before boss |
| 妖怪之海 / Sea of Yokai | 5 | 两个在水下洞穴——注意氧气 / Two in underwater caves — watch oxygen |
| 白刃之塔 / The Blade's Tower | 7 | 三个在隐藏房间——打破木墙 / Three in hidden rooms — break wooden walls |
| 鬼火之村 / Phantom Village | 5 | 一个在屋顶跳跃路径末端 / One at end of rooftop jump path |
| 双龙之城 / Twin Dragons' Keep | 9 | 全游戏最多——仔细探索每个角落 / Most in game — search every corner |

## 温泉位置 / Hot Springs

每个主线关卡有一个温泉——第一次泡永久增加生命恢复量 / Each main mission has one hot spring — first visit permanently boosts HP recovery.

| 关卡 / Mission | 位置 / Location |
|------|------|
| 暗影之都 / Dark Realm | 第二个神社前的右侧岔路 / Right fork before second shrine |
| 妖怪之海 / Sea of Yokai | 下水后左侧洞穴——隐藏在水草后 / Left cave after diving — hidden behind seaweed |
| 白刃之塔 / The Blade's Tower | 三层阳台——从楼梯跳下 / Third floor balcony — jump from stairs |
| 鬼火之村 / Phantom Village | 村后瀑布后面 / Behind waterfall at village rear |

## 探索技巧 / Exploration Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **神社利用 / Use Shrines** | 神社 = 篝火——存档、升级、奉献装备、购买道具。每关必先找到第一个神社 / Shrine = bonfire — save, level up, offer gear, buy items. Find first shrine in every mission |
| **木灵探测器 / Kodama Detector** | 装备上"感应木灵"词缀的物品（或守护灵"白野槌"）——地图上可看到木灵位置 / Equip "Sense Kodama" affix (or use "Isanagami" Guardian Spirit) — see Kodama on map |
| **隐藏墙 / Hidden Walls** | 部分墙壁可以攻击或互动打开——留意墙上有裂缝或颜色不同的 / Some walls can be broken/interacted — look for cracked or differently colored surfaces |
| **可破坏地形 / Destructible Terrain** | 木栅栏、木箱、部分墙壁可用攻击打破——很多藏着道具 / Wooden fences, crates, some walls break with attacks — many hide items |
| **远程引怪 / Ranged Aggro** | 用远程武器或阴阳术引单个敌人——不要被群殴。弓箭/手里剑是清场利器 / Use ranged weapons or Onmyo to pull single enemies. Bows/shurikens are great for pulling |
| **神社解BUFF / Debuff Removal** | 如果中了"盲目"或"麻痹"——回神社参拜即可消除 / If blinded or paralyzed — pray at shrine to cleanse |
