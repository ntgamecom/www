# 仁王2攻略 / Nioh 2 Guide

You play as a half-Yokai protagonist, exploring, fighting, and looting through a demon-infested Warring States period — featuring the deepest combat system and richest build variety in the series.

## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **妖怪化 / Yokai Shift** | 积攒妖怪条后按R2+○变身——大幅提升攻防，可使用妖怪技（不消耗妖力）。变身后有独立血量条 / Fill Yokai gauge → R2+○ to transform — massive attack/defense boost, use Yokai abilities for free. Has separate HP bar |
| **Burst Counter / 猛反** | 对应红光攻击的反制技——用R2+△释放。有三种类型：猛(Brute)硬扛、迅(Feral)闪避反击、幻(Phantom)格挡反击 / Counter against red-glow attacks via R2+△. Three types: Brute (tank), Feral (dodge counter), Phantom (parry counter) |
| **魂核 / Soul Cores** | 击杀妖怪掉落——可装备最多2个。每个提供妖怪技（主动技能）+ 属性加成 + 守护灵词缀加成 / Dropped by Yokai — equip up to 2. Each grants a Yokai Ability (active) + stats + Guardian Spirit synergy |
| **守护灵 / Guardian Spirits** | 类似卧龙的神兽——提供被动加成和妖怪召唤。可装备主/副两个，战斗中切换 / Like Wo Long's Divine Beasts — passives + Yokai summon. Equip primary/secondary, swap mid-combat |
| **残心 / Ki Pulse** | 攻击后按R1原地回气——消除常世（妖怪留下的污秽区域）。完美残心恢复更多精力 + 触发回气效果 / After attacking, press R1 to recover ki — cleanses Yokai Realm pools. Perfect pulse = more ki + buff triggers |
| **三重攻击架势 / Three Stances** | 上段(高伤害/大硬直)、中段(平衡)、下段(快速/低耗精)——每种架势有独立的武技树 / High (damage/heavy), Mid (balanced), Low (fast/low ki) — each stance has its own skill tree |
| **常世 / Yokai Realm** | 妖怪攻击留下紫色区域——站在上面精力和生命恢复降低。用残心消除或翻滚躲开 / Purple pools left by Yokai attacks — reduces ki/HP recovery. Purge with Ki Pulse or dodge out |
| **难度系统 / NG+ Cycles** | 通关后解锁"强者之道"——敌人强化、增加新词缀、新装备等级上限。共5个周目难度 / NG+ unlocks "Way of the Strong" — harder enemies, new affixes, higher gear level. 5 NG+ cycles total |

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **选择你的Burst Counter类型 / Pick Your Burst Counter** | 猛(Brute)最简单——看到红光直接R2+△打出去就行。迅(Feral)需要闪避时机。幻(Phantom)类似格挡。新手推荐**猛** / Brute is easiest — just R2+△ on red glow. Feral needs timing. Phantom is parry-based. **Brute recommended** for beginners |
| **2** | **学会残心 / Master Ki Pulse** | 残心是仁王2最重要的操作——每次攻击后按R1。不残心的话你一管精力打不了多久就会力竭 / Ki Pulse is THE most important skill — press R1 after every attack chain. Without it, you'll be out of ki in seconds |
| **3** | **武技优先学习 / Prioritize Active Skills** | 不要只平A——每个武器都有丰富武技。优先学：踢击（削精）、位移技（追击）、高伤害技 / Don't just R1 spam — every weapon has skills. Priority: kicks (ki dmg), gap closers, high-damage skills |
| **4** | **魂核是核心资源 / Soul Cores Are Key** | 找到喜欢的守护灵→搭配对应魂核→凑齐套装效果。魂核的妖怪技不消耗精力——是持续输出的关键 / Find a Guardian Spirit you like → match Soul Cores → complete set bonuses. Yokai Abilities cost zero ki — key for sustained DPS |
| **5** | **不要忽略装备等级 / Don't Ignore Gear Level** | 仁王2是刷装备游戏——定期更换高等级装备比凑套装更重要。NG+后才需要认真配装 / Nioh 2 is a loot game — regularly upgrade to higher level gear. Don't optimize sets until NG+ |
| **6** | **木灵全收集 / Collect All Kodama** | 每个关卡的木灵(绿色小精怪)提供永久加成——增加药草掉落、提高回复量等。地毯式搜索 / Kodama in each mission give permanent bonuses — more herb drops, better healing, etc. Search every corner |
| **7** | **阴阳术和忍术也很重要 / Onmyo & Ninjutsu Matter Too** | 不要只加体/心/刚——阴阳术( buff/治疗)和忍术(陷阱/隐身)极大提升生存和效率——各加到10-15点 / Don't just pump Body/Heart/Stamina — Onmyo (buffs/heals) and Ninjutsu (traps/stealth) drastically improve QoL. Get each to 10-15 |
| **8** | **温泉回血 / Use Hot Springs** | 每个关卡有温泉（固定在特定位置）——可以回满血+消除常世。第一次泡永久增加生命恢复量 / Each mission has hot springs (fixed positions) — full heal + cleanses Yokai Realm. First visit permanently boosts HP recovery |
