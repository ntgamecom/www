# Boss攻略 / Boss Guide

仁王2的Boss设计以高伤害、快节奏、多阶段为特色。每个Boss都有独特的攻击模式和弱点。本文将覆盖主线及主要支线Boss的打法——重点讲解应对策略、弱点利用和逃课玩法 / Nioh 2's bosses feature high damage, fast pace, and multi-phase design. Each boss has unique attack patterns and weaknesses. This guide covers all main and major side bosses — focusing on counter-strategies, weakness exploitation, and safe tactics.

---

## Boss通用机制 / Universal Boss Mechanics

| 机制 / Mechanic | 说明 / Description |
|------|------|
| **精力伤害 > 血量伤害 / Ki Damage > HP Damage** | Boss精力空了=大硬直→可以打一套完整连段。前期精力伤害比HP伤害更重要 / Boss at zero Ki = big stun → full combo window. Prioritize Ki damage over HP in early phase |
| **Boss精力在人类/妖状态不同 / Boss Ki differs by form** | 人类Boss精力恢复极快——必须持续压制不让回精。妖怪Boss有两条精力条(被打光一条就瘫痪)/ Human bosses regen Ki fast — constant pressure needed. Yokai bosses have two Ki bars (one break = stagger) |
| **混沌状态 / Confusion** | 同时上两种属性异常(水火雷等) → Boss精力不恢复+受到伤害增加50%——终极爆发窗口 / Two elemental statuses at once → Boss Ki stops regen + 50% damage taken — ultimate burst window |
| **妖怪化应对 / Yokai Shift** | 对妖怪Boss时——完美格挡红色攻击=大硬直。妖怪化状态下(用妖力槽)的连续攻击可以快速清空Boss精力 / Perfect block red attacks for big stun on yokai bosses. Yokai Shift rapid attacks quickly drain boss Ki |
| **Boss战前的准备 / Pre-Boss Prep** | 进雾门之前上好Buff(护身符、金刚符、结界符)。提前吃仙药把药自动补充调好 / Buff before entering fog gate (Protection, Steel, Barrier talismans). Pre-configure Elixir auto-restock |

## 主线Boss攻略 / Main Story Boss Guide

### 第一章 / Region 1

#### 1. 马头鬼 / Mezuki

| 属性 / Attribute | 信息 / Info |
|------|------|
| **弱点 / Weakness** | 雷/水 / Lightning/Water |
| **抗性 / Resistance** | 风 / Wind |
| **掉落 / Drops** | 马头鬼魂核 / Mezuki Soul Core |
| **难度 / Difficulty** | ★★☆☆☆ (新手劝退Boss但本身不难) |

**打法**: 马头鬼是新手第一个劝退Boss。它的攻击大开大合: 斧头横扫→侧闪，马头下砸→向后闪，前冲突刺→向侧面闪。**核心策略**: 站在马头鬼左侧绕背——它的攻击判定在正面和右侧较大。下砸后是最大输出窗口(约2秒)。注意它变红光(变身)后增加前冲+二连横扫 combo——此时保持中距离等它出完连段再上 / **Strategy**: Mezuki is the first newbie wall. Attacks are telegraphed: axe sweep → dodge sideways, head slam → dodge back, charge → dodge sideways. **Core tactic**: Circle to its left side — hitboxes are worst in front and right. Head slam = biggest window (~2s). Watch for red aura (transformation) with new charge + double sweep combo — stay mid-range and punish after its combo ends.

#### 2. 夜刀神 / Yatsu-no-Kami

| 属性 / Attribute | 信息 / Info |
|------|------|
| **弱点 / Weakness** | 火 / Fire |
| **抗性 / Resistance** | 毒 / Poison |
| **掉落** | 夜刀神魂核 |
| **难度** | ★★★☆☆ (新手Boss分水岭) |

**打法**: 夜刀神是新手第一个真正的难关——两条蛇臂+蛇身本体。**核心策略**: 开场先把两条蛇臂打掉(打蛇臂本体不掉血)——蛇臂没了后夜刀神的攻击模式大幅简化。蛇臂重生后再次拆掉。**输出窗口**: 地面翻滚(冲撞攻击)→跑到身后输出。毒雾阶段→拉开距离等毒雾散。镰鼬阶段(半血后增加的侧面镰鼬攻击)→向Boss方向闪避 / **Strategy**: Yatsu-no-Kami is the first real challenge — two snake arms + main body. **Core tactic**: Destroy both arms first (they don't damage body) — armless mode is much simpler. Arms respawn → destroy again. **Windows**: ground roll charge → dodge through and hit from behind. Poison cloud phase → disengage until it clears. Scythe wind phase (sub-50% HP) → dodge toward boss.

#### 3. 烟烟罗 / Yamanba (可选/Optional)

| 属性 / Attribute | 信息 / Info |
|------|------|
| **弱点** | 水 |
| **抗性** | 火 |
| **难度** | ★★★☆☆ |

**打法**: 烟烟罗在烟雾中隐身→突然出现攻击。**核心策略**: 保持移动——不要站定。它消失后立即开始跑动/翻滚，出现瞬间完美格挡→变身Boss硬直→一套带走。需要注意的是它的连抓(连续3-4下)——等它出完再反击 / **Strategy**: Yamanba goes invisible in smoke → appears to attack. **Core**: Keep moving — never stand still. When it vanishes, start running/rolling. Perfect block at reappearance → big stun → full combo. Watch for its claw barrage (3-4 hits) — wait for it to finish before countering.

### 第二章 / Region 2

#### 4. 镰鼬 / Kamaitachi

| 属性 / Attribute | 信息 / Info |
|------|------|
| **弱点** | 雷 / Lightning |
| **抗性** | 风 / Wind |
| **难度** | ★★★☆☆ |

**打法**: 镰鼬以速度和连段闻名——攻击间隙极小。**核心策略**: 不要贪刀——每次只打1-2下。镰鼬的攻击有个规律：每次连段后必定后撤→停顿约0.5秒→然后再次冲过来。这个0.5秒就是你出手的时机。**红色攻击**: 旋风斩(全身变红旋转)必须完美格挡——挡到=大硬直，挡不到=大半管血。半血后追加空中连续突刺——向侧面连续闪避 / **Strategy**: Kamaitachi is all about speed and combos — tiny attack windows. **Core**: Don't get greedy — 1-2 hits max per window. Pattern: combo → backstep → 0.5s pause → charge again. That 0.5s is your counter window. **Red attack**: Spinning cyclone (full body red) must be perfect blocked — success = huge stun, failure = half HP gone. Sub-50% adds aerial thrust barrage — dodge sideways repeatedly.

#### 5. 姑获鸟 / Ubume

| 属性 / Attribute | 信息 / Info |
|------|------|
| **弱點** | 火 / Fire |
| **抗性** | 水 / Water |
| **难度** | ★★☆☆☆ |

**打法**: 姑获鸟的攻击偏向法术——投技(范围抓取→会吸血)是主要威胁。**核心策略**: 保持中近距——太远她会频繁放远程法术，太近会被投技抓到。投技前摇很大(她会双手张开→尖叫)→看到前摇立刻向后闪避2次。她放水弹/冰刺时→向侧面跑动即可回避 / **Strategy**: Ubume is caster-leaning — her grab (wide range + HP drain) is the main threat. **Core**: Stay mid-close range — too far = spell spam, too close = grab city. Grab has huge wind-up (arms open → scream) → dodge back twice. Water orbs/ice spikes → run sideways.

### 第三章 / Region 3

#### 6. 真柄直隆 / Magara Naotaka

| 属性 / Attribute | 信息 / Info |
|------|------|
| **弱點** | 水 / Water |
| **抗性** | 火 / Fire |
| **难度** | ★★★★☆ (许多人认为最难的主线Boss) |

**打法**: 真柄使用长太刀——攻击范围极大、伤害极高。**核心策略**: 站在他左侧靠近的位置——他的太刀攻击判定大多从右到左，左侧贴身反而是安全区。**红色攻击**: 居合斩(刀收鞘→红光→极快横斩)——看到收鞘动作立刻完美格挡。**P2(半血后)**: 刀变红→攻击范围再次增大+追加落雷斩(地面雷电范围攻击)——需要连续侧闪3次 / **Strategy**: Magara uses an odachi — huge range, massive damage. **Core**: Stay close on his left side — most attacks swing right-to-left, making close left the safe zone. **Red attack**: Iai slash (sheath → red glow → lightning-fast horizontal cut) — perfect block on sight. **P2 (sub-50%)**: Red blade → even bigger range + adds lightning slam (ground AoE) — dodge sideways 3 times consecutively.

### 第四章+ / Region 4+

#### 7. 浅井长政 / Azai Nagamasa

| 属性 / Attribute | 信息 / Info |
|------|------|
| **弱點** | 雷 / Lightning |
| **抗性** | 水 / Water |
| **难度** | ★★★☆☆ |

**打法**: 浅井是均衡型Boss——近战+法术结合。**核心策略**: 他放冰系法术时(地上冰霜阵→延迟爆炸)→远离范围→等爆炸后冲上去输出。他的近战连段以突刺终结——看到突刺就闪避到侧面然后输出。**P2**: 召唤冰分身→分身的攻击和本体一样快→打掉分身或等分身消失再专注本体 / **Strategy**: Azai is balanced — melee + magic mix. **Core**: Ice spells (frost circle on ground → delayed explosion) → stay out → rush in after explosion. Melee combo ends with a thrust → dodge sideways and counter. **P2**: Ice clones → attacks as fast as original → kill clones or wait for them to expire before focusing main body.

#### 8. 大太法师 / Daidara Bocchi

| 属性 / Attribute | 信息 / Info |
|------|------|
| **弱點** | 火 / Fire |
| **抗性** | 雷 / Lightning |
| **难度** | ★★☆☆☆ |

**打法**: 巨型Boss——本体是手形，场景固定在平台上。**核心策略**: 不要站正面——躲开正面光束/拳头。打手部的发光核心造成最大伤害。**输出窗口**: 拳头砸地后(拳头停留在平台上约3秒)→跑到拳头上打核心 / **Strategy**: Giant boss — hand-shaped, platform arena. **Core**: Don't stand in front — avoid frontal beam/fist. Hit the glowing core on the hands for max damage. **Window**: After fist slam (hand stays on platform ~3s) → run onto fist and hit the core.

## 精力管理与混沌 / Ki Management & Confusion

| 技巧 / Technique | 说明 / Explanation |
|:---:|------|
| **残心 / Ki Pulse** | 攻击后按R1回精——必须养成习惯。配合流转(残心时切换架势)可以额外回精 / Ki Pulse (R1 after attacking) — make it muscle memory. Combine with Flux (switch stance during Ki Pulse) for extra Ki recovery |
| **妖力攻击 / Yokai Abilities** | 魂核技能可以快速打精力——适合在Boss精力接近空时用魂核技一套打空 / Soul Core abilities rapidly drain Ki — perfect for finishing off a nearly-exhausted boss |
| **属性异常表 / Status Effect Table** | 水火雷三属性——积满一条后下次来自不同属性的攻击即触发混沌 / Fire + Water + Lightning = fill one gauge, then hit with a different element to trigger Confusion |

**混沌是通关高周目的核心机制**——给Boss上混沌后(一般用水火双异常或阴阳术)，Boss精力不恢复+受伤增伤50%。混沌持续期间是所有大招的窗口 / **Confusion is the key to NG+ cycles** — once Confused (typically via dual elemental status or Onmyo magic), boss Ki stops regenerating + 50% damage taken. Your entire burst arsenal goes here.

| **机关棍 / Splitstaff** | 大型Boss(大太法师、祸梦鬼) | 上段轻攻击→转换连段打核心 / High-stance light → transform combo hits weak points |

| **手甲 / Fists** | 全能型 / All-purpose | 驱钟馗(突进技能)追击Boss+连续的精力伤害输出 / Drive the Demon (rush skill) for pursuit + continuous Ki damage |
| **薙刀镰 / Switchglaive** | 节奏型Boss | 切换架势变形攻击——一套连段三个架势全覆盖 / Stance switching mid-combo — full coverage in one string |
