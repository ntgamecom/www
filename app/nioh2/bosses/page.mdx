# BOSS攻略 / Boss Strategies

仁王2有约40个Boss——每个Boss都有独特弱点。通用思路：红光→Burst Counter→妖怪技→残心循环 / Nioh 2 has ~40 bosses — each with unique weaknesses. General loop: red attack → Burst Counter → Yokai Ability → Ki Pulse → repeat.

---

## 主线Boss / Story Bosses

### 马头鬼 / Mezuki (第一关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 暗影之都 / Dark Realm |
| **弱点 / Weakness** | 火+雷 / Fire + Lightning |
| **Burst Counter类型** | 红光冲刺——猛反直接打断 / Red charge — Brute counter to interrupt |

**打法:**
- **新手教官**——马头鬼攻击前摇清晰，节奏稳定 / Tutorial boss — clear tells, stable rhythm
- 他使用大砍刀——攻击范围大但速度慢 / Uses large cleaver — wide arcs but slow
- **核心策略**: 保持中距离引诱他出招→躲开→打2-3刀→残心→后退 / **Core**: Mid-range bait → dodge → 2-3 hits → Ki Pulse → back off
- **红光技**: 他蓄力后发光冲刺——用**猛反(R2+△)**直接打断，然后接妖怪技 / Red charge attack — **Brute counter (R2+△)** to interrupt, then follow with Yokai Ability
- 倒地后输出时间充裕——用上段重攻击或武技 / Long punish window on knockdown — use high stance strong attack or skills
- **第二阶段**: 狂暴化——攻击频率加快但破绽更大 / Phase 2: Enraged — faster but more openings

### 夜叉 / Yasha (第二关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 妖怪之海 / Sea of Yokai |
| **弱点 / Weakness** | 雷+净 / Lightning + Purity |
| **Burst Counter** | 红光爪击三连——猛反第一下 / Red 3-claw combo — Brute counter first hit |

**打法:**
- 快速女性妖怪——使用爪子攻击。攻击频率高但每下伤害低 / Fast female Yokai — claw attacks. High frequency but low damage per hit
- **不要被连到**——她的连击可以打掉半管精力，破防后接处决 / **Don't get caught** — her combo drains half your ki, guard break → grapple
- **红光技**: 三连爪击红光——第一下用Burst Counter打断，后面的自动无效 / Red 3-claw combo — counter the first hit, rest auto-cancels
- 她后跳时可能会放远程风刃——侧移躲避 / When she backflips, she may fire wind blades — strafe
- 水底场景注意氧气——打之前确保氧气管是满的 / Underwater — make sure oxygen gauge is full before engaging

### 镰鼬 / Kamaitachi (第三关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 白刃之塔 / The Blade's Tower |
| **弱点 / Weakness** | 火 / Fire |
| **Burst Counter** | 旋转冲锋——猛反或迅反 / Spinning charge — Brute or Feral counter |

**打法:**
- 持双镰鼬的忍者型Boss——速度极快 / Twin-sickle ninja boss — extremely fast
- 他频繁使用远程风刃+突进——不要试图拉开距离，会被风筝 / Frequent wind blades + gap closers — don't try to create distance
- **近身压制**: 贴身打——他的近战攻击范围有限，贴身反而安全 / **Close pressure**: Stay glued to him — melee range is safer than mid-range
- **红光技**: 旋转冲锋——有蓄力提示。猛反打停，然后接妖技 / Red spin charge — has wind-up tell. Brute counter to stop, then Yokai Ability
- 最重要的：控制精力——他的攻击削精高，时刻注意留格挡的精力 / Critical: Manage ki — his attacks drain heavily, always keep ki for blocking

### 雪女 / Yuki-Onna (第四关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 鬼火之村 / Phantom Village |
| **弱点 / Weakness** | 火 / Fire |
| **Burst Counter** | 冰刺突进——猛反 / Ice spike thrust — Brute counter |

**打法:**
- 冰系法师Boss——远距离放冰弹+近身冰剑 / Ice mage boss — ranged ice orbs + close-range ice sword
- 冰弹可以打掉/格挡——近战打碎也行 / Ice orbs can be destroyed/blocked
- **红光技**: 冰刺突进——直线冲刺。横向闪避或Burst Counter / Ice spike thrust — linear charge. Side dodge or Burst Counter
- 她的冰地面会减速+持续伤害——不断移动不要站在冰上 / Her ice floor slows + damages — keep moving, don't stand on ice
- **最关键的**: 带火符或火守护灵——火属性附加后大幅削减她的精力恢复 / **Critical**: Bring fire talisman or fire Guardian Spirit — fire drastically reduces her ki recovery
- 她放全屏冰爆时——跑到房间边缘的石柱后面躲避 / When she casts room-wide ice blast — hide behind pillars at room edges

## Boss通用技巧 / General Boss Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **红光反制 / Counter Red Attacks** | 每个Boss都有2-3种红光攻击——熟悉前摇后100%反制。猛反最稳 / Every boss has 2-3 red attacks — learn tells for 100% counter rate. Brute is most consistent |
| **妖怪技打断 / Yokai Ability Interrupt** | 魂核的妖怪技在释放时带霸体——可以用来打断Boss的攻击节奏 / Soul Core Yokai Abilities have hyper armor during cast — use to interrupt boss rhythm |
| **守护灵召唤 / Guardian Spirit Summon** | 满守护灵条时按R2+○召唤守护灵攻击——造成大量精伤+倒地 / Full Guardian Spirit gauge → R2+○ for summon attack — massive ki damage + knockdown |
| **减速符 / Sloth Talisman** | 阴阳术的"迟钝符"极大降低Boss速度——新手神器。很多Boss被减速后难度骤降 / Onmyo "Sloth Talisman" massively slows bosses — beginner's best friend |
| **结界符 / Barrier Talisman** | 自动残心+加速精力回复——打Boss前必上 / Auto Ki Pulse + accelerated ki recovery — must-have before every boss |
| **忍术隐身 / Ninjutsu: Catwalking** | 跑图神器——敌人不容易发现你。配合"透波"可以直接跑到Boss房 / Catwalking + Suppa = run past all mobs straight to boss |
