# 战斗系统 / Combat System

仁王2的战斗系统是系列最复杂的——融合了魂系的难度、忍龙的快节奏和独特的残心系统 / Nioh 2's combat system is the most complex in the series — blending souls difficulty, Ninja Gaiden speed, and the unique Ki Pulse system.

---

## 架式系统 / Stance System

每种武器有上中下三段架式——切换架式改变攻击模组 / Every weapon has High, Mid, and Low stances — switching stances changes attack movesets.

### 上段 / High Stance

| 属性 / Attribute | 说明 / Description |
|---|---|
| **攻击力 / Damage** | 最高 / Highest |
| **攻速 / Speed** | 最慢 / Slowest |
| **精力消耗 / Ki Cost** | 最大 / Highest |
| **削韧 / Stagger** | 最高 / Highest |
| **适用 / Best For** | 大伤害窗口、破角 / Big damage windows, horn break |

### 中段 / Mid Stance

| 属性 / Attribute | 说明 / Description |
|---|---|
| **攻击力 / Damage** | 中等 / Medium |
| **攻速 / Speed** | 中等 / Medium |
| **精力消耗 / Ki Cost** | 中等 / Medium |
| **削韧 / Stagger** | 中等 / Medium |
| **适用 / Best For** | 万金油——格挡性能最好 / All-rounder — best blocking |

### 下段 / Low Stance

| 属性 / Attribute | 说明 / Description |
|---|---|
| **攻击力 / Damage** | 最低 / Lowest |
| **攻速 / Speed** | 最快 / Fastest |
| **精力消耗 / Ki Cost** | 最小 / Lowest |
| **削韧 / Stagger** | 最低 / Lowest |
| **适用 / Best For** | 快速位移+躲避 / Fast movement + evasion |

## 残心系统 / Ki Pulse

残心是仁王的核心机制——攻击后按R1回精力 / Ki Pulse is Nioh's core mechanic — press R1 after attacking to recover ki.

| 操作 / Input | 说明 / Description |
|---|---|
| **攻击后按R1 / R1 after attack** | 残心——恢复消耗的精力的一部分 / Recover portion of spent ki |
| **完美残心 / Perfect Ki Pulse** | 在蓝色光圈收缩到体内的瞬间按R1——恢复最多精力 + 清除常世 / Press R1 when blue ring collapses — max ki recovery + cleans yokai realm |
| **流转 / Flux** | 残心时切换架式——额外回精 + 获得增益 / Switch stances during ki pulse — extra ki + buff |
| **流转二 / Flux II** | 流转时连续切换两个架式——更多回精 / Switch 2 stances during flux — more ki recovery |

### 残心技巧 / Ki Pulse Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **养成习惯 / Build the Habit** | 每次攻击后按R1——形成肌肉记忆 / Press R1 after every attack — make it muscle memory |
| **流转是进阶 / Flux is Advanced** | 熟练掌握后可以无限连击 / Master Flux for infinite combos |
| **常世清除 / Yokai Realm Cleanse** | 残心清除地上的紫色常世——不清理会阻碍精力恢复 / Cleanses purple pools on ground — they block ki recovery |

## 妖力系统 / Yokai Abilities

仁王2新增了妖怪化能力——半妖状态提供强大的攻击 / Nioh 2 adds Yokai Shift — a half-demon state with powerful attacks.

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **妖力槽 / Yokai Shift Gauge** | 屏幕左侧的紫色槽——攻击+妖技填充 / Purple gauge on left — fills on attacks + yokai skills |
| **妖怪化 / Yokai Shift** | R1+○变身为妖怪形态——HP+妖力消耗 / R1+○ transforms into yokai form — drains HP+gauge |
| **妖怪技 / Yokai Skills** | 装备魂核后——R2+△/○等释放 / Equip Soul Cores — R2+skill to use |
| **妖怪反击 / Burst Counter** | R2+○对红光攻击进行反击——成功造成大硬直 / R2+○ counters red attacks — successful hit causes massive stagger |

### 妖怪反击种类 / Burst Counter Types

| 类型 / Type | 操作 / Input | 效果 / Effect |
|---|---|---|
| **猛 / Brutal** | R2+○正面冲撞 / R2+○ charge forward | 正面硬碰——打中即可 / Head-on collision — needs to connect |
| **迅 / Phantom** | R2+○瞬移闪避 / R2+○ teleport dodge | 瞬移到背后反击——时机宽松 / Teleport behind — generous timing |
| **幻 / Feral** | R2+○高速突进 / R2+○ fast rush | 快速突进反击——范围广 / Fast rush counter — wide range |

## 混沌（Debuff）系统 / Confusion (Debuff) System

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **混沌 / Confusion** | 敌人同时中两种属性异常——大幅增加承受伤害+减缓行动 / Enemy with 2 element statuses — takes massive bonus damage + slows |
| **属性异常 / Element Status** | 火(灼烧)、水(减速)、雷(麻痹)、土(防御降低)、毒(持续伤) / Fire(burn), Water(slow), Lightning(paralyze), Earth(def down), Poison(DOT) |
| **混沌收益 / Confusion Benefit** | 混沌中的敌人精力恢复极慢——可以持续压制到死 / Enemy ki recovery is extremely slow — can keep them locked down |
| **触发方式 / Trigger** | 用阴阳术或守护灵攻击——叠加两种属性 / Use Onmyo magic or Guardian Spirits — stack 2 elements |

## 战斗常见问题 / Combat FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **哪个架式最好用？/ Best stance?** | 看情况——上段打Boss、中段打精英、下段打小兵 / Depends — High for bosses, Mid for elites, Low for mobs |
| **残心总是按不好？/ Can't ki pulse?** | 先练完美残心的蓝光时机——有视觉提示 / Practice the blue ring timing — visual cue helps |
| **妖怪化怎么用？/ When to yokai shift?** | 当妖力槽满+面对强敌时——变身后可以硬吃攻击 / When gauge is full vs strong enemy — can tank hits |
| **魂核怎么选？/ Soul core choice?** | 前期用猛系 + 攻击型魂核——后期换功能型 / Early use Brutal + attack cores — late swap to utility |
| **混沌系统重要吗？/ Is confusion important?** | 极其重要——打Boss必备。混沌后的Boss几乎任你宰割 / Extremely important — essential vs bosses. Makes them trivial |
| **武技和普通攻击配合？/ Skill + normal combo?** | 攻击后接武技取消硬直——残心后可以再接武技。形成无限连段 / Attack → skill to cancel recovery → ki pulse → skill again. Infinite combo potential |
| **阴阳术什么时候用？/ Onmyo usage timing?** | Boss战开场加Buff（护身符+结界符）——打中后上迟钝符+克金符 / Buff at start (Protection+Barrier) — then Sloth+Weakness on hit |
