# 流派构建 / Builds & Equipment

仁王2的Build深度是所有魂系游戏中最高的——武器+守护灵+魂核+套装+词缀，组合无限 / Nioh 2 has the deepest build system of any souls-like — weapons + Guardian Spirits + Soul Cores + sets + affixes, infinite combinations.

---

## 武器流派 / Weapon Builds

### 1. 双刀水形剑流 / Dual Swords Water Sword Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主武器 / Main Weapon** | 双刀 / Dual Swords |
| **推荐守护灵 / Guardian Spirit** | 提马鹫 / Tengen Kujaku |
| **核心武技 / Core Skill** | 水形剑(Water Sword) / 十字斩(Cross Slash) |
| **推荐套装 / Set** | 金吾套 / Kingo's Set |
| **难度 / Difficulty** | ★★☆ |

**核心思路**: 双刀攻速快+水形剑持续输出爆炸——配合提马鹫(全属性吸收)+阴阳术增强 / Dual Swords' fast speed + Water Sword sustained DPS — pair with Tengen Kujaku (all-element absorb) + Onmyo buffs

### 3. 阴阳术混合流 / Onmyo Hybrid Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主武器 / Main Weapon** | 任意——推荐单刀或双刀 / Any — Katana or Dual Swords |

| 配置 / Setup | 推荐 / Recommendation |

## 装备锻造与重铸 / Forging & Tempering

| 操作 / Action | 说明 / Description |
|------|------|
| **锻造 / Forging** | 在铁匠铺用素材制作装备——可以制作特定套装部件 / Craft gear at blacksmith — can target specific set pieces |
| **重铸 / Tempering** | 随机替换装备上的词缀——用"重铸材料"洗出想要的属性 / Reroll affixes on gear — use "Tempering Materials" to chase perfect rolls |
| **魂合 / Soul Matching** | 提升装备等级——用高等级装备喂给低等级的好词缀装备 / Raise gear level — feed high-level gear into good-affix gear |
| **改造 / Remodel** | 改变装备的加点偏向——例如重甲改轻甲属性需求 / Change gear's stat scaling — e.g. heavy armor → lighter stat requirements |

> **第一周目标**：穿等级最高的装备就行。NG+开始认真锻造和重铸 / **NG Goal**: Equip highest level gear. Start forging/tempering in NG+
