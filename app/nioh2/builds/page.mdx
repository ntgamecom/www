# 流派构建 / Builds & Equipment

仁王2的Build深度是所有魂系游戏中最高的——武器+守护灵+魂核+套装+词缀，组合无限。本攻略覆盖从开荒到终局的实用配装方案 / Nioh 2 has the deepest build system of any souls-like — weapons + Guardian Spirits + Soul Cores + sets + affixes create infinite combinations. This guide covers practical builds from early game to endgame.

---

## 武器流派 / Weapon Builds

### 1. 双刀水形剑流 / Dual Swords Water Sword Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主武器 / Main Weapon** | 双刀 / Dual Swords |
| **推荐守护灵 / Guardian Spirit** | 提马鹫 / Tengen Kujaku |
| **核心武技 / Core Skill** | 水形剑(Water Sword) / 十字斩(Cross Slash) |
| **推荐套装 / Set** | 金吾套 / Kingo's Set |
| **难度 / Difficulty** | ★★☆ |

双刀攻速快+水形剑持续输出爆炸——配合提马鹫(全属性吸收)+阴阳术增强。**操作要点**：水形剑要在敌人精力耗尽后才释放，否则容易被反击。十字斩作为追击技——敌人精力空后接十字斩打最大伤害。**属性加点**：主心(体)+副武(技)，保证双刀伤害的同时兼顾负重穿中甲 / Dual Swords' fast speed + Water Sword sustained DPS — pair with Tengen Kujaku (all-element absorb) + Onmyo buffs. **Key timing**: Only use Water Sword after enemy ki is depleted, or you'll get interrupted. Cross Slash as a follow-up for max damage on zero-ki enemies. **Stat priority**: Main Heart + secondary Skill for damage while wearing medium armor.

### 2. 大太刀雪月花流 / Odachi Moonlit Snow Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主武器 / Main Weapon** | 大太刀 / Odachi |
| **推荐守护灵 / Guardian Spirit** | 白虎 / Kato |
| **核心武技 / Core Skill** | 雪月花(Moonlit Snow) / 呼子鸟(Whistling Arrow) |
| **推荐套装 / Set** | 盾无套 + 弁庆套混搭 / Tatenashi + Benkei mix |
| **难度 / Difficulty** | ★☆☆ |

雪月花三连斩——高伤害+长霸体——配合减耗词缀可以连续释放。**操作要点**：雪月花的第三段伤害最高——但如果被打断会损失全部伤害。建议在敌人倒地或精力耗尽时完整打出三段。**属性加点**：主刚(体)+副武(勇)，穿重甲不超重的同时保证精力恢复速度。大太刀距离长，新手容易上手 / Moonlit Snow triple-slash — high damage + hyper armor — with -ki cost affixes for spam. **Key timing**: Third hit deals the most damage but leaves you vulnerable. Only commit to full combo when enemy is down or out of ki. **Stat priority**: Main Stamina + secondary Heart for heavy armor without overweight, plus ki recovery. Long reach makes Odachi beginner-friendly.

### 3. 阴阳术混合流 / Onmyo Hybrid Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主武器 / Main Weapon** | 任意——推荐单刀或双刀 / Any — Katana or Dual Swords |
| **推荐守护灵 / Guardian Spirit** | 玄武 / Kujaku (或葛叶狐 / Kuzunoha) |
| **核心阴阳术 / Core Onmyo** | 迟钝符、结界符、克金符、夺灵符 / Sloth, Barrier, Weakness, Extraction |
| **推荐套装 / Set** | 阴阳师轻装 / Onmyo Warrior set |
| **难度 / Difficulty** | ★☆☆ |

阴阳术BUFF上满→武器附魔→站桩输出。**BUFF循环**：起手结界符(提高精力恢复)→克金符(降防)→迟钝符(减速60%)→夺灵符(攻击回妖力)→武器附魔(属性伤害)→开打。**阴阳术力**：影响BUFF持续时间和效果强度——装备和守护灵上堆阴阳术力词缀。**属性加点**：主咒(忍)+副体(心)，咒99之前优先加咒 / Buff with Onmyo → weapon buff → sustained DPS. **Buff rotation**: Barrier (ki regen) → Weakness (DEF down) → Sloth (60% slow) → Extraction (anima regen) → element weapon → engage. **Onmyo Magic Power**: Affects buff duration and strength — stack on gear and Guardian Spirit. **Stat priority**: Main Magic + secondary Constitution, rush Magic to 99 first.

### 4. 绝境流 / Last Gambit Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主武器 / Main Weapon** | 任何——推荐手甲或双刀 / Any — Fists or Dual Swords |
| **守护灵 / Guardian Spirit** | 玄武 / Genbu |
| **核心词缀 / Core Affix** | 绝境减伤、绝境攻击提升 / Dmg reduction (critical) + ATK up (critical) |
| **套装 / Set** | 猪前立头盔 + 其他通用 / Shuten Doji helm + general |
| **难度 / Difficulty** | ★★☆ |

保持在30%血量以下触发大量增益——包括减伤、攻击力、精力恢复——高风险高回报。**核心机制**：绝境状态下(血量&lt;30%)，绝境减伤词缀叠加可达到接近90%减伤——看似危险实则很硬。**操作要点**：开战前用自残道具(如"晒衣杆")或让小怪把自己打到绝境→然后上Buff开打。**需要注意**：绝境流怕持续伤害(如毒、火)——必须备好解状态道具。对Boss机制不熟悉时慎用 / Stay under 30% HP for massive buffs — damage reduction, ATK up, ki recovery. **Core mechanic**: Stacking "Damage Reduction (Critical)" can reach ~90% reduction — you're actually tankier at low HP. **Setup**: Self-damage with items (e.g., "Clothesline Pole") or let mobs hit you to critical → then buff and engage. **Watch out**: Vulnerable to DoT (poison, fire) — always carry status cleanse. Not recommended when unfamiliar with boss patterns.

### 5. 妖怪化流 / Yokai Shift Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主武器 / Main Weapon** | 任意——推荐手甲或旋棍 / Any — Fists or Tonfa |
| **守护灵 / Guardian Spirit** | 九尾 / Nine-Tails (DLC) 或 白泽 / Byakko |
| **核心词缀 / Core Affix** | 妖力增加量、妖怪化持续时间、妖怪技伤害 / Anima charge, Yokai Shift duration, Yokai ability damage |
| **套装 / Set** | 弁庆套或伊邪那岐 / Benkei or Izanagi set |
| **难度 / Difficulty** | ★★★ |

妖怪化形态下攻防大幅提升，但妖力条管理是关键。**操作要点**：不要在满血时妖怪化——先消耗血量再用妖怪化回血+爆发，一举两得。妖怪化中的重攻击威力最大，但也会消耗更多妖力条。**魂核搭配**：优先选妖力消耗低的魂核(如饿鬼)，在妖怪化形态下频繁释放妖怪技延长变身时间。**属性加点**：主咒(忍)+副体(勇)，堆够阴阳术容量 / Yokai Shift boosts ATK/DEF significantly — anima management is key. **Pro tip**: Don't activate at full HP — take damage first, then shift to heal + burst simultaneously. Heavy attack in Yokai Shift deals the most damage but consumes more gauge. **Soul Core setup**: Prioritize cheap-cost cores (like Gaki) to spam abilities during shift and extend duration. **Stat priority**: Main Magic + secondary Heart for enough Onmyo capacity.

## 加点策略总览 / Stat Priority Summary

| 流派 / Build | 优先属性 / Priority Stats | 说明 / Note |
|:----:|:----:|------|
| **物理近战 / Melee** | 心(体) > 武(勇) > 刚 | 伤害+精力+负重平衡 / Dmg + Ki + Equip load |
| **阴阳术 / Onmyo** | 咒 > 忍 > 体 | 咒99优先——阴阳术容量决定战斗力 / Magic to 99 first |
| **忍术 / Ninjutsu** | 忍 > 技 > 体 | 忍99——飞镖/苦无流核心 / Dex to 99 — Kunai/Shuriken core |
| **绝境 / Last Gambit** | 体 > 刚 > 心 | 血量+防御优先——活着才有输出 / HP + DEF first |
| **妖怪化 / Yokai Shift** | 咒 > 体 > 勇 | 妖力管理+阴阳术辅助 / Anima management + Onmyo support |

## 装备锻造与重铸 / Forging & Tempering

| 操作 / Action | 说明 / Description |
|------|------|
| **锻造 / Forging** | 在铁匠铺用素材制作装备——可以定向制作特定套装部件。一周目不用刻意锻造，穿等级最高的即可 / Craft gear at blacksmith — can target specific set pieces. Don't bother in NG, just equip highest level |
| **重铸 / Tempering** | 随机替换装备上的词缀——用"重铸材料"洗出想要的属性。**优先洗**：绝境减伤、攻击力、属性伤害、精力减耗 / Reroll affixes on gear using Tempering Materials. **Priority affixes**: Critical dmg reduction, ATK, element damage, ki cost reduction |
| **魂合 / Soul Matching** | 提升装备等级——用高等级装备喂给低等级的好词缀装备。注意：魂合费用随次数递增，建议只给核心装备用 / Raise gear level by feeding high-level gear into good-affix gear. Cost increases with each match — use sparingly on core pieces |
| **改造 / Remodel** | 改变装备的加点偏向——例如重甲改为轻甲属性需求。中后期调整装备权重的关键功能 / Change gear's stat scaling. Key for mid-to-late game optimization |

> **一周目目标**：穿等级最高的装备就行。不要花大量时间锻造——NG+才开始认真配装。二周目后解锁稀有度更高的"绿色"装备和"橙色"套装，才是真正的Build开始 / **NG Goal**: Equip highest level gear. Don't waste time forging — NG+ is where building begins. Green rarity gear and Orange sets unlock in NG+, that's the real start.

## 进阶技巧 / Advanced Tips

| 技巧 / Tech | 说明 / Description |
|:----:|------|
| **残心流转 / Ki Pulse Flux** | 攻击后按R1恢复精力+切换架势——进阶玩家的必修课。熟练掌握后精力几乎无限 / Press R1 after attacks to recover ki + stance switch. Master this for nearly infinite ki |
| **妖怪反 / Burst Counter** | 三种妖怪反各有优劣：猛型(正面硬刚)、幻型(精准时机)、迅型(快速闪避)。推荐新手用猛型——容错率高 / Three Burst Counter types: Brute (aggressive), Phantom (precise timing), Feral (evasive). Brute recommended for beginners |
| **混沌状态 / Confusion** | 给敌人上两种以上属性异常→触发混沌→精力不恢复+受伤加深。这是仁王2最强大的输出机制 / Apply 2+ elemental status effects → Confusion → no ki recovery + increased damage taken. The most powerful damage mechanic in the game |
| **忍者/阴阳术容量 / Capacity** | 通过加点提升术容量——每5点对应的主属性(+忍/+咒)增加1容量。优先凑满核心技能的容量需求 / Stats increase capacity — every 5 points in Magic/Dex adds 1 capacity. Prioritize fitting core skills |
