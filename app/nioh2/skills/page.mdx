## 技能常见问题 / Skills FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **先点哪个武器技能？/ First weapon skill?** | 先点出"残心·天/人/地"（流转相关）——再点核心输出技 / Get Ki Pulse skills first — then core damage skills |
| **阴阳术还是忍术？/ Onmyo vs Ninjutsu?** | 都点——阴阳术Buff+忍术复活是万金油组合 / Both — Onmyo buffs + Ninjutsu quick-change are versatile |
| **魂核怎么升级？/ Upgrade soul cores?** | 在神社供奉不需要的魂核——获取魂核精华升级 / Offer unused cores at shrine — get essence to level |
| **守护灵可以换吗？/ Switch guardian spirit?** | 可以——在神社选择守护灵 / Yes — at shrine |
| **奥传技能是什么？/ Mystic Arts?** | 每种武器的终极技能——120万熟练度解锁 / Each weapon's ultimate skill — 1.2M proficiency |
| **阴阳术奥传怎么解锁？/ Onmyo mystic art unlock?** | 完成阴阳师支线任务——阿国线结局奖励 / Complete onmyo mage sub-missions — Okuni questline reward |
| **忍术奥传怎么解锁？/ Ninjutsu mystic art unlock?** | 完成忍者支线任务——半蔵线结局奖励 / Complete ninja sub-missions — Hanzo questline reward |
