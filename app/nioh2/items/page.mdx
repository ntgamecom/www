# 物品收集 / Items & Collectibles

仁王2的收集要素丰富：木灵、温泉、遗发、精华、装备掉落 / Nioh 2 has rich collectibles: Kodama, hot springs, locks of hair, Amrita, gear drops.

---

## 木灵 / Kodama

每个主线关卡有3-9只木灵。提交给神社的大木灵获得永久加成 / Each main mission has 3-9 Kodama. Submit to Great Kodama shrine for permanent bonuses.

### 木灵 Buff 表 / Kodama Buffs

| 提交数 / Submitted | 效果 / Effect |
|:----:|------|
| 5 | 药草掉落率 +1 / Herb drop rate +1 |
| 10 | 仙药(血瓶)掉落率 +1 / Elixir drop rate +1 |
| 15 | 药草回复量 +1 / Herb heal amount +1 |
| 20 | 仙药上限 +1 / Elixir capacity +1 |
| 25 | 精华获得量 +1% / Amrita earned +1% |
| 30+ | 继续叠加奖励 / Stacking bonuses continue |

---

## 温泉 / Hot Springs

第一关泡温泉永久增加**生命恢复量**——全收集后恢复量翻倍 / First soak in each hot spring permanently boosts **HP recovery amount** — all collected = doubled recovery.

| 主线关卡 / Main Missions | 温泉数 / Springs |
|:----:|:----:|
| 本体 / Base Game | 约20个 / ~20 |
| DLC | 约5个 / ~5 |
| **总计** | **25个** |

---

## 遗发 / Locks of Hair

遗发提供忍者/阴阳术技能点——全收集可学满所有技能 / Locks of Hair provide Ninjutsu/Onmyo skill points — all collected = max all skills.

| 类型 / Type | 数量 / Count | 获得方式 / How to Get |
|:----:|:----:|------|
| **忍者遗发 / Ninja Lock** | 约30个 / ~30 | 支线任务奖励 + 隐藏地点 / Side rewards + hidden locations |
| **阴阳术遗发 / Onmyo Lock** | 约30个 / ~30 | 同上 / Same as above |
| **作用 / Use** | 每个遗发 = 1忍者/阴阳术技能点 | 初期加忍/咒到10，开技能树 / Get Ninjutsu/Onmyo to 10, unlock skill trees |
