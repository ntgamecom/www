# 物品收集 / Items & Collectibles

仁王2的收集要素丰富：木灵、温泉、遗发、精华、装备掉落，以及魂核、绘卷、制作书等深度收集内容。合理的收集策略可以大幅降低游戏难度 / Nioh 2 has rich collectibles: Kodama, hot springs, locks of hair, Amrita, gear drops, plus Soul Cores, Scrolls, and smithing texts. Smart collection strategies significantly reduce game difficulty.

---

## 木灵 / Kodama

每个主线关卡有3-9只木灵。提交给神社的大木灵获得永久加成——全收集后药草掉落率+9、仙药上限+9、精华获得量大幅提升 / Each main mission has 3-9 Kodama. Submit to Great Kodama shrine for permanent bonuses — full collection = herb drop +9, elixir cap +9, massive Amrita gain boost.

### 木灵 Buff 表 / Kodama Buffs

| 提交数 / Submitted | 效果 / Effect |
|:----:|------|
| 5 | 药草掉落率 +1 / Herb drop rate +1 |
| 10 | 仙药(血瓶)掉落率 +1 / Elixir drop rate +1 |
| 15 | 药草回复量 +1 / Herb heal amount +1 |
| 20 | 仙药上限 +1 / Elixir capacity +1 |
| 25 | 精华获得量 +1% / Amrita earned +1% |
| 30+ | 继续叠加奖励——最高25级 / Stacking continues — max 25 levels |

**木灵全收集的收益**: 每关的仙药(血瓶)上限从默认的3瓶提升到8瓶(全木灵后)。加上掉落率提升，你几乎不会缺药——这是降低游戏难度最直接的途径 / **Full Kodama benefit**: Elixir cap goes from default 3 to 8 (with all Kodama). Combined with drop rate boost, you'll rarely run out of heals — the single biggest difficulty reducer.

## 温泉 / Hot Springs

第一关泡温泉永久增加**生命恢复量**——全收集后恢复量翻倍。每个主线关卡有1个温泉 / First soak in each hot spring permanently boosts **HP recovery amount** — all collected = doubled recovery. One per main mission.

| 主线关卡 / Main Missions | 温泉数 / Springs |
|:----:|:----:|
| 本体 / Base Game | 约20个 / ~20 |
| DLC 1-3 | 约5个 / ~5 |
| **总计** | **25个** |

**温泉效果**: 默认生命恢复量=每秒恢复约1%HP→全温泉后=每秒约2%HP。**重要**: 恢复量的提升在Boss战中作用有限(战斗中恢复速度不变)，但跑图和清小怪时能省下大量药 / **Effect**: Default recovery = ~1% HP/sec → full springs = ~2%/sec. **Note**: Recovery rate doesn't change during combat, but saves tons of elixirs during exploration and mob clearing.

## 遗发 / Locks of Hair

遗发提供忍者/阴阳术技能点——全收集可学满所有技能。这是游戏中最有价值的收集品之一，因为忍术和阴阳术的被动加成(如"忍术力量"和"阴阳术力量")直接增加武器伤害 / Locks of Hair provide Ninjutsu/Onmyo skill points — full collection unlocks all skills. One of the most valuable collectibles since Ninjutsu/Onmyo power passives directly increase weapon damage.

| 类型 / Type | 数量 / Count | 获得方式 / How to Get |
|:----:|:----:|------|
| **忍者遗发 / Ninja Lock** | 约30个 / ~30 | 支线任务奖励 + 隐藏地点 / Side rewards + hidden locations |
| **阴阳术遗发 / Onmyo Lock** | 约30个 / ~30 | 同上 / Same as above |
| **作用 / Use** | 每个遗发 = 1忍者/阴阳术技能点 | 初期加忍/咒到10，开技能树 / Get Ninjutsu/Onmyo to 10, unlock skill trees |

**遗发获取策略**: 一周目随缘收集——看到就捡。二周目开始补全收集。支线任务会奖励大量遗发——不要跳过全部支线。忍者遗发优先给"速身术"(增加敏捷)和"虎躯术"(跑步加速)；阴阳术遗发优先给"结界符"(自动消除常世)和"护身符"(减伤) / **Collection strategy**: NG+: grab as you find. NG+: full collection run. Side quests reward many locks — don't skip all side content. Ninja priority: Quick-change Scroll + Catwalking. Onmyo priority: Barrier Talisman + Protection Talisman.

## 魂核系统 / Soul Core System

击败妖怪敌人有概率掉落魂核——装备在守护灵上提供技能和属性加成。每个守护灵可装备2个魂核。**魂核稀有度**与装备相同(黄→紫→橙)，高稀有度魂核有更好的词缀和初始等级 / Defeating yokai enemies has a chance to drop Soul Cores — equip on Guardian Spirits for skills + stats. Each spirit holds 2 cores. **Rarity** same as gear (Yellow→Purple→Ethereal), higher rarity = better affixes and starting level.

| 魂核类型 / Type | 特点 / Feature |
|:----:|------|
| **妖怪系 / Yokai** | 攻击型技能——伤害高 / Offensive skills — high damage |
| **人形系 / Human** | 回复/辅助型技能 / Heal/support skills |
| **特殊 / Special** | Boss掉落——效果独特 / Boss drops — unique effects |

**魂核建议**: 优先使用Boss魂核(效果通常最强)。初期魂核每关换一次——保持等级与关卡匹配。三周目开始认真筛选魂核词缀 / **Tip**: Prioritize boss cores (strongest effects). Swap cores every mission early game — keep level matched. Start optimizing affixes from NG++.
