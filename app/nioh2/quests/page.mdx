## 任务常见问题 / Quests FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **支线要不要做完？/ Do all subs?** | 推荐做完——解锁技能、守护灵和最高难度 / Recommended — unlock skills, spirits, and NG+ cycles |
| **什么时候进二周目？/ When to NG+?** | 通关主线后直接进——装备在二周目会快速更新 / After main story — gear upgrades fast in NG+ |
| **稀人是什么？/ What is a visitor?** | 其他玩家被你召唤协助过关——类似于魂系的太阳战士 / Other players summoned to help — like souls' Sunbro |
| **常暗是什么？/ Yokai Realm?** | 妖怪领域的紫色区域——残心可以清理 / Purple yokai area — ki pulse cleanses |
| **木灵有什么用？/ Kodama?** | 收集木灵增加神社提供物品的掉落率 + 回血 / Collect to increase shrine item drops + healing |
| **温泉有什么用？/ Hot springs?** | 每个温泉永久增加体力上限——全收集约+500HP / Each spring permanently increases max HP — full collection ~+500 HP |
| **道场练习有用吗？/ Dojo practice useful?** | 非常重要——可以无风险测试武器连段和熟悉Boss招式 / Very important — risk-free test weapon combos and learn boss patterns |
| **茶室有什么用？/ Teahouse use?** | 联机匹配和稀人召唤——也可以自定义称号和外观 / Online matchmaking + visitor summon — also customize titles and appearance |
| **锻造屋升级材料哪刷？/ Forge upgrade mats?** | 各章节Boss+精英掉落。高周目掉更高级材料 / Bosses + elite drops per chapter. Higher NG = better materials |

| 类型 / Type | 说明 / Description | 奖励特点 / Rewards |
|---|---|---|
