# 新手入门 / Beginner Guide

仁王2的战斗系统比魂系复杂得多——三种架势、残心、妖怪化、魂核……本章帮你理清所有基础 / Nioh 2's combat is far more complex than souls games — three stances, Ki Pulse, Yokai Shift, Soul Cores... This chapter clears the basics.

---

## 核心操作 / Core Controls (PC/PS)

| 操作 / Action | PS | PC | 说明 / Description |
|------|:----:|:----:|------|
| **轻攻击 / Quick Attack** | □ | 鼠标左键 / Left click | 快速连击——各架势攻击动作不同 / Fast chain — different moves per stance |
| **重攻击 / Strong Attack** | △ | 鼠标右键 / Right click | 高伤害/高削精——节奏慢 / High damage/ki damage — slower |
| **武技 / Active Skill** | R1+□/△ | 鼠标侧键 / Mouse side | 装备的武器技能——各武器独立 / Equipped weapon skills — unique per weapon |
| **闪避 / Dodge** | ✕ | Space | 有无敌帧——比魂系短但可用 / Has i-frames — shorter than souls but usable |
| **残心 / Ki Pulse** | R1 | R | **最重要的操作**——攻击后按R1回收精力。不残心打不了几下就没精力了 / **THE most important** — press after attacks to recover ki. No pulse = exhausted in seconds |
| **防御 / Block** | L1 | Q | 格挡——消耗精力。精力为0时被打会破防 / Block — consumes ki. Zero ki = guard break |
| **妖怪技 / Yokai Ability** | R2+□/△/○ | R+1/2/3 | 使用装备的魂核技能——不消耗精力 / Use equipped Soul Core skills — costs zero ki |
| **妖怪化 / Yokai Shift** | R2+○ | R+左键 | 变身妖怪——有独立HP条，大幅提升攻防 / Transform into Yokai — separate HP bar, major buffs |
| **Burst Counter** | R2+△ | R+右键 | 反制红光攻击——三种类型见下文 / Counter red-glow attacks — 3 types below |
| **切换武器 / Weapon Swap** | 方向左/右 / D-pad left/right | 1/2 | 近战武器两把随时切换 / Two melee weapons, swap anytime |
| **切换架势 / Stance Switch** | 上=R1+✕ / 中=R1+○ / 下=R1+△ | 鼠标滚轮 / Scroll | 战斗中随时切换架势——不同动作+不同武技 / Switch stances mid-combat — different movesets |
| **锁定 / Lock On** | R3 | 鼠标中键 / Middle click | 锁定敌人 / Lock onto enemy |
| **道具 / Item** | 方向上下 / D-pad up/down | E/R | 使用快捷道具栏中物品 / Use quick slot items |

## 残心系统详解 / Ki Pulse System

残心是仁王2的精髓——不会残心等于不会玩 / Ki Pulse is the soul of Nioh 2 — not pulsing = not playing correctly.

| 概念 / Concept | 说明 / Description |
|------|------|
| **什么是残心 / What is Ki Pulse** | 攻击会消耗精力——攻击后有一段时间精力会"虚耗"（白色条）。按R1瞬间回收虚耗部分 / Attacks consume ki — after attacking, spent ki appears as a white bar. Press R1 to instantly recover it |
| **为什么要残心 / Why Ki Pulse** | 不残心的话精力恢复极慢——打两套连击就喘气了。残心让你持续输出 / Without pulsing, ki recovers very slowly — 2 combos = exhausted. Pulse = infinite stamina |
| **完美残心 / Perfect Ki Pulse** | 在白条刚好消失的瞬间按R1——出现蓝色光效，额外恢复精力+触发残心BUFF / Press R1 exactly when white bar fades — blue flash for bonus ki + buff triggers |
| **消除常世 / Cleansing Yokai Realm** | 站在常世（紫色区域）中才能残心——残心会消除周围的常世。不在常世中也可以残心 / Stand in Yokai Realm to Ki Pulse — pulse cleanses nearby pools. Can pulse outside realm too |
| **残心BUFF / Pulse Buffs** | 技能树的"残心·天/人/地"——在不同架势下完美残心触发攻击/防御/精力回复加成 / Skill tree passives — perfect pulse in each stance triggers attack/defense/ki recovery buffs |

## 妖怪化与Burst Counter / Yokai Shift & Burst Counter

| 概念 / Concept | 说明 / Description |
|------|------|
| **妖怪条 / Yokai Gauge** | 屏幕左侧紫色条——攻击/被攻击/使用魂核积累。满格=可妖怪化 / Left side purple gauge — filled by attacking/taking damage/using cores. Full = Yokai Shift ready |
| **妖怪化效果 / Shift Effects** | 变身后：攻击力x1.5，精力无限，防御大幅提升，HP耗尽前不会死亡。可使用妖怪技不消耗妖力 / Transformed: 1.5x attack, infinite ki, massive defense, can't die until HP depleted. Use Yokai Abilities free |
| **妖力 / Anima** | 妖怪技消耗的资源——击杀/攻击积累。妖力上限由魂核和守护灵决定 / Resource for Yokai Abilities — gained by killing/attacking. Max determined by Soul Cores & Guardian Spirit |
| **红光攻击 / Red-Glow Attacks** | 敌人身上闪烁红光的大招——可用Burst Counter反制。成功反制=大硬直+大量伤害 / Enemy flash-red special moves — counterable with Burst Counter. Success = big stun + massive damage |
| **猛 Burst Counter / Brute** | R2+△直接一拳打上去——硬扛红光攻击。最简单：看到红光→按R2+△即可 / R2+△ to punch through red attacks. Easiest: see red → press R2+△ |
| **迅 Burst Counter / Feral** | R2+△冲刺闪避——如果刚好穿过红光攻击则反击。需要时机判断 / R2+△ to dodge through red attacks — counter if you phase through. Needs timing |
| **幻 Burst Counter / Phantom** | R2+△展开护盾——格挡红光攻击后反击。类似格挡防反 / R2+△ deploys a barrier — block red attack then counter. Parry-like |

> **推荐**：新手选**猛(Brute)**——最简单直接，不需要复杂的时机判断 / **Recommendation**: Beginners pick **Brute** — simplest, no complex timing needed

## 武器类型与推荐 / Weapon Types

| 武器 / Weapon | 难度 / Difficulty | 特点 / Characteristics |
|:----:|:----:|------|
| **双刀 / Dual Swords** | ★☆☆ | 攻速快、耗精低、容易上手 / Fast, low ki cost, beginner-friendly |
| **太刀 / Katana** | ★★☆ | 均衡型——武技丰富，帅 / Balanced — most versatile skills, looks cool |
| **大太刀 / Odachi** | ★★☆ | 高伤害+长范围，但攻速慢 / High damage + long reach, but slow |
| **锁镰 / Kusarigama** | ★★★ | 上段范围广，下段快。操作复杂 / High stance has wide range, low is fast. Complex |
| **双斧 / Dual Hatchets** | ★★☆ | 中距离投掷+近战结合 / Mid-range throwing + melee hybrid |
| **长枪 / Spear** | ★★☆ | 长距离压制，适合控制节奏 / Long reach for spacing and control |
| **旋棍 / Tonfa** | ★★★ | 攻速最快——高连击+高削精。消耗精力也快 / Fastest attack speed — high combo + ki damage. Ki hungry |
| **机关棍 / Switchglaive** | ★★★ | 变形武器——各架势形态不同。进阶向 / Transformative weapon — different form per stance. Advanced |
| **手甲 / Fists** | ★★★★ | 连击之王——武技可以无限连。极高上限 / Combo king — skills chain infinitely. Highest skill ceiling |
| **薙刀镰 / Splitstaff** | ★★★ | 中距离优秀——可以变长变短 / Great mid-range — extends/retracts |

> **新手推荐**：双刀或大太刀——一个快一个猛，都容易上手 / **Beginner Pick**: Dual Swords (fast) or Odachi (powerful) — both easy to start with

## 初期加点建议 / Early Stat Allocation

| 属性 / Stat | 影响 / Affects | 开局加点 / Early Points |
|:----:|------|:----:|
| **体 / Body** | HP + 部分武器需求 / HP + weapon req | 10 |
| **心 / Heart** | 精力上限 / Ki max | 15 |
| **刚 / Stamina** | 负重上限 / Weight limit | 10 |
| **武 / Strength** | 部分武器需求 + 攻击力 / Weapon req + ATK | 主武器需求 / Main weapon req |
| **技 / Skill** | 部分武器需求 + 技巧伤害 / Weapon req + skill dmg | 主武器需求 / Main weapon req |
| **忍 / Ninjutsu** | 忍术容量 / Ninjutsu capacity | 10 |
| **咒 / Onmyo** | 阴阳术容量 / Onmyo capacity | 10 |

> **标准开局加点**：体10、心15、刚10、忍10、咒10。主武器需求满足后→补心(精力)或体(血量) / **Standard start**: Body10, Heart15, Stamina10, Ninjutsu10, Onmyo10. Meet weapon reqs → Heart (ki) or Body (HP)

## 新手常见问题 / FAQ

| 问题 / Question | 回答 / Answer |
|------|------|
| **精力不够用？/ Always out of ki?** | 记住每次攻击后按R1残心。技能树学"残心·天/人/地"。装"-1攻击精力消耗"词缀 / Ki Pulse after every attack. Learn "Perfect Ki Pulse" passives. Roll "-1 ki cost" affixes |
| **被红光技秒了？/ One-shot by red attacks?** | 看到红光→立刻用Burst Counter（R2+△）。猛(Brute)最简单 / See red → R2+△ immediately. Brute is easiest |
| **装备太多了怎么办？/ Too much loot?** | 在神社可以"奉献(Offer)"不需要的装备——获得精华和材料。后期也可以拆解 / Offer unwanted gear at shrines for Amrita + materials. Later can also disassemble |
| **打不过第一个Boss？/ Stuck on first boss?** | 第一个Boss(马头鬼)弱火——用阴阳术的火符。保持中距离，他出招后打2刀就收 / First boss (Mezuki) is weak to fire — use Onmyo fire talisman. Mid-range, 2 hits then back off |
| **怎么联机？/ How to co-op?** | 使用"常世同行"或"义之刃"——在神社或关卡内召唤。需要联网+PS+/PC / Use "Expedition" or "Benevolent Graves" at shrines. Requires internet + PS+/PC |
| **Build什么时候开始配？/ When to optimize build?** | 第一周目随便穿——挑等级最高的装备。二周目(强者之道)再开始凑套装和词缀 / NG: equip highest level gear. NG+ (Way of the Strong): start optimizing sets |
