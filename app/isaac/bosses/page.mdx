# BOSS与精英 / Bosses & Champions

以撒的结合有100+ Boss——分布在6层(楼层)和多个隐藏路线中。每层Boss从几个固定Boss中随机 / Isaac has 100+ bosses — spread across 6 floors and multiple hidden paths. Each floor's boss is randomly selected from a pool.

## 关键Boss打法 / Key Boss Strategies

### 蒙斯特罗 / Monstro

| 信息 / Info | 说明 / Description |
|------|------|
| **楼层 / Floor** | 地下室 / Basement |
| **类型 / Type** | 跳跃压制型 / Jump-slam |

**打法**: 他会跳起→砸向你→弹幕喷发。保持移动，他跳的落点有红圈预警——走开就行。他跳完后打他2-3枪循环 / He jumps → slams at you → bullet burst. Keep moving — landing has red warning circle. Shoot 2-3 times after each jump

### Mom's Heart (子宫Boss)

| 信息 / Info | 说明 / Description |
|------|------|
| **楼层 / Floor** | 子宫 / Womb |
| **类型 / Type** | 固定位置弹幕战 / Fixed position bullet hell |

**打法**: 
- 战在房间中间——Heart固定在中央，周围有4个门出小兵 / Heart is fixed center, 4 doors spawn adds
- 攻击模式：激光横扫(躲在门缝里)→弹幕环(站角落)→血泪(直线移动躲避) / Laser sweep (hide behind doors) → bullet ring (corners) → blood tears (strafe)
- 优先清小兵——小兵掉落红心和钥匙，维持血量 / Clear adds first — they drop hearts and keys for sustain
- **推荐道具**: 穿透类(妈刀、硫磺火、科技X)——可以同时打Heart和小兵 / Piercing items (Knife, Brimstone, Tech X) — hit both Heart and adds

### 撒旦 / Satan (阴间Boss)

| 信息 / Info | 说明 / Description |
|------|------|
| **楼层 / Floor** | 阴间 / Sheol |
| **类型 / Type** | 多阶段Boss / Multi-phase |

**打法**:
- **P1**: 巨大脚掌踩地+冲击波——跑到两侧安全区打脚掌 / Giant foot stomp + shockwave — run to sides, shoot the foot
- **P2**: 头部出现+激光扫射——站在头部正下方可以躲激光 / Head appears + laser sweep — stand directly below the head to avoid laser
- **P3**: 全屏弹幕+小兵——输出检查。高伤害Build这里轻松过 / Full-screen bullets + adds — DPS check. High-damage build makes this easy

## 精英敌人 / Champions

| 精英类型 / Champion | 特点 / Trait | 应对 / Counter |
|:----:|------|------|
| **红色精英 / Red Champion** | 血量翻倍+移动加速 / Double HP + faster | 风筝打法——不要站撸 / Kite — don't face-tank |
| **紫色精英 / Purple Champion** | 隐身+传送 / Invisible + teleport | 注意脚步声和传送特效 / Watch for footstep sounds and teleport effects |
| **蓝色精英 / Blue Champion** | 冰冻光环——靠近就减速 / Freeze aura — slow on approach | 远程攻击优先 / Ranged attacks preferred |
| **黄色精英 / Yellow Champion** | 分裂——死后变成3个小怪 / Splits into 3 on death | 保持距离，逐个击破 / Keep distance, kill one at a time |

## Boss通用技巧 / General Boss Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **Boss房前准备 / Pre-Boss Prep** | 确保血量健康 + 主动道具充能满 / Full HP + charged active item |
| **环绕类道具 / Orbital Items** | 环绕类(献祭匕首、Cube of Meat)在Boss战中持续输出——非常强 / Orbitals (Sacrificial Dagger, Cube of Meat) provide constant DPS in boss fights |
| **减速/冰冻 / Slow/Freeze** | 减速道具(蜘蛛、冰冻眼泪)对Boss效果极佳——大幅降低难度 / Slow items (Spider, Freeze tears) work great on bosses — massive difficulty reduction |
| **穿透攻击 / Piercing** | 妈刀、硫磺火、科技X——这些道具让Boss战简单一个档次 / Mom's Knife, Brimstone, Tech X — these items trivialize boss fights |
