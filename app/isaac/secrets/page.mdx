# 隐藏与挑战 / Secrets & Challenges

以撒的结合以藏得极深的内容著称——隐藏角色、隐藏路线、34个挑战模式 / Isaac is famous for its deeply hidden content — secret characters, alternate paths, and 34 challenge modes.

---

## 隐藏路线 / Secret Paths

### 首领路线 / Boss Rush Route

| 步骤 / Step | 说明 / Description |
|:----:|------|
| **条件 / Condition** | 在20分钟内到达Mom's Heart / Reach Mom's Heart within 20 minutes |
| **奖励 / Reward** | 进入Boss Rush——连续打所有主线Boss。奖励强力主动道具 / Enter Boss Rush — fight all main bosses in sequence. Reward: powerful active item |

### 贪婪模式 / Greed Mode

| 特性 / Feature | 说明 / Description |
|------|------|
| **解锁 / Unlock** | 初始可用 / Default |
| **玩法 / Gameplay** | 波浪式防守——击败15波敌人后打Boss / Wave defense — 15 waves then Ultra Greed boss |
| **特色 / Special** | 共用金钱池——商店在回合间出现 / Shared money pool — shop between waves |
| **奖励 / Reward** | 解锁Keeper角色 / Unlocks Keeper character |

## 挑战模式 / Challenge Mode

| 挑战 / Challenge | 难度 / Difficulty | 特点 / Feature |
|:----:|:----:|------|
| **C1: 波板糖 / Pitch Black** | ★☆☆ | 不能射眼泪——只能用炸弹和物品 / No tears — bombs and items only |
| **C2: 高脚 / High Brow** | ★★☆ | 只能用"书"型主动道具攻击 / Book-type actives only |
| **C3: 头目突击 / Head Trauma** | ★★★ | 你只有1红心——但攻击力极高 / 1 HP but very high damage |
| **C4: 黑暗降临 / Darkness Falls** | ★★★★ | 永久黑暗——视野极小 / Permanent darkness — tiny vision |
| **C5: 索恩 / The Tank** | ★★★ | 你巨大+移动慢但高攻 + 高血 / Giant + slow but high HP + damage |
| **C6: 太阳神 / Solar System** | ★★★ | 持续掉血——必须在时间耗尽前通关 / Continuous HP drain — speedrun required |
| **C7: 自杀王 / Suicide King** | ★★★★★ | 起始道具会伤害自己——需要小心使用 / Starting item damages you — risky playstyle |
| **C8: 猫猫 / Cat Got Your Tongue** | ★★★★ | 你的眼泪变成猫——攻击方式诡异 / Tears become cats — weird attack pattern |

> **挑战模式奖励**: 每个挑战通关解锁一个新道具或饰品 / Challenge Reward: Each cleared challenge unlocks a new item or trinket
