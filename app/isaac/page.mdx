# 以撒的结合攻略 / The Binding of Isaac Guide

以撒的结合（The Binding of Isaac: Rebirth）是Edmund McMillen开发的Roguelite射击游戏——灵感来自塞尔达传说。你将扮演裸体小孩以撒，用眼泪射击怪物，在随机生成的地牢中收集数百种道具，挑战最终Boss / The Binding of Isaac: Rebirth is a Roguelite shooter by Edmund McMillen — inspired by The Legend of Zelda. Play as naked child Isaac, shoot tears at monsters, collect hundreds of items in randomly generated dungeons, and challenge the final bosses.

## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **道具系统 / Items** | 核心驱动——700+道具，每种改变角色能力。道具之间会产生化学反应——有些组合强到离谱、有些坑死你 / The core driver — 700+ items, each changing your character. Items interact with each other — some OP combos, some trap items |
| **房间类型 / Room Types** | 普通房→Boss房→商店→宝箱房→藏宝房→诅咒房→隐藏房→天使/恶魔房等——每种房有独立规则和奖励 / Normal → Boss → Shop → Treasure → Curse → Secret → Angel/Devil etc — each has unique rules and rewards |
| **路线选择 / Route Choices** | 每层的路径不固定——你可以跳过某些房间，也可以探索隐藏。道具选择+路线规划=策略核心 / Each floor's path is fluid — skip rooms, find secrets. Item choice + route planning = strategic core |
| **血量与红心 / HP & Hearts** | 红心(血量)、魂心(蓝心——优先消耗)、黑心(爆炸时放毒伤)、骨心(不掉恶魔房概率)等——管理血量是生存关键 / Red (HP), Soul (blue — consumed first), Black (poison burst on loss), Bone (doesn't affect Devil deal chance) — HP management is survival |
| **恶魔房 vs 天使房 / Devil vs Angel Deals** | 每层打完Boss有概率出恶魔房(以血量换强力道具)或天使房(免费但通常要求拒绝恶魔房)——核心抉择 / After boss: Devil Room (HP cost for power) or Angel Room (free but requires prior refusal) — critical decision |
| **道具池 / Item Pools** | 每种房间类型对应独立道具池——商店池、Boss池、藏宝池等。理解池机制可以定向刷道具 / Each room type has its own item pool — shop pool, boss pool, treasure pool, etc. Pool knowledge = targeted farming |
| **挑战模式 / Challenges** | 34个规则特殊的挑战——完成解锁新道具和成就 / 34 special-rules challenges — completing them unlocks new items and achievements |

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **速度优先 / Speed is King** | 以撒的地图大、房间多——移速慢=浪费时间+容易受伤。看到移速道具优先拿 / Isaac maps are huge — slow speed = wasted time + more hits. Prioritize speed upgrades |
| **2** | **恶魔房换血值不值？/ Are Devil Deals Worth It?** | 开局：不值（血量不够）。中期：值（强道具可以保命）。前期3血以下不要换 / Early game: No (not enough HP). Mid-game: Yes (power = survival). Skip if under 3 HP |
| **3** | **优先开地图 / Prioritize Map Reveal** | 商店看到地图/指南针——买！知道隐藏房位置=额外道具和资源 / Buy maps/compass from shop — knowing secret room locations = extra items and resources |
| **4** | **道具池策略 / Item Pool Strategy** | 不想在Boss房开出大便道具？先打宝箱房再打Boss——Boss房道具池更优秀 / Don't want poop items from boss? Open treasure room before boss room — boss pool is better |
| **5** | **高伤 > 高射速 / Damage > Fire Rate** | 初期高伤害道具的收益远高于射速。射速可以在后期补 / Early damage items scale better than fire rate. Fire rate can be supplemented later |
| **6** | **不要碰的陷阱道具 / Trap Items to Avoid** | 豆子、牛奶!(除非有回血)、科技.X(十字激光——很多人不习惯)、混沌卡(随机道具池=极不稳定) / Beans, Soy Milk (without heal), Tech X (cross laser — divisive), Chaos (random pools = chaos) |
| **7** | **隐藏房规律 / Secret Room Patterns** | 隐藏房一般连接3个以上房间（在宝箱房/商店/Boss房之间）——用炸弹炸墙壁上有X标记的地方 / Secret rooms typically connect 3+ rooms — bomb walls with crack/X marks near treasure/shop/boss rooms |
