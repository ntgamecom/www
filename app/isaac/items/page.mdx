# 道具图鉴 / Item Catalog

以撒的结合有700+道具——每个道具改变游戏体验。本章列出核心道具分类、强力组合和道具池说明，帮助你在游戏中快速识别和决策 / Isaac has 700+ items — each changes the game. This chapter covers core item categories, OP synergies, and item pool mechanics to help you make fast decisions during runs.

## 道具类型速查 / Item Types

| 类型 / Type | 图标特征 / Icon Feature | 作用 / Effect |
|:----:|------|------|
| **被动道具 / Passive Item** | 金色宝箱 / Gold chest drop | 永久生效——拿了就有效 / Permanent effect on pickup |
| **主动道具 / Active Item** | 宝箱房紫色光环 / Purple aura chest | 需要充能——按空格使用。6格/3格/1格充能不等 / Requires charge — press Space. 6/3/1-bar charge |
| **饰品 / Trinket** | 小饰品 / Small trinket | 副效果——可携带1个（有道具可增加）/ Minor passive — hold 1 (can be increased) |
| **卡牌 / Cards** | 纸牌 / Playing card | 一次性效果——Q键使用 / One-time use — press Q |
| **药丸 / Pills** | 彩色药片 / Colored pill | 随机效果——需要尝试才知道作用 / Random effect — identify by trying |
| **拾取物 / Pickups** | 金币/炸弹/钥匙/红心等 / Coin/bomb/key/heart | 基础资源 / Basic resources |

## 道具池说明 / Item Pools

| 池 / Pool | 包含道具类型 / Contains | 如何获取 / How to Access |
|:----:|------|------|
| **宝箱房池 / Treasure Pool** | 标准道具 / Standard items | 宝箱房金宝箱 / Gold chest in treasure room |
| **Boss房池 / Boss Pool** | Boss级别道具 / Boss-tier items | 击败Boss后 / After defeating boss |
| **商店池 / Shop Pool** | 实用道具 / Utility items | 商店购买 / Buy from shop |
| **恶魔房池 / Devil Pool** | 强力但消耗血量的道具 / Powerful but costs HP | 恶魔房交易 / Devil room deals |
| **天使房池 / Angel Pool** | 免费强力道具 / Free powerful items | 天使房宝箱 / Angel room chest |
| **隐藏房池 / Secret Pool** | 特殊道具 / Special items | 隐藏房内 / Inside secret room |
| **贪婪池 / Greed Pool** | 金币相关道具 / Gold-related items | 贪婪模式 / Greed mode |

**恶魔房 vs 天使房**: 不受伤→恶魔房出现概率高→如果拒绝恶魔房交易→后续出天使房。恶魔房道具强力但要扣血上限——新手优先选恶魔房。天使房道具免费但不一定比恶魔房强。**策略**: 前两层走恶魔房，后两层看Build需求选择 / **Devil vs Angel**: Don't take red heart damage → high devil room chance. Refuse devil deal → subsequent rooms become angel. Devil deals cost HP but are stronger. **Strategy**: First 2 floors take devil deals, later floors decide based on build.
