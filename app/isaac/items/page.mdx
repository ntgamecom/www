## 道具优先级 / Item Priority

| 道具类型 / Type | 优先级 / Priority | 说明 / Reason |
|---|---|---|
| **伤害提升** / Damage Up | ⭐⭐⭐⭐⭐ | 输出=生存=更快解决战斗 / DPS = survival = faster fight resolution |
| **射速提升** / Tears Up | ⭐⭐⭐⭐ | 射速=更高DPS+更好触发效果 / Fire rate = higher DPS + better proc effects |
| **移速提升** / Speed Up | ⭐⭐⭐⭐ | 速度=更好躲弹幕=更安全 / Speed = better dodge = safer runs |
| **血量提升** / HP Up | ⭐⭐⭐ | 血量=容错——但不是越高越好(恶魔房需要血量!) / HP = error margin — but not always good (devil deals need HP!) |
| **射程提升** / Range Up | ⭐⭐⭐ | 特定流派需要——但通用性不如其他 / Niche — less universal than other stats |
| **幸运提升** / Luck Up | ⭐⭐⭐ | 影响多种触发效果——但数值提升不明显 / Affects many proc effects — but impact is subtle |
