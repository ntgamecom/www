## 困难模式 / Hard Mode

| 变化 / Change | 影响 / Impact |
|---|---|
| **敌人血量+25%** | 战斗时间更长——需要更好的输出 / Longer fights — need better DPS |
| **敌人弹幕密度增加** | 走位要求更高——不建议新手直接打困难 / Higher positioning requirement — not recommended for beginners |
| **红心掉落率降低** | 血量管理更难——魂心价值提升 / HP management harder — soul heart value increases |
| **诅咒概率增加** | 更可能出现"黑暗诅咒"(视野缩小)等负面效果 / Higher chance of curses (darkness, blindness, etc.) |
| **Boss Rush时间缩短** | 从20分钟缩短到15分钟 / Reduced from 20 min to 15 min |
| **贪婪模式敌人变强** | 波次更多——经济管理更重要 / More waves — economy management more important |

## 贪婪模式 / Greed Mode

| 要素 / Element | 说明 / Description |
|---|---|
| **模式简介** | 固定波次防守战——每层打完一个商店后进入下一波。击败终极贪婪后通关 / Fixed wave defense — shop after each wave, then next wave. Beat Ultra Greed to win |
| **经济系统** | 金币在贪婪模式中极为重要——每波之间恢复所有血量。用金币买道具比在普通模式中更重要 / Gold is extremely important in Greed Mode — full heal between waves. Spending gold on items matters more than normal mode |
| **推荐角色** | 以撒(D6重roll)、犹大(高伤)、阿撒泻勒(自带飞行+高伤) / Isaac (D6 reroll), Judas (high damage), Azazel (flight + high damage) |
| **推荐道具** | 范围伤害(穿刺+神圣之光)、自动攻击(宝宝类)、经济类道具(金袋+金币降落) / AOE (Piercing + Holy Light), Auto-attack (Familiars), Economy items (Sack of Gold + Dollar) |
| **策略** | 优先拿经济道具→确保每波之间能买关键的伤害道具 → 后期堆伤害 / Prioritize economy items → ensure shop access between waves → stack DPS late |

## 每日挑战 / Daily Challenge

| 要素 / Element | 说明 / Description |
|---|---|
| **每日挑战** | 每天一个固定种子+固定起始角色——所有玩家玩同一局 / Daily fixed seed + fixed starting character — all players play the same run |
| **评分系统** | 分数基于道具收集、金钱、时间等——分数=收集度评分+效率评分 / Scored on item collection, money, time — collection score + efficiency score |
| **排行榜** | 按分数排名——每天重置 / Ranked by score — resets daily |
| **奖励** | 参与即获得"成就点数"——但实际游戏内无物质奖励 / Participation gives "achievement points" — no in-game material reward |
