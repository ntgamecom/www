# 角色攻略 / Character Guide

以撒的结合有34个角色——每个有独特的初始道具和属性。有些角色易上手，有些是进阶挑战。选择合适的角色能显著提升通关率 / Isaac has 34 characters — each with unique starting items and stats. Some are beginner-friendly, some are expert challenges. Choosing the right character significantly improves your win rate.

---

## 新手推荐角色 / Beginner Characters

### 以撒 / Isaac

| 属性 / Stat | 数值 / Value |
|:----:|:----:|
| **初始道具 / Starting Item** | 炸弹×3 / 3 Bombs |
| **血量 / HP** | 3红心 / 3 Red Hearts |
| **解锁 / Unlock** | 初始可用 / Default |

**特点**: 最均衡的角色——没有特殊机制，适合学基础 / Most balanced — no special mechanics, learn the basics
**评价**: ★★★★★ 新手首选——道具池不受限，容错率高 / Best for beginners — unrestricted item pool, high forgiveness

**以撒打法要点**: 以撒的D6(使用?/???模式)是游戏中最强的主动道具之一——每层宝箱房的道具可以无限重roll直到满意。建议每个宝箱房都roll至少2-3次。普通模式的以撒没有D6，但炸弹多的优势让他可以开隐藏房和炸罐子得资源 / **Isaac tips**: D6 (in ???/Blue Baby mode) is one of the best active items — reroll treasure room items until satisfied. Roll at least 2-3 times per room. Normal Isaac has no D6 but 3 bombs let you access secret rooms and pots for extra resources.

### 玛吉 / Maggy

| 属性 / Stat | 数值 / Value |
|:----:|:----:|
| **初始道具 / Starting Item** | 药丸×1 + 回血主动(羔羊之尾) / 1 Pill + Heal active |
| **血量 / HP** | 4红心 / 4 Red Hearts |
| **解锁 / Unlock** | 初始可用 / Default |

**特点**: 最高初始血量 + 自回血——生存能力最强 / Highest starting HP + self-heal — most survivable
**评价**: ★★★★ 适合新手熟悉Boss机制 / Great for learning boss patterns

**玛吉打法要点**: 玛吉的4红心+主动回血让她几乎不会死——非常适合用来学习Boss攻击模式。主动回血每层能用3-4次(每次充能满后使用)。注意: 回血主动道具不会在恶魔房中生效 / **Maggy tips**: 4 hearts + healing active make her nearly unkillable — ideal for learning boss patterns. The healing active can be used 3-4 times per floor. Note: healing actives don't prevent damage in Devil rooms.

### 该隐 / Cain

| 属性 / Stat | 数值 / Value |
|:----:|:----:|
| **初始道具 / Starting Item** | 钥匙×2 + 幸运脚(运气提升) / 2 Keys + Lucky Foot (luck up) |
| **血量 / HP** | 3红心 / 3 Red Hearts |
| **解锁 / Unlock** | 初始可用 / Default |

**特点**: 高运气——开宝箱出好东西概率高、赌博类道具更稳 / High luck — better chests, safer gambling items
**评价**: ★★★★ 经济实惠，适合喜欢商店流的玩家 / Economical, good for shop-heavy runs

**该隐打法要点**: 开局2钥匙让你可以同时开宝箱房和商店。幸运脚让所有药丸效果变为正面、赌博机更容易中奖。该隐的主要策略是**用钥匙开所有能开的箱子**——宝箱出的道具和资源能帮他快速成型 / **Cain tips**: 2 starting keys let you open both treasure room and shop immediately. Lucky Foot makes all pills positive and slot machines more generous. Core strategy: **open every chest you find** — the items and resources snowball quickly.

## 进阶角色 / Advanced Characters

### 亚瑟 / Azazel (新手友好型进阶 / Beginner-friendly advanced)

| 属性 / Stat | 数值 / Value |
|:----:|:----:|
| **初始道具 / Starting Item** | 硫磺火(短距离激光) + 飞行能力 / Brimstone (short-range) + flight |
| **血量 / HP** | 3黑心 / 3 Black Hearts |
| **解锁 / Unlock** | 击败撒旦3次 / Beat Satan 3 times |

**特点**: 开局就有硫磺火+飞行——伤害碾压前期。3黑心提供高容错 / Starts with Brimstone + flight — crushes early game. 3 Black Hearts = high forgiveness
**评价**: ★★★★★ 解锁后强烈推荐——最强角色之一 / Highly recommended after unlock — one of the strongest characters

**亚瑟打法要点**: 亚瑟的硫磺火是短距离版——需要贴脸输出。飞行能力让你可以越过沟壑和岩浆，阵型机动性极高。黑心被击破时爆炸清场，亚瑟几乎不会被小兵包围。**唯一弱点**: 射程短——在空旷Boss房需要靠近输出，增加风险 / **Azazel tips**: Short-range Brimstone = close combat mandatory. Flight lets you bypass pits and spikes — excellent mobility. Black Hearts explode on break, so you're never surrounded. **Only weakness**: short range means you must stay close to bosses.

### 夏娃 / Eve

| 属性 / Stat | 数值 / Value |
|:----:|:----:|
| **初始道具 / Starting Item** | 死鸟(受伤后召唤攻击小兵) + 巴比伦娼妓(低血量加攻速) / Dead Bird + Whore of Babylon |
| **血量 / HP** | 2红心 / 2 Red Hearts |
| **解锁 / Unlock** | 击败特定Boss / Defeat specific boss |

**特点**: 低血量=高输出——高风险的玻璃大炮 / Low HP = high damage — high-risk glass cannon
**评价**: ★★★ 需要熟练控血——新手慎用 / Requires HP management — not for beginners

**夏娃打法要点**: 核心是控在1红心状态触发巴比伦——加攻速+移速。极度过期：掉血=变强，但2红心的容错率极低。**推荐打法**: 开局故意掉到1红心触发巴比伦→全程保持低血量→靠魂心和黑心保命。死鸟在Boss战中输出可观 / **Eve tips**: Stay at 1 red heart to trigger Whore of Babylon (+speed + damage). Very risky: damage taken = stronger, but 2-heart margin is tiny. **Recommended play**: Intentionally drop to 1 heart early → maintain low HP → rely on soul/black hearts for protection. Dead Bird deals solid boss DPS.

### 参孙 / Samson

| 属性 / Stat | 数值 / Value |
|:----:|:----:|
| **初始道具 / Starting Item** | 血之狂暴(击杀加攻击) / Bloody Lust (kills boost damage) |
| **血量 / HP** | 2红心 / 2 Red Hearts |
| **解锁 / Unlock** | 击败特定Boss / Defeat specific boss |

**特点**: 击杀越多越强——连续清怪后伤害爆炸 / Kill streak = damage scaling — snowballs hard
**评价**: ★★★★ 适合激进玩法——清图后打Boss极强 / Great for aggressive play — clear floors then melt bosses

**参孙打法要点**: 血之狂暴每层击杀叠加攻击力——满层后伤害翻倍不止。**核心策略**: 每层先清图(小兵)→攻击力叠满→再打Boss。注意参孙只有2红心——前期血线压力大。推荐优先拿血量道具和魂心 / **Samson tips**: Bloody Lust stacks damage per kill per floor — at max stacks, damage is 2x+. **Core strategy**: Clear floor mobs first (build stacks) → then fight boss. Only 2 red hearts — prioritize HP and soul heart items early.
