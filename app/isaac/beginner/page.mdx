# 新手入门 / Beginner Guide

以撒的结合内容庞大、道具繁多——但核心机制很直观。本章帮你理解基础，活过第一层，并逐步挑战更强的Boss / The Binding of Isaac has immense content and hundreds of items — but the core is intuitive. This chapter helps you survive the first floor and progressively take on tougher bosses.

---

## 核心操作 / Core Controls (PC)

| 操作 / Action | 键位 / Key | 说明 / Description |
|------|:----:|------|
| **移动 / Move** | WASD / 方向键 | 四向移动 / Four-direction movement |
| **射击 / Shoot** | 方向键(箭头) / Arrow keys | 以撒用眼泪射击——可以边移动边射 / Isaac shoots tears — can move and shoot simultaneously |
| **炸弹 / Bomb** | E | 炸开隐藏房/岩石/敌群。核心探索工具 / Blow up secret rooms/rocks/enemies. Core exploration tool |
| **主动道具 / Active Item** | 空格 / Space | 使用当前主动道具(有充能次数) / Use current active item (has charge) |
| **卡牌/药丸 / Card/Pill** | Q | 使用当前持有的卡牌或药丸 / Use held card or pill |
| **地图 / Map** | 按住Tab / Hold Tab | 查看当前层地图 / View current floor map |
| **丢道具 / Drop Item** | 长按Ctrl / Hold Ctrl | 把当前道具丢在地上 / Drop current item on ground |

## 基础资源 / Core Resources

| 资源 / Resource | 说明 / Description |
|------|------|
| **红心 / Red Heart** | 血量——受攻击扣红心。空红心后触发"以撒的魂"形态(半透明，被击中即死) / HP — lose on hit. Zero red hearts = "Isaac's Soul" form (one-hit-kill) |
| **魂心 / Soul Heart** | 蓝色心——优先消耗。不能通过红心补血恢复，但可防止掉恶魔房概率 / Blue heart — consumed first. Can't be refilled by red heart drops, but protects Devil room chance |
| **黑心 / Black Heart** | 黑色心——消耗时爆炸放毒伤，清除周围敌人 / Black heart — explodes with poison on use, clears nearby enemies |
| **金币 / Coins** | 在商店买东西。也影响"贪婪"类道具效果 / Buy items in shops. Also affects "Greed" related item effects |
| **炸弹 / Bombs** | 炸隐藏房、岩石、商店捐款机、特殊障碍 / Blow up secret rooms, rocks, donation machines, special obstacles |
| **钥匙 / Keys** | 开宝箱房(金钥匙)、商店(银钥匙)、某些门 / Open treasure rooms (gold), shops (silver), some doors |
| **电池 / Battery** | 主动道具充能——半格/满格/超级电池 / Charge active items — half/full/super charge |

## 房间类型速查 / Room Type Reference

| 房间 / Room | 图标 / Icon | 说明 / Description |
|:----:|:----:|------|
| **普通房 / Normal** | 无 / None | 清怪得奖励(钱/炸弹/钥匙) / Clear enemies for rewards |
| **Boss房 / Boss** | 骷髅头 / Skull | 每层一个——击败Boss解锁下一层楼梯 / One per floor — beat to unlock next floor |
| **宝箱房 / Treasure** | 金色宝箱 / Gold chest | 免费拿一个道具——最强道具来源 / Free item — best item source |
| **商店 / Shop** | 钱袋 / Coin bag | 用金币买道具——经常有好货 / Buy items with coins — often has good stuff |
| **藏宝房 / Library** | 书架 / Bookshelf | 免费拿一本书型主动道具 / Free book-type active item |
| **诅咒房 / Curse** | 红色 / Red tint | 进房扣血(通常1红心)——但奖励很好 / Costs HP (1 heart) to enter — but rewards are good |
| **天使房 / Angel** | 白色门 / White door | 打完Boss概率出——免费强力道具 / After boss with some chance — free powerful items |
| **恶魔房 / Devil** | 红色门 / Red door | 打完Boss概率出——用血量换强力道具 / After boss with some chance — trade HP for power |
| **隐藏房 / Secret** | 无(墙上裂缝) / None (wall crack) | 用炸弹炸开——通常有道具或钱 / Bomb open — usually items or coins |
| **挑战房 / Challenge** | 红色光圈 / Red aura | 限制条件下清怪——奖励好但危险 / Clear enemies under restrictions — good but risky |

## 游戏进程 / Game Progression

以撒每次游戏有6-12层(取决于路线选择) / Each run has 6-12 floors (depending on path choice):

| 层数 / Floor | 名称 / Name | Boss | 说明 / Note |
|:----:|:----:|:----:|------|
| **F1** | 地下室 / Basement | Montro / Gemini / Duke | 最简单的层——熟悉操作 / Easiest — learn controls |
| **F2** | 洞穴 / Caves | Gurdy / Pin / Chub | 难度上升——注意血量管理 / Harder — watch HP |
| **F3** | 墓穴 / Depths | **Mom** | 第一个真正的挑战——攻击模式多样化 / First real challenge — varied attack patterns |
| **F4** | 子宫 / Womb | **Mom's Heart** | 新手第一个目标——弹幕战 / First major milestone — bullet hell boss |
| **F5 (可选)** | 阴间 / Sheol | Satan | 需要特殊道具解锁——比子宫线更难 / Needs key pieces — harder than Womb path |
| **F5 (可选)** | 大教堂 / Cathedral | Isaac | 天使路线——比阴间线简单 / Angel path — easier than Sheol |
| **F6+** | 宝箱/黑暗 / Chest/Dark Room | ??? / The Lamb | 隐藏路线——最难的Boss / Hidden paths — hardest bosses |

## 新手生存法则 / Survival Tips

| # | 法则 / Rule | 说明 / Description |
|:----:|------|------|
| **1** | **活下来比凑Build重要 / Survive > Build** | 死掉的Build就是没Build——优先拿血量和移速。输出后面会有的 / A dead build is no build — prioritize HP and speed. Damage comes later |
| **2** | **开局二选一 / First Floor Priority** | 第一件宝箱房道具决定前期走向——不管拿到什么，围绕它打 / First treasure room item sets your run direction — whatever it is, build around it |
| **3** | **Boss前清全图 / Clear Floors Before Boss** | 每层的宝箱房和商店在Boss房前先探——拿到好道具再打Boss更稳 / Open treasure and shop before boss — better to power up first |
| **4** | **隐藏房每层炸 / Bomb Every Secret Room** | 隐藏房大概率在宝箱房/Boss房/商店附近——用炸弹探。好道具很多出自隐藏房 / Secret rooms are near treasure/boss/shop rooms — bomb to find. Many great items hide here |
| **5** | **好道具记下来 / Learn the Items** | 700+道具记不住是正常的——新手建议开网页查道具说明 / 700+ items is too many to memorize — use the wiki or item guides |
| **6** | **Mom's Heart不难 / Mom's Heart is Doable** | 新手的第一个目标是打到Mom's Heart(第4层Boss)。坚持几个小时后就能稳定到那里。重点是保持血量健康/First milestone: Mom's Heart (floor 4 boss). A few hours of practice and you'll reach it consistently |
| **7** | **不受伤=恶魔房 / No Damage = Devil Room** | 整层不扣红心(魂心/黑心被扣不影响)→打完Boss有高概率出恶魔房。恶魔房用血量换强力道具/No red heart damage for full floor = high devil room chance. Trade HP for powerful items |
| **8** | **不要捡太多红心 / Don't Hoard Red Hearts** | 魂心优先消耗——但如果你捡了太多红心填充，下层的恶魔房概率降低。控制红心上限 / Red hearts decrease devil room chance. If you want devil deals, keep red hearts low and rely on soul hearts |

## 推荐解锁顺序 / Recommended Unlock Path

1. **用以撒打到Mom's Heart 10次** → 解锁蓝宝(???) / Beat Mom's Heart 10x with Isaac → unlock ???
2. **用玛吉打完整游戏** → 解锁暗黑犹大 / Complete a run with Maggy → unlock Dark Judas

## 常见死因及对策 / Common Deaths & Solutions

| 死因 / Cause | 对策 / Solution |
|---|---|
| **前期被小怪围殴** | 走位不要站定——进房间先往后撤观察敌人分布——再逐个击破 / Keep moving — enter rooms and back off to observe — then pick off enemies one by one |
| **恶魔房换血后血不够** | 除非血量在3红心以上，否则不要轻易换恶魔房道具。蓝心不算恶魔房成本 / Don't take devil deals unless you have 3+ red hearts. Soul hearts don't count for devil deal cost |
| **Boss弹幕太密躲不开** | 保持靠近Boss——大多数Boss的弹幕在近身范围反而比远处稀疏 / Stay close to most bosses — bullet patterns are often sparser at close range |
| **诅咒房伤不起** | 诅咒房扣1红心——如果你魂心多可以进(魂心先扣)。但如果只有红心且血量&lt;3，不要进 / Curse room costs 1 heart — safe if you have soul hearts. Skip if &lt;3 red hearts |
| **宝箱房开出垃圾道具** | 先用D6(以撒的能力)重掷——或者先不开宝箱房，等打完Boss用恶魔房/Angel房再开 / Reroll with D6 (Isaac's ability) — or leave treasure room for after boss and angel/devil room decisions |

## 游戏设置建议 / Game Settings Tips

| 设置 / Setting | 建议 / Recommendation | 原因 / Reason |
|---|---|---|
| **全屏/窗口** | 窗口或全屏——根据个人习惯 / Windowed or fullscreen — personal preference | 没有明显影响——选你觉得舒服的 / No significant difference — pick what's comfortable |
| **游戏速度** | 默认(正常)/ Default (Normal) | 新手不要调快——初期操作不够熟练 / Don't speed up as a beginner — controls need practice |
| **滤镜(CRT/像素)** | 建议关掉CRT滤镜——像素滤镜更清晰 / Turn off CRT filter — pixel filter is clearer | CRF滤镜会模糊画面——影响弹幕判断 / CRT blurs the screen — makes bullet hell harder to read |
| **屏幕抖动** | 建议关掉 / Recommend off | 炸Boss房时抖动会干扰走位 / Screen shake during boss bombs interferes with positioning |
| **音乐音效** | 打开——音乐质量很高 / Keep on — soundtrack is excellent | 以撒的配乐非常棒——特别是Boss战 / Isaac's OST is outstanding — especially boss themes |
