## 实用地图技巧 / Map Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **全清vs速通** | 如果目标是通关——全清(所有房间)更安全。如果目标是Boss Rush或Hush——跳过关卡中不必要的战斗 / If aiming for completion — full clear (all rooms) is safer. If aiming for Boss Rush or Hush — skip unnecessary fights |
| **恶魔房vs天使房** | 如果在第1层就拿到强力道具(如硫磺火)——后面几层不用太在意恶魔房。如果Build较弱——依赖恶魔房交易逆袭 / If you get a strong item (Brimstone) on floor 1 — don't sweat Devil Rooms later. If build is weak — rely on Devil deals for a comeback |
| **Greed模式地图** | 贪婪模式的地图比普通模式小很多——但波次更密集。优先范围伤害和自动攻击 / Greed Mode maps are much smaller than normal mode — but waves are more dense. Prioritize AOE and auto-attacks |
| **Repentance地图** | Repentance增加了更多房间类型和路线选择——地图更大、更复杂。多准备炸弹 / Repentance adds more room types and route choices — maps are larger and more complex. Bring extra bombs |
