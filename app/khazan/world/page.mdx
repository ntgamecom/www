# 地图探索 / World Exploration

第一狂战士：卡赞的世界分为多个区域，每个区域有独特的敌人、收集品和隐藏路径。探索收益极大——隐藏宝箱、锻造素材和技能书都藏在角落 / The world of The First Berserker: Khazan is divided into multiple regions, each with unique enemies, collectibles, and secret paths. Exploration rewards are huge — hidden chests, crafting materials, and skill books are tucked away in corners.

## 亡者海岸 / Coast of the Dead

游戏的起点——一片搁浅了无数沉船的荒凉海岸。卡赞在此苏醒，踏上复仇之路 / The starting point — a desolate coast littered with shipwrecks. Khazan awakens here and begins his path of vengeance.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **沉船海滩 / Shipwreck Beach** | 游戏起点——出生洞穴出口 / Starting cave exit | 第一个篝火、基础商人 / First bonfire, basic merchant |
| **悬崖洞穴 / Cliff Cave** | 左侧悬崖跳下→发现隐藏洞穴 / Left cliff jump → hidden cave | 卡赞雕像x1 + 金币3000 / Khazan Statue x1 + 3000 gold |
| **亡者营地 / Dead Camp** | 反叛军的临时营地——Boss战 / Rebel camp — boss fight | 反叛军首领 + 通往幽暗森林 / Rebel Leader + path to Darkwood |
| **沉船遗迹 / Wreck Remains** | 沙滩上的沉船——内部有宝箱 / Shipwreck hull — chest inside | 铁矿石x5 + 基础防具 / Iron Ore x5 + basic armor |

## 幽暗森林 / Darkwood Forest

密林覆盖的黑暗森林——地面和地下洞穴系统错综复杂。适合仔细探索 / A dark forest of dense trees — complex network of surface paths and underground caves. Rewards thorough exploration.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **森林入口 / Forest Entrance** | 从亡者海岸过来后的第一个区域 / First zone from Coast | 篝火、地图碎片 / Bonfire, map fragment |

| **地下洞窟 / Underground Cave** | 多层洞穴——有瀑布和深潭 / Multi-layer caves — waterfall + deep pool | 双剑武器图纸 + 精炼石 / Dual Blades blueprint + Refined Stone |
| **监视者祭坛 / Watcher's Altar** | 深渊监视者Boss战所在地 / Abyss Watcher boss arena | Boss战 + 通往铁砧城 / Boss fight + path to Anvil City |

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **城门广场 / Gate Plaza** | 城市入口——喷泉广场 + 商人 / City entrance — fountain plaza + merchant | 篝火、军械库钥匙(商人购买) / Bonfire, Armory Key (buy from merchant) |

## 营地与休息点 / Camps & Rest Points

| 功能 / Function | 说明 / Description |
|:----:|------|
| **休息恢复 / Rest** | 完全恢复HP+重置消耗品——同时刷新所有普通敌人(除Boss外) / Full HP + item reset — also respawns all normal enemies (except bosses) |
| **篝火强化 / Bonfire Upgrade** | 使用"篝火余烬"(消耗品)永久提升营地效果——+HP恢复速度在营地附近生效 / Use "Bonfire Embers" to permanently upgrade camp effects — HP regen boost near camp |
| **传送 / Fast Travel** | 已解锁的营地之间可以快速传送——后期探索必备 / Fast travel between unlocked camps — essential for late-game exploration |
| **铁匠铺 / Smithy** | 部分大营地有铁匠——可以强化/分解/锻造装备 / Some major camps have smiths — upgrade/dismantle/craft gear |
| **仓库 / Storage** | 存取不用的装备和素材——负重有限所以善用仓库 / Store excess gear and materials — equip load is limited |

## 探索技巧 / Exploration Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **看到墙壁有裂缝就攻击 / Cracked walls = attack** | 几乎所有隐藏路径都藏在可破坏墙壁后面——用重击或者翻滚撞开 / Almost all secret paths are behind breakable walls — use heavy attack or roll through |
| **悬崖下方通常有东西 / Cliffs often hide secrets** | 看到悬崖边缘时往下看——下方可能有平台/洞穴/宝箱。跳下去前先确认下方有落脚点 / Look down from cliffs — platforms/caves/chests below. Confirm landing spot before jumping |
| **精英怪会刷新 / Elites respawn** | 部分精英怪在休息后会刷新——击杀掉落锻造素材。可以反复刷 / Some elites respawn after resting — kill for crafting materials. Farmable |
| **地图上标记未探索路径 / Mark unexplored paths** | 游戏内地图会显示未探索的分支路径——用标记功能标记后再探索 / In-game map shows unexplored branches — use marker function before exploring |
| **听声音找隐藏 / Listen for secrets** | 靠近隐藏墙壁时——画面会有微微的粒子效果+声音提示 / Near hidden walls — subtle particle effect + sound cue |
