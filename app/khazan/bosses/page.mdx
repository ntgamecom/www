# BOSS攻略 / Boss Strategies

第一狂战士：卡赞的Boss战强调节奏——每个Boss都有独特的攻击模式、快慢刀节奏和阶段转换。理解这些模式比装备更重要 / The First Berserker: Khazan boss fights emphasize rhythm — each boss has unique attack patterns, delayed attack timings, and phase transitions. Understanding patterns matters more than gear.

---

## Boss战通用原则 / Universal Boss Tips

| 原则 / Rule | 说明 / Description |
|:----:|------|
| **前几次只观察不攻击 / Observe first** | 进入Boss战后前30秒不攻击——专注观察攻击模式和快慢刀节奏 / First 30 seconds: don't attack — focus on learning attack patterns and delayed timing |
| **完美格挡 > 闪避 / Parry > Dodge** | Boss的连招最后一下通常是完美格挡窗口——格挡成功=大硬直+反击机会 / The last hit of a boss combo is usually a parry window — success = big stagger + counter chance |
| **阶段转换 = 有动画 / Phase transition** | Boss转阶段时通常有长时间动画——这是免费输出的最大窗口。全力输出 / Boss phase transition animations are free DPS windows. Go all out |
| **属性克制 / Elemental Weakness** | 每个Boss有属性弱点——用正确的附魔/属性武器伤害x1.5 / Each boss has an elemental weakness — correct element = 1.5x damage |
| **不要贪刀 / Don't Get Greedy** | 魂系第一铁律——少打一刀比多挨一刀强 / First rule of souls-likes — better to miss one hit than take one hit |

## 前期Boss / Early Game Bosses

### 反叛军首领 / Rebel Leader

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-60%)** | 三连斩(左→右→下劈) —— 节奏均匀 / Three-slash combo (L→R→down) — even rhythm | 完美格挡第三下→反击。简单 / Parry the third hit → counter. Easy |
| **P2 (60%-0%)** | 追加：旋转斩(360度)+突刺 / Adds: spinning slash (360) + thrust | 旋转斩往后闪避→等突刺时完美格挡。P2同理P1节奏 / Dodge back from spin → parry the thrust. Same rhythm as P1 |
| **属性弱点 / Weakness** | 无(教学Boss) | — |
| **逃课打法 / Cheese** | 拉开距离→等他出三连斩→闪到背后→重击。重复到死 / Keep distance → wait for three-slash → dodge behind → heavy attack. Repeat |

### 深渊监视者 / Abyss Watcher

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-50%)** | 快慢刀——快刀(0.3秒)+慢刀(1.5秒)+突进斩。**快慢刀节奏是关键** / Delayed attacks — fast (0.3s) + slow (1.5s) + rush. **Rhythm is key** |
| **P2 (50%-0%)** | 分身召唤——Boss召唤2个分身同时攻击。真身会发光 / Clone summon — boss spawns 2 clones. Real body glows | 跑位拉开→用AOE清分身→然后集中打真身 / Kite → use AoE to clear clones → focus real body |
| **属性弱点 / Weakness** | 火焰(伤害x1.3) / Fire (1.3x) | 双剑附魔火焰→持续输出 / Dual Blades with fire enchant → sustained DPS |
| **逃课打法 / Cheese** | 长矛+卡距离——刺一下→后退→刺一下→后退。分身阶段跑远等结束再回来打 / Spear + distance abuse — poke → retreat → repeat. Run from clone phase |

## 中期Boss / Mid Game Bosses

### 堕落骑士 / Fallen Knight

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-40%)** | 剑+盾组合——举盾防御→盾击(破防)→突进斩→横扫 / Sword + shield — block → shield bash (guard break) → rush → sweep | 不要正面硬拼——绕背打。盾击时可以向后闪避 / Don't fight head-on — circle behind. Dodge back from shield bash |
| **P2 (40%-0%)** | 丢弃盾牌→双持巨剑→范围攻击x3+旋转升空斩 / Drops shield → dual-wield greatsword → AoE x3 + spinning uppercut | 所有攻击范围大但**后摇也大**——等他挥空后从背后重击 / All attacks have wide range but **long recovery** — punish from behind after whiff |
| **属性弱点 / Weakness** | 冰霜(减速+破盾) / Ice (slow + shield break) | 冰剑/冰附魔→P1快速破盾 / Ice weapon/enchant → fast shield break in P1 |
| **逃课打法 / Cheese** | P1绕柱子——堕落骑士的突进会被柱子挡住。P2没有逃课——练完美格挡 / P1: pillar humping blocks his rush. P2: no cheese — learn parry |

## 后期Boss / Late Game Bosses

### 毁灭之龙 / Dragon of Ruin

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-60%)** | 爪击(左→右→双爪拍地)→火焰吐息(扇形)→尾巴横扫 / Claw (L→R→both) → flame breath (cone) → tail sweep | 打前腿→失衡→倒地输出。火焰吐息往侧面闪 / Hit front legs → stagger → DPS. Side-dodge flame breath |
| **P2 (60%-20%)** | 起飞→空中火焰弹幕→俯冲爪击 / Flies → airborne fire barrage → dive claw | 起飞时锁定头部→闪避火焰弹幕→它俯冲时完美格挡→打头 / Lock head during flight → dodge fire → parry dive → hit head |
| **P3 (20%-0%)** | 狂暴——全屏火焰AOE+连击x5 / Enrage — full-screen fire AoE + 5-hit combo | 火焰AOE时跑到最边缘。连击后完美格挡最后一下→全力输出 / Run to edge during AoE. Parry final hit of combo → max DPS |
| **属性弱点 / Weakness** | 冰霜(伤害x1.5) / Ice (1.5x) | 冰双剑/冰矛附魔 / Ice Dual Blades/Spear enchant |
| **逃课打法 / Cheese** | 长矛站龙头侧面——龙的爪击和吐息都打不到侧面——安全输出 / Spear at dragon side flank — claw and breath can't hit flank — safe DPS |

### 疯狂伊顿 / Insane Eton

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-70%)** | 快慢刀+瞬移突击——节奏多变，没有固定间隔 / Delayed attacks + teleport rush — irregular rhythm |
| **P2 (70%-30%)** | 分身+全屏AOE组合——地上出现圆形红光区域→几秒后爆炸 / Clones + full-screen AoE — red circle zones on ground → delayed explosion | 红光区域之间的缝隙站人——爆炸后立刻跑位 / Stand in gaps between red zones — reposition immediately after explosion |
| **P3 (30%-0%)** | 全屏秒杀技(蓄力5秒)+无限分身+快速连击 / Full-screen one-shot (5s charge) + infinite clones + fast combos | 秒杀技：跑远+用完美格挡挡(可挡但极难)。分身出来后全力AOE清掉 / One-shot: run far OR parry (extremely hard). Clear clones with AoE immediately |
| **属性弱点 / Weakness** | 闪电(伤害x1.2)——所有属性中最高 / Lightning (1.2x) — highest among all elements |
| **逃课打法 / Cheese** | **没有逃课**——伊顿是游戏的技能考试官。练好完美格挡+熟悉快慢刀才能过 / **No cheese** — Eton is the final skill check. Master parry + delayed attack timing or die |
