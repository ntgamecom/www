# 武器系统 / Weapons Guide

## 大剑 / Greatsword

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **攻击风格 / Playstyle** | 蓄力重击+大范围横扫——**一击脱离**是核心节奏 / Charged heavy + wide swings — **hit and retreat** is the core rhythm |
| **核心连招 / Core Combo** | 轻击x2→重击(横扫)→后退恢复精力→循环 / Light x2 → Heavy (sweep) → back off → repeat |
| **蓄力攻击 / Charge ATK** | 按住重击蓄力——满蓄后伤害x2.5+大范围。但蓄力期间不能取消 / Hold heavy for charge — fully charged does 2.5x dmg + massive AoE. Cannot cancel during charge |
| **专属技能 / Unique Skills** | 裂地斩(Ground Splitter)——跳劈+地震AOE。旋风斩(Whirlwind)——360度持续旋转攻击 / Ground Splitter — jump slam + quake AoE. Whirlwind — 360 sustained spin |
| **最佳场景 / Best Use** | Boss大硬直后爆发输出、清群怪。不适合快节奏Boss / Burst damage after boss stagger, crowd clear. Bad for fast-paced bosses |

**大剑Build核心**: 堆**力量**(STR)+**物理攻击力**+**暴击伤害**。精力管理是大剑玩家的终生课题 / **Greatsword build**: Stack **STR** + **physical ATK** + **crit damage**. Stamina management is a lifelong lesson.

## 双剑 / Dual Blades

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **攻击风格 / Playstyle** | 快速连击+元素附魔——通过叠加连击数触发额外特效 / Fast combos + elemental enchants — build combo count for bonus effects |
| **核心连招 / Core Combo** | 轻击x4(连击积累)→重击(终结技)→撤步闪避→循环 / Light x4 (combo building) → Heavy (finisher) → backstep dodge → repeat |
| **属性附魔 / Elemental Enchant** | 双剑可以附魔火焰/冰霜/闪电——附魔后攻击附带属性伤害+特效 / Can enchant with fire/ice/lightning — added elemental dmg + effects on hit |
| **专属技能 / Unique Skills** | 乱舞(Dance of Blades)——20连击——伤害逐步递增。影步(Shadow Step)——向前瞬移+背后攻击 / Dance of Blades — 20-hit combo, scaling dmg. Shadow Step — teleport forward + backstab |
| **最佳场景 / Best Use** | 持续输出型战斗、属性克制场景、**新手开荒首选** / Sustained DPS, elemental countering, **best for beginners** |

**双剑Build核心**: 堆**敏捷**(DEX)+**属性伤害**+**连击相关词条**。双剑的精髓是"永远在攻击"——利用攻速压制敌人 / **Dual Blades build**: Stack **DEX** + **elemental dmg** + **combo-related affixes**. The essence of Dual Blades is "always attacking" — overwhelm enemies with speed.

## 长矛 / Spear

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **攻击风格 / Playstyle** | 中距离刺击+范围横扫——安全的输出距离+优秀的控制能力 / Mid-range thrusts + wide sweeps — safe distance + excellent crowd control |
| **核心连招 / Core Combo** | 重击(突刺→推开距离)→轻击x2(回刺)→后跳→循环 / Heavy (thrust → push back) → Light x2 (return thrust) → back jump → repeat |
| **距离控制 / Range Control** | 长矛攻击距离是最远的——可以在Boss攻击范围外安全输出 / Longest attack range — safe DPS outside boss attack range |
| **专属技能 / Unique Skills** | 旋风刺(Spiral Thrust)——旋转突进+多段伤害。落月斩(Moonsplitter)——跳起后向下刺击——破盾专用 / Spiral Thrust — spinning rush + multi-hit. Moonsplitter — plunge attack — shield breaker |
| **最佳场景 / Best Use** | 对战大型Boss(龙/巨人)、群怪控制、喜欢安全距离的玩家 / Large boss fights (dragons/giants), crowd control, players who prefer safe distance |

**长矛Build核心**: 堆**力量+敏捷均衡**+**攻击范围**+**削韧**(Stagger)。长矛的控制能力在竞技场和Boss战中极强 / **Spear build**: Balanced **STR + DEX** + **attack range** + **stagger**. Spear's control is excellent in arena and boss fights.

## 武器进阶技巧 / Advanced Weapon Techniques

| 技巧 / Technique | 操作 / Input | 说明 / Description |
|:----:|:----:|------|
| **取消后摇 / Cancel Recovery** | 攻击后→完美格挡→取消攻击硬直 | 大剑专用——蓄力攻击后立刻格挡取消后摇 / Greatsword tech — cancel charge attack recovery with block |
| **双剑附魔切换 / Blades Enchant Swap** | 战斗中打开道具栏→切换附魔→关闭 | 利用Boss属性弱点切换附魔——火焰→冰霜→闪电循环 / Exploit boss elemental weakness mid-fight — fire → ice → lightning cycle |
| **长矛卡距离 / Spear Distance Abuse** | 保持长矛最大攻击范围→Boss攻击打不到你但你能打中Boss | 活用长矛范围优势无伤磨血 / Use spear's range advantage to chip damage safely |
| **武器切换连招 / Weapon Swap Combo** | 大剑蓄力→切换双剑→快速追击→切换长矛→横扫收尾 | 高级技巧——需要高负重和高技能点数支持 / Advanced tech — requires high equip load and skill points |
| **空中攻击 / Aerial Attack** | 跳跃+攻击→滞空连击 | 双剑和长矛有空中连招加成。用于躲地面AOE同时输出 / Dual Blades and Spear have aerial combo bonuses. Dodge ground AoE while DPS-ing |
