## 符文系统 / Rune System

符文提供被动增益——镶嵌在装备槽位中 / Runes provide passive bonuses — socketed into gear slots.

### 符文类型 / Rune Types

| 类型 / Type | 效果 / Effect |
|---|---|
| **攻击符文 / Attack Rune** | 增加攻击力+暴击率 / ATK + crit rate |
| **防御符文 / Defense Rune** | 增加防御力+减伤 / DEF + damage reduction |
| **耐力符文 / Stamina Rune** | 增加耐力上限+恢复速度 / Max stamina + recovery rate |
| **狂战士符文 / Berserker Rune** | 强化狂战士模式——延长持续时间+增强效果 / Enhance berserker mode |
| **属性符文 / Element Rune** | 附加火/冰/雷等属性伤害 / Add fire/ice/lightning damage |
| **特殊符文 / Special Rune** | 独特效果——如吸血、反弹 / Unique effects — life steal, thorns |

### 推荐符文组合 / Recommended Rune Sets

| 玩法流派 / Build | 符文组合 / Rune Setup |
|---|---|
| **狂战士爆发流 / Berserker Burst** | 狂战士符文×3 + 攻击符文×2 / Berserker x3 + ATK x2 |
| **坦克生存流 / Tank Survival** | 防御符文×3 + 耐力符文×2 / DEF x3 + Stamina x2 |
| **属性异常流 / Element Status** | 属性符文×3 + 攻击符文×2 / Element x3 + ATK x2 |
| **均衡流 / Balanced** | 每种一个 / One of each |

## 技能常见问题 / Skills FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **先学什么技能？/ First skills?** | 狂战士冲锋 + 怒火连斩——前期核心 / Berserker Rush + Rage Combo — early core |
| **符文怎么镶嵌？/ Socket runes?** | 在铁匠处——选择装备再选择符文 / At blacksmith — select gear then rune |
| **技能可以重置吗？/ Respec skills?** | 可以——使用遗忘卷轴重置 / Yes — use Forget Scroll |
| **被动优先加哪个？/ Priority passive?** | 狂怒之心（更快的愤怒值）/ Rage Heart (faster rage) |
| **符文最高等级？/ Max rune level?** | +10——高周目掉落更高级符文 / +10 — higher NG cycles drop better runes |
| **遗忘卷轴在哪？/ Forget Scroll location?** | 第一章支线任务奖励 + 第二章商人出售 / CH1 side quest reward + CH2 merchant |
| **技能点不够怎么办？/ Not enough SP?** | 清完所有支线 + 探索全图找技能碎片——三周目后可以点满所有 / Complete all side quests + explore for fragments — can max by NG+3 |
| **狂战士符文和攻击符文哪个好？/ Berserker vs ATK rune?** | 狂战士符文适合爆发打法——攻击符文适合持续输出。推荐混搭 / Berserker for burst playstyle — ATK for sustained DPS. Mix recommended |
| **被动技能上限？/ Max passives?** | 所有被动都可以点满——但需要大量技能点。优先核心被动 / All passives can be maxed — but need many SP. Prioritize core ones |
