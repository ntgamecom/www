# 第一狂战士：卡赞攻略 / The First Berserker: Khazan Guide

《第一狂战士：卡赞》是Neople和Nexon开发的硬核动作角色扮演游戏，改编自《地下城与勇士》世界观。你扮演传说中的狂战士卡赞，从死亡中归来，踏上复仇之路。本攻略涵盖战斗系统、武器流派、Boss打法和探索收集 / The First Berserker: Khazan is a hardcore action RPG from Neople and Nexon, set in the Dungeon & Fighter universe. You play as the legendary berserker Khazan, returning from death on a path of revenge. This guide covers combat systems, weapon builds, boss strategies, and world exploration.

## 游戏特色 / Key Features

| 特色 / Feature | 说明 / Description |
|---|---|
| **硬核战斗 / Hardcore Combat** | 基于精力(Stamina)管理的战斗系统——攻击/闪避/格挡都消耗精力 / Stamina-based combat — attack, dodge, and block all consume stamina |
| **三大武器 / Three Weapons** | 大剑(高伤害/慢速)、双剑(快速/连击)、长矛(中距/刺击)——各自独立技能树 / Greatsword (high dmg/slow), Dual Blades (fast/combo), Spear (mid-range/thrust) — each with independent skill trees |
| **技能系统 / Skill System** | 技能点数解锁连招、反击技和被动能力——不同武器不同技能树 / Skill points unlock combos, counterattacks, and passives — different skill trees per weapon |
| **卡赞之力 / Khazan's Power** | 怒意值系统——积累怒意后进入"狂战士状态"——伤害提升+新招式 / Rage meter — enter "Berserker State" at full rage — increased damage + new moves |
| **装备锻造 / Gear Crafting** | 收集素材→锻造/强化装备——装备品质影响词条和孔位 / Collect materials → craft/upgrade gear — gear quality affects stats and slots |

## 新手快速入门 / Quick Start

| 阶段 / Phase | 建议 / Tips |
|---|---|
| **开局 / Start** | 推荐用**双剑**入坑——攻速快、容错率高。大剑对新手的精力管理要求极高 / Start with **Dual Blades** — fast attacks, high margin of error. Greatsword demands precise stamina management |
| **前期1-3章 / Early Game** | 优先加点**精力**(Stamina)——更多精力=更多攻击+闪避。不要只堆攻击力 / Prioritize **Stamina** — more stamina = more attacks + dodges. Don't stack only ATK |
| **中期4-6章 / Mid Game** | 开始注意装备属性和词条——火焰/冰霜等属性克制在后期极重要 / Start paying attention to gear stats and affixes — fire/ice elemental advantages matter in late game |
| **后期7章+ / Late Game** | 魂系核心铁律——**熟悉Boss动作比堆数值更重要** / Souls-like golden rule — **learning boss patterns > stacking numbers** |
