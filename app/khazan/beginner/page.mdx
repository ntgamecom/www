# 新手入门 / Beginner Guide

第一狂战士：卡赞是一款硬核动作RPG，核心战斗围绕精力管理、格挡时机和技能连招展开。本章帮助新手快速上手，避免常见的踩坑 / The First Berserker: Khazan is a hardcore action RPG built around stamina management, parry timing, and skill combos. This chapter helps beginners get started quickly and avoid common pitfalls.

---

## 核心操作 / Core Controls

| 操作 / Action | 效果 / Effect | 技巧 / Tip |
|:----:|------|------|
| **轻击 / Light Attack** | 快速攻击——伤害低、精力消耗少 / Fast attack — low dmg, low stamina cost | 用于压制小怪/补刀 / Use for chipping and finishing |
| **重击 / Heavy Attack** | 慢速高伤——消耗更多精力 / Slow, high dmg — costs more stamina | 用于Boss大硬直时爆发 / Use during boss big openings |
| **格挡 / Block** | 减伤但消耗精力——精力归零会"破防"(大硬直) / Reduces damage but costs stamina — zero stamina = guard break |
| **完美格挡 / Perfect Parry** | 在攻击命中的瞬间格挡——完全抵挡伤害+反击窗口 / Block at the moment of impact — negates all dmg + counter window | **核心技巧**——所有Boss战都要练的必修课 / **Core mechanic** — mandatory for all boss fights |
| **闪避 / Dodge** | 翻滚闪避——有无敌帧 / Roll dodge — has i-frames | 重攻击/范围攻击用闪避而不是格挡 / Use dodge for heavy attacks and AoE, not block |
| **反击 / Counter** | 完美格挡后按攻击→强力反击 / Press attack after perfect parry → powerful counter | 伤害高+削韧——尽量完美格挡后接反击 / High dmg + stagger — always follow up parry with counter |

## 精力管理 / Stamina Management

**精力是第一资源**——没有精力就不能攻击/闪避/格挡。精力归零=必死 / **Stamina is the most important resource** — zero stamina = death sentence.

| 原则 / Rule | 说明 / Description |
|:----:|------|
| **不要打空精力 / Never empty stamina** | 永远留20%精力用于紧急闪避/格挡。精力归零时被攻击→破防→被连死 / Always keep 20% stamina for emergency dodge/block. Guard break = combo death |
| **攻击节奏 / Attack Rhythm** | 2-3次轻击→停→恢复精力→再攻击。不要贪刀 / 2-3 light attacks → pause → stamina regen → attack again. Don't get greedy |
| **精力恢复速度 / Regen Speed** | 精力恢复速度受装备和技能影响。部分装备词条可以提升恢复速度 / Stamina regen speed is affected by gear and skills. Some gear affixes boost regen |
| **满精力不恢复 / Full stamina = wasted regen** | 精力接近满时立刻进攻——不要让精力条闲着 / Attack when stamina is near full — don't let the bar sit idle |
| **狂战士状态 / Berserker State** | 怒意满时触发——精力消耗降低+攻击力上升 / Full rage = reduced stamina cost + increased ATK |

## 技能系统 / Skill System

| 概念 / Concept | 说明 / Description |
|:----:|------|
| **技能点数 / Skill Points** | 升级获得——每级1点。用于解锁武器技能树中的技能 / Earn 1 per level. Use to unlock skills in weapon skill trees |
| **武器技能树 / Weapon Skill Trees** | 大剑、双剑、长矛各有一个独立技能树——每个约15-20个技能 / Each weapon has an independent skill tree — ~15-20 skills each |
| **主动技能 / Active Skills** | 武器连招、特殊攻击、反击技——需要装备到技能栏使用 / Combos, special attacks, counters — must equip to skill slots |
| **被动技能 / Passive Skills** | 常驻效果——+精力/+攻击/+防御/+属性抗性等 / Permanent effects — +stamina/+ATK/+DEF/+elemental resist |
| **技能重置 / Reset** | 可在营地使用金币重置技能点——完整返还 / Full reset available at camp for gold — complete refund |
| **通用技能 / Universal Skills** | 所有武器通用的被动——如精力管理、反击强化、生存技能 / Passives shared across all weapons — stamina management, counter boost, survival |

**加点建议**: 前期优先点出2-3个核心连招技能(每个武器不同)+被动精力技能+反击强化。不要均匀分布——先专精一把武器 / **Early priority**: Unlock 2-3 core combo skills + stamina passives + counter boost. Don't spread thin — master one weapon first.

## 新手推荐武器 / Beginner-Recommended Weapon

| 武器 / Weapon | 难度 / Difficulty | 推荐理由 / Reason |
|:----:|:----:|------|
| **双剑 / Dual Blades** | ★★☆☆☆ | 攻速快→容错率高→精力消耗低→适合学习Boss动作 / Fast attacks = high margin of error = low stamina cost = best for learning boss patterns |
| **大剑 / Greatsword** | ★★★★☆ | 高伤害但精力消耗大+攻击后摇长——不适合新手开荒 / High dmg but huge stamina cost + long recovery — not for beginners |
| **长矛 / Spear** | ★★★☆☆ | 中距离安全输出——但伤害略低、连招复杂 / Safe mid-range combat — but lower dmg, complex combos |

**结论**: 一周目**双剑**开荒→二周目转大剑或长矛 / **TL;DR**: First playthrough with **Dual Blades** → second playthrough switch to Greatsword or Spear.

## 新手常见错误 / Common Beginner Mistakes

| 错误 / Mistake | 后果 / Consequence | 正确做法 / Correct |
|:----:|:----:|------|
| **贪刀 / Greedy attacks** | 精力打空→不能闪避Boss大招→被秒 / Empty stamina → can't dodge boss attack → oneshot | 打2-3下→停→观察Boss动作→再打 / Hit 2-3 times → pause → observe → hit again |
| **只堆攻击 / Pure ATK stacking** | 脆皮→被Boss两下打死。精力不够→无法持续输出 / Glass cannon → 2-shot by boss. No stamina → can't sustain DPS | 精力和血量至少加到30%总属性点再考虑攻击 / Invest 30% of stat points in stamina and HP before ATK |
| **忽略完美格挡 / Ignoring parry** | Boss战难度x2——完美格挡是最大的输出窗口 / Boss fights become 2x harder — parry is your biggest DPS window | 每个Boss花10分钟专门练完美格挡 / Spend 10 minutes per boss practicing perfect parry |
| **装备不升级 / Not upgrading gear** | 伤害刮痧+被秒 / Low dmg + oneshot | 每到一个新区域→先把武器强化到当前上限 / Every new zone → upgrade weapon to current cap |
| **不看Boss动作只按攻略 / Rote following guide** | 遇到同一Boss不同变体时不知所措 / Can't adapt to boss variants | 看攻略了解机制→实战自己观察节奏。每个Boss的"快慢刀"节奏不同 / Learn mechanics from guide → observe rhythm in practice |

## 死亡后怎么办 / After Death

| 步骤 / Step | 操作 / Action |
|:----:|------|
| **1** | 丢失所有未使用的经验值(魂魄)——回到最近存档点 / Lose all unspent XP (souls) — return to last checkpoint |
| **2** | 跑尸回到死亡地点——触碰魂魄即可回收 / Run back to death location — touch soul to recover |
| **3** | 如果在回收前再次死亡——魂魄永久丢失。**先跑尸再打Boss** / If you die before recovery — soul is permanently lost. **Recover before fighting the boss** |
| **4** | 死亡时消耗一个"复活石"可在原地满血复活一次(仅在Boss战中) — 魂系传统 / Use "Revive Stone" to respawn on the spot once (boss fights only) — souls tradition |
