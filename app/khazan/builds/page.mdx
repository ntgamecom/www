# 配装Build / Builds Guide

配装Build是第一狂战士：卡赞中后期核心玩法。属性加点、装备选择、技能搭配和附魔共同决定你的战斗风格 / Builds are the core mid-to-late game mechanic in The First Berserker: Khazan. Stat allocation, gear selection, skill combinations, and enchantments define your playstyle.

---

## 属性加点 / Stat Allocation

| 属性 / Stat | 效果 / Effect | 优先 / Priority |
|:----:|------|:----:|
| **力量 / STR** | +物理攻击力+大剑伤害 / +Physical ATK + Greatsword dmg | 大剑Build ★★★★★ |
| **敏捷 / DEX** | +攻速+双剑伤害+暴击率 / +Attack speed + Dual Blades dmg + Crit rate | 双剑Build ★★★★★ |
| **体力 / VIT** | +最大HP+物理防御 / +Max HP + Physical DEF | 全Build ★★★★☆ |
| **精力 / END** | +最大精力+精力恢复速度 / +Max Stamina + Stamina regen speed | 全Build ★★★★★ |
| **智力 / INT** | +属性伤害+属性抗性 / +Elemental dmg + Elemental resist | 属性Build ★★★☆☆ |

**加点通用原则**: 前期优先精力到20→体力到15→再补主属性到40→最后补副属性 / **General rule**: END to 20 → VIT to 15 → main stat to 40 → secondary stat.

## 开荒Build推荐 / Early Game Builds

### 双剑·火焰流 / Dual Blades Fire Build

| 部位 / Slot | 装备 / Gear | 属性重点 / Stat Focus |
|:----:|:----|:----|
| **武器 / Weapon** | 焰形双剑(Flame Dual Blades) | 火焰伤害+攻速 / Fire dmg + ATK speed |
| **防具 / Armor** | 侦查者套装(Scout Set) | 敏捷+闪避距离 / DEX + dodge distance |
| **饰品 / Accessories** | 火焰之环(Fire Ring) + 精力护符(Stamina Amulet) | +火焰伤害+最大精力 / +Fire dmg + max stamina |
| **技能重点 / Skills** | 乱舞→火焰附魔→影步→精力恢复被动 | Combo→Fire Enchant→Shadow Step→Stamina passive |
| **属性加点 / Stats** | 敏捷25>精力20>体力15>敏捷继续加 | DEX 25 > END 20 > VIT 15 > DEX |

**打法**: 附魔火焰→乱舞连击→触发燃烧DoT→撤步闪避→重复。燃烧伤害在前期非常可观 / **Playstyle**: Enchant fire → Dance of Blades → burn DoT → backstep → repeat. Burn damage is substantial in early game.

### 长矛·防御流 / Spear Defense Build

| 部位 / Slot | 装备 / Gear | 属性重点 / Stat Focus |
|:----:|:----|:----|
| **武器 / Weapon** | 铁矛(Iron Spear) | 基础攻击+削韧 / Base ATK + stagger |
| **防具 / Armor** | 卫兵套装(Guard Set) | 体力+防御+格挡效率 / VIT + DEF + block efficiency |
| **饰品 / Accessories** | 回复戒指(Regen Ring) + 铁壁护符(Bulwark Amulet) | +HP恢复+物理防御 / +HP regen + physical DEF |
| **技能重点 / Skills** | 旋风刺→穿刺→后跳→格挡强化被动 / Spiral Thrust → Impale → Back Jump → Block passive |
| **属性加点 / Stats** | 体力25>精力20>力量20>体力继续 | VIT 25 > END 20 > STR 20 > VIT |

**打法**: 保持距离→刺→退→刺→退。Boss攻击时格挡→完美格挡后反击。稳扎稳打 / **Playstyle**: Keep distance → poke → retreat → repeat. Block boss attacks → parry counter. Slow and steady.

## 毕业Build推荐 / Endgame Builds

### 大剑·狂暴流 / Greatsword Berserker Build

| 部位 / Slot | 装备 / Gear | 属性重点 / Stat Focus |
|:----:|:----|:----|
| **武器 / Weapon** | 狂乱大剑(Madness Greatsword) [伊顿掉落] | 暴击伤害+力量 / Crit dmg + STR |
| **防具 / Armor** | 终焉套装(Finality Set) [伊顿掉落] | 暴击率+暴击伤害+狂战士状态延长 / Crit rate + Crit dmg + Berserker extension |
| **饰品 / Accessories** | 狂战士之环(Berserker Ring) + 力量护符(Strength Talisman) | 怒意积累加速+力量 / Rage build-up + STR |
| **技能重点 / Skills** | 裂地斩→旋风斩→蓄力强化→狂战士状态强化 / Ground Splitter → Whirlwind → Charge boost → Berserker boost |
| **属性加点 / Stats** | 力量50>精力30>体力25 / STR 50 > END 30 > VIT 25 |

**打法**: 积攒怒意→触发狂战士状态→裂地斩起手→旋风斩跟进→最高爆发输出。狂战士状态下大剑攻速+30%、伤害+50% / **Playstyle**: Build rage → trigger Berserker → Ground Splitter → Whirlwind → maximum burst. Berserker: +30% ATK speed, +50% dmg.

### 双剑·属性大师流 / Dual Blades Elemental Master Build

| 部位 / Slot | 装备 / Gear | 属性重点 / Stat Focus |
|:----:|:----|:----|
| **武器 / Weapon** | 元素双剑(Elemental Dual Blades) | 全属性伤害+切换速度 / All elemental dmg + swap speed |
| **防具 / Armor** | 龙鳞套装(Dragon Scale Set) [毁灭之龙掉落] | 全属性抗性+属性伤害 / All elemental resist + elemental dmg |
| **饰品 / Accessories** | 元素宝珠(Elemental Orb) + 连击护符(Combo Talisman) | 连击数增加属性伤害 / Combo count boosts elemental dmg |
| **技能重点 / Skills** | 乱舞→所有属性附魔→影步→闪避强化被动 / Dance of Blades → all elemental enchants → Shadow Step → Evade passive |
| **属性加点 / Stats** | 敏捷50>智力30>精力20 / DEX 50 > INT 30 > END 20 |

**打法**: 根据Boss属性弱点切换附魔→乱舞叠加连击→高属性伤害。一套附魔切换打天下。适合熟悉属性克制的进阶玩家 / **Playstyle**: Switch enchant based on boss weakness → Dance of Blades builds combo → massive elemental dmg. For advanced players who know elemental counters.

## 装备强化指南 / Gear Upgrade Guide

| 等级 / Level | 素材 / Materials | 效果 / Effect | 优先 / Priority |
|:----:|:----|:----:|:----:|
| +1 ~ +3 | 铁矿石+金币 / Iron Ore + Gold | 基础攻击/防御+15% | 每进新区域必强化 / Mandatory per zone |
| +4 ~ +6 | 精炼石+金币 / Refined Stone + Gold | 基础攻击/防御+30%+解锁第1个词条 / Unlock 1st affix | 一周目目标 / Aim for NG |
| +7 ~ +9 | 龙鳞+结晶+金币 / Dragon Scale + Crystal + Gold | 基础+50%+解锁第2个词条 / Unlock 2nd affix | 二周目目标 / Aim for NG+ |
| +10 | 伊顿核心+龙心+大量金币 / Eton Core + Dragon Heart + Gold | 基础+70%+终极词条 / Ultimate affix | 毕业目标 / Endgame goal |

**强化优先级**: 武器>衣服>裤子>手套>鞋子>饰品 / **Priority**: Weapon > Chest > Pants > Gloves > Boots > Accessories.
