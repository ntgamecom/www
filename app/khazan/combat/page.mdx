# 战斗系统 / Combat System

卡赞的战斗系统融合硬核魂系机制与狂战士特色——以耐力管理和愤怒值为核心 / Khazan's combat system blends hardcore souls mechanics with berserker features — centered on stamina management and rage.

---

## 基础战斗 / Basic Combat

### 攻击类型 / Attack Types

| 攻击 / Attack | 操作 / Input | 说明 / Description |
|---|---|---|
| **轻攻击 / Light Attack** | R1 / 鼠标左键 | 快速攻击——积攒愤怒值 / Fast attacks — builds rage |
| **重攻击 / Heavy Attack** | R2 / Shift+左键 | 高伤害+削韧——消耗更多耐力 / High damage + stagger — costs more stamina |
| **跳跃攻击 / Jump Attack** | 跳跃+R2 / Jump+R2 | 可打出敌人硬直 / Can stagger enemies |
| **冲刺攻击 / Running Attack** | 奔跑+攻击 / Sprint+attack | 突进起手技 / Gap closer |
| **处决 / Execution** | 敌人硬直时R1 / R1 when staggered | 高伤害终结技 / High damage finisher |

## 耐力系统 / Stamina System

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **耐力槽 / Stamina Bar** | 绿色槽——攻击+格挡+翻滚消耗 / Green bar — consumed by attacks, blocking, rolls |
| **耐力恢复 / Stamina Regen** | 不消耗时快速恢复 / Fast recovery when idle |
| **耐力见底 / Exhausted** | 耐力归零=大硬直——非常危险 / Zero stamina = big stagger — very dangerous |
| **愤怒值 / Rage Gauge** | 攻击命中+被攻击累积——满了可进入狂战士模式 / Builds on hit/taken — full = berserker mode |

---

## 狂战士模式 / Berserker Mode

狂战士模式是卡赞的核心特色——激活后大幅提升攻防 / Berserker Mode is Khazan's core feature — massively boosts offense and defense.

| 效果 / Effect | 说明 / Description |
|---|---|
| **攻击力提升 / Attack Up** | 所有伤害+30% / All damage +30% |
| **攻速提升 / Speed Up** | 攻击速度+25% / Attack speed +25% |
| **减伤 / Damage Reduction** | 受到伤害-20% / Incoming damage -20% |
| **霸体 / Hyper Armor** | 攻击中不会被轻易打断 / Attacks are hard to interrupt |
| **特殊攻击 / Special Attack** | 解锁狂战士专属技能 / Unlocks berserker-exclusive skills |

### 狂战士激活 / Activation

| 条件 / Condition | 说明 / Description |
|---|---|
| **愤怒值满 / Full Rage** | 持续攻击或受伤积累 / Continuous attack or taking damage |
| **主动激活 / Manual Activate** | L1+R1同时按 / Press L1+R1 together |
| **持续时间 / Duration** | 约15秒——攻击可延长 / ~15 seconds — extended by attacking |
| **冷却 / Cooldown** | 结束后愤怒值归零——重新累积 / Empties on end — need to rebuild |

## 武器系统 / Weapon System

### 武器类型 / Weapon Types

| 武器 / Weapon | 特点 / Features |
|---|---|
| **大剑 / Greatsword** | 高伤害+长范围——但慢 / High damage + range — but slow |
| **双刀 / Dual Blades** | 快速连击——愤怒值累积快 / Fast combos — fast rage build |
| **长矛 / Spear** | 中距离+突刺——安全 / Mid range + thrusts — safe |
| **战斧 / Battle Axe** | 高削韧+破盾 / High stagger + shield break |
| **巨锤 / War Hammer** | 极致削韧——慢但致命 / Max stagger — slow but deadly |

### 武器切换 / Weapon Swap

| 操作 / Input | 说明 / Description |
|---|---|
| **双武器槽 / Dual Weapon Slots** | 可装备两把武器——战斗中随时切换 / Equip two weapons — swap mid-combat |
| **武器切换 / Weapon Swap** | 按方向键左右切换 / D-pad left/right to swap |
| **切换攻击 / Swap Attack** | 切换瞬间按攻击——特殊连击 / Press attack on swap — special combo |
| **战术意义 / Tactical Value** | 大剑+双刀组合——大剑削韧+双刀追击 / Greatsword stagger + dual blade chase |

## 防御与闪避 / Defense & Evasion

### 翻滚 / Rolling

| 负重 / Load | 翻滚质量 / Roll Quality |
|---|---|
| **轻负重 / Light Load** | 快+远 / Fast + far |
| **中负重 / Medium Load** | 标准 / Standard |
| **重负重 / Heavy Load** | 慢+近——危险 / Slow + short — dangerous |

### 格挡 / Blocking

| 盾牌 / Shield | 效果 / Effect |
|---|---|
| **小盾 / Small Shield** | 物防70-85%——可弹反 / Block 70-85% — can parry |
| **中盾 / Medium Shield** | 物防85-95%——最平衡 / 85-95% — most balanced |
| **大盾 / Greatshield** | 物防95-100%——最强防御 / 95-100% — best defense |

### 完美格挡 / Perfect Block

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **完美格挡时机 / Timing** | 攻击命中前瞬间按格挡 / Press block right before impact |
| **完美格挡效果 / Effect** | 不减耐力+弹开敌人——造成大硬直 / No stamina cost + pushes enemy back — big stagger |
| **完美格挡反击 / Counter** | 完美格挡后按攻击键——反击连击 / Press attack after perfect block — counter combo |

## 战斗常见问题 / Combat FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **愤怒值怎么快速累积？/ Fast rage build?** | 双刀轻攻击连击最快——被攻击也累积 / Dual blade light combo fastest — taking damage also builds |
| **狂战士模式什么时候开？/ When to berserker?** | Boss战开场——或敌人有大硬直时 / Boss fight start — or when enemy is staggered |
| **武器怎么搭配好？/ Best weapon combo?** | 大剑（削韧）+ 双刀（输出）/ Greatsword (stagger) + Dual blades (DPS) |
| **完美格挡难吗？/ Perfect block difficulty?** | 比只狼弹反宽松——练习后容易掌握 / More lenient than Sekiro parry — learnable with practice |
| **耐力不够用？/ Not enough stamina?** | 加耐力属性+不要连续攻击太多次——留耐力翻滚 / Level endurance + don't overcommit — save stamina for rolls |
| **负重多少合适？/ Ideal equip load?** | 保持在70%以下——超过70%翻滚极慢。推荐中负重30-70% / Keep under 70% — over 70% fat roll. Medium load 30-70% recommended |
| **处决时机？/ Execution timing?** | 敌人出现大硬直时——通常出现在削韧打满后。处决伤害高且有短暂无敌 / When enemy is fully staggered — usually after poise is broken. High damage + brief invincibility |
