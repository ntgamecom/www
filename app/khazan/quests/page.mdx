## 任务常见问题 / Quests FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **支线会不会错过？/ Missable quests?** | 部分支线有章节时限——进新章节前清完 / Some are chapter-limited — clear before advancing |
| **铁匠升级什么时候开？/ Forge upgrade?** | 第一章完成后解锁 / Unlocks after CH1 completion |
| **技能训练怎么用？/ Skill training?** | 找战士长——消耗技能点学习新技能 / Talk to War Chief — spend skill points |
| **隐藏Boss怎么打？/ Secret boss?** | 需要收集特定钥匙道具——注意探索 / Need specific key item — explore thoroughly |
| **推荐多周目顺序？/ NG+ order?** | 先清完所有支线和隐藏——再进NG+ / Clear all side/secret content first — then NG+ |
| **铁匠升级材料哪刷？/ Forge material farm?** | 第一章矿洞采集铁矿——第二章后精英怪掉落精钢。高周目掉落更高级材料 / Mine iron in CH1 — fine steel from elites after CH2. Higher NG = better materials |
| **据点商人卖什么好东西？/ Hub merchant highlights?** | 海尔加卖药水配方+基础符文。乌尔夫卖技能书。奥拉夫卖武器升级材料 / Helga sells potion recipes + basic runes. Ulf sells skill books. Olaf sells weapon upgrade mats |
| **隐藏支线怎么触发？/ Trigger hidden quests?** | 地图上注意发光的物品和可互动的墙壁——许多隐藏支线通过发现秘密区域触发 / Look for glowing items and interactable walls — many hidden quests triggered by secret area discovery |
