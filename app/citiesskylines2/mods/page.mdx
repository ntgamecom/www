## 模组系统概述 / Mod System Overview

城市天际线2支持丰富的模组和自定义内容，通过Paradox Mods平台和Steam Workshop提供便捷的安装管理。模组可以改变游戏机制、添加新功能、优化性能或提供全新的视觉体验。资产（Asset）则是玩家创建的自定义建筑、车辆、地图等游戏元素。模组和资产极大地扩展了游戏的可玩性和寿命。

Cities: Skylines 2 supports rich mods and custom content, with convenient installation management through the Paradox Mods platform and Steam Workshop. Mods can change game mechanics, add new features, optimize performance, or provide全新的 visual experiences. Assets are player-created custom buildings, vehicles, maps, and other game elements. Mods and assets greatly expand the game's playability and lifespan.

### 安装与管理 / Installation & Management

进入游戏主菜单选择模组管理界面即可浏览和安装模组。Paradox Mods是官方推荐的模组平台，整合在游戏客户端内。Steam Workshop是另一主要模组来源。安装模组前建议阅读模组说明，了解兼容性和前置需求。某些模组之间可能存在冲突，谨慎搭配使用。启用或禁用模组通常需要重启游戏才能生效。建议定期更新已安装的模组以保持兼容性。

Enter the game main menu and select the Mods management interface to browse and install mods. Paradox Mods is the officially recommended mod platform, integrated into the game client. Steam Workshop is another major mod source. Before installing mods, it is recommended to read the mod description to understand compatibility and prerequisites. Some mods may conflict with each other; use combinations cautiously. Enabling or disabling mods usually requires restarting the game to take effect. It is recommended to regularly update installed mods to maintain compatibility.

### 模组安全须知 / Mod Safety Notes

并非所有模组都是安全的。使用模组前应查阅用户评价和下载量数据。信誉良好的模组作者发布的内容通常更可靠。模组可能导致游戏崩溃、存档损坏或性能下降。建议在安装新模组前备份存档。移除模组后，游戏存档中可能包含该模组的数据残留，可能导致后续问题。核心玩法改变类模组对存档的影响最大，需要特别谨慎。

Not all mods are safe. Check user reviews and download statistics before using mods. Content from reputable mod authors is generally more reliable. Mods may cause game crashes, save file corruption, or performance degradation. It is recommended to back up saves before installing new mods. After removing a mod, save files may contain residual data from the mod, potentially causing后续 issues. Core gameplay-changing mods have the most significant impact on saves and require special caution.

## 必备模组推荐 / Essential Mod Recommendations

### 交通管理模组 / Traffic Management Mods

交通管理是城市天际线中最复杂的系统之一，相关模组可以帮助玩家更精细地控制交通流。交通管理模组通常提供以下功能：更详细的交通数据面板、车道连接工具（Lane Connector）允许玩家指定车辆行驶路线、路口控制工具可以精确设置信号灯配时、道路限速调整功能等。这些模组对于解决复杂交通问题和优化道路网络效率非常有价值。

Traffic management is one of the most complex systems in Cities: Skylines, and related mods help players more precisely control traffic flow. Traffic management mods typically provide the following features: more detailed traffic data panels, lane connector tools allowing players to specify vehicle routes, intersection control tools for precise traffic signal timing, and road speed limit adjustment functions. These mods are extremely valuable for solving complex traffic problems and optimizing road network efficiency.

### 资产编辑器 / Asset Editor

资产编辑器允许玩家创建和导入自定义建筑、公园、服务设施等。编辑器提供了丰富的建模工具和纹理选项。玩家可以创建全新建筑或修改现有建筑。创建的资产可以上传到Paradox Mods或Steam Workshop供其他玩家使用。资产编辑器的学习曲线较陡峭，但创作自由度极高。从简单的模型开始练习，逐步掌握复杂的编辑技巧。

The Asset Editor allows players to create and import custom buildings, parks, service facilities, and more. The editor provides rich modeling tools and texture options. Players can create全新的 buildings or modify existing ones. Created assets can be uploaded to Paradox Mods or Steam Workshop for other players to use. The Asset Editor has a steep learning curve but offers极高的 creative freedom. Start practicing with simple models and gradually master more complex editing techniques.

### 地图编辑器 / Map Editor

地图编辑器用于创建自定义游戏地图。玩家可以塑造地形、设置水源、布置自然资源、规划初始道路连接。一张好的自定义地图需要考虑游戏性和美学平衡。地图创建包括地形调整、纹理绘制、资源分布和气候设置等多个步骤。创建的地图可以保存并用于新的游戏存档。地图编辑是创造独特游戏体验的重要工具。

The Map Editor is used to create custom game maps. Players can shape terrain, set water sources, arrange natural resources, and plan initial road connections. A good custom map requires balancing gameplay and aesthetics. Map creation includes multiple steps: terrain adjustment, texture painting, resource distribution, and climate settings. Created maps can be saved and used for new game saves. Map editing is an important tool for creating unique game experiences.

### 代码模组 / Code Mods

代码模组直接修改游戏脚本代码，可以实现最深度的功能改变。这类模组可以重写游戏的核心机制、添加全新的UI界面、创建复杂的自动化管理工具。代码模组需要作者具备编程知识，通常使用C#语言编写。代码模组对游戏版本的兼容性要求较高，每次游戏更新后可能需要模组作者更新代码。代码模组是最强大但也最需要谨慎使用的模组类型。

Code mods directly modify game script code, achieving the deepest functional changes. These mods can rewrite core game mechanics, add全新的 UI interfaces, and create complex automation management tools. Code mods require authors to have programming knowledge, typically written in C#. Code mods have high compatibility requirements with game versions; each game update may require the mod author to update the code. Code mods are the most powerful but also the most cautious type of mods to use.

## 自定义资产 / Custom Assets

### 建筑资产 / Building Assets

自定义建筑资产是最常见的资产类型。玩家可以创建住宅、商业、工业、办公、服务和装饰各类建筑。高质量的建筑资产通常包含精细的模型、纹理和LOD（Level of Detail）层级。自定义建筑可以拥有独特的功能属性，如特定的容量、成本和效果范围。建筑资产需要遵循游戏的技术规范，包括正确的尺寸、功能标签和动画支持。

Custom building assets are the most common asset type. Players can create residential, commercial, industrial, office, service, and decorative buildings. High-quality building assets通常包含 detailed models, textures, and LOD levels. Custom buildings can have unique functional attributes such as specific capacity, cost, and effect range. Building assets must follow the game's technical specifications, including correct dimensions, functional tags, and animation support.

### 车辆资产 / Vehicle Assets

车辆资产包括汽车、公交车、火车、飞机和船舶等。自定义车辆可以替换游戏中的默认车辆或作为新的交通方式加入。车辆资产需要配置速度、容量、维护成本等参数。精美的车辆模型可以为城市增添更多真实感。车辆模组与交通管理模组配合使用效果更佳。车辆资产的创建需要考虑与游戏道路和交通系统的兼容性。

Vehicle assets include cars, buses, trains, aircraft, and ships. Custom vehicles can replace default in-game vehicles or be added as new transportation modes. Vehicle assets require configuring parameters such as speed, capacity, and maintenance costs. Detailed vehicle models add more realism to the city. Vehicle mods work best in配合 with traffic management mods. Vehicle asset creation must consider compatibility with the game's road and traffic systems.

### 地图资产 / Map Assets

自定义地图资产由地图编辑器创建。高质量地图需要精心设计的地形、水系和资源布局。地图的主题风格多样，从温带平原到沙漠绿洲再到山地雪景。地图作者需要考虑游戏性因素：初始高速公路连接位置是否合理，自然资源分布是否平衡，可建造土地是否充足等。好的地图设计可以给玩家带来全新的游戏体验。

Custom map assets are created with the Map Editor. High-quality maps require精心 designed terrain, water systems, and resource layouts. Map themes vary widely, from temperate plains to desert oases and mountain snowscapes. Map authors must consider gameplay factors: whether initial highway connection positions are reasonable, whether natural resource distribution is balanced, whether buildable land is sufficient, etc. Good map design provides players with全新的 game experiences.

## 性能优化 / Performance Optimization

### 模组性能影响 / Mod Performance Impact

模组和资产可能显著影响游戏性能。高多边形数量的模型、高分辨率纹理和复杂的脚本逻辑会增加CPU和GPU负担。建议只安装需要的模组，避免同时启用大量高资源消耗模组。监控游戏帧率和加载时间，发现异常下降时需要排查问题模组。部分模组提供了性能优化选项，如LOD调整和渲染距离控制。

Mods and assets may significantly affect game performance. High-polygon models, high-resolution textures, and complex script logic increase CPU and GPU load. It is recommended to only install needed mods and avoid enabling大量 high-resource-consumption mods simultaneously. Monitor game frame rate and loading times; investigate problem mods when abnormal drops occur. Some mods offer performance optimization options, such as LOD adjustment and render distance control.

### 游戏设置优化 / Game Settings Optimization

除了管理模组，调整游戏画质设置同样重要。降低阴影质量、反射效果和环境光遮蔽可以明显提升帧率。调整视距和LOD细节可以平衡画质和性能。在城市规模增大后，适当降低显示细节有助于保持流畅运行。使用专用显卡和充足的内存是流畅运行大量模组的基础。固态硬盘显著缩短模组的加载时间。

Besides managing mods, adjusting game graphics settings is equally important. Lowering shadow quality, reflection effects, and ambient occlusion significantly improves frame rate. Adjusting view distance and LOD details balances visual quality and performance. As city size increases, appropriately reducing display detail helps maintain smooth operation. Using a dedicated graphics card and sufficient RAM is the foundation for running many mods smoothly. Solid-state drives significantly reduce mod loading times.

### 存档管理 / Save File Management

使用模组后，存档管理需要更加注意。模组更新或移除可能导致存档无法加载。建议在安装重要模组前创建全新存档作为备份。标记使用特定模组的存档，方便日后识别。定期清理不需要的存档，减少存储空间占用。在游戏大版本更新后，旧模组可能不兼容，需要等待模组更新后再继续游戏。

After using mods, save management needs more attention. Mod updates or removal may cause save files to become unloadable. It is recommended to create a fresh save as a backup before installing important mods. Label saves using specific mods for future identification. Regularly clean unnecessary saves to reduce storage usage. After major game version updates, old mods may become incompatible; wait for mod updates before continuing the game.

## 社区资源 / Community Resources

城市天际线拥有活跃的模组社区。Paradox Mods平台、Steam创意工坊、Reddit社区、Discord服务器和专门的模组网站都是获取模组信息和交流经验的渠道。加入社区可以第一时间获得新模组推荐、使用技巧和问题解答。社区内的资深玩家经常发布高质量的教程和指南。参与社区建设可以支持模组作者创作更多优质内容。

Cities: Skylines has an active modding community. The Paradox Mods platform, Steam Workshop, Reddit communities, Discord servers, and dedicated mod websites are all channels for obtaining mod information and exchanging experiences. Joining the community provides timely access to new mod recommendations, usage tips, and problem solutions. Experienced players in the community frequently publish high-quality tutorials and guides. Participating in community building supports mod authors in creating more quality content.

## 模组兼容性与冲突处理 / Mod Compatibility & Conflict Resolution

模组冲突是使用模组时常见的问题。冲突可能表现为游戏崩溃、功能失效或视觉异常。解决冲突的第一步是识别冲突源，通过逐一禁用模组来定位问题模组。查看模组说明中的兼容性信息，避免安装已知冲突的模组组合。使用模组管理工具可以自动检测部分兼容性问题。阅读模组评论区中其他用户的反馈可以了解已知问题。保持模组更新到最新版本可以减少兼容性问题。

Mod conflicts are common issues when using mods. Conflicts may manifest as game crashes, non-functional features, or visual anomalies. The first step in resolving conflicts is identifying the source by disabling mods one by one to locate the problematic mod. Check compatibility information in mod descriptions to avoid installing known conflicting combinations. Using mod management tools can automatically detect some compatibility issues. Reading user feedback in mod comment sections reveals known problems. Keeping mods updated to the latest version reduces compatibility issues.

## 实用模组分类推荐 / Recommended Mod Categories

模组按功能可以分为多个类别，方便玩家按需选择。界面优化类改进游戏UI和信息展示，让数据更清晰易读。功能扩展类添加游戏原版缺乏的功能，如批量操作、自动管理工具。视觉美化类提升画面效果，包括纹理包、光照改进和LUT调色。真实化类提高游戏的真实性，包括更真实的交通行为和经济模型。作弊类提供无限金钱、解锁所有建筑等功能，适合沙盒创意建造。按类别选择模组可以更高效地找到需要的功能。

Mods can be categorized by function for players to choose as needed. UI improvement mods enhance game interface and information display for clearer data reading. Function expansion mods add features lacking in the original game, such as batch operations and automatic management tools. Visual enhancement mods improve graphics, including texture packs, lighting improvements, and LUT color grading. Realism mods increase game authenticity, including more realistic traffic behavior and economic models. Cheat mods provide unlimited money, unlock all buildings, etc., suitable for sandbox creative building. Selecting mods by category helps find needed functions more efficiently.

## 模组开发入门 / Getting Started with Mod Development

对模组开发感兴趣的玩家可以学习创建自己的模组。基础模组开发需要了解游戏的数据结构和脚本语言。官方提供了模组开发文档和API参考。社区论坛中有大量开发教程和示例代码供学习。从简单的模组开始，如修改建筑参数或添加新道路类型。逐步学习更复杂的开发技术，如UI编程和网络功能。开发模组不仅可以满足自己的需求，还可以为社区做出贡献。模组开发是一个持续学习的过程，随着游戏更新不断演进。

Players interested in mod development can learn to create their own mods. Basic mod development requires understanding the game's data structures and scripting language. The official documentation provides mod development API references. Community forums offer大量 development tutorials and example code. Start with simple mods, such as modifying building parameters or adding new road types. Gradually learn more complex development techniques like UI programming and networking. Developing mods not only meets personal needs but also contributes to the community. Mod development is a continuous learning process that evolves with game updates.

## 模组备份与恢复 / Mod Backup & Recovery

定期备份模组和配置文件可以避免因意外问题丢失数据。备份模组文件夹中的所有已安装文件。记录当前使用的模组列表和版本号。保存游戏存档前确认模组配置的稳定性。在游戏大版本更新前备份所有模组和相关配置。出现问题时可以从备份快速恢复工作状态。良好的备份习惯是模组管理的重要部分。建议每次新增重要模组后创建新的备份。

Regularly backing up mods and configuration files prevents data loss from unexpected problems. Back up all installed files in the mods folder. Record the current mod list and version numbers. Confirm mod configuration stability before saving game progress. Back up all mods and related configurations before major game version updates. Quickly restore working status from backups when problems occur. Good backup habits are an important part of mod management. It is recommended to create a new backup after each important mod addition.

## 跨版本兼容性 / Cross-Version Compatibility

游戏版本更新后模组可能出现兼容性问题。检查模组作者是否发布了兼容新版本的更新。关注模组页面上的兼容性标签。临时禁用不兼容的模组等待更新。在游戏设置中可以选择等待模组更新后再升级游戏版本。核心模组通常更新较快，辅助性模组可能更新较慢。创建模组兼容性清单可以帮助管理多版本环境。耐心等待是处理版本兼容问题的最佳策略。

After game version updates, mods may have compatibility issues. Check whether mod authors have released updates compatible with the new version. Pay attention to compatibility tags on mod pages. Temporarily disable incompatible mods while waiting for updates. In game settings, choose to wait for mod updates before upgrading the game version. Core mods usually update quickly, while辅助性 mods may update more slowly. Creating a mod compatibility checklist helps manage multi-version environments. Patience is the best strategy for handling version compatibility issues.
