# 战斗系统 / Combat System

辐射4的战斗系统融合了传统FPS射击与VATS辅助瞄准系统——同时加入动力甲和近战格斗 / Fallout 4 combat blends traditional FPS shooting with the VATS assisted aiming system — plus power armor and melee.

---

## VATS系统 / V.A.T.S.

### VATS基础 / VATS Basics

VATS (Vault-Tec辅助瞄准系统) 可以暂停时间——选择目标部位进行精准攻击 / VATS (Vault-Tec Assisted Targeting System) pauses time — select body parts for precision attacks.

| 部位 / Body Part | 效果 / Effect | 命中率惩罚 / Hit Chance |
|---|---|---|
| **头部 / Head** | 暴击伤害2倍+可能即死 / 2x crit + possible instant kill | -20% |
| **躯干 / Torso** | 标准伤害 / Standard damage | 0% |
| **左/右臂 / Arms** | 降低敌人攻击精度 / Reduce enemy attack accuracy | -15% |
| **左/右腿 / Legs** | 降低敌人移动速度 / Reduce enemy movement speed | -15% |
| **武器 / Weapon** | 可能打落敌人武器 / May disarm enemy | -25% |

**VATS关键参数 / Key VATS Stats**:
| 参数 / Stat | 效果 / Effect | 受什么影响 / Affected By |
|---|---|---|
| AP消耗 / AP Cost | 每次VATS攻击消耗的行动点 / Action points per VATS shot | 武器重量+Perk |
| 命中率 / Hit Chance | 每次攻击的概率 / Per-attack chance | 感知+距离+掩体 / Perception + range + cover |
| 暴击条 / Crit Meter | 满后下次攻击100%暴击 / Full = next attack auto-crit | 运气+暴击perk / Luck + crit perks |
| 暴击倍率 / Crit Multiplier | 暴击造成的额外伤害 / Bonus damage on crit | 武器改装+Perk |

### 武器改造全解 / Weapon Modding

每把武器有7个改造槽——每个改变性能 / 7 mod slots per weapon — each changes performance.

| 改造槽 / Slot | 选项 / Options | 效果 / Effect |
|---|---|---|
| **枪膛 / Receiver** | 舒适/强力/自动/校准/微等 / Standard/Powerful/Auto/Calibrated/Tuned | 决定伤害和射速 / Determines dmg + fire rate |
| **枪管 / Barrel** | 短/中/长/阻击 / Short/Medium/Long/Sniper | 影响射程和后座 / Range + recoil |
| **枪托 / Stock** | 标准/神射手/全自动 / Standard/Marksman/Full Auto | 后座力控制 / Recoil control |
| **瞄具 / Sights** | 标准/反射/准直/狙击镜 / Standard/Reflex/Glow Sight/Scope | 瞄准精度 / Aim precision |
| **弹匣 / Magazine** | 标准/加长/快拔/穿甲 / Standard/Extended/Quick/Armor-Piercing | 容量和换弹 / Capacity + reload |
| **枪口 / Muzzle** | 补偿制退器/消音器 / Compensator/Suppressor | 后座减少或隐蔽 / Recoil reduction or stealth |
| **涂装 / Paint** | 各种颜色迷彩 / Various camo colors | 纯外观(无属性) / Cosmetic only |

## 潜行系统 / Stealth System

### 潜行机制 / Stealth Mechanics

| 因素 / Factor | 对潜行的影响 / Effect on Stealth |
|---|---|
| 光照 / Light | 在阴影中难以被发现——光亮中容易 / Harder in shadows — easier in light |
| 护甲重量 / Armor Weight | 重甲潜行更难 / Heavy armor = harder stealth |
| 移动速度 / Move Speed | 行走比奔跑更安静 / Walk quieter than run |

| 距离 / Distance | 距敌人越远越不易被发现 / Further = safer |
