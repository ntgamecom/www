## 聚落建造 / Settlement Building

### 基础建造指南 / Basic Building Guide

| 元素 / Element | 说明 / Description |
|---|---|
| **床 / Beds** | 每个居民需要一张床——室内床=更好 / One bed per settler — indoor = better |
| **水 / Water** | 水泵产生纯净水+瓶装水(自动放入工坊)/ Pumps produce water + bottles (auto-stored) |
| **食物 / Food** | 每个居民需要1点食物——种植物1点=1食物 / 1 food per settler — 1 plant = 1 food |
| **防御 / Defense** | 防御值需大于食物+水之和——否则被攻击 / Defense > food + water — otherwise raided |
| **电力 / Power** | 发电机提供电力——连接电线到建筑 / Generators → connect wires to buildings |
| **幸福度 / Happiness** | 床、食物、水、防御、装饰——所有因素综合 / Beds, food, water, defense, decor — composite |
