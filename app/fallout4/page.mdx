# 辐射4攻略 / Fallout 4 Guide

《辐射4》是Bethesda Game Studios开发的开放世界后末日角色扮演游戏——核战爆发210年后，你从111号避难所醒来，在波士顿废土寻找儿子并重建文明。从联邦废土到发光海，从兄弟会到铁路，从动力甲到聚落建造——每一寸废土都有故事。本攻略涵盖全部SPECIAL系统与Perk、联邦全区域探索、四大阵营路线以及装备改造与聚落建造完整指南。

Fallout 4 is Bethesda's open-world post-apocalyptic RPG set in the Commonwealth — 210 years after the bombs fell, you emerge from Vault 111 to search for your son and rebuild civilization. From the Commonwealth to the Glowing Sea, from the Brotherhood to the Railroad, from Power Armor to settlement building — every inch of the wasteland tells a story. This guide covers the SPECIAL system and perks, all Commonwealth regions, the four faction routes, and complete weapon modding and settlement building.

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **出门前多加点智力 / Put Points in INT** | 智力直接决定你获得的经验值——智力越高升级越快 / INT directly affects XP gain — more INT = faster leveling |
| **2** | **魅力开局有用 / CHR is Useful Early** | 说服成功率高 + 买卖价格好——帮你前期快速积累资源 / Better persuasion + trade prices — fast early resource gain |
| **3** | **早点拿到动力甲 / Get Power Armor Early** | 康科德自由博物馆顶楼有一套T-45 — 打死亡爪前穿上 / T-45 on top of the Museum of Freedom in Concord |
| **4** | **垃圾就是资源 / Junk is Resource** | 废土上所有垃圾都可以拆解成建造材料——什么都捡 / All junk can be scrapped into building materials — pick up everything |
| **5** | **开局直奔钻石城 / Rush to Diamond City** | 主线引导你去钻石城——路上尽量探索 / Main quest leads to Diamond City — explore along the way |
| **6** | **穿动力甲前把核心装上 / Insert Core Before Entering PA** | 动力甲在架上时需要核心才能进入——T-45需要1个核心 / PA on the frame needs a core to enter — T-45 needs 1 core |
| **7** | **同伴很好用很强 / Companions Are Strong** | 每个同伴都有自己的Perk(好感度满解锁) + 无限负重 / Each has a unique perk (max affinity) + infinite carry capacity |
| **8** | **VATS是你的好朋友 / VATS is Your Friend** | 辐射的VATS系统可以暂停时间瞄准弱点——新手必用 / VATS freezes time to target weak points — essential for beginners |

| 6 | 钻石城 / Diamond City | 进入钻石城——找到尼克·瓦伦坦 / Enter Diamond City — find Nick Valentine |
| 7 | 联邦 / Commonwealth | 自由探索——选择你的阵营道路 / Free exploration — choose your faction path |

## 难易度说明 / Difficulty Guide

| 难度 / Difficulty | 伤害倍率(玩家/敌人) | 说明 / Description |
|---|---|---|
| 非常简单 / Very Easy | 2x / 0.5x | 剧情体验 / Story-focused |
| 简单 / Easy | 1.5x / 0.75x | 轻松探索 / Casual |
| 普通 / Normal | 1x / 1x | 标准体验 / Standard |
| 困难 / Hard | 0.75x / 1.5x | 需要策略 / Needs tactics |
| 非常困难 / Very Hard | 0.5x / 2x | 废土生存 / Wasteland survival |
| **生存模式 / Survival** | 0.5x / 2x + 无快速旅行+需要进食/饮水/睡眠 / No fast travel + need food/water/sleep | **极难——但最有沉浸感** / **Extreme but most immersive** |

**生存模式核心技巧 / Survival Mode Tips**:
1. 聚落是关键——每个聚落都是你的安全屋+补给点 / Settlements are key — each is a safehouse and supply point
2. 饮用水比食物更重要——每个聚落都要建水泵 / Water is more important than food — build water pumps everywhere
3. 睡袋和床——随时存盘(生存模式只能靠睡觉存盘) / Sleeping bags and beds — only way to save in Survival
4. 抗生素必备——废土感染和疾病很多 / Antibiotics are essential — many infections and diseases
5. 带一个同伴——多一个负重+战斗力 / Bring a companion — extra carry capacity and firepower

**动力甲核心提示 / PA Tips**:

3. 改造台：可以升级动力甲部件——需要科学Perk / PA workstation: upgrade parts — needs Science perk
