## 顶级Build推荐 / Top Builds

### 1. 潜行狙击手 / Stealth Sniper

**经典的辐射玩法**——一枪未命中就换个位置 / Classic Fallout — one shot, one kill, reposition.

| 项目 / Aspect | 推荐 / Choice |
|---|---|
| **S.P.E.C.I.A.L** | S3 P9 E3 C3 I6 A7 L6 |
| **核心Perk** | 步枪手(Rifleman)、潜行(Sneak)、睡魔(Mr Sandman)、忍者(Ninja) |
| **推荐武器** | 猎人步枪(消音+狙击镜)/ Hunting Rifle (Sniper), 高斯步枪 / Gauss Rifle |
| **推荐护甲** | 轻甲——暗影改造 / Light armor — Shadowed mod |
| **玩法要点** | 潜行→VATS爆头→如果被发现了就跑→再潜行 / Sneak→VATS headshot→if detected→run→re-sneak |

### 2. 动力甲重炮手 / Power Armor Heavy Gunner

**穿上动力甲拿着重武器碾压一切** / PA + heavy weapons = unstoppable.

| 项目 / Aspect | 推荐 / Choice |
|---|---|
| **S.P.E.C.I.A.L** | S9 P5 E5 C3 I7 A5 L3 |
| **核心Perk** | 重枪手(Heavy Gunner)、装甲商(Armorer)、科学(Science)、核物理学家(Nuclear Physicist) |
| **推荐武器** | 旋转机枪 / Minigun, 加特林激光 / Gatling Laser, 胖子核弹 / Fat Man |
| **推荐护甲** | T-60或X-01动力甲——全部升级到最高 / T-60 or X-01 PA — full upgrades |
| **玩法要点** | 核心充足→开火→继续开火→换核心→开火 / Lots of cores→fire→more fire→swap core→fire |

### 3. 暴击手枪流 / Critical Gunslinger

**高暴击率+手枪——废土牛仔** / High crit + pistols — wasteland cowboy.

| 项目 / Aspect | 推荐 / Choice |
|---|---|
| **S.P.E.C.I.A.L** | S3 P5 E3 C3 I3 A9 L10 |
| **核心Perk** | 双枪客(Gunslinger)、暴击银行(Critical Banker)、暴击强化(Better Criticals)、死神冲刺(Grim Reaper) |
| **推荐武器** | 10mm手枪(高级改造) / 10mm Pistol (fully modded), 西洋左轮 / Western Revolver, 克里木之牙 / Deliverer |
| **推荐护甲** | 轻甲——增加AP恢复 / Light armor — AP regen |
| **玩法要点** | 攒暴击→VATS三枪+暴击→刷新AP→继续 / Build crits→VATS 3 shots + crit→AP refresh→repeat |

### 4. 近战狂战士 / Melee Berserker

**冲到敌人脸上砍翻一切**——最暴力的Build / Run up and smash everything — most violent build.

| 项目 / Aspect | 推荐 / Choice |
|---|---|
| **S.P.E.C.I.A.L** | S10 P1 E7 C3 I3 A9 L4 |
| **核心Perk** | 大联盟(Big Leagues)、根深蒂固(Rooted)、闪电突袭(Blitz)、忍者(Ninja)、独行侠(Lone Wanderer) |
| **推荐武器** | 超级大锤 / Super Sledge, 中国剑 / Chinese Officer Sword, 开膛锯 / Ripper |
| **推荐护甲** | 重甲——全防弹改造 / Heavy armor — max ballistic weave |
| **玩法要点** | Blitz传送到敌人面前→一锤→下一只 / Blitz teleport→smash→next |
