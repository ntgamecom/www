# 技能系统 / Skills & SPECIAL

辐射4取消了前作的技能系统——改为Perk系统——每个Perk有多个等级——并与SPECIAL属性挂钩 / Fallout 4 replaces the old skill system with Perks — each perk has multiple ranks — tied to SPECIAL stats.

---

## SPECIAL属性总览 / SPECIAL Overview

| 属性 / Stat | 英文 / Name | 初始值 / Start | 影响 / Affects |
|---|---|---|---|
| **力量 / S** | Strength | 1-10 | 近战伤害+负重+动力甲能力 / Melee dmg + carry weight + PA |
| **感知 / P** | Perception | 1-10 | VATS命中率+开锁+陷阱发现 / VATS hit chance + lockpick + trap detection |
| **耐力 / E** | Endurance | 1-10 | 生命值+跑步消耗+抗性 / HP + sprint cost + resistances |
| **魅力 / C** | Charisma | 1-10 | 说服成功率+商店价格+聚落幸福度 / Speech + prices + settlement happiness |
| **智力 / I** | Intelligence | 1-10 | 经验获取+改造等级 / XP gain + mod tier |
| **敏捷 / A** | Agility | 1-10 | AP点数+潜行效果 / AP + stealth effectiveness |
| **运气 / L** | Luck | 1-10 | 暴击条填充+战利品质量 / Crit fill + loot quality |

---

### SPECIAL属性提升方式 / How to Increase SPECIAL

| 方式 / Method | 获取 / How to Get |
|---|---|
| **书籍: 特别之书 / You're SPECIAL! Book** | 111避难所出场时拾取 / Vault 111 exit |
| **娃娃 / Bobbleheads** | 每个属性娃娃+1——全地图收集 / +1 per bobblehead |
| **服装和装备 / Clothing & Gear** | 如"研读墨镜"智力+1 / "Road Goggles" +1 Int |
| **酒精/药物 / Alcohol/Drugs** | 临时提升——有副作用 / Temporary — with side effects |
| **Perk(属性点) / Perk Point** | 每级获得一个属性点 / 1 point per level |
| **属性上限 / Cap** | 基础10 + 娃娃1 + 服装临时 = 最多11(基础)+临时 / Base 10 + bobblehead 1 + gear |
