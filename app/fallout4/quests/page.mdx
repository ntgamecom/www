# 任务与阵营 / Quests & Factions

## 主线任务 / Main Quest

辐射4的主线共 **16个核心任务**——从111避难所到学院结局 / 16 core quests — from Vault 111 to the Institute ending.

| 阶段 / Phase | 任务 / Quest | 说明 / Description |
|---|---|---|
| 1 | 战争从未改变 / War Never Changes | 111避难所开场——设定角色 + 见尚恩 / Vault 111 intro — set stats + see Shaun |
| 2 | 走出避难所 / Out of Time | 逃出111——去庇护山丘 / Escape Vault 111 — head to Sanctuary |
| 3 | 乔尔的任务 / The First Step | 去康科德救义勇兵——得动力甲 / Rescue Minutemen in Concord — get PA |
| 4 | 前往钻石城 / Street of Dreams | 去钻石城寻找线索 / Head to Diamond City for leads |
| 5 | 找到侦察兵 / Unlikely Valentine | 在公园街站救尼克·瓦伦坦 / Rescue Nick Valentine from Park Street Station |
| 6 | 获得线索 / Getting a Clue | 调查凯勒——追踪追猎者 / Investigate Kellogg — track the Courser |
| 7 | 重访凯勒 / Reunions | 在福蒂斯大厦击杀凯勒——第一次见到学院合成人 / Kill Kellogg at Fort Hagen — first Synth encounter |
| 8 | 寻找学院 / Dangerous Minds | 用凯勒的大脑芯片追踪学院位置 / Use Kellogg's brain chip to find the Institute |
| 9 | 猎人/猎物 / Hunter/Hunted | 追踪追猎者到绿顶温室 / Track Courser to Greenetech Genetics |
| 10 | 加入阵营 / Institutionalized | 进入学院——见到"尚恩" / Enter the Institute — meet "Shaun" |
| 11 | 学院/阵营 / The Molecular Level | 建造信号拦截器——联系学院 / Build the molecular relay — contact Institute |
| 12 | 阵营选择 / Form Ranks | **关键分支**——选择支持学院/兄弟会/铁路/义勇兵 / **Choose faction** — Institute/BOS/Railroad/Minutemen |
| 13-16 | 最终任务 / Endgame | 根据阵营不同——最终任务流程不同 / Different final quests per faction |

## 四大阵营 / Four Factions

### 义勇兵 / Minutemen

**民兵组织**——保护平民的志愿者联盟 / Militia — volunteer alliance protecting settlers.

| 任务线 / Questline | 说明 / Description |
|---|---|
| 当之无愧 / When Freedom Calls | 康科德救义勇兵——解锁义勇兵 / Rescue at Concord |
| 乔尔的任务 / Taking Independence | 清理城堡——恢复义勇兵总部 / Clear the Castle |
| 老旧的炮台 / Old Guns | 恢复城堡的炮兵防御 / Restore Castle artillery |
| 战利品 / Defend the Castle | 防守城堡——抵御学院攻击 / Defend against Institute |
| 核选项 / The Nuclear Option (义勇兵) | 炸毁学院——义勇兵结局 / Destroy the Institute — Minutemen ending |

**优势 / Pros**: 最直接的善阵营——不需要做出艰难的道德选择 / Most straightforward good faction — no hard moral choices
**劣势 / Cons**: 没有独特的科技/装备奖励 / No unique tech/gear rewards

**加入条件**: 在康科德击败死亡爪后普雷斯顿加入你 / Preston joins after Concord Deathclaw fight

### 钢铁兄弟会 / Brotherhood of Steel

**军事组织**——追求科技控制+消灭非人类 / Military order — seeking tech control + eradicating non-humans.

| 任务线 / Questline | 说明 / Description |
|---|---|
| 呼叫信号 / Fire Support | 剑桥警察局遇到丹斯 / Meet Danse at Cambridge PD |
| 号角 / Call to Arms | 帮助兄弟会清理阿卡迪亚 / Help BOS clear ArcJet Systems |
| 扫描/出征 / Tour of Duty | 各种兄弟会任务 / Various BOS missions |
| 失落的巡逻 / The Lost Patrol | 找回兄弟会失踪的巡逻队 / Find lost BOS patrol |
| 影子 / Show No Mercy | 用兄弟会飞鸟清理超级变种人据点 / Vertibird assault on mutant bases |
| 自由至尊 / Liberty Reprimed | 修复自由至尊——启动巨型机器人 / Repair Liberty Prime |
| 核选项 / The Nuclear Option (兄弟会) | 炸毁学院 / Destroy the Institute |

**优势 / Pros**: 最强的火力——飞鸟信号弹快速旅行 + 自由至尊 + 大量物资 / Best firepower — Vertibird + Liberty Prime + supplies
**劣势 / Cons**: 对合成人和僵尸态度激进——种族主义倾向 / Radical against synths and ghouls — xenophobic

**加入条件**: 完成剑桥警察局的任务后加入 / Complete Cambridge PD missions

### 铁路 / Railroad

**秘密组织**——解放合成人 / Secret organization — liberating synths.

| 任务线 / Questline | 说明 / Description |
|---|---|
| 铁之路 / Road to Freedom | 跟着自由之路的线索找到铁路 / Follow Freedom Trail to RR |
| 交易技巧 / Tradecraft | 诺拉的密码 / Decode PAM's data |
| 波士顿往事 / Boston After Dark | 执行铁路的渗透任务 / RR infiltration missions |
| 地下卧底 / Underground Undercover | 潜入学院——作为双面间谍 / Infiltrate the Institute as a double agent |
| 火箭的红色之光 / Rockets' Red Glare | 攻击兄弟会——铁路结局 / Attack BOS — RR ending |
| 核选项 / The Nuclear Option (铁路) | 炸毁学院 / Destroy the Institute |

**优势 / Pros**: 获得"防弹织物"改造——让你穿普通衣服也有高防御 / Get Ballistic Weave — wear normal clothes with high defense
**劣势 / Cons**: 军事力量最弱——没有重型装备 / Weakest military — no heavy equipment

**加入条件**: 找到自由之路的线索——解码密码——进入总部 / Follow Freedom Trail — decode — enter HQ

### 学院 / The Institute

**联邦最先进的科技组织**——在地下建造了完美的未来 / Most advanced tech org in the Commonwealth — built a perfect future underground.

| 任务线 / Questline | 说明 / Description |
|---|---|
| 学院 / Institutionalized | 第一次进入学院——"见到尚恩" / First visit to the Institute |
| 合成人崛起 / Synth Retention | 回收逃跑的合成人 / Retrieve escaped synths |
| 关闭 / Mankind Redefined | 各种学院任务 / Various Institute missions |
| 局外人 / The Battle for Bunker Hill | 在邦克山触发三方大战 / Three-way battle at Bunker Hill |
| 重组 / Mass Fusion | 从兄弟会手中夺取核心 / Steal the core from BOS |
| 核选项 / The Nuclear Option (学院) | **消灭兄弟会+铁路**——学院结局 / **Eliminate BOS + RR** — Institute ending |

**优势 / Pros**: 最先进的科技——X-1传送 + 合成人帮手 + 学院武器 / Most advanced tech — teleportation + synth helpers
**劣势 / Cons**: **"尚恩"其实是假的**——真正的尚恩是年迈的学院领导人——这有一个道德震撼 / "Shaun" is a synth — real Shaun is elderly Institute leader — massive moral shock

**加入条件**: 在"猎人/猎物"任务后获得进入学院的资格 / After "Hunter/Hunted" — gain access to the Institute

| 阵营 / Faction | 结局后状态 / Post-Game Status | 可继续任务 / Continue | 独特装备 / Unique Gear |

| 义勇兵 / Minutemen | 学院毁——兄弟会存活或毁 / Institute destroyed, BOS alive/dead | 聚落任务继续 / Settlement quests | 无特殊 / None special |
