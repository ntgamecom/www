# 世界与探索 / World & Exploration

## 联邦总览 / Commonwealth Overview

联邦是核战后的波士顿及其周边地区——总面积约 **6平方公里**(约Skyrim的1/5，但密度更高) / Post-nuclear Boston and surroundings — ~6 sq km (1/5 of Skyrim but denser).

### 主要区域 / Major Regions

| 区域 / Region | 等级范围 / Level | 特点 / Traits |
|---|---|---|
| **西北联邦 / NW Commonwealth** | 1-15 | 庇护山丘、康科德、列克星敦——新手区域 / Sanctuary, Concord, Lexington — starter area |
| **波士顿都市 / Boston Urban** | 10-30 | 钻石城、芳邻镇——高密度城市探索 / Diamond City, Goodneighbor — dense urban |
| **东海岸 / East Coast** | 15-35 | 昆西、城堡——义勇兵相关区域 / Quincy, Castle — Minutemen area |
| **发光海 / Glowing Sea** | 30-50 | **极度辐射**——需要全套辐射防护 / **Extreme rads** — full rad protection required |
| **北部 / North** | 10-25 | 联邦理工、麻省总部——学院区域 / CIT, MIT — Institute area |
| **南波士顿 / South Boston** | 15-30 | 宪法号、弹药厂——大量僵尸出没 / USS Constitution, Gunner's Plaza — ghouls |
| **昆西采石场 / Quincy Quarries** | 20-35 | 枪手帮聚集地 / Gunner territory |

## 主要地点指南 / Major Location Guide

### 钻石城 / Diamond City

**联邦最大的定居点**——在芬威公园球场内 / The Commonwealth's largest settlement — inside Fenway Park.

| 设施 / Facility | 说明 / Description |
|---|---|
| **市场 / Market** | 阿图罗武器店、迈娅护甲店、药剂师 / Arturo (weapons), Myrna (armor), Chemist |
| **钻石城电台 / Diamond City Radio** | 播放经典曲目——主持人特拉维斯 / Plays classics — host Travis Miles |
| **公共事件报社 / Publick Occurrences** | 老板派普——信息发布 / Piper — info and rumors |
| **侦探社 / Valentine Detective** | 尼克·瓦伦坦的办公室 / Nick Valentine's office |
| **避难山 / Vault 81** | 附近的地下避难所——有支线+独特的消辐宁 / Nearby vault — quest + unique cure |

### 芳邻镇 / Goodneighbor

**钻石城的竞争对手**——自由港 / Rival to Diamond City — free port.

| 设施 / Facility | 说明 / Description |
|---|---|
| 第三轨酒吧 / The Third Rail | 地下酒吧——有歌手+情报 / Underground bar — singer + intel |
| 凯伯博士的药店 / Dr. Crocker | 汉考克的医生 / Hancock's doctor |
| 废旧商店 / The Rexford Hotel | 可租房间 / Rentable room |

### 剑桥 / Cambridge

| 关键点 / Key Point | 说明 / Description |
|---|---|
| 剑桥警察局 / Cambridge PD | 兄弟会先遣队所在——遇到丹斯 / BOS forward base — meet Danse |
| 剑桥校园 / Cambridge Campus | 大量合成人出没——中后期区域 / Synth territory — mid-late area |

### 城堡 / The Castle

**义勇兵总部**——需要清理后解锁 / Minutemen HQ — clear to unlock.

| 关键点 / Key Point | 说明 / Description |
|---|---|
| 无线电塔 / Radio Tower | 恢复义勇兵电台 / Restore Minutemen radio |
| 军火库 / Armory | 内有大量弹药和胖子核弹 / Lots of ammo and Fat Man |
