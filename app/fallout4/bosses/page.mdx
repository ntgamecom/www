# BOSS攻略 / Boss Fights Guide

辐射4的Boss战包括巨型变种人、传奇合成人、DLC最终Boss以及阵营领袖 / Fallout 4 boss fights include super mutant behemoths, legendary synths, DLC final bosses, and faction leaders.

## 主线Boss / Main Story Bosses

### 克罗格 / Kellogg

| 项目 / Aspect | 说明 / Description |
|---|---|
| **等级 / Level** | 20-40 (取决于玩家等级) / Depends on player level |
| **生命 / HP** | 300-600 |
| **位置 / Location** | 碉堡山地下 / Fort Hagen underground |
| **武器 / Weapon** | .44左轮手枪——高伤害 / .44 Revolver — high damage |

**战斗阶段 / Fight Phases**:
| 阶段 / Phase | 行为 / Behavior | 应对 / Counter |
|---|---|---|
| 准备阶段 / Prep | 激活合成人警卫 / Activates synth guards | 先清理合成人——再面对克罗格 / Clear synths first |
| 远距离 / Range | 使用.44左轮射击 / Shoots with revolver | 利用柱子掩护——不要站桩 / Use pillar cover — don't stand still |
| 近距离 / Close | 扔手雷+近战 / Grenades + melee | 后退保持距离 / Backpedal |
| 残血 / Low HP | 使用治疗针 / Uses stimpaks | 连续输出不给他回血机会 / Sustained DPS |

**推荐打法 / Recommended Strategy**:
1. 进入前确保动力甲满核心和满血 / Full PA fusion core + full HP before entry
2. 先击杀所有合成人警卫——创造一对一局面 / Kill all synth guards first — 1v1
3. 使用手枪或步枪——VATS打头——暴击伤害最大化 / Pistol or rifle — VATS headshots — crit burst
4. 他使用治疗针时——燃烧弹或手雷打断 / When he stimpaks — interrupt with molotov or grenade

### 学院领袖 / Father (Shaun)

| 项目 / Aspect | 说明 / Description |
|---|---|
| **等级 / Level** | 50+ |
| **生命 / HP** | 100 (极脆) / Very fragile |
| **位置 / Location** | 学院内部 / Institute interior |
| **特殊 / Special** | 他使用隐身小子+能量武器 / Uses stealth boy + energy weapon |

**决策 / Decision**:
你可以选择杀死父亲或饶恕他——影响学院线的结局 / You can kill or spare Father — affects Institute ending.

**战斗建议 / Combat Tips**:
- 实际上他很容易击杀——只是周围的合成人护卫很多 / Easy kill — but many synth guards
- 使用VATS找出隐身中的他 / Use VATS to spot him while stealthed
- 如果选择敌对——先清护卫再击杀父亲 / If hostile — clear guards first

## 巨型变种人 / Super Mutant Behemoths

### 已知刷新点 / Known Spawn Locations

| 位置 / Location | 等级 / Level | 特殊 / Special |
|---|---|---|
| **列星顿 / Lexington** | 20+ | 在药房附近徘徊 / Roams near pharmacy |
| **国名银行 / National Bank of Lexington** | 25+ | 后巷重生 / Alley respawn |
| **发光海 / Glowing Sea** | 50+ | 在核爆点附近 / Near the crater |
| **桑树码头 / Tucker's Bridge** | 35+ | 桥下 / Under the bridge |
| **索格斯钢铁厂 / Saugus Ironworks** | 40+ | 工厂门口 / Factory entrance |
| **卡丁车场 / Drive-In** | 随机 / Random | 可以在任何时候出现 / Any time possible |

**巨兽打法 / Behemoth Tactics**:

| 策略 / Strategy | 说明 / Description |
|---|---|
| **远程风筝 / Ranged Kite** | 用狙击枪保持距离打头——后退射击 / Sniper headshots while backpedaling |
| **VATS暴击 / VATS Crit** | 积累暴击条——多次暴击打头——最快击杀 / Build crit bar — multiple head crits |
| **动力甲近战 / PA Melee** | 高力量+超级大锤——根深蒂固+闪电突袭 / High Str + Super Sledge — Rooted + Blitz |
| **爆炸武器 / Explosives** | 导弹发射器或胖子核弹——2-3发击杀 / Missile launcher or Fat Man — 2-3 shots |
| **地形利用 / Terrain** | 巨兽无法进门——利用门口射击 / Can't enter doors — shoot from doorways |

**注意事项 / Warnings**:
- 巨兽投掷汽车和巨石——不要在一个地方站太久 / Behemoths throw cars and boulders — keep moving
- 它们的近战一击可秒杀无动力甲的玩家 / Melee one-shots unarmored players
- 发光海版本带辐射光环——接近会快速积累辐射 / Glowing Sea variant has rad aura

### 麦克森长老 / Elder Maxson

| 项目 / Aspect | 说明 / Description |
|---|---|
| **等级 / Level** | 60+ (随玩家等级) |
| **生命 / HP** | 700+ |
| **护甲 / Armor** | T-60动力甲——高级改造 / T-60 PA — advanced mods |
| **武器 / Weapon** | 加特林激光——高DPS / Gatling Laser — high DPS |
| **位置 / Location** | 普利德温号飞艇 / The Prydwen |

**打法 / Strategy**:
- 使用穿甲武器——T-60护甲很高 / Use armor-piercing weapons — T-60 is tanky
- 打爆他的核融合核心——引发爆炸 / Shoot his fusion core — triggers explosion
- 保持距离——他的加特林激光近距很致命 / Keep distance — his Gatling Laser melts close range
- VATS打头消耗暴击 / VATS head crits
