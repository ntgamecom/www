## SPECIAL属性系统 / SPECIAL System

开局有 **28点** 可分配——每项初始1点，最高10点 / 28 points to allocate — each starts at 1, max 10.

| 属性 / Stat | 效果 / Effect | 关键Perk / Key Perk | 推荐初始值 / Starting Points |
|---|---|---|---|
| **S 力量 / Strength** | 近战伤害+负重 / Melee dmg + carry weight | 装甲商 / Armorer (S3) | 3 (为了装甲商) |
| **P 感知 / Perception** | VATS命中率+开锁 / VATS accuracy + lockpick | 开锁 / Locksmith (P4) | 4 (为了开锁) |
| **E 耐力 / Endurance** | 生命值+辐射抗性 / HP + rad resist | 生命之源 / Life Giver (E3) | 3-5 |
| **C 魅力 / Charisma** | 交易价格+说服成功率 / Prices + persuasion | 地方领导者 / Local Leader (C6) | 3-6 |
| **I 智力 / Intelligence** | **经验获取速度** + 黑客 / **XP gain** + hacking | 枪械迷 / Gun Nut (I3), 科学 / Science (I6) | **6+ (升级速度)** |
| **A 敏捷 / Agility** | AP值+潜行 / AP points + sneak | 潜行 / Sneak (A3) | 3-5 |
| **L 运气 / Luck** | 暴击积累速度+随机奖励 / Crit build + random rewards | 暴击银行 / Critical Banker (L7) | 3+ |

**新手推荐专用分配 / Recommended Starter Build**: S3 P4 E3 C3 I6 A3 L3 (共25点——剩下3点根据喜好加C或L)

## 生存模式入门 / Survival Mode Basics

| 需求 / Need | 效果 / Effect | 解决方法 / Solution |
|---|---|---|
| **饥饿 / Hunger** | -1点力量 / -1 STR | 吃煮熟的食物——不要吃生的(辐射+疾病) / Cooked food — raw gives rads + disease |
| **口渴 / Thirst** | -1点感知 / -1 PER | 喝纯净水——建水泵产瓶装水 / Purified water — build pumps for bottles |
| **疲劳 / Fatigue** | -1点AP / -1 AP | 睡在床上——每小时恢复疲劳 / Sleep in a bed |
| **困倦 / Tiredness** | 减最大AP / Reduced max AP | 连续不睡觉超过24小时会越来越困 / 24+ hours awake = increasing penalties |
| **疾病 / Disease** | 各种负面效果 / Various | 用抗生素或去看医生 / Antibiotics or doctor |

**生存模式关键提示 / Survival Key Tips**:
1. **睡觉即存档**——离开据点前一定睡一觉 / Sleeping is saving — always sleep before leaving base
2. **聚落补给线**——集中资源覆盖全地图 / Supply lines — connect all settlements for shared resources
3. **瓶盖是重量**——每个瓶盖0.000磅，但前代货币和银币更轻/钱更多 / Caps weigh nothing — but pre-war money and silver weigh less per value
4. **携带睡袋**——随时找地方睡 / Carry a sleeping bag — sleep anywhere
5. **不要招惹死亡爪**——除非你穿上动力甲/ Don't fight Deathclaws without PA

## 战斗系统基础 / Combat Basics

### VATS系统 / V.A.T.S.

VATS是辐射的标志性战斗系统——在实时战斗中暂停瞄准弱点 / VATS is Fallout's iconic combat system — pause real-time combat to target weak points.

| 元素 / Element | 说明 / Description |
|---|---|
| **命中率 / Hit Chance** | 显示每个目标的命中率——距离、武器、Perk影响 / Displayed per target — affected by distance, weapon, perks |
| **暴击条 / Critical Meter** | 每命中一次填充——满后按Space触发暴击 / Fills with each hit — press Space for crit |
| **AP消耗 / AP Cost** | VATS中每枪消耗AP——AP用光后退出 / Each shot costs AP — exit when AP is empty |
| **部位瞄准 / Limb Targeting** | 头(高伤害)、躯干(普通)、腿(减速)、手臂(缴械) / Head (dmg), torso (normal), legs (slow), arms (disarm) |

**VATS技巧 / VATS Tips**:
1. 头和躯干是最安全的选择 / Head and torso are safest choices
2. 打腿可以减速敌人——对付近战敌人效果好 / Leg shots slow enemies — great against melee enemies
3. 打武器可以让敌人缴械——对付持RPG/导弹的敌人效果好 / Shoot weapons to disarm — great against explosives

## 新手常见错误 / Common Beginner Mistakes

| # | 错误 / Mistake | 正确做法 / Correct Way |
|---|---|---|
| 1 | 不加智力 / Ignore INT | 智力决定升级速度——智力6+是最佳选择 / INT = XP rate — 6+ is optimal |
| 2 | 不做聚落 / Skip settlements | 聚落提供纯净水(卖钱)+经验+安全屋 / Settlements provide water (money) + XP + safehouses |
| 3 | 乱卖垃圾 / Sell junk randomly | **不卖垃圾**——全部拆成建造材料 / **Never sell junk** — scrap for building materials |
| 4 | 不换同伴 / Stick to one companion | 把每个同伴的好感度刷满——拿到专属Perk后再换 / Max each companion's affinity for their perk |
| 5 | 不捡弹药 / Ignore ammo | 辐射4的弹药可以捡到很多——够用就捡 / FO4 drops plenty of ammo — always pick it up |
| 6 | 硬刚更高等级敌人 / Fight high-level enemies | 初期遇到神话级死亡爪/跟班——跑！/ Run from Mythic Deathclaws and Behemoths |
