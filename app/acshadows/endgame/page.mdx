# 终局内容 / Endgame Content

{/* 终局内容指南 */}

## 终局概述 / Endgame Overview

完成主线剧情只是刺客信条：影的开始。终局内容提供了大量挑战和奖励，让玩家在通关后继续享受游戏。

*Completing the main story is just the beginning of Assassin's Creed Shadows. The endgame offers a wealth of challenges and rewards for players to continue enjoying the game post-story.*

## 通关后内容 / Post-Story Content

### 未完成的任务 / Unfinished Business

通关后，所有未完成的主线和支线任务仍然可以继续进行。此外，还会解锁一批新的终局任务。

*After completing the story, all unfinished main and side quests remain available. Additionally, a new set of endgame quests becomes unlocked.*

| 任务类型 / Quest Type | 数量 / Count | 推荐等级 / Level | 主要奖励 / Main Reward |
|---|---|---|---|
| 终局任务 / Endgame Quests | 15+ | 50 | 传说级装备 / Legendary gear |
| 传奇契约 / Legendary Contracts | 10+ | 50+ | 专属外观 / Exclusive skins |
| 世界事件 / World Events | 20+ | 45+ | 稀有材料 / Rare materials |
| 每日挑战 / Daily Challenges | 每日刷新 / Daily reset | 50 | 金币和材料 / Gold and materials |

### 自由探索 / Free Exploration

通关后地图上会出现新的探索标记和隐藏内容。

*New exploration markers and hidden content appear on the map after story completion.*

- 隐藏洞穴和密室解锁 / Hidden caves and secret rooms unlock
- 传说级宝箱刷新 / Legendary chests respawn
- 特殊动物和材料出现 / Special animals and materials appear
- 新的鸟瞰点标记出现 / New viewpoint markers appear
- 隐藏BOSS在世界地图上游荡 / Hidden bosses roam the world map

## 新游戏+ / New Game+

新游戏+模式允许玩家保留大部分进度重新开始游戏，体验更高的难度和新的奖励。

*New Game+ allows players to restart the game with most of their progress, experiencing higher difficulty and new rewards.*

### 保留内容 / Carried Over Content

| 类别 / Category | 是否保留 / Carried Over | 说明 / Note |
|---|---|---|
| 角色等级 / Character Level | 是 / Yes | 保留当前等级和技能点 / Keep current level and skill points |
| 技能 / Skills | 是 / Yes | 所有已学技能保留 / All learned skills kept |
| 装备 / Equipment | 是 / Yes | 所有武器、护甲和饰品 / All weapons, armor, and accessories |
| 材料 / Materials | 是 / Yes | 所有锻造和升级材料 / All crafting and upgrade materials |
| 金钱 / Money | 是 / Yes | 所有持有的金钱 / All held money |
| 地图进度 / Map Progress | 部分 / Partial | 鸟瞰点保留，任务重置 / Viewpoints kept, quests reset |
| 阵营声望 / Faction Rep | 是 / Yes | 所有阵营声望等级保留 / All faction rep levels kept |
| 收集品 / Collectibles | 是 / Yes | 所有已收集的物品保留 / All collected items kept |

### 新增内容 / New Content in NG+

| 内容 / Content | 说明 / Description |
|---|---|
| 新难度选项 / New Difficulty Option | 新增「传奇（Legendary）」难度 / New Legendary difficulty mode |
| 强化敌人 / Enhanced Enemies | 所有敌人等级提升，新增精英变种 / All enemies leveled up, new elite variants |
| 新装备 / New Equipment | 新游戏+专属传说级装备套装 / NG+ exclusive legendary armor sets |
| 额外技能点 / Extra Skill Points | 更高等级上限，更多技能点 / Higher level cap, more skill points |
| 隐藏结局 / Hidden Ending | 达成特定条件解锁真结局 / Unlock true ending under specific conditions |

### 新游戏+建议 / NG+ Tips

1. 通关后建议先完成所有终局任务再开启新游戏+ / Complete all endgame quests before starting NG+
2. 升级至少两套完整传说级装备 / Upgrade at least two full legendary gear sets
3. 收集足够的材料以应对更高升级需求 / Gather enough materials for higher upgrade demands
4. 新游戏+中敌人攻击模式会变化，需要重新适应 / Enemy attack patterns change in NG+, need to readapt
5. 传说难度下完美格挡的时机窗口更短 / Legendary difficulty has shorter perfect parry windows

## 传承模式 / Legacy Mode

传承模式是一个挑战模式，玩家需要在特定条件下完成一系列战斗和潜行挑战。

*Legacy Mode is a challenge mode where players must complete a series of combat and stealth challenges under specific conditions.*

### 挑战类型 / Challenge Types

| 挑战 / Challenge | 条件 / Condition | 奖励 / Reward |
|---|---|---|
| 暗影试炼 / Shadow Trial | 全程不被发现完成潜行关卡 / Complete stealth section without detection | 传说级潜行护甲 / Legendary stealth armor |
| 武勇试炼 / Valor Trial | 在限定时间内击败所有敌人 / Defeat all enemies within time limit | 传说级战斗护甲 / Legendary combat armor |
| 精准试炼 / Precision Trial | 仅使用远程武器击杀 / Kill using only ranged weapons | 传说级弓 / Legendary bow |
| 生存试炼 / Survival Trial | 连续10波敌人，不恢复生命 / Survive 10 waves without healing | 传说级饰品 / Legendary accessory |
| 完美试炼 / Perfection Trial | 无伤通关 / Complete without taking damage | 专属称号 / Exclusive title |

### 传承等级 / Legacy Ranks

完成挑战后，你会获得传承等级评分。

*After completing challenges, you earn a Legacy Rank score.*

| 等级 / Rank | 评分要求 / Score Required | 解锁奖励 / Unlock Reward |
|---|---|---|
| 铜 / Bronze | 1000-2499 | 基础外观 / Basic skin |
| 银 / Silver | 2500-4999 | 稀有外观 / Rare skin |
| 金 / Gold | 5000-9999 | 史诗级奖励 / Epic reward |
| 白金 / Platinum | 10000-19999 | 传说级武器 / Legendary weapon |
| 钻石 / Diamond | 20000+ | 终极外观 + 称号 / Ultimate skin + title |

## 挑战任务 / Challenge Quests

### 每周挑战 / Weekly Challenges

每周更新的挑战任务，提供持续的游戏动力。

*Weekly rotating challenge quests provide ongoing gameplay motivation.*

- 猎杀指定数量的特定敌人 / Hunt a specific number of certain enemy types
- 在特定区域完成潜行任务 / Complete stealth objectives in specific regions
- 使用特定武器击败精英敌人 / Defeat elite enemies using specific weapons
- 收集指定材料 / Collect specific materials
- 完成特定数量的契约任务 / Complete a specific number of contracts

| 成就名 / Achievement | 条件 / Condition | 奖励点数 / Reward Points |
|---|---|---|
| 影之大师 / Shadow Master | 在传奇难度下完成全部主线 / Complete all main quests on Legendary | 100 Gamerscore |
| 完美主义者 / Perfectionist | 100%完成度 / 100% completion | 150 Gamerscore |
| 双影传奇 / Twin Shadows Legend | 使用双主角各完成一次新游戏+ / Complete NG+ with each protagonist | 200 Gamerscore |
| 不朽 / Immortal | 在传承模式中获得钻石评级 / Achieve Diamond rank in Legacy Mode | 100 Gamerscore |
