## 选择与后果 / Choices and Consequences

本作包含多个重要选择，这些选择会影响剧情走向和结局。

*The game features multiple important choices that affect story direction and endings.*

### 重大选择 / Major Choices

| 章节 / Chapter | 选择 / Choice | 影响 / Impact |
|---|---|---|
| 3 | 信任弥助还是独自行动 / Trust Yasuke or act alone | 影响后续合作任务 / Affects future coop missions |
| 5 | 招募忍者还是武士 / Recruit ninja or samurai | 决定最终阵营 / Determines final faction |
| 6 | 保护京都居民或追击目标 / Protect civilians or pursue target | 影响京都地区的状态 / Affects Kyoto region state |
| 7 | 饶恕或处决反派 / Spare or execute the villain | 导致不同结局分支 / Leads to different ending branches |
| 8 | 最终牺牲选择 / Final sacrifice choice | 决定哪个主角的终局 / Determines which protagonist's ending |

### 结局类型 / Ending Types

| 结局 / Ending | 条件 / Condition | 简述 / Summary |
|---|---|---|
| 影之和平 / Shadow Peace | 多数选择偏向潜行和仁慈 / Mostly stealth and mercy choices | 奈绪江主导的和平结局 / Naoe-led peaceful ending |
| 武士之治 / Samurai Order | 多数选择偏向力量和果断 / Mostly power and decisive choices | 弥助主导的秩序结局 / Yasuke-led order ending |
