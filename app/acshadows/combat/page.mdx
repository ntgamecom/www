## 架势系统 / Posture System

架势（Posture）是本作战斗的核心机制，类似于《只狼》的体干系统。

*Posture is the core combat mechanic, similar to the posture system in Sekiro.*

### 架势条详解 / Posture Bar Details

每个敌人（包括玩家）都有一条架势条。当架势被完全破坏时，敌人将进入失衡状态，你可以发动致命一击。

*Every enemy (including the player) has a posture bar. When posture is fully broken, the enemy enters a staggered state, allowing a lethal strike.*

| 架势状态 / Posture State | 效果 / Effect |
|---|---|
| 满架势 / Full Posture | 敌人防御力全开 / Enemy defense fully active |
| 半架势 / Half Posture | 防御下降，更容易被击退 / Defense reduced, easier to push back |
| 架势崩溃 / Posture Broken | 无法防御，可发动致命一击 / Cannot defend, lethal strike available |

### 恢复架势 / Posture Recovery

- 防御时架势恢复较慢 / Slow recovery when defending
- 闪避后恢复速度提升 / Recovery speed increases after dodging
- 完美格挡可大幅恢复架势 / Perfect parry significantly restores posture

## 格挡与闪避 / Parry and Dodge

### 完美格挡 / Perfect Parry

在敌人攻击命中前的一瞬间按下格挡键，可以触发完美格挡，使敌人短暂硬直。

*Press the block key at the moment before an enemy attack lands to trigger a perfect parry, briefly stunning the enemy.*

- 普通格挡：减少伤害但消耗架势 / Normal block: reduces damage but consumes posture
- 完美格挡：无伤反弹，破坏敌人架势 / Perfect parry: no damage, breaks enemy posture
- 不可格挡攻击：带有红色闪光提示，必须闪避 / Unblockable attacks: indicated by red flash, must dodge

### 闪避系统 / Dodge System

| 闪避类型 / Dodge Type | 操作 / Input | 效果 / Effect |

| 普通闪避 / Normal Dodge | 方向键 + 闪避 / Direction + Dodge | 短距离位移 / Short distance movement |
| 滑步 / Slide | 闪避 + 蹲伏 / Dodge + Crouch | 低姿态移动，可穿过低矮障碍 / Low profile move, can pass under obstacles |
| 侧翻 / Roll | 双击闪避 / Double Tap Dodge | 长距离翻滚，无敌帧较长 / Long roll, longer invincibility frames |

## 战斗策略 / Combat Strategies

### 面对群敌 / Against Groups

- 奈绪江：利用烟雾弹制造混乱，逐个暗杀 / Naoe: use smoke bombs to create chaos, assassinate one by one
- 弥助：使用长枪或薙刀的横扫攻击控制场面 / Yasuke: use yari or naginata sweeping attacks to control the crowd
- 切换掩体，避免被包围 / Use cover, avoid being surrounded

### 面对精英敌人 / Against Elites

- 观察敌人的攻击模式，寻找破绽 / Observe enemy attack patterns, find openings
- 使用完美格挡快速破坏架势 / Use perfect parries to quickly break posture
- 优先清理周围小兵再专注精英 / Clear surrounding minions first, then focus on elites

### 面对BOSS / Against Bosses

- 携带足够的回复道具 / Bring enough healing items
- 使用对应的克制武器 / Use counter weapons
- 学习BOSS的招式前摇 / Learn boss attack tells
- 不要贪刀，保持耐心 / Do not get greedy, stay patient
