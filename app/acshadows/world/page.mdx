## 日本地图 / Map of Japan

刺客信条：影的地图以16世纪的日本为蓝本，分为多个各具特色的区域。每个区域都有独特的地形、文化和任务。

*The map of Assassin's Creed Shadows is based on 16th-century Japan, divided into multiple distinct regions. Each region has unique terrain, culture, and quests.*

## 主要区域 / Major Regions

### 伊贺（Iga）

奈绪江的故乡，以山地和忍者村闻名。这里是忍者文化的中心，遍布隐秘通道和训练场。

*Naoe's homeland, known for mountains and ninja villages. This is the center of ninja culture, filled with hidden passages and training grounds.*

|---|---|
| 地形 / Terrain | 山地森林 / Mountain forest |
| 主要城市 / Main City | 伊贺上野城 / Iga Ueno Castle |

| 推荐等级 / Recommended Level | 1-15 |
| 探索要素 / Exploration | 忍者洞穴、秘密瀑布、训练遗迹 |

日本的首都，天皇和幕府的所在地。繁华的城市区域与宁静的寺庙并存。

## 鸟瞰点 / Viewpoints

鸟瞰点是系列经典机制的回归。同步鸟瞰点可以解锁快速旅行点并揭示周围区域的地图。

*Viewpoints are a returning classic mechanic. Synchronizing viewpoints unlocks fast travel points and reveals surrounding map areas.*

| 鸟瞰点类型 / Type | 难度 / Difficulty | 奖励 / Reward |
|---|---|---|
| 塔楼 / Tower | 低 / Low | 区域地图 + 经验值 / Area map + XP |
| 山峰 / Mountain Peak | 中 / Medium | 区域地图 + 经验值 + 材料 / Area map + XP + materials |
| 城堡主塔 / Castle Keep | 高 / High | 区域地图 + 大量经验值 + 装备 / Area map + large XP + gear |
| 神社鸟居 / Shrine Torii | 低 / Low | 区域地图 + 特殊道具 / Area map + special item |

## 城堡攻略 / Castle Guide

城堡是本作中的重要据点，通常驻有重兵和重要目标。

*Castles are important strongholds in the game, usually heavily guarded with important targets.*

### 城堡类型 / Castle Types

| 类型 / Type | 防守等级 / Defense Level | 建议策略 / Recommended Strategy |
|---|---|---|
| 小型堡垒 / Small Fort | 2-3 | 可强攻可潜行 / Either assault or stealth |
| 中型城塞 / Medium Castle | 4-5 | 建议潜行潜入 / Recommend stealth approach |
| 大型城堡 / Large Castle | 6-8 | 需要充分准备 / Requires full preparation |
| 巨型天守 / Giant Citadel | 9-10 | 推荐组队或高等级 / Recommended with high level |

### 城堡探索要点 / Castle Exploration Tips

- 首先寻找情报商获取城堡地图 / First find informants for castle maps
- 标记所有巡逻路线和巡逻兵 / Mark all patrol routes and guards
- 寻找地下通道或秘密入口 / Find underground passages or secret entrances
- 优先清除警报钟附近的敌人 / Prioritize enemies near alarm bells
- 击败城堡领主获得区域控制权 / Defeat castle lord for area control

## 四季与天气 / Seasons and Weather

### 季节系统 / Season System

游戏中的季节会动态变化，每个季节影响 gameplay 的各个方面。

*Seasons dynamically change in the game, each affecting various aspects of gameplay.*

| 季节 / Season | 效果 / Effect | 持续时间 / Duration |
|---|---|---|
| 春 / Spring | 樱花开放，部分区域可通行 / Cherry blossoms bloom, some areas accessible | 7天游戏内 / 7 in-game days |
| 夏 / Summer | 白天更长，植被茂密提供掩护 / Longer days, dense foliage for cover | 7天游戏内 / 7 in-game days |
| 秋 / Autumn | 落叶增加噪音 / Falling leaves increase noise | 7天游戏内 / 7 in-game days |
| 冬 / Winter | 夜晚更长，水面结冰可行走 / Longer nights, frozen water walkable | 7天游戏内 / 7 in-game days |

### 天气系统 / Weather System

| 天气 / Weather | 对潜行的影响 / Stealth Effect | 对战斗的影响 / Combat Effect |
|---|---|---|
| 晴天 / Clear | 视野开阔，不利潜行 / Open view, bad for stealth | 无影响 / No effect |
| 阴天 / Cloudy | 阴影增多，潜行略好 / More shadows, slightly better | 无影响 / No effect |
| 雨天 / Rainy | 噪音掩盖脚步声 / Rain masks footsteps | 地面湿滑，影响闪避 / Slippery, affects dodge |
| 雷暴 / Thunderstorm | 最佳潜行时机 / Best stealth opportunity | 视线受阻 / Vision obstructed |
| 雾天 / Foggy | 视野大幅降低 / Vision greatly reduced | 远程武器精度下降 / Ranged accuracy reduced |
| 雪天 / Snowy | 脚印可见，追踪风险 / Footprints visible, tracking risk | 寒冷降低耐力回复 / Cold reduces stamina regen |
