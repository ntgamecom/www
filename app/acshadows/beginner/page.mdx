# 新手入门 / Beginner Guide

{/* 新手入门指南 */}

## 欢迎来到刺客信条：影 / Welcome to Assassin's Creed Shadows

欢迎来到16世纪的日本战国时代！本指南将为新玩家提供全面的入门帮助，从角色选择到基本操作，让你快速上手。

*Welcome to 16th-century Sengoku Japan. This guide provides comprehensive help for new players, from character selection to basic controls, helping you get started quickly.*

## 角色选择 / Character Selection

游戏开始后不久，你将面临一个重要选择：以哪位主角为主进行游戏？两种选择将带来完全不同的体验。

*Early in the game, you will face an important choice: which protagonist to primarily play as? Each choice offers a completely different experience.*

### 奈绪江（Naoe）- 忍者路线 / Shinobi Path

| 特性 / Trait | 详情 / Details |
|---|---|
| 战斗风格 / Combat Style | 潜行与敏捷攻击 / Stealth and agile attacks |
| 擅长武器 / Preferred Weapons | 短刀（Kusarigama）、手里剑（Shuriken）、袖剑（Hidden Blade） |
| 移动能力 / Mobility | 极高，可以使用钩爪攀爬 / Very high, can use grappling hook |
| 推荐玩家 / Recommended For | 喜欢《刺客信条》经典潜行玩法的玩家 |
| 难度 / Difficulty | 中等，需要较高的操作技巧 / Medium, requires higher skill |

### 弥助（Yasuke）- 武士路线 / Samurai Path

| 特性 / Trait | 详情 / Details |
|---|---|
| 战斗风格 / Combat Style | 正面强攻与力量压制 / Direct assault and power overwhelming |
| 擅长武器 / Preferred Weapons | 太刀（Katana）、金碎棒（Kanabo）、弓（Bow） |
| 移动能力 / Mobility | 较低，不能攀爬 / Low, cannot climb |
| 推荐玩家 / Recommended For | 喜欢《仁王》或《只狼》战斗风格的玩家 |
| 难度 / Difficulty | 较低，适合新手 / Lower, suitable for newcomers |

## 序章攻略 / Prologue Walkthrough

### 第一阶段：逃离伊贺 / Stage 1: Escape from Iga

游戏序章从伊贺地区开始。奈绪江的村庄遭到织田信长军队的攻击。你需要完成以下目标：

*The prologue begins in the Iga region. Naoe's village is attacked by Oda Nobunaga's army. You need to complete the following objectives:*

1. 跟随引导逃离燃烧的村庄 / Follow the guide to escape the burning village
2. 学习基本潜行技巧 / Learn basic stealth techniques
3. 使用鹰眼视觉标记敌人 / Use Eagle Vision to mark enemies
4. 完成第一次暗杀 / Perform your first assassination
5. 抵达安全区域 / Reach the safe zone

### 第二阶段：弥助的觉醒 / Stage 2: Yasuke's Awakening

序章后期将切换到弥助的视角。你将体验武士的强大力量：

*Later in the prologue, the perspective shifts to Yasuke. You will experience the samurai's immense power:*

1. 学习基础战斗连击 / Learn basic combat combos
2. 掌握格挡与反击 / Master block and counterattack
3. 使用重武器击破盾牌兵 / Use heavy weapons to break shield enemies
4. 第一次BOSS战教学 / First tutorial boss fight

## 基地建设 / Base Building

### 忍者据点 / Shinobi Hideout

随着剧情推进，你将解锁自己的忍者据点。这是你的大本营，可以：

*As the story progresses, you will unlock your own shinobi hideout. This is your home base where you can:*

- **训练场（Training Ground）** — 提升技能熟练度 / Improve skill proficiency
- **武器工坊（Weapon Workshop）** — 锻造和升级武器 / Forge and upgrade weapons
- **情报室（Intelligence Room）** — 解锁新任务和区域 / Unlock new quests and regions
- **花园（Garden）** — 种植材料用于制作 / Grow materials for crafting

### 城堡升级 / Castle Upgrades

弥助可以解锁武士城堡，提供不同的加成：

*Yasuke can unlock a samurai castle, providing different bonuses:*

- **道场（Dojo）** — 解锁新战斗技能 / Unlock new combat skills
- **马厩（Stable）** — 提升坐骑速度和耐力 / Improve horse speed and stamina
- **军械库（Armory）** — 增加武器存储上限 / Increase weapon storage limit
- **瞭望塔（Watchtower）** — 扩大地图视野 / Expand map视野

## 新手建议 / Tips for Beginners

1. 优先探索伊贺和京都周边区域，完成支线任务获取经验 / Prioritize exploring Iga and Kyoto outskirts, complete side quests for XP
2. 奈绪江适合潜行路线，弥助适合正面战斗 / Naoe suits stealth approach, Yasuke suits direct combat
3. 定期升级武器和护甲，不要囤积材料 / Regularly upgrade weapons and armor, do not hoard materials
4. 利用鹰眼视觉频繁扫描周围环境 / Use Eagle Vision frequently to scan surroundings
5. 切换角色应对不同场景 / Swap characters for different scenarios
6. 完成训练挑战获取额外技能点 / Complete training challenges for extra skill points
