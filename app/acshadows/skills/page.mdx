# 技能与升级 / Skills and Upgrades

{/* 技能与升级指南 */}

## 技能系统总览 / Skill System Overview

刺客信条：影采用双角色独立的技能树系统。奈绪江和弥助各自拥有独特的技能树，玩家需要通过积累经验值来解锁技能点。

*Assassin's Creed Shadows features independent skill trees for each character. Naoe and Yasuke each have unique skill trees, and players earn skill points through experience accumulation.*

### 获取技能点 / Earning Skill Points

| 方式 / Method | 获得点数 / Points Gained | 说明 / Note |
|---|---|---|
| 升级 / Level Up | 1点 / 1 Point | 每升一级获得 / Earned per level |
| 完成主线任务 / Main Quests | 1-3点 / 1-3 Points | 视任务难度而定 / Depends on quest difficulty |
| 完成支线任务 / Side Quests | 1点 / 1 Point | 大部分支线奖励 / Most side quests reward this |
| 探索发现 / Exploration | 1点 / 1 Point | 首次发现地标 / First discovery of landmarks |
| 挑战试炼 / Challenge Trials | 2点 / 2 Points | 完成特定挑战 / Complete specific challenges |

## 精通系统 / Mastery System

当角色达到最高等级后，精通系统解锁，允许你继续提升特定方面的能力。

*When a character reaches max level, the Mastery system unlocks, allowing you to further enhance specific abilities.*

| 精通类别 / Mastery Category | 效果 / Effect | 最大等级 / Max Level |
|---|---|---|
| 武器精通 / Weapon Mastery | 提升特定武器伤害 / Increase specific weapon damage | 50级 / Level 50 |
| 生命精通 / Life Mastery | 提升最大生命值 / Increase max HP | 30级 / Level 30 |
| 耐力精通 / Stamina Mastery | 提升耐力上限和恢复速度 / Increase stamina and recovery | 30级 / Level 30 |
| 工具精通 / Tool Mastery | 提升工具效果和携带量 / Increase tool effects and capacity | 20级 / Level 20 |
