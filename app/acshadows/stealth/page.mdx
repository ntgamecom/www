# 潜行与暗杀 / Stealth and Assassination

{/* 潜行与暗杀指南 */}

## 潜行机制 / Stealth Mechanics

本作将潜行系统提升到了全新的高度。光线、噪音、阴影和伪装都将影响你的潜行效果。

*This game elevates the stealth system to new heights. Light, noise, shadows, and camouflage all affect your stealth effectiveness.*

### 光线与阴影 / Light and Shadow

| 环境 / Environment | 被发现概率 / Detection Chance | 说明 / Note |
|---|---|---|
| 完全黑暗 / Complete Darkness | 极低 / Very Low | 最佳潜行状态 / Best stealth state |
| 半阴影 / Partial Shadow | 低 / Low | 保持距离即可 / Keep distance |
| 月光下 / Moonlight | 中 / Medium | 容易被发现 / Easily detected |
| 火把附近 / Near Torch | 高 / High | 尽量避开 / Avoid if possible |
| 完全光亮 / Full Light | 极高 / Very High | 不建议潜行 / Stealth not recommended |

### 噪音系统 / Noise System

你的移动速度会影响产生的噪音大小：

*Your movement speed affects the noise you generate:*

| 移动方式 / Movement | 噪音等级 / Noise Level | 适用范围 / Best Use |
|---|---|---|
| 慢走 / Slow Walk | 无 / None | 靠近敌人背后 / Approaching from behind |
| 正常行走 / Normal Walk | 低 / Low | 一般潜行移动 / General stealth movement |
| 冲刺 / Sprint | 高 / High | 不推荐潜行时使用 / Not recommended during stealth |
| 蹲伏 / Crouch | 极低 / Very Low | 标准潜行姿态 / Standard stealth stance |
| 匍匐 / Prone | 无 / None | 穿越低矮障碍 / Crossing low obstacles |

## 鹰眼视觉 / Eagle Vision

鹰眼视觉是刺客的核心能力，可以感知周围的敌人和重要物品。

*Eagle Vision is the core ability of the Assassin, allowing you to sense enemies and important items around you.*

### 功能 / Functions

- 透视墙壁标记敌人 / See enemies through walls
- 显示可互动物品 / Highlight interactive objects
- 标记巡逻路线 / Mark patrol routes
- 显示警报状态 / Display alert status
- 追踪任务目标 / Track quest objectives

### 升级效果 / Upgrade Effects

| 等级 / Level | 效果 / Effect |
|---|---|
| 基础 / Basic | 标记 20米内的敌人 / Mark enemies within 20m |
| 进阶 / Advanced | 标记 40米内的敌人，显示敌人视野范围 / Mark enemies within 40m, show vision cones |
| 精通 / Master | 标记 60米内的敌人，穿墙显示弱点 / Mark enemies within 60m, show weak points through walls |

## 暗杀技巧 / Assassination Techniques

### 正面暗杀 / Front Assassination

从正面接近敌人时，可以使用快速暗杀，但风险较高。

*When approaching from the front, you can perform a quick assassination, but it is higher risk.*

- 需要敌人未察觉状态 / Requires enemy unaware state
- 暗杀速度较慢 / Slower assassination speed
- 成功后自动隐藏尸体 / Auto-hide body after success

### 背后暗杀 / Back Assassination

从背后接近是最高效的暗杀方式。

*Approaching from behind is the most efficient assassination method.*

- 无声且快速 / Silent and fast
- 自动拖入草丛 / Auto-drag into grass
- 中甲和重甲敌人也可一击必杀 / Also lethal against medium and heavy armor enemies

### 空中暗杀 / Air Assassination

从高处跃下发动暗杀，是刺客的招牌动作。

*Leaping from heights to perform an assassination is the Assassin's signature move.*

- 需要上方立足点 / Requires overhead perch
- 范围判定较大 / Larger hitbox
- 可使用隼落技能强化 / Can be enhanced with Falcon Drop skill

### 环境暗杀 / Environmental Assassination

利用场景中的元素进行暗杀：

*Use environmental elements for assassinations:*

| 方式 / Method | 条件 / Condition | 效果 / Effect |
|---|---|---|
| 草丛暗杀 / Grass Assassination | 躲在草丛中 / Hiding in grass | 将敌人拖入草丛 / Drag enemy into grass |
| 墙角暗杀 / Corner Assassination | 在墙角引诱 / Luring at corner | 在转角处伏击 / Ambush at turn |
| 天花板暗杀 / Ceiling Assassination | 悬挂在天花板 / Hanging on ceiling | 从上方突袭 / Strike from above |
| 水边暗杀 / Water Assassination | 在浅水区 / In shallow water | 将敌人拉入水中 / Pull enemy into water |

## 潜行路线推荐 / Recommended Stealth Path

1. 优先升级鹰眼视觉范围 / Prioritize upgrading Eagle Vision range
2. 学习烟雾弹相关技能 / Learn smoke bomb related skills
3. 获取穿透苦无以应对盾兵 / Get piercing kunai for shielded enemies
4. 解锁空中暗杀技能 / Unlock air assassination skills
5. 装备减少噪音的护甲 / Equip noise-reducing armor
6. 利用夜晚和雨天增加潜行优势 / Use night and rain for stealth advantage
7. 标记所有敌人后再行动 / Mark all enemies before moving
8. 保持耐心，等待最佳时机 / Stay patient, wait for the best moment
