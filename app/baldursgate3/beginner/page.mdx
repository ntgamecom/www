# 新手入门 / Beginner Guide

## D&D 5e基础 / D&D 5e Basics

博德之门3基于D&D 5e规则——以下是关键概念 / BG3 is based on D&D 5e — key concepts:

### 属性 / Ability Scores

| 属性 / Ability | 关联 / Affects | 推荐主属性职业 / Recommended Classes |
|---|---|---|
| 力量 / Strength | 近战命中/伤害、跳跃距离 / Melee hit/dmg, jump | 战士、野蛮人、圣武士 / Fighter, Barbarian, Paladin |
| 敏捷 / Dexterity | 远程命中/伤害、先攻、AC / Ranged hit/dmg, initiative, AC | 游侠、游荡者、武僧 / Ranger, Rogue, Monk |
| 体质 / Constitution | HP、专注检定 / HP, concentration checks | **全职业**——没有职业不重要 / **All classes** — never dump |
| 智力 / Intelligence | 法师法术、调查检定 / Wizard spells, investigation | 法师 / Wizard |
| 感知 / Wisdom | 牧师/德鲁伊/游侠法术、察觉检定 / Cleric/Druid/Ranger spells, perception | 牧师、德鲁伊、武僧 / Cleric, Druid, Monk |
| 魅力 / Charisma | 圣武士/邪术师/吟游诗人法术、对话掷骰 / Paladin/Warlock/Bard spells, dialogue | 圣武士、邪术师、吟游诗人 / Paladin, Warlock, Bard |

## 核心机制 / Core Mechanics

### 动作系统 / Action Economy

| 动作类型 / Action Type | 每回合可用 / Per Turn | 常见用途 / Common Uses |
|---|---|---|
| 标准动作 / Action | 1次 / 1 | 攻击、施法(大部分法术)、冲刺、闪避 / Attack, cast (most spells), Dash, Dodge |
| 附赠动作 / Bonus Action | 1次 / 1 | 副手攻击、部分法术(如治愈真言)、推挤 / Off-hand attack, some spells (Healing Word), Shove |
| 移动 / Movement | 固定距离 / Fixed distance | 移动、跳跃——可以拆分使用 / Move, Jump — can be split |
| 反应 / Reaction | 每回合1次 / 1 per round | 借机攻击、护盾术、某些法术反击 / Opportunity attack, Shield spell, Counterspell |

### 长休 / 短休 / Short Rest / Long Rest

| 休息类型 / Rest | 恢复 / Restores | 消耗 / Cost |
|---|---|---|
| 短休 / Short Rest | 部分生命值 + 某些职业能力(如战技骰) / Some HP + some class features | 每日2次免费 / 2 free per day |
| 长休 / Long Rest | **全部HP + 全部法术位 + 所有能力** / **Full HP + all spell slots + all abilities** | 40补给品(战术家80) / 40 supplies (80 Tactician) |

### 法术等级 / Spell Levels

| 角色等级 / Character Level | 最高法术环位 / Max Spell Level | 说明 / Notes |
|---|---|---|
| 1-2 | 1环 / Level 1 | 基础法术 / Basic spells |
| 3-4 | 2环 / Level 2 | 强力法术出现 / Stronger spells |
| 5-6 | 3环 / Level 3 | **质变**——火球术、加速术等 / **Power spike** — Fireball, Haste |
| 7-8 | 4环 / Level 4 | 高等隐型、放逐术 / Greater Invis, Banishment |
| 9-10 | 5环 / Level 5 | 死云术、召唤元素 / Cloudkill, Conjure Elemental |
| 11-12 | 6环 / Level 6 | 链状闪电、解离术 / Chain Lightning, Disintegrate |

### 对话与检定系统 / Dialogue & Skill Checks

博德之门3的对话系统基于D&D的技能检定——你的选择掷骰结果决定剧情走向。关键机制: **优势(Advantage)**=掷两次取高值——通常来自法术(如"朋友"戏法)或背景。**劣势(Disadvantage)**=掷两次取低值——来自负面状态或装备。**激励点(Inspiration)**=每个角色有4个——消耗一个可以重掷任意检定。建议重要对话前存档 / BG3's dialogue system uses D&D skill checks — your roll determines story outcomes. Key: **Advantage** = roll twice take higher — from spells (Friends cantrip) or background. **Disadvantage** = roll twice take lower — from debuffs. **Inspiration** = 4 per character — reroll any check. Save before important conversations.

```
  ↓

**进入第二章 / Enter Act 2**
```

| 等级 / Level | 区域 / Zone | 关键任务 / Key Quests |
|---|---|---|
| 1-3 | 海滩+翡翠林地周边 / Beach + Grove area | 收割者之路、德鲁伊的困境 / Reaper's Path, Druid's Plight |

## 队友搭配建议 / Party Composition

4人队伍的核心原则: **一个前排坦克 + 一个治疗/辅助 + 两个输出** / Core principle: 1 tank + 1 healer/support + 2 DPS.

| 位置 / Role | 推荐队友 / Recommended Companions | 职业 / Class |
|---|---|---|
| **坦克 / Tank** | 莱埃泽尔(Lae'zel) / 卡拉克(Karlach) | 战士 / 野蛮人 |
| **治疗 / Healer** | 影心(Shadowheart) / 哈尔辛(Halsin) | 牧师 / 德鲁伊 |
| **法系输出 / Magic DPS** | 盖尔(Gale) / 威尔(Wyll) | 法师 / 邪术师 |
| **物理输出 / Physical DPS** | 阿斯代伦(Astarion) / 卡拉克(Karlach) | 游荡者 / 野蛮人 |

**最佳新手队**: 莱埃泽尔(战士坦)+影心(牧师奶)+盖尔(法师AOE)+主角(自由职业) / **Best beginner party**: Lae'zel (tank) + Shadowheart (healer) + Gale (AoE mage) + player (flexible).

## 战斗技巧 / Combat Tips

1. **高地优势 / High Ground**: 高处射击有攻击优势(Advantage) + 伤害加成 / High ground = Attack Advantage + damage bonus
2. **元素联动 / Elemental Combos**: 水面+闪电伤害x2、油面+火焰爆炸、毒云+火焰爆炸 / Water + lightning = 2x, grease + fire = explosion, poison cloud + fire = explosion
3. **推落入坑 / Shove into Chasm**: 把敌人推下悬崖/深坑可以即死(但会丢失掉落物) / Shove enemies off cliffs = instant kill (but lose loot)
4. **药水投掷 / Throw Potions**: 治疗药水扔到队友脚边可以范围回血 / Throw healing potions at teammates' feet for AoE heal
5. **卷轴不限职业 / Scrolls Don't Need Class**: 任何职业都可以使用任何卷轴——全员带几发火球术卷轴 / Any class can use any scroll — everyone carries Fireball scrolls
6. **祝福术是神技 / Bless is OP**: 1环法术——攻击和豁免+1d4——全队最值得常驻的Buff / 1st level spell — +1d4 to attacks and saves — best party buff
7. **加速术 / Haste**: 3环法术——目标每回合多一个动作+额外移动——战斗核心Buff / 3rd level — +1 action + extra movement per turn — core combat buff
8. **隐形术潜行 / Invisibility + Stealth**: 战斗外放隐形术→跑到敌人背后→开战获得突袭轮(免费一回合) / Pre-combat invis → position behind enemies → surprise round (free turn)
9. **敌人先手控制 / Control > Damage**: 恐惧术、沉睡术、塔莎狂笑术——控制敌人比直接输出更高效。被控制的敌人不还手 / Fear, Sleep, Tasha's — control is more efficient than raw damage. Controlled enemies don't fight back
10. **水瓶战术 / Water Bottles**: 扔水瓶到敌人身上=潮湿状态→寒冷/闪电伤害翻倍。每个角色背5个水瓶不亏 / Throw water = Wet condition → cold/lightning damage doubled. Carry 5 water bottles per character
