# 博德之门3攻略 / Baldur's Gate 3 Guide

Baldur's Gate 3 is a CRPG masterpiece by Larian Studios, based on D&D 5e rules. Play as a mind flayer tadpole-infected character, embark on an epic adventure across the Sword Coast, and decide the fate of Faerun. ## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **先确定职业再选种族 / Class Before Race** | 种族提供的基础属性加值影响Build——前期游戏最多人用的组合是邪术师/圣武士/游侠 / Race stat bonuses matter most early — most popular: Warlock/Paladin/Ranger |
| **2** | **长休频繁 / Long Rest Often** | 长休恢复法术位和HP——且推进队友剧情。不要舍不得补给品 / Restores spell slots and HP — also advances companion story. Don't hoard supplies |
| **3** | **活用戏法 / Use Cantrips** | 戏法不耗法术位——法师之手、电爪、魔法伎俩在探索中极其有用 / Cantrips cost no slots — Mage Hand, Shocking Grasp, Minor Illusion are invaluable |
| **4** | **卷轴和药水是资源 / Scrolls & Potions Are Resources** | 法师可以抄卷轴学法术——药水可以扔地上制造范围效果 / Wizards learn spells from scrolls — potions can be thrown for AoE effects |
| **5** | **潜行是神技 / Stealth is OP** | 开战前潜行布阵——用刺客/游侠先手偷袭可秒杀小怪 / Set up before combat — Assassin/Ranger surprise round kills |
| **6** | **高魅力主角建议 / High Cha Recommended** | 主控角色的对话掷骰高魅力受益最大——圣武士/邪术师/吟游诗人做主角对话选项最多 / Main character benefits most from high Charisma for dialogue |
| **7** | **保存频繁 / Save Often** | 遭遇重要战斗前F5快速存档——有些掷骰失败会导致永远错过内容 / Quick save (F5) before important encounters — failed rolls can lock content forever |
| **8** | **洗点自由 / Respec Freely** | 营地中的威尔（Withers）可以提供洗点服务——100金一次随意重置角色 / Withers in camp offers respec for 100 gold — unlimited resets |
| **9** | **搬运重物 / Send Heavy Items to Camp** | 右键→发送到营地——不必背着一堆装备影响移动 / Right-click → Send to Camp — avoid encumbrance |
| **10** | **完成所有队友任务 / Complete All Companion Quests** | 队友任务线影响结局、解锁强力装备、获得专属剧情 / Companion quests affect endings, unlock gear, and have exclusive story content |
