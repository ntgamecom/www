## 相位蜘蛛女王 / Phase Spider Matriarch (第一章)

**位置 / Location**: 染疫村庄井底→相位蜘蛛巢穴 / Blighted Village well → Phase Spider cave
**等级建议 / Level**: 4-5

**Boss特性 / Boss Traits**:
- 会召唤小蜘蛛 / Spawns spiderlings
- 可以传送——瞬移攻击 / Can teleport
- 弱火 / Weak to fire

**打法 / Strategy**:
1. **让一个人去踩蜘蛛蛋**——小蜘蛛群会先把蛋当目标 / Have one character trigger spider eggs — spiderlings aggro on them
2. **炸掉蛋**：用火焰法术/油+火把/炸弹摧毁蜘蛛蛋——防止增援 / Destroy eggs with fire — prevent reinforcements
3. **女王会掉下悬崖**：她传送时可能掉到裂缝里——造成坠落伤害 / She can fall into chasms when teleporting
4. **加速术+祝福术**给近战——快速压血 / Haste + Bless on melee — burn her down fast

**掉落 / Loot**: 相位蜘蛛女王毒囊（制作材料）/ Poison gland (crafting)

## 鬼婆艾瑟琳 / Auntie Ethel (第一章)

**位置 / Location**: 鬼婆沼泽/地下巢穴 / Sunlit Wetlands → underground lair
**等级建议 / Level**: 4-5

**Boss特性 / Boss Traits**:
- 会分身——三个鬼婆互相切换 / Creates clones — 3 hags swapping
- 谈判选项——可以用交易结束战斗 / Can negotiate to end combat
- 弱火 / Weak to fire

**打法 / Strategy**:
1. **识破分身**：真鬼婆身上有特殊标记——用魔法飞弹/区域法术打不中的就是真身 / Real hag has specific mark — Magic Missile/AoE reveals true form
2. **打断变身**：鬼婆会变成大型怪物——用反法术/高伤害打断 / Interrupt her transformation with Counterspell/high damage
3. **别杀玛琳娜**：鬼婆会拿人质威胁——用睡眠术催眠人质再杀鬼婆 / Hag threatens Mayrina — use Sleep on her, then kill hag
4. **拿头发**：把鬼婆打到10%HP以下她会求饶——接受可获得鬼婆头发（属性+1）/ Get her below 10% HP — she begs, accept for +1 stat hair

**掉落 / Loot**: 鬼婆头发（属性+1）/ Hag's Hair (+1 stat), 法力杖 / Power Staff

## 格雷姆 / Grym (第一章——幽暗地域)

**位置 / Location**: 精金熔炉 / Adamantine Forge, Underdark
**等级建议 / Level**: 5-6

**特殊机制 / Special Mechanic**: 格雷姆免疫所有伤害——除非被熔炉的锻锤砸中 / Immune to all damage unless hit by the forge hammer.

**打法 / Strategy**:
1. **拉怪到锻锤下**：一个人站在锻锤控制杆旁，其他人引格雷姆到锻锤正下方 / Kite Grym under the hammer
2. **拉杆砸**：格雷姆站到锻锤下时拉杆——造成大量伤害+使其过热/易伤 / Pull lever when Grym is under hammer — massive damage + overheat bonus
3. **过热时输出**：被砸中后格雷姆过热——这时任何伤害都有效 / Grym overheats after hammer hit — all damage works temporarily
4. **重复2-3次**：引怪→拉杆→输出→循环 / Kite → lever → DPS → repeat
5. **注意熔岩**：场地有熔岩——伤害很高 / Arena has lava — high damage, avoid

**掉落 / Loot**: 精金装备（选择做盔甲或盾或武器）/ Adamantine gear (choose armor/shield/weapon)

## 养鸡人审判官 / Inquisitor W'wargaz (第一章)

**位置 / Location**: 养鸡场——星界位面 / Crèche Y'llek — Astral Plane
**等级建议 / Level**: 6-7

**Boss特性 / Boss Traits**:
- 养鸡人精神士兵包围 / Surrounded by Githyanki mind warriors
- 会使用驱魔武器——伤害+沉默 / Uses a weapon of exorcism — damage + silence

**打法 / Strategy**:
1. **开战前先清小怪**：先在外侧清掉大部分养鸡人战士再开Boss / Clear side Githyanki before engaging boss
2. **法术反制/ Counterspell**：养鸡人会施放沉默术——提前反制 / Githyanki cast Silence — counter it
3. **控制优先**：怪物定身术/催眠纹章控制审判官——集火秒杀 / Hold Person/Hypnotic Pattern — focus fire
4. **吉斯洋基银剑**：如果可以缴械——这把剑是传说武器 / If you can disarm him — the Githyanki Silver Sword is legendary

**掉落 / Loot**: 吉斯洋基银剑（传说）/ Githyanki Silver Sword (legendary)

## 凯瑟里克·索姆 / Ketheric Thorm (第二章)

**位置 / Location**: 月光之塔——最终战 / Moonrise Towers — final battle
**等级建议 / Level**: 8-9

**三个阶段 / Three Phases**:

| 阶段 / Phase | HP范围 / Range | 说明 / Description |
|---|---|---|
| 第一阶段 / Phase 1 | 100%-50% | 月光之塔塔顶——凯瑟里克+追随者 / Moonrise tower top — Ketheric + followers |
| 第二阶段 / Phase 2 | 50%-0% (第一阶段) | 凯瑟里克变成死灵大君——巨大的亡灵形态 / Transforms into Necromancer Lord — giant undead form |
| 第三阶段 / Phase 3 | 独立HP / Separate HP | **暗夜之歌者（如果活着）出现**——配合你打Boss的终极形态 / **Nightsong appears (if alive)** — helps vs final phase |

**核心战术 / Core Tactics**:
1. **第一阶段速杀小怪**：优先清掉追随者——凯瑟里克单打独斗不强 / Kill followers first — Ketheric alone is weak
2. **第二阶段用加速术**：全员加速→集火Boss / Haste whole party → focus fire
3. **暗夜之歌者**：如果你在莎尔试炼场饶了暗夜之歌者——她会来帮忙 / If you spared Nightsong — she joins the fight
4. **法术反制**：凯瑟里克会复活亡灵——反制他的复活术 / Counter his revive spells

## 拉斐尔 / Raphael (第三章——可选Boss)

**位置 / Location**: 九狱——希望之邸 / House of Hope — Nine Hells
**等级建议 / Level**: 11-12

**全游戏最难的可选Boss** / Hardest optional boss in the game.

**Boss战特点 / Battle Features**:
- 拉斐尔有 **666 HP** / Has **666 HP** (fitting for a devil)
- 召唤4个立柱——每根给拉斐尔Buff / Summons 4 pillars — each gives Raphael a buff
- 第二阶段全身火焰——近战接触受伤 / Phase 2: body on fire — melee contact damages

**打法 / Strategy**:
1. **先打立柱**：4根立柱必须优先摧毁——否则拉斐尔无敌 / Destroy 4 pillars first — Raphael is buffed while they stand
2. **拉斐尔的Buff**：每根立柱提供：火焰免疫/物理减伤/法术反射/治疗 / Each pillar gives: fire immunity/phys resist/spell reflect/heal
3. **全员火抗**：开战前喝火焰抗性药水 / Everyone drinks fire resistance potion before fight
4. **召唤魔像**：希望之邸里的地狱魔像可以帮你——说服它反水 / Persuade the divine intervention to help you
5. **怪物定身术**：对拉斐尔有效！——让近战输出全自动暴击 / Hold Monster works on Raphael! — melee auto-crits

**掉落 / Loot**: 拉斐尔的宿敌（传说双手锤）/ Raphael's Foe (legendary maul), 地狱护照 / Helldusk Armor

## 奥林 / Orin (第三章)

**位置 / Location**: 巴尔神殿——下水道 / Temple of Bhaal — sewers
**等级建议 / Level**: 11-12

**Boss特性 / Boss Traits**:
- 变身——变成你队伍中的一个人（复制品）/ Shapeshifts — copies a party member
- 不可阻挡——每回合一次无敌 / Unstoppable — 1 charge per turn of immunity
- 高暴击——刺杀型Boss / High crit — assassin-type boss

**打法 / Strategy**:
1. **破不可阻挡**：用魔法飞弹（每颗弹头消耗一层不可阻挡）/ Magic Missile (each missile removes one Unstoppable charge)
2. **识破真身**：奥林变身后名字旁边有标记——找不同 / Find the real Orin — her name has a special marker
3. **不要让她拿到先攻**：奥林的先攻极高——我方有警觉专长的角色先上 / Her initiative is very high — someone with Alert feat goes first
4. **击倒控制**：摔绊/眩晕/怪物定身——让她无法行动 / Trip/stun/Hold Monster — prevent her from acting

**掉落 / Loot**: 巴尔护甲 / Armor of Bhaal, 血腥之刃 / Bloodthirst (legendary dagger)

## 通用Boss战技巧 / General Boss Tips

1. **战前Buff / Pre-buff**: 加速术、祝福术、防护善恶、英雄盛宴——开战前全部上好 / Haste, Bless, Protection from Evil, Heroes' Feast before fight
2. **控制优先 / Control First**: Boss战第一回合不要输出——先上控制（定身/晕眩/催眠）/ First round = control (Hold/Hypnotic/Stun), not damage
3. **高地优势 / High Ground**: 尽可能占领高地——远程+20%命中和伤害 / Take high ground — +20% hit and damage for ranged
4. **动作如潮开怪 / Action Surge Start**: 战士/圣武士开局动作如潮打出爆发 / Fighter/Paladin Action Surge round 1 for burst
5. **加速术 = 更多控制 / Haste = More Control**: 不一定要加给输出——加速术加给控制法师可以多放一个定身术 / Haste your controller for double crowd control
6. **卷轴大军 / Scroll Army**: 全员带加速术/火球术/法术反制卷轴 / Everyone carries Haste/Fireball/Counterspell scrolls
7. **荣誉模式特殊注意 / Honor Mode Note**: Boss有传说动作——提前查Wiki确认行动模式 / Bosses have legendary actions — check their patterns beforehand
