## 任务概述 / Quest Overview

博德之门3有超过100个任务——分布在三个章节中。许多任务有**多重完成方式和分支选择**——不同的选择影响后续任务和结局。本指南列出关键任务及其最佳选择 / BG3 has 100+ quests across 3 acts. Many quests have **multiple completion methods and branching choices** — choices affect later quests and endings. This guide lists key quests and optimal choices.

### 任务类型 / Quest Types

| 类型 / Type | 说明 / Description | 数量 / Count |
|---|---|---|
| 主线任务 / Main Quest | 推动核心剧情——必须完成才能进入下一章 / Advances main story — required for next act | 15+ |
| 同伴任务 / Companion Quest | 每个同伴的个人任务线——影响结局和伙伴关系 / Each companion's personal story — affects ending and relationship | 10 |
| 支线任务 / Side Quest | 可选——提供经验、装备和背景故事 / Optional — XP, gear, lore | 60+ |
| 隐藏任务 / Hidden Quest | 没有正式任务记录——通过探索触发 / No quest log — explore to trigger | 20+ |

## 第一章任务 / Act 1 Quests

### 主线：夺心魔感染 / Main: Mind Flayer Infection

**触发**: 游戏开始——从螺壳舰坠毁后自动触发 / Trigger: Game start — auto-trigger after nautiloid crash
**目标**: 找到治愈方法——前往德鲁伊林地寻求海量的帮助 / Goal: Find a cure — go to Druid Grove for Halsin's help
**完成方式**: 完成德鲁伊林地任务线→拯救海量→前往月出之塔→在塔中找到治疗信息 / Complete Grove questline → rescue Halsin → go to Moonrise Towers → find cure info
**关键分支**: 可以接受夺心魔蝌蚪的力量(获得强力技能但影响结局)或拒绝(更纯净但少一些能力)。建议至少使用1-2个蝌蚪技能——它们非常强力且单一使用不影响结局 / Can accept tadpole powers (stronger skills but affects ending) or refuse (purer but fewer abilities). Recommend using 1-2 tadpole skills — they're very powerful and single use doesn't affect ending.

### 德鲁伊林地 / Druid Grove

**触发**: 抵达德鲁伊林地后自动触发 / Trigger: Arrive at Druid Grove
**核心任务**: **拯救海量**(进入地精营地/击杀三个地精首领→释放海量)/ **明萨拉的袭击**(她和地精计划攻击林地——可以告密、击杀她、或加入她攻击林地) / Core: **Rescue Halsin** (enter goblin camp/kill 3 goblin leaders → free Halsin) / **Minthara's Attack** (she plans to attack Grove — can warn, kill, or join her)
**奖励**: 海量加入营地(后续关键NPC)——经验值约500——大量装备 / Reward: Halsin joins camp (key NPC) — ~500 XP — lots of gear
**重要分支**: 如果帮明萨拉攻击林地——你失去海量、威尔和卡菈克可能离队——但获得明萨拉作为同伴。通常是邪恶线选择 / If you help Minthara attack the Grove — you lose Halsin, Wyll and Karlach may leave — but gain Minthara as companion. Usually evil playthrough choice.

### 地精营地 / Goblin Camp

包含三个地精首领——**明萨拉**(女祭司)、**地精祭司古特**(地精头子)、**拉格兹林**(食人魔首领)——全部击杀后释放海量。也可以通过**地精女祭司的真言戒指**绕过战斗。**推荐**: 逐个击破——明萨拉最难(可击落吊桥让她掉落)、古特对话中可毒杀、拉格兹林可以直接打也可以挑拨他和座狼的关系 / Contains 3 goblin leaders — **Minthara**, **Priestess Gut**, **Ragzlin** — kill all to free Halsin. Can also bypass via **True Soul Ring**. **Recommend**: Eliminate one by one — Minthara hardest (shoot bridge to drop her), Gut poisoned in dialogue, Ragzlin fought directly or pitted against wargs.

### 幽暗地域 / Underdark

**触发**: 德鲁伊林地地下入口或散塔林会藏匿处 / Trigger: Grove underground entrance or Zhentarim hideout
**任务**: 探索幽暗地域——在**蕈人村**、**奥术塔**、**灰矮人营地**等区域探索。核心奖励包括: **奥术塔的奥术法杖**(法师毕业武器之一)、**蕈人王的护甲**(重甲)、**失落的镜子**(属性加成)。幽暗地域有三条通往第二章的路径——蕈人村传送门、船、或经折磨通道 / Quest: Explore Underdark — Myconid Colony, Arcane Tower, Duergar Camp. Key rewards: **Arcane Staff** (mage BiS), **Myconid Armor** (heavy), **Lost Mirror** (stat boost). Three paths to Act 2: Colony portal, boat, or Mountain Pass.

### 养育间 / Githyanki Creche

**触发**: 经过山隘到达修道院 / Trigger: Cross Mountain Pass to Monastery
**任务**: 和养育人对话→检查治疗仪器(搜刮一切但没有实际治疗效果)→在审讯室找到重要信息→和审判官战斗→进入星界棱镜和君主对话 / Quest: Talk to caretaker → use healing machine (loot everything but no cure) → find intel in interrogation → fight Inquisitor → enter Astral Prism to talk to Emperor
**关键物品**: **吉斯洋基传说武器**(需要击败审判官)、**修道院隐藏的传奇重弩"泰坦弦弓"**——在修道院屋顶的鹰巢中 / Key items: Githyanki legendary weapon (defeat Inquisitor), monastery hidden legendary crossbow "Titanstring Bow" — in eagle nest on monastery roof.

### 第一章隐藏任务 / Act 1 Hidden Quests

**救赎战败的骑士**: 在晋升之路找到一个被囚禁的提夫林骑士——救出后可以在第三章作为盟友出现。**鬼婆的巢穴**: 在沼泽中击败鬼婆——可以选择杀她或放她——放她会获得**鬼婆的头发**(永久+1属性点)——极度推荐——这是全游戏唯一永久属性提升的途径之一 / **Rescue Fallen Knight**: Find imprisoned tiefling knight on Rise Road — rescue for Act 3 ally. **Hag's Lair**: Defeat hag in swamp — kill or spare — sparing gives **Hag's Hair** (permanent +1 stat) — highly recommended — one of the only permanent stat boosts.

## 第二章任务 / Act 2 Quests

### 月出之塔 / Moonrise Towers

**触发**: 通过幽暗地域或山隘到达诅咒之地 / Trigger: Reach Shadow-Cursed Lands via Underdark or Mountain Pass
**核心任务**: 潜入月出之塔→找到凯瑟里克·索姆→收集他弱点的信息→前往暗夜之歌→释放暗夜之歌→返回击杀索姆。**关键NPC**: 贾希拉——可以在月出之战中招募。**重要选择**: 暗夜之歌是杀是留——释放她获得强力盟友+击败索姆的必要条件——杀她则获得**暗夜之矛**(强力武器)但失去贾希拉 + 索姆更难击败 / Core: Infiltrate Moonrise → find Ketheric Thorm → gather info on his weakness → go to Nightsong → free Nightsong → return to kill Thorm. **Key NPC**: Jaheira — can recruit after Moonrise battle. **Key Choice**: Spare or kill Nightsong — spare = strong ally + necessary to beat Thorm — kill = **Nightsong Spear** (strong weapon) but lose Jaheira + harder Thorm fight.

### 诅咒之地的商队 / Shadow-Cursed Lands Caravan

**触发**: 和木之面容旅店附近的NPC对话 / Trigger: Talk to NPC near Last Light Inn
**任务**: 营救被诅咒之地暗影困住的商队。商队在三个不同位置——每救一个获得经验和金币。全部救完后商人出现在旅店——售卖稀有装备 / Quest: Rescue caravans trapped in shadow. Caravans at 3 locations — each gives XP + gold. All rescued = merchant appears at Inn selling rare gear.

## 第三章任务 / Act 3 Quests

### 戈塔什的阴谋 / Gortash's Scheme

**触发**: 进入博德之门后自动触发 / Trigger: Auto-trigger after entering Baldur's Gate
**任务**: 戈塔什已经称帝——需要在城市中寻找盟友对抗他的钢铁守卫和刺客。关键任务线: **钢铁王座**(潜入水下监狱救援盟友)+**钢铁守卫工厂**(破坏钢铁守卫生产)+**戈塔什的加冕**(潜入或强攻他的堡垒)。**可选捷径**: 接受戈塔什的休战协议——他提供情报换取你的不干涉——但最终你还是需要击败他 / Quest: Gortash is Archduke — find allies against his Steel Watch and assassins. Key quests: **Iron Throne** (underwater prison rescue) + **Steel Watch Foundry** (destroy production) + **Gortash's Coronation** (infiltrate or assault his fortress). **Shortcut**: Accept Gortash's truce — he gives intel for non-interference — but you still need to defeat him eventually.

### 最终之战 / Final Battle

三条主线——**戈塔什线**、**奥林线**和**脑线**——都完成后方可进入最终决战。最终战分为三个阶段——城市防守战→脑内突入战→最终脑战。**队友存活**取决于之前的任务选择——有任务的队友在最终战有特殊表现 / Three main questlines — **Gortash**, **Orin**, and **Brain** — all must be completed for final battle. Three phases: City defense → Brain invasion → Final brain fight. **Companion survival** depends on quest choices — quest-completed companions have special moments in final battle.
