# 职业与Build / Classes & Builds

## 总览 / Overview

博德之门3有 **12个基础职业**，等级上限12级。兼职（Multiclass）是构建强力角色的核心——合理兼职比纯职更强 / 12 base classes, max level 12. Multiclassing is key — good multi is stronger than pure.

| Tier | 职业 / Class |
|---|---|
| S | 圣武士 / Paladin, 邪术师 / Warlock, 吟游诗人 / Bard |
| A | 战士 / Fighter, 游侠 / Ranger, 牧师 / Cleric, 法师 / Wizard |
| B | 游荡者 / Rogue, 野蛮人 / Barbarian, 武僧 / Monk, 德鲁伊 / Druid |
| C | 术士 / Sorcerer (后期强) |

---

## ⚔️ 圣武士 / Paladin — S级

**核心定位 / Role**: 近战爆发/坦克/光环 / Melee burst / Tank / Aura

**推荐子职 / Best Subclass**:
- 复仇之誓 / Oath of Vengeance — 最泛用 / Most versatile
- 古贤之誓 / Oath of the Ancients — 控制+治疗 / Control + heal
- 破誓者 / Oathbreaker — 暗黑输出最高 / Highest dark DPS

**核心法术 / Core Spells**:
- 至圣斩 / Smite (消耗法术位附加额外光耀伤害) — 圣武士核心输出 / Smite = bonus radiant damage — core ability
- 命令术 / Command: 跪下 / Approach / 非常强的控制 / Very strong control

**核心专长 / Feats**:
- 巨武器大师 / Great Weapon Master (GWM): -5命中+10伤害 / -5 hit +10 damage
- 凶蛮打手 / Savage Attacker: 掷骰伤害取两次最高 / Roll damage twice, take higher

**推荐Build / Top Build**: 圣武士2级 + 邪术师5级 (契约刃) + 战士5级 / Paladin 2 + Warlock 5 (Pact Blade) + Fighter 5

| 等级 / Level | 职业 / Class | 解锁 / Unlocks |
|---|---|---|
| 1-2 | 圣武士 / Paladin | 至圣斩、战斗风格 / Smite, Fighting Style |
| 3-7 | 邪术师 / Warlock | 契约刃——攻击用魅力 / Pact Blade — attacks with Charisma |
| 8-12 | 战士 / Fighter | 动作如潮、子职、额外专长 / Action Surge, subclass, extra feat |

**效果 / Result**: 每回合3次攻击 + 至圣斩爆发 + 全魅力加点（对话无敌）/ 3 attacks/round + Smite burst + full Cha (godly dialogue)

## 🎸 吟游诗人 / Bard — S级

**核心定位 / Role**: 全能辅助/施法者/技能检定 / Full support/caster/skill monkey

**推荐子职 / Best Subclass**:
- 剑舞学院 / College of Swords — 近战+施法混合 / Melee + caster hybrid
- 逸闻学院 / College of Lore — 技能精通+更多魔法秘密 / Skill monkey + Magical Secrets

**核心法术 / Core Spells**:
- 治愈真言 / Healing Word (附赠动作回血) / Bonus action heal
- 加速术 / Haste (3环——让队友多打一回合) / 3rd level — ally gets +1 action
- 怪物定身术 / Hold Monster (对Boss用——重击必中) / Hold = auto-crit on melee

**推荐Build / Top Build**: 吟游诗人10级 + 战士2级 / Bard 10 + Fighter 2

| 等级 / Level | 职业 / Class | 解锁 / Unlocks |
|---|---|---|
| 1-10 | 吟游诗人 / Bard | 魔法秘密——偷法师法术 / Magical Secrets — steal wizard spells |
| 11-12 | 战士 / Fighter | 动作如潮——一回合两个加速术 / Action Surge — two Hastes in one turn |

**效果 / Result**: 最强辅助——能加血能Buff能输出能对话 / Best support — heals + buffs + DPS + dialogue

- 冠军勇士 / Champion — 暴击翻倍——配合装备 / Double crit range — with gear

**核心技能 / Core Skills**:
- 动作如潮 / Action Surge: 额外一个整轮动作——爆发核心 / Extra full action — core burst
- 战技 / Maneuvers: 摔绊/卸武/精准攻击 / Trip/Disarm/Precision Attack

**推荐Build / Top Build**: 纯职战士12级 / Pure Fighter 12

| 等级 / Level | 解锁 / Unlocks |
|---|---|
| 5 | 额外攻击 / Extra Attack |

**核心定位 / Role**: 远程物理输出/追踪 / Ranged physical DPS / Tracker

**推荐子职 / Best Subclass**:
- 兽王 / Beast Master — 自带宠物（非常强力）/ Pet companion (very strong)

- 荆棘之鞭 / Thorn Whip: 拉近距离 / Pull enemies closer

**推荐Build / Top Build**: 游侠5级 + 游荡者4级 (刺客) + 战士3级 / Ranger 5 + Rogue 4 (Assassin) + Fighter 3

| 等级 / Level | 职业 / Class | 解锁 / Unlocks |

| 6-9 | 游荡者 / Rogue | 暗杀——突袭轮自动暴击 / Assassin — surprise round auto-crit |
| 10-12 | 战士 / Fighter | 动作如潮+战斗大师 / Action Surge + Battle Master |

**效果 / Result**: 第一回合先手6+次攻击——几乎秒杀任何非Boss敌人 / First round: 6+ attacks — deletes any non-boss enemy

## 兼职核心规则 / Multiclassing Rules

| 规则 / Rule | 说明 / Description |
|---|---|
| 属性门槛 / Stat Prerequisites | 兼职需要被兼职职业的主属性≥13 / Need 13+ in the class's primary stat |
| 法术位 / Spell Slots | 全施法者职业等级叠加——扩展法术位表 / Full caster levels stack for spell slots |
| 额外攻击不叠加 / Extra Attack Doesn't Stack | 从不同职业获得的额外攻击不叠加——只有战士11级的3倍攻击例外 / Only Fighter 11's 3 attacks stacks with others |
| 专长 / Feats | 每4个总等级一次专长（不是职业等级）/ Every 4 total levels (not class levels) |
