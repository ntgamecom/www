# 新手入门 / Beginner Guide

荒野大镖客2的世界极其庞大且细节丰富。本章将帮助你快速上手，从操作基础到生存技巧——让你在西部活下去并活得精彩 / Red Dead Redemption 2's world is vast and incredibly detailed. This chapter helps you get started quickly, from control basics to survival tips — keeping you alive and thriving in the West.

## 生存系统 / Survival Systems

### 核心属性 / Core Stats

| 属性 / Stat | 提升方式 / How to Improve | 说明 / Notes |
|------|------|------|
| **生命 / Health** | 运动、战斗、完成挑战 / Exercise, combat, complete challenges | 使用特定烟草可临时提升 / Certain tonics temporarily boost |
| **体力 / Stamina** | 奔跑、游泳、骑马 / Running, swimming, riding | 决定冲刺和特殊动作的持续能力 / Determines sprint and special action duration |
| **死神之眼 / Dead Eye** | 战斗中的精准射击、击杀 / Precise kills in combat | 等级提升延长标记时间/增加自动标记数 / Leveling extends mark time / auto-mark count |

### 体重系统 / Weight System

| 体重状态 / Weight | 效果 / Effect | 如何维持 / How to Maintain |
|------|------|------|
| **过轻 / Underweight** | 体力恢复变慢，但受到的伤害略微减少 / Slower stamina regen, slightly less damage taken | 多吃饭——尤其在营地 / Eat frequently — especially at camp |
| **正常 / Normal** | 平衡 / Balanced | 正常饮食 / Normal diet |
| **超重 / Overweight** | 体力消耗更快，但近战伤害增加 / Stamina drains faster, melee damage increases | 减少进食、增加运动 / Eat less, exercise more |

### 服装温度 / Clothing Temperature

| 服装状态 / Clothing State | 效果 / Effect |
|------|------|
| **适合当前温度 / Suitable** | 无影响 / No effect |
| **太热 / Too Hot** | 体力和生命核心消耗加快 / Stamina and health cores drain faster |
| **太冷 / Too Cold** | 体力和生命核心消耗更快——极端情况下持续掉血 / Stamina and health cores drain much faster — continuous HP loss in extreme cold |

> **建议**：始终在马上备一套厚衣服和一套薄衣服，根据天气切换 / **Tip**: Always carry one warm and one cold outfit on your horse, switch based on weather.

## 荣誉系统 / Honor System

荣誉值影响NPC对你的反应、商店价格、剧情走向和各种奖励 / Honor affects NPC reactions, shop prices, story direction, and various rewards.

### 增加荣誉 / Increase Honor

| 行为 / Action | 荣誉变化 / Change |
|------|:----:|
| 向路人打招呼 / Greet strangers | 小幅增加 / Slight + |
| 帮助陌生人 / Help strangers | 中幅增加 / Moderate + |
| 捐赠营地资金 / Donate to camp | 小幅增加 / Slight + |
| 放走囚犯 / Release prisoners | 中幅增加 / Moderate + |
| 在决斗中不杀对手 / Spare duel opponents | 大幅增加 / Large + |

### 减少荣誉 / Decrease Honor

| 行为 / Action | 荣誉变化 / Change |
|------|:----:|
| 抢劫和偷窃 / Rob and steal | 小幅减少 / Slight − |
| 杀害无辜 / Kill innocents | 大幅减少 / Large − |
| 虐待动物 / Harm animals | 中幅减少 / Moderate − |
| 绑架 / Kidnap | 大幅减少 / Large − |
| 不还债 / Default on debts | 小幅减少 / Slight − |

### 荣誉等级奖励 / Honor Tier Rewards

| 等级 / Tier | 高荣誉奖励 / High Honor | 低荣誉奖励 / Low Honor |
|------|------|------|
| **商店价格 / Shop Prices** | 所有商品打折（最多-50%）/ Discount on all items (up to −50%) | 商品加价（最多+50%）/ Items marked up (up to +50%) |
| **剧情影响 / Story Impact** | 特定结局可解锁 / Certain endings unlockable | 不同的对话选项和结局 / Different dialogue options and ending |
| **特殊服装 / Special Outfits** | 高荣誉专属服装 / High-honor exclusive outfits | 低荣誉专属服装 / Low-honor exclusive outfits |
| **NPC态度 / NPC Attitude** | 更友善，更多帮助 / Friendlier, more helpful | 更敌意，警戒范围扩大 / More hostile, larger detection range |

> **建议**：一周目建议保持高荣誉——剧情体验更好且经济上更划算 / **Tip**: Recommended to maintain high honor on first playthrough — better story experience and more economical.

## 新手十大建议 / Top 10 Beginner Tips

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **先升级营地 / Upgrade Camp First** | 亚瑟的帐篷升级解锁快速旅行。募捐箱升级解锁更多帮派技能 / Arthur's tent upgrade unlocks fast travel. Ledger upgrades unlock gang perks |
| **2** | **尽早做狩猎请求 / Start Hunting Early** | 完美皮毛需要特定武器——用***春田步枪***打大型动物，***小动物弓***打小型猎物 / Perfect pelts need specific weapons — Springfield Rifle for large game, Varmint Rifle for small |
| **3** | **不要急着推进主线 / Don't Rush the Story** | RDR2的开放世界是真正的亮点——花时间探索、打猎、钓鱼、偶遇 / The open world is the real star — take time to explore, hunt, fish, and encounter |
| **4** | **为马备好药 / Stock Horse Medicine** | 马匹是你在西部最重要的伙伴——马死了就永久没了 / Your horse is your most important partner — if it dies, it's gone forever |
| **5** | **利用死神之眼 / Master Dead Eye** | 死神之眼是战斗中的核心机制——升级后可以手动标记多个目标 / Dead Eye is the core combat mechanic — at higher levels, manually mark multiple targets |
| **6** | **随时保留现金储备 / Keep Cash Reserves** | 某些剧情需要花钱、赏金需要交——留至少$200应急 / Some story moments require money, bounties need paying — keep at least $200 |
| **7** | **和帮派成员互动 / Interact with Camp** | 与每个帮派成员对话、参加营地活动——大量的角色发展和支线任务 / Talk to everyone, join camp activities — massive character development and side content |
| **8** | **注意通缉系统 / Watch the Wanted System** | 犯罪会被目击→报告→通缉升级。戴上面罩可以减缓但不会完全阻止 / Crimes are witnessed → reported → wanted level increases. Masks help but don't fully prevent |
| **9** | **收集香烟卡 / Collect Cigarette Cards** | 全144张香烟卡收集完成后邮寄可获得奖励和成就 / Collect all 144 cigarette cards and mail them for rewards and trophies |
| **10** | **自己去体验 / Experience It Yourself** | RDR2最好的玩法是沉浸——少看攻略，多感受这个世界 / RDR2 is best played immersed — experience the world, don't optimize the fun out of it |
