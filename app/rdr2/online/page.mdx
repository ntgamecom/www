# 线上模式 / Red Dead Online

Red Dead Online是荒野大镖客2的独立多人模式。在这片广阔的西部世界中创建你自己的角色，选择职业，与好友一起冒险或对抗 / Red Dead Online is RDR2's standalone multiplayer mode. Create your own character, choose a role, and embark on adventures — or rivalries — with friends in the vast Wild West.

---

## 线上模式概况 / Online Overview

| 项目 / Feature | 说明 / Description |
|------|------|
| **启动方式 / How to Start** | 从主菜单选择"Red Dead Online"——需要网络连接 / Select "Red Dead Online" from the main menu — requires internet |
| **角色创建 / Character Creation** | 自定义你的男性/女性西部角色——外貌、服装、身材均可定制 / Customize your male/female Western character — appearance, clothing, physique |
| **世界共享 / Shared World** | 每个战局最多32名玩家 / Up to 32 players per session |
| **经济系统 / Economy** | 美元（$）+ 金币（Gold Bars）双货币体系 / Dual currency: Dollars ($) + Gold Bars |
| **等级系统 / Rank System** | 最高1000级——每级解锁新内容 / Max rank 1000 — new unlocks per rank |
| **GTA+ / GTA+会员** | 包含RDO福利（如果订阅）/ Includes RDO benefits (if subscribed) |

## 职业系统 / Role System

花费金币解锁职业——每个职业都有专属的等级、技能和玩法 / Unlock roles with Gold Bars — each has unique ranks, skills, and gameplay.

### 赏金猎人 / Bounty Hunter

| 信息 / Info | 说明 / Description |
|------|------|
| **解锁成本 / Cost** | 15金币 / 15 Gold |
| **核心玩法 / Core Gameplay** | 在城镇公告板接取赏金任务→追踪目标→活捉/击杀目标换奖金 / Take bounty posters at town boards → track targets → capture/kill for reward |
| **特色奖励 / Key Rewards** | 套索升级、赏金马车、特别服装 / Reinforced Lasso, Bounty Wagon, exclusive outfits |

| 等级解锁亮点 / Level Highlights | 效果 / Effect |
|:----:|------|
| **5级** | 加强型套索（更难挣脱）/ Reinforced Lasso (harder to escape) |
| **10级** | 赏金猎人马车（装运多个目标）/ Bounty Wagon (transport multiple targets) |
| **20级** | 专属服装 + 特殊技能 / Exclusive outfit + special skill |

## PvP 提示 / PvP Tips

| 提示 / Tip | 说明 / Description |
|------|------|
| **优先升级死神之眼 / Prioritize Dead Eye** | 线上模式的死神之眼技能卡是PvP的核心——解锁和升级技能卡至关重要 / Dead Eye ability cards are the core of PvP — unlocking and upgrading them is essential |
| **技能卡搭配 / Ability Card Builds** | 推荐新手搭配：「稳健之锚」（减少移动时精度损失）+「以眼还眼」（爆头恢复死神之眼）/ Beginner build: "Stability Anchor" + "Eye for an Eye" |
| **使用掩体 / Use Cover** | 和GTA不同，RDR Online的TTK（击杀时间）较短——掩体战是关键 / Unlike GTA, RDR Online has a shorter TTK — cover-based combat is key |
| **武器选择 / Weapon Choice** | 兰卡斯特连发步枪（万金油）+ 泵动式霰弹枪（近战）+ 捕猎步枪（远程）/ Lancaster Repeater (jack of all trades) + Pump Shotgun (close) + Rolling Block Rifle (long) |
| **马匹选择 / Horse Choice** | 速度与体力同样重要。推荐：密苏里狐步马 / Speed and stamina matter equally. Recommended: Missouri Fox Trotter |
| **防禦模式 / Defensive Mode** | 如果不想参与PvP，在自由模式菜单开启「防御模式」减少被恶意攻击的概率 / If not interested in PvP, enable "Defensive Mode" from the free roam menu to reduce griefing |

## 新手入门建议 / Beginner Tips for Online

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **先做主线 / Do Story Missions First** | Online有约20个主线任务，首次完成奖励大量金币和经验 / Online has ~20 story missions — first completions give hefty gold and XP |
| **2** | **用双倍奖励活动 / Play Bonus Events** | Rockstar定期推出双倍金币/经验活动——抓住这些窗口期 / Rockstar regularly runs double gold/XP events — take advantage |
| **3** | **存钱买第一个职业 / Save for First Role** | 收集所有金币买第一个职业——推荐**赏金猎人**（给金币+$）/ Save all gold for your first role — recommended: **Bounty Hunter** (pays gold + $) |
| **4** | **加入一个团队 / Join a Posse** | 组队任务收益更高，安全更有保障 / Posse missions pay more and offer safety in numbers |
| **5** | **创建永久营地 / Set Up Persistent Camp** | 在营地可以烹饪、制作、使用储物箱——是Online的家 / Camp is your Online home — cook, craft, store items |
| **6** | **不要买不需要的东西 / Don't Waste Money** | 前期重点是买职业和升级装备——服装和外观可以后面再买 / Early focus: roles and gear upgrades — cosmetics can wait |
| **7** | **完成每日挑战 / Do Dailies** | 每日挑战是累计金币的最好方式——坚持每天完成几个 / Daily challenges are the best steady gold source — do a few every day |
