| 服装属性 / Property | 效果 / Effect |

| 凉爽值 / Cooling | 炎热地区需要高散热 / High needed in hot regions |
| 风格 / Style | 影响镇上的声望和互动 / Affects town reputation |
| 伪装 / Camouflage | 更好的伪装——潜行加成 / Better camo — stealth bonus |

**服装套装 / Outfit Sets**:
