# 狩猎与制作 / Hunting & Crafting

狩猎是RDR2中最重要也最有趣的生存活动之一。完美皮毛是制作升级装备的关键，从营地升级到传说背包，每一件都需要精心狩猎 / Hunting is one of the most important and rewarding activities in RDR2. Perfect pelts are essential for crafting upgrades — from camp improvements to the Legendary Satchel, every piece requires careful hunting.

---

## 狩猎基础 / Hunting Basics

### 完美皮毛条件 / Perfect Pelt Requirements

| 条件 / Condition | 说明 / Description |
|------|------|
| **动物品质 / Animal Quality** | 必须是**三星完美品质**的动物。用望远镜观察动物状态（一星/二星/三星）/ Must be **3-star perfect quality**. Study animals with binoculars to see their star rating |
| **武器正确 / Correct Weapon** | 不同的动物需要不同类型的武器才能获得完美皮毛——用错武器会降级为二星或一星 / Different animals require different weapon types — wrong weapon downgrades the pelt |
| **一击必杀 / One-Shot Kill** | 命中要害（通常是头部）一击必杀。多枪射击会损伤皮毛 / Hit the vital area (usually the head) for a one-shot kill. Multiple shots damage the pelt |

### 武器选择指南 / Weapon Selection Guide

| 动物类型 / Animal Type | 推荐武器 / Recommended Weapon | 弹药 / Ammo |
|------|------|------|
| **小型动物（鼠、兔、鸟）/ Small (rats, rabbits, birds)** | **小动物弓 / Varmint Rifle** | 小动物弹药 / Varmint ammo |
| **中型动物（狐、山猫、獾）/ Medium (fox, bobcat, badger)** | **连发步枪 / Repeater** | 连发步枪弹药/中空弹 / Repeater ammo/Express |
| **大型动物（鹿、狼、美洲狮）/ Large (deer, wolf, cougar)** | **春田步枪 / Springfield Rifle** 或 **捕猎步枪 / Rolling Block Rifle** | 步枪弹药/中空弹 / Rifle ammo/Express |
| **巨型动物（熊、驼鹿、牛）/ Massive (bear, moose, bison)** | **捕猎步枪 / Rolling Block Rifle** 或 **泵动式霰弹枪 / Pump Shotgun**（独头弹/ Slug） | 高速弹药/独头弹 / High Velocity/Slugs |
| **传奇动物 / Legendary Animals** | **任何强力步枪**——传奇动物不受完美皮毛限制 / **Any powerful rifle** — Legendary pelts aren't graded |

> **建议**：在马上永远备着**小动物弓**和**春田步枪**——覆盖所有常见猎物 / **Tip**: Always carry **Varmint Rifle** + **Springfield Rifle** on your horse — covers all common prey.

### 传说动物狩猎技巧 / Legendary Hunting Tips

| 技巧 / Tip | 说明 / Description |
|------|------|
| **完成"猎熊传说"任务 / Complete 'The Bear Hunter'** | 这是解锁传说动物狩猎的第一步——在瓦伦丁火车站与一位猎人对话触发 / First step to unlocking Legendary hunting — talk to a hunter at Valentine's train station |
| **使用鹰眼追踪 / Use Eagle Eye** | 传说动物的足迹是**金色**的，和普通足迹不同——跟着金色足迹找到动物 / Legendary tracks are **golden** — follow the golden trail |
| **准备大量弹药 / Bring Plenty of Ammo** | 传说动物比其他同类生物皮厚得多——需要多次命中 / Legendary animals are much tougher than their normal counterparts — multiple hits required |
| **加工皮毛 / Process the Pelt** | 传说动物皮**不能卖给普通商人**——必须送到**捕兽人**处加工 / Legendary pelts **can't be sold to regular merchants** — must be processed by the **Trapper** |
| **放弃也没关系 / It's OK to Lose It** | 如果死亡丢失了传说动物皮——几天后动物会重新刷新 / If you die and lose the pelt — the animal will respawn after a few in-game days |

## 钓鱼 / Fishing

### 钓鱼要点 / Fishing Tips

| 要点 / Tip | 说明 / Description |
|------|------|
| **解锁钓鱼 / Unlock Fishing** | 第二章中与杰克一起钓鱼的任务（"一位好人"）/ Fishing with Jack mission in Chapter 2 ("A Fisher of Men") |
| **鱼饵选择 / Bait Selection** | 面包（通用）/ 玉米（草鱼科）/ 奶酪（鲶鱼科）/ 蚯蚓（通用鱼类）/ Bread (universal) / Corn (herbivores) / Cheese (catfish) / Worms (general) |
| **传说鱼 / Legendary Fish** | 全地图有13条传说鱼——需要特殊鱼饵,钓上后邮寄获得奖励 / 13 Legendary Fish across the map — need special lures, mail for rewards |
| **钓鱼赚钱 / Fishing for Money** | 大型鱼（鲟鱼、狗鱼）卖价最高——在圣丹尼斯码头出售 / Large fish (sturgeon, pike) sell for most — sell at the Saint Denis docks |
