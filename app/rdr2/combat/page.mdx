# 战斗系统 / Combat System

荒野大镖客2的战斗系统以真实感著称——枪战具有重量感、弹药种类丰富、死眼系统提供战术优势 / RDR2's combat is renowned for its realism — gunplay has weight, varied ammo types, and Dead Eye provides tactical advantage.

---

## 死眼系统 / Dead Eye

### 死眼等级 / Dead Eye Levels

| 等级 / Level | 解锁方式 / Unlock | 效果 / Effect |
|---|---|---|
| 1 | 自动——第二章初期 / Auto — early Chapter 2 | 瞄准时时间减慢 / Slow motion while aiming |
| 2 | 任务"谁人无罪" / Mission "Who is Not Guilty" | 可手动标记目标——射击后自动锁定 / Manual paint targets — auto lock on fire |
| 3 | 任务"轰炸" / Mission "Bombing" | 可以看到敌人弱点(红色) / See enemy weak spots (red) |
| 4 | 任务"城市之乐" / Mission "Joy of the City" | 标记后目标身上显示命中次数 / Shows hit count after paint |
| 5 | 任务"再见,亲爱的朋友" / Mission "Goodbye, Dear Friend" | 击杀标记目标后自动恢复死眼能量 / Refill on kill |

**死眼使用技巧 / Dead Eye Tips**:
| 技巧 / Technique | 操作方法 / How To |
|---|---|
| 快速标记 / Quick Paint | 鼠标划过目标自动标记——省力 / Sweep mouse across targets |
| 精准标记 / Precision Paint | 逐一点击标记——精确控制弹药 / Click each mark — conserve ammo |
| 单目标多标记 / Multi-Mark Single Target | 在同一目标上标记多次——爆头叠buff / Multiple marks on one target |
| 死眼中断 / Dead Eye Cancel | 按右键解除——节省能量 / Right-click to release early |

**死眼消耗 / Dead Eye Drain**:
- 未标记时: 缓慢消耗 / Slow drain
- 标记时: 每次标记消耗约5% / ~5% per mark
- 射击时: 射击消耗约10%/枪 / ~10% per shot
- 恢复方式: 击杀敌人+服用嚼烟/威士忌 / Kills + chewing tobacco/whiskey

### 武器保养 / Weapon Maintenance

| 状态 / Condition | 效果 / Effect | 修复方法 / Fix |
|---|---|---|
| **完美 / Perfect** | 最佳伤害和精度 / Best dmg and accuracy | 定期清洁 / Regular cleaning |
| **良好 / Good** | 小幅下降 / Slight decrease | 枪油清洁 / Gun oil |
| **脏污 / Dirty** | 明显下降(20-30%) / Notable decrease | 枪油 / Gun oil |
| **极差 / Poor** | 大幅下降(50%+) / Major decrease | 枪油+零件更换 / Oil + parts |
| **损坏 / Broken** | 无法使用 / Unusable | 枪械铺修理 / Gunsmith repair |

**清洁技巧 / Cleaning Tips**:
- 频繁使用的武器——每场战斗后清洁 / Frequently used weapons — clean after each fight
- 随身携带枪油——战前检查 / Carry gun oil — check before battle
- 狙击和稀有武器特别需要保持完美 / Snipers and rare weapons especially need perfect condition

## 马背战斗 / Horseback Combat

### 马战技巧 / Horseback Fighting Tips

| 技巧 / Technique | 操作方法 / How To | 说明 / Description |
|---|---|---|
| **马背射击 / Mounted Shooting** | 骑马时按瞄准键 / Aim while riding | 精度降低——需要高熟练 / Reduced accuracy — need skill |
| **辅助瞄准 / Aim Assist** | 骑马时自动锁定附近敌人 / Auto-lock nearby enemies | 适合新手 / Beginner-friendly |
| **冲刺射击 / Gallop Shooting** | 冲刺中射击——精度极低 / Shoot at full gallop | 高手向——打中很帅 / Advanced |
| **马背换弹 / Mounted Reload** | 骑马时可以换弹——但速度慢 / Reload on horse — slower | 战斗前先装满 / Reload before fight |
| **马匹急停 / Horse Stop** | 急停后精准射击——组合技 / Stop then shoot — combo | 最准确的马战方式 / Most accurate |

**马匹战斗注意事项 / Horse Combat Notes**:
- 马匹受伤会影响移动和控制 / Horse damage affects movement and control
- 马匹死亡是永久性的——除非使用马匹复活药 / Horse death is permanent — unless using Horse Reviver
- 战斗中下马会让马匹自动远离 / Dismounting sends horse to safe distance
- 使用马匹作为移动掩体——但要小心马匹中弹 / Use horse as mobile cover — but watch its health

## 决斗系统 / Duels

### 决斗流程 / Duel Process

| 阶段 / Phase | 操作 / Action | 提示 / Tip |
|---|---|---|
| 接近 / Approach | 按提示键进入决斗 / Press prompt to engage | 不要拖太久——对方会先开枪 / Don't wait too long |
| 对峙 / Standoff | 手放在枪套上等待信号 / Hand on holster | 视角集中在对方身上 / Focus on opponent |
| 拔枪 / Draw | 信号出现后快速按射击键 / Quick-draw on signal | 时机是关键——太快会失误 / Timing is key |
| 标记 / Paint (有死眼时) | 死眼标记目标部位 / Paint target in DE | 打手打掉枪——打头击杀 / Hand = disarm — head = kill |
| 射击 / Fire | 扣动扳机 / Pull trigger | 一锤定音 / One decisive shot |

**决斗要点 / Duel Tips**:
| 对手 / Opponent | 出枪速度 / Draw Speed | 部位建议 / Target |
|---|---|---|
| 普通匪徒 / Common Outlaw | 慢 / Slow | 打手缴械即可 / Shoot hand |
| 赏金猎人头目 / Bounty Leader | 快 / Fast | 打头快速解决 / Headshot |
| 剧情Boss / Story Boss | 非常快 / Very Fast | 预判他的出枪 / Predict draw timing |
