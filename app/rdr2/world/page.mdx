# 世界与探索 / World & Exploration

荒野大镖客2的地图横跨5个虚构的美国州，从白雪皑皑的安巴里诺到炎热干旱的新奥斯汀。每个区域都有独特的生态、城镇和秘密 / RDR2's map spans 5 fictional US states, from snowy Ambarino to arid New Austin. Each region has unique ecosystems, towns, and secrets.

---

## 快速旅行 / Fast Travel

| 方式 / Method | 解锁方式 / Unlock | 限制 / Limitations |
|------|------|------|
| **营地快速旅行 / Camp Fast Travel** | 升级亚瑟的帐篷（$220 + 1皮制工具卷）/ Upgrade Arthur's tent ($220 + 1 Leather Tools) | 仅限营地→已发现的地点 / Camp → discovered locations only |
| **火车 / Train** | 在任何车站购票 / Buy ticket at any station | 只能到沿线车站 / Only station to station |
| **马车 / Stagecoach** | 在马厩/驿站购票 / Buy ticket at stables/coach stops | 城镇之间的有限路线 / Limited routes between towns |
| **电影模式自动骑行 / Cinema Mode Auto-Ride** | 自动可用 / Auto-available | 在地图上设目标点→进入电影模式→马自动前往 / Set waypoint → enter cinema mode → horse rides automatically |

> **建议**：尽快升级亚瑟的帐篷解锁营地快速旅行，这能节省大量跑图时间 / **Tip**: Unlock camp fast travel ASAP by upgrading Arthur's tent — saves enormous travel time.

## 随机事件 / Random Encounters

RDR2的开放世界充满了随机事件——每次经过同一地点都可能触发不同事件 / RDR2's world is packed with random encounters — passing the same spot may trigger different events each time.

| 事件类型 / Event Type | 示例 / Example | 建议 / Tips |
|------|------|------|
| **陌生人求助 / Stranger Needs Help** | 被蛇咬伤的人、被困的旅行者、求救的女士 / Snake-bitten man, stranded traveler, woman in distress | 帮助陌生人通常获得荣誉和奖励 / Helping usually rewards honor + items |
| **伏击 / Ambushes** | 路上被帮派伏击、被劫匪拦路 / Ambushed by gangs or road bandits | 保持警惕——注意路边的尸体是陷阱的信号 / Stay alert — roadside bodies are often trap signals |
| **囚犯/囚车 / Prisoners/Wagons** | 押运中的囚犯、被绑架的人 / Prisoner convoys, kidnap victims | 救出囚犯获得荣誉+可能的奖励 / Rescue for honor + possible reward |
| **野生动物袭击 / Animal Attacks** | 熊/狼/美洲狮袭击路人 / Bear/wolf/cougar attacking someone | 救了人后可能成为朋友——在城镇遇到会送你礼物 / May become a friend and gift you later |
| **随机场景 / Random Scenes** | 决斗、葬礼、婚礼、酒鬼闹事 / Duel, funeral, wedding, drunk brawler | 这只是世界的日常——沉浸即可 / Part of the living world — immerse yourself |

## 探索要点 / Exploration Tips

| 建议 / Tip | 说明 / Description |
|------|------|
| **骑马走小路 / Ride Off-Road** | 主路上跑会错过大量随机事件和隐藏地点。多走小路和野外 / Main roads miss most random events and hidden locations. Explore off the beaten path |
| **夜晚与白天不同 / Night vs Day** | 很多动物和事件只在特定时间出现——例如美洲狮主要在夜间活动 / Many animals and events only appear at specific times — cougars are mostly nocturnal |
| **留意声音和烟 / Watch for Sounds and Smoke** | 远处的枪声、烟柱通常意味着随机事件或隐藏地点 / Gunshots and smoke columns in the distance often indicate events or hidden locations |
| **地图标记 / Map Marking** | 游戏不提供收藏品全图标——善用自定义标记 / The game doesn't show all collectibles on the map — use custom markers liberally |
| **每片区域都值得探索 / Every Area Rewards Exploration** | 每个小屋、每个洞穴、每栋废弃建筑都可能藏着金钱、香烟卡或独特物品 / Every shack, cave, or ruined building may hold money, cigarette cards, or unique items |
