### 平克顿侦探 / Pinkerton Detectives

平克顿是贯穿全剧的敌对势力——战力极强 / Pinkertons are the main antagonistic force — extremely capable.

| 要点 / Point | 说明 / Description |
|---|---|
| 武器 / Weapons | 全自动手枪+步枪 / Semi-auto pistols + rifles |
| 战术 / Tactics | 包抄——压制射击——不给你呼吸时间 / Flanking — suppressing fire |
| 数量 / Numbers | 通常5-10人一组 / Usually 5-10 per group |
| 反应速度 / Response | 极快——一旦被看到就呼叫增援 / Very fast — call reinforcements once spotted |

**对抗平克顿策略 / Pinkerton Counter-Strategy**:
1. 先手至关重要——先发制人 / First strike is critical — pre-empt them
2. 使用死眼快速击杀多人——减少反击火力 / Use Dead Eye for multi-kills
3. 爆炸弹/炸药对他们特别有效 / Explosive rounds/dynamite are very effective
4. 不要在一个位置停留超过3秒 / Don't stay in one spot over 3 seconds
5. 多使用迂回包抄——不要正面硬扛 / Flank them — don't face them head-on

### 迈卡·贝尔 / Micah Bell

| 项目 / Aspect | 说明 / Description |
|---|---|

**搏斗技巧 / Brawl Tips**:
