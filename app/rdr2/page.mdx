# 荒野大镖客2 攻略 / Red Dead Redemption 2 Guide

荒野大镖客2（Red Dead Redemption 2）是Rockstar Games开发的开放世界西部动作冒险游戏，是《荒野大镖客》系列的前传。扮演范德林德帮的成员亚瑟·摩根，在1899年美国西部逐渐消逝的时代中生存、挣扎和抉择 / Red Dead Redemption 2 is an open-world Western action-adventure game by Rockstar Games, a prequel in the Red Dead series. Play as Arthur Morgan, an outlaw in the Van der Linde gang, surviving, struggling, and making choices in the dying days of the American Wild West in 1899.

**平台 / Platforms**: PS4, PS5, Xbox One, Xbox Series X|S, PC | **发售日 / Release**: 2018年10月26日 / Oct 26, 2018 | **类型 / Genre**: 开放世界动作冒险 / Open-World Action Adventure

## 关键特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **无与伦比的开放世界 / Unmatched Open World** | 细节密度空前的西部世界——每个NPC都有独立的生活作息 / A Western world with unprecedented detail density — every NPC lives their own life |
| **沉浸式生存系统 / Immersive Survival** | 需要管理体力、生命、死神之眼、体重、服装温度等 / Manage stamina, health, Dead Eye, weight, clothing temperature, and more |
| **深度荣誉系统 / Deep Honor System** | 你的每一个选择——从打招呼到抢劫——都影响游戏世界的反应 / Every choice — from greeting to robbing — affects the world's response to you |
| **传奇般的叙事 / Legendary Storytelling** | 亚瑟·摩根的救赎之旅被广泛认为是最伟大的游戏故事之一 / Arthur Morgan's redemption arc is widely considered one of gaming's greatest stories |
| **丰富的野外活动 / Rich Outdoor Activities** | 狩猎、钓鱼、骑马、抢劫、扑克、决斗——西部的每一面 / Hunting, fishing, riding, robbing, poker, dueling — every facet of the West |
| **电影级画面 / Cinematic Visuals** | 动态天气、昼夜循环、极致的光影效果——至今仍是画面标杆 / Dynamic weather, day/night cycle, state-of-the-art lighting — still a visual benchmark |
