## 战斗系统 / Combat System

《空洞骑士》的战斗系统看似简单，实则有着非常深的深度。核心由骨钉攻击、灵魂法术、护符搭配和移动技巧四部分组成。掌握这些元素之间的平衡是通关的关键。

Hollow Knight's combat system seems simple on the surface but has considerable depth. The core consists of four elements: Nail attacks, Soul spells, Charm builds, and movement techniques. Balancing these elements is key to clearing the game.

## 骨钉机制 / Nail Mechanics

### 基础攻击 / Basic Attack

骨钉(Nail)是小骑士的主要武器。普通攻击会对前方约一个半身位的范围造成伤害。

The Nail is the little knight's primary weapon. A normal attack deals damage in a range of about one and a half body lengths in front of you.

| 攻击类型 / Attack Type | 伤害 (初始) / Damage (Base) | 说明 / Description |
|---|---|---|
| 普通斩击 / Normal Slash | 5 | 标准地面攻击 / Standard ground attack |
| 空中斩击 / Aerial Slash | 5 | 空中攻击，方向固定 / Aerial attack, fixed direction |
| 下劈 / Downward Slash | 5 | 可弹跳(钉跳) / Enables pogo bounce |
| 蓄力斩 / Great Slash | 15 | 需要蓄力技能 / Requires charging skill |

The Nail can be upgraded at the Nailsmith using Pale Ore, in 4 tiers:

|---|---|---|---|

| 1 | 尖锐骨钉 / Sharpened Nail | 9 | 1 |
| 2 | 钢铁骨钉 / Channeled Nail | 13 | 2 |

### 下劈与钉跳 / Downward Slash and Nail Pogo

下劈(Downward Slash)是游戏中最重要的高阶技巧之一。在空中按下攻击键的同时按向下方向键即可触发。击中敌人或某些物体后，小骑士会向上弹起，实现连续跳跃。

Downward Slash is one of the most important advanced techniques. Trigger it by pressing the attack button and down direction simultaneously while airborne. Hitting an enemy or certain objects bounces the knight upward, enabling continuous jumping.

**钉跳应用场景 / Pogo Applications**:

- 跨越大型深坑 / Crossing large pits
- 穿越尖刺陷阱 / Traversing spike traps
- 在敌人头上连续攻击以保持滞空 / Sustaining aerial combat above enemies
- 到达需要连续跳跃才能到达的高台 / Reaching high platforms requiring continuous jumps

**练习建议 / Practice Tips**:

- 在遗忘十字路用 Crawlid 练习 / Practice on Crawlids in Forgotten Crossroads
- 尝试在尖刺之间连续钉跳 / Try continuous pogo between spikes
- 钉跳节奏比速度更重要 / Rhythm is more important than speed

### 蓄力斩技 / Nail Arts

获得骨钉大师(Master Nail Art)技能后可以学会三种蓄力斩：

After acquiring Nail Art skills, you can learn three charged attacks:

| 技能 / Art | 操作方法 / Input | 效果 / Effect |
|---|---|---|
| 蓄力斩 / Great Slash | 按住攻击后释放 / Hold attack then release | 横向范围大斩击 / Wide horizontal slash |
| 旋风斩 / Cyclone Slash | 按住攻击并旋转 / Hold attack and rotate | 全方位旋转攻击 / Omni-directional spin |
| 冲刺斩 / Dash Slash | 按住攻击并冲刺 / Hold attack and dash | 快速前冲斩击 / Fast forward lunge |

## 法术系统 / Spell System

### 基础法术 / Basic Spells

法术消耗灵魂能量(Soul)释放。灵魂通过攻击敌人积累。

Spells consume Soul energy to cast. Soul is accumulated by attacking enemies.

| 法术 / Spell | 灵魂消耗 / Soul Cost | 效果 / Effect |
|---|---|---|
| 复仇之魂 / Vengeful Spirit | 33 | 向前发射一道灵魂弹幕 / Fires a soul projectile forward |
| 荒芜俯冲 / Desolate Dive | 33 | 猛击地面，造成范围伤害 / Strikes the ground for area damage |
| 嚎叫幽灵 / Howling Wraiths | 33 | 向斜上方发出三道灵魂波 / Fires three soul waves diagonally upward |

|---|---|---|
| 复仇之魂 / Vengeful Spirit | 暗影之魂 / Shade Soul | 灵魂圣殿 / Soul Sanctum |

### 法术策略 / Spell Strategy

**复仇之魂 / Vengeful Spirit**:
- 贯穿效果，可击中多个敌人 / Piercing effect, can hit multiple enemies
- 射程较长，适合远程消耗 / Long range, good for ranged harassment
- 对部分 Boss 可以连续命中两次 / Can hit some bosses twice

**荒芜俯冲 / Desolate Dive**:
- 释放时有短暂无敌帧 / Brief invincibility frames on cast
- 范围伤害，适合应对群敌 / Area damage, good for crowds
- 可用于快速落地 / Useful for fast landing
- 可以震碎部分地面 / Can break certain floors

**嚎叫幽灵 / Howling Wraiths**:
- 高伤害，对大型敌人效果极佳 / High damage, excellent against large enemies
- 上方范围攻击，上方敌人弱点 / Upward attack, useful against overhead enemies
- 升级后尖啸幽灵伤害极高 / Abyss Shriek deals massive damage

## 灵魂管理 / Soul Management

### 灵魂系统详解 / Soul System Explained

灵魂是战斗中的关键资源，既用于治疗也用于施法。

Soul is the critical resource in combat, used for both healing and spellcasting.

- 每次骨钉击中敌人获得 11 点灵魂 / Each Nail hit grants 11 Soul
- 满灵魂容量为 99 点 (三格) / Full capacity is 99 (three charges)
- 灵魂捕手(Soul Catcher)护符使每次攻击获得 14 点灵魂 / Soul Catcher grants 14 per hit
- 灵魂收割者(Soul Eater)护符使每次攻击获得 22 点灵魂 / Soul Eater grants 22 per hit
- 不战斗时灵魂不会自动恢复 / Soul does not regenerate outside combat

### 治疗策略 / Healing Strategy

聚焦(Focus)可以消耗灵魂恢复生命。在战斗中找准治疗时机是关键。

Focus consumes Soul to heal health. Finding the right moment to heal in combat is crucial.

- 每次聚焦消耗 33 点灵魂恢复 1 格血 / Each Focus consumes 33 Soul to heal 1 mask
- 聚焦需要约 1.5 秒引导时间 / Focus requires about 1.5 seconds of channeling
- 在 Boss 攻击间隙或使用大硬直时治疗 / Heal during boss attack gaps or large stun windows
- 快速聚焦(Quick Focus)护符减少引导时间 / Quick Focus charm reduces channeling time
- 深度聚焦(Deep Focus)护符消耗翻倍但治疗两格血 / Deep Focus doubles cost but heals 2 masks

## 敌人行为模式 / Enemy Behavior Patterns

### 敌人分类 / Enemy Classification

了解敌人类型有助于制定战斗策略：

Understanding enemy types helps formulate combat strategies:

| 类型 / Type | 特点 / Characteristics |
|---|---|
| 近战型 / Melee | 接近后攻击，容易预判 / Attacks on approach, easy to predict |
| 远程型 / Ranged | 发射弹幕，需要走位躲避 / Fires projectiles, requires positioning |
| 飞行型 / Flying | 空中移动，需要空中攻击或法术 / Airborne, needs aerial attacks or spells |
| 防御型 / Defensive | 持有盾牌或装甲，需要破防 / Shields or armor, requires guard break |
| 自爆型 / Explosive | 接近后自爆造成大范围伤害 / Explodes on proximity for area damage |

### 通用战斗原则 / Universal Combat Principles

1. 观察敌人的攻击前摇 / Observe enemy attack wind-ups
2. 利用攻击后摇进行反击 / Use attack recovery windows for counterattacks
3. 不要贪刀——尤其是不熟悉的新敌人 / Do not overcommit — especially against unfamiliar enemies
4. 灵活切换攻击和法术 / Flexibly switch between attacks and spells
5. 地形是你的盟友 / Terrain is your ally
6. 保持移动键持续按下 / Keep movement keys pressed continuously
7. 下劈可以改变战局 / Downward slash can turn the tide of battle
8. 善用冲刺的无敌帧 / Use dash invincibility frames wisely

### 灵魂收集节奏 / Soul Collection Rhythm

优秀的战斗节奏应该是：攻击收集灵魂 → 利用灵魂施法或治疗 → 继续攻击。

A good combat rhythm should be: attack to collect Soul → use Soul to cast or heal → continue attacking.

不要一味进攻也不一味防守。在进攻中积累资源，在需要时合理消耗。

Do not be purely offensive or purely defensive. Accumulate resources through offense, and consume them wisely when needed.

Mastering the combat system takes time and practice. Even the strongest bosses can be handled calmly once you fully understand their movement patterns.
