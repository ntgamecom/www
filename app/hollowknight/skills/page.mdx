## 技能与升级 / Skills and Upgrades

在《空洞骑士》中探索圣巢时，你会发现各种能力提升道具。这些技能不仅让你到达新的区域，还极大地丰富了战斗和移动的多样性。本页面详细介绍所有技能的获取方式和实用技巧。

While exploring Hallownest in Hollow Knight, you will discover various ability upgrades. These skills not only let you reach new areas but also greatly enrich combat and movement diversity. This page details the acquisition methods and practical tips for all skills.

## 蛾翼披风 / Mothwing Cloak

### 获取方式 / Acquisition

**位置 / Location**: 苍绿之径 — 在大黄蜂的第一次战斗后获得 / Greenpath — obtained after first Hornet battle
**前置要求 / Prerequisite**: 击败 Hornet (第一次 / First Encounter)
**难度 / Difficulty**: 中等 — 需要击败大黄蜂 / Medium — requires defeating Hornet

蛾翼披风是小骑士获得的第一个移动能力升级，提供了地面和空中的冲刺(Dash)能力。

The Mothwing Cloak is the first movement upgrade acquired by the little knight, granting dash ability on both ground and air.

### 使用技巧 / Usage Tips

- 冲刺有约 0.4 秒的冷却时间 / Dash has approximately 0.4 second cooldown
- 空中冲刺初始只能使用一次，落地后重置 / Air dash refreshes upon landing
- 冲刺过程中有短暂的无敌帧 / Dash grants brief invincibility frames
- 结合跳跃可以跨越较宽的坑 / Combine with jump to cross wider pits
- 常见的应用：跳跃后冲刺，比地面冲刺距离更远 / Common use: jump then dash for longer distance than ground dash

**进阶技巧 / Advanced Technique**:

冲刺取消(Dash Cancel)：在冲刺动画结束后立即按下攻击键可以取消硬直，使你能更快地进入下一个动作。这个技巧在战斗中非常实用。

Dash Cancel: Pressing the attack button immediately after the dash animation ends cancels the recovery frames, allowing you to enter the next action faster. This technique is very useful in combat.

## 螳螂爪 / Mantis Claw

### 获取方式 / Acquisition

**位置 / Location**: 真菌荒地 — 螳螂村顶部 / Fungal Wastes — Mantis Village top
**前置要求 / Prerequisite**: 到达螳螂村顶部房间 / Reach the top room of Mantis Village
**难度 / Difficulty**: 中等 — 需要清理螳螂区域 / Medium — requires clearing the Mantis area

螳螂爪允许小骑士攀爬墙壁和在某些表面上抓附。

The Mantis Claw allows the little knight to climb walls and cling to certain surfaces.

### 使用技巧 / Usage Tips

- 在墙壁上按方向键可以向上攀爬 / Press directional key toward wall to climb upward
- 攀爬会消耗体力，但不会完全耗尽 / Climbing consumes stamina but does not fully deplete
- 从墙壁上跳下时可以立即再次抓住墙壁 / Can immediately re-grab the wall after jumping off
- 结合冲刺和跳跃可以实现高难度攀爬路线 / Combine with dash and jump for difficult climbing routes
- 某些特定的墙壁颜色暗示可以攀爬 / Certain wall colors indicate climbability

**关键应用 / Key Applications**:

- 到达之前无法到达的高台 / Reach previously inaccessible high platforms
- 在 Boss 战中利用墙壁躲避和定位 / Use walls for dodging and positioning in boss fights
- 在螳螂领主的战斗中，墙壁是重要的移动工具 / Walls are critical movement tools in Mantis Lords battle

## 水晶之心 / Crystal Heart

### 获取方式 / Acquisition

**位置 / Location**: 水晶山峰 / Crystal Peak
**前置要求 / Prerequisite**: 到达水晶山峰顶部 / Reach top of Crystal Peak
**难度 / Difficulty**: 中等 / Medium

水晶之心允许小骑士进行超级冲刺(Super Dash)，水平方向高速直线冲刺，可以穿越极长的距离。

The Crystal Heart allows the little knight to perform a Super Dash — a high-speed horizontal dash that covers extremely long distances.

### 使用技巧 / Usage Tips

- 按住聚焦键蓄力，然后释放进行超级冲刺 / Hold the Focus button to charge, release to Super Dash
- 超级冲刺中无法改变方向 / Cannot change direction during Super Dash
- 撞到墙壁或障碍物会停止 / Stops upon hitting walls or obstacles
- 可以用来冲碎特定墙壁 / Can be used to break specific walls
- 搭配冲刺大师护符可以更快蓄力 / Dashmaster charm speeds up charge time

**关键应用 / Key Applications**:

- 在水晶山峰中穿越长距离平台 / Traverse long platforms in Crystal Peak
- 打开需要超级冲刺才能到达的新路线 / Open new routes only accessible via Super Dash
- 在特定 Boss 战中用于快速拉开距离 / Use in specific boss fights to quickly create distance

## 帝王之翼 / Monarch Wings

### 获取方式 / Acquisition

**位置 / Location**: 古老盆地 / Ancient Basin
**前置要求 / Prerequisite**: 击败破碎容器 / Defeat Broken Vessel
**难度 / Difficulty**: 中等偏难 / Medium-Hard

帝王之翼允许小骑士进行二段跳(Double Jump)，在空中额外跳跃一次。

Monarch Wings allow the little knight to perform a double jump — one additional jump while airborne.

### 使用技巧 / Usage Tips

- 在空中按下跳跃键触发二段跳 / Press jump in mid-air to trigger Double Jump
- 二段跳的高度比第一跳略低 / Second jump is slightly lower than the first
- 二段跳后可以接冲刺 / Can dash after Double Jump
- 这是最重要的探索技能之一——有了它几乎可以去任何地方 / One of the most important exploration skills — enables access to nearly anywhere

**关键应用 / Key Applications**:

- 到达绝大多数高台 / Reach most high platforms
- 在战斗中创造更多的躲避空间 / Create more dodging space in combat
- 穿越深坑区域 / Cross deep pit areas
- 在 Boss 战中更容易躲避地面攻击 / Easier to dodge ground attacks in boss fights

## 伊斯玛的眼泪 / Isma's Tear

### 获取方式 / Acquisition

**位置 / Location**: 皇家水道 / Royal Waterways
**前置要求 / Prerequisite**: 穿过皇家水道 / Navigate Royal Waterways
**难度 / Difficulty**: 中等 / Medium

伊斯玛的眼泪让小骑士免疫酸水(Acid)的伤害，可以在酸水中自由游泳。

Isma's Tear makes the little knight immune to acid damage, allowing free swimming in acid water.

### 使用技巧 / Usage Tips

- 进入酸水后会正常游泳，不损失生命 / Swim in acid without taking damage
- 酸水的某些区域仍有隐藏通道 / Some acid areas conceal hidden passages
- 可以在酸水表面使用下劈攻击 / Can use downward slash on acid surface
- 某些区域需要结合二段跳和冲刺才能到达 / Some areas require combining Double Jump and dash to reach

**关键应用 / Key Applications**:

- 穿越皇家水道和王后花园的酸水区域 / Cross acid areas in Royal Waterways and Queen's Gardens
- 到达雾之峡谷的新区域 / Reach new areas in Fog Canyon
- 发现隐藏的护符和收集品 / Discover hidden charms and collectibles

## 梦之钉 / Dream Nail

### 获取方式 / Acquisition

**位置 / Location**: 遗弃十字路 — 梦境区域 / Forgotten Crossroads — Dream area
**前置要求 / Prerequisite**: 获得梦境精华 x1 / Acquire 1 Dream Essence
**难度 / Difficulty**: 简单 / Easy

梦之钉允许小骑士读取生物的思想并收集梦境精华。它也是推进剧情和解锁某些 Boss 战的关键道具。

The Dream Nail allows the little knight to read the thoughts of creatures and collect Dream Essence. It is also a key item for advancing the story and unlocking certain boss battles.

### 升级 / Upgrades

| 等级 / Level | 前置条件 / Prerequisite | 能力 / Ability |
|---|---|---|
| 基础梦之钉 / Base Dream Nail | 从梦境揭示者处获得 / Obtain from Seer | 读取思想 + 收集 1 精华 / Read thoughts + collect 1 Essence |
| 觉醒梦之钉 / Awakened Dream Nail | 收集 1200 精华 / Collect 1200 Essence | 读取思想 + 收集 3 精华 / Read thoughts + collect 3 Essence |

### 使用技巧 / Usage Tips

- 对 Boss 使用梦之钉可以在战斗前获得额外灵魂 / Use Dream Nail on bosses before battles for extra Soul
- 对 NPC 使用可以了解背景故事 / Use on NPCs to learn backstory
- 对尸体使用有时会触发梦境 Boss 战 / Use on corpses to trigger Dream Boss fights sometimes
- 觉醒后效率大幅提升 / Efficiency greatly improves after awakening

**梦之门 / Dreamgate**:

收集 800 精华后获得的能力。允许在任意位置设定传送点，然后随时传送回来。非常实用的探索工具。

Obtained after collecting 800 Essence. Allows setting a teleport point anywhere and returning to it at any time. A very useful exploration tool.

技能系统是《空洞骑士》探索的核心。每次获得新能力，整个圣巢都会向你敞开新的大门——返回旧区域发现之前无法到达的地方，是游戏最令人满足的体验之一。

The skill system is the core of exploration in Hollow Knight. Each new acquisition opens new doors across Hallownest — returning to old areas to discover previously unreachable places is one of the most satisfying experiences in the game.
