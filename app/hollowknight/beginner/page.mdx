## 新手入门 / Beginner Guide

欢迎来到圣巢！本页面将帮助你从一个初出茅庐的小骑士，成长为能够探索整个王国的勇者。我们将从最基础的控制操作讲起，逐步深入。

Welcome to Hallownest! This page will help you grow from a fledgling little knight into a brave explorer capable of traversing the entire kingdom. We will start with the most basic controls and gradually go deeper.

## 基本控制 / Basic Controls

### 键盘与手柄操作 / Keyboard and Controller Controls

操作是游戏体验的基础。以下是默认的按键映射：

Controls are the foundation of the game experience. Here are the default key mappings:

| 动作 / Action | 键盘 / Keyboard | 手柄 / Controller |
|---|---|---|
| 移动 / Move | WASD / 方向键 | 左摇杆 / 十字键 / Left Stick / D-Pad |
| 攻击 / Attack | Z 或 X / Z or X | A 或 B / A or B |
| 跳跃 / Jump | 空格键 / Space | B 或 A / B or A |
| 冲刺 / Dash | 左 Shift / Left Shift | RT 或 RB / RT or RB |
| 使用法术 / Cast Spell | 数字 1 或 Q / 1 or Q | X 或 Y / X or Y |
| 聚焦 / Focus | 数字 2 或 W / 2 or W | 按住 A / Hold A |
| 梦之钉 / Dream Nail | 数字 3 或 E / 3 or E | 按住 B 或 Y / Hold B or Y |
| 快速施法 / Quick Cast | R | LB |
| 地图 / Map | 数字 4 或 Tab / 4 or Tab | Back 或 Select / Back or Select |
| 互动 / Interact | 空格 / Space | A 或 B / A or B |

建议在设置菜单中根据自己的习惯调整按键。手柄模式下建议启用震动以增强沉浸感。

It is recommended to adjust the key bindings in the settings menu according to your habits. For controller mode, enable vibration to enhance immersion.

### 基础动作解析 / Basic Move Breakdown

**移动 (Movement)**: 使用方向键控制角色左右移动和上下攀爬。移动速度受护符影响。

**跳跃 (Jump)**: 单次跳跃有一定高度。搭配冲刺可以扩大横向移动距离。踩踏敌人或部分物体可以重置跳跃（称为"下劈"）。

**攻击 (Attack)**: 默认骨钉攻击范围为前方约一个半身位。空中攻击方向固定为角色面朝方向。

**冲刺 (Dash)**: 在地面和空中均可使用。初始冲刺距离有限，获得螳螂爪后可在墙壁上冲刺跳跃。

## 遗忘十字路 / Forgotten Crossroads

### 初始区域概况 / Starting Area Overview

遗忘十字路是游戏的第一个区域，也是你学习基础机制的地方。这里的敌人相对较弱，地图相对线性，非常适合新手熟悉操作。

Forgotten Crossroads is the first area of the game, where you will learn the basic mechanics. Enemies here are relatively weak and the map is fairly linear, making it ideal for beginners.

**主要敌人 / Main Enemies**:

| 敌人 / Enemy | 描述 / Description |
|---|---|
| Crawlid | 小型爬虫，会朝你冲刺攻击 / Small bug that charges at you |
| Vengefly | 飞行敌人，会从上方俯冲 / Flying enemy that dives from above |
| Leaping Husk | 跳跃攻击的僵尸 / Zombie that attacks with leaping strikes |
| Husk Guard | 持盾守卫，需要先破防 / Shielded guard that requires breaking guard first |

### 地图购买 / Map Purchase

在遗忘十字路中，你会遇到制图师 Cornifer。他会在每个区域的特定位置等待你，靠近后可以购买地图。地图价格因区域而异，遗忘十字路的地图只需要 30 Geo。

In Forgotten Crossroads, you will encounter the cartographer Cornifer. He waits at a specific spot in each area, and you can purchase maps from him. The map for Forgotten Crossroads costs only 30 Geo.

**地图建议 / Map Tips**:

- 尽快购买地图和羽毛笔(Quill)来标注已探索区域 / Buy the map and Quill as soon as possible to mark explored areas
- 在每个区域都先寻找 Cornifer 的踪迹——他哼歌的声音和飘落的纸张 / Look for Cornifer's trail in each area — his humming and falling papers
- 找到长凳坐下后，地图才会更新你的位置 / Your position updates on the map only after sitting on a bench

## 生存技巧 / Survival Tips

### Geo 管理 / Geo Management

Geo 是游戏中的货币，用于购买物品、升级装备和支付服务。死亡后会在原地留下一个阴影(Shade)，需要在击杀或触碰阴影后才能取回已损失的 Geo。

Geo is the currency in the game, used for purchasing items, upgrading equipment, and paying for services. Upon death, you leave a Shade at your death location, and you must defeat or touch it to recover your lost Geo.

**Geo 获取指南 / Geo Acquisition Guide**:

1. 击败敌人掉落 / Defeat enemies for drops
2. 打破宝箱和易碎物品 / Break chests and fragile objects
3. 挖掘发光土堆 / Dig at glowing dirt mounds
4. 完成商人交易 / Complete merchant transactions

**Geo 保存建议 / Geo Saving Tips**:

- 死亡前尽量花光 Geo / Spend Geo before potentially dying
- 优先购买重要物品(如灯笼、护符槽) / Prioritize important purchases (Lantern, Charm Notches)
- 注意 Geo 掉落物在切换场景后会消失 / Note that dropped Geo disappears upon scene transition

### 长凳的重要性 / Importance of Benches

长凳(Bench)是游戏中的休息点。坐下后：

Benches serve as rest points in the game. Sitting down will:

1. 完全恢复生命值 / Fully restore health
2. 补充灵魂能量 / Refill Soul energy
3. 保存游戏进度 / Save game progress
4. 更新地图位置显示 / Update map position display
5. 重置大多数敌人的位置 / Respawn most enemies

**长凳策略 / Bench Strategy**:

- 在挑战强敌前一定要找到最近的长凳 / Always find the nearest bench before challenging a strong enemy
- 注意有些长凳需要交互才能坐下(而不是自动触发) / Note that some benches require interaction to sit
- 部分区域的长凳非常隐蔽，仔细探索 / Some benches are very hidden — explore carefully

## 早期护符 / Early Charms

### 基础护符推荐 / Basic Charm Recommendations

在游戏早期，护符槽有限，选择合适的护符至关重要。

In the early game, with limited charm notches, choosing the right charms is crucial.

| 护符 / Charm | 槽位 / Notch Cost | 效果 / Effect |
|---|---|---|
| Gathering Swarm / 聚集虫群 | 1 | 自动收集附近的 Geo / Auto-collects nearby Geo |
| Stalwart Shell / 坚硬的甲壳 | 2 | 受伤后无敌时间延长 / Extends invincibility after damage |
| Soul Catcher / 灵魂捕手 | 2 | 每次攻击获得更多灵魂 / Gain more Soul per hit |
| Dashmaster / 冲刺大师 | 2 | 提高冲刺频率和灵活性 / Improves dash frequency and flexibility |

**推荐早期配装 / Recommended Early Build**:

Gathering Swarm + Soul Catcher + 剩下的槽位装备 Stalwart Shell。这个组合提供更好的资源收集和生存能力。

Gathering Swarm + Soul Catcher + remaining notches with Stalwart Shell. This setup provides better resource collection and survivability.

### 护符商人 / Charm Merchants

**Salubra (祝福的萨鲁布拉)**: 位于遗忘十字路，出售基础护符和护符槽。每个护符槽 120-900 Geo 不等。

**Salubra (Salubra the Blessing)**: Located in Forgotten Crossroads, sells basic charms and charm notches. Each notch costs 120-900 Geo.

**Sly (斯莱)**: 位于德里茅斯(Dirtmouth)，最早期的商人。出售一些基础道具和地图工具。

**Sly (Sly)**: Located in Dirtmouth, the earliest merchant. Sells basic items and map tools.

## 实用技巧 / Practical Tips

### 下劈技巧 / Downward Slash Technique

下劈(向下攻击)是游戏中最重要的技巧之一。在空中按住向下方向键同时攻击，可以踩踏敌人、尖刺和部分物体弹跳。这个技巧在获得螳螂爪前是穿越某些区域的必要手段。

Downward slash is one of the most important techniques in the game. While airborne, hold down and attack simultaneously to bounce off enemies, spikes, and some objects. This technique is essential for traversing certain areas before acquiring the Mantis Claw.

**下劈练习 / Downward Slash Practice**:

- 在遗忘十字路的简单敌人上练习 / Practice on simple enemies in Forgotten Crossroads
- 学会用下劈越过尖刺陷阱 / Learn to cross spike traps using downward slashes
- 下劈敌人可以让你在深坑上方保持高度 / Downward slashing enemies keeps you above pits
- 连续下劈称为"钉跳"，是高阶技巧 / Continuous downward slashes are called "nail bouncing", an advanced technique

### 灵魂管理 / Soul Management

灵魂是法术和治疗的共用资源。攻击敌人可以收集灵魂，每个普通攻击获得 11 点灵魂(携带灵魂捕手护符时更多)。

Soul is a shared resource for spells and healing. Attacking enemies collects Soul — each normal attack grants 11 Soul (more with Soul Catcher equipped).

**灵魂使用策略 / Soul Usage Strategy**:

- 每格灵魂需要 33 点以完全聚焦恢复 1 格血 / Each Soul Vessel charge requires 33 points to fully focus and heal 1 mask
- 战斗中灵活决定是用灵魂治疗还是释放法术 / Flexibly decide whether to use Soul for healing or spellcasting in combat
- 不要浪费灵魂——满魂时攻击不会继续积累 / Do not waste Soul — attacks do not accumulate when full
- 王后之魂(Queen fragment)增加灵魂容量 / Queen fragment increases Soul capacity

## 早期路线建议 / Early Route Advice

推荐的新手探索路线:

Recommended exploration route for beginners:

1. 从德里茅斯(Dirtmouth)进入遗忘十字路(Forgotten Crossroads)
2. 找到 Cornifer 购买地图 / Find Cornifer and buy the map
3. 击败假骑士(False Knight)获得城市徽章 / Defeat False Knight for the City Crest
4. 探索到绿色之路(Greenpath)入口
5. 返回德里茅斯与制图师妻子对话 / Return to Dirtmouth and talk to the cartographer's wife
6. 在遗忘十字路收集 300 Geo 购买灯笼(Lumafly Lantern)
7. 前往绿色之路继续冒险 / Proceed to Greenpath to continue your adventure

记住：不要害怕死亡。每次死亡都是一次学习机会。圣巢的每个角落都隐藏着秘密，保持好奇心和探索精神。

Remember: do not fear death. Each death is a learning opportunity. Every corner of Hallownest hides secrets — stay curious and keep exploring.
