# 终局内容 / Endgame Content

哈迪斯2的终局从第一次通关才真正开始——热度系统提供了递增的难度，预言挑战和隐藏结局等待探索 / Hades II's endgame truly begins after your first clear — the Heat system provides escalating difficulty, prophecy challenges and hidden endings await discovery.

---

## 热度系统 / Heat System

通关一次后解锁热度系统——通过增加热度来提升难度，热度越高奖励越丰厚 / The Heat system unlocks after first clear — increase difficulty via Heat, higher Heat = better rewards.

| 热度条件 / Heat Condition | 热度值 / Heat Value | 效果 / Effect |
|---|---|---|
| **敌人生命+20%** | 1 | 简单——推荐第一个增加的热度 / Simple — recommended first heat addition |
| **敌人伤害+20%** | 1 | 简单——没什么感觉 / Simple — barely noticeable |
| **精英出现率+30%** | 2 | 中等——更多精英=更多奖励但难度增加 / Medium — more elites = more rewards but harder |
| **商店价格+30%** | 1 | 低影响——钱包痛苦但不算难 / Low impact — wallet pain but manageable |
| **Boss额外增益** | 2 | 中等——Boss增加一个新招式 / Medium — boss gains a new move |
| **恩赐稀有度降低** | 2 | 高影响——更难拿到高级恩赐 / High impact — harder to get high-tier boons |
| **时间限制(7分钟/区域)** | 3 | 极高——强制快速通关。新手别碰 / Very high — enforced speed. Beginners avoid |
| **死亡 defiance次数-1** | 2 | 高影响——容错率大幅降低 / High impact — significantly reduced margin for error |
| **最大生命-30%** | 3 | 极高——很容易暴毙 / Very high — easy to get oneshot |
| **禁止冲刺** | 4 | 超极高——逃跑的核心机制没了 / Extreme — core movement mechanic removed |

**热度推荐**: 新手从5-7热度开始——选敌人生命+20%、敌人伤害+20%、商店价格+30%。熟练后逐步增加到16+热度 / **Heat recommendation**: Beginners start at 5-7 heat — enemy HP+20%, enemy damage+20%, shop price+30%. Gradually increase to 16+ heat as you improve.

## 预言挑战 / Prophecy Challenges

宿命之厅中的"宿命清单"提供了一系列挑战——每完成一项获得奖励 / The "Fate List" in the Fate Hall offers a series of challenges — each completed challenge gives rewards.

| 挑战等级 / Tier | 示例任务 / Example Task | 奖励 / Reward |
|---|---|---|
| **简单** | 用暗影之杖通关1次 / Win once with Shadow Staff | 宝石x10 / 10 Gems |
| **简单** | 获得5个宙斯恩赐 / Obtain 5 Zeus boons | 暗影x3 / 3 Shadows |
| **中等** | 在12热度下通关 / Win at 12 heat | 仙酒x1 / 1 Ambrosia |
| **中等** | 使用6种武器各通关1次 / Win once with each weapon type | 暗影x10 / 10 Shadows |
| **困难** | 在20热度下通关 / Win at 20 heat | 传说恩赐解锁 / Legendary boon unlock |
| **困难** | 不使用任何恩赐通关 / Win without any boons | 隐藏形态材料 / Hidden aspect material |
| **极难** | 在32热度下通关 / Win at 32 heat | 特殊皮肤+成就 / Special skin + achievement |

## 无限模式 / Endless Mode

无限模式在首次通关后解锁——选择"无尽"选项后，击败克洛诺斯后继续闯关，难度持续增加 / Endless Mode unlocks after first clear — after selecting "Endless", continue after Chronos with ever-increasing difficulty.

| 阶段 / Stage | 敌人强度 / Enemy Strength | 奖励 / Rewards |
|---|---|---|
| **克洛诺斯击败后(基础)** | 相当于热度8-10 / Equivalent to Heat 8-10 | 标准通关奖励 / Standard clear rewards |
| **第2轮区域1** | 相当于热度12-15 / Equivalent to Heat 12-15 | 暗影+宝石增加 / Extra Shadows + Gems |
| **第2轮区域2** | 相当于热度18-22 / Equivalent to Heat 18-22 | 仙酒出现 / Ambrosia appears |
| **第2轮区域3** | 相当于热度25-30 / Equivalent to Heat 25-30 | 传奇恩赐概率提升 / Legendary boon rate up |
| **第3轮+** | 指数增长——32+热度 / Exponential — 32+ heat | 资源大量 + 特殊成就 / Lots of resources + special achievement |
