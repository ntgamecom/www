# 哈迪斯2攻略 / Hades II Guide

《哈迪斯2》是Supergiant Games开发的肉鸽动作RPG——以冥界之女梅利诺厄为主角，穿越冥界、色萨利和奥林匹斯，挑战时间泰坦柯罗诺斯。本攻略涵盖全部神明恩赐与Build、五大武器形态、全区域敌人与Boss以及营地升级系统。

Hades II is Supergiant Games' roguelike action RPG starring Melinoe, princess of the Underworld, as she traverses the Underworld, Thessaly, and Olympus to challenge Chronos, the Titan of Time. This guide covers all god boons and builds, five weapon aspects, all region enemies and bosses, and the Crossroads upgrade system.

## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|---|---|
| **双法术系统 / Dual Cast System** | 欧米茄(长按)攻击和特殊——消耗法力——比1代更具策略性 / Omega (hold) attacks and specials — cost magick — more strategic than H1 |
| **法力与魔法 / Magick System** | 每次攻击恢复法力——法术消耗法力——法力管理决定节奏 / Attacks restore magick — spells consume it — management is key |
| **咒语系统 / Incantations** | 在营地可以解锁咒语——永久解锁新功能/升级 / Unlock incantations at the Crossroads — permanent unlocks |
| **卡牌系统 / Arcana Cards** | 塔罗牌系统——提供全局被动Buff——取代1代的镜子 / Tarot card system — global passive buffs — replaces Mirror of Night |
| **装备双武器 / Dual Weapon** | 可以带两把武器随时切换——战斗多样性提升 / Equip 2 weapons and swap mid-run — more combat variety |
| **动物伙伴 / Animal Familiars** | 猫头鹰、青蛙、龙等动物随行——辅助战斗 / Owl, frog, dragon etc. — assist in combat |

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **别急着打冲刺 / Don't Rush Sprint** | 梅利诺厄的冲刺有一个短暂的滑行——开始时感觉笨重——多练手感 / Melinoe's sprint has a brief slide — feels clunky at first — practice timing |
| **2** | **法力管理是关键 / Manage Your Magick** | 法力决定你的欧米茄(强攻击)——保持满法力准备爆发 / Magick fuels Omega attacks — keep it full for burst moments |
| **3** | **优先收集灰烬 / Prioritize Ashes** | 灰烬(Ash)是营地升级的主要资源——比1代的黑暗更早能用 / Ashes are the main Crossroads upgrade resource — available earlier than Darkness was in H1 |
| **4** | **猫头鹰伙伴最早解锁 / Owl Familiar First** | 在营地遇到猫头鹰后完成它的任务线——它会在战斗中捡药水 / Unlock the owl early — it collects potions in combat |
| **5** | **先解锁卡牌格位 / Unlock Arcana Slots First** | 优先级最高的升级：解锁更多塔罗牌格位——被动提升巨大 / Top priority: unlock more Arcana card slots — massive passive buffs |
| **6** | **不要怕换武器 / Try All Weapons** | 5种武器手感完全不同——找到最适合你的 / 5 weapons feel completely different — find your favorite |
| **7** | **每一轮至少打到一个Boss / Reach at Least One Boss** | 每个阶段的Boss掉落独特的升级材料——不打Boss等于白跑 / Each boss drops unique upgrade materials |
| **8** | **奥林匹斯之路需要解锁 / Olympus Route is an Unlock** | 初期只能走冥界路线——需要死几次解锁更多区域 / Initially only Underworld route — die a few times to unlock more |

## 难度系统 / Difficulty System

| 等级 / Level | 说明 / Description |
|---|---|
| **普通 / Normal** | 标准体验——推荐首次游玩 / Standard — recommended for first runs |
| **恐惧誓言 / Vows of Fear** | 类似1代的"惩罚契约"——在营地祭坛设置——增加难度换取更多奖励 / Similar to H1's Pact of Punishment — set at the Crossroads altar — more rewards for more difficulty |

**恐惧誓言示例 / Vow Examples**:
- 敌人伤害+20%/ 100% / +20%/100% enemy damage
- 敌人生命+20% / +20% enemy HP
- 减少时间限制 / Time limit reduction
- 每层减少一个恩赐奖励 / Reduce boon rewards per floor
