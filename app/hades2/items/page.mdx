## 诅咒与强化 / Curses & Enhancements

### 恐惧誓言 / Vows of Fear

在营地祭坛设置——增加难度换取更多奖励 / Set at Crossroads altar — more difficulty = more rewards.

| 誓言 / Vow | 效果 / Effect | 恐惧值 / Fear | 推荐 / Rate |
|---|---|---|---|
| 痛苦誓言 / Vow of Pain | 敌人伤害+20%/级 / Enemy dmg +20%/rank | 1/级 | ⭐⭐⭐ |
| 坚韧誓言 / Vow of Hardiness | 敌人生命+20%/级 / Enemy HP +20%/rank | 1/级 | ⭐⭐ |
| 饥饿誓言 / Vow of Hunger | 药水和食物减半 / Potions and food half effect | 2 | ⭐⭐⭐ |
| 时光誓言 / Vow of Time | 每层限时 / Time limit per floor | 3 | ⭐ |
| 恩赐禁令 / Vow of Denial | 每层减少一个恩赐 / -1 boon per floor | 2 | ⭐⭐ |

**推荐开局设置 / Recommended Starting Setup**:
- 初期恐惧值: 0——先熟悉游戏 / 0 Fear — learn the game
- 中期: 痛苦誓言x2(2恐惧) / Vow of Pain x2 (2 Fear)
- 后期: 痛苦x3+饥饿x1 = 5恐惧——平衡奖励和难度 / Pain x3 + Hunger = 5 Fear
