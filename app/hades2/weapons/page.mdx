# 武器与Build / Weapons & Builds

## 五大武器 / Five Weapons

哈迪斯2目前有 **5种武器**——每种有多个形态(Aspect) / 5 weapons — each with multiple aspects.

| 武器 / Weapon | 解锁条件 / Unlock | 特点 / Traits | 推荐等级 / Tier |
|---|---|---|---|
| **姐妹双刃 / Sister Blades** | 初始 / Starter | 攻速快、双持——近战灵活 / Fast, dual-wield — agile melee | A |
| **月光之斧 / Moonstone Axe** | 初始 / Starter | 慢但伤害高——范围大 / Slow but high dmg — wide range | B |
| **银色之杖 / Silver Staff** | 咒语解锁 / Incantation | 中距+特殊远程——全能 / Mid-range + ranged special — versatile | A |
| **暗影火炬 / Umbral Flames** | Boss掉落解锁 / Boss drop | 远程法术——风筝流 / Ranged magic — kiting build | S |
| **骷髅头骨 / Skull** | 咒语解锁 / Incantation | 魔法爆炸——高爆发 / Magic explosion — high burst | S (高手向) |

## 武器强化 / Weapon Upgrades

在营地武器祭坛消耗材料升级——共5级 / Upgrade at Weapon Altar — 5 levels.

| 等级 / Level | 消耗 / Cost | 效果 / Effect |
|---|---|---|
| 1→2 | 灰烬x5+银币x10 | +5基础伤害 / +5 base dmg |
| 2→3 | 灰烬x10+布料x3 | +5基础伤害 + 特殊效果增强 / +5 dmg + special effect up |
| 3→4 | 灰烬x15+布料x5+珍珠x1 | +10基础伤害 / +10 base dmg |
| 4→5 | 灰烬x25+珍珠x3 | +10基础伤害 + 终极强化特效 / +10 dmg + ultimate upgrade |

**推荐优先升级 / Priority Order**:
1. 暗影火炬(远程安全——新手可风筝) / Umbral Flames — safe ranged kiting
2. 姐妹双刃(帅气+灵活) / Sister Blades — stylish + agile
3. 银之法杖(全能——可攻可守) / Silver Staff — versatile
