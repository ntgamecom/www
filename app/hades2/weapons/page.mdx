## 武器强化 / Weapon Upgrades

在营地武器祭坛消耗材料升级——共5级 / Upgrade at Weapon Altar — 5 levels.

| 等级 / Level | 消耗 / Cost | 效果 / Effect |
|---|---|---|
| 1→2 | 灰烬x5+银币x10 | +5基础伤害 / +5 base dmg |
| 2→3 | 灰烬x10+布料x3 | +5基础伤害 + 特殊效果增强 / +5 dmg + special effect up |
| 3→4 | 灰烬x15+布料x5+珍珠x1 | +10基础伤害 / +10 base dmg |
| 4→5 | 灰烬x25+珍珠x3 | +10基础伤害 + 终极强化特效 / +10 dmg + ultimate upgrade |

**推荐优先升级 / Priority Order**:
1. 暗影火炬(远程安全——新手可风筝) / Umbral Flames — safe ranged kiting
2. 姐妹双刃(帅气+灵活) / Sister Blades — stylish + agile
3. 银之法杖(全能——可攻可守) / Silver Staff — versatile

## 武器切换策略 / Weapon Switching Strategy

| 策略 / Strategy | 说明 / Description |
|---|---|
| **主副武器定位** | 主武器(高伤核心)+副武器(辅助/控制或补属性和类)/ Main (primary DPS) + Offhand (support/control or alternate element) |
| **切换时机** | 主武器攻击→敌人后撤→切远程武器追击→敌人近身→切回近战 / Main weapon attack → enemy retreats → switch to ranged → enemy closes → switch back to melee |
| **欧米茄切换** | 主武器欧米茄(消耗法力)→切副武器欧米茄(消耗剩余法力)→双倍爆发 / Main weapon Omega (spends magick) → switch → offhand Omega (spends remaining magick) → double burst |
| **冷却利用** | 一把武器的特殊技能在冷却时——切换另一把武器继续输出 / When one weapon's special is on cooldown — switch to the other and keep DPSing |
