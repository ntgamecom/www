# BOSS攻略 / Boss Guide

哈迪斯2的Boss战在每个区域的末端——每个Boss有独特的攻击模式和阶段变化。掌握这些Boss战的节奏是从冥界逃脱的关键 / Hades II's boss fights await at the end of each region — each with unique attack patterns and phase changes. Mastering the rhythm of these boss fights is key to escaping the Underworld.

---

## 区域1 Boss / Region 1 Boss

### 赫卡忒 / Hecate

| 属性 / Property | 说明 / Description |
|---|---|
| **区域 / Region** | 哀悼荒野 / Mourning Fields |
| **阶段数 / Phases** | 2个阶段 / 2 phases |
| **主要攻击类型** | 魔法弹幕+近战杖击+分身召唤 / Magic barrage + staff melee + clone summon |
| **难度 / Difficulty** | 中等——新手第一个屏障 / Medium — first real wall for beginners |

**阶段1攻击模式**:
| 攻击 / Attack | 前摇 / Tell | 应对 / Counter |
|---|---|---|
| **魔法球** | 举起法杖聚集能量——发射3个追踪球 / Raises staff, charges energy — 3 homing orbs | 冲刺穿过球体——不要向后跑。向侧面或向Boss方向冲 / Dash through orbs — don't run backward. Dash sideways or toward boss |
| **法杖横扫** | 法杖发光——180度横扫 / Staff glows — 180 degree sweep | 跳跃跳跃躲避——横扫范围很大但上方安全 / Jump to dodge — wide sweep but safe above |
| **地面冲击波** | 法杖砸地——圆形冲击波扩散 / Staff slams ground — expanding ring | 在冲击波到来前跳过——或冲刺到Boss背后 / Jump before impact — or dash behind boss |
| **分身召唤** | 后退——召唤2个分身一起攻击 / Backs away — summons 2 clones | 分身只有1血——快速消灭。不要被分身和本体夹击 / Clones have 1 HP — kill fast. Don't get sandwiched |

**阶段2** (HP≤50%):
| 变化 / Change | 说明 / Description |
|---|---|
| **更密集弹幕** | 魔法球速度翻倍+数量增加到5发 / Orb speed doubled, 5 orbs instead of 3 |
| **旋转攻击** | 持续旋转法杖——360度范围攻击。持续3秒 / Continuous staff spin — 360 degree attack. Lasts 3s |
| **分身增强** | 分身现在会使用魔法球攻击——不只是近战 / Clones now use magic orbs — not just melee |

**推荐Build**: 远程武器(暗影之杖或白骨法杖)+ 冲刺型恩赐(如波塞冬的冲刺击退) / **Recommended build**: Ranged weapon (Staff or Skull) + Dash boon (Poseidon dash knockback)

## 区域2 Boss / Region 2 Boss

### 塞壬海妖三姐妹 / Siren Song

| 属性 / Property | 说明 / Description |
|---|---|
| **区域 / Region** | 维奥蒂亚海岸 / Coast of Viotia |
| **阶段数 / Phases** | 3个阶段——每位姐妹单独战斗 / 3 phases — one sister at a time |
| **顺序** | 随机——每局不同 / Random — changes each run |

**三姐妹详解**:
| 塞壬 / Siren | 攻击模式 / Attack Pattern | 弱点 / Weakness |
|---|---|---|
| **塞壬1: 重型** | 慢速+高伤害——地面冲击波+旋转锯刃。攻击范围大 / Slow + high damage — ground shockwave + spinning saw. Wide range | 移动慢——拉开距离用远程攻击 / Slow movement — ranged attacks from distance |
| **塞壬2: 速度型** | 快速+多段攻击——冲刺连击+空中俯冲。攻击速度快但伤害低 / Fast + multi-hit — dash combo + air dive. Fast but low damage per hit | 用范围攻击——她冲刺后有短暂停顿 / Use area attacks — she pauses briefly after dashing |
| **塞壬3: 魔法型** | 远程弹幕+召唤小兵+传送。最烦人的一个 / Ranged barrage + summon + teleport. Most annoying | 近身压制——她传送后方向固定 / Press close — her teleport direction is fixed |

**推荐Build**: 任意武器+大范围AOE恩赐(如宙斯雷电+波塞冬洪水) / **Recommended build**: Any weapon + large AOE boons (Zeus lightning + Poseidon flood)

## 区域3 Boss / Region 3 Boss

### 克洛诺斯 / Chronos(时间泰坦 / Titan of Time)

| 属性 / Property | 说明 / Description |
|---|---|
| **区域 / Region** | 色萨利盆地 / Thessaly Basin |
| **阶段数 / Phases** | 3个阶段——每个阶段攻击模式大幅变化 / 3 phases — each phase significantly changes attack patterns |
| **难度 / Difficulty** | 极高——目前游戏最难Boss / Very High — currently hardest boss in the game |

**阶段1**(HP 100%-66%):
| 攻击 / Attack | 应对 / Counter |
|---|---|
| **时间利刃** | 召唤延迟出现的斩击——地面上有红色提示区域。1.5秒后斩击出现 / Summons delayed slashes — red indicators on ground. Slash hits after 1.5s | 看到红圈后闪避——不要站在上面。不要提前闪避因为斩击范围追踪 / Dodge after seeing red circle — don't stand on it. Don't dodge early as slash tracks |
| **时间暂停** | 克洛诺斯暂停时间——除了你之外所有东西停止。然后向你的位置发动攻击 / Chronos stops time — everything else freezes. Then attacks your position | 时间暂停期间迅速移动到安全位置——然后用冲刺躲开攻击 / Move to safe position during slow-mo — then dash away from the incoming attack |
| **时钟指针** | 从Boss位置发散出时钟指针形状的弹幕——缓慢旋转 / Clock hand projectiles radiate from boss — slow rotation | 跟随指针旋转方向移动——不要被包围 / Move with the rotation — don't get surrounded |

**阶段2**(HP 66%-33%):
| 新攻击 / New Attack | 说明 / Description |
|---|---|
| **时间裂缝** | 场地上出现时间裂缝——站上面会受到持续伤害+减速 / Time rifts appear on ground — standing on them causes continuous damage + slow | 尽量保持移动——不要站在同一个地方超过2秒 / Keep moving — don't stay in one spot for more than 2 seconds |
| **加速连击** | 克洛诺斯获得攻速加成——近战连击速度翻倍 / Chronos gains attack speed buff — melee combo speed doubles | 更多依赖冲刺——近战硬拼必吃亏 / Rely more on dashing — trading blows in melee loses |
| **沙漏轰炸** | 召唤沙漏从天空落下——地面有黄圈预警 / Summons hourglasses falling from sky — yellow circle warning | 观察地面黄圈——不要站里面。沙漏落地后爆炸留下区域伤害 / Watch ground circles — don't stand in them. Hourglass explodes leaving area damage |

**阶段3**(HP ≤33%):
| 变化 / Change | 说明 / Description |
|---|---|
| **全屏弹幕** | 时间指针+沙漏+裂缝同时出现——全场到处都是伤害区域 / All attack types active simultaneously — entire screen becomes hazard zone |
| **时间狂暴** | 克洛诺斯进入狂暴状态——所有攻击速度+50% / Chronos enrages — all attack speeds +50% |
| **复活无效** | 在这个阶段死亡不会触发"死亡 defiance"的复活——必须0失误 / Death Defiance doesn't trigger in this phase — must play perfectly |

**推荐Build**: 高爆发武器(双刃斧或月光石斧)+ 防御恩赐(雅典娜冲刺反弹+阿芙罗狄蒂减伤) / **Recommended build**: High burst weapon (Axe or Moonstone Axe) + defensive boons (Athena dash reflect + Aphrodite damage reduction)

## Boss战通用技巧 / Boss Fight General Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **冲刺方向** | 大多数情况下——向Boss方向冲刺比向后更安全。向后冲刺往往会被追击范围打中 / In most cases — dashing TOWARD the boss is safer than backward. Backward dashes get caught in chase ranges |
| **恩赐优先级** | 打Boss前——确认你的Build已经成型。冲刺恩赐>施法恩赐>攻击恩赐>特殊恩赐。冲刺最常用 / Before boss — ensure build is ready. Dash boon > Cast boon > Attack boon > Special boon. Dash is most used |
| **纪念品策略** | 区域1用攻击型纪念品(月神雕像)快速拿恩赐→区域2前换成防御型(雅典娜护符)/区域3用爆发型(阿瑞斯护符) / Region 1: offensive keepsake (Selene) → swap to defensive (Athena) before region 2 → burst (Ares) before region 3 |
| **死亡 defiance管理** | 如果还剩1次死亡 defiance——在Boss战前可以考虑回营地补满。不要空血进Boss / If 1 Death Defiance remaining — consider returning to camp to refill. Don't enter boss at low HP |
| **场地利用** | 所有Boss战场地边缘有柱子/障碍物——可以用来挡弹幕和卡Boss走位 / All boss arenas have pillars/obstacles at edges — use them to block projectiles and limit boss movement |
