## 双法术系统详解 / Omega System

| 操作 / Input | 效果 / Effect | 消耗 / Cost |
|---|---|---|
| **欧米茄攻击 / Omega Attack** | 蓄力后释放强攻击——不同武器效果不同 / Charge → powerful attack — varies by weapon | 20-50法力 / 20-50 magick |
| **欧米茄特殊 / Omega Special** | 蓄力后释放强特殊 / Charge → powerful special | 20-40法力 / 20-40 magick |
| **欧米茄法阵 / Omega Cast** | 法阵不扔出——环绕自身——吸引/控制敌人 / Cast stays around you — attracts/controls enemies | 30法力 / 30 magick |
| **奔跑 / Sprint** | 冲刺后按住——撞毁弹丸——减速时间 / Dash then hold — destroys projectiles — slows time | 无消耗 / Free |

**核心循环 / Core Loop**:
```
普通攻击(回法力) → 攒到满蓝 → 欧米茄攻击(爆发) → 继续普攻回蓝
Normal Attack (restore magick) → Full magick → Omega Attack (burst) → Repeat
```

**法力管理 / Magick Management**:
- 攻击(普攻)命中恢复法力——每一击大约回2-5点 / Attacks (normal) restore magick — ~2-5 per hit
- 法力上限100-200——升级可增加 / Max magick 100-200 — upgradable
- 法力不足时——无法使用欧米茄技能——只能靠普攻回蓝 / Out of magick = can't use Omega — only normal attacks to recover

## 新手常见错误 / Common Beginner Mistakes

| # | 错误 / Mistake | 正确做法 / Correct Way |
|---|---|---|
| 1 | 不蓄力只用普攻 / Never use Omega | 欧米茄攻击是主要爆发来源——必须学会蓄力 / Omega attacks are your main burst — must learn to charge |
| 2 | 法力一直空 / Always out of magick | 普攻命中回蓝——打了就跑等回蓝 / Normal attacks restore magick — hit a few times then burst |
| 3 | 不收集灰烬 / Ignore Ash | 灰烬是最重要的升级资源——每个敌人都有可能掉 / Ash is the most important upgrade resource |
| 4 | 冲刺当1代用 / Dash like Hades 1 | 2代的冲刺有滑行——而且可以奔跑撞毁子弹 / Sprint+destroy projectiles is new and powerful |
| 5 | 不带第二把武器 / Only use one weapon | 双武器切换是核心机制——充分利用 / Two weapons are core to the combat system |

## 梅利诺厄能力详解 / Melinoe Abilities

| 能力 / Ability | 基础效果 / Base Effect | 欧米茄效果 / Omega Effect |
|---|---|---|
| **攻击 / Attack** | 快速连击——每次恢复法力 / Fast combo — restores magick per hit | 蓄力大范围攻击——消耗大量法力 / Charge wide attack — costs lots of magick |
| **特殊 / Special** | 武器专属技能——不同武器不同效果 / Weapon-specific — varies by weapon | 蓄力强化特殊——范围+伤害增加 / Charge enhanced special — more range + damage |
| **法阵 / Cast** | 放置或投掷法阵——对范围内敌人造成伤害 / Place or throw cast — damages enemies in area | 欧米茄法阵——控制敌人+持续伤害 / Omega cast — controls enemies + sustained damage |
| **冲刺 / Dash** | 闪避——有短暂无敌帧——冲刺后有一段滑行 / Dodge — brief i-frames — slide after dash | — (无欧米茄版) |
| **奔跑 / Sprint** | 冲刺后按住——持续奔跑——可以撞毁敌人的弹丸 / Hold after dash — sustained run — destroys enemy projectiles | — (无欧米茄版) |
| **召唤 / Call** | 使用神明召唤——需要充能——充能满后可以释放 / Use god call — needs charge — release when full | 蓄力释放——更强大的召唤效果 / Charge release — stronger call effect |

## 角色成长路径 / Progression Path

| 阶段 / Phase | 目标 / Goal | 建议 / Suggestion |
|---|---|---|
| **前10局 (学习期)** | 熟悉操作+收集资源 / Learn controls + gather resources | 不要追求通关——专注于学习敌人攻击模式和收集灰烬 / Don't chase wins — focus on learning enemy patterns and collecting Ashes |
| **10-30局 (成长期)** | 解锁关键升级 / Unlock key upgrades | 优先解锁卡牌格位——每个新格位=巨大提升 / Prioritize Arcana card slots — each new slot = huge power boost |
| **30-50局 (突破期)** | 首次通关 / First clear | 选择最顺手的武器——堆卡牌格位——不要挑战高恐惧值 / Use your best weapon — max Arcana slots — don't use Fear yet |
| **50+局 (精通期)** | 挑战高恐惧值 / Challenge high Fear | 开始使用恐惧誓言——逐步增加难度 / Start using Vows — gradually increase Fear level |
