## 通用Build原则 / Universal Build Principles

| 原则 / Principle | 说明 / Description |
|---|---|
| **恩赐专注** | 不要每种神都拿——专注1-2个神的恩赐。恩赐有等级，同神等级高了有"协同"效果 / Don't pick every god — focus on 1-2 gods. Boons have levels, higher levels unlock "synergy" effects |
| **冲刺恩赐优先** | 冲刺是你最频繁使用的动作——好的冲刺恩赐(雅典娜反弹、波塞冬击退、宙斯闪电)对生存和输出都有巨大提升 / Dash is your most frequent action — good dash boon (Athena reflect, Poseidon knockback, Zeus lightning) boosts both survival and DPS |
| **施法不要忽略** | 施法(Cast)在二代比一代更重要——施法提供额外伤害+控制。不要只靠平A和特殊攻击 / Cast is more important in Hades 2 than Hades 1 — provides extra damage + control. Don't rely only on attack and special |
| **纪念品战术换装** | 开局用一个纪念品(如月神雕像拿月神恩赐)——区域Boss前更换为防御型纪念品(如阿芙罗狄蒂香水) / Start with one keepsake (Selene statue for Selene boons) — swap to defensive keepsake before region boss |

## 进阶操作技巧 / Advanced Combat Techniques

| 技巧 / Technique | 操作 / Input | 效果 / Effect |
|---|---|---|
| **冲刺取消** | 攻击后立即冲刺 / Attack → immediately dash | 取消攻击后摇——攻速实际提升30%+。所有武器通用 / Cancel attack recovery — +30% actual attack speed. Universal |
| **施法引爆** | 施法命中后等待1秒→再次施法 / Cast → wait 1s → cast again | 部分施法(得墨忒耳/阿瑞斯)有"引爆"机制——延迟引爆造成2倍伤害 / Some casts (Demeter/Ares) have "detonate" — delayed 2x damage |
| **蓄力保留** | 蓄力攻击蓄满但不释放——移动中保持蓄力 / Charge attack to max — hold while moving | 双刃斧和月光石斧可以带着蓄力移动——找到最佳时机再释放 / Axe and Moonstone Axe can move while charged — release at perfect moment |
| **双恩赐触发** | 同时按下攻击+特殊攻击 / Press attack + special simultaneously | 某些武器可以同时打出攻击和特殊——双倍瞬间爆发 / Some weapons can attack + special simultaneously — double burst |
| **环境互动** | 把敌人推向岩浆/深渊 / Push enemies into lava/pits | 波塞冬击退流的核心——推到岩浆/深渊直接秒杀 / Poseidon knockback core — push into lava/pits for oneshot |
