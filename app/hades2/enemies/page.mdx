## Boss攻略 / Boss Guide

### 赫卡忒 / Hecate

**位置**: 厄瑞波斯——第3层 / Erebus — Floor 3
**难度**: ★★☆☆☆

**阶段 / Phase**:

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|---|---|---|
| **P1** | 投掷魔法球(4方向弹幕) + 召唤鬼魂 / 4-direction bullet + summon wisps | 冲刺躲避弹幕——优先清鬼魂 / Dash through bullets — clear wisps |
| **P2** (损失50%血) | 地面AOE圆圈——在脚下出现大范围爆炸 + 召唤分身 / Ground AoE circles + clones | 狂奔不要停——看到红圈就冲刺 / Keep sprinting — dash from red circles |
| **P3** (损失25%血) | 全屏弹幕旋转+以上全部 / Full-screen spinning bullet + all previous | 躲在柱子后面——利用奔跑撞毁弹丸 / Hide behind pillars — sprint destroys bullets |

**核心技巧 / Core Tips**:
1. 赫卡忒的弹幕可以被奔跑撞毁——不要只是躲——冲过去 / Her bullets can be sprint-destroyed — charge through them
2. 分身是假的——打一下就消失——不要浪费技能 / Clones disappear in one hit — don't waste Omega
3. 推荐带冲刺次数的恩赐(赫尔墨斯) / Hermes' extra dashes recommended

### 塞壬 / Sirens

**位置**: 色萨利——第3层 / Thessaly — Floor 3
**难度**: ★★★☆☆

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|---|---|---|
| **P1** | 塞壬歌声——范围致晕+召唤海妖 / Song — AoE stun + summon maids | 看到咏唱提示就冲刺离开 / Sprint away when chant starts |
| **P2** | 水龙卷——地面追踪龙卷风+浪击 / Tornado + wave slam | 保持移动——龙卷风转弯慢 / Keep moving — tornadoes turn slow |
| **P3** | 三个塞壬同时出现——分散攻击 / 3 Sirens — split attacks | 先杀其中一个——逐个击破 / Focus one at a time — divide and conquer |

**核心技巧 / Core Tips**:
1. 塞壬的歌声有前摇——看到水面波动就准备冲刺 / Song has a wind-up — prep dash when water ripples
2. 三个塞壬中有一个主要进攻——找到她先击杀 / One Siren is the main attacker — find and kill her first
3. 远程武器(火炬/法杖)在此战有明显优势 / Ranged weapons (flames/staff) have an advantage here

### 柯罗诺斯 / Chronos (最终Boss)

**位置**: 奥林匹斯——第3层 / Olympus — Floor 3
**难度**: ★★★★★

**时间泰坦柯罗诺斯**——目前为止的最终Boss / The Titan of Time — current final boss.

| 阶段 / Phase | HP区间 / HP Range | 攻击模式 / Attack Pattern | 应对 / Counter |
|---|---|---|---|
| **P1** | 100%-66% | 时间减缓(玩家减速)+ 投掷时间之刃 + 召唤拟像(过去的你)/ Time slow + time blades + summon your past self | 时间减速时靠奔跑反击——拟像出现时优先击杀 / Sprint through slow — kill past self fast |
| **P2** | 66%-33% | 全屏时间停止(短暂)+ 大范围时间裂隙 + 召唤多个拟像 / Time stop + time rifts + multiple summons | 时间停止时无法操作——注意停止解除后的闪避 / Can't move during stop — dodge immediately after |
| **P3** | 33%-0% | 以上全部 + 狂暴模式——连击速度快2倍 / All previous + berserk mode — 2x attack speed | 保命优先——用回复药——欧米茄只在你安全的时候放 / Prioritize survival — heal — Omega only when safe |

**柯罗诺斯核心打法 / Chronos Strategy**:

| 准备 / Prep | 建议 / Recommendation |
|---|---|
| **武器推荐** | 暗影火炬(放风筝) 或 姐妹双刃(高爆发) / Umbral Flames (kite) or Sister Blades (burst) |
| **核心恩赐** | 赫尔墨斯(3连冲刺) + 雅典娜(冲刺反弹) / Hermes (triple dash) + Athena (dash reflect) |
| **药水** | 带上至少2瓶回血药 + 1瓶满蓝药 / At least 2 heals + 1 full magick |
| **卡牌** | 守护(减伤+25%)+生命(+50HP)+死神(欧米茄伤害) / Guardian + Life + Death |
| **核心技巧** | 他的时间停止是没有预兆的——看到屏幕变白立刻准备冲刺 / Time stop has no warning — ready dash when screen turns white |

**柯罗诺斯全模式应对 / Chronos Move-by-Move**:

| 攻击 / Move | 视觉效果 / Visual | 应对 / Counter |
|---|---|---|
| 时间之刃 / Time Blade | 黄色光刃飞向你 | 冲刺闪避——不可格挡 / Dash dodge — unblockable |
| 时间裂隙 / Time Rift | 地面出现钟表圈 / Clock circle on ground | 站在圈外——进去会被减速 / Stay outside — inside = slow |
| 拟像召唤 / Past Self | 你的复制体出现 / Your clone appears | 它很脆——优先击杀立即 / Fragile — kill immediately |
| 时间停止 / Time Stop | **全屏变白** / **Full screen white** | 什么都做不了——停了之后立刻冲刺 / Can't act — dash immediately when it ends |
| 时钟冲刺 / Clock Dash | 柯罗诺斯化作时间朝你冲刺 / He becomes time and dashes | 两侧闪避——不要后退 / Dodge sideways — not backward |
