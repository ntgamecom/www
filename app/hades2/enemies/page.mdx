# 敌人图鉴 / Enemy Guide

哈迪斯2的敌人种类比一代更加丰富——每个区域都有独特的敌人阵容和精英变种。了解敌人攻击模式是打通更高热度的关键 / Hades 2 features a more diverse enemy roster than the first game — each region has unique enemy sets and elite variants. Knowing attack patterns is key to climbing higher Heat levels.

---

## 埃瑞波斯区域 / Erebus (第一区域)

| 敌人 / Enemy | 攻击模式 / Attack Pattern | 应对策略 / Counter Strategy |
|------|------|------|
| **女巫学徒 / Witch Apprentice** | 远程追踪弹(紫色球) / Tracking purple orb | 引诱她放完弹幕然后近身——弹幕飞行速度慢，侧移即可 / Lure out the volley then close in — projectiles are slow, just strafe |
| **骷髅兵 / Skeleton** | 近战砍击——死了后可能复活 / Melee slash, may resurrect on death | 打碎后踩一脚防止复活。推荐范围攻击快速清群 / Stomp after killing to prevent resurrection. AoE recommended for groups |
| **痛苦之灵 / Wraith** | 瞬移+近身爪击 / Teleport + claw attack | 保持移动——它会瞬移到你身后。听到瞬移音效立刻翻滚 / Keep moving — it teleports behind you. Roll on teleport sound |
| **地狱犬幼崽 / Hellhound Pup** | 三段咬+冲刺突进 / Triple bite + charge | 最后一下咬有延迟——等它咬空再打。不要被围——犬类敌人喜欢配合围攻 / Third bite has delay — punish the whiff. Don't get surrounded — they coordinate |

### 精英变种 / Elite Variants (Erebus)

| 精英 / Elite | 强化/ Enhanced | 应对 / Strategy |
|------|------|------|
| **精英女巫 / Elite Witch** | 弹幕翻倍+追踪升级——球会分裂 / Double projectiles + homing upgrade, orbs split | 躲到柱子后面——让柱子挡弹幕。近身打断她施法 / Use pillars for cover. Interrupt her cast up close |
| **巨型骷髅 / Giant Skeleton** | 体型2倍+攻击带眩晕 / 2x size, attacks cause stun | 不要站正面——绕背打。精英眩晕恢复很快，打2-3下就走 / Don't engage head-on — flank. Elite stun recovery is fast, hit 2-3 times then reset |

## 哀悼原野 / Mourning Fields (第二区域)

| 敌人 / Enemy | 攻击模式 / Attack Pattern | 应对策略 / Counter Strategy |
|------|------|------|
| **亡魂战士 / Soul Warrior** | 三段连斩+突刺——与一代勒蒜类似 / 3-hit combo + thrust — similar to Zag's style | 完美闪避第三下(最慢)→最大输出窗口。突刺有较大前摇——看到就闪侧 / Perfect dodge the third hit (slowest) = biggest window. Thrust has long wind-up |
| **弓箭手 / Archer** | 远程射击——伤害不高但烦人 / Ranged shot — low damage, annoying | 优先级高——先清弓箭手再处理近战。用冲刺突进贴脸 / Priority target — kill archers first. Dash-strike to close gap |
| **自爆灵 / Exploding Wraith** | 飘向玩家→自爆(范围大)/ Float toward player → explode (wide AoE) | 看到变红就闪避拉开——爆炸范围比看起来大。可以利用它炸其他敌人 / Dash away on red glow — blast radius is bigger than it looks. Trick it into damaging other enemies |
| **美杜莎石像 / Gorgon Statue** | 石化射线——被照到=缓慢+后续受伤增加 / Petrifying beam — slow + increased damage taken | 躲在柱子后面等射线结束。射线方向固定→绕到侧面打 / Use pillars for cover. Ray direction is fixed → flank while it's active |

## 俄刻阿诺斯 / Oceanus (第三区域)

| 敌人 / Enemy | 攻击模式 / Attack Pattern | 应对策略 / Counter Strategy |
|------|------|------|
| **鱿鱼怪 / Squid** | 喷墨(致盲)+触须抓取+冲刺 / Ink (blind) + tentacle grab + charge | 喷墨前摇长→看到就闪到侧面。被致盲=屏幕变黑→立刻翻滚拉开距离 / Ink has long wind-up → dodge sideways on sight. Blinded = black screen → roll away immediately |
| **螃蟹重甲兵 / Crab Heavy** | 大范围钳击+蓄力冲刺 / Wide pincer swing + charged rush | 蓄力冲刺左右闪(不是前后)。钳击范围大→保持中距引诱它出招→闪侧打背 / Dodge charged rush sideways (not backward). Bait the pincer, dodge in, attack from behind |
| **水母群 / Jellyfish Swarm** | 漂浮+触碰带电/ Poison floating + touch stun | 不要站桩——保持移动。远程武器/技能优先——近战打水母群很亏 / Never stand still. Ranged weapons/skills preferred — melee vs jellyfish is bad value |
| **塞壬 / Siren** | 唱歌迷惑(角色反向移动)+水弹 / Singing confuses (reversed controls) + water orbs | 塞哨歌声可以被打断——冲脸打一下就能打断。水弹跳跃躲避 / Siren song is interruptible — one hit stops it. Jump over water orbs |

## 塔耳塔洛斯 / Tartarus (第四区域)

| 敌人 / Enemy | 攻击模式 / Attack Pattern | 应对策略 / Counter Strategy |
|------|------|------|
| **泰坦后裔 / Titan Spawn** | 重拳(范围大)+地板裂缝(延迟爆炸)/ Heavy punch (wide) + floor crack (delayed explosion) | 重拳前摇极长→后闪。地板裂缝→离开裂缝范围。这是Hades2最慢但最痛的敌人之一 / Punch has huge wind-up → dodge back. Floor cracks → leave the area. One of the slowest but hardest-hitting enemies |
| **地狱三头犬(小)/ Cerberus Pup** | 三头喷火(扇形)+尾巴扫击 / 3-headed fire spray + tail sweep | 喷火向侧跑/冲刺不是向后。尾巴扫击范围在背后——站它侧面最安全 / Run sideways for fire spray (not backward). Tail sweep hits behind — safest at its side |
| **复仇女神 / Fury** | 连击(4-6段)+远程投枪——攻击欲望极高 / Long combo + ranged spear — extremely aggressive | 格挡流克星——不能用常规格挡对策。用闪避流或者高爆发在她出招前打硬直 / Counters block builds — use dodge builds or high burst to stagger before she starts |
| **混沌兽 / Chaos Beast** | 瞬移+混乱射线(强制位移)+召唤小怪 / Teleport + chaos ray (forced movement) + summon | 瞬移后有短暂停顿——立刻找到它的新位置。混沌射线→向侧面闪。优先清小怪 / Brief pause after teleport — locate it immediately. Chaos ray → dodge sideways. Clear adds first |

## Boss阶段综述 / Boss Flow Overview

| Boss | 阶段1 / Phase 1 | 阶段2 / Phase 2 | 阶段3 / Phase 3 |
|:---:|:---:|:---:|:---:|
| **赫卡忒 / Hecate** | 魔法弹幕→近身魔杖挥击 | 分身(2个假身)→真身随机位置→打中真身解除 | 巨大化→全屏魔法阵→躲避紫色圈 |
| **刻耳柏洛斯 / Cerberus** | 三头喷火(扇形可闪)+爪击 | 地面火山喷发→躲避地面红圈 | 愤怒状态→三头同时喷火→范围超大 |
| **哈迪斯 / Hades** | 长矛连刺+投矛(远程)/ Spear combo + throw | 召唤亡灵→清小怪优先 | 巨大化→全屏斩击+范围落雷 |

**赫卡忒通用打法的核心思路**: Phase 1 保持中距——引诱她放弹幕→闪避近身打一套。Phase 2 看地上的影子是真身的提示——真身有影子，分身的影子较淡。Phase 3 不要贪刀——她全屏魔法阵的生成速度很快，看到紫色圈立刻闪出范围 / **Hecate General Strategy**: Phase 1: mid-range — bait spell volley → dodge in → combo. Phase 2: shadows reveal the real body — real has a solid shadow, clones have faint ones. Phase 3: don't greed — magic circle spawns fast, dash out of purple zones immediately.

## 热度系统对敌人的影响 / Heat Impact on Enemies

热度(Heat)系统在Hades2中回归——以下热度条件会显著影响敌人强度：

| 热度条件 / Heat Condition | 效果 / Effect | 推荐应对 / Strategy |
|------|------|------|
| **强敌契约 / Hard Labor** | 敌人伤害+20%/40%/60%等 / Enemy damage +20/40/60% | 加强防御类祝福——特别是铜皮/护盾系 / Stack defensive boons |
| **持久苦战 / Lasting Consequences** | 治疗效果-25%/50%/75% / Healing reduced | 选择自带回复的武器/祝福/信物 / Pick self-healing weapons and keepsakes |
| **禁忌仪式 / Forbidden Rites** | 敌人速度+20% / Enemy speed +20% | 提前熟悉敌人攻击前摇——速度提升后窗口变小 / Learn attack tells well — windows shrink |
| **精英倍增 / Routine Inspection** | 每个房间精英怪数量+1 / +1 elite per room | 单点爆发Build优于AOE Build——精英血量高 / Single-target burst > AoE for elite-heavy rooms |
