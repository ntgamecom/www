## 战斗基础 / Combat Basics

《最终幻想 VII 重制版》采用即时动作战斗（Real-time Action）与回合制指令的融合系统。玩家直接操控角色在战场中自由移动、进行普通攻击与闪避。当 ATB 槽（Active Time Battle gauge）蓄满至少一格后，可进入战术模式（Tactical Mode）下达指令，包括施放魔法、使用技能、道具或召唤兽。

**核心机制**：普通攻击可积累 ATB 槽。每个角色拥有两条 ATB 槽（部分武器可增加至三条）。战术模式下时间会大幅减慢，让玩家从容选择指令。成功闪避可减少伤害，但攻击期间的硬直无法取消。防御可大幅减少伤害并防止被击退。

**战斗节奏建议**：初期不要贪刀。攻击 2-3 次后观察敌人动作并适时闪避或防御。HEALING 魔晶石应优先装备给队伍成员，确保至少一人有回复能力。随时注意 ATB 槽的存量——留一格给治疗往往能避免灭团。

## 实时战斗与战术模式 / Real-time vs Tactical Mode

**实时模式（Real-time Mode）**：适合普通杂兵战。通过□键（普通攻击）快速积累 ATB，然后以快捷键（L1+对应按键）直接释放技能。熟练后可流畅衔接普攻与技能。

**战术模式（Tactical Mode）**：按触摸板（PS）或 T 键（PC）进入。时间流速降至 10%，可精确选择目标、切换魔法或技能、查看敌人信息。对于 Boss 战和精英敌人，战术模式几乎不可或缺——尤其在需要精确针对弱点属性施放魔法时。

**建议**：将两者结合使用。普通战斗以实时模式为主，快速击杀；高难度战斗则频繁切换战术模式，确保指令精准。Hard Mode 下战术模式的运用至关重要。

## 角色切换 / Character Switching

战斗中可随时切换当前操控角色（方向键左右）。切换角色有以下重要意义：

1. **AI 效率**：玩家未操控的角色由 AI 控制。AI 会进行普通攻击和防御，但不会自动消耗 ATB 施放技能或治疗。因此需要频繁切换至需要积攒 ATB 的角色。

2. **战术配合**：克劳德近战牵制，切至爱丽丝远程施法，再切至巴雷特使用重火力。熟练的角色轮换是打出高伤害和压制 Boss 的关键。

3. **分散仇恨**：敌人通常追踪玩家当前操控的角色。当被敌人集火时，切换角色可使敌人转移目标，为濒危角色创造回复窗口。

4. **ATB 积累效率**：所有未操控角色都会随时间缓慢积累 ATB，但速度远不如玩家操控角色。高频切换确保队伍整体 ATB 产出最大化。

**快捷键提示**：使用战术模式时可直接选择队友执行指令，无需手动切换。但直接操控角色积累 ATB 效率更高。

## 魔晶石基础 / Materia Basics

魔晶石系统是本作角色定制的核心。魔晶石嵌入武器或防具的链接槽中，赋予角色魔法、能力或属性加成。

**初期魔晶石配置建议**：

- **克劳德**：装备「烈焰」+「雷霆」元素魔法，配合「回复」魔晶石作为副治疗
- **蒂法**：专注物理增伤，「属性」魔晶石链接「烈焰」使武器附火效果
- **爱丽丝**：全魔法输出，「烈焰」「寒气」「雷霆」「疾风」各一，配合「魔法追击」辅助
- **巴雷特**：装备「回复」+「复活」担任主治疗，「烈焰」用于远程属性打击

**魔晶石获取**：初期流程自动获得基础魔晶石。在武器商店可购买部分绿色（魔法）魔晶石。黄色的「支援」魔晶石和蓝色的「指令」魔晶石多通过宝箱或支线任务获得。紫色的「独立」魔晶石最为稀有，通常通过战斗报告或 Boss 掉落获取。

**魔晶石升级**：魔晶石通过战斗获得 AP 积累升级。每升至满级（Master）后可分裂出新的一级魔晶石，实现「复制」效果。在 Hard Mode 终局阶段这是打造最强配置的关键机制。

## 推荐初期魔晶石组合 / Early Materia Loadouts

**全队必备**：
- 至少 2 颗「回复」魔晶石（HEALING）：保证两人可治疗
- 至少 1 颗「复活」魔晶石（REVIVE）：战斗不能时必备
- 每名角色至少 1 种元素魔法（火/冰/雷/风）

**克劳德初期推荐**：烈焰 + 回复 + 提升HP + 铁壁
**蒂法初期推荐**：属性 + 烈焰（链接）+ 提升HP + 提升ATB
**爱丽丝初期推荐**：烈焰 + 寒气 + 雷霆 + 疾风 + 魔法追击
**巴雷特初期推荐**：回复 + 复活 + 提升HP + 烈焰

## 难度选项 / Difficulty Options

游戏提供三种难度模式，以及一个特殊的战斗模式选项：

**Easy（简单模式）**：敌人攻击力大幅降低，适合只想体验剧情的玩家。战斗失败风险极低，普通攻击伤害显著提升。自动回复魔晶石效果增强。

**Normal（普通模式）**：标准难度。需要合理运用格挡、闪避、ATB 管理和战术模式。Boss 战有灭团风险，但装备和策略得当即可顺利通关。适合大部分玩家。

**Classic（经典模式）**：独创难度选项。角色自动进行普通攻击与防御，玩家只需管理 ATB 指令施放技能/魔法。此模式旨在还原原版回合制 RPG 体验。注意：Classic 模式的敌人强度等同于 Normal。

**Hard（困难模式）**：通关后解锁。禁止使用道具，长椅不会恢复 MP，但 EXP/AP 获取量提升。需要深入了解战斗机制才能通关。详见「终局挑战」章节。

**新手建议**：如果系列新手，建议从 Normal 难度开始。前期流程较为线性，可逐步掌握战斗系统。遇到困难的 Boss 可使用战术模式仔细分析敌人弱点。Classic 模式适合怀念原版战斗的老玩家。

## 初期实用技巧 / Early Game Tips

1. **勤于「洞察」（Assess）**：获得「洞察」魔晶石后，每一场新敌人战斗都先使用洞察。了解敌人弱点属性、免疫状态和 break 条件后，战斗难度直线下降。

2. **利用地形要素**：某些场景中的环境物体（如燃料罐、吊灯）可被攻击引爆，对周围敌人造成大量伤害。这是早期效率最高的范围输出手段。

3. **箱子破坏**：场景中的蓝色和橙色箱子务必全部破坏。蓝色箱子含道具或金币，橙色箱子含魔晶石或装备。部分与奖杯相关的魔晶石仅通过箱子和特定宝箱获得。

4. **长椅回复的时机**：长椅可以完全恢复 HP 和 MP。在进入新区域前尽量找到长椅回满状态。Hard Mode 下长椅不会恢复 MP，因此 MP 管理变得至关重要。

5. **支线任务奖励**：完成支线任务可获得珍贵的道具、魔晶石和武器。每个章节可触发支线任务时优先完成。第八章和第九章的支线任务会影响与蒂法和爱丽丝的好感度。

6. **武器技能精通**：每把武器都有一个专属技能。在武器上装备该技能并在战斗中使用熟练度达到 100% 后，该技能将永久解锁，可以在任何武器上装备。尽量在获得新武器后立即练满技能。

7. **闪避 vs 防御**：闪避有无敌帧但较短，防御则提供稳定的减伤。面对无法预判的敌人攻击时，防御比闪避更可靠。铁壁魔晶石可以大幅提升防御性能。

8. **敌人弱点优先级**：先用物理攻击积攒 ATB，然后用弱点魔法打出力竭（Stagger）。力竭状态下的敌人受到的伤害大幅提升，是爆发输出的最佳时机。

## 第一章快速指引 / Chapter 1 Quick Guide

游戏的第一章是教学章节，也是玩家接触战斗系统的起点：

**任务目标**：克劳德和巴雷特潜入壹号魔晄炉安装炸弹。

**流程要点**：
- 开场的潜行教学：跟随巴雷特移动，在守卫背后按○键进行潜行击倒。这里不需要战斗，但练习潜行操作对后续章节有帮助
- 进入魔晄炉后遇到第一群敌人（警卫犬和神罗士兵）。这些敌人血量较低，用普通攻击即可轻松解决
- 继续深入后遇到第一个精英敌人「扫荡机」（Sweeper）。它弱雷属性，使用初期获得的雷霆魔晶石可以快速击杀
- 安装炸弹后触发蝎子哨兵 Boss 战。参战角色为克劳德和巴雷特
- Boss 战后逃离魔晄炉，沿路跑回出口即可——途中的敌人不需要全部击杀

**初始魔晶石**：本章中获得烈焰、寒气、雷霆和回复魔晶石（在克劳德武器上）。这是你最早可用的元素魔法。

## 按键操作说明 / Control Reference

**PlayStation 版**：
- □：普通攻击
- △：切换模式/特殊动作
- ○：互动/确认
- ×：闪避/翻滚
- L1：防御
- R1：按住后 + 对应按键发动指令快捷键
- L2/R2：切换锁定目标
- 触摸板：战术模式
- 选项键：暂停/菜单
- 方向键左右：切换角色
- 方向键上下：切换锁定目标

**PC 版（键鼠默认）**：
- 鼠标左键：普通攻击
- 鼠标右键：防御
- E：切换模式/特殊动作
- 空格：闪避
- T：战术模式
- 1-4：快捷指令
- WASD：移动
- Shift：奔跑
- Q：切换锁定
- F：互动

## 初期成长规划 / Early Progression Guide

前 3 章是游戏的「新手保护区」，建议在这个阶段做好以下准备：

**第 1 章**：熟悉基本操作。重点练习普通攻击→ATB 积累→技能释放的基础循环。不需要急于熟悉所有机制，掌握移动、攻击和治疗即可。

**第 2 章**：角色切换开始变得重要。在爱丽丝加入后尝试在克劳德和爱丽丝之间切换战斗。爱丽丝是远程角色，保持距离施放魔法；克劳德近战牵制。

**第 3 章**：第一个自由探索章节。完成所有 6 个支线任务——这将奖励铁壁魔晶石和大量经验。购买武器升级材料开始升级武器。在第柒区武器店购买钢铁剑（克劳德）和六式（巴雷特）。这是你第一次可以自由配置魔晶石，花时间阅读每个魔晶石的说明。

**经济管理**：
- 初期金币（Gil）比较紧张，优先购买武器和魔晶石
- 恢复剂可以在箱子里大量获得，不需要购买太多
- 以太价格昂贵但前期用得少，遇到打折时再购入
- 支线任务的金币奖励很丰厚，务必完成

## 常见新手陷阱 / Common Beginner Mistakes

1. **忽视防御**：很多玩家在初期的杂兵战中形成不防御的习惯，导致在 Boss 战中频繁被秒。请记住——防御是战斗的基础操作，不是万不得已才用的手段。

2. **只操控一个角色**：不少新手从头到尾只操控克劳德。这会导致其他角色的 ATB 严重不足，Boss 战中回复不及时。养成每 5-10 秒切换一次角色的习惯。

3. **浪费 ATB**：ATB 满格后不使用并不会产生额外收益。看到 ATB 有两格满时立即使用技能或魔法。ATB 溢出等于浪费输出。

4. **忽视魔晶石升级**：魔晶石通过战斗获得 AP 升级。初期魔晶石等级低时效果不明显，但多使用魔法就能快速升级。满级后的魔晶石可以分裂出新的，是实现「人手一颗回复」的关键。

5. **不使用洞察**：洞察是游戏中最有价值的指令之一。很多 Boss 的弱点不是显而易见的，不使用洞察就只能靠猜测或试错。获得洞察魔晶石后请在每场新战斗中使用它。

6. **Boss 战前不调整配置**：每个 Boss 的弱点和抗性不同。进入 Boss 战前根据洞察到的信息调整魔晶石配置，可以大幅降低战斗难度。Boss 房间前的长椅就是提醒你「调整配置」的信号。
