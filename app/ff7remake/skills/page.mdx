## 魔晶石系统综述 / Materia System Overview

魔晶石（Materia）是《最终幻想 VII》系列标志性的角色定制系统。在《重制版》中，魔晶石通过嵌入武器和防具的魔晶石槽位来赋予角色能力。魔晶石分为四种颜色类型：

1. **绿色——魔法魔晶石（Magic Materia）**：提供元素魔法、恢复魔法和状态魔法
2. **黄色——支援魔晶石（Support Materia）**：提供被动增益和特殊机制
3. **蓝色——指令魔晶石（Command Materia）**：提供额外战斗指令
4. **紫色——独立魔晶石（Independent Materia）**：提供属性提升和特殊效果

魔晶石带有「AP」（Ability Points）经验值系统。在战斗中装备魔晶石的武器或防具获得击杀/战斗 AP 后，魔晶石会升级。满级（Master）的魔晶石可以分裂出新的 1 级魔晶石，从而实现「无限复制」。

## 绿色魔法魔晶石 / Green Magic Materia

**元素魔法系列**：

- **烈焰（Fire）** → 大火焰（Fira） → 爆火焰（Firaga）
  - AP 需求：100/300/600
  - 效果：单体火属性伤害。克制冰弱点的敌人
  - 推荐度：★★★★★（全队必备）

- **寒气（Blizzard）** → 暴雪（Blizzara） → 大暴雪（Blizzaga）
  - AP 需求：100/300/600
  - 效果：单体冰属性伤害。克制火弱点的敌人
  - 推荐度：★★★★★（全队必备）

- **雷霆（Thunder）** → 大雷电（Thundara） → 爆雷（Thundaga）
  - AP 需求：100/300/600
  - 效果：单体雷属性伤害。克制机械类敌人
  - 推荐度：★★★★★（全队必备）

- **疾风（Aero）** → 烈风（Aerora） → 暴风（Aeroga）
  - AP 需求：100/300/600
  - 效果：单体风属性伤害。克制飞行类敌人
  - 推荐度：★★★★（部分战斗必备）

**恢复魔法系列**：

- **回复（Healing）** → 中回复（Cura） → 大回复（Curaga）
  - AP 需求：100/300/600
  - 效果：单体 HP 恢复魔法。基础恢复量随魔法攻击力增加
  - 推荐度：★★★★★（全队至少 2 人装备）

- **复活（Revive）** → 全复活（Arise）
  - AP 需求：200/500
  - 效果：战斗不能恢复。Arise 版本回复约 70% HP
  - 推荐度：★★★★★（全队至少 1 人装备，推荐 2 人）

**状态魔法系列**：

- **沉默（Silence）** → 全沉默（Silencera）
  - 效果：使敌人进入沉默状态，无法使用魔法
  - 推荐度：★★★（针对法师型敌人）

- **毒化（Bio）** → 剧毒（Biora） → 猛毒（Bioga）
  - 效果：使敌人进入中毒状态，持续掉血
  - 推荐度：★★★（对 HP 极高的 Boss 有效）

- **停止（Stop）** → 全停止（Stopra）
  - 效果：使敌人停止行动
  - 推荐度：★★★★（对可停止的敌人极具压制力）

## 黄色支援魔晶石 / Yellow Support Materia

**属性（Elemental）**：链接到属性魔法时，普通攻击附带该属性。链接到防具时，对应属性伤害减半或无效化。这是最重要的支援魔晶石之一，可通过第 9 章围墙商业街支线获得。

**HP 吸收（HP Absorb）**：链接到进攻魔法时，魔法伤害的一部分转化为 HP 恢复。在 Hard Mode 下非常实用。

**MP 吸收（MP Absorb）**：链接到进攻魔法时，魔法伤害的一部分转化为 MP 恢复。价格高昂但堪称永动机核心。

**魔法追击（Magic Focus）**：魔法暴击率提升，满级时大幅提升。建议给爱丽丝装备。

**ATB 联动（ATB Assist）**：使用某角色攻击后增加其他角色的 ATB 槽。团队配合的核心魔晶石。

**自动疗伤（Auto-Cure）**：HP 低于 25% 时自动消耗 ATB 使用回复魔法。新手保护向。

**铁壁（Steadfast Block）**：防御时大幅减少伤害，额外积累 ATB。克劳德在 Hard Mode 的生存保障。

## 蓝色指令魔晶石 / Blue Command Materia

**洞察（Assess）**：查看敌人信息。重要性无需赘述——每一场战斗的第一指令。

**跳跃（Jump）**：角色跳跃至空中后下坠攻击，期间免疫地面攻击。对某些 Boss 的特定技能（如房地狱的家具投掷）可用来规避。

**投掷（Throw）**：投掷武器造成伤害。对远程角色巴雷特的补充输出手段。

**偷窃（Steal）**：从敌人身上偷取道具。部分稀有道具只能通过偷窃获得。第 3 章支线任务获得。

**挑战（Challenge）**：嘲讽技能，使敌人强制攻击使用者。巴雷特的核心坦克手段。

**祈祷（Prayer）**：消耗 1 ATB 格治疗全队少量 HP。无 MP 消耗，是 Hard Mode 最重要的回复手段。通过查德利战斗报告获得。

**气势（Chakra）**：消耗 1 ATB 格治疗自身 HP 并解除一个异常状态。同样无 MP 消耗，Hard Mode 必备。

## 紫色独立魔晶石 / Purple Independent Materia

**提升 HP（HP Up）**：增加最大 HP。满级 +1000 HP。全队全员推荐装备。

**提升 MP（MP Up）**：增加最大 MP。满级 +40 MP。法师角色推荐。

**提升 ATB（ATB Up）**：普通攻击额外增加 ATB 获取量。输出角色核心魔晶石。

**提升 Limit（Limit Up）**：Limit 槽积累速度提升。适合容易受伤的前排角色。

**魔法追击（Magic Up）**：魔法伤害提升。满级 +10% 魔法伤害。爱丽丝核心。

**物理追击（Physical Up）**：物理伤害提升。满级 +10% 物理伤害。克劳德/蒂法核心。

**射击追击（Shooting Up）**：远程射击伤害提升。巴雷特核心。

**SP 提升（SP Up）**：SP 获取量增加。升级武器专用。

## 推荐魔晶石搭配 / Recommended Loadouts

### 克劳德——均衡战士 Build

- **武器**：破甲大剑（High ATK）或诸神黄昏（多魔晶石槽）
- **魔晶石**：烈焰 + 雷霆（元素覆盖）/ 铁壁（生存核心）/ 提升 HP ×2 / 提升 ATB / 魔法追击（选）/ 回复（副治疗）
- **战术定位**：前线牵制 + 持续输出，危急时切换至防御姿态

### 蒂法——力竭爆发 Build

- **武器**：神拳（最高力竭积累）
- **魔晶石**：属性 + 烈焰（附火）/ 提升 HP ×2 / 提升 ATB / 物理追击 ×2 / 气势（自恢复）
- **战术定位**：力竭前全力积攒 ATB，力竭后利用正拳/爆裂拳堆叠倍率

### 爱丽丝——纯魔法输出 Build

- **武器**：华丽杖（最高魔攻）/ 奇迹杖（生存）
- **魔晶石**：烈焰 + 寒气 + 雷霆 + 疾风（全元素覆盖）/ 魔法追击 ×2 / 提升 HP / 提升 MP / HP 吸收 / 魔法阵
- **战术定位**：远程魔法炮台，魔法阵双发输出核心

### 巴雷特——坦克治疗 Build

- **武器**：毁灭者（最高 HP）/ 大口径炮（防御）
- **魔晶石**：回复 + 复活（主治疗）/ 提升 HP ×3 / 提升 ATB / 铁壁 / 挑战（AI 自动用）
- **战术定位**：后排治疗 + 坦克，雪崩之魂吸收伤害

## 武器技能精通 / Weapon Proficiency Skills

每把武器都有一个专属技能。装备该武器后在战斗中使用该技能，使用次数达到一定后技能熟练度达到 100%，该技能永久对所有武器开放。

**技能精通重点**：
- 克劳德的「破晄击」和「无尽终结」优先精通
- 蒂法的「正拳」和「爆裂拳」优先精通
- 爱丽丝的「魔法阵」和「光之盾」优先精通
- 巴雷特的「雪崩之魂」和「过热」优先精通

## 召唤魔晶石 / Summon Materia

召唤魔晶石属于紫色独立魔晶石的一种特殊类型。装备后可随时间自动积累召唤槽，满槽后可在战斗中召唤召唤兽。

**召唤兽等级对比**：

| 召唤兽 | 获得难度 | 必杀技评价 | 推荐度 |
|--------|----------|-----------|--------|
| 卡邦库尔 | 系统/特典 | ★★★ | 辅助型，适合新手 |
| 仙人掌怪 | 系统/特典 | ★★★★ | 高伤害单体 |
| 陆行鸟 | 系统/特典 | ★★★ | 恢复型 |
| 湿婆 | ★☆☆ | ★★★★ | 冰属性强力输出 |
| 肥陆行鸟 | ★★★ | ★★★ | 场地控制型 |
| 利维亚桑 | ★★★★ | ★★★★★ | 强力 AOE |
| 巴哈姆特 | ★★★★★ | ★★★★★ | 全游戏最强召唤 |
| 伊弗利特 | 剧情自动 | ★★★★ | 火属性高输出 |

**推荐召唤兽使用**：
- 普通战斗：仙人掌怪（快速清场）
- Boss 战：巴哈姆特 / 利维亚桑（高伤害）
- Hard Mode：巴哈姆特（无消耗爆发输出）
