## 困难模式概览 / Hard Mode Overview

Hard Mode（困难模式）是《最终幻想 VII 重制版》通关后解锁的最高难度模式。在章节选择中选择任意章节后，在难度设置中选择「HARD」即可开启。这不只是一种难度提升，而是对整个游戏策略的彻底重构。

**Hard Mode 核心限制**：
1. **禁止使用道具**：战斗中无法使用恢复剂、以太、凤凰之尾等消耗品。只有魔晶石技能和角色自带技能可用于恢复
2. **长椅不回复 MP**：长椅（Bench）仅恢复 HP，MP 完全不变。这意味着每一场战斗的 MP 都必须精打细算
3. **敌人强度大幅提升**：所有敌人的 HP、攻击力、防御力显著增加。Boss 获得新技能和新阶段模式
4. **EXP/AP 获取量提升**：击杀经验值和魔晶石 AP 获取量约为 Normal 的 2-3 倍
5. **新物品获得**：部分高级魔晶石和饰品的唯一获取途径

**为什么挑战 Hard Mode**：
- 解锁最强配件「诸神黄昏」（Gotterdammerung）
- 获得「巴哈姆特」召唤魔晶石
- 解锁 INTERmission DLC 的战斗模拟器最高难度
- 完成所有战报获取白金奖杯必要条件
- 体验真正的战术深度——逼迫玩家掌握战斗系统

## 困难模式核心策略 / Hard Mode Core Strategies

### 回复策略（无道具环境）

Hard Mode 下唯一的回复来源是魔晶石技能：
- **祈祷（Prayer）**：消耗 1 ATB，全队恢复约 30% HP。无 MP 消耗，是 Hard Mode 最常用的回复技。至少 2 名角色装备。
- **气势（Chakra）**：消耗 1 ATB，单体恢复约 40% HP + 解除一个异常状态。无 MP 消耗，前排角色必备。
- **回复魔法（Cure/Cura/Curaga）**：消耗 MP 的高额单体回复。用于紧急救场——不要随意使用，因为 MP 无法回复。
- **爱丽丝的招式（Healing Wind 等）**：部分角色技能含有回复效果，合理利用。

### MP 管理策略

MP 是 Hard Mode 最紧张的资源：
- **核心原则**：战斗中尽量不使用 MP，只在 Boss 战的关键时刻使用
- **MP 恢复方式**：只有击杀敌人/魔晶石升级时才少量恢复 MP（魔晶石满级时）
- **推荐做法**：Boss 前的杂兵战完全使用物理攻击和无 MP 技能解决
- **魔晶石选择**：「MP 吸收」链接到攻击魔法可以在命中时恢复少量 MP
- **武器升级**：优先升级增加 MP 上限和 MP 恢复速度的节点
- **爱丽丝的 MP 管理**：作为主力魔法输出，优先将节能节点点到最高

### 魔晶石策略

- **全队「提升 HP」×2**：大幅增加生存能力。Hard Mode 中敌人的攻击力可以瞬间将未加 HP 的角色打成濒死。
- **全队「铁壁」**：防御的重要性在 Hard Mode 中翻倍。铁壁的减伤效果是生存的保障。
- **「祈祷」+「气势」**：至少 3 人装备其中之一
- **「属性+元素魔法」（防具链接）**：根据章节点 Boss 调整——例如针对房地狱的火焰将烈焰链接到防具上
- **「洞察」重新使用**：Hard Mode 中部分 Boss 的弱点发生了变化，必须重新使用洞察确认
- **魔法阵的使用**：爱丽丝的魔法阵在 Hard Mode 中极其重要——一次 ATB 换两次魔法，MP 效率翻倍

## 战斗模拟器挑战 / Battle Simulator Challenges

第 17 章神罗总部的战斗模拟器（另可通过查德利挑战）提供了 Hard Mode 的高难度挑战。

### 查德利战斗报告 1-7

基础战斗报告，在 Normal 或 Hard Mode 均可完成：
1. 消灭 10 种怪物（自动进行）
2. 完成 5 次洞察
3. 使用 20 次技能
4. 使用 10 次魔法
5. 力竭 10 次
6. 击败召唤兽（湿婆）
7. 击败召唤兽（肥陆行鸟）

**奖励**：召唤魔晶石、魔晶石槽解锁、SP 点数

### 查德利战斗报告 8-14

进阶报告，推荐在 Hard Mode 完成：
8. 击败召唤兽（利维亚桑）
9. 击败召唤兽（巴哈姆特）
10. 力竭 30 次
11. 使用 50 次技能
12. 完成 VR 三连战
13. 击败特殊召唤兽
14. 全战斗报告积累

**奖励**：高级魔晶石（魔法追击、MP 吸收等）

### 查德利战斗报告 15-20：Top Secrets 挑战

这是游戏中最困难的战斗内容：
- **Top Secrets 1**：连续挑战 3 个强化 Boss 战（湿婆+肥陆行鸟+利维亚桑）
- **Top Secrets 2**：连续挑战 4 个强化 Boss 战（巴哈姆特+伊弗利特+别祖+雷诺&路德）
- **Top Secrets 3**：连续挑战 5 个 Boss（以上全部）
- **报告 20 奖励**：诸神黄昏（Gotterdammerung）——开场 Limit 槽全满的最强配件

**Top Secrets 推荐阵容**：克劳德（主输出）+ 爱丽丝（魔法爆发）+ 巴雷特（治疗+坦克）
**推荐魔晶石**：全队祈祷、艾瑞丝带魔法阵 + HP 吸收、克劳德带铁壁 + 提升 ATB。注意 Top Secrets 的 Boss 连续战斗无法在中间更换魔晶石，必须一穿到底。

## 诸神黄昏攻略 / Gotterdammerung Strategy

诸神黄昏是 Hard Mode 的最终目标，也是白金奖杯的钥匙之一：

**效果**：战斗开始时 Limit 槽立即全满。携带此配件的角色可在开场后直接使用极限技。

**获取方法**：
1. 通关一次游戏（解锁章节选择和 Hard Mode）
2. 在 Hard Mode 下通关所有 18 章（可选，非强制）
3. 完成查德利全部 20 项战斗报告
4. 报告 20 完成后自动获得

**使用建议**：
- 克劳德携带：开场使用「凶斩」快速力竭精英敌人
- 爱丽丝携带：开场使用「惑星守护」保证全队生存
- 巴雷特携带：极限技的高伤害通用方案
- 同时获得 2 个诸神黄昏后，克劳德+爱丽丝各带一个效果最佳

## 白金奖杯攻略 / Platinum Trophy Guide

### 全奖杯列表摘要（共 54 个奖杯）

**白金奖杯（1）**：
- **《最终幻想 VII 重制版》大师**：获得所有奖杯

**金杯（2）**：
1. **毁灭之力**：习得全部角色所有武器技能
2. **战斗大师**：完成所有战斗报告

**银杯（10）**：
1. **更进一步**：使角色 Limit 技能达到 Lv.2
2. **肌肉逞威风**：完成所有引体向上挑战
3. **破箱英雄**：破坏所有箱子
4. **音乐爱好者**：获得全部唱片
5. **飞毛腿**：完成全部摩托追逐时间挑战
6. **敬爱的**：让任意角色好感度达到最高（蒂法/爱丽丝）
7. **毁灭使者**：Hard Mode 通关第 18 章
8. **召唤大师**：获得全部召唤魔晶石
9. **战斗报告者**：全部 20 项战斗报告完成
10. **武器收藏家**：获得全部武器

**其他铜杯（41）**：涵盖剧情解锁、战斗成就、收集要素等。

### 白金攻略路线

**第一步：Normal 通关（约 40 小时）**
- 自由探索，收集所有可见宝箱
- 完成全部 26 个支线任务
- 完成战斗报告 1-14
- 尽量多的使用武器技能以积累精通度
- 收集唱片和破坏箱子（同步进行）

**第二步：Hard Mode 通关（约 25-30 小时）**
- 所有 Boss 都变强，需要针对性策略
- 使用无 MP 消耗的「祈祷」和「气势」代替道具
- 每个 Boss 战前用洞察重新确认弱点
- 利用章节选择补漏支线和收集品
- 完成战斗报告 15-20（Top Secrets）

**第三步：收尾补完（约 5-10 小时）**
- 补完所有剩余的武器技能精通
- 补完唱片收集（共 9 首，分布在各个区域）
- 完成引体向上挑战（蒂法篇）
- 完成摩托追逐时间挑战
- 确认所有箱子已破坏

**特别注意**：「破坏所有箱子」奖杯容易遗漏。全游戏共 90 个箱子，分布在 18 个章节中。建议使用章节选择逐一排查。其中第 9 章围墙商业街的箱子最容易忽略。

## INTERmission DLC / Yuffie Episode

PS5 和 Steam 平台独占的 INTERmission 是发生在主线剧情旁支的故事。尤菲·如月（Yuffie Kisaragi）作为主角登场，携带着她的搭档索侬（Sonon）潜入米德加执行机密任务。

**INTERmission 特点**：
- 游戏时间：约 2-3 小时
- 可操控角色：尤菲 + 索侬
- 新增协同攻击系统（Synergy Abilities）
- 新增 2 种召唤兽：拉姆（Ramuh）
- 新增战斗模拟器挑战
- 尤菲的忍术附魔系统可以自由切换属性

**INTERmission 奖杯（追加）**：
- 追加 6 个新奖杯（1 金 1 银 4 铜）
- 包括：Hard Mode 通关 INTERmission、完成所有尤菲篇模拟器挑战、获得最强武器等

## Hard Mode Boss 特殊攻略 / Hard Mode Boss Notes

### 百耳塞（Hard）：
新增全屏电击技能「电磁毁灭」，必须在场地中的三个掩体间来回切换。弱雷不变，但抗性大幅提升。建议使用「魔法阵+大雷电」快速积累力竭。

### 房地狱（Hard）：
弱点变换速度加快一倍，约 2 秒后切换。建议全队携带全 4 种元素魔晶石。蒂法装备「属性+烈焰」附火，克劳德装备「属性+寒气」附冰。爱丽丝负责补刀剩余两种。使用铁壁减少家具投掷伤害。

### 路法斯·神罗（Hard）：
黑暗恶犬新增全场追击技能。路法斯的「神罗的骄傲」护盾 HP 翻倍，破盾后伤害窗口缩短。克劳德的「反击架势」在此战中价值极高——可以反制路法斯的大部分近战攻击。推荐装备「铁壁」+「回复」（装备在防具上）。

### 杰诺瓦（Hard）：
新增「杰诺瓦的领域」技能，全队受到持续魔法伤害。必须在限定时间内脱离该区域。「离心力场」变为追踪型。爱丽丝的魔法阵必须在脱离领域后使用，否则会被立即打断。

### 萨菲罗斯（Hard）：
超新星技能变为必杀级，不防御则秒杀非满血角色。「逆天的审判」的限定时间缩短。第三阶段新增「陨石」技能，场地大范围落石。建议全队装备「提升 HP」x2，保证血量在 6000+。Limit 技保留至最后阶段。

## Hard Mode 小结 / Final Thoughts on Hard Mode

Hard Mode 虽然挑战性极高，但绝非不可能。它逼迫玩家真正理解战斗系统的每一个细节——从 ATB 管理到力竭机制，从魔晶石搭配到角色切换。当你终于击败 Hard Mode 的萨菲罗斯，获得白金奖杯时，那种成就感是独一无二的。记住：耐心、策略、练习——Hard Mode 的三大支柱。
