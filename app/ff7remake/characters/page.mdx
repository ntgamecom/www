## 克劳德·斯特莱夫 / Cloud Strife

**类型**：均衡近战型 / Balanced Melee Fighter
**武器**：破坏剑（Buster Sword）、秘银剑（Mythril Saber）、钢铁剑（Iron Blade）、狼牙棒（Nail Bat）、破甲大剑（Hardedge）、诸神黄昏（Twilight Sickle）

克劳德是队伍的领袖，定位为全能战士。他的普通攻击速度适中、伤害稳健，是稳定的 ATB 积攒来源。其核心机制为「勇猛模式」（Punisher Mode）——长按□键进入后，攻击变为高伤害的连击，且可自动反击近身物理攻击。

**勇猛模式详解**：在勇猛模式下，克劳德的移动速度降低，但每次攻击造成额外伤害并大幅积累力竭值。勇猛模式下被近战攻击时自动反击，击退敌人并造成硬直。切换至勇猛模式的瞬间也有短暂无敌帧。建议在战斗中频繁切换普通模式（Operator Mode，灵活移动）与勇猛模式（输出爆发）以获得最佳效果。

**专属技能**：
- 奋力斩击（Braver）：高单体伤害，初期主要输出技能
- 破晄击（Cross-slash）：两段斩击使敌人停止，对有护盾的敌人效果极佳
- 无尽终结（Infinity's End）：克劳德最强单体技能，伤害极高但出招慢，适合在敌人力竭时使用
- 凶斩（Triple Slash）：追踪三段斩击，打小型敌人极其有效

**使用技巧**：克劳德的作战思路是利用普通模式的机动性接近敌人，切换勇猛模式打出硬直和伤害，消耗 ATB 使用技能。面对多目标时，勇猛模式的范围攻击非常高效。「反击架势」技能（Counterstance）是一招强力的反击技，能在防御状态下自动反击任意攻击，是克劳德在 Hard Mode 的核心生存技。

## 蒂法·洛克哈特 / Tifa Lockhart

**类型**：高速近战 DPS / Fast Melee DPS
**武器**：皮革手套（Leather Gloves）、金属指虎（Metal Knuckles）、索林（Soul Saber）、音速拳套（Sonic Knuckles）、 premium（Feathered Gloves）、神拳（God's Hand）

蒂法是队伍的近战爆发手。她的攻击速度是全角色中最快的，连击节奏紧凑，能在极短时间内打出大量伤害并迅速积攒 ATB。她的核心机制是「气卦」（Chi）系统——使用技能消耗气卦层数以增强爆发。

**气卦机制详解**：部分蒂法技能拥有「提升力竭倍率」的特殊效果。当蒂法在敌人力竭状态下使用「正拳」（Whirling Uppercut）或「爆裂拳」（Omnistrike）时，可增加目标的力竭伤害倍率（Stagger Damage Multiplier），最高可提升至 200% 以上。这意味着蒂法的连招可以直接放大全队的力竭期爆发伤害。

**专属技能**：
- 正拳（Whirling Uppercut）：快速上勾拳，气卦消耗技，可提升力竭倍率
- 爆裂拳（Omnistrike）：多段连续打击，对力竭状态敌人效果翻倍
- 治疗（Healing Arc）：有恢复效果的扇击，适合蒂法担任副治疗时的选择
- 空翻踢（Tifa's Somersault）：空中攻击，落地追加一段震地攻击
- 最终天堂（Final Heaven）：蒂法最强技能，高伤害单体必杀技

**使用技巧**：蒂法的核心输出窗口在敌人力竭之后。在力竭前以积攒 ATB 为主，力竭后由蒂法先使用正拳/爆裂拳堆叠力竭倍率，然后切换至克劳德或爱丽丝使用高伤害技能进行收割。蒂法的「空翻踢」自带移动属性，可用于快速接近敌人。

## 爱丽丝·盖恩斯巴勒 / Aerith Gainsborough

**类型**：远程魔法输出 / Ranged Magic Caster
**武器**：守护杖（Guard Stick）、银之杖（Silver Staff）、秘银杖（Mythril Rod）、魔法杖（Arcane Scepter）、华丽杖（Imperial Staff）、奇迹杖（Umbrella）

爱丽丝是队伍中唯一的纯法师角色。她的普通攻击为远程魔法弹，可在安全距离持续输出。她的核心机制为「魔法阵」（Arcane Ward）——在场上展开一个魔法阵，站在其中的角色在施放魔法后可免费释放第二次相同魔法。

**魔法阵战略价值**：这是全游戏最强的 buff 技能之一。将魔法阵放置在后排，爱丽丝站在其中施放顶级元素魔法（如大暴雪、大雷电），一次 ATB 消耗可触发两次魔法，爆发极高。近战角色站在魔法阵中也可受益，例如克劳德在魔法阵中使用大火焰可打出双倍魔法伤害。

**专属技能**：
- 暴风雨（Tempest）：蓄力远程魔法攻击，蓄满力后变成范围技
- 神圣之力（Holy）：布置一个持续恢复区域的技能
- 光之盾（Lustrous Shield）：召唤一面可阻挡远程攻击的光盾
- 审判之箭（Ray of Judgment）：爱丽丝最强输出技，大范围魔法打击

**使用技巧**：爱丽丝是队伍中最脆弱的角色，需要保持距离。利用光之盾保护自身，在魔法阵中提供恐怖的魔法爆发。爱丽丝的 ATB 积攒速度是所有角色中最慢的，因此可以多用「暴风雨」积攒 ATB。在面对魔法弱点的敌人时，爱丽丝是绝对的核心输出。建议给她装备「魔法追击」（Magic Focus）魔晶石来进一步提升魔法伤害。

## 巴雷特·华莱士 / Barret Wallace

**类型**：远程坦克 / Ranged Tank
**武器**：加特林机炮序章（Gatling Gun）、六式（Six-shooter）、加农炮（Light Machine Gun）、大口径炮（Big Bertha）、特制机炮（Revolver）、毁灭者（Cannon Ball）

巴雷特是队伍中最坚实的肉盾，同时也是可靠的远程伤害输出者。他的普通攻击是远程连射，可在各种距离安全输出。他的核心机制是「过热」（Overcharge）——通过消耗 ATB 进入过热状态，大幅提升攻击力。

**过热机制详解**：使用「过热」技能后，巴雷特的普通攻击变为更强力的连射（射击次数增加），积攒 ATB 的效率也显著提升。结合「毁灭连击」（Point Blank）等技能，可在短时间内造成可观的远程爆发伤害。

**专属技能**：
- 爆燃射击（Flame Strike）：大伤害的单发狙击，远程狙击精英
- 毁灭连击（Point Blank）：近距离散射，伤害随距离递减
- 雪崩之魂（Lifesaver）：吸收队友部分伤害的嘲讽技能
- 旋转风暴（Charging Circle）：持续范围攻击的旋转射击
- 增压（Rage）：巴雷特最强技能，大幅提升过热状态下的攻击力

**使用技巧**：巴雷特的位置在队伍后方。利用「雪崩之魂」吸引敌人仇恨，保护脆弱的爱丽丝。巴雷特是主治疗角色的最佳人选——他可以在安全的后排持续输出，同时为全队回复血量。面对飞行敌人时，巴雷特是不可或缺的。建议给他装备「回复」和「复活」魔晶石，让巴雷特成为队伍的医疗兵。

## 赤红十三 / Red XIII

**类型**：近战敏捷型（客座角色）/ Agile Melee Fighter (Guest Character)
**出现章节**：第 17-18 章

赤红十三（原名 Nanaki）是一名具有高度智慧的野兽型角色。在本作中他作为客座角色（Guest Character）加入队伍，仅在神罗总部逃脱和最终关卡中可操控。他的战斗风格兼具克劳德的均衡与蒂法的速度。

**独特机制**：赤红的 ATB 技能「守护徽章」（Guard & Counter）可在防御同时自动反击近战攻击。他的普通攻击为三段连击，速度极快。「狂战士模式」（Vengeance Mode）状态下攻击力和 ATB 积攒速度大幅提升。

**专属技能**：
- 突进（Sled Fang）：向前冲刺攻击，追击效果极佳
- 星尘射线（Stardust Ray）：高伤害的远程魔法追击技
- 月牙爪（Lunar Fang）：连续爪击造成高力竭积累
- 大地怒嚎（Howling Moon）：赤红十三最强技能，大范围伤害

**使用技巧**：由于赤红十三在本作中仅可在最后两章使用，玩家没有太多时间熟悉他。建议将他定位为近战副输出，装备元素魔法魔晶石利用其高速度打属性弱点。他在 Hard Mode 的神罗总部章节攻略中是非常重要的战力。

## 队伍协同策略 / Team Synergy Strategies

### 标准均衡队：克劳德 + 蒂法 + 爱丽丝
这是最通用的队伍配置，适合绝大多数战斗场景。克劳德担任前线主坦，吸引仇恨同时稳定输出。蒂法作为副输出和力竭倍率放大器，在敌人力竭时提供爆发。爱丽丝在后排提供魔法伤害全支援。此阵容的弱点是没有专职坦克，克劳德的生存压力较大，需要频繁切换蒂法分担仇恨。

### 坦克魔法队：克劳德 + 巴雷特 + 爱丽丝
适合面对高伤害 Boss 的保守配置。巴雷特使用「雪崩之魂」吸收团队伤害，为爱丽丝创造安全的施法环境。克劳德在巴雷特吸引仇恨后尽情输出。爱丽丝作为主伤害来源，利用魔法阵提供双倍魔法。此阵容的输出窗口是巴雷特过热后的爆发期。缺点是 DPS 偏低，战斗时间较长。

### 双近战速攻队：克劳德 + 蒂法 + 巴雷特
纯物理队伍，适合面对魔法抗性高的敌人。巴雷特在后排提供支援和治疗，克劳德和蒂法在前线交替输出。双近战的优势在于力竭积累速度极快——克劳德的勇猛模式和蒂法的快速连击可以迅速将敌人打入力竭状态。缺点是对飞行敌人和魔法弱点敌人乏力。

### 爱丽丝 + 蒂法 核心组合
这是游戏中后期最强力的爆发组合。爱丽丝在地面展开魔法阵，蒂法将敌人打入力竭后堆叠倍率，切换克劳德使用无尽终结或爱丽丝使用双发大魔法收割。此组合的核心是精确的时机配合——需要在力竭窗口内完成堆叠倍率和输出两个步骤。

## 角色好感度系统 / Affection System

游戏中的部分章节选择会影响蒂法和爱丽丝对克劳德的好感度。好感度不影响主线剧情结局，但会影响第 14 章的特殊场景以及第 18 章的特定过场动画。

**影响好感度的主要因素**：
- 第 3 章支线任务完成情况（蒂法好感度）
- 第 8 章支线任务完成情况（爱丽丝好感度）
- 第 14 章支线任务选择（影响两人）
- 特定对话选项（两者各有关键选择）

**好感度奖励**：
- 高好感度角色的专属约会场景（第 14 章）
- 最终章的特定剧情节面差异
- 奖杯「敬爱的」需要任意角色好感度达到最高

## 武器技能精通指南 / Weapon Proficiency Guide

武器技能精通是本作角色成长的重要部分。每把武器都有一个独一无二的技能，使用该武器在战斗中反复使用这个技能即可提升熟练度。

**快速练技方法**：
1. 获得新武器后立即装备，在下一场战斗中只使用该武器的专属技能
2. 面对低级敌人时可以故意不击杀，反复使用技能直到熟练度满
3. 利用第 3 章和第 8 章的战斗模拟器快速刷熟练度
4. 部分技能（如克劳德的奋力斩击）ATB 消耗仅 1 格，可以更频繁地使用

**优先精通推荐**：
- **克劳德**：破晄击 → 无尽终结 → 凶斩
- **蒂法**：正拳 → 爆裂拳 → 最终天堂
- **爱丽丝**：魔法阵 → 光之盾 → 审判之箭
- **巴雷特**：雪崩之魂 → 过热 → 毁灭连击

**精通后的技能使用**：
精通后的技能不会消失，在任何武器上均可使用。这意味着你可以在最强武器上使用其他武器的专属技能。例如在破甲大剑（最高攻击力）上使用破晄击（高力竭伤害技能），或者在神拳上使用最终天堂。这是最大化角色战斗力的核心机制。
