## 幸存者概述 / Survivor Overview

《雨中冒险2》共有11名可玩幸存者，每个角色都有独特的技能组合和玩法风格。部分角色需要完成特定成就才能解锁。

*Risk of Rain 2* features 11 playable survivors, each with unique skill sets and playstyles. Some characters require completing specific achievements to unlock.

## 突击队员 / Commando

| 属性 / Stats | 数值 / Value |
|---|---|
| 生命值 / HP | 110 |
| 生命恢复 / HP Regen | 1 HP/s |
| 移动速度 / Speed | 7 m/s |
| 护甲 / Armor | 0 |

**技能 / Skills:**
- **普通攻击 / Primary: Double Tap** — 快速双发子弹 / Fast double shot
- **次要攻击 / Secondary: Phase Round** — 穿透弹 / Piercing round
- **技能1 / Utility: Tactical Dive** — 翻滚闪避 / Dodging roll
- **技能2 / Special: Suppressive Fire** — 全自动扫射 / Full-auto barrage

**玩法技巧 / Playstyle Tips:**
突击队员是最平衡的角色，适合任何Build。利用翻滚躲避攻击，保持中距离持续输出。道具方面优先攻击速度和暴击。

Commando is the most balanced survivor, suitable for any build. Use Tactical Dive to avoid attacks and maintain mid-range sustained damage. Prioritize attack speed and crit items.

**最佳道具协同 / Best Item Synergies:**
- 攻击速度类 / Attack Speed items — 大幅提升DPS
- 暴击类 / Crit items — 稳定暴击收益高
- 破甲类 / Armor Shredding — 提升全队输出
