## 如何开始 / How to Start

启动游戏后，主菜单提供以下选项 / After launching the game, the main menu offers:

- **开始游戏 / Play** — 单人游戏或创建联机房间
- **多人游戏 / Multiplayer** — 加入或创建联机房间
- **幸存者选择 / Select Survivor** — 在开始前选择角色
- **道具日志 / Logbook** — 查看已解锁的道具和怪物信息
- **设置 / Settings** — 调整画面、音频和控制选项

- **Play** — Start a solo game or host a multiplayer lobby
- **Multiplayer** — Join or host an online co-op session
- **Select Survivor** — Choose your character before starting
- **Logbook** — View unlocked items and monster lore
- **Settings** — Adjust graphics, audio, and controls

## 难度系统 / Difficulty System

### 时间驱动的难度 / Time-Based Scaling

《雨中冒险2》的难度随游戏时间线性增长。难度等级分为三个阶段：

*Risk of Rain 2* difficulty scales linearly with elapsed time. There are three difficulty tiers:

- **简单 / Easy** — 游戏开始起 5 分钟内 / First 5 minutes of the game
- **中等 / Medium** — 5 到 20 分钟 / 5 to 20 minutes
- **困难 / Hard** — 20 分钟以上 / Beyond 20 minutes

随着难度提升 / As difficulty increases:

- 敌人攻击力提高 / Enemy damage increases
- 敌人生命值提高 / Enemy HP increases
- 更多精英怪生成 / More elite enemies spawn
- BOSS 战更具挑战性 / Boss fights become harder

### 难度模式选择 / Difficulty Mode Selection

| 模式 / Mode | 描述 / Description |
|---|---|
| Drizzle / 细雨 | 低难度，适合新手 / Low difficulty, ideal for beginners |
| Rainstorm / 雨暴 | 标准难度 / Standard difficulty |
| Monsoon / 季风 | 高难度，敌人成长更快 / Hard difficulty, enemies scale faster |
| Eclipse / 日食 | 逐步增加的挑战等级 / Incremental challenge levels |

## 道具品级 / Item Rarity

道具根据颜色分为四个品级 / Items are divided into four rarity tiers by color:

| 品级 / Tier | 颜色 / Color | 代表性道具 / Example |
|---|---|---|
| 普通 / Common | 白色 / White | 黏性炸弹 / Sticky Bomb |
| 罕见 / Uncommon | 绿色 / Green | 汽油桶 / Gasoline |
| 传奇 / Legendary | 红色 / Red | 汞合金 / Mercenary's Gas |
| 虚空 / Void | 黄色 / Yellow | 虚空道具变体 / Void item variants |

道具可以从以下来源获得 / Items can be obtained from:

- **普通宝箱 / Regular Chests** — 需要金钱购买
- **大型宝箱 / Large Chests** — 价格更高，必出绿色以上
- **三选一祭坛 / Shrine of Chance** — 概率出道具
- **传送器BOSS掉落 / Teleporter Boss Drop** — 必掉一件道具
- **隐藏区域** / Hidden areas — 通常有免费道具

{/* ===== 金币与金钱系统 ===== */}

## 金钱系统 / Gold System

击败敌人可获得金币，用于在关卡购买道具。金币不会带到下一关。

Defeating enemies grants gold, used to purchase items. Gold does not carry over between stages.

- 普通敌人掉落少量金币 / Normal enemies drop small amounts
- 精英敌人掉落更多金币 / Elite enemies drop more gold
- BOSS掉落大量金币 / Bosses drop large amounts
- 金钱可以存入**金钱祭坛**获得利息 / Gold can be deposited at Shrine of Blood

## 基础策略 / Basic Strategies

### 初期生存 / Early Game Survival

- 优先收集白色道具增加基础属性 / Focus on white items to boost base stats
- 尽快找到传送器位置 / Find the teleporter location quickly
- 在激活传送器前清理附近的敌人 / Clear nearby enemies before activating teleporter
- 保留金钱以备不时之需 / Save gold for emergency purchases

### 战斗走位 / Combat Positioning

- 始终与敌人保持距离 / Always maintain distance from enemies
- 利用地形高低差作战 / Use elevation differences to your advantage
- 不要被包围 / Avoid being surrounded
- 边走边射 / Shoot while moving

### 道具策略 / Item Strategy

- 道具效果可以叠加 / Item effects stack
- 优先拿攻击类道具提升DPS / Prioritize damage items for DPS
- 不要忽视生存类道具 / Do not overlook survival items
- 绿色道具通常比白色强力 / Green items are generally stronger than white

## 多人模式 / Multiplayer Basics

- 最多支持4名玩家联机 / Supports up to 4 players online
- 道具分配需要团队沟通 / Item distribution requires team communication
- BOSS生命值随玩家数量增加 / Boss HP scales with player count
- 复活队友需要3秒读条 / Reviving teammates takes 3 seconds
- 共享金钱可以购买更贵的道具 / Shared gold can buy expensive items

{/* ===== 常见误区 ===== */}

## 常见误区 / Common Mistakes

1. **停留太久** / Staying too long on one stage — 时间就是难度
2. **忽视白色道具** / Ignoring white items — 基础属性很重要
3. **不看小地图** / Not checking the minimap — 传送器位置一目了然
4. **不利用环境** / Not using the environment — 跳崖可以规避伤害
5. **忘记使用装备** / Forgetting to use equipment — 装备冷却很快

## 结语 / Final Words

《雨中冒险2》是难度曲线非常独特的游戏。初期可能会频繁死亡，但随着对道具系统和角色技能的熟悉，每次游戏都会越来越顺手。关键是要控制节奏，不要贪图探索而浪费太多时间。

*Risk of Rain 2* has a unique difficulty curve. You may die frequently at first, but as you familiarize yourself with the item system and survivor skills, each run will become smoother. The key is controlling the pace — do not waste too much time exploring.
