## 道具系统概述 / Item System Overview

道具是《雨中冒险2》的核心成长系统。道具通过宝箱、祭坛、BOSS击杀等方式获得。同类道具可以叠加，效果非线性增长。

Items are the core progression system in *Risk of Rain 2*. They are obtained from chests, shrines, boss kills, and more. Same items can stack, with effects scaling non-linearly.

## 道具协同组合 / Item Synergy Combos

### 暴击流 / Crit Build

**核心 / Core:** Lens-Maker's Glasses + Predatory Instincts + Harvesting Scythe

暴击流是游戏中最稳定的Build之一。高暴击率触发增攻速和治疗。

Crit build is one of the most consistent builds. High crit rate triggers attack speed and healing.

### 触发流 / Proc Build

**核心 / Core:** Ukulele + AtG Missile Mk. 1 + Gasoline + 57 Leaf Clover

触发流通过多件道具互相触发，形成连锁反应。57叶三叶草大幅提升触发概率。

Proc build chains multiple items together. 57 Leaf Clover massively boosts proc chances.

### 坦克流 / Tank Build

**核心 / Core:** Tougher Times + Medkit + Aegis + Infusion

坦克流注重生存能力，通过盾和格挡承受伤害。

Tank build focuses on survivability through shields and blocking damage.

### 攻速流 / Attack Speed Build

**核心 / Core:** Soldier's Syringe + Berzerker's Pauldron + Predatory Instincts

攻速流叠加攻击速度，配合暴击打出海量DPS。

Attack speed build stacks AS items, pairing with crit for massive DPS.
