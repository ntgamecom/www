## 关卡概述 / Stage Overview

《雨中冒险2》的关卡分为5个循环（Loop）加1个最终区域。每个循环的关卡从对应池中随机选择。通关传送器后进入下一关。

*Risk of Rain 2* stages are divided into 5 loops plus 1 final area. Stages within each loop are randomly selected from a pool. After clearing the teleporter, you proceed to the next stage.

## 新特祭坛 / Newt Altars

新特祭坛隐藏在每个关卡中，使用一枚月球币可以激活。激活后在传送器房间会出现通往和平纪念碑（Bazaar Between Time）的蓝色传送门。

Newt Altars are hidden on each stage. Activate with one lunar coin to spawn a blue portal to the Bazaar Between Time.

| 关卡 / Stage | 常见位置 / Common Location |
|---|---|
| Distant Roost | 悬崖洞穴中 / Cliff caves |
| Titanic Plains | 地图边缘的高台 / Edge high platforms |
| Siphoned Forest | 瀑布后方 / Behind waterfalls |
| Wetland Aspect | 水道深处 / Deep in waterways |
| Abandoned Aqueduct | 滚石陷阱附近 / Near boulder traps |
| Scorched Acres | 火焰区域中心 / Fire zone center |

## 传送器事件策略 / Teleporter Event Strategy

1. **先探索后激活** / Explore before activating — 收集金币买道具
2. **清理传送器周围** / Clear area around teleporter — 移除箭塔类威胁
3. **准备充足后激活** / Activate when ready — 一旦激活无法取消
4. **BOSS集中火力** / Focus the boss — 小怪可以后续清理
5. **利用传送器光环** / Use teleporter radius — 在光环内获得伤害加成
