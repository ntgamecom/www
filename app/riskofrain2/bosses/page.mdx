Activating the teleporter summons a boss. Boss difficulty depends on current game time and stage loop. Bosses drop one item and a large amount of gold.

{/* ===== 巨像 Colossus ===== */}

## 巨像 / Colossus

巨像是游戏中最常见的BOSS，拥有巨大近战攻击和跳跃践踏。

Colossus is the most common boss, with massive melee attacks and a jump stomp.

**攻击模式 / Attack Patterns:**

| 攻击 / Attack | 描述 / Description | 对策 / Counter |
|---|---|---|
| 重拳 / Heavy Fist | 前冲重击 / Charge punch | 侧向闪避 / Side dodge |
| 践踏 / Stomp | 跃起后落地 / Jump then land | 看到起跳后立即移动 / Move when it jumps |
| 横扫 / Sweep | 旋转范围攻击 / Spin AoE | 后退保持距离 / Back off |

**弱点 / Weakness:**
- 动作缓慢，容易预判 / Slow movements, easy to predict
- 头部是弱点区域 / Head is weak point
- 保持中距离输出最安全 / Mid-range is safest

{/* ===== 石泰坦 Stone Titan ===== */}

## 石泰坦 / Stone Titan

石泰坦是远程型BOSS，发射激光束攻击玩家。

Stone Titan is a ranged boss that fires laser beams at players.

**攻击模式 / Attack Patterns:**

| 攻击 / Attack | 描述 / Description | 对策 / Counter |
|---|---|---|
| 激光束 / Laser Beam | 直线持续激光 / Continuous straight laser | 垂直方向移动 / Move perpendicular |
| 落石 / Rock Fall | 召唤陨石坠落 / Summon falling rocks | 注意地面阴影 / Watch ground shadows |
| 震荡波 / Shockwave | 地面冲击波 / Ground shockwave | 跳跃躲避 / Jump to dodge |

**弱点 / Weakness:**
- 激光蓄力时有瞄准线提示 / Laser has aiming line during charge
- 近身时不会使用激光 / No laser at close range
- 侧面输出最安全 / Side attacks are safest

{/* ===== 流浪者 Wandering Vagrant ===== */}

## 流浪者 / Wandering Vagrant

流浪者是飞行BOSS，会召唤小怪并发射弹幕。

Wandering Vagrant is a flying boss that spawns minions and fires barrages.

**攻击模式 / Attack Patterns:**

| 攻击 / Attack | 描述 / Description | 对策 / Counter |
|---|---|---|
| 弹幕 / Barrage | 扇形散射弹丸 / Fan spread projectiles | 横向移动 / Strafe |
| 召唤 / Summon | 召唤小水母 / Spawn small jellyfish | 优先清理小怪 / Clear minions first |
| 自爆 / Nova | 全屏爆炸 / Full screen explosion | 远离中心 / Move away from center |

**弱点 / Weakness:**
- 弹幕弹道缓慢 / Projectiles are slow
- 自爆前有闪烁提示 / Flashes before nova
- 远程职业较好对付 / Ranged characters have advantage

{/* ===== 岩浆蠕虫 Magma Worm ===== */}

## 岩浆蠕虫 / Magma Worm

岩浆蠕虫在地面钻行，间歇性冲出攻击。

Magma Worm burrows through the ground, surfacing to attack intermittently.

**攻击模式 / Attack Patterns:**

| 攻击 / Attack | 描述 / Description | 对策 / Counter |
|---|---|---|
| 钻出 / Emerge | 从地下冲出 / Burst from ground | 注意地面震动 / Watch ground tremors |
| 火球 / Fireball | 喷出多枚火球 / Spit multiple fireballs | 横向移动 / Strafe |
| 岩浆路径 / Lava Trail | 移动路径留下岩浆 / Leave lava trail | 不要停留在岩浆上 / Do not stand on lava |

**弱点 / Weakness:**
- 钻出位置有地面提示 / Ground indicates emerge location
- 头部是主要弱点 / Head is primary weak point
- 持续移动是关键 / Constant movement is key

{/* ===== 恶魔领主 Imp Overlord ===== */}

## 恶魔领主 / Imp Overlord

恶魔领主是近战刺客型BOSS，擅长瞬移和快速攻击。

Imp Overlord is a melee assassin boss,擅长 teleporting and fast attacks.

**攻击模式 / Attack Patterns:**

| 攻击 / Attack | 描述 / Description | 对策 / Counter |
|---|---|---|
| 瞬移斩 / Blink Strike | 瞬移到目标身后攻击 / Teleport behind target | 转身立即移动 / Turn and move |
| 暗影弹 / Shadow Bolt | 远程暗影弹 / Ranged shadow bolt | 侧移躲避 / Side step |
| 暗影之怒 / Shadow Fury | 范围爆炸 / AoE explosion | 观察蓄力后撤 / Watch charge then retreat |

**弱点 / Weakness:**
- 瞬移后有短暂停顿 / Brief pause after teleport
- 暗影弹弹道可见 / Shadow bolt has visible trajectory
- 群体战斗更容易应对 / Easier in group fights

## 极构造体 / Xi Construct

极构造体是机械BOSS，发射导弹和激光。

Xi Construct is a mechanical boss that fires missiles and lasers.

**攻击模式 / Attack Patterns:**

| 攻击 / Attack | 描述 / Description | 对策 / Counter |
|---|---|---|
| 导弹齐射 / Missile Salvo | 多枚追踪导弹 / Multiple homing missiles | 利用障碍物阻挡 / Use obstacles |
| 激光扫射 / Laser Sweep | 扇形激光 / Sweeping laser | 跳跃或蹲下 / Jump or crouch |
| 电磁脉冲 / EMP | 范围减速 / AoE slow | 及时脱离范围 / Leave the area |

**弱点 / Weakness:**
- 导弹可以绕柱躲避 / Missiles can be dodged around pillars
- 核心暴露时是攻击机会 / Core exposed is attack window
- 近距离战斗反而更安全 / Surprisingly safer at close range

## 黄金泰坦 / Aurelionite

黄金泰坦是隐藏BOSS，出现在天体诞生之地 (Gilded Coast)。通过蓝色祭坛进入。

Aurelionite is a hidden boss found in the Gilded Coast. Enter through a blue shrine.

| 攻击 / Attack | 描述 / Description | 对策 / Counter |
|---|---|---|
| 黄金重拳 / Golden Fist | 强化版重拳 / Enhanced punch | 侧闪 / Side dodge |
| 黄金激光 / Golden Laser | 强化版激光 / Enhanced laser | 垂直移动 / Perpendicular movement |
| 黄金召唤 / Golden Summon | 召唤黄金小怪 / Summon golden minions | 先清小怪 / Clear minions first |

**奖励 / Reward:**
击败后掉落传说道具和大量金币 / Drops legendary item and大量 gold

{/* ===== 米斯瑞克斯 Mithrix ===== */}

## 米斯瑞克斯 / Mithrix (最终BOSS / Final Boss)

Mithrix 是游戏的最终BOSS，在Commencement关卡遭遇。战斗分为三个阶段。

Mithrix is the final boss, encountered at Commencement. The fight has three phases.

**第一阶段 / Phase 1:**

| 攻击 / Attack | 描述 / Description | 对策 / Counter |
|---|---|---|
| 锤击 / Hammer Smash | 近战重击 / Melee heavy hit | 看到起手后闪避 / Dodge on wind-up |
| 能量波 / Energy Wave | 地面冲击波 / Ground wave | 跳跃躲避 / Jump over |
| 召唤 / Summon | 召唤小怪 / Summon minions | 先清理 / Clear them |

**第二阶段 / Phase 2:**
Mithrix 飞到高空，发射弹幕和激光 / Mithrix flies up high, fires barrages and lasers

**第三阶段 / Phase 3:**
Mithrix 将所有道具从玩家身上剥离并自己使用，战斗进入最危险阶段。

Mithrix strips all items from the player and uses them himself. This is the most dangerous phase.

**最终阶段策略 / Final Phase Strategy:**
- 不要慌张 / Do not panic
- 躲避为主，寻找输出窗口 / Focus on dodging, find attack windows
- 利用地形遮挡 / Use terrain for cover
- 存活即是胜利 / Survival equals victory

{/* ===== 通用BOSS策略 ===== */}

## 通用BOSS策略 / General Boss Strategies

1. **准备好再激活传送器** / Prepare before activating — 买完道具再打 / Buy items first
2. **保持移动** / Keep moving — BOSS攻击范围大
3. **优先清理小怪** / Clear minions first — 小怪分散注意力
4. **利用地形** / Use terrain — 柱子可以挡弹道
5. **团队配合** / Team coordination — 轮流吸引仇恨
