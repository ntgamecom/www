# 隐藏要素 — 隐藏内容与彩蛋指南
# Secrets — Hidden Content and Easter Eggs Guide

---

{/* ===== 隐藏系统概述 ===== */}

## 隐藏系统概述 / Secrets Overview

《雨中冒险2》中有大量隐藏内容，包括秘密区域、特殊BOSS、隐藏角色和解锁道具。探索隐藏内容是游戏魅力的重要组成部分。

*Risk of Rain 2* is filled with hidden content including secret areas, special bosses, hidden characters, and unlockable items. Exploring secrets is a key part of the game's charm.

## 隐藏区域 / Hidden Areas

### 和平纪念碑 / Bazaar Between Time

通过新特祭坛 (Newt Altar) 使用月球币激活蓝色传送门进入。

Enter through a Newt Altar by spending a lunar coin to activate the blue portal.

| 内容 / Content | 说明 / Description |
|---|---|
| 月球道具商店 / Lunar Item Shop | 使用月球币购买道具 / Buy items with lunar coins |
| 重roll道具机 / 3D Printer | 将一个道具换成同品级随机道具 / Swap an item for random same-tier item |
| 月亮医生 / Lunar Seer | 查看下一关信息 / View next stage info |
| 净化池 / Cleansing Pool | 将月球道具转为普通道具 / Convert lunar to regular items |

### 天体诞生之地 / Gilded Coast

隐藏关卡，通过蓝色祭坛（不是新特祭坛）进入。需要击败黄金泰坦 (Aurelionite)。

Hidden stage, entered through a blue shrine (not Newt Altar). Defeat Aurelionite.

| 内容 / Content | 说明 / Description |
|---|---|
| 黄金泰坦 / Aurelionite | 隐藏BOSS / Hidden boss |
| 黄金宝箱 / Golden Chest | 必出红色道具 / Guaranteed legendary item |
| 金币奖励 / Gold Reward | 大量金币 / Large gold reward |

### 虚空之境 / Void Fields

通过关卡特有虚空传送门进入。该区域持续造成伤害，但道具奖励丰厚。

Enter through a special void portal on stages. This area deals continuous damage but offers rich item rewards.

| 内容 / Content | 说明 / Description |
|---|---|
| 虚空电池 / Void Cells | 激活后进行挑战 / Activate to start challenge |
| 挑战奖励 / Challenge Reward | 完成挑战获得道具 / Complete challenge for items |
| 虚空BOSS / Void Boss | 稀有遭遇 / Rare encounter |

## 怪物日志 / Monster Logs

每个敌人都有一篇日志，击杀时有概率掉落。日志包含该敌人的背景故事和生态信息。

Every enemy has a log entry with a chance to drop on kill. Logs contain backstory and ecological information.

| 日志类型 / Log Type | 掉落概率 / Drop Rate | 收集数量 / Total |
|---|---|---|
| 怪物日志 / Monster Log | 约4% / Approx 4% | 40+ |
| 道具日志 / Item Log | 约4% / Approx 4% | 100+ |
| 环境日志 / Environment Log | 关卡内固定位置 / Fixed positions | 15+ |

## 彩蛋 / Easter Eggs

### 初代《雨中冒险》彩蛋 / Risk of Rain 1 Reference

在散落的文件中，存在指向初代游戏的参考。某些怪物模型也还原了初代的设计。

Among scattered files are references to the first game. Some monster models also replicate the original design.

### 开发者房间 / Developer Room

通过特定条件可以进入开发者房间，里面有制作者名单和特殊互动。

Enter a developer room under specific conditions to find credits and special interactions.

### 音乐彩蛋 / Music Easter Eggs

游戏中的某些片段（如传送器激活时的音效）隐藏着音乐彩蛋。玩家可以通过特定交互触发。

Certain game segments hide music easter eggs. Players can trigger them through specific interactions.

## 探索技巧 / Exploration Tips

1. **检查每个墙角** / Check every corner — 隐藏区域常在边缘
2. **注意声音提示** / Listen for audio cues — 某些隐藏区域有特殊音效
3. **攻击可疑墙壁** / Attack suspicious walls — 隐藏门可能被打破
4. **跳上高处** / Jump to high places — 视觉死角可能隐藏宝箱
5. **多人分头搜索** / Split up in multiplayer — 覆盖更大范围
