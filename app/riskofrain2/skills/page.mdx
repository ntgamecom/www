# 技能与Build — 技能变体与流派指南
# Skills and Builds — Skill Variants and Archetype Guide

---

{/* ===== 技能系统概述 ===== */}

## 技能系统概述 / Skill System Overview

每个幸存者有4个技能槽：普通攻击、次要攻击、技能1、技能2。每个技能有多个变体可以通过完成挑战解锁。

Each survivor has 4 skill slots: Primary, Secondary, Utility, and Special. Each skill has multiple variants unlockable through challenges.

---

{/* ===== 技能变体解锁 ===== */}

## 技能变体统一解锁 / Skill Variant Unlock Conditions

| 技能变体 / Variant | 解锁条件 / Unlock Condition |
|---|---|
| 普通攻击变体 / Primary Variant | 使用该角色击杀 200 个敌人 / Kill 200 enemies with the survivor |
| 次要攻击变体 / Secondary Variant | 使用该角色通关 5 关 / Complete 5 stages with the survivor |
| 技能1变体 / Utility Variant | 使用该角色激活 10 次传送器 / Activate 10 teleporters with the survivor |
| 技能2变体 / Special Variant | 使用该角色击败 BOSS 20 次 / Defeat boss 20 times with the survivor |

## 核心Build流派 / Core Build Archetypes

### 1. 暴击流 / Crit Build

**适用角色 / Best For:** Commando, Bandit, Huntress, Captain

**核心道具 / Core Items:**
- Lens-Maker's Glasses (6+) / 暴击眼镜
- Predatory Instincts / 掠食本能
- Harvesting Scythe / 收割镰刀
- Berzerker's Pauldron / 狂战士肩甲

**玩法 / Playstyle:**
堆叠暴击率至100%，触发攻速加成和治疗。配合高攻速武器打出毁灭性DPS。

Stack crit rate to 100%, triggering attack speed and healing. Pair with high-attack-speed weapons for devastating DPS.

**进阶变体 / Advanced Variant:**
加入 57 Leaf Clover 和 Brilliant Behemoth 使暴击造成范围伤害。

Add 57 Leaf Clover and Brilliant Behemoth for AoE crit damage.

---

### 2. 触发流 / Proc Chain Build

**适用角色 / Best For:** Engineer, Mul-T, Artificer

**核心道具 / Core Items:**
- Ukulele / 尤克里里
- AtG Missile Mk. 1 / AtG导弹
- Gasoline / 汽油桶
- Will-o'-the-wisp / 鬼火
- 57 Leaf Clover (极度推荐) / 57叶三叶草 (highly recommended)
- Tri-Tip Dagger / 三尖匕首

**玩法 / Playstyle:**
每件道具的攻击都有概率触发其他道具的效果，形成连锁反应。57 Leaf Clover 大幅提高触发链启动概率。

Each item attack has a chance to trigger other items' effects, creating chain reactions. 57 Leaf Clover significantly boosts chain initiation chance.

**最佳搭配 / Best Pairing:**
- Engineer 的炮台继承所有触发道具 / Engineer's turrets inherit all proc items
- Mul-T 双倍武器 = 双倍触发机会 / Mul-T's dual weapons = double proc chance

---

### 3. 坦克生存流 / Tank Survival Build

**适用角色 / Best For:** Mul-T, Loader, Rex

**核心道具 / Core Items:**
- Tougher Times (3+) / 熊盾
- Medkit / 医疗包
- Infusion / 输血
- Aegis / 神盾
- Personal Shield Generator / 个人护盾
- Transcendence (Lunar) / 超然 (月球)

**玩法 / Playstyle:**
通过高额减伤和自动恢复承受大量伤害。牺牲部分输出换取极其稳定的生存能力。

Take massive damage through high damage reduction and auto-recovery. Sacrifice some damage for extreme survival consistency.

---

### 4. 移速风筝流 / Speed Kite Build

**适用角色 / Best For:** Huntress, Loader, Mercenary

**核心道具 / Core Items:**
- Paul's Goat Hoof (3+) / 羊蹄
- Energy Drink / 能量饮料
- Wax Quail / 蜡鹌鹑
- Hopoo Feather (2+) / 羽毛
- Grace (Lunar) / 优雅 (月球)

**玩法 / Playstyle:**
极致移速让敌人根本无法追上你。配合跳跃道具穿越地形，边跑边打。

Extreme speed makes enemies unable to catch you. Paired with jump items to traverse terrain while kiting.

---

### 5. 装备冷却流 / Equipment Spam Build

**适用角色 / Best For:** Captain, Commando

**核心道具 / Core Items:**
- Soulbound Catalyst / 灵魂催化剂
- Alien Head / 异形头骨 (CDR)
- Focus Crystal / 聚焦水晶
- Soul Ring / 灵魂戒指

**玩法 / Playstyle:**
大幅缩短装备冷却，不断使用强力装备技能。

Drastically reduce equipment cooldown and spam powerful equipment skills.

---

### 6. 毒伤流 / Poison DoT Build

**适用角色 / Best For:** Acrid, Artificer

**核心道具 / Core Items:**
- Tri-Tip Dagger / 三尖匕首
- Gasoline / 汽油桶
- Will-o'-the-wisp / 鬼火
- Chronobauble / 时间滤镜 (减速)
- N'kuhana's Opinion / 意见

**玩法 / Playstyle:**
Acrid 的毒伤基于敌人最大生命值，搭配持续伤害道具可以高效清场。

Acrid's poison scales with enemy max HP. Paired with DoT items for efficient clearing.

## 如何适应当前Build / How to Adapt Your Build

### 道具评估 / Item Assessment

每关结束后评估当前道具组合，决定Build方向：

After each stage, assess your item pool and decide your build direction:

| 情况 / Situation | 推荐方向 / Direction |
|---|---|
| 已有3+暴击眼镜 / Have 3+ glasses | 走暴击流 / Go crit build |
| 已得尤克里里或AtG / Got Ukulele or AtG | 走触发流 / Go proc build |
| 血量危险，已有熊盾 / Low HP, have bear | 转生存流 / Transition to tank |
| 移速道具多 / Many speed items | 风筝流 / Kite build |

### 道具转换策略 / Item Transition Strategy

- **前期 / Early game**: 优先白色道具堆基础属性
- **中期 / Mid game**: 根据已有道具决定流派
- **后期 / Late game**: 增加传奇道具补强流派弱点

- **Early game**: Prioritize white items for base stats
- **Mid game**: Decide build based on current items
- **Late game**: Add legendaries to cover build weaknesses

## 常见Build误区 / Common Build Mistakes

1. **道具太分散** / Too spread out — 全属性不如专精一项
2. **忽视生存** / Ignoring survival — 死人没有DPS
3. **不看角色特性** / Ignoring character traits — 每个角色有最优道具
4. **不调整策略** / Not adapting — 游戏给的什么道具就用什么Build
5. **过度追求传奇** / Over-pursuing legendaries — 白色道具的叠加效果也不差
