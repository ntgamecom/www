# Endgame — Advanced Strategy Guide

## 终局概述 / Endgame Overview

当你已掌握基础玩法后，真正的挑战才刚刚开始。《雨中冒险2》的终局内容包括最高难度挑战、成就系统以及自我挑战玩法。

Once you have mastered the basics, the real challenge begins. *Risk of Rain 2* endgame content includes the highest difficulty challenges, achievement systems, and self-imposed challenges.

## Monsoon 难度策略 / Monsoon Difficulty Strategies

Monsoon 是标准难度之上的最高初始难度选择。敌人成长速度更快，容错率更低。

Monsoon is the highest selectable starting difficulty. Enemies scale faster and error tolerance is much lower.

### 核心要点 / Core Principles

| 要点 / Point | 说明 / Explanation |
|---|---|
| 时间管理 / Time Management | 每关不超过5分钟 / No more than 5 minutes per stage |
| 道具优先 / Item Priority | 攻击 &gt; 移速 &gt; 防御 / Attack &gt; Speed &gt; Defense |
| 角色选择 / Survivor Choice | 选择你最有信心的角色 / Pick your most confident survivor |
| 不贪图探索 / No Over-Exploration | 有足够道具就走 / Leave when you have enough items |

### Monsoon 推荐角色 / Monsoon Recommended Survivors

| 角色 / Survivor | 理由 / Reason |
|---|---|
| Commando / 突击队员 | 全面，没有明显短板 / Well-rounded, no clear weakness |
| Loader / 装载者 | 高机动 + 高爆发 / High mobility + burst |
| Captain / 船长 | 免费补给信标减少资源压力 / Free beacons reduce resource pressure |

### Monsoon 道具策略 / Monsoon Item Strategy

在Monsoon难度下，道具选择必须更加精准 / Item choices must be more precise on Monsoon:

1. **第1-2关 / Stage 1-2**: 优先攻击速度 + 移速 / Prioritize attack speed + move speed
2. **第3-4关 / Stage 3-4**: 开始堆积暴击和触发链 / Start building crit and proc chains
3. **第5关+ / Stage 5+**: 加入传奇道具和生存道具 / Add legendaries and survival items

## Eclipse 挑战指南 / Eclipse Challenge Guide

Eclipse 是 Monsoon 之上的进阶难度系统，共有8个等级，每级增加一个负面效果修改器。

Eclipse is an advanced difficulty system above Monsoon, with 8 levels that each adds a negative modifier.

### Eclipse 等级列表 / Eclipse Level List

| 等级 / Level | 修改器 / Modifier | 效果 / Effect |
|---|---|---|
| Eclipse 1 | 环境伤害翻倍 / Environmental Damage x2 | 火焰、毒气等伤害加倍 |
| Eclipse 2 | 敌人攻速提升 / Enemy Attack Speed Up | 所有敌人攻击速度+40% |
| Eclipse 3 | 治疗减半 / Healing Halved | 所有治疗效果降低50% |
| Eclipse 4 | 友军伤害 / Friendly Fire | 玩家攻击可以伤害队友 (单人无影响) |
| Eclipse 5 | 开局无道具 / No Starting Item | 游戏开始时没有初始道具 |
| Eclipse 6 | 金钱需求翻倍 / Gold Cost x2 | 所有宝箱和祭坛价格翻倍 |
| Eclipse 7 | 传送器时间翻倍 / Teleporter Time x2 | 传送器充能时间变为两倍 |
| Eclipse 8 | 每关减少最大生命值 / Max HP Reduction Per Stage | 每关开始减少10%最大生命值 |

### Eclipse 策略 / Eclipse Strategies

**Eclipse 1-2:**
这些等级相对友好。注意环境伤害来源，在Eclipse 2 中增加闪避频率。

These levels are relatively forgiving. Watch environmental damage sources and increase dodge frequency on Eclipse 2.

**Eclipse 3-4:**
治疗减半意味着Medkit和Harvesting Scythe的价值下降。优先使用Tougher Times和护盾。Eclipse 4 如果单人可以忽略友军伤害。

Halved healing reduces Medkit and Harvesting Scythe value. Prioritize Tougher Times and shields. Eclipse 4 can be ignored in solo.

**Eclipse 5-6:**
没有开局道具和翻倍金钱意味着前期更加困难。建议在第一关花更多时间积累金币。

No starting item and doubled gold costs mean a much harder early game. Consider spending more time on stage 1 to accumulate gold.

**Eclipse 7-8:**
传送器时间翻倍和血量减少是最大的挑战。建议在激活传送器前清理所有威胁，确保安全充能。Eclipse 8 的血量减少可以通过Infusion和Bison Steak弥补。

Double teleporter time and HP reduction are the biggest challenges. Clear all threats before activating the teleporter. Eclipse 8 HP reduction can be offset with Infusion and Bison Steak.

## 100% 完成指南 / 100% Completion Guide

### 步骤1: 解锁所有角色 / Step 1: Unlock All Survivors

逐一解锁所有11个幸存者并熟悉其技能。

Unlock all 11 survivors one by one and familiarize yourself with their skills.

### 步骤2: 收集全道具 / Step 2: Collect All Items

- 优先解锁道具日志解锁条件 / Prioritize item log unlock requirements
- 利用和平纪念碑获取稀有道具 / Use Bazaar Between Time for rare items
- 虚空领域获得虚空道具 / Void Fields for void items

### 步骤3: 收集全日志 / Step 3: Collect All Logs

- 专注刷特定关卡 / Farm specific stages for monster logs
- 使用幸运道具提高掉落率 / Use luck items to boost drop rates
- 环境日志在固定位置扫描 / Environment logs are at fixed positions

### 步骤4: 全成就 / Step 4: All Achievements

- 从最简单的成就开始 / Start with easiest achievements
- Eclipse 8 留到最后挑战 / Save Eclipse 8 for last

## 循环与无限模式 / Looping and Infinite Mode

通关最终BOSS后，你可以选择进入"循环"模式，重新从第一关开始但保留所有道具。

After defeating the final boss, you can choose to "loop" back to stage 1 while keeping all items.

| 循环数 / Loop | 难度 / Difficulty | 推荐道具数 / Recommended Items |
|---|---|---|
| Loop 1 (1-5关) | 正常 / Normal | 15-25 |
| Loop 2 (6-10关) | 困难 / Hard | 30-40 |
| Loop 3 (11-15关) | 极高 / Very Hard | 45-60 |
| Loop 4+ | 疯狂 / Insane | 60+ |
