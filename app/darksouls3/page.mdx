# 黑暗之魂3攻略 / Dark Souls 3 Guide

黑暗之魂3（Dark Souls 3）是FromSoftware开发的魂系动作RPG终极之作，2016年发售。作为传火之旅的终章，你将在衰败的洛斯里克王国中穿梭，挑战薪王们的化身 / Dark Souls 3 is the culmination of FromSoftware's souls-like action RPG series, released in 2016. As the final chapter of the Age of Fire, you traverse the decaying kingdom of Lothric to challenge the Lords of Cinder.

**平台 / Platforms**: PS4, Xbox One, PC (Steam) | **发售日 / Release**: 2016年3月24日 / Mar 24, 2016 | **类型 / Genre**: 魂系动作RPG / Souls-like Action RPG | **DLC**: 艾雷德尔之烬 / Ashes of Ariandel (2016.10), 环印城 / The Ringed City (2017.3)

## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **快节奏战斗 / Faster Combat** | 相比前作速度大幅提升——更强调进攻和走位 / Significantly faster than previous titles — emphasizes offense and positioning |
| **战技系统 / Weapon Arts** | 每个武器有独特的战技——消耗FP释放 / Unique skill per weapon — consumes FP |
| **FP与蓝瓶 / FP & Ashen Estus** | 新增专注值——用于释放战技和法术 / New Focus Points bar — for weapon arts and spells |
| **质变系统 / Infusions** | 宝石质变——改变武器属性和补正 / Gem infusions — change weapon scaling and damage type |
| **骨灰系统 / Bone Shards** | 升级元素瓶恢复量 / Upgrade Estus healing amount |
| **薪王模式 / Lord of Cinder** | 连续击败四位薪王后解锁最终区域 / Unlock final area after defeating four Lords of Cinder |

## 开局推荐 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **选骑士 / Start as Knight** | 骑士的初始装备和属性分配最适合新手——100%物防盾+高初始生命 / Knight has the best starting gear and stats for beginners — 100% phys shield + high starting HP |
| **2** | **优先加生命和持久 / Prioritize Vigor & Endurance** | 前期全部点数加生命到27（软上限）和持久力到20 / First invest in Vigor to 27 and Endurance to 20 |
| **3** | **举盾绕圈 / Shield Circle** | 拿100%物防盾举盾绕敌人走——等他出完招后攻击 / Hold up a 100% phys shield and circle enemies — attack after they finish their combo |
| **4** | **不要贪刀 / Don't Get Greedy** | 魂系铁律——打1-2下就滚，贪刀=死 / Golden rule: attack 1-2 times then roll — greed = death |
| **5** | **探索每个角落 / Explore Thoroughly** | 游戏有大量隐藏路径、道具和NPC支线——仔细探索每个区域 / Tons of hidden paths, items, and NPC quests — explore every corner |
| **6** | **元素瓶分配 / Estus Allocation** | 前期多分红瓶——法师/圣职可以用蓝瓶。可在祭祀场火女处调整 / More red estus early — mages/clerics can use blue. Adjust at Firelink Shrine |
| **7** | **不要打老太婆 / Don't Kill the Old Lady** | 传火祭祀场的侍女是重要商人——不要攻击NPC / The Shrine Handmaid is a key merchant — don't attack NPCs |
| **8** | **召NPC帮忙 / Summon NPCs** | 很多Boss战前可以召唤金色NPC——能大幅降低难度 / Many boss fights have NPC summon signs — they greatly reduce difficulty |
