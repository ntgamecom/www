# 物品收集 / Items & Collectibles

黑暗之魂3有大量关键收集品——戒指、元素碎片、不死人遗骨、关键道具。本章分类整理最重要的物品 / Dark Souls 3 has many key collectibles — rings, Estus Shards, Undead Bone Shards, and key items. This chapter organizes the most important ones by category.

---

## 戒指推荐 / Best Rings

## 返回骨片 / Coiled Sword Fragment

| 道具 / Item | 说明 / Description |
|------|------|
| **返回骨片 / Homeward Bone** | 无限使用的返回道具——可回到最后休息的篝火 / Infinite-use item — return to last rested bonfire |
| **螺旋剑破片 / Coiled Sword Fragment** | 击败古达后获得——可用来强化元素瓶吗？不，这是用来换道具的 / Item from Iudex Gundyr — used for transposition |

> **重要**：游戏中有**螺旋剑破片**和**返回骨片**两个不同道具——不要混淆 / **Important**: Two different items — Coiled Sword Fragment (transposition) vs Homeward Bone (teleport)
