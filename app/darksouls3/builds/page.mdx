# 流派构建 / Builds & Equipment

黑暗之魂3的Build灵活性极高——物理、魔法、奇迹、咒术各有千秋。本章涵盖所有装备类型和推荐Build / Dark Souls 3 offers incredible build variety — physical, sorcery, miracles, and pyromancies. This chapter covers all equipment types and recommended builds.

## 质变系统 / Infusion System

| 质变宝石 / Gem | 效果 / Effect | 推荐武器 / Best For |
|------|------|------|
| **厚重 / Heavy** | 力量补正大幅提升，其他补正降低 / Greatly increases STR scaling, reduces others | 力量武器 / STR weapons |
| **锋利 / Sharp** | 灵巧补正大幅提升 / Greatly increases DEX scaling | 敏捷武器 / DEX weapons |
| **熟练 / Refined** | 力敏双补正中幅提升 / Increases STR + DEX scaling | 力敏号 / Quality builds |
| **结晶 / Crystal** | 加智力补正+魔力伤害 / INT scaling + magic damage | 法师近战 / Mage melee |
| **雷电 / Lightning** | 加信仰补正+雷伤害 / FTH scaling + lightning damage | 信仰号 / Faith builds |
| **混沌 / Chaos** | 智信双补正+火伤害 / INT+FTH scaling + fire damage | 咒术师 / Pyromancer |
| **暗 / Dark** | 智信双补正+暗伤害 / INT+FTH scaling + dark damage | 暗术流 / Dark builds |
| **锋利/血 / Blood** | 加灵巧补正+出血累积 / DEX scaling + bleed buildup | 出血流 / Bleed builds |

> **建议**：骑士开局用**熟练质变**长剑→中后期转**混沌**或**暗**质变。佣兵开局用**锋利质变**佣兵双刀→PVE无敌 / **Tip**: Knight → **Refined** longsword → late game **Chaos** or **Dark**. Mercenary → **Sharp** Sellsword Twinblades → PVE god mode
