### Boss: 奴隶骑士盖尔 / Slave Knight Gael

| 方面 / Aspect | 说明 / Description |
|---|---|
| **位置 / Location** | 环印城尽头 / End of Ringed City |
| **阶段 / Phases** | 3个阶段——人类→黑暗→雷电 / 3 phases — human form → dark form → lightning form |
| **第二阶段 / Phase 2** | 使用黑暗剑技——大范围横斩+突刺 / Dark sword skills — wide slashes + thrusts |
| **第三阶段 / Phase 3** | 雷电附魔——超快连击+雷电AOE / Lightning buff — super fast combos + lightning AOE |
| **策略 / Strategy** | 二阶段保持中距离——三阶段用大盾硬抗雷电攻击 / Mid range in phase 2 — use greatshield in phase 3 |

### Boss: 吞噬黑暗的米狄尔 / Darkeater Midir

| 方面 / Aspect | 说明 / Description |
|---|---|
| **解锁 / Unlock** | 环印城隐藏区域——需要打碎吊桥 / Hidden area in Ringed City — break bridge |
| **HP** | 全系列最高血量 / Highest HP in series |
| **弱点 / Weakness** | 雷伤害 / Lightning damage |
| **策略 / Strategy** | 只打头——打身体其他部位伤害减半 / Hit only the head — body damage halved |
| **奖励 / Reward** | 米狄尔灵魂——可换月光大剑或暗术 / Midir soul — Moonlight GS or dark spell |
