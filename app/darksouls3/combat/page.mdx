# 战斗系统 / Combat System

黑暗之魂3的战斗是魂系系列的巅峰之一——快速、精准、讲究立回 / Dark Souls 3 combat is among the pinnacle of the souls series — fast, precise, and spacing-focused.

## 近战基础 / Melee Basics

### 攻击类型 / Attack Types

| 攻击 / Attack | 操作 / Input | 说明 / Description |
|---|---|---|
| **轻攻击 / Light Attack** | R1 / 鼠标左键 | 快速连击 / Fast combos |
| **重攻击 / Heavy Attack** | R2 / Shift+左键 | 高伤+削韧——可蓄力 / High damage + poise — chargeable |
| **跳跃攻击 / Jump Attack** | 前冲+R2 / Forward+R2 | 削韧最高 / Highest poise damage |
| **冲刺攻击 / Running Attack** | 奔跑+R1/R2 / Sprint+R1/R2 | 突进起手 / Gap closer |
| **双持攻击 / Dual Wield** | L1 | 双持武器特殊攻击 / Unique dual-wield moveset |
| **武器战技 / Weapon Art** | L2 + R1/R2 | 武器特有技能 / Weapon-specific skill |

### 削韧与硬直 / Poise & Stagger

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **削韧值 / Poise Damage** | 不同武器削韧不同——大锤>大剑>直剑 / Weapon type determines poise damage — hammer > greatsword > straight sword |
| **韧性 / Poise** | 玩家韧性决定能否被打断——穿重甲提升 / Player poise determines stagger resistance — heavy armor helps |
| **处决 / Critical** | 敌人硬直时正面R1——背刺/弹反后也可处决 / R1 in front when staggered — also after backstab/parry |

## 防御与闪避 / Defense & Evasion

### 负重档位 / Equip Load

| 负重 / Load | 翻滚 / Roll | 说明 / Description |
|---|---|---|
| **< 30%** | 快速翻滚 / Fast roll | 最远距离+最多无敌帧 / Farthest + most i-frames |
| **30-70%** | 中翻滚 / Mid roll | 标准 / Standard |
| **70-100%** | 重翻滚 / Fat roll | 极差——勉强能用 / Terrible — barely usable |
| **> 100%** | 不能翻滚 / Can't roll | 最差状态 / Worst case |

| **弹反匕首 / Parrying Dagger** | 30% | 弹反专用 / Parry only |

## 战斗技巧 / Combat Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **精力管理 / Stamina Management** | 不要打空精力——留一丝用来翻滚 / Never deplete stamina — save some for roll |
| **立回 / Spacing** | 在敌人攻击范围边缘引诱出招——然后反击 / Lure attacks at range edge — then punish |
| **R2蓄力 / Charged R2** | 蓄力R2可打跪人形敌人 / Charged R2 knocks down humanoid enemies |
| **背刺判定 / Backstab Range** | 背刺判定比看上去宽松——多练习 / Backstab range is generous — practice |
| **武器长度 / Weapon Reach** | 长武器在PVP中有巨大优势——如教堂骑士大剑 / Long weapons have huge PvP advantage — like Cathedral Knight GS |
