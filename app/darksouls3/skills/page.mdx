## 技能法术常见问题 / Skills & Magic FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **法师怎么开局？/ Mage starting?** | 选魔法师职业——优先加智力和蓝条 / Start as Sorcerer — prioritize INT and FP |
| **信仰战士强吗？/ Faith build viable?** | 很强——阳光之枪+雷电变质的阔剑 / Very strong — Sunlight Spear + Lightning Broadsword |
| **咒术需要双修吗？/ Pyromancy dual stat?** | 是的——智力和信仰都对咒术有加成 / Yes — both INT and FTH boost pyromancies |
| **FP不够用怎么办？/ Low FP?** | 加专注属性——或使用蓝泪石戒指 / Level Attunement — or use Simple infusions |
| **记忆空格怎么增加？/ More attunement slots?** | 加专注属性到一定阈值——10/14/18/24/30各得一格 / Level Attunement to thresholds — 10/14/18/24/30 |
