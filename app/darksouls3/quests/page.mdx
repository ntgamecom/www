## 任务常见问题 / Quests FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **安里在哪里？/ Where is Anri?** | 活祭品之路→卡萨斯墓地→熏烟湖→伊鲁希尔→暗月灵庙 / Road of Sacrifices → Catacombs → Smouldering Lake → Irithyll → Darkmoon Tomb |
| **帕奇骗我怎么办？/ Patches tricked me?** | 原谅他——他以后还有用 / Forgive him — he's useful later |
| **格雷瑞特必死吗？/ Will Greirat die?** | 取决于你救不救他——派他出去偷窃前告诉洋葱 / Depends on rescue — tell Siegward before sending him |
| **怎么洗点？/ How to respec?** | 加入罗莎莉亚指头——献上发青舌头 / Join Rosaria's Fingers — offer Pale Tongues |
| **誓约奖励可以刷吗？/ Farm covenant items?** | 可以——特定敌人低概率掉落 / Yes — specific enemies drop at low rates |
| **暗月之剑刷证攻略？/ Blade proof farming?** | 安诺隆德的银骑士——带金蛇戒指+结晶老者刺剑+硬币 / Anor Londo silver knights — wear Gold Serpent Ring + Crystal Sage Rapier + coins |
| **游魂之王路线要点？/ Lord of Hollows tips?** | 全程不要治愈暗穴——8个暗穴拿满。到最终Boss前召唤尤莉亚 / Don't heal dark sigils — get all 8. Summon Yuria before final boss |
| **洛斯里克王子怎么打？/ Twin Princes strategy?** | 第一阶段打弟弟 / 第二阶段打哥哥（背上的）。注意传送攻击 / Phase 1 hit younger / Phase 2 hit older on his back. Watch teleport attacks |
| **薪王化身怎么打？/ Soul of Cinder strategy?** | 第一阶段多种武器形态——注意曲剑形态连击快。第二阶段保留余火滚雷枪 / Phase 1 multiple weapon forms — watch curved sword combos. Phase 2 dodge lightning |
| **DLC推荐等级？/ DLC recommended level?** | 绘画世界60-80 / 环印城100-130 / Painted World lv60-80 / Ringed City lv100-130 |
| **祭祀场侍女杀死了怎么办？/ Killed handmaid?** | 杀了就不能复活——不过不影响通关。只是不能买她卖的物品 / Can't revive — but doesn't block completion. Just lose her shop |
| **薄暮之国的希里斯任务线？/ Sirris quest?** | 在祭祀场遇到她——帮助她击败入侵者。不要加入罗莎莉亚指头否则会敌对 / Meet at Firelink — help vs invaders. Don't join Rosaria's Fingers or she'll be hostile |
