## 核心机制 / Core Mechanics

### 元素瓶 / Estus Flask

| 机制 / Mechanic | 说明 / Description |
|------|------|
| **红元素瓶 / Estus Flask** | 恢复HP——初始3次，可升级到最多15次 / Restores HP — starts at 3, max 15 charges |
| **蓝元素瓶 / Ashen Estus** | 恢复FP——用于施法和战技 / Restores FP — for spells and weapon arts |
| **分配比例 / Allocation** | 在祭祀场铁匠处调整红蓝瓶分配 / Adjust ratio at Firelink Shrine blacksmith |
| **升级次数 / Charge Upgrade** | 收集元素碎片交给铁匠——共11个 / Collect Estus Shards and give to blacksmith — 11 total |
| **升级恢复量 / Healing Upgrade** | 收集不死人遗骨在祭祀场火祭场强化 / Collect Undead Bone Shards to burn at Firelink |

### 死亡机制 / Death Mechanics

| 死亡后果 / Consequence | 说明 / Description |
|------|------|
| **魂丢失 / Soul Loss** | 死亡时掉落所有魂——在死亡地点捡回 / Drop all souls at death location |
| **捡回魂 / Retrieval** | 触碰血痕捡回——路上死亡则永久丢失 / Touch bloodstain to recover — die en route and souls are lost |
| **余火状态 / Ember** | 使用余火后HP上限+30%——可召唤合作玩家 / Use Ember to +30% HP and enable co-op summoning |
| **余火死亡 / Ember Death** | 死亡后失去余火状态——回到普通状态 / Die and lose Ember state — revert to normal |

### 余火状态 / Ember Form

| 功能 / Function | 说明 / Description |
|------|------|
| **HP上限+30%** | 大幅提升生存能力 / Greatly increases survivability |
| **可召唤合作 / Summon Co-op** | 看到金色召唤标记——召唤其他玩家帮助 / See gold summon signs — summon other players for help |
| **可被入侵 / Open to Invasion** | 余火状态下可能被暗灵入侵 / Can be invaded by dark spirits while embered |
| **余火获取 / Ember Sources** | 击败Boss自动获得/地图上拾取/NPC购买/刷洛斯里克骑士 / Auto-granted on boss kill / found on ground / bought from NPCs / farm Lothric Knights |

### 篝火 / Bonfires

篝火是黑暗之魂3的存档点——功能和法环的赐福类似 / Bonfires are the checkpoints in Dark Souls 3 — similar to Elden Ring's Sites of Grace.

| 功能 / Function | 说明 / Description |
|------|------|
| **休息 / Rest** | 恢复HP/FP/元素瓶，重置所有敌人（Boss除外）/ Restore HP/FP/Estus, respawn all non-boss enemies |
| **传送 / Warp** | 可在已点燃的篝火间自由传送 / Free warp between lit bonfires |
| **记忆 / Attune** | 在篝火处记忆/遗忘法术 / Attune/forget spells at bonfires |
| **注火 / Kindle** | 余火状态下可以注火——增加该篝火的元素瓶恢复量（三代改为遗骨系统）/ Kindle to increase Estus from that bonfire (changed in DS3) |

## 新手常见问题 / FAQ

| 问题 / Question | 回答 / Answer |
|------|------|
| **古达打不过？/ Iudex Gundyr too hard?** | 初始Boss——拉开距离观察招式。第一阶段人类形态可以用盾挡；第二阶段变身后的黑色触手攻击向左侧翻滚 / Starting boss — keep distance and learn attacks. Phase 1 human form: shield block. Phase 2: roll left to dodge black tendrils |
| **魂掉了捡不回来？/ Lost souls?** | 魂可以再刷——洛斯里克高墙的骑士一波约1000魂 / Souls are farmable — Lothric Knights at High Wall give ~1000 souls each |
| **武器怎么升级？/ How to upgrade?** | 传火祭祀场找安德烈铁匠——使用楔形石碎片 / Talk to Andre at Firelink Shrine — requires Titanite Shards |
| **怎么质变武器？/ How to infuse?** | 拿到各色宝石后在铁匠处质变——改变伤害类型和属性补正 / Give infusion gems to Andre to change damage type and scaling |
| **萤火怎么注火？/ How to kindle?** | 三代没有注火——遗骨系统取代。升级元素瓶恢复量 / Kindle system removed — Undead Bone Shards now upgrade healing instead |
| **怎么召唤队友？/ How to summon?** | 使用余火状态 → 看到地上的金色召唤标记 → 触碰召唤 / Use Ember → find gold summon signs on ground → touch to summon |
