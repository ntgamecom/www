## 灰烬墓地 / Cemetery of Ash + 传火祭祀场 / Firelink Shrine

游戏起点和枢纽区域 / Starting area and hub.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **灰烬墓地 / Cemetery of Ash** | 新手教程关——击败古达后正式进入游戏 / Tutorial area — enter the game after defeating Gundyr | 古达 Boss / Iudex Gundyr boss |
| **传火祭祀场 / Firelink Shrine** | 游戏枢纽——升级/强化/商人都在这里 / Main hub — leveling, upgrades, merchants all here | 安德烈(铁匠)、侍女(商人)、火防女(升级)、祭器场(传送) / Andre, Handmaid, Fire Keeper, Shrine Bonfire |

> **关键**：祭祀场中间的大剑是"螺旋剑"——可以初始武器但意义不大。从祭祀场右侧的门出去可快速回到灰烬墓地 / **Key**: Greatsword in the middle is the "Coiled Sword" — skip it. Right door from Firelink leads back to Cemetery

## 洛斯里克高墙 / High Wall of Lothric

游戏的第一个正式关卡 / The game's first proper level.

| 篝火 / Bonfire | 说明 / Description | 重点 / Key Points |
|------|------|------|
| **高墙第一篝火 / High Wall Bonfire** | 起点——出门上楼顶 / Start — go up to the rooftops | 楔形石碎片、蓝泪石戒指 / Titanite Shard, Blue Tearstone Ring |
| **高墙第二篝火 / Tower on the Wall** | 出门后有个喷火龙——快速跑过去 / A dragon breathes fire — sprint past it | 元素碎片 / Estus Shard |
| **冷冽谷的玻尔多 / Vordt** | 区域Boss——大型野兽 / Area boss — large beast | 玻尔多的灵魂 + 余火 / Soul of Vordt + Ember |

**Boss: 冷冽谷的玻尔多 / Vordt of the Boreal Valley**
- 打法很简单——**钻胯下**。第一阶段躲他正下方猛打腿 / Easy fight — **get under him**. Phase 1: stay under his belly and attack
- 第二阶段他开始冲锋和冰霜吐息——同样钻胯下翻滚躲 / Phase 2: charge and frost breath — stay under, dodge roll
- 击败后通往不死聚落 / Defeat to unlock path to Undead Settlement

## 不死聚落 / Undead Settlement

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **聚落中央 / Settlement Center** | 有大量村民和火把敌人 / Lots of villagers and torch enemies | 元素碎片、返回骨片 / Estus Shard, Homeward Bone |
| **大树 / Greatwood Arena** | Boss区域——**诅咒大树** / Greatwood boss arena | 诅咒大树（掉落冶炼炉——用于质变）/ Curse-Rotted Greatwood (drops Transposing Kiln) |
| **罗蕾塔 / Loretta's Bonfire** | 捷径通往Boss / Shortcut to boss | 大皮盾 / Large Leather Shield |

**Boss: 诅咒大树 / Curse-Rotted Greatwood**
- 打他身上的**蛋蛋**——白色的球状物是弱点 / Attack the **eggs/pustules** on his body — these are the weak points
- 第一阶段打腿上的蛋→第二阶段他站起来后打手臂和背上的蛋 / Phase 1: eggs on legs → Phase 2: eggs on arms and back
- 掉落**冶炼炉**——给安德烈可获得Boss魂武器 / Drops **Transposing Kiln** — for boss soul weapons at Andre

## 活祭品之路 + 法兰要塞 / Road of Sacrifices & Farron Keep

| 区域 / Area | 说明 / Description | 关键内容 / Key Content |
|------|------|------|
| **活祭品之路 / Road of Sacrifices** | 沼泽前的通道——有大量乌鸦人和螃蟹 / Path to the swamp — many crow people and crabs | 水晶老者Boss、元素碎片 / Crystal Sage boss, Estus Shard |
| **法兰要塞 / Farron Keep** | 毒沼泽——注意走位，避免陷入毒池 / Poison swamp — watch your step to avoid poison | 熄灭三座火焰→开启深渊监视者Boss门 / Extinguish 3 flames → open Abyss Watchers boss door |

**Boss: 深渊监视者 / Abyss Watchers**
- **第一阶段**: 一个监视者对你攻击——但另有其他监视者会攻击他（包括他自己人）。利用混战从背后偷袭。反复击杀直到进入第二阶段 / Phase 1: One Watcher attacks you — but others attack him too. Use the chaos to backstab. Kill until Phase 2
- **第二阶段**: 监视者获得火焰附魔——攻击更猛范围更大。翻滚躲避火焰连招——等他出完大硬直后输出 / Phase 2: Gains fire enchantment — wider attacks. Roll through fire combos — punish after big finishers
- 掉落**深渊监视者的灵魂** / Drops **Soul of the Abyss Watchers**

## 卡萨斯地下墓穴 / Catacombs of Carthus

| 篝火 / Bonfire | 说明 / Description | 重点 / Key Points |
|------|------|------|
| **墓穴入口 / Catacombs Entrance** | 进门后有很多骷髅——用圣属性武器可以防止它们复活 / Skeletons revive unless killed with holy weapons | 卡萨斯卷轴（给欧贝克）/ Carthus Scroll (give to Orbeck) |
| **霸王沃尼尔 / Wolnir** | Boss——巨大的骷髅王 / Giant skeleton king boss | 打他手上的金色手镯——打碎三个即胜利 / Attack the gold bracelets on his arms — break all 3 to win |

**Boss: 霸王沃尼尔 / High Lord Wolnir**
- 这是游戏中最**简单的Boss之一**——攻击他左右手腕上的金色手镯 / One of the **easiest bosses** — attack the gold bracelets on his wrists
- 打碎三个手镯后他就死了——注意不要被他的毒雾逼到角落 / Break all 3 bracelets and he dies — avoid being cornered by his poison mist

## 幽邃教堂 / Cathedral of the Deep

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **教堂外围 / Cathedral Perimeter** | 有大量巨型奴隶和史莱姆的教堂外院 / Large slavemen and slimes in the courtyard | 元素碎片、誓约"青之守护者" / Estus Shard, Blades of the Darkmoon covenant |
| **Rosaria's Bed Chamber | 幽邃教会的核心——可洗点重置属性 / Core of Cathedral — respec available | 罗莎莉亚指头(契约) + 洗点功能 / Rosaria's Fingers covenant + respec |

**Boss: 幽邃主教群 / Deacons of the Deep**
- 这是**群战Boss**——大量主教围攻你。穿重甲举大盾或使用范围武器（如冰剑）/ **Crowd boss** — many deacons swarm you. Wear heavy armor + greatshield or use AoE weapons (like Irithyll Straight Sword)
- 当出现发红光的**主教长**时——优先击杀他，Boss血量才会减少 / When the red-glowing **Archdeacon** appears — focus him, only his death lowers the boss HP bar

## 冷冽谷 / Irithyll of the Boreal Valley

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **冷冽谷入口 / Irithyll Entrance** | 进入后被银色骑士狙击——跑步前进找掩体 / Silver knight snipers — sprint between cover | 元素碎片 / Estus Shard |
| **沙力万的野兽 / Sulyvahn's Beast** | 可选Boss——桥下的大鳄鱼 / Optional boss — giant crocodile under bridge | 戒指 + 魂 / Ring + souls |
| **亚诺尔隆德 / Anor Londo** | 前作经典场景重现——黑骑士所在地 / Classic return to Anor Londo | 拟态奇迹、太阳公主戒指 / Hidden Body miracle, Ring of the Sun Princess |

**Boss: 教宗沙力万 / Pontiff Sulyvahn**
- **第一阶段**: 使用火焰刀和魔力刀双持攻击。他的攻击节奏快但前摇明显——熟悉后可以用盾挡或翻滚 / Phase 1: Dual-wields fire + magic swords. Fast but telegraphed — shield block or roll
- **第二阶段**: 沙力万召唤一个分身——**分身是透明的**，打分身也可以伤害本体。分身出现后优先击杀分身 / Phase 2: Summons a clone — the **clone is translucent**, damaging it also hurts the boss. Kill clone first
- **关键技巧**: 进门时可背刺开局——节省大量精力 / **Key tip**: You can start the fight with a backstab — saves a lot of effort
- 掉落**沙力万的灵魂** / Drops **Soul of Pontiff Sulyvahn**

## 罪业之都 / Profaned Capital

**Boss: 巨人尤姆 / Yhorm the Giant**
- **特殊Boss**——你不需要和他硬拼 / **Special boss** — you don't need to fight him directly
- 在Boss战场地后方有一把**风暴管束者**（看起来像废铁）/ There's a **Storm Ruler** (looks like scrap metal) at the back of the arena
- 装备风暴管束者→**蓄力战技**（L2长按→R1释放）→发出冲击波造成巨额伤害 / Equip Storm Ruler → **charge the weapon art** (hold L2 → R1) → shockwave deals massive damage
- 大约4-5次蓄力攻击即可击杀尤姆。这是剧情向战斗 / 4-5 charged attacks kill Yhorm. It's a puzzle fight
- 如果**洋葱骑士 Siegward**活着并参加了战斗——他会帮你一起使用风暴管束者 / If **Siegward of Catarina** is alive and joins — he helps with the Storm Ruler too

## 洛斯里克城 + 大书库 / Lothric Castle & Grand Archives

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **洛斯里克城 / Lothric Castle** | 击败三个薪王后通过舞娘到达 / Reach after defeating 3 Lords of Cinder through Dancer | 猎龙铠甲Boss、舞娘Boss / Dragonslayer Armour boss, Dancer boss |
| **大书库 / Grand Archives** | 满布学者的区域——有三胖守护 / Scholar-filled area — guarded by 3 fat wax enemies | 王子双子Boss、结晶老者卷轴 / Lothric & Lorian boss, Crystal Scroll |
| **无主墓地 / Untended Graves** | 隐藏区域——从传火祭祀场进电梯后走隐藏门 / Hidden area — hidden door from Firelink elevator | Champion Gundyr boss + 灭火结局路线 / Champion Gundyr + Usurpation of Fire route |

**Boss: 洛斯里克王子 / Lothric & Lorian, Twin Princes**
- **第一阶段**: 洛里安（哥哥）攻击——他行动缓慢但瞬移到你身边。翻滚躲避瞬移斩 / Phase 1: Lorian (brother) attacks — slow but teleports. Roll to dodge teleport slashes
- **第二阶段**: 洛斯里克（弟弟）骑在哥哥背上——增加远程魔法攻击。此时需要同时击杀两人 / Phase 2: Lothric rides on Lorian's back — adds ranged magic. Must kill both
- **关键**: 洛斯里克会复活洛里安——如果你杀死洛里安而不打洛斯里克，法师会复活他。必须杀死洛斯里克才能结束战斗 / **Key**: Lothric revives Lorian — kill Lothric after Lorian to finish the fight
- 掉落**双王子的灵魂** / Drops **Soul of the Twin Princes**

## 初始火炉 / Kiln of the First Flame

**Boss: 薪王化身 / Soul of Cinder**
- **最终Boss**——传火终章的守门人 / **Final boss** — keeper of the First Flame
- **第一阶段**: 使用多种武器形态——长剑（标准）、曲剑（快速连击）、法杖（远程魔法）、长枪（突刺）。每种形态变换时是输出窗口 / Phase 1: Multiple weapon forms — longsword (standard), curved sword (fast combos), staff (ranged magic), spear (thrusts). Punish during form transitions
- **第二阶段**: 薪王化身获得火焰附魔——使用螺旋剑的连续火焰斩。**注意他的五连斩和雷击**——翻滚躲避 / Phase 2: Gains fire enchantment — coiled sword flame combos. **Watch for 5-hit combo and lightning** — roll through
- 击败后选择结局——传火/灭火/篡火 / Choose ending — Link/End/Usurp the Fire

## DLC区域 / DLC Areas

### 艾雷德尔之烬 / Ashes of Ariandel

| 地点 / Location | 说明 / Description | Boss |
|------|------|------|
| **绘画世界 / Painted World** | 被雪覆盖的绘画世界——通过祭祀场篝火旁的新NPC进入 / Snow-covered painted world — enter via new NPC at Firelink bonfire | 修女芙莉德 / Sister Friede (3阶段) |

**Boss: 修女芙莉德 / Sister Friede**
- **3阶段Boss战**——游戏最难Boss之一 / 3-phase boss fight — one of the hardest in the game
- **P1**: 芙莉德使用镰刀+隐身。她隐身时注意雪花的轨迹找到她的位置 / Scythe + invisibility. Watch snow trail when she goes invisible
- **P2**: 神父艾雷德尔加入战斗——大胖子火焰攻击。优先击杀神父 / Father Ariandel joins — big man fire attacks. Kill the father first
- **P3**: 芙莉德黑化——黑焰镰刀+大招。保持距离，等她出完大硬直后再输出 / Darkened Friede — black flame scythe + ultimate moves. Stay mid-range, punish after big attacks

### 环印城 / The Ringed City

| 地点 / Location | 说明 / Description | Boss |
|------|------|------|
| **环印城 / The Ringed City** | 通过绘画世界的尽头进入——世界终末之地 / Enter through end of Painted World — end of the world area | 恶魔王子、吞噬黑暗的米狄尔、奴隶骑士盖尔 / Demon Prince, Darkeater Midir, Slave Knight Gael |

**关键Boss: 奴隶骑士盖尔 / Slave Knight Gael**
- **最终DLC Boss**——被认为是黑魂系列最精彩的Boss战之一 / **Final DLC boss** — considered one of the best fights in the entire series
- **P1**: 盖尔使用大剑+连发弩——攻擊节奏是"慢→快→慢" / Greatsword + repeating crossbow — attack rhythm is "slow → fast → slow"
- **P2**: 盖尔进入狂暴状态——使用黑暗灵魂之力造成范围伤害。他会跳起后砸地——翻滚躲避 / Enters berserk mode — dark soul AoE damage. Jump slam — roll to dodge
- **P3**: 雷电附体——盖尔召唤雷电攻击大范围区域。**保持近身战斗**——远离反而更容易被雷电击中 / Lightning buff — calls lightning strikes. **Stay close** — being far away makes lightning harder to dodge
