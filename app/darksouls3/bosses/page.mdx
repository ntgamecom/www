# BOSS攻略 / Boss Strategies

黑暗之魂3共有25个Boss（本体19+ DLC 6）——本章按推图顺序详解每个Boss的打法和要点 / Dark Souls 3 has 25 bosses (19 base + 6 DLC) — this chapter covers every boss in recommended order with detailed strategies.

---

## 本体Boss / Base Game Bosses

### 灰烬审判者古达 / Iudex Gundyr

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 灰烬墓地 / Cemetery of Ash |
| **推荐等级 / Level** | 1-5 |
| **弱点 / Weakness** | 雷、打击 / Lightning, Strike |
| **抗性 / Resistances** | 无 / None |

**打法:**
- **第一阶段（人形态）**: 古达使用长戟——前摇长。举盾绕圈，等他出完招后砍1-2刀 / Phase 1 (Human): Uses halberd — slow wind-ups. Shield up, circle, attack 1-2 times after his combo
- **第二阶段（黑蛇形态）**: 古达身上长出黑色触手——攻击范围变大。**不要贪刀**——触手的抓取投技很危险 / Phase 2 (Black Serpent): Grows black tendrils — wider range. **Don't get greedy** — grab attack is dangerous
- 推荐**战士**或**骑士**开局——初始武器和盾牌足够打 / Recommended starting class: **Warrior** or **Knight** — starting gear is sufficient
- 掉落：**螺旋剑** + 余火 / Drops: **Coiled Sword** + Ember

### 冷冽谷的玻尔多 / Vordt of the Boreal Valley

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 洛斯里克高墙 / High Wall of Lothric |
| **推荐等级 / Level** | 15-25 |
| **弱点 / Weakness** | 雷、打击 / Lightning, Strike |
| **抗性 / Resistances** | 冰霜 / Frost |

**打法:**
- **全程钻胯下**——玻尔多的腹部是安全区 / **Stay under him** — his belly is a safe zone
- 第一阶段：他攻击时向前翻滚钻到他正下方——所有攻击都打不到你 / Phase 1: Roll forward under him when he attacks — all his attacks miss
- 第二阶段：他开始冲锋和冰霜吐息——同样钻胯下，他的冰霜吐息喷不到正下方 / Phase 2: Charge + frost breath — same strategy, frost breath can't hit directly underneath
- 掉落：**玻尔多的灵魂**（可换玻尔多冰锤）/ Drops: **Soul of Vordt** (transposable to Vordt's Great Hammer)

### 诅咒大树 / Curse-Rotted Greatwood

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 不死聚落 / Undead Settlement |
| **推荐等级 / Level** | 20-35 |
| **弱点 / Weakness** | 打击、火 / Strike, Fire |
| **抗性 / Resistances** | 出血、毒 / Bleed, Poison |

**打法:**

- 第一阶段：打腿部和侧面的蛋——打爆后木桩会倒地 / Phase 1: Destroy eggs on legs and sides — he falls over after enough damage
- 第二阶段：他站起来后露出新蛋——在他手臂和背部 / Phase 2: New eggs appear on arms and back
- 注意躲避他的抓取和**放屁坐**——伤害很高 / Watch for grab attack and **butt slam** — high damage
- 掉落：**冶炼炉**（给安德烈可制作Boss魂武器）/ Drops: **Transposing Kiln** (give to Andre for boss weapons)

### 深渊监视者 / Abyss Watchers

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 法兰要塞 / Farron Keep |
| **推荐等级 / Level** | 35-50 |
| **弱点 / Weakness** | 雷、出血 / Lightning, Bleed |
| **抗性 / Resistances** | 火 / Fire |

**打法:**
- **第一阶段**: 一个红眼监视者攻击你——但很快会有第二个（灰眼）加入，他会攻击红眼监视者！这是混战——等待他们互殴时从背后**背刺**红眼 / Phase 1: One red-eye Watcher attacks you — but a second (grey-eye) joins and attacks the first! Use the chaos to **backstab** the red-eye
- 你需要击杀红眼监视者——灰眼是你的帮手（虽然是暂时的）/ You must kill the red-eye — the grey-eye is your ally (temporarily)
- 第二阶段：击败一轮后，监视者全员死亡→一个监视者获得**火焰附魔**复活 / Phase 2: After one round, all Watchers die → one revives with **fire enchantment**
- 火焰监视者攻击范围变大，有火焰连斩和跳劈。翻滚躲避——他的大硬直是他砸地后的长时间收招 / Fire Watcher has wider combo and jumping slam — roll through, punish after ground slam
- 掉落：**深渊监视者的灵魂**——可换法兰大剑或狼骑士大剑 / Drops: **Soul of the Abyss Watchers** — transpose to Farron Greatsword or Wolf Knight's Greatsword

### 教宗沙力万 / Pontiff Sulyvahn

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 冷冽谷 / Irithyll |
| **推荐等级 / Level** | 55-70 |
| **弱点 / Weakness** | 出血、打击 / Bleed, Strike |
| **抗性 / Resistances** | 魔力、雷 / Magic, Lightning |

**打法:**
- **背刺开局**：进入雾门后往前走一步→沙力万会缓慢转身→趁机**背刺他** / **Backstab start**: Enter fog → take one step → Sulyvahn slowly turns → **backstab him**
- 第一阶段：双持火焰刀+魔力刀——他的连招长但可举盾挡（推荐大盾）/ Phase 1: Fire + magic swords — long combos but blockable (greatshield recommended)
- 第二阶段：沙力万召唤一个**透明分身**。**分身一出现立刻击杀**——打分身也能伤害本体 / Phase 2: Summons a **transparent clone** — **kill the clone immediately** — damaging the clone hurts the boss too
- 他的**延迟斩**非常坑——看到抬手不要立刻滚，等剑快落下时再滚 / His **delayed slash** is tricky — don't roll on wind-up, roll when the sword is about to fall
- 如果不举盾，练习翻滚节奏是关键——他大部分攻击的翻滚时机是"看到出手→等0.5秒→滚" / Without a shield, practice roll timing — "see attack → wait 0.5s → roll"

### 巨人尤姆 / Yhorm the Giant

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 罪业之都 / Profaned Capital |
| **推荐等级 / Level** | 65-75 |
| **特殊机制 / Special** | 需要风暴管束者 / Storm Ruler required |

**打法:**
- **特殊机制Boss**——不需要普通的砍杀 / **Puzzle boss** — normal attacks are ineffective
- 在Boss战场地后方捡**风暴管束者**（一把破损的大剑）/ Pick up **Storm Ruler** at the back of the arena (a broken greatsword)
- 装备风暴管束者→**双手持**→按住**战技键（L2）蓄力**→剑身发出冲击→释放（R1/R2）→发出冲击波 / Equip Storm Ruler → **two-hand it** → **hold weapon art (L2) to charge** → blade glows → release (R1/R2) → shockwave
- 每次蓄力冲击波造成约25%血量伤害——4-5次即可击杀 / Each shockwave deals ~25% HP — kill in 4-5 hits
- 如果你做了**洋葱骑士 Siegward**支线——他会出现在Boss战场协助，也使用风暴管束者，大幅加快战斗 / If you completed **Siegward's questline** — he appears and helps with his own Storm Ruler

### 洛斯里克双王子 / Twin Princes: Lothric & Lorian

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 大书库 / Grand Archives |
| **推荐等级 / Level** | 75-85 |
| **弱点 / Weakness** | 雷、出血 / Lightning, Bleed |
| **抗性 / Resistances** | 魔力、暗 / Magic, Dark |

**打法:**
- **第一阶段（洛里安）**: 洛里安使用火焰大剑——他会**瞬移**到你身边攻击。关键：他瞬移后不要立刻滚——等剑光出现后再滚 / Phase 1 (Lorian): Fire greatsword — **teleports** to you. Key: don't roll immediately after teleport — roll when the flash appears
- 他有一招长时间蓄力的范围爆炸——看到后跑开或者连续向后翻滚躲避 / Watch for the long-charge AoE explosion — run away or chain roll backward
- **第二阶段（双王子）**: 洛里安骑在洛斯里克背上——增加了远程魔法弹幕攻击。需要同时打两个人的血量 / Phase 2 (Twin Princes): Lorian rides Lothric — adds ranged magic barrages. Must deplete both HP bars
- **关键机制**: 如果你杀死洛里安而洛斯里克活着，洛斯里克会**复活**洛里安。你必须打死洛里安的当前血条后，趁洛斯里克还没复活他之前攻击洛斯里克本人（在他背上）**/ Key mechanic**: If you kill Lorian while Lothric lives, Lothric **revives** him. You must damage Lothric (on Lorian's back) before he can revive
- 推荐使用**高DPS武器**——快速且能在洛斯里克施法间隙打到背上的人 / Use **high DPS weapons** — fast enough to hit Lothric during casting windows

### 薪王化身 / Soul of Cinder

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 初始火炉 / Kiln of the First Flame |
| **推荐等级 / Level** | 80-95 |
| **弱点 / Weakness** | 雷 / Lightning |
| **抗性 / Resistances** | 火（第二阶段）/ Fire (Phase 2) |

**打法:**
- **最终Boss**——他代表了所有传火者的化身 / **Final boss** — embodiment of all who linked the fire
- **第一阶段**: 他会变换四种武器形态 / Phase 1: Switches between 4 weapon forms:
  - **长剑形态**: 标准连击——最好应对 / Longsword: standard combos — easiest to handle
  - **曲剑形态**: 快速连击——拉开距离等他出完招 / Curved Sword: fast combos — keep distance, punish after
  - **法杖形态**: 远程魔法——冲到脸上打 / Staff: ranged magic — rush him
  - **长枪形态**: 突刺——侧向翻滚 / Spear: thrusts — roll sideways
- 每种形态切换时是**最佳输出窗口** / Form transitions are **the best DPS windows**
- **第二阶段**: 薪王化身拔出螺旋剑——获得**火焰附魔**。他的招牌五连斩非常致命——第一下横斩→第二下竖斩→第三下横斩→第四下竖斩→第五下大范围爆炸——全部翻滚躲避 / Phase 2: Pulls out Coiled Sword — **fire enchantment**. His 5-hit combo is deadly: horizontal → vertical → horizontal → vertical → AoE explosion — roll through all
- 还会使用**雷枪投掷**和**阳光之枪**——看到他举枪时横向翻滚 / Also uses **Lightning Spear** and **Sunlight Spear** — roll sideways when he winds up
- 掉落：**薪王的灵魂** / Drops: **Soul of the Soul of Cinder**

### 修女芙莉德 / Sister Friede (Ashes of Ariandel)

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 绘画世界 / Painted World |

- **P2**: 神父艾雷德尔加入——大胖子火焰+冰霜双重攻击。**优先击杀神父**——他和芙莉德在同一个位置时两人都打得到 / Father Ariandel joins — fire + frost attacks. **Focus the father** — when close together, AoE hits both
- **P3**: 芙莉德觉醒——黑焰镰刀。速度极快，她的**黑焰大范围斩击**需要完美翻滚。注意她的**跳起抓取投技**——看到她跳起立刻向后滚 / Friede awakens — black flame scythe. Extremely fast. **Black flame AoE slash** needs perfect roll timing. **Jumping grab** — roll back immediately when she jumps

### 奴隶骑士盖尔 / Slave Knight Gael (Ringed City)

- **P2**: 盖尔狂暴——使用**黑暗灵魂**之力。他会释放大范围暗爆——跳起或翻滚躲避。他的**七连斩**是致命招式，不断向后翻滚躲避 / Goes berserk — **Dark Soul** power. Large dark explosion — jump or roll. His **7-hit combo** is deadly — chain roll backward

- 建议使用**出血武器**——盖尔出血抗性不高 / Use **bleed weapons** — Gael's bleed resistance is moderate
