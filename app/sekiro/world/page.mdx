## 破旧寺院 / Dilapidated Temple

游戏枢纽区域——经过序章后到达。佛雕师所在的寺庙 / Game hub — reached after the prologue. Home of the Sculptor.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **破旧寺院 / Temple** | 佛雕师在此——升级忍义手 / Sculptor upgrades prosthetic tools | 忍义手升级、不死半兵卫(练习) / Prosthetic upgrades, Undying Soldier (training) |
| **寺院井底 / Well** | 序章所在——后期可回来拿道具 / Prologue location — return for items later | 忍义手的基本工具(手里剑) / Shuriken prosthetic tool |

## 苇名城 城外 / Ashina Outskirts

游戏第一个正式区域——从破旧寺院出发，经过城门通往苇名城 / The game's first proper area — from the temple through the gates to Ashina Castle.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **城门外 / Outskirts Gate** | 有大量足轻兵——暗杀清图 / Many ashigaru — stealth clear | 地图碎片、葫芦种子 / Map fragment, Gourd Seed |
| **城门道 / Gate Path** | 有精英敌人"山内典膳" / Elite "Tenzen Yamauchi" | 佛珠 / Prayer Bead |
| **虎口阶梯 / Staircase** | 大量敌人——突击Boss鬼庭形部雅孝的路线 / Route to Gyoubu boss | 鬼庭形部雅孝Boss——骑马的武士将军 / Gyoubu boss — mounted samurai general |
| **城邑进口 / Castle Gate** | 过了桥就是苇名城主城入口 / Cross bridge to Ashina Castle entrance | 赤鬼(精英) / Chained Ogre (mini-boss) |

**Boss: 鬼庭形部雅孝 / Gyoubu Oniwa**
- 骑马Boss——用钩绳拉近+攻击。他跑远时用钩绳飞到他身边 / Mounted boss — use grappling hook to close distance. When he rides far, grapple to him
- 打法：挡住他的攻击→钩绳拉近距离→攻击2-3刀→重复 / Strategy: Block → grapple → attack 2-3 times → repeat
- 掉落：**机关伞**（忍义手）/ Drops: **Loaded Umbrella** (prosthetic)

## 平田宅邸 / Hirata Estate (可选 / Optional)

通过破旧寺院佛像前的铃铛进入——是回忆场景。强烈推荐早期探索以获得强力道具 / Enter via the bell in front of the Buddha statue at the temple — a memory scene. Highly recommended to explore early for powerful items.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **宅邸外围 / Estate Path** | 竹林——有大量忍者 / Bamboo grove — many shinobi | 佛珠、金钱袋 / Prayer Bead, money pouch |
| **宅邸正门 / Estate Gate** | 有精英"忍者猎人" / Elite "Shinobi Hunter" | 佛珠 / Prayer Bead |
| **宅邸内部 / Estate Interior** | 着火建筑——有大量物品 / Burning building — many items | 吹火筒(忍义手) / Flame Vent (prosthetic) |
| **平田宅邸 Boss / Hirata Boss** | 蝴蝶夫人——可选Boss / Lady Butterfly — optional boss | **樱花枝**(关键道具) / **Sakura Branch** (key item) |

**Boss: 蝴蝶夫人 / Lady Butterfly**
- 第一阶段：她近战攻击——格挡+攻击打崩架势。她跳上钢丝时用**手里剑**打下来 / Phase 1: Melee — deflect + attack for posture. **Shuriken** knocks her off wires
- 第二阶段：增加幻影小兵——不要慌，跑动躲避到幻影消失或用**鸣钟**秒杀他们 / Phase 2: Adds phantom minions — stay mobile until they disappear, or use **Snap Seeds** to instantly kill them
- 关键：**不要给她喘息机会**——保持贴身压制 / **Key: Don't give her breathing room** — stay close and aggressive
- 掉落：**樱花枝**（关键道具——用于开启另一个结局路线）/ Drops: **Sakura Branch** (key item — used for alternate ending)

## 苇名城 主城 / Ashina Castle

游戏中期核心区域——多层垂直结构，连接多个区域 / Core mid-game area — multi-layered vertical structure connecting to multiple regions.

| 楼层 / Floor | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **主城一楼 / Ground Floor** | 大量武士和弓箭手——建议从屋顶暗杀 / Many samurai and archers — stealth from rooftops | 苇名流秘籍(技能书)、武士们 / Ashina Style skill book, samurai |
| **主城二楼 / Second Floor** | 有天狗(苇名一心)——给你"苇名流秘籍" / Tengu (Isshin) — gives "Ashina Style" skill book | 苇名流技能、葫芦种子 / Ashina skills, Gourd Seed |
| **主城顶层 / Top Floor** | **苇名弦一郎Boss** / Genichiro boss fight | 弦一郎——之后获得"不死斩" / Genichiro — then acquire "Mortal Blade" |
| **主城地下 / Underground** | 苇名之底的入口 / Entrance to Ashina Depths | 隐匿房间——有重要NPC / Hidden room — important NPC |
| **瞭望塔 / Watchtower** | 连接城外和主城 / Connects outskirts to castle | 多个佛珠 / Multiple prayer beads |

**Boss: 苇名弦一郎 / Genichiro Ashina**
- **游戏试金石**——如果你能打过他，你就掌握了只狼的核心战斗 / **Game's skill check** — if you can beat him, you've mastered Sekiro's core combat
- **第一阶段**: 弓+刀——他的箭射速中等但伤害高。近战格挡节奏是"2连→停顿→3连" / Phase 1: Bow + sword — arrows are dangerous. Deflect rhythm is "2-hit → pause → 3-hit"
- **第二阶段**: 增加雷反——他跳起放雷时**跳跃接雷→在空中按攻击**把雷打回去，造成大量架势伤害+眩晕 / Phase 2: Adds lightning reversal — **jump to catch lightning → press attack mid-air** to throw it back, massive posture damage + stun
- **第三阶段**: 实际上和第二阶段一样——但他会更激进地使用雷反。用好雷反可以快速击杀 / Phase 3: Same as phase 2 but more aggressive lightning. Lightning reversal wins the fight
- 掉落：**不死斩**（关键道具）/ Drops: **Mortal Blade** (key item)

## 坠落之谷 / Sunken Valley

从主城出来后经过毒沼泽到达——有大量远程火枪兵 / Reach via poison swamp after leaving the castle — many ranged gunners.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **毒沼 / Poison Swamp** | 有毒气的峡谷——快速通过或用解毒粉 / Poison gas canyon — sprint through or use Antidote Powder | 佛珠、铁屑 / Prayer Bead, Scrap Iron |
| **铁炮要塞 / Gun Fort** | 大量火枪兵——利用钩绳快速近身 / Many gunners — use grappling to close distance quickly | 火枪精英——长爪蜈蚣 / Gun Fort elite — Long-arm Centipede |
| **狮子猿巢穴 / Ape Nest** | **狮子猿Boss场地** / Guardian Ape boss arena | 狮子猿（Boss）/ Guardian Ape |

**Boss: 狮子猿 / Guardian Ape**
- **第一阶段**: 野兽形态——疯狂的爪击+放毒+丢屎。**格挡他的攻击**——他虽然凶但攻击前摇大。他丢屎时向前跑 / Phase 1: Beast form — crazy claw swipes + poison + poop throw. **Deflect his attacks** — fierce but telegraphed. Run forward when he throws poop
- **第二阶段**: 狮子猿被忍杀后原地复活——**无头状态**用剑攻击。这阶段的攻击节奏变了——他的大范围横斩用跳跃踩头化解 / Phase 2: Revives headless — attacks with a sword. **新节奏** — wide horizontal slash countered by jumping on his head
- 关键：第二阶段他有一招大喊+下斩——完美格挡后他会长时间硬直 / **Key**: His roar + overhead slam in phase 2 — perfect deflect leaves him stunned for a long punish
- 掉落：**爱哭鬼的指环** + 忍义手材料 / Drops: **Malcontent's Ring** + prosthetic material

## 金刚山·仙峰寺 / Senpou Temple

从苇名城通过地牢到达——佛教寺院区域 / Reach via the dungeon from Ashina Castle — Buddhist temple area.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **仙峰寺山道 / Temple Path** | 有大量武僧——可暗杀 / Many monks — stealthable | 佛珠、仙峰寺拳法技能书 / Prayer Bead, Senpou skills |
| **正殿 / Main Hall** | 寺内中心——有重要NPC / Temple center — important NPC | 变若神子——给米 / The Divine Child of Rejuvenation — gives rice |
| **幻廊 / Illusory Hall** | **假破戒僧Boss** / Folding Screen Monkeys boss | 三只猴子——用隐形+走路轻手轻脚的方式一个个暗杀 / 3 monkeys — stealth kill each one without alerting the others |
| **寺院后山 / Temple Rear** | 通往菩萨谷 / Path to Bodhisattva Valley | 忍者突击——风车、铁伞 / Shinobi ambush |

**Boss: 假破戒僧 / Folding Screen Monkeys**
- "解谜Boss"——不是战斗Boss。三只隐形的猴子——需要利用环境击杀 / "Puzzle boss" — not a combat boss. Three invisible monkeys — use the environment to kill them
- **红猴**: 在点燃的烛台附近——熄灭火会让他现身 / Red monkey: Near lit candles — extinguish fire to reveal
- **绿猴**: 你在草丛中走时他会靠近你——引诱到铃铛处摇铃/ In grass — lure to the bell and ring it
- **紫猴**: 在宫内的暗处——等他坐下时暗杀 / In dark room — stealth kill when sitting

## 苇名之底 / Ashina Depths

从苇名城地下通道进入的阴暗区域——水下也有探索内容 / Dark area entered through Ashina Castle underground — includes underwater exploration.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **苇名之底 / Depths** | 阴暗洞穴+毒沼——有大量敌人 / Dark caves + poison swamp — many enemies | 佛珠、金刚屑 / Prayer Bead, Adamantite Scrap |
| **水生村 / Mibu Village** | 水下村庄——村民都是水生怪物 / Underwater village — villagers are water creatures | 破戒僧(精英)、尊贵饵食 / Corrupted Monk (mini-boss), Precious Bait |
| **隐藏森林 / Hidden Forest** | 被迷雾覆盖的区域——迷路时跟着火光走 / Mist-covered area — follow the lights when lost | 无首(精英) / Headless (mini-boss) |

**Boss: 无首狮子猿 / Headless Ape (双猿 / Twin Apes)**
- **第一阶段**: 单独的狮子猿（无头形态）——和之前一样打 / Phase 1: Solo headless ape — same as before
- **第二阶段**: 加入第二只白色狮子猿（普通形态）——**1v2**。关键：白色狮子猿的血少——优先击杀白色猿，然后1v1黑色无头猿 / Phase 2: A second white ape joins — **1v2**. **Key**: The white ape has less HP — kill it first, then 1v1 the black headless ape
- 掉落：**佛珠** + 忍义手材料 / Drops: **Prayer Bead** + prosthetic material

## 源之宫 / Fountainhead Palace

**最终区域**——通过仙峰寺的水底洞穴到达。一个被水淹没的绝美神秘之地 / **Final area** — reachable through a water cave from Senpou Temple. A beautiful, flooded mystical realm.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **源之宫入口 / Palace Entrance** | 瀑布——用钩绳上升 / Waterfall — grapple up | 佛珠、源之宫地图 / Prayer Bead, map |
| **宫之内部 / Palace Interior** | 贵族区域——敌人打起来很烦但都可以暗杀 / Noble area — annoying enemies but all stealthable | 金色回生(精英) / Golden Revitalizing (mini-boss) |
| **巨大樱树 / Great Sakura Tree** | **樱龙Boss**——最终Boss之一 / Divine Dragon boss — one of the final bosses | 樱龙的眼泪(关键道具) / Tears of the Dragon (key item) |
| **水下区域 / Underwater** | 可潜水探索——大量宝物 / Dive-able areas — lots of treasures | 佛珠、宝鲤鳞片 / Prayer Bead, Fulminated Mercury |

**Boss: 破戒僧 / Corrupted Monk (源之宫版)**
- 真身——比假破戒僧难得多。三段架势条 / Real version — much harder than the fake. Three health nodes
- 打法：他的攻击范围极大但前摇明显。**鞭炮**可以打断他的攻击。**突刺危**用识破，**横扫危**跳跃踩头 / Wide attacks but telegraphed. Use **Firecrackers** to interrupt. **Mikiri** thrusts, **jump** sweeps
- 用**飞雪**（神之飞雪）附魔可以造成额外伤害 / Use **Divine Confetti** for extra damage

**Boss: 樱龙 / Divine Dragon**
- **剧情Boss**——不算太难。需要利用钩绳在空中移动→接近樱龙→打树上发光的小龙→积累忍杀条 / **Story boss** — not very hard. Grapple to clouds → approach Divine Dragon → hit the glowing wisps on branches → build deathblow meter
- 关键：**雷反**——樱龙的雷攻击用跳跃接住打回去 / **Key**: **Lightning reversal** — catch its lightning and throw it back
- 掉落：**樱龙的眼泪**——用于结局 / Drops: **Tears of the Divine Dragon** — for ending

| Boss | 地点 / Location | 说明 / Description |
|------|------|------|
| **苇名一心 / Isshin Ashina** | 苇名城顶层 / Ashina Castle top | 用火焰攻击的老人——第一阶段用刀，第二阶段加入火焰附魔。注意他的**一文字斩**—跳跃躲避 / Old man with fire — phase 1 sword, phase 2 fire enchantment. Watch for **Ichimonji** — jump to dodge |

| Boss | 地点 / Location | 说明 / Description |
|------|------|------|

| **第二-四阶段: 剑圣苇名一心 / Sword Saint Isshin** | 同上 / Same | **共4个阶段**——游戏真正的最终Boss / **4 phases** — the true final boss |
