# 新手入门 / Beginner Guide

只狼的战斗系统与魂系截然不同——放弃翻滚，拥抱格挡。本章涵盖所有核心机制 / Sekiro's combat is fundamentally different from souls games — abandon rolling, embrace parrying. This chapter covers all core mechanics.

---

## 核心操作 / Core Controls (PC/PS)

| 操作 / Action | PS | PC | 说明 / Description |
|------|:----:|:----:|------|
| **攻击 / Attack** | R1 | 鼠标左键 / Left click | 普通攻击——连续攻击压制敌人架势 / Normal attack — chain attacks to pressure posture |
| **格挡 / Deflect** | L1 | 鼠标右键 / Right click | **核心操作**——精准格挡（防反）增加敌人架势。比魂系盾牌重要一百倍 / **Core mechanic** — precise deflect builds enemy posture. 100x more important than souls blocking |
| **闪避 / Dodge** | ○ | Space | 几乎没有无敌帧——主要用于位移而非躲攻击 / Almost no i-frames — for positioning, not dodging |
| **跳跃 / Jump** | ✕ | Space(按住) / Hold Space | 跳跃踩头（对下段危）/ Jump to head-stomp (vs sweep perilous attacks) |
| **钩绳 / Grappling Hook** | R2 | Shift | 飞檐走壁的核心——可抓钩到特定位置 / Core mobility — grapple to marked points |
| **忍义手 / Prosthetic** | L2 | Q | 使用装备的忍义手工具 / Use equipped prosthetic tool |
| **使用道具 / Use Item** | □ | E | 使用快捷道具（药丸/葫芦）/ Use quick item (pellets/gourd) |
| **潜行 / Stealth** | 蹲伏+背后 / Crouch + behind | 从背后暗杀秒杀——对大多数小兵和部分精英有效 / Backstab for instant kill — works on most mobs and some elites |
| **忍杀 / Deathblow** | △ + R1 | F | 架势崩了后执行忍杀——减少敌人一条命 / Execute deathblow on broken posture — depletes one health node |
| **锁定 / Lock On** | R3 | 鼠标中键 / Middle click | 锁定敌人 / Lock-on to enemy |
| **切换目标 / Switch Target** | R3 → 方向 / R3 + direction | 鼠标中键+方向 / Middle click + direction | 在多个敌人间切换锁定 / Switch lock between multiple enemies |
| **危 - 识破 / Mikiri Counter** | 见突刺危→○向前 / vs thrust → forward ○ | 识破突刺危——对突刺攻击使用可大幅增加架势伤害 / Counter thrust perilous — press dodge toward thrust for massive posture damage |
| **危 - 跳跃踩头 / Jump Counter** | 见横扫危→跳跃 / vs sweep → jump | 跳跃踩敌人头来化解横扫攻击 / Jump on enemy head to counter sweep attacks |

## 架势系统详解 / Posture System

架势是只狼战斗的核心——比血量更重要 / Posture is the core of Sekiro's combat — more important than HP.

| 概念 / Concept | 说明 / Description |
|------|------|
| **架势条 / Posture Bar** | 屏幕上方——你的架势在左边，敌人在右边。满格=架势崩溃 / Top of screen — yours on left, enemy on right. Full = posture broken |
| **架势恢复 / Posture Recovery** | 不攻击/格挡时架势自动恢复。血量越低恢复越慢 / Auto-recovers when idle. Lower HP = slower recovery |
| **打崩架势 / Break Posture** | 攻击命中+精准格挡增加敌人架势。崩了=可忍杀 / Attacks + deflections build enemy posture. Broken = deathblow |
| **精准格挡 / Perfect Deflect** | 在攻击命中瞬间按L1——不扣HP+大幅增加敌人架势。火花效果变大说明成功了 / Press L1 at the moment of impact — no HP loss + big posture damage. Bigger sparks = success |
| **防反后反击 / Deflect Counter** | 成功后你可以立刻反击——打乱敌人节奏 / After a perfect deflect, you can instantly counter-attack — disrupts enemy rhythm |
| **敌人的忍杀 / Enemy Deathblow** | 你的架势崩了=被破防——长时间硬直。非常危险 / Your posture breaks = staggered for a long time. Very dangerous |

> **核心原则**：不要被动防守——保持进攻。攻击压制敌人的架势恢复。防反配合攻击打崩架势 / **Core Principle**: Don't be passive — stay aggressive. Attacks prevent enemy posture recovery. Deflect + attack to break posture

## 血量与忍杀 / Health & Deathblow System

| 机制 / Mechanic | 说明 / Description |
|------|------|
| **血量节点 / Health Nodes** | 敌人头上的黄色菱形——每个节点 = 一条命。需要忍杀一次消耗一个节点 / Yellow diamonds above enemy HP bar — each = one life. One deathblow removes one |
| **Boss多条命 / Multi-Life Bosses** | 大部分Boss有2-3条命——需要多次打崩架势+忍杀 / Most bosses have 2-3 health nodes — need multiple posture breaks + deathblows |
| **架势与血量关系 / HP-Posture Relation** | 敌人血量越低，架势恢复越慢。优先削血再打架势是有效策略 / Lower enemy HP = slower posture recovery. Chip HP first, then break posture |
| **处决伤害 / Execute Damage** | 忍杀造成固定伤害——不管敌人剩余血量多少直接移除一条命 / Deathblow deals fixed damage — removes one node regardless of remaining HP |

## 复活机制 / Resurrection System

| 机制 / Mechanic | 说明 / Description |
|------|------|
| **复活之力 / Resurrection Power** | 初始有1次复活机会。击杀敌人积累复活之力——画面左侧的红色圆环 / Start with 1 resurrection charge. Kill enemies to accumulate — red ring on left side of screen |
| **立即复活 / Immediate Resurrect** | 死亡后按R1/R2选择是否复活——复活后敌人注意力重置 / Press button after death to resurrect — enemy aggro resets |
| **复活充能 / Recharge** | 坐鬼佛完全恢复复活次数。战斗中击杀多个敌人也可以累积一次 / Rest at idol for full recharge. Killing multiple enemies in combat can also build one charge |
| **不可复活 / Unavailable** | 如果你在短时间内连续死亡两次——会出现"不可复活"——不能再原地复活 / Die twice in quick succession = "Unavailable" — can no longer resurrect in that fight |
| **龙咳 / Dragonrot** | 频繁死亡导致NPC"龙咳"——降低NPC相关支线的运气。可用龙胤露滴治疗 / Frequent deaths cause NPCs to catch "Dragonrot" — reduces luck on NPC questlines. Cure with Dragon's Blood Droplet |

> **策略**：每场战斗有两次复活机会（第一次免费，第二次需要击杀充能）。不要浪费第一次——如果局势不利，可以等敌人松懈后复活 / **Strategy**: Two resurrections per fight. Don't waste the first one — if outmatched, wait for enemies to relax before resurrecting

## 危攻击应对 / Perilous Attack Counters

"危"字出现表示敌人使用无法格挡的特殊攻击——有三种类型，各有应对方法 / "危" (Perilous) indicates an unblockable special attack — three types with different counters.

### 突刺危 / Thrust Perilous

| 特征 / Tell | 应对 / Counter |
|------|------|
| 刀收在身体一侧准备突刺 / Blade pulled back for a lunge | **识破（向前闪避）** / **Mikiri Counter (dodge forward)** |
| 常见于：剑圣、武士类敌人 / Common on: sword saints, samurai | 识破成功造成巨额架势伤害 / Successful mikiri deals massive posture damage |

### 横扫危 / Sweep Perilous

| 特征 / Tell | 应对 / Counter |
|------|------|
| 刀横向大挥扫下肢 / Wide low sweep | **跳跃踩头** / **Jump on enemy's head** |
| 常见于：大型野兽、持长柄武器敌人 / Common on: large beasts, polearm enemies | 跳跃后可以再踩一次——持续造成架势伤害 / Jump again off their head for extra posture damage |

### 抓取危 / Grab Perilous

| 特征 / Tell | 应对 / Counter |
|------|------|
| 双手张开冲过来 / Arms open, charging | **向后跳或向侧面闪避** / **Backdash or dodge sideways** |
| 常见于：怨鬼、大型怪物 / Common on: Demon of Hatred, large monsters | 不能格挡不能识破——只能靠走位躲避 / Can't deflect or mikiri — pure positioning |

> **训练**：在破旧寺院的不死半兵卫处可以无限练习识破和格挡 / **Training**: Practice mikiri and deflect infinitely at the Undying Soldier in Dilapidated Temple

## 新手常见问题 / FAQ

| 问题 / Question | 回答 / Answer |
|------|------|
| **为什么翻滚躲不开攻击？/ Rolling doesn't dodge?** | 只狼的翻滚**没有无敌帧**——不要试图用翻滚躲攻击。用**格挡(L1)**替代 / Rolls have **no i-frames** — don't use them to dodge. Use **deflect (L1)** instead |
| **蝴蝶夫人打不过？/ Lady Butterfly too hard?** | 不要逃跑——压着她打。她跳上钢丝后用手里剑打下来 / Don't run — stay aggressive. Use shuriken to knock her off wires |
| **纸人不够用？/ Out of spirit emblems?** | 初期尽量节省纸人——主要靠刀战。钱多时在商人处买纸人 / Conserve emblems early — rely on your sword. Buy from merchants when you have gold |
| **鬼佛是什么？/ What are Sculptor's Idols?** | 只狼的篝火——存档+传送+升级。坐鬼佛重置所有敌人 / Sekiro's bonfires — save, warp, upgrade. Sitting respawns all enemies |
| **金币掉了怎么办？/ Lost gold?** | 死亡掉一半金币。击杀敌人或坐鬼佛后，掉落的金币自动捡回 / Die and lose half. Gold is auto-recovered on kill or sitting at idol |
