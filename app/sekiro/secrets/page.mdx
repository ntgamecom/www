# 隐藏要素与终局 / Secrets & Endgame

只狼虽不是开放世界——但隐藏内容极其丰富，终局挑战深刻考验你的操作 / Sekiro may not be open world — but its hidden content is extremely rich, and the endgame challenges truly test your skills.

|---|---|

## 怨恨之鬼 / Demon of Hatred

| 方面 / Aspect | 说明 / Description |
|---|---|
| **背景 / Lore** | 佛雕师化身——曾经的忍义手忍者怨恨之火爆发 / The Sculptor's transformation — his hatred erupts |
| **位置 / Location** | 苇名城城外——最终战时 / Ashina outskirts — during final invasion |
| **推荐 / Recommended** | 击败后获得"爱哭鬼"强化材料 / Gives Malcontent upgrade material |
| **阶段 / Phases** | 3个阶段——阶段3有火圈AOE / 3 phases — phase 3 has fire ring AOE |

| 阶段 / Phase | 策略 / Strategy |
|---|---|
| **1** | 近战砍腿——注意踩地和抓取 / Hit legs — watch for stomp and grab |
| **2** | 火手挥击——跳起来躲火焰AOE / Fire hand sweep — jump to avoid fire AOE |
| **3** | 火圈+连击——用爱哭鬼戒指震慑 / Fire ring + combos — use Malcontent to stun |

## 多周目挑战 / NG+ Challenges

| 难度 / Difficulty | 说明 / Description |
|---|---|
| **普通新周目 / Normal NG+** | 继承所有技能+忍义手——敌人变强 / Inherit all — enemies stronger |
| **钟鬼模式 / Demon Bell Mode** | 敌人攻击+200%，掉落率+30% / Enemy attack +200%, drop rate +30% |
| **交符模式 / Kuro's Charm Challenge** | 不完美格挡会扣血——极高难度 / Imperfect blocking costs HP — extreme difficulty |
| **钟鬼+交符 / Both Active** | 最高难度——几乎任何失误都致命 / Maximum difficulty — almost any mistake is fatal |

### 回报 / Rewards

| 挑战 / Challenge | 奖励 / Reward |
|---|---|
| **钟鬼 / Demon Bell** | 更好的掉落——更多材料+纸人 / Better drops — more materials + emblems |
| **交符 / Kuro's Charm** | 攻击力增加——以及成就感 / Attack power increase — and bragging rights |
| **双开 / Both** | 全成就+真·通关 / Full achievements + true completion |
