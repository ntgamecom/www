# 任务与NPC / Quests & NPCs

只狼的NPC支线数量不如魂系列多——但每个NPC都有深刻的故事和重要的影响 / Sekiro has fewer NPC quests than the souls series — but each character has a deep story and significant impact.

---

## 主要NPC / Major NPCs

### 佛雕师 / The Sculptor

| 信息 / Info | 说明 / Description |
|---|---|
| **位置 / Location** | 破旧寺院 / Dilapidated Temple |
| **功能 / Function** | 强化忍义手 / Upgrades prosthetic tools |
| **所需材料 / Materials** | 铁屑、磁铁屑、蜡、金钱 / Scrap iron, magnetite, wax, gold |
| **背景 / Lore** | 曾经的忍义手忍者——被称为"猩猩" / Former shinobi with prosthetic — called "Orangutan" |
| **结局 / Fate** | 怨鬼 = 佛雕师化身为怨恨之鬼 / Demon of Hatred = Sculptor transformed |

### 变若之子 / The Divine Heir (九郎 / Kuro)

| 信息 / Info | 说明 / Description |
|---|---|
| **位置 / Location** | 苇名城主城——天守阁 / Ashina Castle — top floor |
| **关系 / Relationship** | 狼的主人——你需要保护他 / Wolf's master — you must protect him |

| **结局影响 / Ending Impact** | 决定你能达成的结局——三个结局中有两个取决于他 / Determines possible endings — 2 of 3 endings depend on him |

### 苇名一心 / Isshin Ashina

## 结局路线 / Ending Routes

只狼有三个主要结局 / Sekiro has three main endings.

### 1. 断绝不死 / Immortal Severance (标准结局)

| 步骤 / Step | 说明 / Description |
|---|---|
| **1** | 收集所有必需品——不死斩 + 馨香水莲 + 结宿之石 / Gather all key items — Mortal Blade + Lotus + Stone |
| **2** | 击败苇名弦一郎 + 剑圣苇名一心 / Defeat Genichiro + Sword Saint Isshin |
| **3** | 使用不死斩终结九郎的不死诅咒 / Use Mortal Blade to sever Kuro's immortality |
| **结果 / Result** | 狼失去忍义手——佛雕师变回普通人 / Wolf loses prosthetic — Sculptor becomes human |

### 2. 归复常人 / Purification (真结局)

| 步骤 / Step | 说明 / Description |
|---|---|
| **1** | 从永真处获得"义父的守护铃" / Get "Father's Bell Charm" from Emma |
| **2** | 回到平田宅邸——击败义父/蝴蝶夫人幻影 / Return to Hirata — defeat Owl/Lady Butterfly phantom |
| **3** | 获得"常樱之花" / Get "Everblossom" |
| **4** | 在最终战前给九郎常樱之花 / Give Everblossom to Kuro before final fight |
| **5** | 使用不死斩牺牲自己——让九郎恢复凡人 / Use Mortal Blade to sacrifice self — Kuro becomes mortal |
| **结果 / Result** | 狼死亡——九郎活下来成为凡人 / Wolf dies — Kuro lives as mortal |

### 3. 龙之还乡 / Return (隐藏结局)

| 步骤 / Step | 说明 / Description |
|---|---|
| **1** | 在仙峰寺遇到变若之子 / Meet Divine Child of Rejuvenation at Senpou |
| **2** | 给她"干柿子"和"鲜柿子" / Give her "Dried Fruit" and "Fresh Fruit" |
| **3** | 她变成龙胤容器 / She becomes a Dragon's Heritage vessel |
| **4** | 收集"冰泪"——在龙泉河边冻雨区域 / Get "Frozen Tears" — in Fountainhead frozen rain area |
| **5** | 最终战前给九郎冰泪 / Give Frozen Tears to Kuro before final fight |
| **6** | 变若之子带九郎前往西方——归还龙胤 / Divine Child takes Kuro to the West — return the Dragon's Heritage |
| **结果 / Result** | 狼存活——九郎和变若之子前往西方 / Wolf survives — Kuro and Divine Child go West |

## 联机与多周目 / NG+ & Features

| 周目 / NG Cycle | 难度 / Difficulty | 说明 / Description |
|---|---|---|
| **新周目 / NG+** | +30% | 继承所有技能和忍义手——但难度提升 / Inherit all skills and tools — increased difficulty |
| **多周目 / NG+2+** | 递增 / Increasing | 敌人血量+伤害继续提升 / Enemy HP + damage scale up |
