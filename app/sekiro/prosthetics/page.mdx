# 忍义手与技能 / Prosthetics & Skills

只狼的成长体现在三个维度：忍义手升级、流派技能学习、被动技能解锁。本章详解全部内容 / Sekiro's progression has three dimensions: prosthetic upgrades, combat art schools, and latent skills. This chapter covers everything.

---

## 忍义手工具 / Prosthetic Tools

忍义手是只狼的核心装备——通过破旧寺院的佛雕师升级。所有工具消耗纸人 / Prosthetic tools are core equipment — upgraded by the Sculptor at the Dilapidated Temple. All tools consume spirit emblems.

| 等级 / Level | 效果 / Effect | 材料 / Materials | 推荐度 / Rating |
|------|------|------|:----:|
| **手里剑 / Shuriken** | 基础远程攻击——打断敌人攻击 / Basic ranged attack — interrupts enemy attacks | 初始获得 / Starting tool | ★★★★★ |
| **陀螺手里剑 / Charged Shuriken** | 蓄力后穿透多个敌人 / Charge to pierce multiple enemies | 铁屑x50 + 铜屑x20 | ★★★☆☆ |

## 技能升级顺序推荐 / Recommended Skill Order

| 阶段 / Stage | 优先学习的技能 / Priority Skills | 理由 / Why |
|------|------|------|
| **游戏前期 / Early** | 识破 → 空中格挡 → 忍者之眼 → 流水 → 空中忍杀 | 战斗核心技能——先搞这些 / Core combat skills — get these first |
| **游戏中期 / Mid** | 一文字二连 → 不死斩 → 最终反击 → 看破 | 强力战斗技+保命 / Strong combat arts + safety net |
| **游戏后期 / Late** | 绝技·不死斩 → 登鲤/下鲤 → 体力增强 → 纸人增加 | 封顶输出 / Max out damage |

> **必拿技能**: 识破、空中格挡、一文字二连、不死斩、最终反击——没有这些游戏难度翻倍 / **Must-have skills**: Mikiri, Mid-air Deflect, Ichimonji Double, Mortal Draw, Last Chance — without these, the game is 2x harder
