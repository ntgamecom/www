# 物品收集 / Items & Collectibles

只狼的收集品主要集中在佛珠、葫芦种子和义手升级材料上——全部找齐是满血量和满义手的必要条件 / Sekiro's collectibles focus on prayer beads, gourd seeds, and prosthetic upgrade materials — finding all is essential for max HP and full prosthetics.

### 琉璃获取指南 / Lapis Lazuli Guide

每周目共可以获得**6颗琉璃**（需解锁所有）+ 二周目重置 / 6 per NG cycle + NG+ reset:

| # | 获取方式 / Method | 说明 / Details |
|:----:|------|------|
| 1 | 源之宫——水下宝箱 / Underwater chest | 潜入宫殿前的湖底 / Dive in lake before palace |
| 2 | 源之宫——宝鲤鳞片兑换 / Carp Scale exchange | 用7片宝鲤鳞片跟坛子贵人换 / Exchange 7 carp scales with Pot Noble |
| 3 | 源之宫——宝鲤鳞片兑换(第二次) / Second exchange | 再用7片鳞片换 / Another 7 scales |
| 4 | 源之宫——击败破戒僧奖励 / Corrupted Monk reward | 击败源之宫的破戒僧 / Defeat Fountainhead Corrupted Monk |
| 5 | 二周目+击败源之宫Boss / NG+ boss drop | BOSS轮换掉落 / Boss rotation drops |
| 6 | 二周目+坛子贵人 / NG+ Pot Noble | 鳞片兑换重置 / Scale exchange resets |

> **建议**：优先升级对游戏体验提升最大的义手——琉璃手里剑、琉璃伞、附牙斩 / **Tip**: Prioritize upgrades that impact gameplay most — Lazulite Shuriken, Suzaku Umbrella, Flame Sword

## 降灵 / Sugars & Spirits

降灵提供短暂的增益效果——分为"糖"（持续时间中等）和"降灵"（持续时间短+额外效果）/ Sugars and spirits provide temporary buffs — "sugars" (medium duration) and "spirits" (shorter duration + bonus effect).

| 降灵 / Item | 效果 / Effect | 获取 / Location |
|------|------|------|
| **月隐糖 / Gachiin's Sugar** | 大幅降低被侦测——潜行流必备 / Greatly reduce detection — stealth essential | 商人购买、地图拾取 / Buy, pickups |
| **月隐降灵 / Gachiin's Spirit** | 同月隐糖+纸人消耗降为0 / Same + 0 emblem consumption | 隐藏森林——无首掉落 / Hidden Forest — Headless drop |
| **刚幹糖 / Ungo's Sugar** | 大幅提升物理减伤——生存流 / Greatly increase physical defense — survival | 商人购买 / Buy |
| **刚幹降灵 / Ungo's Spirit** | 同刚幹糖+纸人消耗降为0 / Same + 0 emblem consumption | 苇名城地下——无首掉落 / Ashina underground — Headless drop |
| **夜叉糖 / Yashariku's Sugar** | 大幅提升攻击力+大幅降低减伤——高回报高风险 / Greatly increase attack + reduce defense — glass cannon | 商人购买 / Buy |
| **夜叉降灵 / Yashariku's Spirit** | 同夜叉糖+纸人消耗降为0 / Same + 0 emblem consumption | 苇名之底——无首掉落 / Ashina Depths — Headless drop |
| **哈将糖 / Ako's Sugar** | 提升攻击力——通用进攻buff / Increase attack — general offensive buff | 全地图拾取 / Map pickups |
| **哈将降灵 / Ako's Spirit** | 同哈将糖+纸人消耗降为0 / Same + 0 emblem consumption | 平田宅邸——无首掉落 / Hirata Estate — Headless drop |

> **建议**：打Boss前吃哈将糖或夜叉糖（取决于你对自己的信心）。潜行时吃月隐糖 / **Tip**: Ako's for general boss fights, Yashariku's for confidence, Gachiin's for stealth
