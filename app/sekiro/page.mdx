# 只狼攻略 / Sekiro: Shadows Die Twice Guide

只狼（Sekiro: Shadows Die Twice）是FromSoftware开发的日本战国题材动作冒险游戏。与魂系不同，只狼强调精准格挡、架势系统和潜行——你需要扮演忍者"狼"，救回主人并复生复仇 / Sekiro is an action-adventure game by FromSoftware set in Sengoku-era Japan. Unlike souls games, Sekiro emphasizes precise parrying, posture mechanics, and stealth — you play as the shinobi "Wolf," rescuing your lord and seeking revenge.

## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **架势系统 / Posture** | 核心战斗机制——打崩敌人架势可发动忍杀。L1精准格挡比闪避更重要 / Core combat — break enemy posture for a deathblow. L1 parry > dodge |
| **复活机制 / Resurrection** | 死亡后可消耗复活之力原地复活——每次复活需要击杀敌人充能 / Die and resurrect on the spot — recharge by killing enemies |
| **钩绳 / Grappling Hook** | 自由飞檐走壁——大幅改变探索方式 / Free vertical mobility — transforms exploration |
| **忍义手 / Prosthetic Tools** | 各种特殊工具——手里剑、斧头、鞭炮、火筒等——可升级 / Special tools — shuriken, axe, firecrackers, flame vent — upgradeable |
| **潜行暗杀 / Stealth** | 从背后/空中暗杀秒杀小兵——清图利器 / Backstab/air assassinate for instant kills — essential for area clear |
| **战斗技 / Combat Arts** | 可装备的特殊攻击招式——消耗纸人 / Special attack skills — consumes spirit emblems |
| **复活之力 / Resurrection Power** | 需要击杀敌人积累——死亡时消耗复活。全游戏有两条命机制 / Accumulate by killing — consume on death. Essentially two lives |

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **学会格挡，不要闪避 / Master Parry, Not Dodge** | 只狼的翻滚没有无敌帧——防反（L1精准格挡）是核心。放弃魂系的翻滚习惯 / Rolling has no i-frames — deflecting (L1 timing) is core. Unlearn souls dodge habits |
| **2** | **打崩架势再忍杀 / Break Posture for Deathblow** | 大部分敌人需要打崩架势条才能忍杀。持续攻击+精准格挡是破敌关键 / Most enemies need posture broken for deathblow. Aggression + parry = victory |
| **3** | **重视潜行 / Prioritize Stealth** | 先用钩绳绕后暗杀清掉大部分小兵——不要正面硬打一群 / Use grappling to stealth-clear most enemies — never fight groups head-on |
| **4** | **攒钱买袋 / Save Money for Pouches** | 金币在你死后掉一半。商人卖"钱袋"——买了后死亡不会掉（钱袋里的钱安全）/ Gold drops on death. Buy "purses" from merchants — purse gold is safe |
| **5** | **探索每个角落 / Explore Everywhere** | 用钩绳搜刮屋顶、悬崖——佛珠、葫芦种子和义手材料都在隐蔽位置 / Grapple to every rooftop and cliff — prayer beads, seeds, and upgrade mats are hidden |
| **6** | **不要跳过精英 / Don't Skip Mini-Bosses** | 精英敌人掉落佛珠（4颗增加HP上限）——杀光每个精英 / Mini-bosses drop prayer beads (4 = +HP) — kill every one you find |
| **7** | **鬼佛优先 / Prioritize Sculptor's Idols** | 鬼佛是存档点——比魂系篝火复活更慢。记得多坐鬼佛 / Idols are checkpoints — slower respawn than souls bonfires. Rest often |
| **8** | **纸人管理 / Manage Spirit Emblems** | 忍义手和战斗技消耗纸人——上限20。可以用钱买，路上捡。不够用时依赖纯刀战 / Prosthetics and arts consume spirit emblems (max 20). Buy or find them. Save when low |
