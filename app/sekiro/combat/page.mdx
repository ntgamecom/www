# 战斗系统 / Combat System

只狼的战斗系统与所有魂系截然不同——核心是格挡而非翻滚，节奏极快 / Sekiro's combat is fundamentally different from all souls games — the core is deflecting, not rolling, with a lightning-fast rhythm.

---

## 格挡系统详解 / Deflection System

### 格挡与完美格挡 / Block vs Perfect Deflect

| 动作 / Action | 操作 / Input | 效果 / Effect |
|---|---|---|
| **格挡 / Block** | 按住L1 / Hold L1 | 减伤但扣HP和架势——不推荐长期使用 / Reduces damage but costs HP and posture — not sustainable |
| **完美格挡 / Perfect Deflect** | 攻击命中瞬间按L1 / Press L1 at impact | 不扣HP + 大幅增加敌人架势——火花变大=成功 / No HP loss + massive posture damage — larger sparks = success |
| **空中格挡 / Air Deflect** | 空中按L1 / L1 in air | 可在空中格挡——对某些攻击有效 / Can deflect while airborne |

### 架势机制 / Posture System

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **架势条 / Posture Bar** | 屏幕上方——你的左侧/敌人右侧。满格=崩溃 / Top of screen — yours left, enemy right. Full = broken |
| **架势恢复 / Posture Recovery** | 不进攻时自动恢复。血量越低恢复越慢 / Auto-recovers when idle. Lower HP = slower recovery |
| **攻击压制 / Attack Pressure** | 持续攻击敌人会阻止其架势恢复——保持进攻 / Continuous attacks prevent enemy posture recovery — stay aggressive |
| **格挡扣架势 / Blocking Costs** | 普通格挡会扣你的架势——完美格挡不扣 / Normal block costs your posture — perfect deflect does not |

## 危攻击应对 / Perilous Attack Counters

| 危类型 / Type | 标识 / Indicator | 应对方法 / Counter |
|---|---|---|
| **突刺危 / Thrust** | 敌人收刀做突刺动作 / Blade pulled back for thrust | **识破**——向前按○/Space / **Mikiri** — forward dodge |
| **横扫危 / Sweep** | 横向大范围横扫 / Wide horizontal swing | **跳跃踩头**——跳跃后再次按跳 / **Jump on head** — jump then jump again |
| **抓取危 / Grab** | 双手张开冲过来 / Arms open charging | **向后闪避或侧闪** / **Backdash or side dodge** |

### 识破技巧 / Mikiri Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **时机 / Timing** | 在突刺即将命中你的瞬间按○——不要太早 / Press dodge right before thrust connects — not too early |
| **方向 / Direction** | 默认向后识破——但向前识破更安全且增加架势伤害 / Default is neutral — forward mikiri is safer and deals more posture damage |
| **训练 / Practice** | 在破旧寺院的不死半兵卫处无限练习 / Practice infinitely at Dilapidated Temple's undying soldier |

## 忍杀系统 / Deathblow System

### 血量节点 / Health Nodes

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **黄色菱形 / Yellow Diamonds** | 敌人血条上方的黄色菱形 = 需要忍杀的次数 / Diamonds above HP = deathblows needed |
| **普通小兵 / Normal Soldiers** | 1条命——可直接暗杀 / 1 node — can stealth kill |
| **精英怪 / Elites** | 1-2条命——通常2次忍杀 / 1-2 nodes — usually 2 deathblows |
| **Boss** | 2-3条命——最少2次最多3次忍杀 / 2-3 nodes — minimum 2, maximum 3 deathblows |
| **架势崩溃 / Posture Break** | 打崩架势后接近敌人按R1/△处决 / On break, approach + R1/△ for deathblow |

### 暗杀 / Stealth Deathblow

| 场景 / Scenario | 操作 / Action |
|---|---|
| **背后暗杀 / Backstab** | 从敌人背后按住攻击——对小兵和部分精英有效 / Hold attack from behind — works on soldiers and some elites |
| **空中暗杀 / Air Assassinate** | 从高处跳下暗杀——需要忍杀之技/空中暗杀技能 / Drop from height — needs skill |
| **草丛暗杀 / Grass Assassinate** | 在草丛中潜伏后暗杀 / Ambush from grass |
| **钩绳暗杀 / Grapple Assassinate** | 钩到敌人附近后空中暗杀——强推技能 / Grapple then air assassinate — recommended skill |

## Boss战斗策略 / Boss Combat

### 通用原则 / General Principles

| 原则 / Principle | 说明 / Description |
|---|---|
| **进攻是最好的防守 / Aggression Rules** | 持续攻击压制敌人架势恢复——直到敌人完美格挡你 / Keep attacking to suppress posture recovery — until enemy perfect deflects you |
| **弹反节奏 / Deflect Rhythm** | 每个敌人有固定的攻击节奏——记住以后可以无缝弹反 / Every enemy has fixed attack rhythm — learn for flawless deflection |
| **血量优先 / HP First** | 削血量可以降低敌人架势恢复速度——先削血再打架势 / Chip HP to slow posture recovery — HP first, posture second |
| **纸人管理 / Emblem Management** | 忍义手需要纸人——不要过度依赖。普通刀战是核心 / Prosthetics cost emblems — don't over-rely. Sword is primary |

### 各Boss核心策略 / Key Boss Strategies

| Boss | 核心策略 / Key Strategy |
|---|---|
| **鬼庭形部雅孝 / Gyoubu Oniwa** | 用鞭炮惊吓他的马——创造大攻击窗口 / Firecracker to scare horse — creates huge opening |
| **蝴蝶夫人 / Lady Butterfly** | 压制到死——她跳钢丝用手里剑打下来 / Pressure nonstop — shuriken when she jumps on wires |
| **苇名弦一郎 / Genichiro Ashina** | 弹反他的弓箭——突进危用识破 / Deflect his arrows — mikiri his thrusts |
| **狮子猿 / Guardian Ape** | 第一段死斗——第二段用长枪拉虫，弹反刀攻击 / Phase 1 deathmatch — Phase 2 spear pull, deflect sword |
| **怨鬼 / Demon of Hatred** | 用爱哭鬼戒指——跳起来躲火攻击 / Use Malcontent ring — jump to avoid fire attacks |
| **剑圣苇名一心 / Sword Saint Isshin** | 弦一郎阶段压着他打——一心阶段完美弹反 + 识破 / Aggressive on Genichiro — perfect deflect + mikiri on Isshin |

## 战斗常见问题 / Combat FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **为什么弹反总失败？/ Can't deflect?** | 注意不是按住L1——是在攻击命中瞬间按一下 / Don't hold L1 — tap it right as attack connects |
| **识破怎么按？/ How to mikiri?** | 看到"危"字+突刺动作——向前按○/Space / See "危" + thrust — press forward dodge |
| **纸人不够用？/ Out of emblems?** | 找商人买——或刷苇名城的武士 / Buy from merchants — or farm Ashina soldiers |
| **龙咳怎么办？/ Dragonrot?** | 用龙胤露滴治愈——不影响主线进度 / Cure with Dragon's Blood Droplet — doesn't affect story |
| **不死半兵卫怎么无限练？/ Infinite practice?** | 在破旧寺院与他对话选择"挑战"——死了不会真死 / Talk at Dilapidated Temple — choose "challenge" — no real death |
| **给精英怪上了两管血怎么办？/ Two health nodes?** | 需要两次忍杀——打崩架势两次。暗杀可以秒一条命 / Need 2 deathblows — break posture twice. Stealth kills one node instantly |

## 实战技巧 / Advanced Combat Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **拼刀节奏 / Blade Rhythm** | 每个Boss的攻击有固定节奏——先听节奏再按格挡。熟悉后你可以闭着眼睛弹反 / Every boss has a fixed rhythm — listen first, then deflect. Once familiar, you can deflect blind |
| **垫步绕后 / Step Dodge** | 虽然无无敌帧——但垫步可以快速绕到敌人背后。配合冲刺攻击使用 / No i-frames — but can quickly circle behind. Use with sprint attack |
| **跳跃踩头追击 / Jump Stomp Chase** | 跳跃踩头后——落地前可以再接一次攻击。对下段危用完后可以继续连击 / After jump stomp — attack before landing. Follow up sweep perilous counter with combo |
| **手里剑追斩 / Shuriken Chase** | 手里剑命中后按突进——可以快速接近被击中的敌人。对空中敌人效果更好 / Shuriken hit + rush — fast gap close. Better vs airborne enemies |
| **鞭炮+追击 / Firecracker Chase** | 鞭炮吓到敌人后——立刻突进攻击。对野兽类敌人效果尤其好 / Firecracker stuns — immediately rush attack. Especially good vs beasts |
| **雨天火焰 / Rain Fire** | 雨天使用大炮/火焰类忍义手——伤害降低。但雷电类伤害提升 / Rain reduces fire tool damage — but lightning damage increases |
| **背后识别 / Backstab Awareness** | 未发现你的敌人——背后有蓝色光圈提示。可以暗杀 / Unaware enemies have blue aura behind them — can stealth kill |
| **雨天战斗 / Rain Combat** | 雨天会影响火焰忍义手的伤害——但雷电伤害增加。注意环境对战斗的影响 / Rain reduces fire tool damage but increases lightning damage. Environment matters |
| **精英怪暗杀 / Elite Stealth** | 大部分精英怪可以暗杀一条命——只剩一条血。利用这点降低难度 / Most elites can be stealth-killed for one health node — use this to reduce difficulty |

## 精英怪应对 / Elite Enemy Tactics

| 精英怪 / Elite | 策略 / Strategy | 忍义手推荐 / Tool |
|---|---|---|
| **武士大将 / Samurai General** | 先暗杀清掉周围小兵——然后面对面对决。注意他的居合斩 / Stealth clear mobs first — then duel. Watch for Iai slash | 手里剑（打断）/ Shuriken (interrupt) |
| **蛇眼白藤/白秋 / Snake Eyes** | 枪兵——距离远但攻击慢。格挡她的连击后反击 / Spear user — long range but slow. Deflect combo then punish | 机关伞（防子弹）/ Umbrella (bullet block) |
| **七本枪 / Seven Ashina Spears** | 识破他的突刺危是关键——不识字破极难打 / Mikiri his thrust is key — near impossible without it | 长枪（拉近）/ Spear (pull close) |
| **孤影众 / Lone Shadow** | 快速连击型——注意他的踢击连段。弹反后立刻反击 / Fast combo type — watch for kick combos. Deflect then retaliate | 鞭炮（制造空隙）/ Firecracker (create opening) |
| **苇名流 / Ashina Elite** | 极速居合斩——反应要快。用机关伞格挡或预判弹反 / Lightning fast Iai — need quick reflexes. Use umbrella or predict deflect | 机关伞（防居合）/ Umbrella (block Iai) |
