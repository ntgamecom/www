# BOSS攻略 / Boss Strategies

只狼共有约40个Boss和精英敌人——本章按类型和难度排列所有关键Boss的详细打法 / Sekiro has ~40 bosses and mini-bosses — this chapter covers all critical boss strategies organized by type and difficulty.

---

## 主线Boss / Story Bosses

### 鬼庭形部雅孝 / Gyoubu Oniwa

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 苇名城城外——虎口阶梯 / Ashina Outskirts — Staircase |
| **命数 / Health Nodes** | 1条 / 1 node |
| **弱点 / Weakness** | 钩绳攻击 / Grappling hook attack |

**打法:**
- 骑马Boss——机动性高。他用大范围横扫攻击 / Mounted boss — highly mobile. Wide sweeping attacks
- **核心策略**: 他跑远后按R2用钩绳飞到身边——空中攻击一刀然后落地继续输出 / **Core strategy**: When he rides far, press R2 to grapple to him — mid-air attack then continue on ground
- 不要试图追他跑——等他跑开后用钩绳 / Don't chase him — let him run and then grapple
- 他的攻击节奏清晰——连续格挡后反击。他起身举枪时是突刺危——识破 / Clear attack rhythm — deflect chain then counter. Spear wind-up = thrust perilous — mikiri
- 掉落：**机关伞**（忍义手）/ Drops: **Loaded Umbrella** (prosthetic)

### 蝴蝶夫人 / Lady Butterfly

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 平田宅邸 / Hirata Estate |
| **命数 / Health Nodes** | 2条 / 2 nodes |
| **弱点 / Weakness** | 手里剑（打断钢丝）/ Shuriken (wire interrupt) |

**打法:**
- **第一阶段**: 她使用踢击和幻术刀——节奏快但硬直短。保持贴身猛攻——你的攻击力压制她的架势恢复 / Phase 1: Kicks + illusion sword — fast but short windows. Stay aggressive — attacks prevent her posture recovery
- 她跳上钢丝时——立刻用**手里剑**打下来（造成坠落伤害+倒地）/ When she jumps on wires — use **Shuriken** to knock her down (fall damage + stun)
- 她的连击最后一下通常有较大硬直——完美格挡后反击 / Her combo finisher has a longer recovery — perfect deflect then counter
- **第二阶段**: 增加幻影小兵——不要慌，跑动躲避约20秒后幻影消失。也可以使用**鸣种**（Snap Seed）立即清除幻影 / Phase 2: Adds phantom minions — run and dodge for ~20 seconds until they vanish. Or use **Snap Seed** to instantly clear them
- 蝴蝶夫人的幻影消失后会从空中落下一——锁定她后用手里剑打下来接攻击 / After phantoms fade, she descends from above — lock on, shuriken, then attack
- 掉落：**樱花枝**（关键道具）/ Drops: **Sakura Branch** (key item)

### 苇名弦一郎 / Genichiro Ashina

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 苇名城顶层 / Ashina Castle top |
| **命数 / Health Nodes** | 3条 / 3 nodes |
| **弱点 / Weakness** | 雷反（第二阶段+）/ Lightning reversal (Phase 2+) |

**打法:**
- **R1**: 弓+刀模式——他的弓箭伤害高但射击前摇长。近战时注意他的**突刺危**——用识破 / Phase 1: Bow + sword — arrows hit hard but are telegraphed. Watch for **thrust perilous** — mikiri counter
- 他的经典连招：2刀→停顿→3刀突刺。节奏是"挡-挡-停-挡-挡-挡-识破" / Classic combo: 2 hits → pause → 3 hits + thrust. Rhythm: "deflect-deflect-pause-deflect-deflect-deflect-mikiri"
- **P2**: 弦一郎增加雷攻——跳起+刀插雷云→雷击。**接雷反**: 他跳起时你也跳→接住雷（空中会被电）→落地前按**攻击**把雷打回去 / Phase 2: Adds lightning — he jumps + katana catches lightning → strikes. **Lightning reversal**: jump when he jumps → catch lightning (you get stunned) → press **attack** before landing to throw it back
- **P3**: 同第二阶段——但他更频繁使用雷反。用好雷反可以3-4次解决P3 / Phase 3: Same as P2 but more lightning. 3-4 lightning reversals end the phase
- 掉落：**不死斩**（关键道具）/ Drops: **Mortal Blade** (key item)

### 狮子猿 / Guardian Ape

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 坠落之谷——狮子猿巢穴 / Sunken Valley — Ape Nest |
| **命数 / Health Nodes** | 2条 / 2 nodes |
| **弱点 / Weakness** | 火、完美格挡他的大招 / Fire, perfect deflect his big swing |

**打法:**
- **P1（野兽形态）**: 疯狂爪击连续攻击——格挡！他的招式虽然兇猛但前摇明显。主要注意： / Phase 1 (Beast form): Wild claw swipes — deflect! Fierce but telegraphed.
  - **丢屎**: 他弯腰捡东西→丢出一团——向前跑躲开然后攻击 / **Poop throw**: He bends to pick up → throws — sprint forward under it and attack
  - **滚地**: 他躺下滚来滚去——跳起来躲避 / **Roll**: He flops and rolls — jump to dodge
  - **连打**: 长时间乱拍——连续格挡7-8下 / **Combo**: Extended flurry — chain deflect 7-8 hits
- **P2（无头持剑）**: 狮子猿被忍杀后复活——拔出自己的剑 + 用脖子当刀鞘 / Phase 2 (Headless sword): Revives — draws its own sword
  - 攻击节奏完全变了——变得更有规律。他的大范围横斩接竖劈——完美格挡后倒地 / Attack rhythm changes completely — more choreographed. Wide horizontal → vertical — perfect deflect = long stun
  - **危: 大喊+下砸**: 他后退吸一口气——大喊（可以格挡）→然后跳起重砸地面——**完美格挡这招**让他长时间硬直 / **Perilous: Roar + slam**: Backs up and inhales → roar (blockable) → jumps for ground slam — **perfect deflect this** for a huge punish window
  - **危: 横扫**: 他用剑横扫——**跳跃踩头** / **Perilous sweep**: Horizontal sword sweep — **jump on head**
  - 用**长矛忍义手**可以勾出他脖子里的蜈蚣——增加架势伤害 / Use **Spear prosthetic** to pull the centipede from its neck — massive posture damage
- 掉落：**爱哭鬼的指环** / Drops: **Malcontent's Ring**

### 破戒僧 / Corrupted Monk (源之宫 / Fountainhead Palace)

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 源之宫——桥 / Fountainhead Palace — Bridge |
| **命数 / Health Nodes** | 3条 / 3 nodes |
| **弱点 / Weakness** | 鞭炮、飞雪 / Firecrackers, Divine Confetti |

**打法:**
- **战斗的马拉松**——需要耐心打崩三次架势 / A marathon — need to break posture three times
- 她的攻击范围极大但节奏固定——多练习可以完美格挡所有招式 / Wide attacks but fixed rhythm — practice for perfect deflects on all moves
- **鞭炮打法**: 她出招前扔鞭炮可以打断并造成大量架势伤害——每次硬直可以打4-5刀。最多用3-4次鞭炮 / **Firecracker tactic**: Throw firecrackers during her wind-up to interrupt + massive posture damage — 4-5 free hits each time. Use 3-4 times max
- **飞雪（神之飞雪）**: 附魔后额外造成25%伤害——强烈推荐使用 / **Divine Confetti**: 25% bonus damage — highly recommended
- 她的**突刺危**用识破；**横扫危**跳跃踩头 / **Mikiri** thrusts; **jump** sweeps
- **毒刀打法**: 用"琉璃之手里剑"或"毒刃"让她中毒——持续掉血，降低架势恢复速度 / **Poison tactic**: Use "Lazulite Shuriken" or poison blade — poison reduces posture recovery speed
- 掉落：**源之宫之香**（关键道具）/ Drops: **Palace Incense** (key item)

### 樱龙 / Divine Dragon

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 源之宫——巨大樱树 / Fountainhead Palace — Great Sakura Tree |
| **命数 / Health Nodes** | 1条 / 1 node |
| **特殊机制** | 解谜式Boss——不是战斗 / Puzzle boss — not a combat fight |

**打法:**
- **第一阶段**: 穿越雷云到樱龙面前——用钩绳在浮石之间移动 / Phase 1: Cross the storm to reach the Divine Dragon — grapple between floating stones
- **第二阶段**: 樱龙出现后——它用巨大的剑攻击。你的目标不是打它——而是打它周围枝条上发光的小龙（精魂）/ Phase 2: Divine Dragon appears — attacks with a giant sword. Your target isn't the dragon itself — hit the glowing wisps (spirit trees) on branches around it
- 每次打掉几个精魂后——积累一格忍杀条。积累3格后可以忍杀 / Each batch of wisps builds one deathblow node. Build 3 nodes → deathblow
- **雷反**: 樱龙的雷云攻击——站在被雷击的浮石上→接雷→打回去 / **Lightning reversal**: Stand on lightning-struck stones → catch lightning → throw it back
- 三阶段忍杀后——获得**樱龙的眼泪** / After 3 deathblows — acquire **Tears of the Divine Dragon**

### 剑圣苇名一心 / Sword Saint Isshin (最终Boss / Final Boss)

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 苇名城野外 / Ashina field |
| **命数 / Health Nodes** | P1(弦一郎)1条 + P2-P4(一心)3条 = 共4阶段 / 1 (Genichiro) + 3 (Isshin) = 4 phases |
| **弱点 / Weakness** | 识破突刺、雷反、火 / Mikiri thrusts, lightning reversal, fire |

**公认最难的Boss战之一——但也是最公平、最精彩的一场** / **One of the hardest but fairest and most epic boss fights**

**第一阶段: 苇名弦一郎（再战）/ Phase 1: Genichiro (rematch)**
- 比第一次弱了很多——你变强了 / Much weaker than before — you've improved
- 注意他的**不死斩**——有红色发光蓄力，看到后立刻跑向侧面 / Watch for his **Mortal Blade** attack — red glow charge, sprint sideways
- 快速解决——不要浪费药和纸人 / Finish quickly — don't waste healing or emblems

**第二阶段: 剑圣苇名一心P1 / Phase 2: Isshin P1**
- 使用十字枪+剑——远程用枪横扫，近身用剑斩 / Spear + sword — ranged spear sweeps, close-range sword slashes
- 他的**突刺危**非常多——用识破！这是最有效的架势伤害来源 / Lots of **thrust perilous** — **mikiri!** This is your best posture damage source
- 他后退时可能会开枪——连续翻滚躲避 / When he backs up, he might fire his pistol — chain dodge

**第三阶段: 剑圣苇名一心P2 / Phase 3: Isshin P2**
- 增加蓄力大范围横扫+雷攻 / Adds charged wide sweep + lightning attacks
- **经典连招**: 4连枪击+突刺——格挡4下+识破收尾 / **Classic combo**: 4 spear hits + thrust — deflect 4x + mikiri
- 他的**雷反机会**: 他跳起放雷时——使用雷反！这是P2和P3的最佳输出窗口 / **Lightning reversal**: When he jumps for lightning — reversal it! Best damage window in P2 and P3

**第四阶段: 剑圣苇名一心P3 / Phase 4: Isshin P3**
- 同P2——但他更频繁使用雷攻 / Same as P2 but more frequent lightning
- 保持冷静——你已经打到这里了。雷反+识破是胜利的关键 / Stay calm — you've made it this far. Lightning reversal + mikiri = victory

> **关键技巧**：一心的攻击节奏是"慢→快→慢"——不要被他的延迟斩骗到。看到闪光就格挡，但大枪举过头顶的招式有延迟——等0.5秒再挡 / **Key tip**: Isshin's rhythm is "slow → fast → slow" — don't be fooled by delayed attacks. Deflect on flash, but overhead spear charge has delay — wait 0.5s before deflecting

## 可选Boss / Optional Bosses

### 怨鬼 / Demon of Hatred

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 苇名城 城外——隐匿之端 / Ashina Outskirts — Gate |
| **命数 / Health Nodes** | 3条 / 3 nodes |
| **弱点 / Weakness** | 火（吹火筒+火斧）/ Fire (Flame Vent + Flame Axe), **爱哭鬼的指环** / **Malcontent's Ring** |

**打法:**
- **这是一个魂系Boss**——是的，在只狼里混进了一个血源诅咒风格的巨型恶魔Boss / **This is a souls boss** — yes, a Bloodborne-style demon boss snuck into Sekiro
- **关键: 格挡他的攻击是可以的**——但他的火伤不能用肉身格挡（会着火）/ **Key: You CAN deflect his attacks** — but fire attacks burn through your guard
- **P1**: 爪击+火拳——他的攻击大开大合。始终保持在中距离——别太近也别太远 / Claw swipes + fire fist — telegraphed attacks. Stay at mid-range — not too close, not too far
- **P2**: 增加火圈AOE——跳起来躲避+火球三连 / Adds fire ring AoE — jump to dodge + 3-fireball burst
- **P3**: 增加火焰旋风——连续跑动躲避 / Adds fire whirlwind — keep sprinting to dodge
- **爱哭鬼的指环**：可以在P3使用三次——每次让怨鬼停住6秒，给你大量输出时间 / **Malcontent's Ring**: Can be used 3 times in P3 — stuns him for 6 seconds each time
- **跳过打法**: 如果你不想打——在怨鬼所在区域的悬崖边，可以**让他跳崖自杀**（需要精准走位）/ **Skip strategy**: You can make him **jump off the cliff** with precise positioning
