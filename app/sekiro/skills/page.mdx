# 技能与法术 / Skills & Magic

只狼没有传统法术体系——技能通过技能树学习，忍义手提供工具型能力 / Sekiro has no traditional magic system — skills are learned through trees, and the prosthetic arm provides tool-based abilities.

---

## 技能系统 / Skill System

技能通过消耗技能点（经验值累积）在鬼佛处学习 / Skills are learned at Sculptor's Idols using skill points (accumulated experience).

### 技能类型 / Skill Types

| 类型 / Type | 说明 / Description |
|---|---|
| **流派技能 / Combat Arts** | 主动攻击技能——装备后替代普通R1攻击 / Active attack skills — replaces normal R1 when equipped |
| **被动技能 / Passive Skills** | 永久增益——自动生效 / Permanent buffs — always active |
| **忍义手技能 / Prosthetic Skills** | 强化忍义手的功能——如追斩等 / Enhance prosthetic tools — like follow-up slash |
| **体术技能 / Body Skills** | 增强移动和生存能力——如空中弹反 / Enhance movement and survival — like air deflect |

## 技能常见问题 / Skills FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **先学什么技能？/ First skills?** | 识破（最优先）→ 一字斩 → 空中弹反 → 钩绳攻击 / Mikiri (#1) → Ichimonji → Air Deflect → Grapple Attack |
| **技能点可以洗吗？/ Can I respec?** | 不能——点了就不能撤回。选好再点 / No — skills are permanent. Choose wisely |
| **忍义手哪个最好用？/ Best prosthetic?** | 手里剑（通用）+ 鞭炮（野兽Boss）+ 长枪（铠甲精英） / Shuriken (general) + Firecracker (beast bosses) + Spear (armored elites) |
| **纸人上限怎么提升？/ Increase emblem cap?** | 找削符师——每次提升上限/消耗金币 / Talk to Malcontent — increases cap for gold |
| **全技能要多少点？/ All skills cost?** | 约75-80技能点——一周目刷不满，建议二周目 / ~75-80 SP — can't max in one NG, need NG+ |
| **技能点获取效率最高？/ Best SP farm?** | 后期虎口阶梯——紫衣忍者无限刷新。装备+经验提升道具 / Antechamber late game — purple ninjas respawn. Use XP boosting items |
| **忍义手升级优先级？/ Prosthetic upgrade priority?** | 手里剑→鞭炮→机关长枪→鸦羽→其他。优先升级常用的 / Shuriken → Firecracker → Spear → Mist Raven → rest |
