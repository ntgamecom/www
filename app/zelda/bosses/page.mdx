# BOSS攻略 / Boss Strategy Guide

## 主线BOSS / Main Story Bosses

### 急冻盖拉 / Colgera — 风之神殿BOSS

**位置**：海布拉山脉上空的风之神殿（Wind Temple）
**推荐准备**：火箭盾、弓箭、火焰果

急冻盖拉是一个巨大的冰属性骨龙，战斗分为三个阶段。

| 阶段 / Phase | 攻击方式 / Attack Pattern | 应对策略 / Strategy |
|---|---|---|
| 第一阶段 | 空中飞行，射出冰弹 | 使用火箭盾升空，射箭攻击三个弱点 |
| 第二阶段 | 盘旋冲刺，召唤冰块 | 滑翔躲避冰柱，利用上升气流追击 |
| 第三阶段 | 狂暴模式，快速冰弹连射 | 保持移动，用范围弓箭攻击头部弱点 |

**推荐打法**：使用火箭盾直接升空到与BOSS同一高度，在空中用弓箭攻击其身体上的三个蓝色发光弱点。每个弱点被破坏后BOSS会损失大量生命。使用火焰果箭矢可以有效融化冰甲。

### 熔岩巨蜥 / Marbled Gohma — 火之神殿BOSS

**位置**：死亡之山的火之神殿（Fire Temple）
**推荐准备**：阿沅（Yunobo）的贤者能力、双手武器、耐火装备

熔岩巨蜥是一只覆盖着坚硬熔岩外壳的巨大蜘蛛状BOSS。

**核心机制**：
- BOSS身体覆盖熔岩外壳，普通攻击无效
- 需要利用阿沅的滚动能力撞击BOSS使其翻倒
- BOSS翻倒后爬上天花板，需要利用通天术（Ascend）追击
- 反复三次即可击败

**推荐打法**：
1. 召唤阿沅，瞄准BOSS让其滚动撞击
2. BOSS翻倒后露出腹部弱点，使用高攻击力武器全力输出
3. BOSS爬到天花板后，使用通天术直接穿上去
4. 靠近弱点使用阿沅再次撞击
5. 重复到击败为止

### 泥沼章鱼 / Mucktorok — 水之神殿BOSS

**位置**：拉聂尔地区的水之神殿（Water Temple）
**推荐准备**：希多（Sidon）的贤者能力、电击武器、冰冻武器

泥沼章鱼是一个灵活的章鱼型BOSS，会在泥浆中快速移动。

**核心机制**：
- BOSS会在场地中喷洒泥浆阻碍移动
- 使用希多的水盾能力可以清除泥浆
- BOSS暴露时攻击其本体
- 冰属性攻击可以减速BOSS

**推荐打法**：
1. 利用希多的能力制造水盾，在泥浆中自由移动
2. 接近BOSS后使用冰冻武器冻结
3. 冻结后进行连击输出
4. BOSS的弱点会周期性暴露，抓住时机全力攻击
5. 注意躲避BOSS的泥浆喷射

### 女王吉波德 / Queen Gibdo — 雷之神殿BOSS

**位置**：格鲁德沙漠上空的雷之神殿（Lightning Temple）
**推荐准备**：露珠（Riju）的贤者能力、电击武器、火焰武器

女王吉波德是吉波德蜂群的女王，会召唤大量手下。

**核心机制**：
- BOSS的甲壳覆盖全身，正面攻击无效
- 使用露珠的闪电能力可以破除甲壳
- 吉波德群对火焰和电击有弱点
- BOSS会周期性召唤大量吉波德

**推荐打法**：
1. 使用露珠的能力制造闪电区域
2. 引导闪电击中BOSS破除护甲
3. 破甲后使用高伤害武器近战输出
4. 清理召唤的小兵（使用火焰果箭矢）
5. 重复破除护甲到击败

### 魔王加侬多夫 / Demon King Ganondorf

**位置**：地底最深处的魔王城堡
**推荐准备**：大师剑、完整精力、防瘴气装备、全恢复料理、向阳草

加侬多夫是游戏最终BOSS，也是战斗难度最高的敌人。

**核心机制**：
- BOSS拥有四阶段变身（人形态、魔兽形态、瘴气形态、黑龙形态）
- 人形态可以弹反和林克时间闪避
- 魔兽形态使用大范围攻击
- 瘴气形态会减少最大生命值
- 黑龙形态体型巨大，需要在空中战斗

| 阶段 / Phase | 关键策略 / Key Strategy |
|---|---|
| 第一形态 / Humanoid | 完美弹反和林克时间反击，无法直接格挡的攻击要闪避 |
| 第二形态 / Beast | 保持距离射箭，攻击暴露的弱点 |
| 第三形态 / Gloom | 使用向阳草料理治疗瘴气伤害，保持高血量 |
| 第四形态 / Dragon | 在空中使用弓箭射击，利用上升气流 |

**推荐打法**：
- 第一形态：练习弹反，成功后使用大师剑爆发输出
- 第二形态：使用火箭盾升空，空中林克时间射箭
- 第三形态：优先保持血量，使用向阳草解瘴气
- 第四形态：利用跳起的上升气流空战

## 世界BOSS / World Bosses

### 古栗欧克 / Gleeok

古栗欧克是多头龙型BOSS，分布在三个不同区域。

| 类型 / Type | 位置 / Location | 特点 / Feature |
|---|---|---|
| 火焰古栗欧克 / Flame Gleeok | 格鲁德沙漠 | 火属性攻击 |
| 冰冻古栗欧克 / Frost Gleeok | 海布拉雪山 | 冰属性攻击 |
| 雷电古栗欧克 / Thunder Gleeok | 阿卡莱高原 | 雷属性攻击 |
| 王者古栗欧克 / King Gleeok | 地底 | 三种属性 |

**通用打法**：
- 使用火箭盾或高处获得制空权
- 射箭击中每个头的眼睛可以击落BOSS
- 使用对应抗性装备减少元素伤害
- 落地后使用高伤害武器攻击身体

### 岩石巨人 / Stone Talus

分布在海拉鲁各处的岩石巨人，击败后掉落大量矿石。

- **普通岩石巨人**：爬上去攻击顶部的矿石核心
- **稀有岩石巨人**：核心在背后，需要使用通天术
- **熔岩石巨人**：使用冰冻箭冷却岩浆
- **冰冷岩石巨人**：使用火焰箭融化冰层

### 西诺克斯 / Hinox

独眼巨人型怪物，散布在海拉鲁各处。

- **普通西诺克斯**：攻击眼睛使其倒地
- **骷髅西诺克斯**：夜晚出现，弱点是火焰
- **蓝色西诺克斯**：更高级版本，血量更高

**推荐打法**：使用混乱花制造内乱，或使用弓箭射击眼睛后近战输出。

### 巨霸迦马 / Frox

地底世界特有的BOSS，体型巨大。

| 类型 / Type | 特点 / Feature |
|---|---|
| 巨霸迦马 / Frox | 基础型号，吞噬攻击 |
| 巨霸迦马（蓝） / Blue Frox | 更高血量和攻击力 |
| 巨霸迦马（夜光） / Obsidian Frox | 会释放瘴气 |

**推荐打法**：
- 在BOSS张嘴吞噬时投掷炸弹花到其口中
- BOSS翻倒后攻击背后的矿石核心
- 使用光亮花种子照亮场地便于闪避

### 幻影加侬 / Phantom Ganon

在地底大树根附近出现的加侬分身。

- 使用大师剑战斗有额外伤害加成
- 可以弹反和林克时间
- 击败后掉落黑暗武器和瘴气材料
