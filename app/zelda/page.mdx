# 塞尔达传说王国之泪 / Zelda: Tears of the Kingdom

## 游戏概述 / Game Overview

《塞尔达传说 王国之泪》（The Legend of Zelda: Tears of the Kingdom）是任天堂（Nintendo）于2023年5月12日在Nintendo Switch平台发布的开放世界动作冒险游戏。作为2017年《旷野之息》（Breath of the Wild）的正统续作，本作在继承前作核心玩法的基础上，引入了更加宏大的三层地图结构和全新的创意机制。

本指南将帮助你全面了解海拉鲁的每一个角落——从漂浮的空岛到幽深的地底，从战斗技巧到建造创意。无论你是初次踏入海拉鲁的新手，还是想要挑战全收集的资深玩家，这里都有你需要的攻略信息。

### 游戏特色 / Key Features

本作被誉为"开放世界的巅峰之作"（Open-air masterpiece），其核心特色包括：

- **三层世界**（Three-Layer World）：游戏地图分为天空群岛（Sky Islands）、海拉鲁地表（Hyrule Surface）和地底世界（The Depths）三个独立而又相互连接的层面，总面积为前作的数倍
- **究极手**（Ultrahand）：全新的建造系统，允许玩家将几乎任何物体组合在一起，创造出载具、武器和机关
- **倒转乾坤**（Recall）：操控物体的时间流动，使其沿原路返回
- **余料建造**（Fuse）：将材料与武器或箭矢融合，创造全新的装备
- **通天术**（Ascend）：从下方直接穿透天花板到达上方
- **蓝图**（Autobuild）：保存并自动建造你设计过的作品
