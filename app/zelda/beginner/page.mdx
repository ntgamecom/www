# 新手入门 / Beginner Guide

## 初始空岛流程 / Great Sky Island Walkthrough

游戏开始时，林克（Link）在黑暗中被神秘之手唤醒，发现自己身处空中岛屿——初始空岛（Great Sky Island）。这是整个游戏的教程区域，你将在这里学习所有基本能力。

### 第一步：觉醒 / Awakening

从觉醒室（Room of Awakening）出发，你会遇到第一个重要角色——劳鲁（Rauru），他将引导你完成初始教学。穿过走廊，在门口获得第一把武器和盾牌。出门后，你会发现周围环绕着美丽的云海和漂浮的岛屿碎片。

### 第二步：获得能力 / Obtain Abilities

初始空岛总共有四个能力需要在神庙中获取：

| 能力 / Ability | 神庙 / Temple | 位置区域 / Location |
|---|---|---|
| 究极手 / Ultrahand | 建造之祠 | 初始空岛中部 |
| 余料建造 / Fuse | 锻造之祠 | 东侧悬崖附近 |
| 通天术 / Ascend | 上升之祠 | 北侧高地 |
| 倒转乾坤 / Recall | 时光之祠 | 西侧山谷 |

### 第三步：时之神殿 / Temple of Time

收集完四种能力后，前往初始空岛中心的时之神殿（Temple of Time）。在神殿顶部，你将获得本作至关重要的道具——**滑翔伞**（Paraglider）。有了滑翔伞，你就可以跳出初始空岛，降落到海拉鲁大陆了。

## 滑翔伞获取与使用 / Paraglider Acquisition & Usage

滑翔伞是本作最重要的移动工具之一。获得滑翔伞后：

- 在空中按A键打开滑翔伞，缓慢下降
- 可以利用上升气流（如点燃草丛产生的热气流）获得高度
- 配合精力（Stamina）管理，可以滑翔极远的距离
- 在空中可以射箭、投掷武器或使用道具

### 精力管理 / Stamina Management

精力是前期最需要注意的数值。初期精力圈只有一圈，任何攀爬、游泳、滑翔和冲刺都会消耗精力：

- **精力料理**： cooked dishes that restore or temporarily extend stamina
- **精力药水**：使用精力鲈鱼（Staminoka Bass）等材料制作
- **克洛格果实**（Korok Seeds）：找到海利亚人（Hestu）可扩充武器、盾牌和弓箭袋，但无法直接提升精力
- **神庙祝福**：每四个祝福之光（Light of Blessing）可在女神像处兑换精力容器或心之容器

推荐优先升级精力到两圈以上，这将大大提升探索效率。

## 烹饪基础 / Cooking Basics

烹饪是《王国之泪》中恢复生命和获得增益效果的核心方式。

### 烹饪规则 / Cooking Rules

- 在料理锅（Cooking Pot）前手持材料投入锅中即可烹饪
- 最多可同时投入五种材料
- 同类型效果会叠加但不会超过上限
- 不要使用怪物材料（怪物精华除外）与非怪物材料混合烹饪——这会做成" dubious food"（可疑料理）

### 重要料理配方 / Important Recipes

| 料理 / Dish | 材料 / Ingredients | 效果 / Effect |
|---|---|---|
| 精力炒饭 / Stamina Rice Balls | 精力草 + 海拉鲁米 | 恢复精力 |
| 生命炖菜 / Hearty Stew | 生命萝卜 + 任何食材 | 完全恢复+临时黄心 |
| 耐火药 / Fireproof Elixir | 耐火蜥蜴 + 怪物精华 | 防火 |
| 速度药水 / Speed Potion | 速速胡萝卜 + 怪物精华 | 移动速度提升 |
| 潜行药水 / Stealth Potion | 潜行草 + 怪物精华 | 降低脚步声 |

### 烹饪小技巧 / Cooking Tips

- 放入龙角（Dragon Horn）可延长料理持续时间至30分钟
- 星星碎片（Star Fragment）也是优秀的持续时间延长材料
- 怪物精华（Monster Extract）会随机产生正面或负面效果
- 使用佐纳乌能源（Zonaite）材料会影响料理效果

## 海拉鲁第一步 / First Steps in Hyrule

从初始空岛降落后，你会到达海拉鲁大陆。此时的目标是前往监视堡垒（Lookout Landing）。

### 监视堡垒 / Lookout Landing

监视堡垒位于海拉鲁平原中央，是游戏的核心据点。在这里你可以：

- 遇到重要NPC：普尔亚（Purah）、乔舒亚（Josha）、洛贝利（Robbie）等
- 接受主线任务"海拉鲁大地异变"（Regional Phenomena）
- 解锁相机功能（Camera）和冒险笔记
- 开始调查地底世界

### 初始建议 / First Action Items

1. **激活塔**（Activate Towers）：找到并激活散落在各地的监视堡垒塔（Skyview Tower），解锁地图
2. **寻找神庙**（Find Shrines）：每座神庙都是一个快速传送点和能力强化机会
3. **收集材料**（Gather Materials）：不要放过任何可采集的植物和矿物
4. **与NPC对话**（Talk to NPCs）：很多任务和信息都藏在对话中
5. **标记兴趣点**（Mark Points of Interest）：使用地图标记功能和图钉

## 核心机制速览 / Core Mechanics Overview

### 武器耐久度 / Weapon Durability

所有武器、盾牌和弓箭都有耐久度限制，使用一定次数后会损坏。这意味着你需要不断更换武器。建议：

- 前期多使用旅人、士兵等基础武器
- 不要吝啬使用余料建造强化武器
- 宝箱和敌人营地是获取武器的主要来源

### 快速旅行 / Fast Travel

激活神庙、塔和部分关键地点后即可解锁快速传送点。你可以在任何时候打开地图传送（除了在特定战斗或剧情中）。

### 天气系统 / Weather System

海拉鲁有动态天气系统：

- **雨天**：攀爬会打滑（可以装备防滑套装）
- **雷暴**：金属装备会引雷（需换上非金属装备）
- **高温/低温**：需要相应的抗性装备或料理
- **沙暴**：在格鲁德沙漠常见，降低能见度

## 常见误区 / Common Mistakes

1. **忽略烹饪**：只吃生食材效率很低，烹饪后效果翻倍
2. **不会用余料建造**：Fuse是提升伤害的核心机制
3. **不扩充背包**：早找伯库林（Hestu）扩充背包容量
4. **忘记标记**：地图上遇到重要地点要记得标记
5. **硬刚强敌**：前期遇到强敌可以绕道，不必硬拼
