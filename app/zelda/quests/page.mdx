# 任务指南 / Quest Guide

## 任务分类 / Quest Categories

《王国之泪》的任务系统分为三大类别。与《旷野之息》相比，本作的任务数量更多，各任务之间的关联性也更强。

| 任务类型 / Quest Type | 图标 / Icon | 数量 / Count | 说明 / Description |
|---|---|---|---|
| 主线任务 / Main Quest | 星形 | 约15个 | 推进剧情的关键任务 |
| 挑战 / Side Quest | 气泡 | 约60个 | 海拉鲁各地的支线故事 |
| 冒险 / Side Adventure | 菱形 | 约31个 | 包含多重步骤的大型支线 |
| 神庙挑战 / Shrine Quest | 神庙图标 | 约30个 | 需要特定条件解锁的神庙 |

## 主线剧情 / Main Story

### 第一阶段：海拉鲁大地异变 / Regional Phenomena

游戏前期的主要任务线。在监视堡垒与普尔亚（Purah）对话后触发。

**任务流程**：
1. 与普尔亚对话，了解海拉鲁四地的异常现象
2. 选择前往任意一个区域进行调查
3. 在每个区域找到对应的神殿，击败BOSS，解放贤者
4. 推荐顺序：风之神殿（利特）→ 火之神殿（鼓隆）→ 水之神殿（卓拉）→ 雷之神殿（格鲁德）

| 区域 / Region | 初始NPC / Starting NPC | 核心奖励 / Core Reward |
|---|---|---|
| 利特村 / Rito Village | 卡西瓦的弟子 | 丘栗的贤者能力 |
| 鼓隆城 / Goron City | 阿沅 | 阿沅的贤者能力 |
| 卓拉领地 / Zora's Domain | 优娜 | 希多的贤者能力 |
| 格鲁德小镇 / Gerudo Town | 露珠 | 露珠的贤者能力 |

### 第二阶段：海拉鲁城堡异变 / Crisis at Hyrule Castle

完成四个区域的神殿后，海拉鲁城堡出现异象。

**触发条件**：完成所有四个区域变化任务

**任务要点**：
1. 前往监视堡垒与普尔亚汇报
2. 前往海拉鲁城堡调查
3. 与幻影加侬战斗
4. 发现城堡下方的地底入口
5. 解锁前往地底的路径

### 第三阶段：寻访塞尔达的踪迹 / Guidance from Ages Past

在地底寻找第五位贤者米涅鲁的过程中，你需要搜集塞尔达公主留下的记忆。

**触发条件**：完成海拉鲁城堡调查

**关键步骤**：
1. 前往地底大废矿
2. 与米涅鲁的灵魂对话
3. 搜集四大区域的魔像部件
4. 解锁第五贤者米涅鲁
5. 获得魂之贤者能力

### 第四阶段：龙之泪 / The Dragon's Tears

12段记忆的收集任务，揭示塞尔达公主穿越到过去的真相。

**推荐观看顺序**（按时间线）：

| 序号 / No. | 地画位置 / Geoglyph Location | 剧情概要 / Story Summary |
|---|---|---|
| 1 | 海拉鲁平原东侧 | 塞尔达穿越到过去，遇到劳鲁和索尼娅 |
| 2 | 哈特诺村北侧 | 塞尔达学习古代科技 |
| 3 | 卓拉领地西侧 | 塞尔达观察各个种族 |
| 4 | 死亡之山南侧 | 塞尔达见到古代的鼓隆族 |
| 5 | 格鲁德沙漠北侧 | 塞尔达见到古代的格鲁德族 |
| 6 | 利特村东侧 | 塞尔达见到古代的利特族 |
| 7 | 海布拉山南侧 | 塞尔达学习封印之力 |
| 8 | 阿卡莱高原西侧 | 塞尔达与索尼娅的对话 |
| 9 | 费罗尼地区北侧 | 加侬多夫的背叛 |
| 10 | 塔邦达地区 | 塞尔达变身龙（关键剧情） |
| 11 | 拉聂尔海 | 龙之泪的真相 |
| 12 | 海拉鲁城堡东侧 | 塞尔达的最终决定 |

**重要提示**：观看全部12个回忆后，将触发重要剧情。

### 第五阶段：讨伐魔王加侬多夫 / Destroy Ganondorf

最终决战任务。

**触发条件**：完成所有主线任务和龙之泪

**最终战准备清单**：
- 大师剑（满能量）
- 完成所有主线贤者任务
- 至少10心以上（推荐15+）
- 大量恢复料理（全恢复+黄心）
- 向阳草料理（应对瘴气）
- 完整三圈精力
- 最强装备和融合材料

## 重要支线挑战 / Notable Side Quests

### 卢利村重建 / Rebuilding Lurelin Village

帮助卢利村从海盗袭击中恢复的系列任务。

| 阶段 / Stage | 任务 / Quest | 奖励 / Reward |
|---|---|---|
| 1 | 清理海盗营地 | 卢比 |
| 2 | 修复房屋 | 旅馆免费住宿 |
| 3 | 重建商店 | 购买稀有物品 |
| 4 | 码头重建 | 出海钓鱼权限 |

**建议**：尽早完成该任务线，可以获得稳定的免费住宿和特殊交易权限。

### 哈特诺村选举 / The Hateno Village Election

帮助哈特诺村选择新村长。

**参与方式**：
1. 与缇科（Teki）对话触发
2. 收集村中三位候选人的签名
3. 帮助每位候选人完成一个小任务
4. 选择支持的候选人
5. 最终影响：村内商店会因村长人选不同而略有不同

### 摄影协会 / The Photo Society

完成洛贝利的相机任务后解锁。

- 拍摄特定怪物、材料和物品的照片
- 完成图鉴（Hyrule Compendium）的各分类
- 每完成一个分类获得卢比奖励
- 全收集后获得特殊奖励

### 马厩系列任务 / Stable Quest Series

各地驿站的系列任务：

- **白色马匹**：找到并登记特殊的白马
- **金色马匹**：在特定区域捕获金色马
- **巨马**：驯服巨大的马（注意需要更高的精力）
- **马具收集**：通过驿站挑战获得特殊马具

## 冒险记忆 / Side Adventures

### 寻找精灵喷泉 / Great Fairy Fountain

恢复四座大妖精喷泉的系列冒险：

1. 小精灵（Cotera）：海拉鲁平原，费用最低
2. 森林精灵（Kayla）：海布拉地区
3. 沙漠精灵（Mija）：格鲁德地区
4. 火山精灵（Tera）：死亡之山地区

每座喷泉的激活都需要支付一笔卢比和完成附近的音乐演奏任务。

### 地底地图绘制 / Mapping the Depths

探索地底世界并激活所有树根：

- 树根数量：120个（对应地表120座非祝福神庙位置）
- 每个树根激活后解锁周围地底地图
- 激活所有树根是地底全探索的基础
- 推荐使用载具或马匹在地底运输
