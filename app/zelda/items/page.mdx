## 传说武器 / Legendary Weapons

### 大师剑 / Master Sword

大师剑是本作最具标志性的武器。

- **攻击力**：30（常态）/ 60（对瘴气敌人）
- **耐久度**：无限（但需要回复能量）
- **获取方法**：完成"失落之剑"（The Lost Sword）主线任务
- **特殊能力**：在瘴气区域和靠近加侬时攻击力翻倍
- **注意**：需要至少两圈精力才能拔出

## 料理与药剂 / Meals & Elixirs

### 料理分类 / Meal Categories

| 类型 / Type | 效果 / Effect | 代表材料 / Key Material |
|---|---|---|
| 恢复类 / Recovery | 恢复红心 | 生命萝卜、苹果、禽肉 |
| 精力类 / Stamina | 恢复或增加精力 | 精力草、精力鲈鱼 |
| 攻击类 / Attack Up | 增加攻击力 | 大剑草、攻击材料 |
| 防御类 / Defense Up | 增加防御力 | 铠甲草、硬甲材料 |
| 潜行类 / Stealth Up | 降低脚步声 | 潜行草、潜行材料 |
| 速度类 / Speed Up | 增加移动速度 | 速速胡萝卜、速速材料 |
| 抗性类 / Resistance | 温度/电击抗性 | 耐火蜥蜴、防寒材料 |
| 向阳类 / Sunny | 治疗瘴气 | 向阳草 |
| 精力持久类 / Enduring | 增加临时精力上限 | 毅力蘑菇、毅力胡萝卜 |

### 高级料理配方 / Advanced Recipes

- **生命全恢复+黄心**：5个生命萝卜（Hearty Radish）一起烹饪
- **30分钟攻击力UP**：4个大剑草（Razorclaw Crab）+ 1个龙角
- **三级精力持久料理**：3个毅力蘑菇（Endura Carrot）+ 2个精力草
- **全属性抗性**：各有单一属性的抗性料理无法叠加

### 药剂制作 / Elixir Making

药剂需要使用怪物材料（怪物精华、莱尼尔的角等）搭配特定效果的虫子或蜥蜴制作：

- 怪物精华 + 耐火蜥蜴 = 耐火药剂
- 怪物精华 + 速速蛙 = 速度药剂
- 4个怪物材料 + 1个虫 = 延长持续时间（但不会增加效果等级）
