# 能力系统 / Ability System

## 林克的五大能力 / Link's Five Abilities

《王国之泪》中林克拥有五种全新的能力，取代了前作的希卡石板能力。这些能力是探索和战斗的核心。

### 究极手 / Ultrahand

究极手是《王国之泪》中最具变革性的能力，允许你抓取、移动和固定几乎任何物体。

**基本操作**：
- 使用究极手抓取物体（最大同时抓取一个）
- 按L/R旋转物体
- 使用绿色粘液连接物体
- 可以拆卸已连接的物体

**核心用途**：
1. **建造载具**（Vehicle Building）：将风扇+操纵杆+平台组合成飞行器或车辆
2. **搭建桥梁**：将木板或铁板连接到悬崖之间
3. **建造机关**：组合弹簧和铁板制作弹射装置
4. **搬运重物**：利用究极手搬运金属箱子或巨石
5. **战斗应用**：将巨型物体投向敌人

**进阶技巧**：
- 粘合点越多，结构越稳固
- 不对称设计会导致载具倾斜
- 使用货箱（Wagon）的轮子可以提高地面速度
- 用究极手调整冰龙头的角度来改变方向

### 倒转乾坤 / Recall

倒转乾坤允许你将物体的时间流逆转，使其沿原路返回。

**基本操作**：对准物体按L键激活，物体会沿之前的运动轨迹倒回。

**核心用途**：

| 用途 / Use | 示例 / Example |
|---|---|
| 重返空岛 | 让落下的岩石倒回高空，站在上面到达空岛 |
| 反弹飞行器 | 对敌人的飞行器使用倒转，使其返回 |
| 暂停坠落 | 对下落的物体使用，可以站在上面 |
| 回收装备 | 对打飞出去的武器使用，将其召回 |
| 破解机关 | 许多神庙谜题需要倒转旋转或移动的平台 |

**重要技巧**：
- 倒转乾坤可以同时对多个物体使用（如果它们运动轨迹相同）
- 站在被倒转的物体上时可以移动和攻击
- 倒转的物体具有碰撞判定，可以用来攻击敌人
- 对高速运动的物体使用倒转效果最好

### 余料建造 / Fuse

余料建造是本作战斗系统的核心。具体内容已在"战斗系统"页面详细说明，这里列出能力的进阶用法。

**武器融合进阶**：
- 盾牌+火箭：火箭盾原地升空
- 盾牌+矿车：在铁轨上高速滑行
- 盾牌+弹簧：弹开接近的敌人
- 武器+左纳尼乌姆：对构造体特攻
- 箭矢+蓝宝石：大范围冰冻
- 箭矢+红宝石：大范围爆炸

**融合与耐久度**：
- 融合材料不会改变武器的攻击速度或动作模组（枪仍然是枪动作）
- 融合后的武器耐久度提升约20%-30%
- 武器损坏后融合材料会掉落，可以回收利用
- 制造余料建造武器消耗的是武器本体耐久度

### 通天术 / Ascend

通天术允许你直接从下方穿透天花板到达上方。

**基本操作**：在有天花板的地方下方按L键，林克会化作光柱穿透上方。

**使用条件**：
- 上方必须有可到达的空间
- 天花板不能太厚（不超过约50米）
- 部分特殊区域的天花板无法穿透

**核心用途**：
1. **快速到达高处**：摆脱爬山的烦恼
2. **逃出洞穴**：从洞穴底部直接穿到地面
3. **突破关卡**：很多神庙谜题的核心解法
4. **潜入时的快速撤退**：被敌人发现时可以瞬间穿墙离开
5. **战斗优势**：从下方突袭高处的敌人

**进阶技巧**：
- 在洞穴地底如果找不到路，抬头看看能不能用通天术
- 先使用火箭盾飞到高处，再用通天术可以穿到更高的地方
- 站在移动的平台上使用通天术，但穿出后平台会留在原位

### 蓝图 / Autobuild

蓝图能力可以保存和自动建造你设计过的作品。

**获取方法**：在地底的大废矿（Great Abandoned Central Mine）获得

**基本功能**：
- 保存你使用究极手建造的结构
- 使用左纳乌能源自动重建保存的设计
- 可以收藏最多8个设计蓝图

**自动填充**：如果你没有保存设计中所需的材料，蓝图会自动消耗附近的材料补足。

**能源消耗**：
- 每个部件的能源消耗不同
- 风扇消耗9能源
- 操纵杆消耗6能源
- 大轮胎消耗12能源
- 小轮胎消耗6能源
- 平台消耗3能源

**进阶用途**：
1. **快速部署**：战斗中快速召唤预先设计的武器平台
2. **过河造桥**：保存一个简易桥梁设计随时调用
3. **矿车建造**：设计标准化的矿车用于地底运输
4. **复制敌人**：蓝图还可以复制用究极手抓起的敌人结构

## 贤者的化身能力 / Sage Avatar Abilities

在完成四个区域神殿后，四位贤者（Sage）会给予你他们的化身（Avatar），在战斗中协助你。

### 丘栗 / Tulin — 利特贤者

**能力**：制造旋风（Gust），吹飞敌人和远程攻击

| 能力 / Ability | 说明 / Description |
|---|---|
| 主动技能 | 按A键释放旋风，吹飞前方敌人 |
| 被动效果 | 在探索时自动攻击附近的敌人 |
| 解锁方式 | 完成风之神殿 |
| 推荐搭配 | 弓箭流、需要控场的战斗 |

**使用技巧**：
- 丘栗的旋风可以吹散沙暴
- 在雪山地区特别有用
- 可以把敌人吹下悬崖造成地形杀

### 阿沅 / Yunobo — 鼓隆贤者

**能力**：滚动撞击（Rolling Charge），冲撞敌人和障碍物

| 能力 / Ability | 说明 / Description |
|---|---|
| 主动技能 | 按A键让阿沅向前滚动撞击 |
| 被动效果 | 自动冲撞附近的敌人 |
| 解锁方式 | 完成火之神殿 |
| 推荐搭配 | 近战型战斗、采矿 |

**使用技巧**：
- 可以撞碎矿石墙
- 对岩属性敌人有额外伤害
- 打破障碍物的首选能力

### 希多 / Sidon — 卓拉贤者

**能力**：水盾（Water Shield），包围林克的水属性屏障

| 能力 / Ability | 说明 / Description |
|---|---|
| 主动技能 | 按A键获得水盾屏障 |
| 被动效果 | 自动喷水攻击附近敌人 |
| 解锁方式 | 完成水之神殿 |
| 推荐搭配 | 火焰环境、防守型战斗 |

**使用技巧**：
- 水盾可以抵挡一次攻击
- 在高温环境下提供冷却效果
- 搭配卓拉武器获得武器强化加成

### 露珠 / Riju — 格鲁德贤者

**能力**：闪电领域（Lightning Field），召唤闪电攻击

| 能力 / Ability | 说明 / Description |
|---|---|
| 主动技能 | 按A键在露珠周围制造闪电区域 |
| 被动效果 | 自动释放闪电攻击附近敌人 |
| 解锁方式 | 完成雷之神殿 |
| 推荐搭配 | 群战、属性配合 |

**使用技巧**：
- 闪电可以击落敌人手中的武器
- 在水中区域闪电扩散范围更大
- 对金属护甲的敌人特别有效

### 米涅鲁 / Mineru — 魂之贤者

**能力**：操控建造机甲（Construct Mech）

| 能力 / Ability | 说明 / Description |
|---|---|
| 主动技能 | 按A键召唤机甲并进入操控模式 |
| 被动效果 | 自动攻击附近敌人 |
| 解锁方式 | 完成魂之神殿（地底） |
| 推荐搭配 | 探索地底、强力战斗 |

**使用技巧**：
- 机甲免疫瘴气
- 在机甲上可以融合武器到手臂
- 机甲可以采矿和破坏障碍物
- 是地底探索的最佳伙伴
