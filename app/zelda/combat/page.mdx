# 战斗系统 / Combat System

## 余料建造 / Fuse System

余料建造（Fuse）是本作战斗系统的核心创新机制。通过将武器与材料融合，你可以创造出全新的武器，大幅提升攻击力、耐久度或赋予特殊效果。

### 武器融合 / Weapon Fusing

拿着武器，按住L键选择材料即可融合。融合规则如下：

| 武器类型 / Weapon | 推荐融合材料 / Recommended Fuse | 效果 / Effect |
|---|---|---|
| 剑类 / Sword | 蜥蜴刃、大魔像核心 | 高攻击力加成 |
| 大剑 / Greatsword | 熔岩矿、银莱尼尔的刃角 | 极高攻击力 |
| 枪 / Spear | 冰莱尼尔的刃角、电蝙蝠翅膀 | 元素效果+远程攻击 |
| 棍棒 / Club | 巨石、铁箱 | 钝器打击 |
| 弓 / Bow | 元素宝石、古代箭（剧情） | 特殊箭矢效果 |

### 特殊融合 / Special Fuse Combinations

- **武器+盾牌**：创造出可以攻击的盾（盾击伤害提升）
- **盾牌+矿车**：在轨道上滑行
- **盾牌+火箭**：火箭盾，原地升空
- **盾牌+弹簧**：弹飞靠近的敌人
- **武器+左纳尼乌姆**：提升对构造体的伤害

### 融合攻击力计算 / Fuse Damage Calculation

融合武器的总攻击力 = 基础武器攻击力 + 融合材料攻击力。融合材料的攻击力由材料本身决定：

- 银莱尼尔刃角：55攻击力
- 白鬃莱尼尔刃角：44攻击力
- 蓝鬃莱尼尔刃角：33攻击力
- 红鬃莱尼尔刃角：22攻击力
- 魔像核心III：30攻击力
- 莫力布林刃角：25攻击力

弓箭同样可以使用余料建造进行融合。按住R键射箭后，按住十字键上选择融合材料。

## 核心战斗技巧 / Core Combat Techniques

### 林克时间 / Flurry Rush

林克时间是游戏中最重要的回避反击技巧。

触发方法：
1. 在敌人即将击中你的瞬间向后或侧向跳跃（后空翻）
2. 成功闪避后，时间会变慢
3. 按Y键发动连续攻击（Flurry Rush）

练习要点：
- 每种敌人的攻击节奏不同，需要熟悉
- 后空翻适合躲避水平挥砍
- 侧跳适合躲避突刺攻击
- 可以预判而非反应——提前观察敌人动作

### 完美防御 / Perfect Parry

用盾牌在敌人攻击的瞬间格挡，可以弹开敌人攻击并使其中门大开。

- 按A键举盾
- 在攻击到达前的瞬间松开A键
- 成功弹反后敌人会短暂硬直
- 可以弹反守护者激光（如果有盾牌的话）

### 潜行攻击 / Stealth Attack

通过潜行接近敌人发起的强力攻击。

- 从背后偷袭：8倍伤害
- 从高处跳劈：可叠加潜行倍率
- 装备潜行套装（Sheikah Armor）降低脚步声
- 使用潜行药水进一步提升效果
- 烟雾蘑菇和混乱花是潜行流派的好伙伴

## 元素系统 / Elemental System

### 火焰 / Fire

- 造成持续燃烧伤害
- 可以点燃杂草和木质武器
- 制造上升气流（用于滑翔）
- 融化冰系障碍
- 火+冰 = 蒸发（造成额外伤害）

### 冰冻 / Ice

- 完全冻结敌人（无法行动）
- 被冰冻的敌人受到的首击为三倍伤害
- 可以在水面上行走
- 冻结的敌人不会掉落悬崖

### 雷电 / Electricity

- 麻痹敌人，使其掉落武器
- 在水中导电范围扩大
- 金属装备引雷
- 雷电+水 = 大范围电击

## 武器类型解析 / Weapon Type Analysis

### 单手武器 / One-Handed Weapons

- 速度最快，可以持盾同时攻击
- 适合连击和一打一
- 代表武器：旅人之剑、王族之剑、大师剑

### 双手武器 / Two-Handed Weapons

- 攻击力最高，范围大
- 无法持盾，但可以蓄力旋转攻击
- 代表武器：碎岩大剑、王族大剑

### 长枪 / Spears

- 攻击距离最远
- 可以保持安全距离攻击
- 蓄力可连续突刺
- 代表武器：王族之枪、光鳞之枪

### 弓 / Bows

- 远程攻击
- 二段蓄力增加射程和伤害
- 可配合余料建造使用特殊弓箭
- 代表武器：王族之弓、大鹫弓、黄昏之光弓

## 特殊战斗场景 / Special Combat Scenarios

### 空中战斗 / Aerial Combat

- 从高处跳下时按ZR键进入林克时间射箭
- 可以使用火箭盾升空后射击
- 利用滑翔伞调整位置
- 空中射箭每次命中都算暴击

### 载具战斗 / Vehicle Combat

- 在建造的载具上射箭
- 在战车上安装火焰发射器和光束发射器
- 使用风扇制造移动炮台
- 在空中飞行器上投掷炸弹花

### 元素配合 / Element Combos

- 先冰冻再锤击：碎冰效果，造成大量伤害
- 先电击再攻击：敌人无法还手
- 使用火焰武器攻击冰冻敌人：瞬间蒸发
