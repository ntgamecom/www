# 海拉鲁世界 / The World of Hyrule

## 三层地图架构 / Three-Layer Map Structure

《王国之泪》最令人惊叹的设计之一就是其三层的世界架构。与前作《旷野之息》相比，本作的地图不仅向上延伸到天空，还向地下延伸至深渊。三层地图总面积约为前作的三到四倍，每一层都有独特的生态、资源和挑战。

| 层级 / Layer | 面积 / Area | 高度范围 / Height | 主要特征 / Key Feature |
|---|---|---|---|
| 天空群岛 / Sky Islands | 约30% 地表面积 | 800-2000m | 浮空岛屿、佐纳乌遗迹 |
| 海拉鲁地表 / Surface | 100% 前作地图 | 0-800m | 城镇、森林、沙漠、雪山 |
| 地底世界 / Depths | 约100% 地表面积 | -500-0m | 完全黑暗、特殊材料、BOSS |

## 天空群岛 / Sky Islands

天空群岛是本作新增的最大亮点之一。在游戏初期，大量岛屿漂浮在海拉鲁上空，等待探索。

### 主要空岛区域 / Major Sky Regions

- **初始空岛**（Great Sky Island）：游戏的起始区域，教程所在地
- **风之神殿**（Wind Temple）：位于海布拉山脉上空的暴风眼中心
- **雷之神殿**（Lightning Temple）：格鲁德沙漠上空的闪电云层中
- **水之神殿**（Water Temple）：拉聂尔山上方水域环绕的岛屿
- **火之神殿**（Fire Temple）：死亡之山上空的熔岩区域
- **龙之岛**（Dragonhead Island）：雷龙所在的浮空区域

天空群岛大量使用**佐纳乌科技**（Zonai Technology），包括各种装置、能量核心和遗迹建筑。在空岛上你可以找到大量的左纳乌能源（Zonaite）和左纳乌装置（Zonai Devices）。

### 天空探索技巧 / Sky Exploration Tips

- 使用**发射台**（Launch Platforms）可以快速到达高处的岛屿
- **火箭盾**（Rocket Shield）是前期探索空岛的好帮手
- 记录空岛之间的**气流通道**（Air Currents），可以用滑翔伞快速移动
- 每个空岛群通常有一个**魔像**（Golem）BOSS守护宝箱

## 海拉鲁地表 / Hyrule Surface

海拉鲁地表基本上是前作《旷野之息》地图的扩展版本，但增加了大量新内容、洞穴和变化。

### 新增地表内容 / New Surface Additions

- **洞穴系统**（Cave System）：遍布海拉鲁的洞穴，里面可能有魔尤伊（Bubbulfrog）和珍贵矿石
- **井**（Wells）：很多地区新增了可探索的水井
- **深穴**（Chasm）：通往地底的巨大坑洞，地表共有多个入口
- **地画**（Geoglyphs）：分布在海拉鲁各处的地面图案，与龙之泪任务相关
- **监视堡垒**（Lookout Landing）：取代了前作中的初始台地

### 驿站系统 / Stable System

海拉鲁各主要道路旁都有驿站，提供以下服务：

1. 寄存和取回马匹
2. 注册新的马匹
3. 购买马具和缰绳
4. 睡觉恢复生命
5. 接受部分支线任务

## 地底世界 / The Depths

地底世界是《王国之泪》最具革命性的新增内容。这片完全黑暗的地下空间覆盖了与地表相同的面积，充满了危险和宝藏。

### 地底世界的规则 / Depths Rules

- **完全黑暗**：需要携带光亮花种子（Brightbloom Seed）或使用发光装备
- **瘴气**（Gloom）：紫色的有毒雾气会减少你的最大生命值，需要阳光料理或特定装备
- **树根**（Lightroots）：地底的传送点，与地表的神庙位置对应
- **BOSS**：每个主要区域都有地底BOSS镇守

### 关键地底区域 / Key Depths Areas

| 地底区域 / Depths Area | 对应地表区域 | 主要BOSS |
|---|---|---|
| 中央大废矿 / Central Great Mine | 海拉鲁平原 | 魔像 |
| 海布拉地底 / Hebra Depths | 海布拉山脉 | 地底古栗欧克 |
| 拉聂尔地底 / Lanayru Depths | 拉聂尔地区 | 地底岩石巨人 |
| 格鲁德地底 / Gerudo Depths | 格鲁德沙漠 | 巨霸迦马 |
| 死亡之山地底 / Eldin Depths | 死亡之山 | 熔岩巨兽 |

### 树根与神庙对应关系 / Lightroot-Shrine Correspondence

每个地底树根（Lightroot）的位置与地表的一座神庙（Shrine）完全对应。如果你在地表发现了一座神庙但不记得具体位置，可以从地底同一坐标的树根位置推测出来。反之亦然。这是一个极其有用的导航技巧。

### 地底探索必备品 / Essential Depths Gear

- **光亮花种子**（Brightbloom Seeds）：照亮黑暗，无限投掷
- **防瘴气装备**（Gloom-resistant gear）：如残旧套装升级后
- **向阳草料理**（Sunny Herb dishes）：治疗瘴气伤害
- **大量箭矢**：地底有大量敌人
- **佐纳乌能源**：用于点亮载具和照明

## 龙之泪与地画 / Dragon Tears & Geoglyphs

完成龙之泪（Dragon Tears）任务是了解《王国之泪》剧情的关键。海拉鲁大陆上有12处地画（Geoglyphs），每个地画对应一段记忆，展示塞尔达公主在过去的经历。

### 触发与流程 / Trigger & Flow

1. 在监视堡垒触发龙之泪任务线
2. 与**英帕**（Impa）对话开始寻找地画
3. 前往每个地画处触发回忆
4. 收集全部12个回忆后获得重要剧情信息

每个龙之泪的回忆解锁都会揭示塞尔达公主穿越到过去的故事。强烈建议按照特定顺序观看，以获得最佳的叙事体验。

## 快速旅行网络 / Fast Travel Network

### 监视堡垒塔 / Skyview Towers

海拉鲁全域共有15座监视堡垒塔。激活后可以解锁该区域的地图信息。每座塔通常需要通过一个小型解谜才能进入。

### 神庙传送 / Shrine Warping

激活过的神庙可以作为快速传送点。这是游戏中最常用的传送方式，共计152个传送点。
