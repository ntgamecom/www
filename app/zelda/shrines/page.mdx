# 神庙挑战 / Shrine Challenges

## 神庙系统概述 / Shrine System Overview

神庙（Shrine）是《王国之泪》中的小型迷宫和挑战关卡。与《旷野之息》的120座神庙不同，本作共有152座神庙分布在天空和海拉鲁地表。每座神庙首次通关后获得一个祝福之光（Light of Blessing），积累四个可以兑换心之容器或精力容器。

### 神庙类型 / Shrine Types

| 类型 / Type | 数量 / Count | 说明 / Description |
|---|---|---|
| 解谜类 / Puzzle | 约70座 | 需要利用能力和环境机关解谜 |
| 祝福类 / Blessing | 约40座 | 通过外部解谜或完成任务后直接获得奖励 |
| 战斗类 / Combat | 约22座 | 与固定配置的敌人战斗 |
| 试炼类 / Proving Grounds | 约12座 | 限制装备的生存战斗挑战 |
| 其他特殊类 / Special | 约8座 | 包含独特机制的混合类型 |

### 祝福之光分布 / Light of Blessing Distribution

整个海拉鲁共有152个祝福之光，可以兑换以下内容：

| 兑换项目 / Exchange | 所需数量 / Cost | 最大次数 / Max |
|---|---|---|
| 心之容器 / Heart Container | 4 | 最多+40心（含初始心和神殿心） |
| 精力容器 / Stamina Vessel | 4 | 最多3圈精力 |
| 总计 / Total | 152个祝福之光 | 38次兑换 |

### 海布拉地区 / Hebra（约18座）

海布拉地区的神庙主要分布在雪山和天空区域。

- **Ijo-o Shrine**（解谜）：使用通天术穿过冰柱
- **Gikaku Shrine**（祝福）：在雪山完成特定解谜后出现
- **Sisart Shrine**（战斗）：对战冰属性莫力布林
- **Oshozan-u Shrine**（试炼）：限制装备的雪地战斗

**寒冷天气注意**：海布拉地区的很多神庙入口在寒冷区域，建议携带防寒装备或抗寒料理。

### 塔邦达边境 / Tabantha（约14座）

- **Ananki Shrine**（解谜）：利用气流平台
- **Sakun-bon Shrine**（祝福）：完成房车任务后解锁
- **Jochi-ihiga Shrine**（战斗）：对战火焰构造体

### 拉聂尔地区 / Lanayru（约16座）

以水相关的解谜为主。

- **Gatanisis Shrine**（解谜）：使用卓拉能力制作水桥
- **Boshigaki Shrine**（祝福）：在瀑布顶部找到入口
- **Moshakabe Shrine**（试炼）：限制装备的水中战斗

**卓拉能力配合**：拉聂尔地区很多神庙推荐先完成卓拉领地的主线任务，利用希多的能力解谜。

### 阿卡莱地区 / Akkala（约12座）

- **Sinatanika Shrine**（解谜）：使用火箭盾到达高处平台
- **Rasiwak Shrine**（祝福）：完成监视堡垒任务解锁

### 格鲁德地区 / Gerudo（约18座）

## 神庙解谜通用技巧 / General Shrine Puzzle Tips

### 究极手技巧 / Ultrahand Tips

- 旋转视角可以帮助你更精确地连接物体
- 使用究极手将大球体移动到凹陷中
- 连接多个平台可以制作更长的桥
- 把风扇和平台组合可以创造简易电梯

### 倒转乾坤技巧 / Recall Tips

- 对旋转的齿轮使用倒转可以改变机关状态
- 将滚动的球体倒转送回起点
- 站在被倒转的平台上升至高处
- 对快速移动的障碍物使用倒转创造安全窗口

### 通天术技巧 / Ascend Tips

- 通天术可以从下层直接到达目标区域
- 部分神庙有隐藏的天花板通道
- 通天后到达的区域可能有隐藏宝箱
- 这是很多神庙的逃课（skip）方法

### 余料建造技巧 / Fuse Tips

- 将灯融合到盾牌上获得照明盾
- 将矿石融合到武器上改变攻击属性
- 将火箭融合到盾牌上方便攀爬
- 部分神庙需要特制的融合武器

## 快速刷祝福之光 / Fast Light of Blessing Farming

如果你希望快速积攒祝福之光：

1. **优先祝福类神庙**：只需要找到入口即可获得奖励
2. **使用传送列表**：按照区域逐一清理效率最高
3. **天空群岛的神庙**：很多需要复杂的解谜但奖励相同
4. **标记未完成的神庙**：在地图上用图钉标记已找到但未完成的神庙
