# 究极手建造 / Ultrahand Building

## 建造系统基础 / Building System Basics

究极手（Ultrahand）建造系统是《王国之泪》最具创意和自由度的机制。通过组合左纳乌装置（Zonai Devices）和普通物体，你可以创造出近乎无限的机械设计。

### 建造三要素 / Three Core Elements

| 要素 / Element | 说明 / Description | 示例 / Examples |
|---|---|---|
| 结构件 / Structure | 提供框架和平台 | 木板、石板、铁板、木箱 |
| 动力源 / Power | 提供移动能力 | 风扇、轮子、火箭、螺旋桨 |
| 控制系统 / Control | 提供操控功能 | 操纵杆、转向轮 |

### 核心建造原则 / Core Building Principles

1. **对称性**（Symmetry）：不平衡的设计会导致载具偏向一侧
2. **重心**（Center of Gravity）：重心越低、越居中，稳定性越好
3. **结构强度**：连接点越多越稳固，但重量也越大
4. **能量管理**：复杂的机械消耗电池更快，需要备用电源

## 实用建造方案 / Practical Build Designs

### 初级载具 / Beginner Vehicles

**1. 简易飞行器 / Basic Flying Machine**
- **材料**：2个风扇 + 1个操纵杆 + 1个平台
- **步骤**：
  1. 将平台作为基座
  2. 在平台后方安装两个风扇（一左一右朝下偏后）
  3. 在平台中央安装操纵杆
  4. 方向控制：前倾=前进，后倾=悬停/后退
- **评价**：最简单实用的飞行器，操作直观，适合初期探索

**2. 地面越野车 / Off-Road Vehicle**
- **材料**：4个小轮子 + 1个操纵杆 + 1个平台
- **步骤**：
  1. 平台作为车身
  2. 前方安装两个小轮子作为转向轮
  3. 后方安装两个小轮子作为驱动轮
  4. 安装操纵杆于平台中央
- **评价**：适合海拉鲁地表探索，比马更快更灵活

**3. 简易小船 / Simple Boat**
- **材料**：2个风扇 + 1个操纵杆 + 1个木板平台 + 2个浮筒/木箱
- **步骤**：
  1. 木板作为船体
  2. 两侧安装浮筒保持稳定
  3. 后方安装风扇（水平朝后）
  4. 安装操纵杆
- **评价**：过河渡海的最佳选择

### 中级建造 / Intermediate Builds

**4. 六扇飞行器 / Six-Fan Flying Machine**
- **材料**：6个风扇 + 1个操纵杆 + 1个大平台
- **结构**：
  - 四个风扇垂直朝下提供升力
  - 两个风扇水平朝后提供前进推力
  - 平台中央安装操纵杆
- **评价**：稳定性和载重能力远超简易版，可运送左纳乌矿等重物

**5. 坦克战车 / Battle Tank**
- **材料**：2个大轮子 + 2个火焰发射器 + 1个操纵杆 + 1个铁板车身
- **结构**：
  - 铁板作为防护车身
  - 前方安装火焰发射器
  - 两侧安装大轮子（适应崎岖）
  - 操纵杆位于车身保护下
- **评价**：非常适合清理地底和怪物营地

**6. 火箭升空器 / Rocket Ascender**
- **材料**：3个火箭 + 1个平台 + 1个操纵杆 + 1个滑翔伞（使用中）
- **使用方式**：
  1. 站在平台上
  2. 同时激活三个火箭
  3. 到达最高点后使用滑翔伞
- **评价**：快速到达极高空岛的最优方案

### 高级建造 / Advanced Builds

**7. 战斗机甲 / Battle Mech**
- **材料**：2个大轮子（腿部）+ 1个操纵杆 + 2个光束发射器 + 1个大平台（身体）+ 1个小平台（头部）
- **结构设计**：
  - 大轮子作为机甲腿部
  - 大平台作为身体
  - 小平台作为头部（安装操纵杆）
  - 在机甲手臂位置安装光束发射器
- **操作方式**：
  - 使用操纵杆控制移动
  - 光束发射器自动攻击
  - 机甲可以踩踏小型敌人
- **评价**：娱乐性大于实用性，但非常酷

**8. 自动采矿机 / Auto-Mining Rig**
- **材料**：2个弹簧 + 2个大锤（武器）+ 1个平台 + 1个风扇
- **机制**：
  - 弹簧驱动大锤反复敲击地面
  - 风扇使机器缓慢移动
  - 被动采集沿途矿石
- **评价**：懒人采矿法，效率中等但非常有趣

**9. 飞行战舰 / Flying Warship**
- **材料**：6个风扇 + 2个光束发射器 + 2个火焰发射器 + 1个操纵杆 + 1个大平台 + 2个小平台
- **结构**：
  - 主平台作为舰体
  - 四个升力风扇 + 两个推进风扇
  - 两侧安装武器
  - 弹药仓（载货区）放置炸弹花
- **评价**：终极探索战斗一体机，能源消耗巨大

**10. 科洛格传送装置 / Korok Transportation Device**
- **材料**：1个风扇 + 1个平台 + 1个操纵杆 + 笼子结构
- **用途**：
  - 将科洛格运送到其朋友处
  - 使用笼子结构固定科洛格
  - 适用于地形复杂的科洛格运送任务
- **评价**：专门解决最令人头疼的科洛格运送任务

### 蓝图管理技巧 / Blueprint Management Tips

1. 保存通用方案而非特化方案
2. 最常用的方案放在前两个位置
3. 定期更新改进设计
4. 共享蓝图码（Zonaite Code）可以和其他玩家交流设计
5. 蓝图自动填充材料时优先使用地上的物体而非胶囊

## 建造常见问题 / Common Building Issues

| 问题 / Issue | 原因 / Cause | 解决方案 / Solution |
|---|---|---|
| 载具倾斜 | 重心不对称 | 检查两侧重量是否平衡 |
| 飞行器下落 | 升力不足 | 增加风扇或减少重量 |
| 转向困难 | 缺少转向机制 | 安装转向轮或增加推力差 |
| 结构解体 | 连接点不足 | 增加粘合点加固 |
| 能源不够 | 电池容量不足 | 升级电池或携带备用 |

## 电池系统 / Battery System

电池决定了你能使用左纳乌装置多长时间。

| 电池等级 / Tier | 容量 / Capacity | 升级所需能源 / Required |
|---|---|---|
| 初始 / Base | 1节（3格） | --- |
| 二级 / Tier 2 | 2节（6格） | 100左纳乌能源 |
| 三级 / Tier 3 | 3节（9格） | 100左纳乌能源 |
| 最高 / Max | 16节（48格） | 4500+左纳乌能源 |

**升级方法**：在地底废矿（Mines）使用左纳乌矿兑换能源结晶（Energy Crystal），然后在能源锻造炉（Energy Forge）处升级电池。
