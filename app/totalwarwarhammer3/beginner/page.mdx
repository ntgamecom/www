# 新手入门 / Beginner Guide

全面战争：战锤3是一款集回合制战略和实时战斗于一体的宏大策略游戏。对新手来说，理解核心机制比精通操作更重要——选对派系、管好经济、打对仗，你就能征服凡世帝国 / Total War: Warhammer 3 combines turn-based strategy with real-time battles. For beginners, understanding core mechanics matters more than mastering controls — pick the right faction, manage your economy, fight smart, and you'll conquer the mortal empires.

---

## 核心游戏循环 / Core Game Loop

| 回合 / Turn Phase | 操作 / Actions |
|:----:|------|
| **战略地图回合 / Campaign Turn** | 移动军队(蓝色区域=行军距离)→建设建筑→招募单位→外交谈判→事务官行动→结束回合 / Move armies (blue zone = range) → build buildings → recruit units → diplomacy → agent actions → end turn |
| **战斗阶段 / Battle Phase** | 触发战斗→布阵(部署阶段)→实时指挥→胜利/撤退→回到战略地图 / Battle triggered → deploy → real-time command → victory/retreat → back to campaign |
| **关键资源 / Key Resources** | 金币(建筑/招募/外交)、食物(人口增长/军队维持)、魔法之风(战斗中使用法术) / Gold (buildings/recruit/diplomacy), Food (population/army upkeep), Winds of Magic (spells in battle) |

## 界面详解 / UI Guide

| 界面 / UI | 功能 / Function |
|:----:|------|
| **顶部信息栏 / Top Bar** | 回合数、金币、食物/其他资源、魔法之风库存、外交状态 / Turn count, gold, food/other resources, winds of magic reserve, diplomatic status |
| **左下角军队 / Army Panel** | 选择军队后显示——将领等级、兵种列表、移动力 / Shows selected army — lord level, unit list, movement range |
| **右下角小地图 / Minimap** | 战略地图全览——战斗中小地图同样重要 / Campaign map overview — equally important in battle for unit positioning |
| **城市面板 / City Panel** | 点击城市打开——建筑序列、招募队列、驻军信息 / Click city → building chain, recruitment queue, garrison info |
| **外交面板 / Diplomacy** | 查看/操作与其他派系的关系——条约、宣战、交易 / View/manage relations — treaties, war declarations, trade |

## 胜利条件 / Victory Conditions

| 胜利类型 / Victory Type | 条件 / Conditions |
|:----:|------|
| **短期胜利 / Short Victory** | 完成该派系特定目标(通常是占领特定省份+消灭特定敌人)——约50-80回合 / Faction-specific goals (usually control certain provinces + defeat certain enemies) — ~50-80 turns |
| **长期胜利 / Long Victory** | 更严格的条件——占领更多省份+消灭更多敌人——约100-150回合 / Stricter conditions — more provinces + more enemies — ~100-150 turns |
| **凡世帝国征服 / Domination** | 占领所有行省首府——真正的全图制霸 / Control all provincial capitals — true total map domination |
| **魔域战役胜利 / Realm of Chaos** | 进入混沌魔域→收集四神灵魂→击败最终Boss——有回合限制 / Enter Chaos Realms → collect 4 gods' souls → defeat final boss — turn-limited |

## 经济管理入门 / Economy Basics

| 建筑 / Building | 效果 / Effect | 优先建造 / Priority |
|:----:|:----|:----:|
| **主城序列 / Main Settlement** | 每级+人口+建筑槽+驻军——优先升到3级 / Each level = pop + building slots + garrison — priority to level 3 |
| **收入建筑 / Income Building** | 直接+金币/回合——如市场、工坊 / Direct + gold/turn — e.g., markets, workshops |
| **增长建筑 / Growth Building** | +人口增长→更快升主城——如粮仓 / +population growth → faster capital leveling — e.g., granaries |
| **军事建筑 / Military Building** | 解锁高级兵种招募——只建在有兵种加成的行省 / Unlock elite unit recruitment — only build in provinces with faction bonuses |
| **墙 / Walls** | 防御——被围攻时提供驻军+城墙战斗 / Defense — provides garrison + wall battle when besieged |

**经济铁律**: 不要所有城都建军事建筑。**2-3个经济行省**(全收入建筑)+**1-2个军事行省**(全军事建筑)=最优配置。行省首府升3级解锁全部建筑槽后再规划 / **Rule**: Don't build military in every province. **2-3 economic provinces** (pure income) + **1-2 military provinces** (pure recruitment) = optimal. Level capital to 3 first for all building slots.

## 战斗基础 / Battle Basics

| 概念 / Concept | 说明 / Description |
|:----:|------|
| **兵种相克 / Unit Counters** | 反大型(长矛/戟兵)克制骑兵/巨兽。反步兵(剑盾/斧头)克制步兵。远程克制慢速近战 / Anti-large (spears/halberds) beat cavalry/monsters. Anti-infantry (swords/axes) beat infantry. Ranged beats slow melee |
| **士气 / Morale** | 兵种有士气值——受到损伤/侧翼被攻击/将领死亡→士气下降→白旗溃逃 / Units have morale — taking damage/flanked/general killed → morale drops → route |
| **阵型 / Formation** | 步兵前排→远程/火炮中排→骑兵/巨兽两翼或后备。这是基础"锤砧战术" / Infantry front → ranged/artillery mid → cavalry/monsters on flanks or reserve. The classic "hammer and anvil" |
| **地形 / Terrain** | 高坡=远程射程+伤害优势、森林=伏击/隐蔽加成、河流=跨河冲锋惩罚 / High ground = ranged range+dmg, forest = ambush/hidden, river crossing = charge penalty |
| **魔法 / Magic** | 战斗中消耗魔法之风施法——范围伤害(Bombardment)、buff/debuff、召唤等 / Use winds of magic in battle — AoE damage, buff/debuff, summons etc |

**锤砧战术**: 步兵线(砧)吸住敌人→骑兵/巨兽(锤)从侧翼或背后冲锋→敌人崩溃。这是全战最经典、最有效的战术 / **Hammer and Anvil**: Infantry (anvil) holds enemy → cavalry/monsters (hammer) charges flank/rear → enemy breaks. The most classic and effective TW tactic.

## 新手易犯错误 / Common Beginner Mistakes

| 错误 / Mistake | 正确做法 / Correct |
|:----:|------|
| **经济崩溃 / Bankrupt** | 不要超出经济能力暴兵——保持每回合金币正收入。留2000+金币应急 / Don't over-recruit — keep per-turn gold positive. Keep 2000+ gold reserve |
| **多线开战 / Multi-front War** | 同时宣战2+敌人=被车轮战耗死。各个击破+外交稳住未交战的派系 / Fighting 2+ enemies = death by attrition. Divide and conquer + diplomacy to pacify others |
| **将领裸奔送死 / Lord Solo** | 传奇领主很强但不是无敌——不要让他单挑满编军队。魔法+守卫保护 / Legendary lords are strong but not invincible — don't 1v20. Use magic and bodyguards |
| **不建城墙 / No Walls** | 边境城市一定要建城墙——AI更喜欢打无墙城市。城墙=多一轮防守时间 / Border cities MUST have walls — AI prioritizes unwalled settlements. Walls = one extra turn to defend |
| **纯堆高级兵 / Full Elite Stack** | 高级兵维护费高——1队满编精英=3队混合部队的战斗力/金币比。杂兵+精英混编更高效 / Elite units cost too much — 1 elite stack = 3 mixed stacks in cost-efficiency. Mix chaff with elites |
