## 部队组成基础 / Army Composition Basics

全面战争战锤3的部队没有固定的"最强配兵"——不同种族、不同领主、不同对手需要完全不同的部队组合。但有一些放之四海而皆准的原则 / TWW3 has no single "best army" — different races, lords, and enemies need different compositions. But some universal principles apply.

### 配兵基本原则 / Core Composition Principles

| 原则 / Principle | 说明 / Description |
|---|---|
| 前排+后排 / Frontline + Backline | 任何军队都需要肉盾和输出——比例约1:1 / Every army needs tanks and DPS — roughly 1:1 ratio |
| 反大+反步 / Anti-Large + Anti-Infantry | 同时准备这两种能力——根据对手调整 / Prepare both — adjust based on enemy |
| 魔法支援 / Magic Support | 至少带一个施法者——魔法是战锤的核心 / Bring at least one caster — magic is Warhammer's core |
| 机动部队 / Mobile Units | 骑兵/飞行单位——处理远程和炮兵 / Cavalry/flyers — deal with ranged and artillery |
| 领主强度 / Lord Strength | 领主是单军团最强的单位——投资升级领主 / Lord is the strongest unit in any army — invest in them |

### 标准部队构成 / Standard Army Template

```
满编20单位 / 20-unit stack:
- 领主 / Lord: 1
- 英雄 / Hero: 1-2 (施法者 + 战士 / Caster + Fighter)
- 前排步兵 / Frontline Infantry: 4-6
- 远程步兵 / Ranged Infantry: 4-6
- 骑兵/怪物 / Cavalry/Monsters: 2-4
- 炮兵 / Artillery: 2-4
- 机动特种 / Special Mobile: 1-2
```

### 魔法使用技巧 / Magic Usage Tips

1. **远程施法者不要靠近前线**——施法距离够远 / Keep ranged casters away from frontline — spells have long range
2. **AOE魔法等敌人聚拢后再放**——让步兵缠住后施法 / Wait for enemy blob before AOE — let infantry engage first
3. **魔法之风很宝贵**——不要在小规模战斗中浪费 / Winds of Magic are precious — don't waste on small skirmishes
4. **群体Buff魔法(如回春/护盾)通常比单体伤害更划算** / Group buff spells (Regrowth/Shield) are often better value than single-target damage

### 省份经济发展策略 / Province Development

1. **每个省份至少有一个金币建筑**——保证基础收入 / Every province needs at least one gold building — base income
2. **主城优先造科技建筑**——科技=/=力量 / Capital: priority on tech buildings — tech = power
3. **边境城市优先造城墙**——防止被偷家 / Border cities: priority on walls — prevent back-caps
4. **不要把资源浪费在内陆安全省份的军事建筑上** / Don't build military buildings in safe interior provinces

## 军队运营 / Army Management

### 多线开战原则 / Multi-Front War Principles

1. **不要同时在两个以上战线开战** / Never fight on more than 2 fronts
2. **每个前线至少2支满编** / Each front needs at least 2 full stacks
3. **防守前线用性价比高的低级兵**——主力进攻用精锐 / Defense front: cost-effective low-tier units — offense: elites
4. **同盟的领土可作为缓冲区**——利用AI的领土防守你的边界 / Ally territory = buffer zone — use AI land to guard your borders

### 损耗管理 / Attrition Management

| 损耗类型 / Attrition Type | 来源 / Source | 应对 / Counter |
|---|---|---|
| 腐蚀 / Corruption | 敌对地区 / Hostile territory | 建筑反腐蚀 / Build anti-corruption |
| 混沌损耗 / Chaos Attrition | 混沌之地 / Chaos Wastes | 混沌抗性装备/技能 / Chaos resist gear/skills |
| 急行军疲劳 / March Attrition | 连续急行军 / Forced march spam | 每2回合急行军后正常走一回合 / Normal move every 2 turns |
| 围攻损耗 / Siege Attrition | 长期包围 / Prolonged siege | 攻城塔/大炮——减少围城时间 / Siege towers/cannons |

## 特殊配兵 / Special Builds

### 无敌坦克流 / Unkillable Tank Build

**要求 / Requirements**：矮人 / Dwarfs + 高护甲领主 + 生命法师

**核心 / Core**：
- 领主全套护甲+生命再生装备 / Lord with max armor + regeneration gear
- 生命法师全程加血 / Life mage healing throughout
- 步兵线全部是重甲长须战士 / Frontline: all Ironbreakers
- 炮兵在后排输出 / Artillery output from back

**战术 / Tactic**：缓慢推进——敌人撞上你的钢铁城墙然后被炮兵打碎 / Slow push — enemy crashes into your steel wall, artillery shreds them.

### 放风筝致死流 / Kite to Death Build

**要求 / Requirements**：高机动远程种族（木精灵/基斯里夫/暗精灵）

**核心 / Core**：
- 全部远程——一个近战都不要 / All ranged — zero melee
- 带2-3个近战英雄用来缠住敌人将领 / 2-3 melee heroes to pin enemy lords
- 飞行单位骚扰敌人炮兵 / Flyers harass enemy artillery

**战术 / Tactic**：敌人永远追不上你——全程被射杀 / Enemy never catches you — shot to death while chasing.

### 怪物派系流 / Monster Mash Build

**要求 / Requirements**：能召唤怪物的种族（野兽人/吸血鬼海岸/鼠人）

**核心 / Core**：
- 全部大型怪物——不要步兵 / All large monsters — no infantry
- 施法者负责增益和迪 buff / Casters for buffs and debuffs
- 1-2个单位拉仇恨——其它单位输出 / 1-2 units tank aggro — others DPS

**战术 / Tactic**：怪物聚团冲垮敌阵——撞散后逐个击杀 / Monster blob crashes enemy line — scatter and kill individually.
