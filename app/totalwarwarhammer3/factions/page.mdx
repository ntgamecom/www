# 派系详解 / Factions Guide

全面战争：战锤3有20+个可玩派系，每个派系有独特的机制、兵种树和玩法风格。选对派系能事半功倍 / Total War: Warhammer 3 has 20+ playable factions, each with unique mechanics, unit rosters, and playstyles. Choosing the right faction makes a huge difference.

---

## 秩序阵营 / Order Factions

### 震旦天朝 / Grand Cathay

| 特性 / Mechanic | 说明 / Description |
|:----:|------|
| **阴阳和谐 / Yin-Yang Harmony** | 建筑和将领技能分阴阳——平衡阴阳获得全局加成(+收入/+补员/+法力等)。失衡则全派系惩罚 / Buildings and lord skills are Yin or Yang — balance them for global buffs (+income/+replenishment/+magic). Unbalance = global penalty |
| **长城屏障 / Great Wall Bastion** | 震旦北部长城——每回合防御混沌入侵。守住长城=巨额收入+补员 / Northern wall defending against Chaos. Hold it = massive income + replenishment |
| **五行罗盘 / Wu Xing Compass** | 选一个方向获得被动效果——+金币/+增长率/+法力/+外交/+补员 / Choose a direction for passive bonus — gold/growth/magic/diplomacy/replenishment |
| **龙裔领主 / Dragon Lords** | 传奇领主是人形/巨龙双形态——巨龙形态下近战+魔法极大增强 / Legendary lords have human/dragon forms — dragon form massively boosts melee + magic |

**兵种特色**: 均衡派系——远程(火箭炮兵/龙弩手)优秀、步兵稳固、骑兵一般。拥有龙(巨兽)和陶俑守卫(超重步) / **Roster**: Balanced — excellent ranged (rocket artillery/crossbows), solid infantry, average cavalry. Has dragons (monsters) and Terracotta Sentinels (super-heavy infantry).

### 基斯里夫 / Kislev

| 特性 / Mechanic | 说明 / Description |
|:----:|------|
| **支持率 / Support** | 通过战斗和建筑积累支持率→到达阈值解锁派系效果——最高解锁冰卫(精英弓手) / Build support via battles and buildings → thresholds unlock faction effects — max unlocks Ice Guard (elite archers) |
| **冰霜魔法 / Ice Magic** | 专属冰霜法术——减速/冰冻敌人。冰法可以通过冰霜阁升级 / Exclusive frost spells — slow/freeze enemies. Ice witches upgrade via Ice Court |
| **母国号召 / The Motherland** | 防御战时+补员+士气——在本土作战时更强大 / +replenishment +morale when defending homeland |
| **虔诚信仰 / Orthodoxy** | 建造教堂获得信仰值→解锁祭司事务官+独特Buff / Build churches for devotion → unlock Patriarch agents + unique buffs |

**兵种特色**: 冰阵射手(射箭时冻结近战敌人)+熊骑兵(破甲冲击)+重甲步兵(沙皇卫队)。冰魔法控场很强 / **Roster**: Streltsi (shoot while freezing melee attackers) + bear cavalry (armor-piercing charge) + heavy infantry (Tzar Guard). Ice magic has excellent crowd control.

## 毁灭阵营 / Destruction Factions

### 恐虐 / Khorne

| 特性 / Mechanic | 说明 / Description |
|:----:|------|
| **血祭血神 / Blood for the Blood God** | 近战杀敌获得"颅骨"→消耗颅骨解锁技能/科技。鼓励持续战斗 / Kill in melee for "skulls" → spend on skills/tech. Rewards constant fighting |
| **战争熔炉 / War Forge** | 通过战斗解锁装备——屠夫越多装备越好 / Unlock gear through battle — more slaughter = better equipment |
| **无法外交 / No Diplomacy** | 不能签订任何和平条约——只能战斗到一方灭亡。纯战狂派系 / Cannot sign peace treaties — only fight until one side dies. Pure warmonger |

**兵种特色**: 全游戏最强近战步兵——狂战士+混沌武士+嗜血狂魔(巨兽)。**完全没有远程**。纯近战冲锋流 / **Roster**: Best melee infantry in game — berserkers + Chaos Warriors + Bloodthirsters (monsters). **Zero ranged units**. Pure melee rush.

### 纳垢 / Nurgle

| 特性 / Mechanic | 说明 / Description |
|:----:|------|
| **瘟疫传播 / Plague** | 创造瘟疫→散播到敌人城市/军队→减员+Debuff。可自定义瘟疫效果 / Create plagues → spread to enemy cities/armies → attrition + debuffs. Customizable plague effects |
| **循环建筑 / Cyclical Buildings** | 建筑随时间自动升级→不需要手动升级。但前期等待时间长 / Buildings auto-upgrade over time — no manual upgrading. But slow early game |
| **枯荣机制 / Cycle of Decay** | 纳垢单位在战斗中缓慢回复HP——越打越强 / Nurgle units slowly regenerate HP in battle — stronger the longer they fight |

**兵种特色**: 全游戏最高血量单位——纳垢灵(便宜杂兵)+纳垢兽(巨兽)+大不净者(超巨兽)。毒属性攻击降低敌人属性 / **Roster**: Highest HP units in game — nurglings (cheap chaff) + beasts of Nurgle (monsters) + Great Unclean Ones (super-monsters). Poison attacks debuff enemies.

## 其他核心派系 / Other Core Factions

| 派系 / Faction | 核心机制 / Core Mechanic | 兵种特色 / Roster Style |
|:----:|:----|:----|
| **奸奇 / Tzeentch** | 魔法之风更强大——法力恢复+法术强度。变道(更改敌军行军路线) / Stronger winds of magic — regen + spell intensity. Change the Paths (redirect enemy armies) | 远程魔法射手+粉妖/蓝妖+飞盘骑兵。全游戏机动性最强的派系之一 / Magical ranged + horrors + disc cavalry. Most mobile faction |
| **色孽 / Slaanesh** | 诱惑(给敌军开Buff→使其叛变)+追求感官(完成任务+属性) / Seduction (buff enemy → seduce units) + Pursuit of Pleasure (quests + stats) | 高速破甲——全游戏最快的骑兵和步兵。但护甲极低——不能被远程打 / Fastest cavalry and infantry in game. But paper armor — must avoid ranged |
| **食人魔王国 / Ogre Kingdoms** | 营地(可移动的招募/经济建筑)+契约(接雇佣任务→金币+好东西) / Camps (mobile recruit/econ) + Contracts (mercenary jobs → gold + rewards) | 巨兽步兵——食人魔公牛+铁胃(远程巨兽)+石角兽(超巨兽)。全派系最强单体步兵 / Monstrous infantry — Ogres + Ironblasters (ranged monstrous) + Stonehorns. Best monstrous infantry |
| **高等精灵 / High Elves** | 影响力(外交资源——可操控其他派系关系)→缔结联盟/制造分裂 / Influence (diplomatic currency — manipulate other factions' relations) | 全游戏最强弓箭手——阿瓦隆姐妹+龙王子骑兵+巨龙。全能派系 / Best archers — Sisters of Avelorn + Dragon Princes + dragons. Jack-of-all-trades |
| **蜥蜴人 / Lizardmen** | 神圣几何(城市星盘Buff)+古圣的意志(全球Buff目标) / Geomantic Web (city network buffs) + Great Plan (global buff objectives) | 恐龙!——棘背龙+暴龙+三角龙+远古巨龙。巨型单位统治战场 / Dinosaurs! — Cold Ones + Carnosaurs + Stegadons + Ancient Dragons. Big units dominate |
| **斯卡文鼠人 / Skaven** | 粮食(食物→人口/军队)+次元石(独特资源)+地下帝国(地表城市下的暗影网络) / Food (population/army) + Warpstone (unique resource) + Under-Empire (shadow network under cities) | 人海+武器小组——鼠人奴隶(最便宜杂兵)+次元炮+瘟疫僧。疯狂科技 / Swarm + weapon teams — Skavenslaves (cheapest chaff) + Warp Lightning Cannon + Plague Monks. Insane tech |
| **吸血鬼海岸 / Vampire Coast** | 海盗契约(打劫)+舰船机制(港口有收益加成)/腐化传播 / Pirate Contracts (raiding) + Ship Mechanics (port bonuses) + Vampiric Corruption | 亡灵火枪手+僵尸+炮船+巨像(巨型亡灵机械)。远程火力流派 / Undead gunners + zombies + cannon ships + Necrofex Colossus (giant undead mech). Ranged firepower |
