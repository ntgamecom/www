# 传奇领主 / Legendary Lords

传奇领主是全面战争：战锤3的核心——每个领主有独特技能树、专属装备和派系机制。选对传奇领主=赢在起跑线 / Legendary Lords are the core of Total War: Warhammer 3 — each has a unique skill tree,专属 gear, and faction mechanics. Choosing the right lord = winning before the campaign starts.

---

## 震旦天朝 / Grand Cathay

### 飙龙·妙影 / Miao Ying, the Storm Dragon

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **人形形态 / Human Form** | 近战+远程(弓)双修——范围雷系法术。优秀领主杀手 / Hybrid melee + ranged (bow) — AoE lightning spells. Excellent lord killer |
| **龙形形态 / Storm Dragon Form** | 变为巨型雷龙——近战+雷AOE——攻城/清兵/巨兽战极强 / Transforms into giant lightning dragon — melee + AoE lightning — amazing in siege/clearing/monster duels |
| **派系加成 / Faction Bonus** | 长城驻军+50%强度、龙裔事务官-25%维持费 / Great Wall garrison +50% strength, Dragon-blooded agents -25% upkeep |
| **专属装备 / Unique Gear** | **天龙戟(Storm Dragon Halberd)**——反大型+窃取生命 / Anti-large + life steal |
| **推荐开局 / Strategy** | 守住长城→统一震旦北部→南下收服元伯/其他龙裔。防御型龟壳玩法 / Hold the wall → unite northern Cathay → confederate Zhao Ming. Defensive turtle playstyle |

### 镔龙·元伯 / Zhao Ming, the Iron Dragon

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **人形形态 / Human Form** | 近战+金属法——破甲+防御Buff。比妙影更硬 / Melee + Lore of Metal — armor-piercing + defense buffs. Tankier than Miao Ying |
| **龙形形态 / Iron Dragon Form** | 变为金属巨龙——高护甲+金属AOE——近战硬度极高 / Transforms into metallic dragon — high armor + metal AoE — extremely durable |
| **派系加成 / Faction Bonus** | 贸易收入+30%、与食人魔外交+50友好(可早期雇佣食人魔兵) / Trade income +30%, +50 Ogre diplomacy (can hire ogre units early) |
| **专属装备 / Unique Gear** | 玄铁甲(Iron Dragon Armor)——+护甲+物理抗性 / +armor +physical resistance |
| **推荐开局 / Strategy** | 经济优先→贸易扩张→统一震旦南部。比妙影更偏经济/扩张型 / Economy first → trade expansion → unite southern Cathay. More economic/expansionist than Miao Ying |

## 基斯里夫 / Kislev

### 冰雪女王·卡捷琳 / Katarin, the Ice Queen

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **战斗风格 / Playstyle** | 冰法师——远程施法型领主。冰风暴+减速控场。**不要冲前排** / Ice mage — ranged caster. Ice storm + slow. **Don't frontline** |
| **派系加成 / Faction Bonus** | 冰霜阁法师+2等级、冰卫维持费-20% / Ice Court mages +2 levels, Ice Guard upkeep -20% |
| **专属装备 / Unique Gear** | 冰霜法杖(Frost Staff)——+冰法强度+法力回复 / +ice magic strength + mana regen |
| **推荐开局 / Strategy** | 统一基斯里夫中部→建立防御线→对抗北方混沌。前期压力大——建议优先建墙 + 培养冰法部队 / Unite central Kislev → build defense line → fight northern Chaos. High early pressure — prioritize walls + Ice Guard |

### 大正教会牧首·康斯坦丁 / Kostaltyn

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **战斗风格 / Playstyle** | 近战领主+支持Buff——带领军队冲锋。比卡捷琳更能抗线 / Melee lord + support buffs — leads from the front. Tankier than Katarin |
| **派系加成 / Faction Bonus** | 牧师+2等级、教堂收入+30%、熊骑兵维持费-15% / Patriarchs +2 levels, church income +30%, bear cavalry upkeep -15% |
| **专属装备 / Unique Gear** | 圣熊之爪(Sacred Bear Claws)——+破甲+冲锋加成 / +armor-piercing +charge bonus |
| **推荐开局 / Strategy** | 和卡捷琳竞争支持率→统一基斯里夫。更多依靠熊骑兵和牧师Buff打正面战 / Compete with Katarin for support → unite Kislev. Relies on bear cavalry + Patriarch buffs for frontline battles |

## 混沌四神 / Chaos Gods

### 恐虐·放血鬼·斯卡布兰德 / Skarbrand

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **战斗风格 / Playstyle** | **全游戏最强近战单体**——冲锋→击杀→冲锋循环。没有远程、没有魔法、纯物理碾压 / **Strongest melee lord in game** — charge → kill → repeat. No ranged, no magic, pure physical dominance |
| **派系加成 / Faction Bonus** | 战斗收入+50%(战后金钱翻倍)、军队移动力+10%(战斗后) / Battle income +50%, post-battle army movement +10% |
| **独有机制 / Unique Mechanic** | 不能使用魔法——但杀敌回血+每场战斗额外移动。纯战狂玩法 / No magic — but heal on kill + extra post-battle movement. Pure slaughter |
| **推荐开局 / Strategy** | 一路向南杀向震旦/高等精灵领地。不要停战、不要防守——一直进攻 / March south toward Cathay/High Elves. Never stop, never defend — only attack |

### 纳垢·慈父·库嘎斯 / Ku'gath Plaguefather

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **战斗风格 / Playstyle** | 巨兽型施法领主——近战强+纳垢法(回血+毒+召唤)。**非常硬** / Monster caster lord — strong melee + Lore of Nurgle (heal+poison+summon). **Extremely tanky** |
| **派系加成 / Faction Bonus** | 纳垢灵维持费-30%、瘟疫效果+20%强度 / Nurglings upkeep -30%, plague effects +20% |
| **独有机制 / Unique Mechanic** | 可以酿造独特瘟疫配方→自定义瘟疫传播给敌人 / Brew unique plague recipes → customize plagues to spread to enemies |
| **推荐开局 / Strategy** | 占据混沌荒原→缓慢扩张→用瘟疫削弱敌人再进攻。**纳垢前期节奏慢**——请耐心 / Take Chaos Wastes → slow expansion → weaken enemies with plague before attacking. **Nurgle early game is slow** — be patient |

### 奸奇·万变之主·卡洛斯 / Kairos Fateweaver

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **战斗风格 / Playstyle** | **全游戏最强法师**——掌握全部奸奇法术(全游戏最全面的法术派系)+两个头=双倍视野 / **Strongest wizard in game** — full Lore of Tzeentch (most versatile lore) + two heads = double map视野 |
| **派系加成 / Faction Bonus** | 魔法之风上限+30、法力恢复速度+25%、蓝妖/粉妖维持费-25% / Winds of magic reserve +30, mana regen +25%, horrors upkeep -25% |
| **独有机制 / Unique Mechanic** | "变道"(强制改变敌军行军方向)+预知(选择战斗中获得的Buff) / "Change the Paths" (redirect enemy armies) + Foretelling (choose battle buffs) |
| **推荐开局 / Strategy** | 占据南极→通过变道机制操控其他派系→用无穷无尽法力碾压敌人 / Take South Pole → manipulate factions via Change the Paths → crush enemies with infinite magic |

### 色孽·愉悦王子·纳卡里 / N'Kari

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **战斗风格 / Playstyle** | 极速近战领主——冲锋+破甲+机动性全游戏第一。**不能站桩——打完就跑** / Fastest melee lord — charge + AP + best mobility. **Cannot stand still — hit and run** |
| **派系加成 / Faction Bonus** | 诱惑机制(可以策反敌军单位)——骑兵+10%冲锋加成 / Seduction mechanic (can convert enemy units) — cavalry +10% charge bonus |
| **独有机制 / Unique Mechanic** | 色孽追求狂潮——完成任务链获得属性+神器+Buff / Slaanesh's Pursuit of Pleasure — quest chains for stats + artifacts + buffs |
| **推荐开局 / Strategy** | 目标是高等精灵领土(奥苏安)。诱惑高等精灵部队为你而战——以其人之道还治其人之身 / Target Ulthuan (High Elf homeland). Seduce High Elf units to fight for you — turn their own roster against them |
