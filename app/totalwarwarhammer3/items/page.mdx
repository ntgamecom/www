## 装备系统概述 / Equipment System

全面战争战锤3的装备系统分为**武器**、**护甲**、**附身符**和**魔法物品**四大类。装备可提升领主/英雄的属性和技能——或者提供特殊能力 / TWW3's equipment system is divided into **Weapons**, **Armor**, **Talismans**, and **Enchanted Items**. Equipment boosts lord/hero stats and skills — or provides special abilities.

### 装备品质 / Equipment Quality

| 品质 / Quality | 颜色 / Color | 获得方式 / How to Get |
|---|---|---|
| 基础 / Basic | 白色 / White | 战斗掉落 / Battle loot |

| 槽位 / Slot | 可装备 / Equippable | 说明 / Description |
|---|---|---|

| 坐骑坐骑 / Mount | 特定领主 / Specific lords | 改变单位类型和属性 / Changes unit type and stats |

## 神器刷取指南 / Artefact Farming Guide

### 战斗掉落机制 / Battle Loot Mechanics

| 战斗类型 / Battle Type | 掉落品质 / Loot Quality | 备注 / Note |
|---|---|---|
| 普通战斗 / Normal battle | 白色/绿色 / White/Green | — |
| 英雄行动 / Hero Action | 绿色/蓝色 / Green/Blue | 刺杀/破坏成功率影响品质 / Assassination/Sabotage success affects quality |
| 攻城战 / Siege battle | 绿色/蓝色 / Green/Blue | 胜利后额外掉落 / Bonus loot on win |
| 神器任务 / Quest battle | 紫色/金色 / Purple/Gold | 固定掉落——唯一 / Fixed drop — unique |
| 终局Boss战 / Endboss battle | 金色 / Gold | 传奇物品 / Legendary items |

### 刷取效率建议 / Farming Efficiency Tips

1. **低阶物品不需要专门刷**——战斗自然掉落就够用 / Low-tier items don't need targeted farming — natural drops suffice
2. **蓝色以上物品靠神器任务**——每完成一个领主的神器任务可获得专属装备 / Blue+ items come from quest battles — complete lord quests for unique gear
3. **让英雄在地图上执行行动**——可以随机获得装备 / Send heroes on map actions — random equipment drops
4. **击败敌方法师领主**——有更高概率掉落魔法物品 / Defeating enemy wizard lords — higher magic item drop rate

## 神器任务指南 / Quest Battle Guide

### 卡尔·弗兰兹 — 盖尔·玛拉兹任务 / Karl Franz — Ghal Maraz Quest

**解锁等级 / Unlock Level**：15

**任务目标 / Objective**：击败叛变的帝国领主和他们的混沌同盟 / Defeat rebel Empire lords and their Chaos allies.

**推荐配置 / Recommended Setup**：
- 带2-3个坦克——对抗混沌巨人和骑兵 / Bring 2-3 Steam Tanks — counter Chaos Giants and cavalry
- 火枪手4队——躲在坦克后面输出 / 4 Handgunners — shoot from behind tanks
- 光明法师——光网控住敌人骑兵 / Light wizard — Net enemy cavalry

**奖励 / Reward**：盖尔·玛拉兹——+20近战攻击 + 主动技能"毁灭打击" / Ghal Maraz — +20 MA + active "Destroy" ability

### 泰瑞昂 — 太阳之盾任务 / Tyrion — Sun Shield Quest

**解锁等级 / Unlock Level**：12

**任务目标 / Objective**：在奥苏安的森林中击败暗精灵入侵部队 / Defeat Dark Elf invaders in the forests of Ulthuan.

**推荐配置 / Recommended Setup**：
- 凤凰守卫——可挡住暗精灵黑卫 / Phoenix Guard — counter Black Guard
- 弓箭手——射爆暗精灵远程 / Archers — outrange Dark Elf ranged
- 生命法师——回血续航 / Life mage — sustain healing

**奖励 / Reward**：太阳之盾——+20护甲 + 30%火抗 + 主动技能"太阳耀斑" / Sun Shield — +20 armor + 30% fire resist + active "Solar Flare"
