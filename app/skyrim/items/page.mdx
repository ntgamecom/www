| 小麦 + 蓝山花 / Wheat + Blue Mountain Flower | 生命恢复 + 强化生命 / Restore Health + Fortify Health | 中高 / Mid-high |

### 炼金100速刷 / Fast Alchemy Leveling

1. 采集大量蓝山花 + 蝴蝶翅膀(或金光菊+巨魔苔) / Gather Blue Mountain Flower + Butterfly Wing
2. 在每个城镇的炼金台批量生产 / Bulk craft at any alchemy table
3. 把药水卖给商人 → 买空他们的原料 → 继续造 / Sell to vendors → buy their ingredients → repeat
4. 炼金100后——配合附魔100——可以做出强化锻造+强化附魔的药水/装备循环 / Level 100 alchemy + enchanting → Fortify Smithing + Fortify Enchanting loop

### 炼金锻造附魔循环 / Alchemy-Enchanting-Smithing Loop

**天际最强的装备强化循环**——让你的装备几乎无敌 / The most OP gear loop in Skyrim — become nearly invincible.

| 步骤 / Step | 操作 / Action | 结果 / Result |
|---|---|---|
| 1 | 炼金做出"强化附魔"药水 / Brew "Fortify Enchanting" potion | 附魔强度+25% / +25% enchantment power |
| 2 | 喝药后附魔4件装备"强化炼金" / Enchant "Fortify Alchemy" on 4 gear pieces | 每次附魔增加炼金效果 / Each piece boosts alchemy |
| 3 | 穿上附魔装备→再做更强附魔药水 / Wear alchemy gear → brew stronger potion | 强度再+10% / +10% more power |
| 4 | 重复1-3→直到效果上限 / Repeat until cap | 最终附魔+炼金+锻造全最大化 / Max all three |
| 5 | 最后做"强化锻造"药水+锻造装备 / Brew "Fortify Smithing" + improve gear | **单件护甲1000+，武器伤害500+** |
