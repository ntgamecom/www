### 屠龙策略 / Dragon Slaying Strategies

| 策略 / Strategy | 适合流派 / Build | 说明 / Description |
|---|---|---|
| **弓箭远程 / Ranged Archery** | 潜行弓 / Stealth Archer | 保持距离——射弱点——龙落地前可射4-5箭 / Keep distance — shoot between attacks |
| **马战 / Mounted Combat** | 任何 / Any | 骑召唤的马可无限重击——不消耗耐力 / Summoned horse — infinite power attacks |
| **元素抗性 / Elemental Resist** | 任何 / Any | 龙的大部分伤害来自吐息——堆抗性到85% / Most dragon damage is breath — stack resist to 85% |
| **龙吼控制 / Shout Control** | 任意 / Any | 不卸之力打断吐息——减速吼提供射窗 / Unrelenting Force interrupts breath |
| **近战锁头 / Melee Headlock** | 双手/双持 / 2H/Dual | 龙落地后打头——巨龙倒地时暴击 / Hit head when dragon lands — crit while stunned |
| **法师风筝 / Mage Kite** | 毁灭法 / Destruction | 冲击Perk无限打断——保持移动 / Impact perk infinite stun — keep moving |

**抗性堆叠 / Resistance Stacking**:
- 火焰抗性上限: 85% (诺德50% + 附魔/药水35%) / Fire resist cap: 85%
- 寒霜抗性上限: 85% (黑暗精灵50% + 附魔/药水35%) / Frost resist cap: 85%
- 魔法抗性上限: 85% (布莱顿25% + 魔神祝福+附魔) / Magic resist cap: 85%

### 米拉克详细攻略 / Miraak Detailed Guide

**位置 / Location**: 索瑟姆众神殿 / Apocrypha, Solstheim
**推荐等级 / Recommended**: 50+
**阶段 / Phases**: 3阶段——会吸龙魂恢复生命 / 3 phases — steals dragon souls to heal

**阶段1攻击模式 / Phase 1 Attack Pattern**:
| 攻击 / Attack | 应对 / Counter |
|---|---|
| 冰刺法术 / Ice Spike | 横向移动躲避 / Strafe sideways |
| 龙吼-不卸之力 / Shout-Unrelenting | 保持距离——被推后退再接近 / Keep range — re-approach after stagger |
| 召唤魔人 / Summon Dremora | 快速击杀——否则数量优势 / Kill quickly — avoid being outnumbered |

**阶段2攻击模式 / Phase 2 Attack Pattern**:
| 攻击 / Attack | 应对 / Counter |
|---|---|
| 闪电风暴 / Lightning Storm | 躲到柱子后面 / Hide behind pillars |
| 龙吼-旋风精力 / Shout-WW Sprint | 他位移——改变攻击方向 / He repositions |
| 召唤龙(仅一次) / Summon Dragon | 龙会提供龙魂用于恢复 / Dragon gives soul to heal |

**阶段3攻击模式 / Phase 3 Attack Pattern**:
| 攻击 / Attack | 应对 / Counter |
|---|---|
| 所有攻击组合 / All attacks combined | 保持机动——用恢复 / Stay mobile — use restoration |
| 龙魂吸收 / Soul Absorb | 无法阻止——等他吸完再输出 / Can't stop — wait and re-engage |
| 最终必杀 / Final Mechanism | 击败后他的灵魂被赫玛·莫拉吞噬 / His soul claimed by Hermaeus Mora |
