# 种族与技能 / Races & Skills

## 技能树总览 / Skill Trees

天际的18项技能分为三个类别——每个技能有Perk树 / 18 skills in 3 categories — each with a perk tree.

### 战士系 / Combat (6)

| 技能 / Skill | 核心Perk / Key Perk | 说明 / Description |
|---|---|---|
| **单手 / One-Handed** | 剑/斧/锤精通 + 双持乱舞 | 剑=暴击, 斧=流血, 锤=无视护甲 / Sword=crit, Axe=bleed, Mace=armor pen |
| **双手 / Two-Handed** | 横扫 + 强力暴击 | 范围攻击——大剑/战斧/战锤 / AoE swings — greatsword/battleaxe/warhammer |
| **弓箭 / Archery** | 稳扎稳打 + 猎人纪律 | 远距狙杀——潜行弓是最强流派之一 / Range sniping — stealth archer is the most OP build |
| **格挡 / Block** | 护盾之墙 + 元素防护 | 盾牌减伤+盾击——元素防护挡法术 / Shield dmg reduction + bash — Elemental Protection for spells |
| **锻造 / Smithing** | 秘制 + 全能锻造 | 制造/强化武器护甲——可以升级到传奇品质 / Craft/improve gear — upgrade to Legendary quality |
| **重甲 / Heavy Armor** | 铁拳 + 反射伤害 | 高防御——铁拳让护甲拳击造成护甲值伤害 / High defense — Fists of Steel adds gauntlet armor to punches |

### 法系 / Magic (6)

| 技能 / Skill | 核心Perk / Key Perk | 说明 / Description |
|---|---|---|
| **毁灭 / Destruction** | 双重施法 + 冲击 | 元素伤害法术——冲击perk可以打断施法 / Elemental dmg spells — Impact perk staggers enemies |
| **恢复 / Restoration** | 双重施法 + 恢复精通 | 治疗+防护法术——亡灵法术对不死族造成伤害 / Heal + ward spells — turns undead |
| **变化 / Alteration** | 魔法护甲 + 魔法吸收 | 橡木/铁/石皮术提升护甲——魔法吸收回蓝 / Flesh spells boost armor — Magic Absorption refunds MP |
| **幻术 / Illusion** | 寂静施法 + 幻术精通 | 平静/恐惧/狂暴法术——影响敌人精神状态 / Calm, fear, frenzy spells — manipulate enemy AI |
| **召唤 / Conjuration** | 双魂 + 元素之力 | 召唤生物+亡灵——灵魂陷阱填充灵魂石 / Summon atronachs + undead — Soul Trap fills soul gems |
| **附魔 / Enchanting** | 附魔精通 + 双附魔 | 为装备附魔——可以给一件装备附两个效果 / Enchant gear — put 2 effects on one item |

### 潜行系 / Stealth (6)

| 技能 / Skill | 核心Perk / Key Perk | 说明 / Description |
|---|---|---|
| **潜行 / Sneak** | 潜行密探 + 暗影战士 | 潜行中更难被发现——暗杀15倍(匕首) / Harder to detect — 15x sneak attack (dagger) |
| **开锁 / Lockpicking** | 开锁大师 + 速行者 | 更容易开锁+不会被破锁器卡住 / Easier locks + picks won't break |
| **偷窃 / Pickpocket** | 夜盗 + 金手指 | 更容易偷东西+可以偷装备 / Easier pickpocket + steal equipped items |
| **轻甲 / Light Armor** | 敏捷防御 + 无声移动 | 耐力恢复加速+移动无声 / Stamina regen + silent movement |
| **炼金 / Alchemy** | 炼金精通 + 毒剂师 | 制作药水和毒药——天际最强的印钞机 / Make potions and poisons — best money maker |
| **演讲 / Speech** | 商人 + 说服 | 买卖价格更好+可以贿赂守卫 / Better prices + bribe guards |

## 最强Build推荐 / Top Builds

### 1. 潜行弓箭手 / Stealth Archer

**最简单的强力Build**——天际经典梗 / The easiest OP build — the iconic Skyrim meme.

| 项目 / Aspect | 推荐 / Choice |
|---|---|
| **种族 / Race** | 木精灵 / Bosmer (弓箭开局+2) 或 虎人/Khajiit (潜行+2) |
| **主要技能 / Primary** | 弓箭、潜行、锻造、附魔、炼金、轻甲 |
| **关键Perk** | 潜行密探(潜行)、稳扎稳打(弓箭)、猎人纪律(弓箭) |
| **护甲 / Armor** | 轻甲——夜莺套或龙鳞甲 / Light — Nightingale or Dragonscale |
| **武器 / Weapon** | 魔族弓 / Daedric Bow (最好), 奥瑞埃尔之弓 / Auriel's Bow (DLC) |
| **核心打法** | 潜行→爆头→被发现→跑→再潜行→爆头 / Sneak→headshot→run→re-sneak→headshot |

**潜行弓升级路径**:
1. 初期(1-10): 溪木镇的Faendal可以教你弓箭(免费——让他当随从然后偷钱回来) / Faendal trains for free (follower + steal gold back)
2. 中期(10-30): 暗影之石(潜行时隐形) + 夜莺套/古老暗影套 / Shadow Stone + Nightingale/Ancient Shroud
3. 后期(30+): 双附魔弓(火伤+吸魂) + 潜行100 → 暗影战士Perk(战斗中蹲下隐身) / Dual-enchanted bow + Shadow Warrior perk

### 2. 双持狂战士 / Dual-Wield Berserker

**最高DPS的Build**——双持武器输出爆炸 / Highest DPS build — insane dual-wield output.

| 项目 / Aspect | 推荐 / Choice |
|---|---|
| **种族 / Race** | 兽人 / Orsimer (狂战士种族技能+双倍伤害) |
| **主要技能 / Primary** | 单手、格挡(只需要左手有盾或不带)、重甲、锻造、附魔 |
| **关键Perk** | 双持乱舞(攻速翻倍)、狂风打击(双持重击) / Dual Flurry (speed), Dual Savagery (power attack) |
| **护甲 / Armor** | 魔族甲 / Daedric Armor |
| **武器 / Weapon** | 双持魔族剑——附魔: 吸血+火伤 / Dual Daedric swords — Absorb Health + Fire |

**双持核心机制**:
- 双持时右键=左手攻击，左键=右手攻击，同时按=双持攻击 / Left+Right click = dual-wield attack
- 双持攻击的攻速极快——配合吸血附魔你就是不死之身 / Dual-wield attack speed is insane with Absorb Health

### 3. 毁灭法师 / Destruction Mage

**全屏AOE清场**——双手毁灭法术的极致 / Full-screen AoE — peak destruction magic.

| 项目 / Aspect | 推荐 / Choice |
|---|---|
| **种族 / Race** | 高精灵 / Altmer (+50法力起手) 或 暗精灵/Dunmer (毁灭+2, 火抗50%) |
| **主要技能 / Primary** | 毁灭、恢复、变化、召唤、附魔 |
| **关键Perk** | 双重施法(双手施放加强法术)、冲击(施法打断) / Dual Casting, Impact (stagger on dual cast) |
| **护甲 / Armor** | 法袍——最好是玛格努斯之杖+大法师长袍 / Robes — Archmage's Robes + Staff of Magnus |
| **法术 / Spells** | 焚化(火)、冰风暴(冰)、闪电风暴(电) / Incinerate (fire), Icy Spear (frost), Lightning Storm (shock) |

**法师核心提示 / Mage Tips**:
1. 冲击Perk是毁灭法师的核心——双重施法可以把任何敌人强制打断/击退 / Impact is essential — dual-cast staggers any enemy
2. 变化系学了"魔法护甲"三perk后+石皮术=300+护甲 / Alteration armor perks + Ebonyflesh = 300+ armor
3. 恢复系推荐"死者复苏"——回蓝的同时恢复生命 / Restoration perks help with sustain
4. 三元素中闪电最快(瞬发)——火焰持续伤害——冰霜减速 / Lightning is instant, Fire is DoT, Frost slows

### 4. 幻影刺客 / Shadow Mage

**将幻术与潜行结合的终极控制玩法** / Ultimate control build — Illusion + Stealth.

| 项目 / Aspect | 推荐 / Choice |
|---|---|
| **种族 / Race** | 布莱顿 / Breton (魔抗+25%) |
| **主要技能 / Primary** | 幻术、潜行、单手(匕首)、轻甲、炼金 |
| **关键Perk** | 寂静施法(潜行中无声施法)、暗影战士(战斗中蹲下隐身) / Quiet Casting, Shadow Warrior |
| **武器 / Weapon** | 魔族匕首——附魔: 吸血+灵魂陷阱 / Daedric Dagger — Absorb Health + Soul Trap |

**核心打法**:
1. 用狂暴术让敌人内斗——然后用恐惧术让幸存者逃跑 / Cast Frenzy on groups — then Fear the survivors
2. 逐一潜行刺杀死剩最后一个 / Stealth-kill the last one standing
3. 如果被发现——用平静术让敌人冷静——然后走到背后致命一击 / If spotted: Calm spell → walk behind → backstab

## 技能升级技巧 / Skill Leveling Tips

| 技能 / Skill | 快速升级方法 / Fast Leveling |
|---|---|
| **锻造 / Smithing** | 制作大量铁匕首(1-30) → 矮人弓(30-70) → 魔族/龙甲(70-100) / Iron daggers → Dwarven bows → Daedric/Dragon |
| **附魔 / Enchanting** | 制作大量铁匕首后附魔"驱散"——最值钱/经验最多 / Enchant crafted gear with "Banish" — most XP and value |
| **炼金 / Alchemy** | 蓝山花+蝴蝶翅膀(最贵配方) / Blue Mountain Flower + Butterfly Wing (most valuable) |
| **潜行 / Sneak** | 躲在灰胡子的背后一直潜行(挂机10分钟到100) / Hide behind Greybeards and sneak — 10 min to 100 |
| **幻术 / Illusion** | 在雪漫市场区放"轻声术" / Cast "Muffle" in Whiterun market |
| **毁灭 / Destruction** | 打自己的马(回蓝后继续打——但别打死) / Attack your horse (don't kill it) |
| **恢复 / Restoration** | 站在火焰/陷阱上对自己回血 / Stand in fire/trap and heal yourself |
| **演讲 / Speech** | 大量买卖——炼金卖药水 / Mass buy/sell — sell potions from alchemy |
