# 流派构建 / Builds Guide

天际的开放技能系统允许玩家创造任何流派——但专精6-8项技能才能达到最强 / Skyrim's open skill system allows any build — but specializing in 6-8 skills is necessary for endgame power.

---

## 流派分类 / Build Categories

| 大类 / Category | 子类 / Subtypes | 核心技能 / Core Skills | 难度 / Difficulty |
|---|---|---|---|
| **战士 / Warrior** | 剑盾、双手、双持 | 单手/双手 + 重甲/轻甲 + 格挡 + 锻造 | 简单 / Easy |
| **法师 / Mage** | 纯法、战斗法、召唤法 | 毁灭/召唤/恢复 + 变化 + 附魔 | 中等 / Medium |
| **盗贼 / Thief** | 潜行弓、刺客、盗贼 | 潜行 + 弓/单手 + 轻甲 + 炼金 | 简单~中等 / Easy-Medium |
| **混合 / Hybrid** | 魔剑士、夜刃、战斗法师 | 混合以上 / Hybrid of above | 困难 / Hard |

## 战士流派 / Warrior Builds

### 剑盾战士 / Sword and Board

| 项目 / Aspect | 推荐 / Recommendation |
|---|---|
| **种族 / Race** | 诺德(Nord) 或 帝国(Imperial) |
| **主要技能 / Main Skills** | 单手(100) + 格挡(100) + 重甲(100) + 锻造(100) |
| **辅助技能 / Support Skills** | 恢复(50) + 附魔(100) |
| **护甲 / Armor** | 魔族重甲 / Daedric Heavy Armor |
| **武器 / Weapon** | 魔族单手剑 / Daedric Sword |
| **盾牌 / Shield** | 魔族盾牌 / Daedric Shield |
| **站立之石 / Standing Stone** | 战士之石 / Warrior Stone |

**Perk优先级 / Perk Priority**:
1. 单手：剑精通5/5 + 暴击 + 流血 / Armsman 5/5 + Critical Charge + Savage Strike
2. 格挡：护盾之墙5/5 + 盾牌冲锋 + 元素防护 / Shield Wall 5/5 + Shield Charge + Elemental Protection
3. 重甲：铁拳 + 反射伤害 + 条件反射 / Fists of Steel + Reflect Blows + Conditioning
4. 锻造：全能锻造 + 秘制 + 龙甲锻造 / Arcane Blacksmith + Daedric Smithing + Dragon Armor

**核心打法 / Core Playstyle**:
利用盾牌格挡减少伤害——盾击打断敌人——单手剑快速输出 / Block with shield — bash to interrupt — fast 1H attacks.

**优缺点 / Pros & Cons**:
| 优点 / Pros | 缺点 / Cons |
|---|---|
| 生存能力极高 / Extremely tanky | 输出不如双手或双持 / Lower DPS than 2H or dual |
| 适合新手 / Beginner-friendly | 清群怪速度较慢 / Slow crowd clearing |
| 几乎不耗血瓶 / Almost no health potions needed | 对法师需要特别策略 / Need special tactics for mages |

### 双持狂战 / Dual-Wield Berserker

| 项目 / Aspect | 推荐 / Recommendation |
|---|---|
| **种族 / Race** | 兽人(Orsimer) —— 狂暴能力完美配合双持 / Berserker Rage pairs perfectly |
| **主要技能 / Main Skills** | 单手(100) + 轻甲(100) + 锻造(100) + 附魔(100) |
| **辅助技能 / Support Skills** | 格挡(仅双持闪避Perk) + 炼金(50) |
| **武器 / Weapons** | 双持魔族剑 / Dual Daedric Swords |
| **护甲 / Armor** | 龙鳞轻甲 / Dragonscale Light Armor |
| **站立之石 / Standing Stone** | 恋人之石 / Lover Stone |

**Perk优先级 / Perk Priority**:
1. 单手：双持乱舞3/3 + 双持蛮力 + 毁灭打击 / Dual Flurry 3/3 + Dual Savagery + Devastating Strike
2. 轻甲：敏捷移动 + 轻甲强化 + 无声移动 / Agile Defender + Custom Fit + Silence
3. 锻造：秘制 + 魔族锻造 / Arcane Blacksmith + Daedric Smithing

**核心打法 / Core Playstyle**:
双持时双倍攻击速度——左右键交替攻击——瞬间打出巨额伤害 / Dual-wield double attack speed — alternate left/right — insane burst damage.

## 法师流派 / Mage Builds

### 纯毁灭法 / Pure Destruction Mage

| 项目 / Aspect | 推荐 / Recommendation |
|---|---|
| **种族 / Race** | 高精灵(Altmer) —— 额外50法力 / +50 Magicka |
| **主要技能 / Main Skills** | 毁灭(100) + 恢复(100) + 变化(100) + 附魔(100) |
| **辅助技能 / Support Skills** | 召唤(50) + 炼金(50) |
| **护甲 / Armor** | 无甲——变化术护甲 / Unarmored — flesh spells |
| **站立之石 / Standing Stone** | 法师之石 / Mage Stone |

**Perk优先级 / Perk Priority**:
1. 毁灭：双持施法 + 冲击 + 精通毁灭/每系 / Dual Casting + Impact + Augmented Flames/Frost/Shock
2. 变化：魔法护甲3/3 + 魔法吸收 / Mage Armor 3/3 + Magic Absorption
3. 恢复：恢复精通 + 双持恢复 / Regeneration + Dual Casting Restoration
4. 附魔：附魔师5/5 + 双倍附魔 / Enchanter 5/5 + Extra Effect

**核心打法 / Core Playstyle**:
双持毁灭法术——冲击Perk每次施法打断敌人——保持距离风筝 / Dual-cast destruction — Impact staggers on every cast — kite at range.

### 召唤法 / Conjurer Summoner

| 项目 / Aspect | 推荐 / Recommendation |
|---|---|
| **种族 / Race** | 布莱顿(Breton) —— 25%魔抗保生存 / 25% Magic Resist |
| **主要技能 / Main Skills** | 召唤(100) + 毁灭(50) + 变化(50) + 附魔(100) |
| **核心法术 / Key Spells** | 召唤魔人大君 / Conjure Dremora Lord |
| **站立之石 / Standing Stone** | 法师之石 / Mage Stone |

**Perk优先级 / Perk Priority**:
1. 召唤：双魂 + 元素之力 + 亡灵束缚 / Twin Souls + Atromancy + Necromancy
2. 毁灭：仅火焰系——补刀用 / Fire branch only — for finishing
3. 附魔：减少召唤法力消耗 / Reduce conjuration cost to 0

**核心打法 / Core Playstyle**:
同时召唤两个魔人大君——自己躲在后面扔火球 / Summon two Dremora Lords — hide behind them and fireball.

## 盗贼流派 / Thief Builds

### 潜行弓 / Stealth Archer (最强流派)

| 项目 / Aspect | 推荐 / Recommendation |
|---|---|
| **种族 / Race** | 木精灵(Bosmer) —— 弓箭初始加成 / Starting Archery bonus |
| **主要技能 / Main Skills** | 潜行(100) + 弓箭(100) + 轻甲(100) + 炼金(100) |
| **辅助技能 / Support Skills** | 幻术(50-幻影) + 锻造(100) |
| **武器 / Weapon** | 龙骨弓 + 魔族匕首 / Dragonbone Bow + Daedric Dagger |
| **站立之石 / Standing Stone** | 暗影之石 / Shadow Stone |

**潜行弓倍率伤害 / Stealth Archer Damage Multiplier**:
| 潜行状态 / Stealth State | 弓 / Bow | 匕首 / Dagger |
|---|---|---|
| 未发现 / Undetected | 3x (5x 暗影战士Perk) / 5x with Deadly Aim | 15x (30x 暗影斗篷) / 30x with Assassin's Blade |
| 战斗后重新潜行 / Re-enter stealth | 3x | 15x |
| 暗影斗篷潜行 / Shadow Cloak | 3.5x | 15x |

**核心打法 / Core Playstyle**:
在敌人未发现时远程爆头——三倍伤害——大部分普通敌人一箭带走 / Snipe from afar while undetected — 3x damage — most enemies die in one arrow.

## 混合流派 / Hybrid Builds

### 魔剑士 / Spellsword

| 项目 / Aspect | 推荐 / Recommendation |
|---|---|
| **种族 / Race** | 暗精灵(Dunmer) —— 初始毁灭+单手加成 / Starting Destruction + 1H |
| **主要技能 / Main Skills** | 单手(100) + 毁灭(75) + 恢复(75) + 轻甲(100) |
| **核心打法 / Playstyle** | 左手法术——右手剑——保持距离 / Spell in left hand — sword in right — control distance |

**推荐法术组合 / Recommended Spell Combos**:
1. 左手：石甲术 / Flesh Spell + 右手剑 / Sword = 常规战斗 / Standard combat
2. 左手：火焰斗篷 / Flame Cloak + 右手剑 / Sword = 近战Fire AOE
3. 左手：治愈术 / Healing + 右手剑 / Sword = 战时治疗 / Combat healing
4. 左手：麻痹术 / Paralyze + 右手剑 / Sword = 控杀单个强敌 / Control single strong enemy
