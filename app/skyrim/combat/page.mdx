# 战斗系统 / Combat System

天际的战斗系统分为近战、远程和魔法三大流派——每种都有独特的战斗节奏 / Skyrim's combat divides into melee, ranged, and magic — each with unique pacing.

---

## 近战系统 / Melee Combat

### 武器类型 / Weapon Types

| 武器 / Weapon | 特点 / Traits | 每秒伤害 / DPS | 攻击速度 / Speed |
|---|---|---|---|
| **单手剑 / One-Handed Sword** | 速度均衡——暴击加成 / Balanced — crit bonus | 中等 / Medium | 快 / Fast |
| **单手斧 / One-Handed Axe** | 流血效果——持续伤害 / Bleed effect — DoT | 中等 / Medium | 中等 / Medium |
| **单手锤 / One-Handed Mace** | 无视护甲——对重甲有效 / Armor ignore — good vs heavy | 中等偏低 / Medium-low | 慢 / Slow |
| **双手剑 / Greatsword** | 横扫范围攻击——可打多个 / Sweep attacks — hits multiple | 高 / High | 中等 / Medium |
| **双手战斧 / Battleaxe** | 高伤害+流血 / High damage + bleed | 很高 / Very High | 慢 / Slow |
| **双手战锤 / Warhammer** | 极高伤害+破甲 / Very high damage + armor ignore | 极高 / Very high | 极慢 / Very Slow |
| **匕首 / Dagger** | 潜行背刺15倍伤害 / 15x sneak attack | 低(正面) / Low (frontal) | 极快 / Very Fast |

**最佳近战策略 / Best Melee Strategy**:
1. 剑盾组合是最稳健的打法——防御+低体力消耗 / Sword + shield is the safest — block + low stamina use
2. 双持武器爆发力最高——但无防御手段 / Dual-wielding has highest burst — but no block
3. 双手武器适合清理群敌——横扫Perk是核心 / Two-handed weapons excel at crowd clearing — Sweep perk is key

### 格挡系统 / Block System

| 操作 / Action | 效果 / Effect | 说明 / Description |
|---|---|---|
| 右键按住 / Hold Right-Click | 格挡 / Block | 减少正面伤害 / Reduce frontal damage |
| 格挡时攻击 / Attack While Blocking | 盾击 / Bash | 打断敌人攻击 / Interrupt enemy attacks |
| 完美时机格挡 / Timed Block | 弹反 / Parry | 敌人硬直——可反击 / Enemy stagger — counter-attack |
| 快速格挡 / Quick Block | 紧急防御 / Emergency block | 双持时唯一的防御手段 / Only defense for dual-wield |

**格挡Perk关键节点 / Key Block Perks**:
- 护盾之墙 / Shield Wall: 每级提升20%格挡效果 / +20% block effectiveness per rank
- 快速退敌 / Quick Reflexes: 时间减慢——易格挡 / Slow time for easier blocking
- 元素防护 / Elemental Protection: 格挡时50%元素伤害减免 / 50% elemental damage reduction when blocking
- 盾牌冲锋 / Shield Charge: 冲刺中持盾冲倒敌人 / Sprint with shield to knock down enemies

### 耐力管理 / Stamina Management

| 动作 / Action | 耐力消耗 / Stamina Cost | 备注 / Note |
|---|---|---|
| 普通攻击 / Normal Attack | 15-25 | 取决于武器重量 / Depends on weapon weight |
| 重击 / Power Attack | 60-80 | 消耗巨大——但可击倒敌人 / High cost — knocks down |
| 格挡 / Block | 取决于伤害 / Depends on damage | 格挡Perk可减少消耗 / Block perks reduce cost |
| 冲刺 / Sprint | 20/秒 / per second | 战斗中慎用 / Use sparingly in combat |
| 弓箭瞄准 / Bow Draw | 持续消耗 / Continuous | 瞄准越久消耗越多 / Longer draw = more cost |

**耐力恢复技巧 / Stamina Recovery Tips**:
- 耐力低时不攻击——等恢复后再输出 / Don't attack with low stamina — wait for regen
- 蔬菜汤/牛肉汤提供持续耐力恢复 / Vegetable soup/beef stew give continuous stamina regen
- 恢复系法术"快速恢复"可提升恢复速度 / Restoration spell "Fast Healing" improves regen
