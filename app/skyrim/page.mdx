# 上古卷轴5攻略 / The Elder Scrolls V: Skyrim Guide

《上古卷轴5：天际》是Bethesda Game Studios开发的开放世界奇幻RPG——天际省，龙裔归来之时，古老预言终将实现。从溪木镇到冬堡，从战友团到黑暗兄弟会，从锻造附魔到龙吼龙语——这是一段无人能够抗拒的史诗之旅。本攻略涵盖全部种族与技能Build、九大领地探索、全阵营任务链以及装备锻造附魔完整指南。

The Elder Scrolls V: Skyrim is Bethesda's open-world fantasy RPG set in the province of Skyrim — where the Dragonborn returns to fulfill an ancient prophecy. From Riverwood to Winterhold, from the Companions to the Dark Brotherhood, from smithing and enchanting to Shouts and dragon words — an epic journey no one can resist. This guide covers all races and builds, the nine holds, all faction questlines, and complete equipment crafting.

## 游戏框架 / Game Systems

### 等级系统 / Leveling System

天际没有职业等级——你做任何事都会提升对应的技能 / No character classes — every action levels the corresponding skill.

| 技能类别 / Category | 技能 / Skills | 升级方式 / How to Level |
|---|---|---|
| **战士系 / Combat** | 单手、双手、弓箭、格挡、锻造、重甲 / One-Handed, Two-Handed, Archery, Block, Smithing, Heavy Armor | 使用对应武器/技能 / Use the relevant weapon/skill |
| **法系 / Magic** | 毁灭、恢复、变化、幻术、召唤、附魔 / Destruction, Restoration, Alteration, Illusion, Conjuration, Enchanting | 施放对应法术 / Cast relevant spells |
| **潜行系 / Stealth** | 潜行、开锁、偷窃、轻甲、炼金、演讲 / Sneak, Lockpicking, Pickpocket, Light Armor, Alchemy, Speech | 潜行/开锁/交易等 / Sneak, pick locks, trade, etc. |

**升级机制 / Level-Up**:
- 任何技能提升都会增加总体经验 / Any skill increase adds to overall XP
- 每升一级获得10点HP/MP/SP分配 + 一个Perk点 / Level up: +10 HP/MP/SP + 1 perk point
- 最高等级无上限——但Perk点有限(最高约80级前获得全部)/ No level cap, but perk points are finite

### 战斗系统 / Combat System

| 元素 / Element | 说明 / Description |
|---|---|
| **左手/右手 / Left/Right Hand** | 双持武器、法术+武器、双持法术——任意组合 / Dual wield, spell+weapon, dual spells — any combo |
| **格挡 / Block** | 盾牌/武器格挡——减少伤害 + 可以盾击破防 / Shield/weapon block — reduce damage + bash to stagger |
| **潜行攻击 / Sneak Attack** | 潜行状态下近战=3倍伤害(匕首=15倍) / Sneak melee = 3x damage (dagger = 15x with perks) |
| **龙吼 / Shouts** | 解锁后按Z使用——有冷却时间 / Press Z to use Shouts — cooldown-based |
| **随从 / Followers** | 可以招募一个随从同行——帮你战斗/负重 / Hire one follower — fights and carries loot |
