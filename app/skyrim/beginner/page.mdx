## 核心机制解析 / Core Mechanics

### 龙吼系统 / Shouts

龙吼是龙裔的特殊能力——由三个龙语组成 / Shouts are the Dragonborn's special ability — composed of three Words of Power.

**学习龙吼 / Learning Shouts**:
1. 找到铭刻龙语的"龙墙" / Find "Word Walls" with inscribed words
2. 吸收龙的灵魂(杀龙) / Absorb dragon souls (by killing dragons)
3. 在魔法菜单中消耗龙魂解锁 / Spend dragon souls in the Magic menu

### 随从系统 / Followers

天际中你可以招募一个随从同行 / You can have one follower at a time.

**重要随从位置 / Key Followers**:

| 随从 / Follower | 位置 / Location | 特长 / Specialty | 婚姻 / Marriage |
|---|---|---|---|
| 莱迪亚 / Lydia | 雪漫龙临堡 / Dragonsreach, Whiterun | 单手+重甲 / 1H + Heavy | 可 / Yes |
| 法恩达尔 / Faendal | 溪木镇 / Riverwood | 弓箭 / Archery | 可 / Yes |
| 简拉萨 / Jenassa | 雪漫醉猎人旅馆 / Drunken Huntsman | 双持 / Dual-wield | 可 / Yes |
| 马科里奥 / Marcurio | 裂谷城蜜蜂与倒钩旅馆 / Riften Bee and Barb | 毁灭法师 / Destruction | 可 / Yes |
| 塞兰达 / Serana | 黎明守卫DLC / Dawnguard DLC | 全能法师 / Versatile mage | 不可 / No |

**随从核心技巧 / Follower Tips**:
1. 随从可以无限负重——命令他们捡地上的东西 / Followers have unlimited carry capacity if you command them to pick items
2. 随从不会触发陷阱——让随从走前面探路 / Followers don't trigger traps — send them first
3. 随从死亡是永久性的——如果不小心杀了随从可以读档 / Follower death is permanent — reload if they die
4. **重要**：如果随从在任务中失踪——等待3天会回到原来招募地点 / Missing followers return to original location after 3 days

### 婚姻系统 / Marriage

可以通过**玛拉项链**和特定NPC对话结婚 / Marry specific NPCs with a **Mara's Amulet**.

**结婚收益 / Marriage Benefits**:
- 配偶开了一家商店——每天给你100金币 / Spouse opens a store — 100 gold/day
- 同住时获得"爱人"buff——技能升级+15% / "Lover's Comfort" buff — 15% skill improve rate

**可结婚NPC推荐 / Recommended Marriage Candidates**:
- 莱迪亚 / Lydia (雪漫) — 女战士—最强的战斗型配偶 / Female warrior — strongest fighter spouse
- 伊索尔达 / Ysolda (雪漫) — 商人型—开最赚钱的商店 / Merchant type — most profitable shop
- 穆伊瑞 / Muiri (马卡斯城) — 炼金师—提供炼金产品 / Alchemist — provides potions

## 新手常见错误 / Common Beginner Mistakes

| # | 错误 / Mistake | 正确做法 / Correct Way |
|---|---|---|
| 1 | 技能过于分散 / Spread skills too thin | 集中练6-8项技能——全能角色中期很弱 / Focus on 6-8 skills — jack-of-all-trades falls off at mid game |
| 2 | 不升级锻造和附魔 / Skip smithing & enchanting | 这两个技能决定装备上限——后期最强的来源 / These two skills define endgame power |
| 3 | 卖掉所有材料 / Sell all crafting mats | 矿石、灵魂石、草药全都留着——炼金附魔要用 / Keep ores, soul gems, ingredients — you'll need them |
| 4 | 不看任务描述跳对话 / Skip dialogue | 天际的任务提示很模糊——看清NPC说了什么 / Quest markers are vague — listen to NPCs |
| 5 | 负重满了还捡 / Over-encumbered looting | 价值/重量比>10的才捡——其他不要 / Value/weight ratio > 10 only |
| 6 | 不打随从MOD / No follower mods | 原版随从AI很低——强烈推荐"Amazing Follower Tweaks" / Vanilla follower AI is terrible |
