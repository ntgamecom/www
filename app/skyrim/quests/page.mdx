# 任务与阵营 / Quests & Factions

## 主线任务 / Main Quest

天际的主线共 **18个任务**——从逃离海尔根到击杀奥杜因 / 18 main quests — from escaping Helgen to killing Alduin.

| 章节 / Chapter | 任务 / Quest | 说明 / Description |
|---|---|---|
| 1 | 出逃 / Unbound | 从海尔根龙袭中逃出 / Escape Helgen dragon attack |
| 2 | 风暴怒吼之前 / Before the Storm | 去雪漫报告龙袭 / Report to Whiterun |
| 3 | 风暴怒吼 / The Way of the Voice | 爬七千级台阶去高吼峰 / Climb the 7000 steps |
| 4 | 黑降 / The Horn of Jurgen Windcaller | 从荒瀑古坟取号角 / Retrieve the Horn |
| 5 | 龙吼 / The Way of the Voice (续) | 学习不卸之力 / Learn Unrelenting Force |
| 6 | 约根之风 / The Horn of Jurgen Windcaller (续) | 黑降——首次遭遇龙裔 / First encounter with Delphine |
| 7 | 刀锋会 / A Blade in the Dark | 在凯尔娜塔杀第一条龙 / Kill first dragon at Kynesgrove |
| 8 | 外交豁免 / Diplomatic Immunity | 潜入梭默大使馆 / Infiltrate Thalmor Embassy |
| 9 | 不速之客 / Thalmor Embassy (续) | 找到刀锋会需要的情报 / Find the intel |
| 10 | 奥杜因之墙 / Alduin's Wall | 去天空天堂谷解读奥杜因之墙 / Decode Alduin's Wall at Sky Haven Temple |
| 11 | 世界之喉 / The Throat of the World | 学习召唤巨龙 / Learn "Call Dragon" |
| 12 | 艾恩盖尔 / The Mountain | 用召唤巨龙叫出奥达威英 / Summon Odahviing |
| 13 | 屠龙者 / Dragonslayer | 前往英灵殿——击杀奥杜因 / Go to Sovngarde — kill Alduin |
| 14 | 龙裔(DLC) / Dragonborn | 去索瑟海姆——对抗米拉克 / Go to Solstheim — fight Miraak |

**主线推荐分段 / Recommended Pacing**:
- 1-6: 前期(建议5-10级做) / Early (levels 5-10)
- 7-10: 中期(10-20级) / Mid (10-20)
- 11-13: 后期(20-30级) / Late (20-30)
- 14(DLC): 建议30级以后 / Recommended 30+

## 公会阵营 / Guilds & Factions

### 战友团 / The Companions

**天际最古老的佣兵公会**——在雪漫城的勇者之家 / Oldest mercenary guild in Skyrim — Jorrvaskr, Whiterun.

| 任务名 / Quest | 说明 / Description |
|---|---|
| 加入战友团 / Take Up Arms | 在勇者之家挑战任意成员——成为新兵 / Challenge a member — become a recruit |
| 荣耀证明 / Prove Your Honor | 完成第一个任务——成为正式成员 / Complete first job — become a full member |
| 银手 / The Silver Hand | 发现战友团的秘密——他们是狼人 / Discover the secret — they are werewolves |
| 血之荣光 / Blood's Honor | 成为战友团领袖 / Become the leader of the Companions |
| 纯粹天性 / Purest of Nature | **最终任务**——寻找治愈狼人的方法 / Find the cure for lycanthropy |

**战友团奖励 / Rewards**:
- 成为狼人——变身有1000额外生命值 / Become a werewolf — 1000 extra HP in beast form
- 天空熔炉的使用权 ——天际最好的锻造台 / Access to Skyforge — best smithy
- 可以治愈狼人——如果后期不想要这个能力 / Can cure lycanthropy if unwanted later

### 法师学院 / College of Winterhold

**天际唯一的魔法学院** / The only magic academy in Skyrim.

| 任务名 / Quest | 说明 / Description |
|---|---|
| 第一课 / First Lessons | 进入学院——展示魔法 / Enter the College — demonstrate magic |
| 萨沃斯·阿兰兹 / Under Saarthal | 发现阿其姆斯的萨尔之杖 / Discover the Staff of Magnus |
| 玛格努斯之杖 / The Staff of Magnus | 前往拉布林西安取杖 / Retrieve the Staff |
| 玛格努斯之眼 / The Eye of Magnus | **最终任务**——阻止阿其姆斯 / Stop Ancano |

**法师学院奖励 / Rewards**:
- 大法师长袍——100%法力恢复 + 减15%施法消耗 / Archmage's Robes — 100% mana regen + 15% spell cost reduction
- 学院各系教授的魔法训练 / Skill training from all schools' professors
- 元素大厅各系法术导师 / Spell vendors for all schools

| 任务名 / Quest | 说明 / Description |
|---|---|
| 顺风顺水 / A Chance Arrangement | 遇见布林乔夫——加入公会 / Meet Brynjolf — join the guild |

| 黑暗中的黑暗 / Darkness Returns | **最终任务**——重塑公会荣光 / Restore the Guild to glory |

| 渗透 / Breaching Security | 渗透沼泽洞穴监狱 / Infiltrate the swamp cave prison |
| 西塞罗的疯狂 / The Cure for Madness | 处理西塞罗 / Deal with Cicero |
| 死亡代替 / Death Incarnate | **最终任务**——击杀指挥官 / Kill the Commander |

**黑暗兄弟会奖励 / Rewards**:
- 影蓝马——幽灵马 / Shadowmere — ghost horse
- 远古兄弟会套装(最强刺客装备) / Ancient Shroud set (best assassin gear)
- 每次契约任务获得300-1200金币不等的报酬 / 300-1200 gold per contract

## 内战任务线 / Civil War

内战分为帝国军团 vs 风暴斗篷——选择一方加入 / Imperial Legion vs Stormcloaks.

| 阵营 / Side | 领袖 / Leader | 优势 / Strengths | 任务数 / Quests |
|---|---|---|---|
| 帝国军团 / Imperial Legion | 图留斯将军 / General Tullius | 正规军——有利的城市初始控制 / Professional army | ~12 |
| 风暴斗篷 / Stormcloaks | 乌弗瑞克·风暴斗篷 / Ulfric | 本地人支持——游击战优势 / Local support | ~12 |

**选择建议 / Recommendation**:
- 帝国：如果你喜欢秩序+商业繁荣 / If you prefer order + commerce
- 风暴斗篷：如果你喜欢诺德传统自由 / If you prefer Nord tradition + freedom

**解锁成就/奖杯**——推荐在完成主线后再做内战线 / Both sides unlock achievements — recommended post-main-quest
