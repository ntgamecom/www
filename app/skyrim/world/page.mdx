# 世界与探索 / World & Exploration

天际省分为 **9个领地**——每个领地有一个主城和一个领主 / Skyrim is divided into **9 Holds** — each has a capital city and a Jarl.

| 领地 / Hold | 主城 / Capital | 地形 / Terrain | 特点 / Traits |
|---|---|---|---|
| **雪漫 / Whiterun** | 雪漫城 / Whiterun | 平原 / Plains | 天际中心——最适合起步的区域 / Central — best starting area |
| **东陲 / Eastmarch** | 风盔城 / Windhelm | 冻原+温泉 / Tundra + hot springs | 风暴斗篷总部 / Stormcloak HQ |
| **裂谷 / The Rift** | 裂谷城 / Riften | 森林+湖泊 / Forest + lakes | 盗贼公会总部 + 秋色美景 / Thieves Guild + autumn scenery |
| **苍原 / The Pale** | 晨星城 / Dawnstar | 海岸+雪原 / Coast + snow | 小城镇——隐藏的魔神任务 / Small town — hidden Daedric quest |
| **白地 / Hjaalmarch** | 莫萨尔城 / Morthal | 沼泽 / Swamp | 吸血鬼出没 / Vampire-infested |
| **边塞 / The Reach** | 马卡斯城 / Markarth | 山地+峡谷 / Mountains + canyons | 锻莫遗址+弃誓者 / Dwemer ruins + Forsworn |
| **海望 / Haafingar** | 孤独城 / Solitude | 海岸+山崖 / Coast + cliffs | 帝国军团总部 / Imperial Legion HQ |
| **冬堡 / Winterhold** | 冬堡 / Winterhold | 雪山+冰原 / Snowy + icebergs | 法师学院所在 / College of Winterhold |
| **福克瑞斯 / Falkreath** | 福克瑞斯城 / Falkreath | 森林 / Dense forest | 黑暗兄弟会相关 / Dark Brotherhood connections |

## 隐藏要素 / Hidden Secrets

### 站立石 / Standing Stones

天际共有 **13个站立石**——每个提供特定的能力增益 / 13 Standing Stones — each provides a specific ability buff.

**激活效果** - 只能激活一个 / Only one active at a time.

| 石头 / Stone | 位置 / Location | 效果 / Effect |
|---|---|---|
| **战士之石 / Warrior Stone** | 溪木镇附近 / Near Riverwood | 战士系技能升级+20% / Combat skills +20% |
| **法师之石 / Mage Stone** | 裂谷城附近 / Near Riften | 法系技能升级+20% / Magic skills +20% |
| **盗贼之石 / Thief Stone** | 孤独城附近 / Near Solitude | 潜行系技能升级+20% / Stealth skills +20% |
| **恋人之石 / Lover Stone** | 马卡斯附近 / Near Markarth | 所有技能升级+15% / All skills +15% |
| **君王之石 / Lord Stone** | 冬堡以北 / North of Winterhold | +50护甲 +25%魔抗 / +50 armor +25% MR |
| **暗影之石 / Shadow Stone** | 裂谷附近 / Near Riften | 潜行时隐形/秒 / Invisibility while sneaking |

**最佳组合**:
1. 前期：战士/法师/盗贼石 (看你的Build) / Early: Warrior/Mage/Thief
2. 中期：恋人之石 (全方位+15%) / Mid: Lover Stone (+15% all)
3. 后期：君王之石 (50护甲+25%魔抗——生存向) / Late: Lord Stone (50 armor +25% MR)

### 巴兰兹雅之石 / Stones of Barenziah

共 **24块**——收集后获得"巴兰兹雅王冠"——大幅增加宝石发现率 / 24 stones — get "Crown of Barenziah" — massively increased gem find rate.

| 信息 / Info | 说明 / Description |
|---|---|
| 全部收集后效果 / Full collection | 每个容器内发现1-3颗宝石(价值300-1000+) / 1-3 gems per container (300-1000+ each) |
| 难度 / Difficulty | **非常麻烦**——分布在全地图各建筑内 / Very tedious — spread across entire map |
| 建议 / Tip | 不需要刻意收集——除非你追求全成就 / Don't bother unless going for achievements |

### 红奈恩根 / Crimson Nirnroot

在布莱克瑞奇找到 **30株红奈恩根**——交给萨沃斯·阿兰兹获得奖励 / Find 30 Crimson Nirnroot in Blackreach — give to Sarthis Aren for reward.

| 奖励 / Reward | 说明 / Description |
|---|---|
| 强化耐力药剂 x3 / Fortify Stamina x3 | 恢复200点耐力，5分钟内+200最大耐力 / Restore 200 stamina, +200 max for 5 min |

## 快速旅行系统 / Fast Travel

**通达各个已发现的主要城镇和地标** / Travel to any discovered major location.

**限制 / Restrictions**:
- 负重超载时不能快速旅行 / Cannot fast travel while over-encumbered
- 战斗中不能用 / Cannot use in combat
- 室内/地下城不能用 / Cannot use indoors or in dungeons
- 未被发现的地点不能用 / Unvisited locations unavailable

**代替方案 / Alternatives**:
1. 马车——到达各主城(即使没去过) / Carriage — to any capital (even unvisited)
2. 召唤马(阿瓦克) / Summon Arvak (Dawnguard)
3. 缓落术龙吼——飞下悬崖快速下落 / Become Ethereal Shout — jump off cliffs
4. 保持轻负重——随时快速旅行 / Stay unencumbered for instant fast travel
