## 技能系统概述 / Skills System Overview

《崩坏：星穹铁道》的角色技能系统包括行迹（Traces）、星魂（Eidolons）和额外的属性加成节点。理解每个角色的技能优先级是高效培养的关键。

The character skill system includes Traces, Eidolons, and bonus stat nodes. Understanding each character's skill priority is key to efficient investment.

## 行迹系统 / Trace System

行迹是角色的技能树，包含主动技能、被动天赋和属性加成节点。每个角色需要投入大量资源来解锁全部行迹。

Traces are the character's skill tree, containing active skills, passive talents, and stat bonus nodes. Each character requires significant resources to fully unlock.

### 行迹分类 / Trace Categories

**核心行迹 / Core Traces**：

- 普通攻击（普攻）：基础攻击技能，回复战技点
- 战技：消耗战技点的主动技能，通常是主要输出或治疗手段
- 终结技：需要能量释放的强力技能
- 天赋：常驻被动效果，通常提供特殊机制

- Basic ATK: Foundational attack, restores SP
- Skill: SP-consuming active, usually the main ability
- Ultimate: Powerful skill requiring energy
- Talent: Permanent passive providing special mechanics

**额外能力 / Bonus Abilities**：需要解锁前置节点的高级被动，提供诸如暴击率提升、能量回复效果加成等强化。每个角色有三个额外能力。

Advanced passives requiring unlocked prerequisites. Provide crit rate boosts, energy regen effects, etc. Each character has three bonus abilities.

**属性节点 / Stat Nodes**：提供攻击力、防御力、生命值、速度、暴击率等属性提升的小节点。

Small nodes providing stat boosts: ATK, DEF, HP, SPD, Crit Rate.

### 行迹升级优先级 / Trace Upgrade Priority

通用优先级原则 / General priority rules：

1. **终结技**：通常提供最高的伤害倍率或核心效果，优先升级
2. **战技**：主要循环技能，第二优先级
3. **天赋**：常驻效果，第三优先级
4. **普通攻击**：对于大部分角色优先级最低，但部分角色例外（如丹恒·饮月、镜流、青雀）

1. Ultimate: Usually the highest multiplier or core effect. Priority 1
2. Skill: Main rotation ability. Priority 2
3. Talent: Permanent passive. Priority 3
4. Basic ATK: Lowest priority for most. Exceptions: Dan Heng IL, Jingliu, Qingque

**辅助角色注意**：对于纯辅助角色，优先升级关键增益/治疗行迹即可，无需点满全部。

For pure supports, prioritize key buff/heal Traces. No need to max everything.

**输出角色注意**：主C建议全部行迹升满（包括普攻在某些情况下）。

Main DPS should max all Traces (Basic ATK included in some cases).

### 行迹升级材料 / Trace Upgrade Materials

行迹升级需要三种类型的材料：

Trace upgrades require three material types:

- **拟造花萼（赤）材料**：根据命途不同而不同，分为绿色/蓝色/紫色品质
- **周本材料**：历战余响掉落，用于高级行迹解锁（8级以上）
- **信用点**：所有升级操作都需要消耗信用点

- **Calyx Crimson materials**: Vary by Path, in green/blue/purple quality
- **Weekly boss materials**: From Echo of War, for level 8+ Traces
- **Credits**: Required for all upgrade operations

建议有计划地刷取材料，不要等需要时再刷。利用游戏内预览功能查看所需全部材料。

Plan material farming in advance using the in-game preview to see total requirements.

## 星魂系统 / Eidolon System

星魂相当于其他游戏中的命座系统，通过获得重复角色来解锁。每个角色有6个星魂等级，不同的星魂提供不同的强化效果。

Eidolons are equivalent to constellation systems in other games, unlocked by obtaining duplicate characters. Each character has 6 Eidolon levels offering various enhancements.

### 重要星魂 / Notable Eidolons

**关键星魂示例 / Key Eidolon Examples**：

- 镜流 E1：对非弱点敌人也有暴击率加成
- 布洛妮娅 E1：释放战技后一定概率恢复战技点
- 罗刹 E1：结界内队友攻击力提升
- 符玄 E1：全队暴击伤害提升
- 阮·梅 E1：击破弱点时无视部分防御
- 黄泉 E2：额外无视防御和伤害加成

- Jingliu E1: Crit rate even against non-weakness enemies
- Bronya E1: Chance to recover SP after Skill
- Luocha E1: Team ATK boost in field
- Fu Xuan E1: Team crit DMG boost
- Ruan Mei E1: DEF ignore on break
- Acheron E2: Additional DEF ignore and DMG boost

**四星关键星魂 / Notable 4-star Eidolons**：

- 停云 E6：终结技额外回复能量
- 佩拉 E4：战技降低冰抗
- 青雀 E4：进入战斗获得额外战技点
- 驭空 E6：释放终结技后全队速度提升

- Tingyun E6: Ultimate grants extra energy
- Pela E4: Skill reduces Ice RES
- Qingque E4: Start battle with extra SP
- Yukong E6: Ultimate boosts team SPD

星魂的获取成本较高，建议优先抽取角色的0命版本，再根据资源情况考虑关键星魂。

Eidolons are costly. Prioritize obtaining the C0 character first, then consider key Eidolons based on resources.

## 技能循环 / Skill Rotation

每个队伍有最佳的输出循环，合理安排技能释放顺序可以最大化伤害输出。

Each team has an optimal rotation. Properly ordering skill usage maximizes damage output.

**常见循环示例 / Common Rotation Example**：

1. 辅助开战技施加增益/减益
2. 副C使用战技补充伤害或施加DoT
3. 主C使用战技进行主要输出
4. 生存位根据需要治疗或套盾
5. 终结技在关键时机插入使用

1. Support uses Skill for buffs/debuffs
2. Sub-DPS applies damage or DoT
3. Main DPS uses Skill for primary damage
4. Sustain heals or shields as needed
5. Ultimates deployed at key moments

**战技点管理示例 / SP Management Example**：

产点角色（辅助）：普攻为主，战技为辅
耗点角色（主C）：战技为主
在战技点紧张时，辅助应适当减少战技使用

SP-positive characters (supports): Basic ATK mostly, Skill occasionally
SP-negative characters (DPS): Skill mostly
When SP is tight, supports should reduce Skill usage

## 材料 farming 策略 / Material Farming Strategy

**周常优先 / Weekly Priority**：每周三次数历战余响是最高优先级，掉落材料不可替代。

3 weekly Echo of War runs are top priority. Materials are irreplaceable.

**行迹材料 / Trace Materials**：根据角色命途选择对应的拟造花萼（赤）。建议在双倍活动期间大量刷取。

Select the correct Calyx Crimson per character Path. Farm heavily during double drop events.

**突破材料 / Ascension Materials**：根据角色属性选择凝滞虚影。每个角色需要特定属性的材料。

Select Stagnant Shadow per character element. Each character needs element-specific materials.

**信用点 / Credits**：拟造花萼（金）专门产出信用点，结合角色经验书一起刷。后期非常缺信用点。

Calyx Golden produces Credits and EXP books. Credits become scarce in late game.

## 角色培养规划 / Character Building Plan

建议每周制定培养计划，合理分配资源：

Plan weekly builds and allocate resources efficiently:

- 周一至周三：刷取行迹材料
- 周四至周五：刷取突破材料
- 周末：刷取信用点和经验书
- 每周：三次历战余响优先完成

- Mon-Wed: Trace material farming
- Thu-Fri: Ascension material farming
- Weekend: Credits and EXP books
- Weekly: 3 Echo of War runs

按照这个节奏可以稳步提升队伍整体实力。

This rhythm ensures steady team-wide improvement.
