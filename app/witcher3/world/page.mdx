# 世界探索 / World Exploration

## 区域总览 / Region Overview

巫师3共有 **5 个主要区域**（含DLC）/ 5 major regions (with DLCs).

| 区域 / Region | 等级范围 / Level | 主线任务数量 / Main Quests | 特色 / Highlights |
|---|---|---|---|
| 白果园 / White Orchard | 1-6 | 序章 | 新手村——建议全清 / Tutorial zone, full clear recommended |
| 威伦 / Velen | 5-20 | 10+ | 最大区域——战争废墟、血腥男爵主线 / Largest — war-torn, Bloody Baron main quest |
| 诺维格瑞 / Novigrad | 10-20 | 8+ | 自由之城——帮派、女巫猎人、特莉丝 / Free City — gangs, witch hunters, Triss |
| 史凯利杰 / Skellige | 15-30 | 10+ | 群岛——航海、问号很多在水下 / Islands — sailing, many "?" underwater |
| 凯尔莫罕 / Kaer Morhen | 20-25 | 3+ | 猎魔人堡垒——剧情转折点 / Witcher fortress — story turning point |
| 陶森特 / Toussaint (DLC) | 35-50 | 10+ | 血与酒——阳光明媚的吸血鬼之乡 / Blood and Wine — sunny vampire country |

### 希里命运任务 / Ciri's Fate Quests

希里的生死取决于以下5个选择——至少做**3个正面选择**才能存活 / Ciri's life depends on 5 choices — need **3 positive ones**.

1. **国王的赌局 / The King's Gambit** —— 带希里去见史凯利杰领袖 / Take Ciri to Skellige leaders
2. **穿越时空 / The Final Trial** —— 让希里独自面对精灵实验室 / Let Ciri face elven lab alone
3. **战场上的血 / Blood on the Battlefield** —— 陪希里见父亲恩希尔 / Accompany Ciri to meet father Emhyr
4. **复仇 / The King's Gambit** —— 和希里一起破坏实验室 / Trash the lab together with Ciri
5. **暴风雨前的平静 / Calm Before the Storm** —— 给希里买项链而不是喝酒 / Buy Ciri a necklace instead of drinking

## Gwent卡牌收集 / Gwent Card Collection

昆特牌是巫师3内置的卡牌游戏——共约 **300+ 张可收集卡牌** / Gwent is an in-game card game — ~300+ collectible cards.

**重要牌手 / Key Players**：
- 白果园旅馆老板 / White Orchard innkeeper
- 乌鸦窝血腥男爵 / Bloody Baron (Crow's Perch)
- 诺维格瑞帕西弗罗拉 / Passiflora (Novigrad)
- 史凯利杰凯尔卓 / Kaer Trolde (Skellige)
- 各大旅店老板 / Every innkeeper
- 陶森特鲍克兰 / Beauclair (Toussaint)

**收集成就 / Completion**: 需要收集全部卡牌——建议开局就买/赢所有旅店老板 / Need to collect all cards — start early by buying/winning from every innkeeper
