# 新手入门 / Beginner Guide

## 三大学派路线 / Three School Paths

游戏初期虽然没有明确的"职业选择"，但技能点和装备方向决定了你的玩法风格 / No explicit class, but skill point allocation and gear define playstyle.

| 学派 / School | 核心属性 / Core Stat | 玩法风格 / Playstyle |
|---|---|---|
| 猫学派 / Cat School | 轻甲 + 暴击 / Light armor + crit | 高速攻速、高暴击、高闪避——玻璃大炮 / Fast attacks, high crit, glass cannon |
| 熊学派 / Bear School | 重甲 + 强击 / Heavy armor + strong attack | 高防御、高生命、重击破防——坦克 / High defense, heavy attacks — tank |
| 狮鹫学派 / Griffin School | 中甲 + 法印 / Medium armor + Signs | 法印强化、元素伤害——施法者 / Sign intensity, spellcaster |

**其他学派 / Other Schools**：
- **狼学派 / Wolf School**：均衡流派，混合攻击和法印 / Balanced — hybrid attacks and signs
- **蝮蛇学派 / Viper School**：DLC石之心——暴击和毒伤 / Hearts of Stone — crit and poison focused
- **飞狮学派 / Manticore School**：DLC血与酒——炼金专精 / Blood and Wine — alchemy focused

## 技能系统 / Skill System

### 技能槽 / Skill Slots

共 **12 个技能槽**，解锁于等级 1/2/4/8/12/16/20/24/28/32/36/40 / 12 skill slots at specific levels. **突变诱发物（Mutagen）** 插入技能行左端——颜色匹配(红/蓝/绿)提供额外属性加成 / Mutagen matching colors (red/blue/green) gives bonus stats. 次世代更新后技能槽翻倍至24 / Next-gen update doubled to 24.

### 技能树 / Skill Trees

| 树 / Tree | 颜色 / Color | 核心技能 / Key Skills |
|---|---|---|
| 战斗 / Combat | 红色 / Red | 速攻/强攻强化、暴击率/暴击伤害、肾上腺点数 / Fast/strong attack, crit, adrenaline |
| 法印 / Signs | 蓝色 / Blue | 5种法印各5级强化 + 替换模式 / 5 signs x 5 tiers + alternate modes |
| 炼金 / Alchemy | 绿色 / Green | 魔药/煎药/剑油/炸弹增强——毒性上限 / Potion/decoction/oil/bomb enhancements |
| 通用 / General | 无色 / Colorless | 美食家、百折不挠等强力被动 / Gourmet, Undying passives |

### 突变系统 / Mutation System (血与酒 DLC)

击败狄拉夫后解锁——12种强大的被动技能，仅可装备1个 / Unlocked after Detlaff boss — 12 passives, equip 1 only.

| 突变 / Mutation | 效果 / Effect |
|---|---|
| 瘾头 / Euphoria | 每点毒性增加攻击力和法印强度 / Each toxicity = +attack +sign intensity |
| 魔法涌流 / Magic Sensibilities | 法印强度与暴击率挂钩 / Sign intensity tied to crit rate |
| 致死 / Deadly Counter | 反击必杀 / Counterattack instant kill |

## 战斗基础 / Combat Basics

1. **昆恩是神技 / Quen is King**：高难度下永远保持昆恩护盾——挨打不扣血 / Keep Quen active on higher difficulties — absorbs one hit
2. **闪避 vs 翻滚 / Dodge vs Roll**：轻按闪避（短距离无硬直）vs翻滚（长距离耗耐力）/ Dodge (short, no animation lock) vs Roll (long, costs stamina)
3. **反击破防 / Counterattack**：敌人攻击瞬间格挡可触发反击硬直——对人形敌人极有效 / Parry at right moment for counter stun — great vs humans
4. **亚登打鬼 / Yrden for Ghosts**：妖灵、日间妖灵等幽灵系敌人——亚登陷阱使其显形可伤害 / Specters need Yrden to become damageable
5. **剑油常驻 / Always Apply Oil**：次世代更新后剑油无限使用——战斗前记得涂对应油 / Infinite oils next-gen — always apply the right one before combat
6. **煎药强力但有毒 / Decoctions Strong but Toxic**：煎药持续到冥想——毒性占用大，配合瘾头突变使用 / Decoctions last until meditation — high toxicity, use with Euphoria mutation
