# 巫师3攻略 / The Witcher 3: Wild Hunt Guide

《巫师3：狂猎》是CD Projekt Red开发的开放世界动作RPG，改编自安杰伊·萨普科夫斯基的同名小说系列。你将扮演猎魔人杰洛特，在战争肆虐的庞大开放世界中寻找养女希里，同时面对来自狂猎的致命威胁。本攻略涵盖两个DLC——《石之心》和《血与酒》的全部内容。

The Witcher 3: Wild Hunt is an open-world action RPG by CD Projekt Red, based on Andrzej Sapkowski's novel series. Play as Geralt of Rivia, a witcher hunting for his adopted daughter Ciri across a war-torn continent while facing the deadly Wild Hunt. This guide covers both DLCs — Hearts of Stone and Blood and Wine.

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **探索白果园 / Explore White Orchard Fully** | 开局地图的所有"？"清完是值得的——获得技能点、草药配方、猎魔人装备图纸 / Clear all "?" on the starting map — ability points, recipes, witcher gear diagrams |
| **2** | **勤用昆恩法印 / Use Quen Often** | 昆恩护盾吸收一次伤害——高难度下是你的生存保障 / Quen absorbs one hit — essential for survival on higher difficulties |
| **3** | **多做支线任务 / Do Side Quests** | 支线不仅影响结局走向，还提供大量经验和装备。不要跳过 / Side quests affect endings and provide huge XP and gear. Don't skip |
| **4** | **搜刮一切 / Loot Everything** | 草药、炼金材料、武器装备——所有东西都有用。但注意不要在守卫面前偷 / Herbs, alchemy mats, gear — everything has use. Don't steal in front of guards |
| **5** | **撕掉旧装备 / Dismantle Old Gear** | 铁匠可以分解装备获得升级材料——比直接卖更划算 / Blacksmiths dismantle gear for upgrade materials — better than selling |
| **6** | **多看公告板 / Check Notice Boards** | 公告板刷新合约和支线任务——每个村庄都有 / Notice boards have new contracts and side quests — check every village |
| **7** | **用对剑油和魔药 / Use Oils & Potions** | 剑油提供对特定怪物类型+50%伤害，魔药提供强力Buff。次世代更新后剑油无限使用 / Oils give +50% dmg vs specific monster types. Next-gen patch made oils infinite-use |
| **8** | **做寻宝任务 / Do Treasure Hunts** | 猎魔人套装图纸从寻宝任务获得——4个等级的套装是毕业装 / Witcher gear diagrams from treasure hunts — 4 tiers, best-in-slot gear |
| **9** | **升级马鞍袋 / Upgrade Saddlebags** | 增加负重上限——你不会后悔的 / Increases carry weight — you won't regret it |
| **10** | **DLC建议等级 / DLC Recommended Level** | 石之心 30+级 / 血与酒 35+级 / Hearts of Stone 30+ / Blood and Wine 35+ |

| 项目 / Item | 说明 / Description |
|---|---|
| **开启条件 / Prerequisite** | 完成"血腥男爵"主线 / Complete "Bloody Baron" main quest |
| **推荐等级 / Level** | 30+ |
| **剧情时长 / Length** | 约10小时 / ~10 hours |
| **核心内容 / Content** | 镜子大师欧吉尔德·伊佛瑞克的契约——三个愿望任务线 / Gaunter O'Dimm's contract — three wish quests |
| **特色奖励 / Rewards** | 蝮蛇学派套装、符文之语（符文匠）、爱丽丝钢剑 / Viper gear, Runewright, Iris steel sword |
