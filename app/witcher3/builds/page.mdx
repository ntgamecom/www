# 流派构建 / Build Guides

## 总览 / Overview

巫师3的Build主要围绕 **战斗/法印/炼金** 三棵树搭配——搭配对应学派装备效果最大化 / Builds center on Combat/Signs/Alchemy trees — pair with matching witcher gear for max effect.

---

## 🐱 猫学派暴击流 / Cat School Crit Build

极致攻速和暴击伤害——玻璃大炮中的玻璃大炮 / Maximum attack speed and crit damage — ultimate glass cannon.

**毕业装备 / Endgame Gear**：
- 极品猫学派套装（轻甲）/ Mastercrafted Cat School gear (light armor)
- 钢剑/银剑：猫学派极品剑 / Cat School swords

**核心技能 / Core Skills**：

| 技能 / Skill | 点数 / Points | 效果 / Effect |
|---|---|---|
| 速攻强化 / Fast Attack | 5/5 | 速攻伤害+25% / +25% fast attack dmg |
| 速攻暴击 / Precision Strike | 5/5 | 速攻暴击率+10% / +10% fast attack crit |
| 暴击伤害 / Crushing Blows | 5/5 | 暴击伤害+50% / +50% crit damage |
| 肌肉记忆 / Muscle Memory | 5/5 | 速攻速+25% / +25% fast attack speed |
| 回旋 / Whirl | 5/5 | 速攻旋风——对群神技 / Spin attack — amazing vs groups |
| 猫学派技巧 / Cat School Techniques | 3/3 | 穿轻甲时暴击+50%/攻速+20% / +50% crit +20% speed in light armor |

**突变诱发物 / Mutagens**: 全部红色（攻击力）/ All red (attack power)

**推荐突变 / Mutation**: **致死 / Deadly Counter** — 反击必杀，配合高攻速 / Counter instant kill with fast attacks

**煎药推荐 / Decoctions**：
- **海克娜 / Ekimmara**: 攻击回血 / Attack = HP regain
- **土元素 / Golem**: +30% 攻击力 / +30% attack power

**战术 / Tactics**:
1. 速攻起手叠伤害——回旋对群 / Fast attack chain + Whirl for groups
2. 闪避为主——不能挨打 / Dodge only — cannot take hits
3. 昆恩护盾必须常驻 / Quen shield mandatory at all times

**强度评估 / Tier**: S — 输出天花板，但需要操作 / Highest DPS, high skill required

## 🐻 熊学派坦克流 / Bear School Tank Build

根本打不死——高防御、高HP、重击碾压 / Nearly immortal — high defense, high HP, heavy attack focus.

**毕业装备 / Endgame Gear**：
- 极品熊学派套装（重甲）/ Mastercrafted Bear gear (heavy armor)
- 熊学派双剑 / Bear School swords

**核心技能 / Core Skills**：

| 技能 / Skill | 点数 / Points | 效果 / Effect |
|---|---|---|
| 强攻强化 / Strong Attack | 5/5 | 强攻伤害+25% / +25% strong attack dmg |
| 破防 / Power Strike | 5/5 | 强攻无视部分护甲 / Strong attack ignores armor |
| 怒气爆发 / Fury | 5/5 | 强攻暴击+10% / +10% strong attack crit |
| 不动如山 / Steadfast | 5/5 | 强攻击中回复耐力 / Strong attack = stamina regain |
| 逆流 / Counterflow | 3/5 | 格挡不耗耐力 / Block costs no stamina |
| 熊学派技巧 / Bear School Techniques | 3/3 | 穿重甲时+100%耐力恢复/+20%强攻 / +100% stamina regen +20% strong in heavy armor |

**突变诱发物 / Mutagens**: 全部红色（攻击力）/ All red (attack power)

**推荐突变 / Mutation**: **肾上腺素爆发 / Adrenaline Burst** — 击杀回肾上腺 / Kill = adrenaline

**煎药推荐 / Decoctions**：
- **古代鹿首精 / Ancient Leshen**: +50% HP / +50% max HP
- **狮鹫 / Griffin**: 格挡回血 / Block = HP regain
- **叉尾龙 / Forktail**: 被击时回血 / Hit = HP regain

**战术 / Tactics**:
1. 强攻蓄力——破防+击倒接处决 / Charged strong attack = knockdown + execution
2. 硬扛伤害喝药——煎药持续回血 / Tank damage while decoctions heal
3. 逆流格挡不耗耐力——面对群敌不虚 / Free blocking with Counterflow

**强度评估 / Tier**: A+ — 生存最强，输出足够但不如猫 / Best survival, good DPS but not top

## 🦅 狮鹫派法印流 / Griffin School Sign Build

用伊格尼和阿尔德清场——完全不依赖剑 / Clear rooms with Igni and Aard — barely need swords.

**毕业装备 / Endgame Gear**：
- 极品狮鹫学派套装（中甲）/ Mastercrafted Griffin gear (medium armor)
- 极品狮鹫双剑（法印强度加成）/ Griffin swords (Sign intensity bonus)

**核心技能 / Core Skills**：

| 技能 / Skill | 点数 / Points | 效果 / Effect |
|---|---|---|
| 伊格尼强化 / Melt Armor | 5/5 | 伊格尼伤害+25%，降低敌人护甲 / Igni +25% dmg, reduces armor |
| 伊格尼替换 / Firestream | 2/2 | 持续火焰喷射 / Continuous fire spray |
| 阿尔德强化 / Aard Intensity | 5/5 | 阿尔德伤害+25% / Aard +25% dmg |
| 阿尔德替换 / Aard Sweep | 2/2 | 扇形阿尔德 / Cone AoE Aard |
| 昆恩替换 / Active Shield | 2/2 | 昆恩移动回血 / Moving shield regen |
| 法印强化 / Sign Intensity | 5/5 | 全法印强度+25% / All signs +25% |
| 狮鹫学派技巧 / Griffin School Techniques | 3/3 | 穿中甲时法印+20%/耐力恢复+10% / +20% sign +10% stamina in medium armor |

**突变诱发物 / Mutagens**: 全部蓝色（法印强度）/ All blue (sign intensity)

**推荐突变 / Mutation**: **魔法涌流 / Magic Sensibilities** — 法印强度=暴击率 / Sign intensity = crit rate

**煎药推荐 / Decoctions**：
- **远古吸血女妖 / Ancient Vampire**: 法印消耗-50% / Sign stamina cost -50%
- **女夜魔 / Succubus**: 法印强度+30% / Sign intensity +30%

**战术 / Tactics**:
1. 伊格尼点燃+持续伤害——配合舞动之星炸弹 / Igni burn + Dancing Star bomb combo
2. 阿尔德横扫清场——击倒接处决 / Aard Sweep knockdown + execution
3. 昆恩主动护盾回血——生存无忧 / Active Quen shield for sustain

**强度评估 / Tier**: A+ — 法印成型后清场速度极快，但Boss战稍弱 / Great clear speed, weaker vs bosses

## 🧪 瘾头炼金流 / Euphoria Alchemy Build

炼金流派之王——同时喝3-4瓶煎药，攻击力和法印强度翻倍 / King of alchemy — drink 3-4 decoctions simultaneously, double attack power + sign intensity.

**毕业装备 / Endgame Gear**：
- 极品飞狮学派套装（DLC血与酒）/ Grandmaster Manticore gear (Blood and Wine)
- 任意高伤害剑 / Any high-damage swords

**核心技能 / Core Skills**：

| 技能 / Skill | 点数 / Points | 效果 / Effect |
|---|---|---|
| 耐力强化 / Endurance Training | 5/5 | 毒性上限+30 / Toxicity cap +30 |
| 炼金术 / Alchemy | 5/5 | 煎药持续时间+25% / Decoction duration +25% |
| 协同作用 / Synergy | 5/5 | 突变诱发物效果+100% / Mutagen effects +100% |
| 剑油强化 / Poisoned Blades | 5/5 | 剑油中毒伤害 / Oil applies poison DoT |
| 耐性增强 / Tolerance | 5/5 | 毒性抗性+100% / Toxicity resistance +100% |
| 快速代谢 / Fast Metabolism | 5/5 | 毒性自动下降+20%/s / Toxicity decays +20%/s |

**突变诱发物 / Mutagens**: 3个绿色（毒性/生命） + 红色或蓝色随意 / 3 green + rest red/blue

**推荐突变 / Mutation**: **瘾头 / Euphoria** — 每点毒性+攻击力和法印强度 — 核心 / Each toxicity = +attack +sign intensity — CORE SKILL

**煎药推荐 / Decoctions (同时喝3-4瓶)**：
- **海克娜 / Ekimmara Decoction**: 攻击回血 / HP on attack
- **叉尾龙 / Forktail Decoction**: 受伤回血 / HP on hit taken
- **古代鹿首精 / Ancient Leshen Decoction**: +50% HP
- **狮鹫 / Griffin Decoction**: 格挡回血 / HP on block

**战术 / Tactics**:
1. 战斗前喝3-4瓶煎药——毒性冲到极限 / Drink 3-4 decoctions before combat — max toxicity
2. 瘾头突变将高毒性转化为巨额攻击力和法印加成 / Euphoria converts toxicity to massive attack + sign bonus
3. 协同作用+绿色突变进一步放大 / Synergy + green mutagens amplify further
4. 就算只穿普通装备，攻击力和法印也远超其他流派 / Even with basic gear, damage and signs exceed other builds

**强度评估 / Tier**: S+ — 游戏最强Build（版本答案）/ Strongest build in the game — the meta answer
