## 战斗系统概述 / Combat Overview

**中文：** 巫师3的战斗系统融合了剑术、法印（魔法）、炼金术（剑油、煎药、魔药）和十字弓四种核心玩法。猎魔人的战斗不仅仅是砍杀，更需要提前准备——针对不同的怪物类型涂抹对应的剑油、喝下合适的煎药、使用正确的法印。最高难度"死而无憾"下，准备工作直接决定战斗成败。

**English:** The Witcher 3's combat combines four core systems: swordsmanship, Signs (magic), alchemy (oils, decoctions, potions), and the crossbow. Combat is about preparation — applying the right oil, drinking the right decoction, and using the right Sign for each monster type. On Death March difficulty, preparation determines victory or defeat.

| 战斗系统 / System | 类型 / Type | 核心作用 / Core Role |
|---|---|---|
| 剑术 / Swordsmanship | 物理近战 / Physical Melee | 主要输出手段 / Primary damage |
| 法印 / Signs | 魔法 / Magic | 控制、防护、辅助 / Control, Shield, Utility |
| 炼金术 / Alchemy | 药剂 / Potions | 增强、回复、抗性 / Buffs, Healing, Resist |
| 十字弓 / Crossbow | 远程 / Ranged | 空中怪物、水下 / Flying, Underwater |

{/* ===== 剑术系统 / Swordsmanship ===== */}

## 剑术系统 / Swordsmanship

**中文：** 剑术是杰洛特最基础的战斗方式。钢剑对付人类和类人生物，银剑对付怪物。攻击分为轻击（快速、低伤害）和重击（慢速、高伤害）。闪避和格挡是生存的关键——闪避适合躲避大型怪物，格挡（招架）适合对付人类敌人的反击。完美格挡（在命中瞬间格挡）可以击晕人类敌人。

**English:** Swordsmanship is Geralt's foundational combat style. The steel sword is for humans and humanoids; the silver sword is for monsters. Light attacks are fast with low damage; heavy attacks are slow with high damage. Dodge is for avoiding big monster attacks; parry is for countering human enemies. A perfect parry (parry at the moment of impact) stuns human opponents.

**剑术技能 / Sword Skills**:

| 技能 / Skill | 效果 / Effect |
|---|---|
| 肌肉记忆 / Muscle Memory | 轻击伤害+5%/级 / Light attack damage +5%/level |
| 精准重击 / Precise Strikes | 重击暴击率+5%/级 / Heavy attack crit rate +5%/level |
| 剑舞 / Whirl | 轻击持续消耗耐力旋转攻击，范围伤害 / Spin attack with light attacks, AoE damage |
| 怒气爆发 / Rend | 重击蓄力造成高额伤害 + 破甲 / Charged heavy attack, high damage + armor break |
| 回旋回避 / Fleet Footed | 闪避时伤害减免+25% / Damage reduction during dodge +25% |

**核心技巧 / Core Tips**:
1. 轻击-闪避循环：轻击2-3下 → 闪避 → 重复 / Light attack-dodge loop: 2-3 hits → dodge → repeat
2. 重击起手打断大型怪物攻击 / Heavy attack to interrupt large monster attack wind-ups
3. 利用剑舞清理群怪 / Use Whirl for crowd clear
4. 怒气爆发针对高护甲精英怪和Boss / Rend for high-armor elites and bosses

{/* ===== 法印系统 / Sign System ===== */}

## 法印系统 / Sign System

**中文：** 法印是五种基础魔法，不需要消耗耐力（而是消耗活力）。活力的恢复速度决定法印的使用频率。可以通过技能和装备减少法印消耗、增加强度。

**English:** Signs are five basic magic spells that consume Stamina instead of HP. Stamina regen rate determines Sign frequency. Skills and gear can reduce Sign cost and increase intensity.

| 法印 / Sign | 效果 / Effect | 最佳用途 / Best Use |
|---|---|---|
| 阿尔德 / Aard | 击退/击倒敌人 / Telekinetic push | 破盾、击倒悬崖边敌人 / Break shields, knock off cliffs |
| 亚克西 / Axii | 魅惑/混乱敌人 / Charm enemy | 对话选项、让敌人内斗 / Dialogue, making enemies fight |
| 伊格尼 / Igni | 火焰伤害 / Fire damage | 点燃油桶、燃烧群怪 / Ignite oil, burn groups |
| 昆恩 / Quen | 护盾 / Shield | 吸收伤害，生存核心 / Damage absorb, survival core |
| 亚登 / Yrden | 减速陷阱 / Slow trap | 对付灵体类怪物 / Against wraiths and specters |

**法印使用技巧 / Sign Usage Tips**:
- **昆恩是生存的核心**：战斗中始终维持昆恩护盾，一旦被破立刻重新施放 / Quen is survival: keep it active at all times, recast immediately when broken
- 伊格尼对食尸鬼和水鬼类怪物有奇效 / Igni is very effective against ghouls and drowners
- 亚登打日间妖灵和夜间妖灵是强制需求 / Yrden is mandatory for wraith-type enemies
- 阿尔德可以打断正在施法的敌人 / Aard interrupts enemy spellcasting

{/* ===== 炼金术 / Alchemy ===== */}

## 炼金术系统 / Alchemy

**中文：** 炼金术是巫师3中最强大的辅助系统。剑油提供对特定怪物类型的伤害加成，煎药提供强力长效Buff，魔药提供短时增益和生命回复。每次冥想时如果背包里有烈酒（如矮人烈酒），会自动补充所有已制作的魔药和剑油——材料只需要收集一次。

**English:** Alchemy is the most powerful support system. Oils provide damage bonus against specific monster types. Decoctions grant powerful long-lasting buffs. Potions give short-term boosts and HP regen. Meditation auto-refills all crafted potions and oils using alcohol in your inventory — ingredients only need to be gathered once.

**剑油 / Oils**:

| 剑油 / Oil | 克制怪物 / Monster Type |
|---|---|
| 吊死者之油 / Hanged Man's Oil | 人类 / Humans |
| 食尸生物油 / Necrophage Oil | 食尸鬼、水鬼 / Ghouls, Drowners |
| 混兽油 / Hybrid Oil | 狮鹫、翼手龙 / Griffins, Wyverns |
| 诅咒生物油 / Cursed Oil | 狼人、吸血女妖 / Werewolves, Bruxae |
| 虫胶 / Insectoid Oil | 蟹蜘蛛、巨型蝎 / Arachnomorphs, Scorpions |
| 龙兽油 / Draconid Oil | 叉尾龙、龙 / Forktails, Dragons |

**煎药（核心战斗药剂） / Decoctions (Core Combat Potions)**:

| 煎药 / Decoction | 效果 / Effect |
|---|---|
| 海克娜 / Ekimmara | 造成伤害回复生命 / Deal damage to heal |
| 叉尾龙 / Forktail | 攻击回耐力 / Attacks restore stamina |
| 古老吸血鬼 / Ancient Leshen | 法印强度+100%，但耐力回复-50% / Sign intensity +100%, stamina regen -50% |
| 变形蛛 / Arachas | 背包重量越少护甲越高 / Less inventory weight = more armor |

## 十字弓系统 / Crossbow

**中文：** 十字弓在巫师3中定位为辅助武器，主要用于水下战斗（一箭必杀）和打落飞行怪物。伤害不如剑术，但特定场景不可替代。部分弩箭有特殊效果——爆炸箭、击倒箭等。

**English:** The crossbow is a support weapon, essential for underwater combat (one-shot kill) and grounding flying monsters. Its damage is lower than swords, but certain situations demand it. Special bolts include explosive and knockdown variants.

**十字弓技巧 / Crossbow Tips**:
1. 水下战斗：十字弓是水下唯一可用武器，一箭击杀任何水下生物 / Underwater: crossbow one-shots all aquatic enemies
2. 打龙：飞行怪物（狮鹫、翼手龙）被弩箭射中会落地 / Shoot flying monsters (griffins, wyverns) to ground them
3. 爆头伤害：弩箭爆头有额外伤害加成 / Headshots grant bonus damage

{/* ===== 战斗流派 / Combat Builds ===== */}

## 战斗流派 / Combat Builds

**中文：** 三种主流战斗流派：剑术流、法印流和炼金流。可以根据喜好混合。

**English:** Three main combat builds: Swordsmanship, Signs, and Alchemy. Mix and match as preferred.

**剑术流 / Swordsmanship Build**:
- 核心技能：肌肉记忆、精准重击、剑舞、怒气爆发 / Core: Muscle Memory, Precise Strikes, Whirl, Rend
- 推荐装备：猫学派装备（增加暴击和攻速）/ Cat School gear (crit + attack speed)
- 玩法：近身肉搏，利用剑舞清群，怒气爆发打精英 / Close combat, Whirl for groups, Rend for elites

**法印流 / Sign Build**:
- 核心技能：昆恩强化、伊格尼强化、亚登强化 / Core: Quen upgrades, Igni upgrades, Yrden upgrades
- 推荐装备：狮鹫学派装备（增加法印强度）/ Griffin School gear (Sign intensity)
- 玩法：伊格尼点燃群怪，昆恩护盾自保，亚登打灵体 / Igni for groups, Quen for survival, Yrden for wraiths

**炼金流 / Alchemy Build**:
- 核心技能：耐性提高、组织变异、协同作用 / Core: Heightened Tolerance, Tissue Transformation, Synergy
- 推荐装备：熊学派装备（增加生命和抗性）/ Bear School gear (HP + resistance)
- 玩法：喝三瓶煎药获得超强Buff，剑油增伤，魔药回复 / Triple decoction for massive buffs, apply oil, chug potions

{/* ===== 战斗技巧汇总 / Combat Tips Summary ===== */}

## 战斗技巧汇总 / Combat Tips Summary

**中文：** 最高难度"死而无憾"下的核心生存法则。

**English:** Core survival rules for Death March difficulty.

1. **昆恩护盾常驻**：每次战斗前先上昆恩，被打破后立即风筝等耐力回复再补 / Always have Quen active before engaging, reapply when broken
2. **剑油不能忘**：遇到新怪物第一时间查看剑油需求 / Always apply appropriate oil for each monster type
3. **吃煎药**：至少喝一个煎药再打高难战斗 / Drink at least one decoction before hard fights
4. **不要贪刀**：砍2-3下就闪避，不要试图连到耐力耗尽 / Hit 2-3 times then dodge, never exhaust your stamina
5. **利用翻滚**：对付大型怪物用翻滚（长按闪避）躲避范围攻击 / Use roll (hold dodge) to avoid large monster AoE attacks
6. **法印灵活切换**：灵体用亚登，人类用阿尔德破盾，群怪用伊格尼 / Switch Signs: Yrden vs wraiths, Aard vs shields, Igni vs groups
7. **水下一定要装弩箭**：否则水下战斗等于送死 / Always equip crossbow bolts before diving
8. **食物**：低难度靠食物回血消耗品足够，高难度必须靠煎药和魔药 / Food is enough on low difficulties; decoctions and potions are mandatory on Death March
