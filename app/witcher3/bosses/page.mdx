## 狮鹫 / Griffin (白果园)

**位置 / Location**: 白果园, 被毁的村庄附近 / White Orchard, near ruined village
**等级建议 / Level**: 2-4

**Boss特性 / Boss Traits**：
- 飞行——大部分时间在天上 / Flies — mostly airborne
- 俯冲攻击——高伤害 / Dive attack — high damage
- 落地点短暂硬直 / Lands briefly, vulnerable window

**打法 / Strategy**:
1. **引诱俯冲 / Lure Dive**：站在开阔地——狮鹫俯冲时闪避后砍几刀 / Stand in open — dodge dive, get a few hits
2. **十字弓打下来 / Crossbow Takedown**：飞行时用十字弓射——有一定几率打落 / Shoot with crossbow to ground it
3. **阿尔德打断 / Aard Interrupt**：俯冲瞬间阿尔德可以把它打翻在地 / Aard at dive moment to stagger it
4. **活用伊格尼 / Use Igni**：点燃造成持续伤害 / Ignite for burn DoT

**掉落 / Loot**: 狮鹫战利品（+5%经验+5%暴击）/ Griffin trophy (+5% XP, +5% crit)

## 血腥男爵的豪猪 / Botchling (乌鸦窝)

**位置 / Location**: 乌鸦窝地下 / Crow's Perch basement
**等级建议 / Level**: 8-10

**机制 / Mechanic**: 这是一个救赎任务——可以选择净化（战斗）或释放 / Redemption quest — can purify (fight) or release.

**战斗打法 / Combat Strategy**:
1. **月亮粉尘 / Moon Dust**：月尘炸弹阻止豪猪变形隐身 / Moon Dust stops it from phasing
2. **亚登法印 / Yrden**：亚登陷阱让它显形——核心技能 / Yrden trap reveals it — essential
3. **恶灵剑油 / Specter Oil**：+50%对恶灵伤害 / +50% damage vs specters
4. **阿尔德打断**：它释放尖叫技能时阿尔德可打断 / Aard interrupts its scream attack

**净化奖励 / Purification Reward**: 血腥男爵的好感 + 完整个任务线 / Baron's favor + complete questline

## 林中夫人 / Crones (威伦沼泽)

**位置 / Location**: 康庄之地 / Baldur, Velen
**等级建议 / Level**: 12-14

虽然是剧情Boss，但需要合理战术 / Story boss that needs good strategy.

**打法 / Strategy**:
1. **先清除小怪 / Clear Adds First**：场景中的仆从优先杀——不影响Boss刷新 / Kill the servants first — they don't affect boss spawn
2. **逐个击破 / Fight One at a Time**：三姐妹会轮流上场——每次只有一个主动攻击 / They take turns — only one fights at a time
3. **伊格尼输出 / Igni DPS**：对林中夫人特别有效 / Igni is very effective against them
4. **昆恩常驻 / Quen Always**：她们的技能伤害高——保持护盾 / High damage abilities — keep Quen up

## 伊勒瑞斯 / Imlerith (凯尔莫罕)

**位置 / Location**: 凯尔莫罕庭院 / Kaer Morhen courtyard
**等级建议 / Level**: 20-25

狂猎将军之一——高难度分水岭Boss / One of the Wild Hunt generals — difficulty spike boss.

**Boss阶段 / Boss Phases**:

| 阶段 / Phase | HP范围 / Range | 行为 / Behavior |
|---|---|---|
| 一阶段 | 100%-60% | 重锤攻击 + 召唤冰刺 / Heavy mace + ice spikes |
| 二阶段 | 60%-30% | 冲刺连击 + AoE冰爆 / Charge combo + AoE ice blast |
| 三阶段 | 30%-0% | 狂暴——攻速大幅提升 / Enrage — attack speed spikes |

**打法 / Strategy**:
1. **绕后攻击 / Flank Attack**：伊勒瑞斯的正面攻击范围大——绕到背后输出 / Wide attack arcs — attack from behind
2. **闪避不要格挡 / Dodge Don't Block**：他的重锤攻击会破防——闪避更安全 / His heavy attacks break guard — dodge instead
3. **亚克西/ Axii**：亚克西可以短暂眩晕他——输出窗口 / Axii stuns him briefly — DPS window
4. **利用地形 / Use Terrain**：凯尔莫罕庭院有柱子和台阶——可以卡位 / Use pillars and stairs for positioning

**掉落 / Loot**: 狂猎将军战利品 / Wild Hunt general trophy

## 卡兰希尔 / Caranthir (迷雾之岛)

**位置 / Location**: 迷雾之岛 / Isle of Mists
**等级建议 / Level**: 22-26

狂猎法师——擅长远程冰系攻击 / Wild Hunt mage — ice magic specialist.

**打法 / Strategy**:
1. **贴身近战 / Close Range**：卡兰希尔的远程法术伤害极高——贴身后他的近战较弱 / His ranged spells are deadly — close range reduces his effectiveness
2. **阿尔德打断施法 / Aard Interrupt**：他施法时阿尔德可打断——创造输出机会 / Interrupt his casting with Aard
3. **防火药 / Golden Oriole**：虽然他是冰系，但金色金莺可以解毒+回血 / Golden Oriole gives poison immunity + heal
4. **不要踩冰面 / Avoid Ice Floor**：他释放的冰面会减速——跳过去或用阿尔德清除 / Ice floor slows — jump over or clear with Aard

## 艾瑞汀 / Eredin (最终Boss)

**位置 / Location**: 史凯利杰海岸 / Skellige coast
**等级建议 / Level**: 25-30

狂猎之王——游戏最终Boss / King of the Wild Hunt — final boss.

**Boss阶段 / Boss Phases**:

| 阶段 / Phase | HP范围 / Range | 行为 / Behavior |
|---|---|---|
| 一阶段 | 100%-60% | 剑术对攻——瞬移绕后 / Sword duel — teleport flank |
| 二阶段 | 60%-25% | 召唤狂猎猎犬 + 冰冻箭 / Summon hounds + ice arrows |
| 三阶段 | 25%-0% | 开启传送门——召唤杂兵 / Opens portals — summons minions |

**打法 / Strategy**:
1. **学会反制瞬移 / Counter Teleport**：艾瑞汀瞬移到背后时立刻闪避或转身格挡 / When he teleports behind, dodge or turn-block immediately
2. **优先清狗 / Clear Hounds**：二阶段的猎犬烦人——伊格尼或回旋速清 / Kill hounds fast with Igni or Whirl
3. **三阶段打游击 / Third Phase Kite**：不要被杂兵包围——跑动+阿尔德击倒各个击破 / Don't get surrounded — kite with Aard knockdowns
4. **雷霆+燕子 / Thunderbolt + Swallow**：全程保持攻击Buff和回血 / Keep Thunderbolt + Swallow active throughout

**掉落 / Loot**: 艾瑞汀银剑 / Eredin's silver sword

## DLC Boss: 狄拉夫 / Detlaff (血与酒)

**位置 / Location**: 暗之地——陶森特地下 / The Underworld — Toussaint underground
**等级建议 / Level**: 40-45

**巫师3全游戏最难的Boss**，没有之一 / **Hardest boss in the entire game**, bar none.

**Boss阶段 / Boss Phases**：

| 阶段 / Phase | HP范围 / Range | 行为 / Behavior | 难度 / Difficulty |
|---|---|---|---|
| 一阶段 | 100%-65% | 人形态——快速剑术+回避 / Humanoid — fast sword + evade | ★★★★ |
| 二阶段 | 65%-30% | 蝙蝠群——全屏AoE蝙蝠轰炸 / Bat swarm — full screen bat barrage | ★★★★★ |
| 三阶段 | 30%-0% | 变异形态——触手+远程射击 / Mutated form — tentacles + ranged | ★★★★☆ |

**一阶段打法 / Phase 1**：
1. 他的攻击速度极快——熟悉闪避节奏 / Extremely fast attacks — learn dodge timing
2. 他闪避后立刻追击——输出窗口很短 / Short DPS windows after his dodge
3. 不要贪刀——砍2-3刀就闪避 / 2-3 hits then dodge — don't overcommit

**二阶段——最难阶段 / Phase 2 — Hardest Phase**：
1. **地面红色圆圈提示**——蝙蝠群从圆圈喷出 / Red circles on ground — bat swarms erupt from them
2. **保持移动**——原地站着必中蝙蝠群 / Keep moving — standing still = guaranteed hit
3. 连续闪避 + 翻滚 + 奔跑躲蝙蝠群 / Chain dodge + roll + sprint to avoid bats
4. **蝙蝠群结束后有短暂输出窗口**——全力输出 / Short DPS window after bat swarm ends
5. 推荐准备**北风炸弹**——可以冻住他打断蝙蝠群 / Northern Wind bomb freezes him, interrupting bats

**三阶段 / Phase 3**：
1. 触手可以破坏——优先打掉 / Tentacles are destructible — prioritize them
2. 远程地面触手会喷毒——闪避 / Ground tentacles spray poison — dodge
3. 换银剑——他变异后仍然是吸血鬼 / Silver sword — he's still a vampire
4. 吸血剑油 + 黑血魔药 / Vampire oil + Black Blood potion

**推荐准备 / Recommended Prep**:
- 燕子-超级 / Superior Swallow
- 雷霆-超级 / Superior Thunderbolt
- 黑血-超级 / Superior Black Blood
- 舞动之星 + 北风炸弹 / Dancing Star + Northern Wind
- 海克娜煎药 / Ekimmara Decoction

## DLC Boss: 欧吉尔德 / Olgierd von Everec (石之心)

**位置 / Location**: 伊佛瑞克庄园 / Von Everec estate
**等级建议 / Level**: 32-36

**特殊机制 / Special Mechanic**：欧吉尔德被镜子大师诅咒——战斗中拥有不死之身 / Cursed by Gaunter O'Dimm — functionally immortal in combat.

**打法 / Strategy**:
1. **他不可杀死**——只需要把他打到一定HP以下触发剧情 / Can't kill him — reduce HP to trigger story event
2. 爱丽丝（他的钢剑）会释放鬼魂干扰——闪避 + 昆恩 / Iris (his sword) spawns ghosts — dodge + Quen
3. 他使用火法——伊格尼对他无效 / He uses fire magic — Igni is useless
4. 剑油用**诅咒生物油 / Cursed Oil**——他是诅咒生物 / Cursed Oil — he's a cursed being
5. **速攻为主**——他的强攻反击速度极快 / Use fast attacks — his heavy counter is too fast

## DLC Boss: 镜子大师 / Gaunter O'Dimm (石之心)

**位置 / Location**: 伊佛瑞克庄园——异空间 / Von Everec estate — otherworldly dimension
**等级建议 / Level**: 35-40

**特殊机制 / Special Mechanic**：不是传统战斗——是**谜题+平台挑战** / Not a traditional fight — **puzzle + platforming challenge**.

**流程 / Walkthrough**：
1. 在异空间追逐镜子大师 / Chase Gaunter through twisted dimension
2. 躲避他的环境攻击——闪电、冰柱、火焰 / Dodge environmental attacks — lightning, icicles, fire
3. 打碎发光的水晶才能伤害他 / Break glowing crystals to damage him
4. 利用阿尔德快速摧毁水晶 / Use Aard to destroy crystals quickly
5. 阶段3：站在指定的圆形平台上躲避攻击 / Phase 3: Stand on specific circular platforms to avoid attacks

**奖励 / Reward**：
- **选择1**：拯救欧吉尔德——获得爱丽丝钢剑（最强钢剑之一）+ 马鞍 / Save Olgierd — get Iris (best steel sword) + saddle
- **选择2**：让镜子大师杀死欧吉尔德——获得无限吃喝/无限食物/无限酒 / Let Gaunter kill Olgierd — infinite food/drink
- **选择3**：与镜子大师赌斗——获得**恶魔马鞍/最佳马鞍** + 无限吃喝 / Bet with Gaunter — best saddle + infinite food

**推荐选择 / Recommendation**: 选1——爱丽丝是游戏最强钢剑之一 / Option 1 — Iris is one of the best steel swords in the game
