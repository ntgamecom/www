## 技能系统概述 / Skill System Overview

**中文：** 巫师3的技能系统分为四大类：剑术（战斗）、法印（魔法）、炼金术和一般能力。每类技能下有多个技能分支。技能需要装备到技能槽位才能生效——每升一级获得一个技能点，技能槽位随等级解锁。普通技能最多5级，随着等级提升效果增强。突变物可以插入技能槽中央的变异槽，提供额外的属性加成。

**English:** The Witcher 3 skill system has four trees: Combat (Swordsmanship), Signs (Magic), Alchemy, and General. Skills must be slotted to take effect. Each level grants 1 skill point. Skills have up to 5 ranks, scaling in effect. Mutagens slot into the center of the skill tree for bonus attributes.

| 技能树 / Tree | 核心作用 / Core Role | 关键技能数 / Key Skills |
|---|---|---|
| 剑术 / Combat | 物理输出、暴击、耐力 / Physical DPS, Crit, Stamina | 12+ |
| 法印 / Signs | 魔法控制、防护、元素伤害 / Magic, Control, Shield | 15+ |
| 炼金术 / Alchemy | 药剂、炸弹、剑油 / Potions, Bombs, Oils | 15+ |
| 一般 / General | 通用增益 / Universal Bonuses | 8+ |

{/* ===== 剑术技能树 / Combat Skill Tree ===== */}

## 剑术技能树 / Combat Skill Tree

**中文：** 剑术技能树专注提高杰洛特的物理近战能力。分为基础剑术、远程（十字弓）和近战进阶三部分。

**English:** The Combat tree focuses on Geralt's physical melee capability, divided into basic swordsmanship, crossbow, and advanced melee.

**基础剑术 / Basic Swordsmanship**:

| 技能 / Skill | 效果（满级5级）/ Effect (Max Rank 5) |
|---|---|
| 肌肉记忆 / Muscle Memory | 轻击伤害+25% / Light attack damage +25% |
| 精准重击 / Precise Strikes | 重击暴击率+25% / Heavy attack crit rate +25% |
| 回旋回避 / Fleet Footed | 闪避时机体伤害-100% / Invulnerable during dodge |
| 战斗狂热 / Combat Acumen | 肾上腺素生成+30% / Adrenaline generation +30% |

**进阶剑术 / Advanced Swordsmanship**:

| 技能 / Skill | 效果 / Effect |
|---|---|
| 剑舞 / Whirl | 按住轻击消耗耐力旋转攻击，每秒造成5次伤害 / Spin attack, 5 hits/sec |
| 怒气爆发 / Rend | 重击蓄力后释放，伤害+50%，50%破甲 / Charged heavy, +50% dmg, 50% armor pierce |
| 致命精准 / Killing Spree | 每个敌人击杀后暴击率+5% / Each kill grants +5% crit rate |
| 血流成河 / Bloodbath | 击杀后伤害+15%，可叠加 / Kill grants +15% damage, stacking |

**十字弓技能 / Crossbow Skills**:
- 精准射击：弩箭伤害+15% / Crossbow damage +15%
- 诱饵设计：弩箭攻击有几率击倒小型敌人 / Crossbow attacks may knock down small enemies

{/* ===== 法印技能树 / Sign Skill Tree ===== */}

## 法印技能树 / Sign Skill Tree

**中文：** 法印技能树强化五种法印的效果。每个法印有三个强化技能：基础强化、替代形态和持续性强化。

**English:** The Signs tree enhances all five Signs. Each Sign has three upgrade skills: base upgrade, alternate mode, and duration/persistence upgrade.

**阿尔德法印 / Aard**:
- 阿尔德强击：伤害和击退距离+25% / Damage and knockback +25%
- 阿尔德扫荡：替代形态，360度范围击退 / Alternate: 360-degree AoE knockback
- 阿尔德震荡：增加晕眩时间 / Increased stun duration

**亚克西法印 / Axii**:
- 亚克西迷惑：魅惑时间+25% / Charm duration +25%
- 亚克西傀儡师：替代形态，控制敌人攻击友军 / Alternate: puppet enemy to attack allies
- 亚克西思维控制：降低法印消耗 / Reduced Sign cost

**伊格尼法印 / Igni**:
- 伊格尼强化：火焰伤害+25% + 点燃几率 / Fire damage +25%
- 伊格尼喷射：替代形态，持续喷射火焰 / Alternate: continuous flame stream
- 伊格尼灼烧：增加燃烧持续时间 / Increased burn duration

**昆恩法印 / Quen**:
- 昆恩强化：护盾吸收量+25% / Shield absorption +25%
- 昆恩护盾：替代形态，护盾破裂后自动恢复 / Alternate: shield auto-regens after break
- 昆恩持续：护盾持续时间+30% / Shield duration +30%

**亚登法印 / Yrden**:
- 亚登强化：陷阱伤害+30%，减速效果+20% / Trap damage +30%, slow +20%
- 亚登炮台：替代形态，放置自动攻击炮台 / Alternate: auto-attack turret
- 亚登持续：陷阱持续时间+30% / Trap duration +30%

{/* ===== 炼金术技能树 / Alchemy Skill Tree ===== */}

## 炼金术技能树 / Alchemy Skill Tree

**中文：** 炼金术技能树是游戏中上限最高的技能分支。正确加点的炼金术可以让你的战斗力翻倍。

**English:** The Alchemy tree has the highest power ceiling in the game. Properly specced alchemy can double your combat effectiveness.

| 技能 / Skill | 效果（满级）/ Effect (Max Rank) |
|---|---|
| 耐性提高 / Heightened Tolerance | 毒素上限+60%，可以喝更多煎药 / Toxicity limit +60%, drink more decoctions |
| 组织变异 / Tissue Transformation | 每个已装备的绿色突变 +500 HP / Each green mutagen slot +500 HP |
| 协同作用 / Synergy | 突变物提供的属性加成+60% / Mutagen bonus +60% |
| 恢复力 / Refreshment | 喝药恢复生命值+25% / Potion use heals +25% |
| 效率提高 / Efficiency | 炸弹/剑油/药剂的制作量+1 / Crafting yield +1 |
| 传染性 / Contagion | 剑油效果+50% / Oil effectiveness +50% |

{/* ===== 突变系统 / Mutagen System ===== */}

## 突变系统 / Mutagen System

**中文：** 突变系统是石之心DLC引入的大型技能扩展，血与酒DLC进一步增加了四个终极突变。突变位于技能树中心的四个大圆形槽位中，可以插入突变技能，提供强大的被动效果。

**English:** The Mutagen system was introduced in Hearts of Stone and expanded in Blood and Wine with four Greater Mutagens. Mutagens fit into four central circular slots, granting powerful passives.

**终极突变 / Greater Mutagens**:

| 突变 / Mutagen | 效果 / Effect | 获得方式 / Acquisition |
|---|---|---|
| 致命精准 / Killing Spree | 剑术：使用剑舞时耐力不消耗 / Whirl costs no stamina | 血与酒主线 / B&W main quest |
| 魔法潮涌 / Magic Surge | 法印：法印暴击造成200%额外伤害 / Sign crits deal +200% dmg | 血与酒主线 / B&W main quest |
| 毒性之血 / Toxic Blood | 炼金：受到伤害时对周围敌人造成毒素伤害 / Take dmg = poison nearby enemies | 血与酒主线 / B&W main quest |
| 肾上腺素爆发 / Adrenaline Rush | 战斗击杀回复50%生命 / Kill restores 50% HP | 血与酒主线 / B&W main quest |

## 技能搭配推荐 / Recommended Builds

**中文：** 以下是经过验证的几种高效技能搭配。

**English:** Here are several proven high-efficiency skill builds.

**剑术专精 / Pure Combat**:
- 主动技能：肌肉记忆(5)、精准重击(5)、剑舞(5)、怒气爆发(5)、回旋回避(3)、致命精准(3)
- 突变搭配：3个红色大突变，1个致命精准终极突变 / 3 red Greater Mutagens, Killing Spree Greater Mutagen
- 装备：猫学派套装 / Cat School gear

**法印控制 / Sign Control**:
- 主动技能：伊格尼强化(5)、伊格尼喷射(3)、昆恩强化(5)、昆恩护盾(3)、亚登强化(5)、亚登炮台(3)
- 突变搭配：3个蓝色大突变，1个魔法潮涌终极突变 / 3 blue Greater Mutagens, Magic Surge Greater Mutagen
- 装备：狮鹫学派套装 / Griffin School gear

**炼金坦克 / Alchemy Tank**:
- 主动技能：耐性提高(5)、协同作用(5)、组织变异(5)、传染性(3)、恢复力(3)、效率提高(3)
- 突变搭配：3个绿色大突变，1个毒性之血终极突变 / 3 green Greater Mutagens, Toxic Blood Greater Mutagen
- 装备：熊学派套装 / Bear School gear

**混合流 / Hybrid Build**:
- 组合剑术和炼金，利用剑油和煎药配合近战输出 / Mix Combat + Alchemy for oil + decoction + sword synergy
- 核心：剑舞 + 耐性提高 + 协同作用 / Core: Whirl + Heightened Tolerance + Synergy
- 装备：飞狮学派套装（均衡型）/ Manticore School gear (balanced)

## 技能点获取 / Skill Point Sources

**English:** Over 100 skill points available in the game. Primary sources below.

| 升级至100级 / Leveling to 100 | 100 | 每次升级固定1点 / 1 per level |

| 血与酒突变任务 / B&W Mutagen Quest | 4 | 解锁终极突变时奖励 / Mutagen unlocks |
