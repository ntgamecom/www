## 任务系统概述 / Quest System Overview

**中文：** 巫师3的任务系统极其庞大，包含主线、支线、猎魔委托、寻宝任务和随机事件。主线决定游戏结局方向，支线影响关键角色命运和可选结局，猎魔委托是金钱和经验的主要来源，寻宝任务解锁各学派套装图纸。

**English:** The Witcher 3 has a massive quest system: main quests, side quests, witcher contracts, treasure hunts, and random encounters. Main quests determine endings, side quests affect character fates and ending options, contracts are the main gold/XP source, and treasure hunts unlock witcher school gear schematics.

| 任务类型 / Type | 数量 / Count | 重要性 / Importance |
|---|---|---|
| 主线 / Main Quests | 30+ | 决定结局 / Determine ending |
| 支线 / Side Quests | 90+ | 角色命运、装备奖励 / Character fates, gear |
| 猎魔委托 / Contracts | 40+ | 金钱经验来源 / Gold + XP |
| 寻宝 / Treasure Hunts | 30+ | 学派装备图纸 / School gear diagrams |

{/* ===== 主线流程 / Main Quest Flow ===== */}

## 主线流程 / Main Quest Flow

**中文：** 主线分为两条主要路线——寻找希里和对抗狂猎。关键选择点会记录在"剧情日志"中，影响最终结局。

**English:** The main story has two branches — finding Ciri and fighting the Wild Hunt. Key choices are tracked in the "Quest Log" and affect the ending.

**序幕 / Prologue — 白果园**:

1. **丁香与醋栗**：追踪叶奈法的线索，学习基础猎魔人工作 / Track Yennefer's trail, learn basic witcher work
2. **狩魔猎人的所见所闻**：完成白果园的猎魔委托，积累经验和金钱 / Complete contracts in White Orchard
3. **狂猎**：首次遭遇狂猎追兵，保护叶奈法 / First encounter with Wild Hunt, protect Yennefer

**第一章 / Act 1 — 威伦与诺维格瑞**:

1. **血腥男爵**：在乌鸦窝寻找希里的线索，面对血腥男爵的家庭悲剧 / Find Ciri's trail at Crow's Perch, face the Bloody Baron's family tragedy
2. **逃脱**：帮助男爵的妻子安娜和女儿塔玛拉 / Help the Baron's wife Anna and daughter Tamara
3. **诺维格瑞之火**：在诺维格瑞寻找丹德里恩和希里的线索 / Search Novigrad for Dandelion and Ciri's trail
4. **破碎之花**：与特莉丝一起揭开诺维格瑞地下世界 / Uncover Novigrad's underworld with Triss

**关键选择 / Key Choices**:
- 血腥男爵的结局：选择救男爵的女儿还是妻子 / The Baron's fate: save daughter or wife
- 凯拉·梅兹的命运：让她去凯尔莫罕还是杀了她 / Keira Metz: send to Kaer Morhen or kill her
- 特莉丝留下还是离开诺维格瑞 / Triss stays or leaves Novigrad

**第二章 / Act 2 — 史凯利格群岛**:

1. **国王已死**：参与史凯利格的王位选举 / Participate in Skellige's king election
2. **乌德维克之王**：帮助哈尔玛打败冰霜巨人 / Help Hjalmar defeat the ice giant
3. **恶灵缠身**：帮助克拉茨阻止恶灵 / Help Crach an Craite stop a vengeful spirit
4. **最后的准备**：召集盟友去凯尔莫罕 / Rally allies for the Battle of Kaer Morhen

**关键选择 / Key Choices**:
- 支持哈尔玛还是塞尔维斯成为史凯利格国王 / Support Hjalmar or Cerys as Skellige king
- 史凯利格王位继承者影响后期援军 / King choice affects endgame allies

**第三章 / Act 3 — 凯尔莫罕之战与结局**:

1. **凯尔莫罕之战**：保卫凯尔莫罕对抗狂猎，希里的命运在此决定 / Defend Kaer Morhen against the Wild Hunt, Ciri's fate is decided
2. **终极对决**：在斯提加城堡决战狂猎之王艾瑞汀 / Final battle against Eredin at Undvik
3. **白狼归来**：根据希里的状态获得不同结局 / Different endings based on Ciri's fate

**结局条件 / Ending Conditions**:
- 希里成为女皇：带希里去见恩希尔皇帝 + 完成相关支线 / Ciri becomes empress: visit Emhyr + related side quests
- 希里成为猎魔人：和希里保持亲密关系，支持她的决定 / Ciri becomes a witcher: maintain good relationship
- 希里死亡：做出太多错误选择 / Ciri dies: too many bad choices

{/* ===== 重要支线 / Key Side Quests ===== */}

## 重要支线 / Key Side Quests

**中文：** 以下支线强烈建议完成，它们影响角色命运、结局选项和最强装备获取。

**English:** These side quests are highly recommended — they affect character fates, ending options, and best gear.

**凯拉·梅兹 / Keira Metz**:
- 任务：魔法之灯、老鼠之塔 / Magic Lamp, Tower of Mice
- 影响：能否邀请凯拉去凯尔莫罕参战 / Whether Keira joins the Battle of Kaer Morhen
- 注意：老鼠之塔的选择直接影响凯拉的命运 / Tower of Mice choice directly affects Keira's fate

**特莉丝·梅丽葛德 / Triss Merigold**:
- 任务：生死交关、逃出生天 / Now or Never, Matter of Life and Death
- 影响：叶奈法与特莉丝的竞争关系 / The Yen vs Triss romance competition
- 重要：在"逃出生天"中选择表白确定感情线 / Declare romance in "Now or Never"

**叶奈法 / Yennefer**:
- 任务：最后的愿望 / The Last Wish
- 影响：确定和叶奈法的感情线 / Confirms Yennefer romance path
- 注意：如果同时向特莉丝和叶奈法表白会导致两者都失去 / Declaring for both Triss and Yen loses both

**血腥男爵 / Bloody Baron**:
- 任务：男爵的家族事务系列 / The Baron's family quest chain
- 影响：男爵命运、安娜命运、树之心选择 / Baron's fate, Anna's fate, Tree Heart choice
- 三个结局：男爵存活、男爵死亡、男爵带着妻子去治病 / Three endings: alive, dead, or leaves for treatment

**林中夫人 / Ladies of the Wood**:
- 任务：呢喃山丘 / Whispering Hillock
- 关键选择：释放树之心还是杀掉 / Free or kill the Tree Spirit
- 影响：直接影响血腥男爵支线的结局 / Directly affects the Bloody Baron's quest ending

{/* ===== DLC任务 / DLC Quests ===== */}

## DLC任务 / DLC Quests

**中文：** 两个大型DLC——石之心和血与酒——分别提供了10+小时的全新任务内容。

**English:** Two major DLCs — Hearts of Stone and Blood and Wine — each add 10+ hours of new quest content.

**石之心 / Hearts of Stone**:
- 触发条件：完成主线"血腥男爵"后，在诺维格瑞七猫旅馆接告示 / After "Bloody Baron", read notice at Seven Cats Inn
- 核心人物：镜子大师冈特·欧迪姆、欧吉尔德·冯·埃弗瑞克 / Gaunter O'Dimm, Olgierd von Everec
- 重要任务：芝麻开门、冤有债主、婚姻回忆 / Open Sesame, Dead Man's Party, Scenes from a Marriage
- 结局选择：帮助欧吉尔德对抗镜子大师，或帮助镜子大师收取灵魂 / Help Olgierd vs Gaunter, or let Gaunter claim the soul
- 奖励：爱丽丝（钢剑）、欧吉尔德的不死特性 / Iris (steel sword), Olgierd's immortality

**血与酒 / Blood and Wine**:
- 触发条件：等级40+，完成凯尔莫罕之战后 / Level 40+, after Battle of Kaer Morhen
- 核心人物：女爵安娜·亨利叶塔、狄拉夫 / Duchess Anna Henrietta, Detlaff
- 重要任务：酒国的使节、恶兽现身、终结之路 / Envoys of Wine, Beast of Toussaint, The End Path
- 结局选择：三条路线——杀狄拉夫、救狄拉夫、席安娜路线 / Three paths: kill Detlaff, save Detlaff, Syanna path
- 奖励：鸽派（银剑）、自己拥有的葡萄酒庄园 / Gesheft (silver sword), own vineyard

{/* ===== 猎魔委托 / Witcher Contracts ===== */}

## 猎魔委托指南 / Witcher Contracts

**中文：** 委托是主要的游戏内经济来源。每个委托可以通过对话讨价还价提高报酬。正确利用怪物的弱点是委托的关键。

**English:** Contracts are the main income source. Bargain for higher pay via Axii dialogue options. Exploit monster weaknesses for easier kills.

| 委托类型 / Type | 推荐法印 / Sign | 剑油 / Oil |
|---|---|---|
| 食尸鬼 / Ghouls | 伊格尼 / Igni | 食尸生物油 / Necrophage Oil |
| 灵体 / Wraiths | 亚登 / Yrden | 灵体油 / Specter Oil |
| 狼人 / Werewolves | 伊格尼（银制武器）/ Igni (moon dust bomb) | 诅咒生物油 / Cursed Oil |
| 狮鹫 / Griffins | 阿尔德（击落）/ Aard (ground them) | 混兽油 / Hybrid Oil |
| 吸血鬼 / Vampires | 昆恩（防御吸血鬼冲撞）/ Quen (block charge) | 吸血鬼油 / Vampire Oil |
- 讨价还价技巧：使用亚克西法印可以提升报酬10%-20% / Use Axii to increase payment by 10%-20%
- 调查现场仔细：阅读委托板上的详细信息可以了解怪物行为模式 / Read contract details for monster behavior patterns
- 昆特牌：不要忘记和委托人打昆特牌收集卡牌 / Don't forget to play Gwent with contractors

## 寻宝任务 / Treasure Hunts

**中文：** 寻宝任务是获取学派装备图纸的主要方式。每个学派的图纸需要完成对应的寻宝任务才能获得。

**English:** Treasure hunts are the primary way to obtain school gear diagrams. Each school's diagrams require their own treasure hunt quest.

| 学派 / School | 起始任务 / Starting Quest | 起始位置 / Location |
|---|---|---|
| 猫学派 / Cat | 猫学派装备寻宝图 / Cat School Gear | 诺维格瑞七猫旅馆告示板 / Seven Cats Inn notice board |
| 熊学派 / Bear | 熊学派装备寻宝图 / Bear School Gear | 史凯利格凯尔缪雷旅店 / Kaer Muire inn, Skellige |
| 狮鹫学派 / Griffin | 狮鹫学派装备寻宝图 / Griffin School Gear | 威伦吊死鬼之树 / Hanged Man's Tree, Velen |
| 狼学派 / Wolf | 狼学派装备寻宝图 / Wolf School Gear | 凯尔莫罕城堡 / Kaer Morhen Castle |
| 飞狮学派 / Manticore | 飞狮学派装备寻宝图 / Manticore School Gear | 陶森特白鸦葡萄园 / White Orchard, Toussaint |

**寻宝提示 / Treasure Hunt Tips**:
- 每个寻宝任务有多个藏宝点，需要用猎魔人感官追踪 / Each hunt has multiple cache points to track with Witcher Senses
- 藏宝点通常被怪物守卫 / Cache points are usually guarded by monsters
- 获得图纸后找对应等级的工匠制作 / Take diagrams to a craftsman of matching rank
