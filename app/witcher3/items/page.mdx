## 突变诱发物 / Mutagens

颜色决定了技能行左端的属性加成——**颜色匹配时效果加倍** / Color determines row bonus — **matching color doubles effect**.

| 颜色 / Color | 效果 / Effect | 匹配加成 / Matched Bonus |
|---|---|---|
| 红色 / Red | +攻击力 / +Attack Power | +攻击力 ×2 |
| 蓝色 / Blue | +法印强度 / +Sign Intensity | +法印强度 ×2 |
| 绿色 / Green | +生命值 / +HP | +生命值 ×2 |
| 无色 / Colorless | +经验值 / +XP | — |

**次世代更新**：次世代版本中突变诱发物可以拆解、升级、合成——蓝/红/绿三种碎片10:1合成 / Next-gen: mutagens can be dismantled, upgraded, and crafted (10:1 ratio for pieces → full)

## 符文之语 / Runewords (石之心 DLC)

在符文匠（Runewright）处给装备镶嵌符文之语——3个等级 / Runewright in Hearts of Stone adds runewords to equipment — 3 tiers.

| 等级 / Tier | 费用 / Cost | 解锁效果 / Effect |
|---|---|---|
| 1级 | 5000金 | 基础符文 / Basic runes |
| 2级 | 15000金 | 中型符文之语——3个插槽组合触发特效 / Medium runewords — 3-slot combos |
| 3级 | 25000金 | 高级符文之语——最强特效 / Advanced runewords — best effects |

**推荐符文之语 / Recommended Runewords**：
- **平衡 / Balance** (3×红色): 轻甲视为中甲——触发狮鹫学派技巧同时穿轻甲 / Light armor counts as medium — triggers Griffin Techniques in light gear
- **魔法师 / Mage** (3×蓝色): 法印释放不消耗耐力——无限法印 / Signs cost no stamina — unlimited casting
- **防暴 / Eruption** (3×红色): 击杀敌人时产生火焰爆炸 / Kill = fire explosion
- **漩涡 / Whirl** (3×红色): 回旋技能可边移动边释放 / Whirl can be used while moving

## Gwent卡牌 / Gwent Cards

昆特牌共约 **300+张**——分为五大阵营 / ~300+ cards in 5 factions.

| 阵营 / Faction | 特点 / Trait | 核心卡 / Key Cards |
|---|---|---|
| 北方领域 / Northern Realms | 获胜抽一张 / Win = draw one | 丹德里恩、攻城系列 |
| 尼弗迦德 / Nilfgaard | 平局胜 / Tie goes to Nilfgaard | 希里、莫夫兰·沃里斯 |
| 松鼠党 / Scoia'tael | 先手选择 / Choose who goes first | 矮人突击队、精灵弓箭手 |
| 怪物 / Monsters | 随机留场一张 / Random card stays | 吸血鬼、食尸鬼集群 |
| 史凯利杰 / Skellige | 第一轮结束后死亡牌回场 / Dead cards return round 2 | 奎特家族、燃烧 |

**全收集指南 / Completion Guide**：
1. 每个**旅店老板**都卖基础卡——全部买完 / Buy all cards from every innkeeper
2. **主要NPC牌手**（血腥男爵、卓尔坦、帕西弗罗拉等）/ Key NPC opponents
3. **昆特牌任务**——"大赌局"、"昆特高手"等 / Gwent quests — "High Stakes", "Gwent Master" etc.
4. **陶森特**——血与酒DLC新增约50张卡 / Toussaint — ~50 new cards in Blood and Wine
