## 任务系统概述 / Quest System Overview

**中文：** 剑星的任务分为主线任务和支线任务。主线任务推动剧情发展，解锁新区域和Boss战。支线任务提供经验值、技能点、装备和服装收集。部分支线有时间段限制，需要在推进主线前完成。

**English:** Stellar Blade's quests are divided into main quests and side quests. Main quests advance the story and unlock new areas and boss fights. Side quests provide XP, skill points, gear, and costume collectibles. Some side quests have time limits and must be completed before main story progression.

| 任务类型 / Type | 说明 / Description |
|---|---|
| 主线 / Main Quests | 核心剧情推进 / Core story progression |
| 支线 / Side Quests | 收集品、技能点、服装 / Collectibles, skill points, costumes |
| 告示板委托 / Bulletin Board | 重复性任务，刷经验材料 / Repeatable, farm XP and materials |

{/* ===== 主线流程 / Main Quest Flow ===== */}

## 主线流程 / Main Quest Flow

**中文：** 剑星的主线剧情分为七个主要章节，从第七空降小队抵达地球开始，到最终决战Alpha Nayptil结束。

**English:** The main story is divided into seven chapters, from the 7th Airborne Squad's arrival on Earth to the final battle against Alpha Nayptil.

| 章节 / Chapter | 区域 / Area | 核心事件 / Key Event |
|---|---|---|
| 第一章 / Ch 1 | 埃多斯7号 / Eidos 7 | 降落地球，首次遭遇奈提巴 / Land on Earth, first Naytiba encounter |
| 第二章 / Ch 2 | 大沙漠 / Great Desert | 穿越废土，寻找避难所 / Cross wasteland, find refuge |
| 第三章 / Ch 3 | 埃多斯城 / Eidos City | 探索城市废墟 / Explore city ruins |
| 第四章 / Ch 4 | 矩阵11号 / Matrix 11 | 深入地下设施 / Descend into underground facility |
| 第五章 / Ch 5 | 轨道电梯 / Space Elevator | 攀登轨道电梯 / Climb the space elevator |
| 第六章 / Ch 6 | 太空站 / Space Station | 真相揭露 / Truth revealed |
| 第七章 / Ch 7 | 最终区域 / Final Area | 最终Boss战 / Final boss battle |

**全章节关键节点 / Key Story Beats**:

- **第一章末尾**: 解锁技能系统和营地传送 / Unlock skill system and camp fast travel
- **第二章**: 获得无人机战斗模块 / Acquire drone combat module
- **第三章**: 遇到莉莉，解锁支线任务系统 / Meet Lily, unlock side quest system
- **第四章**: 爆发技能解锁 / Burst skills unlocked
- **第五章**: 获得最终武器升级 / Final weapon upgrade
- **第六章**: 关键剧情转折，影响结局选择 / Key story twist, affects ending choice
- **第七章**: 最终选择——多个结局 / Final choice — multiple endings

## 莉莉好感度系统 / Lily Affinity System

**中文：** 莉莉是伊芙的同伴，她的好感度影响结局解锁。通过完成莉莉的专属支线和给她礼物提升好感度。

**English:** Lily is Eve's companion. Her affinity affects ending unlocks. Increase affinity through Lily-specific side quests and gift-giving.

**好感度提升方式 / Affinity Methods**:
- 完成莉莉的支线任务：+30好感度 / Complete Lily's side quests: +30 affinity
- 给莉莉礼物（收集的特殊物品）：+5~15好感度 / Give Lily gifts (special collectibles): +5~15 affinity
- 带莉莉战斗（莉莉作为AI随从）：每次战斗胜利+1 / Fight with Lily (AI companion): +1 per battle win

## 告示板委托指南 / Bulletin Board

**中文：** 告示板委托是可重复接取的刷素材任务。每个区域都有独立的告示板，完成任务后刷新新委托。

**English:** Bulletin board requests are repeatable material farming quests. Each region has its own board that refreshes with new requests after completion.

**委托类型 / Request Types**:

| 类型 / Type | 要求 / Requirement | 奖励 / Reward |
|---|---|---|
| 狩猎委托 / Hunt Request | 击败指定数量的特定敌人 / Defeat specific number of enemies | 金币+材料 / Gold + materials |
| 收集委托 / Gather Request | 收集指定数量的材料 / Collect specific number of materials | 金币+经验 / Gold + XP |
| 钓鱼委托 / Fish Request | 钓到指定鱼种 / Catch specific fish type | 金币+技能点道具 / Gold + skill point item |
| 摄影委托 / Photo Request | 拍摄指定场景的照片 / Take photo of specific location | 金币+外观道具 / Gold + cosmetic item |

## 全结局解锁路线 / Ending Route Map

**中文：** 以下是解锁所有结局的路线图。

**English:** Route map for unlocking all endings.

1. **一周目**：随便选择，解锁"回归"或"牺牲"结局——不需要刻意收集 / First playthrough: choose freely, unlock "Return" or "Sacrifice" — no special collection needed
2. **二周目**：完成全支线+莉莉好感度MAX，解锁"觉醒"或"真结局"/ NG+: complete all side quests + max Lily affinity for "Awakening" or "True Ending"
3. **速通**：记录全结局后再选择自己喜欢的结局存档继续探索 / Speedrun: record all endings, then continue with preferred ending save for exploration
4. **白金**：真结局需要全支线+全收集品——建议在二周目完成 / Platinum: True Ending requires all side quests + all collectibles — recommended in NG+
