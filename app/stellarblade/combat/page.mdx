# 战斗与技能 / Combat & Skills

剑星的战斗系统深度远超表面所见。理解技能树、装备搭配和连招机制是成为高手的必经之路 / Stellar Blade's combat system runs deeper than it first appears. Mastering the skill tree, gear builds, and combo mechanics is the path to mastery.

---

## 技能树详解 / Skill Tree Breakdown

技能树分为三大分支：贝塔、爆发、连招 / The skill tree has three branches: Beta, Burst, and Combo.

### 贝塔技能 / Beta Skills

消耗贝塔能量释放的强力技能，是战斗中主要爆发手段 / Powerful skills consuming Beta energy — your main burst tool in combat.

| 技能 / Skill | 能量消耗 / Cost | 效果 / Effect | 推荐度 / Priority |
|------|:----:|------|:----:|
| **贝塔冲击 / Beta Strike** | 1格 | 前方大范围剑气，削韧值高 / Wide frontal slash, high stagger | ★★★★★ |
| **闪电突击 / Lightning Rush** | 1格 | 快速突进+多段攻击 / Fast gap-closer + multi-hit | ★★★★☆ |
| **破盾斩 / Shield Breaker** | 1格 | 专破持盾敌人，伤害一般 / Specifically breaks guarded enemies, mediocre damage | ★★★☆☆ |
| **升龙斩 / Rising Slash** | 2格 | 浮空敌人→空中追击 / Launches enemies → air combo follow-up | ★★★★☆ |
| **旋风流 / Blade Storm** | 2格 | 大范围持续旋转攻击，清场神技 / Wide AoE spinning attack, excellent for clearing | ★★★★★ |
| **裂地斩 / Ground Splitter** | 3格 | 高伤害地面AOE，对大体型敌人额外伤害 / High-damage ground AoE, bonus vs large enemies | ★★★★☆ |

> **优先学习**：贝塔冲击 → 旋风流 → 闪电突击。破盾斩视情况 / **Priority**: Beta Strike → Blade Storm → Lightning Rush. Shield Breaker situationally.

### 爆发技能 / Burst Skills

消耗爆发能量的终极技能，通过完美闪避/完美格挡积攒 / Ultimate skills consuming Burst energy, gained from perfect dodges/parries.

| 技能 / Skill | 能量消耗 / Cost | 效果 / Effect | 推荐度 / Priority |
|------|:----:|------|:----:|
| **爆发连斩 / Burst Combo** | 1格 | 快速追击连续斩击，泛用输出 / Rapid follow-up slashes, versatile DPS | ★★★★★ |
| **审判之剑 / Sword of Judgment** | 1格 | 单体高伤+必定打断敌人动作 / High single-target damage + guaranteed interrupt | ★★★★☆ |
| **时空减速 / Time Slow** | 2格 | 全场减速，创造巨大输出窗口 / Full-field slow, creates massive damage window | ★★★★★ |
| **护盾爆发 / Shield Burst** | 2格 | 大范围爆炸+自身无敌帧 / Wide explosion + invincibility frames | ★★★★☆ |
| **终结技 / Final Strike** | 3格 | 极高单体伤害，处决效果 / Extreme single-target damage, execution effect | ★★★★☆ |

> **优先学习**：爆发连斩 → 时空减速 → 终结技 / **Priority**: Burst Combo → Time Slow → Final Strike.

### 连招技能 / Combo Skills

扩展基础连招的被动技能，改变□和△的攻击模式 / Passive skills that extend basic combos, modifying □ and △ attack patterns.

| 技能 / Skill | 效果 / Effect | 推荐度 / Priority |
|------|------|:----:|
| **连招扩展1 / Combo Extension 1** | □□□□后追加一段攻击 / Add an extra hit after □□□□ | ★★★★★ |
| **连招扩展2 / Combo Extension 2** | □□□□□后追加终结击 / Add a finisher after □□□□□ | ★★★★★ |
| **蓄力重击 / Charge Heavy** | 按住△蓄力，释放强力破防攻击 / Hold △ for a charged guard-breaking strike | ★★★★☆ |
| **空中连击 / Air Combo** | 浮空后可追击更多段 / Extra air hits after launch | ★★★☆☆ |
| **反击追击 / Counter Pursuit** | 完美格挡后可接特殊追击 / Special follow-up after perfect parry | ★★★★★ |
| **闪避反击 / Dodge Counter** | 完美闪避后可接特殊反击 / Special counter after perfect dodge | ★★★★★ |

> **优先学习**：连招扩展 → 反击追击 → 闪避反击 → 蓄力重击 / **Priority**: Combo Extensions → Counter Pursuit → Dodge Counter → Charge Heavy.

## 装备系统 / Gear System

### 外脊装备 / Exospine

外脊是核心装备，提供流派定义性的加成 / Exospine defines your build with transformative bonuses.

| 外脊 / Exospine | 获取方式 / Location | 效果 / Effect | 适用流派 / Build |
|------|------|------|------|
| **速度外脊 / Speed Exospine** | 废土主线 / Wasteland main quest | 攻击速度+20%，闪避距离+30% / +20% attack speed, +30% dodge distance | 快速攻击流 / Fast Attack |
| **力量外脊 / Strength Exospine** | 大沙漠主线 / Great Desert main quest | 攻击力+25%，防御-15% / +25% attack, -15% defense | 玻璃大炮 / Glass Cannon |
| **防御外脊 / Defense Exospine** | 废土支线 / Wasteland side quest | 防御+30%，受到伤害-20% / +30% defense, -20% damage taken | 坦克流 / Tank |
| **贝塔外脊 / Beta Exospine** | 大沙漠支线 / Great Desert side quest | 贝塔能量获取+40%，爆发能量获取-20% / +40% Beta gain, -20% Burst gain | 贝塔技能流 / Beta Build |
| **爆发外脊 / Burst Exospine** | 后期区域 / Late game area | 爆发能量获取+40%，贝塔能量获取-20% / +40% Burst gain, -20% Beta gain | 爆发技能流 / Burst Build |
| **生命外脊 / Vitality Exospine** | 钓鱼奖励 / Fishing reward | HP+40%，药水回复量+50% / +40% HP, +50% potion heal | 生存流 / Survival |

### 身体核心 / Body Cores

核心升级提升基础属性上限 / Core upgrades increase base stat caps.

| 核心类型 / Core Type | 效果 / Effect | 最大可获取数 / Max Obtainable |
|------|------|:----:|
| **HP核心 / HP Core** | 提升最大HP / Increase max HP | 约30个 / ~30 |
| **贝塔核心 / Beta Core** | 提升贝塔能量上限 / Increase Beta cap | 约15个 / ~15 |
| **爆发核心 / Burst Core** | 提升爆发能量上限 / Increase Burst cap | 约10个 / ~10 |
| **武器核心 / Weapon Core** | 提升基础攻击力 / Increase base attack | 约8个 / ~8 |

**寻找核心的小技巧 / Core Hunting Tips**:
- 探索地图每个角落——核心通常藏在不显眼的位置 / Explore every corner — cores are often hidden in obscure spots
- 完成支线任务获得核心奖励 / Complete side quests for core rewards
- 精英敌人首次击杀掉落核心 / Elite enemies drop cores on first kill
- 钓鱼获得的箱子可能开出核心 / Fishing chests may contain cores

## 无人机系统 / Drone System

无人机是你的远程伙伴，在战斗中提供火力支援 / Your drone provides ranged support in combat.

### 无人机升级 / Drone Upgrades

在营地使用材料升级无人机 / Upgrade the drone at camps using materials.

| 升级 / Upgrade | 效果 / Effect | 材料 / Materials |
|------|------|------|
| **攻击力 / Attack** | 增加射击伤害 / Increase shot damage | 金币 + 零件 / Gold + Parts |
| **射速 / Fire Rate** | 增加射击频率 / Increase fire rate | 金币 + 零件 / Gold + Parts |
| **弹夹容量 / Magazine** | 增加弹夹上限 / Increase magazine size | 金币 + 弹药包 / Gold + Ammo Packs |
| **特殊射击 / Special Shot** | 解锁蓄力射击 / Unlock charged shot | 金币 + 核心 / Gold + Cores |

### 无人机技能 / Drone Skills

| 技能 / Skill | 效果 / Effect | 解锁方式 / Unlock |

| **普通射击 / Normal Shot** | 三连发基础射击 / 3-round basic shot | 默认 / Default |
| **蓄力射击 / Charged Shot** | 单发高伤，削韧值高 / Single high-damage shot, high stagger | 无人机升级 / Drone upgrade |
| **导弹射击 / Missile Shot** | 追踪导弹，适合空中敌人 / Tracking missile, great for airborne enemies | 技能树 / Skill tree |
| **弹幕 / Barrage** | 全弹发射，大范围AOE / Full salvo, wide AoE | 技能树 / Skill tree |

## 战斗进阶技巧 / Advanced Combat Tips

| 技巧 / Tip | 说明 / Description |
|------|------|
| **取消后摇 / Animation Cancel** | 轻攻击最后一段可以用闪避或格挡取消硬直，提前进入防御状态 / Cancel the final hit's recovery with dodge or guard for earlier defense |
| **切换目标 / Target Switch** | 不要死锁一个敌人——利用R3切换目标，兼顾多方向威胁 / Don't tunnel one enemy — use R3 to switch targets and maintain awareness |
| **空中优势 / Air Advantage** | 升龙斩浮空后→空中连击→落地前接爆发技能，打出高额伤害 / Launch → air combo → Burst skill before landing for massive damage |
| **贝塔能量管理 / Beta Management** | 平时用□连招攒贝塔，不要一有就放。留1格应急闪避或格挡 / Build Beta with □ combos, don't spend immediately. Keep 1 bar for emergency |
| **环境利用 / Environment** | 部分场景有可爆炸物或悬崖——把敌人引过去造成额外伤害 / Some areas have explosives or ledges — bait enemies into them for extra damage |
