# 剑星 攻略 / Stellar Blade Guide

剑星（Stellar Blade）是由Shift Up开发的科幻动作冒险游戏。你将扮演伊芙（Eve），在末世地球上与入侵的奈提巴（Naytiba）战斗，拯救人类最后的希望。本攻略涵盖从入门到白金的全方位指导 / Stellar Blade is a sci-fi action-adventure game developed by Shift Up. You play as Eve, fighting against the invading Naytibas on a post-apocalyptic Earth to save humanity's last hope. This guide covers everything from beginner tips to platinum trophy guidance.

**平台 / Platform**: PlayStation 5 | **发售日 / Release**: 2024年4月26日 / April 26, 2024 | **类型 / Genre**: 动作冒险 / Action Adventure

## 游戏概况 / Game Overview

| 项目 / Feature | 说明 / Description |
|------|------|
| **开发商 / Developer** | Shift Up（韩国）/ Shift Up (South Korea) |
| **总监 / Director** | 金亨泰 / Hyung-Tae Kim |
| **游戏类型 / Genre** | 科幻动作冒险 / Sci-fi Action Adventure |
| **视角 / Perspective** | 第三人称 / Third-person |
| **战斗风格 / Combat Style** | 快节奏近战 + 远程无人机 / Fast-paced melee + ranged drone |

## 关键特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **流畅的战斗系统 / Fluid Combat** | 融合闪避、格挡、反击的即时动作系统，节奏紧凑 / Real-time action blending dodge, parry, and counter for fast-paced combat |
| **丰富的技能树 / Deep Skill Tree** | 贝塔技能、爆发技能、连招技能三大分支，高度自定义 / Three branches: Beta, Burst, and Combo skills for deep customization |
| **装备定制 / Gear Customization** | 外脊装备（Exospine）+ 身体核心（Body Core）+ 无人机升级 / Exospine + Body Core + Drone upgrades |
| **多结局叙事 / Multiple Endings** | 玩家的选择影响最终结局（3种结局）/ Player choices affect the ending (3 endings) |
| **服装收集 / Costume Collection** | 超过30套可收集服装，部分需要探索隐藏区域 / 30+ collectible costumes, some requiring hidden area exploration |
| **BOSS挑战模式 / Boss Challenge** | 通关后解锁BOSS RUSH模式，挑战极限 / Post-game Boss Rush mode for ultimate challenge |
| **钓鱼小游戏 / Fishing Minigame** | 在绿洲放松钓鱼，收集鱼类图鉴 / Relax at the Oasis with fishing, complete the fish encyclopedia |
| **新游戏+ / New Game+** | 继承装备和技能开始二周目，新增更高难度 / Carry over gear and skills into NG+ with increased difficulty |
