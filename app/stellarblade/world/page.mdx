# 世界与收集 / World & Collectibles

剑星的世界分为多个风格迥异的区域，每个都充满了隐藏的秘密和收集品。本章详解全区域的探索要点和收集品位置 / Stellar Blade's world is divided into several distinct regions, each full of hidden secrets and collectibles. This chapter covers exploration tips and collectible locations for all areas.

---

## 区域总览 / Region Overview

| 区域 / Region | 类型 / Type | 主线阶段 / Story Phase | 特点 / Features |
|------|------|:----:|------|
| **埃多斯7号 / Eidos 7** | 线性关卡 / Linear | 序章-第2章 / Prologue-Ch.2 | 城市废墟，入门区域 / Urban ruins, introductory area |
| **废土 / Wasteland** | 开放区域 / Open Zone | 第3-5章 / Ch.3-5 | 荒漠废墟，最大的可探索区域 / Desert ruins, largest explorable zone |
| **大沙漠 / Great Desert** | 开放区域 / Open Zone | 第6-8章 / Ch.6-8 | 广袤沙漠，地下遗迹 / Vast desert with underground ruins |
| **脊柱 / The Spire** | 线性关卡 / Linear | 第9-10章 / Ch.9-10 | 最终区域，高塔攀登 / Final area, tower ascent |
| **绿洲 / Oasis** | 安全区域 / Safe Zone | 废土阶段解锁 / Unlocked in Wasteland | 营地、商人、钓鱼点 / Camp, merchant, fishing spot |

## 埃多斯7号 / Eidos 7

### 可收集物品 / Collectibles

| 物品类型 / Type | 数量 / Count | 说明 / Notes |
|------|:----:|------|
| **文件 / Documents** | 8 | 分布在各个房间和走廊 / Scattered across rooms and corridors |
| **罐子 / Cans** | 3 | 需要破坏所有易碎物品才能发现 / Requires breaking all breakable objects |
| **服装 / Costumes** | 2 | 一条在隐藏房间，一条在主线路径上 / One in hidden room, one on main path |
| **身体核心 / Body Cores** | 3 | BOSS战奖励 + 探索 / Boss rewards + exploration |

### 探索要点 / Exploration Tips

- 序章区域在事件后**无法返回**，确保一次性收集完 / The prologue area is **unreturnable** after the event — collect everything in one pass
- 每个有锁的门都需要找到对应的钥匙卡 / Locked doors require finding corresponding keycards
- 易碎墙壁后用无人机导弹击碎 / Breakable walls can be destroyed with drone missiles

## 废土 / Wasteland

### 可收集物品 / Collectibles

| 物品类型 / Type | 数量 / Count | 说明 / Notes |
|------|:----:|------|
| **文件 / Documents** | 25 | 分布在各个废墟建筑中 / Scattered across ruined buildings |
| **罐子 / Cans** | 16 | 部分需要完成解谜才能获得 / Some require puzzle completion |
| **服装 / Costumes** | 8 | 包括主线奖励、支线奖励、隐藏位置 / Main quest, side quest, and hidden locations |
| **身体核心 / Body Cores** | 10 | 探索 + 精英 + 支线 / Exploration + elites + side quests |
| **传送点 / Waypoints** | 6 | 激活后快速移动 / Activate for fast travel |

### 重要地点 / Key Locations

| 地点 / Location | 说明 / Description |
|------|------|
| **废土入口 / Wasteland Entrance** | 主线到达的第一个区域。有商人 / First area on arrival. Has a merchant |
| **中部废墟 / Central Ruins** | 最大的建筑群，多层探索 / Largest building complex, multi-floor exploration |
| **地下设施 / Underground Facility** | 需要主线推进才能进入。有重要BOSS战 / Requires story progression to enter. Key boss fight |
| **绿洲 / Oasis** | 安全区。商人、钓鱼、休息 / Safe zone. Merchant, fishing, rest |
| **太阳能塔 / Solar Tower** | 高处眺望点，有收集品 / High vantage point with collectibles |

### 探索要点 / Exploration Tips

- **白天和夜晚**部分区域可访问性不同——有些收集品只能在特定时间获得 / Some areas differ between **day and night** — certain collectibles are time-locked
- 废土地图有很多垂直结构——注意寻找可攀爬的墙面 / The Wasteland has lots of verticality — look for climbable walls
- 沙漠中的流沙会把你拉下去——但下方可能有隐藏通道 / Quicksand pulls you down — but there may be hidden passages below
- 绿洲的钓鱼点是游戏唯一钓鱼处，建议尽早解锁 / The Oasis fishing spot is the only one in the game — unlock it early

## 大沙漠 / Great Desert

### 可收集物品 / Collectibles

| 物品类型 / Type | 数量 / Count | 说明 / Notes |
|------|:----:|------|
| **文件 / Documents** | 28 | 分布广泛，部分在隐藏洞穴 / Widespread, some in hidden caves |
| **罐子 / Cans** | 20 | 大沙漠是罐子最多的区域 / Great Desert has the most cans |
| **服装 / Costumes** | 10 | 包括需要完成特定支线才能获取的 / Including some locked behind specific side quests |
| **身体核心 / Body Cores** | 12 | 探索 + 精英 + 支线 / Exploration + elites + side quests |
| **传送点 / Waypoints** | 8 | 覆盖全区域 / Covering the entire zone |

### 重要地点 / Key Locations

| 地点 / Location | 说明 / Description |
|------|------|
| **沙漠入口 / Desert Entrance** | 从废土进入大沙漠的入口 / Entry point from Wasteland |
| **地下遗迹 / Underground Ruins** | 超大型地下迷宫，主线必经之路 / Massive underground labyrinth, required for main story |
| **绿洲营地 / Oasis Camp 2** | 第二个安全区，有NPC和商人 / Second safe zone with NPCs and merchant |
| **古代神殿 / Ancient Temple** | 隐藏区域，有强力BOSS和独特奖励 / Hidden area with powerful boss and unique rewards |
| **沙漠堡垒 / Desert Fortress** | 大型建筑群，多层结构 / Large building complex, multi-level |

### 探索要点 / Exploration Tips

- 大沙漠比废土大得多——善用传送点快速移动 / Great Desert is much larger than Wasteland — use waypoints liberally
- 地下遗迹容易迷路——建议从入口开始向右探索 / The Underground Ruins are easy to get lost in — explore right from the entrance
- 沙漠中有热浪天气效果，会持续掉血——尽快移动到室内或阴影区域 / Heat wave weather causes continuous HP drain — move indoors or to shade
- 隐藏洞穴入口通常被石块或藤蔓遮挡，用无人机攻击清除 / Hidden cave entrances are often blocked by rocks or vines — clear with drone attacks

## 脊柱 / The Spire

### 可收集物品 / Collectibles

| 物品类型 / Type | 数量 / Count | 说明 / Notes |
|------|:----:|------|
| **文件 / Documents** | 12 | 最终区域，文件集中在主线路径上 / Final zone, documents along main path |
| **罐子 / Cans** | 6 | 散落在塔内各层 / Scattered across tower floors |
