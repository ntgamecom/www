## 终局概述 / Endgame Overview

**中文：** 剑星的终局内容包括完成主线结局、新游戏Plus（NG+）模式、Boss挑战模式和全收集。主线完成后可以继续探索并完成未完成的支线和收集。NG+继承技能和部分装备，同时增加新的敌人配置和更高级的装备。

**English:** Stellar Blade's endgame includes completing the main story endings, New Game+ mode, Boss Challenge mode, and full completion. After the main story, you can continue exploring for unfinished quests and collectibles. NG+ carries over skills and some gear, adds new enemy configurations and higher-tier equipment.

## 结局路线详解 / Ending Routes

**中文：** 剑星有四个结局——普通结局、牺牲结局、觉醒结局和真结局。其中真结局是全收集玩家的最终目标。

**English:** Stellar Blade has four endings — Return, Sacrifice, Awakening, and True Ending. The True Ending is the ultimate goal for completionists.

**解锁真结局的条件 / True Ending Requirements**:

1. 完成全部支线任务 / Complete all side quests
2. 莉莉好感度达到最高 / Max Lily affinity
3. 收集全部50+文件 / Collect all 50+ documents
4. 收集全部30+录音 / Collect all 30+ audio logs
5. 在最终章节前完成"记忆拼图"支线 / Complete "Memory Puzzle" side quest before final chapter

**真结局流程 / True Ending Flow**:
1. 最终选择中出现第三个选项（觉醒）/ A third option (Awakening) appears in the final choice
2. 伊芙发现世界的真相——人类和奈提巴的起源 / Eve discovers the truth about humanity and Naytiba origins
3. 莉莉牺牲自己激活远古装置 / Lily sacrifices herself to activate the ancient device
4. 伊芙用装置重塑世界，人类和奈提巴共存 / Eve reshapes the world using the device, humans and Naytiba coexist
5. 隐藏过场动画揭示新世界的和平景象 / Hidden cutscene shows the peaceful new world

## 新游戏Plus指南 / New Game+ Guide

**中文：** NG+在主线通关后解锁。继承所有技能、服装和收集品进度。敌人配置发生变化——部分区域会出现新敌人类型。同时解锁更高级的武器升级。

**English:** NG+ unlocks after completing the main story. Carries over all skills, costumes, and collectible progress. Enemy configurations change — new enemy types appear in some areas. Higher-tier weapon upgrades are also unlocked.

**NG+继承内容 / NG+ Carry Over**:

| 继承 / Carries Over | 不继承 / Does Not Carry Over |
|---|---|
| 所有技能 / All skills | 任务进度 / Quest progress |
| 服装和外观 / Costumes & cosmetics | 钥匙物品 / Key items |
| 收集品进度 / Collectible progress | 传送点解锁 / Fast travel unlocks |
| 武器等级 / Weapon levels | 部分剧情物品 / Some story items |
| 无人机等级 / Drone levels | 贝塔/爆发能量槽上限 / Beta/Burst energy caps (re-earned) |

**NG+新增内容 / NG+ New Content**:
- 新敌人颜色变种，攻击模式略有变化 / New enemy color variants with slightly different attack patterns
- 武器最高等级扩展至X级 / Weapon max level extended to X
- 新增隐藏Boss"终极伊芙" / New hidden boss "Ultimate Eve"
- 新增服装×2 / 2 new costumes

## Boss挑战模式 / Boss Challenge Mode

**中文：** 通关后解锁Boss挑战模式。在营地选择"挑战"选项，可以依次挑战全部Boss。挑战模式会记录最佳时间和无伤成绩。

**English:** Boss Challenge mode unlocks after completing the game. Select "Challenge" at camps to fight all bosses in sequence. The mode records best times and no-hit runs.

**Boss挑战顺序 / Boss Challenge Order**:

1. 第一波：前期精英奈提巴×3 / Wave 1: 3 early elite Naytibas
2. 第二波：中期Boss×2 / Wave 2: 2 mid-game bosses
3. 第三波：后期Boss×2 + 精英奈提巴×2 / Wave 3: 2 late-game bosses + 2 elite Naytibas
4. 最终波：Alpha Nayptil + 隐藏Boss / Final Wave: Alpha Nayptil + hidden boss

**挑战模式奖励 / Challenge Rewards**:
- 无伤击败任何Boss：获得"完美战士"成就 / No-hit any boss: "Perfect Warrior" achievement
- 全波次挑战完成：获得金色武器外观 / Complete all waves: gold weapon skin
- 全Boss无伤：解锁隐藏技能"时停" / All bosses no-hit: unlock hidden skill "Time Stop"

## 终局收集路线 / Endgame Collection Route

**中文：** 终局阶段建议按以下顺序完成全收集。

**English:** Recommended order for endgame full completion.

1. **优先完成所有支线**：支线提供技能点、关键装备和莉莉好感度 / Complete all side quests first — they provide skill points, key gear, and Lily affinity
2. **全服装收集**：使用扫描器寻找隐藏通道，每个章节仔细探索 / Collect all costumes: use scanner for hidden passages, explore each chapter thoroughly
3. **全文件录音**：文件50+和录音30+分布在所有章节，从第一章开始按顺序收集 / Collect all 50+ documents and 30+ audio logs, start from Ch 1 in order
4. **Boss挑战模式**：全收集完成后挑战Boss战，检验Build强度 / After full collection, challenge Boss mode to test build strength
5. **NG+**：以上全部完成后进入NG+，体验新敌人配置和新增装备 / After all above, start NG+ for new enemy configs and exclusive gear
