## 技能系统概述 / Skill System Overview

**中文：** 剑星的技能系统分为四大类：贝塔技能（消耗贝塔能量）、爆发技能（消耗爆发能量）、无人机技能和被动技能。技能点通过升级和完成特定任务获得。每个技能树有多个分支，玩家可以根据自己的战斗风格自由加点。所有技能最终都可以点满，但前期需要优先选择核心技能。

**English:** Stellar Blade's skill system has four categories: Beta Skills (consume Beta energy), Burst Skills (consume Burst energy), Drone Skills, and Passive Skills. Skill points are earned through leveling and completing specific quests. Each skill tree has multiple branches for build customization. All skills can eventually be maxed, but early-game priority matters.

| 技能类型 / Type | 消耗资源 / Resource | 核心作用 / Core Role |
|---|---|---|
| 贝塔技能 / Beta Skills | 贝塔能量 / Beta Energy | 主动攻击技能 / Active attack abilities |
| 爆发技能 / Burst Skills | 爆发能量 / Burst Energy | 强力终结技能 / Powerful finishers |
| 无人机技能 / Drone Skills | 无（冷却）/ None (cooldown) | 远程支援攻击 / Ranged support attacks |
| 被动技能 / Passives | 无 / None | 属性强化 / Stat enhancements |

{/* ===== 贝塔技能树 / Beta Skill Tree ===== */}

## 贝塔技能树 / Beta Skill Tree

**中文：** 贝塔技能是剑星战斗的核心，消耗黄色贝塔能量条。每个技能都有独特的攻击动作和用途。

**English:** Beta skills are the core of combat, consuming the yellow Beta energy bar. Each skill has unique attack animations and uses.

| 技能 / Skill | 贝塔消耗 / Cost | 效果 / Effect |
|---|---|---|
| 贝塔突击 / Beta Strike | 1格 / 1 bar | 快速突进斩击，打断敌人动作 / Fast lunge slash, interrupts enemy actions |
| 猛击 / Slam | 1格 / 1 bar | 下砸攻击，击倒中小型敌人 / Ground slam, knocks down small-medium enemies |
| 旋风刃 / Blade Whirlwind | 2格 / 2 bars | 旋转攻击，范围伤害 / Spinning AoE attack |
| 穿刺 / Pierce | 2格 / 2 bars | 突刺攻击，破甲效果 / Thrust attack, armor piercing |
| 裂空斩 / Sky Splitter | 3格 / 3 bars | 跳起后下劈，高额单体伤害 / Jumping overhead slash, high single-target damage |

**贝塔技能使用技巧 / Beta Skill Tips**:
- 贝塔能量通过普通攻击和完美闪避回复 / Beta energy regenerates through normal attacks and perfect dodges
- 贝塔技能可以取消普通攻击的后摇 / Beta skills can cancel normal attack recovery frames
- 不同Boss对不同贝塔技能有抗性，需要灵活切换 / Different bosses have different resistances
- 旋风刃是清群怪的首选技能 / Blade Whirlwind is the best crowd clear skill

{/* ===== 爆发技能树 / Burst Skill Tree ===== */}

## 爆发技能树 / Burst Skill Tree

**中文：** 爆发技能是消耗蓝色爆发能量条的强力技能。爆发能量通过完美闪避、招架和处决积攒。爆发技能是面对精英怪和Boss时的核心输出手段。

**English:** Burst skills are powerful abilities consuming the blue Burst energy bar. Burst energy builds through perfect dodges, parries, and executions. Burst skills are the primary damage tool against elites and bosses.

| 技能 / Skill | 爆发消耗 / Cost | 效果 / Effect |
|---|---|---|
| 爆发连击 / Burst Combo | 1格 / 1 bar | 快速多段连击 / Fast multi-hit combo |
| 强力爆发 / Power Burst | 2格 / 2 bars | 蓄力后释放高伤害攻击 / Charged high-damage attack |
| 回旋爆裂 / Spiral Burst | 2格 / 2 bars | 旋转释放剑气，远程范围攻击 / Spinning blade beam, ranged AoE |
| 终结爆发 / Final Burst | 3格 / 3 bars | 全屏范围大伤害攻击 / Full-screen high-damage attack |
| 时间爆发 / Temporal Burst | 3格 / 3 bars | 短暂时间减慢后发动连击 / Brief time slow then unleash combo |

**爆发能量积攒技巧 / Burst Energy Tips**:
- 完美招架（Parry）积攒最多爆发能量 / Perfect parries generate the most burst energy
- 处决倒地敌人直接恢复2格爆发能量 / Executing downed enemies restores 2 full bars
- 爆发技能释放期间有无敌帧 / Burst skills grant invincibility frames during animation

{/* ===== 无人机技能树 / Drone Skill Tree ===== */}

## 无人机技能树 / Drone Skill Tree

**中文：** 无人机是剑星的远程支援单位，提供多种辅助攻击方式。无人机技能不消耗能量但有冷却时间。

**English:** The drone is Eve's ranged support unit, providing various auxiliary attack modes. Drone skills don't consume energy but have cooldowns.

| 技能 / Skill | 冷却 / Cooldown | 效果 / Effect |
|---|---|---|
| 无人机射击 / Drone Shot | 3秒 / 3s | 远程射击，造成物理伤害 / Ranged shot, physical damage |
| 导弹齐射 / Missile Salvo | 8秒 / 8s | 发射多枚追踪导弹 / Fire multiple homing missiles |
| 护盾生成 / Shield Generator | 15秒 / 15s | 生成临时护盾吸收伤害 / Generate temporary damage shield |
| 治疗脉冲 / Heal Pulse | 20秒 / 20s | 恢复生命值 / Restore HP |
| 电磁脉冲 / EMP Burst | 25秒 / 25s | 范围电击，对机械敌人特效 / AoE electric, effective vs machines |

**无人机升级 / Drone Upgrades**:
- 在营地使用材料升级无人机，提升伤害和效果 / Upgrade drone at camps using materials to boost damage and effects
- 高级无人机模块可以通过支线任务获得 / Advanced drone modules from side quests
- 不同无人机技能有专属升级路线 / Each drone skill has its own upgrade path

| 空中战斗 / Air Combat | 空中攻击伤害+25% / Air attack damage +25% |

{/* ===== 技能搭配推荐 / Skill Builds ===== */}

## 技能搭配推荐 / Recommended Builds

**中文：** 根据不同战斗风格推荐的技能搭配。

**English:** Recommended skill builds for different playstyles.

**招架反擊流 / Parry Counter Build**:
- 核心技能：招架强化、爆发充能、强力爆发、终结爆发 / Core: Parry Boost, Burst Charge, Power Burst, Final Burst
- 玩法：完美招架→积攒爆发能量→爆发技能输出 / Perfect parry → build burst energy → burst skill damage
- 推荐装备：增加招架判定的装备 / Gear with parry window extension

**闪避风筝流 / Dodge Kite Build**:
- 核心技能：闪避强化、贝塔充能、贝塔突击、裂空斩 / Core: Dodge Boost, Beta Charge, Beta Strike, Sky Splitter
- 玩法：完美闪避→贝塔技能输出→拉开距离 / Perfect dodge → beta skill damage → create distance
- 推荐装备：增加闪避距离的装备 / Gear with dodge distance increase

**无人机消耗流 / Drone Harass Build**:
- 核心技能：无人机射击强化、导弹齐射升级、电磁脉冲 / Core: Drone Shot boost, Missile Salvo upgrade, EMP Burst
- 玩法：远程无人机消耗 → 近战补刀 / Ranged drone harassment → melee cleanup
- 推荐装备：增加无人机伤害的装备 / Gear with drone damage boost

{/* ===== 技能升级优先级 / Skill Upgrade Priority ===== */}

## 技能升级优先级 / Upgrade Priority

**中文：** 前期技能点有限，建议按以下顺序加点。

**English:** Skill points are limited early on. Recommended upgrade order.

1. **招架强化 / Parry Boost** — 增加完美招架窗口，是最重要的生存技能 / Extends parry window, most important survival skill
2. **贝塔突击 / Beta Strike** — 核心起手技能，打断敌人 + 输出 / Core opener, interrupt + damage
3. **爆发充能 / Burst Charge** — 加快爆发能量积攒 / Faster burst energy buildup
4. **闪避强化 / Dodge Boost** — 增加完美闪避窗口 / Extends dodge window
5. **强力爆发 / Power Burst** — 主要爆发输出手段 / Primary burst damage tool
6. **治疗脉冲 / Heal Pulse** — 节省恢复道具 / Save healing items
7. **暴击强化 / Crit Boost** — 后期提升伤害的关键 / Key for late-game damage scaling

| 支线任务 / Side Quests | 20+ | 全支线奖励总计 / Total from all side quests |

| 钓鱼大赛 / Fishing Contest | 2 | 完成钓鱼收集后获得 / After completing fishing collection |
