# 新手入门 / Beginner Guide

剑星是一款以战斗为核心的动作游戏，但理解其系统和资源管理同样重要。本章带你从零开始 / Stellar Blade is a combat-focused action game, but understanding its systems and resource management is equally important. This guide takes you from the very beginning.

---

## 游戏目标 / Game Objective

你扮演伊芙（Eve），从轨道空降到被奈提巴占领的地球。目标是前往各个区域，击败奈提巴的首领，最终揭示地球沦陷的真相 / You play as Eve, orbital-dropping onto a Naytiba-infested Earth. Your goal is to travel across regions, defeat Naytiba leaders, and ultimately uncover the truth behind Earth's fall.

---

## 基础战斗 / Basic Combat

### 核心操作 / Core Controls

| 操作 / Action | PS5按键 / Button | 说明 / Description |
|------|:----:|------|
| **轻攻击 / Light Attack** | □ | 快速连击，积攒贝塔能量 / Fast combo, builds Beta energy |
| **重攻击 / Heavy Attack** | △ | 强力攻击，打断敌人动作 / Powerful strike, interrupts enemy actions |
| **闪避 / Dodge** | × | 翻滚闪避。完美闪避触发慢动作 / Roll dodge. Perfect dodge triggers slow motion |
| **格挡 / Guard** | L1 | 防御攻击。完美格挡弹反敌人 / Block attacks. Perfect parry deflects enemies |
| **贝塔技能 / Beta Skill** | R1 + □/△ | 消耗贝塔能量的强力技能 / Powerful skills consuming Beta energy |
| **爆发技能 / Burst Skill** | L1 + □/△/×/○ | 消耗爆发能量的终极技能 / Ultimate skills consuming Burst energy |
| **无人机射击 / Drone Shot** | R2 | 远程射击，牵制和削韧 / Ranged shots for harass and stagger |
| **锁定 / Lock-on** | R3 | 锁定当前敌人 / Lock onto current enemy |
| **跳跃 / Jump** | ✕(按住) | 跨越障碍/躲地面攻击 / Cross obstacles / dodge ground attacks |

### 战斗基本原则 / Core Combat Principles

| 原则 / Principle | 说明 / Description |
|------|------|
| **观察敌人动作 / Watch Enemy Patterns** | 每个敌人都有固定的攻击前摇——记住它们是你无伤的关键 / Every enemy has telegraphed attacks — learning them is key to no-damage fights |
| **格挡优于闪避 / Block > Dodge** | 大多数攻击可以格挡，完美格挡还能创造反击窗口 / Most attacks are blockable; perfect parries create counter windows |
| **不可格挡攻击 / Unblockable Attacks** | 红光攻击不可格挡，必须闪避 / Red-glowing attacks are unblockable — must dodge |
| **保持贝塔能量 / Maintain Beta Energy** | 贝塔技能是主要爆发手段，普攻积攒能量 / Beta skills are your main burst — build energy with normal attacks |
| **利用无人机 / Use Your Drone** | 无人机可以远程削韧、打断敌人施法，是安全输出手段 / Drone chips away at stagger, interrupts casts — safe damage option |

### 资源系统 / Resource System

| 资源 / Resource | 上限 / Cap | 获得方式 / Gain | 用途 / Use |
|------|:----:|------|------|
| **HP / HP** | 固定 + 核心提升 / Fixed + Core upgrades | 药水、休息 / Potions, rest | 生存 / Survival |
| **贝塔能量 / Beta Energy** | 默认3格 / 3 bars default | 轻攻击命中 / Light attacks hit | 施放贝塔技能 / Cast Beta skills |
| **爆发能量 / Burst Energy** | 默认2格 / 2 bars default | 完美闪避/格挡 / Perfect dodge/parry | 施放爆发技能 / Cast Burst skills |
| **SP（技能点）/ Skill Points** | 无上限 / No cap | 升级获得 / Gain on level up | 学习技能 / Unlock skills |

## 初期升级路线 / Early Progression Path

| 阶段 / Phase | 目标 / Goal | 建议 / Tips |
|------|------|------|
| **埃多斯7号 / Eidos 7** | 完成序章，熟悉基础操作 / Complete prologue, learn basics | 多练习完美格挡和完美闪避——这是后续战斗的核心 / Practice perfect parry and dodge — they're essential later |
| **废土 / Wasteland** | 第一个开放区域，探索和主线并行 / First open zone, explore + main quest | 拾取所有可见的收集品，开传送点。优先升级**贝塔能量上限** / Grab all visible collectibles, activate waypoints. Prioritize **Beta Energy cap** upgrades |
| **大沙漠 / Great Desert** | 第二个开放区域 / Second open zone | 难度开始上升。确保学习了至少2个贝塔技能 / Difficulty ramps up. Make sure you have at least 2 Beta skills |
| **脊柱 / The Spire** | 最终区域 / Final area | 准备最终决战——确保所有药水升级完毕，装备强化到位 / Prepare for the final battles — max potion upgrades and gear |

## 新手常见误区 / Common Newbie Mistakes

| 误区 / Mistake | 正确做法 / Correct |
|------|------|
| **只进攻不防御 / Only Attack** | 剑星的战斗节奏是"攻-守-攻"循环。盲目进攻=暴毙 / Combat is an "attack-defend-attack" loop. Mindless aggression = death |
| **忽略无人机 / Ignore Drone** | 无人机在远程牵制、破盾和AOE清理小怪时极其有用 / Drone is invaluable for ranged harass, shield breaking, and AoE clear |
| **不升级药水 / Don't Upgrade Potions** | 在营地升级药水容量和效果至关重要——这是最划算的投资 / Upgrade potion capacity and effect at camps — best value investment |
| **跳过支线 / Skip Side Quests** | 支线提供SP、核心和独特服装——值得做 / Side quests reward SP, cores, and unique costumes — worth doing |
| **忽略钓鱼 / Ignore Fishing** | 钓鱼奖励包括SP和核心，还能解锁奖杯 / Fishing rewards SP and cores, plus trophy unlocks |

## 难度选择建议 / Difficulty Recommendations

| 难度 / Difficulty | 适合人群 / Best For |
|------|------|
| **剧情模式 / Story Mode** | 想体验剧情、不擅长动作游戏的玩家 / Players who want the story without combat frustration |
| **普通模式 / Normal Mode** | **推荐大多数玩家**。有挑战但不至于卡关 / **Recommended for most**. Challenging but fair |
| **困难模式 / Hard Mode** | 动作游戏高手、二周目玩家 / Action game veterans, NG+ players |

> **建议**：一周目选普通，专心探索和收集。二周目新游戏+开启困难挑战 / **Tip**: Play Normal on first run for exploration and collection. Save Hard for NG+.
