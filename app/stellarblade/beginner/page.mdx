# 新手入门 / Beginner Guide

剑星是一款以战斗为核心的动作游戏，但理解其系统和资源管理同样重要。本文将从零开始教你如何高效推图、合理分配资源，为高难Boss战做好准备 / Stellar Blade is a combat-focused action game, but understanding its systems and resource management is equally important. This guide takes you from the very beginning, covering efficient progression, resource allocation, and boss prep.

---

## 游戏目标 / Game Objective

你扮演伊芙（Eve），从轨道空降到被奈提巴占领的地球。目标是穿越四大区域——埃多斯7号、废土、大沙漠、脊柱——击败奈提巴的首领，揭示地球沦陷的真相。每个区域都有主线Boss、支线任务和大量收集品 / You play as Eve, orbital-dropping onto a Naytiba-infested Earth. Your goal is to traverse four regions — Eidos 7, Wasteland, Great Desert, and The Spire — defeating Naytiba leaders and uncovering Earth's fall. Each region has main bosses, side quests, and many collectibles.

### 战斗基本原则 / Core Combat Principles

剑星的战斗系统围绕一个循环：敌人攻击→你防御(格挡或闪避)→反击→敌人在硬直→输出→等下一次敌人攻击 / Stellar Blade's combat revolves around a cycle: enemy attacks → you defend (block or dodge) → counter → enemy is stunned → damage → wait for next enemy attack.

| 原则 / Principle | 说明 / Description |
|------|------|
| **蓝光可格挡，红光必须闪 / Blue = Block, Red = Dodge** | 敌人攻击时身上的闪光颜色决定应对方式。蓝色=可以格挡/完美格挡，红色=必须闪避 / Enemy glow color tells you: blue = blockable/parryable, red = must dodge |
| **完美格挡 > 普通格挡 / Perfect Parry > Normal Block** | 完美格挡弹反敌人创造反击窗口、攒爆发能量。普通格挡只免伤 / Perfect parry deflects enemies, creates counter windows, builds Burst energy. Normal block only prevents damage |
| **普攻攒贝塔，防反攒爆发 / Light attack = Beta, Parry = Burst** | 轻攻击积攒贝塔能量(用于贝塔技能)，完美防御/闪避积攒爆发能量(用于爆发技能) / Light attacks build Beta (for Beta skills), perfect parries/dodges build Burst (for Burst skills) |
| **贝塔技能有霸体 / Beta Skills = Hyper Armor** | 释放贝塔技能时不会被敌人打断——可以用来强行反打 / Beta skills can't be interrupted during cast — use to force a counter |

## 区域攻略 / Region Strategy

### 埃多斯7号策略 / Eidos 7 Strategy

埃多斯7号是游戏的序章区域——线性关卡，不可返回。这里是建立战斗习惯的关键时期 / Eidos 7 is the prologue — linear, unreturnable. This is the critical period for building combat habits.

| 建议 / Tip | 说明 / Description |
|------|------|
| **每个房间搜干净 / Search every room** | 门后和角落藏有核心和药水——离开后无法再回来 / Cores and potions hide behind doors and corners — no return after leaving |
| **练习完美格挡 / Practice perfect parry** | 用这里的低级敌人练习格挡时机——后续区域没有这样的练习机会 / Use weak enemies here to practice parry timing — later zones offer no such practice |
| **破坏所有易碎物品 / Break everything** | 罐子和收集品藏在木箱和玻璃后面——见到就破坏 / Cans and collectibles hide behind crates and glass — break everything |
| **不急推进 / Take your time** | 没有时间限制，熟悉操作最重要 / No time limit — building muscle memory is the priority |

### 废土策略 / Wasteland Strategy

废土是第一个开放区域——从线性关卡到开放世界的转变需要调整玩法 / Wasteland is the first open zone — the shift from linear to open world requires adjustment.

| 建议 / Tip | 说明 / Description |
|------|------|
| **优先开传送点 / Unlock waypoints first** | 废土地图很大，传送点减少跑路时间——见到信号塔就去激活 / Wasteland is huge — waypoints minimize travel. Activate every tower you see |
| **白天/夜晚规划 / Plan day/night** | 部分收集品和NPC仅在特定时间出现——到达新区域后先确认时间 / Some collectibles and NPCs only appear at specific times — check time on arrival |
| **绿洲早点解锁 / Unlock Oasis early** | 绿洲提供商人、钓鱼和休息——是废土的核心枢纽，主线推到能去就去 / Oasis offers merchant, fishing, and rest — the Wasteland hub. Go as soon as story allows |
| **打不过先探索 / Stuck? Explore** | 如果某场战斗太难，有可能走错了方向——先探索周边区域拿核心和SP / If a fight is too hard, you may be underleveled — explore for cores and SP first |

### 大沙漠策略 / Great Desert Strategy

大沙漠比废土更大、更强、更复杂——进入前需要充分准备 / Great Desert is bigger, harder, and more complex than Wasteland — prepare thoroughly before entering.

| 建议 / Tip | 说明 / Description |
|------|------|
| **准备至少2个贝塔技能 / Have 2+ Beta skills** | 大沙漠敌人血量大幅提升——单贝塔技能不够用 / Enemy HP spikes in Great Desert — one Beta skill won't cut it |
| **备足药水和金币 / Stock potions and gold** | 大沙漠商人稀少，药水购买机会有限。离开废土前买够10瓶以上 / Merchants are scarce in Great Desert. Stock 10+ potions before leaving Wasteland |
| **热浪天气对策 / Heat wave counter** | 热浪时室外持续掉血——看到天气变化立即找室内或阴影区域 / Heat wave causes continuous HP drain — immediately seek indoors or shade |
| **地下遗迹不迷路 / Navigate underground ruins** | 遗迹是多层迷宫——从入口向右探索，每层记住返回路径 / The ruins are a multi-floor maze — explore right from entrance, memorize return paths on each floor |

### 脊柱策略 / The Spire Strategy

脊柱是最終区域——这里没有回头路，所有准备工作在此完成 / The Spire is the final area — no return, all prep must be done before entering.

| 建议 / Tip | 说明 / Description |
|------|------|
| **进入前100%准备 / 100% prep before entry** | 确保药水满级、核心全镶嵌、技能学完——进入后无法返回营地 / Max potions, socketed cores, learned skills — no camp return after entry |
| **注意分叉路 / Watch for branches** | 脊柱有多条岔路——一条通向主线，其他通向收集品和隐藏奖励 / The Spire has branching paths — one leads to main story, others to collectibles |
| **最终商人 / Final merchant** | 最终Boss门前有一个商人——最后一次补充机会 / There's a merchant before the final boss — last chance to restock |
| **心理准备 / Mental prep** | 最终Boss连续极端难度三阶段——打完一阶段不是结束而是刚刚开始 / Final boss is 3 phases of extreme difficulty — beating phase 1 is just the beginning |

### 钓鱼基础 / Fishing Basics

| 步骤 / Step | 操作 / Action |
|------|------|
| **1. 获得鱼饵 / Get bait** | 从商人处购买——不同鱼饵吸引不同鱼类 / Buy from merchants — different bait attracts different fish |
| **2. 抛竿 / Cast** | 站在码头按□抛竿，按住□控制力度 / Stand on the dock, press □ to cast, hold □ for distance |
| **3. 等待咬钩 / Wait for bite** | 观察浮标下沉——当浮标完全入水时按○收线 / Watch the bobber — when fully submerged, press ○ to hook |

| **5. 重复 / Repeat** | 继续抛竿直到钓到目标鱼类 / Keep casting until you catch your target fish |

## 新手常见误区 / Common Newbie Mistakes

| 误区 / Mistake | 正确做法 / Correct |
|------|------|
| **只进攻不防御 / Only Attack** | 剑星的战斗节奏是"攻-守-攻"循环。盲目进攻=暴毙。每个攻击窗口只打3-4下就收手 / Combat is "attack-defend-attack" loop. Trade 3-4 hits per window, then reset |
| **忽略无人机 / Ignore Drone** | 无人机远程牵制、破盾、AOE清小怪——打Boss时保持无人机持续输出，DPS提升显著 / Drone provides ranged harass, shield break, AoE clear. Keep firing during boss fights for significant DPS |
| **不升级药水 / Skip Potion Upgrades** | 营地升级药水容量和效果——这是最划算的投资。满级药水=一瓶回满60%+HP / Upgrade potion capacity and effect at camps — best value investment. Max potions = 60%+ heal per use |
| **跳过支线 / Skip Side Quests** | 支线提供大量SP、核心和独特服装——主线卡关时先做支线 / Side quests reward SP, cores, and unique costumes. If stuck on main, do side quests first |
| **忽略钓鱼 / Ignore Fishing** | 钓鱼奖励SP和核心——还有奖杯。每次路过钓鱼点花5分钟 / Fishing rewards SP and cores plus trophies. Spend 5 minutes at each fishing spot |
| **不锁定敌人 / Not Locking On** | 战斗中锁定敌人(R3)可以保持视角锁定——尤其对敏捷型敌人和Boss / Lock on (R3) keeps camera focused — essential for agile enemies and bosses |
| **不懂背刺 / No Back Attacks** | 对小怪绕到背后攻击=必定暴击。巡逻敌人可以潜行到背后→一击必杀 / Flanking = guaranteed crit on normal enemies. Stealth to patrol backs for instant kills |
| **不升级装备 / Skip Gear Upgrades** | 在营地及时升级无人机和外脊——战斗压力会显著降低。每到一个新营地去看看升级选项 / Upgrade drone and exospine at camps regularly — it significantly lowers combat pressure. Check upgrades at every new camp |
| **忽视简单营地 / Skip Easy Camps** | 每个营地都提供回复和传送——激活所有路过的营地，方便后续快速移动 / Every camp provides heal and fast travel — activate all camps you pass for convenient later travel |
| **乱花金币 / Waste Gold** | 前期金币有限——优先升级药水容量和购买鱼饵。不要买不需要的消耗品 / Gold is limited early — prioritize potion upgrades and fishing bait. Don't buy unnecessary consumables |
| **不检查地图 / Never Check Map** | 按触摸板查看地图——标记未探索区域和收集品。迷路或找不到目标时先看地图 / Press touchpad to check the map — mark unexplored areas and collectibles. Check map when lost or stuck |

## 难度选择建议 / Difficulty Recommendations

| 难度 / Difficulty | 适合人群 / Best For |
|------|------|
| **剧情模式 / Story Mode** | 剧情优先的玩家——战斗压力小 / Players prioritizing story — minimal combat pressure |
