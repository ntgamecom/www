# BOSS攻略 / Boss Strategies

剑星的BOSS战是游戏的核心亮点，每个BOSS都有独特的攻击模式和阶段转换。本章详解每个BOSS的机制和打法 / Boss fights are the highlight of Stellar Blade. Each boss has unique attack patterns and phase transitions. This chapter covers every boss's mechanics and strategies.

---

## 通用BOSS技巧 / General Boss Tips

| 技巧 / Tip | 说明 / Description |
|------|------|
| **熟悉前摇 / Learn the Tells** | 每个BOSS的攻击前明确——记住哪些可以格挡、哪些必须闪避 / Every boss has clear tells — learn which attacks are blockable and which must be dodged |
| **完美格挡优先 / Prioritize Perfect Parry** | 完美格挡不仅化解攻击，还创造反击窗口和爆发能量 / Perfect parry not only nullifies attacks but creates counter windows and Burst energy |
| **善用时空减速 / Use Time Slow** | **时空减速**爆发技能对几乎所有BOSS都极为有效——创造大量输出时间 / **Time Slow** Burst skill is effective against nearly every boss — creates massive damage windows |
| **保持贝塔能量 / Keep Beta Reserve** | 至少保留1格贝塔能量用于紧急情况——某些BOSS有快速连招需要打断 / Keep at least 1 Beta bar for emergencies — some bosses have rapid combos needing interruption |
| **用地形规避 / Use Terrain** | 部分BOSS的大范围攻击可以用柱子或地形遮挡 / Some bosses' wide attacks can be blocked by pillars or terrain |
| **无人机压制 / Drone Pressure** | BOSS战中用无人机持续攻击可以维持削韧，加速失衡 / Sustained drone fire maintains stagger pressure, accelerating stagger |
| **不要贪刀 / Don't Get Greedy** | 剑星BOSS战的节奏是"防→反→防"——贪刀是暴毙的第一原因 / Combat rhythm is "defend → counter → defend" — greed is the #1 cause of death |

## 第一章BOSS：布鲁斯 / Brutus

**位置 / Location**: 埃多斯7号-竞技场 / Eidos 7 - Arena

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★☆☆☆（入门 / Introductory） |
| **类型 / Type** | 重型人形 / Heavy Humanoid |
| **弱点 / Weakness** | 攻击后摇长，闪避后有大输出窗口 / Long attack recovery, large window after dodge |

### 阶段1（100%-50%）/ Phase 1 (100%-50%)

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|
| 巨拳横扫 / Giant fist sweep | 格挡可 / Blockable | 完美格挡后追击 / Perfect parry → counter |
| 践踏 / Stomp | 格挡可 / Blockable | 闪避到侧方 / Dodge sideways |
| 冲刺冲撞 / Charge tackle | **不可格挡（红光）/ Unblockable (red)** | 侧向闪避 / Side dodge |
| 双拳砸地 / Double fist slam | 格挡可 / Blockable | 格挡第一下→闪避范围AOE / Block first → dodge AoE |

### 阶段2（50%-0%）/ Phase 2 (50%-0%)

新增攻击 / New attacks:
| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|
| 三连砸地 / Triple slam | 格挡可 / Blockable | 格挡×3 → 反击 / Block ×3 → counter |
| 全屏震波 / Full-screen shockwave | **不可格挡（红光）/ Unblockable (red)** | 跳跃躲避 / Jump over |

**打法要点 / Tips**:
- 这是教学BOSS——重点练习完美格挡的时机 / This is a tutorial boss — focus on perfect parry timing
- 阶段2的三连砸地是非常好的反击窗口 / Phase 2's triple slam is a great counter window
- 打败布鲁斯后获得**力量外脊**的早期版本 / Defeating Brutus grants an early version of Strength Exospine

## 第二章BOSS：潜行者 / Stalker

**位置 / Location**: 废土-地下设施 / Wasteland - Underground Facility

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★★☆☆（中等 / Medium） |
| **类型 / Type** | 敏捷人形 / Agile Humanoid |
| **弱点 / Weakness** | 攻击后有短暂硬直，但窗口极短 / Brief post-attack stun, but very short window |

### 阶段1（100%-50%）/ Phase 1 (100%-50%)

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|
| 快速三连斩 / Fast triple slash | 格挡可 / Blockable | 连续完美格挡→反击 / Chain perfect parry → counter |
| 瞬移突刺 / Teleport thrust | **不可格挡（红光）/ Unblockable (red)** | 听到音效立刻闪避 / Dodge immediately on sound cue |
| 投掷飞刃 / Thrown blade | 格挡可 / Blockable | 可格挡反弹 / Can be parried back |
| 后跳射击 / Backstep shot | 格挡可 / Blockable | 格挡或闪避接近 / Block or dodge close |

### 阶段2（50%-0%）/ Phase 2 (50%-0%)

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|
| 分身攻击 / Clone attack | 特殊 / Special | 先找出真身（观察动作差异）→ 攻击真身 / Find the real one (spot movement differences) → attack it |
| 暗影弹幕 / Shadow barrage | **不可格挡（红光）/ Unblockable (red)** | 连续闪避 / Chain dodge |
| 终结连击 / Final combo | 格挡可 / Blockable | 全部格挡后有大硬直→满输出 / Block all → big stun → full damage |

**打法要点 / Tips**:
- 这是第一个真正的挑战。潜行者速度快、攻击密集 / First real challenge. Fast with dense attack patterns
- 分身阶段不要慌张——真身的攻击比分身更流畅 / Don't panic during clones — the real one's attacks are smoother
- **强烈建议**在打这个BOSS前学习了**反击追击**技能 / **Strongly recommended** to have **Counter Pursuit** before this fight
- 奖励：**速度外脊** / Reward: **Speed Exospine**

## 第三章BOSS：阿比斯 / Abyss

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★★☆☆（中等 / Medium） |
| **类型 / Type** | 大型奈提巴 / Large Naytiba |
| **弱点 / Weakness** | 行动缓慢，但伤害极高 / Slow but deals massive damage |

### 阶段1（100%-60%）/ Phase 1 (100%-60%)

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|
| 巨爪横扫 / Claw sweep | 格挡可 / Blockable | 格挡→输出 / Block → attack |
| 触须穿刺 / Tentacle pierce | **不可格挡（红光）/ Unblockable (red)** | 地面出现标记时闪避 / Dodge when ground marker appears |
| 酸液喷射 / Acid spit | 格挡可 / Blockable | 侧移或格挡 / Sidestep or block |
| 尾部扫击 / Tail sweep | 格挡可 / Blockable | 跳跃或后闪 / Jump or back dodge |

### 阶段2（60%-0%）/ Phase 2 (60%-0%)

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|

- 推荐外脊：**防御外脊**（存活率大幅提升）/ Recommended Exospine: **Defense Exospine** (greatly improves survival)

**位置 / Location**: 大沙漠-地下遗迹 / Great Desert - Underground Ruins

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★★★☆（困难 / Hard） |
| **类型 / Type** | 人形BOSS / Humanoid Boss |
| **弱点 / Weakness** | 攻击节奏固定，可预测 / Fixed attack rhythm, predictable |

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|
| 剑气斩 / Blade wave | 格挡可 / Blockable | 完美格挡→反击 / Perfect parry → counter |
| 瞬移连击 / Teleport combo | **部分不可格挡 / Partially unblockable** | 前2下格挡，红光部分闪避 / Block first 2, dodge red parts |
| 暗影突袭 / Shadow assault | **不可格挡（红光）/ Unblockable (red)** | 侧闪 / Side dodge |
| 飞刀投掷 / Knife throw | 格挡可 / Blockable | 格挡反弹 / Parry back |

### 阶段2（60%-30%）/ Phase 2 (60%-30%)

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|
| 分身剑阵 / Clone sword array | 特殊 / Special | 观察影子轨迹→闪避最后一击 / Watch shadow patterns → dodge final strike |
| 快速连斩+终结 / Fast combo + finisher | 混合 / Mixed | 格挡前3下→闪避红光终结 / Block first 3 → dodge red finisher |
| 暗黑领域 / Dark zone | **不可格挡（红光）/ Unblockable (red)** | 离开黑色区域 / Leave dark zone |

### 阶段3（30%-0%）/ Phase 3 (30%-0%)

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|
| 全力猛攻 / All-out assault | 混合 / Mixed | 连续格挡+闪避，等待反击时机 / Chain block + dodge, wait for counter window |
| 终极斩击 / Ultimate slash | **不可格挡（红光）/ Unblockable (red)** | 完美闪避→大量输出时间 / Perfect dodge → large damage window |
| 残影连击 / Afterimage combo | 格挡可 / Blockable | 全部完美格挡→处决级反击窗口 / All perfect parries → execution-level window |

**打法要点 / Tips**:
- **蕾雯是游戏中最难的BOSS之一**——做好反复尝试的准备 / **Raven is one of the hardest bosses in the game** — be prepared for multiple attempts
- 阶段3是纯粹的操作检测——每个失误都可能致命 / Phase 3 is pure skill check — every mistake is potentially fatal
- **时空减速**在这里极其有效——在阶段3开始时使用 / **Time Slow** is extremely effective here — use it at Phase 3 start
- 推荐外脊：**速度外脊**（更快攻速=更快积攒贝塔）/ Recommended: **Speed Exospine** (faster attacks = faster Beta generation)
- 奖励：**独特服装** / Reward: **Unique Costume**

| **类型 / Type** | 终极奈提巴 / Ultimate Naytiba |
| **弱点 / Weakness** | 部分攻击有固定节奏，可以利用反击 / Certain attacks have fixed rhythm for counters |

### 阶段1（100%-70%）/ Phase 1 (100%-70%)

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|

- 阶段3的连续光剑风暴是纯耐力考验——保持冷静，看准每个攻击的时机 / Phase 3's blade storm is an endurance test — stay calm and time each parry
- 推荐装备：**贝塔外脊** + 满级HP核心 + 满级药水 / Recommended: **Beta Exospine** + max HP Cores + max potions
- **最后一击**必须通过完美格挡触发——不能普通攻击杀死 / **Final blow** must be triggered by perfect parry — cannot be killed by normal attacks

## 隐藏BOSS：远古守卫 / Ancient Guardian

**位置 / Location**: 大沙漠-古代神殿（隐藏区域）/ Great Desert - Ancient Temple (hidden area)

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★★★☆（困难 / Hard） |
| **解锁条件 / Unlock** | 在大沙漠找到并激活3个古代石碑 / Find and activate 3 ancient tablets in Great Desert |

**打法要点 / Tips**:
- 远古守卫的攻击节奏与主线BOSS不同——更慢但单发伤害更高 / Slower attack rhythm than story bosses, but higher single-hit damage
- 它的红光攻击前摇很短——需要快速反应 / Red attack tells are very short — quick reflexes needed
- 击败后获得**最强外脊**之一 / Defeating it grants one of the **best Exospines** in the game
- 建议等级：35+ / Recommended level: 35+

## BOSS挑战模式 / Boss Challenge Mode

通关后解锁BOSS挑战模式，可重新挑战所有BOSS / Unlocked after completing the game — replay all bosses.

| 模式 / Mode | 说明 / Description |
|------|------|
| **普通挑战 / Normal Challenge** | 标准难度，练习不限次数 / Standard difficulty, unlimited practice |
| **困难挑战 / Hard Challenge** | BOSS大幅强化，新增攻击模式 / Significantly buffed bosses with new attack patterns |
| **无伤挑战 / No-Damage Challenge** | 被击中一次即失败 / Failed on first hit |

**挑战模式奖励 / Challenge Rewards**: 独特外观、奖杯、二周目继承道具 / Unique cosmetics, trophies, NG+ carryover items
