# BOSS攻略 / Boss Strategies

剑星的BOSS战是游戏的核心亮点，每个BOSS都有独特的攻击模式和阶段转换。本文详解每个BOSS的机制和应对策略，从教学Boss到最终Boss全覆盖 / Boss fights are the highlight of Stellar Blade. Each boss has unique attack patterns and phase transitions. This guide covers every boss from tutorial to final encounter.

---

## 通用BOSS技巧 / General Boss Tips

剑星的战斗系统围绕格挡(完美格挡)和闪避(完美闪避)展开——掌握这两项是所有Boss战的基础 / Stellar Blade's combat revolves around Perfect Parry and Perfect Dodge — mastering both is the foundation of all boss fights.

| 技巧 / Tip | 说明 / Description |
|------|------|
| **熟悉前摇 / Learn the Tells** | 每个BOSS的攻击有明显的起手动画——记住哪些可以格挡、哪些必须闪避。蓝光攻击可格挡，红光攻击必须闪避 / Every boss has clear tells — learn which are blockable (blue glow) and which must be dodged (red glow) |
| **完美格挡优先 / Prioritize Perfect Parry** | 完美格挡不仅化解攻击，还创造反击窗口和爆发能量。完美格挡后接轻攻击可以触发追击技——伤害+削韧 / Perfect parry creates counter windows and Burst energy. Follow with light attack for pursuit skill — damage + stagger |
| **善用时空减速 / Use Time Slow** | **时空减速**爆发技能对几乎所有BOSS都极为有效——创造大量输出时间。建议在进入战斗后第一个爆发技能就用时空减速——开局多打30%血 / Time Slow (Burst skill) creates massive damage windows. Use it as your first Burst skill after engaging for a 30% HP advantage |
| **保持贝塔能量 / Keep Beta Reserve** | 至少保留1格贝塔能量用于紧急情况——某些BOSS有快速连招需要用贝塔技能打断。贝塔技带霸体，被抓到连招时可以用来反打 / Keep 1 Beta bar for emergencies — Beta skills have hyper armor and can counter boss combos |
| **用地形规避 / Use Terrain** | 部分BOSS的大范围攻击可以用柱子或地形遮挡——例如阿比斯的全屏酸雨可以躲在柱子后面 / Some bosses' wide attacks can be blocked by pillars — e.g., Abyss's acid rain can be negated by pillars |
| **无人机压制 / Drone Pressure** | BOSS战中用无人机持续攻击可以维持削韧、加速失衡。无人机攻击攒贝塔能量的效率也很高 / Sustained drone fire maintains stagger pressure and builds Beta energy efficiently |
| **不要贪刀 / Don't Get Greedy** | 剑星BOSS战的节奏是"防→反→防"——贪刀是暴毙的第一原因。打完一套安全连招后立刻退回防守姿态 / Combat rhythm is "defend → counter → defend" — greed is the #1 cause of death. After your combo, immediately return to defensive stance |

## 第一章BOSS：布鲁斯 / Brutus (教学Boss)

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★☆☆☆（入门 / Introductory） |
| **类型 / Type** | 重型人形 / Heavy Humanoid |
| **弱点 / Weakness** | 攻击后摇长，闪避后有大输出窗口 / Long attack recovery |

布鲁斯是教学Boss——专门用来教你完美格挡和闪避。他的攻击速度偏慢，前摇明显，非常适合练习 / Brutus is the tutorial boss — designed to teach perfect parry and dodge. Slow attacks with clear tells, perfect for practice.

**阶段1（100%-50%）**: 

| 攻击 / Attack | 类型 / Type | 应对 / Counter |
|------|------|------|
| 巨拳横扫 / Fist sweep | 格挡可 / Blockable | 完美格挡→追击 / Perfect parry → pursuit |
| 践踏 / Stomp | 格挡可 / Blockable | 闪避到侧方 / Dodge sideways |
| 冲刺冲撞 / Charge tackle | **不可格挡（红光）/ Unblockable (red)** | 侧向闪避 / Side dodge |
| 双拳砸地 / Double fist slam | 格挡可 / Blockable | 格挡第一下→闪避范围AOE / Block first → dodge AoE |

**阶段2（50%-0%）**: 新增三连砸地(格挡×3→反击)和全屏震波(红光——跳跃躲避) / Adds triple slam (block ×3 → counter) and full-screen shockwave (red — jump over).

**详细攻击阶段分析 / Detailed Pattern Breakdown**:

| 攻击模式 / Pattern | 触发条件 / Trigger | 伤害 / Damage | 反击时机 / Counter Window |
|------|------|:----:|------|
| **巨拳横扫 / Fist Sweep** | 玩家在中近距离 / Player mid-close range | 中等 / Medium | 完美格挡后——约2秒输出窗口 / After perfect parry — ~2s window |
| **双拳砸地 / Double Slam** | 玩家在正面短距离 / Player directly in front | 高 / High | 格挡第一拳后闪避AOE→1.5秒窗口 / Block first → dodge AoE → 1.5s window |
| **冲刺冲撞 / Charge Tackle** | 玩家在远距离 / Player at range | 高（红光）/ High (red) | 侧向闪避后——约1秒窗口 / After side dodge — ~1s window |
| **全屏震波(P2) / Shockwave (P2)** | 阶段转换时必定释放 / Always on phase transition | 中 / Medium | 跳跃时机：看到地面波纹扩散的瞬间 / Jump timing: the moment ground ripple appears |

**NG+变化 / NG+ Changes**:
- P1新增2连冲刺冲撞(红光×2→连续闪避) / P1 adds double charge tackle (red ×2 → chain dodge)
- P2震波速度加快——跳跃时机要求更精确 / P2 shockwave is faster — tighter jump timing
- 全屏震波后追加一次快速砸地 / Shockwave now followed by a quick slam

## 第二章BOSS：潜行者 / Stalker

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★★☆☆（中等 / Medium） |
| **类型 / Type** | 敏捷人形 / Agile Humanoid |
| **弱点 / Weakness** | 攻击后短暂硬直——窗口极短 / Brief post-attack stun, very short |

潜行者是第一个真正的挑战——速度快、攻击密集，考验你的反应能力 / Stalker is the first real challenge — fast with dense attack patterns, testing reactions.

**阶段1（100%-50%）**: 快速三连斩(连续完美格挡→反击)、瞬移突刺(红光——听声音闪避)、投掷飞刃(可格挡反弹)、后跳射击(格挡或闪避接近) / Fast triple slash (chain parry → counter), Teleport thrust (red — dodge on sound cue), Thrown blade (parry back), Backstep shot (block or dodge close).

**阶段2（50%-0%）**: 分身攻击(找出真身——观察动作差异)、暗影弹幕(红光——连续闪避)、终结连击(全部格挡后大硬直) / Clone attack (find real one — spot movement differences), Shadow barrage (red — chain dodge), Final combo (block all → big stun → full damage).

**详细攻击阶段分析 / Detailed Pattern Breakdown**:

| 攻击模式 / Pattern | 类型 / Type | 阶段 / Phase | 反击窗口 / Counter Window |
|------|:----:|:----:|------|
| **快速三连斩 / Fast Triple Slash** | 全部格挡 / All blockable | P1 | 完美格挡×3后——约3秒输出窗口 / After 3 perfect parries — ~3s window |
| **瞬移突刺 / Teleport Thrust** | 红光——必须闪避 / Red — must dodge | P1 | 闪避后约1秒——追刺很短 / ~1s after dodge — very short |
| **投掷飞刃 / Thrown Blade** | 格挡可反弹 / Blockable, parryable | P1 | 反弹后硬直——中距离约2秒窗口 / After parry bounce — ~2s at mid-range |
| **分身攻击 / Clone Attack** | 特殊机制 / Special mechanic | P2 | 真身攻击后——约1.5秒窗口。看动作流畅度判断真身 / ~1.5s after real one attacks. Identify via smoother movement |
| **暗影弹幕 / Shadow Barrage** | 红光×5连续 / Red ×5 chain | P2 | 连续完美闪避5次后——约3秒大窗口 / After 5 perfect dodges — ~3s big window |
| **终结连击 / Final Combo** | 格挡×4+红光终结 / Block ×4 + red finisher | P2 | 前4下格挡→闪避红光→约4秒大窗口 / Block 4 → dodge red → ~4s huge window |

**NG+变化 / NG+ Changes**:
- 瞬移突刺变为高速连刺——红光×3连续闪避 / Teleport becomes rapid thrusts — red ×3 chain dodge
- 分身攻击出现3个分身(原来是2个) / Clone attack spawns 3 clones (was 2)
- P2新增暗影领域——地面出现黑色区域，站在其中持续掉血 / P2 adds Shadow Zone — dark areas on ground, DoT while standing in them

## 第三章BOSS：阿比斯 / Abyss

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★★☆☆（中等 / Medium） |
| **类型 / Type** | 大型奈提巴 / Large Naytiba |
| **弱点 / Weakness** | 行动缓慢，但伤害极高 / Slow but deals massive damage |

大型Boss——行动缓慢但单次攻击伤害极高。关键是不要被任何一下打中 / Large boss — slow but each hit is devastating. Don't get hit even once.

**阶段1（100%-60%）**: 巨爪横扫(格挡→输出)、触须穿刺(红光——地面标记闪现避)、酸液喷射(侧移或格挡)、尾部扫击(跳跃或后闪) / Claw sweep (block → attack), Tentacle pierce (red — dodge ground marker), Acid spit (block or sidestep), Tail sweep (jump or back dodge).

**阶段2（60%-0%）**: 全屏酸雨(红光——跑到无酸区域)、狂暴连击(格挡全部4下→大型反击窗口)、吞噬(被抓时快速按键挣脱) / Full-screen acid rain (red — run to safe zone), Berserk combo (block all 4 → big counter window), Devour (mash button to escape).

**详细攻击阶段分析 / Detailed Pattern Breakdown**:

| 攻击模式 / Pattern | 类型 / Type | 阶段 / Phase | 特征 / Tell | 应对 / Counter |
|------|:----:|:----:|------|------|
| **巨爪横扫 / Claw Sweep** | 格挡可 / Blockable | P1 | 大范围横挥前摇 / Wide wind-up | 完美格挡→2秒输出 / Perfect parry → 2s damage |
| **触须穿刺 / Tentacle Pierce** | 红光 / Red | P1 | 地面标记——红色圆圈 / Ground marker — red circle | 闪避出圈 / Dodge out of circle |
| **酸液喷射 / Acid Spit** | 格挡可 / Blockable | P1 | 低头蓄力——向前方扇形喷射 / Lowers head, charges — frontal cone | 侧移或格挡 / Sidestep or block |
| **尾部扫击 / Tail Sweep** | 红光——不可格挡 / Red — unblockable | P1 | 转身尾部发光 / Turns, tail glows | 跳跃或后闪 / Jump or back dodge |
| **全屏酸雨 / Acid Rain** | 特殊机制 / Special | P2 | 仰天嘶吼→全屏酸雨 / Roars skyward → full rain | 跑向无酸安全区——3个固定位置 / Run to dry safe zone — 3 fixed spots |
| **狂暴连击 / Berserk Combo** | 格挡×4 / Block ×4 | P2 | 连续4次爪击——速度递增 / 4 rapid claw strikes — increasing speed | 连续完美格挡×4→5秒输出窗口 / 4 perfect parries → 5s window |
| **吞噬 / Devour** | 特殊 / Special | P2 | 张嘴前冲——抓住后大幅掉血 / Opens mouth, charges — grab = big damage | 被抓后疯狂按键O挣脱 / Mash ○ to escape |

**NG+变化 / NG+ Changes**:
- 酸雨从3个安全区变为2个——站位要求更精确 / Acid rain safe zones reduced from 3 to 2 — more precise positioning needed
- 狂暴连击从4下增加到6下——但反击窗口延长到7秒 / Berserk combo increased from 4 to 6 hits — counter window extended to 7s
- 新增触须乱舞——场地多根触须无差别攻击，需要持续闪避 / New: Tentacle Frenzy — multiple tentacles attack randomly, continuous dodging required
- 酸液喷射后地面残留酸液区域——持续伤害 / Acid leaves puddles after spit — area DoT

## 第四章BOSS：蕾雯 / Raven

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★★★☆（困难 / Hard） |
| **类型 / Type** | 人形BOSS / Humanoid Boss |
| **弱点 / Weakness** | 攻击节奏固定，可预测 / Fixed attack rhythm, predictable |

蕾雯是游戏中最难的BOSS之一——三阶段战斗，每个阶段攻击模式不同。做好反复尝试的准备 / Raven is one of the hardest bosses — 3-phase fight with different patterns per phase. Be prepared for multiple attempts.

**阶段1（100%-60%）**: 剑气斩(完美格挡→反击)、瞬移连击(前2下格挡→红光部分闪避)、暗影突袭(红光——侧闪)、飞刀投掷(格挡反弹) / Blade wave (perfect parry → counter), Teleport combo (block first 2 → dodge red parts), Shadow assault (red — side dodge), Knife throw (parry back).

**阶段2（60%-30%）**: 分身剑阵(观察影子轨迹→闪避最后一击)、快速连斩+终结(格挡前3下→闪避红光终结)、暗黑领域(红光——离开黑色区域) / Clone sword array (watch shadow patterns → dodge final strike), Fast combo + finisher (block first 3 → dodge red finisher), Dark zone (red — leave dark zone).

**阶段3（30%-0%）**: 全力猛攻(连续格挡+闪避→等待反击时机)、终极斩击(红光——完美闪避→大量输出时间)、残影连击(全部完美格挡→处决级反击窗口) / All-out assault (chain block + dodge → wait for counter window), Ultimate slash (red — perfect dodge → large damage window), Afterimage combo (all perfect parries → execution-level window).

**详细攻击阶段分析 / Detailed Pattern Breakdown**:

| 攻击模式 / Pattern | 类型 / Type | 阶段 / Phase | 特征 / Tell | 应对 / Counter |
|------|:----:|:----:|------|------|
| **剑气斩 / Blade Wave** | 格挡可 / Blockable | P1 | 剑蓄力→横斩发出剑气 / Sword charges → horizontal wave | 完美格挡→反击追击 / Perfect parry → counter pursuit |
| **瞬移连击 / Teleport Combo** | 格挡×2→红光 / Block×2→red | P1 | 消失→出现→前2下格挡→第3下红光 / Vanishes → appears → block 2 → dodge 3rd | 格挡→格挡→闪避 / Block → block → dodge |
| **暗影突袭 / Shadow Assault** | 红光 / Red | P1 | 化为黑影冲刺——速度快 / Becomes shadow, charges fast | 听到声效瞬间闪避 / Dodge the moment sound plays |
| **飞刀投掷 / Knife Throw** | 格挡可反弹 / Blockable, parryable | P1 | 后跳+投掷飞刀×3 / Backstep + throws 3 knives | 连续完美格挡反弹回去伤害蕾雯 / Chain parry to reflect damage back at Raven |

| **暗黑领域 / Dark Zone** | 特殊 / Special | P2 | 场地出现黑圈——持续伤害 / Dark circles on ground — DoT | 迅速离开黑色区域 / Leave dark zone immediately |
| **终极斩击 / Ultimate Slash** | 红光——完美闪避窗口 / Red — perfect dodge window | P3 | 全屏锁定→巨大剑气斩下 / Full-screen lock-on → massive blade falls | 完美闪避→约5秒输出窗口——此战最大输出机会 / Perfect dodge → ~5s window — biggest damage opportunity |
| **残影连击 / Afterimage Combo** | 全部完美格挡 / All perfect parry | P3 | 蕾雯产生多个残影连续攻击 / Multiple afterimages attack in sequence | 全部完美格挡→7秒处决级反击窗口 / All perfect parries → 7s execution-level window |

**NG+变化 / NG+ Changes**:

- 狂暴状态判定——战斗时间超过8分钟后BOSS攻击频率大幅增加 / Enrage timer — after 8 min, boss attack frequency increases dramatically

## 最终BOSS：命运守护者 / Destiny Guardian

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★★★★（极难 / Extreme） |
| **类型 / Type** | 终极奈提巴 / Ultimate Naytiba |
| **弱点 / Weakness** | 部分攻击固定节奏，可以利用反击 / Fixed rhythm on certain attacks |

游戏最难战斗——三阶段+超高伤害。建议满级满装备再来挑战 / Hardest fight in the game — 3 phases with extreme damage. Come at max level with max gear.

**阶段1（100%-70%）**: 光剑连斩(连续完美格挡×4→反击)、能量波(跳跃或格挡)、瞬移突进(红光——听声音闪避)、重力场(拉开距离等消失) / Light saber combo (chain parry ×4 → counter), Energy wave (jump or block), Teleport rush (red — dodge on sound), Gravity field (create distance).

**阶段2（70%-35%）**: 光翼斩(格挡→反击——范围极大)、追踪能量弹(红光——绕场跑让它们撞墙)、全屏激光(红光——躲柱子后)、分身召唤(速清分身→集中本体) / Light wing slash (block → counter, wide range), Tracking orb (red — run to make them hit walls), Full-screen laser (red — hide behind pillar), Clone summon (clear clones fast → focus main).

**阶段3（35%-0%）**: 连续光剑风暴(格挡/闪避交替约15秒——纯耐力考验)、末日冲击(红光——场地边缘安全)、终极处决(最后阶段——全部完美格挡后触发结局动画) / Continuous blade storm (alternating block/dodge for ~15s — endurance test), Doom impact (red — safe at edges), Ultimate execution (final phase — all perfect parries trigger ending cutscene).

**详细攻击阶段分析 / Detailed Pattern Breakdown**:

| 攻击模式 / Pattern | 类型 / Type | 特征 / Tell | 应对 / Counter |
|------|:----:|------|------|
| **光剑连斩 / Light Saber Combo** | 格挡可×4 / Blockable×4 | 连续4下光剑挥舞——速度均匀 / 4 steady light saber swings | 连续完美格挡×4→3秒反击窗口 / Chain perfect parry×4→3s counter |
| **能量波 / Energy Wave** | 格挡可 / Blockable | 剑上举→地面冲击波扩散 / Sword raised → ground wave spreads | 跳跃或格挡——跳跃可从上方避开 / Jump or block — jump avoids entirely |
| **瞬移突进 / Teleport Rush** | 红光——不可格挡 / Red — unblockable | 消失→从前方突进——伴随音效 / Vanishes → rushes from front with sound cue | 声效响起的瞬间完美闪避 / Perfect dodge at sound cue |
| **重力场 / Gravity Field** | 特殊 / Special | 剑插地→地面紫色引力场 / Sword planted → purple gravity zone | 迅速后撤到范围外等待消失 / Retreat fast, wait for it to dissipate |

| **全屏激光 / Full-screen Laser** | 红光——不可格挡 / Red — unblockable | 翅膀发光→全屏激光横扫 / Wings glow → screen-wide laser sweep | 躲在场景柱子后面 / Hide behind pillars |

| **末日冲击 / Doom Impact** | 红光——场地边缘安全 / Red — safe at edges | 全屏冲击——场地中心伤害最高 / Full-screen blast — center = highest damage | 跑到场地边缘 / Run to arena edge |

## 隐藏BOSS：远古守卫 / Ancient Guardian

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **难度 / Difficulty** | ★★★★☆（困难 / Hard） |
| **解锁条件 / Unlock** | 大沙漠找到并激活3个古代石碑 / Find 3 ancient tablets in Great Desert |
| **推荐等级 / Recommended Level** | 35+ |

远古守卫攻击节奏与主线BOSS不同——更慢但单发伤害更高。红光攻击前摇很短——需要快速反应。击败后获得游戏中最强外脊之一 / Slower attack rhythm but higher single-hit damage than story bosses. Red attack tells are very short — quick reflexes needed. Defeating it grants one of the best Exospines.

**详细攻击阶段分析 / Detailed Pattern Breakdown**:

| 攻击模式 / Pattern | 类型 / Type | 特征 / Tell | 应对 / Counter |
|------|:----:|------|------|
| **巨锤砸地 / Hammer Slam** | 格挡可 / Blockable | 高举石锤→砸地造成大范围震波 / Raises stone hammer → ground slam + shockwave | 完美格挡锤击→跳跃闪避震波 / Perfect parry hammer → jump over shockwave |
| **旋转横扫 / Spin Sweep** | 格挡可——连续2次 / Blockable — 2 hits | 旋转石锤360度横扫×2 / 360° spin sweep×2 | 连续格挡×2→约2秒窗口 / Block×2 → ~2s window |
| **岩石投掷 / Rock Throw** | 红光——不可格挡 / Red — unblockable | 从地面拔起岩石砸向玩家 / Pulls rock from ground, throws at player | 侧向闪避——投掷有抛物线轨迹 / Side dodge — projectile follows arc |
| **大地震击 / Earth Shock** | 红光——跳跃闪避 / Red — jump dodge | 双拳砸地→全屏震波——前摇大但速度快 / Double fist slam → full-screen shockwave — big tell, fast execution | 看到砸地动作立即跳跃 / Jump immediately on slam motion |
| **连锤追击 / Hammer Chase** | 格挡可×3 / Blockable×3 | 连续3次向前砸地——每次前移 / 3 consecutive forward slams — advances each hit | 连续完美格挡×3→约3秒窗口 / Chain perfect parry×3 → ~3s window |
| **狂怒爆发 / Rage Burst** | 红光——特殊 / Red — special | HP低于30%时触发——全身发光→连续AOE爆发 / Triggered below 30% HP — glows → continuous AoE bursts | 拉开距离等待爆发结束——约10秒 / Create distance, wait it out — ~10s |

**打法要点**: 这个Boss的攻击全是大开大合的动作——每个攻击后都有安全窗口。它的难点不在于连招长度，而在于单发伤害(被打中=半血)。保持耐心，不要试图速杀——4-5分钟慢慢磨死。注意二周目中该BOSS新增第三种攻击模式——召唤落石，需要不断移动 / **Tips**: All attacks are telegraphed with safe windows. Difficulty is in single-hit damage (one hit = half HP). Patience is key — 4-5 minute slow kill is the safe approach. In NG+, the boss gains a third phase with falling rocks — constant movement required.

**NG+变化 / NG+ Changes**:
- HP翻倍 / HP doubled
- 新增P2.5阶段(40%HP)——召唤落石+连续地震波 / New P2.5 phase (40% HP) — falling rocks + chain shockwaves
- 大地震击变为连续3次——需要连续跳跃3次 / Earth Shock becomes triple — 3 consecutive jumps required
- 击败后获得暴君外脊 / Reward: Tyrant Exospine
