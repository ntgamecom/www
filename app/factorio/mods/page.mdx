## 模组生态 / Mod Ecosystem

Factorio 拥有最活跃的模组社区之一。通过内置的模组门户 (Mod Portal)，你可以一键安装和更新模组。模组分为三大类：大型改造模组、质量提升模组和功能扩展模组。

Factorio has one of the most active modding communities. Using the built-in Mod Portal, you can install and update mods with one click. Mods fall into three categories: overhaul mods, quality-of-life mods, and extension mods.

## 大型改造模组 / Overhaul Mods

### 太空探索 / Space Exploration (SE)
由 Earendel 开发。将游戏扩展到整个星系。你需要建造轨道平台、探索其他星球、处理跨星球的物流。流程可达 500+ 小时。

Developed by Earendel. Expands the game to an entire star system. Build orbital platforms, explore other planets, manage interplanetary logistics. Playtime can exceed 500 hours.

#### 核心特性 / Core Features
- 多星球基地 / Multi-planet bases
- 轨道平台 / Orbital platforms
- 太空缆车 / Space elevators
- 天体科学包 / Astro science packs
- 能量束传输 / Energy beam transmission

### 科雷斯托里奥 2 / Krastorio 2 (K2)
由 Krastor 开发。保留原版核心感觉但大大丰富。更多资源、更多建筑、更多科技。约 100-150 小时流程。

Developed by Krastor. Retains the core Factorio feel while greatly expanding content. More resources, more buildings, more technologies. About 100-150 hours of gameplay.

#### 核心特性 / Core Features
- 新矿石：绿铁矿、蓝铁矿 / New ores: mineral ores
- 生物处理 / Biological processing
- 高级军事 / Advanced military
- 超级建筑 / Super buildings
- 大矿机 / Big miners

### 鲍勃模组 / Bob's Mods
由 Bobingabout 开发。模组全家桶，增加大量新矿石、新中间品、新建筑和新科技。极度复杂。

Developed by Bobingabout. A full mod suite adding many new ores, intermediates, buildings, and technologies. Extremely complex.

#### 核心模组 / Core Mods
- Bob's Ores — 新矿物 / New minerals
- Bob's Electronics — 多级电路 / Multi-tier circuits
- Bob's Assembly — 多级组装机 / Multi-tier assemblers
- Bob's Power — 多种发电方式 / Multiple power generation methods

### 天使模组 / Angel's Mods
由 Arch666Angel 开发。与鲍勃模组高度兼容，专注于原材料加工链。多层级的矿石处理流程。

Developed by Arch666Angel. Highly compatible with Bob's mods, focusing on raw material processing chains. Multi-tier ore processing.

#### 核心特性 / Core Features
- 贫化矿石处理 / Lean ore processing
- 多级冶炼 / Multi-stage smelting
- 废料回收 / Waste recycling
- 生物燃料 / Bio fuel

### 异星工业 / Pyanodon's Mods (Py)
由 Pyanodon 开发。终极复杂模组系列。流程可能超过 1000 小时。不建议新手尝试。

Developed by Pyanodon. The ultimate complex mod series. Playtime can exceed 1000 hours. Not recommended for beginners.

#### 核心特性 / Core Features
- 数十种新矿石 / Dozens of new ores
- 真实化学配方 / Realistic chemical recipes
- 复杂建筑链 / Complex building chains
- 动物养殖 / Animal farming

## 质量提升模组 / Quality of Life (QoL) Mods

### 必备 QoL 模组 / Essential QoL Mods

#### 工厂规划器 / Factory Planner
游戏内计算器，比网页版更方便。可视化显示生产线所需的所有建筑和材料。

In-game calculator, more convenient than web versions. Visualizes all buildings and materials needed for a production line.

#### 速率计算器 / Rate Calculator
选中一组建筑，实时计算总输入/输出速率。调试生产瓶颈的利器。

Select a group of buildings and instantly calculate total input/output rates. Essential for debugging production bottlenecks.

#### 模块放入器 / Module Inserter
一键为所有建筑分配插件。支持从箱子中取用模块，并支持混合不同的模块组合。

Assign modules to all buildings with one click. Supports pulling modules from chests and mixing different module combinations.

#### 度娘 / Helmod
另一个工厂规划器，支持多级递归计算。适合复杂的模组环境。

Another factory planner supporting multi-level recursive calculations. Suitable for complex modded environments.

#### 远距离 / Long Reach
增加你的交互距离。不再需要走到建筑旁才能操作。

Increases your interaction range. No longer need to walk next to buildings to operate them.

#### 自动填充 / Auto Fill
向建筑中自动添加燃料和弹药。大幅减少手动操作。

Automatically adds fuel and ammo to buildings. Greatly reduces manual operations.

#### 背包管理 / Inventory Management
- Even Distribution — 均分物品到多个箱子 / Evenly distribute items across chests
- Bottleneck — 显示建筑是否在运行 / Shows whether buildings are running
- Todo List — 记录任务 / Track tasks

## 模组管理 / Mod Management

### 安装与更新 / Installation and Updates
在游戏标题页点击"模组"按钮。搜索模组并点击"安装"。推荐在开始新游戏前完成安装。

Click the "Mods" button on the game title screen. Search for a mod and click "Install". Recommended to install before starting a new game.

### 兼容性 / Compatibility
大型模组通常标注兼容性。建议查看模组描述和讨论区。K2+SE 是常见的组合。

Overhaul mods usually note compatibility. Check mod descriptions and discussion forums. K2+SE is a common combination.

### 存档安全 / Save Safety
备份存档后安装模组。卸载模组可能导致存档损坏。大型模组的存档不能卸载该模组。

Back up saves before installing mods. Uninstalling a mod can corrupt saves. Saves for overhaul mods cannot have the mod removed.

### 蓝图字符串 / Blueprint Strings
Factorio 使用长字符串分享蓝图。用 Ctrl+C 复制建筑，然后 Ctrl+V 导出字符串。你可以在论坛和网站分享它们。

Factorio uses long strings to share blueprints. Press Ctrl+C to copy buildings, then Ctrl+V to export the string. Share them on forums and websites.
