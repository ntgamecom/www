## 科研系统概述 / Research System Overview

Factorio 的科技树是游戏进程的主干。你必须生产科技包并送入实验室，才能解锁新的配方、建筑和升级。科技包共有 7 种，从易到难排列。

Factorio's tech tree is the backbone of game progression. You must produce science packs and feed them into labs to unlock new recipes, buildings, and upgrades. There are 7 types of science packs, ordered from easiest to hardest.

## 七种科技包 / Seven Science Packs

### 1. 自动化科技包 (红色) / Automation Science Pack (Red)
- 配方：1 齿轮 + 1 铁板 / Recipe: 1 Gear + 1 Iron Plate
- 所需建筑：组装机 1 / Building: Assembler 1
- 前置科技：无 / Prerequisite: None
- 产量建议：5-10 包/分钟 / Recommended rate: 5-10 packs/min

### 2. 物流科技包 (绿色) / Logistic Science Pack (Green)
- 配方：1 机械臂 + 1 传送带 / Recipe: 1 Inserter + 1 Belt
- 所需建筑：组装机 1 / Building: Assembler 1
- 前置科技：物流学 / Prerequisite: Logistics
- 产量建议：12-24 包/分钟 / Recommended rate: 12-24 packs/min

### 3. 军事实力科技包 (黑色) / Military Science Pack (Black)
- 配方：1 穿甲弹匣 + 1 手榴弹 + 1 墙壁 / Recipe: 1 Piercing Round + 1 Grenade + 1 Wall
- 所需建筑：组装机 2 / Building: Assembler 2
- 前置科技：军事实力 / Prerequisite: Military
- 产量建议：6-12 包/分钟 / Recommended rate: 6-12 packs/min

### 4. 化工科技包 (蓝色) / Chemical Science Pack (Blue)
- 配方：1 发动机单元 + 1 红板 + 1 硫磺 / Recipe: 1 Engine Unit + 1 Red Circuit + 1 Sulfur
- 所需建筑：组装机 2，化工厂 / Building: Assembler 2, Chemical Plant
- 前置科技：石油加工 / Prerequisite: Oil Processing
- 产量建议：8-16 包/分钟 / Recommended rate: 8-16 packs/min

### 5. 生产科技包 (黄色) / Production Science Pack (Yellow)
- 配方：1 电炉 + 1 生产模块 1 + 1 机械臂蓝 / Recipe: 1 Electric Furnace + 1 Prod Module 1 + 1 Blue Inserter
- 所需建筑：组装机 3 / Building: Assembler 3
- 前置科技：先进材料加工 2 / Prerequisite: Advanced Material Processing 2
- 产量建议：8-12 包/分钟 / Recommended rate: 8-12 packs/min

### 6. 高级科技包 (紫色) / Utility Science Pack (Purple)
- 配方：1 飞行机械人框架 + 1 速度模块 1 + 1 低密度结构 / Recipe: 1 Flying Bot Frame + 1 Speed Module 1 + 1 LDS
- 所需建筑：组装机 3 / Building: Assembler 3
- 前置科技：机械人学 / Prerequisite: Robotics
- 产量建议：6-10 包/分钟 / Recommended rate: 6-10 packs/min

### 7. 太空科技包 (白色) / Space Science Pack (White)
- 获取方式：通过火箭发射卫星获得 / Obtained by launching satellites via rocket
- 所需建筑：火箭发射场 + 卫星 / Building: Rocket Silo + Satellite
- 前置科技：火箭学 + 卫星 / Prerequisite: Rocketry + Satellite
- 用途：无限循环科技 / Used for infinite repeatable research

## 实验室设置 / Lab Setup

### 实验室布局 / Lab Layout
实验室之间可以通过机械臂传递科技包。将实验室排成行或网格，机械臂从一端传递到另一端。

Labs can pass science packs between each other using inserters. Arrange labs in rows or grids with inserters passing packs from one to the next.

### 实验室与传送带 / Labs and Belts
用传送带将 7 种科技包送到实验室群的一侧。使用机械臂将科技包从传送带送入实验室。

Use belts to deliver all 7 science pack types to one side of your lab array. Use inserters to move packs from belt to lab.

### 实验室数量 / How Many Labs
初期 4-8 个实验室即可。中后期扩展到 16-32 个。最终可以成百上千。

Start with 4-8 labs. Expand to 16-32 in mid-game. Hundreds in the endgame.

## 科技树优先级 / Research Priority

### 第一阶段 / Phase 1 (Red + Green)
优先研发：自动化 → 物流学 → 钢加工 → 电子学 → 石油加工（通往蓝瓶的关键）

Priority: Automation → Logistics → Steel → Electronics → Oil Processing (key to unlocking blue science)

### 第二阶段 / Phase 2 (Blue + Military)
优先研发：石油加工 → 硫磺加工 → 塑料 → 电池 → 先进电子学 → 发动机 → 飞行机械人

Priority: Oil Processing → Sulfur → Plastic → Batteries → Advanced Electronics → Engines → Robotics

### 第三阶段 / Phase 3 (Yellow + Purple)
优先研发：铝土加工 → 先进材料加工 → 生产模块 3 → 速度模块 3 → 火箭学

Priority: Aluminum Processing → Advanced Materials Processing → Productivity Module 3 → Speed Module 3 → Rocketry

### 无限循环科技 / Infinite Repeatable Research
- 采矿生产力 (Mining Productivity) — 最优先 / Top priority
- 机械人速度 (Robot Speed) — 高优先 / High priority
- 武器伤害 (Weapon Damage) — 战斗时优先 / Priority during combat
- 插件效果倍数 (Inserter Bonuses) — 中等优先 / Medium priority

## 火箭发射指南 / Rocket Launch Guide

### 火箭配方 / Rocket Recipe
火箭需要四种组件：
- 火箭燃料 (Rocket Fuel) — 100 份
- 低密度结构 (LDS) — 100 份
- 火箭控制单元 (RCU) — 100 份
- 卫星 (Satellite) — 1 份（额外，用于太空科技包）

Rockets require four ingredients:
- Rocket Fuel — 100 units
- Low Density Structure (LDS) — 100 units
- Rocket Control Unit (RCU) — 100 units
- Satellite — 1 unit (optional, for space science)

### 火箭发射场 / Rocket Silo Setup
发射场占地约 12x12。周围用机械臂供给四类组件。使用带产能模块的组装机生产部件以获得额外产量。

The silo occupies roughly 12x12 tiles. Feed all four ingredient types via inserters. Use productivity modules in component assemblers for bonus output.

### 首次发射 / First Launch
目标：75 包/分钟科技 + 火箭发射。你需要大约 10-15 台炼油厂、全生产线和稳定的供电系统。

Target: 75 SPM science + rocket launches. You will need approximately 10-15 refineries, a full production line, and stable power.
