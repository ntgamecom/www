| 问题 / Question | 回答 / Answer |

| **推荐一周目流程？/ First playthrough?** | 探索全图+清理支线+收集遗物 / Full explore + clear side quests + collect relics |

| **每个章节推荐等级？/ Chapter recommended levels?** | 序章1-5 / 第一章10-30 / 第二章30-50 / 第三章50-70 / 失落京城70+ / Prologue 1-5 / CH1 10-30 / CH2 30-50 / CH3 50-70 / Lost Capital 70+ |
| **据点功能什么时候全部解锁？/ Hub full unlock?** | 第一章完成后解锁铁匠+药庐 / 第二章解锁藏书阁 / 第三章解锁训练场 / CH1: forge + apothecary / CH2: library / CH3: training yard |
