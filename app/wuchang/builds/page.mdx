# 配装Build / Builds Guide

明末：渊虚之羽的Build围绕武器流派和羽化系统展开。正确的属性加点+装备选择+法术搭配能让你事半功倍 / Builds in Wuchang revolve around weapon styles and the Feather system. Correct stat allocation + gear selection + spell combos make the game significantly easier.

---

## 开荒Build / Early Game Build

| 部位 / Slot | 装备 / Gear | 说明 / Notes |
|:----:|:----|:----|
| **主武器 / Main** | 铁刀(Iron Blade) | 初始武器可升级——开荒够用 / Starting weapon, upgradeable — sufficient for early game |
| **副武器 / Secondary** | 木棍(Wood Staff) | 群怪控制——旋风棍清杂兵 / Crowd control — Whirlwind Staff for mob clear |
| **法术 / Spells** | 土·护盾术 + 水·冰霜术 | 土护盾保命+冰霜术控场 / Earth Shield for survival + Water Frost for control |
| **防具 / Armor** | 旅人套装(Traveler Set) | 初期防御+中负重 / Early DEF + mid equip load |
| **属性加点 / Stats** | 体力20>精力15>羽化上限10>力量20 | 生存优先→再补输出 / Survival first → then DPS |

## 物理流Build / Physical Build

| 部位 / Slot | 装备 / Gear | 说明 / Notes |
|:----:|:----|:----|
| **主武器 / Main** | 青龙偃月刀(Blue Dragon Blade) | 力量型大范围刀——高伤害+高削韧 / STR-based wide blade — high dmg + high stagger |
| **副武器 / Secondary** | 白蜡枪(Ash Spear) | 对单突刺——Boss战补伤害 / Single-target thrust — boss DPS supplement |
| **法术 / Spells** | 金·锋刃术(破甲)+ 土·护盾术(霸体) | 锋刃术附魔+破甲 / Metal enchant for AP + Earth Shield for hyper armor |
| **防具 / Armor** | 将军套装(General Set) | 高物理防御+高韧性 / High physical DEF + high poise |
| **护身符 / Talisman** | 力量护符(STR Talisman) + 羽化加速符 | +力量+羽化值积累速度 / +STR + Feather build speed |
| **属性加点 / Stats** | 力量50>体力35>精力25>羽化上限15 | 纯物理——正面碾压 / Pure physical — face-tank and smash |

**打法**: 羽化反击积累羽化值→开羽化形态→青龙刀重击连招→毁天灭地 / **Playstyle**: Feather Counter build meter → Feather Form → blade heavy combos → devastation.

| **法术 / Spells** | 金·锋刃术(暴击率) + 火·烈焰术(AOE) | 锋刃术增加暴击率 / Metal enchant for crit rate boost |
| **防具 / Armor** | 刺客套装(Assassin Set) | 轻甲+暴击伤害+移动速度 / Light armor + crit dmg + move speed |

| **属性加点 / Stats** | 敏捷50>体力25>精力20>羽化上限15 | 高速暴击——反复横跳 / Fast crits — hit and run |
