# 世界探索 / World Exploration

明末：渊虚之羽的巴蜀世界设计灵感来自真实地理——锦官城、青城山、都江堰、乐山大佛等地标均有再现。地图多层且充满隐藏路径 / Wuchang's Shu region is inspired by real geography — JinGuan City, Qingcheng Mountain, Dujiangyan, Leshan Giant Buddha are all recreated. The map is multi-layered and full of secret paths.

## 锦官城 / JinGuan City

蜀地首府——明朝西南重镇，现已被妖化军队占领。城市分上下两层：地面街道和地下排水系统 / The capital of Shu — Ming Dynasty southwestern stronghold, now overrun by corrupted forces. Two layers: surface streets and underground sewer system.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **南城门 / South Gate** | 游戏起点——破败的城门 + 第一个篝火 / Starting point — ruined gate + first bonfire | 篝火、初级商人 / Bonfire, starter merchant |
| **商业街 / Market Street** | 主街——大量杂兵+可进入的建筑 / Main street — many mobs + enterable buildings | 隐藏武器库入口(井口) + 法书·土护盾 / Hidden armory (well) + Earth Shield spell |
| **将军府 / General's Mansion** | 妖化游击将军所在地——Boss战 / Corrupted General boss arena | Boss战 + 通往青城山路径 / Boss fight + path to Qingcheng |
| **地下排水系统 / Sewers** | 城市地下管网——连接多个隐藏区域 / Underground pipe network — connects multiple hidden areas | 玄铁双刀图纸、灵药种子、隐藏Boss / Dual Blade blueprint, herb seeds, secret boss |

## 青城山 / Qingcheng Mountain

道教名山——现在是妖道的老巢。山中道观林立，树林茂密，洞穴错综复杂 / Famous Taoist mountain — now the sorcerer's lair. Temples dot the mountain, dense forests, complex caves.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **山门道观 / Mountain Gate Temple** | 山脚入口——已废弃的道观 / Mountain entrance — abandoned temple | 商人、法书·木荆棘术 / Merchant, Wood Thorn spell |
| **三清殿 / SanQing Hall** | 山中主殿——妖道第一阶段战斗 / Main hall — sorcerer phase 1 | 隐藏暗门(香炉推动) + 灵药种子 / Secret door (incense burner) + herb seeds |
| **密林区域 / Dense Forest** | 后山密林——大量陷阱+隐藏洞穴 / Back mountain forest — many traps + hidden caves | 记忆碎片x2、精英妖兽 / Memory fragments x2, elite beasts |
| **青城山顶 / Mountain Summit** | 山顶祭坛——青城山妖道Boss战 / Summit altar — Qingcheng Sorcerer boss | Boss战 + 通往都江堰 / Boss fight + path to Dujiangyan |

## 都江堰 / Dujiangyan

伟大的古代水利工程——现在被蛟龙占据。水域面积巨大，大量区域需要潜水探索 / The great ancient irrigation system — now occupied by the Flood Dragon. Massive water areas, much underwater exploration required.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **飞沙堰 / Flying Sand Weir** | 入口区域——激流+水下通道 / Entrance — rapids + underwater passages | 水抗戒指、法书·水冰霜术 / Water resist ring, Water Frost spell |
| **宝瓶口 / Bottleneck Canal** | 狭窄水道——水下有隐藏洞穴 / Narrow canal — hidden underwater cave | 凶兽核x2、大量矿石 / Beast Cores x2, many ores |
| **离堆公园 / Lidui Park** | 江心小岛——蛟龙出没区域 / River island — dragon's domain | 传送点、记忆碎片 / Fast travel, memory fragment |
| **鱼嘴分水堤 / Fish Mouth Dam** | 都江堰核心——蛟龙Boss战 / Dam center — Flood Dragon boss | Boss战 + 通往乐山大佛 / Boss fight + path to Leshan |

## 乐山大佛 / Leshan Giant Buddha

世界最大的石佛——这座巨佛内部隐藏着通往皇城废墟的秘密通道。上下多层垂直地形 / The world's largest stone Buddha — inside hides a secret passage to the Imperial Ruins. Multi-layer vertical terrain.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **大佛脚下 / Buddha's Feet** | 大佛基座——入口区域 / Buddha's base — entrance area | 篝火、商人 / Bonfire, merchant |
| **大佛内部 / Inside the Buddha** | 佛身内部有禅室——隐藏大量宝箱 / Interior meditation chambers — many hidden chests | 终极法术·五行归一、佛骨舍利 / Ultimate spell, Buddha relic |
| **左臂平台 / Left Arm Platform** | 跳到手掌→手臂→肩膀→收集品 / Jump hand → arm → shoulder → loot | 武器强化素材+全属性抗性戒指 / Upgrade materials + all-resist ring |
| **大佛头顶 / Buddha's Crown** | 顶部——护法金刚Boss战所在地 / Top — Guardian Deity boss arena | Boss战 + 通往皇城废墟 / Boss fight + path to Imperial Ruins |

|:----:|------|------|

| **太和殿 / Hall of Supreme Harmony** | 主殿——最终Boss第一阶段战场 / Main hall — final boss phase 1 | 记忆碎片、龙椅隐藏密室 / Memory fragment, throne secret room |
| **后宫 / Imperial Harem** | 后宫区域——隐藏了大量收集品 / Harem quarters — many collectibles hidden | 渊虚之羽碎片x1、终极武器素材 / Feather shard, ultimate weapon materials |
| **皇城地宫 / Palace Underground** | 隐藏区域——需要通过龙椅密室进入 / Hidden zone — enter via throne room | 最终武器图纸、隐藏Boss记忆 / Final weapon blueprint, secret boss memory |

## 探索技巧 / Exploration Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **墙壁裂缝可破坏 / Cracked walls** | 明末中有大量可破坏墙壁——用重击或滚翻撞开。通常藏着宝箱或捷径 / Many breakable walls — use heavy attack or roll through. Usually hide chests or shortcuts |
| **水底有东西 / Underwater secrets** | 都江堰等水域区域——水底有宝箱和隐藏通道。注意氧气条 / Underwater zones have chests and hidden passages. Watch oxygen bar |
| **高处佛像有平台 / Buddha platforms** | 乐山大佛区域可以在佛像身上攀爬跳跃——每个平台都有收集品 / Climb and jump on the Buddha statue — each platform holds collectibles |
| **道士的提示 / Taoist hints** | 部分NPC道士会给你隐藏宝物的提示——对话不要跳过 / Some Taoist NPCs give hints about hidden treasures — don't skip dialogue |
| **火把照亮隐藏墙壁 / Torch illuminates secrets** | 部分隐藏门需要火把/火焰法术照亮才能看到——看到墙壁颜色不一致就试试火 / Some hidden doors need torch/fire spell to reveal — try fire on walls with inconsistent texture |
