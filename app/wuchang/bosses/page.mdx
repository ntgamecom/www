# BOSS攻略 / Boss Strategies

明末：渊虚之羽的Boss设计结合了中国神话和明末历史——从妖化的明军将领到上古凶兽。每个Boss有多阶段和独特的羽化反击时机 / Wuchang's boss designs blend Chinese mythology with Ming Dynasty history — from corrupted Ming generals to ancient mythical beasts. Each boss has multiple phases and unique Feather Counter timings.

---

## 前期Boss / Early Game Bosses

### 妖化游击将军 / Corrupted General

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-50%)** | 长枪突刺x3→横扫→跳劈。节奏均匀 / Spear thrust x3 → sweep → jump slam. Even rhythm | 前三次突刺节奏一样——完美格挡第三下→反击。跳劈时向两侧闪避 / Thrusts have same rhythm — parry 3rd → counter. Side-dodge the jump slam |
| **P2 (50%-0%)** | 追加：妖化——全身黑气——双倍攻速+分身攻击 / Adds: corruption aura — double ATK speed + clone attack | 分身出现时→跑→等分身消失再打。P2不要主动进攻——等他出完连招再反击 / Run from clone → wait for it to fade. P2: don't engage — counter after his combos |
| **羽化反击时机 / Counter Window** | 突刺第三下和跳劈落地瞬间 / 3rd thrust and jump slam landing | 这两个时机的完美闪避触发羽化反击——伤害可观 / Perfect dodge on these two for big Feather Counter damage |

### 锦官城守将 / JinGuan City General

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-40%)** | 大刀——大范围横斩+蓄力竖劈+铁山靠 / Greatsword — wide sweep + charge vertical + shoulder tackle | 铁山靠前摇短(0.3秒)——需要快速反应。大范围横斩可以跳起来躲 / Shoulder tackle is fast (0.3s) — react quickly. Jump over the wide sweep |
| **P2 (40%-0%)** | 追加：召唤明军亡灵士兵——场上同时有3-4个小兵干扰 / Summons Ming soldier spirits — 3-4 adds on field | 优先清小兵→再打Boss。不要被包围 / Clear adds first → then boss. Don't get surrounded |
| **羽化反击时机** | 铁山靠和蓄力竖劈落地时 / Shoulder tackle and charge vertical landing | 铁山靠完美闪避→触发羽化反击→直接打断他后续连招 / Perfect dodge shoulder tackle → trigger Counter → cancel his follow-up combo |

## 中期Boss / Mid Game Bosses

### 青城山妖道 / Qingcheng Sorcerer

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-60%)** | 法术远程攻击(火球/冰刺) + 传送 + 近身杖击 / Ranged spell (fireball/ice) + teleport + staff melee | 法术可以跑位躲——不要翻滚(翻滚无敌帧短)。近身杖击是他的弱点——骗他传送→他出现时立刻攻击 / Run-dodge spells (roll i-frames are short). Staff melee is his weakness — bait teleport → attack immediately on reappearance |
| **P2 (60%-0%)** | 追加：五行法阵——地面出现五行圈→不同颜色不同效果(火圈烧伤/冰圈减速) / Adds: Five Elements circle — colored ground circles with different effects (fire=burn, ice=slow) | 站在没有圈的地方打。优先破坏场地边缘的法阵核心(发光点) / Stand in empty space. Prioritize destroying circle cores (glowing points) at arena edges |
| **属性弱点** | 物理伤害(他有高法抗低物抗) / Physical dmg (high magic resist, low physical) | 用物理Build打他比法术快2倍 / Physical build kills him 2x faster than magic |

### 都江堰蛟龙 / Dujiangyan Flood Dragon

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-60%)** | 水柱喷射(直线)+尾巴横扫+爪击+水下潜行突袭 / Water jet (line) + tail sweep + claw + underwater ambush | 水柱横向跑。尾巴横扫跳起来躲。水下潜行时水面有波纹→预判位置闪避 / Run perpendicular from water jet. Jump over tail sweep. Ripples on water surface telegraph ambush |
| **P2 (60%-20%)** | 召唤水龙卷(3个)+全屏水弹幕 / Summons 3 water tornadoes + full-screen water barrage | 水龙卷会缓慢追踪——不要被逼到角落。水弹幕时站在龙的身体下方是安全区 / Tornadoes slowly track — don't get cornered. Stand under dragon's body during barrage — safe zone |
| **P3 (20%-0%)** | 狂暴——全屏水淹(涨水)+连击x6 / Enrage — arena floods + 6-hit combo | 涨水时跑到场地最高的平台上。连击最后一下有完美闪避窗口 / Run to highest platform during flood. Final hit of combo has perfect dodge window |
| **羽化反击时机** | 水下突袭出现瞬间和水龙卷消失后 / Underwater ambush emergence and post-tornado | 这两个时机最容易触发羽化反击——积累羽化值后P3开羽化形态爆发 / Easiest counter windows — build meter, then pop Feather Form in P3 for burst |

### 乐山大佛·护法金刚 / Leshan Guardian Deity

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-50%)** | 巨型拳头砸地(AOE)+掌风(推人)+召唤石柱(阻碍移动) / Giant fist slam (AoE) + palm wind (push) + stone pillar summon (obstacle) | 拳头砸地前地面有裂纹——看到就闪避。掌风可以跳起来躲。石柱后可以利用柱子挡掌风 / Ground cracks before fist slam — dodge on sight. Jump over palm wind. Use pillars to block wind |
| **P2 (50%-20%)** | 追加：大佛金身(伤害减免50%)+掌心激光(横扫)+大量石柱 / Golden form (50% dmg reduction) + palm laser (sweep) + mass pillars | 激光横扫时躲在石柱后面。金身期间不要浪费技能——绕圈跑拖时间 / Hide behind pillars during laser sweep. Don't waste skills during golden form — kite until it ends |
| **P3 (20%-0%)** | 狂怒——双拳密集锤地(连续5次)+碎石弹幕 / Frenzy — double fist slam x5 + debris barrage | 连续5次锤地保持移动——不要停在同一位置。碎石弹幕向Boss方向跑(安全区) / Keep moving during 5x slam — never stay still. Run TOWARD boss during debris barrage (safe spot) |
| **羽化反击时机** | 拳头砸地瞬间和激光开始横扫前 / Fist slam landing and laser sweep startup | P1的拳砸地最好反击——时机宽裕。P2的激光横扫前闪避到Boss背后也可触发 / P1 fist slam is easiest counter. P2 laser sweep startup dodge behind boss also works |

## 后期Boss / Late Game Bosses

### 最终Boss：渊虚之羽 / Fallen Feathers (Final Boss)

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-60%)** | 人形态——快慢刀剑术+羽化瞬移+暗影突刺 / Human form — delayed swordplay + Feather teleport + shadow thrust | 快慢刀节奏变化极大——需要靠**目押**而非记忆节奏。不要贪刀 / Delayed attack rhythm varies wildly — rely on **visual reaction** not memorization. Don't get greedy |
| **P2 (60%-25%)** | 半羽化——双翼展开+大范围羽刃风暴+召唤暗影无常(分身) / Half-Feather — wings out + wide feather storm + summons shadow clone | 羽刃风暴时→连续向后闪避x4→等风暴结束。分身出来时速度打死→拖久了Boss会回血 / Back-dodge x4 during feather storm. Kill clone quickly — prolonged fight = boss heals |
| **P3 (25%-0%)** | 完全羽化——全屏羽毛弹幕+瞬移连击x8+终极秒杀技(蓄力3秒全屏AOE) / Full Feather — screen-filling feather barrage + 8-hit teleport combo + ultimate one-shot (3s charge AoE) | 秒杀技：看到蓄力立刻跑到地图**最边缘**——中心伤害最高。瞬移连击最后一下可以羽化反击 / One-shot: run to **edge** immediately on charge. Last hit of teleport combo is counterable |
| **属性弱点 / Weakness** | 交替——P1弱火、P2弱冰、P3弱雷 / Alternating — P1 fire, P2 ice, P3 lightning | 战斗中需要切换法术属性——建议带火+冰+雷三个法术 / Switch spell elements mid-fight — bring all three |

## 隐藏Boss / Secret Boss

### 上古凶兽·梼杌 / Taowu (Ancient Beast)

| 要素 / Factor | 说明 / Description |
|:----:|------|
| **出现条件 / Condition** | 收集全部12个"记忆碎片"→返回初始祭坛→触发隐藏Boss战 / Collect all 12 "Memory Fragments" → return to starting altar → trigger secret boss |
| **攻击模式 / Attacks** | 全属性攻击(五行轮流)+地面塌陷+召唤小凶兽 / All five elements rotation + ground collapse + summons mini-beasts |
| **打法 / Strategy** | 五行属性轮流切换防御——土属性时站远、火属性时近身。召唤小怪时速度AOE清掉 / Switch defensive stance per element — earth = keep distance, fire = close range. Clear summons with AoE |
| **奖励 / Reward** | 解锁真结局 + 最强武器"渊虚之羽"(全属性+全抗性) / Unlocks true ending + best weapon "Fallen Feathers" (all elements + all resistances) |
