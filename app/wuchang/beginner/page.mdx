# 新手入门 / Beginner Guide

明末：渊虚之羽是一款正统魂系游戏——高难度、高回报、强调背板和节奏。跟其他魂系不同的是，它的羽化系统提供了更主动的进攻手段 / Wuchang: Fallen Feathers is a traditional souls-like — high difficulty, high reward, emphasizing pattern memorization and rhythm. What sets it apart is the Feather system, which provides more aggressive offensive options.

---

## 核心机制 / Core Mechanics

| 机制 / Mechanic | 说明 / Description |
|:----:|------|
| **羽化值 / Feather Meter** | 攻击命中→积累羽化值。满值时→按特定按键触发"羽化形态" / Build on hit → full meter → trigger "Feather Form" |
| **羽化形态 / Feather Form** | 进入后：攻击力x1.5+移动速度提升+精力无限——持续15秒。期间可以使用**羽化终结技** / Active: ATK x1.5 + move speed boost + infinite stamina — 15 seconds. Can use **Feather Finisher** during this time |
| **羽化反击 / Feather Counter** | **核心机制**——敌人攻击命中前0.2秒内完美闪避/格挡→触发羽化反击(时停+大量伤害+大量羽化值) / **Core mechanic** — perfect dodge/block within 0.2s of enemy attack → triggers Counter (slow-mo + massive dmg + big Feather gain) |
| **精力系统 / Stamina** | 同魂系——攻击/闪避/格挡都消耗精力。精力归零=破防=被连 / Same as souls — attack/dodge/block all cost stamina. Zero stamina = guard break = combo death |
| **法术系统 / Magic** | 五行法术(金木水火土)——各有不同效果。消耗法力。法术可以通过探索获得的法书解锁 / Five Elements spells — each with unique effects. Consume mana. Unlocked via spell books found through exploration |

---

## 羽化系统详解 / Feather System Guide

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **被动积累 / Passive build** | 普通攻击命中积累羽化值——效率一般但不消耗资源 / Normal attacks build Feather — mediocre efficiency but resource-free |
| **主动积累 / Active build** | 羽化反击(完美闪避/格挡)是最高效的积累方式——一次反击可积累30%+羽化值 / Feather Counter (perfect dodge/block) is most efficient — one counter builds 30%+ meter |
| **羽化形态使用时机 / Feather Form timing** | Boss大硬直时开→15秒无限精力+高伤害→打满输出 / Activate during boss big openings → 15s of infinite stamina + high dmg → maximum DPS |
| **羽化终结技 / Feather Finisher** | 羽化形态中按重击+轻击同时→大范围终结技——约等于5次重击伤害 / In Feather Form → press heavy+light simultaneously → wide finisher — ~5x heavy attack damage |
| **不要浪费羽化值 / Don't waste meter** | 打小怪时可以不开羽化形态——留给Boss战。羽化值在战斗外会缓慢衰减 / Don't use Feather Form on mobs — save for bosses. Meter slowly decays outside combat |

## 属性加点 / Stat Allocation

| 属性 / Stat | 效果 / Effect | 优先 / Priority |
|:----:|------|:----:|
| **体力 / VIT** | +HP+物理防御 / +HP + Physical DEF | ★★★★★ (优先30) |
| **精力 / END** | +最大精力+精力恢复速度 / +Max Stamina + Stamina regen | ★★★★★ (优先25) |
| **力量 / STR** | +刀/枪/棍伤害 / +Blade/Spear/Staff dmg | ★★★★☆ (物理Build) |
| **敏捷 / DEX** | +双刀/剑伤害+暴击率 / +Dual Blades/Sword dmg + Crit rate | ★★★★☆ (敏捷Build) |
| **智力 / INT** | +法术伤害+羽化值积累速度 / +Magic dmg + Feather build speed | ★★★☆☆ (法系Build) |
| **羽化上限 / Feather Cap** | +最大羽化值 / +Max Feather meter | ★★★★☆ (全Build) |

**新手建议加点顺序**: 体力15→精力15→羽化上限10→体力25→精力20→主属性30→体力35 / **Beginner stat order**: VIT 15 → END 15 → Feather Cap 10 → VIT 25 → END 20 → Main stat 30 → VIT 35.

|:----:|:----:|:------|

| **双刀 / Dual Blades** | ★★★★☆ | 极速连击+暴击——需要高熟练度 / Ultra-fast combos + crit — requires high skill |

## 新手常见错误 / Common Beginner Mistakes

| 错误 / Mistake | 正确做法 / Correct |
|:----:|------|
| **不用羽化反击 / Not using Feather Counter** | 羽化反击是游戏的核心——练好完美闪避。每个Boss花5分钟专门练 / Feather Counter is the core — practice perfect dodge. Spend 5 min per boss just practicing |
| **羽化值满了不用 / Holding full meter** | 羽化值满了不用等于浪费——Boss大硬直时立刻开 / Full meter not used = waste. Pop it during boss recovery windows |
| **忽视法术 / Ignoring magic** | 五行法术非常强力——火焰法术清杂兵、水法术减速、土法术护盾 / Five Elements magic is very strong — fire for AoE clear, water for slow, earth for shield |
| **堆错属性 / Wrong stats** | 确定武器流派再堆属性。不要力敏智都加——**专精一项** / Decide your weapon first, then stack its main stat. Don't spread — **specialize** |
