# 技能与法术 / Skills & Magic

明末渊虚之羽的技能系统围绕"羽化"概念设计——玩家可以解锁强大的羽化能力和武器技能 / Wuchang's skill system is designed around the "Fallen Feathers" concept — players unlock powerful feather abilities and weapon skills.

---

## 羽化系统 / Feather Skills

羽化是游戏的核心能力——消耗羽毛资源释放强力技能 / Feather skills are the game's core abilities — consume feather resources to unleash powerful attacks.

### 基础羽化技能 / Basic Feather Skills

| 技能 / Skill | 羽毛消耗 / Feather Cost | 效果 / Effect |
|---|---|---|
| **羽刃 / Feather Blade** | 1 | 远程投掷羽毛刃——牵制+打断 / Ranged feather throw — harass + interrupt |
| **羽盾 / Feather Shield** | 1 | 瞬间生成护盾——吸收一次攻击 / Instant shield — absorb one hit |
| **羽翼突击 / Wing Rush** | 2 | 快速突进——造成伤害+击退 / Fast rush — damage + knockback |
| **羽化风暴 / Feather Storm** | 3 | 全方位羽毛弹幕——范围AOE / Omni-directional feather barrage — AOE |

### 进阶羽化技能 / Advanced Feather Skills

| 技能 / Skill | 羽毛消耗 / Feather Cost | 解锁条件 / Unlock |
|---|---|---|
| **羽化·天降 / Feather Descent** | 3 | 完成特定支线 / Complete specific side quest |
| **羽化·影袭 / Shadow Feather** | 2 | 击败暗影Boss / Defeat shadow boss |
| **羽化·净化 / Purify Feather** | 2 | 找到遗物碎片 / Find relic fragment |
| **羽化·重生 / Rebirth Feather** | 全部 / All | 最终技能——满血复活 / Ultimate skill — full revival |

### 羽毛资源管理 / Feather Resource Management

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **羽毛上限 / Max Feathers** | 初始3个——升级可增加到6个 / Start with 3 — upgrade to 6 max |
| **羽毛恢复 / Feather Regen** | 击败敌人恢复+特定地点采集 / Enemy kills + specific gather points |
| **羽化值 / Feather Gauge** | 屏幕底部的羽毛图标——消耗+恢复 / Feather icons at screen bottom — spend + recover |

## 技能升级 / Skill Upgrades

### 升级方式 / Upgrade Methods

| 方式 / Method | 说明 / Description |
|---|---|
| **技能点 / Skill Points** | 升级获得——用来解锁技能树节点 / Gained on level up — spend on skill tree |
| **羽毛强化 / Feather Enhance** | 找特定NPC消耗材料升级羽化技能 / Specific NPC — spend materials to upgrade |
| **武器熟练度 / Weapon Mastery** | 使用武器战斗累积熟练度——解锁武器技能 / Use weapon to build mastery — unlock weapon skills |

### 技能树 / Skill Tree

| 分支 / Branch | 说明 / Description |
|---|---|
| **羽化分支 / Feather Branch** | 强化所有羽化技能 / Enhance all feather skills |
| **武器分支 / Weapon Branch** | 解锁+强化武器技能 / Unlock + enhance weapon skills |
| **生存分支 / Survival Branch** | 增加HP+防御+回复 / HP + defense + recovery |
| **探索分支 / Exploration Branch** | 增加掉率+感知+潜行 / Drop rate + perception + stealth |

## 技能常见问题 / Skills FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **优先加哪个属性？/ Priority stat?** | 推荐优先加敏捷（攻速+闪避收益高）/ DEX first (attack speed + dodge) |
| **羽毛不够用？/ Not enough feathers?** | 加灵性属性+找增加羽毛上限的遗物 / Level Spirit + find feather cap relics |
| **最强羽化技能？/ Best feather skill?** | 羽翼突击（性价比最高）/ Wing Rush (best value) |
| **武器技能可以换吗？/ Swap weapon skills?** | 可以在篝火处更换装备的技能 / Swap at bonfires |
| **遗物可以升级吗？/ Upgrade relics?** | 可以——找遗物匠人消耗材料升级 / Yes — relic artisan uses materials |
| **武器熟练度怎么提升？/ Increase weapon mastery?** | 用该武器打怪——击败敌人累积熟练度。换武器打不同敌人保持效率 / Use the weapon to fight — kills build mastery. Swap weapons to vary |
| **遗物碎片在哪？/ Relic fragment locations?** | 第一章隐藏宝箱+精英掉落+支线奖励——全图共12个 / CH1 hidden chest + elite drops + quest rewards — 12 total across map |
| **武器技能可以全部学会吗？/ Can I learn all weapon skills?** | 可以——但需要大量技能点。推荐先专注1-2种武器 / Yes — but needs many skill points. Recommend focus on 1-2 weapons first |
| **遗物可以同时装备多个吗？/ Equip multiple relics?** | 可以——最多同时装备3个遗物。注意遗物效果叠加 / Yes — max 3 relics equipped simultaneously. Effects stack |
