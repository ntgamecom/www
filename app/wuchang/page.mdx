# 明末：渊虚之羽攻略 / Wuchang: Fallen Feathers Guide

《明末：渊虚之羽》是一款国产魂系动作角色扮演游戏，背景设定在明末乱世——崇祯年间，天下大乱，妖邪横生。玩家扮演失忆的女剑士无常，在巴蜀之地追寻记忆碎片，封印渊虚之羽的诅咒 / Wuchang: Fallen Feathers is a Chinese-developed souls-like action RPG set during the Ming Dynasty's fall. Play as Wuchang, an amnesiac female swordswoman, traversing the Shu region to recover lost memories and seal the cursed "Fallen Feathers."

## 游戏特色 / Key Features

| 特色 / Feature | 说明 / Description |
|---|---|
| **明末国风背景 / Ming Dynasty Setting** | 崇祯年间的巴蜀——古建筑+山水画风+中式怪物设计 / 1644 Sichuan — ancient architecture + ink-wash art + Chinese mythological monsters |
| **羽化系统 / Feather System** | 核心机制——积累"羽化值"→触发"羽化形态"——攻防大幅提升+特殊技能 / Core mechanic — build "Feather meter" → enter "Feather Form" — massive ATK/DEF boost + special skills |
| **双武器系统 / Dual Weapons** | 剑/刀/枪/棍/双刀——每种武器有独立技能树。可在战斗中切换 / Sword/Blade/Spear/Staff/Dual Blades — each with independent skill tree. Switch mid-combat |
| **羽化反击 / Feather Counter** | 在敌人攻击命中前完美闪避或格挡→触发羽化反击——时停+大伤害 / Perfect dodge or block before enemy hit → triggers Feather Counter — slow-mo + massive damage |
| **五行法术 / Five Elements Magic** | 金木水火土五种法术——消耗法力施放。每种有独特效果 / Metal/Wood/Water/Fire/Earth — each with unique effects. Consume mana to cast |
| **巴蜀世界 / Shu World** | 多层地图设计——锦官城、青城山、都江堰、乐山大佛等真实地点改编 / Multi-layer map — JinGuan City, Qingcheng Mountain, Dujiangyan, Leshan Giant Buddha reimagined |

## 新手快速入门 / Quick Start

| 阶段 / Phase | 建议 / Tips |
|---|---|
| **开局 / Start** | 推荐**刀**开局——攻速+伤害均衡。优先学习**羽化反击**——这是游戏的核心技能 / Start with **Blade** — balanced speed and damage. Prioritize **Feather Counter** — the core mechanic |
| **前期 / Early** | 加点优先**体力**和**羽化值上限**——生存>输出。前期不要追求完美Build / Prioritize **VIT** and **Feather cap** — survival > DPS. Don't chase perfect builds early |
| **中期 / Mid** | 开始补**主属性**(力/敏/智)——根据武器选择。探索隐藏区域获取强力法术 / Start investing in main stat (STR/DEX/INT) based on weapon. Explore hidden areas for powerful spells |
| **后期 / Late** | Build成型——专注于Boss战和收集全成就。隐藏Boss和真结局需要全收集 / Build complete — focus on boss fights and achievement hunting. Secret ending requires full collection |
