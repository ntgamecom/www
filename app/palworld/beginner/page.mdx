# 幻兽帕鲁 新手入门 / Palworld Beginner Guide

## 版本说明 / Version Info

适配最新正式版 / Compatible with the latest release version

## 开局路线 / Starting Route

### 1. 角色创建与初始区域 / Character Creation & Starting Area

游戏开始后，你会在**初始高原**醒来。首先调整角色外观，然后跟随教程指引收集基本资源 / You wake up on the **Initial Plateau**. Customize your character, then follow the tutorial to gather basic resources.

- **敲石头**收集石头和燧石 / **Mine rocks** for stone and flint
- **砍树**收集木材 / **Chop trees** for wood
- **采集灌木**获得浆果 / **Harvest bushes** for berries

### 2. 制作第一把工具 / Craft Your First Tools

打开制作菜单（按 B 键），依序制作 / Open the crafting menu (press B) and craft in order:

1. **石斧（Stone Hatchet）**—— 砍树效率更高 / Better tree chopping
2. **石镐（Stone Pickaxe）**—— 采石效率更高 / Better stone mining
3. **火把（Torch）**—— 照亮夜间，驱赶低级帕鲁 / Light the night, repel low-level Pals

### 3. 搭建临时营地 / Set Up a Temporary Camp

选择一个平坦区域，放置 / Choose a flat area and place:

- **篝火（Campfire）** —— 烹饪食物、保暖 / Cook food, stay warm
- **工作台（Workbench）** —— 制作更多工具和武器 / Craft more tools and weapons
- **床（Bed）** —— 设置重生点 / Set your spawn point
- **箱子（Chest）** —— 存放物品 / Store items

### 4. 捕捉第一只帕鲁 / Catch Your First Pal

- 将**帕鲁球（Pal Sphere）**放在快捷栏 / Place **Pal Spheres** in your hotbar
- 找到低等级帕鲁（如**企丸丸（Pengullet）**或**毛掸儿（Cattiva）**）/ Find low-level Pals like **Pengullet** or **Cattiva**
- **削弱它**——攻击直到血量变低 / **Weaken it** — attack until HP is low
- **投掷帕鲁球**——瞄准并按 Q（PC）/ **Throw Pal Sphere** — aim and press Q (PC)
- 成功捕获后，帕鲁会加入你的队伍 / On success, the Pal joins your party

### 5. 建立永久基地 / Build a Permanent Base

找到合适位置建立**帕鲁终端（Pal Box）**，选择地势平坦、靠近资源点的位置 / Find a good spot for the **Pal Box** — flat terrain near resources.

- **帕鲁终端**是基地核心，升级它可扩展建造范围 / **Pal Box** is the base core, level it to expand building range
- 建造**饲料箱（Feed Box）**让帕鲁自动获取食物 / Build a **Feed Box** so Pals auto-feed
- 建造**温泉（Hot Spring）**恢复帕鲁的SAN值（理智）/ Build a **Hot Spring** to restore Pal SAN (sanity)
- 建造**帕鲁床（Pal Bed）**让帕鲁休息 / Build **Pal Beds** for Pals to sleep

## 基础机制 / Core Mechanics

### 帕鲁工作类型 / Pal Work Types

每只帕鲁有不同工作适应性，合理安排工作 / Each Pal has different work suitabilities, assign wisely:

| 工作类型 / Work Type | 图标 / Icon | 用途 / Use |
|------|------|----------|
| **生火 / Kindling** | 🔥 | 冶炼金属、烹饪 / Smelt metal, cook |
| **浇水 / Watering** | 💧 | 种植作物 / Water crops |
| **种植 / Planting** | 🌱 | 播种 / Plant seeds |
| **发电 / Generating** | ⚡ | 为设备供电 / Power devices |
| **手工 / Handiwork** | 🛠️ | 制作物品 / Craft items |
| **采集 / Gathering** | 🌾 | 收集成熟作物 / Collect crops |
| **伐木 / Logging** | 🪓 | 砍伐木材 / Cut wood |
| **采矿 / Mining** | ⛏️ | 开采矿石 / Mine ores |
| **搬运 / Transporting** | 📦 | 运输物品到箱子 / Transport items to chests |
| **冷却 / Cooling** | ❄️ | 冷藏食物 / Refrigerate food |

### 帕鲁属性 / Pal Stats

- **HP（血量）**：战斗中的生命值 / Hit points in combat
- **攻击（Attack）**：影响造成的伤害 / Affects damage dealt
- **防御（Defense）**：减少受到的伤害 / Reduces damage taken
- **SAN（理智）**：工作时会下降，降到0会罢工并生病 / Decreases while working, at 0 they stop and get sick
- **饱腹度（Fullness）**：需要食物补充，耗尽会昏倒 / Needs food, when empty they faint

### 科技树 / Technology Tree

- 按 **Tab** 打开科技菜单 / Press **Tab** to open technology menu
- 升级获得**科技点（Technology Points）**，用于解锁新配方 / Level up to gain **Technology Points** for unlocking new recipes
- **古代科技点（Ancient Technology Points）**通过击败BOSS获得 / **Ancient Technology Points** are earned by defeating bosses
- 优先解锁：帕鲁球升级版、**滑翔伞（Glider）**、**抓钩枪（Grappling Gun）** / Priority unlocks: upgraded Pal Spheres, **Glider**, **Grappling Gun**

## 捕捉技巧 / Catching Tips

| 技巧 / Tip | 说明 / Description |
|------|----------|
| **背击加成 / Back Bonus** | 从背后投掷帕鲁球提高捕获率 / Throwing from behind increases catch rate |
| **等级差 / Level Gap** | 帕鲁球等级必须接近或高于目标等级 / Sphere tier must match or exceed target level |
| **异常状态 / Status Effects** | 睡眠、烧灼、麻痹状态下捕获率大幅提升 / Sleep, burn, and paralysis greatly boost catch rate |
| **生命值 / HP** | 目标血量越低，捕获率越高 / Lower HP = higher catch rate |
| **巨型帕鲁 / Tower Pals** | 关底BOSS无法被普通帕鲁球捕获 / End bosses cannot be caught with normal spheres |

## 前期BOSS / Early Bosses

### 佐伊与暴电熊（第一个塔主）/ Zoe & Grizzbolt (First Tower Boss)

- **推荐等级**：15级以上 / Recommended level: 15+
- **帕鲁推荐**：草系帕鲁（克制电系）/ Recommended Pals: Grass type (counters Electric)
- **核心机制**：暴电熊使用范围电击，保持移动 / Grizzbolt uses area lightning, keep moving
- **打法**：远程攻击为主，躲避地面电圈 / Ranged attacks, dodge ground lightning circles
- **奖励**：古代科技点 / Reward: Ancient Technology Points

### 叶胖达（野外BOSS）/ Penking (Field Boss)

- **位置**：初始高原西南方的湖泊 / Location: Lake southwest of Initial Plateau
- **推荐等级**：12级以上 / Recommended level: 12+
- **打法**：冰水属性，弱电系 / Ice/Water type, weak to Electric
- **奖励**：古代科技点 + 珍贵材料 / Reward: Ancient Technology Points + rare materials

## 避坑要点 / Tips & Pitfalls

- **不要在一级打塔主**——先去探索和升级，10级前挑战塔主必死 / **Don't fight tower bosses at level 1** — explore and level up first
- **帕鲁球要多做**——前期至少准备50个普通帕鲁球 / **Make plenty of Pal Spheres** — at least 50 standard spheres early on
- **基地选址很重要**——选择靠近矿石和木材的位置，后期方便自动采集 / **Base location matters** — choose near ore and wood nodes for automated gathering
- **注意帕鲁的SAN值**——太低会让帕鲁生病，工作效率大幅下降 / **Watch Pal SAN** — low SAN causes illness and massive work efficiency drop
- **不要用低等级帕鲁球抓高等级帕鲁**——概率极低，浪费资源 / **Don't use low-tier spheres on high-level Pals** — near-zero catch rate, waste of resources
- **尽早解锁滑翔伞和抓钩枪**——大幅提升探索效率 / **Unlock Glider and Grappling Gun early** — huge exploration efficiency boost
- **搭建屋顶会让帕鲁在雨天继续工作** / **Build a roof** — Pals will keep working in the rain
- **夜间会刷出更强的帕鲁**，白天相对安全 / **Night spawns stronger Pals** — daytime is safer
