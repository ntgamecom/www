## 制作优先级 / Crafting Priority

| 等级阶段 / Level Phase | 优先制作 / Priority Craft |
|:----:|:----|
| **1-10** | 石斧→石镐→木弓→布甲→皮甲→普通帕鲁球 / Stone tools → Wood Bow → Cloth → Leather → Basic Spheres |
| **10-20** | 金属斧镐→金属弩→金属甲→帕鲁背包→滑翔伞 / Metal tools → Crossbow → Metal Armor → Pack → Glider |
| **20-30** | 手枪→霰弹枪→耐热/寒甲→帕鲁鞍具→中级帕鲁球 / Handgun → Shotgun → Resist Armor → Saddles → Better Spheres |
| **30-40** | 突击步枪→精炼甲→帕鲁护甲→精炼工具→高级帕鲁球 / AR → Refined Armor → Pal Armor → Refined tools → High Spheres |
| **40-50** | 火箭筒→传说防具→各帕鲁专属终极装备→传说帕鲁球 / Rocket Launcher → Legendary Armor → Ultimate Pal Gear → Legendary Spheres |
