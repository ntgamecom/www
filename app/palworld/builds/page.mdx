# 基地建设与配种 / Base Building & Breeding

幻兽帕鲁的基地建设和配种是中后期核心玩法。一个高效的基地能自动产出大量资源，而配种可以定向培育出高属性帕鲁 / Base building and breeding are the core mid-to-late game mechanics in Palworld. An efficient base auto-produces resources, while breeding lets you target high-stat Pals.

---

## 基地选址 / Base Location Guide

| 位置 / Location | 优势 / Pros | 劣势 / Cons | 推荐用途 / Best For |
|:----:|:----|:----|:----|
| **初始高原 / Starting Plateau** | 平坦开阔、靠近初期资源 / Flat, open, near early resources | 后期资源匮乏 / Late-game resource poor | 1号基地——新手期 / First base — early game |
| **矿山区 / Mining Area** | 大量矿石+煤炭——自动采矿 / Many ore+coal nodes — auto-mining | 防御位置差 / Poor defensive position | 资源产出基地 / Resource base |
| **海滩平地 / Beach Flat** | 超大平坦面积——适合大型基地 / Huge flat area — perfect for large bases | 无特殊资源 / No special resources | 主基地/畜牧基地 / Main base/ranch |
| **沙漠绿洲 / Desert Oasis** | 硫磺+石英+____/Sulfur+Quartz | 炎热——需要耐热帕鲁 / Hot — need heat-resistant Pals | 中后期资源基地 / Mid-game resource base |
| **雪山温泉 / Snow Hot Spring** | 冰矿+大量纯水晶 / Ice ore + pure crystal | 寒冷——需要耐寒帕鲁 / Cold — need cold-resistant Pals | 后期高级材料 / Late-game materials |

**推荐2-3基地配置**: 1个主基地(生产+居住) + 1个矿场(自动采集) + 1个畜牧基地(配种+牧场) / **Recommended 2-3 base setup**: 1 main base (production+living) + 1 mining base (auto-gather) + 1 ranch (breeding+pets).

## 配种系统 / Breeding System

**配种是获得高品质帕鲁的最佳途径**——遗传父母双方属性+有概率变异出更优技能 / **Breeding is the best way to get high-quality Pals** — inherits parent stats + chance to mutate better skills.

| 配种要点 / Key Point | 说明 / Description |
|:----:|------|
| **繁殖场建造 / Build** | 解锁等级15→制作繁殖场→放置到基地 / Unlock at lv15 → build breeding farm → place in base |
| **放入帕鲁 / Place Pals** | 一公一母同种或指定组合→放入繁殖场→开始 / One male + one female (same species or specific combo) → place in farm → start |
| **蛋糕 / Cake** | 配种消耗蛋糕——放入繁殖场附件的饲料箱 / Breeding consumes cake — place in nearby feed box |
| **蛋孵化 / Egg Hatching** | 获得蛋→放入孵化器→等待孵化 / Get egg → place in incubator → wait for hatch |
| **温度控制 / Temperature** | 部分蛋需要特定温度(火焰蛋=热/冰霜蛋=冷) / Some eggs need specific temperature (fire=hot/ice=cold) |
| **属性继承 / Stat Inheritance** | 后代继承父母双方的基础属性区间——父母越好=后代越强 / Offspring inherits stat range from both parents — better parents = stronger offspring |

## 基地管理技巧 / Base Management Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **SAN值管理 / SAN Management** | 温泉>床铺>食物。SAN值掉的太快→检查温泉数量是否足够 / Hot spring > beds > food. SAN dropping fast? Check hot spring count |
| **负重管理 / Weight Management** | 设置仓库位置在工坊旁边——减少帕鲁搬运距离 / Place storage next to workshop — reduce transport distance |
| **帕鲁分工 / Pal Assignment** | 在终端里手动调整每个帕鲁的工作优先级——1=最高 / Manually adjust work priority in terminal — 1=highest |
| **不要超过等级上限 / Stay within base limit** | 基地等级越高→可放的帕鲁越多→但超过15只后SAN消耗加速 / Higher base level = more Pals = SAN drain accelerates beyond 15 |
| **大基地要分区 / Zone your base** | 将生产区/居住区/畜牧区分开——避免帕鲁跑太远浪费时间 / Separate production/living/ranch zones — reduce Pal travel time |
