## 初始高原 / Initial Plateau

**推荐等级 / Level**: 1-10

游戏开始的地方。温和的气候，基础资源丰富，适合建立第一个基地 / The starting area. Mild climate, abundant basic resources, ideal for a first base.

**资源 / Resources**:
- 🪨 石头（Stone）、燧石（Flint）
- 🪵 木材（Wood）
- 🫐 浆果（Berries）
- ⛏️ 少量金属矿石（少量 Ore）

**常见帕鲁 / Common Pals**:
企丸丸（Pengullet）、毛掸儿（Cattiva）、捣蛋猫（Vixy）、皮皮鸡（Chikipi）、火绒狐（Foxparks）、波霸牛（Mozzarina）

**要点 / Tips**:
- 🏠 这里是建立**主基地**的最佳地点 / Best location for your **main base**
- 🚫 没有强力敌人，适合熟悉游戏机制 / No strong enemies, perfect for learning mechanics
- 🌙 夜间出现的**寐魔（Daedream）**是不错的早期战斗帕鲁 / **Daedream** (night spawn) is a solid early combat Pal

---

## 风蚀丘陵 / Windswept Hills

**推荐等级 / Level**: 5-15

离开初始高原后的第一个主区域，地形起伏，有大量矿石资源 / The first major area after the plateau, rolling terrain with abundant ore.

**资源 / Resources**:
- ⛏️ 金属矿石（Ore）—— 大量，建矿点首选 / Abundant, best for ore base
- 硫磺（Sulfur）—— 制作火药和子弹用 / Used for gunpowder and ammo
- 🪵 木材（Wood）、石头（Stone）

**常见帕鲁 / Common Pals**:
疾风隼（Galeclaw）、雷角马（Univolt）、紫霞鹿（Eikthyrdeer）、连理龙（Dinossom）、羽箭射手（Robinquill）

**塔主BOSS / Tower Boss**: **佐伊&暴电熊（Zoe & Grizzbolt）** Lv.15
- 弱点：草系 / Weakness: Grass type
- 推荐准备：Lv.15+，带草系帕鲁 / Recommended: Lv.15+, bring Grass Pals

**要点 / Tips**:
- ⛏️ 这里的**金属矿石**是建立自动采集基地的最佳选择 / **Ore nodes** here are perfect for an automated mining base
- 🦅 捕捉**疾风隼（Galeclaw）**——它的滑翔能力非常实用 / Catch **Galeclaw** — its gliding ability is extremely useful
- 🐴 **雷角马（Univolt）**是优秀的早期坐骑 / **Univolt** is an excellent early mount

---

## 沼泽岛 / Marsh Island

**推荐等级 / Level**: 10-20

湿润的沼泽地带，水属性帕鲁丰富，有不少优质金属矿 / Humid marshlands, rich in Water-type Pals, has refined ore nodes.

**资源 / Resources**:
- 优质金属矿石（Refined Ore）
- 皮革（Leather）—— 从鳄类帕鲁掉落 / Dropped by croc-type Pals
- 蘑菇（Mushroom）

**常见帕鲁 / Common Pals**:
滑水蛇（Fuack）、鲤鱼王（Gobfin）、鳄霸王（Broncherry）、浴火雀（Suzaku）

**要点 / Tips**:
- 💧 带草系帕鲁克制水属性敌人 / Bring Grass Pals to counter Water types
- 🐊 鳄霸王（Broncherry）是非常优秀的搬运帕鲁 / Broncherry is an excellent Transporting Pal
- 🗺️ 岛中央有一个**地下城入口** / There's a **dungeon entrance** in the center of the island

---

## 竹林 / Bamboo Groves

**推荐等级 / Level**: 15-25

茂密的竹林区域，草系和龙系帕鲁聚集地 / Dense bamboo forest, home to Grass-type and Dragon-type Pals.

**资源 / Resources**:
- 🪵 大量木材（Abundant Wood）
- 🌿 纤维（Fiber）
- 🎋 竹笋（Bamboo Shoot）—— 烹饪材料 / Cooking ingredient

**常见帕鲁 / Common Pals**:
花冠龙（Petallia）、叶胖达（Penking）（BOSS）、精灵龙（Elizabee）、魔渊龙（Azurobe）

**塔主BOSS / Tower Boss**: **莉莉&百合女王（Lily & Lyleen）** Lv.25
- 弱点：火系 / Weakness: Fire type
- 推荐准备：Lv.25+，带火系帕鲁 / Recommended: Lv.25+, bring Fire Pals

**要点 / Tips**:
- 🔥 火系帕鲁在这里非常有用 / Fire Pals are very useful here
- 🐉 **魔渊龙（Azurobe）**是强大的龙系战斗帕鲁 / **Azurobe** is a powerful Dragon-type combat Pal
- 🏆 击败塔主后可获得**百合女王（Lyleen）**的完美工作帕鲁 / Defeating the tower unlocks Lyleen — an excellent worker Pal

---

## 干涸沙漠 / Desiccated Desert

**推荐等级 / Level**: 25-35

炎热的沙漠区域，地面系和火系帕鲁为主 / Hot desert zone, mostly Ground-type and Fire-type Pals.

**资源 / Resources**:
- 🪨 煤矿石（Coal）—— 制作精炼金属锭 / Used for refined metal ingots
- 石英（Quartz）—— 制作电路板和玻璃 / Used for circuit boards and glass
- 🔥 硫磺（Sulfur）

**常见帕鲁 / Common Pals**:
霹雳犬（Boltmane）、熔岩兽（Arsox）、沙漠之蝶（Warsect）、地脉猪（Fuddler）

**塔主BOSS / Tower Boss**: **艾克赛尔&暴灵熊（Axel & Orserk）** Lv.35
- 弱点：地系 / Weakness: Ground type
- 推荐准备：Lv.35+，带地系帕鲁 / Recommended: Lv.35+, bring Ground Pals

**要点 / Tips**:
- ☀️ 沙漠白天极端高温，需要耐热装备 / Extreme daytime heat — need heat-resistant gear
- 🌙 夜间会刷新更强的帕鲁 / Stronger Pals spawn at night
- 🏜️ 沙漠深处有**古代科技祭坛** / Ancient Technology Altar in the deep desert

---

## 星界山脉 / Astral Mountains

**推荐等级 / Level**: 30-40

高海拔山脉，昼夜温差大，龙系和暗系帕鲁常见 / High-altitude mountains with extreme temperature swings. Dragon-type and Dark-type Pals.

**资源 / Resources**:
- 💎 纯石英（Pure Quartz）
- 💠 钻石（Diamond）—— 高级装备材料 / High-tier equipment material

**常见帕鲁 / Common Pals**:
空涡龙（Jetragon）（传说级）、冥铠蝎（Menasting）、雷鳞龙（Relaxisurus）

**塔主BOSS / Tower Boss**: **马库斯&空涡龙（Marcus & Faleris）** Lv.40
- 弱点：暗系 / Weakness: Dark type
- 推荐准备：Lv.40+，带暗系帕鲁 / Recommended: Lv.40+, bring Dark Pals

**要点 / Tips**:
- 🧊 山顶极其寒冷，必备**防寒装备** / Extremely cold at peak — **cold-resistant gear** required
- 🐉 **空涡龙（Jetragon）**是传说级帕鲁，击败塔主后可在野外遇到 / **Jetragon** is a legendary Pal, found in the wild after defeating the tower
- ⛰️ 这里有多条**探索路线**，建议用飞行坐骑探索 / Multiple **exploration routes** — use a flying mount

---

## 冰风岛 / Ice Wind Island

**推荐等级 / Level**: 40-45

极寒冰封岛屿，冰系帕鲁栖息地 / Frozen island, Ice-type Pal habitat.

**资源 / Resources**:
- ❄️ 冰矿（Ice Ore）
- 🔵 蓝宝石（Sapphire）
- 纯石英（Pure Quartz）

**常见帕鲁 / Common Pals**:
寒冰兽（Frostallion）（传说级）、冰霜犬（Foxcicle）、白绒雪怪（Wumpo）、冰刺鼠（Jolthog Cryst）

**塔主BOSS / Tower Boss**: **维克多&暗黑犬（Victor & Shadowbeak）** Lv.45
- 弱点：龙系 / Weakness: Dragon type
- 推荐准备：Lv.45+，带龙系帕鲁 / Recommended: Lv.45+, bring Dragon Pals

**要点 / Tips**:
- 🥶 需要**高级防寒装备**才能在此生存 / Need **advanced cold-resistant gear** to survive
- 🐎 **寒冰兽（Frostallion）**是最强的传说级帕鲁之一，击败塔主后在野外可遇 / **Frostallion** is one of the strongest legendary Pals
- 🧊 冰面很滑，建议用飞行坐骑 / Icy surfaces — use a flying mount

---

## 火山岛 / Volcanic Island

**推荐等级 / Level**: 45-50

The most dangerous area. Extreme heat, high-level Pals, and the final tower boss.

极端危险区域，高温、高等级帕鲁和最终塔主 / The most dangerous area. Extreme heat, high-level Pals, the final tower boss.

**资源 / Resources**:
- ⛏️ 精炼金属矿石（Refined Metal Ore）
- 🔴 红宝石（Ruby）
- 🔥 硫磺（大量）/ Sulfur (abundant)
- 🌋 古代文明零件（Ancient Civilization Parts）

**常见帕鲁 / Common Pals**:
炽焰龙（Blazamut）、炎魔狮（Pyrin Noct）、熔岩龟（Reptiro）、混沌骑士（Astegon）

**塔主BOSS / Tower Boss**: **比约恩&霜牙王（Bjorn & Bastigor）** Lv.50
- 弱点：龙系 / Weakness: Dragon type
- 推荐准备：Lv.50，顶配装备，大量药品 / Lv.50, max gear, plenty of potions

**要点 / Tips**:
- 🔥 需要**高级耐热装备** / Need **advanced heat-resistant gear**
- 🦎 **混沌骑士（Astegon）**是顶级采矿帕鲁 / **Astegon** is a top-tier Mining Pal
- 👑 最终塔主战后解锁**通关后内容** / Clearing the final tower unlocks **post-game content**

---

## 樱花岛 / Sakurajima

**推荐等级 / Level**: 30-50

大型更新新增的岛屿，地形复杂，有独特的古代遗迹 / Added in a major update. Complex terrain with unique ancient ruins.

**资源 / Resources**:
- 🪙 钴矿（Cobalt Ore）
- 🏛️ 古代文明零件（Ancient Civilization Parts）
- 稀有果实（Rare Fruits）

**常见帕鲁 / Common Pals**:
比克鸟（Beegarde）、海灵龙（Selyne）、光棱鹿（Celesdir）

**要点 / Tips**:
- 🗺️ 岛上有多处**古代遗迹**，值得仔细探索 / Multiple **ancient ruins** worth thorough exploration
- ⚔️ 这里的帕鲁等级跨度大（30-50），小心高等级区域 / Pal levels range from 30-50, watch out for high-level zones
- 🏆 有独立的野外BOSS "**光棱鹿（Celesdir）**"非常强大 / Independent field boss **Celesdir** is extremely powerful
