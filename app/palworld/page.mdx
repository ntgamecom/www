## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|---|---|
| **帕鲁系统 / Pal System** | 游戏中100+种帕鲁——每种有独特技能、工作能力和伙伴技能。帕鲁可以战斗、工作、骑乘、配种 / 100+ Pal species — each with unique skills, work abilities, and partner skills. Pals can fight, work, mount, and breed |
| **基地建设 / Base Building** | 自由建造基地——布置工作设施让帕鲁自动生产、加工、制作。从篝火到自动化生产线 / Free-form base building — assign Pals to workstations for automated production. From campfires to assembly lines |

| **帕鲁配种 / Pal Breeding** | 配种系统深度极高——通过配种遗传被动技能、提升IV值、产出稀有帕鲁 / Deep breeding system — inherit passive skills, improve IVs, produce rare Pals through breeding |
| **多人联机 / Multiplayer** | 最多32人同服务器——可以合作建基地、交易帕鲁、PVP竞技 / Up to 32 players per server — co-op base building, Pal trading, PVP arena |

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **先抓基础帕鲁 / Catch Basic Pals First** | 开局不要急着打怪——先抓棉悠悠、捣蛋猫等基础帕鲁来采集和建设工作 / Don't rush combat — catch Lamball, Cattiva etc. for gathering and building work |
| **2** | **造帕鲁球工坊 / Build Pal Sphere Workshop** | 有了帕鲁球才能抓帕鲁——尽快解锁帕鲁球生产线 / Need spheres to catch Pals — unlock sphere production ASAP |
| **3** | **据点选址 / Pick a Good Base Location** | 找靠近资源(金属矿/石头/木材)+水源+平坦地形的区域——前期效率翻倍 / Near resources (ore/stone/wood) + water + flat terrain — doubles early efficiency |
| **4** | **等级优先 / Prioritize Leveling** | 等级解锁科技树——采集、制作、战斗都加经验。优先做任务/探索升级 / Leveling unlocks tech tree — gather, craft, fight for XP. Prioritize quests/exploration |
| **5** | **注意帕鲁体力 / Watch Pal SAN** | 帕鲁工作会消耗体力(SAN)——体力低时工作速度变慢。造温泉和床给它们恢复 / Pals consume SAN while working — low SAN = slow work. Build hot springs and beds for recovery |
| **6** | **早开传送点 / Unlock Fast Travel Early** | 地图上的传送雕像务必激活——后期来回跑的时间成本极高 / Activate fast travel statues on the map — travel time cost is huge late game |
| **7** | **盾牌很重要 / Shields Matter** | 游戏前中期盾牌可以帮你挡大量伤害——比堆血量高效 / Early-mid game shields block lots of damage — more efficient than HP stacking |

## 基础操作 / Basic Controls

| 操作 / Action | 键位 / Key (PC) | 说明 / Note |
|---|---|---|
| **冲刺 / Sprint** | Shift / Left Shift | 消耗体力 / Costs stamina |
| **翻滚 / Dodge Roll** | 空格 / Space | 无敌帧——可以躲攻击 / I-frames — can dodge attacks |
| **跳跃 / Jump** | 空格(连按)/ Space (press hold) | 二段跳需要帕鲁技能 / Double jump needs Pal skill |
| **骑乘 / Mount** | F | 骑帕鲁——不同帕鲁速度/能力不同 / Ride Pals — different speeds/abilities |
| **帕鲁球投掷 / Sphere Throw** | 鼠标左键 / Left Click | 瞄准+投掷——打到帕鲁背后更高捕获率 / Aim + throw — back hit = higher catch rate |
| **建筑 / Build** | B | 打开建筑菜单 / Open build menu |
| **地图 / Map** | M | 传送点必须解锁后才可使用 / Need to discover fast travel points |
| **背包 / Inventory** | Tab | 查看物品/装备 / View items/equipment |
| **帕鲁管理 / Pal Management** | P | 查看所有已捕获的帕鲁 / View all caught Pals |
| **科技 / Technology** | T | 科技点可解锁新配方 / Spend tech points to unlock recipes |
| **古代科技 / Ancient Tech** | Ctrl+T | 用古代科技点解锁高级物品 / Spend ancient tech points for endgame items |

## 新手10条黄金法则 / 10 Golden Rules

| # | 法则 / Rule | 说明 / Explanation |
|---|---|---|
| 1 | **先抓低级帕鲁** | 不要急着抓高等级——先抓10只低等级解锁科技点升级 / Don't rush high levels — catch 10 low-level Pals first for tech XP |
| 2 | **造篝火+床** | 第一天必须造——否则晚上冻死+帕鲁不干活 / Build campfire + bed on Day 1 — or freeze at night & Pals won't work |
| 3 | **留3只帕鲁在基地** | 出门探险前确保基地有3只帕鲁工作——否则基地效率为0 / Leave 3 Pals at base before exploring — or base efficiency drops to 0 |
| 4 | **升级传送点** | 遇到新传送点第一时间激活——后面传送省时间 / Activate every fast travel point immediately — saves time later |
| 5 | **做够帕鲁球** | 永远不要只带10个球出门——最少带50个基础球+20个超级球 / Never leave base with only 10 spheres — minimum 50 basic + 20 mega |
| 6 | **抓一只飞行帕鲁** | 尽早抓一只飞行的(如疾风隼/天羽龙)——跑图效率翻倍 / Catch a flying Pal early — doubles map travel speed |
| 7 | **升级工作台** | 每解锁新科技就去升级工作台——否则不能制作高级物品 / Upgrade workbench each time you unlock new tech |
| 8 | **学会帕鲁球制作** | 前期缺球是因为没点"帕鲁球制作"科技——点了之后自己搓 / Unlock Pal Sphere crafting tech — make your own |
| 9 | **抓迅雷鸟** | 前期最好的交通帕鲁——速度快+可战斗 / Beakon is the best early transport — fast + combat capable |
| 10 | **别单挑塔主Boss** | 等级差超过10级去打塔主Boss=送死——先练级 / Fighting tower bosses 10+ levels above you = suicide — level up first |
