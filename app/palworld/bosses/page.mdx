# 全BOSS攻略 / All Bosses Guide

幻兽帕鲁的BOSS分为**塔主BOSS**（主线推进）和**野外BOSS/Alpha帕鲁**（探索挑战）。每个BOSS都有独特的机制和弱点 / Palworld bosses are divided into **Tower Bosses** (main story progression) and **Field Bosses / Alpha Pals** (exploration challenges). Each boss has unique mechanics and weaknesses.

---

## 塔主BOSS / Tower Bosses

塔主是每个主要区域的最终挑战，击败后获得**古代科技点**和**奖杯** / Tower bosses are the ultimate challenge of each major region, rewarding **Ancient Technology Points** and **Trophies**.

---

### 1. 佐伊 & 暴电熊 / Zoe & Grizzbolt

- **位置 / Location**: 风蚀丘陵·雷恩盗猎者高塔 / Windswept Hills · Rayne Syndicate Tower (坐标 / Coords: 112, -368)
- **推荐等级 / Recommended Level**: 15+
- **弱**点 / **Weakness**: 草系 / Grass type（克制电系 / Counters Electric）

**技能机制 / Mechanics**:
- 暴电熊使用**范围电击**和**冲锋攻击** / Grizzbolt uses **area lightning** and **charge attacks**
- 地面会出现**蓝色电圈**，延迟后造成伤害 / Blue lightning circles appear on the ground with delayed damage
- 佐伊会从一旁射击干扰 / Zoe shoots from the side to harass

**打法建议 / Strategy**:
| 阶段 / Phase | 行动 / Action |
|------|----------|
| **开场 / Start** | 放出草系帕鲁（推荐**花冠龙 Petallia**或**连理龙 Dinossom**）/ Deploy Grass Pal (**Petallia** or **Dinossom** recommended) |
| **电圈 / Circles** | 不断移动，电圈出现后1秒闪避 / Keep moving, dodge 1 sec after circles appear |
| **冲锋 / Charge** | 向侧方闪避，拉开距离 / Dodge sideways, create distance |
| **输出 / Damage** | 电圈间隙使用草系技能攻击 / Attack with Grass skills between lightning phases |

**奖励 / Rewards**: 古代科技点×1, 暴电熊雕像 / Ancient Tech Points ×1, Grizzbolt Statue

### 2. 莉莉 & 百合女王 / Lily & Lyleen

- **位置 / Location**: 竹林·自由战士高塔 / Bamboo Groves · Free Pal Alliance Tower (坐标 / Coords: 180, 35)
- **推荐等级 / Recommended Level**: 25+
- **弱点 / Weakness**: 火系 / Fire type（克��草系 / Counters Grass）

**技能机制 / Mechanics**:
- 百合女王使用**藤蔓缠绕**和**范围花粉**攻击 / Lyleen uses **vine wrap** and **area pollen** attacks
- 莉莉会**治疗**百合女王 / Lily **heals** Lyleen
- 藤蔓缠绕会将你固定在原地 / Vine wrap immobilizes you

**打法建议 / Strategy**:
| 阶段 / Phase | 行动 / Action |
|------|----------|
| **优先目标 / Priority** | **优先击败莉莉本人**，她死后百合女王防御大幅下降 / **Focus Lily first** — Lyleen's defense drops significantly after Lily falls |
| **藤蔓 / Vine** | 被缠住时切换帕鲁解控 / Switch Pals to break free when wrapped |
| **花粉 / Pollen** | 看到花粉扩散时向外跑 / Run outward when pollen spreads |
| **输出 / Damage** | 火系技能高效输出 / Fire skills deal massive damage |

**奖励 / Rewards**: 古代科技点×2, 百合女王雕像 / Ancient Tech Points ×2, Lyleen Statue

### 3. 艾克赛尔 & 暴灵熊 / Axel & Orserk

- **位置 / Location**: 干涸沙漠·PIDF高塔 / Desiccated Desert · PIDF Tower (坐标 / Coords: -62, -430)
- **推荐等级 / Recommended Level**: 35+
- **弱点 / Weakness**: 地系 / Ground type（克制电系 / Counters Electric）

**技能机制 / Mechanics**:
- 暴灵熊使用**高电压远程攻击**和**范围落雷** / Orserk uses **high-voltage ranged attacks** and **area lightning strikes**
- 场地地面带电，站久了会麻痹 / The floor becomes electrified, standing still causes paralysis
- **艾克赛尔**使用榴弹枪远程攻击 / **Axel** uses a grenade launcher for ranged attacks

**打法建议 / Strategy**:
| 阶段 / Phase | 行动 / Action |
|------|----------|
| **开场 / Start** | 接地系帕鲁，推荐**沙漠之蝶 Warsect** / Deploy Ground Pal (**Warsect** recommended) |
| **落雷 / Stikes** | 不要停在一个位置超过3秒 / Don't stay in one spot for more than 3 seconds |
| **榴弹 / Grenades** | 看到抛物线轨迹后横向移动 / Move laterally when you see the grenade arc |
| **输出 / Damage** | 地系技能破防，近战输出最佳 / Ground skills break defense, best to fight close |

**奖励 / Rewards**: 古代科技点×2, 暴灵熊雕像 / Ancient Tech Points ×2, Orserk Statue

### 4. 马库斯 & 空涡龙 / Marcus & Faleris

- **位置 / Location**: 星界山脉·天爪山高塔 / Astral Mountains · Pal Genetic Tower (坐标 / Coords: 441, 514)
- **推荐等级 / Recommended Level**: 40+
- **弱点 / Weakness**: 暗系 / Dark type

**技能机制 / Mechanics**:
- 空涡龙是**飞行系**，持续在空中攻击 / Faleris is a **Flying type**, attacks continuously from the air
- **火焰龙卷**和**俯冲攻击**交替使用 / Alternates between **fire tornadoes** and **dive attacks**
- 马库斯会从龙背上射击 / Marcus shoots from the dragon's back

**打法建议 / Strategy**:
| 阶段 / Phase | 行动 / Action |
|------|----------|
| **开场 / Start** | 带暗系帕鲁（推荐**冥铠蝎 Menasting**或**噬魂兽 Helzephyr**）/ Bring Dark Pals (**Menasting** or **Helzephyr** recommended) |
| **高空 / Air** | 用自己的飞行帕鲁升空作战 / Use your flying Pal to engage in the air |
| **火焰龙卷 / Tornado** | 龙卷轨迹固定，提前预判横向移动 / Tornadoes follow a fixed path, strafe preemptively |
| **俯冲 / Dive** | 侧闪后反击，他会有短暂硬直 / Side dodge then counter — brief recovery window |

**奖励 / Rewards**: 古代科技点×3, 空涡龙雕像 / Ancient Tech Points ×3, Faleris Statue

### 5. 维克多 & 暗黑犬 / Victor & Shadowbeak

- **位置 / Location**: 冰风岛·暗黑高塔 / Ice Wind Island · Dark Tower (坐标 / Coords: -102, 659)
- **推荐等级 / Recommended Level**: 45+
- **弱点 / Weakness**: 龙系 / Dragon type

**技能机制 / Mechanics**:
- 暗黑犬使用**暗影球**和**范围暗爆** / Shadowbeak uses **shadow orbs** and **area dark explosions**
- **暗影领域**会降低视野 / **Shadow field** reduces visibility
- 维克多会使用**吸血技能**为自己回血 / Victor uses **life-steal skills**

**打法建议 / Strategy**:
| 阶段 / Phase | 行动 / Action |
|------|----------|
| **开场 / Start** | 带龙系帕鲁（推荐**魔渊龙 Azurobe**或**精灵龙 Elizabee**）/ Bring Dragon Pals (**Azurobe** or **Elizabee** recommended) |
| **暗影球 / Orbs** | 连续闪避，暗影球有追踪特性 / Continuous dodging — orbs have tracking |
| **暗影领域 / Shadow** | 视野受限时保持移动，注意听声音提示 / Keep moving when visibility drops, listen for audio cues |
| **吸血 / Life-steal** | 看到他施法吸血时立刻打断 / Interrupt immediately when he starts life-steal casting |

**奖励 / Rewards**: 古代科技点×3, 暗黑犬雕像 / Ancient Tech Points ×3, Shadowbeak Statue

### 6. 比约恩 & 霜牙王 / Bjorn & Bastigor

- **位置 / Location**: 火山岛·终极高塔 / Volcanic Island · Ultimate Tower (坐标 / Coords: -657, -134)
- **推荐等级 / Recommended Level**: 50（满级）/ Max level
- **弱点 / Weakness**: 龙系 / Dragon type

**技能机制 / Mechanics**:
- 霜牙王是**冰系终极BOSS**，全屏冰霜攻击 / Bastigor is the **ultimate Ice boss**, full-screen frost attacks
- **暴风雪**持续造成冰冻伤害 / **Blizzard** deals continuous freezing damage
- **冰锥坠落**需要精确闪避 / **Ice spike fall** requires precise dodging
- 比约恩使用**火箭筒**远程攻击 / Bjorn uses a **bazooka** for ranged attacks

**打法建议 / Strategy**:
| 阶段 / Phase | 行动 / Action |
|------|----------|
| **准备 / Prep** | **满级满装备**，带足药品和复活道具 / **Max level + max gear**, plenty of potions and revives |
| **帕鲁推荐 / Pals** | **空涡龙 Jetragon**或**混沌骑士 Astegon**（龙系克制）/ **Jetragon** or **Astegon** (Dragon type counters Ice) |
| **暴风雪 / Blizzard** | 看到暴风雪前兆时寻找**掩体** / Find **cover** when you see the blizzard warning |
| **冰锥 / Ice Spikes** | 地面出现蓝圈后1.5秒闪避 / Dodge 1.5 seconds after blue circles appear |
| **火箭筒 / Bazooka** | 听到发射音效立刻侧移 / Move laterally on hearing the launch sound |
| **输出 / Damage** | 暴风雪间隙是唯一输出窗口，全力攻击 / Blizzard gaps are the ONLY damage windows — go all out |

**奖励 / Rewards**: 古代科技点×5, 霜牙王雕像, 通关奖杯 / Ancient Tech Points ×5, Bastigor Statue, Clear Trophy

## BOSS战通用技巧 / General Boss Tips

| 技巧 / Tip | 说明 / Description |
|------|----------|
| **等级压制 / Level Advantage** | 高于BOSS等级5级会让战斗轻松很多 / Being 5+ levels above the boss makes it much easier |
| **属性克制 / Type Advantage** | **草克电 → 电克水 → 水克火 → 火克草 → 地克电 → 龙克暗 → 暗克龙** |
| **药品准备 / Potions** | 每种BOSS战至少带10个中级恢复药 / Bring at least 10 mid-tier potions per boss fight |
| **帕鲁搭配 / Pal Loadout** | 带5只帕鲁：1克制系+1坦克+1治疗+2备用 / 5 Pals: 1 counter + 1 tank + 1 healer + 2 backup |
| **远程武器 / Ranged Weapons** | 弓或枪可以在安全距离输出 / Bow or gun allows safe ranged damage |
| **闪避优先 / Dodge First** | 不熟悉BOSS时先学会闪避所有技能再输出 / Learn all dodge timings before focusing on damage |
| **复活道具 / Revives** | 至少带3个帕鲁复活道具 / Bring at least 3 Pal revive items |
