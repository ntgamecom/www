# 技能系统 / Skills System

幻兽帕鲁的技能系统分为两大类：帕鲁的技能体系和玩家的科技技能体系。两者独立发展——帕鲁通过战斗/配种获得技能，玩家通过升级解锁科技 / Palworld's skill system has two categories: Pal skills and player tech skills. They develop independently — Pals gain skills via combat/breeding, players unlock tech via leveling.

---

## 帕鲁技能 / Pal Skills

### 技能类型 / Skill Types

| 技能类型 / Type | 数量 / Count | 说明 / Description |
|---|---|---|
| **主动技能 / Active** | 每个帕鲁3个 / 3 per Pal | 战斗中自动释放——可手动选择释放时机 / Auto-cast in battle — can also manual trigger |
| **被动技能 / Passive** | 每个帕鲁最多4个 / Max 4 per Pal | 永久生效的固有特质——通过配种遗传 / Permanent traits — inherited through breeding |
| **伙伴技能 / Partner** | 每个帕鲁1个 / 1 per Pal | 唯一技能——决定帕鲁的特殊功能 / Unique skill — defines Pal's special function |

### 主动技能获取 / Active Skill Acquisition

| 获取方式 / Method | 概率/条件 / Probability/Condition |
|---|---|
| **升级自动学习** | 帕鲁在每个特定等级自动学习新技能——每个帕鲁的技能池是固定的 / Pals auto-learn skills at specific levels — each Pal has a fixed skill pool |
| **技能果实** | 在地图上找到技能果实(发光的树)——可以直接让帕鲁学会特定技能 / Find Skill Fruits (glowing trees) on the map — teach any Pal a specific skill |
| **技能重置** | 使用"技能重置药"可以重置帕鲁的技能——重新选择 / Use "Skill Reset Potion" to reset Pal's skills — choose again |
| **配种继承** | 父母双方的技能都有可能遗传给子代——配种时大概率继承被动技能 / Both parents' skills may pass to offspring — passives are highly inheritable |

## 配种技能继承 / Breeding Skill Inheritance

配种是获取特定技能组合的核心方法 / Breeding is the core way to get specific skill combinations.

| 继承规则 / Inheritance Rule | 说明 / Description |
|---|---|
| **被动技能继承** | 父母共有4个被动技能槽。子代有概率随机继承父母双方的被动技能——最高继承3-4个 / Parents have 4 passive slots each. Children randomly inherit from both parents — max 3-4 skills |
| **技能突变** | 配种有小概率产生新技能(不在父母技能池中)——通常是低概率稀有技能 / Small chance of new skills (not in parent pool) — usually low-probability rare skills |
| **配种配方** | 特定父母组合固定产出特定帕鲁——利用配方规划你的完美帕鲁 / Specific parent pairs produce specific offspring — use recipes to plan your perfect Pal |
| **IV值(个体值)** | 每个帕鲁有隐藏的个体值——影响属性。通过配种可以逐步提升IV值 / Each Pal has hidden IVs — affects stats. Can gradually improve IVs through breeding |

## 被动技能遗传 / Passive Skill Inheritance

| 规则 / Rule | 说明 / Explanation |
|---|---|
| **父母技能** | 子代有概率继承父母的被动技能——包括金技能 / Offspring has chance to inherit both parents' passives — including gold skills |
| **技能数量上限** | 每只帕鲁最多4个被动技能——多余会替换 / Each Pal max 4 passives — extras replace |
| **变异概率** | 配种时有低概率产生新技能(变异)/ Low chance of new skill mutation during breeding |
| **金技能** | 金色名字的被动技能是最强的——如"传说级攻击" / Gold-name passives are strongest — e.g. "Legendary Attack" |
| **技能等级** | 同被动技能有等级——如"攻击Lv1","攻击Lv2","攻击Lv3"/ Same passive has tiers — "ATK Lv1", "ATK Lv2", "ATK Lv3" |

**最佳被动技能列表**: 传说级攻击(+30%ATK) > 传说级防御(+30%DEF) > 传说级生命(+30%HP) > 快速工作(+50%工作速度) > 幸运(+15%捕获率) / **Best passives**: Legendary ATK > Legendary DEF > Legendary HP > Swift Worker > Lucky.
