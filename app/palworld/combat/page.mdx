# 战斗系统 / Combat System

幻兽帕鲁的战斗系统结合了帕鲁(Pal)与玩家武器两种战斗方式——帕鲁负责技能输出和坦度，玩家负责走位、控制、补刀 / Palworld's combat system combines Pal abilities with player weaponry — Pals handle skill damage and tanking, while the player handles positioning, crowd control, and finishing blows.

---

## 帕鲁战斗 / Pal Combat

### 帕鲁技能类型 / Pal Skill Types

| 技能类型 / Skill Type | 描述 / Description | 战术 / Tactics |
|---|---|---|
| **主动技能 / Active Skills** | 帕鲁在战斗中自动释放——每个帕鲁最多3个主动技能 / Auto-cast in battle — max 3 active skills per Pal | 搭配合适的属性——针对弱点打克制 / Match elements to exploit weaknesses |
| **被动技能 / Passive Skills** | 帕鲁的固有天赋——如"攻击手冠军"(+20%攻击)/ Inherent traits — e.g. "Ferocious" (+20% attack) | 配种时优先选择好被动——后天无法改变 / Prioritize during breeding — unchangeable after capture |
| **伙伴技能 / Partner Skills** | 每个帕鲁独有的特殊技能——如"焰煌"的坐骑技能 / Unique special skill per Pal — e.g. "Blazehowl" mount skill | 战斗中可切换帕鲁释放伙伴技能——冷却时间较长 / Switch Pals mid-battle for partner skill — long cooldown |

### 帕鲁战斗管理 / Pal Battle Management

| 策略 / Strategy | 说明 / Description |
|---|---|
| **切换帕鲁** | 战斗中按帕鲁快捷键切换——每个帕鲁都有独立CD。利用切换来规避伤害/让不同属性帕鲁输出 / Switch Pals during combat — each has independent cooldowns. Use switching to dodge hits and cycle elements |
| **收放帕鲁** | 将帕鲁收回再释放可以重置其位置——可以用来躲避大范围AOE / Recall + resummon resets Pal position — useful for dodging large AOEs |
| **帕鲁数量** | 战斗中最多同时派出1只帕鲁——但背包中可以带5只。战斗中可以随时切换 / Only 1 Pal out at a time — but carry up to 5 in party. Switch freely mid-combat |
| **帕鲁体力** | 帕鲁在战斗中消耗体力(SAN)——体力耗尽后战斗力下降。用温泉/药品回复 / Pals consume SAN in battle — combat effectiveness drops when exhausted. Use hot springs/meds to recover |

## 高级战斗技巧 / Advanced Combat

| 技巧 / Technique | 说明 / Description |
|---|---|
| **骑乘战斗** | 部分帕鲁可以骑乘——骑乘时使用帕鲁的伙伴技能(如火龙的火焰喷射)。骑乘战斗机动性强 / Mountable Pals — use partner skill while mounted (e.g. Fire Dragon's flamethrower). High mobility |
| **空中战斗** | 飞行帕鲁可以骑乘飞行——从空中攻击地面敌人。对付飞行BOSS也有效 / Flying mounts enable aerial combat — attack ground enemies from above. Also effective vs flying bosses |
| **抓取战斗** | 前期可以丢"帕鲁球"砸敌人——有极低概率直接捕捉。PVP中也可以用作控制 / Early game: throw Pal Spheres at enemies — very low catch rate. Also usable as CC in PVP |
| **环境利用** | 利用地形躲避BOSS技能——把BOSS引到开阔地带打。避免在狭窄通道被围殴 / Use terrain to dodge boss skills — lure bosses to open areas. Avoid getting pinned in corridors |
| **状态效果** | 部分武器和帕鲁技能带有状态效果(燃烧、冰冻、麻痹等)。叠加状态可以造成额外伤害 / Some weapons and Pal skills have status effects (burn, freeze, shock). Stacking statuses deals bonus damage |

## PVP战斗 / PVP Combat

PVP目前主要通过竞技场进行——玩家1v1或组队对战 / PVP is currently done through the Arena — 1v1 or team battles.

| PVP要素 / PVP Element | 说明 / Description |
|---|---|
| **装备要求** | 传说级武器+传说级防具是竞技场标配 / Legendary weapons + armor are the Arena standard |
| **帕鲁选择** | 高攻高速帕鲁(如"神龙"、"焰煌")在PVP中表现突出——坦克型帕鲁(如"三头蛇")拖时间给别人输出 / High speed/attack Pals (Jetragon, Blazehowl) excel — tanks (Hydra) stall for team DPS |
| **PVP技巧** | 频繁切换帕鲁打乱对手节奏——用远程武器消耗——近战时放帕鲁挡住 / Frequently switch Pals to disrupt opponent rhythm — poke with ranged — recall Pal for melee block |
| **属性克制** | PVP中属性克制效果显著——针对对手帕鲁配置调整自己的队伍 / Element advantage is significant in PVP — adjust your party based on opponent's Pal composition |

## 战斗状态效果 / Combat Status Effects

| 效果 / Effect | 触发方式 / Trigger | 效果 / Result | 应对 / Counter |
|---|---|---|---|
| **燃烧(Burn)** | 火属性攻击 / Fire attack | 持续火伤——每秒掉血 / Damage over time — DPS per second | 跳入水中/使用抗火装备 / Jump in water / use fire resist gear |
| **冰冻(Freeze)** | 冰属性攻击 / Ice attack | 无法行动1-3秒 / Unable to act 1-3s | 使用抗冰装备 / Use freeze resist gear |
| **麻痹(Paralysis)** | 电属性攻击 / Electric attack | 移速降低50% + 随机停止 / Speed -50% + random stun | 使用抗电装备 / Use shock resist gear |
| **中毒(Poison)** | 暗/草系攻击 / Dark/Grass attack | 持续毒伤——比燃烧弱但时间长 / DoT — weaker than burn but longer | 解毒药/等待恢复 / Antidote / wait for recovery |
| **昏迷(Stun)** | 重击类攻击 / Heavy attacks | 无法行动0.5-1秒 / Unable to act 0.5-1s | 高防御可以减少昏迷几率 / High DEF reduces stun chance |
| **恐惧(Fear)** | 特定帕鲁技能 / Specific Pal skills | 帕鲁短暂失控——乱跑 / Pal briefly out of control — runs around | 收回帕鲁重新召唤 / Recall and resummon |
