## 主要角色 / Main Characters

### 里昂·斯科特·肯尼迪 / Leon Scott Kennedy

**基本信息 / Basic Info**：
- 年龄 / Age：27岁（2004年）/ 27 (2004)
- 职业 / Occupation：特工（直属总统）/ Federal Agent (directly under the President)
- 前作 / Previous：生化危机2（1998年）警察新人 / RE2 (1998) RPD rookie

**背景故事 / Background**：
浣熊市事件6年后——里昂已经成为美国政府最顶尖的特工之一。他被派往西班牙某村庄营救被绑架的总统女儿阿什莉·格雷厄姆 / 6 years after Raccoon City — Leon has become one of the US government's top agents. He is sent to a remote Spanish village to rescue the kidnapped President's daughter, Ashley Graham.

**性格特点 / Personality**：
- 冷面幽默——即使面对绝境也会说出俏皮话 / Deadpan humor — jokes even in dire situations
- 极具责任感——绝不放弃任务目标 / Deeply responsible — never abandons mission objectives
- 战斗能力极强——精通格斗和枪械 / Extremely capable — master of CQC and firearms

**经典台词 / Iconic Quotes**：
- "Where's everyone going? Bingo?"
- "You're small time."
- "No thanks, bro."

**战斗风格 / Fighting Style**：
里昂的战斗风格以实用为主——擅长利用体术击倒敌人后追击。他的标志性动作是**回旋踢**（正面体术）/ Leon's fighting style is practical — uses melee knockdowns for follow-up shots. His signature move is the **roundhouse kick**.

### 阿什莉·格雷厄姆 / Ashley Graham

**基本信息 / Basic Info**：
- 年龄 / Age：20岁 / 20
- 职业 / Occupation：白宫实习生 / White House intern
- 身份 / Status：美国总统之女 / Daughter of the US President

**背景故事 / Background**：
阿什莉在从大学回家的路上被不明组织绑架——被带到西班牙的一个偏远村庄。绑架者试图利用她体内的寄生体控制美国政府 / Ashley was abducted on her way home from college, taken to a remote Spanish village. Her captors sought to use the parasite inside her to control the US government.

**重制版变化 / Remake Changes**：
重制版中的阿什莉比原版更加主动——不再是单纯的被救援对象。她会帮里昂打开通道、合作解谜、甚至在战斗中提供帮助 / Ashley in the Remake is much more proactive — no longer a pure damsel in distress. She helps open paths, solves puzzles, and even assists in combat.

**阿什莉AI行为 / Ashley AI Behavior**：
| 指令 / Command | 效果 / Effect |
|---|---|
| 跟随 / Follow | 阿什莉紧跟里昂——但会主动躲避敌人 / Follows closely — actively dodges enemies |
| 等待 / Wait | 待在原地不动——不会引敌人 / Stays put — doesn't attract enemies |
| 隐身模式 / Stealth | 蹲下潜行——降低被发现的概率 / Crouch — reduces detection chance |

**经典台词 / Iconic Quotes**：
- "Leon, help!"
- "I'm not going to be a burden anymore."

### 路易斯·塞拉 / Luis Sera

**基本信息 / Basic Info**：
- 年龄 / Age：约30岁 / ~30
- 职业 / Occupation：前研究员 / Former researcher
- 国籍 / Nationality：西班牙 / Spanish

**背景故事 / Background**：
曾是萨拉查家族的研究员——负责寄生体（Las Plagas）的研究。发现真相后叛逃——试图带着研究资料逃离村庄 / Former researcher for the Salazar family — worked on the Las Plagas parasite. Defected after discovering the truth — tried to escape with research data.

**角色变化 / Character Changes**：
重制版中路易斯的戏份大大增加——角色更加丰满——不再只是一个送钥匙的工具人 / In the Remake, Luis has significantly more screen time — a much more fleshed-out character, no longer just a key delivery NPC.

**经典台词 / Iconic Quotes**：
- "You're not a bad guy yourself, Leon."

## 反派角色 / Antagonists

### 奥斯蒙·萨德勒 / Osmund Saddler

**基本信息 / Basic Info**：
- 身份 / Status：光明教教主 / Cult leader of Los Illuminados
- 年龄 / Age：约50岁 / ~50
- 动机 / Motive：控制全世界的寄生体计划 / Global domination via Las Plagas

**背景故事 / Background**：
萨德勒发现了古时留下的寄生体"Las Plagas"并利用它们建立了宗教组织。他计划利用寄生体控制各国政要——最终统治世界 / Saddler discovered the ancient Las Plagas parasite and built a religious organization around it. His plan was to control world leaders via the parasite and rule the world.

**Boss战阶段 / Boss Fight Phases**：
| 阶段 / Phase | 攻击模式 / Attack Pattern | 弱点 / Weak Point |
|---|---|---|
| 第一阶段 / Phase 1 | 触手攻击 + 召唤教徒 / Tentacle + spawn cultists | 胸口眼睛 / Eye on chest |
| 第二阶段 / Phase 2 | 多触手扫射 + 酸液弹 / Multi-tentacle sweep + acid spit | 同一个眼睛（需要更高精准度）/ Same eye (tighter window) |
| 第三阶段 / Phase 3 | 巨大化后全屏攻击 / Giant form — full screen attacks | 眼睛——但移动更快 / Eye — faster movement |

**经典台词 / Iconic Quotes**：
- "You will become one of us, one of the Illuminados."

### 拉蒙·萨拉查 / Ramon Salazar

**基本信息 / Basic Info**：
- 身份 / Status：城堡领主 / Castle Lord
- 年龄 / Age：虽老但身高如幼童 / Elderly but child-sized
- 特点 / Trait：疯狂且自负 / Insane and arrogant

**背景故事 / Background**：
萨拉查家族世世代代守护着Las Plagas的封印。年轻的拉蒙受到萨德勒的蛊惑——打开了封印并将寄生体释放出来 / The Salazar family had guarded the Las Plagas seal for generations. Young Ramon was manipulated by Saddler into breaking the seal and releasing the parasite.

**Boss战打法 / Boss Fight**：
| 阶段 / Phase | 机制 / Mechanic | 策略 / Strategy |
|---|---|---|
| 第一阶段 / Phase 1 | 萨拉查站在高处——召唤杂兵 / Calls adds from above | 使用狙击枪打他——或者不理他清小兵 / Snipe him or ignore and clear adds |
| 第二阶段 / Phase 2 | 墙上的脸型怪物（巨大化）/ Giant face monster on wall | 打眼球——然后打张开的大嘴里的舌头 / Shoot eyes, then tongue in open mouth |

### 杰克·克劳瑟 / Jack Krauser

**基本信息 / Basic Info**：
- 身份 / Status：前美军士兵 / Former US soldier
- 年龄 / Age：约40岁 / ~40
- 动机 / Motive：追求力量——雇佣兵 / Pursuit of power — mercenary

**背景故事 / Background**：
里昂在特工训练时期的前辈兼好友。在浣熊市事件后退役——转而成为雇佣兵。接受了Las Plagas寄生以获得更强的力量 / Leon's senior and friend from agent training. Retired after Raccoon City — became a mercenary. Accepted Las Plagas for enhanced strength.

**Boss战打法 / Boss Fight**：
| 阶段 / Phase | 攻击 / Attack | 应对 / Counter |
|---|---|---|
| 第一阶段 / Phase 1 | 冲锋+突击步枪 / Charge + AR fire | 保持距离——用狙击枪打 / Distance — sniper |
| 第二阶段 / Phase 2 | 变异——触手攻击 / Mutated — tentacle | 打暴露的心脏弱点 / Shoot exposed heart |
| QTE阶段 / QTE Phase | 连续按键对抗 / Button mashing struggle | 不要按错——否则断头 / Don't miss — instant fail |

**经典台词 / Iconic Quotes**：
- "You're just a rookie who got lucky."

- 特点 / Trait：体型巨大——拥有超人的力量 / Giant build — superhuman strength

**背景故事 / Background**：
村庄的管理者——是萨德勒忠诚的追随者。他负责筛选村民进行寄生——维持村庄的秩序 / Village overseer — Saddler's loyal follower. Responsible for selecting villagers for parasitism and maintaining order.

| 阶段 / Phase | 攻击 / Attack | 应对 / Counter |

| 第二阶段 / Phase 2 | 变异——后背伸出巨大触手 / Mutated — huge tentacle from back | 打暴露的寄生体核心 / Shoot exposed parasite core |

## 重要NPC / Important NPCs

### 商人 / The Merchant

**基本信息 / Basic Info**：
- 身份 / Status：神秘武器商 / Mysterious arms dealer
- 特征 / Trait：穿大衣——蒙面——永远在安全屋等你 / Coated, masked, always waiting in safe rooms
- 经典台词 / Iconic Quote："What are you buyin'?"

**商人的作用 / Merchant Role**：
| 服务 / Service | 说明 / Description |
|---|---|
| 购买武器 / Buy Weapons | 各种枪械和近战武器 / All firearms and melee weapons |
| 改造武器 / Upgrade Weapons | 提升武器性能 / Improve weapon stats |
| 购买弹药 / Buy Ammo | 补充各种口径弹药 / Restock all ammo types |
| 出售宝物 / Sell Treasures | 以比塞塔收购宝物 / Buy treasures for pesetas |
| 接受悬赏 / Accept Requests | 完成悬赏任务换尖晶石 / Complete requests for Spinels |

**商人的秘密 / Merchant Secrets**：
- 没有人知道商人的真实身份 / No one knows the Merchant's true identity
- 他似乎知道Las Plagas的全部历史 / He seems to know everything about Las Plagas
- 有粉丝理论认为他是一代角色转世 / Fan theory: he's a reincarnation of a RE1 character

### 英格丽德·汉尼根 / Ingrid Hunnigan

**基本信息 / Basic Info**：
- 身份 / Status：联邦特工——里昂的联络员 / Federal agent — Leon's radio contact
- 声音 / Role：全程为里昂提供情报支援 / Provides intel throughout the mission

**她的作用 / Her Role**：
汉尼根是里昂在任务中的眼睛和耳朵——提供卫星情报、目标指引和战术建议 / Hunnigan is Leon's eyes and ears — provides satellite intel, objective guidance, and tactical advice.

**经典台词 / Iconic Quotes**：
- "Leon, can you hear me?"
- "The target is Ashley Graham. She's being held in the village church."
