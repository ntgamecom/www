商人悬赏任务之一——在每个章节区域打碎全部蓝色徽章可获得尖晶石奖励 / One of the Merchant's requests — destroy all blue medallions in each area for Spinel rewards.

### 蓝徽章收集技巧 / Medallion Tips

- 使用狙击枪打——距离远精度高 / Use rifle — long range precision
- 打破的徽章会发出清脆的玻璃碎裂声 / Broken medallions make a distinct glass shatter sound
- 可以在章节选择里补漏——通关后回到章节收集 / Can revisit via chapter select after finishing game
- 全部打完后找商人领取额外尖晶石奖励 / All done = extra Spinel reward from Merchant

## 彩蛋 / Easter Eggs

### 湖主雕像 / Lake Lord Statue

在湖区——如果你用火箭筒打湖边的湖主雕像——雕像的嘴会张开并吐出一颗宝石 / In the lake area — shoot the Lake Lord statue with a rocket launcher — its mouth opens and spits out a gem.

### 打字机彩蛋 / Typewriter Easter Egg

在城堡的一个房间中——有一台打字机。如果你在上面反复按使用键——里昂会说关于打字机保存的吐槽 / In a castle room — there's a typewriter. Spamming the use key triggers Leon's comment about typewriter saves.

### 商人对话 / Merchant Dialogues

不去商人那里——或者反复和商人对话——会触发不同的台词 / Ignoring the Merchant or talking to him repeatedly triggers different voice lines.

**隐藏台词 / Hidden Line**：在通过特定剧情后和商人说话——商人会说："Stranger, I'm glad you made it." / Talk to Merchant after certain story events — he says: "Stranger, I'm glad you made it."

### 彩蛋服装 / Easter Egg Costumes

阿什莉的骑士甲有一套隐藏的猫耳装——需要在新游戏+中完成四个额外的挑战才能获得 / Ashley's Knight Armor has a hidden cat ear variant — requires completing four additional challenges in NG+.
