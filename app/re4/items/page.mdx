## 商人的问候 / The Merchant

**"Stranger, stranger. Now that`s a weapon."**

商人是游戏中最重要的 NPC，在各个安全屋出现，提供武器买卖、改造升级、道具交易等服务。他会在每个主要区域出现1-2次。

The Merchant is the most important NPC in the game, appearing in safe rooms throughout each major area. He offers weapon sales, upgrades, item trading, and more.

### 商人服务 / Merchant Services

| 服务 Service | 说明 Description |
|---|---|
| 购买武器 Buy Weapons | 购买新武器，价格随难度提升 / Buy new weapons, prices scale with difficulty |
| 出售武器 Sell Weapons | 出售不需要的武器换取比塞塔 / Sell unwanted weapons for pesetas |
| 改造升级 Upgrade | 使用比塞塔提升武器属性 / Use pesetas to upgrade weapon stats |
| 购买道具 Buy Items | 购买弹药、治疗品、公文包等 / Buy ammo, healing items, cases, etc. |
| 交易尖晶石 Trade Spinels | 用蓝色徽章换取的尖晶石购买特殊道具 / Trade spinels for exclusive items |
| 修理匕首 Repair Knife | 恢复匕首耐久度 / Restore knife durability |

## 比塞塔与货币 / Pesetas & Currency

**比塞塔（Pesetas）** 是游戏中的通用货币，通过击杀敌人、搜刮宝物和完成悬赏获得。

Pesetas are the main currency, earned by defeating enemies, looting treasures, and completing requests.

### 赚钱技巧 / Money Making Tips

1. **收集所有宝藏 / Collect All Treasures**: 每个区域都有隐藏宝藏，仔细搜刮 / Every area has hidden treasures, search carefully
2. **完成蓝徽章悬赏 / Complete Blue Medallion Requests**: 每次悬赏奖励尖晶石和比塞塔 / Each request rewards spinels and pesetas
3. **出售多余武器 / Sell Unused Weapons**: 确定不用的武器及时出售换钱 / Sell weapons you do not need
4. **宝石镶嵌增值 / Gem镶嵌 Value Boost**: 将宝石嵌入宝藏在商人处出售可获更高价格 / Mount gems in treasures for higher sell prices
5. **优先投资公文包 / Invest in Cases**: 更大空间意味着可以携带更多宝藏 / More space means more treasure capacity

## 出售宝藏的最佳策略 / Best Treasure Selling Strategy

1. **保留钻石保留宝石 / Hold Until Full**: 尽量在镶嵌满宝石后再出售宝藏 / Sell treasures only after mounting all gems
2. **优先镶嵌亚历山大石 / Prioritize Alexandrite**: 价格最高的宝石，组合增值最大 / Most valuable gem, best组合 value
3. **皇家权杖配红宝石 / Crown + Ruby Combo**: 皇冠配3颗红宝石可获得极高倍率 / Crown with 3 rubies yields high multiplier
4. **不要乱卖宝石 / Do Not Sell Loose Gems**: 未镶嵌的宝石出售价格很低 / Loose gems sell for very little
5. **使用商人挂件 / Use Merchant Charm**: 增加商人收购价格 / Increases merchant buy prices
