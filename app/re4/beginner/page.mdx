### 移动射击 /移动 Shooting

重制版中你可以 **边移动边射击（move while shooting）**，但移动中准度会下降。建议在关键时刻停下脚步精确瞄准。

You can move and shoot simultaneously in the remake, but accuracy decreases while moving. Stop to aim precisely during critical moments.

## 瞄准技巧 / Aiming Tips

- **爆头优先 Headshots First**: 射击头部可触发硬直，接体术秒杀普通敌人 / Shooting heads stuns enemies for a melee follow-up
- **腿部射击 Leg Shots**: 打腿让敌人跪下，争取时间逃跑 / Shoot legs to stagger, buying time to escape
- **精准瞄准 Precision Aiming**: 按住瞄准键后不要移动，准星会随时间收缩提高精度 / Hold aim without moving to tighten reticle over time
- **激光瞄准器 Laser Sight**: 部分武器可装备激光瞄准器，显著提高盲射精度 / Equip laser sight for significantly improved hip-fire accuracy
- **弱点攻击 Weak Points**: 寄生虫寄生的敌人头部的寄生虫是弱点 / Parasite heads are weak points on infected enemies

### 瞄准技巧总结 / Aiming Summary Table

| 部位 Target | 效果 Effect | 推荐时机 Best Used |
|---|---|---|
| 头部 Head | 硬直触发体术 / Stun + melee | 单体敌人 / Single enemy |
| 腿部 Legs | 减速跪下 / Slow + kneel | 逃跑/控场 / Escape / crowd control |
| 寄生虫 Parasite | 高伤害一击必杀 / High damage instant kill | 寄生虫暴露时 / When exposed |
| 膝盖 Knees | 敌人跪下后可触发处决 / Kneel then execute | 搭配霰弹 / With shotgun |

## 刀反系统 / Parry Mechanic

### 如何刀反 / How to Parry

按下 **格挡键（Parry button, LB/L1/Q）** 在敌人攻击命中前的瞬间进行匕首格挡。成功挡下后敌人会露出大硬直，此时可发动体术或反击。

Press the Parry button (LB/L1/Q) just before an enemy attack lands. A successful parry leaves the enemy wide open for a melee follow-up or counterattack.

### 刀反要点 / Parry Tips

| 项目 Item | 说明 Description |
|---|---|
| 判定窗口 Timing | 攻击命中前约0.3秒 / Approximately 0.3 seconds before impact |
| 耐久消耗 Durability Cost | 每次格挡消耗匕首5-10%耐久 / Each parry costs 5-10% knife durability |
| 可格挡攻击 Parryable | 近战攻击、投掷物（斧头/镰刀）/ Melee attacks, thrown items (axes/sickles) |
| 不可格挡 Unparryable | 抓取攻击、爆炸、Boss特定技能 / Grabs, explosions, certain boss attacks |
| 战斗刀 Combat Knife | 耐久更高，更适合频繁刀反 / Higher durability, better for frequent parrying |
| 靴刀 Boot Knife | 耐久低，建议留作最后手段 / Low durability, save as last resort |

### 匕首耐久度 / Knife Durability

匕首有耐久度系统，使用匕首攻击和刀反都会消耗耐久。耐久耗尽后匕首会损坏，需在商人处修理或购买新的。

The knife has a durability system. Both attacks and parries consume durability. When depleted, the knife breaks and must be repaired or replaced at the Merchant.

### 难度对比 / Difficulty Comparison

| 难度 Difficulty | 敌人强度 Enemy Strength | 资源掉落 Resources | 商人价格 Prices | 推荐玩家 Recommended For |
|---|---|---|---|---|

| 标准 Standard | 中 Medium | 正常 Normal | 正常 Normal | 普通玩家 / Average players |

### 难度对游戏的影响 / Difficulty Effects

**辅助模式 Assisted**:
- 自动回血至黄色状态 / Auto-heals to yellow health
- 辅助瞄准 Assist aim
- QTE 自动完成 / QTEs auto-complete

**硬核模式 Hardcore**:
- 敌人攻击欲望更强 / Enemies more aggressive
- 存档次数受限（打字机色带）/ Limited saves (typewriter ribbons)
- 无自动回血 / No auto-healing

**专家模式 Professional**:
- 仅限新游戏+或通关后解锁 / Unlocked after completion or New Game+
- 无自动瞄准 / No aim assist
- 敌人配置变化 / Enemy placement changes
- 完美格挡要求更严格 / Tighter parry windows

## 生存技巧 / Survival Tips

1. **搜刮一切 Search Everything**: 每个角落都有宝藏和资源，仔细搜索每个区域 / Every corner has treasures and resources, search thoroughly
2. **优先升级公文包 Upgrade Case First**: 初始格子有限，优先购买公文包升级 / Limited slots at first, prioritize case upgrades
3. **匕首是第二条命 Knife Is a Second Life**: 被敌人抓住时可用匕首挣脱，关键时刻救你一命 / Use knife to escape grabs in critical moments
4. **蛇形走位 Zigzag Movement**: 面对投掷武器时蛇形移动减少被击中概率 / Zigzag against thrown weapons to avoid damage
5. **活用闪避弹 Use Flash Grenades**: 对寄生虫敌人有即死效果，遇到大批寄生敌时优先使用 / Instant kill on plaga enemies, use against large groups
6. **黄草务必合成 Always Combine Yellow Herbs**: 永久提升血量上限是通关关键 / Permanent HP increase is critical for survival
7. **先杀弓手 Archers First**: 弓箭手远程威胁大，优先清除 / Archers are a high long-range threat, eliminate first
8. **管理背包空间 Inventory Management**: 不常用的武器可暂存箱中，保证背包有空间捡物资 / Store unused weapons to free inventory space
