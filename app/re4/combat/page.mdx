## 战斗技巧 / Combat Techniques

### 体术系统 / Melee System

| 触发方式 Trigger | 体术动作 Melee Move | 效果 Effect |
|---|---|---|
| 爆头硬直 Headshot Stun | 回旋踢 Roundhouse Kick | 击飞周围敌人 / Knocks back nearby enemies |
| 腿部硬直 Leg Stun | 下段踢 Low Kick | 倒地敌人 / Knocks enemy down |
| 背后潜行 Behind Stealth | 暗杀 Stealth Kill | 一击必杀 / Instant kill |
| 刀反成功 Parry Success | 反击刺击 Counter Stab | 高伤害 / High damage |

### 潜行系统 / Stealth System

- 蹲下接近敌人背后可触发暗杀提示 / Crouch and approach from behind for stealth kill prompt
- 暗杀对普通村民和教徒有效 / Works on normal villagers and cultists
- 部分敌人（装甲兵、 Boss ）无法暗杀 / Not effective on armored enemies or bosses
- 暗杀不消耗匕首耐久 / Stealth kills do not consume knife durability

### 体术策略 / Melee Strategy

1. **爆头 + 体术 Headshot + Melee**: 最基础的连段，节省弹药 / Basic combo to save ammo
2. **体术控场 Melee Crowd Control**: 回旋踢可击倒周围多个敌人 / Roundhouse kick knocks down surrounding enemies
3. **倒地追击 Downed Follow-up**: 敌人倒地后可用匕首处决或补枪 / Execute downed enemies with knife or gun
4. **刀反反击 Parry Counter**: 成功后接体术再补刀 / Parry then melee then finish
