## 新游戏+系统 / New Game Plus

通关任意难度后开启新游戏+——继承所有武器、改造、宝物和金钱 / After completing any difficulty, NG+ unlocks — carry over all weapons, upgrades, treasures, and money.

### NG+继承内容 / NG+ Carryover

| 继承 / Carry Over | 不继承 / Does Not Carry |
|---|---|
| 全部武器及改造 / All weapons + upgrades | 小钥匙 / Small keys |
| 全部宝物和宝石 / All treasures + gems | 尖晶石进度 / Spinel progress |
| 金钱 / Pesetas | 消耗品（药/弹药）/ Consumables |
| 手提箱升级 / Case upgrades | 章节评价 / Chapter ranks |

**NG+核心策略 / NG+ Core Strategy**：
1. 先攒200万比塞塔购买无限火箭筒 / Save 2M ptas for Infinite Rocket Launcher
2. 使用无限火箭筒快速通关专家难度 / Use it to speedrun Professional
3. 然后解锁芝加哥打字机和手炮 / Then unlock Chicago Sweeper and Handcannon
4. 最终目标：全武器无限弹药 + 全成就 / End goal: all infinite ammo weapons + all achievements

## 佣兵模式 / Mercenaries Mode

### 模式说明 / Mode Overview

佣兵模式是独立计时挑战——在限定时间内杀敌获得分数——时间可击杀特定敌人延长 / Mercenaries is a timed challenge — kill enemies for score within the time limit — certain kills extend time.

**核心机制 / Core Mechanics**：
- 基础时间：120秒 / Base time: 120s
- 击败特定敌人加时间 / Specific enemies add time
- 连杀达到指定数量获得奖杯奖励 / Kill streak = bonus points
- 全部地图获得S级评价解锁奖励 / S rank on all maps = rewards

### 地图与评价 / Maps & Ranks

| 地图 / Map | 敌人类型 / Enemy Type | S级分数 / S Rank Score | 推荐角色 / Best Character |
|---|---|---|---|
| 村庄 / Village | 村民 / Ganados | 150,000 | 汉克 / Hunk |
| 城堡 / Castle | 教徒 / Cultists | 150,000 | 威斯克 / Wesker |
| 孤岛 / Island | 士兵 / Soldiers | 150,000 | 克拉泽 / Krauser |

**S级策略 / S Rank Strategy**：
1. 优先击杀紫色标记的特殊敌人——它们给最多时间加成 / Priority: purple-marked special enemies — highest time bonus
2. 连杀15-20个后——会有金色敌人出现——击杀可得大量分数 / After 15-20 killstreak — gold enemy spawns — massive points
3. 不要停下来——地图上有时间奖励沙漏——收集它们 / Never stop moving — collect time bonus hourglasses on map
4. 保存大招（R+近战）应对紧急情况 / Save special ability (R + melee) for emergencies

### 汉克打法 / Hunk Strategy

汉克有处决技能——正面处决不仅秒杀还加时间——是冲S级的最佳选择 / Hunk has execution skill — face execution instakills and adds time — best for S rank.

**连招 / Combo**：体术击倒 > 处决动画 > 无敌帧——循环 / Melee knockdown > execution animation > invincibility frames — repeat.
