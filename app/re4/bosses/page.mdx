# 首领攻略 / Boss Guide

本页面详细分析所有首领战和精英敌人的打法策略，提供站位建议、弱点分析和推荐装备。

This page covers all boss fights and elite enemy strategies, including positioning advice, weak point analysis, and recommended equipment.

---

## 主要首领 / Main Bosses

### 电锯姐妹花 / Bella Sisters

| 属性 Stat | 数值 Value |
|---|---|
| 出现章节 Chapter | 第1章 / Chapter 1 |
| 弱点 Weak Point | 头部 Head |
| 推荐武器 Weapon | W-870 霰弹枪 / Shotgun |
| 危险等级 Threat | 中 / Medium |

**战术要点 / Strategy**:

- 保持距离，不要被逼入角落 / Keep distance, do not get cornered
- 使用霰弹枪射击头部造成硬直 / Use shotgun headshots to stun
- 场景中有油桶可爆炸杀伤 / Explosive barrels in the arena
- 姐妹花同时出现，注意走位避免被包围 / Both sisters appear, position carefully to avoid being surrounded
- 先集中攻击一个，减少压力 / Focus on one first to reduce pressure
- QTE 躲避电锯攻击 / QTE dodge chainsaw attacks

### El Gigante（巨人）

| 属性 Stat | 数值 Value |
|---|---|
| 出现章节 Chapter | 第3章 / Chapter 3 |
| 弱点 Weak Point | 背部寄生虫 / Plaga on back |
| 推荐武器 Weapon | 步枪 / Rifle |
| 危险等级 Threat | 中高 / Medium-High |

**战术要点 / Strategy**:

- 巨人共有两次遭遇战（村庄和城堡）/ Two encounters (village and castle)
- 引诱巨人攻击柱子可造成硬直 / Lure giant into pillars for stun
- 翻到巨人背上攻击寄生虫弱点 / Climb onto its back and attack the plaga weak point
- 射击头部也可造成伤害，但效率低 / Head shots deal damage but are inefficient
- 霰弹枪近距离轰击腹部效果也不错 / Shotgun blasts to the stomach are effective up close
- 注意巨人的抓取和重砸攻击 / Watch for grab and slam attacks
- 闪光弹可造成短暂硬直 / Flash grenades cause brief stun

### Del Lago（湖之主）

| 属性 Stat | 数值 Value |
|---|---|
| 出现章节 Chapter | 第4章 / Chapter 4 |
| 弱点 Weak Point | 头部 Head |
| 推荐武器 Weapon | 鱼叉 / Harpoon |
| 危险等级 Threat | 中 / Medium |

**战术要点 / Strategy**:

- 驾驶小船使用鱼叉攻击 / Drive the boat and use the harpoon
- 瞄准头部伤害最高 / Aim for the head for maximum damage
- 当湖之主潜入水下时准备躲避冲撞 / When it dives, prepare to dodge its charge
- 冲撞攻击横移即可躲避 / Side-step to avoid charges
- 甩尾攻击向后躲避 / Back up to avoid tail sweeps
- 保持移动，不要停船 / Keep moving, never stop the boat
- 攻击节奏：投掷鱼叉 - 收回 - 重复 / Attack rhythm: throw harpoon - reel in - repeat

### Bitores Mendez（门德斯）

| 属性 Stat | 数值 Value |
|---|---|
| 出现章节 Chapter | 第6章 / Chapter 6 |
| 弱点 Weak Point | 头部、眼睛 / Head, eye |
| 推荐武器 Weapon | 霰弹枪 + 步枪 / Shotgun + Rifle |
| 危险等级 Threat | 高 / High |

**第一阶段 Phase 1 - 正常形态 / Normal Form**:

- 门德斯会使用铁棍攻击，攻击范围大 / Mendez attacks with an iron bar, wide range
- 射击头部积累硬直值 / Shoot the head to build up stun
- 硬直后可绕背输出 / Stun then circle behind for damage
- 利用场景中的柱子躲避攻击 / Use pillars for cover
- 注意他的冲刺攻击，及时闪避 / Watch for his charge attack, dodge in time

**第二阶段 Phase 2 - 变异形态 / Mutated Form**:

- 门德斯变异为巨大怪物 / Mendez mutates into a large monster
- 弱点变为背后的巨大眼睛 / Weak point becomes the giant eye on its back
- 使用霰弹枪轰击正面创造输出机会 / Use shotgun blasts to the front to create openings
- 绕到背后攻击眼睛 / Circle behind to attack the eye
- 场景中有油桶可利用 / Exploit explosive barrels in the arena
- 保持移动，不要站在原地输出 / Keep moving, never stand still to attack
- 火焰喷射器对此战效果极佳 / Flamethrower is highly effective here

### Verdugo（右手）

| 属性 Stat | 数值 Value |
|---|---|
| 出现章节 Chapter | 第8章 / Chapter 8 |
| 弱点 Weak Point | 身体任何部位 / Any body part |
| 推荐武器 Weapon | 马格南 + 霰弹 / Magnum + Shotgun |
| 危险等级 Threat | 高 / High |

**战术要点 / Strategy**:

- 房间中有氮气罐可冷冻 Verdugo / Nitrogen tanks in the room can freeze Verdugo
- 冷冻后 Verdugo 会硬直并暴露弱点 / Freeze causes stun and exposes weak points
- 冷冻期间全力输出 / Maximize damage during freeze
- 氮气罐有限，合理使用 / Nitrogen tanks are limited, use wisely
- Verdugo 移动极快，保持距离 / Verdugo is very fast, maintain distance
- 它的尾巴攻击范围极大 / Its tail attack has huge range
- 推荐使用马格南或高威力步枪 / Use magnum or high-power rifle
- 也可以不击杀直接过关（不开阀门逃生）/ Optional skip: escape without opening the valve

### Ramon Salazar（拉蒙·萨拉扎）

| 属性 Stat | 数值 Value |
|---|---|
| 出现章节 Chapter | 第10章 / Chapter 10 |
| 弱点 Weak Point | 本体（头顶的萨拉扎）/ The core (Salazar on top) |
| 推荐武器 Weapon | 火箭筒 / Rocket Launcher |
| 危险等级 Threat | 中高 / Medium-High |

**战术要点 / Strategy**:

- 萨拉扎驾驶巨型机甲战斗 / Salazar pilots a giant mechanical statue
- 机甲弱点为头顶的萨拉扎本体 / Weak point is Salazar himself on top of the statue
- 射击本体造成大量伤害 / Shoot the core for massive damage
- 机甲会发射导弹和使用触手攻击 / The statue fires missiles and uses tentacle attacks
- 利用场景中的柱子躲避 / Use pillars for cover
- 最简单打法：购买火箭筒一发秒杀 / Easiest strat: buy a rocket launcher for one-shot kill
- 正常打法：步枪精确射击头部本体 / Normal strat: precise rifle shots to the head core
- 机甲倒地时冲上前近战输出 / Rush in for melee damage when the statue falls

### Krauser（克劳瑟）

| 属性 Stat | 数值 Value |
|---|---|
| 出现章节 Chapter | 第11章（第一次）/ 第15章（最终战）|
| 弱点 Weak Point | 头部 / Head |
| 推荐武器 Weapon | 战斗匕首 + 步枪 / Combat Knife + Rifle |
| 危险等级 Threat | 极高 / Very High |

**第一次遭遇 / First Encounter（第11章）**:

- 匕首对决，必须使用刀反 / Knife duel, parrying is essential
- Krauser 的攻击模式有规律 / Krauser`s attack patterns are predictable
- 成功刀反后快速攻击 / Parry then quickly attack
- 他后撤时会扔闪光弹，注意躲避 / He throws flash grenades when retreating, dodge them
- 场景中有手枪和弹药补给 / Handgun and ammo supply in the arena
- 坚持一段时间后剧情结束 / Survive for a set time for the cutscene to trigger

**最终战 / Final Battle（第15章）**:

- Krauser 使用弓箭和变异手臂攻击 / Krauser uses a bow and mutated arm attacks
- 第一阶段：远程对射，利用掩体 / Phase 1: ranged exchange, use cover
- 射击 Krauser 头部阻止他射箭 / Shoot his head to interrupt his bow shots
- 第二阶段：近身格斗，他手臂变异为巨刃 / Phase 2: close combat, arm mutates into blade
- 此阶段刀反非常有效 / Parrying is highly effective in this phase
- 成功刀反后使用马格南或步枪输出最大伤害 / After successful parry, use magnum or rifle for max damage
- 保持血量，他的连击伤害很高 / Keep health up, his combos deal heavy damage
- 场景中的油桶和爆炸物可利用 / Use explosive barrels and objects in the arena

### Osmund Saddler（奥斯蒙德·萨德勒）

| 属性 Stat | 数值 Value |
|---|---|
| 出现章节 Chapter | 第16章 / Chapter 16 |
| 弱点 Weak Point | 眼睛 Eye |
| 推荐武器 Weapon | 马格南 / 火箭筒 / Magnum / Rocket Launcher |
| 危险等级 Threat | 极高 / Very High |

**第一阶段 Phase 1**:

- 萨德勒以巨型怪物形态出现 / Saddler appears as a giant monster
- 射击他身体各处的眼睛 / Shoot the eyes scattered on his body
- 每打掉一只眼睛会造成硬直 / Each eye destroyed causes a stun
- 场上有四座金色雕像可使用 / Four golden statues in the arena
- 将雕像推倒砸向萨德勒可造成大量伤害 / Push statues onto Saddler for massive damage
- 节省弹药，主要依靠雕像攻击 / Save ammo, rely mainly on statue attacks

**第二阶段 Phase 2**:

- 萨德勒缩小但更灵活 / Saddler shrinks but becomes more agile
- 触手攻击范围广，持续移动躲避 / Tentacle attacks have wide range, keep moving
- 射击本体头部积累伤害 / Shoot the main body head for damage
- Ada 会扔下一发火箭筒 / Ada drops a rocket launcher
- **拾取火箭筒一发结束战斗 / Pick up the rocket launcher for a one-shot finish**
- 也可以使用马格南持续输出 / Alternatively, use magnum for sustained damage

**推荐打法 / Recommended Strategy**:

```
1. 第一阶段推动四座雕像攻击 / Phase 1: push all four statues
2. 第二阶段捡起火箭筒 / Phase 2: grab the rocket launcher
3. 一发带走 / One shot finish
```

## 精英敌人 / Elite Enemies

### 再生者 / Regenerator

| 属性 Stat | 数值 Value |
|---|---|
| 出现章节 Chapter | 第12章 / Chapter 12 |
| 弱点 Weak Point | 体内蓝色物体 / Blue objects inside |
| 推荐武器 Weapon | 热成像步枪 / Thermal scope rifle |
| 危险等级 Threat | 高 / High |

- 需要热成像仪看到体内的蓝色寄生虫 / Need thermal scope to see blue plagas inside
- 射击所有蓝色寄生虫后方可击杀 / Shoot all blue parasites to kill
- 不攻击时再生者会缓慢恢复生命 / Regenerator slowly heals if not attacked
- 它会抱住里昂吸血，及时挣脱 / It grabs Leon and drains health, break free quickly
- 腿被打断后仍会爬行追击 / It crawls and chases even with broken legs
- 建议使用高精度武器逐个点杀寄生虫 / Use high-precision weapons to pick off parasites

### 铁爪 / Iron Maiden

- 再生者的变异体，身上带刺 / Variant of Regenerator with spikes
- 近战攻击附带额外伤害 / Melee attacks deal extra damage
- 同样使用热成像仪攻击体内寄生虫 / Same thermal scope strategy
- 保持更远距离，避免被刺伤 / Keep more distance to avoid spike damage

### 装甲兵 / Armored Enemy

| 属性 Stat | 数值 Value |
|---|---|
| 推荐武器 Weapon | 霰弹枪 / Shotgun |
| 策略 Strategy | 打掉头盔后爆头 / Remove helmet then headshot |

装甲兵从头到脚覆盖盔甲，普通手枪伤害很低。使用霰弹枪射击头部打掉头盔，然后爆头击杀。

Armored enemies are covered head to toe. Handguns deal minimal damage. Use a shotgun to blast off the helmet, then go for a headshot.
