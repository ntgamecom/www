## 升级系统概述 / Upgrade System Overview

生化危机4重制版的成长系统分为**武器升级**（商人处使用比塞塔/k改造）和**宝物收集**（合成宝石提升价值）。相比原版，重制版的改造更灵活——改造武器会改变外观 / RE4 Remake's progression system is split into **weapon upgrades** (from the Merchant with pesetas) and **treasure combining** (gem socketing to increase value). Remake upgrades are more flexible — upgraded weapons change appearance.

### 改造货币 / Upgrade Currencies

| 货币 / Currency | 用途 / Use | 获取方式 / How to Get |
|---|---|---|
| 比塞塔 / Pesetas | 武器改造+弹药购买 / Weapon upgrades + ammo | 击杀敌人/场景拾取/宝物出售 / Kills, pickups, treasure |
| 尖晶石 / Spinels | 兑换专属武器和配件 / Exclusive gear | 任务奖励/蓝徽章 / Requests, blue medallions |
| 宝石 / Gems | 嵌在宝物上提升售价 / Socket into treasures | 敌人掉落/宝箱 / Enemy drops, chests |

## 专业技能 / Pro Skills

### 格挡与体术 / Parry & Melee

重制版新增的核心技能——完美格挡后可以接体术 / New core skill in Remake — perfect parry into melee follow-up.

| 技能 / Skill | 操作 / Input | 效果 / Effect |
|---|---|---|
| 格挡 / Parry | 防守键时机精准 / Block at right moment | 弹开近战攻击并硬直敌人 / Deflect melee and stagger enemy |
| 完美格挡 / Perfect Parry | 攻击命中前0.1秒格挡 / Block 0.1s before hit | 大幅硬直——可以接正面体术 / Long stagger — into face melee |
| 正面体术 / Roundhouse | 格挡后按近战 / Melee after parry | 范围击倒周围敌人 / AoE knockdown |
| 背后体术 / Behind Kick | 从背后接近按近战 / Melee from behind | 秒杀普通敌人 / Instant kill basic enemies |
| 处决 / Execution | 倒地敌人附近按近战 / Melee near downed enemy | 高伤害终结 / High damage finisher |

### 刀的使用 / Knife Usage

**小刀 / Combat Knife**：
- 默认武器——可格挡/刺杀/破坏箱子 / Default — parry, stab, break boxes
- 每个敌人背后刺杀消耗5%耐久 / Backstab costs 5% durability per enemy
- 可修复/升级——独占改造后耐久无限 / Repairable/upgradable — exclusive = infinite durability

**靴刀 / Boot Knife**：
- 获得时间较晚——伤害更高但耐久低 / Later to obtain — higher dmg but lower durability
- 独占改造后也是无限耐久 / Exclusive also gives infinite durability

## 升级规划建议 / Upgrade Planning

### 一周目推荐 / First Playthrough

1. **初期**：SG-09手枪 > 火力LV2 > W-870霰弹 > 火力LV2 / Early: SG-09 > Power LV2 > W-870 > Power LV2
2. **中期**：半自动狙击枪 > 独占改造 / Mid: Semi-Auto Rifle > Exclusive Upgrade
3. **后期**：打击者霰弹 > 独占改造 > 红9或黑尾 > 独占 / Late: Striker > Exclusive > Red9 or Blacktail > Exclusive

### 新游戏+/专家难度 / NG+/Professional

1. **优先解锁**：防弹背心+鸡头帽——生存优先 / Priority: Vest + Chicken Hat — survival first
2. **武器选择**：红9 + 半自动狙 + 打击者——三把独占改造全满 / Red9 + Semi-Auto Rifle + Striker — all exclusive upgrade
3. **无限火箭筒**：通关后花200万购买——专家难度最优解 / Infinite Rocket Launcher: 2M ptas — Professional easy mode
