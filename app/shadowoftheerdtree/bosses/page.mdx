# BOSS攻略 / Boss Strategies

黄金树之影的Boss战难度普遍高于本体——每个Boss都有复杂的多阶段设计和极高的伤害。本攻略提供全Boss详细打法 / Shadow of the Erdtree boss fights are universally harder than the base game — each boss features complex multi-phase designs and extreme damage. This guide covers strategies for all bosses.

---

## 通用Boss战技巧 / Universal Boss Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **幽影树庇护等级决定难度 / Blessing level = difficulty** | 打不过Boss时→先去探索收集碎片。不要硬刚 / Can't beat a boss? Go explore for fragments first. Don't brute force |
| **骨灰和仿生泪滴 ≠ 轻松赢 / Summons ≠ easy win** | DLC中Boss会频繁绕过骨灰攻击玩家。灵灰越高骨灰越有用 / Bosses bypass summons frequently in DLC. Higher Revered Ashes = better summons |
| **跳劈是神技 / Jump attack is king** | DLC Boss削韧门槛高——但跳劈的削韧值依然优秀。配合硬尖露滴更强 / DLC bosses have high stagger thresholds — jump attacks still have great stagger. Pair with Stonebarb Cracked Tear |
| **属性防御很重要 / Elemental defense matters** | 每个Boss有主要的伤害类型——对应的防护护符/油脂减伤可达50% / Each boss has a primary damage type — correct defense talisman/grease can reduce dmg by 50% |
| **不要锁定大型Boss / Don't lock on large bosses** | 神兽舞狮/梅瑟莫等大型Boss——锁定时镜头乱飞。不锁定打反而更稳 / Don't lock on large bosses like Dancing Lion or Messmer — free camera is more stable |

## 主线Boss / Main Bosses

### 神兽舞狮 / Divine Beast Dancing Lion

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-50%)** | 冰霜吐息→四连爪击→旋转AOE→闪电突进 / Frost breath → 4-claw combo → spin AoE → lightning rush | 冰霜吐息往侧面跑。四连爪最后一下延迟——等0.5秒再翻滚 / Side-run from frost breath. Last claw is delayed — roll 0.5s late |
| **P2 (50%-0%)** | 新增：全属性AOE轮换(冰→雷→风)→范围扩大 / Adds: full elemental AoE rotation (frost→lightning→wind) → wider range | AOE轮换时跑远→等动画结束→再近身打。不要试图在AOE时输出 / Run far during AoE rotation → wait for animation to end → close in |
| **弱点 / Weakness** | 打击属性 + 圣属性 / Strike damage + Holy | 大锤+圣油脂 = 最高效率 / Hammer + Holy Grease = best DPS |
| **推荐打法 / Strategy** | 不锁定→站左手侧→打前腿。跳劈输出。P2 AOE时跑远→等完再上 / No lock-on → left-hand side → hit front legs. Jump attacks for damage. Run from P2 AoEs |

### 双月骑士蕾拉娜 / Rellana, Twin Moon Knight

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-50%)** | 双持魔法剑——卡利亚剑术(快慢刀连击) + 月魔法投掷 / Dual magic swords — Carian swordplay (delayed combos) + moon magic throw | 快慢刀节奏：第一刀快→停顿→第二刀快→停顿→第三刀慢。目押第三刀翻滚 / Rhythm: fast → pause → fast → pause → slow. Roll on 3rd |
| **P2 (50%-0%)** | 追加：双月斩(大范围月弧x2)+月光追踪弹 / Adds: Twin Moon slash (wide arc x2) + homing moon projectiles | 双月斩向她的**后方**翻滚(不要向侧面)。追踪弹不停跑位——不要停 / Roll **behind** her for Twin Moon (not sideways). Keep moving for homing projectiles |
| **弱点 / Weakness** | 魔法抗性低——她自己是魔法师 / Low magic resist — she's a mage | 用魔法打她伤害可观。但需要小心她的魔法追踪弹 / Magic deals solid damage. Watch for her magic projectiles |
| **推荐打法 / Strategy** | 盾刺流或居合流——防守反击是最安全的方式。P2拉开距离等她出完双月斩再打 / Guard counter or Iaijutsu — safest approach. In P2, wait out Twin Moon before attacking |

### 穿刺者梅瑟莫 / Messmer the Impaler

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-50%)** | 火矛刺击(快)+火矛横扫(慢)+暗影突击(瞬移) / Fire spear thrust (fast) + sweep (slow) + shadow rush (teleport) | 所有攻击都有前摇特效——看清是刺还是扫。刺→向左滚。扫→向右滚 / All attacks have visual telegraphed effects. Thrust → roll left. Sweep → roll right |
| **P2 (50%-0%)** | 追加：蛇形火矛(自动追踪)+全屏火焰AOE(起跳后砸地)+蛇群召唤 / Adds: homing snake spear + full-screen fire AoE (jump slam) + snake summon | 火焰AOE时跑到**最边缘**——中心范围最大→越远越好。蛇群出现时优先闪避 / Run to **edge** for fire AoE — center has widest radius. Prioritize dodging snakes |
| **弱点 / Weakness** | 雷属性 / Lightning | 雷油脂附魔武器——伤害x1.3 / Lightning grease — 1.3x damage |
| **推荐打法 / Strategy** | 大盾+刺剑——举盾-> poke -> 举盾。梅瑟莫的攻击打不动大盾(需要65+强韧度) / Greatshield + thrusting sword — hold block → poke → repeat. Messmer can't break greatshields (65+ guard boost) |

## 隐藏Boss / Optional Bosses

### 癫狂之王米德拉 / Midra, Lord of Frenzied Flame

| 要素 / Factor | 说明 / Description |
|:----:|------|
| **位置 / Location** | 艾尼尔·伊利姆→地下→癫狂教堂 / Enir-Ilim → underground → Church of Frenzy |
| **进入条件 / Access** | 击败梅瑟莫后→探索教堂地下的隐藏通道 / After Messmer → explore hidden passage under church |
| **攻击类型 / Attack Type** | 狂暴火焰——所有攻击附带癫狂积累 / Frenzied flame — all attacks build Frenzy buildup |
| **核心机制 / Mechanic** | 战斗中有大量癫狂爆发——需要**清醒干肝**清癫狂 / Heavy Frenzy buildup during fight — need **Clarifying Boluses** to clear |
| **打法 / Strategy** | 保持中距离——引诱他出大硬直招式(蓄力突刺/砸地)→翻滚到背后输出 / Keep mid-range — bait his big recovery moves (charge thrust/slam) → roll behind → DPS |
| **奖励 / Reward** | 癫狂之王大剑 + 癫狂火焰祷告 / Midra's Greatsword + Frenzied Flame incantation |

### 指头之母 / Metyr, Mother of Fingers

| 要素 / Factor | 说明 / Description |
|:----:|------|
| **位置 / Location** | 指头遗迹→最底部 / Finger Ruins → bottom |
| **攻击类型 / Attack Type** | 重力魔法 + 召唤小指头 + 激光扫射 / Gravity magic + spawns small fingers + laser sweep |
| **打法 / Strategy** | 激光扫射时横向奔跑——不要往远处跑。小指头出来时先AE O清掉——否则被包围 / Run perpendicular to laser sweep — not away. Clear spawned fingers with AoE immediately to avoid being overwhelmed |
| **注意 / Note** | 战斗场地有大量水塘——移动速度降低——翻滚变慢。可以穿提升水塘移动速度的装备 / Arena has water pools — reduced movement speed and slower rolls. Equip gear that boosts mobility in water |
| **奖励 / Reward** | 指头之母追忆 + 重力魔法 / Remembrance + Gravity sorcery |

## 最终Boss / Final Boss

### 约定之王拉塔恩 / Promised Consort Radahn

**全游戏最难Boss**——没有之一。二阶段的光污染和快慢刀组合让无数玩家崩溃 / **The hardest boss in any FromSoftware game** — bar none. Phase 2's visual clutter and delayed attack combos have broken countless players.

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对 / Counter |
|:----:|:----:|------|
| **P1 (100%-40%)** | 重力剑+光速连击——快慢刀+突进+AOE重力波 / Gravity sword + light-speed combos — delayed attacks + rush + AoE gravity wave | 所有快慢刀的共同点：**刀光闪了之后等0.3-0.5秒再翻滚** / All delayed attacks share one tell: **roll 0.3-0.5s after the blade flashes** |
| **P2 (40%-0%)** | 追加：米凯拉附身——全屏光污染AOE+光之矛追踪+分身连击 + 1/3血量时全屏秒杀(米凯拉擒抱) / Adds: Miquella possession — full-screen light AoE + homing holy spears + clone combos + 1/3 HP grab one-shot | 光污染时**不要锁定**——不锁定才能看清动作。分身连击——往边缘跑然后等最后一下翻滚 / **Don't lock on** during light show. Clone combo → run to edge → roll on final hit |
| **逃课打法 / Cheese** | **抗拒刺+大盾**——二人转打法。或者血王叉(蒙格温圣矛)+仿生泪滴。但最稳的还是练好翻滚时机 / **Antspur Rapier + Greatshield** — poke and strafe. Or Mohgwyn's Sacred Spear + Mimic Tear. But best is learning roll timing |
| **推荐装备 / Gear** | 圣抗护符(Holyproof Dried Liver + Haligdrake Talisman +3) + 大盾 + 刺剑 / Holy damage protection + Greatshield + thrusting sword |
| **幽影树碎片建议 / Blessing** | 至少Lv 15+ / At least Lv 15+ |
