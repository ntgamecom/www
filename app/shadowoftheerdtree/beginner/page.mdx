# 新手入门 / Beginner Guide

黄金树之影的难度远高于本体——即使你150级进DLC，也可能被第一只小怪秒杀。理解DLC独有的机制是活下去的关键 / Shadow of the Erdtree is significantly harder than the base game — even at level 150, the first enemy can one-shot you. Understanding DLC-exclusive mechanics is key to survival.

---

## 幽影树庇护系统 / Scadutree Blessing System

**这是DLC最重要的机制**——没有幽影树庇护，你的人物在DLC区域攻防严重不足 / **This is the most important DLC mechanic** — without Scadutree Blessing, your character's ATK and DEF in DLC areas are severely lacking.

| 等级 / Level | 碎片数量 / Fragments | 攻击力加成 / ATK Multiplier | 减伤率 / DMG Negation |
|:----:|:----:|:----:|:----:|
| Lv 1 | 0 (初始) | 1.00x | 0% |
| Lv 5 | 15 | 1.30x | 15% |
| Lv 10 | 31 | 1.60x | 25% |
| Lv 15 | 47 | 2.00x | 35% |
| Lv 20 | 60 | 2.50x | 45% |

**关键结论**: 收集幽影树碎片比升100级还重要。探索地图>刷级 / **Key takeaway**: Collecting Scadutree Fragments is more important than gaining 100 levels. Explore > grind.

**碎片大致分布**: 每个区域5-8个——部分在明显位置(教堂/米凯拉十字架)，部分在隐藏路径里 / **Distribution**: ~5-8 per zone — some obvious (churches/Miquella Crosses), some in secret paths.

---

## 灵灰庇护系统 / Revered Spirit Ash Blessing

| 等级 / Level | 灵灰数量 / Ashes | 骨灰和灵马加成 / Summon & Torrent Bonus |
|:----:|:----:|:----:|
| Lv 1 | 0 | +0% HP/DMG |
| Lv 5 | 8 | +20% HP/DMG |
| Lv 10 | 16 | +40% HP/DMG |
| Lv 15 | 25 | +60% HP/DMG |

**如果依赖骨灰**: 优先收集灵灰——骨灰在DLC中很容易被秒杀——灵灰可以大幅提升骨灰的生存能力 / **If you rely on summons**: Prioritize Revered Ashes — summons get one-shot in DLC without them.

## 开局路线推荐 / Recommended Starting Route

| 步骤 / Step | 位置 / Location | 操作 / Action |
|:----:|:----|------|
| **1** | 墓地平原(初始区域) / Gravesite Plain | 不要直接冲向第一个Boss——先往**东南方向**探索→收集3-4个幽影树碎片 / Don't rush the first boss — explore **southeast** first → collect 3-4 Scadutree Fragments |
| **2** | 三叉口十字标记 / Three-Path Cross | 这里是第一个米凯拉十字架——互动获得幽影树碎片。同时解锁地图导航提示 / First Miquella's Cross — interact for Scadutree Fragment. Unlocks map navigation hints |
| **3** | 塔之镇 / Belurat, Tower Settlement | 从墓地平原向西→到达塔之镇——神兽舞狮在镇顶 / Gravesite Plain west → Belurat — Divine Beast Dancing Lion at top |
| **4** | 恩希斯城 / Castle Ensis | 击败舞狮后向北→恩希斯城→双月骑士 / After Dancing Lion → north to Castle Ensis → Rellana |
| **5** | 幽影城 / Shadow Keep | 恩希斯城后东北方向→幽影城→穿刺者梅瑟莫 / After Ensis → northeast to Shadow Keep → Messmer |

**新手建议**: 进DLC后前2小时**不要打Boss**——专注于收集碎片+探索地图。碎片等级到6-8后再开始打Boss / **Beginner tip**: First 2 hours in DLC **don't fight bosses** — focus on fragments + exploration. Start bossing at blessing level 6-8.

## DLC vs 本体差异 / DLC vs Base Game Differences

| 差异 / Difference | 说明 / Description |
|:----:|------|
| **敌人伤害翻倍 / Enemy DMG doubled** | DLC敌人的基础伤害约是本体的1.5-2倍——即使幽影树碎片满级也高于本体 / DLC enemy base damage is ~1.5-2x base game — even at max blessing it's harder |
| **快慢刀泛滥 / Delayed attacks everywhere** | DLC的Boss和小怪都有大量快慢刀——目押时机需要重新适应 / Bosses and mobs all use delayed attacks — timing requires re-learning |
| **视野/Boss体积 / Visual clarity issues** | 部分Boss(尤其是拉塔恩)的粒子效果极多——看不清动作。拉开距离看更清楚 / Some bosses (especially Radahn) have extreme particle effects — fight from distance for better visibility |
| **骨灰削弱 / Summons nerfed** | 骨灰在DLC中仇恨值降低——Boss会频繁绕过骨灰打你。但灵灰强化后有好转 / Summons have reduced aggro in DLC — bosses bypass them frequently. Improved with Revered Ashes |
| **地图没有小地图 / No mini-map** | DLC地图没有右下角小地图——完全靠环境辨别方向 + 米凯拉十字架指向目标方向 / No mini-map — navigate purely by environment + Miquella Crosses point toward objectives |

## 新手常见错误 / Common Beginner Mistakes

| 错误 / Mistake | 后果 / Consequence | 正确做法 / Correct |
|:----:|:----:|------|
| **不打舞狮直接冲 / Rush past Dancing Lion** | 错失了前期关键的幽影树碎片+地图碎片——后面更难打 / Miss early Scadutree Fragments + map pieces — everything gets harder | 舞狮是第一个门槛——打过它再往前。它的攻击有节奏可循 / Dancing Lion is the first gate — beat it before moving on. Its attacks have rhythm |
| **忽略灵灰 / Ignoring Revered Ashes** | 骨灰被Boss一刀秒——等于没有召唤 / Summon gets one-shot — useless | 灵灰和幽影树碎片同样重要——两个都要收集 / Both are equally important — collect both |
| **用本体武器硬打 / Using base weapons only** | DLC新武器面板高+有专属战技——不换新武器=少30%伤害 / New DLC weapons have higher stats + unique skills — not swapping = 30% less DPS | 至少试试反手剑和轻大剑——战技强度很高 / At least try Backhand Blade and Light Greatsword — their skills are very strong |
| **等级歧视 / Level gatekeeping** | 觉得自己200级了DLC随便打——然后被小怪两刀秒 / Thinking level 200 = easy mode — then getting two-shot by a mob | 幽影树碎片等级决定你的DLC强度——不是等级 / Scadutree Blessing level determines your DLC strength — not your character level |
