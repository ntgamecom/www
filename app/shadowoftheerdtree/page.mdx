# 艾尔登法环DLC：黄金树之影攻略 / Elden Ring: Shadow of the Erdtree Guide

《黄金树之影》是艾尔登法环的大型DLC，带来全新的"幽影之地"——面积相当于游戏本体1.5倍的巨大地图。包含10+新Boss、100+新武器、8个新魔法学派和全新的"幽影树碎片"成长系统。本攻略涵盖DLC全部内容 / Shadow of the Erdtree is Elden Ring's massive DLC, featuring the "Land of Shadow" — a map 1.5x the size of the base game. Includes 10+ new bosses, 100+ new weapons, 8 new magic schools, and the new "Scadutree Fragment" leveling system. This guide covers all DLC content.

## DLC核心机制 / Core DLC Mechanics

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **幽影树碎片 / Scadutree Fragments** | DLC专属升级系统——收集碎片在赐福处强化→提升攻击力和减伤率(仅限DLC区域生效) / DLC-only leveling — collect fragments and bless at Grace → boosted ATK and damage negation (DLC area only) |
| **灵灰庇护 / Revered Spirit Ashes** | 类似幽影树碎片——专门强化骨灰和灵马——收集灵灰提升召唤物强度 / Similar to Scadutree — strengthens Spirit Ashes and Torrent — collect Revered Ashes for stronger summons |
| **幽影之地 / Land of Shadow** | 全新独立地图——没有小地图——靠环境和十字标记导航。地图碎片需要收集 / All-new map — no mini-map — navigate by environment and Cross markers. Map fragments are collectibles |
| **新武器类型 / New Weapon Types** | 反手剑(Backhand Blade)、轻大剑(Light Greatsword)、格斗术(Dryleaf Arts)、兽爪拳套(Beast Claw)、调香瓶(Smithscript)等8种新武器类型 / 8 new weapon categories |
| **新成长系统 / New Progression** | 幽影树庇护独立于角色等级——探索越多→越强——鼓励全面探索 / Scadutree Blessing is separate from character level — explore more = get stronger — encourages full exploration |

## DLC进入条件 / DLC Access Requirements

| 条件 / Requirement | 说明 / Description |
|:----:|------|
| **击败碎星拉塔恩 / Defeat Starscourge Radahn** | 盖利德(Redmane Castle)→Boss战 / Caelid (Redmane Castle) → boss fight |
| **击败蒙格 / Defeat Mohg, Lord of Blood** | 化圣雪原(Mohgwyn Palace)→蒙格温王朝 / Consecrated Snowfield (Mohgwyn Palace) |
| **到达蒙格温王朝 / Reach Mohgwyn Palace** | 通过化圣雪原传送门或梵雷任务线 / Via Consecrated Snowfield teleporter or Varre's questline |
| **触摸神人茧 / Touch the Cocoon** | 蒙格Boss房后方→神人茧→触摸→被传送至幽影之地 / Behind Mohg's boss room → cocoon → touch → teleported to Land of Shadow |

**建议进入等级**: 120-150级——装备至少+15~+20武器。DLC难度远高于本体后期区域 / **Recommended level**: 120-150 — weapons at least +15~+20. DLC is significantly harder than base game late areas.
