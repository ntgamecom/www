# 配装Build / Builds Guide

黄金树之影新增了8种武器类型和100+新武器，Build可能性大幅扩展。以下是DLC最强Build推荐 / Shadow of the Erdtree adds 8 new weapon types and 100+ new weapons, massively expanding build possibilities. Here are the strongest DLC builds.

---

## 反手剑出血流 / Backhand Blade Bleed Build

| 部位 / Slot | 装备 / Gear | 说明 / Notes |
|:----:|:----|:----|
| **主武器 / Main Weapon** | 反手剑(Backhand Blade) + 血质变 / Blood infusion | DLC开局即可获得——墓地平原→三叉口→捡尸体 / Available at DLC start — Gravesite Plain → Three-Path Cross → corpse pickup |
| **战技 / Ash of War** | 盲目突刺(Blind Spot) | 无敌帧+绕背攻击——DLC最强战技之一 / I-frames + backstab attack — one of the best AoWs in DLC |
| **护符 / Talismans** | 鲜血欢愉(Blood Exultation) + 米莉森的义手 + 腐败剑翼徽章 + 勾爪护符 | 出血流标准配置 / Standard bleed setup |
| **防具 / Armor** | 任何中甲——保持70%以下负重 / Any medium armor — keep under 70% equip load |
| **加点 / Stats** | 灵巧50>感应50>血量40>精力30 / DEX 50 > ARC 50 > VIT 40 > END 30 |

**打法**: 盲目突刺绕背→打出血→触发鲜血欢愉+攻击力上升→重复。出血伤害在DLC中依然统治级 / **Playstyle**: Blind Spot behind → trigger bleed → Blood Exultation proc + ATK up → repeat. Bleed is still king in DLC.

| 部位 / Slot | 装备 / Gear | 说明 / Notes |
|:----:|:----|:----|
| **主武器 / Main Weapon** | 轻大剑(Light Greatsword) + 优质质变 / Quality infusion | 双月骑士掉落武器——拥有独特的"待机->斩击"战技 / Drops from Rellana — unique "stance → slash" skill |
| **战技 / Ash of War** | 待机月斩(Rellana's Twin Moons) | 双月骑士同名技能——双月弧光+高削韧 / Rellana's signature — twin moon arcs + high stagger |
| **护符 / Talismans** | 绿龟(Dragoncrest Shield) + 大壶众武 + 魔力龙徽 + 处决者 | 防御反击核心配置 / Guard counter core setup |

**打法**: 举盾→等Boss攻击→格挡→立刻轻击(防反)→后退恢复精力→重复。轻大剑防反距离远+伤害高 / **Playstyle**: Hold shield → block boss attack → light attack (guard counter) → back off → repeat. Light Greatsword guard counter has long range + high damage.

## 法师Build / Mage Build

DLC新增多个强力法术——尤其是**辉石魔砾**的进阶版本和**重力魔法** / DLC adds many powerful spells — especially advanced versions of Glintstone Pebble and Gravity magic.

| 部位 / Slot | 装备 / Gear | 说明 / Notes |
|:----:|:----|:----|
| **主武器 / Main Weapon** | 母亲之杖(Mother's Staff) [指头之母掉落] | 强化重力魔法+召唤魔法 / Boosts gravity + summon sorceries |
| **法术 / Spells** | 辉石流星(Glintstone Meteor) + 重力旋涡(Gravity Vortex) + 指头召唤(Call of Fingers) | DLC新增强力法术 / New powerful DLC spells |
| **护符 / Talismans** | 魔法师块(Magic Scorpion) + 葛孚雷 + 拉达冈肖像 + 魔力龙徽 | 标准法师配置 / Standard mage setup |
| **防具 / Armor** | 轻甲/中甲——保持低负重提高翻滚距离 / Light/medium armor — low equip load for better roll distance |
| **加点 / Stats** | 智力80>血量40>精力25>灵巧20 / INT 80 > VIT 40 > END 25 > DEX 20 |

**打法**: 距离=生命。辉石流星→追踪+高伤。重力漩涡→聚怪+控场。保持距离·持续风筝 / **Playstyle**: Distance = life. Glintstone Meteor → homing + high dmg. Gravity Vortex → group + control. Keep distance, kite constantly.

## 幽影树庇护重要提醒 / Scadutree Blessing Reminder

**幽影树庇护是DLC最重要的数值系统**——本体+25级在DLC里没有庇护=被两刀砍死。优先把庇护升到至少15级再去打梅瑟莫 / **Scadutree Blessing is the most important progression system in the DLC** — base game level 25 without blessing = dead in 2 hits. Prioritize raising blessing to at least 15 before fighting Messmer.

| 庇护等级 / Level | 攻击倍率 / ATK Multi | 减伤率 / DMG Negation |
|:----:|:----:|:----:|
| 0 | 1.0x | 0% (两刀死 / dead in 2 hits) |
| 5 | 1.3x | 20% |
| 10 | 1.6x | 35% |
| 15 | 1.9x | 45% |
| 20 | 2.2x | 55% |
| 满级(25) / Max | 2.5x | 60% |
