## 隐藏Boss / Hidden Bosses

DLC中有几个**完全没有标记在地图上**的隐藏Boss——需要特殊触发条件 / DLC has several hidden bosses with **no map marker** — requiring special triggers.

### 古龙塞努萨克斯 / Ancient Dragon Senusax

**位置**: 尖刺山——但不会在第一次到达时出现 / Location: Jagged Peak — not present on first arrival
**触发条件**: 在尖刺山击杀所有小龙(约12只)——然后回到尖刺山山顶——古龙会从天而降 / Trigger: kill all small dragons on Jagged Peak (~12) — return to summit — dragon descends from sky
**等级建议**: 170+ / Level: 170+
**奖励**: 古龙岩锻造石×3——龙鳞碎片×5——塞努萨克斯的龙心脏(新龙飨祷告) / Rewards: Ancient Dragon Stone ×3 — Dragon Scale ×5 — Senusax's Heart (new Dragon Communion incantation)
**打法**: 弱圣属性——推荐用圣律剑刃战技——打头有额外伤害。二阶段飞行时用猎犬步法躲避空中雷击——落地后输出 / Weak to holy — use Sacred Blade AoW — bonus damage on head. Phase 2: Bloodhound Step for aerial lightning — punish on landing.

### 隐形的刺客 / Invisible Black Knife Assassin

**位置**: 幽影亚坛——一个不起眼的废墟中——需要走过特定路线才能触发 / Location: Scadu Altus — unremarkable ruin — needs specific walking path
**触发条件**: 在废墟中保持不锁定——走对角线路径——在第三个柱子旁边停下——等待约10秒——刺客出现 / Trigger: unlock camera — walk diagonal path — stop at third pillar — wait ~10s — assassin appears
**奖励**: 黑刀护甲套装(轻装) + 黑刀的匕首 / Rewards: Black Knife Armor Set (light) + Black Knife Dagger
**打法**: 他会隐身——听脚步声判断位置——用**哨兵火把**或**猎犬步法**反隐——当他现身时用冲击力高的武器打断 / Invisible — listen for footsteps — use **Sentry Torch** or **Bloodhound Step** to reveal — interrupt with high poise weapon when visible.

### 米凯拉的梦魇 / Miquella's Nightmare

**位置**: 谷底森林的最深处——需要通过一个隐藏墙壁 / Location: Deepest Abyssal Woods — through hidden wall
**触发条件**: 在谷底森林找到5个隐藏的梦境碎片(发光的地面)——全部收集后回到中央树根——隐藏墙打开 / Trigger: find 5 hidden dream shards (glowing ground) in Abyssal Woods — return to central root — hidden wall opens
**奖励**: 米凯拉的梦境记忆——完整剧情片段——以及"梦魇大剑"——全游戏唯一带催眠效果的武器 / Rewards: Miquella's Dream Memory — full story scene — and "Nightmare Greatsword" — only weapon with sleep effect
**打法**: 梦魇是一个谜题Boss——不直接攻击——而是创造幻象——你需要找到真正的米凯拉幻影才能造成伤害——其他幻象会浪费你的资源 / Puzzle boss — doesn't attack directly — creates illusions — find real Miquella illusion to deal damage — others waste resources.
