## Buff叠加指南 / Buff Stacking Guide

DLC中可同时叠加多个Buff——了解叠加规则对高难度战斗至关重要 / Multiple buffs can stack in DLC — understanding stacking rules is crucial for hard fights.

### Buff叠加规则 / Stacking Rules

| 类别 / Category | 叠加 / Stack | 例子 / Example |
|---|---|---|
| 所有 Buff(同类不叠) / Body Buffs | 同类别最后一个生效 / Last cast wins | 黄金树立誓 + 火焰庇护 = 2个叠加(不同类) / Golden Vow + Flame Protect = 2 buffs (different types) |
| 武器附魔 / Weapon Buffs | 互斥——最后一个生效 / Mutually exclusive — last cast wins | 神圣之刃 + 火焰附魔油脂 = 后者覆盖前者 / Sacred Blade + Fire Grease = grease overwrites |
| 灵药 / Physick | 独立——和所有Buff叠加 / Independent — stacks with all | 灵药 + 黄金树立誓 + 火焰庇护 = 3重Buff / Physick + Golden Vow + Flame Protect = triple stack |
| 护符被动 / Talisman Passives | 全部叠加——无冲突 / All stack — no conflict | 龙徽大盾 + 珍珠龙辉 + 圣龙徽 = 全减伤 / All defense talismans = all stack |
| 装备效果 / Armor Effects | 特定套装效果独立叠加 / Specific set bonuses independent | 蘑菇王冠(中毒+攻击)可以和任何Buff共存 / Mushroom Crown works with all |

### 推荐Buff组合 / Recommended Buff Combos

**通用输出Buff**: 黄金树立誓(+15%所有伤害) → 火焰庇护(+20%减伤) → 灵药(+各类属性) → 武器附魔油脂。**Boss战极限Buff**: 黄金树立誓 → 火焰庇护 → 灵药(加攻击力+减伤) → 决心战技 → 砍——这一套可以让你第一刀打出约5000伤害 / **All-purpose ATK**: Golden Vow (+15% all dmg) → Flame Protect (+20% DEF) → Physick (dmg/DEF) → weapon grease. **Boss burst**: Golden Vow → Flame Protect → Physick (ATK+DEF) → Determination AoW → attack — first hit deals ~5000 damage.

## 属性补正变化 / Stat Scaling Changes

DLC调整了部分属性的软上限——最显著的是**感应**的出血累积效率上限提升了——从45提升到了60。**信仰**Build在DLC中大幅受益——新增大量信仰加成装备。**智力**Build在DLC中相对被削弱——很多Boss有魔法抗性——但仍然可用。**力量**和**敏捷**Build的补正率没有变化——但新武器的基础伤害更高——所以总伤害提升 / DLC adjusts some stat soft caps — most notably: **Arcane** bleed buildup cap raised from 45 to 60. **Faith** builds benefit most — many new faith-scaling items. **INT** builds relatively weaker — many bosses have magic resist — still viable. **STR** and **DEX** scaling unchanged — but new weapons have higher base damage — so total DPS is up.

## 进阶战斗技巧 / Advanced Combat Techniques

**战技取消**: 部分战技有后摇——可以通过翻滚取消——提前开始输出。死角一击的后摇可以用翻滚取消约0.3秒——让你更快接续攻击。**跳跃躲下段**: DLC很多Boss有下段横扫——跳起来可以完全回避——比翻滚更好——跳起来还能接跳劈反击。**盾牌反击时机**: 大盾防御反击的最佳时机是Boss连招的最后一下——挡完立刻重击——大部分敌人会被击退。**交叉步法**: 连续向同一方向翻滚会被Boss的追踪攻击命中——交替向左/右翻滚可以迷惑Boss的追踪AI / **AoW Cancel**: some skills have recovery that can be roll-canceled — start next attack ~0.3s faster. Blind Spot is very roll-cancel-friendly. **Jump Low Sweeps**: many DLC bosses have low sweeps — jumping completely avoids them — better than rolling — follow with jump attack counter. **Guard Counter Timing**: best after boss combo's last hit — block then immediate heavy — most enemies stagger. **Cross-Step**: rolling same direction repeatedly gets caught by tracking attacks — alternate left/right rolls to confuse tracking AI.
