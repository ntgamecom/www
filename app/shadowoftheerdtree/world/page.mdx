# 幽影之地探索 / World Exploration

幽影之地是黄金树之影的舞台——面积约为宁姆格福+利耶尼亚+盖利德的总和。地图没有传统小地图，完全依靠环境和米凯拉的十字标记导航 / The Land of Shadow is the stage of Shadow of the Erdtree — roughly the size of Limgrave + Liurnia + Caelid combined. There is no traditional mini-map; navigation relies entirely on the environment and Miquella's Cross markers.

## 米凯拉十字标记系统 / Miquella's Cross System

**这是幽影之地的核心导航方式**——米凯拉在旅途中留下了多个十字标记。每个十字标记指向下一个目标方向 / **This is the core navigation system** — Miquella left Cross markers along his journey. Each Cross points toward the next objective.

| 十字 / Cross | 位置 / Location | 指向方向 / Points Toward |
|:----:|:----|:----:|
| **三叉口十字** | 墓地平原·三叉口 / Gravesite Plain | 指向塔之镇方向 / Toward Belurat |
| **塔之镇十字** | 塔之镇·入口 / Belurat entrance | 指向恩希斯城 / Toward Castle Ensis |
| **恩希斯十字** | 恩希斯城·城门口 / Castle Ensis gate | 指向幽影城 / Toward Shadow Keep |
| **幽影城十字** | 幽影城·教堂 / Shadow Keep church | 指向艾尼尔·伊利姆 / Toward Enir-Ilim |
| **艾尼尔十字** | 艾尼尔·伊利姆·入口 / Enir-Ilim entrance | 指向最终Boss / Toward final boss |

**每个十字标记互动后获得一个幽影树碎片**——所以每个十字都值得去 / **Each Cross grants a Scadutree Fragment when interacted with** — so visit every Cross.

## 探索技巧 / Exploration Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **跟着乌鸦走 / Follow the birds** | 幽影之地的乌鸦(大乌鸦)通常会带你找到隐藏物品/路径 / Big birds in Land of Shadow often lead to hidden items/paths |
| **注意地面发光 / Watch for glowing ground** | 幽影树碎片通常在小黄金树附近的地面发光处——或者教堂内的小雕像前 / Scadutree Fragments are often near glowing ground by minor Erdtrees or in churches |
| **悬崖边缘看下面 / Check cliff edges** | 大量隐藏路径需要从看似是死路的悬崖跳下——下方90%有平台 / Many secret paths require jumping off seemingly fatal cliffs — 90% have a platform below |
| **米凯拉十字都是有用的 / Every Cross matters** | 不仅给碎片——十字指向的方向通常是下一个重要目标的方向 / Not just for fragments — Crosses point toward the next important objective |
| **灵马二段跳找新路 / Torrent double-jump** | 幽影之地的地形设计充分利用灵马的二段跳——看到高台就试试能不能跳上去 / Land of Shadow terrain is designed for Torrent's double-jump — try jumping to every elevated platform |
