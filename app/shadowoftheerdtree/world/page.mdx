## 幽影之地总览 / Land of Shadow Overview

幽影之地是艾尔登法环黄金树幽影DLC的全新大陆——位于交界地之外——通过蒙格温王朝的米凯拉之茧进入。地图面积约为本体宁姆格福+利耶尼亚的总和——垂直设计更复杂——分多层地形——大量隐藏洞穴和地下区域 / The Land of Shadow is the new continent in Elden Ring's Shadow of the Erdtree DLC — outside the Lands Between — entered via Miquella's Cocoon in Mohgwyn Palace. Map size is roughly Limgrave + Liurnia combined — with more complex vertical design — multi-layer terrain — many hidden caves and underground areas.

## 地图碎片位置 / Map Fragment Locations

和本体一样——幽影之地的地图需要找到地图碎片才能解锁。共4个地图碎片——完整展开后覆盖全部DLC区域 / Like base game — map fragments needed to reveal the full map. 4 fragments total — full reveal covers all DLC areas.

### 碎片1：墓地平原 / Fragment 1: Gravesite Plain

**位置**: 从蒙格温王朝进入幽影之地后——直走——在第一个岔路口右转——地图碎片在石碑下方 / Location: After entering from Mohgwyn — go straight — first fork turn right — fragment at base of obelisk
**解锁区域**: 墓地平原全图——塔之镇贝瑞特周边——恩希斯城入口 / Reveals: Gravesite Plain full — Belurat area — Castle Ensis entrance

### 碎片2：幽影亚坛 / Fragment 2: Scadu Altus

**位置**: 恩希斯城出口——过桥后右转——在一个高台上的石碑处 / Location: After Castle Ensis exit — cross bridge — turn right — on high platform at obelisk
**解锁区域**: 幽影亚坛全图——暗影要塞入口——谷底森林入口提示 / Reveals: Scadu Altus full — Shadow Keep entrance — Abyssal Woods entry hint

### 碎片3：谷底森林 / Fragment 3: Abyssal Woods

**位置**: 谷底森林入口附近——在一条隐藏小路尽头的树根下 / Location: Near Abyssal Woods entrance — at end of hidden path under tree root
**解锁区域**: 谷底森林全图——尖刺山入口提示 / Reveals: Abyssal Woods full — Jagged Peak entry hint

### 碎片4：尖刺山 / Fragment 4: Jagged Peak

**位置**: 尖刺山山脚——在龙族废墟中的石碑处 / Location: Jagged Peak base — obelisk inside dragon ruins
**解锁区域**: 尖刺山全图——艾尼尔-伊利姆入口提示 / Reveals: Jagged Peak full — Enir-Ilim entry hint

## 重要地点 / Important Locations

### 墓地平原 / Gravesite Plain

DLC的起始区域——广阔的荒野——散布着废墟、地下墓地和小型营地。**主要地标**: **岔路教堂**(第一个篝火——商人)、**水边废墟**(隐藏地下入口)、**角人村落**(断壁残垣——有角人NPC)、**墓地平原地下墓穴**(小型地下城——获得第一个大盾护符)。这里也是收集幽影树碎片的主要区域——至少有15个碎片可以在此区域找到 / Starting area — vast wilderness — scattered ruins, catacombs, small camps. **Key landmarks**: **Fork Church** (first bonfire — merchant), **Waterside Ruins** (hidden underground entrance), **Hornsent Village** (ruins — Hornsent NPC), **Gravesite Catacombs** (small dungeon — first Greatshield talisman). Main Scadutree Fragment area — at least 15 fragments here.

### 幽影亚坛 / Scadu Altus

DLC最大的区域——一个巨大的高原——包含多个子区域。**东部**: 腐烂湖——腐败状态危险但藏有强力腐败装备。**西部**: 龙族牧场——多只龙型敌人游荡——但也是古龙岩的主要来源。**中部**: 米凯拉十字路线——一条由十字标记指引的道路——通往暗影要塞。**南部**: 角人王城遗址——探索价值高——有隐藏Boss。幽影亚坛是整个DLC的核心——大部分支线任务和收集品都在这里 / The largest region — a giant plateau with multiple sub-areas. **East**: Rot Lake — dangerous rot but has powerful rot gear. **West**: Dragon Pasture — many dragon enemies — main source of Ancient Dragon Stones. **Central**: Miquella Cross Route — path marked by crosses — leads to Shadow Keep. **South**: Hornsent City Ruins — high exploration value — hidden boss. Scadu Altus is the DLC's core — most side quests and collectibles here.

### 谷底森林 / Abyssal Woods

一个被黑暗笼罩的森林——能见度极低——还有特殊的**视线干扰**机制——走远了会看不到路。这里被癫火感染者占据——战斗有持续的癫火累积——非常危险但奖励丰厚。**癫火抗体护符**和**哨兵火把**是探索的必要装备——前者降低癫火累积速度——后者增加能见度 / A dark-shrouded forest — very low visibility — special **view interference** — can't see far paths. Inhabited by frenzy-infected enemies — constant madness buildup during combat — dangerous but rewarding. **Madness Resist Charm** and **Sentry Torch** are essential — first reduces madness buildup — second increases visibility.

### 尖刺山 / Jagged Peak

一座布满龙族遗迹的山峰——山顶有古龙Boss战——沿途需要和大量小龙战斗。这里的龙族敌人掉落龙心脏和龙鳞碎片——是龙飨Build玩家的天堂。建议先清理全部小龙再挑战古龙——否则会被围殴 / A mountain covered in dragon ruins — ancient dragon boss at summit — many small dragons on the way. Dragon enemies here drop dragon hearts and scale fragments — paradise for Dragon Communion builds. Clear small dragons first before challenging the ancient dragon — or get ganked.

### 暗影要塞 / Shadow Keep

主线核心地下城——梅瑟莫的居所——多层结构——内有大量精英火焰骑士。**重要房间**: 梅瑟莫的议事厅(主线BOSS战前)、暗影要塞图书馆(大量道具+米凯拉剧情)、隐藏水道(通往梅瑟莫的私人房间)。这个地下城设计精巧——有多条路线——充分探索约2小时 / Main story core dungeon — Messmer's residence — multi-level — many elite Fire Knights. **Key rooms**: Messmer's Council Hall (pre-boss), Shadow Keep Library (many items + Miquella lore), Hidden Waterway (leads to Messmer's private chamber). Well-designed dungeon — multiple routes — ~2 hours full exploration.

## 世界设计特点 / World Design Features

| 特点 / Feature | 说明 / Description |
|---|---|
| 垂直探索 / Vertical Exploration | 幽影之地多层设计——悬崖、地下洞穴、楼顶——很多区域需要绕路上山或跳下 / Multi-layer design — cliffs, caves, rooftops — many areas require detours up or drops down |
| 隐藏墙壁 / Hidden Walls | 数量远超本体——几乎所有看起来可疑的墙壁都可以打碎 / Far more than base game — almost all suspicious walls can be broken |
| 幻影地面 / Illusory Floors | 少量幻影地板——踩上去掉到隐藏区域——注意观察地板的不同纹理 / A few illusory floors — fall to hidden areas — look for different texture |
| 毒池陷阱 / Poison Swamp | 有两个大型毒池区域——比本体毒池更危险——需要抗毒装备 / 2 large poison swamp areas — more dangerous than base game — need poison resist |
| 回归性传送 / Fall-back Teleports | 从高处跳下后会触发传送回高处的机制——反复探索同一区域 / Fall from height triggers teleport back up — revisit same area multiple times |
