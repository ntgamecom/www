## 终局概述 / Endgame Overview

**中文：** 对马岛之魂的终局内容包括完成主线结局、新游戏Plus（NG+）、奇谭模式（多人合作）和全收集。主线完成后可以继续自由探索，完成未完成的支线和收集品。NG+继承所有技能和装备，但敌人更强，同时新增了专属护身符和装备升级。

**English:** Ghost of Tsushima's endgame includes completing the main story endings, New Game+ (NG+), Legends mode (co-op multiplayer), and full completion. After the main story, free exploration continues for unfinished content. NG+ carries over all skills and gear with stronger enemies, exclusive charms, and gear upgrades.

| 终局内容 / Endgame Content | 说明 / Description |
|---|---|
| 多结局 / Multiple Endings | 最终选择影响结局 / Final choice affects ending |
| 新游戏Plus / New Game+ | 继承进度，更高难度，新装备 / Carry over progress, new gear |
| 奇谭模式 / Legends Mode | 2-4人合作PVE模式 / 2-4 player co-op PVE |
| 全收集 / Full Completion | 神社/传说动物/铠甲/荣誉之柱 / Shrines, animals, armor, pillars |

## 结局选择 / Ending Choices

**中文：** 对马岛之魂有两个主要结局，由最终任务"对马岛的命运"中的选择决定。

**English:** Ghost of Tsushima has two main endings, determined by your choice in the final quest "Fate of Tsushima".

**结局一：荣誉之死 / The Honorable Death**:

- 选择：给志村大人一个武士的荣誉死亡 / Give Lord Shimura an honorable samurai death
- 结果：仁杀死志村，继承志村铠甲 / Jin kills Shimura, inherits his armor
- 获得：志村的铠甲+护身符 / Shimura's Armor + Charm
- 结局动画：仁独自离开，成为真正的战鬼 / Jin leaves alone, becoming the true Ghost

**结局二：饶恕 / The Spare**:

- 选择：放过志村大人 / Spare Lord Shimura
- 结果：志村活下去但颜面尽失，仁放弃武士身份 / Shimura lives but loses honor, Jin abandons samurai identity
- 获得：仁的铠甲（白色战鬼铠甲）/ Jin's Armor (white Ghost Armor variant)
- 结局动画：仁和结奈一起离开，自由地活下去 / Jin leaves with Yuna, living freely

**选择的影响 / Impact of Choice**:
- 两个结局都有专属奖杯 / Each ending has a dedicated trophy
- 铠甲奖励不同，但属性强度相当 / Different armor rewards but comparable stats
- 不影响NG+继承内容 / No impact on NG+ carryover

## 新游戏Plus指南 / New Game+ Guide

**中文：** NG+在主线通关后解锁。以下是NG+的继承规则和新增内容。

**English:** NG+ unlocks after completing the main story. Inheritance rules and new content below.

**继承内容 / NG+ Carry Over**:

| 继承 / Carries Over | 不继承 / Does Not Carry Over |
|---|---|
| 所有技能和架势 / All skills and stances | 任务进度 / Quest progress |
| 所有铠甲和外观 / All armor and skins | 地图探索进度 / Map exploration |
| 所有武器和护身符 / All weapons and charms | 蒙古领土进度 / Mongol territory progress |
| 收集品进度 / Collectible progress | 部分剧情物品 / Some story items |
| 升级材料 / Upgrade materials | 资源节点状态 / Resource node states |

**NG+新增内容 / NG+ New Content**:
- 新增"仁之道"护身符（2倍伤害+2倍受伤）/ New "Path of Jin" Charm (2x damage, 2x damage taken)
- 所有铠甲增加第四级升级 / All armor gains a 4th upgrade tier
- 新的剑刃外观（NG+专属）/ New sword kit skins (NG+ exclusive)
- 新的鞍具外观 / New saddle skins
- 敌人配置变化，部分区域会出现更强敌人 / Enemy configuration changes with stronger foes

## 奇谭模式 / Legends Mode

**中文：** 奇谭模式是对马岛之魂的在线合作PVE模式。2-4名玩家组队，选择四种职业之一，挑战故事任务、生存任务和突袭任务。

**English:** Legends Mode is Ghost of Tsushima's online co-op PVE mode. 2-4 players team up, choose one of four classes, and tackle story missions, survival missions, and raids.

**四大职业 / Four Classes**:

| 职业 / Class | 角色定位 / Role | 特殊能力 / Special Ability |
|---|---|---|
| 武士 / Samurai | 近战坦克 / Melee Tank | 爆气斩——大范围近战攻击 / Explosive Blade — wide AoE melee |
| 猎人 / Hunter | 远程DPS / Ranged DPS | 瞄准之眼——延长子弹时间 / Eye of Aim — extended bullet time |
| 浪人 / Ronin | 治疗+宠物 / Healer + Pet | 治愈气息——范围治疗 / Healing Breath — AoE heal |
| 刺客 / Assassin | 潜行爆发 / Stealth Burst | 暗影打击——隐身突袭 / Shadow Strike — stealth attack |

**奇谭模式模式 / Legends Game Modes**:
- **故事任务 / Story Missions**: 双人合作剧情任务（共9关）/ 2-player co-op story (9 missions)
- **生存战 / Survival**: 四人防守据点 / 4-player wave defense
- **突袭 / Raid**: 四人高难挑战 / 4-player high-difficulty challenge
- ** rivals / 竞争**: 四人2v2竞速赛 / 4-player 2v2 speedrun

**奇谭模式终局 / Legends Endgame**:
- 装备等级上限110 / Gear score cap 110
- 梦魇难度每周挑战 / Nightmare difficulty weekly challenges
- 全职业满级奖励金色装备 / Max all classes for gold gear rewards

## 致命难度挑战 / Lethal Difficulty

**中文：** 致命难度是游戏最高难度，敌我双方都受到大幅伤害。这个难度下生存是第一要务。

**English:** Lethal is the highest difficulty — both player and enemies deal massively increased damage. Survival is the top priority.

**致命难度规则 / Lethal Rules**:
- 玩家受到伤害×4 / Player takes 4x damage
- 敌人受到伤害×2 / Enemies take 2x damage
- 完美格挡窗口不变 / Perfect parry window unchanged
- 敌人AI更激进 / Enemy AI is more aggressive

**致命难度生存技巧 / Lethal Survival Tips**:
- 昆恩护盾（指格挡）是最重要的生存技能——始终保持格挡姿态 / Blocking is the most important survival skill — always keep guard up
- 不要正面面对超过3个敌人 / Never face more than 3 enemies head-on
- 利用战鬼武器和暗杀减少正面战斗 / Use ghost weapons and assassinations to reduce direct confrontations
- 保持移动，不要在一个位置停留超过2秒 / Keep moving, never stay in one spot for more than 2 seconds
- 优先升级铠甲和护身符再推进主线 / Prioritize armor and charm upgrades before advancing main story

| 二级 / Tier 2 | 防御+30%，解锁外观 / Def +30%, unlocks look | 丝绸+钢 / Silk + Steel |

| 四级 / Tier 4 (NG+) | 防御+100%，最强外观 / Def +100%, best look | 荣誉之柱掉落材料 / Pillar of Honor materials |
