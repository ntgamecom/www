## 任务系统概述 / Quest System Overview

**中文：** 对马岛之魂的任务系统分为主线任务、传说故事（神话任务）、支线任务和角色任务线。主线任务推进对抗蒙古人的剧情，传说故事解锁强力技能和装备，角色任务线深化同伴的故事。支线任务提供经验值、资源和护身符。

**English:** Ghost of Tsushima's quest system includes main quests, mythic tales, side quests, and character questlines. Main quests advance the Mongol conflict story. Mythic tales unlock powerful skills and gear. Character questlines deepen companion stories. Side quests provide XP, resources, and charms.

| 任务类型 / Type | 说明 / Description |
|---|---|
| 主线 / Main Quests | 推进蒙古战争剧情 / Advance Mongol war story |
| 传说故事 / Mythic Tales | 解锁传说武器和技能 / Unlock legendary gear and skills |
| 角色任务线 / Character Tales | 同伴伙伴的个人故事 / Companion personal stories |
| 支线 / Side Quests | 额外经验和收集品 / Bonus XP and collectibles |

## 主线流程 / Main Quest Flow

**中文：** 主线分为三个主要幕：严原、丰玉和上县。

**English:** The main story has three acts: Izuhara, Toyotama, and Kamiagata.

**第一幕 / Act 1 — 严原 / Izuhara**:

| 任务 / Quest | 目标 / Goal | 核心事件 / Key Event |
|---|---|---|
| 对马岛之魂 / Ghost of Tsushima | 逃出对马城 / Escape Castle Tsushima | 开场，学习基础战斗 / Intro, learn basic combat |
| 境井的旅程 / Jin's Journey | 到达黄金寺 / Reach Golden Temple | 遇到百合子 / Meet Yuriko |
| 铁钩 / The Iron Hook | 找到铁匠 / Find the blacksmith | 解锁抓钩 / Unlock grappling hook |
| 汗王 / The Khan | 救出叔叔 / Rescue uncle | 第一次见到汗王 / First Khan encounter |
| 战鬼诞生 / The Ghost is Born | 击败龙三 / Defeat Ryuzo | 解锁战鬼架势 / Unlock Ghost Stance |

**第二幕 / Act 2 — 丰玉 / Toyotama**:

| 任务 / Quest | 目标 / Goal |
|---|---|
| 新天地 / A New Horizon | 进入丰玉地区 / Enter Toyotama region |
| 师父 / The Sensei | 寻找剑术师父 / Find the sword sensei |
| 盟友 / The Ally | 结盟仁佑架 / Ally with Yuna |
| 攻城的准备 / Preparations for the Attack | 集结盟友 / Gather allies |
| 对马城之战 / Battle of Castle Tsushima | 攻城战 / Castle siege |

**第三幕 / Act 3 — 上县 / Kamiagata**:

| 任务 / Quest | 目标 / Goal |
|---|---|
| 阴影之中 / In the Shadow | 进入上县 / Enter Kamiagata |
| 最后的武士 / The Last Samurai | 面对志村 / Face Shimura |
| 对马岛的命运 / Fate of Tsushima | 最终战汗王 / Final battle vs Khan |

## 传说故事 / Mythic Tales

**中文：** 传说故事是解锁最强技能和武器的关键任务。每个传说故事都讲述一个对马岛的古老传说。

**English:** Mythic Tales are key quests that unlock the strongest skills and weapons. Each tells an ancient Tsushima legend.

| 传说 / Tale | 奖励 / Reward | 解锁位置 / Location |
|---|---|---|
| 不屈之弓 / Unyielding Bow | 天之一弓（传说长弓）/ Heavenly Bow (legendary longbow) | 严原东部神社 / Izuhara east shrine |
| 水之面具 / Water's Mask | 流水架势（新剑术架势）/ Water Stance (new sword stance) | 严原南部海滩 / Izuhara south beach |
| 天之剑 / Heaven's Sword | 天之一刀（强力技能）/ Heaven's Strike (powerful technique) | 丰玉东部悬崖 / Toyotama east cliff |
| 雷霆鼓 / Thunder Drums | 雷电护身符 / Thunder Charm | 上县山顶 / Kamiagata peak |
| 鬼的传说 / The Ghost's Legend | 战鬼架势+战鬼铠甲 / Ghost Stance + Ghost Armor | 上县隐藏洞穴 / Kamiagata hidden cave |

**传说故事攻略 / Mythic Tale Tips**:
- 每个传说故事包含解谜和战斗两部分 / Each includes puzzle and combat segments
- 完成传说故事后获得的装备是游戏中最强的 / Gear from mythic tales is among the best in the game
- 建议在进入第二幕前完成至少两个传说故事 / Complete at least 2 mythic tales before Act 2

{/* ===== 角色任务线 / Character Tales ===== */}

## 角色任务线 / Character Tales

**中文：** 同伴角色的个人任务线。完成这些任务可以解锁同伴的独特能力和装备。

**English:** Companion character personal questlines. Completing these unlocks unique companion abilities and gear.

**结奈 / Yuna (盗贼)**:
- 任务链：结奈的故事（共5个）/ Yuna's Tale (5 quests)
- 核心任务："偷窃信念"——和结奈一起潜入蒙古营地 / Core: "Stealing Conviction" — infiltrate Mongol camp with Yuna
- 奖励：结奈的护身符（潜行移动速度+20%）/ Yuna's Charm (stealth speed +20%)

**石川师父 / Sensei Ishikawa (弓术)**:
- 任务链：石川的故事（共6个）/ Ishikawa's Tale (6 quests)
- 核心任务："师父与弟子"——面对石川的叛徒弟子 / Core: "Sensei and Student" — face Ishikawa's rogue student
- 奖励：石川的护身符（弓箭爆头伤害+30%）/ Ishikawa's Charm (bow headshot dmg +30%)

**政子夫人 / Lady Masako (复仇)**:
- 任务链：政子的故事（共8个）/ Masako's Tale (8 quests)
- 核心任务："复仇之终"——找到杀害政子家人的真凶 / Core: "End of Revenge" — find her family's true killer
- 奖励：政子的护身符（近战伤害+15%）/ Masako's Charm (melee dmg +15%)

**仁佑架 / Kenji (商人)**:
- 任务链：仁佑架的故事（共4个）/ Kenji's Tale (4 quests)
- 核心任务："仁佑架的救赎"——帮仁佑架解决债务问题 / Core: "Kenji's Redemption" — help Kenji with his debt
- 奖励：仁佑架的护身符（交易价格-20%）/ Kenji's Charm (vendor prices -20%)

**百合子 / Yuriko (药师)**:
- 任务链：百合子的故事（共3个）/ Yuriko's Tale (3 quests)
- 奖励：百合子的护身符（治疗效果+50%）/ Yuriko's Charm (healing effect +50%)

**中文：** 游戏中的关键选择影响剧情走向和结局。

| 最终Boss / Final Boss | 杀汗王/俘虏汗王 / Kill Khan/Capture Khan | 改变结局动画 / Changes ending cinematic |

## 蒙古领土解放 / Mongol Territory Liberation

**中文：** 每张地图上有多个蒙古领土据点，解放后可以清除该区域的蒙古人并解锁快速旅行点。

**English:** Each map has multiple Mongol territory outposts. Liberating them clears Mongol presence and unlocks fast travel points.

**解放奖励 / Liberation Rewards**:
- 每个据点解放：经验值+资源+赏金 / Per outpost: XP + resources + bounty
- 区域全解放：特殊护身符奖励 / Region full liberation: special charm reward
- 全地图解放：隐藏铠甲"对马之鬼" / Full map liberation: hidden Ghost Armor

**解放策略 / Liberation Strategy**:
- 先暗杀蒙古将领，减少敌人数目后强攻 / Assassinate Mongol leaders first, then assault
- 使用战鬼武器快速清理群怪 / Use ghost weapons for quick crowd clear
- 每个据点的蒙古将领掉落领土解放钥匙 / Each outpost's Mongol leader drops the liberation key

{/* ===== 传说故事解锁顺序 / Mythic Tale Order ===== */}

## 传说故事解锁顺序 / Mythic Tale Order

**中文：** 传说故事可以按任意顺序完成，但推荐按以下顺序解锁以获得平缓的难度曲线。

**English:** Mythic tales can be completed in any order, but this order provides a smoother difficulty curve.

1. **不屈之弓**（获得传说长弓，提升远程能力）/ Unyielding Bow: legendary longbow, ranged boost
2. **水之面具**（解锁流水架势，克制枪兵）/ Water's Mask: Water Stance, counters spearmen
3. **天之剑**（获得天之一刀，强力单体技能）/ Heaven's Sword: Heaven's Strike, powerful single-target
4. **雷霆鼓**（获得雷电护身符，范围攻击）/ Thunder Drums: Thunder Charm, AoE attack
5. **鬼的传说**（获得战鬼架势+战鬼铠甲）/ The Ghost's Legend: Ghost Stance + Ghost Armor
