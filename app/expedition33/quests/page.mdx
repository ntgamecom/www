## 主线任务流程 / Main Story Quests

33号远征队的主线分为**序章+三章+终章**——每章包含多个区域和主线任务。全程10-15小时——全收集20+小时 / Expedition 33's main story spans **Prologue + 3 Chapters + Finale** — each chapter has multiple zones and quests. Full run: 10-15 hours — 20+ hours for completionist.

## 角色专属任务 / Character Exclusive Quests

每个队友都有专属任务链——完成后解锁角色的**终极技能**和**专属装备**。**艾伦的专属**: 在光辉之城和贫民窟NPC对话触发——帮他找回被盗的家族弓——获得终极技"穿心箭"。**莉莎的专属**: 机械平原找到她导师留下的研究笔记——获得终极技"机械大军"。**塞拉的专属**: 深渊裂隙中发现她失落的魔法书——获得终极技"元素灾变" / Each ally has an exclusive quest chain — unlocks **ultimate skill** and **exclusive gear**. **Allen's Quest**: Radiant City slums NPC dialog — recover stolen family bow — unlocks "Heart Piercer" ult. **Lisa's Quest**: Find mentor's research notes in Machine Plains — unlocks "Mech Army" ult. **Sera's Quest**: Find lost spellbook in Abyss Rift — unlocks "Elemental Cataclysm" ult.

## 终局任务 / Endgame Quests

通关后解锁: **深渊裂隙深层**(10-20层)——每层有精英+Boss。**挑战竞技场**(10层)——限制条件下的战斗挑战。**光之记忆收集**——寻找所有隐藏记忆碎片解锁完整剧情。完成全部终局内容后获得**白金成就**和**隐藏武器外观"光之终焉"** / Post-game content: **Deep Abyss Rift** (floors 10-20) — elite + boss per floor. **Challenge Arena** (10 floors) — restricted battle challenges. **Light Memory Collection** — find all hidden memory fragments for full story. Completing all endgame content grants **Platinum Achievement** and **hidden weapon appearance "Light's End"**.

## 全章节主线流程 / Full Chapter Main Quest Walkthrough

### 第一章: 觉醒 / Chapter 1: Awakening

**开局** → 主角在废墟中醒来——跟随光之指引前进——学会基础移动和交互 / **Start**: Protag wakes in ruins — follow light guidance — learn basic move and interact.

**废墟探索** → B1层(普通敌人+3个宝箱) → B2层(精英怪+回声点1) → B3层(战争巨像Boss战)。B3层有两个分支: 左侧是Boss(主线)右侧是隐藏区域(回声点2+光之碎片1) / **Ruins exploration**: B1 (normal enemies+3 chests) → B2 (elite+Echo 1) → B3 (War Colossus boss). B3 has 2 branches: left=Boss (story) right=hidden area (Echo 2+Fragment 1).

**战争巨像Boss战**: 第一阶段→巨像使用手臂横扫和践踏——注意QTE防御。第二阶段(50%HP)→巨像眼睛发光——释放全屏激光——躲在柱子后面。弱点: 打眼睛造成额外伤害。击败后获得雷霆之心遗物 / **War Colossus**: Phase 1 → arm sweep+stomp — QTE guard. Phase 2 (50% HP) → eye glows — screen laser — hide behind pillars. Weakness: eye shots deal extra dmg. Drops Thunder Heart relic.

**前往城镇**: 击败巨像后从战场左侧门出去——到达第一章城镇——获得基础装备和第一个传送点 / **To town**: Left gate from arena — Ch.1 town — basic gear + first waypoint.

### 第二章: 扩张 / Chapter 2: Expansion

**港口到达**: 解锁艾伦和塞拉——队伍变为3人——从港口接取多个支线任务 / **Port arrival**: Unlock Allen+Sera — party of 3 — pick up side quests at port.

**火山区域**: 主要挑战区域——推荐等级20-25。火山上半段(普通怪+材料采集)→火山口(精英+Boss前存档点)→火山内部(火山领主的巢穴)。火山领主弱冰——推荐塞拉冰系技能主打 / **Volcano zone**: Main challenge — lvl 20-25 recommended. Upper slope (normal+mats) → crater (elite+save) → inner (Volcano Lord lair). Volcano Lord weak to ice — Sera ice skills.

**海岸探索**: 可选区域——有重要支线和隐藏区域——还有一只猫(猫之商人线索) / **Coast**: Optional zone — important side quest + hidden area + cat (Cat Merchant clue).

### 第三章: 真相 / Chapter 3: Truth

**学院区**: 情报中心——图书馆有大量技能书和隐藏通道 / **Academy**: Info hub — library has skill books + hidden passage.

**光之森林**: 收集光之碎片——光之守护者Boss——弱暗属性——推荐艾伦暗影Build / **Light Forest**: Light Fragments — Light Guardian boss — weak to dark — Allen shadow build.

**暗影森林**: 高难度区域——推荐等级32+——暗影领主——弱光——每次暗影爆发后记得驱散恐惧 / **Shadow Forest**: High difficulty — lvl 32+ — Shadow Lord — weak to light — cleanse fear after Shadow Burst.

### 第四章: 终焉 / Chapter 4: End

**生命神殿**: 试炼挑战——获得维克多和莉莎的终极武器 / **Life Temple**: Trial challenges — get Victor and Lisa ultimate weapons.

**深渊**: 最终区域——深渊之主最终战——三个阶段不断切换属性。最终阶段的全屏即死需要用光之碎片×5来抵挡 / **Abyss**: Final zone — Abyss Lord final battle — 3 phases with alternating elements. Final phase screen instant kill — block with Light Fragment x5.

**终局**: 通关后——回到各个区域收集遗漏——深渊裂隙——竞技场——全成就 / **Endgame**: Post-clear — return to zones for collectibles — Abyss Rift — Arena — all achievements.
