# 角色图鉴 / Characters Guide

33号远征队有6名可玩角色，每个角色有独特的技能树、装备和战斗定位。队伍最多3人，合理搭配是通关关键 / Clair Obscur: Expedition 33 has 6 playable characters, each with unique skill trees, equipment, and combat roles. Parties are limited to 3 — good composition is key to victory.

---

## 角色总览 / Character Overview

| 角色 / Character | 定位 / Role | 武器 / Weapon | 核心属性 / Core Stat | 难度 / Difficulty |
|:----:|:----:|:----:|:----:|:----:|
| **古斯塔夫 / Gustave** | 坦克+输出 / Tank + DPS | 巨剑 / Greatsword | 力量 / STR | ★★☆☆☆ |

| **泽农 / Zenon** | 刺客+爆发 / Assassin + Burst | 双匕首 / Dual Daggers | 敏捷 / DEX | ★★★★☆ |
| **维克多 / Victor** | 工程师+召唤 / Engineer + Summon | 扳手 / Wrench | 力量 / STR | ★★★☆☆ |

## 古斯塔夫 / Gustave

**远征队的领袖**——使用巨剑的前排战士。高HP+高物理伤害，操作简单适合新手 / **Expedition leader** — greatsword-wielding frontline fighter. High HP + high physical damage, simple to play.

| 技能 / Skill | AP消耗 | 效果 / Effect |
|:----:|:----:|------|
| **猛击 / Power Strike** | 2 | 单体物理伤害x1.8 + 50%概率击退 / Single target phys dmg x1.8 + 50% knockback |
| **旋风斩 / Whirlwind** | 3 | 全体物理伤害x1.2 / AoE phys dmg x1.2 |
| **战斗怒吼 / War Cry** | 2 | 全队攻击力+25%持续3回合 / Party ATK +25% for 3 turns |
| **终极技·碎山 / Ultimate: Mountain Breaker** | 4 | 全体物理伤害x3.0——蓄力一回合后释放 / AoE phys dmg x3.0 — charges for 1 turn |

**加点推荐**: 力量>体力>敏捷——纯物理站桩 / **Skill priority**: STR > VIT > DEX — pure physical tank.

---

## 玛艾勒 / Maelle

**远征队的医生**——唯一专职治疗。任何队伍都必带的角色 / **Expedition medic** — only dedicated healer. Must-have in every party.

| 技能 / Skill | AP消耗 | 效果 / Effect |
|:----:|:----:|------|
| **治愈术 / Heal** | 2 | 单体回复HPx2.0 / Single target heal x2.0 |
| **群体治愈 / Party Heal** | 3 | 全体回复HPx1.5 / Party heal x1.5 |
| **净化 / Purify** | 1 | 清除全队所有负面状态 / Remove all debuffs from party |
| **复活 / Revive** | 3 | 复活阵亡队友+50%HP / Revive fallen ally +50% HP |
| **终极技·生命之泉 / Ultimate: Life Spring** | 4 | 全队完全回复HP+全状态免疫3回合 / Full party full heal + all status immunity 3 turns |

**加点推荐**: 智力>体力>速度——治疗量=智力x2 / **Skill priority**: INT > VIT > SPD — heal amount = INT x2.

---

## 露娜 / Luna

**冰霜法师**——擅长控制和元素连携。冰+水连携=敌方跳过一回合，极强 / **Ice mage** — excels at control and elemental combos. Water+Ice combo = enemy skips a turn, extremely powerful.

| 技能 / Skill | AP消耗 | 效果 / Effect |
|:----:|:----:|------|
| **冰枪 / Ice Lance** | 2 | 单体冰霜伤害x1.6 + 40%冻结 / Single ice dmg x1.6 + 40% freeze |
| **暴风雪 / Blizzard** | 3 | 全体冰霜伤害x1.2 + 减速3回合 / AoE ice dmg x1.2 + slow 3 turns |
| **水幕 / Water Veil** | 2 | 给全队上"潮湿"状态→下个冰系技能必冻结 / Apply "Wet" to party → next ice skill guarantees freeze |
| **终极技·永冻 / Ultimate: Permafrost** | 4 | 全屏冰霜伤害x2.5 + 全体冻结2回合 / Full-screen ice dmg x2.5 + freeze all 2 turns |

**加点推荐**: 智力>速度>体力——先手冰冻控制 / **Skill priority**: INT > SPD > VIT — freeze control goes first.

---

## 泽农 / Zenon

**暗影刺客**——最高单体爆发。需要队友配合(冰冻或击晕后进场)才能发挥最大价值 / **Shadow assassin** — highest single-target burst. Needs team setup (freeze or stun) for maximum value.

| 技能 / Skill | AP消耗 | 效果 / Effect |
|:----:|:----:|------|
| **暗影突袭 / Shadow Strike** | 2 | 单体暗影伤害x2.0——背后攻击时x3.0 / Single shadow dmg x2.0 — x3.0 on backstab |
| **淬毒 / Poison Blade** | 1 | 武器涂毒——3回合内攻击附加毒DoT / Poison weapon — attacks add poison DoT for 3 turns |
| **烟雾弹 / Smoke Bomb** | 2 | 全队闪避率+50%持续2回合 / Party evasion +50% for 2 turns |
| **终极技·死神降临 / Ultimate: Death's Arrival** | 4 | 单体暗影伤害x5.0——如果目标血量低于30%则直接秒杀 / Single shadow dmg x5.0 — instant kill if target HP &lt;30% |

**加点推荐**: 敏捷>暴击>速度——追求一击必杀 / **Skill priority**: DEX > CRIT > SPD — one-shot build.

---

## 维克多 / Victor

**工程师**——可以设置炮台、召唤机械宠物攻击。适合喜欢召唤流的玩家 / **Engineer** — sets turrets, summons mechanical pets. For summon-style players.

| 技能 / Skill | AP消耗 | 效果 / Effect |
|:----:|:----:|------|
| **架设炮台 / Set Turret** | 2 | 召唤一个炮台——每回合自动攻击(2回合) / Summon turret — auto-attacks each turn (2 turns) |
| **自爆机器人 / Bomb Bot** | 2 | 召唤自爆机器人→下回合爆炸→全体伤害x2.0 / Summon bomb bot → explodes next turn → AoE dmg x2.0 |
| **机械护盾 / Mech Shield** | 1 | 全队获得护盾(吸收伤害x1.5)持续2回合 / Party gains shield (absorb dmg x1.5) for 2 turns |
| **终极技·巨像 / Ultimate: Colossus** | 4 | 召唤巨型机械(3回合)——每回合攻击+嘲讽全部敌人 / Summon giant mech (3 turns) — attacks + taunts all enemies each turn |

**加点推荐**: 力量>体力>智力——召唤物伤害=力量x1.5 / **Skill priority**: STR > VIT > INT — summon dmg = STR x1.5.

---

## 艾莉丝 / Iris

**弓手**——远程物理输出+侦查(地图上标记敌人和宝箱)。探索时必带 / **Archer** — ranged physical DPS + scout (marks enemies and chests on map). Must-have for exploration.

| 技能 / Skill | AP消耗 | 效果 / Effect |
|:----:|:----:|------|
| **精准射击 / Precise Shot** | 2 | 单体物理伤害x2.0——无视护甲 / Single phys dmg x2.0 — ignores armor |
| **箭雨 / Rain of Arrows** | 3 | 全体物理伤害x1.3 / AoE phys dmg x1.3 |
| **标记 / Mark Target** | 1 | 标记敌人——全队对该目标伤害+30%持续3回合 / Mark enemy — party deals +30% dmg to target for 3 turns |
| **终极技·穿心 / Ultimate: Heart Piercer** | 4 | 单体物理伤害x4.0——如果身上有标记则x6.0 / Single phys dmg x4.0 — x6.0 if target is marked |

**加点推荐**: 敏捷>暴击>速度——先手标记+爆发 / **Skill priority**: DEX > CRIT > SPD — mark first, then burst.

## 阵容推荐 / Team Recommendations

| **Boss战 / Boss Fight** | 古斯塔夫+玛艾勒+艾莉丝 | 坦克抗+Boss标记+稳扎稳打 / Tank holds + mark + consistent DPS |
