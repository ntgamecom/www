## 详细角色分析 / Character Analysis

### 古斯塔夫 / Gustave

**解锁**: 序章自动加入 / Unlock: Auto-join in Prologue
**专属机制**: **狂怒值**——每次受到伤害或造成伤害积累狂怒——满100时下一次攻击必定暴击且不消耗AP / **Rage Meter** — builds on taking/dealing damage — at 100, next attack auto-crits and costs 0 AP.

**核心技能**: **巨剑连斩**(2AP)——3连击物理伤害——每击独立QTE。**战吼**(1AP)——获得嘲讽+防御力+50% 2回合。**旋风斩**(2AP)——周身AOE——每个命中敌人回复5%HP。**终极技·裂地斩**(4AP)——全屏物理伤害——对受嘲讽敌人伤害x2 / **Core Skills**: **Greatsword Combo** (2AP) — 3-hit phys — each independent QTE. **War Cry** (1AP) — Taunt + 50% DEF for 2 turns. **Whirlwind** (2AP) — AoE around self — 5% HP per hit. **Ultimate: Earth Splitter** (4AP) — full-screen phys — 2x damage vs taunted.

**最佳搭配**: 和主角配合前排双坦——古斯塔夫抗伤害、主角输出。推荐和维克多组队——双坦克阵容在困难模式下非常稳健 / **Best Pairing**: Frontline duo with Protagonist — Gustave tanks, Protagonist DPS. Recommend with Victor — dual tank comp very stable on Hard.

### 主角 / Protagonist

**解锁**: 游戏开始 / Unlock: Game start
**专属机制**: **光之能量**——每次使用光系技能积累光能——满10层时光系技能不消耗AP且伤害x1.5 / **Light Energy** — Light skill uses build energy — at 10 stacks, next Light skill costs 0 AP and does 1.5x damage.

**核心技能**: **光之斩**(1AP)——单体光系。**光之护盾**(1AP)——自身获得护盾+光能。**光之爆发**(2AP)——范围光系+击退。**光之冲锋**(2AP)——突进+眩晕。**终极技·光之审判**(4AP)——全屏光系——对暗影单位伤害x3 / **Core Skills**: **Light Slash** (1AP) — single Light. **Light Shield** (1AP) — self shield + Light energy. **Light Burst** (2AP) — AoE Light + knockback. **Light Charge** (2AP) — gap close + stun. **Ultimate: Light Judgment** (4AP) — full-screen Light — 3x vs Shadow.

**最佳搭配**: 和塞拉组队(法术队核心)或和艾伦组队(物理输出互补)。主角万能——适应任何队伍缺的位置 / **Best Pairing**: With Sera (mage core) or Allen (phys complement). Protagonist is versatile — fills any gap in party.

### 艾伦 / Allen

**解锁**: 第一章光辉之城 / Unlock: Ch.1 Radiant City
**专属机制**: **瞄准**——进入瞄准模式——下次攻击必定暴击+无视50%防御——但需要消耗1额外AP / **Aim** — enter aiming mode — next attack auto-crits + ignores 50% defense — costs 1 extra AP.

**核心技能**: **精准射击**(2AP)——高单体伤害+无视30%防御。**毒箭**(1AP)——低伤害但中毒3回合(每回合10%HP)。**快速射击**(3AP)——3连射——可分别瞄准不同目标。**鹰眼**(1AP)——自身暴击率+30% 3回合。**终极技·箭雨**(4AP)——全屏多段物理——对受中毒敌人每段+20%伤害 / **Core Skills**: **Precise Shot** (2AP) — high single target + 30% armor pen. **Poison Arrow** (1AP) — low dmg but poison 3T (10% HP/turn). **Rapid Shot** (3AP) — 3 shots — can target different enemies. **Eagle Eye** (1AP) — self +30% crit 3 turns. **Ultimate: Arrow Rain** (4AP) — full-screen multi-hit phys — +20% per hit vs poisoned.

**最佳搭配**: 和塞拉组队(艾伦单点Boss、塞拉清小怪)或和主角组队(主角提供光系辅助) / **Best Pairing**: With Sera (Allen single-target boss, Sera AoE clear) or Protagonist (Light support).

### 塞拉 / Sera

**解锁**: 第三章深渊裂隙 / Unlock: Ch.3 Abyss Rift
**专属机制**: **元素切换**——战斗中可以切换火/冰/电三种元素——每切换一次获得一层"元素精通"(法术伤害+10% 3回合) / **Element Switch** — switch between Fire/ICE/Lightning in combat — each switch grants 1 stack "Elemental Mastery" (+10% magic damage 3 turns).

**核心技能**: **火焰风暴**(2AP)——范围火焰+点燃。**冰霜新星**(2AP)——范围冰霜+减速。**闪电链**(2AP)——3目标闪电——跳跃伤害递减15%。**奥术护盾**(1AP)——自身获得护盾+回复10%HP。**终极技·元素灾变**(4AP)——全屏三元素连击——第一段点燃→第二段冰冻→第三段过载 / **Core Skills**: **Fire Storm** (2AP) — AoE fire + burn. **Frost Nova** (2AP) — AoE frost + slow. **Chain Lightning** (2AP) — 3 targets — 15% chain damage reduction. **Arcane Shield** (1AP) — self shield + 10% HP. **Ultimate: Elemental Cataclysm** (4AP) — full-screen 3-element combo — burn→freeze→overload.

**最佳搭配**: 和主角组队——主角提供光系补刀和防御。和莉莎组队——莉莎的召唤物可以保护塞拉 / **Best Pairing**: With Protagonist (Light finisher + defense) or Lisa (minions protect Sera).

### 莉莎 / Lisa

**解锁**: 第二章机械平原 / Unlock: Ch.2 Machine Plains
**专属机制**: **机械改造**——可以在战斗外改造自己的机械臂——更换不同的模块改变技能效果——共3种模块: 攻击型、防御型、支援型 / **Mech Mods** — customize mech arm outside combat — swap modules to change skill effects — 3 types: Attack, Defense, Support.

**核心技能**: **召唤机器人**(2AP)——召唤一个小机器人攻击敌人——存在3回合。**机械护盾**(1AP)——全队获得护盾。**修复光线**(1AP)——治疗一个队友15%HP。**炸药投掷**(2AP)——范围物理+击倒。**终极技·机械大军**(4AP)——召唤3个强化机器人——持续到战斗结束——每个机器人有独立的QTE攻击 / **Core Skills**: **Summon Robot** (2AP) — summons 1 robot — lasts 3 turns. **Mech Shield** (1AP) — party shield. **Repair Beam** (1AP) — heal ally 15% HP. **Grenade Toss** (2AP) — AoE phys + knockdown. **Ultimate: Mech Army** (4AP) — summon 3 enhanced robots — permanent — each has independent QTE.

**最佳搭配**: 适合任何队伍——纯辅助/召唤定位。和维克多组队保护能力最强 / **Best Pairing**: Fits any party — pure support/summon. With Victor for maximum protection.

### 维克多 / Victor

**解锁**: 第二章后半 / Unlock: Late Ch.2
**专属机制**: **铁壁**——每回合获得一层铁壁(减伤5%)——最多10层——被破盾后重置。铁壁层数越高格挡率越高 / **Iron Wall** — gains 1 stack per turn (5% DR) — max 10 — resets on shield break. Higher stacks = higher block chance.

**核心技能**: **嘲讽**(1AP)——全敌人强制攻击维克多2回合。**盾墙**(2AP)——格挡率+100% 3回合+反击伤害。**破甲打击**(1AP)——降低敌人防御30% 3回合。**地震踏**(2AP)——范围物理+眩晕1回合。**终极技·钢铁堡垒**(4AP)——全队无敌1回合+全队回复20%HP / **Core Skills**: **Taunt** (1AP) — force all enemies to attack Victor 2 turns. **Shield Wall** (2AP) — 100% block 3 turns + counter. **Armor Break** (1AP) — -30% enemy def 3 turns. **Stomp** (2AP) — AoE phys + stun 1 turn. **Ultimate: Iron Fortress** (4AP) — party invincible 1 turn + 20% party heal.

**最佳搭配**: 任何缺坦克的队伍。和古斯塔夫组队形成铜墙铁壁前排——极难被击杀 / **Best Pairing**: Any party needing a tank. With Gustave for impenetrable frontline.
