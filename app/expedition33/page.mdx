# 33号远征队攻略 / Clair Obscur: Expedition 33 Guide

《33号远征队》是一款回合制角色扮演游戏，设定在一个被神秘"绘画"诅咒的世界——每年一个数字年龄的人会被抹去消失。玩家组成第33号远征队，踏上摧毁绘画、拯救人类的旅程 / Clair Obscur: Expedition 33 is a turn-based RPG set in a world cursed by a mysterious "Painting" — each year, everyone of a specific age is erased. Players form Expedition 33 to destroy the Painting and save humanity.

## 游戏背景 / Game Background

世界的"绘画者"每年在画布上抹去一个数字——所有该年龄的人瞬间消失化为灰烬。2034年，轮到33岁了。主角古斯塔夫(Gustave)和新招募的远征队成员一起，踏上摧毁绘画者的朝圣之旅 / The "Painter" erases one number from the canvas each year — everyone of that age vanishes into ash. In 2034, it's age 33's turn. Protagonist Gustave and the newly assembled Expedition 33 embark on a pilgrimage to destroy the Painter.

## 游戏特色 / Key Features

| 特色 / Feature | 说明 / Description |
|---|---|
| **独特世界观 / Unique Setting** | 油画风格渲染——整个世界如同一幅动起来的油画 / Oil-painting art style — the entire world looks like a living painting |
| **回合制+QTE战斗 / Turn-based + QTE** | 传统回合制指令+攻击时按QTE时机判定增加伤害/效果 / Traditional turn-based commands + QTE timing for bonus damage/effects |
| **行动点系统 / Action Point System** | 每个角色每回合有行动点(AP)——攻击/技能/道具消耗不同AP / Each character has AP per turn — attacks/skills/items cost different AP |
| **元素反应 / Elemental Reactions** | 水+冰=冻结、火+油=爆炸、雷+水=传导——元素组合策略 / Water+Ice=Freeze, Fire+Oil=Explode, Lightning+Water=Conduct — element combo strategy |
| **角色羁绊 / Character Bonds** | 远征队成员之间有羁绊系统——战斗中使用组合技需要羁绊值 / Bond system between characters — combo skills require bond points |
| **油画画风 / Oil Painting Art** | 全程实时渲染的油画风格——每一帧都是壁纸 / Real-time oil painting rendering — every frame is wallpaper-worthy |

## 新手快速入门 / Quick Start

| 建议 / Tip | 说明 / Description |
|:----:|------|
| **开局角色 / Start** | 古斯塔夫(均衡)+玛艾勒(治疗)是最稳的开局组合 / Gustave (balanced) + Maelle (healer) = safest start |
| **战斗节奏 / Battle Rhythm** | 不要浪费行动点——每个角色每回合至少用1AP攻击/技能。留1AP防御/闪避 / Don't waste AP — each character uses at least 1AP per turn for attack/skill. Save 1AP for defense/dodge |
| **QTE练习 / QTE Practice** | 战斗中有大量QTE判定——完美判定(绿色区域)伤害x1.5。花时间练好QTE = 战斗难度降半 / Perfect QTE (green zone) = 1.5x damage. Mastering QTE halves battle difficulty |
| **探索优先 / Explore First** | 主线推进前先探索地图——隐藏宝箱有强化素材和遗物 / Explore maps before progressing main story — hidden chests contain upgrade materials and relics |
