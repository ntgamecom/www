## 角色Build概述 / Character Build Overview

33号远征队的Build系统围绕每个角色的独特技能树和装备搭配展开。每个角色有专属武器类型和技能倾向——合理分配技能点和装备词缀可以让角色发挥最大效能 / Expedition 33's build system centers on each character's unique skill tree and gear setup. Each character has exclusive weapon types and skill tendencies — allocating skill points and gear affixes optimally maximizes effectiveness.

### 属性优先级 / Stat Priority

| 属性 / Stat | 效果 / Effect | 适用角色 / Best For |
|---|---|---|
| 力量 / Strength | 物理伤害+近战技能伤害 / Phys + melee skill damage | 坦克、近战DPS / Tank, melee DPS |
| 敏捷 / Dexterity | 暴击率+闪避率+远程伤害 / Crit + dodge + ranged | 刺客、弓手 / Assassin, archer |
| 智力 / Intelligence | 法术伤害+元素抗性 / Magic damage + elemental resist | 法师、辅助 / Mage, support |
| 体质 / Constitution | 生命上限+物理抗性 / Max HP + phys resist | 所有角色(至少点一些) / All (some) |
| 精神 / Spirit | 技能能量回复+治疗量 / Skill energy regen + healing | 治疗、辅助 / Healer, support |

### 属性分配策略 / Stat Allocation Strategy

**前期(1-20级)**: 主属性点满→体质点到10→剩余点副属性。优先确保你能装备当前等级的最强武器——武器有力量/敏捷需求。**中期(20-40级)**: 主属性到30→体质到20→开始点副属性。输出角色优先堆满主属性。**终局(40+)**: 主属性50+→体质30+→副属性适量。终局装备的附加属性可以补充不足的副属性 / **Early (1-20)**: Max main stat → Con 10 → remaining in secondary. Prioritize equipping best weapons — weapons have Str/Dex requirements. **Mid (20-40)**: Main stat 30 → Con 20 → start secondary. DPS prioritize maxing main stat. **Endgame (40+)**: Main stat 50+ → Con 30+ → moderate secondary. Endgame gear affixes can supplement weaker stats.

## 全角色Build推荐 / Character Build Recommendations

### 主角(光之战士) / Protagonist (Light Warrior)

**定位**: 平衡型输出 / Role: Balanced DPS
**推荐武器**: 单手剑+盾 / 双手剑 / Recommended Weapon: Sword+Shield / Greatsword

**技能优先**: 光之斩(Max)→光之护盾(3)→光之爆发(Max)→光之冲锋(3)→终极技·光之审判(Max)。**被动优先**: 光之精通(Max)→战斗回复(3)→剑术精通(3)→铁壁(3) / **Skill Priority**: Light Slash (Max) → Light Shield (3) → Light Burst (Max) → Light Charge (3) → Ultimate: Light Judgment (Max). **Passive Priority**: Light Mastery (Max) → Battle Regen (3) → Sword Mastery (3) → Iron Wall (3).

**属性加点**: 力量35→体质25→敏捷20—保持平衡的攻防能力。**装备词缀**: 物理伤害+光系伤害>暴击率>生命值>防御力。推荐独特装备: **光辉之剑**——每击叠加光系伤害——最多10层/ **Stat Spread**: Str 35 → Con 25 → Dex 20 — balanced offense/defense. **Gear Affixes**: Phys damage + Light damage > Crit rate > HP > Defense. Recommended Unique: **Radiant Sword** — stacking Light damage per hit — max 10 stacks.

**强度评价: A级**——万金油角色，适合任何队伍。前期强后期中规中矩。弱点: 没有特别突出的领域 / **Tier: A** — versatile, fits any party. Strong early, solid later. Weakness: no exceptional specialty.

### 莉莎(机械师) / Lisa (Mech Engineer)

**定位**: 召唤/辅助 / Role: Summon/Support
**推荐武器**: 手枪+工具 / 机械臂 / Recommended Weapon: Pistol+Tool / Mech Arm

**技能优先**: 召唤机器人(Max)→机械护盾(3)→修复光线(Max)→炸药投掷(3)→终极技·机械大军(Max)。**被动优先**: 机械精通(Max)→节省材料(3)→快速修复(3)→连锁爆炸(3) / **Skill Priority**: Summon Robot (Max) → Mech Shield (3) → Repair Beam (Max) → Grenade Toss (3) → Ultimate: Mech Army (Max). **Passive Priority**: Mech Mastery (Max) → Material Saver (3) → Quick Repair (3) → Chain Explosion (3).

**属性加点**: 智力30→精神25→体质15——召唤物伤害受智力加成。**装备词缀**: 召唤物伤害>技能能量回复>精神>生命值。独特推荐: **机械核心**——增加1个召唤物上限——性价比最高 / **Stat Spread**: Int 30 → Spirit 25 → Con 15 — summon damage scales with Int. **Gear Affixes**: Summon damage > Energy regen > Spirit > HP. Unique: **Mech Core** — +1 summon cap — best value.

**强度评价: A+级**——召唤流的最强角色，单人能力和队伍适配性都很好。弱点: 本体脆弱、召唤物AI有时会发呆 / **Tier: A+** — best summoner, strong solo and in parties. Weakness: squishy body, minion AI occasionally inconsistent.

### 维克多(重装兵) / Victor (Heavy Armor)

**定位**: 坦克/控制 / Role: Tank/Controller
**推荐武器**: 重盾+单手锤 / 巨斧 / Recommended Weapon: Tower Shield+Hammer / Greataxe

**技能优先**: 嘲讽(Max)→盾墙(Max)→破甲打击(3)→地震踏(Max)→终极技·钢铁堡垒(Max)。**被动优先**: 铁壁(Max)→生命强化(Max)→反伤(3)→盾牌精通(Max) / **Skill Priority**: Taunt (Max) → Shield Wall (Max) → Armor Break (3) → Stomp (Max) → Ultimate: Iron Fortress (Max). **Passive Priority**: Iron Wall (Max) → HP Boost (Max) → Thorns (3) → Shield Mastery (Max).

**属性加点**: 体质40→力量20→精神10——纯坦克方向。**装备词缀**: 防御力>生命值>物理抗性>元素抗性>生命回复。独特推荐: **不破之盾**——格挡时回复5%生命——坦克生存质变 / **Stat Spread**: Con 40 → Str 20 → Spirit 10 — pure tank. **Gear Affixes**: Defense > HP > Phys resist > Elemental resist > HP regen. Unique: **Unbreakable Shield** — 5% HP restore on block — game-changer for tank.

**强度评价: S级(坦克) / B级(输出)**——最强坦克没有之一。但纯输出Build远不如专业DPS角色。推荐作为副坦+辅助控制使用 / **Tier: S (tank) / B (DPS)** — best tank in game, but pure DPS build far behind dedicated DPS. Recommended as off-tank + controller.

### 艾伦(游侠) / Allen (Ranger)

**定位**: 远程物理DPS / Role: Ranged Physical DPS
**推荐武器**: 长弓/双持弩 / Recommended Weapon: Longbow / Dual Crossbows

**技能优先**: 精准射击(Max)→毒箭(3)→快速射击(Max)→鹰眼(Max)→终极技·箭雨(Max)。**被动优先**: 弓术精通(Max)→暴击强化(Max)→弱点识别(3)→疾风步(3) / **Skill Priority**: Precise Shot (Max) → Poison Arrow (3) → Rapid Shot (Max) → Eagle Eye (Max) → Ultimate: Arrow Rain (Max). **Passive Priority**: Bow Mastery (Max) → Crit Boost (Max) → Weak Point (3) → Wind Step (3).

**属性加点**: 敏捷40→力量20→体质15——极致输出方向。**装备词缀**: 暴击率>暴击伤害>远程伤害>攻速。独特推荐: **猎龙弓**——对Boss伤害+30%——终局Boss战MVP / **Stat Spread**: Dex 40 → Str 20 → Con 15 — max DPS. **Gear Affixes**: Crit rate > Crit damage > Ranged damage > Attack speed. Unique: **Dragon Hunter Bow** — +30% boss damage — endgame boss MVP.

**强度评价: S级**——全游戏最强单体DPS，终局Boss战表现最佳。弱点: 群战能力弱、本体脆皮需要保护 / **Tier: S** — best single-target DPS, excels in endgame bosses. Weakness: poor AoE, squishy.

### 塞拉(法师) / Sera (Mage)

**定位**: 法术AoE DPS / Role: Magic AoE DPS
**推荐武器**: 法杖/魔法书 / Recommended Weapon: Staff / Spellbook

**技能优先**: 火焰风暴(Max)→冰霜新星(3)→闪电链(Max)→奥术护盾(3)→终极技·元素灾变(Max)。**被动优先**: 元素精通(Max)→魔力增幅(Max)→施法速度(3)→能量回复(3) / **Skill Priority**: Fire Storm (Max) → Frost Nova (3) → Chain Lightning (Max) → Arcane Shield (3) → Ultimate: Elemental Cataclysm (Max). **Passive Priority**: Elemental Mastery (Max) → Power Boost (Max) → Cast Speed (3) → Energy Regen (3).

**属性加点**: 智力40→精神20→体质15——纯法系输出。**装备词缀**: 法术伤害>元素穿透>施法速度>法力回复。独特推荐: **元素法典**——法术有几率双重施法——DPS翻倍 / **Stat Spread**: Int 40 → Spirit 20 → Con 15 — pure caster DPS. **Gear Affixes**: Magic damage > Elemental pen > Cast speed > Mana regen. Unique: **Elemental Codex** — chance to dual-cast spells — 2x DPS.

**强度评价: S级**——清图效率第一，AoE伤害天花板。弱点: 单体伤害不如艾伦、蓝耗高需要管理 / **Tier: S** — best clear speed, AoE damage ceiling. Weakness: single target below Allen, high mana cost.

## 终局Build要点 / Endgame Build Tips

终局(通关后)内容包括**深渊裂隙深层**(10-20层)和**挑战竞技场**。推荐终局队伍: 艾伦(主C)+塞拉(副C+AoE)+维克多(坦克)+莉莎(召唤/辅助)。全体装备至少+8强化——武器+10。所有主属性堆到50+。准备不同属性伤害的武器应对怪物抗性——物理+火焰+冰霜三种伤害类型均衡分配 / Endgame content includes **Deep Abyss Rift** (floors 10-20) and **Challenge Arena**. Recommended endgame party: Allen (main DPS) + Sera (AoE sub-DPS) + Victor (tank) + Lisa (summon/support). Gear at least +8 enhance — weapons +10. All main stats 50+. Prepare multiple damage types — balanced physical + fire + frost for enemy resistances.

## Build进阶技巧 / Advanced Build Tips

**艾伦爆发连段**: 暗影步(背击+暴击) → 疾风斩×3(连击叠满1.45倍) → 终结技(斩杀线40%HP以下)。装备暗影套装+暴击饰品——暴击率堆到50%+——暴击伤害250%+。总爆发=约2万伤害——除终局Boss外直接秒杀 / **Allen burst combo**: Shadowstep (back crit) → Wind Slash x3 (combo stack 1.45x) → Execute (under 40% HP). Shadow set+CR gear — 50%+ CR — 250%+ CRIT DMG. Total burst ~20k dmg — oneshots non-endgame bosses.

**塞拉元素循环**: 火焰风暴(叠灼烧) → 冰霜新星(蒸发1.5x) → 闪电链(超载AoE) → 元素末日(终极技——属性连锁)。装备元素套装+混沌法杖——AP恢复越快伤害越高。此循环需要约80AP(3回合积累)——爆发后清理全场 / **Sera element loop**: Firestorm (Burn) → Frost Nova (Vaporize 1.5x) → Chain Lightning (Overload AoE) → Element Doom (ulti — element chain). Element set+Chaos Staff — more AP regen = more dmg. Takes ~80AP (3 turns charge) — full screen wipe.
