# 世界探索 / World Exploration

33号远征队的世界由多个手绘油画风格区域组成——每个区域有独特的视觉风格、敌人种类和隐藏秘密。探索是获取强力装备和遗物的主要途径 / Expedition 33's world is composed of hand-painted oil painting style zones — each with unique visual style, enemy types, and hidden secrets. Exploration is the main source of powerful gear and relics.

### 环境互动 / Environment Interaction

世界场景中有多种可互动物体: **可破坏墙壁**(裂缝墙面——用力量检定或爆炸物破坏)、**隐藏开关**(书架上不显眼的书、地板上的特殊砖块)、**可攀爬表面**(发光的凸起墙面——可用攀爬技能)、**光之反射镜**(将光之碎片放入后反射光线打开通道)。**第3章前尽量完成所有探索**——通关后部分区域会永久关闭 / World has various interactable objects: **Breakable walls** (cracked walls — STR check or explosives), **Hidden switches** (unobtrusive books, special floor tiles), **Climbable surfaces** (glowing protrusions — use climb skill), **Light Reflectors** (insert Light Fragment to reflect beam and open path). **Complete all exploration before Ch.3** — some zones become permanently closed post-game.

## 传送点系统 / Waypoint System

每个区域有3-6个传送点。激活后可以快速移动。推荐到达新区第一时间找到并激活所有传送点——节省大量跑路时间。部分传送点隐藏在分支路径中——探索时注意小地图上的非活动传送点图标 / Each zone has 3-6 waypoints. Activate for fast travel. Recommended: find and activate all upon entering a new zone — saves massive travel time. Some waypoints hidden on branching paths — watch minimap for inactive icons.

## 世界Boss / World Bosses

部分区域有**巡逻型世界Boss**——会在地上巡逻——击败后掉落独特装备。**白崖镇的腐败巨龙**(等级8)——在悬崖周围飞行——用远程攻击打下来。**光辉之城的暗影巡视者**(等级18)——夜晚在城市街道巡逻——只在游戏内"夜晚时间"出现。**机械平原的巨型机械蛇**(等级28)——在地图上钻地移动——需要引诱到特定位置用场景机关击杀。**深渊巡礼者**(等级38)——深渊裂隙1-5层随机出现——击杀获得深渊专属遗物 / Some zones have **roaming World Bosses** — patrol the ground — drop unique gear on defeat. **White Cliff Corrupted Dragon** (lvl 8) — flies around cliffs — use ranged to ground it. **Radiant City Shadow Warden** (lvl 18) — patrols streets at night — only appears in-game "nighttime". **Machine Plains Giant Mech Snake** (lvl 28) — burrows through ground — lure to specific spot for environment kill. **Abyss Pilgrim** (lvl 38) — random spawn Abyss Rift F1-5 — drops Abyss-exclusive relic.

## 探索优先级 / Exploration Priority

**进入新区域后按此顺序探索**: 1. 激活所有传送点 2. 收集地图上的明显宝箱 3. 探索分支路径寻找隐藏区域 4. 完成对应区域的支线任务 5. 收集区域内光之碎片 6. 寻找回声点和猫 7. 挑战世界Boss(如果等级足够) / **Priority on entering new zone**: 1. Activate all waypoints 2. Collect visible chests 3. Explore branches for hidden zones 4. Complete zone side quests 5. Collect Light Fragments 6. Find Echo Points and Cats 7. Challenge World Boss (if level appropriate).

## 地图探索小贴士 / Map Exploration Tips

1. **每个新区域先找传送点**: 找到后赶紧激活——方便死亡后快速返回 / **Find waypoints first**: Activate immediately — quick return after death.

2. **收集品有提示**: 光之碎片在地图上显示为金色光点——回声点显示为蓝色光波——宝箱显示为白色光点。大世界地图可以看到已收集/剩余数量 / **Collectible hints**: Fragments = gold dots, Echoes = blue waves, chests = white dots. World map shows collected/remaining counts.

3. **隐藏墙壁的识别**: 有隐藏墙壁的墙面通常有细微的颜色差异——或者墙边有异常的道具摆放(比如单独的花瓶、垫子)。攻击5次即可破坏——注意听攻击音效反馈(打隐藏墙是空心声)/ **Hidden wall detection**: Walls with secrets have subtle color difference or unusual object placement (single vase, mat). Attack 5 times to break — listen for hollow sound.

4. **回头探索**: 获得新能力后(比如潜水、暗影视野)——回到之前的区域——之前去不了的地方现在可以进入了——很多光之碎片需要回头收集 / **Backtracking**: After getting new abilities (dive, shadow vision) — return to earlier zones — previously inaccessible areas open — many fragments need backtracking.

5. **地图标记**: 游戏内置地图可以自定义标记点——遇到打不开的门/解不开的谜——先标记——等获得对应的能力或道具后再回来 / **Map markers**: In-game map has custom markers — find locked door/puzzle — mark it — return with correct ability/item.

## 全区域主线通关路线速览 / All Zones Speedrun Route

**速通路线(4小时内)**: 第一章废墟→直奔B3层→战争巨像→城镇(跳过支线)→第二章港口(收艾伦+塞拉)→火山→火山领主→第三章学院(只拿关键道具)→光之森林→光之守护者→不走暗影森林(浪费时间)→第四章神殿→试炼→深渊→最终Boss。全程不探索任何分支——不打任何非必要精英——对话全部快进。1-2章约1小时——3章约1.5小时——4章约1.5小时 / **Speedrun route (under 4h)**: Ch.1 ruins→rush B3→Colossus→town (skip side)→Ch.2 port (get Allen+Sera)→volcano→Lord→Ch.3 academy (key items only)→Light Forest→Guardian→skip Shadow Forest→Ch.4 temple→trials→Abyss→final boss. No exploration — no non-essential elites — fast-forward all dialogue. Ch.1-2 ~1h — Ch.3 ~1.5h — Ch.4 ~1.5h.

**全收集路线(50-80小时)**: 每个区域地毯式搜索——每个角落都走到——每面墙攻击检查——每个NPC对话3次以上——回头检查新能力区域。建议: 1周目随意玩(30-40小时)——通关后用终局装备和新能力去做全收集(20-40小时) / **Completionist route (50-80h)**: Every zone fully combed — every corner — attack every wall — talk to NPCs 3+ times — backtrack with new abilities. Recommend: playthrough 1 casual (30-40h) — full collect with endgame gear (20-40h).
