# 世界探索 / World Exploration

33号远征队的世界由多个手绘油画风格区域组成——每个区域有独特的视觉风格、敌人种类和隐藏秘密。探索是获取强力装备和遗物的主要途径 / Expedition 33's world is composed of hand-painted oil painting style zones — each with unique visual style, enemy types, and hidden secrets. Exploration is the main source of powerful gear and relics.

---

## 区域总览 / Region Overview

| 区域 / Region | 推荐等级 / Level | 视觉风格 / Visual Style | 核心敌人 / Enemies |
|:----:|:----:|:----|:----:|
| **白崖镇 / White Cliff Town** | 1-5 | 海岸白垩悬崖+小镇 / Coastal chalk cliffs + town | 被抹去的游魂、腐败动物 / Erased souls, corrupted animals |
| **灰烬森林 / Ashen Forest** | 5-10 | 烧焦的黑森林——灰烬飘落 / Burnt black forest — falling ash | 灰烬亡灵、火焰元素 / Ash undead, fire elementals |
| **冰封湖 / Frozen Lake** | 10-15 | 冰封的湖面+冰晶森林 / Frozen lake surface + crystal forest | 冰霜亡灵、冰元素 / Frost undead, ice elementals |
| **锈蚀工厂 / Rusted Factory** | 15-20 | 工业废墟——齿轮+管道 / Industrial ruins — gears + pipes | 机械亡灵、腐蚀兽 / Mechanical undead, corrosion beasts |

## 白崖镇 / White Cliff Town

游戏的起点——一个被"抹除"灾难摧毁的海滨小镇。玩家从这里开始远征 / The starting point — a seaside town destroyed by the "Erase" disaster. Players begin their expedition here.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **中心广场 / Town Square** | 游戏初始位置——有商人和任务板 / Starting position — merchant and quest board | 首个商人、可接支线任务 / First merchant, side quests available |
| **教堂 / Church** | 二楼有隐藏房间——调查画像触发事件 / Second floor hidden room — examine painting | 遗物·治愈之泉、白崖镇支线触发 / Relic: Healing Spring, side quest trigger |
| **码头 / Dock** | 破船调查可进入隐藏洞穴/ Broken ship leads to hidden cave | 遗物·水手护符 + 水上移动能力 / Relic: Sailor's Charm + water movement |
| **白崖灯塔 / Lighthouse** | 区域末端——击败守关Boss后前往灰烬森林 / Zone end — defeat gate boss to reach Ashen Forest | 白崖镇Boss战 / White Cliff boss fight |

## 灰烬森林 / Ashen Forest

被烧焦的黑色森林——灰烬不断从天而降。这里的火焰元素需要用水系技能克制 / A charred black forest — ash falls constantly. Fire elementals here are weak to water skills.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **燃烧林地 / Burning Woods** | 森林入口区域——大量火焰敌人 / Forest entrance — many fire enemies | 火抗戒指、树洞隐藏入口 / Fire resist ring, hidden tree hole |
| **隐藏洞穴 / Hidden Cave** | 需要用水系技能灭火后进入 / Extinguish fire with water skill to enter | 遗物·时光沙漏 + 强化石 / Relic: Hourglass + upgrade stones |
| **灰烬神庙 / Ash Temple** | 灰烬森林中部的重要地标 / Central landmark | 传送点 + 记忆碎片 / Fast travel + memory fragment |
| **焦枯王座 / Burnt Throne** | 关底区域——守关Boss战 / End zone — gate boss | 灰烬森林Boss + 通往冰封湖路径 / Boss + path to Frozen Lake |

## 冰封湖 / Frozen Lake

巨大的冰封湖面——湖下有隐藏的水下城市遗迹。冰霜系敌人对火系技能脆弱 / A massive frozen lake — beneath lies a hidden underwater city ruin. Frost enemies are weak to fire skills.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **湖面冰原 / Lake Surface** | 冰面行走——注意滑行机制影响移动 / Icy surface — sliding mechanic affects movement | 冰晶森林入口、湖面战斗 / Crystal forest entrance, surface fights |
| **冰晶森林 / Crystal Forest** | 湖岸的冰晶树林——有隐藏宝箱 / Ice crystal grove on shore — hidden chests | 武器强化素材、冰抗戒指 / Upgrade materials, ice resist ring |
| **湖底神殿 / Underwater Temple** | 打破冰面跳入水下→到达城市遗迹 / Break ice → dive → city ruins | 法书·终极冰霜 + 勇者之证 / Spellbook: Permafrost + Hero's Proof |
| **冰封祭坛 / Frozen Altar** | 关底区域——守关Boss战 / End zone — gate boss | 冰封湖Boss + 通往锈蚀工厂 / Boss + path to Rusted Factory |

## 锈蚀工厂 / Rusted Factory

工业革命风格的机械废墟——齿轮、管道和传送带构成复杂的立体地图。机械敌人对雷系脆弱 / Industrial revolution-style mechanical ruins — gears, pipes, and conveyor belts create complex vertical maps. Mechanical enemies weak to lightning.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **工厂入口 / Factory Entrance** | 破败的铁门——进入后是传送带区域 / Rusted iron gate — conveyor belt zone inside | 传送点、工程师NPC支线 / Fast travel, engineer NPC quest |
| **传送带区 / Conveyor Belt Zone** | 跳到移动传送带上探索 / Jump on moving belts to explore | 隐藏区域入口(跳上逆向传送带) / Hidden zone entry (reverse belt) |
| **核心车间 / Core Workshop** | 工厂中心——巨型齿轮机关解谜 / Center — giant gear puzzle | 终极武器素材、矿石大量 / Endgame weapon materials |
| **熔炉大厅 / Furnace Hall** | 关底区域——守关Boss战 / End zone — gate boss | 锈蚀工厂Boss + 通往画师之塔 / Boss + path to Painter's Tower |

## 画师之塔 / Painter's Tower

最终区域——一座无限旋转的油画楼梯构成的神秘高塔。每一层都是不同的油画世界 / The final zone — a mysterious tower with infinitely spinning painted staircases. Each floor is a different painted world.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **塔基大厅 / Tower Lobby** | 入口区域——墙壁油画开始活动 / Entrance — wall paintings come alive | 传送点、颜色钥匙需要收集 / Fast travel, color key collection |
| **油画迷宫 / Painted Maze** | 1-3层——每层颜色主题不同(红/蓝/黄) / Floors 1-3 — each has color theme (red/blue/yellow) | 颜色敌人、各层钥匙碎片 / Color enemies, key fragments per floor |
| **隐藏画室 / Hidden Studio** | 3层中间的隐藏平台——看似会摔死但实际安全 / Hidden platform at 3F center — looks deadly but safe | 真结局道具 + 遗物·画师之笔 / True ending item + Painter's Brush |
| **塔顶画室 / Top Studio** | 最终Boss战——画师本尊 / Final boss — the Painter | 游戏结局 + 通关后内容 / Game ending + post-game content |

## 探索技巧 / Exploration Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **艾莉丝在队伍时地图显示宝箱 / Iris shows chests on map** | 艾莉丝的被动技能"鹰眼"——地图上会标记所有可互动的宝箱和收集品 / Iris's passive "Eagle Eye" marks all interactive chests and collectibles on the map |
| **油画可以互动 / Paintings are interactive** | 部分区域的油画可以互动——调查→触发隐藏事件/传送/宝箱 / Some zone paintings are interactive — examine → hidden event/teleport/chest |
| **颜色钥匙 / Color Keys** | 某些门需要对应的颜色钥匙——钥匙在对应颜色的精英怪身上 / Some doors need color keys — dropped by same-color elite enemies |
| **传送点 / Fast Travel** | 每个区域有2-3个传送点(篝火)——激活后可快速移动。探索时优先找传送点 / 2-3 fast travel points per zone — activate for fast movement. Prioritize finding them during exploration |
| **水上移动 / Water Movement** | 部分区域需要水上行走能力——完成白崖镇的支线获得"水手护符"后才能去 / Some zones need water-walking ability — complete White Cliff Town side quest for "Sailor's Charm" |
