# 战斗系统 / Combat System

33号远征队的战斗是回合制指令+实时QTE的结合。策略深度在行动点分配、元素连携和QTE完美判定之间 / Expedition 33's combat blends turn-based commands with real-time QTE. Strategic depth lies in AP allocation, elemental combos, and perfect QTE timing.

---

## 战斗流程 / Battle Flow

| 阶段 / Phase | 操作 / Actions |
|:----:|------|
| **1. 指令阶段 / Command Phase** | 选择每个角色的行动(攻击/技能/道具/防御)——每个行动消耗AP / Choose each action (attack/skill/item/defend) — each costs AP |
| **2. 行动阶段 / Action Phase** | 按速度顺序执行行动——每个行动命中时触发QTE / Actions execute in speed order — QTE triggers on hit |
| **3. 敌方阶段 / Enemy Phase** | 敌人行动——玩家可以按防御键触发防御QTE / Enemies act — press defend for defense QTE |
| **4. 回合结束 / Turn End** | 结算持续伤害/回复效果——进入下一回合 / Tick DoT/Heal — next turn |

### 行动顺序 / Turn Order

行动顺序由**速度(SPD)**属性决定——速度最高的角色最先行动。行动条在屏幕上方显示——可以看到下三个行动者。**加速/减速效果**可以影响行动顺序——将敌人的行动推迟非常有效。注意: 等待(Wait)会让角色在本回合不行动但下回合提前行动 / Turn order determined by **Speed (SPD)** — highest goes first. Turn bar at top of screen shows next 3 actors. **Haste/Slow effects** manipulate order — delaying enemy turns is very effective. Note: Wait skips current turn but moves character earlier next turn.

## 技能分类 / Skill Categories

### 物理技能 / Physical Skills

消耗AP: 1-3 / 适用角色: 古斯塔夫、主角、艾伦 / 特点: 受力量/敏捷加成——物理伤害——可触发暴击 / Cost: 1-3 AP. For: Gustave, Protagonist, Allen. Features: STR/DEX scaling — physical damage — can crit.

**连击技能**: 多次命中——每次命中独立QTE——全部完美判定时最后一击伤害x2。例如主角的"光之连斩"——3连击——全部完美QTE时额外追加一次光属性伤害 / **Combo Skills**: Multi-hit — each hit has independent QTE — all perfect = final hit damage x2. Example: Protagonist's "Light Combo" — 3 hits — perfect all = bonus Light damage.

**蓄力技能**: 一回合蓄力——下回合自动释放——伤害x2但蓄力期间受伤增加50%。适合在敌人被控制时使用 / **Charge Skills**: Charge one turn — auto-release next turn — damage x2 but +50% damage taken during charge. Best used when enemies are CC'd.

### 法术技能 / Magic Skills

消耗AP: 2-3 / 适用角色: 塞拉、主角(光系) / 特点: 受智力加成——元素属性——可触发元素反应 / Cost: 2-3 AP. For: Sera, Protagonist (Light). Features: INT scaling — elemental — triggers reactions.

**火焰法术**: 高伤害+点燃效果(每回合持续伤害, 3回合)。点燃状态在受到冰霜法术时解除但不额外效果 / **Fire Spells**: High damage + Burn (DoT, 3 turns). Burn removed by Frost spells with no extra effect.

**冰霜法术**: 中等伤害+减速效果(敌人速度-30%, 2回合)。冰冻+闪电可以触发**超导**——降低敌人防御50%持续2回合 / **Frost Spells**: Medium damage + Slow (-30% SPD, 2 turns). Frost + Lightning triggers **Superconduct** — -50% enemy defense for 2 turns.

**闪电法术**: 中等伤害+暴击率提升(自身+10%暴击, 3回合)。对受冰冻的敌人暴击率额外+20% / **Lightning Spells**: Medium damage + self-crit buff (+10% crit, 3 turns). +20% crit vs Frozen enemies.

**光系法术**: 主角专属——对暗影系敌人伤害x2——治疗队友50%伤害值的HP——全能型技能 / **Light Spells**: Protagonist exclusive — 2x damage vs Shadow enemies — heal party for 50% damage dealt — versatile.

|---|---|---|

| 火焰+冰霜 / Fire + Frost | 蒸发——两个效果抵消——无额外 / Vaporize — both cancel | 避免 / Avoid |

| 光系+暗影 / Light + Shadow | 净化——额外100%伤害——只对暗影敌人 / Purify — +100% vs shadow | 极高(暗影敌人) / Very High |

## 战斗策略 / Battle Strategies

### 群战策略 / AoE Strategy

面对3个以上敌人时——优先使用塞拉的**火焰风暴**或闪电链清场。QTE以命中更多敌人为目标——范围技能的完美QTE可以扩大范围50%。第二优先级控制敌人——使用冰霜法术减速或主角的盾击眩晕——减少同时面对的敌人数量 / Against 3+ enemies — prioritize Sera's Fire Storm or Chain Lightning. QTE aim for maximum targets — perfect QTE expands AoE by 50%. Second priority: CC — Frost spells slow or Protagonist's Shield Stun — reduce active enemies.

### Boss战策略 / Boss Strategy

Boss战的关键是**阶段转换管理**。大部分Boss在70%/40%HP时转换阶段——转换前有短暂停顿——此时使用大招(4AP终极技)可以获得全额伤害。Boss战中保留至少1人防御——确保在Boss放大招时有角色可以存活。**破防机制**: 部分Boss需要先破坏护盾才能造成伤害——使用高AP技能集中破盾 / Boss key is **phase transition management**. Most bosses transition at 70%/40% HP — pause before transition — use ults for full damage. Keep at least 1 character defending — ensures survivability during boss ults. **Shield Break**: Some bosses need shield destroyed first — focus high-AP skills.

## 状态效果一览 / Status Effects

**增益**: 攻击力上升(+50%伤害, 3回合)、防御力上升(-50%受伤, 3回合)、加速(+50%速度, 3回合)、再生(每回合回复10%HP, 3回合)、暴击率提升(+20%暴击, 3回合)。**减益**: 燃烧(每回合15%HP, 3回合)、冰冻(速度-50%, 2回合——受攻击解除)、眩晕(无法行动, 1回合)、中毒(每回合10%HP+减治疗, 3回合)、减速(速度-30%, 2回合) / **Buffs**: ATK Up (+50% dmg, 3T), DEF Up (-50% dmg taken, 3T), Haste (+50% SPD, 3T), Regen (10% HP/turn, 3T), Crit Up (+20%, 3T). **Debuffs**: Burn (15% HP/turn, 3T), Frost (-50% SPD, 2T — breaks on hit), Stun (can't act, 1T), Poison (10% HP/turn + -heal, 3T), Slow (-30% SPD, 2T).

## 队伍协作技巧 / Teamwork Tips

**连携攻击**: 两个角色对同一个敌人使用同属性攻击时——第二次攻击获得20%伤害加成。**交替控制**: 一控一输出——一个角色控制敌人时另一个角色的暴击率+25%。**集火优先级**: 治疗者>高输出敌人>控制者>坦克——优先击杀能治疗或高输出的敌人。**待机配合**: 在Boss即将转换阶段时全队待机——下回合全员满AP爆发——可以在阶段转换窗口打出最高输出 / **Chain Attack**: Two characters using same element on same enemy — 2nd hit +20% damage. **Alternate Control**: One CC, one DPS — DPS gets +25% crit vs CC'd enemy. **Focus Priority**: Healer > High DPS > Controller > Tank. **Wait Combo**: All party members wait before boss phase transition — full AP burst next turn during transition window.

## QTE判定详解 / QTE Timing Details

33号远征队的QTE分为三种类型——每种有不同的判定窗口和成功率 / Three types of QTEs in Expedition 33 — each with different timing and success windows:

| QTE类型 / Type | 判定窗口 / Window | 完美QTE奖励 / Perfect Bonus | 失败惩罚 / Failure Penalty |
|---|---|---|---|
| 攻击QTE / Attack QTE | 0.5秒——指示器到达高亮区时按 / 0.5s — press when indicator hits highlight zone | 伤害+30%——额外效果触发 / +30% dmg — bonus effect | 伤害减半 / Half damage |
| 防御QTE / Defend QTE | 0.3秒——敌人攻击前出现红色提示 / 0.3s — red indicator before enemy attack | 完全防御——反击+20%伤害 / Full block — counter +20% dmg | 受到全额伤害 / Full damage taken |
| 闪避QTE / Dodge QTE | 0.2秒——比防御窗口更短——黄色提示 / 0.2s — yellow indicator — shorter than block | 无伤+触发一次免费攻击机会 / No damage + free attack | 受到150%伤害 / 150% damage taken |

**完美QTE练习**: 每种技能有不同的QTE节奏——建议在训练场逐一练习主力技能的完美QTE。一般规律: 技能速度越快→QTE窗口越短——大招的QTE通常比小技能长——但失败惩罚也更大 / **Practice**: Each skill has different QTE rhythm — practice main skills at training ground. Rule: Faster skills = shorter window — ultimate skills have longer window but bigger failure penalty.

## 连招系统详解 / Combo System Detailed Guide

**基础连招**: 同一角色连续使用技能——每次连击增加15%伤害——上限3连(1.45倍)。连击在角色被击中或使用普通攻击时重置 / Basic combo: Same character chain skills — +15% dmg per hit — cap 3 hits (1.45x). Combo resets on hit taken or normal attack.

**接力连招**: 角色A最后一下→角色B接上→全队获得"接力"Buff(+25%暴击率持续1回合)。接力连招需要角色A的攻击动画结束时立即按下角色B的快捷——完美衔接可获得额外+10%暴击伤害 / Relay combo: Char A final hit → Char B chains → party gets "Relay" buff (+25% CR — 1 turn). Relay needs immediate button press at end of A's animation — perfect link = extra +10% CRIT DMG.

**终极连招**: 接力3次后(ABCD4人)——第4人使用终极技能——获得"终焉之力"——终极技能伤害翻倍。这是游戏中最顶级的输出手段——推荐在Boss破防/虚弱时使用 / Ultimate combo: Relay 3 times (A→B→C→D) — D uses ultimate — "End Power" — ultimate dmg x2. Best burst in game — use during boss weakness phase.

**QTE等级判定**: 不同QTE结果有不同的奖励 / QTE Grade System:

| QTE等级 / Grade | 判定窗口 / Window | 奖励 / Bonus |
|---|---|---|
| 完美 / Perfect | ≤0.1秒偏差 / ≤0.1s off | 伤害+30%——恢复2AP / Dmg+30% — restore 2AP |
| 良好 / Good | ≤0.3秒偏差 / ≤0.3s off | 伤害+15%——恢复1AP / Dmg+15% — restore 1AP |
| 普通 / Normal | ≤0.5秒偏差 / ≤0.5s off | 标准伤害 / Standard dmg |
| 失误 / Miss | >0.5秒偏差 / >0.5s off | 伤害减半——无AP恢复 / Dmg halved — no AP restore |

**每角色最优QTE帧**: 每个角色的完美QTE窗口帧不同。艾伦: 攻击动画挥刀至最高点时按。塞拉: 法阵收缩到最小时按。主角: 盾牌挡在前方时按。维克多: 光球到最高点时按。莉莎: 召唤物出现瞬间按 / Each character has unique perfect frame — learn timing in training room.

## 战斗站位与仇恨系统 / Battle Positioning & Aggro

**仇恨机制**: 坦克(主角/古斯塔夫)使用嘲讽技能强制Boss攻击自己——每嘲讽1次增加仇恨值(持续5回合)。DPS和治疗的高伤害/高治疗会累积仇恨——超出坦克仇恨值后Boss会转头攻击 / Aggro: Tank (Protag/Gustave) taunt forces boss attack — 1 taunt = aggro value (5 turns). DPS/heal high actions build aggro — exceeds tank aggro = boss switches target.

**站位推荐**: 坦克>前排——DPS>中排——治疗>后排。物理DPS(艾伦)和坦克站同侧(吃相同AOE)——法系(塞拉)和后排(维克多/莉莎)站另一侧——避免全队被范围攻击同时覆盖 / Recommended formation: Tank front, DPS mid, Heal back. Phys DPS (Allen) same side as tank — magic (Sera) + backline (Victor/Lisa) other side — avoid full party AoE coverage.

**换位技巧**: 战斗中每回合可以免费换位1次——额外换位消耗10AP。面对范围攻击时——把脆皮拉到安全位置——让坦克去吃AOE——这是最关键的操作技巧 / Position swap: 1 free swap per turn — extra swaps cost 10AP. Pull squishies to safety against AoE — tank eats the hit — most critical combat technique.

**防御时机**: 不是所有攻击都需要防御——普通攻击可以硬吃——必杀技必须防御。必杀技前摇特征: Boss身上出现红光+屏幕边缘闪烁——此时按防御键进入QTE防御模式——完美防御可以完全无伤并恢复2AP。学会区分普通攻击和必杀技是进阶第一课 / **Guard timing**: Not all attacks need guarding — normal hits can be tanked — ults must be guarded. Ultimate tell: boss glows red+screen edge flash — press guard for QTE guard mode — perfect guard = zero dmg + 2AP restore. Distinguishing normal vs ultimate is advance lesson #1.
