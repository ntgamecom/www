# 新手入门 / Beginner Guide

33号远征队的回合制战斗有着独特的行动点系统和QTE判定——理解这两个核心机制是玩好游戏的关键 / Clair Obscur: Expedition 33's turn-based combat features a unique Action Point system and QTE timing — mastering these two core mechanics is key to playing well.

---

## 行动点系统 / Action Point System

| 概念 / Concept | 说明 / Description |
|:----:|------|
| **基础AP / Base AP** | 每个角色每回合获得4点行动点 / Each character gets 4 AP per turn |
| **技能消耗 / Skill Cost** | 技能消耗1-3AP，终极技消耗4AP / Skills cost 1-3 AP, ultimates cost 4 AP |
| **道具消耗 / Item Cost** | 大部分道具消耗1AP / Most items cost 1 AP |
| **防御 / Defend** | 消耗1AP——75%减伤+回复少量HP / Costs 1 AP — 75% damage reduction + small HP regen |
| **待机 / Wait** | 不消耗AP——角色跳过回合——下回合获得+1额外AP / Costs 0 AP — skip turn — next turn gets +1 bonus AP |
| **AP上限 / AP Cap** | 最大AP为10——通过装备和被动技能提升 / Max AP is 10 — increased via gear and passives |

**AP管理核心技巧**: 不要每回合都用完所有AP——留1-2AP用于防御或应对突发情况。待机战术在需要爆发时非常有效——待机一回合下回合有5AP可以连放两个2AP技能 / **AP Management Tip**: Don't spend all AP every turn — save 1-2 AP for defense or emergencies. Wait tactic is great for burst setup — wait one turn for 5 AP next turn to chain two 2-AP skills.

## QTE系统 / QTE System

每个攻击和技能命中时触发QTE——**在命中瞬间再次按下技能键**触发完美判定。普通命中 = 100%伤害。完美命中 = 150%伤害 + 额外效果(击退/眩晕/附加状态)。**防御QTE**: 被攻击时按防御键——完美防御 = 完全免伤 + 反击机会 / Every attack and skill triggers QTE on hit — **press skill key again at impact moment** for perfect timing. Normal hit = 100% damage. Perfect hit = 150% damage + bonus effect (knockback/stun/status). **Defense QTE**: Press defend when attacked — perfect defend = full immunity + counterattack chance.

### QTE练习方法

在学院的训练室可以无限次练习QTE。推荐新玩家在这里花10分钟熟悉每个角色的QTE节奏。**进阶技巧**: 完美QTE也受角色敏捷属性影响——敏捷越高完美判定窗口越宽。堆敏捷的角色更容易打出完美QTE / Training room at Academy allows unlimited QTE practice. Recommended: 10 min to learn each character's rhythm. **Advanced**: Perfect QTE window also scales with Dexterity — higher Dex = wider window. Dex-stacking characters land perfect QTEs more easily.

## 装备基础 / Equipment Basics

装备分为**白色(普通)>绿色(优秀)>蓝色(精良)>紫色(史诗)>橙色(传说)**五个品质。装备可以通过**锻造**升级(+1至+5)——每次升级提升基础属性20%。**镶嵌系统**: 装备有插槽——可以镶嵌宝石获得额外属性——宝石也可以合成升级。推荐优先升级武器(+5比+0的伤害翻倍) / Gear tiers: **White > Green > Blue > Purple > Orange**. Gear can be upgraded via **crafting** (+1 to +5) — +20% base stats per level. **Socket System**: Gear has sockets for gems — gems also upgradeable via crafting. Priority: upgrade weapons first (+5 = 2x damage vs +0).

## 新手常见错误 / Common Newbie Mistakes

1. **AP用完不留防御**——经常被打到残血。留1AP防御可以大幅度降低生存压力 / Spending all AP without saving for defense — leads to critical HP. Save 1 AP for defend.
2. **忽视QTE**——完美QTE是150%伤害而不完美是100%——长期差距巨大 / Ignoring QTE — perfect = 150% vs normal = 100% — massive long-term gap.
3. **不升级装备**——前期锻造材料充足——把武器升到+3可以提高50%输出 / Not upgrading gear — plenty of early crafting mats — +3 weapon = +50% DPS.
4. **忽略支线任务**——支线给予大量经验和永久属性奖励——第七章前尽可能多完成 / Skipping side quests — they give big XP and permanent stat rewards — do as many as possible before Ch.3.
5. **队伍搭配单一**——不要全队同一属性伤害——遇到抗性Boss会打不动 / One-dimensional party — don't use same damage type — countered by resist bosses.

## 推荐新人Build / Recommended New Build

**主角(力量流)+艾伦(敏捷流)+塞拉(智力流)**: 三种伤害类型均衡——适应任何情况。主角坦克+输出、艾伦单体爆发、塞拉AOE清图。AP分配: 主角主防御+控制——艾伦主输出——塞拉辅助清图。这个组合可以无压力通关全剧情 / **Protagonist (STR) + Allen (DEX) + Sera (INT)**: Balanced three damage types — adapts to any situation. Protagonist tank+DPS, Allen single-target burst, Sera AoE clear. AP: Main defend+control, Allen main DPS, Sera support+clear. This combo clears all story content easily.

## 新手指南常见错误 / Common Beginner Mistakes

1. **忽视AP管理**: 前期AP非常有限——不要把所有AP都用在攻击上——至少保留1-2AP用于防御和紧急闪避。AP用光的角色在BossAOE时无法防御——可能被秒 / Ignoring AP management — keep 1-2 AP for defense and emergency dodge. Empty AP = can't defend against boss AoE = possible one-shot.
2. **不升级装备**: 很多新玩家只顾升等级不管装备——但33号远征队的装备提升约等于3-5级的属性收益。推荐每到一个新区域更新武器和胸甲 / Not upgrading gear — equipment boost equals 3-5 levels of stats. Update weapon and chest armor every new zone.
3. **无视QTE练习**: 完美QTE=伤害+30%——前期可以不追求——但中后期不练完美QTE会卡Boss。推荐在训练场花10分钟练主力技能的QTE节奏 / Ignoring QTE practice — perfect QTE = +30% damage. Fine to ignore early game but mid-late game boss walls require perfect QTE. Practice main skills at training ground.
4. **单一属性队伍**: 全队同一种属性遇到抗性Boss无法处理——至少两种属性输出。推荐物理+法系混搭 / Single element team — get walled by resistant bosses. At least 2 damage types. Recommend phys+magic hybrid.
5. **不探索隐藏路径**: 33号远征队的地图有大量分支路径——很多连主路都不显示——里面有更好的装备、更多的AP恢复点和捷径 / Not exploring hidden paths. Maps have many branches not visible from main path — better gear, more AP recovery points, and shortcuts.

## 推荐升级路线 / Recommended Progression

**第一章** → 主角单人战——熟悉基础QTE和AP系统。**第二章** → 队伍扩大到3人——开始练习配合——优先解锁艾伦和塞拉——物理+法系=覆盖全面。**第三章** → 队伍满5人——开始深入角色Build——每个角色确定主方向。**第四章** → 终章前准备——装备全强化——收集隐藏遗物——准备终局Boss战。**终局** → 通关后深渊裂隙+竞技场——刷终极装备和成就 / **Ch.1**: Solo protagonist — learn QTE and AP basics. **Ch.2**: Party of 3 — practice combos — unlock Allen+Sera first. **Ch.3**: Full 5-person party — build characters — each pick main direction. **Ch.4**: Pre-final prep — max gear — collect hidden relics — prepare for final boss. **Endgame**: Abyss Rift + Arena — farm ultimate gear and achievements.

## 战斗系统进阶理解 / Advanced Combat Understanding

**QTE完美时机**: 每个角色攻击动画不同——完美QTE时机不同——艾伦的疾风斩在第三段动画的闪光瞬间按——塞拉的火焰风暴在法阵完全展开时按——主角的铁壁在被击中的前一帧按。训练场(第三章解锁)可以反复练习每个技能的QTE时机 / **Perfect QTE Timing**: Each character has unique animation timing — Allen's Wind Slash = 3rd animation flash moment — Sera's Firestorm = when glyph fully expands — Protag's Iron Wall = frame before hit. Training room (Ch.3) for practice.

**AP管理策略**: 战斗初期不急着用AP——先攒到50AP(上限)——然后爆发: 先用加Buff技能(增加伤害/暴击)——再用高AP消耗技能(终极技/高倍率技能)——保留10-20AP用于紧急回避和治疗。维克多的时空锁(15AP)在Boss蓄力时使用可以打断必杀技——这是最关键的AP用途之一 / **AP Management**: Early fight — save AP to 50 cap — then burst: buff first (dmg/CR buffs) — then high-cost skills (ultimates) — keep 10-20AP for dodge/heal. Victor's Time Lock (15AP) can interrupt boss charge attacks — most critical AP use.

**元素反应系统**: 火+冰=蒸发——1.5倍伤害——降低敌人抗性。火+雷=超载——AoE爆炸——周围敌人击退。冰+雷=超导——降低敌人防御20%持续3回合。三种以上异常状态同时存在=元素风暴——全队额外+50%伤害持续1回合。**推荐连招**: 塞拉先火焰风暴→然后冰霜新星=蒸发→接着闪电链=超载→最后暴风雪=继续叠冰异常 / **Element Reactions**: Fire+Ice = Vaporize (1.5x dmg, lower enemy resist). Fire+Lightning = Overload (AoE blast, knockback). Ice+Lightning = Superconduct (lower enemy DEF 20% — 3 turns). 3+ status effects = Elemental Storm (+50% dmg 1 turn). **Recommended combo**: Sera Firestorm → Frost Nova (Vaporize) → Chain Lightning (Overload) → Blizzard (stack ice).

## 游戏设置优化 / Game Settings Optimization

**推荐设置**: 战斗速度→1.25x(默认1x太长——1.5x太快看不清QTE)。QTE提示→开启"提前指示"(画面边缘闪光——比音效提示更快)。自动存档→开启。对话自动推进→关闭(免得错过关键信息)。视角→战斗视角改为"动态"(更好观察战场全貌) / **Recommended settings**: Battle speed 1.25x (default too slow — 1.5x too fast for QTE). QTE hint: enable "Advance Warning" (screen edge flash — faster than audio). Auto-save: ON. Auto-advance dialogue: OFF. Camera: "Dynamic" for better battlefield view.

**按键设置**: QTE确认键→建议绑定到手感最好的键(空格键或左键)。快捷换位→绑定到鼠标侧键。快捷技能→1234对应4个角色——QERT对应4个技能槽——F为终极技能。**手柄推荐**: QTE用A/B/X/Y——换位用LB/RB——技能用方向键+ABXY组合 / **Keybinds**: QTE confirm → bind to most comfortable key (Space or left click). Quick swap → mouse side button. Quick skills: 1234=characters, QERT=skill slots, F=ultimate. **Controller**: QTE with A/B/X/Y, swap with LB/RB, skills with dpad+ABXY.
