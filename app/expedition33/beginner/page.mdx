# 新手入门 / Beginner Guide

33号远征队的回合制战斗有着独特的行动点系统和QTE判定——理解这两个核心机制是玩好游戏的关键 / Clair Obscur: Expedition 33's turn-based combat features a unique Action Point system and QTE timing — mastering these two core mechanics is key to playing well.

---

## 行动点系统 / Action Point System

| 概念 / Concept | 说明 / Description |
|:----:|------|
| **基础AP / Base AP** | 每个角色每回合获得4点行动点(Action Points) / Each character gets 4 AP per turn |
| **攻击消耗 / Attack Cost** | 轻击=1AP、重击=2AP、技能=2-3AP、强力技能=4AP / Light=1AP, Heavy=2AP, Skill=2-3AP, Ultimate=4AP |
| **防御消耗 / Defense Cost** | 闪避=1AP、格挡=1AP、反击=2AP(需要完美格挡后) / Dodge=1AP, Block=1AP, Counter=2AP (after perfect block) |
| **道具消耗 / Item Cost** | 使用道具=1AP / Use item = 1AP |
| **AP恢复 / AP Recovery** | 每回合自动恢复全部AP。部分技能和遗物可以增加每回合AP上限 / Full AP restore each turn. Some skills and relics increase AP cap |
| **未使用AP / Unused AP** | 本回合未消耗的AP不会累积到下回合——**每回合用光AP** / Unused AP does NOT carry over — **spend all AP each turn** |

**核心战术**: 每回合尽量用完全部AP——留AP不用=浪费输出。除非需要留1AP应急闪避 / **Core tactic**: Spend all AP each turn — leaving AP unused = wasted DPS. Exception: save 1AP for emergency dodge.

## QTE系统 / QTE System

| 判定 / Judgment | 区域 / Zone | 效果 / Effect |
|:----:|:----:|------|
| **完美 / Perfect** | 绿色区域中心 / Center of green zone | 伤害x1.5 + 额外效果触发 / 1.5x damage + bonus effect |
| **良好 / Good** | 绿色区域边缘 / Edge of green zone | 伤害x1.0 / 1.0x damage |
| **失误 / Miss** | 绿色区域外 / Outside green zone | 伤害x0.5 / 0.5x damage |

**QTE类型**: 攻击时按圈/空格(伤害加成)、闪避时按圈(无敌帧延长)、格挡时按圈(减伤率提升)、连携技时按顺序按键(全队增伤) / **QTE types**: Attack (dmg boost), Dodge (i-frame extension), Block (dmg reduction), Combo (party-wide dmg buff).

## 新手成长路线 / Beginner Progression Path

| 阶段 / Phase | 目标 / Goal | 推荐行动 / Recommended Actions |
|:----:|:----|:----|
| **1. 熟悉基础 / Learn Basics** (1-2小时) | 掌握AP管理和QTE / Master AP + QTE | 完成白崖镇探索→做QTE练习模式→熟悉4角色技能 / Complete White Cliff → QTE practice → learn 4 skills per char |
| **2. 装备积累 / Gear Up** (2-5小时) | 武器+3, 防具升级 / Weapon +3, armor upgraded | 灰烬森林探全图→收集铁矿石→升级古斯塔夫和露娜武器 / Explore Ashen Forest fully → collect Iron Ore → upgrade Gustave + Luna |
| **3. 元素精通 / Element Master** (5-10小时) | 熟练掌握元素连携 / Master elemental combos | 冰封湖→学习冰+水冻结连携→购买全回复药→刷精炼钢 / Frozen Lake → learn Freeze combo → buy Mega Potions → farm Refined Steel |
| **4. 阵容成型 / Party Setup** (10-15小时) | 确定主力4人阵容 / Finalize main 4 | 锈蚀工厂→解锁维克多终极技能→收集防具套装→武器+4 / Rusted Factory → Victor's ultimate → collect armor sets → weapon +4 |
| **5. 终局准备 / Endgame Prep** (15-20小时) | 全队+5装备, 终极技能全解锁 / All +5 gear, all ultimates | 画师之塔→收集最终武器蓝图→全隐藏宝藏→准备最终Boss / Painter's Tower → final blueprints → all secrets → prep for final boss |

## 队伍搭配 / Party Composition

| 角色 / Character | 定位 / Role | 推荐搭配 / Synergies |
|:----:|:----:|:----|
| **古斯塔夫(Gustave)** | 前排输出+坦克 / Frontline DPS + Tank | 配玛艾勒(治疗)或露娜(控制) / Pair with Maelle (heal) or Luna (control) |
| **玛艾勒(Maelle)** | 治疗+辅助 / Healer + Support | 通用——任何队伍都需要 / Universal — every party needs |
| **露娜(Luna)** | 法师+控制 / Mage + Control | 配古斯塔夫(冰→物理增伤) / Pair with Gustave (freeze → phys dmg bonus) |
| **泽农(Zenon)** | 刺客+爆发 / Assassin + Burst | 配露娜(冰冻后暴击必中) / Pair with Luna (guaranteed crit on frozen) |
| **维克多(Victor)** | 工程师+召唤 / Engineer + Summon | 配任意前排角色 / Pair with any frontline |
| **艾莉丝(Iris)** | 弓手+标记 / Archer + Mark | 配泽农(标记+背刺=爆发) / Pair with Zenon (mark + backstab = burst) |

**开荒推荐**: 古斯塔夫+玛艾勒+露娜——打不过就换维克多(召唤物吸引仇恨) / **Beginner party**: Gustave + Maelle + Luna. Swap Victor for fights where you need a tank.

## 元素系统 / Elemental System

| 元素 / Element | 效果 / Effect | 连携 / Combo |
|:----:|:------|:----:|
| **物理 / Physical** | 基础伤害——无特殊效果 / Base damage | — |
| **火焰 / Fire** | 灼烧DoT(3回合) / Burn DoT (3 turns) | 火+油=爆炸(范围伤害x2) / Fire+Oil=Explosion (AoE x2) |
| **冰霜 / Ice** | 减速(减AP恢复速度)+概率冻结 / Slow (+reduce AP regen) + chance to freeze | 水+冰=冻结(目标跳过一回合) / Water+Ice=Freeze (skip 1 turn) |
| **闪电 / Lightning** | 传导+概率麻痹 / Conduct + chance to paralyze | 雷+水=传导(伤害溅射到相邻敌人) / Lightning+Water=Conduct (chain to adjacent enemies) |
| **暗影 / Shadow** | 减攻击+恐惧(概率逃跑) / ATK down + fear (chance to flee) | 暗+火=烟雾(闪避率上升) / Shadow+Fire=Smoke (+evasion) |

## 新手常见错误 / Common Beginner Mistakes

| 错误 / Mistake | 正确做法 / Correct |
|:----:|------|
| **忽略QTE / Ignoring QTE** | 完美QTE的1.5倍伤害在Boss战中等于多打50%输出——练好QTE / Perfect QTE = 50% more damage in boss fights — practice it |
| **AP不用完 / Not spending AP** | 每回合4AP至少用3AP。留太多AP=输出浪费 / Spend at least 3AP per turn. Wasted AP = wasted damage |
| **忽视元素反应 / Ignoring elements** | 冰+水=冻结(敌方跳过一回合)——这比任何单体技能都强 / Water+Ice=Freeze (enemy skips a turn) — stronger than any single skill |
| **不升级装备 / Skipping gear upgrades** | 武器每升级一次=伤害+15%——素材在探索中获得。每到一个新区域先找铁匠 / Each weapon upgrade = +15% damage — materials found through exploration. Find smith in each new zone |
| **不用道具 / Not using items** | 回复药和Buff道具在Boss战中极其重要——不要省着不用 / Potions and buff items are crucial in boss fights — don't hoard them |
