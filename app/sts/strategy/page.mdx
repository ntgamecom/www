# 策略技巧 / Strategy & Tips

杀戮尖塔不是单纯比谁的卡牌更强——路线选择、经济管理、卡牌取舍和风险控制同样重要。本章覆盖通关策略的核心原则 / Slay the Spire isn't just about having stronger cards — route planning, economy management, card selection and risk control are equally important. This chapter covers the core principles of winning strategy.

---

## 路线选择 / Route Planning

### 地图要素 / Map Elements

| 图标 / Icon | 类型 / Type | 建议 / Advice |
|---|---|---|
| **怪物 / Monster** | 普通战斗——获得金币+有机会掉落卡牌 / Normal fight — gold + card drop chance | 前期多打怪——积累金币和卡牌 / Fight more early — accumulate gold and cards |
| **精英 / Elite** | 强力战斗——必定掉落遗物 / Hard fight — guaranteed relic drop | 根据你的强度决定——打得过就全打。打不过绕路 / Fight if you're strong enough — skip if under-leveled |
| **休息处 / Rest Site** | 回血或强化卡牌 / Heal or Smith | 新手太容易选择回血——实际上强化更重要。除非生命&lt;50% / Beginners heal too much — smith is usually better unless HP &lt;50% |
| **商店 / Shop** | 购买卡牌、遗物、药水 / Buy cards, relics, potions | 优先购买关键遗物和核心卡 / Prioritize key relics and core cards |
| **宝箱 / Treasure** | 获得遗物——普通或稀有 / Get relic — common or rare | 必去——免费遗物 / Always take — free relic |
| **事件 / Event** | 随机事件——有好有坏 / Random event — good or bad | 了解事件结果后可以做最优选择 / Optimal choices available with event knowledge |

### 路线决策 / Route Decisions

| 场景 / Scenario | 推荐路线 / Route |
|---|---|
| **前期(Act 1)** | 多打怪+精英——积累金币和经验。1个休息处足够。目标是3-4精英 / Fight many + elites — accumulate gold and XP. 1 rest site is enough. Aim for 3-4 elites |
| **中期(Act 2)** | 更多商店+事件。精英选择性打——Act 2的精英比Act 1难很多 / More shops + events. Selective elite fights — Act 2 elites are much harder |
| **后期(Act 3)** | 2-3精英+1-2商店+1休息处。确保装备打心/最终Boss / 2-3 elites + 1-2 shops + 1 rest. Ensure readiness for Heart/final boss |
| **低血量时** | 优先休息处——死在路上等于白打 / Prioritize rest sites — dying on the road wastes the run |
| **经济充裕时** | 商店多的路线——花钱买关键卡 / Routes with more shops — spend gold on key cards |

## 经济管理 / Economy Management

| 原则 / Principle | 说明 / Description |
|---|---|
| **攒钱到商店** | Act 1的商店在第一个Boss前——你需要至少$150-200来买关键遗物或卡牌 / Act 1 shop is before the first boss — need $150-200 for key relic or card |
| **跳过不必要的卡牌奖励** | 三张牌都不适合你的Build？跳过——不要被"免费"诱惑增加牌组厚度 / None of 3 cards fit your build? Skip — don't bloat your deck with freebies |
| **删牌优先级** | 初始的"打击"和"防御"是废牌——在商店花$75删掉它们。优先删"打击" / Starting Strikes and Defends are junk — pay $75 to remove them. Prioritize Strike removal |
| **药水值得买** | $50-100的药水在关键时刻救你一命——特别是烟雾弹和力量药水 / $50-100 potions can save a run — especially Smoke Bomb and Strength Potion |
| **遗物优先于卡牌** | 同样的$150——买一个稀有遗物比买3张普通卡更有价值 / Same $150 — a rare relic is more valuable than 3 common cards |

## 卡牌评估原则 / Card Evaluation Principles

### 什么时候拿牌 / When to Take Cards

| 情况 / Situation | 决策 / Decision |
|---|---|
| **缺输出** | 拿高伤害攻击牌——特别是AOE(如"旋风斩"、"扫射") / Take high-damage attacks — especially AOE (Whirlwind, Sweeping Beam) |
| **缺防御** | 拿高效叠甲牌——如"金属化"、"护甲" / Take efficient block cards — Metallicize, Leap |
| **缺抽牌** | 拿过牌——如"战斗专注"、"快速检索" / Take draw cards — Battle Trance, Quick Slash |
| **你的核心卡已经足够** | 跳过——牌组越精简越好 / Skip — thinner deck is better |
| **看到稀有卡但不好用** | 跳过——稀有卡不一定适合你的Build / Skip — rare doesn't always mean right for your build |

### 牌组大小管理 / Deck Size Management

| 牌组大小 / Deck Size | 评价 / Evaluation |
|---|---|
| **15-25张** | 极精简——适合无限连击Build。观者最容易达到 / Very thin — suitable for infinite builds. Watcher achieves this easiest |
| **25-35张** | 理想范围——足够的伤害+防御+过牌。大部分通关的牌组在这个范围 / Ideal range — enough damage + block + draw. Most winning decks in this range |
| **35-45张** | 偏大——仍然可以通关但需要额外的过牌卡 / Large — can still win but needs extra draw power |
| **45+张** | 太大——抽牌不稳定。建议在商店删牌 / Too large — inconsistent draws. Remove cards at shop |

## 通用策略 / General Strategy

### 战斗策略 / Combat Strategy

| 策略 / Strategy | 说明 / Description |
|---|---|
| **锋面清理** | 除非特定情况——先打伤害最低的敌人(通常是小兵)来减少每回合受到的伤害总量 / Unless specific scenario — kill lowest-damage enemy first to reduce total incoming damage |
| **精英战斗节奏** | 精英战斗需要规划——先在普通战斗中测试Build，再打精英 / Elite fights need planning — test your build on normal fights first, then take elites |
| **药水使用** | "留着以后用"的心态会让你死——好的药水现在用可能就是活下来的关键 / "Save for later" mindset kills — a good potion now might be the key to survival |
| **生命是资源** | 不要为了保持满血而牺牲输出机会。生命值只是一种资源——只要不死就可以用 / Don't sacrifice DPS to stay at full HP. HP is just a resource — spend it as long as you don't die |

### 各阶段策略重点 / Phase-Specific Strategy

| 阶段 / Phase | 重点 / Focus |
|---|---|
| **Act 1(第1-3关)** | 找AOE攻击牌——应对"三奴隶"等群战。优先升级核心攻击牌 / Find AOE attacks — for Slavers and other group fights. Upgrade core attacks first |
| **Act 2(第4-6关)** | 补足防御——Act 2的敌人单发伤害很高。找到你的Build核心卡 / Supplement block — Act 2 enemies hit hard per hit. Find your build core cards |
| **Act 3(第7-9关)** | 综合强化。确保你的Build已经成型——升满核心卡 / Comprehensive strengthening. Ensure your build is complete — upgrade all core cards |
| **Act 4(心)** | 律动对策(每打一张牌掉1血)——需要高效卡牌而不是大量小卡 / Beat of Death counter (1 dmg per card) — need efficient cards, not many small cards |

## 常见新手错误 / Common Beginner Mistakes

| 错误 / Mistake | 频率 / Frequency | 解决方法 / Fix |
|---|---|---|
| **拿太多卡牌** | 极高 / Very High | 卡牌不是越多越好——15-25张是理想范围。学会跳过(C牌都差时) / More cards isn't better — 15-25 is ideal. Learn to skip when all 3 are bad |
| **休息处总是回血** | 高 / High | 除非血量危急(&lt;40%)——选择强化卡牌。强化后的卡牌可以让你少受更多伤害 / Unless HP is critical (&lt;40%) — choose Smith. Upgraded cards prevent more damage than healing |
| **忽视精英战斗** | 高 / High | 精英掉落的遗物是关键——遗物是永久性增强，比卡牌更重要。但Act 2的精英比Act 1难很多 / Elite relics are key — permanent upgrades more important than cards. But Act 2 elites are much harder than Act 1 |
| **不看敌人意图** | 中 / Medium | 观察敌人头上的图标——攻击/防御/强化。对应更换策略 / Watch enemy intent icons — attack/block/buff. Adapt strategy accordingly |
| **不删初始卡** | 中 / Medium | 初始的"打击"和"防御"牌效率很低——在商店花$75删掉。优先删"打击" / Starting Strikes and Defends are inefficient — pay $75 to remove them. Prioritize Strike removal |
| **低估药水** | 中高 / Medium-High | 药水能救命——烟雾弹可以让你从必死精英手中逃命。力量药水爆发用 / Potions save runs — Smoke Bomb escapes lethal elites. Strength Potion for burst |

## 高阶技巧 / Advanced Techniques

### 卡牌计数 / Card Counting

| 技巧 / Technique | 说明 / Description |
|---|---|
| **牌组循环预测** | 如果你知道牌组的剩余牌——可以预判下一轮会抽到什么牌。特别是精简牌组(&lt;15张)时 / If you know remaining deck cards — predict next draw. Especially useful with thin decks (&lt;15 cards) |
| **概率计算** | 牌组中核心卡的张数/总牌数=抽中概率。如果你有2张催化剂在30张牌中——抽到一张的概率约6.7% / Core cards / total = draw chance. 2 Catalysts in 30 cards = ~6.7% chance per card |
| **精英战斗容错** | 如果你在Act 1遇到"三奴隶"精英——AOE是必需品。没有AOE时考虑绕路 / If Act 1 has "Slavers" elite — AOE is mandatory. Consider alternative route without AOE |

### 进阶资源利用 / Advanced Resource Use

| 资源 / Resource | 利用方式 / Usage |
|---|---|
| **生命值** | 生命值是可以消耗的资源——如果你确定下一场精英战斗能赢但会掉血，打是值得的 / HP is a consumable resource — if you're sure you can win the next elite fight but take damage, it's worth it |
| **金币** | 不要只花在卡牌上——遗物优先。商店中的"删牌"服务是性价比最高的消费 / Don't just spend on cards — relics first. Card removal at shop is the best value purchase |
| **药水** | "留着关键时刻用"——但关键时刻往往就是现在。如果你现在不用可能会死——那就用 / "Save for emergencies" — the emergency is usually now. If not using means you might die — use it |
| **强化机会** | 优先强化最核心、使用次数最多的牌——而不是想"以后可能会用到"的牌 / Upgrade the card you use most in combat — not cards you "might use later" |

## 进阶20模式下调整 / A20 Adjustments

进阶20下游戏发生了以下变化——策略需要相应调整 / At A20, the game changes significantly — strategy must adapt.

| 变化 / Change | 影响 / Impact | 应对 / Response |
|---|---|---|
| **怪物血量+50%** | 战斗时间变长——续航更重要 / Longer fights — sustain matters more | 拿一些恢复手段(如"燃烧之血"、"回复") / Take healing sources (Burning Blood, Regen) |
| **精英更难** | Act 1精英已经够难——Act 2精英更致命 / Act 1 elites are already tough — Act 2 elites are lethal | 确保有AOE再打Act 2精英 / Ensure AOE before fighting Act 2 elites |
| **Boss获得额外能力** | 所有Boss增加一个新招式——有些能力很致命 / Each boss gains a new move — some are lethal | 了解每个Boss的A20新增能力——提前防备 / Learn each boss's new A20 move — prepare in advance |
| **商店价格翻倍** | 经济压力大——不能随心所欲买东西 / Economic pressure — can't buy freely | 更谨慎消费——核心遗物优先 / Spend wisely — prioritize core relics |
| **事件负面概率增加** | 事件更多是坏的——"诅咒"出现率提高 / Events are more often bad — higher curse rate | 有"蓝蜡烛"或"净化"时选择高风险事件 / Take risky events only with Blue Candle or Purify |
