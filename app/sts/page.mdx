## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|---|---|
| **卡牌构筑 / Deck Building** | 每局游戏起始约10张打击/防御——通过获取新卡牌逐步构筑你的专属卡组。每张卡牌的加入和移除都在塑造最终构筑 / Start with ~10 strikes/defends — build your unique deck by adding/removing cards each run |
| **遗物系统 / Relics** | 强力被动道具——获得后永久生效(消除后消失)。遗物之间的联动可以产生极其强大的组合效果 / Powerful passive items — permanent once obtained (lost on removal). Relic synergies create extremely powerful combos |
| **四角色系统 / Four Characters** | 每个角色有独特的卡牌池、遗物和机制——铁甲(烧伤/格挡)、静默(毒/弃牌)、机器人(球体/聚焦)、观者(姿态/神格) / Each has unique cards, relics, mechanics — Ironclad(burns/block), Silent(poison/discard), Defect(orbs/focus), Watcher(stances/divinity) |
| **路线选择 / Route Planning** | 在地图上选择路线——篝火→商店→精英→Boss。不同路线意味着不同的风险和收益 / Choose your path on the map — campfires → shops → elites → boss. Routes = different risks and rewards |
| **进阶模式 / Ascension** | 通关后解锁进阶模式——1-20级。每级增加新负面效果——20级是真正的终局挑战 / Unlock after beating the game — levels 1-20. Each adds new penalties — Ascension 20 is the true endgame |
| **每日挑战 / Daily Climb** | 每日随机种子+特殊规则——所有玩家同种子比拼分数 / Daily random seed + special rules — all players compete on same seed |

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **删牌比加牌重要 / Remove Cards > Add Cards** | 卡组越精越好——商店看到"移除"服务一定用。减少废牌=提高关键牌上手率 / Tighter deck = better — always use shop removal. Fewer bad cards = more consistent key card draws |
| **2** | **精英怪值得打 / Elites Are Worth It** | 精英掉落遗物(非常强力)+更多金币。但要注意——精英战会扣血——评估自己能不能打 / Elites drop relics (very powerful) + more gold. But consider your HP — assess if you can handle them |
| **3** | **篝火升级优先于回血 / Upgrade Over Heal** | 残血时很想回血——但升级卡的收益通常更大。除非血量濒危(&lt;30%)，否则尽量升级 / You want to heal — but upgrading cards is usually better. Unless critically low (&lt;30%), prioritize upgrades |
| **4** | **路线规划看地图 / Plan Route on Map** | 开地图时确认精英位置和Boss类型——根据你的构筑选择最有利的路线 / Confirm elite locations and boss type — choose the most favorable route for your build |
| **5** | **前期多拿攻击牌 / Prioritize Attacks Early** | Act 1的小怪伤害不高但你需要快速击杀——至少拿3-4张优质攻击牌再考虑防御 / Act 1 mobs aren't tough but you need burst — get 3-4 good attacks before considering defense |
| **6** | **药水不要省 / Use Potions Generously** | 药水到下一层不会保留——该用就用，尤其是在精英战和Boss战中 / Potions don't carry forward — use them freely, especially in elite and boss fights |

## 游戏阶段 / Game Phases

| 阶段 / Phase | 包含内容 / Content |
|---|---|
| **Act 1** | 3个Boss之一(史莱姆Boss/守护者/六火亡魂)——新手区——卡组初步成型 / One of 3 bosses (Slime Boss/Guardian/Hexaghost) — starter area — deck takes initial shape |

## 常见问题 / FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **哪个角色最简单？** / Easiest character? | 战士(Ironclad)——力量流简单粗暴，血量高容错大 / Ironclad — strength build is straightforward, high HP, forgiving |
| **进阶模式有什么变化？** / Ascension changes? | 每级增加一个负面效果——如精英怪更强、Boss血量增多、商店涨价等。A20是最终挑战 / Each level adds a penalty — stronger elites, more boss HP, shop price hikes. A20 is the ultimate challenge |
| **牌组多大合适？** / Ideal deck size? | 大多数流派20-30张最佳——过牌流可以更大但不要超过35 / Most builds: 20-30 cards ideal — draw builds can go bigger but keep under 35 |
| **删牌还是买牌？** / Remove or add cards? | 优先删牌！牌组越精越好——减少废牌=提高关键牌上手率 / Prioritize removal! Tighter deck = better consistency — fewer bad cards = more key draws |
| **什么时候该跳过卡牌奖励？** / When to skip card rewards? | 牌组已经不需要新卡的时候——不要为了"可能有用"而拿牌 / Skip when your deck doesn't need anything new — don't take cards "just in case" |

## 核心策略 / Core Strategy

| 阶段 / Phase | 策略 / Strategy |
|---|---|
| **选择角色** / Pick Character | 新手推荐战士(Ironclad)——力量流简单 / Beginners: Ironclad — strength build is straightforward |
| **开局路线** / Starting Path | Act1多打精英(拿遗物)——至少打2个精英 / Act1: hunt elites for relics — at least 2 elites |
| **篝火优先** / Campfire Priority | 升级 > 回血——除非血量&lt;30%——卡牌升级的收益远大于几点HP / Upgrade > heal — unless HP < 30% — card upgrade value far outweighs a few HP |
| **删牌选择** / Card Removal | 优先删除打击(Strike)——防御好牌多但好攻击牌也多——打击是前期最废的 / Remove Strikes first — good block cards are plentiful but good attacks are too — Strikes are the weakest starter card |
| **商店策略** / Shop Strategy | 删牌 > 关键牌 > 遗物 > 药水——删牌是商店最值的消费 / Removal > Key card > Relic > Potion — removal is the best shop purchase |

## 药水用途 / Potion Usage

| 药水 / Potion | 效果 / Effect | 使用时机 / When to Use |
|---|---|---|
| **力量药水 / Strength Potion** | +2力量(战斗内)/ +2 Strength (combat only) | 精英战/Boss战开场吃——爆发输出最大化 / Pop at elite/boss fight start — max burst damage |
| **格挡药水 / Block Potion** | +12格挡 / +12 Block | 紧急保命——敌人要打出大伤害时 / Emergency survival — when enemy is about to hit hard |
| **能量药水 / Energy Potion** | +2能量(本回合)/ +2 Energy (this turn) | 需要爆发的时候——配合大牌打combo / When you need burst — combo with big cards |
| **敏捷药水 / Dexterity Potion** | +4敏捷(战斗内)/ +4 Dexterity (combat only) | 配合格挡卡——每个格挡卡多4格挡 / Combo with block cards — each block card gains +4 |
| **幽灵药水 / Ghost in a Jar** | 获得1层无实体 / Gain 1 Intangible | 面对高伤害攻击时——完美抵消一次大伤害 / Against a massive hit — perfectly negates one big attack |
| **抽牌药水 / Draw Potion** | 抽5张牌 / Draw 5 cards | 需要关键牌时——缺过牌时用 / When you need a key card — or lack draw power |
