# 战士攻略 / Ironclad Guide

战士（Ironclad）是杀戮尖塔的第一位角色，擅长高伤害输出和强力格挡，核心机制是**燃烧**和**力量** / The Ironclad is the first character, excelling at high damage output and powerful blocking. Core mechanics: **Exhaust** and **Strength**.

**初始遗物 / Starting Relic**: 燃烧之血（Burning Blood）—— 每场战斗结束后回复6HP / Heal 6 HP after each combat

---

## 核心机制 / Core Mechanics

| 机制 / Mechanic | 说明 / Description |
|------|------|
| **燃烧（Exhaust）** | 打出后从牌组中移除的卡牌，很多强力卡靠燃烧触发额外效果 / Cards that remove themselves from the deck — many powerful cards gain extra effects when exhausted |
| **力量（Strength）** | 每点力量使攻击卡伤害+1（多段攻击每段+1）/ Each strength adds +1 damage to attack cards (+1 per hit for multi-hit attacks) |
| **易伤（Vulnerable）** | 受到攻击伤害+50% / Take 50% more attack damage |
| **虚弱（Weak）** | 造成的攻击伤害减半 / Deal 50% less attack damage |

---

## 主流流派 / Main Builds

### 1. 力量战 / Strength Build

**核心思路**：叠加力量，用高伤害或多段攻击秒杀敌人 / Stack strength, then unleash high-damage or multi-hit attacks.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **观察弱点 / Spot Weakness** | 1 | 每回合获得3点力量直到回合结束 / Gain 3 Strength this turn |
| **活动肌肉 / Inflame** | 1 | 获得2（3）点力量 / Gain 2(3) Strength |
| **突破极限 / Limit Break** | 1 | 力量翻倍（可无限循环）/ Double your Strength (can loop infinitely) |
| **重刃 / Heavy Blade** | 2 | 造成14点伤害×力量倍率（力量×5）/ Deal 14 dmg × Strength multiplier (×5) |
| **飞剑回旋镖 / Sword Boomerang** | 1 | 随机敌人造成3次×4（6）点伤害 / Deal 3×4(6) to random enemies |
| **恶魔形态 / Demon Form** | 3 | 每回合获得3（4）点力量 / Each turn gain 3(4) Strength |

**核心遗物 / Key Relics**:
- **手里剑（Shuriken）**：每回合打3张攻击牌后获得1点力量 / Gain 1 Strength after playing 3 attacks
- **苦无（Kunai）**：每回合打3张攻击牌后获得1点敏捷 / Gain 1 Dexterity after playing 3 attacks
- **壶铃（Kettlebell）**：每场战斗第一次攻击附带易伤 / First attack each combat applies Vulnerable

**打法要点 / Tips**:
- 前期尽快拿到力量来源（活动肌肉/观察弱点）/ Grab a strength source early (Inflame / Spot Weakness)
- **突破极限**是多段力量的核心，但需要先叠一定力量才有收益 / **Limit Break** is key but needs base strength first
- 多段攻击（如飞剑回旋镖）受力量加成极大 / Multi-hit attacks benefit massively from strength

### 2. 防战 / Block Build

**核心思路**：通过格挡卡输出伤害（如**全身撞击**），铁壁防御的同时造成高伤 / Deal damage through block cards (e.g., **Body Slam**), building an iron defense while dealing damage.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **全身撞击 / Body Slam** | 1(0) | 造成等于当前格挡值的伤害 / Deal damage equal to current Block |
| **坚不可摧 / Impervious** | 2 | 获得30（40）点格挡 / Gain 30(40) Block |
| **金属化 / Metallicize** | 1 | 每回合获得3（4）点格挡 / Gain 3(4) Block each turn |
| **壁垒 / Barricade** | 3(2) | 格挡不会在回合结束时消失 / Block does not expire at end of turn |
| **双持 / Dual Wield** | 1 | 复制手牌中的一张攻击卡或技能卡 / Duplicate an Attack or Skill card |
| **幽灵形态 / Ghostly Armor** | 1 | 获得10（13）点格挡，并生成一张虚空卡 / Gain 10(13) Block, ethereal |

**核心遗物 / Key Relics**:
- **船首像（Captain's Wheel）**：每场战斗前3回合获得额外格挡 / Gain extra block first 3 turns
- **考古学家帽（Orichalcum）**：回合结束时如果没有格挡，获得6点格挡 / Gain 6 Block at end of turn if no Block
- **棘刺（Thorn）**：受到攻击时反弹伤害 / Thorns: deal damage when attacked

**打法要点 / Tips**:
- **壁垒**是防战的核心，没拿到壁垒时格挡每回合重置 / **Barricade** is the core — without it, block resets each turn
- **全身撞击**配合壁垒可以一回合打出极高伤害 / **Body Slam** + **Barricade** = massive one-turn damage
- 双持可以复制格挡卡，大幅提升防御能力 / Dual Wield can duplicate block cards for massive defense

### 3. 灼烧战 / Exhaust Build

**核心思路**：利用燃烧机制激活强力效果，燃烧越多越强 / Use exhaust mechanics to trigger powerful effects — the more you exhaust, the stronger you get.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **腐化 / Corruption** | 3(2) | 技能卡变为0费且打完后燃烧 / Skills cost 0 and Exhaust when played |
| **黑暗之拥 / Dark Embrace** | 2(1) | 当有卡牌被燃烧时，抽1张牌 / When a card is Exhausted, draw 1 card |
| **狂怒 / Rage** | 1 | 每打出一张攻击卡获得3（5）点格挡 / Gain 3(5) Block each time you play an Attack |
| **燃烧之刃 / Fiend Fire** | 2 | 燃烧手牌中的所有卡，每张造成7（10）伤害 / Exhaust ALL cards in hand, deal 7(10) per card |
| **重组 / Reaper** | 2 | 对所有敌人造成4（5）伤害，回复等量HP / Deal 4(5) to ALL enemies, heal equal to damage dealt |

**核心遗物 / Key Relics**:
- **死灵之书（Dead Branch）**：每次燃烧后生成一张随机卡牌 / Generate a random card when a card is exhausted
- **符��金字塔（Runic Pyramid）**：回合结束不弃牌（配合腐化极强）/ Don't discard at end of turn (extremely strong with Corruption)

**打法要点 / Tips**:
- **腐化 + 黑暗之拥**是灼烧流的绝对核心组合 / **Corruption + Dark Embrace** is the core combo
- 死灵之书配合腐化会让战斗变得疯狂但非常强力 / Dead Branch + Corruption makes combat chaotic but extremely powerful
- **重组**在腐化作用下变成0费全体攻击+吸血，非常强力 / **Reaper** becomes 0-cost AoE + lifesteal under Corruption

### 4. 灼热攻击流 / Searing Blow Build

**核心思路**：只升级一张**灼热攻击（Searing Blow）**，在篝火处反复升级 / Upgrade only **Searing Blow** repeatedly at campfires.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 说明 / Description |
|------|------|
| **灼热攻击 / Searing Blow** | 基础伤害12点，每次升级+4点，可无限升级 / Base 12 dmg, +4 per upgrade, no upgrade cap |
| **双持 / Dual Wield** | 复制灼热攻击 / Duplicate Searing Blow |
| **武装 / Armaments** | 升级手牌中所有卡 / Upgrade all cards in hand |

**打法要点 / Tips**:
- 篝火全部用来升级灼热攻击 / Use ALL campfires to upgrade Searing Blow
- 拿到双持后一回合可以打出多张高伤灼热攻击 / With Dual Wield, play multiple high-damage Searing Blows per turn
- 删掉所有其他攻击卡，保证每回合抽到灼热攻击 / Remove all other attacks to draw Searing Blow every turn

## 战士通用策略 / General Strategy

| 阶段 / Phase | 策略 / Strategy |
|------|------|
| **Act 1 前期** | 多拿攻击牌打精英。**飞剑回旋镖**和**重刃**是优质输出 / Prioritize attacks for elites. **Sword Boomerang** and **Heavy Blade** are excellent |
| **Act 1 中期** | 加入一些格挡和过牌，不要只拿攻击 / Add some block and draw, don't go full damage |
| **Act 1 后期** | 确认流派方向，针对BOSS调整 / Commit to an archetype, tailor for the boss |
| **Act 2** | 急需AOE（**死亡收割**、**旋风斩**），Act 2群怪极多 / You need AoE badly — Act 2 has many multi-enemy fights |
| **Act 3** | 完善牌组运转，确保打BOSS的爆发输出 / Refine your engine, ensure burst damage for the boss |

## 进阶技巧 / Advanced Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **腐化优先级** | 拿到腐化(Corruption)后——牌组里的所有技能牌变为0费但被消耗——配合黑暗之拥(Dark Embrace)可以无限抽牌 + 壁垒(Barricade)无限格挡 / Corruption makes all skills cost 0 but exhaust — with Dark Embrace gives infinite draw + Barricade gives infinite block |
| **死亡收割续航** | 死亡收割(Reaper)AOE+回血——配合力量叠加 = 一次回满血。是战士最稳定的恢复手段 / Reaper = AoE + heal — with strength stacking = full heal. Most reliable sustain for Ironclad |
| **灼热攻击流** | 核心是把牌组缩小到只剩灼热攻击+过牌——篝火全部升灼热。6级灼热=12+4×6=36基础伤害 / Reduce deck to Searing Blow + draw cards — all campfires upgrade it. Level 6 = 12+4×6=36 base dmg |
| **双持爆发** | 双持(Dual Wield)+恶魔形态(Demon Form)=三份恶魔形态——每回合+6力量 / Dual Wield + Demon Form = triple Demon Form — +6 strength per turn |
| **壁垒防战** | 壁垒(Barricade)+全身撞击(Body Slam)=格挡不消失+用格挡打伤害——最稳的通关方式之一 / Barricade + Body Slam = block doesn't expire + block = damage — one of the most consistent winning strategies |
