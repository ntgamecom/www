# 故障机器人攻略 / Defect Guide

故障机器人（Defect）是第三位角色，以球体（Orbs）为核心机制，通过召唤闪电、冰霜、黑暗等球体来输出和防御 / The Defect is the third character, centered on Orbs — summoning lightning, frost, and dark orbs for damage and defense.

**初始遗物 / Starting Relic**: 裂变核心（Cracked Core）—— 战斗开始时召唤一个闪电球体 / Summon a Lightning Orb at the start of combat

---

## 核心机制 / Core Mechanics

### 球体系统 / Orb System

球体由卡牌召唤，最多同时存在3个球体，按顺序依次激活 / Orbs are summoned by cards, max 3 active orbs in a row, activated in order each turn.

| 球体 / Orb | 效果 / Effect |
|------|------|
| **闪电球 / Lightning** | 回合开始时对随机敌人造成3（4）电系伤害 / Deal 3(4) Lightning damage to a random enemy at start of turn |
| **冰霜球 / Frost** | 回合开始时获得2（3）格挡 / Gain 2(3) Block at start of turn |
| **黑暗球 / Dark** | 回合开始时伤害翻倍（初始6），打出时释放累计伤害 / Doubles damage each turn (starts at 6), released when Evoked |
| **等离子球 / Plasma** | 回合开始时获得1点能量 / Gain 1 energy at start of turn |
| **虚空球 / Void** | 无效果，占位用 / No effect, placeholder |

**激发（Evoke）**：召唤第4个球体时最旧的球体被**激发**，触发其效果并移除 / When a 4th orb is summoned, the oldest orb is **Evoked** — its effect triggers and it's removed.

### 聚焦 / Focus

每点聚焦使球体效果+1（闪电伤害+1，冰霜格挡+1）/ Each Focus adds +1 to orb effects (+1 lightning damage, +1 frost block).

## 主流流派 / Main Builds

### 1. 冰雷球 / Frost-Lightning Hybrid

**核心思路**：闪电球输出，冰霜球防御，聚焦提升两者效果 / Lightning for damage, Frost for defense, Focus boosts both.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **球状闪电 / Ball Lightning** | 2(1) | 造成7（10）伤害，召唤1个闪电球 / Deal 7(10) dmg, channel 1 Lightning |
| **冷静头脑 / Coolheaded** | 1 | 抽1（2）张牌，召唤1个冰霜球 / Draw 1(2) cards, channel 1 Frost |
| **冰川 / Glacier** | 2 | 获得7（10）格挡，召唤2个冰霜球 / Gain 7(10) Block, channel 2 Frost |
| **雷暴 / Thunder Strike** | 0 | 每召唤一个闪电球造成7（12）伤害 / Deal 7(12) per Lightning channeled this combat |
| **聚焦 / Focus** | 2(1) | 获得2（3）聚焦 / Gain 2(3) Focus |
| **偏折电容 / Defragment** | 1 | 获得1（2）聚焦 / Gain 1(2) Focus |

**核心遗物 / Key Relics**:
- **数据金（Data Disk）**：战斗开始时获得1聚焦 / Gain 1 Focus at start of combat
- **对称符号（Symbiotic Virus）**：战斗开始时召唤1个黑暗球 / Channel 1 Dark at start
- **符文电容器（Runic Capacitor）**：球体位上限于+2（可同时存5个球）/ +2 Orb slots (up to 5)

**打法要点 / Tips**:
- **聚焦**是球体系的核心中的核心，有多少拿多少 / **Focus** is the MOST important stat for orb builds — take every one you see
- **冰川**是机器人的最强格挡卡之一 / **Glacier** is one of the Defect's best block cards
- 球体数量上限提升（电容/+球体槽遗物）配合聚焦效果极佳 / Orb slot upgrades synergize extremely well with Focus

### 2. 黑暗球爆发流 / Dark Orb Burst

**核心思路**：培养一个黑暗球，让它在回合间反复翻倍，最后释放打出毁灭性伤害 / Nurture a Dark Orb to multiply its damage each turn, then Evoke for devastating burst.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **黑暗浪潮 / Darkness** | 1 | 召唤1个黑暗球（升级后召唤2个）/ Channel 1(2) Dark |
| **多重释放 / Multi-Cast** | 1(2) | 依次激发球体多次 / Evoke your next orb X times |
| **双重释放 / Dualcast** | 1 | 激发第一个球体2次 / Evoke your first orb 2 times |
| **循环 / Loop** | 1 | 每回合自动激发最右侧球体 / Automatically evokes the rightmost orb each turn |
| **散热片 / Heatsinks** | 1 | 每召唤一个球体抽1（2）张牌 / Draw 1(2) card whenever you channel an orb |

**核心遗物 / Key Relics**:
- **龙之泪（Gold-Plated Cables）**：你的球体每次触发时额外激发一次效果 / Your orbs trigger an extra time
- **符文电容器（Runic Capacitor）**：额外球体槽 / Extra orb slots

**打法要点 / Tips**:
- 黑暗球在场上每回合伤害翻倍，放得越久越强 / Dark Orbs double damage each turn — the longer they sit, the stronger they get
- **多重释放**可以一次性打出超高爆发 / **Multi-Cast** delivers massive burst in one turn
- 保留一个额外的球体槽来"藏"黑暗球，让它持续成长 / Save an extra orb slot to let your Dark Orb keep growing

### 3. 电击无限流 / Lightning Infinite

**核心思路**：通过特定组合实现无限循环，每回合召唤大量闪电球 / Create an infinite loop, channeling massive Lightning each turn.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **超频 / Overclock** | 0 | 抽2（3）张牌，消耗1HP / Draw 2(3) cards, lose 1 HP |
| **重编程 / Reboot** | 2(1) | 洗回所有手牌和弃牌堆，重抽等量牌 / Shuffle all cards back, draw equal amount |
| **启用科学 / Machine Learning** | 1 | 每回合多抽1（2）张 / Draw 1(2) extra card each turn |
| **融合 / Fission** | 1(0) | 激发所有球体，获得等量能量 / Evoke ALL orbs, gain equal energy |
| **缓冲 / Buffer** | 2(1) | 抵挡下1（2）次伤害 / Prevent next 1(2) instances of damage |

**打法要点 / Tips**:
- 需要大量过牌和能量来源 / Needs lots of draw and energy sources
- **融合**在大量球体被激发时可以给你巨量能量 / **Fission** with many orbs = massive energy
- 核心是让牌组足够小，实现不断循环 / Core is a small deck that cycles consistently

### 4. 爪击流 / Claw Build

**核心思路**：**爪击（Claw）**每次打出伤害+2（3），多张爪击互相叠加，越打越强 / **Claw** gains +2(3) damage each time played — multiple Claws stack each other.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **爪击 / Claw** | 0 | 造成3（5）伤害，每打出一次此卡永久+2（3）伤害 / Deal 3(5) dmg, permanently gains +2(3) dmg each time played |
| **全息影像 / Hologram** | 1 | 从弃牌堆回收一张牌到手上 / Retrieve a card from your discard pile to your hand |
| **均衡 / Equilibrium** | 2(1) | 获得13（16）格挡，手牌全保留到下一回合 / Gain 13(16) Block, retain all cards |
| **重编程 / Reboot** | 2(1) | 洗回所有手牌和弃牌堆 / Shuffle all cards back, redraw |

**打法要点 / Tips**:
- 爪击流是**机器人最欢乐的流派**，多张爪击互相叠加 / **Claw is the Defect's meme build** — multiple Claws stacking is hilarious
- 全息影像是爪击流的核心，可以反复回收爪击 / **Hologram** is key — repeatedly retrieve Claws from discard
- 需要大量过牌来保证每回合抽到爪击 / Needs lots of draw to ensure Claw is in hand each turn

| **Act 1 前期** | **球状闪电**和**冰冷触摸**是优质前期卡，撑起早期输出和防御 / **Ball Lightning** and **Chill** are excellent early picks for damage and defense |
