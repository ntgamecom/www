# 隐藏要素 / Secrets & Unlockables

杀戮尖塔表面简单，但有丰富的隐藏内容和解锁要素——从隐藏角色到特殊事件，从成就挑战到彩蛋 / Slay the Spire appears simple but has rich hidden content and unlockables — from hidden characters to special events, achievement challenges to easter eggs.

---

## 角色解锁 / Character Unlock

| 角色 / Character | 解锁条件 / Unlock Condition |
|---|---|
| **铁甲战士 / Ironclad** | 默认解锁 / Default |
| **静默猎人 / Silent** | 使用铁甲战士通关一次 / Complete one run with Ironclad |
| **故障机器人 / Defect** | 使用静默猎人通关一次 / Complete one run with Silent |
| **观者 / Watcher** | 使用故障机器人通关一次 / Complete one run with Defect |

**解锁顺序**: 铁甲战士 → 静默猎人 → 故障机器人 → 观者。每个角色首次通关后解锁下一个 / **Unlock order**: Ironclad → Silent → Defect → Watcher. Each first clear unlocks the next character.

## 卡牌解锁 / Card Unlock

每个角色初始有部分卡牌——其他卡牌需要通过提升角色等级解锁 / Each character starts with a subset of cards — the rest unlock as the character's level increases.

| 角色等级 / Character Level | 解锁内容 / Unlock |
|---|---|
| **2级 / Level 2** | 解锁3张稀有卡 / Unlock 3 rare cards |
| **3级 / Level 3** | 解锁2张普通卡 + 1张稀有卡 / Unlock 2 common + 1 rare card |
| **4级 / Level 4** | 解锁3张罕见卡 / Unlock 3 uncommon cards |
| **5级 / Level 5** | 解锁2张普通卡 + 1张稀有卡 / Unlock 2 common + 1 rare card |
| **6级 / Level 6** | 解锁3张罕见卡 / Unlock 3 uncommon cards |
| **7级 / Level 7** | 解锁2张普通卡 + 1张稀有卡 / Unlock 2 common + 1 rare card |
| **8级 / Level 8** | 解锁3张罕见卡 / Unlock 3 uncommon cards |
| **9级 / Level 9** | 解锁2张普通卡 + 1张稀有卡 / Unlock 2 common + 1 rare card |
| **10级 / Level 10** | 解锁5张稀有卡——角色卡牌全解锁 / Unlock 5 rare cards — character's card pool complete |

**升级方式**: 每次通关获得角色经验——胜利+大量经验，失败+少量经验 / **Leveling**: Each run gives character XP — win = large XP, loss = small XP.

| **极简主义 / Minimalist** | 牌组≤5张牌时通关 / Win with ≤5 card deck | 观者最容易——单张"愤怒"+大量遗物 / Watcher is easiest — single "Rage" + lots of relics |

| **蓝宝石 / Sapphire** | 故障机器人通关 / Win with Defect | 冰球+集中流最稳——闪电球DPS / Frost orb + focus build — lightning orbs for DPS |
| **紫水晶 / Amethyst** | 观者通关 / Win with Watcher | 观者最强但风险也最高——暴怒+警惕切换 / Watcher is strongest but risky — Wrath + Calm switching |

## 隐藏Boss / Hidden Bosses

| Boss / Boss | 出现条件 / Condition | 应对 / Strategy |
|---|---|---|
| **矛盾 / The Corrupt Heart** | Act 3中持有"红钥匙"、"绿钥匙"和"蓝钥匙"——才能打开通往矛盾的传送门 / Hold Ruby Key, Emerald Key, and Sapphire Key in Act 3 to open the Heart portal | 需要全面防御——律动效果每打一张牌掉1血。推荐叠冰球防御的机器人 / Need comprehensive defense — Beat of Death damages per card. Defect with frost orbs is best |

**钥匙获取方式**:
| 钥匙 / Key | 获取方法 / Method |
|---|---|
| **红钥匙 / Ruby Key** | 在休息处选择"回忆"——获得红钥匙(代替回血或强化) / At rest site, choose "Recall" — get Ruby Key (instead of heal or smith) |
| **绿钥匙 / Emerald Key** | 击败任意精英——获得绿钥匙(取代遗物掉落) / Defeat any elite — get Emerald Key (replaces relic drop) |
| **蓝钥匙 / Sapphire Key** | 在宝箱处选择"取出"——获得蓝钥匙(取代宝箱中的遗物) / At chest, choose "Take Key" — get Sapphire Key (replaces chest relic) |

**获取代价**: 三把钥匙分别取代：回血/强化机会、精英遗物掉落、宝箱遗物。选择拿钥匙意味着牺牲这些资源——需要仔细权衡 / **Cost**: Three keys replace: heal/smith chance, elite relic drop, chest relic. Choosing keys means sacrificing these resources — weigh carefully.

## 隐藏数值 / Hidden Mechanics

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **随机种子 / Seed System** | 每个存档有随机种子——决定了整局游戏的所有掉落和事件。输入指定的种子可以复现同一局 / Each run has a seed — determines all drops and events. Input an exact seed to replay the same run |
| **稀有度保底 / Rarity Protection** | 连续不出稀有卡/遗物时——出率逐渐提升 / When no rare cards/relics appear for a while — drop rate gradually increases |
| **进阶系统隐藏变化** | 除了面板上描述的进阶变化——一些Boss和怪物还有隐藏的数据改动(如进阶18开始精英出现率变化) / Beyond described ascension changes — some bosses and enemies have hidden stat changes (elite spawn rate changes from A18) |
| **遗物池权重** | 不同遗物的出现概率不同——部分遗物有"降低出现率"标签，有些遗物只能出现一次 / Different relics have different weightings — some have "reduced appearance" tag, some can only appear once |
| **稀有卡牌保底** | 每局至少出现1张稀有卡——如果前半局没看到，后半局一定会出 / At least 1 rare card per run — if none in first half, guaranteed in second half |

## 每日挑战 / Daily Climb

| 要素 / Element | 说明 / Description |
|---|---|
| **每日挑战模式** | 每天一个固定种子+特殊规则——所有玩家玩同一个挑战 / Daily fixed seed + special rules — all players share the same challenge |
| **排名系统** | 根据分数排名——分数来自通关速度、精英击杀数、金币等 / Ranked by score — speed, elite kills, gold, etc. |
| **特殊模组** | 每天可能有特殊模组——如"所有敌人伤害翻倍"、"抽牌加倍" / Daily modifiers — "All enemies double damage", "Double draw", etc. |
| **奖励** | 每日挑战通关获得积分——积分不影响游戏内容 / Points on completion — no in-game rewards, purely for bragging rights |
