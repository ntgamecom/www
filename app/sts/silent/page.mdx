# 猎人攻略 / Silent Guide

猎人（Silent）是第二位角色，擅长毒系持续伤害和卡牌运转，核心机制是**毒**和**弃牌** / The Silent is the second character, excelling at poison damage and deck cycling. Core mechanics: **Poison** and **Discard**.

**初始遗物 / Starting Relic**: 蛇之戒指（Ring of the Snake）—— 战斗开始时多抽2张牌 / Draw 2 extra cards at the start of combat

---

## 核心机制 / Core Mechanics

| 机制 / Mechanic | 说明 / Description |
|------|------|
| **毒（Poison）** | 每回合减少敌人HP，层数-1。毒层数越高伤害越大，适合持久战 / Deals HP damage per turn, stacks decay by 1. Great for prolonged fights |
| **弃牌（Discard）** | 主动弃牌触发效果，如额外能量、抽牌、格挡 / Discarding cards triggers effects like extra energy, draw, or block |
| **敏捷（Dexterity）** | 每点敏捷使格挡卡获得的格挡值+1 / Each Dexterity adds +1 Block from Block cards |

---

## 主流流派 / Main Builds

### 1. 毒贼 / Poison Build

**核心思路**：叠毒让敌人每回合掉大量HP，自己专注防御和运转 / Stack poison so enemies take massive damage each turn while you focus on defense and cycling.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **毒雾 / Noxious Fumes** | 1 | 每回合给所有敌人2（3）层毒 / Apply 2(3) poison to ALL enemies each turn |
| **催化剂 / Catalyst** | 1(0) | 毒层数翻倍（3倍）/ Double (Triple) poison stacks |
| **致命毒药 / Deadly Poison** | 1 | 施加5（7）层毒 / Apply 5(7) poison |
| **毒药溅射 / Poison Splash** | 2 | 施加8（11）层毒，溅射其他敌人4层 / Apply 8(11) poison, splash 4 to other enemies |
| **闷燃毒雾 / Bouncing Flask** | 2 | 随机敌人3次×3（5）层毒 / Apply 3×3(5) poison to random enemies |
| **残影 / Blur** | 1 | 获得5（8）格挡，格挡不会在回合结束消失 / Gain 5(8) Block, block doesn't expire this turn |

**核心遗物 / Key Relics**:
- **蛇之戒指（Snake Ring）**：开局多抽2张 / Draw +2 at start
- **黑石护符（Blackstone）**：毒伤害+1 / Poison deals +1 damage
- **毒蛇标本（Snecko Skull）**：施加毒时额外+1层 / Apply +1 extra poison when applying poison

**打法要点 / Tips**:
- **催化剂**是毒贼的核心爆发，可以瞬间让毒层数翻倍/三倍 / **Catalyst** is the key burst — can instantly double/triple poison stacks
- **毒雾**是优质AOE毒来源，站场每回合免费叠毒 / **Noxious Fumes** is excellent passive AoE poison
- 毒贼前期较弱的，需要格挡和过牌支撑到Act 2 / Poison build is weak early — needs block and draw to survive until Act 2

### 2. 弃牌贼 / Discard Build

**核心思路**：通过主动弃牌获得额外收益——能量、抽牌、格挡等 / Actively discard cards for bonus energy, draw, block, and more.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **战术大师 / Tactician** | 0 | 弃牌时获得2（3）能量 / Gain 2(3) energy when discarded |
| **计算下注 / Calculated Gamble** | 1(0) | 弃掉所有手牌，抽等量牌 / Discard your hand, draw that many cards |
| **杂技 / Acrobatics** | 1 | 抽3（4）张牌，弃1张 / Draw 3(4) cards, discard 1 card |
| **内脏切除 / Eviscerate** | 3(2) | 本回合每弃一张牌费用-1，造成6（10）×3伤害 / Costs -1 per card discarded this turn, deal 6(10)×3 |
| **坚韧 / Leg Sweep** | 2 | 获得12（15）格挡，施加2（3）层虚弱 / Gain 12(15) block, apply 2(3) weak |
| **刀舞 / Blade Dance** | 1 | 加入2（3）张0费5伤的小刀到手中 / Add 2(3) 0-cost 5-dmg Shivs to hand |

**核心遗物 / Key Relics**:
- **符文金字塔（Runic Pyramid）**：回合结束不弃牌（配合弃牌卡可控性更高）/ Don't discard at end of turn (more control over discards)
- **铜钹（Tingsha）**：每次弃牌对所有敌人造成3点伤害 / Deal 3 damage to all enemies when you discard
- **木雕（Hovering Kite）**：每弃2张牌获得1点能量 / Gain 1 energy every 2 cards discarded

**打法要点 / Tips**:
- **战术大师**让弃牌流能量极其充裕 / **Tactician** gives the build abundant energy
- **铜钹**让弃牌本身变成AOE伤害 / **Tingsha** turns discarding into AoE damage
- **计算下注**+**战术大师**可以形成强力过牌+能量循环 / **Calculated Gamble** + **Tactician** = powerful draw + energy loop

### 3. 小刀贼 / Shiv Build

**核心思路**：生成大量0费小刀（Shivs），配合力量/精准加成造成海量伤害 / Generate tons of 0-cost Shivs, buffed by strength and accuracy for massive damage.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **刀舞 / Blade Dance** | 1 | 加入2（3）张小刀 / Add 2(3) Shivs to hand |
| **无限刀 / Infinite Blades** | 1 | 每回合开始加入1（2）张小刀 / At start of turn add 1(2) Shiv |
| **终止 / Finisher** | 1 | 本回合每打出1张小刀造成6（8）伤害 / Deal 6(8) per Shiv played this turn |
| **精准 / Accuracy** | 1 | 小刀伤害+4（6）/ Shivs deal +4(6) damage |
| **疾速铁锤 / Storm of Steel** | 1 | 弃任意张牌，每张加入1张小刀 / Discard any number of cards, add 1 Shiv per card |
| **刀刃乱舞 / Cloak and Dagger** | 1 | 获得6格挡，加入1（2）张小刀 / Gain 6 Block, add 1(2) Shiv |

**核心遗物 / Key Relics**:
- **腕带（Wrist Blade）**：小刀伤害+4 / Shivs deal +4 damage
- **手里剑（Shuriken）**：每回合打3张牌获得1力量（小刀也算）/ Gain 1 Strength after 3 attacks (Shivs count)
- **苦无（Kunai）**：每回合打3张牌获得1敏捷 / Gain 1 Dexterity after 3 attacks

**打法要点 / Tips**:
- **精准**的伤害加成对小刀至关重要 / **Accuracy**'s damage buff is crucial for Shivs
- 小刀配合力量遗物（手里剑）可以一回合叠大量力量 / Shivs with Shuriken can stack massive strength in one turn
- **终止**配合大量小刀可以一回合打出极高爆发 / **Finisher** with many Shivs = huge burst damage

### 4. 弃牌无限流 / Discard Infinite

**核心思路**：通过特定卡牌组合实现无限循环，一回合走完整个牌组无限次 / Achieve an infinite loop with specific card combos, cycling your deck infinitely in one turn.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 说明 / Description |
|------|------|
| **战术大师 / Tactician** | 弃牌获得能量 / Energy on discard |
| **计算下注 / Calculated Gamble** | 弃光手牌重新抽 / Discard hand, redraw |
| **杂技 / Acrobatics** | 抽牌+弃牌 / Draw + discard |
| **反射 / Reflex** | 弃牌时抽牌 / Draw on discard |

**打法要点 / Tips**:
- 核心是**战术大师**+弃牌卡形成的能量正循环 / Core is **Tactician** + discard cards for net positive energy
- 牌组必须足够小（10张以下）/ Deck must be very small (under 10 cards)
- 必须有**符文金字塔**来控制手牌 / Need **Runic Pyramid** for hand control

## 猎人通用策略 / General Strategy

| 阶段 / Phase | 策略 / Strategy |
|------|------|
| **Act 1 前期** | 猎人前期弱，多拿**刀舞**和**毒药**撑前期战力 / Silent is weak early — grab **Blade Dance** and **Deadly Poison** for early damage |
| **Act 1 中期** | 加入**杂技**和格挡卡，保证存活 / Add **Acrobatics** and block cards for survivability |

| **Act 2** | 弃牌流和毒贼在Act 2起飞，大量AoE和过牌让战斗掌控力大增 / Discard and poison builds shine in Act 2 with better draw and AoE |
| **Act 3** | 确保有足够爆发输出打BOSS，毒贼需要催化剂，小刀贼需要精准 / Ensure burst damage for Act 3 bosses — Catalyst for poison, Accuracy for Shivs |

## 进阶技巧 / Advanced Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **弃牌无限** | 战术大师(Tactician)+ 杂技(Acrobatics)形成能量正循环——配合计算下注(Calculated Gamble)过全牌组——不需要特定遗物就能达成 / Tactician + Acrobatics for positive energy loop + Calculated Gamble for full deck draw — no specific relic needed |
| **毒药叠加** | 催化剂(Catalyst)将敌人毒层数×3——一次上毒→催化剂=三层毒。配合毒雾(Noxious Fumes)每回合叠毒 + 催化剂 = 爆发毒杀 / Catalyst triples poison — apply poison once → Catalyst = triple. With Noxious Fumes (auto-poison) + Catalyst = burst poison kill |
| **小刀爆发** | 精准(Accuracy) + 刀舞(Blade Dance)=每回合3× 8+4伤害=36点/回合。配合时间之杖(Wrist Blade)=每刀+4伤害 = 60/回合 / Accuracy + Blade Dance = 3x (8+4)=36 dmg/turn. With Wrist Blade (+4 per shiv) = 60/turn |
| **残影保留** | 残影(Blur)让格挡保留到下一回合——配合坚毅( fortified)可以做到每回合起手50+格挡 / Blur makes block carry over — with fortified can achieve 50+ block at start of each turn |
| **夜魇爆发** | 夜魇(Nightmare)复制一张手牌——可以复制催化剂(→3×3×毒)、刀舞(→3×3=9刀)、幽魂形态(→3层无实体)/ Nightmare duplicates a hand card — can copy Catalyst (3x triple poison), Blade Dance (9 shivs), Ghostly Form (3 intangible) |
