# 物品与遗物 / Items & Relics

遗物是杀戮尖塔中永久性的增益——一旦获得就持续整局。选择合适的遗物并围绕它们构建Build是通关的关键 / Relics in Slay the Spire are permanent buffs — once acquired they last the entire run. Choosing the right relics and building around them is key to victory.

---

## 遗物稀有度 / Relic Rarity

| 稀有度 / Rarity | 说明 / Description | 出现方式 / Source |
|---|---|---|
| **普通 / Common** | 基础效果——前期帮助大，后期效果有限 / Basic effects — strong early, limited late | 普通宝箱、小怪掉落、商店购买 / Normal chests, enemy drops, shop |
| **罕见 / Uncommon** | 中等效果——通常在中期发挥关键作用 / Medium effects — usually key in mid-game | 精英掉落、普通宝箱、商店 / Elite drops, chests, shop |
| **稀有 / Rare** | 强力效果——可以决定Build方向 / Strong effects — can define a build | 精英掉落、Boss掉落、商店 / Elite drops, boss drops, shop |
| **Boss遗物 / Boss** | 极其强力——但通常有代价(如+1能量但禁止回血) / Extremely strong — but usually with a cost (+1 energy but no healing) | 击败Boss后三选一 / Choose 1 of 3 after boss defeat |
| **事件遗物 / Event** | 特殊事件中获得的遗物——不能在其他地方获得 / Special event relics — no other acquisition method | 特定事件选择 / Specific event choices |
| **商店遗物 / Shop** | 商店专属遗物——只能购买 / Shop exclusive — must purchase | 商店($150-350) / Shop ($150-350) |

## 遗物获取策略 / Relic Acquisition Strategy

| 阶段 / Phase | 优先获取 / Priority | 说明 / Reason |
|---|---|---|
| **Act 1** | 输出型遗物(力量/伤害加成) / Damage relics | 需要快速击杀精英和Boss——防御可以靠卡牌 / Need to kill elites and boss fast — block can come from cards |
| **Act 2** | 防御型+经济遗物 / Block + Economy | Act 2的敌人伤害高——需要叠甲手段。另外需要经济在商店买关键卡 / Act 2 enemies hit hard — need block. Also need gold for key shop cards |
| **Act 3** | 能量遗物+综合型 / Energy + general | Act 3需要高能量来打出大组合——+1能量的Boss遗物是关键 / Need high energy for big combos — +1 Energy boss relics are crucial |
| **Act 4(心)** | 防御型+律动对策 / Block + Beat of Death counter | 对心的战斗中每打一张牌掉1血——所以需要高效的叠甲而不是大量小牌 / Beat of Death damages per card — need efficient block, not many small cards |
