# BOSS攻略 / Boss Guide

杀戮尖塔的Boss是每个Act的最终挑战——每个Boss有独特的攻击模式和机制。了解每个Boss的行动规律是通关的关键 / Slay the Spire's bosses are the final challenge of each Act — each has unique attack patterns and mechanics. Understanding each boss's behavior is key to winning.

---

## Act 1 Boss / Act 1 Bosses

### 六火亡魂 / The Hexaghost

| 属性 / Property | 说明 / Description |
|---|---|
| **生命值 / HP** | 240-260(进阶14+: 264-286) / 240-260 (A14+: 264-286) |
| **特殊机制** | 根据你的血量在第1回合生成6层"灼伤"进牌组——然后每回合从灼伤池中抽1-2张。血越低抽得越少 / Generates 6 "Burn" cards into discard based on HP on turn 1 — draws 1-2 per turn from pool. Lower HP = fewer Burns |
| **强力攻击 / Strong Attack** | "分裂"——6x6的AOE伤害，如果你有灼伤在手，伤害翻倍 / "Inferno" — 6x6 AOE, double damage if you hold Burns |
| **难度 / Difficulty** | 中等——对新手来说灼伤机制很烦人 / Medium — Burn mechanic is annoying for beginners |

**应对策略**: 尽量降低血量到40%以下进场——这样第一回合只生成1张灼伤。回合结束时弃掉灼伤牌。力量战和毒贼对这个Boss都很轻松 / **Strategy**: Enter with HP below 40% — only 1 Burn generated. Discard Burns at end of turn. Strength build and Poison build both handle this boss easily.

### 守护者 / The Guardian

| 属性 / Property | 说明 / Description |
|---|---|
| **生命值 / HP** | 240-255(进阶14+: 264-280) / 240-255 (A14+: 264-280) |
| **特殊机制** | 前2个回合处于"攻击模式"(伤害+30%)——然后转"防御模式"(护甲+40点)。循环切换 / First 2 turns in "Offensive Mode" (+30% damage) — then "Defensive Mode" (+40 block). Switches cyclically |
| **强力攻击** | "冲撞"——在防御模式下每回合蓄力——然后在下回合造成30点伤害 / "Charge Up" in Defensive Mode each turn — then deals 30 damage next turn |
| **难度 / Difficulty** | 低——机制最简单 / Low — simplest mechanics |

**应对策略**: 在防御模式期间叠甲防御——攻击模式时输出。注意它蓄力时用虚弱减少伤害。毒贼和机器人对这个Boss特别轻松 / **Strategy**: Block during defensive mode — DPS during offensive mode. Apply Weak when it's charging up. Poison and Defect have an easy time here.

### 史莱姆王 / The Slime Boss

| 属性 / Property | 说明 / Description |
|---|---|
| **生命值 / HP** | 140-150(进阶14+: 154-165) / 140-150 (A14+: 154-165) |
| **特殊机制** | 半血时分列为两个小史莱姆——大史莱姆(110HP)和小史莱姆(30HP)。如果分列时你已经清理了所有"黏泥"则不分列 / Splits at half HP into Big Slime (110HP) and Small Slime (30HP). If no "Slime" minions present, no split |
| **强力攻击** | "黏液喷吐"——对玩家造成伤害并在弃牌堆加入2张黏泥 / "Goop Spray" — damages player and adds 2 Slimes to discard |
| **难度 / Difficulty** | 中高——新手经常因为分列后输出不够而翻车 / Medium-High — beginners often die after split from insufficient DPS |

**应对策略**: 集中输出把史莱姆王打到半血以下触发分列——然后优先击杀小史莱姆(30HP)。分列前的回合记得叠甲防御 / **Strategy**: Focus damage to trigger split — then kill Small Slime (30HP) first. Block on the turn before split.

## Act 2 Boss / Act 2 Bosses

### 铜制机械人偶 / The Bronze Automation

| 属性 / Property | 说明 / Description |
|---|---|
| **生命值 / HP** | 290-310(进阶14+: 319-341) / 290-310 (A14+: 319-341) |
| **特殊机制** | 每3回合强化——获得4层"人工护甲"(每层6点护甲)。需要攻击来去除护甲 / Buffs every 3 turns — gains 4 stacks of "Plated Armor" (6 block each). Requires attacks to strip |
| **强力攻击** | "超频"——8x3的AOE攻击 / "Hyper Beam" — 8x3 AOE |
| **难度 / Difficulty** | 中等——对叠甲不够的Build有压力 / Medium — pressures builds with weak block |

**应对策略**: 在它强化前输出压低血量。用多段攻击快速剥除人工护甲。机器人用闪电球自动攻击破甲效果很好 / **Strategy**: Reduce HP before it buffs. Use multi-hit attacks to strip Plated Armor quickly. Defect's Lightning orbs auto-strip armor effectively.

### 收集者 / The Collector

| 属性 / Property | 说明 / Description |
|---|---|
| **生命值 / HP** | 290-310(进阶14+: 319-341) / 290-310 (A14+: 319-341) |
| **特殊机制** | 召唤两个小兵——"拷问者"和"幼虫"。只要小兵在，收集者获得额外护甲 / Summons two minions — "Torch Head" and "Maw". Gains extra block while minions are alive |
| **强力攻击** | "吸取"——造成伤害并回血 / "Drain" — damages and heals |
| **难度 / Difficulty** | 高——小兵管理是关键 / High — minion management is crucial |

**应对策略**: 优先击杀小兵——没有小兵后收集者不再获得额外护甲且伤害大幅降低。战士的AOE攻击和猎人的毒雾很有效 / **Strategy**: Kill minions first — without them, Collector's block and damage drop significantly. Ironclad's AOE and Silent's Crippling Cloud are effective.

### 铜制机械人偶 / The Champ

| 属性 / Property | 说明 / Description |
|---|---|
| **生命值 / HP** | 300-320(进阶14+: 330-352) / 300-320 (A14+: 330-352) |
| **特殊机制** | 半血前每回合强化一种属性(力量/敏捷)。半血后"狂暴"——每次行动2次 / Buffs one stat (STR/DEX) per turn before 50% HP. "Execute" phase at 50% — acts twice per turn |
| **强力攻击** | "处决"——多段攻击。半血后威胁极大 / "Execute" — multi-hit attack. Extremely dangerous after 50% HP |
| **难度 / Difficulty** | 高——半血后攻击翻倍对防御不足的Build是致命打击 / High — double actions after 50% HP kills weak-block builds |

**应对策略**: 在半血前把防御堆起来——虚弱和易伤是关键。不要在半血临界点时血量太低——因为它会连续两次攻击。推荐叠足够的甲后一次性爆发打进半血并快速击杀 / **Strategy**: Stack block before 50% HP — Weak and Vulnerable are crucial. Don't enter execute phase with low HP. Save burst damage to push through execute phase quickly.

## Act 3 Boss / Act 3 Bosses

### 觉醒者 / The Awakened One

| 属性 / Property | 说明 / Description |
|---|---|
| **生命值 / HP** | 240-260(第一形态) / 240-260 (Phase 1) |
| **特殊机制** | 每当你使用能力牌时获得2力量。第一形态死后复活为第二形态——比第一形态更强 / Gains 2 STR each time you play a Power. Revives into Phase 2 — stronger than Phase 1 |
| **难度 / Difficulty** | 极高——对能力牌多的Build威胁最大 / Very High — most dangerous for Power-heavy builds |

**应对策略**: 对能力牌少的角色(如力量战、毒贼)比较轻松。如果你有很多能力牌——在最后一回合一次性打出全部能力牌直接打进第二形态。第二形态要注意防御 / **Strategy**: Easy for low-Power builds (Strength Ironclad, Poison Silent). If you have many Powers — play them all on the final turn before entering Phase 2. Prioritize block in Phase 2.

### 时间吞噬者 / The Time Eater

| 属性 / Property | 说明 / Description |
|---|---|
| **生命值 / HP** | 300-320(进阶14+: 330-352) / 300-320 (A14+: 330-352) |
| **特殊机制** | 每12张牌填满一个"时间"条——填满时你的回合结束，它获得额外回合 / Fills a "time" bar every 12 cards — when full, ends your turn and gives it an extra turn |
| **难度 / Difficulty** | 极高——对0费卡/小卡组Build非常克制 / Very High — hard-counter for 0-cost/small-card builds |

**应对策略**: 控制出牌数量——在一回合内不要超过12张牌。大卡组和高质量卡牌是正解。战士的重击流(单张高伤卡)和机器人的冰球流对这个Boss有优势 / **Strategy**: Control card count — don't play more than 12 cards per turn. Big cards and high-quality decks are correct. Ironclad's heavy strike builds and Defect's frost orb builds excel here.

### 矛盾 / The Corrupt Heart(最终Boss / Final Boss)

| 属性 / Property | 说明 / Description |
|---|---|
| **生命值 / HP** | 300-400(根据进阶等级) / 300-400 (varies by ascension) |
| **特殊机制** | 第1回合给5张"伤口"进牌组。每回合强化。叠加强力"律动"效果(每打一张牌掉1血) / Turn 1: adds 5 Wounds. Buffs each turn. Gains "Beat of Death" (1 damage per card played) |
| **难度 / Difficulty** | 最高——需要全面应对 / Highest — requires full preparation |
| **应对策略** | 需要叠甲能力和伤口处理能力。机器人最适合——冰球防御+集中+递归 / Needs block generation and status handling. Defect excels — Frost orbs + Focus + Recursion |

|---|---|---|
| **六火亡魂** | 简单 / Easy | 力量战——快速叠力量+重刃爆发。腐化战也可以 / Strength build — fast STR + Heavy Blade burst. Corruption build works too |
| **守护者** | 简单 / Easy | 在防御模式叠甲——攻击模式输出。燃烧可以让它持续掉血 / Block in defense mode — DPS in attack mode. Combust provides steady damage |
| **史莱姆王** | 中等 / Medium | 注意血量控制——在半血前一回合叠好甲。用AOE清小史莱姆 / Manage HP before split — block the turn before. AOE clears small slimes |
| **收集者** | 困难 / Hard | 优先AOE清小兵——黑暗之拥+腐化组合收益很高 / Kill minions first with AOE — Corruption + Dark Embrace is excellent here |
| **铜制机械人偶** | 中等 / Medium | 用多段攻击(如旋风斩)剥甲——注意半血后的双动模式 / Multi-hit (Whirlwind) to strip armor — watch for execute phase |
| **觉醒者** | 困难 / Hard | 限制能力牌使用——力量战不用能力牌也能打 / Limit Power cards — Strength build works without Powers |
| **时间吞噬者** | 中等 / Medium | 重击流——每次出牌都打大伤害。矮子里面挑高个的卡组 / Big attack build — make every card count. Thin but high-impact deck |
| **矛盾** | 极难 / Very Hard | 需要腐化+黑暗之涌的无限叠甲组合——否则律动效果太疼 / Need Corruption + Dark Embrace for infinite block — Beat of Death is brutal otherwise |

| Boss / Boss | 难度 / Difficulty | 推荐策略 / Strategy |
|---|---|---|
| **六火亡魂** | 简单 / Easy | 毒贼——叠毒后Boss每回合掉血。不需要管灼伤 / Poison build — poison does the work. Ignore Burns |
| **守护者** | 简单 / Easy | 精准小刀流——快速低压输出。防御模式时叠甲 / Blade dance build — fast DPS. Block during defense mode |
| **史莱姆王** | 困难 / Hard | 毒贼比较慢——半血后会分列。推荐小刀流快速打进半血 / Poison is slow here — split will happen. Blade build pushes through faster |
| **铜制机械人偶** | 困难 / Hard | 多发飞刀破甲效率高——配合精准和力量 / Multi-hit blades strip armor fast — with Accuracy + Strength |
| **收集者** | 中等 / Medium | 毒雾(Crippling Cloud)+催化剂——先毒小兵再毒Boss / Poison Cloud + Catalyst — poison minions first, then boss |
| **觉醒者** | 中等 / Medium | 毒贼完美应对——能力牌少，主要靠技能牌叠毒 / Poison build excels — few Powers, mostly skills for poison |
| **时间吞噬者** | 极难 / Very Hard | 小刀贼被严重克制——每回合出牌数有限。推荐毒贼(每回合出牌少) / Blade build is heavily countered — limited cards per turn. Use poison build instead |
| **矛盾** | 困难 / Hard | 律动效果对小刀贼非常致命——每把小刀掉1血。毒贼相对好一些 / Beat of Death kills blade builds — 1 damage per blade. Poison build is manageable |

|---|---|---|
| **全Boss综合评价** | 最稳定 / Most Consistent | 冰球流是所有Boss的通用解——冰球防御+电球输出+集中提升 / Frost orb build works against ALL bosses — frost block + lightning DPS + focus |
| **觉醒者** | 简单 / Easy | 机器人的能力牌不多——而且冰球不受能力牌限制 / Defect doesn't rely heavily on Powers — frost orbs work regardless |
| **时间吞噬者** | 中等 / Medium | 冰球流出牌量可控——冰球/电球自动输出不占用出牌次数 / Frost build controls card count — orbs auto-DPS without playing cards |
| **矛盾** | 中等 / Medium | 冰球防御完美应对律动效果——冰球叠甲不触发律动 / Frost block bypasses Beat of Death — orb block doesn't trigger it |

|---|---|---|
