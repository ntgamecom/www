# 观者攻略 / Watcher Guide

观者（Watcher）是第四位也是最后一位角色，以**姿态切换**为核心机制，拥有极强的爆发力和独特的能量系统 / The Watcher is the fourth and final character, centered on **Stance Switching**, with exceptional burst damage and a unique energy system.

**初始遗物 / Starting Relic**: 纯净之水（Pure Water）—— 战斗开始时获得一张**神格姿态**（进入神格） / Gain a **Deva Form** (enter Divinity) at the start of combat

**注意**：观者是公认的最强角色，上限极高，但操作难度也最大 / Note: The Watcher is widely considered the strongest character — highest ceiling but also most demanding to play.

---

## 核心机制 / Core Mechanics

### 姿态系统 / Stance System

观者可以在三种姿态间切换，每次进入新姿态会覆盖旧姿态 / The Watcher switches between stances — entering a new stance overwrites the old one.

| 姿态 / Stance | 效果 / Effect |
|------|------|
| **愤怒 / Wrath** | 造成的攻击伤害翻倍，受到的攻击伤害也翻倍 / Deal AND take double attack damage |
| **平静 / Calm** | 退出此姿态时获得2点能量 / When you exit this stance, gain 2 energy |
| **神格 / Divinity** | 造成的攻击伤害×3，每回合退出 / Deal TRIPLE attack damage, exits at end of turn |

### 保留（Retain）

观者有很多卡牌带有**保留**关键字——留在手中不会被强制弃牌 / The Watcher has many **Retain** cards — stay in hand and aren't discarded at end of turn.

### 预见（Scry）

观者可以控制牌堆顶部的卡牌顺序，安排未来的抽牌 / The Watcher can manipulate the top cards of the draw pile to arrange future draws.

## 主流流派 / Main Builds

### 1. 红蓝姿态流 / Wrath-Calm Stance Dance

**核心思路**：在愤怒和平静之间反复切换，愤怒打输出，平静回能量，循环往复 / Switch between Wrath and Calm repeatedly — Wrath for damage, Calm for energy — loop infinitely.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **警惕 / Vigilance** | 2(1) | 获得8（12）格挡，进入平静 / Gain 8(12) block, enter Calm |
| **暴怒 / Eruption** | 2(1) | 造成8（10）伤害，进入愤怒 / Deal 8(10) dmg, enter Wrath |
| **内心平静 / Inner Peace** | 1(0) | 如果在平静则抽3张牌，否则进入平静 / If in Calm draw 3 cards, otherwise enter Calm |
| **化作即空 / Empty Mind** | 1(0) | 退出当前姿态，抽2（3）张牌 / Exit your stance, draw 2(3) cards |
| **不惧死亡 / Fear No Evil** | 1 | 造成8（11）伤害，如果敌人有意图攻击则进入平静 / Deal 8(11) dmg, enter Calm if enemy intends to attack |
| **切断凡念 / Cut Through Fate** | 1 | 造成6（9）伤害，预见2（3）张 / Deal 6(9) dmg, Scry 2(3) |

**核心遗物 / Key Relics**:
- **念珠（Prayer Beads）**：战斗开始时进入平静 1 回合 / Enter Calm for 1 turn at start
- **紫水晶（Teardrop Locket）**：战斗开始时进入愤怒 / Enter Wrath at start
- **尼利宝珠（Necronomicon）**：每场战斗第一张2费以上攻击卡打两次 / First 2+ cost attack each combat is played twice

**打法要点 / Tips**:
- 核心循环：愤怒→打伤害→平静→获能量→再愤怒→再打 / Core loop: Wrath → damage → Calm → energy → Wrath → damage
- **化作即空**是退出姿态+过牌的神卡，几乎必拿 / **Empty Mind** is stance exit + draw, almost always take it
- 有2张**暴怒**可以把一张升级（减费）然后轮换使用 / Two **Eruptions** (one upgraded) allows alternating

### 2. 神格爆发流 / Divinity Burst

**核心思路**：快速进入神格（×3伤害），一回合打出毁灭性输出 / Enter Divinity (×3 damage) quickly, then unleash devastating damage in one turn.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **天眼 / Brilliance** | 2(1) | 造成8（12）伤害，每点神格能量+2伤害 / Deal 8(12) dmg, deals +2 per Mantra |
| **虔信 / Devotion** | 1 | 每回合获得1（2）神格能量 / Gain 1(2) Mantra each turn |
| **斋戒 / Prostrate** | 1 | 获得4（5）格挡，获得1神格能量，预见2 / Gain 4(5) block, gain 1 Mantra, Scry 2 |
| **洞察 / Study** | 2(1) | 加入一张洞察到手上 / Add an Insight to your hand |
| **冥想 / Meditate** | 1 | 放一张弃牌堆的卡到手上，进入平静 / Put 1 card from discard to hand, enter Calm |

**核心遗物 / Key Relics**:
- **怜悯（Duality）**：每打一张技能牌获得1点敏捷直到回合结束 / Gain 1 temporary Dexterity per skill played
- **冰沙（Ice Cream）**：能量不会消失，可以积累多回合能量 / Energy does not expire, can accumulate across turns

**打法要点 / Tips**:
- 神格需要积累10点**神格能量**才能触发 / Divinity requires 10 **Mantra** to trigger
- 进入神格后伤害×3，配合**天眼**一回合可以打出破百伤害 / Triple damage in Divinity + **Brilliance** = 100+ damage in one turn
- **斋戒**是高效的神格积累卡 / **Prostrate** is efficient Mantra gain

### 3. 以静制动 / Retain Build

**核心思路**：利用大量保留卡，规划好手牌，在合适的时机打出爆发 / Use Retain cards extensively, plan your hand, and burst at the perfect moment.

**关键卡牌 / Key Cards**:

| 卡牌 / Card | 费用 / Cost | 说明 / Description |
|------|:----:|------|
| **征服 / Conclude** | 1(0) | 对所有敌人造成8（12）伤害，退出当前姿态 / Deal 8(12) to ALL enemies, exit your stance |
| **审判 / Judgment** | 2(1) | 如果敌人HP<=30（40），直接击杀 / If enemy HP ≤ 30(40), kill them |
| **破灭 / Ragnarok** | 3 | 对所有敌人造成5（6）伤害×6（7）次？不对——实际是5次×5（6）/ Wait, let me check - it hits all enemies 5(6) times for 5(6) damage. Actually Ragnarok is: Deal 5(6) damage 5(6) times to ALL enemies. |
| **涅槃 / Nirvana** | 1 | 获得6（9）格挡，预见4（6）/ Gain 6(9) Block, Scry 4(6) |
| **威吓 / Intimidate** | 0(0) | 对所有敌人施加1（2）层虚弱 / Apply 1(2) Weak to ALL enemies |

**打法要点 / Tips**:
- 观者的保留卡让她可以完美规划几回合后的输出 / Retain cards let you plan multiple turns ahead
- **审判**可以直接秒杀杂兵，对付高血量敌人时效果一般 / **Judgment** insta-kills trash mobs but is less useful vs high-HP enemies
- **征服**配合愤怒姿态可以打出极高AOE / **Conclude** in Wrath stance = massive AoE

### 4. 无限循环流 / Infinite Loop

**核心思路**：观者拥有杀戮尖塔中最容易实现的无限循环，只需要特定几张卡 / The Watcher has the easiest infinite loop in Slay the Spire, requiring only specific cards.

**核心组合 / Core Combo**:
- **暴怒（Eruption）**或**警惕（Vigilance）** / **Eruption** or **Vigilance**
- **内心平静（Inner Peace）**或**像即空（Empty Mind）** / **Inner Peace** or **Empty Mind**
- **不惧死亡（Fear No Evil）**（备选）/ **Fear No Evil** (alternative)
- 牌组总卡数≤10张 / Total deck size ≤ 10

**打法要点 / Tips**:
- 观者的无限循环是**最实用**的无限之一，因为姿态切换自带伤害和能量 / The Watcher's infinite is one of the **most practical** — stance switching inherently provides damage and energy
- 只需要1费进入愤怒 + 1费退出进入平静 + 姿态退出给2能量 = 能量正循环 / 1 cost Wrath + 1 cost exit to Calm + 2 energy from Calm = net positive energy

## 观者通用策略 / General Strategy

| 阶段 / Phase | 策略 / Strategy |
|------|------|
| **Act 1 前期** | 观者前期极强，**暴怒**升级后1费进入愤怒，配合任何攻击卡都能打出高伤 / Watcher is extremely strong early — upgraded **Eruption** at 1 cost makes any attack card hit hard |
| **Act 1 中期** | 找退出姿态和过牌的卡（**化作即空**/**内心平静**），确保姿态循环 / Find stance exit and draw cards for consistent stance cycling |
| **Act 1 后期** | 观者打Act 1 BOSS通常很轻松，注意防止暴毙（愤怒态受双倍伤害）/ Watcher handles Act 1 bosses easily — just avoid dying in Wrath (double damage taken) |
| **Act 2** | 小心被秒杀！愤怒态下Act 2的精英伤害极高。优先拿格挡和退出姿态卡 / Watch out for being oneshot! Act 2 elites hit extremely hard in Wrath. Prioritize block and stance exit |
| **Act 3** | 观者到Act 3基本成型，注意不要被心脏的律动伤害打死 / Watcher is usually fully built by Act 3 — watch out for the Heart's beat damage |

## 观者最重要技巧 / Most Important Tips

- **愤怒态下你也是脆皮**——愤怒双倍伤害是双向的，你打敌人双倍，敌人打你也是双倍 / **Wrath works both ways** — you deal double but also take double. One mistake = death
- **上**的节奏非常重要——不要在愤怒态下结束回合 / **Stance management is everything** — never end your turn in Wrath unless you know you won't take damage
- **观者不需要太多卡**——15张以内足矣，精简即力量 / **Watcher doesn't need many cards** — 15 or fewer, lean = powerful
- **最容易达成无限的职业**——利用姿态切换实现能量和抽牌的正循环 / **Easiest infinite in the game** — stance switching enables positive energy + draw loops
