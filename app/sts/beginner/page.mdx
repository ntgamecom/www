# 新手入门 / Beginner Guide

杀戮尖塔是一款卡牌构建Roguelike游戏——每次开始都是全新的旅程，用随机获得的卡牌构建你的牌组，爬上高塔、击败BOSS / Slay the Spire is a deck-building roguelike — every run is a fresh journey. Build your deck from randomly offered cards, climb the Spire, and defeat the bosses.

---

## 核心概念 / Core Concepts

### 能量系统 / Energy System

每回合获得3点能量（可提升），每张卡牌有能量费用，消耗能量才能打出 / You get 3 energy per turn (upgradeable). Each card costs energy to play.

- **初始能量**：3点 / Starting energy: 3
- **提升方式**：获得**Boss遗物**（如能量核心）、某些事件 / Upgrade via **Boss Relics** (e.g., Energy Core), certain events

### 生命与休息 / HP & Rest

- **生命值（HP）**：归零则游戏结束 / HP reaches 0 = run ends
- **篝火（Campfire）**：在篝火处可以**休息**（回复30%HP）或**升级卡牌** / At campfires: **Rest** (heal 30% HP) or **Upgrade** a card
- **药水（Potions）**：战斗中可使用，提供一次性效果 / Use potions during combat for one-time effects

### 卡牌类型 / Card Types

| 类型 / Type | 颜色 / Color | 说明 / Description |
|------|:----:|------|
| **攻击 / Attack** | 红色 / Red | 造成伤害 / Deal damage |
| **技能 / Skill** | 绿色 / Green | 格挡、抽牌、增益等 / Block, draw, buff, etc. |
| **能力 / Power** | 蓝色 / Blue | 整场战斗生效的被动效果 / Passive effects lasting the entire combat |

### 卡牌稀有度 / Card Rarity

| 稀有度 / Rarity | 颜色 / Color | 出现概率 / Rate |
|------|:----:|:----:|
| **普通 / Common** | 白色 / White | 最常见 / Most common |
| **罕见 / Uncommon** | 蓝色 / Blue | 中等 / Medium |
| **稀有 / Rare** | 金色 / Gold | 最稀有 / Rarest |

### 遗物 / Relics

遗物是永久生效的特殊物品，不可移除 / Relics are permanent special items that cannot be removed.

- **普通遗物 / Common Relics**：基本效果 / Basic effects
- **罕见遗物 / Uncommon Relics**：中等效果 / Medium effects
- **稀有遗物 / Rare Relics**：强力效果 / Powerful effects
- **BOSS遗物 / Boss Relics**：击败BOSS后获得，通常有正面效果也有负面代价 / Earned from bosses, usually have both benefits and drawbacks

### 药水 / Potions

战斗中可携带最多2瓶药水，提供一次性强力效果 / Carry up to 2 potions into combat for one-time powerful effects.

| 常用药水 / Key Potions | 效果 / Effect |
|------|------|
| **烈焰药水 / Fire Potion** | 造成20点伤害 / Deal 20 damage |
| **格挡药水 / Block Potion** | 获得12点格挡 / Gain 12 block |
| **能量药水 / Energy Potion** | 获得2点能量 / Gain 2 energy |
| **力量药水 / Strength Potion** | 获得2点力量 / Gain 2 strength |
| **敏捷药水 / Dexterity Potion** | 获得2点敏捷 / Gain 2 dexterity |
| **再生药水 / Regeneration Potion** | 每回合回复5HP持续5回合 / Regen 5 HP/turn for 5 turns |

## 通用技巧 / General Tips

| 技巧 / Tip | 说明 / Description |
|------|----------|
| **路线规划 / Path Planning** | 先看地图终点，规划最佳路线。精英怪给遗物，篝火升级，问号房随机事件 / Plan your path based on boss. Elites = relics, campfires = upgrades, ? rooms = events |
| **早期抓攻击牌 / Early Attacks** | Act 1前期多拿攻击牌，否则打精英和BOSS输出不够 / Prioritize attack cards early in Act 1 or you won't have enough damage for elites and the boss |
| **删牌比加牌更重要 / Remove > Add** | 商店优先删除初始的**打击**和**防御**，让牌组更精简 / Prioritize removing basic **Strikes** and **Defends** at shops for a leaner deck |
| **卡组不宜过大 / Keep Deck Lean** | 15-25张卡是最佳区间，太多导致关键牌抽不到 / 15-25 cards is the sweet spot — too many means you won't draw key cards |
| **遗物影响卡牌选择 / Relics Guide Picks** | 拿到某系遗物就优先选配合的卡——如拿到**蛇之戒指**后走弃牌流 / If a relic supports an archetype (e.g., **Snake Ring** for discard), lean into it |
| **血量是资源 / HP Is a Resource** | 不要害怕掉血去拿更好的卡或遗物。吃下伤害换取强度是值得的 / Don't be afraid to take HP loss for better cards/relics. Tanking damage for power is often worth it |
| **药水留着打BOSS / Save Potions** | 除非要死了，否则药水尽量留给精英和BOSS战 / Unless you're about to die, save potions for elites and bosses |
| **看地图终点选BOSS / Check Boss Early** | Act 1开局就确认最终BOSS，围绕它构建卡组 / Check the final boss at the start of Act 1 and build your deck around countering it |

## 进阶建议 / Advanced Tips

- **了解敌人意图（Intent）**：敌人头上图标显示即将采取的行动，据此决策 / Learn enemy **intents** — the icon above enemies shows their next action, plan accordingly
- **牌组协同 > 单卡强度**：两张普通卡配合好，强过一张稀有卡没配合 / **Deck synergy > single card power** — two common cards that synergize beat an unsupported rare card
- **战斗奖励选择**：缺输出拿攻击，缺防御拿格挡，后期拿能力卡 / **Choose rewards wisely** — need damage → attacks; need defense → blocks; late game → power cards
- **不要执着于固定流派**：根据拿到的卡和遗物灵活调整 / **Don't force an archetype** — adapt based on the cards and relics you're offered
- **高进阶需注意**：进阶15以上时精英和BOSS大幅加强，需要更优路线规划 / **High Ascension note**: above Ascension 15, elites and bosses are significantly stronger, requiring better path planning
- **遗物互动的隐藏配合**：如**符文金字塔** + **战术**可无限循环 / **Hidden synergies**: e.g., **Runic Pyramid** + **Tactician** enables infinite loops

## Act BOSS弱点 / Act Boss Weaknesses

| BOSS / Boss | 弱点 / Weakness | 应对 / Counter |
|---|---|---|
| **史莱姆Boss(Act 1)** | 分身后血量共享——AOE快速清小史莱姆 / Split HP shared — AOE to clear small slimes fast | 留AOE牌——分身后一波清 / Save AOE for split phase |
| **守护者(Act 1)** | 攻击模式固定——格挡后反击 / Fixed pattern — block then counter | 叠格挡等它攻击——反弹伤害 / Stack block, let it attack, reflect damage |
| **六火亡魂(Act 1)** | 第一回合强化状态不打——做防御准备 / First turn buffs — prepare defense | 第一回合叠格挡——第二回合全力输出 / First turn: block. Second turn: full damage |
| **铜制机械人偶(Act 2)** | 召唤小怪时弱AOE——本体不强 / Summon phase weak to AOE — boss is weak itself | 留AOE清召唤物——集中打本体 / AOE for summons, focus boss |
| **收藏家(Act 2)** | 每回合变强——需要速战速决 / Scales every turn — needs fast kill | 前期全力输出——不能拖长 / Go all-in early — can't drag out |
| **书呆子(Act 2)** | 攻击模式固定——第一回合弱 / Fixed pattern — weak first turn | 第一回合叠BUFF——第二回合爆发 / First turn buff, second turn burst |
| **觉醒者(Act 3)** | 能力越多越强——转型卡组对他无效 / More powers = stronger — punishes power-heavy decks | 少拿能力牌——以技能和攻击为主 / Fewer powers — focus on skills and attacks |
| **时间吞噬者(Act 3)** | 每12张牌行动一次——控制出牌数 / Reacts every 12 cards — control hand count | 拿大牌——少出牌但每张高质量 / Play few but high-impact cards |
| **堕落之心(Act 3)** | 叠力量+多段攻击——需要格挡应对 / Scales strength + multi-hit — needs block | 叠大量格挡——不要给他叠力量的机会 / Huge block — don't let it scale strength |

## 新手指南总结 / Beginner Summary

| 关键点 / Key Point | 一句话 / One Liner |
|---|---|
| **删牌 > 加牌** / Removal > Addition | 牌组越小越稳定——商店删牌是最好投资 / Smaller deck = more consistent — shop removal is the best investment |
| **路线规划** / Route Planning | 看地图规划路线——精英给遗物但危险——篝火升级优于回血 / Plan route on map — elites give relics but are risky — upgrade over heal |
| **Act阶段重点** / Act Focus | Act1拿攻击牌——Act2平衡攻防——Act3验证构筑 / Act1: attacks — Act2: balance — Act3: verify build |
| **遗物决定流派** / Relics Define Build | 拿到核心遗物后围绕它构筑——不要逆遗物而行 / Build around your key relics — don't fight against them |
