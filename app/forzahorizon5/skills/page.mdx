## 技能系统总览 / Skills System Overview

地平线5的技能系统由角色技能（Perks）和车辆技能（Car Mastery）两大部分组成。通过完成各种特技动作积累技能点数，消耗技能点数解锁各类加成。合理规划技能点数的使用可以大幅提升游戏效率和收益。

The FH5 skills system consists of two main parts: Character Perks and Car Mastery. Accumulate Skill Points by completing various stunt actions, then spend them to unlock various bonuses. Strategic skill point allocation can significantly improve game efficiency and rewards.

## 技能点数获取 / Earning Skill Points

### 技能连锁 / Skill Chain

技能连锁是获取技能点数的主要方式。连续完成特技动作而不中断，可以累加评分乘数。乘数越高，最终获得的技能点数越多。

The skill chain is the primary way to earn Skill Points. Continuously complete stunt actions without interruption to accumulate a score multiplier. The higher the multiplier, the more Skill Points you earn at the end.

| 动作 Action | 分数 Points | 说明 Description |
|:-----------|:----------|:----------------|
| 漂移 Drift | +50 每段 / per segment | 保持漂移状态可持续得分 / Sustained drifting scores continuously |
| 烧胎 Burnout | +200 | 原地烧胎 / Stationary burnout |
| 空中飞跃 Air Jump | +100 起 / base | 飞跃距离越远分数越高 / Further jumps score more |
| 两轮驾驶 Two Wheels | +200 | 车辆侧倾至两轮行驶 / Car balanced on two wheels |
| 近距贴车 Near Miss | +100 | 与对向车流近距离擦过 / Close pass with oncoming traffic |
| 逆向行驶 Oncoming | +50 每辆 / per car | 在逆行车道行驶 / Driving in oncoming lane |
| 甩尾 E-Drift | +100 | 长距离漂移 / Long sustained drift |

**连锁中断原因 / Chain Break Causes**:
- 车辆完全停止超过 2 秒 / Vehicle fully stopped for over 2 seconds
- 严重撞击到墙壁或障碍物 / Heavy collision with wall or obstacle
- 翻滚 / Vehicle rollover
- 倒车行驶 / Reversing
- 退出车辆 / Exiting the vehicle

### 连锁技巧 / Chain Tips

**保持动量 / Maintain Momentum**:
技能连锁的核心是保持车辆持续运动。即使在开阔直道上，也可以通过小幅转向（蛇形行驶）来避免连锁中断。在城市区域，利用车流进行近距贴车是维持连锁的好方法。

The key to skill chains is keeping the vehicle in constant motion. Even on open straights, use slight steering (weaving) to avoid chain breaks. In city areas, using traffic for near misses is a great way to maintain the chain.

**安全连锁区域 / Safe Chain Areas**:
- 瓜纳华托城市街道（车流量大，近距贴车机会多）/ Guanajuato city streets (heavy traffic, many near-miss opportunities)
- 高速公路 Highways（高速行驶，适合飞跃和极速）/ High-speed driving, good for jumps and top speed
- 农田 Farmland（开阔空间，少量障碍物）/ Open space, few obstacles
- 海滩 Beach（平整长距离沙地）/ Flat long sand stretch

**高效刷技能点 / Efficient Skill Point Farming**:
1. 在瓜纳华托的城市街道中以中速行驶 / Drive at medium speed in Guanajuato city streets
2. 在车流中穿梭获得近距贴车 + 漂移分数 / Weave through traffic for near misses + drift points
3. 连锁乘数达到 5x 以上后寻找安全路段结算 / After reaching 5x+ multiplier, find a safe spot to cash in
4. 购买 Casa Bella 房产解锁漂移套件额外加分 / Buy Casa Bella property for额外 drift bonus points

## 技能歌曲 / Skill Songs

技能歌曲是地平线5的独特机制。游戏中会随机播放带有金色标记的电台歌曲。在技能歌曲播放期间，所有技能连锁获得的分数翻倍（2x 基础分）。这是刷技能点数的黄金时间。

Skill Songs are a unique FH5 mechanic. Radio stations randomly play songs marked with a golden icon. During a Skill Song, all skill chain scores are doubled (2x base). This is the golden time for farming Skill Points.

### 技能歌曲机制 / Skill Song Mechanics

- 触发方式: 收听任何游戏内电台，歌曲随机触发 / Listen to any in-game radio station, songs trigger randomly
- 持续时间: 约 2-3 分钟 / Approximately 2-3 minutes
- 效果: 所有技能动作基础分数翻倍 / All skill action base scores doubled
- 提示: 屏幕左上角会出现金色音符图标 / A golden note icon appears in the top-left corner
- 叠加: 技能歌曲的 2x 与连锁乘数叠加计算（最高可达 20x）/ Skill Song 2x stacks with chain multiplier (up to 20x)

**技能歌曲策略 / Skill Song Strategy**:
当技能歌曲播放时，立即停止当前活动，专注做特技连锁。如果附近有 PR 特技（漂移区、测速区间），也可以利用技能歌曲的双倍加成来冲击高分记录。

When a Skill Song plays, immediately stop current activities and focus on stunt chaining. If near a PR Stunt (drift zone, speed trap), use the double bonus to challenge high score records.

| 服装服装 Fashion | 服装和外观物品掉落率 / Clothing and cosmetic drop rate | 低 Low |
| 特技特技 Stunts | PR特技分数加成 / PR stunt score bonus | 中 Medium |

### 推荐加点顺序 / Recommended Perk Order

**第一优先 / First Priority**: 地平线技能树 Horizon Tree
- 核心加成: 所有影响力获取 +15% / All Influence gain +15%
- 快速旅行折扣: 从 50% 折扣到完全免费 / Fast travel discount from 50% to free
- 这是最重要的技能树，尽早点满 / This is the most important tree, max it early

**第二优先 / Second Priority**: 竞速技能树 Racing Tree
- 核心加成: 赛事 Cr 奖励 +20% / Event Cr reward +20%
- 赛季赛奖励加成 / Seasonal event reward bonus
- 加速资金积累 / Accelerate credit accumulation

**第三优先 / Third Priority**: 车辆技能树 Cars Tree
- 核心加成: 车展车辆价格折扣 / Autoshow car price discount
- 转盘车辆概率提升 / Increased wheelspin car probability
- 节省大量 Cr / Save significant Cr

**第四优先 / Fourth Priority**: 特技技能树 Stunts Tree
- 核心加成: PR特技分数加成 / PR stunt score bonus
- 完成三星挑战更容易 / Easier to complete 3-star challenges
- 对全成就玩家重要 / Important for completionists

**第五优先 / Fifth Priority**: 探索技能树 Exploration Tree
- 加成: 发现地标和广告牌额外奖励 / Extra rewards for discovering landmarks and boards
- 推荐在探索地图时逐步解锁 / Unlock gradually during map exploration

**最后 Last**: 服装技能树 Fashion Tree
- 纯外观加成 / Pure cosmetic bonuses
- 对游戏玩法没有实质影响 / No substantive gameplay impact

## Forza LINK 语音系统 / Forza LINK Voice System

Forza LINK 是地平线5的语音指令系统。玩家可以通过语音指令与队友和 AI 车手进行交流。该功能在 Xbox 和 PC 上均可使用。

Forza LINK is the voice command system in FH5. Players can communicate with teammates and AI drivers through voice commands. This feature is available on both Xbox and PC.

### 常用语音指令 / Common Voice Commands

**导航指令 Navigation Commands**:
- "Set a destination" — 设置导航目的地
- "Route to closest event" — 导航到最近的赛事
- "Route to fast travel" — 导航到最近的快速旅行点
- "Cancel route" — 取消导航

**社交指令 Social Commands**:
- "Follow me" — 邀请队友跟随
- "Let is race" — 发起竞速挑战
- "Nice driving" — 赞扬队友驾驶表现
- "Good game" — 比赛结束后的礼貌用语

**车辆指令 Vehicle Commands**:
- "Switch car" — 更换车辆
- "Upgrade car" — 升级当前车辆
- "Show stats" — 显示车辆性能数据

### Forza LINK 使用技巧 / Forza LINK Tips

- 使用 Forza LINK 可以减少在菜单中操作的时间 / Using Forza LINK reduces menu navigation time
- 在驾驶过程中语音指令比手动操作更安全 / Voice commands are safer than manual操作 during driving
- 部分指令需要联网才能使用 / Some commands require online connection
- 可以在设置中关闭 Forza LINK 功能 / Can be disabled in settings

## 车辆 Mastery / Car Mastery

每辆车都有独立的 Mastery 技能树，需要消耗技能点数解锁。Mastery 加成包括性能提升、Cr 奖励和影响力加成。

Each car has an independent Mastery skill tree that requires Skill Points to unlock. Mastery bonuses include performance improvements, Cr rewards, and Influence bonuses.

### Mastery 技能树结构 / Mastery Tree Structure

| 技能类型 Skill Type | 效果 Effect | 技能点数成本 Cost |
|:------------------|:-----------|:----------------|
| 基础加成 Basic Bonus | 影响力 +5% 到 +15% / Influence +5% to +15% | 1-3 点 / points |
| Cr 奖励 Cr Reward | 一次性 Cr 奖励 5,000 到 50,000 / One-time Cr reward | 2-5 点 / points |
| 车辆性能 Performance | 操控、加速提升 / Handling, acceleration boost | 3-7 点 / points |
| 稀有奖励 Rare Reward | 限时获得独占部件或车辆 / Limited exclusive parts or cars | 5-10 点 / points |
| 大师技能 Master Skill | 全力解锁后获得的终极加成 / Ultimate bonus after full unlock | 10-15 点 / points |

有些车辆的 Mastery 树中提供了高额 Cr 奖励。投入少量技能点数即可获得数倍于成本 Cr 回报。

Some car Mastery trees offer substantial Cr rewards. Investing a few Skill Points can yield several times the cost in Cr returns.

**大师技能 / Master Skills**:
每辆车的 Mastery 树顶端有一个"大师技能"，通常是该车辆独有的特殊加成。解锁大师技能需要先解锁路径上的所有前置技能。

At the top of each car Mastery tree is a "Master Skill", typically a unique bonus specific to that car. Unlocking the Master Skill requires unlocking all prerequisite skills on the path.
