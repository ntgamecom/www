# BOSS攻略 / Boss Fights Guide

天国拯救2的Boss战主要是人形敌人——但护甲、武器和战斗技巧的差异创造了独特的挑战 / KCD2 boss fights are primarily humanoid — but armor, weapon, and skill differences create unique challenges.

---

## 竞技场Boss / Tournament Bosses

### 拉迪亚茨克竞技场 / Rattay Tourney

| 轮次 / Round | 对手 / Opponent | 武器 / Weapon | 风格 / Style | 难度 / Difficulty |
|---|---|---|---|---|
| 第一轮 / Round 1 | 见习骑士 / Squire | 长剑 / Longsword | 基础——测试基本功 / Basic | 低 / Low |
| 第二轮 / Round 2 | 老兵 / Veteran | 长剑+盾 / Sword+Shield | 防守反击型 / Counter-fighter | 中等 / Medium |
| 第三轮 / Round 3 | 黑骑士 / Black Knight | 双手剑 / Greatsword | 纯进攻——高伤害 / Aggressive | 中高 / Medium-High |
| 半决赛 / Semi-Final | 精英骑士 / Elite Knight | 钉头锤+盾 / Mace+Shield | 破甲流 / Armor-breaking | 高 / High |
| 决赛 / Final | 冠军 / Champion | 随机 / Random | 掌握一切技巧 / All skills | 极高 / Extreme |

**竞技场冠军打法 / Champion Strategy**:

| 要点 / Key | 说明 / Detail |
|---|---|
| 战前准备 / Prep | 确保装备耐久满+修理工具 / Full repair + repair kit |
| 观察 / Observe | 前几轮观察对手的出招习惯 / Watch opponent patterns |
| 耐力管理 / Stamina | 不要一直攻击——保持耐力 / Don't spam attack |
| 大师格挡 / Master Strike | 这是获胜关键 / Key to victory |
| 利用刺击 / Use Thrust | 刺击对板甲最有效 / Thrust is best vs plate |

## 库曼指挥官 / Cuman Commanders

| 指挥官 / Commander | 位置 / Location | 武器 / Weapon | 护甲 / Armor |
|---|---|---|---|
| 库曼队长 / Cuman Captain | 强攻击败 / Cuman Camp | 弯刀 / Scimitar | 东方板甲 / Eastern Plate |

**对抗库曼技巧 / Cuman Tips**:
- 他们擅长马战——下马战斗更有优势 / Dismount for advantage
- 弯刀攻击速度快——格挡时机需要适应 / Fast attacks — adapt parry
- 库曼人不会大师格挡——你可以用连招破防 / They don't master strike
- 先杀弓手——他们的箭很准 / Kill archers first

## Boss战准备 / Boss Fight Preparation

### 战前检查清单 / Pre-Fight Checklist

| 准备项 / Item | 说明 / Detail |
|---|---|
| **板甲耐久 / Plate Durability** | 全部100% 或 带修理套件 |
| **武器 / Weapon** | 磨刀到100%——带备用武器 |
| **药水 / Potions** | 马利筋剂+止痛剂+救世主药剂 |
| **食物 / Food** | 携带充足食物——维持体力 |
| **修理套件 / Repair Kit** | 战斗中武器会磨损——需要修理 |
| **弓箭 / Bows** | 远程处理弓手或引怪 |

### 战斗通用技巧 / General Combat Tips

| 技巧 / Tip | 说明 / Detail |
|---|---|
| 一对一 / 1v1 | 尽量一对一——不要被包围 / Try for 1v1 |
| 地形利用 / Terrain | 利用斜坡和树木阻挡追兵 / Use terrain |
| 后退战斗 / Backpedal | 边打边退——不陷入包围 / Walk backward |
| 保护头部 / Protect Head | 头部被命中伤害最高——头盔很重要 |
| 耐力管理 / Stamina | 耐力低时不要攻击——先恢复 / Regen stamina |
| 观察对手 / Observe | 每个对手习惯不同——先观察再打 |
