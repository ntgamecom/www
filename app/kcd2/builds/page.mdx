# 流派构建 / Builds Guide

天国拯救2没有传统的职业系统——而是通过技能和属性的组合形成不同的玩法风格 / Kingdom Come Deliverance 2 has no class system — playstyles are formed through skill and attribute combinations.

---

## 主要属性 / Main Attributes

### 属性概览 / Attribute Overview

| 属性 / Attribute | 效果 / Effect | 提升方法 / Level Method |
|---|---|---|
| **力量 / Strength** | 近战伤害+耐用+力量型武器需求 / Melee dmg + carry weight | 近战战斗+重体力劳动 / Melee + heavy work |
| **敏捷 / Agility** | 攻速+轻型武器需求+潜行 / Attack speed + light weapons + stealth | 轻型武器+潜行+弓箭 / Light weapons + bows |
| **活力 / Vitality** | 生命+耐力+负重 / HP + stamina + carry | 跑步+游泳+受击 / Run + swim + hit |
| **口才 / Speech** | 说服+商店价格+声望 / Persuasion + prices + rep | 对话+阅读+辩论 / Dialogue + books |
| **酒神 / Drinking** | 酒精抗性+醉酒buff / Alcohol resistance + drunk buffs | 喝酒 / Drink alcohol |
| **药神 / Herbology** | 炼金配方+材料效果 / Alchemy recipes + herb effects | 采集草药+炼金 / Gather herbs + alembic |

---

### 属性成长推荐 / Attribute Growth

| 流派 / Build | 优先属性 / Priority | 次要属性 / Secondary |
|---|---|---|
| 近战骑士 / Knight | 力量 > 活力 > 敏捷 / Str > Vit > Agi | 口才 / Speech |
| 潜行刺客 / Assassin | 敏捷 > 活力 > 力量 / Agi > Vit > Str | 药神 / Herbology |
| 弓箭游侠 / Ranger | 敏捷 > 活力 / Agi > Vit | 药神+口才 / Herb + Speech |
| 炼金术士 / Alchemist | 药神 > 口才 > 活力 / Herb > Speech > Vit | 其他随意 / Any |
| 全能型 / All-Round | 平均分配——活力优先 / Balanced — Vit first | 口才 / Speech |

## 战斗流派 / Combat Builds

### 长剑流 / Longsword Build

| 项目 / Aspect | 推荐 / Recommendation |
|---|---|
| **武器 / Weapon** | 长剑——全长1.5米以上 / Longsword — 1.5m+ |
| **护甲 / Armor** | 板甲——全身覆盖 / Full plate armor |
| **核心技能 / Core Skills** | 剑术 + 力量 + 活力 / Sword + Strength + Vitality |
| **战斗风格 / Style** | 大师格挡+反击——保持中距离 / Master strike + riposte — mid range |

**长剑优势 / Longsword Advantages**:
| 优势 / Pro | 说明 / Detail |
|---|---|
| 攻击距离长 / Long Range | 比短武器先命中敌人 / Hits before shorter weapons |
| 大师格挡窗口大 / Big Master Strike Window | 更容易触发完美格挡 / Easier to master strike |
| 伤害高 / High Damage | 板甲对砍时伤害可观 / Decent vs armor |
| 可刺击 / Can Thrust | 刺击对板甲有效 / Thrust effective vs plate |

**长剑连招推荐 / Longsword Combos**:
| 连招 / Combo | 输入 / Input | 效果 / Effect |
|---|---|---|
| 蝎子尾 / Scorpion's Tail | 左-右-下 / Left-Right-Bottom | 快速三连——破防 / Fast triple |
| 晨星 / Morning Star | 上-左-右 / Top-Left-Right | 高位起手——针对头盔 / Starts high — targets helmet |
| 蓝刃 / Blue Blade | 右-刺-左 / Right-Thrust-Left | 刺+横斩——对板甲有效 / Thrust + slash |

## 非战斗流派 / Non-Combat Builds

### 口才外交流 / Speech/Dipomacy Build

| 项目 / Aspect | 推荐 / Recommendation |
|---|---|
| **核心属性 / Core** | 口才 (20) + 魅力 (关注服装) / Speech (20) + Charisma (clothing) |
| **支持技能 / Support** | 阅读 + 酒神 / Reading + Drinking |
| **玩法 / Playstyle** | 避免战斗——用对话解决问题 / Avoid fights — talk your way out |
| **服装 / Clothes** | 高魅力服装——整洁华丽 / High charisma — clean and fancy |

**口才流战斗能力 / Speech Build Combat**:
- 技能点有限——战斗能力较弱 / Limited combat ability
- 但不不能完全避免战斗——建议保留基础战斗技能 / Can't always avoid — keep basic combat
- 用对话选项可以避免很多不必要的战斗 / Dialog options avoid many fights
- 更高的口才=更好的商店价格 / Better speech = better prices

### 射手流 / Archer Build

| 项目 / Aspect | 推荐 / Recommendation |
|---|---|
| **武器 / Weapon** | 长弓/轻弩 / Longbow/Crossbow |
| **护甲 / Armor** | 中甲——平衡保护和移动 / Medium armor |
| **核心技能 / Core Skills** | 弓箭 + 敏捷 + 活力 / Bows + Agility + Vitality |
| **战斗风格 / Style** | 远程消耗——近战时才切剑 / Range first — melee backup |

**弓 vs 弩对比 / Bow vs Crossbow**:

| 对比 / Compare | 弓 / Bow | 弩 / Crossbow |
|---|---|---|
| 射速 / Fire Rate | 快——可连续射 / Fast — continuous | 慢——每击后需上弦 / Slow — reload per shot |
| 伤害 / Damage | 中——取决于弓 + 箭 / Medium | 高——但弩机品质影响 / High |
| 精度 / Accuracy | 需要技巧——无准星偏移感 / Skill-based | 更准——机械瞄准 / More accurate |
| 隐蔽性 / Stealth | 声音小 / Quiet | 声音大 / Loud |
| 移动中射击 / Move-Fire | 可以 / Yes | 不可以——必须站立 / Must stand still |

## 属性点分配建议 / Stat Allocation

| 等级 / Level | 推荐加点 / Recommended |
|---|---|
| 1-5 | 活力优先——基础生存 / Vitality first — base survival |
| 6-10 | 主攻属性(力量/敏捷) / Main combat stat (Str/Agi) |
| 11-15 | 口才提升——任务更容易 / Speech — missions easier |
| 16-20 | 满级主攻属性和活力 / Max combat + vitality |
| 20+ | 补满辅助技能 / Fill support skills |
