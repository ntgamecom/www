# 战斗系统 / Combat

KCD2 的战斗系统以其真实性和深度著称。掌握这套系统需要耐心练习，但一旦精通，你将能以一敌多。

The combat system in KCD2 is renowned for its realism and depth. Mastering it takes patience and practice, but once mastered, you can take on multiple opponents.

## 基础战斗机制 / Core Combat Mechanics

### 战斗姿态与方向 / Stance & Direction

战斗基于五个攻击方向：上、下、左、右和刺击。你的鼠标或右摇杆控制出招方向，对手的防御方向决定格挡效果。

Combat uses five attack directions: top, bottom, left, right, and thrust. Your mouse or right stick controls the direction of each attack, while the opponent's guard direction determines block effectiveness.

| 方向 / Direction | 攻击类型 / Attack Type | 最佳应对 / Best Counter |
|---|---|---|
| 上方 / Top | 竖劈 / Vertical Slash | 上格挡 / Block Top |
| 左侧 / Left | 横斩 / Horizontal Slash (Left) | 左格挡 / Block Left |
| 右侧 / Right | 横斩 / Horizontal Slash (Right) | 右格挡 / Block Right |
| 下方 / Bottom | 上撩 / Upward Slash | 下格挡 / Block Low |
| 刺击 / Thrust | 穿刺 / Stab | 格挡刺击 / Block Thrust |

### 耐力系统 / Stamina System

耐力是战斗的核心资源。攻击、格挡和被击中都会消耗耐力。耐力耗尽时，任何攻击都会造成全额伤害并更容易导致失衡。

Stamina is the core resource of combat. Attacking, blocking, and being hit all consume stamina. When stamina is depleted, all attacks deal full damage and are more likely to stagger you.

**耐力管理原则 / Stamina Management Principles**:
- 保持至少 30% 耐力用于防御 / Keep at least 30% stamina for defense
- 不要连续攻击超过三次 / Do not chain more than three attacks
- 后退恢复耐力后再进攻 / Fall back to recover stamina before re-engaging
- 格挡消耗比闪避更少的耐力 / Blocking consumes less stamina than dodging

## Master Strike 系统 / Master Strike System

Master Strike 是战斗系统的核心高级技巧，也是逆转战局的关键。

Master Strike is the core advanced technique and the key to turning the tide of battle.

### 什么是 Master Strike / What is Master Strike

Master Strike 是一种完美格挡后立即发动反击的技巧。它无视对手的护甲，造成固定伤害并打断对手攻击。

Master Strike is a perfect block followed by an immediate counter-attack. It ignores armor, deals fixed damage, and interrupts the opponent's attack chain.

### 学习 Master Strike / Learning Master Strike

你必须与 Rattay 的 Bernard 教官学习这个技能。支付 50 格罗申参加高级训练课程。

You must learn this skill from Captain Bernard in Rattay. Pay 50 Groschen for the advanced training course.

**学习步骤 / Learning Steps**:
1. 完成 Rattay 主线任务直到 Bernard 成为训练师
   Complete Rattay main quests until Bernard becomes available as a trainer
2. 选择「我想练习战斗 / I want to practice combat」
3. 选择「高级技巧 / Advanced techniques」
4. 支付训练费用 / Pay the training fee
5. 跟随屏幕提示完成训练 / Follow on-screen prompts to complete training

### 执行时机 / Execution Timing

Master Strike 的核心在于时机。当敌人攻击动画开始的瞬间，朝相同方向格挡。

The core of Master Strike is timing. Block in the same direction the instant the enemy's attack animation begins.

**判断技巧 / Timing Tips**:
- 观察对手肩膀动作 / Watch the opponent's shoulder movement
- 绿色盾牌图标出现时立即格挡 / Block immediately when the green shield icon appears
- 不要预判方向，等看到攻击再反应 / Do not predict direction; react when you see the attack
- 练习模式下可开启攻击方向指示器 / Enable the attack direction indicator in practice mode

## 武器类型 / Weapon Types

### 长剑 / Longsword

最均衡的武器，双手使用，攻击距离远，适合新手。

The most balanced weapon. Two-handed, long reach, suitable for beginners.

| 属性 / Stat | 数值 / Value |
|---|---|
| 攻击速度 / Speed | 中等 / Medium |
| 伤害 / Damage | 中等 / Medium |
| 穿甲 / Armor Penetration | 高 / High |
| 推荐等级 / Recommended Level | 1-20 |

### 短剑与匕首 / Short Sword & Dagger

速度快但伤害低，配合盾牌使用效果更佳。

Fast but low damage; best used with a shield.

| 属性 / Stat | 数值 / Value |
|---|---|
| 攻击速度 / Speed | 快 / Fast |
| 伤害 / Damage | 低 / Low |
| 穿甲 / Armor Penetration | 低 / Low |
| 推荐等级 / Recommended Level | 1-10 |

### 钝器 / Mace

对抗重甲敌人的最佳选择，但速度慢。

Best against heavily armored enemies, but slow.

| 属性 / Stat | 数值 / Value |
|---|---|
| 攻击速度 / Speed | 慢 / Slow |
| 伤害 / Damage | 高 / High |
| 穿甲 / Armor Penetration | 极高 / Very High |
| 推荐等级 / Recommended Level | 5-20 |

### 长柄武器 / Polearm

攻击范围最大，但无法在战斗中灵活切换方向。

Longest reach, but cannot switch attack directions flexibly in combat.

| 属性 / Stat | 数值 / Value |
|---|---|
| 攻击速度 / Speed | 慢 / Slow |
| 伤害 / Damage | 极高 / Very High |
| 穿甲 / Armor Penetration | 高 / High |
| 推荐等级 / Recommended Level | 10-20 |

### 弓 / Bow

远程攻击武器，需要练习瞄准。箭矢类型影响伤害和穿甲能力。

Ranged weapon requiring practice. Arrow type affects damage and armor penetration.

| 箭矢 / Arrow Type | 伤害 / Damage | 穿甲 / Armor Pen |
|---|---|---|
| 普通箭 / Standard | 50 | 低 / Low |
| 穿甲箭 / Piercing | 35 | 高 / High |
| 猎箭 / Hunting | 60 | 中 / Medium |
| 火矢 / Fire Arrow | 40 + 燃烧 / Burning | 低 / Low |

## 连招系统 / Combo System

连招是预先设定的攻击序列。成功执行连招会造成额外伤害并击退敌人。

Combos are pre-set attack sequences. Successfully executing a combo deals bonus damage and knocks back enemies.

### 基础连招列表 / Basic Combo List

| 连招名称 / Combo Name | 攻击顺序 / Sequence | 效果 / Effect |
|---|---|---|
| 横扫 / Scarmaker | 左 - 右 - 上 / Left-Right-Top | 高伤害 / High Damage |
| 穿刺 / Wrath Strike | 刺 - 刺 - 右 / Thrust-Thrust-Right | 穿甲 / Armor Piercing |
| 连环斩 / Run Through | 上 - 左 - 刺 / Top-Left-Thrust | 击退 / Knockback |
| 双重斩 / Double Cut | 右 - 左 - 下 / Right-Left-Bottom | 出血 / Bleeding |

### 连招技巧 / Combo Tips

- 学习连招需要从 Bernard 处购买技能书
- Combo scrolls are purchased from Bernard
- 在实战中时使用，不要强求连招
- Use combos opportunistically in real combat
- 连招必须在耐力充足时才能完整执行
- Combos require sufficient stamina to execute fully
- 被格挡会中断连招
- Combos are interrupted if any hit is blocked

角色分为四个装备部位，每个部位最多有三层防护。

Characters have four equipment slots, each with up to three armor layers.

| 腿部 / Legs | 腿甲 / Greave | 护膝 / Knee Pad | 裤子 / Hose |

| 属性 / Stat | 含义 / Meaning |

| 钝击防御 / Blunt Defense | 对抗锤和钝器 / Defense against maces and blunt weapons |

## 训练技巧 / Training Tips

1. 每天与 Bernard 练习至少 15 分钟 / Practice with Bernard for at least 15 minutes daily
2. 使用木剑训练避免受伤 / Use wooden swords to avoid injury during training
3. 先掌握完美格挡再学 Master Strike / Master perfect block before learning Master Strike
4. 在实战中优先使用 Master Strike / Prioritize Master Strike in real combat
5. 学会撤退：不要恋战 / Learn to retreat; do not overcommit
6. 利用地形：狭窄通道可限制敌人数量
   Use terrain: narrow passages limit enemy numbers
7. 骑马战斗：骑射或冲刺攻击 / Mounted combat: horse archery or charge attacks

## 高级技巧 / Advanced Techniques

### 假动作 / Feints

改变攻击方向来迷惑对手的格挡。当你开始攻击动画时，快速切换到另一个方向。

Change attack direction mid-swing to confuse blocking. Switch directions quickly after starting the attack animation.

### 组合目标 / Target Switching

在多人战斗中频繁切换目标，防止被包围时遭到背刺。

In multi-enemy fights, switch targets frequently to prevent backstabs while surrounded.

### 踢击与组合 / Kick & Combo

踢击可以破防并创造攻击窗口。连续攻击 - 踢击 - 攻击是基础连段。

Kicks break guard and create attack windows. The basic chain: attack - kick - attack.
