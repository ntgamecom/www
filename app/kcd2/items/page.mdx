**护甲层叠机制 / Layer System**:
- 护甲是多层的——内层+外层=更全面防御 / Multiple layers = better coverage
- 例如: 锁甲内衬 + 板甲胸甲 = 最高物理防御 / Mail + Plate = max def
- 层数越多越重——但保护越好 / More layers = heavier but safer
- 缺少任何部位都会导致该部位容易被重伤 / Missing any part = vulnerability
