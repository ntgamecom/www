# 技能与属性 / Skills & Perks

KCD2 拥有丰富的技能系统，分为主要属性和次要技能。合理规划技能发展路线能大幅提升游戏体验。

KCD2 features a comprehensive skill system divided into primary attributes and secondary skills. Planning your skill progression greatly enhances the gameplay experience.

## 主要属性 / Primary Attributes

### 力量 / Strength

影响近战伤害、负重能力和某些对话选项。通过战斗和使用重型武器提升。

Affects melee damage, carry capacity, and certain dialogue options. Improved through combat and using heavy weapons.

| 等级 / Level | 效果 / Effect |
|---|---|
| 0-5 | 基础近战伤害 / Base melee damage |
| 6-10 | 负重 +15 / Carry Capacity +15 |
| 11-15 | 解锁强力连招 / Unlock powerful combos |
| 16-20 | 终极力量 Perk / Ultimate Strength perk |

**推荐 Perk / Recommended Perks**:
- **铁腕 / Iron Grip**: 减少武器体力消耗 / Reduces weapon stamina cost
- **破甲者 / Armor Breaker**: 增加对护甲敌人的伤害 / Increases damage against armored enemies
- **熊之力 / Bear Strength**: 负重 +25 / Carry Capacity +25

### 敏捷 / Agility

影响攻击速度、弓箭伤害和潜行效果。通过使用轻型武器和弓箭提升。

Affects attack speed, bow damage, and stealth effectiveness. Improved through using light weapons and bows.

| 等级 / Level | 效果 / Effect |
|---|---|
| 0-5 | 基础攻击速度 / Base attack speed |
| 6-10 | 弓箭伤害 +10% / Bow damage +10% |
| 11-15 | 解锁敏捷连招 / Unlock agility combos |
| 16-20 | 终极敏捷 Perk / Ultimate Agility perk |

**推荐 Perk / Recommended Perks**:
- **轻灵 / Light Foot**: 降低移动噪音 / Reduces movement noise
- **快速射击 / Fast Draw**: 增加弓箭拉弓速度 / Increases bow draw speed
- **猫之优雅 / Cat's Grace**: 掉落伤害减半 / Half fall damage

### 活力 / Vitality

影响生命值、耐力和恢复速度。通过跑步、游泳和承受伤害提升。

Affects health, stamina, and recovery speed. Improved through running, swimming, and taking damage.

| 等级 / Level | 效果 / Effect |
|---|---|
| 0-5 | 基础生命值 / Base health |
| 6-10 | 耐力恢复 +15% / Stamina recovery +15% |
| 11-15 | 生命值 +20 / Health +20 |
| 16-20 | 终极活力 Perk / Ultimate Vitality perk |

**推荐 Perk / Recommended Perks**:
- **铁肺 / Iron Lungs**: 耐力恢复速度翻倍 / Doubles stamina recovery speed
- **坚韧 / Toughness**: 固定减伤 +3 / Flat damage reduction +3
- **恢复 / Recuperation**: 非战斗状态下缓慢恢复生命
   Slowly regenerate health outside combat

### 口才 / Speech

影响对话成功率、砍价能力和谎言识别。通过对话、演讲和阅读提升。

Affects dialogue success rate, haggling ability, and lie detection. Improved through conversation, speech, and reading.

| 等级 / Level | 效果 / Effect |
|---|---|
| 0-5 | 基础说服率 / Base persuasion rate |
| 6-10 | 商人价格 -10% / Merchant prices -10% |
| 11-15 | 解锁特殊对话选项 / Unlock special dialogue options |
| 16-20 | 终极口才 Perk / Ultimate Speech perk |

**推荐 Perk / Recommended Perks**:
- **巧舌如簧 / Silver Tongue**: 说服成功率 +15% / Persuasion success +15%
- **讨价还价 / Haggler**: 买卖价格各提升 10% / Buy/sell prices improved by 10%
- **洞察 / Empathy**: 更容易识破谎言 / Easier to detect lies

## 次要技能 / Secondary Skills

### 阅读 / Reading

初始为零，需找抄写员学习。阅读技能影响书籍使用和任务解法。

Starts at zero; must learn from a scribe. Affects book usage and quest solutions.

**学习地点 / Learning Location**: Rattay 抄写员处支付 100 格罗申学习 / Learn from the Rattay scribe for 100 Groschen

**收益 / Benefits**:
- 可以阅读技能书（获得经验加成）
- Can read skill books (gain experience bonuses)
- 解锁某些任务的特殊解法
- Unlocks special solutions for certain quests
- 破解密码和文字谜题
- Decipher codes and text puzzles

### 潜行 / Stealth

影响潜行移动速度、背刺伤害和被发现几率。

Affects stealth movement speed, backstab damage, and detection chance.

| 等级 / Level | 效果 / Effect |
|---|---|
| 0-5 | 基础潜行速度 / Base sneak speed |
| 6-10 | 背刺伤害 x2 / Backstab damage x2 |
| 11-15 | 夜间隐形增强 / Enhanced night invisibility |
| 16-20 | 终极潜行 Perk / Ultimate Stealth perk |

**推荐 Perk / Recommended Perks**:
- **暗影行者 / Shadow Stride**: 潜行移动速度 +20% / Sneak speed +20%
- **无声杀手 / Silent Killer**: 背刺不会触发警报 / Backstabs do not trigger alarms
- **夜猫 / Night Owl**: 夜间潜行效果增强 30% / Night stealth improved by 30%

### 开锁 / Lockpicking

用于开启上锁的箱子和门。难度取决于锁的等级。

Used to open locked chests and doors. Difficulty depends on lock level.

| 锁等级 / Lock Level | 推荐技能 / Recommended Skill | 收益 / Reward |
|---|---|---|
| 简单 / Easy | 1-3 | 普通物品 / Common items |
| 中等 / Medium | 4-7 | 优质物品 / Quality items |
| 困难 / Hard | 8-12 | 稀有物品 / Rare items |
| 极难 / Very Hard | 13-15 | 传奇物品 / Legendary items |
| 大师 / Master | 16-20 | 史诗物品 / Epic items |

**训练技巧 / Training Tips**:
- 在 Rattay 铁匠处购买练习锁 / Buy practice locks from the Rattay blacksmith
- 夜间行动降低被发现风险 / Act at night to reduce detection risk
- 开锁前保存 / Save before attempting locks
- 使用幸运之骰 Perk 降低开锁难度
- Use the Lucky Die perk to reduce lock difficulty

### 炼金 / Alchemy

制作药水和毒药。需要使用炼金台按配方制作。

Brew potions and poisons. Requires an alchemy bench and recipes.

**基础配方 / Basic Recipes**:

| 药水 / Potion | 效果 / Effect | 材料 / Ingredients |
|---|---|---|
| 救星酒 / Saviour Schnapps | 手动保存 / Manual save | 荨麻 + 缬草 / Nettle + Valerian |
| 治疗剂 / Healing Potion | 恢复生命 / Restore health | 荨麻 + 聚合草 / Nettle + Comfrey |
| 夜莺 / Night Hawk | 夜视 / Night vision | 颠茄 + 薄荷 / Belladonna + Mint |
| 射手酒 / Bowman's Brew | 增强弓箭 / Boost bow | 蒲公英 + 蓍草 / Dandelion + Yarrow |

**炼金技巧 / Alchemy Tips**:
- 按正确顺序添加材料 / Add ingredients in the correct order
- 煮沸时间影响药水质量 / Boil time affects potion quality
- 使用新鲜材料获得最佳品质 / Use fresh ingredients for best quality
- 阅读炼金配方书解锁新配方 / Read alchemy recipe books to unlock recipes

### 工艺 / Craftsmanship

包括锻造和维修装备。使用铁砧制作和修理物品。

Includes blacksmithing and equipment repair. Use an anvil to craft and repair items.

| 等级 / Level | 解锁 / Unlock |
|---|---|
| 0-5 | 基础维修 / Basic repair |
| 6-10 | 制作普通武器 / Craft common weapons |
| 11-15 | 制作优质装备 / Craft quality gear |
| 16-20 | 制作传奇装备 / Craft legendary gear |

**推荐 Perk / Recommended Perks**:
- **能工巧匠 / Skilled Artisan**: 制作装备品质 +1 tier / Craft quality +1 tier
- **精打细算 / Thrifty**: 维修成本减半 / Repair cost halved
- **锻造大师 / Master Smith**: 解锁传奇装备制作 / Unlock legendary crafting

## 技能升级策略 / Leveling Strategy

### 高效升级路线 / Efficient Leveling Path

1. **1-5 级**: 在 Rattay 周边采集草药（采集 + 活力）
   Gather herbs around Rattay (gathering + vitality)
2. **5-10 级**: 日常与 Bernard 战斗训练（力量 + 敏捷 + 活力）
   Daily combat training with Bernard (strength + agility + vitality)
3. **10-15 级**: 探索地图各地的营地（实战提升）
   Explore camps across the map (real combat improvement)
4. **15-20 级**: 完成高难度任务和挑战
   Complete high-difficulty quests and challenges

### 通用技巧 / General Tips

- 技能书提供大量经验 / Skill books provide large experience boosts
- 向训练师支付费用可快速升级 / Paying trainers provides quick level-ups
- 不同活动提升不同技能 / Different activities improve different skills
- 平衡发展避免偏科 / Balanced development avoids weaknesses
- Perk 可以在洗点处重置 / Perks can be reset at a respec location
