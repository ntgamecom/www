# 任务攻略 / Quests

KCD2 的任务设计以非线性选择著称。你的决定会影响 NPC 的命运、阵营关系，甚至结局走向。

KCD2's quest design is renowned for non-linear choices. Your decisions affect NPC fates, faction relations, and even the ending.

## 主线任务结构 / Main Quest Structure

主线分为多个章节，每个章节包含数个主要任务和可选目标。

The main story is divided into chapters, each containing several main quests and optional objectives.

### 第一章：复仇之始 / Chapter 1: The Beginning of Vengeance

| 任务 / Quest | 推荐等级 / Level | 关键选择 / Key Choice |
|---|---|---|
| 序章 / Prologue | 1 | 是否帮助泰丽莎 / Help Theresa or not |
| 抵达 Rattay / Arrival in Rattay | 2 | 如何获取装备 / How to acquire gear |
| 训练之日 / Training Day | 3 | 训练时间长短 / Training duration |
| 猎狼 / Hunting Wolves | 4 | 是否杀狼或驱赶 / Kill or drive away wolves |

**关键提示 / Key Tips**:
- 第一章的核心是熟悉游戏机制 / Chapter 1 focuses on learning game mechanics
- 尽量完成所有对话选项获得背景信息 / Exhaust all dialogue options for lore
- 不要跳过 Bernard 的训练 / Do not skip Bernard's training
- 收集地图上的路标点以解锁快速旅行 / Collect waypoints to unlock fast travel

### 第二章：政治漩涡 / Chapter 2: Political Intrigue

| 任务 / Quest | 推荐等级 / Level | 关键选择 / Key Choice |
|---|---|---|
| 领主之议 / Council of Lords | 5 | 支持哪一方 / Which faction to support |
| 失窃的文书 / Stolen Documents | 6 | 偷窃或谈判 / Steal or negotiate |
| 夜袭行动 / Night Raid | 7 | 潜行或强攻 / Stealth or assault |
| 背叛之刃 / Blade of Betrayal | 8 | 揭发或隐瞒 / Reveal or conceal |

**选择后果 / Choice Consequences**:
- 支持 Radzig 提升声望但可能得罪其他贵族
  Supporting Radzig increases reputation but may offend other nobles
- 偷窃文书可能导致后续任务对话选项减少
  Stealing documents may reduce dialogue options in later quests
- 夜袭中选择强攻会损失友军 NPC
  Choosing assault in the night raid loses allied NPCs
- 隐瞒背叛者信息会影响最终结局
  Concealing traitor information affects the final ending

### 第三章：王权之争 / Chapter 3: The Crown Struggle

| 任务 / Quest | 推荐等级 / Level | 关键选择 / Key Choice |
|---|---|---|
| 联盟之约 / Alliance Pact | 10 | 与哪国结盟 / Which kingdom to ally with |
| 攻城战 / Siege Battle | 12 | 指挥左翼或右翼 / Command left or right flank |
| 暗影中的交易 / Deal in Shadows | 14 | 信任或怀疑 / Trust or doubt |
| 最终决战 / Final Confrontation | 16 | 饶恕或处决 / Spare or execute |

### 结局分支 / Ending Branches

游戏有多个结局，取决于你在关键任务中的选择。

The game has multiple endings depending on your choices in crucial quests.

| 结局类型 / Ending Type | 触发条件 / Trigger Condition |
|---|---|
| 和平结局 / Peace Ending | 第三章中选择外交解决 / Choose diplomatic resolution in Chapter 3 |
| 复仇结局 / Vengeance Ending | 所有决斗中选择斩杀 / Choose execution in all duels |
| 统治结局 / Dominion Ending | 帮助特定贵族夺权 / Help a specific noble seize power |
| 隐退结局 / Retirement Ending | 完成所有支线后选择归隐 / Complete all side quests then retire |

## 重要支线任务 / Notable Side Quests

### Rattay 地区 / Rattay Region

**1. 好邻居 / Good Neighbor**

帮助 Rattay 居民解决纠纷。任务有多条解决路线。

Help Rattay residents resolve a dispute. Multiple resolution paths exist.

- 说服双方可获得双方奖励 / Persuade both sides for double rewards
- 偏袒一方会降低另一方声望 / Taking sides reduces reputation with the other
- 偷窃关键物品可快速完成 / Stealing a key item offers a quick solution

**2. 失踪的猎人 / The Missing Hunter**

追踪在森林中失踪的猎人。涉及追踪技能和战斗。

Track a hunter lost in the woods. Involves tracking skill and combat.

- 尽早出发，时间过长猎人可能死亡 / Leave early; the hunter may die if delayed
- 带上治疗药水以防万一 / Bring healing potions just in case
- 猎人身旁的宝箱需要中等开锁技能 / The chest near the hunter requires medium lockpicking

###  Talmberg 地区 / Talmberg Region

**3. 城堡之秘 / Castle Secret**

探索 Talmberg 城堡隐藏的地下通道。

Explore hidden passages beneath Talmberg Castle.

- 需要开锁技能 8+ / Requires lockpicking skill 8+
- 发现秘密房间可获得传奇装备 / Secret room contains legendary gear
- 被守卫发现会导致声望下降 / Getting caught by guards reduces reputation

**4. 炼金师的需求 / The Alchemist's Request**

为当地炼金师收集稀有材料。

Collect rare ingredients for a local alchemist.

- 部分材料在危险区域 / Some ingredients are in dangerous areas
- 推荐战斗等级 8+ 再尝试 / Recommended combat level 8+
- 完成后可学习稀有配方 / Rewards a rare recipe upon completion

## 任务通用技巧 / General Quest Tips

### 对话选择 / Dialogue Choices

| 对话类型 / Type | 效果 / Effect |
|---|---|
| 说服 / Persuade | 需要高口才 / Requires high Speech |
| 威胁 / Intimidate | 需要高力量 / Requires high Strength |
| 欺骗 / Lie | 可能被发现 / May be discovered |
| 支付 / Pay | 消耗格罗申 / Costs Groschen |
| 怜悯 / Mercy | 影响声望 / Affects reputation |

### 探索提示 / Exploration Tips

- 注意对话中的暗示和线索 / Pay attention to hints and clues in dialogue
- 任务日志提供详细目标和背景 / Quest journal provides detailed objectives and lore
- 与 NPC 对话时选择所有可选项 / Select all available options when talking to NPCs
- 支线任务有截止期限 / Some side quests have time limits
- 夜晚任务需要潜行装备 / Night quests require stealth gear

### 战斗任务准备 / Combat Quest Preparation

1. 提前保存游戏 / Save the game beforehand
2. 准备足够的治疗药水 / Prepare enough healing potions
3. 修理所有装备 / Repair all equipment
4. 检查任务区域的地形 / Check the terrain of the quest area
5. 准备对应敌人弱点的武器 / Prepare weapons matching enemy weaknesses

## 任务推荐顺序 / Recommended Quest Order

1. 完成 Rattay 所有基础支线（等级 2-5）
   Complete all basic side quests in Rattay (level 2-5)
2. 推进主线第二章至「领主之议」（等级 5-7）
   Advance main story Chapter 2 to "Council of Lords" (level 5-7)
3. 探索地图完成区域支线（等级 8-12）
   Explore the map completing regional side quests (level 8-12)
4. 推进主线第三章（等级 12-16）
   Advance main story Chapter 3 (level 12-16)
5. 最终章前完成所有剩余支线（等级 16-20）
   Complete all remaining side quests before the final chapter (level 16-20)
