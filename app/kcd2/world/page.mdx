# 世界探索 / World & Exploration

波希米亚的广阔世界充满了秘密、资源和危险。了解各个区域的特点和探索技巧将帮助你在这片中世纪土地上茁壮成长。

The vast world of Bohemia is filled with secrets, resources, and dangers. Understanding each region's characteristics and exploration techniques will help you thrive in this medieval land.

## 地图区域 / Map Regions

### 斯卡利茨地区 / Skalitz Region

游戏起始区域，剧情发生后大部分被毁。前期探索价值有限，但后期可以返回寻找隐藏物品。

Starting area, mostly destroyed after the prologue. Limited early-game value, but returnable later for hidden items.

| 地点 / Location | 类型 / Type | 推荐等级 / Level |
|---|---|---|
| 斯卡利茨废墟 / Skalitz Ruins | 剧情地点 / Story | 1-3 |
| 斯卡利茨周边林地 / Skalitz Woods | 采集区 / Gathering | 1-5 |
| 矿洞 / Abandoned Mine | 探索点 / Explore | 3-8 |

Core game region with the most complete facilities and quests.

**主要设施 / Key Facilities**:

- 市场 / Market: 商人集中区 / Merchant hub
- 铁匠铺 / Blacksmith: 购买和维修武器 / Buy and repair weapons
- 抄写员 / Scribe: 学习阅读技能 / Learn reading skill
- 澡堂 / Bathhouse: 清洁和恢复状态 / Clean and restore status

**周边村落 / Surrounding Villages**:

|---|---|---|
| 拉泰 / Ledetchko | 农业村落 / Farming | 旅馆、磨坊 / Inn, Mill |
| 萨莫普什 / Samopesh | 猎人村落 / Hunting | 猎人工坊 / Hunter's Workshop |
| 梅霍杰迪 / Merhojed | 畜牧村落 / Livestock | 马厩、牧场 / Stables, Pasture |
| 诺伊霍夫 / Neuhof | 贸易村落 / Trade | 市场、驿站 / Market, Waystation |

###  Talmberg 地区 / Talmberg Region

山地地形，拥有强大的城堡和丰富的矿藏。

Mountainous terrain with a powerful castle and rich mineral deposits.

**特色地点 / Notable Locations**:
- Talmberg 城堡: 易守难攻的堡垒 / An impenetrable fortress
- 采矿小镇 / Mining Town: 铁和银矿资源 / Iron and silver deposits
- 山间小径 / Mountain Passes: 危险但捷径 / Dangerous but shortcuts
- 猎人小屋 / Hunter's Cabins: 休息和补给点 / Rest and supply points

High-level areas on the map's edges with the strongest enemies and richest loot.

**警告 / Warning**: 推荐等级 12+ 再探索这些区域 / Recommended level 12+ before exploring these areas

| 东部山脉 / Eastern Mountains | 极高 / Very High | 传奇护甲 / Legendary armor |
| 南部沼泽 / Southern Swamp | 高 / High | 稀有炼金材料 / Rare alchemy ingredients |

## 快速旅行 / Fast Travel

### 如何使用 / How to Use

快速旅行需要通过发现路标点解锁。打开地图，点击已发现的路标点即可快速移动。

Fast travel is unlocked by discovering waypoints. Open the map and click discovered waypoints to fast travel.

**限制 / Limitations**:
- 负重过高时无法使用 / Cannot use when over-encumbered
- 战斗中无法使用 / Cannot use in combat
- 被通缉时部分路标点禁用 / Some waypoints disabled while wanted
- 快速旅行途中可能遭遇随机事件 / Random events may occur during fast travel

### 随机事件 / Random Events

快速旅行时有几率触发随机事件，包括：

Fast travel has a chance to trigger random events, including:

| 事件 / Event | 概率 / Chance | 应对 / Response |
|---|---|---|
| 强盗伏击 / Bandit Ambush | 30% | 战斗或逃跑 / Fight or flee |
| 旅行商人 / Traveling Merchant | 15% | 交易机会 / Trade opportunity |
| 路遇难民 / Refugees | 10% | 给予食物或金钱 / Give food or money |
| 动物袭击 / Animal Attack | 10% | 战斗 / Combat |
| 找到物品 / Found Item | 5% | 拾取 / Loot |

## 炼金术系统 / Alchemy System

### 炼金台使用 / Using the Alchemy Bench

炼金台由以下部分组成：研钵、蒸馏器、煮锅和药剂瓶架。

The alchemy bench consists of: mortar, alembic, cauldron, and phial rack.

**炼金步骤 / Alchemy Steps**:
1. 阅读配方或凭记忆制作 / Read recipe or brew from memory
2. 准备材料（新鲜度影响品质） / Prepare ingredients (freshness affects quality)
3. 处理材料（研磨、切割等） / Process ingredients (grind, cut, etc.)
4. 按顺序添加材料入锅 / Add ingredients to cauldron in order
5. 控制煮沸时间（沙漏计时） / Control boil time (hourglass timing)
6. 蒸馏装瓶 / Distill and bottle

### 全部配方 / Complete Recipes

| 药水 / Potion | 材料 / Ingredients | 效果 / Effect |
|---|---|---|
| 救星酒 / Saviour Schnapps | 荨麻 x1 + 缬草 x2 | 保存游戏 / Save game |
| 治疗剂 / Healing Potion | 荨麻 x1 + 聚合草 x1 | 治疗 30 HP / Heal 30 HP |
| 强效治疗 / Strong Healing | 荨麻 x2 + 聚合草 x2 | 治疗 60 HP / Heal 60 HP |
| 夜莺 / Night Hawk | 颠茄 x1 + 薄荷 x2 | 夜视 5 分钟 / Night vision 5 min |
| 射手酒 / Bowman's Brew | 蒲公英 x1 + 蓍草 x1 | 弓箭伤害 +50% for 5 min |
| 毒药 / Poison | 曼德拉草 x1 + 毒鹅膏 x1 | 武器涂毒 / Poison weapon |
| 镇定剂 / Lullaby Potion | 罂粟 x2 + 薄荷 x1 | 敌人沉睡 / Put enemy to sleep |
| 活力剂 / Energy Potion | 鼠尾草 x2 + 薄荷 x1 | 恢复精力 / Restore energy |

### 炼金等级提升 / Leveling Alchemy

- 每次成功制作获得经验 / Experience per successful brew
- 使用新鲜材料额外经验加成 / Fresh ingredients give bonus XP
- 阅读配方书解锁新配方 / Reading recipe books unlocks new recipes
- 制作高品质药水获得更多经验 / High-quality potions give more XP

## 开锁系统 / Lockpicking System

### 开锁机制 / How Lockpicking Works

开锁需要操作两个摇杆：一个控制扭矩扳手（压力），一个控制开锁器（旋转）。

Lockpicking requires operating two controls: one for torque (pressure) and one for the pick (rotation).

**难度因素 / Difficulty Factors**:
- 锁的等级决定难度 / Lock level determines difficulty
- 光线影响操作精度 / Light affects precision
- 醉酒状态降低开锁能力 / Drunk status reduces skill
- 开锁 Perk 可降低难度 / Lockpick perks reduce difficulty

### 开锁技巧 / Lockpicking Tips

1. 从简单锁开始练习 / Start with easy locks
2. 听到咔嗒声时保持位置 / Hold position when you hear a click
3. 使用「万用钥匙」Perk 降低难度 / Use "Skeleton Key" perk to reduce difficulty
4. 开锁前保存以防失败 / Save before attempting locks
5. 夜间开锁降低被发现的几率 / Pick locks at night for lower detection chance

## 声望系统 / Reputation System

### 声望影响 / How Reputation Works

每个区域和阵营有独立的声望值。声望影响商人价格、任务可用性和 NPC 态度。

Each region and faction has an independent reputation value. Reputation affects merchant prices, quest availability, and NPC attitudes.

| 声望等级 / Level | 效果 / Effect |
|---|---|
| 100+ / 爱戴 / Loved | 最优价格 / Best prices, 特殊对话 / Special dialogue |
| 75-99 / 尊敬 / Respected | 良好价格 / Good prices, 更多任务 / More quests |
| 50-74 / 中立 / Neutral | 标准价格 / Standard prices |
| 25-49 / 不喜 / Disliked | 价格上涨 / Prices increase, 部分任务关闭 / Some quests locked |
| 0-24 / 憎恨 / Hated | 价格最高 / Worst prices, 守卫攻击可能 / Guards may attack |

### 提升声望 / Increasing Reputation

- 完成任务 / Complete quests
- 捐赠金钱给教堂 / Donate money to churches
- 对 NPC 使用友善对话选项 / Use friendly dialogue options
- 击败区域内的强盗 / Defeat bandits in the area
- 参与城镇活动 / Participate in town events

### 降低声望 / Decreasing Reputation

- 偷盗被抓住 / Caught stealing
- 谋杀平民 / Murdering civilians
- 拒绝帮助 NPC / Refusing to help NPCs
- 违反当地法律 / Breaking local laws
- 穿着敌方阵营装备 / Wearing enemy faction gear

## 探索建议 / Exploration Advice

### 必备装备 / Essential Gear

1. 火把 / Torch: 夜间探索必需 / Essential for night exploration
2. 开锁器 / Lockpicks: 至少 10 个备用 / At least 10 spares
3. 治疗药水 / Healing Potions: 至少 3 瓶 / At least 3 potions
4. 食物 / Food: 足够 3 天的量 / Enough for 3 days
5. 睡袋 / Bedroll: 可在野外露营 / Allows camping in the wild

### 探索路线 / Exploration Routes

- 沿主路探索最安全 / Main roads are safest
- 森林小径隐藏着宝藏和危险 / Forest paths hide treasure and danger
- 河流可引导你到达主要城镇 / Rivers lead to major towns
- 高地俯瞰可发现兴趣点 / High ground vantage points reveal points of interest
