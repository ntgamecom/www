# 第二章 黄风岭 完整攻略

## 版本说明

适配最新正式版

## 路线流程

1. 从第一章结束的**黑风山渡口**出发，穿过山洞抵达**黄风岭入口**
2. 沿沙道前行，遇到第一个土地庙（**黄风岭·山口**），激活后继续前行
3. 穿过沙暴区域，抵达**沙门村**，清理村内鼠妖
4. 在沙门村深处触发精英怪**沙国王**，击败后获得【定风珠】碎片
5. 从沙门村后山进入**流沙河**区域，在河床寻找**沙大郎**BOSS
6. 击败沙大郎后通过地下洞穴进入**黄风寺**外围
7. 在黄风寺外围遇到**虎先锋·黄风版**（比第一章更强）
8. 击败虎先锋后进入**黄风寺正殿**，面对**黄风大王**
9. 使用定风珠削弱黄风大王的沙暴护盾后击败他
10. 从黄风寺后门进入**浮屠塔**，挑战隐藏BOSS**金蝉子**（可选）

## BOSS打法

### 沙国王

- **技能机制**：沙暴护体+远程沙刺，地面有流沙减速区域
- **核心弱点**：护体沙暴有持续时间，结束后进入虚弱
- **躲避时机**：沙刺出手时向侧方闪避，流沙区域用跳跃通过
- **连招建议**：等他沙暴结束 → 定身术 + 轻×3 + 蓄力重棍
- **逃课打法**：远程蓄力风筝，等他沙暴消失再打一轮

### 沙大郎

- **技能机制**：巨型蝎狮，尾刺有毒+钻地突袭
- **核心弱点**：钻地后露背时有长硬直
- **躲避时机**：钻地时不断移动，出现瞬间向侧方闪避
- **连招建议**：闪避钻地突袭 → 轻×5 + 重击 → 广智变身追击
- **逃课打法**：站在高台上等他钻地后撞墙硬直，跳下去打一套

### 虎先锋（黄风岭版）

- **技能机制**：比第一章虎先锋多一个分身技能，召唤两个幻影同时攻击
- **核心弱点**：分身存在时真身会蓄力大招，击破分身可打断
- **躲避时机**：分身出现时优先闪避并清除分身
- **连招建议**：清分身 → 定身术 + 满蓄力重棍 → 拉开
- **逃课打法**：边跑边打分身，等真身大招被中断后冲上去打一套

### 黄风大王

- **技能机制**：超大范围沙暴+沙尘龙卷+召唤小妖
- **核心弱点**：使用**定风珠**可以驱散沙暴护盾，使其进入倒地状态
- **第一阶段**：沙暴护盾+远程沙刃，此时攻击无效
- **第二阶段**：护盾被定风珠驱散后，近战连击+龙卷风
- **连招建议**：
  - 开局直接使用定风珠 → 冲上去轻×5 + 重击
  - 他起身后闪避龙卷风 → 等定风珠CD → 再次使用定风珠
  - 重复以上流程
- **逃课打法**：保持距离用定风珠+蓄力重棍循环，尽量不近战

### 金蝉子（隐藏BOSS）

- **触发条件**：在浮屠塔找到五颗佛珠后，在塔顶大殿触发
- **技能机制**：佛法金光+瞬移突进+禁锢咒
- **躲避时机**：金光蓄力时向侧方连续闪避
- **连招建议**：闪避金光后 → 轻×4 + 重击 → 禁锢咒时使用定身术解控
- **逃课打法**：利用场景柱子卡位，蓄力重棍偷刀

## 避坑要点

- 定风珠是本章最关键道具！打黄风大王前务必集齐三个碎片合成
- 沙门村的鼠妖群不要硬冲，用广智变身的火焰AOE清群效率最高
- 流沙河的毒池区域右侧有隐藏通道通往宝箱
- 虎先锋分身阶段不要贪刀，必须先清分身
- 黄风大王的沙暴阶段攻击无效，别浪费法力
- 浮屠塔需要跳跃技巧，掉下去要从底层重新爬
- 金蝉子战败后传送回浮屠塔入口，不是死亡
