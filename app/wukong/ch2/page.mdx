# 第二章：黄风岭 / Chapter 2: Yellow Wind Ridge

## 版本说明 / Version Info

适配最新正式版 / Compatible with the latest release version

## 路线流程 / Route Walkthrough

1. 从第一章结束的**黑风山渡口**出发，穿过山洞抵达**黄风岭入口** / From the Chapter 1 dock at **Black Wind Mountain**, go through the cave to reach **Yellow Wind Ridge Entrance**
2. 沿沙道前行，遇到第一个土地庙（**黄风岭·山口**），激活后继续前行 / Follow the sandy path to the first shrine (**Yellow Wind Ridge Pass**), activate and continue
3. 穿过沙暴区域，抵达**沙门村**，清理村内鼠妖 / Cross the sandstorm area to reach **Sand Gate Village (Shamencun)**, clear the rat yaoguai
4. 在沙门村深处触发精英怪**沙国王**，击败后获得**定风珠**碎片 / Deeper in the village, trigger elite **Sand King**, defeat for a **Wind Tamer** fragment
5. 从沙门村后山进入**流沙河**区域，在河床寻找**沙大郎**BOSS / From the back hill, enter **Quicksand River** area, search the riverbed for **Sand Lang** boss
6. 击败沙大郎后通过地下洞穴进入**黄风寺**外围 / Defeat Sand Lang, go through the underground cave to reach **Yellow Wind Temple** outskirts
7. 在黄风寺外围遇到**虎先锋·黄风版**（比第一章更强） / Outside the temple, encounter **Tiger Vanguard (Yellow Wind version)** — stronger than Chapter 1
8. 击败虎先锋后进入**黄风寺正殿**，面对**黄风大王** / Defeat Tiger Vanguard and enter the **main hall** to face **Yellow Wind Sage**
9. 使用定风珠削弱黄风大王的沙暴护盾后击败他 / Use the Wind Tamer to weaken his sandstorm shield, then defeat him
10. 从黄风寺后门进入**浮屠塔**，挑战隐藏BOSS**金蝉子**（可选） / Enter **Floating Pagoda** from the back gate to challenge hidden boss **Golden Cicada** (optional)

## BOSS打法 / Boss Strategies

### 沙国王 / Sand King

- **技能机制 / Mechanics**: 沙暴护体+远程沙刺，地面有流沙减速区域 / Sandstorm shield + ranged sand spikes, ground has quicksand slow zones
- **核心弱点 / Weakness**: 护体沙暴有持续时间，结束后进入虚弱 / Sandstorm shield has a duration, he becomes vulnerable when it expires
- **躲避时机 / Dodge Timing**: 沙刺出手时向侧方闪避，流沙区域用跳跃通过 / Dodge sideways when sand spikes fire, jump over quicksand
- **连招建议 / Combo**: 等他沙暴结束 → 定身术 + 轻×3 + 蓄力重棍 / Wait for sandstorm to end → Immobilize + light ×3 + charged heavy
- **逃课打法 / Cheese**: 远程蓄力风筝，等他沙暴消失再打一轮 / Ranged charged staff, kite until his sandstorm dissipates

### 沙大郎 / Sand Lang

- **技能机制 / Mechanics**: 巨型蝎狮，尾刺有毒+钻地突袭 / Giant manticore, poisonous tail stab + burrow ambush
- **核心弱点 / Weakness**: 钻地后露背时有长硬直 / Long recovery when surfacing after burrowing
- **躲避时机 / Dodge Timing**: 钻地时不断移动，出现瞬间向侧方闪避 / Keep moving while he burrows, dodge sideways the moment he surfaces
- **连招建议 / Combo**: 闪避钻地突袭 → 轻×5 + 重击 → 广智变身追击 / Dodge burrow ambush → light ×5 + heavy → Guangzhi transformation pursuit
- **逃课打法 / Cheese**: 站在高台上等他钻地后撞墙硬直，跳下去打一套 / Stand on a high platform, jump down and combo when he hits the wall

### 虎先锋（黄风岭版）/ Tiger Vanguard (Yellow Wind Version)

- **技能机制 / Mechanics**: 比第一章虎先锋多一个分身技能，召唤两个幻影同时攻击 / Gains a clone skill, summons two phantoms to attack simultaneously
- **核心弱点 / Weakness**: 分身存在时真身会蓄力大招，击破分身可打断 / Real body charges a big move while clones exist — destroying clones interrupts it
- **躲避时机 / Dodge Timing**: 分身出现时优先闪避并清除分身 / Prioritize dodging and clearing clones when they appear
- **连招建议 / Combo**: 清分身 → 定身术 + 满蓄力重棍 → 拉开 / Clear clones → Immobilize + fully charged heavy → retreat
- **逃课打法 / Cheese**: 边跑边打分身，等真身大招被中断后冲上去打一套 / Hit and run on clones, rush in when the real one's big move is interrupted

### 黄风大王 / Yellow Wind Sage

- **技能机制 / Mechanics**: 超大范围沙暴+沙尘龙卷+召唤小妖 / Massive sandstorm + sand tornadoes + summon minions
- **核心弱点 / Weakness**: 使用**定风珠**可以驱散沙暴护盾，使其进入倒地状态 / Using the **Wind Tamer** dispels his sandstorm shield, putting him in a downed state
- **第一阶段 / Phase 1**: 沙暴护盾+远程沙刃，此时攻击无效 / Sandstorm shield + ranged sand blades, attacks are ineffective
- **第二阶段 / Phase 2**: 护盾被定风珠驱散后，近战连击+龙卷风 / After shield is dispelled, melee combos + tornadoes
- **连招建议 / Combo**:
  - 开局直接使用定风珠 → 冲上去轻×5 + 重击 / Open with Wind Tamer → rush light ×5 + heavy
  - 他起身后闪避龙卷风 → 等定风珠CD → 再次使用定风珠 / Dodge tornadoes → wait for Wind Tamer cooldown → use again
  - 重复以上流程 / Repeat
- **逃课打法 / Cheese**: 保持距离用定风珠+蓄力重棍循环，尽量不近战 / Keep distance, cycle Wind Tamer + charged heavy, avoid melee

### 金蝉子（隐藏BOSS）/ Golden Cicada (Hidden Boss)

- **触发条件 / Trigger**: 在浮屠塔找到五颗佛珠后，在塔顶大殿触发 / Find five Prayer Beads in Floating Pagoda, activate at the top floor hall
- **技能机制 / Mechanics**: 佛法金光+瞬移突进+禁锢咒 / Buddhist golden light + teleport strike + binding curse
- **躲避时机 / Dodge Timing**: 金光蓄力时向侧方连续闪避 / Continuous sideways dodges during golden light charge
- **连招建议 / Combo**: 闪避金光后 → 轻×4 + 重击 → 禁锢咒时使用定身术解控 / Dodge golden light → light ×4 + heavy → use Immobilize to break binding curse
- **逃课打法 / Cheese**: 利用场景柱子卡位，蓄力重棍偷刀 / Use pillars to block line of sight, charged heavy sneak attacks

## 避坑要点 / Tips & Pitfalls

- **定风珠是本章最关键道具**！打黄风大王前务必集齐三个碎片合成 / **The Wind Tamer is THE key item** — collect all three fragments and synthesize before fighting Yellow Wind Sage
- **沙门村的鼠妖群不要硬冲**，用广智变身的火焰AOE清群效率最高 / **Don't charge into the rat horde** — use Guangzhi's fire AoE for maximum clear efficiency
- **流沙河的毒池区域右侧有隐藏通道**通往宝箱 / **Right side of the poison area** has a hidden passage to a chest
- **虎先锋分身阶段不要贪刀**，必须先清分身 / **During clone phase**, don't get greedy — clear clones first
- **黄风大王的沙暴阶段攻击无效**，别浪费法力 / **Sandstorm phase makes him immune**, don't waste mana
- **浮屠塔需要跳跃技巧**，掉下去要从底层重新爬 / **Floating Pagoda requires jumping skill** — fall and you'll climb from the bottom
- **金蝉子战败后传送回浮屠塔入口**，不是死亡 / **Defeat teleports you to the entrance**, it's not death
