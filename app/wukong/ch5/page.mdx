# 第五章 火焰山 完整攻略

## 版本说明

适配最新正式版

## 路线流程

1. 从盘丝洞渡口乘船抵达**火焰山·焦土地庙**
2. 穿过焦土平原，清理火焰小妖，进入**烈焰峡谷**
3. 在峡谷中段遭遇精英怪**焰魔**，击败后获得火焰抗性
4. 穿过峡谷到达**牛魔王城堡**外围，遭遇**红孩儿**
5. 击败红孩儿后进入城堡正门，面对**牛魔王**第一阶段
6. 牛魔王败退后追入城堡深处**熔岩大厅**
7. 在熔岩大厅击败牛魔王第二阶段
8. 从城堡后门进入**火云洞**，面对最终BOSS**夜叉王**
9. 击败夜叉王后在火云洞最深处发现隐藏BOSS**铁扇公主**（可选）

## BOSS打法

### 焰魔

- **技能机制**：火焰护体+岩浆喷发+火球连射
- **核心弱点**：火焰护体有能量条，耗尽后熄灭
- **躲避时机**：岩浆喷发前地面变红，向外跑
- **连招建议**：等他护体熄灭 → 定身术 + 轻×5 + 重击 → 拉开
- **逃课打法**：利用场景水潭灭火，站在水里打可以减少火焰伤害

### 红孩儿

- **技能机制**：火焰三尖枪+火尖枪连刺+召唤火凤
- **核心弱点**：火凤可以被定身术定住
- **躲避时机**：三尖枪连刺最后一下范围最大，向后闪
- **连招建议**：
  - 定身术定住火凤 → 打红孩儿本体
  - 火尖枪连刺闪避后 → 轻×4 + 重击
  - 他召唤火凤时是输出窗口
- **逃课打法**：先让火凤撞墙消失再打本体

### 牛魔王（第一阶段·城堡外）

- **技能机制**：巨斧横扫+践踏冲击+召唤牛魔小兵
- **核心弱点**：践踏冲击前有抬腿动作
- **躲避时机**：看到他抬腿立刻向侧方闪避
- **连招建议**：
  - 闪避践踏 → 冲到脚边输出2秒 → 拉开
  - 召唤小兵时用广智变身清场
  - 巨斧横扫往他身后闪
- **逃课打法**：保持在中距离引他出践踏，利用后摇输出

### 牛魔王（第二阶段·熔岩大厅）

- **技能机制**：巨大化+熔岩AOE+冲撞攻击
- **核心弱点**：冲撞会撞到墙壁，眩晕数秒
- **躲避时机**：冲撞时向侧方跑，引他撞墙
- **连招建议**：
  - 引他撞墙 → 眩晕时冲过去满蓄力重棍 + 轻×5
  - 熔岩AOE时站在他背后
  - 重复撞墙→输出循环
- **逃课打法**：利用场景熔岩池旁边的安全区域卡位输出

### 夜叉王（最终BOSS）

- **技能机制**：三头六臂形态+火焰刀+黑暗领域
- **核心弱点**：黑暗领域中可以看他的眼睛红光判断位置
- **第一形态**：三头六臂+火焰刀乱舞
  - 火焰刀范围极大，保持在他背后输出
  - 六臂攻击有节奏，闪避第一下后反向闪避第二下
- **第二形态**：黑暗领域+瞬移突袭
  - 黑暗领域时不断移动
  - 看到红光（他眼睛）立刻闪避
  - 等黑暗散去后再反击
- **连招建议**：
  - 第一形态：绕背轻×4 → 重击 → 拉开
  - 第二形态：黑暗中闪避突袭 → 黑暗散去瞬间定身术 + 满蓄力
- **逃课打法**：黑暗领域时贴墙跑，减少被突袭的方向

### 铁扇公主（隐藏BOSS）

- **触发条件**：火云洞最深处使用芭蕉扇（在城堡宝箱中找到）
- **技能机制**：芭蕉扇风暴+乾坤圈远程攻击
- **核心弱点**：芭蕉扇风暴释放期间她是无敌的
- **躲避时机**：风暴出现时躲到石柱后面
- **连招建议**：
  - 躲风暴 → 等她收扇 → 定身术 + 输出
  - 乾坤圈有轨迹，侧闪即可
- **逃课打法**：全程用石柱挡住风暴和乾坤圈

## 避坑要点

- 火焰山的熔岩地面会持续掉血，看地面颜色判断是否安全
- 焰魔给的火焰抗性非常重要，不打焰魔直接往后走会很痛苦
- 红孩儿的火凤定身后只有很短输出窗口，提前蓄力
- 牛魔王第二阶段引他撞墙是最重要的输出机会
- 夜叉王的黑暗领域不要乱滚，看红光闪避
- 芭蕉扇在城堡二楼，不打铁扇公主也建议拿（后面有用）
- 熔岩大厅有一个隐藏存档点，在右侧熔岩瀑布后面
