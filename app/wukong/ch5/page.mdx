# 第五章：火焰山 / Chapter 5: Flaming Mountain

## 版本说明 / Version Info

适配最新正式版 / Compatible with the latest release version

## 路线流程 / Route Walkthrough

1. 从盘丝洞渡口乘船抵达**火焰山·焦土地庙** / Take the boat from Spider Cave dock to **Flaming Mountain · Scorched Earth Shrine**
2. 穿过焦土平原，清理火焰小妖，进入**烈焰峡谷** / Cross the scorched plain, clear fire imps, enter **Blazing Canyon**
3. 在峡谷中段遭遇精英怪**焰魔**，击败后获得火焰抗性 / Midway through the canyon, encounter elite **Flame Demon**, defeat for fire resistance
4. 穿过峡谷到达**牛魔王城堡**外围，遭遇**红孩儿** / Cross the canyon to **Bull King's Castle** outskirts, encounter **Red Boy**
5. 击败红孩儿后进入城堡正门，面对**牛魔王**第一阶段 / Defeat Red Boy and enter the castle to face **Bull King** Phase 1
6. 牛魔王败退后追入城堡深处**熔岩大厅** / After Bull King retreats, chase him into the **Lava Hall**
7. 在熔岩大厅击败牛魔王第二阶段 / Defeat Bull King Phase 2 in the Lava Hall
8. 从城堡后门进入**火云洞**，面对最终BOSS**夜叉王** / Enter **Fire Cloud Cave** through the castle's back gate, face the final boss **Yaksha King**
9. 击败夜叉王后在火云洞最深处发现隐藏BOSS**铁扇公主**（可选）/ After defeating Yaksha King, discover hidden boss **Princess Iron Fan** (optional)

## BOSS打法 / Boss Strategies

### 焰魔 / Flame Demon

- **技能机制 / Mechanics**: 火焰护体+岩浆喷发+火球连射 / Fire shield + magma eruption + fireball volley
- **核心弱点 / Weakness**: 火焰护体有能量条，耗尽后熄灭 / Fire shield has a stamina bar, extinguishes when depleted
- **躲避时机 / Dodge Timing**: 岩浆喷发前地面变红，向外跑 / Ground turns red before magma eruption, run outward
- **连招建议 / Combo**: 等他护体熄灭 → 定身术 + 轻×5 + 重击 → 拉开 / Wait for shield to die → Immobilize + light ×5 + heavy → retreat
- **逃课打法 / Cheese**: 利用场景水潭灭火，站在水里打可以减少火焰伤害 / Use water pools to extinguish fire, fight from water for reduced fire damage

### 红孩儿 / Red Boy

- **技能机制 / Mechanics**: 火焰三尖枪+火尖枪连刺+召唤火凤 / Flame trident + fiery spear thrusts + summon fire phoenix
- **核心弱点 / Weakness**: 火凤可以被定身术定住 / The fire phoenix can be immobilized
- **躲避时机 / Dodge Timing**: 三尖枪连刺最后一下范围最大，向后闪 / Last spear thrust has widest range, dodge backward
- **连招建议 / Combo**:
  - 定身术定住火凤 → 打红孩儿本体 / Immobilize the phoenix → attack Red Boy
  - 火尖枪连刺闪避后 → 轻×4 + 重击 / Dodge spear thrusts → light ×4 + heavy
  - 他召唤火凤时是输出窗口 / His phoenix summon is an attack window
- **逃课打法 / Cheese**: 先让火凤撞墙消失再打本体 / Let the phoenix crash into a wall first

### 牛魔王（第一阶段·城堡外）/ Bull King (Phase 1 · Outside Castle)

- **技能机制 / Mechanics**: 巨斧横扫+践踏冲击+召唤牛魔小兵 / Giant axe sweep + stomp shockwave + summon bull minions
- **核心弱点 / Weakness**: 践踏冲击前有抬腿动作 / Leg-raising tell before stomp shockwave
- **躲避时机 / Dodge Timing**: 看到他抬腿立刻向侧方闪避 / Dodge sideways the moment he raises his leg
- **连招建议 / Combo**:
  - 闪避践踏 → 冲到脚边输出2秒 → 拉开 / Dodge stomp → rush to his feet for 2 seconds → retreat
  - 召唤小兵时用广智变身清场 / Clear minions with Guangzhi transformation
  - 巨斧横扫往他身后闪 / Dodge axe sweep by rolling behind him
- **逃课打法 / Cheese**: 保持在中距离引他出践踏，利用后摇输出 / Keep mid-range to bait his stomp, punish the recovery

### 牛魔王（第二阶段·熔岩大厅）/ Bull King (Phase 2 · Lava Hall)

- **技能机制 / Mechanics**: 巨大化+熔岩AOE+冲撞攻击 / Giant form + lava AoE + charge attack
- **核心弱点 / Weakness**: 冲撞会撞到墙壁，眩晕数秒 / Charge crashes into walls, stunning him for seconds
- **躲避时机 / Dodge Timing**: 冲撞时向侧方跑，引他撞墙 / Run sideways when he charges, bait him into the wall
- **连招建议 / Combo**:
  - 引他撞墙 → 眩晕时冲过去满蓄力重棍 + 轻×5 / Bait wall charge → during stun, fully charged heavy + light ×5
  - 熔岩AOE时站在他背后 / Stand behind him during lava AoE
  - 重复撞墙→输出循环 / Repeat wall charge → damage cycle
- **逃课打法 / Cheese**: 利用场景熔岩池旁边的安全区域卡位输出 / Use safe zones near lava pools

### 夜叉王（最终BOSS）/ Yaksha King (Final Boss)

- **技能机制 / Mechanics**: 三头六臂形态+火焰刀+黑暗领域 / Three-headed six-armed form + flame blade + dark realm
- **核心弱点 / Weakness**: 黑暗领域中可以看他的眼睛红光判断位置 / In dark realm, watch for the red glow of his eyes
- **第一形态 / Form 1**: 三头六臂+火焰刀乱舞 / Three heads six arms + flame blade flurry
  - 火焰刀范围极大，保持在他背后输出 / Flame blade has huge range, stay behind him
  - 六臂攻击有节奏，闪避第一下后反向闪避第二下 / Six-arm attacks have a rhythm
- **第二形态 / Form 2**: 黑暗领域+瞬移突袭 / Dark realm + teleport ambush
  - 黑暗领域时不断移动 / Keep moving in the dark realm
  - 看到红光（他眼睛）立刻闪避 / Dodge on seeing red glow (his eyes)
  - 等黑暗散去后再反击 / Counterattack after the dark dissipates
- **连招建议 / Combo**:
  - 第一形态：绕背轻×4 → 重击 → 拉开 / Circle behind light ×4 → heavy → retreat
  - 第二形态：黑暗中闪避突袭 → 黑暗散去瞬间定身术 + 满蓄力 / Dodge ambush in darkness → Immobilize + full charge when dark dissipates
- **逃课打法 / Cheese**: 黑暗领域时贴墙跑，减少被突袭的方向 / Hug the wall during dark realm to reduce ambush angles

### 铁扇公主（隐藏BOSS）/ Princess Iron Fan (Hidden Boss)

- **触发条件 / Trigger**: 火云洞最深处使用芭蕉扇（在城堡宝箱中找到）/ Use the Plantain Fan (found in castle chest) at the deepest part of Fire Cloud Cave
- **技能机制 / Mechanics**: 芭蕉扇风暴+乾坤圈远程攻击 / Plantain fan storm + Universe Ring ranged attacks
- **核心弱点 / Weakness**: 芭蕉扇风暴释放期间她是无敌的 / She's invulnerable during the fan storm
- **躲避时机 / Dodge Timing**: 风暴出现时躲到石柱后面 / Hide behind stone pillars when the storm appears
- **连招建议 / Combo**:
  - 躲风暴 → 等她收扇 → 定身术 + 输出 / Hide from storm → wait for her to retract fan → Immobilize + damage
  - 乾坤圈有轨迹，侧闪即可 / Universe Ring has a trajectory, side dodge
- **逃课打法 / Cheese**: 全程用石柱挡住风暴和乾坤圈 / Use stone pillars to block both storm and rings

## 避坑要点 / Tips & Pitfalls

- **火焰山的熔岩地面会持续掉血**，看地面颜色判断是否安全 / **Lava ground deals continuous damage** — watch ground color for safe zones
- **焰魔给的火焰抗性非常重要**，不打焰魔直接往后走会很痛苦 / **Flame Demon's fire resistance is crucial** — skipping him makes the rest painful
- **红孩儿的火凤定身后只有很短输出窗口**，提前蓄力 / **Phoenix has a short window after Immobilize**, charge up in advance
- **牛魔王第二阶段引他撞墙是最重要的输出机会** / **Baiting wall charges is THE key damage opportunity** in Phase 2
- **夜叉王的黑暗领域不要乱滚**，看红光闪避 / **Don't panic roll in dark realm** — watch for the red glow
- **芭蕉扇在城堡二楼**，不打铁扇公主也建议拿（后面有用） / **Grab the Plantain Fan** even if you skip Princess Iron Fan
- **熔岩大厅有一个隐藏存档点**，在右侧熔岩瀑布后面 / **Hidden checkpoint** in Lava Hall behind the right lava waterfall
