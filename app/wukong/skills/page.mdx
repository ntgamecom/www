## 技能系统概述 / Skill System Overview

**中文：** 黑神话悟空的技能系统分为六大类：棍法（主动攻击技能）、法术（魔法技能）、身法（被动/移动技能）、属性（基础属性增强）、变身和毫毛。技能点通过升级获得，每升一级获得1点。在土地庙可以洗点重置所有技能。

**English:** Black Myth: Wukong's skill system has six categories: Stances (active attack skills), Spells (magic), Marital Arts (passive/movement), Stats (basic attribute enhancement), Transformations, and Strands. Each level grants 1 skill point. Respeccing is available at shrines.

| 技能类别 / Category | 说明 / Description |
|---|---|
| 棍法 / Stances | 三种棍法的连段和重击强化 / Combo and heavy attack upgrades for 3 stances |
| 法术 / Spells | 主动魔法技能 / Active magic abilities |
| 身法 / Marital Arts | 闪避、格挡等被动技能 / Dodge, block passives |
| 属性 / Stats | 生命、法力、体力、攻击力等基础属性 / HP, mana, stamina, attack |
| 变身 / Transformations | 解锁和强化变身能力 / Unlock and enhance transformations |
| 毫毛 / Strands | 加强毫毛召唤技能 / Enhance strand summoning skill |

## 属性加点推荐 / Stat Allocation

**中文：** 属性加点可以提升基础能力。建议优先加满核心属性。

**English:** Stat upgrades improve baseline abilities. Prioritize maxing core stats.

| 属性 / Stat | 效果 / Effect | 推荐优先级 / Priority |
|---|---|---|
| 生命 / HP | 每点+20最大生命 / +20 HP per point | ★★★★ |
| 法力 / Mana | 每点+15最大法力 / +15 Mana per point | ★★★★★ |
| 体力 / Stamina | 每点+10最大体力 / +10 Stamina per point | ★★★★ |
| 攻击 / Attack | 每点+3攻击力 / +3 Attack per point | ★★★★★ |
| 防御 / Defense | 每点+2防御力 / +2 Defense per point | ★★★ |

**加点策略 / Leveling Strategy**:
- 前期优先：法力 > 攻击 > 生命 > 体力 > 防御 / Early priority: Mana > Atk > HP > Stamina > Def
- 中期均衡： 各项2：1均衡加点 / Mid: 2:1 ratio across core stats
- 后期：主加攻击和法力，其他根据需要 / Late: focus Atk + Mana, others situational

## 技能重置 / Skill Reset

**中文：** 在土地庙可以随时重置所有技能点，消耗少量材料。重置后所有技能点返还，可以重新分配。

**English:** Respec all skill points at any shrine for a small material cost. All points are refunded for reallocation.

- 重置消耗：1000灵蕴（游戏内货币）/ Cost: 1000 Will (in-game currency)
- 重置次数：无限次 / Unlimited
- 注意：重置不会影响属性上限解锁，只会返还技能点 / Note: reset only refunds skill points, not stat caps
- 建议：每章开始时重置一次，根据章节获得的道具和新法术调整配点 / Recommended: respec at each new chapter to adjust for new gear and spells
