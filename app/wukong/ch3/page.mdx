# 第三章 新西天 完整攻略

## 版本说明

适配最新正式版

## 路线流程

1. 从黄风岭渡口乘船抵达**新西天·苦海岸**
2. 沿冰海岸边前行，清理雪地狼妖，抵达**苦海北岸土地庙**
3. 穿过冰封湖面到达**雷音寺**外围，遭遇精英怪**不空**
4. 击败不空后进入**雷音寺广场**，触发**黄眉大王**第一阶段战斗
5. 黄眉大王败退后追入**雷音寺大雄宝殿**，进行第二阶段战斗
6. 击败黄眉大王后从后山前往**雪山之巅**
7. 在雪山之巅遇到**青背龙**，可选BOSS
8. 沿雪山栈道下行抵达**浮屠界**，清理狱卒和囚魂
9. 在浮屠界最深处触发**不白**BOSS战
10. 击败不白后通过传送门回到雷音寺后方，面对最终BOSS**金铙主持**

## BOSS打法

### 不空

- **技能机制**：冰系法术+禅杖近战，能冻结天命人
- **核心弱点**：施法期间有较长前摇
- **躲避时机**：冰霜圈出现时立刻闪避出范围
- **连招建议**：闪避冰霜圈 → 冲脸轻×4 + 蓄力重棍 → 被冻结时切广智变身解控
- **逃课打法**：利用地形高低差，远程蓄力等他施法硬直

### 黄眉大王（第一阶段·广场）

- **技能机制**：金铙音波攻击+召唤金铙分身
- **核心弱点**：金铙分身每次出现的位置固定
- **躲避时机**：金铙音波有扩散轨迹，向两侧闪避
- **连招建议**：清掉金铙分身 → 定身术 + 轻×5 + 重击 → 拉开
- **逃课打法**：先清分身再打本体，不要同时面对多个目标

### 黄眉大王（第二阶段·大雄宝殿）

- **技能机制**：巨大化金身+范围冲击波+召唤佛陀幻影
- **核心弱点**：冲击波落地前有金色光圈提示
- **躲避时机**：金色光圈收缩到最大时闪避（完美时机）
- **连招建议**：
  - 闪避冲击波 → 冲到他脚下输出2秒 → 拉开
  - 召唤佛陀时优先用定身术打断
  - 金身状态时不要硬打，等他变回正常大小
- **逃课打法**：全程保持中距离，等他出完冲击波再靠近打一轮就跑

### 青背龙（可选BOSS）

- **技能机制**：飞行+冰霜吐息+落冰攻击
- **核心弱点**：落地后短暂硬直
- **躲避时机**：冰霜吐息向侧方跑，不要后退
- **连招建议**：他落地瞬间 → 定身术 + 满蓄力重棍 + 广智变身追击
- **逃课打法**：等他自己落地，不打飞行阶段

### 不白

- **技能机制**：巨大白骨形态+骷髅召唤+骨刺AOE
- **核心弱点**：召唤骷髅时本体不动
- **躲避时机**：骨刺蓄力时闪避三次（连续）
- **连招建议**：
  - 清骷髅 → 冲脸输出3秒 → 拉开
  - 骨刺AOE时连续侧闪
  - 残血时白骨形态变大，此时攻击他身后
- **逃课打法**：保持距离让骷髅聚集，用广智变身火焰AOE清场

### 金铙主持（最终BOSS）

- **技能机制**：金铙远程攻击+近身禅杖+音波陷阱
- **核心弱点**：音波陷阱踩到后会被定在原地
- **第一形态**：远程为主，金铙飞出攻击
- **第二形态**：近战为主，禅杖连击+音波陷阱
- **连招建议**：
  - 第一形态：闪避金铙 → 趁他收回时靠近输出
  - 第二形态：注意地面音波陷阱，绕到他背后输出
  - 陷阱被定住时立刻切广智变身解控
- **逃课打法**：贴着墙壁打可以减少音波陷阱的威胁范围

## 避坑要点

- 苦海岸的冰面可以打破，下方有隐藏洞穴和宝箱
- 雷音寺广场的铜钟可以敲响，会暂时眩晕周围小怪
- 黄眉大王第二阶段不要站在正前方，冲击波正面范围最大
- 青背龙不打不影响主线，但鳞片能造非常好的冰属性武器
- 浮屠界的囚魂可以不打，但击败后给的灵蕴很多
- 雪山栈道有几处冰面滑脚，走的时候按住慢走键
- 金铙主持的音波陷阱在低血量时密度翻倍，注意脚下
