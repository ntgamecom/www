# 第四章 盘丝洞 完整攻略

## 版本说明

适配最新正式版

## 路线流程

1. 从新西天渡口乘船进入**盘丝洞·洞窟入口**
2. 穿过蛛网密布的洞穴，到达**盘丝洞前厅土地庙**
3. 清理前厅蜘蛛卵和小蜘蛛群，遭遇精英怪**黑手道人**
4. 击败黑手道人后获得【蜘蛛令】，可打开后续机关门
5. 深入洞窟抵达**蜘蛛巢穴**，触发**紫蛛儿**BOSS战
6. 击败紫蛛儿后从巢穴后方的竖井垂降到底层
7. 底层为**毒液池**区域，小心毒水，沿墙壁走
8. 穿过毒液池到达**盘丝洞正殿**，面对**百眼魔君**
9. 击败百眼魔君后从正殿后门进入**祭祀坛**
10. 在祭祀坛触发隐藏BOSS**毒王**（可选，需持毒王令）

## BOSS打法

### 黑手道人

- **技能机制**：多臂攻击+暗器投掷+烟雾隐身
- **核心弱点**：烟雾隐身时移动有足迹轨迹
- **躲避时机**：暗器投掷前有收手动作
- **连招建议**：
  - 隐身时根据足迹预判位置 → 蓄力重棍等他现身
  - 现身瞬间打出满蓄力
  - 他多臂乱舞时拉开距离
- **逃课打法**：贴墙站减少烟雾隐身时的追击范围

### 紫蛛儿

- **技能机制**：蛛丝缠绕+产卵召唤小蜘蛛+毒液喷射
- **核心弱点**：产卵时本体不动
- **躲避时机**：蛛丝出手时向侧方闪避
- **连招建议**：
  - 起手定身术打断蛛丝 → 轻×5 + 重击
  - 产卵时优先清卵，否则小蜘蛛会累积
  - 毒液喷射前后摇长，绕侧输出
- **逃课打法**：绕圈跑等小蜘蛛聚集 → 广智变身火焰AOE清场 → 打本体

### 百眼魔君

- **技能机制**：多眼激光+石化凝视+召唤毒蛛
- **核心弱点**：看他的眼睛会被石化，需要背对或使用【避邪符】
- **第一阶段**：激光扫射+石化凝视
  - 不要直视他的眼睛，用侧移保持他在画面边缘
  - 激光有蓄力轨迹，向垂直方向跑
- **第二阶段**：召唤毒蛛+全屏毒雨
  - 毒雨有地面印记，在印记出现后闪避
  - 优先清理毒蛛，避免被围攻
- **连招建议**：
  - 第一阶段：侧移避开凝视 → 冲到他侧面轻×4 + 重击 → 拉开
  - 第二阶段：清毒蛛 → 等他放完毒雨 → 定身术 + 满蓄力
- **逃课打法**：保持中远距离，等他放完激光再靠近打

### 毒王（隐藏BOSS）

- **触发条件**：持毒王令进入祭祀坛深处
- **技能机制**：毒雾领域+毒爪连击+剧毒炸弹
- **核心弱点**：毒雾领域有范围限制，出范围可解毒
- **躲避时机**：毒爪连击最后一下是重击，闪避后有大硬直
- **连招建议**：
  - 毒雾时退出范围 → 等他出来 → 轻×3 + 重击
  - 毒爪连击闪避最后一下 → 定身术 + 蓄力重棍
- **逃课打法**：全程保持毒雾范围外，用远程蓄力磨血

## 避坑要点

- 盘丝洞的蜘蛛卵见到就要打掉，否则走过后会孵化出小蜘蛛追你
- 黑手道人的烟雾隐身阶段不要乱跑，看足迹轨迹
- 紫蛛儿的蛛丝缠绕中了之后可以用广智变身的火焰烧断
- 百眼魔君的凝视可以用避邪符完全免疫，进BOSS战前务必拿
- 毒液池的毒水伤害极高，不要试图游泳
- 毒王令一定要捡，否则错过隐藏BOSS和最好的毒属性武器材料
- 祭祀坛的传送门通往盘丝洞各个区域，善用可以快速移动
