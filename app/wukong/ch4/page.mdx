# 第四章：盘丝洞 / Chapter 4: Spider Cave

## 版本说明 / Version Info

适配最新正式版 / Compatible with the latest release version

## 路线流程 / Route Walkthrough

1. 从新西天渡口乘船进入**盘丝洞·洞窟入口** / Take the boat from New Western Paradise dock to **Spider Cave · Cavern Entrance**
2. 穿过蛛网密布的洞穴，到达**盘丝洞前厅土地庙** / Traverse the web-covered tunnels to **Spider Cave Front Hall Shrine**
3. 清理前厅蜘蛛卵和小蜘蛛群，遭遇精英怪**黑手道人** / Clear spider eggs and small spiders, encounter elite **Black Hand Daoist**
4. 击败黑手道人后获得**蜘蛛令**，可打开后续机关门 / Defeat him to obtain the **Spider Token**, which opens mechanism doors ahead
5. 深入洞窟抵达**蜘蛛巢穴**，触发**紫蛛儿**BOSS战 / Go deeper to **Spider Nest**, triggering the **Purple Spider** boss fight
6. 击败紫蛛儿后从巢穴后方的竖井垂降到底层 / After defeating her, descend through the shaft behind the nest to the bottom
7. 底层为**毒液池**区域，小心毒水，沿墙壁走 / The bottom level is a **Poison Pool** area, watch out for toxic water, stick to the walls
8. 穿过毒液池到达**盘丝洞正殿**，面对**百眼魔君** / Cross the poison pool to **Spider Cave Main Hall**, face **Hundred-Eyed Daoist**
9. 击败百眼魔君后从正殿后门进入**祭祀坛** / After defeating him, enter the **Sacrificial Altar** through the hall's back door
10. 在祭祀坛触发隐藏BOSS**毒王**（可选，需持毒王令）/ Trigger hidden boss **Poison King** at the altar (optional, requires Poison King Token)

## BOSS打法 / Boss Strategies

### 黑手道人 / Black Hand Daoist

- **技能机制 / Mechanics**: 多臂攻击+暗器投掷+烟雾隐身 / Multi-arm attacks + throwing daggers + smoke invisibility
- **核心弱点 / Weakness**: 烟雾隐身时移动有足迹轨迹 / Leaves footprints while moving in smoke
- **躲避时机 / Dodge Timing**: 暗器投掷前有收手动作 / Arm retraction before throwing daggers
- **连招建议 / Combo**:
  - 隐身时根据足迹预判位置 → 蓄力重棍等他现身 / Predict position from footprints → charged heavy ready
  - 现身瞬间打出满蓄力 / Unleash full charge the moment he appears
  - 他多臂乱舞时拉开距离 / Back off during multi-arm flurry
- **逃课打法 / Cheese**: 贴墙站减少烟雾隐身时的追击范围 / Fight against a wall to reduce his pursuit range during smoke

### 紫蛛儿 / Purple Spider

- **技能机制 / Mechanics**: 蛛丝缠绕+产卵召唤小蜘蛛+毒液喷射 / Web wrap + egg-laying summon small spiders + poison spit
- **核心弱点 / Weakness**: 产卵时本体不动 / Stationary while laying eggs
- **躲避时机 / Dodge Timing**: 蛛丝出手时向侧方闪避 / Dodge sideways when web fires
- **连招建议 / Combo**:
  - 起手定身术打断蛛丝 → 轻×5 + 重击 / Open with Immobilize to interrupt web → light ×5 + heavy
  - 产卵时优先清卵，否则小蜘蛛会累积 / Prioritize clearing eggs, otherwise small spiders pile up
  - 毒液喷射前后摇长，绕侧输出 / Poison spit has long startup, attack from the side
- **逃课打法 / Cheese**: 绕圈跑等小蜘蛛聚集 → 广智变身火焰AOE清场 → 打本体 / Circle until spiders group → Guangzhi fire AoE → attack main body

### 百眼魔君 / Hundred-Eyed Daoist

- **技能机制 / Mechanics**: 多眼激光+石化凝视+召唤毒蛛 / Multi-eye laser + petrifying gaze + summon poison spiders
- **核心弱点 / Weakness**: 看他的眼睛会被石化，需要背对或使用**避邪符** / Looking at his eyes petrifies you — face away or use the **Warding Charm**
- **第一阶段 / Phase 1**: 激光扫射+石化凝视 / Laser sweep + petrifying gaze
  - 不要直视他的眼睛，用侧移保持他在画面边缘 / Don't look directly at his eyes, strafe to keep him at screen edge
  - 激光有蓄力轨迹，向垂直方向跑 / Laser has a charge trajectory, run perpendicular
- **第二阶段 / Phase 2**: 召唤毒蛛+全屏毒雨 / Summon poison spiders + full-screen poison rain
  - 毒雨有地面印记，在印记出现后闪避 / Poison rain has ground markers, dodge after they appear
  - 优先清理毒蛛，避免被围攻 / Prioritize clearing poison spiders
- **连招建议 / Combo**:
  - 第一阶段：侧移避开凝视 → 冲到他侧面轻×4 + 重击 → 拉开 / Strafing avoids gaze → rush to his side light ×4 + heavy → retreat
  - 第二阶段：清毒蛛 → 等他放完毒雨 → 定身术 + 满蓄力 / Clear spiders → wait for poison rain → Immobilize + full charge
- **逃课打法 / Cheese**: 保持中远距离，等他放完激光再靠近打 / Keep mid-to-long range, approach after his laser

### 毒王（隐藏BOSS）/ Poison King (Hidden Boss)

- **触发条件 / Trigger**: 持毒王令进入祭祀坛深处 / Enter deepest part of the Sacrificial Altar with the Poison King Token
- **技能机制 / Mechanics**: 毒雾领域+毒爪连击+剧毒炸弹 / Poison mist field + poison claw combo + toxic bombs
- **核心弱点 / Weakness**: 毒雾领域有范围限制，出范围可解毒 / Poison mist has a limited range, leaving it removes the poison
- **躲避时机 / Dodge Timing**: 毒爪连击最后一下是重击，闪避后有大硬直 / The last hit of claw combo is a heavy strike, big recovery after dodge
- **连招建议 / Combo**:
  - 毒雾时退出范围 → 等他出来 → 轻×3 + 重击 / Exit poison mist → wait for him → light ×3 + heavy
  - 毒爪连击闪避最后一下 → 定身术 + 蓄力重棍 / Dodge last claw hit → Immobilize + charged heavy
- **逃课打法 / Cheese**: 全程保持毒雾范围外，用远程蓄力磨血 / Stay outside the poison mist, chip with ranged charges

## 避坑要点 / Tips & Pitfalls

- **盘丝洞的蜘蛛卵见到就要打掉**，否则走过后会孵化出小蜘蛛追你 / **Destroy spider eggs on sight** — they hatch into chasing spiders
- **黑手道人的烟雾隐身阶段不要乱跑**，看足迹轨迹 / **During smoke invisibility**, watch for footprint trails
- **紫蛛儿的蛛丝缠绕中了之后可以用广智变身的火焰烧断** / **Web bind can be burned off** with Guangzhi transformation's fire
- **百眼魔君的凝视可以用避邪符完全免疫**，进BOSS战前务必拿 / **Warding Charm nullifies the gaze** — grab it before the boss fight
- **毒液池的毒水伤害极高**，不要试图游泳 / **Poison Pool water deals massive damage** — don't swim
- **毒王令一定要捡**，否则错过隐藏BOSS和最好的毒属性武器材料 / **Don't miss the Poison King Token** — best poison weapon material
- **祭祀坛的传送门通往盘丝洞各个区域**，善用可以快速移动 / **Sacrificial Altar teleports** connect to various areas, use for fast travel
