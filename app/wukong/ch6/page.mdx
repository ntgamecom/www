# 第六章：花果山 / Chapter 6: Mountain of Flowers and Fruit

## 版本说明 / Version Info

适配最新正式版 / Compatible with the latest release version

## 路线流程 / Route Walkthrough

1. 从火焰山渡口乘船返回**花果山·水帘洞入口** / Take the boat from Flaming Mountain dock to **Mountain of Flowers and Fruit · Water Curtain Cave Entrance**
2. 进入水帘洞，激活**水帘洞土地庙**，这里是最终章的安全区 / Enter and activate **Water Curtain Cave Shrine** — the final chapter's safe zone
3. 从水帘洞出来到达**花果山山脚**，清理天兵天将 / Exit to **Mountain Base**, clear Heavenly Soldiers
4. 沿山路向上，遇到**四大天王**逐一挑战（顺序固定） / Climb the mountain path, encounter the **Four Heavenly Kings** one by one (fixed order)
5. 击败四大天王后到达**花果山山顶** / After defeating them, reach **Mountain Peak**
6. 在山顶触发**二郎神**BOSS战 / At the peak, trigger the **Erlang Shen** boss fight
7. 击败二郎神后进入**天庭裂隙**，面对最终BOSS**大圣残躯** / Defeat Erlang Shen and enter **Heavenly Rift** to face the final boss **Great Sage's Remnants**
8. 击败大圣残躯后观看结局动画 / Watch the ending cutscene after defeating the final boss
9. 二周目解锁：继承部分装备重新开始，难度提升 / New Game+ unlocked: inherit some equipment, restart with increased difficulty

## BOSS打法 / Boss Strategies

### 增长天王（四大天王·第一位）/ Virūḍhaka (Growth Heavenly King · First)

- **技能机制 / Mechanics**: 长剑+青光护盾+召唤天兵 / Longsword + cyan light shield + summon soldiers
- **核心弱点 / Weakness**: 青光护盾需要连续攻击破盾 / Cyan shield requires sustained attacks to break
- **躲避时机 / Dodge Timing**: 长剑突刺时向侧方闪避 / Dodge sideways during sword thrust
- **连招建议 / Combo**:
  - 开局连续攻击破盾 → 盾破后轻×5 + 重击 / Open with sustained attacks to break shield → light ×5 + heavy
  - 召唤天兵时用AOE清掉 / Clear summoned soldiers with AoE
- **逃课打法 / Cheese**: 保持移动，他的突刺有轨迹预判 / Keep moving, his thrust has a predictable trajectory

### 多闻天王（四大天王·第二位）/ Dhṛtarāṣṭra (Dharma Heavenly King · Second)

- **技能机制 / Mechanics**: 混元伞+雷电攻击+瞬移 / Universe Umbrella + lightning attacks + teleport
- **核心弱点 / Weakness**: 混元伞张开时他不能移动 / He can't move while the umbrella is open
- **躲避时机 / Dodge Timing**: 雷电从天而降，地面有蓝色光圈 / Lightning strikes from above, blue circles on the ground
- **连招建议 / Combo**:
  - 他开伞时绕到背后输出 / Circle behind when his umbrella opens
  - 雷电光圈出现后闪避，不要连续站同一个位置 / Dodge after lightning circles appear
- **逃课打法 / Cheese**: 等他开伞再打，不开伞时保持距离 / Only attack when umbrella is open

### 广目天王（四大天王·第三位）/ Virūpākṣa (Wide-Eyed Heavenly King · Third)

- **技能机制 / Mechanics**: 赤龙缠绕+火焰吐息+飞龙骑脸 / Red dragon缠绕 + flame breath + dragon dive
- **核心弱点 / Weakness**: 赤龙是他的弱点，攻击赤龙头部可中断技能 / The red dragon is his weak point — attacking its head interrupts skills
- **躲避时机 / Dodge Timing**: 火焰吐息转向你的方向时向侧面跑 / Run sideways when flame breath turns toward you
- **连招建议 / Combo**:
  - 优先攻击赤龙头部 → 中断吐息后打本体 / Prioritize attacking the dragon's head → interrupt breath then hit main body
  - 飞龙骑脸时向两侧闪避 / Side dodge during dragon dive
- **逃课打法 / Cheese**: 专注打龙，龙被打掉后天王就废了一半 / Focus the dragon — once it's down, the king is half-defeated

### 持国天王（四大天王·第四位）/ Vaiśravaṇa (Wealth Heavenly King · Fourth)

- **技能机制 / Mechanics**: 琵琶音波+范围眩晕+召唤四天王合体技 / Lute sonic waves + area stun + summon Four Kings combined attack
- **核心弱点 / Weakness**: 音波攻击有波形轨迹 / Sonic attacks have visible wave patterns
- **躲避时机 / Dodge Timing**: 琵琶弹奏时注意地面音波扩散轨迹 / Watch the ground sonic wave spread during lute playing
- **连招建议 / Combo**:
  - 闪避音波后近身打断他弹奏 / Close distance after sonic wave dodge to interrupt his playing
  - 合体技时全力闪避，不要输出 / Full focus on dodging during the combined attack
- **逃课打法 / Cheese**: 贴身打不让他弹琵琶 / Stay glued to him to prevent lute playing

### 二郎神 / Erlang Shen (Yang Jian)

- **技能机制 / Mechanics**: 三尖两刃刀+天眼激光+七十二变（变身各种妖兽）/ Three-pointed blade + heavenly eye laser + 72 Transformations
- **核心弱点 / Weakness**: 天眼激光蓄力位置固定，站在他背后可规避 / Laser has a fixed charge position — stand behind him
- **第一阶段 / Phase 1**: 人形态 + 三尖两刃刀连击 / Human form + three-pointed blade combos
  - 刀法流畅，每一刀都有不同角度 / Flowing sword style, each strike from a different angle
  - 连击最后一下是横扫，范围最广 / Final hit is a wide sweep
- **第二阶段 / Phase 2**: 天眼激光 + 七十二变 / Heavenly eye laser + 72 Transformations
  - 天眼激光蓄力时绕到背后输出 / Circle behind during laser charge
  - 七十二变：变成虎→攻击方式同虎先锋；变成鹰→远程风刃 / Tiger form → same as Tiger Vanguard; Eagle → ranged wind blades
- **连招建议 / Combo**:
  - 第一阶段：闪避完整套连击 → 定身术 + 满蓄力重棍 / Dodge the full combo → Immobilize + fully charged heavy
  - 第二阶段：绕背躲激光 → 趁变身切换间隙输出 / Circle behind to dodge laser → damage during transformation switch
  - 七十二变时用广智变身对拼 / Match transformations with Guangzhi
- **逃课打法 / Cheese**: 激光阶段是最好输出时机，其他时间保持距离磨血 / The laser phase is the best damage window

### 大圣残躯（最终BOSS）/ Great Sage's Remnants (Final Boss)

- **技能机制 / Mechanics**: 如意金箍棒（可长可短）+筋斗云瞬移+分身术+定身术 / Ruyi Jingu Bang (extendable) + Jindou Cloud teleport + clone technique + Immobilize
- **核心弱点 / Weakness**: 分身术时只有真身有影子 / During clone technique, only the real body casts a shadow
- **第一阶段 / Phase 1**: 基础战斗 / Basic combat
  - 金箍棒范围超大，注意闪避节奏 / Jingu Bang has huge range, watch your dodge rhythm
  - 他会用和你一样的定身术！被定住时切广智变身解控 / He uses the SAME Immobilize — switch to Guangzhi to break free
  - 筋斗云瞬移后会接一个突进攻击 / Jindou Cloud teleport is followed by a charging attack
- **第二阶段 / Phase 2**: 全力爆发 / Full power
  - 分身术召唤三个残影同时攻击 / Clone technique summons three phantoms
  - 金箍棒伸长时横扫全场 / Extended Jingu Bang sweeps the entire arena
  - 血量极低时会触发一次全屏AOE，必须完美闪避 / At critically low HP, triggers a full-screen AoE requiring perfect dodge
- **连招建议 / Combo**:
  - 第一阶段：等他出完一套 → 轻×3 + 重击 → 立刻拉开 / Wait for his set → light ×3 + heavy → immediately retreat
  - 第二阶段：看到分身先找真身（看影子） → 定身术 + 满蓄力 / When clones appear, find the real one (check shadows) → Immobilize + full charge
  - 全屏AOE：地面出现金色光圈时跳跃+闪避（完美闪避时机）/ Full-screen AoE: jump + dodge when the golden ring appears (perfect dodge timing)
- **逃课打法 / Cheese**: 保持最远距离，等他出完大硬直技能再靠近 / Stay at max range, approach only after high-recovery skills

## 避坑要点 / Tips & Pitfalls

- **四大天王的战斗可以逐个击破**，每打完一个可以回水帘洞休整 / **Fight the Four Kings one at a time** — rest at Water Curtain Cave between each
- **二郎神的变身切换间隙是固定套路**，记熟就是输出窗口 / **Transformation switches follow a fixed pattern** — learn it for damage windows
- **大圣残躯的定身术和你的一样**，会被广智变身解除 / **His Immobilize works like yours** — break it with Guangzhi transformation
- **最终战不要省药**，这是最后一关了 / **Don't hoard potions** — this is the last fight
- **分身术看影子找真身是最快方法** / **Look for shadows** to identify the real body during clones
- **全屏AOE的完美闪避时机是金色光圈收缩到最小时** / **Perfect dodge timing** is when the golden ring contracts to its smallest
- **二周目会继承葫芦升级、珍稀材料和法术**，建议至少打完二周目解锁真结局 / **NG+ carries over gourd upgrades, rare materials, and spells** — recommended for the true ending
- **击败大圣残躯后可以自由探索已解锁的所有区域** / **Explore all previously unlocked areas** after defeating the final boss
