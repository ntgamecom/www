# 第六章 花果山 完整攻略

## 版本说明

适配最新正式版

## 路线流程

1. 从火焰山渡口乘船返回**花果山·水帘洞入口**
2. 进入水帘洞，激活**水帘洞土地庙**，这里是最终章的安全区
3. 从水帘洞出来到达**花果山山脚**，清理天兵天将
4. 沿山路向上，遇到**四大天王**逐一挑战（顺序固定）
5. 击败四大天王后到达**花果山山顶**
6. 在山顶触发**二郎神**BOSS战
7. 击败二郎神后进入**天庭裂隙**，面对最终BOSS**大圣残躯**
8. 击败大圣残躯后观看结局动画
9. 二周目解锁：继承部分装备重新开始，难度提升

## BOSS打法

### 增长天王（四大天王·第一位）

- **技能机制**：长剑+青光护盾+召唤天兵
- **核心弱点**：青光护盾需要连续攻击破盾
- **躲避时机**：长剑突刺时向侧方闪避
- **连招建议**：
  - 开局连续攻击破盾 → 盾破后轻×5 + 重击
  - 召唤天兵时用AOE清掉
- **逃课打法**：保持移动，他的突刺有轨迹预判

### 多闻天王（四大天王·第二位）

- **技能机制**：混元伞+雷电攻击+瞬移
- **核心弱点**：混元伞张开时他不能移动
- **躲避时机**：雷电从天而降，地面有蓝色光圈
- **连招建议**：
  - 他开伞时绕到背后输出
  - 雷电光圈出现后闪避，不要连续站同一个位置
- **逃课打法**：等他开伞再打，不开伞时保持距离

### 广目天王（四大天王·第三位）

- **技能机制**：赤龙缠绕+火焰吐息+飞龙骑脸
- **核心弱点**：赤龙是他的弱点，攻击赤龙头部可中断技能
- **躲避时机**：火焰吐息转向你的方向时向侧面跑
- **连招建议**：
  - 优先攻击赤龙头部 → 中断吐息后打本体
  - 飞龙骑脸时向两侧闪避
- **逃课打法**：专注打龙，龙被打掉后天王就废了一半

### 持国天王（四大天王·第四位）

- **技能机制**：琵琶音波+范围眩晕+召唤四天王合体技
- **核心弱点**：音波攻击有波形轨迹
- **躲避时机**：琵琶弹奏时注意地面音波扩散轨迹
- **连招建议**：
  - 闪避音波后近身打断他弹奏
  - 合体技时全力闪避，不要输出
- **逃课打法**：贴身打不让他弹琵琶

### 二郎神

- **技能机制**：三尖两刃刀+天眼激光+七十二变（变身各种妖兽）
- **核心弱点**：天眼激光蓄力位置固定，站在他背后可规避
- **第一阶段**：人形态 + 三尖两刃刀连击
  - 刀法流畅，每一刀都有不同角度
  - 连击最后一下是横扫，范围最广
- **第二阶段**：天眼激光 + 七十二变
  - 天眼激光蓄力时绕到背后输出
  - 七十二变：变成虎→攻击方式同虎先锋；变成鹰→远程风刃
- **连招建议**：
  - 第一阶段：闪避完整套连击 → 定身术 + 满蓄力重棍
  - 第二阶段：绕背躲激光 → 趁变身切换间隙输出
  - 七十二变时用广智变身对拼
- **逃课打法**：激光阶段是最好输出时机，其他时间保持距离磨血

### 大圣残躯（最终BOSS）

- **技能机制**：如意金箍棒（可长可短）+筋斗云瞬移+分身术+定身术
- **核心弱点**：分身术时只有真身有影子
- **第一阶段**：基础战斗
  - 金箍棒范围超大，注意闪避节奏
  - 他会用和你一样的定身术！被定住时切广智变身解控
  - 筋斗云瞬移后会接一个突进攻击
- **第二阶段**：全力爆发
  - 分身术召唤三个残影同时攻击
  - 金箍棒伸长时横扫全场
  - 血量极低时会触发一次全屏AOE，必须完美闪避
- **连招建议**：
  - 第一阶段：等他出完一套 → 轻×3 + 重击 → 立刻拉开
  - 第二阶段：看到分身先找真身（看影子） → 定身术 + 满蓄力
  - 全屏AOE：地面出现金色光圈时跳跃+闪避（完美闪避时机）
- **逃课打法**：保持最远距离，等他出完大硬直技能再靠近

## 避坑要点

- 四大天王的战斗可以逐个击破，每打完一个可以回水帘洞休整
- 二郎神的变身切换间隙是固定套路，记熟了就是输出窗口
- 大圣残躯的定身术和你的一样，会被广智变身解除
- 最终战不要省药，这是最后一关了
- 分身术看影子找真身是最快方法
- 全屏AOE的完美闪避时机是金色光圈收缩到最小时
- 二周目会继承葫芦升级、珍稀材料和法术，建议至少打完二周目解锁真结局
- 击败大圣残躯后可以自由探索已解锁的所有区域
