## 战斗系统概述 / Combat Overview

**中文：** 黑神话悟空的战斗系统围绕棍法、法术、变身和毫毛四大核心玩法展开。天命人可以使用三种棍法（劈棍法、立棍法、戳棍法）进行基础攻击，消耗法力释放法术，积攒神力变身成为击败过的妖怪，使用毫毛召唤猴群助战。战斗的核心是灵活切换棍法、管理法力资源、选择合适的法术应对不同敌人。

**English:** Black Myth: Wukong's combat revolves around four core systems: Stances, Spells, Transformations, and Strands. The Destined One uses three stances (Smash, Pillar, Thrust) for basic attacks, consumes mana for spells, builds focus for transformations into defeated foes, and uses strands to summon monkey allies. Core combat is about stance switching, mana management, and choosing the right spell for each enemy.

| 战斗系统 / System | 资源 / Resource | 核心作用 / Core Role |
|---|---|---|
| 棍法 / Stances | 体力 / Stamina | 基础攻击和连段 / Basic attacks and combos |
| 法术 / Spells | 法力 / Mana | 控制、输出、生存 / Control, damage, survival |
| 变身 / Transformations | 神力 / Focus | 变身为妖怪战斗 / Transform into defeated enemies |
| 毫毛 / Strands | 毫毛充能 / Strand Charge | 召唤猴群支援 / Summon monkey allies |

## 棍法系统 / Stance System

**中文：** 三种棍法各有特点，适用于不同的战斗场景。战斗中可以在三种棍法之间自由切换，组成连段。

**English:** Each of the three stances has unique characteristics suited to different combat scenarios. Switch freely between stances during combat for combos.

**劈棍法 / Smash Stance**:

- 特点：速度快、伤害高、适合快速压制 / Fast attacks, high damage, good for aggressive pressure
- 轻击连段：4连击，最后一下击倒 / 4-hit combo, final hit knocks down
- 重击：蓄力后释放强力下劈 / Charged heavy overhead smash
- 核心技能：劈棍连打（增加连击段数）/ Smash Combo (extends combo length)
- 推荐场景：面对小型敌人群、需要快速输出的战斗 / Small enemy groups, burst damage situations

**立棍法 / Pillar Stance**:

- 特点：攻击范围大、可攻击高处敌人、格挡性能好 / Wide arcs, hits high enemies, good blocking
- 轻击连段：3连击+范围横扫 / 3-hit combo + wide sweep
- 重击：立棍格挡后反击 / Pillar block then counter
- 核心技能：铁壁格挡（格挡时减少耐力消耗）/ Iron Wall (block stamina reduction)
- 推荐场景：群怪、大型Boss、需要防守反击的战斗 / Crowds, large bosses, counter-attack scenarios

**戳棍法 / Thrust Stance**:

- 特点：攻击距离远、穿透力强、精准打击 / Long range, piercing, precision strikes
- 轻击连段：2连突刺+上挑 / 2 thrusts + uppercut
- 重击：蓄力突刺，可穿透多个敌人 / Charged thrust, pierces multiple enemies
- 核心技能：穿心突刺（增加突刺伤害和距离）/ Piercing Thrust (increased damage and range)
- 推荐场景：远距离起手、对付持盾敌人、安全输出 / Ranged openers, shielded enemies, safe DPS

## 法术系统 / Spell System

**中文：** 法术消耗法力值，提供多样化的战斗能力。可以在土地庙（检查点）重置法术配置。法力通过击败敌人和饮用葫芦恢复。

**English:** Spells consume mana and provide diverse combat abilities. Respec spells at shrines (checkpoints). Mana recovers through defeating enemies and drinking from the gourd.

| 法术 / Spell | 法力消耗 / Cost | 效果 / Effect |
|---|---|---|
| 定身术 / Immobilize | 30 | 定身敌人2-3秒，造成额外伤害破除定身 / Freeze enemy 2-3s, bonus dmg breaks freeze |
| 安身法 / Rock Solid | 25 | 召唤石柱格挡一次攻击，石柱存在期间减伤 / Summon stone pillar to block one hit, damage reduction while active |
| 聚形散气 / Cloud Step | 35 | 隐身+留下残影，残影爆炸造成伤害 / Invisibility + decoy, decoy explodes for damage |
| 毫毛法 / Pluck of Many | 50 | 拔出一根毫毛召唤猴群助战，猴群继承部分属性 / Summon monkey allies, they inherit some stats |
| 变身术 / Transformation | 100(神力) / Focus | 变身为已击败的妖怪，拥有独立的技能和生命条 / Transform into a defeated foe with unique skills and HP |

**法术使用技巧 / Spell Usage Tips**:
- 定身术是性价比最高的法术——控制+额外伤害 / Immobilize is the best value spell — control + bonus damage
- 聚形散气可以躲避不可闪避的攻击 / Cloud Step can dodge undodgeable attacks
- 安身法在Boss战中可以抵消一次致命攻击 / Rock Solid negates one lethal hit in boss fights
- 毫毛法在群怪和Boss战中都非常强力 / Pluck of Many is powerful in both crowd and boss fights
- 变身术的神力通过攻击积攒，注意不要浪费 / Focus for transformation builds through attacks, don't waste it

## 变身系统 / Transformation System

**中文：** 击败特定妖怪后可以解锁对应的变身形态。每种变身都有独立的生命值条、攻击技能和属性。变身后会持续消耗神力，神力耗尽或生命值为零时恢复原形。

**English:** Defeating certain enemies unlocks their transformation form. Each transformation has its own HP bar, attack skills, and stats. Transformations consume focus over time and revert when focus or HP reaches zero.

**推荐变身 / Recommended Transformations**:

| 变身 / Form | 获得章节 / Chapter | 特点 / Traits |
|---|---|---|
| 广智 / Guangzhi | 第一章 / Ch 1 | 快速火刀攻击，高DPS / Fast fire-blade attacks, high DPS |
| 幽魂 / Spirit | 第一章 / Ch 1 | 高防御，范围冲击波 / High defense, AoE shockwave |
| 不白 / Bubai | 第三章 / Ch 3 | 冰属性攻击，减速敌人 / Ice attacks, slows enemies |
| 百眼魔君 / Hundred-Eyed Daoist | 第四章 / Ch 4 | 远程法术攻击，范围毒雾 / Ranged spell attacks, poison mist |
| 夜叉王 / Yaksha King | 第五章 / Ch 5 | 高伤害近战，血色爆发 / High melee damage, blood burst |

## 通用战斗技巧 / General Combat Tips

**中文：** 以下技巧适用于所有章节的战斗。

**English:** These combat tips apply across all chapters.

1. **闪避（翻滚）有无敌帧**：完美闪避可以触发慢动作效果，创造输出窗口 / Dodges have iframes, perfect dodge triggers slow-mo for attack windows
2. **棍势切换连段**：轻击+重击可以打出特殊连段，不同棍法组合不同 / Light + heavy attack combos vary by stance
3. **跳跃攻击**：跳跃后重击可打出下砸攻击，有高削韧效果 / Jumping heavy attack deals high stagger
4. **喝葫芦时机**：BOSS战中寻找安全窗口喝葫芦恢复 / Find safe windows to drink gourd during boss fights
5. **法术配合连段**：定身术起手 → 毫毛召唤 → 变身爆发 / Immobilize → Pluck of Many → Transformation burst
6. **蓄力重击**：不在战斗中可以提前蓄力，进战斗后释放 / Pre-charge heavy attack before entering combat
7. **土地庙升级**：每个土地庙可以升级葫芦容量和恢复量 / Upgrade gourd capacity and recovery at shrines

## 棍势系统 / Focus System

**中文：** 棍势是悟空的特殊资源系统。通过连续攻击和完美闪避积攒棍势，消耗棍势可以释放强力技能。不同棍法消耗棍势的方式不同——劈棍法消耗棍势打出爆发连段，立棍法消耗棍势强化格挡反击，戳棍法消耗棍势释放穿透突刺。

**English:** Focus is Wukong's special resource. Build focus through consecutive attacks and perfect dodges. Spend focus on powerful techniques. Each stance spends focus differently — Smash stance spends on burst combos, Pillar stance on enhanced counterattacks, Thrust stance on piercing thrusts.

**棍势积攒方式 / Focus Building Methods**:

| 方式 / Method | 棍势获得 / Focus Gained |
|---|---|
| 轻击命中 / Light attack hit | +0.1 |
| 重击命中 / Heavy attack hit | +0.2 |
| 完美闪避 / Perfect dodge | +0.3 |
| 格挡成功 / Successful block | +0.15 |
| 处决敌人 / Execute enemy | +1.0 |

**棍势消耗技能 / Focus Spending Skills**:
- 劈棍爆发：消耗3棍势，释放5连击终结技，伤害极高 / Smash Burst: spend 3 focus, 5-hit finisher, massive damage
- 立棍反击：消耗2棍势，格挡后发动范围反击 / Pillar Counter: spend 2 focus, AoE counter after block
- 戳棍穿刺：消耗2棍势，释放穿透性突刺，无视防御 / Thrust Pierce: spend 2 focus, piercing thrust, ignores defense

## 敌人类型与应对 / Enemy Types

**中文：** 游戏中的敌人分为普通妖怪、精英妖怪和Boss三种。不同类型的敌人需要不同的应对策略。

**English:** Enemies are divided into normal yaoguai, elite yaoguai, and bosses. Each type requires different strategies.

**普通妖怪 / Normal Yaoguai**:
- 数量多，攻击模式简单 / Numerous, simple attack patterns
- 推荐：劈棍法快速清场 / Recommended: Smash stance for quick clearing
- 注意：部分普通妖怪会自爆或召唤同伴 / Some explode on death or summon allies

**精英妖怪 / Elite Yaoguai**:
- 攻击力高，有特殊技能 / High damage, special abilities
- 推荐：立棍法防御反击或戳棍法远程消耗 / Recommended: Pillar stance counter or Thrust stance poke
- 掉落：精魄、珍玩、制作材料 / Drops: Spirits, curios, crafting materials

**Boss妖怪 / Boss Yaoguai**:
- 多阶段战斗，攻击模式随HP变化 / Multi-phase fights, patterns change with HP
- 推荐：法术配合变身爆发输出 / Recommended: Spells + transformation burst
- 关键：找到每个Boss的攻击窗口和弱点 / Key: find each boss's attack windows and weaknesses

{/* ===== 战斗资源管理 / Resource Management ===== */}

## 战斗资源管理 / Resource Management

**中文：** 战斗中的资源管理决定胜负。法力用于释放法术，体力用于攻击和闪避，棍势用于爆发技能。合理分配资源是通关高难度的关键。

**English:** Resource management determines victory or defeat. Mana is for spells, stamina for attacks and dodges, focus for burst skills. Proper resource allocation is key to clearing difficult content.

- **法力管理**：定身术和聚形散气是最高效的法术，优先保留法力使用这两个 / Prioritize mana for Immobilize and Cloud Step
- **体力管理**：不要连续翻滚超过3次，保留体力进行反击 / Don't roll more than 3 times consecutively, save stamina for counters
- **葫芦管理**：每次土地庙休息后葫芦自动补满，战斗中尽量喝到最后一口 / Gourds auto-refill at shrines, try to save one charge for emergencies
- **变身时机**：在Boss第二阶段变身为最佳时机，此时Boss攻击模式变化最大 / Transform at boss phase 2 for maximum impact
