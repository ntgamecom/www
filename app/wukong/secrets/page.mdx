**第三章 新西天 / Chapter 3: New Western Paradise**:

- **收集品：舍利子×7 / Collectible: Relics x7** — 分布在佛寺各处的发光点 / Glowing spots throughout the temple

**第四章 盘丝洞 / Chapter 4: Spider Cave**:

- **隐藏Boss：紫蛛儿 / Hidden Boss: Purple Spider** — 在盘丝洞最深处，需要破坏蜘蛛卵触发 / Deepest part of Spider Cave, destroy spider eggs to trigger
- **隐藏区域：天庭遗址 / Secret Area: Heavenly Court Ruins** — 洞窟中的一个隐藏传送点 / Hidden teleport in the cave

- **隐藏区域：芭蕉洞 / Secret Area: Plantain Cave** — 需要火焰抗性道具进入 / Requires fire resistance item to enter

## 真结局条件 / True Ending

**中文：** 黑神话悟空有多结局，真结局需要完成特定隐藏内容。

**English:** Black Myth: Wukong has multiple endings. The true ending requires specific hidden content.

**真结局解锁条件 / True Ending Requirements**:

1. 完成全部六个章节的主线 / Complete all six chapters' main paths
2. 击败每章的隐藏Boss（幽魂、沙大郎、不白、紫蛛儿、火焰牛魔、真大圣残躯）/ Defeat each chapter's hidden boss
3. 收集全部佛经残卷（共24卷）/ Collect all Sutra Fragments (24 total)
4. 在第六章回到水帘洞触发最终剧情 / Return to Water Curtain Cave in Ch 6 for final story

**真结局内容 / True Ending Content**:
- 揭示天命的真正含义——天命人继承了悟空的记忆和力量 / Reveals the true meaning of destiny — the Destined One inherits Wukong's memories and power
- 如来佛祖出现，给予最终考验 / The Buddha appears for the final test
- 解锁隐藏过场动画，交代所有角色的命运 / Hidden cutscene explaining all characters' fates
- 获得最终武器"如意金箍棒·真" / Obtain the final weapon "Ruyi Jingu Bang·True"

## 地图探索技巧 / Exploration Tips

**中文：** 探索隐藏内容的有效方法。

**English:** Effective methods for discovering hidden content.

- 注意可破坏的墙壁——有些看似实心的墙壁可以用重击打碎 / Watch for destructible walls — some seemingly solid walls can be broken with heavy attacks
- 探索每个岔路——隐藏Boss和收集品往往在主线路径之外 / Explore every fork — hidden bosses and collectibles are usually off the main path
- 与NPC反复对话——部分隐藏任务需要多次对话触发 / Talk to NPCs repeatedly — some hidden quests trigger after multiple conversations
- 使用侦查法术——某些区域需要用特定法术或技能才能看到隐藏路径 / Use detection spells — some hidden paths are only visible with certain abilities
- 记录返回——遇到暂时无法打开的门或机关，标记位置稍后回来 / Note blocked paths — mark inaccessible areas and return later with new abilities
