## 道具系统概述 / Item System Overview

**中文：** 黑神话悟空中的装备道具分为武器、护甲、葫芦、丹药、珍玩和精魄六大类。武器决定基础攻击力，护甲提供防御和套装效果，葫芦用于恢复生命，丹药提供临时Buff，珍玩提供被动增益，精魄提供额外技能。大部分装备可以通过击败Boss、探索隐藏区域和完成支线获得。

**English:** Items in Black Myth: Wukong are divided into six categories: Weapons, Armor, Gourds, Pills, Curios, and Spirits. Weapons determine base attack, armor provides defense and set bonuses, gourds restore HP, pills grant temporary buffs, curios provide passive bonuses, and spirits grant additional skills. Most gear is obtained through boss defeats, hidden area exploration, and side content.

| 道具类型 / Type | 说明 / Description |
|---|---|
| 武器 / Weapons | 棍棒类，提升攻击力和特殊效果 / Staff-type, attack power and special effects |
| 护甲 / Armor | 头/身/足三件套，有套装效果 / Head/Body/Feet, set bonuses |
| 葫芦 / Gourds | 恢复生命，可升级容量和恢复量 / HP recovery, upgradable capacity and amount |
| 丹药 / Pills | 临时属性增益 / Temporary stat buffs |
| 珍玩 / Curios | 被动增益效果 / Passive bonus effects |
| 精魄 / Spirits | 击败精英后获得的特殊技能 / Special skills from elite defeats |
