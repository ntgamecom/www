# 黑神话悟空 第一章完整攻略

## 版本说明

适配最新正式版

## 路线流程

1. 从**初始土地庙（苍狼林）**出发，沿唯一路径前进，击败路边小狼妖获取灵蕴
2. 穿过溪谷抵达**第一座寺庙废墟**，左拐上山坡遇到第一个精英怪**牯护院**
3. 击败牯护院后继续沿山脊走，进入**竹林深处**，遇到第二个精英怪**广智**
4. 击败广智获得【广智变身】能力，原路返回寺庙废墟右前方通往**白雾泽**
5. 白雾泽中有多个岔路：
   - 主路直走触发**灵虚子**BOSS战
   - 右侧岔路通往【马哥】支线
   - 左侧水塘有隐藏道具
6. 击败灵虚子后穿过石门进入**翠竹林**，沿路下行遇到**刀狼教头**
7. 继续深入抵达**蛇径**，小心毒池和蛇妖
8. 穿过蛇径到达**卧虎寺**门前，触发**虎先锋**BOSS战
9. 击败虎先锋进入**卧虎寺地窖**，面对第一章最终BOSS**黑风大王**
10. 击败黑风大王后进入**黑风洞**，挑战隐藏BOSS**金池长老**（可选）

## BOSS打法

### 牯护院

- **技能机制**：近战三板斧（挥砍+突刺+下砸），动作较慢
- **躲避时机**：每次出招后有约2秒硬直，可在其身后输出
- **连招建议**：轻攻击×3 → 重攻击蓄力 → 闪避拉开
- **逃课打法**：远程蓄力棍法，保持距离风筝

### 广智

- **技能机制**：双刀快速连击，有火焰附魔状态
- **躲避时机**：火焰附魔时不要对拼，等附魔结束再输出
- **连招建议**：等他三连斩打完 → 轻×2 + 重击 → 后撤步
- **逃课打法**：利用地形柱子卡位，蓄力重棍偷刀

### 灵虚子

- **技能机制**：大型狼妖，扑击范围大，甩尾有击退
- **躲避时机**：扑击前有起跳动作，向侧方闪避
- **连招建议**：闪避扑击后 → 轻×4 → 重击 → 定身术
- **逃课打法**：保持中距离，等他出完扑击再靠近打一套就跑

### 虎先锋

- **技能机制**：老虎形态二连扑+三连抓，残血时进入狂暴
- **躲避时机**：二连扑第一次向侧闪，第二次向后闪
- **连招建议**：闪避完二连扑 → 轻×3 + 重击 → 广智变身压起身
- **逃课打法**：利用广智变身的火属性克制，配合定身术打一轮就跑

### 黑风大王（最终BOSS）

- **技能机制**：第一阶段人形（剑术连招），第二阶段黑风形态（范围AOE）
- **躲避时机**：
  - 人形阶段：剑光闪烁时立刻闪避
  - 黑风阶段：地面黑圈出现后1秒闪避
- **连招建议**：
  - 人形：等他五连斩打完 → 定身术 + 满蓄力重棍
  - 黑风：闪避黑风冲击 → 轻×5 + 重击收尾
- **逃课打法**：保持远距离用蓄力棍磨血，定身术CD好了就打一套

### 金池长老（隐藏BOSS）

- **技能机制**：召唤分身+范围佛法攻击
- **躲避时机**：分身出现时优先清分身
- **连招建议**：清完分身 → 靠近打一套 → 拉开等下一波
- **逃课打法**：攒满四段棍势，等他出招硬直时一发满蓄力打掉半管血

## 避坑要点

- 不要跳过广智：广智变身是第一章最有用的能力，后期打虎先锋和黑风大王都靠它
- 白雾泽的毒池不要硬闯：从右侧绕路过去，直接踩毒池会快速掉血
- 虎先锋门前先清小怪：卧虎寺门前有大量刀狼教众，不清直接触发BOSS会被群殴
- 灵蕴不要乱花：优先升级葫芦和丹药，武器可以等第二章再换
- 黑风大王二阶段别贪刀：黑风形态的AOE伤害极高，打一套就撤
- 金池长老不打也不影响通关：但是打了给大量灵蕴和佛珠，建议打
