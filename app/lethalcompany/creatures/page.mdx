## 怪物系统 / Creature System

《致命公司》中的怪物种类丰富，各有独特的行为模式和弱点。了解每种怪物是提高生存率的关键。怪物可以分为三类：**被动生物**（不会主动攻击）、**领地生物**（保护特定区域）和**猎食者**（主动追击玩家）。

*Lethal Company* features a diverse roster of creatures, each with unique behavior patterns and weaknesses. Understanding each creature is key to improving survival rates. Creatures fall into three categories: **passive creatures** (do not attack), **territorial creatures** (guard specific areas), and **predators** (actively hunt players).

## 被动生物 / Passive Creatures

## 蜂鸟 / Hummingbird

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 危险等级 / Danger | 无 / None |
| 速度 / Speed | 快 / Fast |
| 攻击性 / Aggression | 无 / None |

蜂鸟是完全无害的生物，在空中飞行。它们会发出嗡嗡声可以提示附近没有其他危险。

Hummingbirds are completely harmless creatures that fly through the air. Their buzzing can signal that no other danger is nearby.

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 危险等级 / Danger | 无 / None |
| 速度 / Speed | 中等 / Medium |
| 攻击性 / Aggression | 无 / None |

兔子是环境生物，不会对玩家造成威胁。它们听到响动会逃跑。

Rabbits are environmental creatures that pose no threat to players. They flee when they hear loud noises.

## 领地生物 / Territorial Creatures

## 孢子蜥蜴 / Spore Lizard

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 危险等级 / Danger | 低 / Low |
| 速度 / Speed | 中等 / Medium |
| 弱点 / Weakness | 光线 / Light |
| 出现星球 / Moons | Experimentation, Assurance, Vow |

孢子蜥蜴在受到惊吓时会释放致盲孢子云。它本身不会直接伤害玩家，但孢子效果会使你暂时失明。用灯光照射可以使其退避。

The Spore Lizard releases a blinding spore cloud when startled. It does not directly damage players, but the spore effect causes temporary blindness. Shining a light at it will make it retreat.

**应对方法** / Counter: 保持距离。如果被孢子击中，原地等待效果消失。Maintain distance. If hit by spores, stay still until the effect wears off.

## 地鼠 / Bunker Spider

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 危险等级 / Danger | 中 / Medium |
| 速度 / Speed | 慢-中 / Slow-Med |
| 弱点 / Weakness | 速度慢，可规避 / Slow, avoidable |
| 出现星球 / Moons | Offense, March, Rend |

地鼠是在室内结网的蜘蛛类生物。它会在走廊织网，玩家触碰蛛网会触发警报。地鼠会追踪触发警报的玩家，但速度较慢。

The Bunker Spider is a spider-like creature that webs indoor corridors. Touching its web triggers an alert. The spider pursues players who trigger its web but moves slowly.

**应对方法** / Counter: 看到蛛网就绕路。如果触发警报，跑远即可摆脱。Avoid webs when spotted. If alerted, run far enough to lose aggro.

## 猎食者 / Predators

## 无眼狗 / Eyeless Dog

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 危险等级 / Danger | 高 / High |
| 速度 / Speed | 快 / Fast |
| 弱点 / Weakness | 不能看，靠听力 / Blind, relies on hearing |
| 出现星球 / Moons | 全部 / All |

无眼狗是游戏中最危险的怪物之一。它没有视觉，完全依靠听觉追踪猎物。如果你保持静止和安静，它会从你身边走过而不会发现你。

The Eyeless Dog is one of the most dangerous creatures in the game. It has no vision and relies entirely on hearing to track prey. If you stay still and silent, it can walk right past you without detection.

**应对方法** / Counter:
- 被追逐时立即停止移动 / Stop moving immediately when chased
- 蹲下可以减少移动声音 / Crouching reduces movement noise
- 扔出物品制造噪音引开它 / Throw items to create noise diversions
- 千万不要跑进开阔地 / Never run across open ground

## 森林守卫 / Forest Keeper

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 危险等级 / Danger | 极高 / Very High |
| 速度 / Speed | 中等 / Medium |
| 弱点 / Weakness | 室内无法进入 / Cannot enter buildings |
| 出现星球 / Moons | 室外大型星球 / Large outdoor moons |

森林守卫是巨大的类人生物，在室外巡逻。它们无法进入建筑物，但会守候在出口附近。被抓住即死。

Forest Keepers are giant humanoid creatures that patrol outdoor areas. They cannot enter buildings but will wait near exits. Being caught results in instant death.

**应对方法** / Counter: 观察其巡逻路线。利用建筑物进出口转移。在它们背对时快速移动。Watch patrol patterns. Use building entrances to reposition. Move quickly when they face away.

## 蜈蚣机械 / Coil-Head

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 危险等级 / Danger | 高 / High |
| 速度 / Speed | 静止或极快 / Static or very fast |
| 弱点 / Weakness | 必须持续观察 / Must be watched continuously |
| 出现星球 / Moons | 室内区域 / Indoor areas |

蜈蚣机械是一种恐怖的机械生物。当你看着它时，它静止不动。但一旦你移开视线，它会以极快的速度向你移动。多人时可以轮流盯着它行动。

The Coil-Head is a terrifying mechanical creature. It remains still while you watch it. The moment you look away, it moves toward you at extreme speed. Multiple players can take turns watching it to move safely.

## 精英怪物 / Elite Creatures

## 小丑 / Jester

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 危险等级 / Danger | 极高 / Very High |
| 速度 / Speed | 慢→极快 / Slow→Extreme |
| 弱点 / Weakness | 需要时间启动 / Requires wind-up time |
| 出现星球 / Moons | Titan, Artifice, Rend |

小丑开始时是一个无害的音乐盒。一段时间后会变成致命的追杀形态。在它变身前有充足的时间撤离。

The Jester starts as a harmless music box. After a wind-up period, it transforms into a lethal chasing form. You have ample time to evacuate before it transforms.

**应对方法** / Counter: 听到音乐盒声音立即离开该区域。Leaver the area immediately upon hearing the music box.

## 鬼魂女孩 / Ghost Girl

| 属性 / Attribute | 详情 / Detail |
|---|---|
| 危险等级 / Danger | 极高 / Very High |
| 速度 / Speed | 中等 / Medium |
| 弱点 / Weakness | 不能进入光照区域 / Avoids lit areas |
| 出现星球 / Moons | 全部 / All (低概率 / Low chance) |

鬼魂女孩是一种稀有且极为致命的生物。她会缠住一名玩家，其他玩家看不到她。被缠住的玩家必须保持移动并寻找光源来驱赶她。

The Ghost Girl is a rare and extremely lethal creature. She latches onto one player and is invisible to others. The targeted player must keep moving and seek light sources to ward her off.
