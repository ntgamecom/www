# 技巧策略 — 高级玩法与效率攻略
# Strategies Guide — Advanced Play & Efficiency Tips

---

{/* ===== 策略概述 ===== */}

## 策略思维 / Strategic Mindset

《致命公司》不只是一款拾荒模拟器——它是一款 **风险管理游戏**。每一秒你都在做出权衡：是继续深入寻找高价值物品，还是带着现有战利品安全返回？高级玩家的核心能力是做出最优决策。

*Lethal Company* is not just a scavenging simulator — it is a **risk management game**. Every second you make trade-offs: push deeper for high-value items or return safely with current loot? The core skill of advanced players is making optimal decisions.

## 拾荒路线规划 / Scavenging Route Planning

## 高效搜索原则 / Efficient Search Principles

1. **顺时针搜索法** / Clockwise sweep — 从飞船出发，沿顺时针方向清理建筑，最后绕回飞船
2. **分层搜索** / Tiered search — 先搜索主要建筑，有多余时间再搜外围
3. **标记已探索区域** / Mark explored areas — 使用废品或涂鸦标记已完成搜索的建筑
4. **注意时间分配** / Time allocation — 大型建筑预留更多时间，小型建筑速战速决

## 星球特定路线 / Moon-Specific Routes

| 星球 / Moon | 推荐路线 / Recommended Route |
|---|---|
| Experimentation | 主建筑 → 仓库 → 地下室 → 撤离 |
| Assurance | 地面仓库 → 地下通道 → 主建筑 → 撤离 |
| Offense | 东侧建筑 → 地下区域 → 西侧建筑 → 撤离 |
| March | 北仓库 → 中央建筑 → 南洞穴 → 撤离 |
| Titan | 入口大厅 → 东翼 → 西翼 → 地下层 → 撤离 |

## 时间管理 / Time Management

## 游戏内时间节奏 / In-Game Time Rhythm

每个游戏日分为四个时段：

Each game day is divided into four phases:

| 时段 / Phase | 时间 / Time | 建议行动 / Recommended Action |
|---|---|---|
| 早晨 / Morning | 8:00-10:00 | 登录系统，选择星球，检查天气 |
| 上午 / Late Morning | 10:00-14:00 | 主要搜索时段，效率最高 |
| 下午 / Afternoon | 14:00-18:00 | 搬运废品回船，处理紧急情况 |
| 傍晚 / Evening | 18:00-24:00 | 最后一次搬运，准备撤离或夜间生存 |

## 时间节省技巧 / Time-Saving Tips

- **多人分工**：一个人在飞船操作终端，其他人搜索
- **预先打包**：将废品堆在建筑出口附近，最后一次性搬运
- **不要反复进出**：每次往返飞船都会消耗大量时间
- **了解你的队友**：知道谁跑得快、谁能扛，合理分配任务

- **Divide roles**: One person operates the ship terminal while others search
- **Pre-stack**: Pile scrap near building exits, then haul all at once
- **Minimize round trips**: Each trip to the ship consumes significant time
- **Know your team**: Assign tasks based on who runs faster, who can carry more

## 高级通讯策略 / Advanced Communication

## 信息传递协议 / Information Protocol

有效的通讯需要结构化信息。以下是一个推荐的通讯格式：

Effective communication requires structured information. Here is a recommended format:

**发现报告 / Discovery Report**: "位置 [区域名]，发现 [物品名]，价值 [估值]，需要搬运 / Location [area], found [item], value [estimate], needs hauling"

**危险警告 / Danger Warning**: "[怪物名] 在 [位置]，[人数] 注意 / [Creature] at [location], watch out [players]"

**求救信号 / Distress Signal**: "[名字] 被困在 [位置]，需要救援，有 [怪物] 在附近 / [Name] trapped at [location], need rescue, [creature] nearby"

## 语音技巧 / Voice Chat Tips

- 使用 **PTT (按下说话)** 模式避免背景噪音
- 危急时用简短关键词：**"跑" "回来" "别动"**
- 不要同时多人说话
- 建立团队暗号（如 "红色" = 危险，"绿色" = 安全）

- Use **PTT (Push-to-Talk)** mode to avoid background noise
- Use short keywords in emergencies: **"Run" "Come back" "Freeze"**
- Avoid multiple people speaking at once
- Establish team code words (e.g., "Red" = danger, "Green" = safe)

## 高级移动技巧 / Advanced Movement

## 基础移动优化 / Basic Movement Optimization

- **蹲跳** / Crouch jumping — 蹲下后跳跃可以穿过矮洞
- **冲刺取消** / Sprint cancel — 冲刺后短暂松开再冲刺可以略微提升平均速度
- **滑铲** / Slide — 从高处下落时按蹲可以滑铲一小段距离
- **跳跃爬升** / Jump climbing — 连续跳跃可以爬上某些看似无法攀爬的障碍物

## 负重管理 / Weight Management

携带物品会影响你的移动速度。不同物品有不同的重量：

Carrying items affects your movement speed. Different items have varying weights:

| 重量等级 / Weight Tier | 速度影响 / Speed Effect | 示例物品 / Example Items |
|---|---|---|
| 轻 / Light | 几乎无影响 / Negligible | 螺丝刀、对讲机、橡皮鸭 |
| 中 / Medium | 轻微减速 / Slight slow | 金条、铁管、茶杯 |
| 重 / Heavy | 明显减速 / Noticeable slow | 机械零件、花瓶、钟表 |
| 极重 / Very Heavy | 大幅减速 / Major slow | 雕像、核心、大型机械 |

## 资源管理 / Resource Management

## 资金分配策略 / Credit Allocation Strategy

| 阶段 / Phase | 建议支出 / Recommended Spending |
|---|---|
| 第一轮 / Round 1 | 全部存钱 / Save everything |
| 第二轮 / Round 2 | 购买 1-2 个 Pro 手电筒 / Buy 1-2 Pro Flashlights |
| 第三轮 / Round 3 | 购买飞船升级 / Buy ship upgrades |
| 第四轮+ / Round 4+ | 投资高级装备 / Invest in advanced gear |

## 电池管理 / Battery Management

- 手电筒电池有限，关闭时自动充电
- 建议交替使用：一支手电筒用到低电量，切换另一支，让第一支充电
- 专业手电筒续航是普通的两倍
- 备用电池可以购买，但价格不便宜

- Flashlight batteries are limited; they recharge while turned off
- Alternate usage: Use one flashlight until low, switch to another while the first recharges
- Pro Flashlight has double the battery life of the standard one
- Spare batteries can be purchased but are pricey

## 风险计算 / Risk Calculation

## 何时冒险 / When to Take Risks

| 情景 / Scenario | 建议 / Recommendation |
|---|---|
| 配额即将到期 | 冒险深入高风险区域 |
| 已有足够废品 | 不要冒险，保守行动 |
| 队友已经阵亡 | 优先保命而非拾荒 |
| 发现高价值物品 | 集结团队一起运输 |
| 天气恶劣 | 除非必要，不要外出 |

## 止损原则 / Loss Mitigation

当你意识到情况不妙时：

When you realize things are going wrong:

1. 立即评估损失 / Immediately assess the losses
2. 判断能否挽回 / Determine if recovery is possible
3. 如果不能，立即撤退 / If not, retreat immediately
4. 保存现有资源比冒险获得更多更重要 / Preserving current resources matters more than risking for more

## 进阶技巧 / Advanced Tips

1. **双人搬运法**：两个人各拿一边搬运大型废品，速度更快
2. **声音预判**：学习每种怪物声音的方位和距离判断
3. **空投点利用**：在某些星球可以利用地形落差快速下降
4. **探测棒使用**：雷达助推器可以帮你监视大片区域
5. **记忆训练**：努力记忆每个星球的地图布局

1. **Two-person carry**: Two players carrying a large scrap item together moves faster
2. **Audio prediction**: Learn to judge each creature's direction and distance by sound
3. **Drop point use**: Use terrain drops to descend quickly on certain moons
4. **Radar booster use**: Radar Boosters help monitor large areas
5. **Memory training**: Memorize the layout of each moon
