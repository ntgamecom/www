# 新手入门 / Beginner Guide

## 操作基础 / Basic Controls

### PC版默认键位 / PC Default Controls

| 按键 / Key | 功能 / Action |
|---|---|
| W/A/S/D | 移动 / Move |
| 鼠标左键 / Mouse 1 | 射击 / Fire |
| 鼠标右键 / Mouse 2 | 瞄准 / Aim |
| R | 换弹 / Reload |
| Space | 跳跃 / Jump |
| Shift | 冲刺(跑步)/加速(开车) / Sprint (on foot) / Accelerate (driving) |
| Ctrl | 蹲下 / Crouch |
| E | 上车/下车 / Enter/Exit Vehicle |
| F | 近战 / Melee |
| Caps Lock | 特殊能力 / Special Ability |
| Q | 掩护 / Take Cover |
| Z | 切换武器 / Switch Weapon |
| X | 使用装备 / Use Item |
| M | 互动菜单 / Interaction Menu |
| V | 切换视角 / Change Camera |
| TAB | 暂停菜单 / Pause Menu |

**切换角色 / Character Switch**:
- **Alt键**: 切换至下一位主角 / Switch to next character
- **方向键下(↓)**: 打开角色选择轮盘——可查看每位主角目前的位置和状态 / Open character wheel — see location and status of each

## 三位主角 / Three Protagonists

### 迈克尔·迪圣塔 / Michael De Santa

| 属性 / Attribute | 说明 / Description |
|---|---|
| **背景 / Background** | 退休银行劫匪——FIB证人保护计划 / Retired bank robber — FIB witness protection |
| **特殊能力 / Special** | **子弹时间**——射击时放慢时间，精准爆头 / Bullet Time — slow-motion aiming for precise headshots |
| **能力特点 / Strengths** | 射击技能最高、驾驶优秀 / Best shooting stats, good driving |
| **开局房产 / Property** | 洛圣都罗克福德山豪宅 / Mansion in Rockford Hills |
| **剧情定位 / Role** | 任务策划者——制定抢劫计划 / Heist planner — designs the strategy |

**迈克尔核心技巧**:
1. 特殊能力在全队战斗时极为强力——尤其面对大量敌人 / Special ability is OP in combat — especially against groups
2. 迈克尔的身份暴露后会有FIB和黑帮追捕 / Post-exposure, FIB and mobsters hunt him
3. 迈克尔和家庭的关系影响部分剧情 / Family relationships affect story segments

### 富兰克林·克林顿 / Franklin Clinton

| 属性 / Attribute | 说明 / Description |
|---|---|
| **背景 / Background** | 街头混混——收车员出身 / Street hustler — repo man |
| **特殊能力 / Special** | **驾驶专注**——开车时放慢时间，轻松漂移过弯 / Driving Focus — slow-motion driving for easy drifting |
| **能力特点 / Strengths** | 驾驶技能最高、潜行优秀 / Best driving, good stealth |
| **开局房产 / Property** | 洛圣都张伯伦山小公寓 / Small apartment in Chamberlain Hills |
| **剧情定位 / Role** | 执行者——最年轻的成员、司机和打手 / The muscle — youngest member, driver and enforcer |

**富兰克林核心技巧**:
1. 特殊能力在赛车任务和追逐中完全改变游戏体验 / Special ability transforms racing and chase missions
2. 富兰克林联系线广——和他叔叔的互动提供额外任务 / Has connections through his uncle — extra missions
3. 最终的结局选择取决于富兰克林的决策 / Ending choice is Franklin's to make

### 崔佛·菲利普斯 / Trevor Philips

| 属性 / Attribute | 说明 / Description |
|---|---|
| **背景 / Background** | 军事飞行员出身——堕落的疯子和毒贩 / Former military pilot — degenerate lunatic and drug dealer |
| **特殊能力 / Special** | **狂战士**——受到的伤害减半、对敌伤害加倍 / Berserker — take half damage, deal double damage |
| **能力特点 / Strengths** | 飞行技能最高、力量最高 / Best flying, strongest in melee |
| **开局房产 / Property** | 桑迪海岸的拖车屋 / Trailer in Sandy Shores |
| **剧情定位 / Role** | 暴力专家——脏活累活全包 / Violence expert — handles all the dirty work |

**崔佛核心技巧**:
1. 狂战士模式最适合同大量敌人正面硬杠 / Berserker mode is best for going head-on against groups
2. 崔佛有自己独特的支线任务线和帮派生意 / Trevor has unique side mission chains and gang businesses
3. 飞行相关任务——崔佛是最佳人选 / Trevor is the best choice for flying missions

## 技能升级指南 / Skill Leveling

| 技能 / Skill | 最大效果 / Max Effect | 升级方法 / Leveling Method |
|---|---|---|
| **体力 / Stamina** | 无限冲刺 / Unlimited sprint | 持续跑步/骑自行车 / Run or ride a bike |
| **射击 / Shooting** | 减少后座力、弹道更集中 / Less recoil, tighter spread | 在弹药店射击场练习或战斗 / Ammu-Nation range or combat |
| **力量 / Strength** | 近战伤害翻倍 / Double melee damage | 近战攻击、打网球、打高尔夫 / Melee fights, tennis, golf |
| **潜行 / Stealth** | 提升潜行移动速度 / Faster stealth movement | 潜行状态下击杀敌人 / Kill enemies while sneaking |
| **飞行 / Flying** | 减少飞行时的颠簸 / Smoother aircraft handling | 在飞行学校练习或驾驶飞机 / Flight school or fly planes |
| **驾驶 / Driving** | 更好的转向和车辆控制 / Better turning and control | 在公路上高速行驶避免碰撞 / Drive fast without crashing |

**全技能100%快速方法 / Fastest Way to Max All Skills**:
1. 飞行 → 飞行学校完成所有铜牌以上即可 / Flight school all bronze+
2. 射击 → 弹药店里用机枪扫射靶子——但费钱 / Ammu-Nation range with SMG — costs money
3. 力量 → 在洛圣都码头找一个陌生人无限近战 / Find an NPC at the docks and keep punching
4. 体力和驾驶 → 自然升级——多跑多开 / Natural progression through gameplay

## 赚钱入门 / Money-Making for Beginners

### 主线内赚钱 / Making Money in Story Mode

| 方法 / Method | 收入 / Income | 说明 / Description |
|---|---|---|
| **主线和抢劫 / Story & Heists** | 各角色约4000万 / ~40M per character | 完成全部抢劫——选好分赃比例 / Complete all heists with good crew splits |
| **暗杀股票 / Assassination Stocks** | 各角色可达20亿 / Up to 2B | 通关后才做刺杀任务——选好股票赚最大 / Do assassination missions post-game with stock tips |
| **随机事件 / Random Events** | 几千到几万 / A few thousand | 路上随机事件可以拿现金 / On-foot random events give cash |
| **房产投资 / Properties** | 被动收入 / Passive income | 购买电影院、出租车公司等定期收租 / Buy theaters, taxi companies for weekly income |
| **收集品 / Collectibles** | 10万+ / 100K+ | 信件碎片全部收集=10万 / All letter scraps = 100K |

### 主线最佳投资时间线 / Best Investment Timeline

1. **初期(任务1-20)**: 别做暗杀任务——等通关后 + 把所有钱投入BAWSAQ股票 / Skip assassination missions until post-game + invest in BAWSAQ
2. **中期(任务21-40)**: 每个暗杀任务前存盘——买入指定股票卖出赚N倍 / Save before each assassination — buy recommended stocks
3. **后期(任务40-69)**: 通关后做全部暗杀任务 → 三位主角各20亿+ / Post-game: do all assassinations → 2B+ per character
4. **最终/结局后**: 购买全房产、全改车——剩下的就是成就 / Buy all properties, all mods — rest is for achievements

## 线上模式入门 / GTA Online Quick Start

**注意**: 线上模式的人物与主线完全独立——需要单独创建角色 / Note: Online character is completely separate from story mode.

| 等级 / Level Range | 建议 / Recommendation |
|---|---|
| **1-10** | 完成"死亡从天降"(NPC差事)、"逃离洛圣都"、"和平共处"等差事升级 / Do "Death From Above", "Out of Towners", "Co-Operative" missions |
| **10-30** | 购买地堡/夜总会——开始被动收入 / Buy bunker/nightclub — start passive income |
| **30-50** | 完成全福银行抢劫——四人配合 / Complete Fleeca Heist — 4-player coordination |
| **50-100** | 太平洋银行抢劫、末日豪劫——大量金钱 / Pacific Standard, Doomsday Heist — big money |
| **100+** | 佩里科岛抢劫——单人即可12分钟完成150万 / Cayo Perico — solo 12-min run for 1.5M |

## 新手常见错误 / Common Beginner Mistakes

| # | 错误 / Mistake | 正确做法 / Correct Way |
|---|---|---|
| 1 | 前期乱花钱 / Waste money early | 把钱存起来——等到暗杀任务用股票赚大钱 / Save for stock investments in assassination missions |
| 2 | 不升级角色技能 / Ignore skills | 通关后想做100%才发现技能不够 / Can't get 100% completion without max skills |
| 3 | 错过随机事件 / Miss random events | 在路上看到蓝点/问号——一定要去看看 / Always check blue dots/question marks on the map |
| 4 | 在线模式裸奔 / Play Online without armor | 弹药店买足防弹衣和零食——这是你的命 / Buy armor and snacks at Ammu-Nation — they save lives |
| 5 | 主线做完就删 / Delete after story | 通关后才是真正的开始——收集、金牌、线上 / Post-game is the real start — collectibles, gold medals, Online |
