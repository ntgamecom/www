| **1** | **主线优先做到"闪亮登场" / Rush to "The Jewel Store Job"** | 解锁抢劫任务系统——每个抢劫都可以选择不同策略，影响收益 / Unlocks heist system — each heist has multiple approaches affecting payout |

| **6** | **保险与改车 / Insurance & Modding** | 线上模式买的载具一定要上保险——否则永久丢失 / Always insure purchased vehicles in Online — otherwise permanent loss |

## 难易度说明 / Difficulty Guide

GTA 5没有传统难度选择——难度来自任务失败条件 / GTA 5 has no difficulty selection — difficulty comes from mission fail conditions.

| 难度来源 / Source | 说明 / Description |
|---|---|
| **任务检查点 / Checkpoints** | 大部分任务失败可从检查点重来 / Most missions have checkpoints on failure |
| **任务金牌 / Gold Medals** | 每个任务有金牌条件(时间/精准/特殊要求) / Each mission has gold medal criteria (time/accuracy/special) |
| **在线模式PVP / Online PVP** | 线上模式的PVP对战是真正的难度来源 / Real difficulty comes from Online PVP |
| **抢银行难度 / Heist Difficulty** | 抢劫可选手法和准备任务——不同选择影响难度和报酬 / Heists have multiple approach options affecting difficulty and payout |

**全金牌技巧 / Gold Medal Tips**:
- 大部分任务的精准度要求在70-80%——少用扫射多用爆头 / Most require 70-80% accuracy — aim for headshots, avoid spray
- 时间要求通常是"在XX分钟内完成"——熟悉路线后冲刺 / Time requirements are per-mission — practice routes then speedrun
- 部分任务的金牌条件需要换主角操作——注意切换时机 / Some gold conditions require specific character actions — time switches carefully
