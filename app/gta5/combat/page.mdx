# 战斗系统 / Combat System

GTA5的战斗系统结合了第三人称射击、近战格斗和三位主角的独特特殊能力 / GTA5's combat system combines third-person shooting, melee fighting, and three unique protagonist special abilities.

---

## 射击系统 / Shooting Mechanics

### 基础射击 / Basic Shooting

| 操作 / Action | 键位 / Key | 说明 / Description |
|---|---|---|
| 瞄准 / Aim | 鼠标右键 / Mouse 2 | 进入精确瞄准模式 / Enter precision aiming |
| 射击 / Fire | 鼠标左键 / Mouse 1 | 开火射击 / Fire weapon |
| 爆头 / Headshot | 瞄准头部射击 | 2倍伤害——秒杀普通敌人 / 2x damage — instant kill on normal enemies |
| 盲射 / Blind Fire | 不瞄准直接射击 | 精度下降但移动速度不减 / Reduced accuracy, full move speed |
| 掩体射击 / Cover Shooting | Q键贴墙后射击 | 减少受到的伤害 / Reduces incoming damage |
| 切换武器 / Switch Weapon | 鼠标滚轮 / Scroll | 快速切换主副武器 / Quick weapon swap |
| 子弹时间 / Bullet Time | 迈克尔专用能力 | 射击时时间减慢 / Slow motion while shooting |

**射击技巧 / Shooting Tips**:
- 爆头是最高效的杀敌方式——节约弹药 / Headshots are the most efficient way to conserve ammo
- 利用掩体系统可以减少大部分伤害 / Use the cover system to reduce most incoming damage
- 不同武器有不同的弹道和后座力模式 / Each weapon has different bullet trajectories and recoil patterns

### 武器类型 / Weapon Types

| 类别 / Category | 代表武器 / Examples | 特点 / Traits |
|---|---|---|
| **手枪 / Pistols** | AP手枪、重型左轮 / AP Pistol, Heavy Revolver | 单手——可在奔跑时射击 / One-handed — can fire while running |
| **冲锋枪 / SMGs** | 微型冲锋枪、冲锋枪 / Micro SMG, SMG | 高射速、近距离压制 / High fire rate, close-range suppression |
| **突击步枪 / Assault Rifles** | 特种卡宾、高级步枪 / Special Carbine, Advanced Rifle | 全能——中远距离精确射击 / Versatile — accurate at mid-long range |
| **霰弹枪 / Shotguns** | 泵动霰弹、突击霰弹 / Pump Shotgun, Assault Shotgun | 近距离一发致命——清理房间最佳 / 1-hit kill at close range |
| **狙击枪 / Sniper Rifles** | 重型狙击、射手步枪 / Heavy Sniper, Marksman Rifle | 超远程击杀——用于暗杀和压制 / Long-range elimination |
| **重型武器 / Heavy Weapons** | 火箭筒、加特林 / RPG, Minigun | 大范围破坏——对付载具和直升机 / Area destruction — anti-vehicle |
| **近战 / Melee** | 棒球棍、小刀、电击枪 / Bat, Knife, Stun Gun | 无声击杀——潜行时最佳 / Silent kills — best for stealth |
| **投掷 / Throwables** | 手雷、燃烧弹、粘性炸弹 / Grenade, Molotov, Sticky Bomb | 范围伤害——对付聚集的敌人 / Area damage — against grouped enemies |

**武器改装 / Weapon Modifications**:

| 配件 / Attachment | 效果 / Effect | 价格 / Price |

| 消音器 / Suppressor | 消除枪声和枪口火焰——潜行必备 / Silent shots — stealth essential | $5,000-$15,000 |
| 瞄准镜 / Scope | 放大视野——远距离精度提升 / Zoom magnification — long-range accuracy | $2,000-$10,000 |
| 加长弹匣 / Extended Mag | 弹容量翻倍——减少换弹次数 / Double magazine capacity | $3,000-$8,000 |
| 握把 / Grip | 减少后座力——提升连续射击精度 / Reduced recoil | $1,500-$5,000 |
| 枪口制动器 / Muzzle Brake | 进一步减少后座力 / Further recoil reduction | $2,000-$6,000 |
| 特殊弹药 / Special Ammo | 曳光弹、穿甲弹等 / Tracer, armor-piercing, etc. | $500-$2,000 |

| 弹药类型 / Ammo Type | 效果 / Effect |
|---|---|
| 曳光弹 / Tracer Rounds | 弹道可见——更容易调整射击 / Visible bullet trajectory — easier aim correction |

## 近战系统 / Melee System

### 近战攻击类型 / Melee Attack Types

| 攻击 / Attack | 操作 / Action | 效果 / Effect |
|---|---|---|
| 普通攻击 / Basic Attack | F键 | 基础近战伤害 / Basic melee damage |
| 重击 / Heavy Attack | 蓄力F键 | 2倍伤害——击倒效果 / 2x damage — knockdown |
| 冲刺攻击 / Sprint Attack | 冲刺中按F | 带冲击力的飞踢或冲撞 / Sprinting kick or tackle |
| 暗杀 / Stealth Kill | 潜行接近按F | 一击必杀——不触发警报 / Instant kill — no alarm triggered |
| 反制 / Counter | 近战时及时按F | 格挡后反击——对拳击任务有效 / Block then counter |

**力量属性影响 / Strength Impact**:
- 力量满级后近战伤害翻倍 / Max Strength doubles melee damage
- 解锁更快的出拳速度和连击 / Unlocks faster punch speed and combos
- 体力消耗减少 / Reduced stamina consumption

## 特殊能力系统 / Special Abilities

每位主角拥有独特的特殊能力——消耗能力条 / Each protagonist has a unique special ability — consumes the ability bar.

### 能力对比 / Ability Comparison

| 角色 / Character | 能力 / Ability | 效果 / Effect | 最佳使用场景 / Best Use |
|---|---|---|---|
| **迈克尔 / Michael** | 子弹时间 / Bullet Time | 战斗中时间减慢——精准射击 / Slow motion — precision shooting | 面对大量枪手时 / Facing large groups of shooters |
| **富兰克林 / Franklin** | 驾驶专注 / Driving Focus | 驾驶时时间减慢——轻松过弯 / Slow motion while driving | 赛车和追逐任务 / Races and chase missions |
| **崔佛 / Trevor** | 狂战士 / Berserker | 伤害减半、输出翻倍 / Take half damage, deal double | 近战混战——正面突破 / Melee brawls — direct assault |

**能力恢复 / Ability Recharge**:
- 迈克尔：完成特殊动作(爆头、开车特技) / Michael: special actions (headshots, stunts)
- 富兰克林：危险驾驶(近距超车、极速行驶) / Franklin: risky driving (near-misses, top speed)
- 崔佛：造成伤害和承受伤害 / Trevor: dealing and receiving damage
- 通用恢复：击杀敌人、完成任务 / Universal: killing enemies, completing missions

## 通缉系统 / Wanted System

### 通缉等级 / Wanted Levels

| 等级 / Level | 星星数 / Stars | 警方反应 / Police Response |
|---|---|---|
| 1 | ★ | 一名警车追逐——轻微犯罪 / One patrol car — minor crime |
| 2 | ★★ | 2-3辆警车——开始直升机支援 / 2-3 cars — helicopter support begins |
| 3 | ★★★ | 警车+直升机——设路障 / Cars + helicopter — roadblocks deployed |
| 4 | ★★★★ | 全城警力——大量直升机+警车 / City-wide — many helicopters and cars |
| 5 | ★★★★★ | FIB出动——直升机+SUV特工队 / FIB deployed — choppers + SUV agents |
| 6 | ★★★★★★ | 军队介入——坦克+武装直升机 / Military — tanks + attack helicopters |

**消除通缉的方法 / Ways to Lose the Wanted Level**:

| 方法 / Method | 说明 / Description |
|---|---|
| **逃离搜索区 / Escape Search Zone** | 离开通缉圈——等待倒计时消失 / Leave the red search zone — wait for timer |
| **改车油漆 / Respray Vehicle** | 在改车王改漆——立即消除通缉 / Pay N Spray — instantly lose wanted level |
| **躲藏 / Hide** | 离开车辆躲在角落——等搜索结束 / Leave vehicle and hide — wait for search to end |
| **贿赂警察 / Bribe Cops** | 部分特殊事件中可以贿赂 / Some special events allow bribery |
| **脱衣舞俱乐部 / Strip Club** | 进入后等通缉消除（仅限1-2星）/ Enter and wait (level 1-2 only) |
| **穿警服 / Police Uniform** | 部分任务获得的警服可降低通缉 / Police uniform from missions reduces wanted level |

**通缉注意事项 / Wanted System Notes**:
- 通缉状态下不能保存游戏 / Cannot save while wanted
- 直升机探测范围很大——优先打掉直升机 / Helicopters have large detection range — take them out first
- 通缉共享给当前操作的角色——切换角色有时能重置位置 / Wanted status follows current character — switching may reset position

## 战斗策略 / Combat Strategies

### 不同情况对策 / Situation-Specific Tactics

| 场景 / Scenario | 推荐策略 / Recommended Tactic |
|---|---|
| **室内近战 / Indoor CQB** | 霰弹枪+手雷——先扔闪后突入 / Shotgun + grenade — flash then enter |
| **野外枪战 / Open Field** | 狙击消灭远处目标——步枪清理剩余 / Snipe distant targets — rifle clean remaining |
| **驾车追逐 / Vehicle Chase** | 富兰克林驾驶专注——副驾射击轮胎 / Franklin's driving focus — shoot tires from passenger seat |
| **直升机追击 / Helicopter Chase** | 狙击手或火箭筒打直升机——重型左轮也可 / Sniper or RPG — heavy revolver works too |
| **潜行渗透 / Stealth Infiltration** | 消音武器+潜行击杀——避免触发警报 / Suppressed weapons + stealth kills — avoid alarms |
| **正面强攻 / Direct Assault** | 崔佛狂战士+重型护甲——硬碰硬 / Trevor berserker + heavy armor — brute force |

**团队切换战术 / Character Switching Tactics**:
- 任务中可以在三位主角间切换 / Switch between protagonists during missions
- 将一名主角留在安全位置作为后备 / Keep one character at safe position as backup
- 切换后可命令AI控制的主角自动战斗 / AI-controlled characters auto-fight after switching
- 注意：主角死亡任务即失败 / Note: protagonist death = mission failure
