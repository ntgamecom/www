# 线上模式 / GTA Online

## 总览 / Overview

GTA Online是GTA 5的多人模式——2013年推出至今持续更新 / Online multiplayer mode of GTA 5 — continuously updated since 2013.

**特点 / Features**:
- 1-30人自由畅玩洛圣都 / Free roam in LS with 1-30 players
- 差事任务、抢劫、竞速、PVP改版多种模式 / Missions, heists, races, PVP modes
- 持续的免费DLC——新内容不断加入 / Constant free DLC — new content added regularly
- 独立的经济系统——角色与主线完全分离 / Independent economy — characters completely separate from story

**重点提醒 / Key Notes**:
1. 线上模式的金币(GTA$)和主线完全不互通 / GTA$ in Online is completely separate from story mode
2. 鲨鱼卡可以用现实货币直接购买GTA$ / Shark Cards let you buy GTA$ with real money
3. **游戏内的恶意玩家很多**——建议开被动模式 / Lots of griefers — use Passive Mode
4. 差事任务的匹配速度取决于平台在线人数 / Matchmaking speed depends on platform population

### 全福银行差事 / The Fleeca Heist

| 信息 / Info | 说明 / Description |
|---|---|
| **人数 / Players** | 2 |

|---|---|

| **准备任务 / Preps** | 5个 |

### 太平洋银行抢劫 / Pacific Standard Job

| 信息 / Info | 说明 / Description |
|---|---|
| **人数 / Players** | 4 |
| **准备任务 / Preps** | 5个 |
| **最终任务 / Finale** | 太平洋银行抢劫 / Pacific Standard Bank |
| **推荐报酬 / Payout** | $500,000-$1,000,000 |

**核心提示 / Tips**:

2. 人质管理——不让警察突入 / Manage hostages to delay police
3. 逃跑路线：跳上摩托——一路到码头 / Escape: bikes to the docks
4. **最重要**：所有成员要集合拿钱——不要单独行动 / Everyone must collect cash together

### 佩里科岛抢劫 / The Cayo Perico Heist

**目前最快单人赚钱方法** / Currently the fastest solo money-maker.

| 信息 / Info | 说明 / Description |
|---|---|
| **人数 / Players** | 1-4 (单人完全可行) / Solo-able |
| **准备任务 / Preps** | 6-7个 / 6-7 |
| **最终任务 / Finale** | 潜入佩里科岛——偷走目标物品 / Infiltrate Cayo Perico — steal the target |
| **推荐报酬 / Payout** | $900,000-$1,500,000 (单人) / Solo |

**佩里科岛单人攻略 / Solo Cayo Perico Guide**:

**准备阶段 / Prep Phase**:
1. 买Kosatka潜艇 / Buy Kosatka submarine
2. 侦查阶段——飞到岛上用摄像机拍目标 / Recon phase — fly to island, scope targets
3. 准备任务：割炬、炸药、武器、船等 / Preps: blowtorch, explosives, weapons, vehicle

**最终任务 / Finale (Solo Route)**:
1. 从排水管道潜入——割炬进入 / Enter via drainage tunnel — use blowtorch
2. 沿地下通道直接进入金库 / Follow underground tunnel to vault
3. 偷走主要目标(科卡因或文件) / Steal primary target (cocaine or files)
4. 原路返回——游泳逃离 / Return the same way — swim escape
5. 时间：熟练后12-15分钟 / Time: 12-15 minutes once practiced

**最优化路线 / Optimal Loot**:
- 主要目标：粉钻(最高价) / Pink Diamond (highest value)
- 次要目标：桌上有可卡因优先拿 / Secondary: cocaine from tables

## PVP技巧 / PVP Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **被动模式 / Passive Mode** | 不想打的时候打开——无敌 / Invincibility toggle |
| **被粘弹追 / Stickies** | 听到粘弹黏上的声音——立刻跳车/跳伞 / Jump out/parachute when sticky attaches |
| **地堡/公寓快传 / Quick Teleport** | 在互动菜单中进入地堡/夜总会等可以从任何地方快速传送 / Enter properties from Interaction Menu to teleport |
| **CEO保镖 / CEO Bodyguard** | 招募保镖可以获得额外血量和护甲 / Bodyguards get bonus HP and armor |
| **空中卡位 / Jet Camping** | 连续用Hydra/天煞空中轰炸——很难被反制 / Continuous jet bombing — very hard to counter |
| **太空行者 / Micro SMG** | 开车时的最佳副武器 / Best drive-by weapon |
| **瞄准模式 / Aim Assist** | 辅助瞄准是全平台特有的——善用/防着 / Assist aim is platform feature |

## 快速赚钱路径 / Fastest Money Path

### 单人 / Solo Path

1. 投资Kosatka ($2.2M) → 刷佩里科岛 (1.5M/次, 1小时/次) / Buy Kosatka → Cayo Perico (1.5M/run, 1hr)
2. 买夜总会 → 绑定地堡+可卡因+大麻 → 每天收 $50K+ / Nightclub → link bunker+coke+weed → $50K/day
3. 买地堡 → 升级设备 → 满库存销售 ($250K/次) / Bunker → upgrade → full sale ($250K/run)
4. 买恐霸 → 刷VIP任务 ($15K/5分钟) / Terrorbyte → VIP missions ($15K/5min)

### 团队 / Team Path

1. 全做一遍简单抢劫(全福→人道→太平洋) / Do all original heists (Fleeca → Humane → Pacific)
2. 末日豪劫三章 ($1.2M/章) / Doomsday Act I-III ($1.2M/act)
3. 团队佩里科岛 — 每人 $1.5M/次 / Team Cayo Perico — $1.5M each/run
4. 钻石赌场抢劫 — 难度高收益高 / Diamond Casino Heist — high risk, high reward
