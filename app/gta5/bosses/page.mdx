# BOSS攻略 / Boss Fights Guide

GTA5没有传统意义上的Boss战——但存在多个高难度战斗场景——需要策略和装备准备 / GTA5 has no traditional boss fights — but features several high-difficulty combat scenarios requiring strategy and gear.

---

## 高难度任务战斗 / Hard Mission Encounters

### 全福银行差事 / The Fleeca Job (Online)

**推荐等级 / Recommended Level**: 10+
**推荐装备 / Recommended Gear**: 防弹衣+零食+穿甲手枪 / Armor + snacks + AP Pistol

| 阶段 / Phase | 敌人数量 / Enemies | 难点 / Difficulty |
|---|---|---|
| 接近银行 / Approach | 2-3 | 潜行暗杀守卫 / Stealth kill guards |
| 钻金库 / Drill Vault | 0 | 保护钻机不被警察打断 / Protect the drill |
| 逃脱 / Escape | 无限警察 / Infinite cops | 开车到目标点——躲避通缉 / Drive to objective — evade wanted level |

**策略 / Strategy**:
1. 驾驶员角色负责查看GPS路线 / Driver role handles GPS routing
2. 钻金库期间——枪手守住入口 / During drilling — gunner holds the entrance
3. 逃脱时走地下隧道——减少警察拦截 / Escape through underground tunnels

### 太平洋银行差事 / Pacific Standard Heist (Online)

**推荐等级 / Recommended Level**: 50+
**推荐装备 / Recommended Gear**: 重型防弹衣+特种卡宾 / Heavy armor + Special Carbine

| 阶段 / Phase | 敌人类型 / Enemy Type | 数量 / Count |
|---|---|---|
| 冲入银行 / Breach | 安保 + 警员 / Security + Police | 8-10 |
| 装钱 / Bag Money | 无限警察 / Infinite cops | 持续增援 / Continuous |
| 摩托车逃脱 / Bike Escape | 警察追兵 / Police pursuit | 无限 / Infinite |
| 泛舟河上 / Boat Escape | 直升机 + 警艇 / Chopper + Police boats | 各有3-4个 / 3-4 each |

**策略 / Strategy**:
1. 进入银行后立即清场——避免腹背受敌 / Clear the bank lobby immediately — avoid pincer attacks
2. 装钱时一人守入口——一人守后门 / During bagging — one watches front, one watches back
3. 摩托车阶段不要跟随导航——走铁路更安全 / Skip GPS on bikes — take the railway instead
4. 泛舟阶段集中火力打直升机 / Focus fire on the helicopter during boat phase

### 佩里科岛抢劫 / Cayo Perico Heist (Online)

**推荐等级 / Recommended Level**: 100+
**推荐装备 / Recommended Gear**: 消音武器+防水袋 / Suppressed weapons + drainage tool

| 区域 / Zone | 守卫数量 / Guards | 危险等级 / Danger |
|---|---|---|
| 机场 / Airport | 4-6 | 中等 / Medium |
| 主码头 / Main Dock | 3-5 | 低 / Low |
| 北码头 / North Dock | 2-3 | 低 / Low |
| 庄园外围 / Compound Outside | 6-8 | 高 / High |
| 庄园内部 / Compound Inside | 4-6 | 极高 / Very High |
| 下水道 / Drainage Tunnel | 1 | 低 / Low |

**单人潜行策略 / Solo Stealth Strategy**:
1. 从北侧下水道进入——直通庄园内部 / Enter via north drainage tunnel — straight to compound
2. 按固定路线暗杀守卫——记住巡逻时间 / Assassinate guards on fixed routes — memorize patrol timings
3. 偷取主目标后从悬崖跳海逃脱 / Take primary target — cliff jump to escape
4. 全程不触发警报可额外获得精英奖励 / No alarms triggered = elite bonus

## 故事模式关键战斗 / Story Mode Key Battles

### 北扬克顿抢劫 / North Yankton Prologue

| 要素 / Element | 说明 / Description |
|---|---|
| 敌人类型 / Enemy | 警员 + 巡逻车 / Police + Patrol cars |
| 地形 / Terrain | 雪地——能见度低 / Snow — low visibility |
| 特殊机制 / Gimmick | 雪地打滑——车辆难控制 / Snowy roads — poor vehicle traction |

**建议 / Tips**:
- 使用第一人称提高瞄准精度 / Use first-person for better aim
- 利用雪堆作为掩体——白色背景容易被发现 / Use snowbanks as cover
- 开车逃跑时提前减速——雪地刹车距离翻倍 / Brake early in snow — stopping distance doubles

### 无伤击杀精英敌人技巧 / Elite Enemy Tips

| 掩体切换 / Cover Switching | Q键贴墙后A/D移动到相邻掩体 / Q for cover then A/D to adjacent |

| 载具碾压 / Vehicle Run-Over | 用高速载具直接碾压步兵 / Run over foot soldiers with vehicles |
