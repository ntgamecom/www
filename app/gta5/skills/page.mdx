# 技能系统 / Skills System

GTA5拥有六项角色技能——每项最高100%——提升后获得实战加成 / GTA5 has six character skills — each caps at 100% — providing combat and gameplay bonuses.

---

## 技能总览 / Skills Overview

### 六项技能对比 / Six Skills Comparison

| 技能 / Skill | 满级效果 / Max Effect | 升级速度 / Level Speed | 是否关键 / Critical |
|---|---|---|---|
| **体力 / Stamina** | 无限冲刺 / Unlimited sprint | 中等 / Medium | 探索必备 / Essential for exploration |
| **射击 / Shooting** | 减少后座、弹道集中 / Less recoil, tighter spread | 慢 / Slow | 战斗核心 / Core for combat |
| **力量 / Strength** | 近战伤害翻倍 / Double melee damage | 中等 / Medium | 近战流必备 / Essential for melee |
| **潜行 / Stealth** | 潜行移动速度翻倍 / Double stealth speed | 非常慢 / Very Slow | 全100%必须 / Required for 100% |
| **飞行 / Flying** | 飞行颠簸减少 / Smoother aircraft handling | 快 / Fast | 飞行任务需要 / Needed for flight missions |
| **驾驶 / Driving** | 转向控制提升 / Better turning control | 快 / Fast | 全游戏通用 / Universally useful |

| 打高尔夫 / Golf | 低 / Low | 仅微量提升 / Minimal gain |

骑自行车在海滨大道来回骑行——约20分钟游戏内时间即可满级 / Bike back and forth on the beach boardwalk — about 20 minutes in-game to max.

## 射击技能 / Shooting

### 升级方法 / Leveling Methods

| 方法 / Method | 效率 / Efficiency | 说明 / Description |
|---|---|---|
| 弹药店射击场 / Ammu-Nation Range | 极佳 / Excellent | 花钱买弹药——用机枪扫射 / Spend money on ammo — spray with SMG |
| 实战射击 / Combat | 中等 / Medium | 任务中爆头击杀 / Headshot kills in missions |
| 飞行学校射击 / Flight School Range | 低 / Low | 仅少数任务涉及 / Only a few missions |

**射击场快速升级 / Shooting Range Fast Leveling**:
1. 进入任意弹药店射击场 / Enter any Ammu-Nation shooting range
2. 选择"目标射击"模式——使用冲锋枪 / Select "Target Shooting" — use SMG
3. 对着所有靶子持续扫射——不要求精度 / Continuously fire at all targets — accuracy not required
4. 每轮结束后立即重新开始 / Restart immediately after each round
5. 约10-15分钟从0升到100% / Takes about 10-15 minutes from 0 to 100%

**满级效果详解 / Max Level Effects**:
| 效果 / Effect | 数值 / Value |
|---|---|
| 后座力减少 / Recoil Reduction | 约40% / ~40% |
| 弹道扩散减少 / Bullet Spread Reduction | 约50% / ~50% |
| 换弹速度提升 / Reload Speed Increase | 约20% / ~20% |
| 盲射精度提升 / Hip-Fire Accuracy | 约30% / ~30% |

## 力量技能 / Strength

### 升级方法 / Leveling Methods

| 方法 / Method | 效率 / Efficiency | 说明 / Description |
|---|---|---|
| 近战攻击NPC / Melee Fighting NPCs | 极佳 / Excellent | 在码头找路人无限打斗 / Fight random NPCs at the docks |
| 打网球 / Tennis | 极佳 / Excellent | 一局网球可升10-15% / One tennis match = 10-15% |
| 打高尔夫 / Golf | 中等 / Medium | 挥杆力量决定提升量 / Swing power determines gain |
| 铁人三项 / Triathlons | 中等 / Medium | 兼具体力和力量提升 / Combines stamina and strength |

**最快升级方法 / Fastest Method**:
1. 去洛圣都码头区域 / Go to the Los Santos docks area
2. 找一个落单的NPC——持续按F攻击 / Find a lone NPC — continuously press F to attack
3. NPC倒地后等待站起——继续攻击 / Wait for them to get up — attack again
4. **注意**：不要击杀NPC——击杀后会刷新延迟 / Note: don't kill them — death causes respawn delay
5. 约15分钟可满级 / Takes about 15 minutes to max

**满级效果详解 / Max Level Effects**:
| 效果 / Effect | 数值 / Value |
|---|---|
| 近战伤害提升 / Melee Damage Increase | 100% (翻倍) / 100% (double) |
| 出拳速度提升 / Punch Speed | 约50% / ~50% |
| 体力消耗减少 / Stamina Cost Reduction | 约30% / ~30% |
| 翻越障碍速度 / Vault Speed | 约20% / ~20% |

## 潜行技能 / Stealth

### 升级方法 / Leveling Methods

| 方法 / Method | 效率 / Efficiency | 说明 / Description |
|---|---|---|
| 潜行击杀 / Stealth Kills | 中等 / Medium | 在敌人背后暗杀 / Assassinate enemies from behind |
| 潜行移动 / Stealth Movement | 慢 / Slow | 按住Ctrl潜行移动 / Hold Ctrl and move while sneaking |

**快速满级方法 / Fastest Method**:
1. 找一个有大量敌人且可反复刷的区域——如叛军据点 / Find an area with respawning enemies — rebel outposts
2. 潜行到敌人背后按F暗杀 / Stealth approach and press F to assassinate
3. 被发现就逃跑重新潜行 / If detected, escape and re-enter stealth
4. 也可在洛圣都国际机场大厅内潜行绕圈——不会被攻击 / Alternatively, stealth walk inside LSIA terminal — no attacks

**潜行技巧 / Stealth Tips**:
- 潜行时移动速度取决于技能等级 / Movement speed in stealth depends on skill level
- 满级后潜行速度与正常行走速度相近 / Max level stealth speed is similar to normal walk speed
- 夜间的潜行效果更好——减少被发现的概率 / Nighttime stealth is more effective — reduced detection
- 穿深色服装降低被发现概率 / Dark clothing reduces detection chance

| 高速驾驶 / High-Speed Driving | 快 / Fast | 在高速公路上全速行驶 / Drive at top speed on highways |

## 技能与线上模式 / Skills in GTA Online

GTA Online的技能系统与故事模式不同——通过等级解锁能力 / GTA Online skills differ from story mode — abilities unlock via level.

| 线上能力 / Online Ability | 解锁方式 / Unlock Method |
|---|---|
| 最大生命值 / Max HP | 每升20级提升一次 / Every 20 levels |
| 呼吸时间 / Breath Time | 潜水面罩或等级解锁 / Rebreather or level |
| 驾驶能力 / Driving | 等级自然提升 / Natural level progression |
| 飞行能力 / Flying | 在飞行学校完成课程 / Flight school completion |
