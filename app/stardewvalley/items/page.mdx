# 物品与收集 / Items & Collectibles

## 钓鱼全图鉴 / Fishing Guide

星露谷共有 **70+种鱼类**——分布在河流/湖泊/海洋/矿井/沙漠/姜岛 / 70+ fish species — rivers/lakes/ocean/mine/desert/Ginger Island.

### 钓鱼技巧 / Fishing Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **鱼竿升级 / Rod Upgrades** | 竹竿(免费)→玻璃鱼竿(1800金)→铱星鱼竿(7500金) / Bamboo(free)→Fiberglass(1800g)→Iridium(7500g) |
| **鱼饵 / Bait** | 虫子(挖虫肉+1木头制作) / Bugs (crafted from bug meat) |
| **钓具 / Tackle** | 铱星杆可以装钓具——陷阱钩(减少鱼逃脱)最有用 / Iridium Rod uses tackle — Trap Bobber is best |
| **钓鱼小游戏** | 按住=上浮 松开=下沉——目标是让绿条包住鱼 / Hold = rise, release = fall — keep green bar on the fish |
| **钓鱼等级影响** | 等级越高——绿条越大——鱼越容易钓 / Higher level = bigger green bar |

## 金色核桃 / Golden Walnuts

姜岛有 **130个金色核桃**——用于解锁姜岛设施 / Ginger Island has 130 Golden Walnuts — unlock island facilities.

| 获取方式 / Method | 数量 / Count |
|---|---|
| 岛上各处拾取 / Pick up around the island | 10+ |
| 挖地 / Till soil | 10+ |
| 钓鱼 / Fish (特定位置) | 10+ |
| 打怪掉落 / Monster drops | 20+ |
| 解谜 / Puzzles | 30+ |
| 齐先生的核桃房 / Mr Qi's Walnut Room | 50+ |

**核桃解锁顺序 / Walnut Unlock Priority**:
1. 鹦鹉特快(10个)——快速旅行 / Parrot Express (10) — fast travel
2. 农场小屋(20个)——可以住在姜岛种田 / Island Farmhouse (20) — stay on island
3. 姜岛商人(10个)——交易特殊物品 / Island Trader (10) — special trades
4. 齐先生核桃房(100个)——解锁齐先生任务 / Mr Qi's Walnut Room (100) — Qi quests
