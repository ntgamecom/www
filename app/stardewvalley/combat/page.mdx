# 战斗系统 / Combat System

星露谷物语的战斗主要集中在矿井、骷髅洞穴和火山地牢。虽然战斗不是游戏的核心主题——但了解战斗系统对深入探索至关重要 / Stardew Valley's combat is primarily in the Mines, Skull Cavern, and Volcano Dungeon. While combat isn't the game's central focus — understanding the combat system is essential for deep exploration.

## 战斗技巧 / Combat Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **武器格挡** | 剑类武器右键可以格挡——减少80%伤害并击退敌人。面对高伤怪(暗影兵/铱矿蟹)时极为有用 / Sword right-click blocks — 80% damage reduction + knockback. Essential vs high-damage enemies |
| **走砍技巧** | 攻击→走位→攻击的循环——不要站在原地连砍。打一下走一下可以有效避免被包围 / Attack → reposition → attack loop — don't stand still. Move after each hit to avoid being surrounded |
| **楼梯策略** | 在骷髅洞穴中，如果被大量怪物包围——直接铺楼梯下去。活着比打怪重要 / In Skull Cavern, if surrounded by too many monsters — place a ladder and descend. Survival over killing |
| **时间管理** | 矿井和骷髅洞穴中，时间在房间内继续走——但切换楼层时暂停。规划好下矿时间，晚上12点前要准备离开 / Time continues in rooms but pauses on floor transition. Plan mining time — prepare to leave before midnight |
| **食物准备** | 下矿前带够食物——奶酪、沙拉、紫蘑菇。不要带会加属性的食物(速度/攻击力等)——战斗中吃普通回血食物就行 / Bring enough food before mining — Cheese, Salad, Purple Mushroom. Don't waste stat-boosting food mid-combat |

## 防御策略 / Defense Strategy

| 策略 / Strategy | 说明 / Description |
|---|---|
| **走位第一 / Positioning First** | 永远不要站定输出——保持移动等敌人攻击结束再反击 / Never stand still — keep moving, wait for enemies to finish their attack then counter |
| **卡门打法 / Doorway Tactics** | 站在门口攻击——敌人一次只能进一个——避免被围殴 / Stand in doorways — enemies enter one at a time — avoids being swarmed |
| **地形利用 / Terrain Use** | 利用岩石、水体等障碍物——让敌人绕路——你趁机输出 / Use rocks, water as obstacles — enemies path around — DPS while they path |
| **楼梯战术 / Staircase Tactics** | 在骷髅洞穴中——如果被包围直接铺楼梯走——不要贪 / In Skull Cavern — if surrounded, staircase out immediately — don't be greedy |
| **放风筝 / Kiting** | 对近战敌人——往后退同时攻击——边退边打——永远保持距离 / Against melee enemies — backpedal while attacking — maintain distance at all times |
| **优先级击杀 / Priority Kills** | 先杀远程敌人(骷髅法师、辣椒霸王龙)——再处理近战——最后处理坦克 / Kill ranged first (Skeleton Mage, Pepper Rex) — then melee — then tanks |
