# 技能系统 / Skills System

星露谷物语有5种技能——耕种、采矿、采集、钓鱼、战斗。每个技能有10个等级，5级和10级可以选择专精方向，专精选择决定了你的游戏风格 / Stardew Valley has 5 skills — Farming, Mining, Foraging, Fishing, Combat. Each has 10 levels with specialization choices at levels 5 and 10 that define your playstyle.

---

## 耕种 / Farming

| 等级 / Level | 解锁 / Unlock |
|---|---|
| **1级** | 稻草人、初级肥料配方 / Scarecrow, Basic Fertilizer recipe |
| **2级** | 石栏、蛋黄酱机配方 / Stone Fence, Mayonnaise Machine recipe |
| **3级** | 蜂箱、速生肥料配方 / Bee House, Speed-Gro recipe |
| **4级** | 熏鱼机、高级肥料配方 / Smoker, Quality Fertilizer recipe |
| **5级** | **选择专精**：农耕人(农产品价值+10%)或牧场主(动物产品价值+20%) / **Choose**: Tiller (+10% crop value) or Rancher (+20% animal product value) |
| **6级** | 芝士机、织布机、硬木栏配方 / Cheese Press, Loom, Hardwood Fence recipe |
| **7级** | 树木肥料配方 / Tree Fertilizer recipe |
| **8级** | 产油机、酒桶、高级洒水器配方 / Oil Maker, Keg, Quality Sprinkler recipe |
| **9级** | 种子机、铱制洒水器配方 / Seed Maker, Iridium Sprinkler recipe |
| **10级** | **二次专精**：农耕人→农耕大师(农产品价值再+10%)或工匠(加工品价值+40%) / 牧场主→牧羊人(棚内动物繁殖加速)或养鸡人(动物产品价值+20%) / Tiller→Artisan(+40% artisan goods) or Agriculturist(+10% all crops) / Rancher→Shepherd(barn animals) or Coopmaster(coop animals) |

**推荐**: 5级选农耕人→10级选工匠。工匠路线(酿酒+果酱)是游戏中最赚钱的路线 / **Recommendation**: Tiller at 5 → Artisan at 10. Artisan route (wine + preserves) is the most profitable in the game.

## 采矿 / Mining

| 等级 / Level | 解锁 / Unlock |
|---|---|
| **1级** | 樱桃炸弹配方 / Cherry Bomb recipe |
| **2级** | 楼梯配方(99石头)/ Ladder recipe (99 stone) |
| **3级** | 矿工特供配方 / Miner's Treat recipe |
| **4级** | 熔炼炉升级——铜→铁 / Smelter upgrade — copper → iron |
| **5级** | **选择专精**：矿工(每矿脉+1矿石)或地质学家(宝石出现率x2)/ **Choose**: Miner (+1 ore per vein) or Geologist (2x gem chance) |
| **6级** | 超级炸弹配方 / Mega Bomb recipe |
| **7级** | 熔炼炉升级——铁→金 / Smelter upgrade — iron → gold |
| **8级** | 铱矿熔炼配方 / Iridium Smelting recipe |
| **9级** | 宝石复制机、加料熔炉配方 / Crystalarium, Furnace upgrade |
| **10级** | **二次专精**：矿工→铁匠(金属块价值+50%)或勘探者(煤出现率x2)/ 地质学家→宝石学家(宝石价值+30%)或挖掘者(晶球出现率x2)/ Miner→Blacksmith(+50% metal bar value) or Prospector(2x coal) / Geologist→Gemologist(+30% gem value) or Excavator(2x geodes) |

**推荐**: 5级选矿工(前期缺矿石)——10级选勘探者(煤永远不够用) / **Recommendation**: Miner at 5 (early ore shortage) → Prospector at 10 (coal always scarce).

## 采集 / Foraging

| 等级 / Level | 解锁 / Unlock |
|---|---|
| **1级** | 野生种子(春季)配方 / Wild Seeds (Spring) recipe |
| **2级** | 野生种子(夏季)配方 / Wild Seeds (Summer) recipe |
| **3级** | 窑炉、野生种子(秋季)配方 / Charcoal Kiln, Wild Seeds (Fall) recipe |
| **4级** | 野生种子(冬季)配方 / Wild Seeds (Winter) recipe |
| **5级** | **选择专精**：采集者(采集时可获得双倍)或森林之子(所有树掉落更多木材)/ **Choose**: Gatherer (2x foraged items) or Forester (more wood from trees) |
| **6级** | 避雷针、野炊工具配方 / Lightning Rod, Cookout Kit recipe |
| **7级** | 野生种子(全年)配方——可以在任何季节播种 / Wild Seeds (All Seasons) recipe — plantable any season |
| **8级** | 传送图腾：海滩、山脉、农场配方 / Warp Totems: Beach, Mountains, Farm recipe |
| **9级** | 雨水图腾配方 / Rain Totem recipe |
| **10级** | **二次专精**：采集者→植物学家(采集品总是最高品质)或追踪者(地图显示采集品位置)/ 森林之子→伐木工(所有树掉落硬木)或护林人(树掉落更多种子) / Gatherer→Botanist (always gold quality) or Tracker (show on map) / Forester→Lumberjack (trees drop hardwood) or Tapper (syrup value +25%) |

**推荐**: 5级选采集者(双倍采集收益高)→10级选植物学家(采集品全金星的收益在早期很大) / **Recommendation**: Gatherer at 5 (double foraged items) → Botanist at 10 (gold quality always sells well).

## 钓鱼 / Fishing

| 等级 / Level | 解锁 / Unlock |
|---|---|
| **1级** | 训练用鱼竿 / Training Rod |
| **2级** | 玻璃纤维鱼竿(可装鱼饵)/ Fiberglass Rod (bait compatible) |
| **3级** | 蟹笼配方、海鲜汤配方 / Crab Pot recipe, Chowder recipe |
| **4级** | 碳素鱼竿(Carbon Fiber Rod——可装鱼饵+浮标)/ Carbon Fiber Rod (bait + bobber) |
| **5级** | **选择专精**：渔夫(鱼价值+25%)或捕蟹者(蟹笼成本-25%) / **Choose**: Fisher (+25% fish value) or Trapper (-25% crab pot cost) |
| **6级** | 回收机、旋式鱼饵配方 / Recycling Machine, Spinner recipe |
| **7级** | 软木塞浮标、寻宝者浮标、铅制浮标配方 / Cork Bobber, Treasure Hunter, Lead Bobber recipe |
| **8级** | 鱼饵制造机配方 / Bait Maker recipe |
| **9级** | 磁铁浮标、陷阱浮标配方 / Magnet Bobber, Trap Bobber recipe |
| **10级** | **二次专精**：渔夫→垂钓大师(鱼价值+50%)或海盗(宝箱出现率x2)/ 捕蟹者→水手(蟹笼不再出垃圾)或诱饵大师(蟹笼不需要饵料) / Fisher→Angler(+50% fish value) or Pirate(2x treasure) / Trapper→Mariner(no junk) or Luremaster(no bait needed) |

**推荐**: 5级选渔夫(前期钱很重要)→10级选垂钓大师(鱼价值+50%) / **Recommendation**: Fisher at 5 (early gold) → Angler at 10 (+50% fish value).

## 综合技能策略 / Overall Skill Strategy

| 早期重点(第1年) / Early Focus (Year 1) | 中期重点(第2年) / Mid Focus (Year 2) | 后期 / Late Game |
|---|---|---|
| **耕种5级+钓鱼5级**——赚钱基础 / Farming 5 + Fishing 5 — income base | 所有技能到10级——解锁所有配方 / All skills to 10 — unlock all recipes | 工匠路线全面铺开——酿酒+果酱+腌菜 / Full artisan route — wine + preserves + pickles |
| 采矿尽早到5级——解锁炸弹配方 / Mining to 5 early — unlock bombs | 采集7级+耕种6级——自动浇水/永久种子 / Foraging 7 + Farming 6 — sprinklers + seeds | 矿井+骷髅洞穴无限深挖——铱矿/钻石/自动采集 / Infinite mine + Skull Cavern — iridium/gems/automation |

**技能升级速度**:
- 耕种: 慢——主要靠收获作物。每个季节只能收几次 / Slow — mainly from harvesting crops
- 采矿: 中——矿石+宝石+晶球。炸矿效率高 / Medium — ore + gems + geodes. Bombs speed it up
- 采集: 快——每天都在涨。捡野果+砍树 / Fast — daily gains. Picking berries + chopping wood
- 钓鱼: 最快——高难度鱼(如春季鱼王)一把+100+经验 / Fastest — difficult fish (Spring King) give 100+ XP per catch
- 战斗: 中——杀怪。矿井100层刷一圈约500-1000经验 / Medium — kill monsters. Full mine clear ~500-1000 XP

## 专精重置 / Specialization Reset

如果你后悔了专精选择——在火车站附近的雕像处花10,000金币可以重置所有专精 / If you regret specialization choices — the statue near the Railroad station lets you reset all specializations for 10,000 gold.

| 雕像位置 / Statue Location | 操作 / Action | 费用 / Cost |
|---|---|---|
| 铁路场景(温泉上方)——需要修好巴士才能前往 / Railroad area (above spa) — need bus repaired | 放入10,000金币——所有专精取消——可以重新选择 / Insert 10,000 gold — all specializations reset — choose again | 10,000金币 / 10,000 gold |

**注意**: 重置后你需要在5级和10级重新选择专精——重置次数不限 / **Note**: After reset, choose new specializations at 5 and 10 — unlimited resets.

## 技能经验速刷方法 / Skill XP Farming Methods

| 技能 / Skill | 速刷方法 / Fast XP Method |

| **耕种** | 种植大量高价值作物(蓝莓/蔓越莓/上古水果) + 每天浇水 = 最快 / Mass plant high-value crops + water daily = fastest |

| **战斗** | 下矿洞反复刷层——杀的怪越多经验越多——骷髅洞穴效率最高 / Grind mine floors — more kills = more XP — Skull Cavern most efficient |
