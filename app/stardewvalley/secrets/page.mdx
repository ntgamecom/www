## 秘密笔记 / Secret Notes

到达冬1日后——在山姆的房间触发剧情后获得"放大镜"——之后砍树、挖地、钓鱼、杀怪时有概率获得"秘密笔记"/ After Winter 1 — trigger cutscene in Sam's room to get "Magnifying Glass" — then chance to find "Secret Notes" from chopping, digging, fishing, killing monsters.

|---|---|---|
| **#1** | "在矿井入口附近的灌木丛中藏了东西……" / "Something hidden in bushes near mine entrance…" | 隐藏雕像(装饰品)/ Hidden Statue (decoration) |
| **#2** | "刘易斯镇长的紫色裤子在玛妮的卧室里……" / "Mayor Lewis's purple shorts are in Marnie's bedroom…" | 触发剧情——刘易斯的反应 / Trigger cutscene — Lewis's reaction |
| **#3** | "在煤矿森林的湖里钓到一条奇怪的鱼……" / "A strange fish in Cindersap Forest lake…" | 春季鱼王——传说鱼之一 / Spring Fish King — Legendary Fish |
| **#4** | "在山姆卧室的箱子里找到秘密……" / "A secret in Sam's bedroom chest…" | 山姆的特殊剧情 / Sam special cutscene |
| **#5** | "在社区中心附近的栅栏旁……" / "Near the fence by Community Center…" | 隐藏的"兔子脚"剧情——获得永久好运 / Hidden "Rabbit's Foot" event — permanent luck |
| **#6-#25** | 更多笔记——给出Boss弱点、最爱礼物、隐藏宝箱位置等 / More notes — boss weaknesses, loved gifts, hidden chest locations | 每个笔记解开一个隐藏小秘密 / Each note unlocks a hidden secret |

| 类型 / Type | 数量 / Count | 稀有点 / Rare Finds | 建议 / Tip |
|---|---|---|---|

| **全收集奖励** | 95/95 | 生锈的钥匙(开下水道)+ 星露谷英雄奖杯 / Rusty Key (sewers) + Stardew Valley Hero Trophy | |
