## 第一年春完整攻略 / Spring Year 1 Guide

### 第1天 / Day 1

| 时间 / Time | 任务 / Task |
|---|---|
| 早上 / Morning | 收拾残局——清理家门口和木箱周围的杂物 / Clear debris near your house and the wooden box |
| 中午 / Noon | 去鹈鹕镇——和所有村民打招呼 / Go to Pelican Town — greet all villagers |
| 下午 / Afternoon | 去皮埃尔店买防风草种子(15个=75金) + 花椰菜种子(3个=240金) / Buy 15 parsnips (75g) + 3 cauliflowers (240g) |
| 晚上 | 种下所有种子——浇水 / Plant all seeds — water them |
| 睡前 | 存盘点——建议保留50金应急 / Before sleep — keep 50g emergency |

### 第2-7天 / Days 2-7

| 天数 / Day | 重点 / Priority |
|---|---|
| 2 | 继续清理农场——砍树+碎石——得木头和石头 / Clear farm — chop trees + break rocks |
| 3 | 防风草熟了(种下后4天成熟)——继续种植第二轮 / Parsnips harvest (4 days) — plant second batch |
| 4 | 去海滩找威利——获得**竹竿**——开始钓鱼 / Visit Willy — get **Bamboo Pole** — start fishing |
| 5 | 雨天——去矿井(如果已开) / Rainy day — go to Mines (if unlocked) |
| 6-7 | 攒够2000金币——去皮埃尔店**升级背包** / Save 2000g — **upgrade backpack** |

### 春8-13 / Days 8-13

| 重点 / Focus | 说明 / Description |
|---|---|
| 升级背包 / Backpack Upgrade | 2000金——绝对的第一优先级 / 2000g — absolute #1 priority |
| 攒钱 / Save Money | 目标是**蛋蛋节(春13)**买草莓种子 → 每颗100金 / Goal: buy **Strawberry seeds** at Egg Festival (Spring 13) — 100g each |
| 钓鱼赚钱 / Fish for Money | 每天钓鱼可以赚500-1500金 / Fishing earns 500-1500g/day |
| 解锁社区中心 / Unlock CC | 春5天后进城——触发社区中心剧情 / Enter town after Spring 5 — trigger CC cutscene |

### 春13：蛋蛋节 / Spring 13: Egg Festival

| 活动 / Event | 说明 / Description |
|---|---|
| 彩蛋寻宝 / Egg Hunt | 上午9点在小镇广场开始——和阿比盖尔比赛找彩蛋 / 9AM at town square — race Abigail for eggs |
| 买到草莓种子 / Buy Strawberry Seeds | **最重要**——把全部钱买草莓种子(100金/颗) / **Most important** — spend all gold on Strawberry seeds (100g each) |
| 种植草莓 / Plant Strawberries | 当天种下——春13种→春16第一次收获→春28之前可以收2次 / Plant same day — harvest Spring 16 → 2 harvests by Spring 28 |

### 春14-28 / Days 14-28

| 重点 / Focus | 说明 / Description |
|---|---|
| 草莓管理 / Strawberry Care | 每天浇水——每3天收获一次——最后一次收春26 / Water daily — harvest every 3 days — last harvest Spring 26 |
| 社区中心献祭 / CC Bundles | 开始查看献祭包——每种作物留一个 / Check bundles — keep one of each crop |
| 钓鱼练级 / Level Fishing | 争取钓鱼3-4级——解锁更好的鱼竿 / Try for Fishing 3-4 — unlock better rods |
| 下矿 / Mining | 有空就去矿井——收集铜矿+铁矿 + 杀怪拿武器 / Mine when possible — copper + iron + weapon drops |

## 每日推荐流程 / Recommended Daily Routine

| 时间段 / Time | 活动 / Activity |
|---|---|
| **6:00-7:00** | 起床——检查邮箱——查看每日任务 / Wake up — check mail — daily quests |
| **7:00-9:00** | 浇水+收菜+照顾动物 / Water + harvest + tend animals |
| **9:00-12:00** | 去镇上送礼物+接任务 / Go to town — give gifts + accept quests |
| **12:00-16:00** | 下矿井/钓鱼/砍树/收集 / Mine/Fish/Chop/Forage |
| **16:00-18:00** | 回家——放东西+整理 / Return — store items + organize |
| **18:00-20:00** | 加工设备(放果酒/奶酪等) / Process artisan goods |
| **20:00-00:00** | 继续下矿或钓鱼——睡前回家 / Mine or fish more — return before 2AM pass out |
| **注意** | 凌晨2:00不睡觉会昏倒——扣1000金 + 丢失物品 / Passing out at 2AM costs 1000g + item loss |

## 新手常见错误 / Common Beginner Mistakes

| # | 错误 / Mistake | 正确做法 / Correct Way |
|---|---|---|
| 1 | 把作物全卖掉不留种 / Sell all crops | 每种至少留1-2个——社区中心献祭包会用到 / Keep 1-2 of each for Community Center bundles |
| 2 | 铺路铺太大的田地 / Over-plant farmland | 第一天没有洒水器——浇不完那么多地 / No sprinklers early — can't water it all |
| 3 | 不去社区中心 / Ignore Community Center | 献祭奖励是游戏中最有价值的(温室+矿车+巴士) / Bundle rewards are the best (greenhouse + minecarts + bus) |
| 4 | 砍掉所有树 / Chop all trees | 木材永远不够用——建一个树场 / You'll never have enough wood — plant a tree farm |
| 5 | 和Joja Mart走/ Choose Joja route | Joja路线只是花钱买过——丢失了游戏最有趣的部分 / Joja is pay-to-win — misses the most fun content |
| 6 | 不开鱼塘 / Skip Fish Ponds | 鱼塘后期可以产铱星+珍珠——值得建 / Fish ponds produce iridium + pearls late-game |
