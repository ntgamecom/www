# 星露谷物语攻略 / Stardew Valley Guide

《星露谷物语》是ConcernedApe开发的农场经营模拟RPG——继承爷爷的破旧农场，在鹈鹕镇开始全新的生活。种田、采矿、钓鱼、战斗、交友、恋爱——渔夫种地的终极休闲体验。本攻略涵盖全季节作物规划、村民好感度与婚姻、矿井与战斗指南以及全收集品攻略。

Stardew Valley is ConcernedApe's farm simulation RPG — inherit your grandfather's old farm and start a new life in Pelican Town. Farming, mining, fishing, combat, friendship, romance — the ultimate relaxing experience. This guide covers seasonal crop planning, villager relationships and marriage, mine and combat guide, and complete collectibles.

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **标准农场最适合新手 / Standard Farm for Beginners** | 其他农场地图都有耕地限制——标准农场最大最平 / Other maps restrict farming space — Standard is largest and flattest |
| **2** | **第一年春的目标 / Spring Year 1 Goal** | 种满防风草(前3天种)→攒钱买草莓(春13蛋蛋节) / Plant parsnips (days 1-3) → save for Strawberries (Spring 13 Egg Festival) |
| **3** | **先升级背包 / Upgrade Backpack First** | 皮埃尔店的背包升级(2000→10000金)——增加容量是最重要的投资 / Pierre's backpack upgrade — most important early investment |
| **4** | **前7天必去海滩 / Visit Beach First Week** | 威利的钓鱼竿免费——每天钓鱼可以赚一大笔 / Willy gives free fishing rod — fishing is great early money |
| **5** | **献祭包是第一优先级 / Community Center First** | 完成献祭包解锁温室+矿车等——比乔家路线好得多 / Completing bundles unlocks greenhouse, minecarts, etc. — better than Joja route |
| **6** | **洒水器改变人生 / Sprinklers Change Everything** | 耕种2级解锁基础洒水器——6级解锁优质洒水器——农活自动化 / Farming 2 = basic sprinkler, 6 = quality sprinkler — automate watering |
| **7** | **下雨天去挖矿 / Mine on Rainy Days** | 下雨天不用浇水——是下矿的最佳时机 / No watering on rainy days — best time to mine |
| **8** | **别砍光树和草 / Don't Clear Everything** | 留一些树和草——草可以割(干草喂动物) / Leave some trees and grass — grass becomes hay for animals |

## 技能系统 / Skill System

| 技能 / Skill | 升级方法 / Leveling | 5级专精 / Lv5 Specialization | 10级专精 / Lv10 Specialization |
|---|---|---|---|
| **耕种 / Farming** | 收获作物+照顾动物 / Harvest crops + animals | 农耕人(产品+10%) / Tiller (+10% crops) | 工匠(工匠设备+40%) / Artisan (+40% artisan) |
| | | 牧场主(动物+20%) / Rancher (+20% animals) | 畜牧人(动物+10%) / Shepherd (+10% animals) |
| **采矿 / Mining** | 敲石头+挖矿 / Break rocks + mine ores | 矿工(矿石+1) / Miner (+1 ore) | 铁匠(金属锭价值+50%) / Blacksmith (+50% metal bars) |
| | | 地质学家(宝石概率+50%) / Geologist (+50% gems) | 挖掘者(宝石双倍) / Excavator (double gems) |
| **采集 / Foraging** | 拾取+砍树 / Pick up + chop trees | 护林人(木材+25%) / Forester (+25% wood) | 伐木工(硬木出现) / Lumberjack (hardwood) |
| | | 采集者(双倍概率) / Gatherer (double chance) | 植物学家(铱星品质) / Botanist (iridium quality) |
| **钓鱼 / Fishing** | 钓鱼成功 / Catch fish | 渔夫(鱼价值+25%) / Fisher (+25% fish) | 垂钓者(鱼价值+50%) / Angler (+50% fish) |
| | | 捕蟹者(蟹笼便宜) / Trapper (cheaper crab pots) | 水手(蟹笼无垃圾) / Mariner (no junk in pots) |
| **战斗 / Combat** | 击杀怪物 / Kill monsters | 战士(伤害+10%) / Fighter (+10% dmg) | 野蛮人(暴击+50%) / Brute (+50% crit) |
| | | 侦察兵(暴击率+50%) / Scout (+50% crit rate) | 特技者(暴击威力翻倍) / Acrobat (crit power x2) |

**推荐技能路线 / Recommended Skill Path**:
- 耕种: 农耕人→工匠(最赚钱的路线) / Tiller→Artisan (most profitable)
- 采矿: 矿工→铁匠(赚钱) 或 地质学家→挖掘者(宝石收藏) / Miner→Blacksmith or Geologist→Excavator
- 采集: 采集者→植物学家(铱星品质采集品——疯狂赚钱) / Gatherer→Botanist (iridium forage = money)
- 钓鱼: 渔夫→垂钓者(卖鱼最赚) / Fisher→Angler (best fish money)
- 战斗: 侦察兵→特技者(暴击流) / Scout→Acrobat (crit build)

## 难易度说明 / Difficulty Notes

星露谷没有传统难度选择——但有几种模式区分 / No difficulty selection — but different play styles:

| 模式 / Mode | 说明 / Description |
|---|---|
| **普通 / Normal** | 标准体验——死亡扣1000金+丢物品(最多丢失1件) / Normal — death costs 1000g + item loss (max 1 item) |
| **收入比 / Profit Margin** | 创建农场时可选25%/50%/75%/100%——变相提高难度 / Set to 25-75% — effectively harder |
| **联机 / Multiplayer** | 存档文件独立——可随时加入退出 / Separate save files — join/leave anytime |
