| 活动 / Activity | 说明 / Description |

| **升级工具 / Upgrade Tools** | 把所有工具升级到至少金质 / Upgrade all tools to at least Gold |

| **温室 / Greenhouse** | 如果解锁了温室——里面全年可种 / Unlocked greenhouse — year-round planting inside |

| 洒水器 / Sprinkler | 耕种等级 / Farming | 材料 / Materials | 覆盖 / Coverage |
|---|---|---|---|
| **基础洒水器 / Basic** | 2级 | 铜锭x1+铁锭x1 | 前后左右4格 / 4 tiles |
| **优质洒水器 / Quality** | 6级 | 铁锭x1+金锭x1+精炼石英x1 | 3x3范围(8格) / 3x3 (8 tiles) |

**建议 / Tips**:
- 春第一年：手动浇水（别浪费材料做基础洒水器）
- 夏第一年：优先做出6-8个**优质洒水器**——自动浇水100格 / Prioritize 6-8 Quality Sprinklers — auto-water 100 tiles
- 第二年：换铱星洒水器——一步到位 / Year 2: Iridium Sprinklers — one-and-done

- **上古果实(Ancient Fruit)** x 100-120株——用铱星洒水器自动浇水 / 100-120 Ancient Fruit plants — Iridium Sprinklers
