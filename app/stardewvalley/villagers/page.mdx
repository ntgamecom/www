## 送礼指南 / Gift Guide

### 策略 / Strategy

| 策略 / Strategy | 说明 / Description |
|---|---|
| **每周只能送两次** / 2 gifts per week | 生日当天可以送一次 + 另外一周送一次 / Birthday gift + one extra per week |
| **生日礼物4倍好感** / Birthday = 4x affinity | 生日送最爱礼物=320点好感 / Birthday + loved = 320 points |
| **星露谷展览会送礼** / Stardew Fair | 秋天展览会可以用星星币购买特殊礼物 / Buy special gifts with Star Tokens |
| **普遍最爱** / Universal Love | 黄金南瓜、珍珠、魔法糖冰棍 / Golden Pumpkin, Pearl, Magic Rock Candy |

## 婚姻系统 / Marriage

### 结婚条件 / Requirements

| 条件 / Condition | 说明 / Description |
|---|---|
| 10心好感 / 10 Hearts | 必须和对象有10心好感度 / 10 hearts required |
| 送花束 / Give Bouquet | 在皮埃尔店买花束(200金)——送后解锁10心 / Buy Bouquet (200g) from Pierre's |
| 人鱼吊坠 / Mermaid's Pendant | **暴风雨天去海滩找老水手**——在涨潮时花5000金买 / **Rainy day at the beach** — 5000g from Old Mariner |
| 婚礼 / Wedding | 送吊坠后3天——在镇上举办婚礼 / 3 days after pendant — wedding ceremony |

| 孩子 / Children | 结婚后可以生孩子(最多2个) / Have children (max 2) |
| 永远不会离婚(除非你选择)/ Never leave | **除非你找刘易斯镇长离婚(50000金)** / Unless you divorce (50,000g) |
