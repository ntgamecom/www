矿井共 **120层**——每10层一个主题——越往下怪物越强、矿石越丰富 / 120 floors — theme every 10 floors — harder monsters, richer ores deeper down.

## 头骨山洞 / Skull Cavern

**解锁条件**: 完成巴士修理献祭包——买票去卡利科沙漠 / Unlock: complete bus bundle — buy ticket to Calico Desert.

| 信息 / Info | 说明 / Description |
|---|---|
| 层数 / Floors | **无限层**——用电梯但电梯布局随机 / **Infinite floors** — elevator but random layout |
| 难度 / Difficulty | 极高——敌人比矿井强数倍 / Extreme — enemies are way stronger |
| 宝藏 / Loot | 铱星矿石极多——最快的赚钱方式 / Tons of iridium — fastest money in the game |
| 最终奖励 / Final Reward | 100层后遇到齐先生——获得"蛇之牛奶"(永久+25血) / Mr. Qi at floor 100 — "Snake Milk" (+25 HP permanent) |

### 头骨山洞攻略核心 / Key Skull Cavern Strategy

| 准备 / Prep | 说明 / Description |
|---|---|
| **高级炸弹 / Bombs** | 带50+个炸弹——炸弹挖矿比镐快得多 / 50+ bombs — much faster than pickaxe |
| **堆楼梯 / Staircases** | 99石头=1个楼梯——带30+个直接跳层 / 30+ staircases to skip floors |
| **食物 / Food** | 带**奶酪(200血+)**或**咖啡(+速度)** / Cheese (200+ HP) or Coffee (speed) |
| **武器 / Weapon** | **银河剑**——没有这个就不用去 / **Galaxy Sword** — don't go without it |
| **最佳时间** | **最佳运气日**去(看电视→算命) + 食用**幸运午餐** / **Best luck day** (check TV) + eat **Lucky Lunch** |

**头骨山洞三层核心循环 / Core Skull Cavern Loop**:
1. 丢炸弹炸石头 → 找洞跳下(比楼梯快) / Throw bombs → find hole (faster than stairs)
2. 跳洞=直接下落3-15层——可以快速下到100层 / Hole = skip 3-15 floors — fastest way to floor 100
3. 有铱矿就停→收铱→继续下 / Stop for iridium nodes → mine → continue
4. 效率最大化：一层停留不超过30秒 / Max efficiency: less than 30 seconds per floor
