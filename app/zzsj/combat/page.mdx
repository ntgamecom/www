## 战斗基础 / Combat Basics

主宰世界采用实时战斗系统（不是回合制）。玩家通过虚拟摇杆移动，点击技能按钮释放技能。战斗流畅度直接影响游戏体验。

Master of the World uses a real-time combat system (not turn-based). Players move with a virtual joystick and tap skill buttons to cast abilities. Combat fluidity directly affects the gaming experience.

## 技能类型 / Skill Types

### 核心技能 / Core Skill

每个职业有一个核心技能，是职业机制的关键。

Each class has one core skill that is key to the class mechanic.

| 职业 / Class | 核心技能 / Core Skill | 效果 / Effect |
|---|---|---|
| 御天 / Yutian | 元素调和 / Elemental Harmony | 切换法器元素属性 / Switch artifact element |
| 万剑 / Wanjian | 剑意爆发 / Sword Intent Burst | 消耗剑意层数造成额外伤害 / Consume stacks for bonus damage |
| 凌霄 / Lingxiao | 战意觉醒 / Battle Spirit Awaken | 战意满时强化全部技能 / Enhance all skills at full Battle Spirit |

### 主动技能 / Active Skills

主动技能需要玩家手动释放，通常有冷却时间（CD）。每个职业有4-6个主动技能。

Active skills require manual activation, have cooldowns (CD). Each class has 4-6 active skills.

| 技能类别 / Category | 特点 / Features |
|---|---|
| 输出技能 / DPS Skills | 主要伤害来源 / Main damage source |
| 控制技能 / CC Skills | 眩晕、定身、减速 / Stun, root, slow |
| 位移技能 / Mobility | 冲刺、闪避 / Dash, dodge |
| 辅助技能 / Support | 加血、加盾、增益 / Heal, shield, buff |
| 终结技 / Finisher | 高伤害大招 / High-damage ultimate |

### 被动技能 / Passive Skills

被动技能不需要主动释放，在满足条件时自动触发。它们提供属性加成或触发额外效果。

Passive skills do not require activation and trigger automatically when conditions are met. They provide stat bonuses or trigger additional effects.

## 战斗机制 / Combat Mechanics

### 冷却时间 / Cooldown (CD)

每个技能使用后需要等待一定时间才能再次使用。合理管理CD是战斗的关键。

Each skill has a cooldown period before it can be used again. Managing CDs is key to combat.

- 小技能CD通常为3-8秒 / Minor skills usually have 3-8s CD
- 大招CD通常为20-60秒 / Ultimates usually have 20-60s CD
- 使用减CD装备或技能可以缩短等待时间 / Use CD reduction gear or skills

### 技能连招 / Skill Combos

技能连招是按照特定顺序释放技能以最大化伤害输出的技巧。

Skill combos are sequences of skills used in a specific order to maximize damage output.

**御天连招示例** / Yutian Combo Example:
```
冰霜领域 → 冰冻术 → 寒冰刺 → 雷电术
Frost Field → Freeze → Ice Spike → Lightning
```

**万剑连招示例** / Wanjian Combo Example:
```
剑影步 → 剑气斩 → 剑气斩 → 万剑归宗
Sword Step → Slash → Slash → Ten Thousand Swords
```

**凌霄连招示例** / Lingxiao Combo Example:
```
战吼 → 破军枪法 → 龙吟 → 破军枪法
War Cry → Spear Art → Dragon Roar → Spear Art
```

### 闪避与走位 / Dodge and Positioning

- 闪避有独立CD，通常2-3秒 / Dodge has its own CD, usually 2-3s
- 完美闪避可触发增益效果 / Perfect dodge triggers buff effects
- 利用地形遮挡远程攻击 / Use terrain to block ranged attacks

## 自动战斗系统 / Auto-Battle System

自动战斗适合日常刷怪和简单副本，但在高难度挑战和高段位PVP中不建议使用。

Auto-battle is suitable for daily farming and easy dungeons, but not recommended for difficult content or high-rank PVP.

**优点** / Pros:
- 节省操作精力 / Saves manual effort
- 适合挂机刷材料 / Good for AFK farming
- 反应速度更快 / Faster reaction time

**缺点** / Cons:
- 不会躲避技能 / Does not dodge skills
- 不会合理控制资源 / Poor resource management
- 容易被PVP对手预判 / Predictable in PVP

**自动战斗设置** / Auto-Battle Settings:
| 设置 / Setting | 选项 / Options |
|---|---|
| 技能使用 / Skill Usage | 全部/核心/自动 / All / Core Only / Auto |
| 目标选择 / Targeting | 最近/血量最低/威胁最大 / Nearest / Lowest HP / Highest Threat |
| 拾取设置 / Loot | 自动拾取/仅拾取稀有 / Auto-loot / Rare Only |
| 药品使用 / Potion Use | 自动喝药/手动 / Auto-potion / Manual |

## PVE与PVP战斗差异 / PVE vs PVP Combat

| 方面 / Aspect | PVE | PVP |
|---|---|---|
| 目标数量 / Targets | 多怪群 / Multiple mobs | 单体或少目标 / Single or few |
| 技能选择 / Skill Choice | AOE优先 / AoE priority | 单体控制优先 / Single-target CC priority |
| 走位要求 / Positioning | 较低 / Lower | 极高 / Very high |
| 属性侧重 / Stats | 输出为主 / DPS focus | 暴击+穿透 / Crit + Pierce |
| 自动战斗 / Auto-Battle | 可用 / Usable | 不推荐 / Not recommended |

## 战斗技巧 / Combat Tips

1. 在技能CD期间使用普攻填充 / Use basic attacks during skill CDs
2. 合理利用控制技能打断敌人施法 / Use CC to interrupt enemy casting
3. 注意观察敌人技能预警圈 / Watch enemy skill warning circles
4. 与队友配合控制链 / Coordinate CC chains with teammates
5. 在安全位置等待技能CD / Reposition safely while waiting for CDs
