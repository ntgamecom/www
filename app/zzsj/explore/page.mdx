## 世界探索 / World Exploration

主宰世界拥有广阔的开放世界地图，包含多个风格各异的区域。探索世界不仅能欣赏美景，还能发现隐藏宝藏、挑战世界首领和体验独特的修仙氛围。

Master of the World features a vast open world with multiple distinct regions. Exploring the world not only lets you enjoy scenic views but also discover hidden treasures, challenge world bosses, and experience the unique cultivation atmosphere.

## 传送系统 / Teleportation System

### 飞行坐骑 / Flying Mounts

达到20级后解锁飞行功能。飞行坐骑可以大幅提升移动速度并穿越地形障碍。

Flying is unlocked at level 20. Flying mounts greatly increase movement speed and allow traversal over terrain obstacles.

| 坐骑 / Mount | 获取方式 / Acquisition | 速度 / Speed |
|---|---|---|
| 仙鹤 / Celestial Crane | 主线赠送 / Main Quest Reward | +60% |
| 飞剑 / Flying Sword | 任务获得 / Quest Reward | +80% |
| 灵狐 / Spirit Fox | 商店购买 / Shop Purchase | +100% |
| 神龙 / Divine Dragon | 活动限时 / Limited Event | +120% |

### 传送点 / Waypoints

地图上分布着多个传送点，解锁后可以快速传送。

Multiple waypoints are scattered across the map. Once unlocked, you can fast travel between them.

- 点击地图上的传送点图标 / Click waypoint icons on the map
- 消耗少量银两 / Costs a small amount of Silver
- 战斗中不可传送 / Cannot teleport in combat
- 推荐优先解锁所有传送点 / Recommended to unlock all waypoints early

## 隐藏宝箱 / Hidden Chests

世界各地隐藏着宝箱，打开可获得银两、经验丹和稀有道具。

Hidden chests are scattered throughout the world. Opening them rewards Silver, XP elixirs, and rare items.

| 宝箱类型 / Chest Type | 外观 / Appearance | 刷新 / Respawn | 奖励 / Reward |
|---|---|---|---|
| 木宝箱 / Wood Chest | 棕色木箱 / Brown | 每日 / Daily | 银两、低级强化石 / Silver, Low Stones |
| 银宝箱 / Silver Chest | 银色金属 / Silver | 3天 / 3 Days | 银两、中级强化石 / Silver, Mid Stones |
| 金宝箱 / Gold Chest | 金色发光 / Golden Glow | 7天 / 7 Days | 大量银两、高级材料 / Lots Silver, High Mats |
| 稀有宝箱 / Rare Chest | 紫色光芒 / Purple Glow | 随机 / Random | 真灵碎片、装备 / Spirit Frags, Gear |

**寻宝技巧** / Treasure Hunting Tips:
- 查看网上社区分享的宝箱坐标 / Check community-shared coordinates
- 在悬崖、瀑布和建筑物角落寻找 / Look at cliffs, waterfalls, building corners
- 使用飞行坐骑更容易到达高处宝箱 / Use flying mounts for hard-to-reach chests
- 组队时共享宝箱位置 / Share chest locations with party

## 论道系统 / Dao Discussion System

论道是主宰世界中独特的冥想悟道系统。在特定的风景优美的地方静坐，可以获得修为加成。

Dao Discussion is a unique meditation and enlightenment system in Master of the World. Sitting quietly in specific scenic locations grants cultivation bonuses.

**论道地点** / Dao Discussion Locations:
| 地点 / Location | 区域 / Area | 效果 / Effect |
|---|---|---|
| 观云台 / Cloud Viewing Platform | 青云山 / Azure Cloud Mt. | 经验获取+5% / XP Gain +5% |
| 听瀑布 / Waterfall Listening | 落霞谷 / Sunset Valley | 修为获取+10% / Cultivation Gain +10% |
| 望月亭 / Moon Viewing Pavilion | 新手村 / Novice Village | 真灵经验+5% / Spirit XP +5% |
| 悟道崖 / Enlightenment Cliff | 天界之塔 / Celestial Tower | 技能经验+10% / Skill XP +10% |

**论道技巧** / Dao Tips:
- 组队论道效果叠加 / Party meditation stacks effects
- 每日论道有上限时间 / Daily cap on meditation time
- 使用熏香可提升论道效率 / Use incense to boost meditation efficiency
- 论道时无法移动，请选择安全地点 / Cannot move while meditating, choose safe spots

## 探索小贴士 / Exploration Tips

1. **开图优先** — 优先解锁所有传送点 / Unlock all waypoints first
2. **日常采集** — 每个区域有特产材料，每日可采集 / Each area has daily gathering materials
3. **隐藏任务** — 与特定NPC对话可触发隐藏任务 / Talk to specific NPCs for hidden quests
4. **风景打卡** — 游戏中设有拍照点和成就 / Photo spots and achievements available
5. **飞行为王** — 飞行可以到达步行无法到达的地方 / Flying reaches areas inaccessible on foot
6. **注意等级** — 高等级区域怪物更强，注意安全 / Higher-level areas have stronger mobs
