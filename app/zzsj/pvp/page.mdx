## 竞技场 / Arena

竞技场是1v1的公平对战模式，根据段位匹配实力相近的对手。

The arena is a 1v1 fair match mode that pairs you with opponents of similar rank.

## 公会战详解 / Guild War Details

公会战是公会之间的领地争夺战，每周举行两次。

Guild War is a territory battle between guilds, held twice per week.

**参与条件** / Entry Requirements:
- 公会等级≥2 / Guild Level 2+
- 玩家等级≥30 / Player Level 30+
- 公会活跃度达标 / Guild must meet activity threshold

**战斗规则** / Battle Rules:
1. 每方最多20人参战 / Max 20 players per side
2. 占领三个据点获取积分 / Capture 3 strongholds for points
3. 击杀敌方玩家获得积分 / Kill enemy players for points
4. 15分钟内积分高者获胜 / Higher points within 15 min wins

**公会战奖励** / Guild War Rewards:

| 结果 / Result | 公会奖励 / Guild Reward | 个人奖励 / Personal Reward |
|---|---|---|
| 胜利 / Victory | 公会资金+500,000 / Funds +500,000 | 功勋+200, 帮贡+500 |
| 失败 / Defeat | 公会资金+100,000 / Funds +100,000 | 功勋+50, 帮贡+100 |

## PVP技巧 / PVP Tips

1. **了解对手** — 熟悉各职业技能和套路 / Learn all class skills and patterns
2. **控制优先** — 先手控制决定胜负 / First CC often decides the match
3. **走位风筝** — 近战贴脸，远程保持距离 / Melee close in, ranged kite
4. **技能预判** — 预判对手的位移和闪避 / Predict opponent mobility and dodges
5. **资源管理** — 不要一次性交出所有技能 / Do not blow all skills at once
6. **真灵搭配** — PVP选择控制和生存型真灵 / Choose CC and survival spirits for PVP
7. **装备调整** — PVP装备侧重暴击和穿透 / PVP gear focuses on crit and pierce

## 公会建议 / Guild Tips

1. **选择活跃公会** — 活跃度高的公会有更多福利 / Active guilds have more benefits
2. **每日捐献** — 稳定获取帮贡 / Daily donation for steady contribution
3. **参与公会战** — 公会战奖励非常丰厚 / Guild wars offer great rewards
4. **公会社交** — 积极交流找到固定队伍 / Active communication leads to fixed teams
5. **公会技能** — 优先升级攻击和生命 / Prioritize ATK and HP guild skills
6. **帮助新人** — 帮助低等级公会成员可获得额外帮贡 / Helping low-level members gives bonus contribution
