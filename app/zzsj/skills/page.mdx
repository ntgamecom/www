## 神通系统 / Divine Ability System

神通是主宰世界中角色技能的总称。分为核心技能、主动技能和被动技能三大类。每个职业都有独特的技能树，玩家可以根据自己的玩法偏好进行搭配。

Divine Abilities (Shentong) are the collective name for character skills in Master of the World. They are divided into three categories: Core Skills, Active Skills, and Passive Skills. Each class has its own unique skill tree, and players can customize their loadout based on playstyle.

## 核心技能详解 / Core Skills

### 御天 — 元素调和 / Yutian — Elemental Harmony

元素调和允许御天在三种元素形态之间切换。

Elemental Harmony allows Yutian to switch between three elemental forms.

| 形态 / Form | 强化效果 / Enhanced Effect |
|---|---|
| 火焰形态 / Fire | 技能附带灼烧，每秒造成30%法术伤害 / Skills apply burn, 30% magic DPS |
| 冰霜形态 / Ice | 技能附带减速，降低目标移动速度40% / Skills apply slow, reduce move speed by 40% |
| 雷电形态 / Lightning | 技能有20%概率连锁攻击额外2个目标 / 20% chance to chain attack 2 extra targets |

### 万剑 — 剑意 / Wanjian — Sword Intent

使用技能积攒剑意，满层后终极技能伤害翻倍。

Build Sword Intent stacks with skills; at max stacks, ultimate damage doubles.

| 剑意层数 / Stacks | 效果 / Effect |
|---|---|
| 1层 | 暴击率+5% / Crit Rate +5% |
| 3层 | 技能伤害+10% / Skill DMG +10% |
| 5层（满） | 万剑归宗伤害翻倍 / Ultimate DMG x2 |

### 凌霄 — 战意 / Lingxiao — Battle Spirit

受到或造成伤害积攒战意，战意满时强化全部技能。

Accumulate Battle Spirit from taking or dealing damage; when full, all skills are enhanced.

| 战意状态 / State | 效果 / Effect |
|---|---|
| 50% | 技能CD减少10% / Skill CD -10% |
| 100%（满） | 全部技能伤害+30%持续8秒 / All skill DMG +30% for 8s |

## 技能搭配小贴士 / Skill Tips

1. **优先升级核心技能** — 核心技能对整体战斗影响最大 / Prioritize core skill upgrades
2. **根据场景切换** — PVE和PVP使用不同的技能搭配 / Switch loadouts for PVE vs PVP
3. **CD管理** — 注意技能CD搭配，避免同时进入CD / Manage cooldowns to avoid downtime
4. **控制链配合** — 多人配合时可形成控制链 / Coordinate CC chains in parties
5. **资源分配** — 不要平均升级，集中资源给主技能 / Focus resources on main skills
