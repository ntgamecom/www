## 装备系统 / Equipment System

装备是提升战力的核心途径。主宰世界拥有完整的装备体系，包括多个装备栏位、品质等级、强化、宝石镶嵌和打造系统。

Equipment is the core path to increasing combat power. Master of the World has a complete equipment system including multiple gear slots, quality tiers, enhancement, gem socketing, and crafting.

## 装备强化 / Equipment Enhancement

强化是提升装备属性的基础方式。消耗银两和强化石可提升装备等级。

Enhancement is the basic way to improve equipment stats. Consume Silver and Enhancement Stones to raise gear levels.

| 强化等级 / Level | 成功率 / Success Rate | 所需材料 / Materials |
|---|---|---|
| +1 to +5 | 100% | 银两 + 低级强化石 / Silver + Low Stone |
| +6 to +10 | 80% | 银两 + 中级强化石 / Silver + Mid Stone |
| +11 to +15 | 60% | 银两 + 高级强化石 / Silver + High Stone |
| +16 to +20 | 40% | 银两 + 顶级强化石 / Silver + Top Stone |
| +21 to +25 | 25% | 银两 + 传说强化石 / Silver + Legendary Stone |
| +26 to +30 | 15% | 银两 + 神话强化石 / Silver + Mythic Stone |

**强化小贴士** / Enhancement Tips:
- 强化失败不会掉级 (最高+15前) / No downgrade on failure (pre +15)
- +15以上失败可能掉级 / Failure above +15 may cause downgrade
- 使用保护符可防止掉级 / Use protection scrolls to prevent downgrade
- 优先强化武器和饰品 / Prioritize weapon and accessory enhancement

## 宝石镶嵌 / Gem Socketing

每个装备有1-4个宝石孔。宝石提供额外属性加成。

Each gear piece has 1-4 gem sockets. Gems provide additional stat bonuses.

| 宝石类型 / Gem Type | 属性 / Stat |
|---|---|
| 红宝石 / Ruby | 攻击 / Attack |
| 蓝宝石 / Sapphire | 防御 / Defense |

| 黄宝石 / Topaz | 暴击 / Crit |
| 紫宝石 / Amethyst | 穿透 / Pierce |

**宝石合成** / Gem Crafting:
3个同级宝石可合成1个更高级宝石。

3 same-level gems can be combined into 1 higher-level gem.

| 3级 / Lv.3 | +50 |

## 装备打造 / Equipment Crafting

通过收集材料，可以在铁匠处打造高品质装备。

By collecting materials, you can craft high-quality equipment at the blacksmith.

**所需材料** / Required Materials:
- 装备图纸 / Equipment Blueprint (副本掉落 / Dungeon drop)
- 打造石 / Crafting Stone (分解装备获得 / From gear dismantling)
- 五行精华 / Elemental Essence (活动奖励 / Event reward)
- 银两 / Silver

**可打造品质** / Craftable Quality:
| 图纸品质 / Blueprint Quality | 可打造品质 / Result Quality |
|---|---|
| 蓝色图纸 / Blueprint Blue | 蓝色-紫色 / Rare-Epic |
| 紫色图纸 / Blueprint Purple | 紫色-传说 / Epic-Legendary |
| 橙色图纸 / Blueprint Orange | 传说-神话 / Legendary-Mythic |

## 装备养成建议 / Gear Advice

1. **随等级更新** — 每10级更换一次装备 / Replace gear every 10 levels
2. **强化继承** — 换装备时强化等级可继承 / Enhancement level transfers to new gear
3. **套装效果** — 同套装2/4/6件有额外加成 / Set bonuses at 2/4/6 pieces
4. **属性取舍** — 优先主属性再考虑套装 / Prioritize main stat before set bonuses
5. **分解无用装备** — 获得强化材料和打造石 / Dismantle useless gear for materials
6. **不要过度强化过渡装** — 集中资源给毕业装备 / Do not over-enhance temporary gear
