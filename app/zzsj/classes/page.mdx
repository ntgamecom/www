## 职业总览 / Class Overview

主宰世界目前有三个职业：御天、万剑、凌霄。每个职业拥有独特的武器、技能体系和战斗风格。选择适合你的职业是成功的第一步。

Master of the World currently features three classes: Yutian (Arcane), Wanjian (Myriad Swords), and Lingxiao (Soaring Firmament). Each class has unique weapons, skill systems, and combat styles. Choosing the right class is the first step to success.

## 御天 / Yutian — 远程法系输出

御天是使用法器的远程法术输出职业。擅长通过元素法术造成大量范围伤害，拥有优秀的控制技能。

Yutian is a ranged magic DPS class wielding artifact orbs. They excel at dealing massive area damage through elemental spells and possess excellent crowd control abilities.

**核心机制** / Core Mechanic: **元素调和** / Elemental Harmony — 御天的技能会根据当前法器的元素属性附加额外效果 (火焰/冰霜/雷电)。切换元素可以适应不同战斗场景。Yutian's skills gain bonus effects based on the current artifact's element (Fire/Ice/Lightning). Switching elements adapts to different battle scenarios.

**推荐定位** / Recommended Role: 后排输出 / Backline DPS

| 技能类型 / Skill Type | 描述 / Description |
|---|---|
| 火焰形态 / Fire Form | 高爆发、灼烧效果 / High burst, burn effect |
| 冰霜形态 / Ice Form | 减速、冰冻控制 / Slow, freeze control |
| 雷电形态 / Lightning Form | 连锁伤害、眩晕 / Chain damage, stun |

**优势** / Strengths:
- 远程安全输出 / Safe ranged DPS
- 强大的AOE能力 / Strong AoE potential
- 多种控制手段 / Multiple crowd control options

**劣势** / Weaknesses:
- 身板脆弱 / Squishy
- 被近身后较为被动 / Weak when engaged in melee
- 对技能命中率要求高 / Requires high skill accuracy

**适合玩家** / Suitable for:
- 喜欢远程风筝打法 / Players who enjoy ranged kiting
- 擅长走位和预判 / Good positioning and prediction
- PVE团队副本的重要输出 / Key DPS in PVE raids

## 万剑 / Wanjian — 近战物理爆发

万剑是使用长剑的近战物理爆发职业。以极高的瞬间爆发能力著称，擅长在短时间内倾泻大量伤害。

Wanjian is a melee physical burst DPS class wielding a longsword. Famous for extremely high burst damage, they excel at unloading massive damage in a short window.

**核心机制** / Core Mechanic: **剑意** / Sword Intent — 使用技能积攒剑意层数，层数越高，终结技伤害越强。Build up Sword Intent stacks through skill usage; higher stacks amplify finisher damage.

**推荐定位** / Recommended Role: 刺客型近战 / Assassin-type Melee

| 核心技能 / Core Skill | 效果 / Effect |
|---|---|
| 万剑归宗 / Ten Thousand Swords | 终极爆发技，消耗所有剑意 / Ultimate burst, consumes all Sword Intent |
| 剑影步 / Sword Step | 突进并提升暴击率 / Dash forward and increase crit rate |
| 剑气斩 / Sword Slash | 范围伤害，积攒剑意 / AoE damage, builds Sword Intent |
| 剑盾 / Sword Shield | 减伤技能，短暂无敌 / Damage reduction, brief invincibility |

**优势** / Strengths:
- 极高的单体爆发 / Extremely high single-target burst
- 帅气连招 / Stylish combos
- 有位移和无敌技能 / Has mobility and invincibility

**劣势** / Weaknesses:
- 需要贴脸输出 / Requires melee range
- 技能循环要求精准 / Requires precise skill rotation
- 持续输出一般 / Average sustained DPS

**适合玩家** / Suitable for:
- 喜欢高爆发职业 / Players who enjoy burst classes
- 追求操作感和连招 / Who enjoy combos and mechanical skill
- PVP单体击杀能力强 / Strong in PVP single-target kills

## 凌霄 / Lingxiao — 近战均衡战士

凌霄是使用长枪的近战均衡职业。攻守兼备，既有不错的输出也有优秀的生存能力，是团队中可靠的前排。

Lingxiao is a balanced melee class wielding a spear. With both solid damage and excellent survivability, they are reliable frontliners in any team composition.

**核心机制** / Core Mechanic: **战意** / Battle Spirit — 受到伤害或造成伤害时积攒战意值，战意满时技能获得强化效果。Battle Spirit accumulates when taking or dealing damage. When full, skills gain enhanced effects.

**推荐定位** / Recommended Role: 战士 / Frontline Warrior

| 核心技能 / Core Skill | 效果 / Effect |
|---|---|
| 破军枪法 / Spear Art | 主要输出技能，战意强化后范围增大 / Main DPS skill, enhanced AoE with Battle Spirit |
| 龙吟 / Dragon Roar | 嘲讽周围敌人 / Taunt nearby enemies |
| 战吼 / War Cry | 提升全队攻击力 / Boost party attack |
| 铁壁 / Iron Wall | 大幅度提升防御和韧性 / Greatly increase defense and toughness |

**优势** / Strengths:
- 生存能力强 / High survivability
- 团队辅助能力强 / Strong team support
- 持续作战能力优秀 / Excellent sustained combat
- 适合新手入门 / Beginner-friendly

**劣势** / Weaknesses:
- 爆发能力不如万剑 / Lower burst than Wanjian
- 远程能力有限 / Limited ranged capability
- 机动性一般 / Average mobility

**适合玩家** / Suitable for:
- 喜欢坦克/战士角色 / Players who enjoy tank/warrior roles
- 注重团队配合 / Team-oriented players
- 新手首选 / Best first class
