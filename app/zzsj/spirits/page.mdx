## 真灵系统简介 / Spirit System Overview

真灵是主宰世界中的宠物伙伴系统。它们不仅外观可爱，还能在战斗中提供强大的帮助。真灵可以通过多种方式获得和培养。

Spirits (Zhenling) are the pet companion system in Master of the World. Not only are they adorable, but they also provide powerful assistance in battle. Spirits can be obtained and raised through various methods.

## 真灵培养系统 / Spirit Training System

### 升级 / Level Up

真灵通过吞噬其他真灵或经验丹来提升等级。等级提高会增加基础属性。

Spirits gain levels by consuming other spirits or XP elixirs. Higher levels increase base stats.

- 满级通常为60级 / Max level is usually 60
- 每10级需要突破材料 / Breakthrough materials needed every 10 levels
- 突破后可解锁新技能 / Breakthrough unlocks new skills

### 升星 / Star Rating

升星是提升真灵品质上限的关键途径。

Star rating is the key path to raising a spirit's quality ceiling.

| 星级 / Stars | 所需碎片 / Fragments Needed | 效果 / Effect |
|---|---|---|
| 0 → 1★ | 10 | 基础属性+5% / Base stats +5% |
| 1 → 2★ | 20 | 基础属性+10% / Base stats +10% |
| 2 → 3★ | 40 | 解锁新技能 / Unlock new skill |
| 3 → 4★ | 80 | 基础属性+15% / Base stats +15% |
| 4 → 5★ | 160 | 技能强化 / Skill enhancement |
| 5 → 6★ | 320 | 终极形态外观 / Ultimate form appearance |

### 启灵 / Enlightenment

启灵是让真灵觉醒隐藏潜力的系统。

Enlightenment is the system for awakening a spirit's hidden potential.

- 每次启灵随机获得一条附加属性 / Each enlightenment grants a random bonus stat
- 可以重置启灵属性以重新培养 / Can reset enlightenment stats to retrain
- 高品质真灵启灵上限更高 / Higher-tier spirits have higher enlightenment caps

## 真灵培养小贴士 / Spirit Tips

1. **不要平均培养** — 集中资源培养一只主真灵 / Focus resources on one primary spirit
2. **每日免费召唤必做** — 每日有免费召唤次数 / Use daily free summons
3. **碎片合成** — 碎片足够时合成比召唤更划算 / Crafting from fragments is more efficient
4. **属性搭配** — 真灵属性应与主角属性互补 / Spirit stats should complement your character
5. **合体时机** — 在爆发阶段使用真灵合体 / Use spirit fusion during burst phases
