## 任务与活动系统 / Quests & Events System

主宰世界拥有丰富的任务和活动系统，是获取经验、装备、货币和各类培养材料的主要途径。合理安排每日活动可以最大化收益。

Master of the World features a rich quest and event system, which is the main way to obtain experience, equipment, currency, and various cultivation materials. Properly arranging daily activities maximizes your gains.

## 任务类型 / Quest Types

### 主线任务 / Main Storyline Quests

主线任务是推动游戏剧情的核心任务，提供大量经验和关键功能解锁。

Main quests drive the game story and provide massive experience and key feature unlocks.

- 不可跳过 / Cannot be skipped
- 推荐优先完成 / Recommended priority
- 自动寻路引导 / Auto-navigation guidance

### 支线任务 / Side Quests

支线任务提供额外的剧情体验和奖励。

Side quests provide additional story experience and rewards.

- 可选择性完成 / Optional
- 奖励包括经验、银两和道具 / Rewards include XP, Silver, and items
- 一些支线任务有前置条件 / Some have prerequisites

### 师门任务 / Sect Quests

每日可完成10次的师门任务，是稳定的经验来源。

Complete 10 times daily. A stable source of experience.

| 任务类型 / Type | 说明 / Description |
|---|---|
| 采集 / Gathering | 采集指定材料 / Gather specific materials |
| 送信 / Delivery | 将信件送达指定NPC / Deliver letters to NPCs |
| 讨伐 / Elimination | 击杀指定怪物 / Kill specific monsters |
| 巡逻 / Patrol | 在指定区域巡逻 / Patrol designated areas |

## 任务技巧 / Quest Tips

1. **组队完成** — 组队有经验加成，最高20% / Party XP bonus up to 20%
2. **双倍时间** — 关注游戏内的双倍经验活动通知 / Watch for double XP events
3. **活动日历** — 使用游戏内活动日历规划时间 / Use in-game event calendar
4. **离线挂机** — 离线时自动积累挂机收益 / Offline AFK accumulates rewards
5. **活跃度宝箱** — 达到一定活跃度可领取额外奖励 / Activity chests for extra rewards
6. **活动周历** — 每周有不同主题，奖励翻倍 / Weekly themes with doubled rewards
7. **小号养大号** — 可以创建小号获取额外资源 / Alt characters for extra resources
