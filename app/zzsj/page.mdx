## 世界观设定 / World Setting

主宰世界是一款基于中国古代神话与仙侠世界观的大型多人在线角色扮演游戏（MMORPG）。玩家将扮演一名修行者，从凡界出发，历经修真、渡劫、飞升等阶段，逐步探索广袤的仙侠世界。

Master of the World is a large-scale MMORPG rooted in ancient Chinese mythology and the xianxia (immortal hero) genre. Players take on the role of a cultivator, starting from the mortal realm and progressing through stages of cultivation, tribulation transcendence, and ascension, gradually exploring a vast xianxia world.

**游戏背景关键词 / Setting Keywords:**

- **修真** / Cultivation — 玩家通过修炼提升境界 / Players improve their realm through cultivation practices
- **渡劫** / Tribulation — 突破境界时需要经历天劫考验 / Breakthroughs require surviving heavenly tribulations
- **飞升** / Ascension — 达到最高境界后飞升至仙界 / Reaching the highest realm allows ascension to the celestial realm
- **仙魔之争** / Immortal vs Demon — 仙道与魔道之间的永恒战争 / The eternal war between the Immortal path and the Demon path

## 快速上手 / Quick Start Guide

1. **创建角色** / Create Character — 选择职业（御天、万剑、凌霄）和外观 / Choose your class and appearance
2. **完成新手引导** / Complete Tutorial — 跟随主线任务了解基础操作 / Follow the main storyline to learn basic controls
3. **提升等级** / Level Up — 主线任务→师门任务→抓鬼→雪山试炼 / Main quests → Sect quests → Ghost Hunt → Snow Trial
4. **加入公会** / Join a Guild — 达到15级后解锁公会系统 / Unlocked at level 15
5. **参与活动** / Join Events — 每日活动、限时活动获取丰厚奖励 / Daily and limited events for rich rewards

## 成长路径 / Progression Path

```
凡界 (Mortal Realm) → 练气 (Qi Refinement) → 筑基 (Foundation)
→ 金丹 (Golden Core) → 元婴 (Nascent Soul) → 化神 (Divine Transformation)
→ 合体 (Body Integration) → 大乘 (Mahayana) → 渡劫 (Tribulation)
→ 飞升 (Ascension)
```

每个境界分为 **前期** / Early、**中期** / Mid、**后期** / Late、**圆满** / Peak 四个小阶段。

Each realm is divided into four sub-stages: Early, Mid, Late, and Peak.
