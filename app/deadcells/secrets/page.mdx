# 隐藏要素 / Secrets & Hidden Content

死亡细胞有丰富的隐藏要素——从隐藏房间到特殊符文，从隐藏Boss到秘密结局。本章汇总所有需要探索才能发现的内容 / Dead Cells is filled with hidden content — from secret rooms to special runes, hidden bosses to secret endings. This chapter covers everything that requires exploration to discover.

---

## 符文系统 / Rune System

符文是永久解锁的能力——找到后所有存档都能使用 / Runes are permanent abilities — once found, usable across all save files.

| 符文 / Rune | 位置 / Location | 效果 / Effect | 解锁难度 / Difficulty |
|---|---|---|---|
| **藤蔓符文 / Vine Rune** | 有罪者大道(Promenade of the Condemned) — 击败精英怪"红皮大锤" / Defeat elite "Shieldbearer" | 在特定藤蔓处放下绳子——可以攀爬 / Deploy vine at specific points — climbable | ★ 简单——第一个符文 / Easy — first rune |
| **传送符文 / Teleportation Rune** | 猛毒下水道(Toxic Sewers) — 击败精英怪 / Defeat elite enemy | 在紫色传送石处传送——可以到达新区域 / Teleport at purple teleport stones — access new areas | ★★ 中等——需要找到入口 / Medium — need to find entrance |
| **爬墙符文 / Wall Climb Rune** | 雾锁港湾(Slumbering Sanctuary) — 击败精英怪"大锤" / Defeat elite "Golem" | 可以在特定墙壁上爬行——解锁大量新路线 / Climb specific walls — unlocks many new routes | ★★★ 较难——关卡本身有难度 / Harder — the level itself is tough |
| **冲击符文 / Impact Rune** | 旧下水道(Ancient Sewers) — 击败精英怪 / Defeat elite enemy | 落地时产生冲击波——可以打破地面的脆弱地板 / Create shockwave on landing — break fragile floors | ★★ 中等——需要传送符文先 / Medium — requires Teleportation Rune first |
| **探索符文 / Explorer Rune** | 钟楼(Clock Tower) — 击败精英怪 / Defeat elite "Pirate Captain" | 在小地图上显示全关卡的探索进度——不会漏隐藏房间 / Reveals full map exploration status — never miss secrets | ★★ 推荐尽早拿 / Recommended early |

**符文获取顺序建议**: 藤蔓 → 传送 → 冲击 → 爬墙 → 探索。前3个符文在新游戏+之前就可以全部拿到 / **Rune acquisition order**: Vine → Teleport → Impact → Wall Climb → Explorer. First 3 obtainable before new game+

## 隐藏区域 / Secret Areas

### 通过符文解锁的区域 / Rune-Accessible Areas

| 区域 / Area | 进入方式 / Entry Method | 奖励 / Reward |
|---|---|---|
| **藤蔓区域** | 使用藤蔓符文爬上去——每个关卡都有1-2个藤蔓点 / Use Vine Rune to climb — 1-2 per level | 通常有奖励宝箱或金币 / Bonus chests or gold |
| **传送区域** | 用传送符文到达的高台——通常有诅咒宝箱 / Teleport to high platforms — usually cursed chests | 诅咒宝箱给强力装备 / Cursed chests give powerful gear |
| **地下室** | 用冲击符文砸碎脆弱地板——进入地下 / Smash fragile floor with Impact Rune — enter underground | 隐藏Boss、宝箱或蓝图 / Hidden bosses, chests or blueprints |
| **墙壁路径** | 用爬墙符文进入高墙上的通道——跳过大量普通区域 / Wall climb into high wall passages — skip large normal areas | 快速通道+奖励 / Shortcut + rewards |

### 无符文隐藏区域 / Non-Rune Secret Areas

| 位置 / Location | 发现方法 / Discovery Method | 内容 / Content |
|---|---|---|
| **隐藏墙壁 / Hidden Walls** | 攻击看起来可疑的墙壁——有些墙壁可以打破 / Attack suspicious-looking walls — some are breakable | 金币、食物或图纸 / Gold, food, or blueprints |

| **计时门 / Timed Doors** | 在限定时间内到达特定位置(如2分钟内到某个关卡) / Reach specific location within time limit | 高级装备和金币 / High-level gear and gold |
| **无伤门 / Perfect Doors** | 整关无伤通过——到达关底时出现 / Complete a level without taking damage — appears at exit | 高级图纸和装备 / Premium blueprints and gear |

|---|---|---|---|

| **湖之女 / The Lady of the Lake** | 湖(水域关卡) / Lake (water level) | 特殊事件触发——需要特定武器或符文 / Special event trigger — need specific weapon or rune | ★★ 简单——更多是彩蛋 / Easy — more of an easter egg |

## 蓝图收集 / Blueprint Collection

蓝图用于解锁新装备——找到后带到收藏家处花费细胞解锁 / Blueprints unlock new gear — find and bring to the Collector, spend cells to unlock.

| 获取方式 / Method | 概率 / Chance | 说明 / Description |
|---|---|---|
| **击杀敌人掉落** | 0.4%-1.7% | 特定敌人掉落特定蓝图——需要反复刷 / Specific enemies drop specific blueprints — requires farming |
| **精英怪掉落** | 100% | 每个精英怪首次击杀必定掉落一个蓝图 / First elite kill in a run guarantees a blueprint drop |
| **隐藏房间** | 50%-100% | 隐藏房间内通常固定有蓝图 / Secret rooms usually have guaranteed blueprints |
| **Boss掉落** | 100% | 每个Boss首次击杀必定掉落一个蓝图 / First Boss kill per run guarantees a blueprint |
| **诅咒宝箱** | 100% | 诅咒宝箱必定包含一个蓝图 / Cursed chests always contain a blueprint |
| **无伤门奖励** | 100% | 无伤通过一关后必定获得一个蓝图 / Perfect level clear guarantees a blueprint |

**刷蓝图推荐关卡**: 监狱深处(Prison Depths)的精英怪 + 有罪者大道的盾兵——这两个最容易反复刷 / **Best farming levels**: Prison Depths elites + Promenade shieldbears — easiest to repeat farm.
