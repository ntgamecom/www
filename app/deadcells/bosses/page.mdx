# BOSS攻略 / Boss Strategies

死亡细胞的Boss战节奏快、伤害高——但每个Boss都有明显破绽。掌握攻击模式即可无伤。注意Boss房前有回血泉——满血进战 / Dead Cells boss fights are fast and deadly — but every boss has clear tells. Learn the patterns for no-hit wins. Use the heal fountain before entering.

---

## 主线Boss / Main Bosses

### 看守者 / The Concierge

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 黑色大桥 / Black Bridge |
| **难度 / Difficulty** | ★☆☆ — 最简单的Boss，用来教新手看前摇 / Tutorial boss for learning tells |
| **弱点 / Weakness** | 远程风筝、盾反冲刺 / Ranged kiting, parry his charge |

看守者是死亡细胞的第一个Boss——他的攻击模式非常固定。**P1(100%-70%HP)**: 慢速近战——他出手后翻滚到背后打2-3刀。他转身就是攻击信号——不要贪刀。**P2(70%-30%)**: 增加地面火焰AOE——站在火焰范围外继续输出，火焰持续约5秒。**P3(30%-0%)**: 增加远程火球——2-3发连射，翻滚轻松躲避 / The Concierge is Dead Cells' first boss — very predictable patterns. P1 (100%-70%): Slow melee — roll behind after his attack, 2-3 hits. His turn = incoming attack — don't get greedy. P2 (70%-30%): Ground fire AoE — stand outside. Fire lasts ~5s. P3 (30%-0%): Fire orbs 2-3 shot volley — easy roll dodge.

**盾反技巧**: 他的冲刺是最容易反的技能——看到冲刺动作→按盾牌→眩晕+大硬直→全力输出2-3秒。完美盾反后可以接一个回血/技能 / **Parry tip**: His charge is the easiest parry — see charge windup → press shield → stun + punish window. Follow up with heal or skill use.

**推荐Build**: 任何Build都能过——新手建议用暴虐(红)近战Build，不依赖特殊技能。带盾牌可以大幅降低难度 / **Any build works** — beginners should use Brutality (red) melee. A shield makes this fight significantly easier.

### 肿胀眼魔 / The Conjunctivius

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 沉睡庇护所 / Insufferable Crypt |
| **难度 / Difficulty** | ★★☆ |
| **弱点 / Weakness** | 高输出Build、快速破坏触手 / High DPS, destroy tentacles fast |

飞行Boss——触手攻击模式。场地有高低平台可以利用。**触手阶段**: 四根触手从四角伸出——优先打掉。触手会发射弹幕(弧线)。每根触手约3-4刀可摧毁。**P2(60%-0%)**: 巨大眼球发射激光+掉落毒液。激光有前摇——看到蓄力就躲到平台后。毒液在地面形成毒池——跳跃躲避 / Flying boss with tentacle attack patterns. Has platforms. **Tentacle phase**: 4 tentacles from corners — kill first. Each takes 3-4 hits. They fire curved projectiles. **P2 (60%-0%)**: Giant eye laser + poison drops. Laser has windup — hide behind platform. Poison pools on ground — jump over.

**推荐策略**: 远程Build最好打——触手阶段站在远处安逸输出，弹幕容易躲。近战Build需要全程跑动+跳跃——利用平台高度差躲避毒池。带"蜘蛛符文"可以爬墙到安全位置 / **Best strategy**: Ranged build is safest — stand at distance and DPS tentacles. Melee builds need constant movement + jumping — use platforms. Spider Rune lets you climb to safe positions.

**常见失误**: 1) 打触手时站在场地中央——被四面夹击。2) 激光时躲在平台后但露出了头——激光判定比看上去大。3) P2毒池+弹幕同时出现时慌乱跳错方向 / **Common mistakes**: 1) Standing center during tentacles — surrounded. 2) Head exposed behind platform during laser. 3) Panic jumping into poison during P2 bullet hell.

### 时光守护者 / Time Keeper

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 时钟塔 / Clock Tower |
| **难度 / Difficulty** | ★★★ — 死亡细胞的第一个真正的操作检测 / First real skill check |
| **弱点 / Weakness** | 盾反、高爆发Build / Parry, burst damage build |

快速Boss——她的攻击速度极快，动作流畅。**P1(100%-70%)**: 连击刀法——3-4连击后短暂硬直。节奏是快-快-慢-快。格挡或翻滚全部4下后就是你的输出时间。**P2(70%-30%)**: 增加时间减速(画面变蓝)——她的速度变得更快，你的动作变慢。这时候翻滚时机要比平时提前——因为你的翻滚变慢了。**P3(30%-0%)**: 增加全屏弹幕+瞬移——最难阶段。弹幕有规律：先两边→中间→四处散开 / Speed boss — lightning-fast attacks. P1 (100%-70%): 3-4 chain strikes then stun. Rhythm: fast-fast-slow-fast. Parry/roll all 4 then punish. P2 (70%-30%): Time slow (blue screen) — she's faster, you're slower. Roll EARLIER than usual. P3 (30%-0%): Full-screen projectiles + teleport. Projectile pattern: sides → middle → spread.

**盾反核心**: 她的所有攻击都可以盾反——这是打这个Boss最稳的方式。节奏是快-快-停-快——注意那个"停"就是假动作，再等一下才是第四下。练习盾反这个Boss可以极大提升你的整体盾反水平 / **Parry is key**: All her attacks are parryable. Rhythm: fast-fast-pause-fast — that "pause" is a feint, wait for the 4th hit. Practice parrying this boss — it will dramatically improve your overall parry skill.

**推荐Build**: 盾牌+高暴发武器(双匕首/巨镰)。战术(紫)Build也可以——用冰弓远程减速她，保持距离输出。变异推荐带"反击"——盾反后+80%伤害 / **Recommended build**: Shield + burst weapon (Twin Daggers/Scythe). Tactics (Purple) works too — Ice Bow for slow. Mutation: "Counterattack" for +80% damage after parry.

### 国王之手 / Hand of the King

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 王座之巅 / Throne Room |
| **难度 / Difficulty** | ★★★★ (0细胞) / ★★★★ (0BC) |
| **弱点 / Weakness** | 盾反、高机动性 / Parry, high mobility |

0细胞最终Boss——大锤+冰/火地面攻击。**P1(100%-70%)**: 大锤横扫+砸地。翻滚时机：看到起手→延迟0.5秒→翻滚。**冰阶段(70%-40%)**: 地面结冰+减速——跳跃可以避免被冰住但会消耗体力。不跳的话地面减速持续约3秒。**火阶段(40%-20%)**: 地面裂缝喷火——站在喷火之间的安全区域输出。喷火顺序固定——观察后预判。**P3(20%-0%)**: 增加远程冲击波——跳跃躲避。同时他会更频繁地攻击 / 0BC final boss. Giant hammer + ice/fire floor attacks. P1 (100%-70%): Hammer sweep + slam. Roll timing: see windup → 0.5s delay → roll. Ice phase (70%-40%): Ground ice + slow — jump to avoid (costs stamina). Fire phase (40%-20%): Floor jets — stand in safe zones between jets. Pattern is fixed. P3 (20%-0%): Shockwaves — jump to dodge. More aggressive.

**盾反核心**: 重击都可以盾反——反到后打一套连招。他的冲刺重击最明显——看到黑烟冲刺→完美盾反→输出。但是注意他的有些攻击带延迟——不要太早按盾 / **Parry is key**: Heavy hits are all parryable. His charge attack has the clearest tell — black smoke charge → parry → punish. Be careful of delayed attacks — don't parry too early.

**推荐Build**: 盾牌是必须的——这个Boss没有盾会难很多。暴虐(红)Build带闪光弹可以瞬移躲他的地面AOE。变异推荐"复仇"+ "反击" / **Recommended build**: Shield is mandatory. Brutality (red) with Phaser to teleport-dodge ground AOEs. Mutations: "Vengeance" + "Counterattack".

### 巨人 / The Giant (DLC/1细胞+)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 守护者避难所 / Guardian's Haven |
| **难度 / Difficulty** | ★★★★★ — 死亡细胞最难Boss之一 / One of the hardest in Dead Cells |
| **弱点 / Weakness** | 远程高DPS——近战在这个Boss非常危险 / High DPS ranged — melee is extremely risky here |

1细胞后可选最终Boss——比国王之手难得多。眼睛激光+地面手拍+弹幕——全屏攻击密度极高。**核心机制**: 巨人的弱点是**眼睛**——只有眼睛受到全额伤害。攻击他身体部位的伤害极低(约10%)。所以必须瞄准眼睛输出 / 1BC+ optional final boss — much harder than HotK. Eye laser + ground slams + projectiles — dense screen-wide attacks. **Core mechanic**: Weak spot is the **EYE** — only the eye takes full damage. Body parts take ~10% damage. Must aim for the eye.

**阶段划分**: 
- **P1(100%-60%)**: 眼睛发射激光(蓄力→红色瞄准线→发射——躲在最低平台可以躲开)+单手拍地(跳跃或翻滚)
- **P2(60%-30%)**: 双手交替拍地+弹幕。弹幕从两边向中间扫射——站在中间缝隙可以躲
- **P3(30%-0%)**: 激光+双手拍+弹幕全开——最疯狂的阶段。保持高度集中

**攻击节奏**: 
- 眼睛激光：蓄力约2秒→发射→4秒冷却
- 手拍：每只手拍地→抬起→再拍——循环2次后休息
- 召唤小手：P2开始每隔一段时间召唤——必须快速清理，否则会被围死

P1 (100%-60%): Eye laser (charge → red line → fire — hide behind lowest platform) + single hand slam. P2 (60%-30%): Dual hand slam + projectile barrage from sides toward center — stand in center gap. P3 (30%-0%): Everything at once — maximum concentration required.

**推荐Build**: 战术(紫)远程Build是唯一推荐——用多重射击弓+集束箭/火焰箭。站在最左侧或最右侧平台输出眼睛。变异推荐"弹药"(+1远程弹药)+ "死灵术"(回血)/ **Recommended build**: Tactics (Purple) ranged only. Multi-knock bow + cluster/flammable arrows. Stand on leftmost or rightmost platform aiming at eye. Mutations: "Ammo" + "Necromancy" (heal).

## 隐藏Boss / Secret Bosses

### 女王 / The Queen (DLC2)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 女王堡 / Queen's Castle |
| **难度 / Difficulty** | ★★★★★ |
| **弱点 / Weakness** | 盾反——她的连招全部可反 / Parry — her entire combo chain is parryable |

速度比时光守护者更快——连招长达8-10下。**核心策略**: 不要主动进攻——等她打完一套再打2-3刀。她的攻击节奏是多变的——每套连招的长度不同。盾反是唯一稳过的方案 / Faster than Time Keeper — combos up to 8-10 hits. **Core strategy**: Don't attack first — wait for her to finish a combo, then 2-3 hits. Her combo length varies. Parry is the only reliable approach.

## Boss通用技巧 / General Boss Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **盾反 > 翻滚 / Parry > Dodge** | 盾反免伤+创造输出窗口——翻滚只是躲开。打Boss时带盾永远是正解 / Parry negates damage AND creates windows — dodge just avoids. Always carry a shield for bosses |
| **进Boss房前满血 / Enter at Full HP** | Boss房前的回血泉用掉——不要省。0细胞时满血可能也会死——残血进=自杀 / Use the heal fountain before every boss. Even at 0BC, half HP can mean death |
| **变异切换 / Swap Mutations** | 打Boss前换专门打Boss的变异。推荐："死灵术"(击杀回血)、"反击"(盾反后+80%)、"不屈"(+20%HP)。打完Boss再换回推图用变异 / Switch to boss-specific mutations before entering. Recommended: "Necromancy" (heal on kill for adds), "Counterattack" (+80% after parry), "Dead Inside" (+20% HP) |
| **节奏观察 / Learn the Rhythm** | 每个Boss的攻击节奏固定——前3次死亡是用来学习的。观察→记住→应对。每次死后问自己：我刚才为什么死？下次怎么躲？/ Every boss has fixed attack rhythm — your first 3 deaths are for learning. Observe → remember → react. After each death ask: why did I die? How do I avoid it next time? |
| **食物管理 / Food Management** | Boss房前商店可能卖食物——留够钱。如果钱不够，回前面关卡找隐藏食物 / Shops before bosses often sell food — save gold. If short, backtrack for hidden food |
| **传奇装备 / Legendary Items** | 传奇装备(金色边框)自带一个额外变异效果——打Boss前如果有商店，看看有没有传奇武器卖 / Legendary items (gold border) have an innate mutation effect — check shops before bosses |
| **Boss细胞变化 / Boss Changes by BC** | 1细胞开始Boss新增攻击模式。2细胞Boss速度加快。3细胞Boss伤害翻倍。4-5细胞Boss全程加速+新增秒杀技——之前练好的节奏需要重新适应 / From 1BC, bosses gain new attacks. 2BC: faster. 3BC: double damage. 4-5BC: full speed + new oneshot mechanics — relearn patterns |
