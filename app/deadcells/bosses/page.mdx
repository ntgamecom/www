# BOSS攻略 / Boss Strategies

死亡细胞的Boss战节奏快、伤害高——但每个Boss都有明显破绽。掌握其攻击模式即可无伤 / Dead Cells boss fights are fast and deadly — but every boss has clear tells. Learn the patterns for no-hit wins.

---

## 主线Boss / Main Bosses

### 看守者 / The Concierge

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 黑色大桥 / Black Bridge |
| **难度 / Difficulty** | ★☆☆ |
| **弱点 / Weakness** | 远程风筝、盾反他冲刺 / Ranged kiting, parry his charge |

**打法:**
- 最简单的Boss——他会冲向你+近战横扫 / Easiest boss — charges + melee sweep
- **P1**: 慢速近战——他出手后翻滚到背后打2-3刀 / Slow melee — roll behind after his attack, 2-3 hits
- **P2**: 增加地面火焰AOE——站在火焰范围外继续输出 / Adds ground fire AoE — stand outside fire, keep hitting
- **P3**: 增加远程火球——翻滚躲避 / Adds fire orbs — roll to dodge
- **盾反**：他的冲刺可以用盾牌反制——眩晕+大量伤害 / Parry his charge — stun + massive damage
- **无伤技巧**: 不要贪刀——他转身就是攻击信号 / No-hit tip: Don't get greedy — his turn = attack incoming

---

### 肿胀眼魔 / The Conjunctivius

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 沉睡庇护所 / Insufferable Crypt |
| **难度 / Difficulty** | ★★☆ |
| **弱点 / Weakness** | 高输出Build、快速破坏触手 / High DPS, destroy tentacles fast |

**打法:**
- 飞行Boss——触手攻击模式。场地有平台 / Flying boss — tentacle attack patterns. Has platforms
- **触手阶段**: 四根触手——优先打掉。触手会喷弹幕 / Tentacle phase: 4 tentacles — kill first. They fire projectiles
- **红光阶段**: 巨大眼球发射激光——躲在平台后 / Giant eye laser — hide behind platforms
- **P2**: 掉落毒液——地面毒池需要跳跃躲避 / Drops poison — jump over ground pools
- **推荐**: 远程Build最好打——触手阶段可以站在远处输出 / Ranged build is easiest — safe distance during tentacle phase

### 时光守护者 / Time Keeper

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 时钟塔 / Clock Tower |
| **难度 / Difficulty** | ★★★ |
| **弱点 / Weakness** | 盾反、高爆发Build / Parry, burst damage build |

**打法:**
- **快速Boss**——她的攻击速度极快，动作流畅 / Speed boss — lightning-fast attacks, fluid movement
- **P1**: 连击刀法——3-4连击后硬直。格挡或翻滚 / Chain sword strikes — 3-4 combo then stun. Block or roll
- **P2**: 增加时间减速(画面变蓝)——她速度更快，你变慢 / Adds time slow (blue screen) — she's faster, you're slower
- **P3**: 增加全屏弹幕+瞬移——最难阶段 / Adds full-screen projectiles + teleport — hardest phase
- **盾反**: 她的所有攻击都可以反——节奏是快-快-停-快 / All attacks are parryable — rhythm is fast-fast-pause-fast
- **推荐**: 盾牌是这个Boss的答案——带盾反一切 / Shield is the answer — parry everything

### 国王之手 / Hand of the King

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 王座之巅 / Throne Room |
| **难度 / Difficulty** | ★★★★ (0细胞) / ★★★★ (0BC) |
| **弱点 / Weakness** | 盾反、高机动性 / Parry, high mobility |

**打法:**
- 0细胞最终Boss——大锤+冰/火地面攻击 / 0BC final boss — giant hammer + ice/fire ground attacks
- **P1**: 他使用大锤横扫+砸地——翻滚时机：看到起手→0.5秒后滚 / Giant hammer sweep + slam — roll timing: see windup → 0.5s delay → roll
- **冰阶段**: 地面结冰+减速——跳起来避免被冰住 / Ice floor + slow — jump to avoid freezing
- **火阶段**: 地面喷火——站在安全区域输出 / Floor fire jets — stand in safe zones
- **P3**: 增加远程冲击波——跳跃躲避 / Adds shockwaves — jump to avoid
- **盾反核心**: 他的重击都可以盾反——成功反到后可以打一套 / Shield parry is key — parry his heavy hits for big punish windows

### 巨人 / The Giant (DLC/1细胞+)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Location** | 守护者避难所 / Guardian's Haven |
| **难度 / Difficulty** | ★★★★★ |
| **弱点 / Weakness** | 远程高DPS / High DPS ranged build |

**打法:**
- 1细胞后可选最终Boss——比国王之手难 / 1BC+ optional final boss — harder than Hand of the King
- 眼睛激光+地面手拍+弹幕——全屏高密度攻击 / Eye laser + ground slams + projectiles — dense screen-wide attacks
- **弱点是眼睛**——只有眼睛受到全额伤害 / Weak spot is the EYE — only the eye takes full damage
- **推荐**：战术(紫)Build用远程武器输出眼睛 / Tactics (Purple) build — ranged weapon focus on eye
- 他召唤的小手必须快速清掉——否则被围死 / Small hands must be killed fast — or get overwhelmed

## Boss通用技巧 / General Boss Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **盾反 > 翻滚 / Parry > Dodge** | 盾反可以反大部分Boss攻击——不仅免伤还创造输出窗口 / Parry negates damage AND creates punish windows |
| **进Boss房前满血 / Enter at Full HP** | Boss房前有回血泉——不要浪费。保持满血进Boss / Use the heal fountain before boss door |
| **变异切换 / Swap Mutations** | 打Boss前换打Boss专用的变异——如"死灵术"、"反击" / Switch to boss-fight mutations — e.g., "Necromancy", "Counterattack" |
| **节奏观察 / Learn the Rhythm** | 每个Boss的攻击有固定节奏——死了不要紧，记住节奏下次就能躲 / Each boss has a fixed attack rhythm — death is fine, learn the pattern |
