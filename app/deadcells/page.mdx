## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **属性系统 / Stats** | **紫(战术)**:**红(暴虐)**:**绿(生存)**——三种属性对应不同武器和技能。集中加点比平均分配强得多 / **Purple(Tactics)**:**Red(Brutality)**:**Green(Survival)** — three stats for different weapon types. Focused > balanced |
| **Boss细胞 / Boss Cells** | 通关后解锁难度等级——0→5细胞。细胞越多敌人越强、奖励越好、新区域解锁 / NG+ cycles — 0→5 cells. More cells = harder enemies, better rewards, new biomes |
| **武器锻造 / Blueprint System** | 击败敌人/开宝箱获得蓝图——在收藏家处用细胞解锁。解锁后可掉落 / Kill/chest for blueprints — unlock at Collector with cells. Then available in runs |
| **传奇锻造所 / Legendary Forge** | 在收藏家处用细胞升级武器/技能品质——最高到S级。提高刷出高品质装备的概率 / Upgrade gear quality with cells — up to S-tier. Higher quality = more damage affixes |
| **变异 / Mutations** | 每关选择被动技能（最多3个）——可随时在关卡内重置。构筑核心 / Passive skills (max 3) — resettable within a run. Core of your build |
| **卷轴 / Scrolls** | 关卡中的属性卷轴——选对应属性增加伤害（乘法叠加）。全收集是通关关键 / Stat scrolls in biomes — multiplicative damage scaling. Full collection = key to winning |

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **集中加点 / Focus Your Stats** | 不要平均分配属性——全加一种。死亡细胞的伤害是乘法叠加——紫/红/绿选一主加 / Don't spread stats — put ALL into one color. Damage multiplies — pick Purple/Red/Green and commit |
| **2** | **跑图全探索 / Full Explore Each Biome** | 每个关卡都有卷轴——漏一个卷轴=少30%+伤害。时间奖励没有卷轴重要 / Every biome has scrolls — missing one = 30%+ less damage. Time bonus is never worth skipping scrolls |
| **3** | **盾牌是你的朋友 / Shields Are Great** | 很多人忽略盾牌——但盾牌格挡可以免除所有伤害+眩晕敌人。举盾时机是关键 / Many ignore shields — but parrying negates all damage + stuns enemies. Timing is key |
| **4** | **选对路线 / Pick the Right Route** | 不是所有路线都一样——某些路线卷轴多、某些路线Boss简单。了解路线等于赢了50% / Not all routes are equal — some have more scrolls, some easier bosses. Route knowledge = 50% of winning |
| **5** | **不要怕死 / Don't Fear Death** | 死是这游戏的一部分——每次死亡你都会解锁更多武器/变异。死亡=进步 / Death is part of the game — each death unlocks more weapons/mutations. Death = progress |
| **6** | **早期投资回收 / Invest in Recycling Early** | 用细胞解锁"回收"技术——分解装备获得金钱。钱在商店买卷轴和好装备 / Unlock "Recycling" tech early — break down gear for gold. Gold buys scrolls and gear in shops |
| **7** | **自定义模式很友好 / Custom Mode is Fair** | 如果默认模式太难——自定义模式可以关闭某些武器、调整难度，奖杯不影响 / If default is too hard — Custom Mode lets you ban weapons, tweak difficulty. Achievements still work |
