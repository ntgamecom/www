# 新手入门 / Beginner Guide

死亡细胞的战斗节奏快、死亡惩罚轻——但系统比看起来深得多。本章帮你打好基础 / Dead Cells is fast-paced with light death penalties — but deeper than it looks. This chapter builds your foundation.

---

## 核心操作 / Core Controls (PC)

| 操作 / Action | 键位 / Key | 说明 / Description |
|------|:----:|------|
| **攻击 / Attack** | 鼠标左键 / Left click | 主武器攻击 / Main weapon attack |
| **技能 / Skill** | 右键 / Right click | 副武器或技能 / Secondary weapon or skill |
| **翻滚 / Roll** | Shift | 翻滚有无敌帧——核心闪避技能 / Roll has i-frames — core dodge |
| **跳跃 / Jump** | 空格 / Space | 二段跳可解锁 / Double-jump is unlockable |
| **互动 / Interact** | E | 开门/捡东西/对话 / Open doors, pick up items |
| **切换武器 / Swap** | Q | 切换到背包第二把武器 / Switch to backpack weapon |
| **喝药 / Heal** | R | 使用血瓶——限量，Boss细胞越多限制越大 / Use flask — limited, more restricted at higher BC |
| **架盾 / Shield** | 按住右键 / Hold right click | 格挡——精准格挡(Parry)可免伤+眩晕 / Block — parry = no damage + stun |
| **头锤 / Homunculus** | 长按R / Hold R | 远程头锤——可拉怪/触发机关/Eltie无伤控 / Remote rune — pull enemies, trigger switches, control elites |

## 三属性系统详解 / Three Stats

| 属性 / Stat | 对应颜色 / Color | 适合武器 / Weapons | 核心玩法 / Playstyle |
|:----:|:----:|------|------|
| **暴虐 / Brutality** | 红 / Red | 近战武器 + 部分技能 / Melee + some skills | 贴脸爆发——高速近战 / Aggressive close-range burst |
| **战术 / Tactics** | 紫 / Purple | 远程武器 + 陷阱/技能 / Ranged + traps/skills | 远程风筝——高伤脆皮 / Ranged kiting — glass cannon |
| **生存 / Survival** | 绿 / Green | 重型武器 + 盾牌 / Heavy weapons + shields | 坦克——高血量+盾反 / Tanky — high HP + shield parry |

> **核心原则**：**全加一种**。死亡细胞的伤害是乘法叠加——紫色/红色/绿色选一种主加，另一个两个属性不加。分散加点=伤害不够 / **Golden Rule**: **Put ALL points into one color**. Damage multiplies within a color — pick one and commit. Splitting = not enough damage

## 细胞升级优先级 / Cell Upgrade Priority

| 升级项目 / Upgrade | 细胞数 / Cells | 优先级 / Priority | 理由 / Why |
|:----:|:----:|:----:|------|
| **血瓶充能 / Flask Charge** | 15 | ★★★★★ | 多一次回血——生存第一 / Extra heal — survival first |
| **回收 / Recycling** | 25 | ★★★★★ | 分解装备得钱——商店买卷轴 / Break gear for gold — buy scrolls |
| **锻造初始装备 / Starting Gear** | 30 | ★★★★ | 开局指定武器——减少随机性 / Start with chosen weapon — less RNG |
| **金币背包 / Gold Backpack** | 20 | ★★★★ | 死亡保留金币——经济系统核心 / Keep gold on death — economy core |
| **随机突变槽 / Random Mutation** | 45 | ★★★ | 开局多一个变异——构筑灵活度 / Start with extra mutation — build flexibility |
| **传奇锻造所 / Legendary Forge** | 100+ | ★★★★★ | 后期最重要——提高装备品质 / The most important late game — gear quality up |

## 新手常见误区 / Common Mistakes

| 误区 / Mistake | 正确做法 / Correct |
|------|------|
| 三属性平均加点 / Spreading stats | 全加一种颜色——最多加一点血到其他颜色 / Pour all into one color — at most one point in others for HP |
| 盾牌不用 / Ignoring shields | 盾反免伤+眩晕——至少带一个盾牌 / Parry negates damage + stuns — carry at least one shield |
| 赶时间拿奖励门 / Rushing for timed doors | 时间奖励只有金钱和低级装备——不如全探索拿卷轴 / Timed doors give gold/low gear — full explore for scrolls is better |
| 不变异 / Not picking mutations | 每关在突变祭坛选一个——打Boss前记得切换 / Pick one per biome — switch before boss fights |
| 不用药省到Boss / Hoarding heals for boss | 血量满时不掉补给——残血了赶紧喝 / Full HP = no flask drops — heal when low, don't hoard |
