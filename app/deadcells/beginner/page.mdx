# 新手入门 / Beginner Guide

死亡细胞的战斗节奏快、死亡惩罚轻——但系统比看起来深得多。本章帮你打好从0细胞到5细胞的基础 / Dead Cells is fast-paced with light death penalties — but deeper than it looks. This chapter builds your foundation from 0BC to 5BC.

---

## 核心操作 / Core Controls (PC)

| 操作 / Action | 键位 / Key | 说明 / Description |
|------|:----:|------|
| **攻击 / Attack** | 鼠标左键 / Left click | 主武器攻击 / Main weapon attack |
| **技能 / Skill** | 右键 / Right click | 副武器或技能 / Secondary weapon or skill |
| **翻滚 / Roll** | Shift | 翻滚有无敌帧——核心闪避技能。学会翻滚时机比什么都重要 / Roll has i-frames — core dodge. Mastering roll timing is more important than any weapon |
| **跳跃 / Jump** | 空格 / Space | 二段跳可解锁(蜘蛛符文后)。跳跃可以躲地面攻击和陷阱 / Double-jump unlockable (Spider Rune). Jump dodges ground attacks and traps |
| **互动 / Interact** | E | 开门/捡东西/对话。注意隐藏互动点——墙壁裂缝可以用互动检查 / Open doors, pick up items. Check wall cracks with interact |
| **切换武器 / Swap** | Q | 切换到背包第二把武器——用于切换远程/近战或双近战Build / Switch to backpack weapon — for ranged/melee or dual melee builds |
| **喝药 / Heal** | R | 使用血瓶。每关只能在Boss房前补一次——规划使用时机。0-4细胞可以补，5细胞全程不能补！/ Use flask. Replenish once per biome before boss. At 5BC, no refills at all! |
| **架盾 / Shield** | 按住右键 / Hold right click | 防御——精准格挡(Parry)免伤+眩晕。**新手最容易忽略的核心技巧**——带个盾牌游戏难度减半 / Block — parry negates damage + stuns. **Most overlooked core mechanic** — carrying a shield halves the difficulty |
| **头锤 / Homunculus** | 长按R / Hold R | 远程头锤——可以拉怪、触发机关、无伤控精英。高细胞难度下可以用来引怪一个个杀 / Remote攻击 — pull enemies, trigger switches, control elites. At high BC, use to aggro enemies one by one |

## 三属性系统详解 / Three Stats

选择一种主属性并**全加**——这是死亡细胞最重要的一条规则 / Pick ONE main color and **pour everything into it** — Dead Cells' most important rule.

| 属性 / Stat | 颜色 / Color | 适合武器 / Weapons | 核心玩法 / Playstyle |
|:----:|:----:|------|------|
| **暴虐 / Brutality** | 红 / Red | 近战武器 + 部分技能 / Melee + some skills | 贴脸爆发——高攻速+高伤害但风险大。适合喜欢快节奏的玩家 / Aggressive close-range — fast + high damage but risky |
| **战术 / Tactics** | 紫 / Purple | 远程武器 + 陷阱/技能 / Ranged + traps/skills | 远程风筝——伤害最高但最脆(血量减半)。高细胞难度下"玻璃大炮" / Ranged kiting — highest damage but lowest HP. "Glass cannon" at high BC |
| **生存 / Survival** | 绿 / Green | 重型武器 + 盾牌 / Heavy weapons + shields | 坦克——血量两倍+盾反。容错率最高，适合新手过渡到高细胞 / Tankiest — 2x HP + shields. Highest margin for error, best for learning high BC |

> **核心原则**：全加一种！每3点同属性=伤害翻倍。分散加点=最终Boss打不动。想换流派就重新开一局——不要在一局内分摊 / **Golden Rule**: All into ONE! Every 3 points in same color = double damage. Splitting = can't kill final boss. Start a new run to try a different color.

**新手属性选择建议**:
- **0-1细胞**: 推荐生存(绿)——容错率高，可以多犯错
- **2-3细胞**: 暴虐(红)——伤害和生存的平衡，适合大多数玩家
- **4-5细胞**: 战术(紫)——最高伤害是唯一出路，因为敌人血太厚了

**Beginner color advice**: 0-1BC: Survival (green) — highest margin for error. 2-3BC: Brutality (red) — balanced. 4-5BC: Tactics (purple) — highest damage is the only way.

## 武器类型速览 / Weapon Type Overview

| 类型 / Type | 示例 / Example | 特点 / Traits |
|------|------|------|
| **单手武器 / One-Handed** | 铁剑、双匕首、血刃 / Iron Sword, Twin Daggers, Blood Sword | 攻速快、可副手持盾——最全面的武器类型 / Fast, can off-hand shield — most versatile |
| **双手武器 / Two-Handed** | 大剑、巨镰、雷锤 / Broadsword, Scythe Claw, Thunder Hammer | 伤害高、范围大但攻速慢——适合生存Build / High damage, wide arcs but slow — for Survival builds |
| **远程武器 / Ranged** | 弓、弩、吹箭 / Bow, Crossbow, Blowpipe | 安全输出——适合战术Build。注意弹药管理 / Safe damage — for Tactics builds. Watch ammo |
| **盾牌 / Shield** | 突击盾、巨盾 / Assault Shield, Cudgel | 格挡+盾反——**每个Build都建议带**。新手最大的误区是不拿盾 / Parry tool — **recommended for every build**. Biggest beginner mistake is skipping shields |
| **技能 / Skills** | 闪光弹、捕兽夹、炮台 / Phaser, Wolf Trap, Turret | 爆发/控制/辅助——看Build需求选 / Burst/control/utility — pick based on build |

## 细胞升级优先级 / Cell Upgrade Priority

前期细胞有限——花对地方能让你少死100次 / Early cells are scarce — smart spending saves 100+ deaths.

| 升级 / Upgrade | 细胞 / Cells | 优先级 / Priority | 理由 / Why |
|:----:|:----:|:----:|------|
| **血瓶充能 / Flask Charge** | 15 | ★★★★★ | +1瓶药=+1条命。最便宜最有价值的升级 / +1 heal = +1 life. Cheapest most valuable upgrade |
| **回收 / Recycling** | 25 | ★★★★★ | 分解装备得金币→商店买卷轴和装备。经济基础 / Salvage gear for gold → shop scrolls and gear. Economy foundation |
| **锻造初始装备 / Starting Gear** | 30 | ★★★★ | 开局指定武器——减少随机性，让Build更可控 / Start with specific weapon — less RNG, more consistent builds |
| **金币背包 / Gold Backpack** | 20 | ★★★★ | 死亡保留金币——越早解锁积累越多 / Keep gold on death — earlier unlock = more cumulative wealth |
| **随机突变槽 / Mutation Slot** | 45 | ★★★ | 开局多一个变异——构筑更灵活 / Extra starting mutation — more build flexibility |
| **传奇锻造所 / Legendary Forge** | 100+ | ★★★★★ | 提升装备基础品质——后期最重要，但前期先解锁上面那些 / Increases base gear quality — most important late game, but unlock others first |

## 新手常见误区 / Common Mistakes

| 误区 / Mistake | 正确做法 / Correct |
|------|------|
| 三属性平均加点 / Splitting stats evenly | 全加一种颜色——最多加1点到其他颜色只为那点HP / Pour all into one color — at most 1 point in others for the HP bonus |
| 盾牌不用 / Ignoring shields | 盾反免伤+眩晕+反击窗口——至少带一个盾牌。单手武器+盾是最稳的组合 / Parry = no damage + stun + counter window. One-handed + shield is the safest combo |
| 赶时间拿奖励门 / Rushing for timed doors | 时间奖励只有金钱和低级装备——不如全探索拿卷轴和秘密。杀光所有敌人=更多细胞和掉落 / Timed doors give gold and low gear — full exploration = more scrolls and secrets. Kill all enemies = more cells and drops |
| 不变异切换 / Not swapping mutations | 每关在突变祭坛选一个——打Boss前记得切换为Boss战变异。推图用生存变异，打Boss用输出变异 / Pick one per biome at mutation altar — swap to boss mutations before boss door. Survival mutations for biomes, damage mutations for bosses |
| 不用药省到Boss / Hoarding heals | 血量满时不掉药水补给——残血了赶紧喝，空血瓶才能掉新的。到Boss房前只剩半血=大概率翻车 / Full HP = no flask drops. Heal when low — empty flask = new drops. Entering boss room at half HP = high chance of death |
| 不看地图 / Ignoring the map | 按Tab打开地图——标注所有未探索区域、隐藏门、奖励门。全探索=全资源 / Press Tab for the map — marks unexplored areas, secret doors, and reward doors. Full clear = full resources |

## Boss细胞系统 / Boss Cell System

| 细胞等级 / BC | 主要变化 / Key Changes | 建议准备 / Preparation |
|:----:|------|------|
| **0细胞** | 基础游戏——敌人少、伤害低、有药水补给 / Tutorial mode — fewer enemies, low damage, flask refills | 通关=解锁1细胞。目标：全探索，熟悉战斗 |
| **1细胞** | 敌人数量×2 + 新增精英敌人 / 2x enemies + new elites | 必须学会用盾 / Must learn to use shields |
| **2细胞** | 敌人速度+30% + 传送系统(你没看到的敌人可能会传送到你身边) / +30% enemy speed + teleport system | 移动Build成型 / Solidify your build |
| **3细胞** | 敌人伤害×2 + 诅咒宝箱更多 / 2x enemy damage + more cursed chests | 无伤意识——被打两下就死 / No-hit mentality — 2 hits = death |
| **4细胞** | 敌人全程加速 + 感染机制(站毒里持续掉血) + 敌人回血 / Constant enemy speed + malaise + enemy healing | 必须速战速决 / Must finish fights fast |
| **5细胞** | 全程感染加速 + 隐藏Boss"收藏家" + 没有药水补给 / Max malaise + secret boss "The Collector" + no flask refills | 最终挑战——完美操作是唯一出路 / Ultimate challenge — perfection is the only way |

## 从0到5细胞的学习路径 / 0BC to 5BC Learning Path

| 阶段 / Phase | 目标 / Goal | 关键技能 / Key Skill |
|:----:|------|------|
| **0-1细胞** | 通关——理解基础机制 / Beat the game — learn basics | 属性加点、武器选择、基础盾反 / Stat allocation, weapon choice, basic parry |
| **1-2细胞** | 学会盾反——无伤清图 / Master parry — no-damage biome clears | 盾反+翻滚配合。每个敌人都可以反——练习 / Parry + roll combos. Everything is parryable — practice |
| **2-3细胞** | 找到你的Build偏好 / Find your preferred build | 尝试所有三种颜色——找到最顺手的 / Try all 3 colors — find your comfort zone |
| **3-4细胞** | 无伤Boss——学会每个Boss的节奏 / No-hit bosses — learn every boss rhythm | Boss攻击模式记忆——不用看血条也知道Boss在打哪一套 / Memorize boss patterns — know the combo without looking |
| **5细胞** | 真结局通关——击败"收藏家" / True ending — beat "The Collector" | 一切完美的整合——属性、Build、操作、节奏 / Perfection of everything |

## 进阶技巧 / Advanced Tips

| 技巧 / Technique | 说明 / Description |
|---|---|
| **格挡取消 / Parry Cancel** | 某些攻击后摇可以用格挡取消——减少硬直时间 / Some attack recovery can be cancelled with parry — reduces animation lock |
| **滚取消 / Roll Cancel** | 翻滚可以取消攻击后摇——攻击→翻滚→攻击=更快连段 / Roll cancels attack recovery — attack→roll→attack = faster combos |
| **墙壁跳连击 / Wall Jump Combo** | 利用墙壁跳跃重置空中攻击——空中连段可以打出落地前的额外伤害 / Wall jump resets aerial attacks — mid-air combos deal extra damage before landing |
| **陷阱放置技巧 / Trap Placement** | 陷阱类武器(集束炸弹、捕兽夹)放在怪物出生点/必经之路上——效率最大化 / Place traps (Cluster Bomb, Bear Trap) on monster spawn points/pathways for max efficiency |
| **属性切换策略 / Stat Switching** | 拿到强力武器后可以灵活切换主属性——不一定要死守一开始的选择 / After finding a powerful weapon, flexibly switch main stat — don't lock into initial choice |

## 武器特性详解 / Weapon Mechanics

| 武器机制 / Mechanic | 说明 / Description |
|---|---|
| **暴击条件 / Crit Conditions** | 大部分武器有暴击条件(如背刺、空中攻击、连击后)——理解条件可翻倍伤害 / Most weapons have crit conditions (backstab, airborne, combo finisher) — meeting them doubles damage |
| **传奇武器 / Legendary** | 金色边框——自带全属性适应+强力特效——任何流派都可以用 / Gold border — auto-scales with all stats + powerful effect — usable in any build |
| **无色武器 / Colorless** | 无色边框——跟随你最高属性——灵活性最高 / Colorless border — scales with your highest stat — maximum flexibility |
| **武器词缀 / Affixes** | 武器上随机生成的附加效果——如火伤、毒伤、冰冻等——词缀可以决定武器强度 / Random bonus effects on weapons — fire, poison, freeze, etc. — affixes can make or break a weapon |

## 常见错误 / Common Mistakes

| # | 错误 / Mistake | 正确做法 / Correct |
|---|---|---|
| 1 | 平均分配属性 / Spreading stats | 全加一种——死亡细胞的伤害乘法叠加——分散=伤害大减 / ALL in one color — damage scales multiplicatively — spreading = massive damage loss |
| 2 | 跳过关卡内的卷轴 / Skipping scrolls | 每个关卡全探索拿齐卷轴——漏一个=少30%+伤害 / Full explore every biome for all scrolls — missing one = 30%+ less damage |
| 3 | 忽视盾牌 / Ignoring shields | 副手带盾牌——格挡是无伤通关的核心 / Equip shield in offhand — parrying is core to no-hit runs |
| 4 | 选错路线 / Wrong route | 根据当前Build选择关卡——暴虐走红色路线、战术走紫色路线 / Choose biomes based on build — brutality for red, tactics for purple |
| 5 | 不投资回收 / Skipping Recycling | 用细胞解锁回收技术——分解装备换钱——钱买卷轴和好装备 / Unlock Recycling with cells — break gear for gold — buy scrolls and gear |

**核心理念**: 集中加点+盾牌格挡+全探索=死亡细胞成功的三要素 / **Core**: Focus stats + shield parry + full explore = Dead Cells success formula
