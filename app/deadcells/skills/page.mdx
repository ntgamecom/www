# 技能系统 / Skills System

死亡细胞的技能系统由主动技能(手雷/炮塔/陷阱等)和被动变异组成。合理搭配技能和变异是Build成功的关键 / Dead Cells' skill system consists of active skills (grenades/turrets/traps/etc.) and passive mutations. Proper skill and mutation synergy is key to a successful build.

---

## 变异系统 / Mutation System

变异是被动技能——开局可以选择2个，每关结束时可以更换/增加。最多4个变异槽 / Mutations are passive skills — pick 2 at start, can change/add at end of each level. Max 4 mutation slots.

### 通用变异 / General Mutations

| 变异 / Mutation | 属性 / Stat | 效果 / Effect |
|---|---|---|
| **狂乱 / Frenzy** | 暴虐 / Brutality | 暴击时+30%攻速——持续5秒 / +30% attack speed on crit — 5s duration |
| **支撑 / Support** | 战术 / Tactics | 附近有炮塔/陷阱时+15%伤害 / +15% damage when near turret/trap |
| **复仇 / Vengeance** | 生存 / Survival | 受到伤害后+60%伤害——持续10秒(冷却30秒) / +60% damage after taking damage — 10s (30s cooldown) |
| **近距离作战 / Melee** | 暴虐 / Brutality | 近战击杀后恢复5%生命——受暴虐属性加成 / 5% HP restore on melee kill — scales with Brutality |
| **远距离 / Distance** | 战术 / Tactics | 远程击杀后恢复5%生命——受战术属性加成 / 5% HP restore on ranged kill — scales with Tactics |
| **死灵法术 / Necromancy** | 生存 / Survival | 击杀敌人恢复少量生命——受生存属性加成 / Small HP restore on kill — scales with Survival |
| **弹药 / Ammo** | 战术 / Tactics | 所有远程武器弹药+3——对弓箭和弩类关键 / +3 ammo for all ranged weapons — critical for bows |

## 技能升级与传奇等级 / Skill Upgrades & Legendary Levels

| 等级 / Level | 效果 / Effect |
|---|---|
| **I(基础)** | 技能基础效果 / Skill base effect |
| **II(强化)** | 伤害/持续时间+20% / Damage/Duration +20% |
| **III(进阶)** | 伤害/持续时间+40% + 可能增加额外效果(如冰冻手雷范围+50%) / Damage/Duration +40% + possible extra effect |
| **IV(稀有)** | 伤害/持续时间+60% + 额外的特效 / +60% + extra special effect |
| **V(传奇)** | 伤害/持续时间+100% + 传奇特效(如双管炮塔改为四向射击) / +100% + legendary effect (Double Crossb-o-matic becomes quadruple) |

**传奇技能获取**: 只能在4细胞难度以上的"传奇祭坛"或"诅咒宝箱"中获得——非常稀有但效果极强 / **Legendary skill acquisition**: Only from "Legendary Altars" or "Cursed Chests" in 4+ BC — extremely rare but game-changing.

| 来源 / Source | 概率 / Chance | 说明 / Description |
|---|---|---|
| **关卡中掉落** | 随机 / Random | 击败敌人或打开宝箱——随机获得技能 / Defeat enemies or open chests — random skill drops |

| **图纸解锁** | 固定掉落 / Fixed | 找到技能图纸后到收藏家处用细胞解锁 / Find blueprint → unlock at Collector with cells |

| **诅咒宝箱** | 100% | 诅咒宝箱(开箱后受诅咒——杀10个敌人解除)必定给高级技能 / Cursed chest (cursed on open — kill 10 enemies) always gives high-tier skill |

## 变异重铸 / Mutation Reforging

在5细胞难度下解锁的功能——允许在关卡中重铸变异 / Unlocked at 5 BC — allows reforging mutations during a level.

| 重铸方式 / Method | 消耗 / Cost | 效果 / Effect |
|---|---|---|
| **重铸一个变异** | 20,000金币 / 20,000 gold | 将选中的变异换成另一个随机变异 / Replace selected mutation with random one |
| **重铸所有变异** | 50,000金币 / 50,000 gold | 将所有变异全部重铸为新组合 / Replace all mutations with new set |
| **锁定变异** | 10,000金币 / 10,000 gold | 锁定某个变异——重铸时不会被改变 / Lock a mutation — won't be changed on reforging |

**重铸策略**: 如果打Boss前发现当前变异不够好(如泡塔流拿了近战变异)——花金币重铸 / **Reforging strategy**: If mutations don't match your build before a boss — spend gold to reforge.
