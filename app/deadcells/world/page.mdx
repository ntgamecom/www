# 关卡与路线 / Biomes & Routes

死亡细胞的路线选择直接决定你的Build强度和通关概率——最优路线要多拿卷轴和好装备，也要避开不适合当前Build的关卡 / Route choice in Dead Cells directly affects your power level and win rate — optimal routes maximize scrolls and gear quality while avoiding biomes that counter your build.

---

## 卷轴机制详解 / Scroll Mechanics

| 类型 / Type | 说明 / Description |
|------|------|
| **威力卷轴 / Power Scroll** | 三选一属性——全加主属性。双色卷轴(如暴虐+战术)可以选主属性。**重要**：永远不要平均加点——集中一个属性伤害翻倍最快 / Choose 1 of 3 stats — always pick your main color. Dual scrolls (e.g. Brutality+Tactics) let you stay on-color. **Crucial**: never split stats — focusing one color doubles damage every 3 points |
| **碎片卷轴 / Scroll Fragment** | 收集特定数量合成一个完整卷轴——全图探索确保不漏。每关固定碎片数量，有时在隐藏区域 / Collect X fragments for a full scroll — full clear to avoid missing. Fixed per biome, sometimes in secrets |
| **诅咒卷轴 / Cursed Scroll** | 给2级主属性但附加10层诅咒——10连杀后解除。**管理诅咒**：拿诅咒卷轴前确保附近有足够小怪可以杀，或者有盾牌/陷阱可以安全解诅咒 / +2 main stat but applies 10-hit curse. **Cursed scroll management**: ensure enough mobs nearby before picking, or use shield/traps for safe curse removal |
| **双色卷轴、三色卷轴 / Dual/Triple Scrolls** | 没有你的主属性时——选对整体Build最有益的副属性。暴虐副选战术(增加技能伤害)，生存副选暴虐(增加近战伤害) / When main color isn't available, pick the secondary that best supports your build |

> **卷轴分配原则**：全加一种属性！每3点同属性伤害翻倍——暴虐加近战伤害，战术加远程和技能伤害，生存加生命值和回复。全紫/全红/全绿是唯一正确选择 / **Golden Rule**: ALL into one color! Every 3 points in same color doubles damage. Brutality = melee, Tactics = ranged/skills, Survival = HP/healing. Commit fully.

## 各关卡敌人详解 / Enemy Breakdown by Biome

### 囚犯宿舍 / Prisoners' Quarters
- **僵尸 / Zombie**：最基础敌人——慢速，会复活一次。砍倒后不要马上走开，等它复活再补一刀 / Slow basic enemy, revives once. Wait for revive then kill again
- **弓箭手 / Archer**：远程——优先击杀。盾反或翻滚接近 / Priority target. Roll or parry to close distance
- **盾兵 / Shieldbearer**：正面格挡——绕背或破盾技能 / Blocks frontal attacks — flank or use shield-break skills

### 有毒下水道 / Toxic Sewers
- **毒虫 / Poisoner**：死后留下毒池——注意走位 / Leaves poison pool on death — watch positioning
- **鼠群 / Rat Swarm**：数量多伤害低——范围武器轻松清理 / Large groups, low damage — trivial with AOE
- **自爆蝙蝠 / Bomber**：听到滋滋声立刻翻滚 / Roll on hearing sizzle sound

### 时钟塔 / Clock Tower
- **钟表怪 / Clockwork**：攻击节奏固定——掌握节奏后容易无伤 / Fixed attack rhythm — easy once learned
- **刺客 / Assassin**：隐身+背刺——注意脚步声和空气波纹 / Invisible + backstab — listen for footsteps, watch for air distortion
- **尖刺陷阱 / Spike Trap**：间歇性弹出——看地面缝隙判断 / Periodic spikes — check floor gaps
