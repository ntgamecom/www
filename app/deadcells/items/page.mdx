# 物品收集 / Items & Unlocks

死亡细胞的收集驱动是蓝图(BP)——击败特定敌人/开宝箱获得蓝图，在收藏家用细胞解锁 / Dead Cells' collection system is Blueprints (BP) — defeat specific enemies/open chests for BPs, unlock at the Collector with cells.

---

## 核心收集要素 / Core Collectibles

| 物品 / Item | 作用 / Purpose | 总量 / Total |
|:----:|------|:----:|
| **蓝图 / Blueprints** | 解锁新武器/技能/变异 / Unlock new weapons/skills/mutations | 200+ |
| **符文 / Runes** | 解锁新能力（爬墙、传送等）/ Unlock new abilities (wall climb, teleport etc) | 6 |
| **武器图纸 / Weapon BPs** | 特定武器解锁——击杀精英或隐藏宝箱 / Unlock specific weapons — elite kills or hidden chests | 150+ |
| **皮肤 / Outfits** | 外观——Boss无伤等条件解锁 / Cosmetics — no-hit boss, etc | 50+ |
| **收藏家物品 / Collector Items** | 细胞兑换——花、观景点等 / Cell exchange — flowers, viewpoints etc | 20+ |

## 蓝图获取Tips / Blueprint Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **精英必杀 / Kill All Elites** | 每个精英敌人有概率出专属蓝图——不管多难都打 / Each elite type has a unique BP drop — always kill them |
| **隐藏宝箱 / Check Every Corner** | 隐藏宝箱(金色光圈)出高品质装备+蓝图——全图探索 / Gold aura chests = high quality gear + BPs — full explore |
| **Boss无伤 / No-Hit Bosses** | 无伤击败Boss出特定皮肤——挑战党的目标 / No-hit boss kills unlock special outfits |
| **自定义模式刷蓝图 / Custom Mode Farming** | 自定义模式可以关掉已解锁的武器——提高特定蓝图掉落率 / Custom Mode can disable unlocked weapons — increases specific BP drop rates |
