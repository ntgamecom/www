# 物品收集 / Items & Unlocks

死亡细胞的收集驱动是蓝图(BP)——击败特定敌人/开宝箱获得蓝图，在收藏家用细胞解锁。合理规划收集顺序能大幅加速成长 / Dead Cells' collection system is Blueprints (BP) — defeat specific enemies/open chests for BPs, unlock at the Collector with cells. Smart planning massively accelerates progression.

---

## 核心收集要素 / Core Collectibles

| 物品 / Item | 作用 / Purpose | 总量 / Total |
|:----:|------|:----:|
| **蓝图 / Blueprints** | 解锁新武器/技能/变异 — 每次解锁都扩展Build可能性 / Unlock new weapons/skills/mutations | 200+ |
| **符文 / Runes** | 解锁新能力（爬墙、传送等）— 必须收集以解锁全部关卡 / Unlock new abilities (wall climb, teleport) — essential for full biome access | 6 |
| **武器图纸 / Weapon BPs** | 特定武器解锁 — 击杀精英或隐藏宝箱 / Unlock specific weapons — elite kills or hidden chests | 150+ |
| **皮肤 / Outfits** | 外观 — Boss无伤等条件解锁，纯收藏向 / Cosmetics — no-hit boss kills, purely collectible | 50+ |
| **收藏家物品 / Collector Items** | 细胞兑换 — 花、观景点等，全收集有成就 / Cell exchange items — flowers, viewpoints, achievement-related | 20+ |

## 蓝图获取技巧 / Blueprint Farming Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **精英必杀 / Always Kill Elites** | 每个精英敌人有概率出专属蓝图。精英身上有金色光环——很好辨认。不管多难都打——可能爆极品 / Each elite type has a unique BP drop. Golden aura makes them easy to spot. Always fight them |
| **隐藏宝箱全搜 / Search Every Corner** | 金色光圈宝箱出高品质装备+蓝图。每个关卡至少1-2个隐藏点——注意墙壁裂缝和可疑地面 / Gold-aura chests = high quality gear + BPs. Walls with cracks or suspicious floors often hide them |
| **Boss无伤 / No-Hit Boss Kills** | 无伤击败Boss解锁专属皮肤。低细胞时练无伤最简单——伤害低、容错高，适合练习Boss攻击模式 / Easier to practice no-hit at low BC — lower damage, more forgiving |
| **自定义模式刷 / Custom Mode Farming** | 自定义模式可以关闭已解锁的武器——降低武器池数量→提高特定新蓝图的掉落率。刷特定图纸时效率翻倍 / Disable unlocked weapons → smaller weapon pool → higher chance for specific new BPs. Doubles farm efficiency |
| **收藏家DSR / Collector's DSR** | 细胞不要攒着——打到就花。细胞死亡后会丢失一半——但花掉的就是赚到 / Spend cells as you get them. You lose half on death — spent cells are safe |
| **Boss细胞门 / Boss Cell Doors** | 每关有需要对应细胞等级才能打开的门——门后是高价值宝箱、新武器和蓝图。前期先解锁细胞等级，再回来开这些门 / Each BC level unlocks special doors — contain high-value chests, weapons, and BPs. Unlock BC first, then revisit |
