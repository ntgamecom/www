# 终局内容 / Endgame Content

死亡细胞的终局从第一次通关后才真正开始——Boss细胞系统提供了5个递增的难度等级。本章覆盖5细胞难度、真结局路线及终局挑战 / The Dead Cells endgame truly begins after your first clear — the Boss Cell system provides 5 escalating difficulty levels. This chapter covers 5 BC difficulty, the true ending route, and endgame challenges.

---

## Boss细胞系统 / Boss Cell System

击败最终Boss"王之手"后获得第一个Boss干细胞——使用细胞可以开启更高难度 / Defeat the final boss "Hand of the King" to get your first Boss Cell — use cells at the start to enable higher difficulty.

| 细胞等级 / BC Level | 击杀Boss / Boss Required | 难度变化 / Changes |
|---|---|---|
| **0细胞(0BC)** | 无(默认) / None (default) | 基础难度——适合新手熟悉操作 / Base difficulty — learn the ropes |
| **1细胞(1BC)** | 王之手(首次击杀) / Hand of the King (first kill) | 敌人血量+70%，伤害+30%。部分敌人增加新招式 / +70% enemy HP, +30% damage. Some enemies gain new moves |
| **2细胞(2BC)** | 王之手(1BC难度) / Hand (at 1BC) | 精英怪不再掉落食物。全关卡经济减少 / Elites no longer drop food. Reduced economy across all levels |
| **3细胞(3BC)** | 王之手(2BC) | 商店价格上涨50%。所有敌人生成速度加快 / Shop prices +50%. Enemy spawn rate increased |
| **4细胞(4BC)** | 王之手(3BC) | **感染(Malaise)机制引入**——随时间积累感染值，满值后持续掉血。击杀敌人可降低感染 / **Malaise mechanic** — accumulates over time, constant HP loss at max. Kills reduce malaise |
| **5细胞(5BC)** | 巨人(4BC) / The Giant (4BC) | 感染速度大幅增加 + 新关卡"观星实验所"和最终Boss"收藏家" / Malaise rate greatly increased + new level "Astrolab" and final boss "Collector" |

## 真结局路线 / True Ending Route

5细胞难度下的完整通关路线——打开最终结局 / Complete route at 5 BC difficulty — unlocks the true ending.

### 关卡路线 / Level Route

| 阶段 / Phase | 关卡 / Level | 注意事项 / Notes |
|---|---|---|
| **1** | 囚者宿舍 → 有罪者大道 / Prisoners' Quarters → Promenade | 快速清图——感染值随时间累积 / Clear fast — malaise accumulates over time |
| **2** | 壁垒 → 雾锁港湾 / Ramparts → Slumbering Sanctuary | 推荐经过壁垒(大量细胞来源) / Recommended: Ramparts (lots of cells) |
| **3** | 旧下水道 → 守护者避难所 / Ancient Sewers → Guardian's Haven | 可选择"钟楼"但避难所路线更快 / Can take Clock Tower route but Haven is faster |
| **4** | 击败巨人 → 获得"人工菌种" → 进入观星实验所 / Defeat Giant → get "Artificial Malaise" → enter Astrolab | 巨人是最难的中间Boss——确保装备足够 / Giant is the toughest mid-boss — ensure gear is ready |
| **5** | 观星实验所 → 击败收藏家 / Astrolab → Defeat The Collector | 最终关卡——大量精英怪 + 最终Boss / Final level — many elites + final boss |
| **6** | 观看真结局动画 / True ending cutscene | |

### 收藏家Boss战详解 / The Collector Boss Fight

| 阶段 / Phase | 攻击模式 / Attack Pattern | 应对方法 / Counter |
|---|---|---|
| **阶段1(100%-66%HP)** | 近战连击+远程弹幕。召唤小兵 / Melee combo + ranged barrage. Summon minions | 翻滚躲连击——弹反远程弹幕 / Roll through combos — parry projectiles |
| **阶段2(66%-33%HP)** | 地面AOE冲击波+传送突刺。小兵变强 / Ground AOE shockwave + teleport stab. Stronger minions | 跳跃躲冲击波——注意传送方向 / Jump over shockwave — watch teleport direction |
| **阶段3(33%-0%HP)** | 全屏弹幕+频繁传送+快速近战。Boss狂暴 / Full-screen barrage + frequent teleport + fast melee. Boss enrages | 保持移动——不要贪刀。用盾牌弹反 / Keep moving — don't greedy. Use shield parry |

**推荐Build**: 盾牌(必须)+高DPS主武器(如刺剑/巨剑)。推荐生存系变异——弹反流收益最高 / **Recommended build**: Shield (mandatory) + high DPS weapon (rapier/greatsword). Survival mutations — parry build offers highest payoff.

## Boss Rush模式 / Boss Rush Mode

Boss Rush是在DLC"Return to Castlevania"后加入的模式——连续挑战Boss队列 / Boss Rush is a mode added post-"Return to Castlevania" DLC — consecutively fight boss gauntlets.

| 难度 / Difficulty | Boss数量 / Boss Count | 奖励 / Reward |
|---|---|---|
| **简单 / Easy** | 3个Boss轮换 / 3 random bosses | 金币+少量细胞 / Gold + small cells |
| **普通 / Normal** | 5个Boss轮换 / 5 random bosses | 一般细胞量 / Medium cells |
| **困难 / Hard** | 8个Boss轮换 / 8 random bosses | 大量细胞+限定武器皮肤 / Lots of cells + exclusive weapon skins |
| **完美挑战 / Perfect Challenge** | 全Boss无伤通过 / All bosses no-hit | 最强限定皮肤——金色套装 / Best exclusive skin — gold outfit |

## 5细胞生存指南 / 5 BC Survival Guide

### 感染机制详解 / Malaise Mechanics

| 感染数值 / Malaise Level | 效果 / Effect |
|---|---|
| **0-20%** | 无效果 / No effect |
| **20-40%** | 移动速度降低5% / Move speed -5% |
| **40-60%** | 移动速度降低10% + 攻击速度降低5% / Move speed -10%, attack speed -5% |
| **60-80%** | 移动速度降低15% + 攻击速度降低10% / Move speed -15%, attack speed -10% |
| **80-99%** | 移动速度降低20% + 攻击速度降低15% + 受到伤害+10% / Move speed -20%, attack speed -15%, damage taken +10% |
| **100%(满)** | 持续掉血——每0.5秒掉1%最大生命值 / Constant HP loss — 1% max HP per 0.5s |

**感染控制方法**:
| 方法 / Method | 效果 / Effect |
|---|---|
| **击杀敌人** | 每个击杀减少3%感染值 / Each kill reduces malaise by 3% |
| **使用药瓶** | 减少10%感染值 / Reduces malaise by 10% |
| **食物** | 减少5%感染值 / Reduces malaise by 5% |
| **特定变异** | 如"杀菌者"变异可以被动降低感染积累 / Mutation "Sterilizer" passively reduces malaise accumulation |
| **保持移动** | 静止不动时感染积累速度翻倍——永远保持移动 / Standing still doubles malaise accumulation — always keep moving |

### 5细胞路线推荐 / 5 BC Route Recommendations

| 路线 / Route | 优点 / Pros | 缺点 / Cons |
|---|---|---|
| **路线A: 最短路线** | 囚者宿舍→有罪者大道→壁垒→雾锁港湾→旧下水道→守护者避难所→巨人→观星实验所 / PQs→Promenade→Ramparts→Sanctuary→Ancient Sewers→Haven→Giant→Astrolab | 关卡最少——感染值积累少 / Fewest levels — less malaise accumulation | 装备等级可能不够 / Gear level may be low |
| **路线B: 装备路线** | 囚者宿舍→猛毒下水道→旧下水道→庇护所→雾锁港湾→钟楼→时间守护者→王座之巅→观星实验所 / PQs→Toxic Sewers→Ancient Sewers→Sanctuary→Clock Tower→Time Keeper→Throne→Astrolab | 装备等级高——有更多商店 / Higher gear level — more shops | 路线长——感染值高 / Longer route — higher malaise |

**推荐选择**: 新手用路线A——感染值管理比装备等级更重要。熟悉后可以尝试路线B拿更高装备 / **Recommendation**: Beginners take Route A — malaise management trumps gear level. After familiarizing, try Route B for higher gear.
