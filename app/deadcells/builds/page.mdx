# 武器与Build / Weapons & Builds

死亡细胞的武器分为三个属性颜色——选择Build就是选择一种属性然后全加到底。但真正的高手会根据武器和变异灵活调整。本章提供详细的Build方案和武器评测 / Dead Cells weapons are divided into 3 colors — choosing a build means picking a color and committing fully. But true masters adapt based on weapon and mutation availability. This chapter provides detailed build plans and weapon reviews.

---

## 武器类型总览 / Weapon Types

| 类型 / Type | 属性 / Stat | 特点 / Features |
|:----:|:----:|------|
| **近战 / Melee** | 暴虐(红) / Brutality | 高伤害、贴脸输出、杀敌回复 / High damage, close-range, kill healing |
| **远程 / Ranged** | 战术(紫) / Tactics | 安全距离、陷阱辅助、脆皮高输出 / Safe range, trap support, glass cannon |
| **盾牌 / Shield** | 生存(绿) / Survival | 格挡/盾反、高血量、重型武器 / Block/parry, high HP, heavy weapons |
| **技能 / Skills** | 视类型而定 / Varies | 手雷、炮塔、陷阱、治疗等 / Grenades, turrets, traps, heals etc |

## 经典Build推荐 / Popular Builds

### 1. 暴虐近战流 / Brutality Melee Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **武器 / Weapon** | 均衡之刃(Balanced Blade) / 怨恨之刃(Melee) |
| **盾牌 / Shield** | 前线盾(Front Line Shield) / 惩罚者(Punishment) |
| **技能 / Skills** | 碎肉者(Slicer) + 捕熊夹(Stun Grenade) |
| **变异 / Mutations** | 狂战士(Berserker)、复仇(Vengeance)、杀敌回血(What Doesn't Kill Me) |
| **属性加点 / Scrolls** | 全红 / All Red |
| **难度 / Difficulty** | ★☆☆ —— 最简单直接的Build / Easiest build |

**核心思路**: 贴脸猛攻——暴虐有杀敌减CD和回血，清图效率极高 / Close-range aggression — Brutality has kill-based CD reduction and healing, great clear speed

**变异详解**: 狂战士(击杀+移速+减伤)是暴虐流核心——保持Buff不断则小怪基本打不中你。复仇(受伤后增伤10秒)打Boss时触发。杀敌回血提供续航。**技能搭配**: 碎肉者放脚下→敌人踩上去持续受伤→你上去近战补刀。捕熊夹负责控制精英/Boss / **Mutation details**: Berserker (kill = speed + DR) is core — keep buff active and mobs can't hit you. Vengeance (damage boost after hit) for bosses. What Doesn't Kill Me for sustain. **Skill combo**: Slicer on ground → enemies take DoT → you melee finish. Stun Grenade for elite/boss control.

### 2. 战术炮塔流 / Tactics Turret Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **武器 / Weapon** | 电鞭(Electric Whip) / 多重射击弓(Multi-knock Bow) |
| **技能 / Skills** | 双炮塔×2（或炮塔+诱饵）/ Double Turret (or Turret + Decoy) |
| **变异 / Mutations** | 支援(Support)、死灵术(Dead Inside)、远程(Efficiency) |
| **属性加点 / Scrolls** | 全紫 / All Purple |
| **难度 / Difficulty** | ★★☆ —— 远程安全，但脆皮 / Safe range, but very squishy |

**核心思路**: 炮塔拉仇恨→你在远处安全输出。打Boss时炮塔持续输出你负责躲 / Turrets tank aggro → you DPS from safety. Bosses: turrets DPS while you dodge

**电鞭优势**: 自动瞄准+穿墙——在炮塔吸引火力的同时你可以站在安全位置无脑攻击。**多重射击弓**: 穿透+多段伤害——对群怪极强。**变异选择**: 支援(炮塔附近增伤)是核心——站炮塔旁边。死灵术(+40%血量)弥补战术流的低血量短板。Efficiency(技能CD减少)让炮塔覆盖更持续 / **Electric Whip**: auto-aim + wall pierce — attack safely while turrets tank. **Multi-knock Bow**: pierce + multi-hit — great vs groups. **Mutations**: Support (dmg near turrets) is core — stand next to them. Dead Inside (+40% HP) fixes Tactics' squishiness. Efficiency reduces skill cooldowns for better uptime.

### 3. 生存盾反流 / Survival Shield Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **武器 / Weapon** | 巨锤(Giant Hammer) / 烤炉之刃(Oven Knife) |
| **盾牌 / Shield** | 任何盾牌 + 盾反相关变异 / Any shield + parry mutations |
| **技能 / Skills** | 巨人之矛(Giant Whistle) + 冰墙(Ice Wall) |
| **变异 / Mutations** | 盾反治疗(What Doesn't Kill Me)、拳头(Armadillopack)、反击(Counterattack) |
| **属性加点 / Scrolls** | 全绿 / All Green |
| **难度 / Difficulty** | ★★★ —— 高容错但需要盾反熟练 / High survivability but needs parry skill |

**核心思路**: 生存属性+重型武器=高血量+高伤害。盾反是核心——防住一切+反击 / Survival stat + heavy weapons = high HP + high dmg. Parry is core — block everything + counter

**盾反机制**: 在敌人攻击命中前0.2秒按盾牌键(默认右键)。成功=完全格挡+敌人硬直+触发盾牌特效。**Armadillopack**变异让翻滚也能触发盾反——大幅降低盾反门槛。**巨锤**的蓄力攻击在盾反后接上能打出恐怖伤害 / **Parry timing**: press shield (right click default) 0.2s before hit. Success = full block + enemy stun + shield effect. **Armadillopack** mutation lets your roll parry — massively lowers skill floor. **Giant Hammer** charged attack after parry delivers devastating damage.

### 4. 元素毒血流 / Elemental DoT Build (进阶 / Advanced)

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **武器 / Weapon** | 蛇牙(Serpent Fang) / 毒匕首(Poison Dagger) |
| **技能 / Skills** | 火焰手雷 + 腐蚀毒云 / Fire Grenade + Corrosive Cloud |
| **变异 / Mutations** | 连击(Combo)、开放伤口(Open Wounds)、毒大师(Alchemist) |
| **属性加点 / Scrolls** | 全红(暴虐)或全紫(战术) / All Red (Brutality) or All Purple (Tactics) |
| **难度 / Difficulty** | ★★★☆ —— DOT管理需要意识 / DOT management required |

**核心思路**: 上毒→引爆→敌人持续掉血。DOT伤害不受距离影响，放完就跑放风筝 / Apply poison/bleed → detonate → enemies die over time. DOT damage isn't range-dependent — apply and kite.
