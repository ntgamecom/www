# 武器与Build / Weapons & Builds

死亡细胞的武器分为三个属性颜色——选择Build就是选择一种属性然后全加到底 / Dead Cells weapons are divided into 3 colors — choosing a build means picking a color and committing fully.

---

## 武器类型总览 / Weapon Types

| 类型 / Type | 属性 / Stat | 特点 / Features |
|:----:|:----:|------|
| **近战 / Melee** | 暴虐(红) / Brutality | 高伤害、贴脸输出、杀敌回复 / High damage, close-range, kill healing |
| **远程 / Ranged** | 战术(紫) / Tactics | 安全距离、陷阱辅助、脆皮高输出 / Safe range, trap support, glass cannon |
| **盾牌 / Shield** | 生存(绿) / Survival | 格挡/盾反、高血量、重型武器 / Block/parry, high HP, heavy weapons |
| **技能 / Skills** | 视类型而定 / Varies | 手雷、炮塔、陷阱、治疗等 / Grenades, turrets, traps, heals etc |

---

## 经典Build推荐 / Popular Builds

### 1. 暴虐近战流 / Brutality Melee Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **武器 / Weapon** | 均衡之刃(Balanced Blade) / 怨恨之刃(Melee) |
| **盾牌 / Shield** | 前线盾(Front Line Shield) / 惩罚者(Punishment) |
| **技能 / Skills** | 碎肉者(Slicer) + 捕熊夹(Stun Grenade) |
| **变异 / Mutations** | 狂战士(Berserker)、复仇(Vengeance)、杀敌回血(What Doesn't Kill Me) |
| **属性加点 / Scrolls** | 全红 / All Red |
| **难度 / Difficulty** | ★☆☆ —— 最简单直接的Build / Easiest build |

**核心思路**: 贴脸猛攻——暴虐有杀敌减CD和回血，清图效率极高 / Close-range aggression — Brutality has kill-based CD reduction and healing, great clear speed

### 2. 战术炮塔流 / Tactics Turret Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **武器 / Weapon** | 电鞭(Electric Whip) / 多重射击弓(Multi-knock Bow) |
| **技能 / Skills** | 双炮塔×2（或炮塔+诱饵）/ Double Turret (or Turret + Decoy) |
| **变异 / Mutations** | 支援(Support)、死灵术(Dead Inside)、远程(Efficiency) |
| **属性加点 / Scrolls** | 全紫 / All Purple |
| **难度 / Difficulty** | ★★☆ —— 远程安全，但脆皮 / Safe range, but very squishy |

**核心思路**: 炮塔拉仇恨→你在远处安全输出。打Boss时炮塔持续输出你负责躲 / Turrets tank aggro → you DPS from safety. Bosses: turrets DPS while you dodge

### 3. 生存盾反流 / Survival Shield Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **武器 / Weapon** | 巨锤(Giant Hammer) / 烤炉之刃(Oven Knife) |
| **盾牌 / Shield** | 任何盾牌 + 盾反相关变异 / Any shield + parry mutations |
| **技能 / Skills** | 巨人之矛(Giant Whistle) + 冰墙(Ice Wall) |
| **变异 / Mutations** | 盾反治疗(What Doesn't Kill Me)、拳头( Armadillopack)、反击(Counterattack) |
| **属性加点 / Scrolls** | 全绿 / All Green |
| **难度 / Difficulty** | ★★★ —— 高容错但需要盾反熟练 / High survivability but needs parry skill |

**核心思路**: 生存属性+重型武器=高血量+高伤害。盾反是核心——防住一切+反击 / Survival stat + heavy weapons = high HP + high dmg. Parry is core — block everything + counter
