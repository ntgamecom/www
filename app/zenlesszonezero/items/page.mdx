## 物品系统概述 / Item System Overview

绝区零拥有丰富的物品系统，包括代理人的装备、升级材料和消耗品。合理管理和分配物品资源是提升战斗力的关键。

ZZZ features a comprehensive item system including character equipment, upgrade materials, and consumables. Proper management and allocation of item resources is key to improving combat effectiveness.

## W-Engine / W-引擎

W-Engine是代理人的专属武器装备。每位代理人可以装备一把W-Engine，提供基础属性加成和特殊效果。

W-Engines are character-exclusive weapon equipment. Each Agent can equip one W-Engine, providing base stat bonuses and special effects.

**稀有度 / Rarity**: S级、A级、B级。S级拥有最高基础属性和最强的特效。（S-rank, A-rank, B-rank. S-rank has the highest base stats and strongest effects.

**类型 / Types**: 根据代理人专精分类。（Categorized by Agent Specialty:
- 强攻型 / Attack Type
- 击破型 / Stun Type
- 支援型 / Support Type
- 防护型 / Defense Type
- 异常型 / Anomaly Type

**获取方式 / Acquisition Methods**:
- 信号搜索（卡池）/ Signal Search (gacha)
- 商店兑换 / Shop exchange
- 活动奖励 / Event rewards
- 锻造制作 / Crafting

**S级W-Engine推荐 / Recommended S-Rank W-Engines**:
- **深海访客 / Deep Sea Visitor**（艾莲专属）：冰伤加成、暴击伤害提升。（Ellen's signature, Ice DMG bonus, Crit DMG increase.
- **玉竹长矛 / Jade Bamboo Spear**（朱鸢专属）：以太伤加成、连携攻击增伤。（Zhu Yuan's signature, Ether DMG bonus, Chain Attack damage increase.
- **钢铁狱犬 / Steel Prison Dog**（11号专属）：火伤加成、攻击力提升。（Soldier 11's signature, Fire DMG bonus, ATK increase.

**A级W-Engine推荐 / Recommended A-Rank W-Engines**:

- **聚宝箱 / Treasure Box**: 击破型W-Engine，失衡值提升。（Stun W-Engine, Daze increase.
- **咆哮 / Howl**: 支援型W-Engine，攻击增益。（Support W-Engine, ATK buff.

## 驱动盘 / Drive Discs

驱动盘是类似"圣遗物"系统的装备部件。每位代理人可装备6件驱动盘，分为1
6号位。

Drive Discs are similar to the artifact/relic system. Each Agent can equip 6 Drive Discs, divided into slots 1 through 6.

**编号位置 / Slot Numbers**:
- 1号位：固定基础攻击力 / Slot 1: Fixed base ATK
- 2号位：固定基础防御力 / Slot 2: Fixed base DEF
- 3号位：固定基础生命值 / Slot 3: Fixed base HP
- 4号位：主属性随机（暴击率、暴伤、攻击%）/ Slot 4: Random main stat (Crit Rate, Crit DMG, ATK%)
- 5号位：主属性随机（属性伤害、穿透、攻击%）/ Slot 5: Random main stat (Elemental DMG, PEN, ATK%)
- 6号位：主属性随机（能量回复、冲击、异常精通）/ Slot 6: Random main stat (Energy Regen, Impact, Anomaly Proficiency)

**套装效果 / Set Bonuses**: 装备2件或4件同套装驱动盘可激活套装效果。（Equipping 2 or 4 pieces from the same set activates set bonuses.

**推荐套装 / Recommended Sets**:
- **混沌重金属 / Chaos Metal**: 物理伤害套装。（Physical DMG set.
- **极地重金属 / Polar Metal**: 冰属性伤害套装。（Ice DMG set.
- **炎狱重金属 / Inferno Metal**: 火属性伤害套装。（Fire DMG set.
- **雷暴重金属 / Thunder Metal**: 电属性伤害套装。（Electric DMG set.
- **灵魂摇滚 / Soul Rock**: 以太属性伤害套装。（Ether DMG set.
- **摇摆爵士 / Swing Jazz**: 能量回复和支援套。（Energy Regen and Support set.

## Bardic Needle / 吟游针筒

吟游针筒（Bardic Needle）是一种特殊的消耗品，使用后可让玩家在空洞内的特定位置快速恢复状态。它可以在探索时提供关键的生命值和能量回复。

The Bardic Needle is a special consumable that allows players to quickly restore their status at specific locations within Hollows. It provides critical HP and energy recovery during exploration.

**获取途径 / Acquisition**:
- 空洞探索奖励 / Hollow exploration rewards
- 补给站购买 / Supply station purchase
- 活动赠送 / Event gifts

**使用建议 / Usage Tips**: 保留在遇到精英战或Boss战前使用。（Save for use before elite encounters or boss fights.

## 丁尼 / Dennies

丁尼是绝区零中的基础通用货币，几乎所有操作都需要消耗丁尼。它是游戏中最重要的经济资源。

Dennies is the basic universal currency in ZZZ. Almost all operations consume Dennies. It is the most important economic resource in the game.

**获取方式 / Acquisition Methods**:
- 委托奖励 / Commission rewards
- 空洞探索 / Hollow exploration
- 丁尼副本 / Dennie stages
- 活动 / Events
- 商店出售不需要的物品 / Selling unwanted items at shops

**主要消耗途径 / Main Expenditure**:
- 角色升级 / Character leveling
- 技能升级 / Skill upgrades
- 驱动盘强化 / Drive Disc enhancement
- W-Engine升级 / W-Engine leveling
- 商店购物 / Shop purchases

**丁尼管理建议 / Dennie Management Tips**:
- 不要把丁尼花在无关紧要的过渡装备上 / Do not spend Dennies on unimportant transitional gear
- 每日丁尼副本值得清理 / Daily Dennie stages are worth clearing
- 活动优先换取丁尼 / Prioritize Dennies from event shops
- 不要用菲林购买丁尼，性价比极低 / Never use Polychrome to buy Dennies, extremely poor value

## 菲林 / Polychrome

菲林是游戏中的高级货币，主要用于抽卡。获取途径包括日常、周常、深渊和活动。

Polychrome is the premium currency, primarily used for gacha pulls. Acquisition includes dailies, weeklies, endgame, and events.

**推荐用途 / Recommended Use**: 仅用于购买加密母带抽限定卡池。（Only use for buying Encrypted Master Tapes on limited banners.

**信号余烬兑换 / Signal Ember Exchange**: 使用抽卡副产物兑换W-Engine等道具。（Use gacha byproducts to exchange for W-Engines and other items.

**玩具店 / Souvenir Shop**: 使用丁尼购买基础材料。（Buy basic materials with Dennies.

**补给站 / Supply Station**: 空洞内的临时商店，消耗丁尼购买消耗品。（Temporary shop inside Hollows, buy consumables with Dennies.
