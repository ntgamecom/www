# 收藏与成就 / Unlocks & Achievements

Balatro的解锁系统驱动着游戏进度——每赢一局、每用一种牌组通关，都会解锁新内容 / Balatro's unlock system drives progression — each win, each deck clear unlocks new content.

---

## 牌组解锁 / Deck Unlocks

| 牌组 / Deck | 解锁条件 / Unlock Condition |
|:----:|------|
| **红/蓝/绿 / Red/Blue/Green** | 初始可用 / Default |
| **黑 / Black** | 任意牌组赢一局 / Win any run |
| **黄 / Yellow** | 赢两局 / Win 2 runs |
| **乱序 / Chaos** | 赢五局 / Win 5 runs |
| **魔法 / Magic** | 用黑牌组赢一局 / Win with Black Deck |
| **星云 / Nebula** | 赢所有基础牌组 / Win with all base decks |
| **斑纹 / Checkered** | 用魔法牌组赢 / Win with Magic Deck |
| **幽灵 / Spectral** | 赢所有牌组 / Win with all decks |
| **斑马 / Zebra** | 隐藏条件 / Hidden |

---

## 小丑牌解锁 / Joker Unlocks

| 小丑 / Joker | 解锁条件 / Unlock |
|:----:|------|
| **男爵 / Barron** | 不买任何小丑牌赢一局 / Win without buying any Jokers |
| **吸血鬼 / Vampire** | 用斑纹牌组赢 / Win with Checkered Deck |
| **鬼牌 / Joker** | 初始可用 / Default |
| **钢× / Steel Joker** | 牌组中有5张钢牌时过关 / Pass a round with 5 Steel cards in deck |
| **露营者 / Campfire** | 用幻灵牌组赢 / Win with Spectral Deck |
| **超新星 / Supernova** | 玩同一手牌型100次 / Play the same hand type 100 times |

## 挑战模式 / Challenge Mode

挑战模式提供20个规则修改的预设关卡——完成解锁新小丑牌和牌组 / Challenge Mode has 20 modified preset runs — completing them unlocks new Jokers and decks.

| 挑战 / Challenge | 难度 / Difficulty | 特殊规则 / Special Rule |
|:----:|:----:|------|
| **节俭之家 / Frugal House** | ★☆☆ | 不能用塔罗牌和星球牌 / No Tarot or Planet cards |
| **富人的烦恼 / Rich's Woes** | ★★☆ | 起始只有$1，利息为0 / Start with $1, no interest |
| **魔法大师 / Magic Master** | ★★★ | 只能用幻灵牌——没有小丑牌 / Spectral only — no Jokers |
| **小丑的报复 / Joker's Revenge** | ★★★★ | 不能买小丑牌——只能用初始小丑 / Can't buy Jokers — starting Joker only |
| **黄金风暴 / Gold Rush** | ★★★★ | 所有卡牌变成金卡——但需要$$来保留 / All cards are Gold — costs $ to keep |
| **最后的赌注 / Last Bet** | ★★★★★ | 只有1条命——输一局就结束 / One life only — lose once and it's over |

## 成就目标 / Achievement Goals

| 成就 / Achievement | 要求 / Requirement |
|:----:|------|
| **通关者 / Victor** | 任意牌组通关 / Win any run |
| **全牌组通关 / Deckmaster** | 所有牌组通关 / Win with all decks |
| **全挑战 / Challenger** | 完成所有挑战模式 / Complete all challenges |
| **积分破e / Exponential** | 单局积分达到1e12(一万亿) / Single-run score reaches 1e12 |
| **金手指 / Golden Touch** | 一局中打出100张金卡 / Play 100 Gold cards in one run |
| **全成就 / Completionist** | 解锁全部内容 / Unlock everything |
