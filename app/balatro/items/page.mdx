# 收藏与成就 / Unlocks & Achievements

Balatro的解锁系统驱动着游戏进度——每赢一局、每用一种牌组通关，都会解锁新内容。合理规划解锁顺序可以大幅加快成长速度 / Balatro's unlock system drives progression — each win and each deck clear unlocks new content. Smart unlock planning accelerates your growth significantly.

---

## 牌组解锁 / Deck Unlocks

| 牌组 / Deck | 解锁条件 / Unlock Condition | 推荐用途 / Best Use |
|:----:|------|------|
| **红 / Red** | 初始可用 / Default | 每回合多一次弃牌——适合搜索关键牌的Build / Extra discard per round — good for combo-seeking builds |
| **蓝 / Blue** | 初始可用 / Default | 多一手牌位——适合需要大量手牌的星球牌体系 / Extra hand slot — great for Planet card strategies |
| **绿 / Green** | 初始可用 / Default | 无利息但每回合给$2——适合快节奏低分通关 / No interest but $2/round — good for fast low-score clears |
| **黑 / Black** | 任意牌组赢一局 / Win any run | +1小丑位但-1手牌——最强后期牌组之一 / +1 Joker slot but -1 hand — one of the best late-game decks |
| **黄 / Yellow** | 赢两局 / Win 2 runs | 起始$10额外资金——快速建立经济优势 / Start with extra $10 — fast economy snowball |
| **乱序 / Chaos** | 赢五局 / Win 5 runs | 消费牌面随机——高风险高回报。适合追求大分的玩家 / Consumable cards random — high risk, high reward |
| **魔法 / Magic** | 用黑牌组赢一局 / Win with Black Deck | 起始带幻灵牌——前期强力爆发 / Start with Spectral card — strong early burst |
| **星云 / Nebula** | 赢所有基础牌组 / Win with all base decks | 起始望远镜——每局获得一张免费星球牌 / Starts with Telescope — free Planet card per round |
| **斑纹 / Checkered** | 用魔法牌组赢 / Win with Magic Deck | 牌组全黑桃红心——最适合打同花的牌组 / Deck is all Spades/Hearts — best for flush builds |
| **幻灵 / Spectral** | 赢斑纹牌组 / Win with Checkered Deck | 每回合免费幻灵牌——最强消费牌组 / Free Spectral per round — strongest consumable deck |

## 小丑牌解锁 / Joker Unlocks

小丑牌是Balatro战斗力的核心——不同小丑牌的解锁条件鼓励探索各种玩法 / Jokers are the core of your power — unlock conditions encourage exploring different playstyles.

| 小丑 / Joker | 解锁条件 / Unlock | 实用度 / Usefulness |
|:----:|------|:----:|
| **男爵 / Barron** | 不买任何小丑牌赢一局 / Win without buying Jokers | ★★★ 限定流——配合钢铁牌打高倍率 / Niche — Steel card synergy for huge mult |
| **吸血鬼 / Vampire** | 用斑纹牌组赢 / Win with Checkered Deck | ★★★★★ 每次出牌后强化一张——成长型核心 / Enhances a card per hand played — scaling powerhouse |
| **露营者 / Campfire** | 用幻灵牌组赢 / Win with Spectral Deck | ★★★☆ 每跳过一张牌+0.5倍——适合大量买卖的操作流 / +0.5x per card skipped — fits heavy-buy play |
| **超新星 / Supernova** | 玩同一手牌型100次 / Play a hand type 100 times | ★★★★★ 每回合手牌倍率随使用次数增加——简单暴力 / Hand mult increases with uses — simple and strong |
| **钢× / Steel Joker** | 牌组中有5张钢牌过关 / Pass with 5 Steel cards | ★★★☆ 钢牌有效倍率——需要配合钢铁卡牌 / Steel synergy — needs Steel card support |
| **DNA / DNA** | 用红牌组赢 / Win with Red Deck | ★★★★★ 第一张出牌复制进牌组——最强成长小丑之一 / Copies first played card into deck — top-tier scaling |
| **钢铁× / Steel×** | 牌组全为钢牌时过关 / Win with all Steel deck | ★★★★ 全钢牌补强 / Steel deck finisher |
| **三重 / Triplet** | 不使用任何消费牌赢一局 / Win without consumables | ★★★ | ★★★ 三条Build核心 / Three of a kind build core |
| **古老 / Ancient** | 10回合内通关 / Clear within 10 rounds | ★★★★ 每回合切换增强花色——倍率极高但需调整战术 / Rotating suit enhancement each round — high mult |

**高效解锁小丑的技巧**: 针对特定条件专门配装。比如解锁男爵时，选绿牌组(多弃牌)+专注打高牌/对子快速通关，小丑牌用初始的完全不买。解锁超新星则一直打同一种手型(推荐两对或三条) / **Pro tip for efficient unlocks**: Tailor your build for each unlock condition. For Barron: Green Deck (extra discards) + focus High Pair, skip all Joker purchases. For Supernova: play the same hand type every round (Two Pair or Three of a Kind recommended).

## 挑战模式 / Challenge Mode

挑战模式提供20个规则修改的预设关卡——完成解锁新小丑牌和牌组。按难度从简到难挑战 / Challenge Mode has 20 modified preset runs — completing them unlocks new Jokers and decks. Work from easiest to hardest.

| 挑战 / Challenge | 难度 / Difficulty | 特殊规则 / Special Rule | 通关技巧 / Tip |
|:----:|:----:|------|------|
| **节俭之家 / Frugal House** | ★☆☆ | 不能用塔罗牌和星球牌 / No Tarot or Planet cards | 纯靠小丑牌倍率——选倍率型小丑而非加成型 / Pure Joker mult — choose ×mult over +mult |
| **富人的烦恼 / Rich's Woes** | ★★☆ | 起始$1，利息为0 / Start with $1, no interest | 多用经济型小丑(如桶金)赚钱 / Use economy Jokers (e.g. Golden) to generate income |
| **魔法大师 / Magic Master** | ★★★ | 只能用幻灵牌——没有小丑牌 / Spectral only — no Jokers | 靠幻灵牌强化牌组质量——每回合精打细算 / Rely on Spectral cards to enhance deck — use carefully each round |
| **小丑的报复 / Joker's Revenge** | ★★★★ | 不能买小丑牌——只能用起始小丑 / Can't buy Jokers — starting Joker only | 牌组强化至上的打法——主攻星球牌提升手牌基础倍率 / Focus on deck enhancement — Planet cards for base hand mult |
| **黄金风暴 / Gold Rush** | ★★★★ | 所有卡牌变金卡——但需要花钱保留 / All cards are Gold — costs $ to keep | 优先积攒经济——金卡每回合给$3但弃牌也要$。留够本金 / Build economy first. Gold cards give $3/round but cost to discard. Keep cash reserves |
| **最后的赌注 / Last Bet** | ★★★★★ | 只有1条命——输一局就结束 / One life only — lose once done | 极度保守——每一回合都假设是最后一回合。确保每局都过，不贪分 / Extremely conservative — play every round like it's your last. Prioritize survival over score |

**挑战模式通用技巧**: 打挑战之前先用对应牌组练习几局熟悉机制。每个挑战的前3回合决定成败——如果前3局没有拿到合适的小丑或牌型，直接重开可以节省时间 / **General challenge tips**: Practice with similar decks before attempting. The first 3 rounds determine success — if you don't find suitable Jokers or hands, restart early to save time.

## 高效解锁路线图 / Efficient Unlock Roadmap

1. **前10小时**：用红/蓝牌组通关2-3次，解锁黑牌组和黄牌组。熟悉基础机制
2. **10-30小时**：用黑牌组通关解锁魔法牌组→斑纹→幻灵。同步解锁容易的小丑牌
3. **30-50小时**：用幻灵牌组挑战高难度解锁。开始解锁挑战模式前5个
4. **50+小时**：完成剩余挑战和全牌组通关。追求1e12成就

1. **First 10h**: Clear 2-3 runs with Red/Blue Deck to unlock Black and Yellow. Learn basics.
2. **10-30h**: Progress Black → Magic → Checkered → Spectral. Unlock easy Jokers along the way.
3. **30-50h**: Use Spectral Deck for hard unlocks. Start first 5 challenge modes.
4. **50h+**: Complete remaining challenges and all-deck clears. Chase 1e12 achievement.
