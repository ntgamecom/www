## 经济运营核心 / Economy Management Core

### 利息机制详解 / Interest Mechanics

| 规则 / Rule | 说明 / Description |
|---|---|
| **基础利息** | 每回合结束时每持$5给$1利息——上限$5(需持$25) / Each $5 held = $1 interest at end of round — cap $5 (need $25) |
| **桶金提升** | 桶金小丑将上限提高到$10(需持$50) / Piggy Bank raises cap to $10 (need $50) |
| **工资收入** | 每回合固定$3工资——与利息叠加 / $3 base salary per round — stacks with interest |
| **大盲奖金** | 击败大盲(Big Blind)通常给$3-8——额外收入来源 / Beating Big Blind gives $3-8 bonus — extra income |
| **利息计算时机** | 利息在每回合开始时计算——所以上回合存的钱这回合产生利息 / Interest calculated at start of round — so last round's savings earns next round's interest |

### 经济阶段 / Economy Phases

| 阶段 / Phase | 目标金钱 / Target Gold | 策略 / Strategy |
|---|---|---|
| **前期(Ante 1-3)** | $10-25 | 优先买小丑牌——不够$25就花掉，够$25就留着吃利息 / Prioritize buying Jokers — if below $25, spend; if above $25, save for interest |
| **中期(Ante 4-6)** | $25-40 | 保持$25以上吃满利息。多余的钱刷新商店找核心小丑 / Keep $25+ for max interest. Extra gold → reroll for core Jokers |
| **后期(Ante 7-8)** | $40-60+ | 利息已经不重要了——把钱全花在找×Mult小丑上 / Interest no longer matters — dump all gold into finding ×Mult Jokers |
| **无尽模式** | 随意 / Whatever | 钱多得花不完——每关全刷商店 / Infinite gold — full reroll every shop |

**关键规则**: 前期省钱 > 中期合理消费 > 后期全花光 / **Key rule**: Early save > Mid spend wisely > Late dump all

## 牌组操控 / Deck Manipulation

### 牌组改造工具 / Deck Modification Tools

| 工具 / Tool | 效果 / Effect | 使用技巧 / Usage Tip |
|---|---|---|
| **死神 / Death** | 选择两张牌——左侧摧毁，右侧加入牌组(保留增强) / Select 2 cards — left destroyed, right added (keeps enhancement) | 用来升级低牌(2-3→K/A)。两张牌位置决定：左毁右留 / Upgrade low cards (2-3→K/A). Left→destroy, right→keep |
| **力量 / Strength** | 一张牌的点数+1(如3→4, K→A) / One card rank +1 (3→4, K→A) | 配合死神做牌组升级——先把2变3，3变4，然后死神升级更高 / Pair with Death for sequencing upgrades |
| **倒吊人 / Hanged Man** | 摧毁牌组中两张牌 / Destroy 2 cards in deck | 牌组压缩的核心工具——每次使用减少2张牌。尽量对低牌使用 / Core deck thinning tool — removes 2 cards per use. Use on low-value cards |
| **女祭司 / High Priestess** | 选择一张牌→复制两张加入牌组 / Select a card → add 2 copies to deck | 复制你的核心牌(如K或♥牌)。配合死神无限复制 / Duplicate your key cards (Kings or ♥). Pair with Death for infinite copies |
| **隐士 / Hermit** | 手中金钱翻倍(上限$20) / Double held gold (cap $20) | 前期神器——$10→$20。但要保证吃利息前使用 / Early game god-tier — $10→$20. Use before interest calculation |
| **魔术师 / Magician** | 一张牌获得"幸运"增强 / One card gets Lucky enhancement | 幸运牌在计分时1/2概率给+20倍率——稳定触发时极强 / Lucky cards have 50% +20 Mult chance — extremely strong when consistent |

### 牌组压缩技巧 / Deck Thinning Techniques

| 技巧 / Technique | 说明 / Description |
|---|---|
| **删除低牌 / Delete Low Cards** | 使用倒吊人移除2,3,4,5点——这些牌对同花/对子没有帮助。目标是30-40张精简牌组 / Use Hanged Man to remove 2,3,4,5 — these don't help flushes/pairs. Target 30-40 card deck |
| **复制高牌 / Duplicate High Cards** | 用女祭司复制K,A或特定花色的牌——增加核心牌的抽中概率 / Use High Priestess to duplicate Kings, Aces, or specific suits — increase core card draw rate |
| **浓缩花色 / Suit Concentration** | 把牌组转成单一花色(推荐♥或♠) → 同花抽中率从20%提升到90%+ / Convert deck to single suit (♥ or ♠ recommended) → flush draw rate from 20% to 90%+ |
| **转换点数 / Rank Conversion** | 用力量+死神的组合把低牌逐步升级为高牌 / Use Strength + Death combo to progressively upgrade low ranks to high ranks |

## 分数最大化技巧 / Score Maximization

### 计分顺序优化 / Scoring Order Optimization

计分顺序：从左到右计分牌，然后触发小丑效果。每个小丑的触发顺序也可以调整 / Scoring goes: left to right on played cards, then Joker effects. Joker order can be rearranged.

| 技巧 / Technique | 说明 / Description |
|---|---|
| **筹码牌放左边** | 如果小丑提供+筹码或+倍率，把+筹码放最左边(先触发)→+倍率放中间→×倍率放右边(最后触发)。乘法在最后效果最好 / Put +Chips leftmost → +Mult middle → ×Mult rightmost. Multiplication last is best |
| **小丑位置调整** | 拖拽小丑牌调整触发顺序——筹谋(Blueprint)复制右边小丑，所以把想要复制的放它右边 / Drag Jokers to reorder — Blueprint copies Joker to its right, so put your best ×Mult to its right |
| **触发顺序优化** | 如果有多张牌触发(如血瓶×2、钢牌×1.5)——确保倍率计算顺序最优 / With multiple triggered effects (Bloodstone ×2, Steel ×1.5) — ensure optimal calculation order |

## 高阶概念 / Advanced Concepts

| 概念 / Concept | 说明 / Description |
|---|---|
| **重触发链 / Retrigger Chain** | 用Hack(重触发2-4) + Sock and Buskin(人头牌重触发) + 血瓶(♥×2) → 一张牌可以触发3-4次×2判定 / With Hack(retrigger 2-4) + Sock(retrigger face) + Bloodstone(♥×2) → one card triggers ×2 check 3-4 times |
| **钢牌复制循环 / Steel Duplication Loop** | DNA复制K → 幻灵牌圣杯转钢牌 → 抽到钢牌后自动计分——每回合+1钢牌。随着钢牌增多，倍率指数增长 / DNA copy King → Spectral Chalice to Steel → drawn Steels auto-score — +1 Steel per round. Exponential growth |
| **等离子牌组平衡 / Plasma Deck Balance** | 等离子牌组筹码和倍率取平均——所以如果你有1000筹码和2倍率，实际是(1000+2)/2=501 × 501=251001分。这意味着你需要筹码和倍率都高 / Plasma Deck averages chips and mult — 1000 chips × 2 mult becomes (1000+2)/2=501 × 501=251001. You need both chips AND mult high |
| **塔罗牌循环 / Tarot Cycle** | 愚者(复制最后使用的塔罗牌) → 配合死神无限复制K → 一只愚者可以复制出4+张K / The Fool (copies last Tarot) → with Death loops for infinite King copies → one Fool can yield 4+ Kings |

**核心心法**: 灵活 > 死板。最优秀的Balatro玩家不是死记硬背Build——而是根据商店给出的牌灵活调整路线 / **Core mindset**: Flexibility > Rigidity. The best Balatro players don't force builds — they adapt to what the shop gives them.

## 小丑牌运营节奏 / Joker Operation Tempo

### 小丑牌更换策略 / Joker Replacement Strategy

| 场景 / Scenario | 操作 / Action |
|---|---|
| **看到当前小丑的升级版** | 直接替换——不要犹豫。例如从+4 Mult升级到+8 Mult / Replace immediately — no hesitation. Example: +4 Mult → +8 Mult |
| **经济小丑过了前期** | 黄金小丑在Ante 5之后收益降低——如果商店出现核心×Mult小丑但槽位满了，卖掉黄金小丑换 / Golden Joker's value drops after Ante 5 — if shop shows a core ×Mult but slots are full, sell Golden |
| **槽位满了遇到传奇小丑** | 卖掉最低效的+倍率小丑。传奇小丑(如男爵、Triboulet)的价值远高于普通+倍率 / Sell the weakest +Mult Joker. Legendary Jokers (Baron, Triboulet) outclass basic +Mult by orders of magnitude |
| **临时小丑用完即弃** | 斗牛士(Boss战中赚钱)、赊账(应急购物)——用完就卖掉回收资金 / Matador (earns money during Boss), Credit Card (emergency buy) — sell immediately after use |

## 常见错误总结 / Common Mistake Summary

| 错误 / Mistake | 频率 / Frequency | 解决方法 / Fix |
|---|---|---|
| **过度刷新商店** | 极高 / Very High | 每关控制在1-3次刷新。只有后期缺×Mult时才大量刷 / Limit to 1-3 rerolls per shop. Only mass-reroll late game for ×Mult |

| **不保留消费牌** | 中等 / Medium | 手里常备2-3张有用塔罗牌。关键时刻一张死神可以救局 / Keep 2-3 useful Tarot cards in hand. A timely Death can save a run |
| **Boss预告不看** | 中等 / Medium | 每次选关前看下一个Boss——提前准备对策 / Check next Boss before every blind selection — prepare counters in advance |
