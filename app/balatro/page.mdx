## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **小丑牌 / Jokers** | 游戏核心——每个小丑牌提供独特加成（+倍率、×倍率、触发效果等）。最多可持有5个（某些方式可增加）/ The heart of the game — each Joker gives a unique buff (+mult, ×mult, trigger effects, etc). Hold up to 5 (can be increased) |
| **牌型 / Poker Hands** | 标准德州扑克牌型——高牌→对子→两对→三条→顺子→同花→葫芦→四条→同花顺→皇家同花顺。每种牌型基础分数不同 / Standard poker hands — High Card → Royal Flush. Each has a base score |
| **星球牌 / Planet Cards** | 升级指定牌型的等级——每升一级+基础分数和倍率。牌型等级是积分的基石 / Upgrade a specific hand's level — each level adds base chips and mult. Hand level is your score foundation |
| **塔罗牌 / Tarot Cards** | 一次性消耗——改变牌组（如：把♠2变成♠A、给牌加附魔、转换花色等）/ Single-use — modify your deck (e.g., ♠2→♠A, add enhancement, change suit, etc) |
| **幻灵牌 / Spectral Cards** | 强力的消耗牌——代价通常很大（如：摧毁手牌、复制小丑牌等）/ Powerful consumables — usually with a cost (destroy hand, duplicate joker, etc) |
| **倍率系统 / Mult System** | 分数 = 基础筹码 × 倍率。小丑牌的加法/乘法倍率叠加是冲高分的关键 / Score = base chips × mult. Additive + multiplicative mult stacking from Jokers is key to high scores |
| **盲注 / Blinds** | 每关三个盲注：小盲(白送)→大盲(有奖励)→首领盲(有负面效果)——达到目标分数过关 / Three per ante: Small (free) → Big (reward) → Boss (debuff) — meet target score to pass |

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **专注一种牌型 / Focus on One Hand** | 不要均衡升级所有牌型——选一种（推荐同花或对子）然后用星球牌猛升 / Don't spread hand levels — pick one (Flush or Pair recommended) and spam Planet cards |
| **2** | **多拿小丑牌 / Prioritize Jokers** | 小丑牌是分数引擎——前期每关买1-2个小丑牌。没有好小丑牌你再好的牌型也打不过后期 / Jokers are your scoring engine — buy 1-2 per shop early. No good Jokers = can't keep up with late antes |
| **3** | **经济管理 / Manage Your Money** | 利息收入是最稳定的经济来源——每回合持有$5得$1利息（上限$5）。尽量保持$25+过回合 / Interest is your best income — $5 per $5 held (cap $5). Try to keep $25+ at end of each round |
| **4** | **优化牌组 / Thin Your Deck** | 牌越少越好——用塔罗牌移除低牌（2-5点），增加高牌和同花牌。牌组越精简越稳定 / Fewer cards = better — use Tarot to remove low cards (2-5), add high cards and matching suits. Tighter deck = more consistent |
| **5** | **注意Boss盲注 / Watch Boss Blind Debuffs** | 每个Boss盲注有负面效果——有些禁止同花，有些限制出牌次数等。提前规划Build应对 / Each boss has a debuff — some ban Flush, some limit plays, etc. Plan your build around upcoming bosses |
| **6** | **倍率乘法优先 / Prioritize ×Mult Jokers** | 加法倍率('+Mult')很多，但乘法倍率('×Mult')才是冲高分的关键。×Mult小丑牌是后期核心 / +Mult is common, but ×Mult Jokers are the key to huge scores. ×Mult Jokers are your late-game core |
| **7** | **不要忽略牌增强 / Don't Ignore Card Enhancements** | 塔罗牌给牌加"多重"、"幸运"、"狂野"等附魔——一张多重牌=自带倍率，积累起来非常可观 / Tarot enhancements like "Mult", "Lucky", "Wild" — a single Mult card = built-in multiplier, they stack up fast |
