# 首领盲注攻略 / Boss Blind Guide

小丑牌的Boss盲注每个Ante的最后一道关卡——它们有独特的负面效果，会限制你的牌型或小丑牌。了解每个Boss的效果并提前准备是通关的关键 / Boss Blinds are the final challenge of each Ante — each has a unique debuff that restricts your hands or Jokers. Knowing every Boss effect and preparing ahead is the key to consistent wins.

---

## Boss机制概述 / Boss Mechanics

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **Boss出现频率** | 每个Ante的第3个盲注(小盲→大盲→首领盲)。每8个Ante通关一次 / Every 3rd blind of each Ante (Small → Big → Boss). Clear 8 Antes to win |
| **Boss池 / Boss Pool** | 按Ante分段解锁——前4个Ante使用基础Boss池，从Ante 5开始加入更难的Boss / Unlock by Ante tier — first 4 Antes use base pool, Ante 5+ introduces harder bosses |
| **重复规则 / Repeat Rule** | 同一次游戏中不会出现重复的Boss / No duplicate Bosses appear in a single run |
| **标签跳过 / Skip Tags** | 跳过Boss盲注的标签可以抵消Boss效果——极为重要 / Boss Skip tags can negate the Boss effect — extremely valuable |
| **斗牛士小丑 / Matador Joker** | Boss盲注中每弃一次牌得$4——可以临时买进再卖掉 / Earns $4 per discard during Boss blinds — buy and sell as needed |

## Ante 1-4 Boss盲注 / Boss Blinds (Early Game)

这些Boss相对温和——主要干扰基础玩法 / These bosses are relatively mild — they mainly disrupt basic play.

### 眼盲 / The Eye

| 属性 / Property | 说明 / Description |
|---|---|
| **效果 / Effect** | 你不能重复使用上一轮出过的牌型 / You cannot play the same hand type as the previous round |
| **难度 / Difficulty** | 中等 / Medium |
| **应对策略 / Counter** | 准备两种牌型——比如第一轮出同花，第二轮出对子。如果你只有一种牌型，可以先打一回合高牌转换牌型 / Prepare two hand types — play Flush first, then Pair. If you only have one hand type, waste a round with High Card to reset |

### 墙盲 / The Wall

| 属性 / Property | 说明 / Description |
|---|---|
| **效果 / Effect** | 所需得分x1.5倍 / Required score multiplied by 1.5x |
| **难度 / Difficulty** | 低 / Low |
| **应对策略 / Counter** | 纯数值压力——如果你的Build正常发育，问题不大。如果分数不够，在商店买+倍率小丑或升级牌型 / Pure stat check — if your build is on track, this is fine. If struggling, buy +Mult Jokers or level your hand |

### 轮盘 / The Wheel

| 属性 / Property | 说明 / Description |
|---|---|
| **效果 / Effect** | 每张塔罗牌和幻灵牌有1/7概率触发两次 / Each Tarot and Spectral card has a 1/7 chance to activate twice |
| **难度 / Difficulty** | 低 / Low |
| **应对策略 / Counter** | 这个Boss其实帮你——多出的塔罗/幻灵效果白送。尽量多买塔罗牌来利用这个效果 / This Boss helps you — free double Tarot/Spectral procs. Buy more Tarot cards to abuse this |

### 鱼钩 / The Hook

| 属性 / Property | 说明 / Description |
|---|---|
| **效果 / Effect** | 每出一张牌随机弃掉一张手牌 / Each card played randomly discards one card from hand |
| **难度 / Difficulty** | 中高 / Medium-High |
| **应对策略 / Counter** | 手牌越小越好——不要保留手牌。尽量每次出牌把手牌打完 / Keep hand size small — don't hoard cards. Try to play most of your hand each round |

### 瓢虫 / The Lady Luck

| 属性 / Property | 说明 / Description |
|---|---|
| **效果 / Effect** | 每张幸运牌有翻倍概率(1/2→1/2) ——但实际上没有变 / Each Lucky card has double chance (1/2→1/2) — effectively unchanged |
| **难度 / Difficulty** | 极低 / Very Low |
| **应对策略 / Counter** | 没有实际负面效果——放心打 / No real debuff — play normally |

## Ante 5-8 Boss盲注 / Boss Blinds (Late Game)

这些Boss的负面效果更加严厉——很多能直接废掉你的Build / These bosses have much harsher debuffs — many can completely negate your build.

### 水印 / The Watermark

| 属性 / Property | 说明 / Description |
|---|---|
| **效果 / Effect** | 只有特定花色的牌被计分 / Only cards of a specific suit score |
| **难度 / Difficulty** | 高 / High |
| **应对策略 / Counter** | 如果牌组只有一种花色，被禁就完了。确保牌组有至少2种花色——或者用塔罗牌"女祭司"转换花色 / If your deck is one suit and it's banned, you lose. Keep at least 2 suits in your deck — or use The High Priestess Tarot to change suit |

### 利爪 / The Flint

| 属性 / Property | 说明 / Description |
|---|---|
| **效果 / Effect** | 每轮只给你1次出牌和1次弃牌 / You only get 1 hand and 1 discard per round |
| **难度 / Difficulty** | 极高 / Very High |
| **应对策略 / Counter** | 极少数能直接废掉你的Boss。如果你靠多次出牌叠分(Build靠多次出牌叠加触发型小丑)会被废。确保你一次出牌就能过 / One of the few run-killers. If you rely on multiple hands per round (trigger stacking builds), you're in trouble. Ensure one hand can clear the round |

### 海洋 / The Ocean

| 属性 / Property | 说明 / Description |
|---|---|
| **效果 / Effect** | 每张计分的♥/♦-1筹码 / Each scoring ♥/♦ gets -1 Chips |
| **难度 / Difficulty** | 中等 / Medium |
| **应对策略 / Counter** | 红牌减筹码但黑牌不受影响。切换到全♠/♣牌型不出红牌即可 / Red cards lose chips but blacks are unaffected. Switch to ♠/♣ only |

### 钟表 / The Clock

| 属性 / Property | 说明 / Description |
|---|---|
| **效果 / Effect** | 每回合结束时弃掉你手牌中点数最小的牌 / Discards your lowest rank card at end of round |
| **难度 / Difficulty** | 中等 / Medium |
| **应对策略 / Counter** | 如果你保留大量K(男爵流)——影响不大。如果你是看手牌Build可能会被打乱 / If you keep Kings (Baron build) — minimal impact. If you need specific held cards, this is disruptive |

### 圣杯 / The Grail

| 属性 / Property | 说明 / Description |
|---|---|
| **效果 / Effect** | 每张计分牌必须有一张塔罗/星球/幻灵牌在手中才能计分 / Each scoring card requires a held Tarot/Planet/Spectral to count |
| **难度 / Difficulty** | 高 / High |
| **应对策略 / Counter** | 手牌中至少保留5张消耗牌——每回合预留好。考验你的资源管理能力 / Keep at least 5 consumables in hand at all times — reserve them each round. Tests your resource management |

## Ante 8+ Boss盲注 / Boss Blinds (Endless Mode)

无尽模式中的Boss继续随机循环——但负面效果不能靠"跳过标签"抵消了，且所有Boss的分数要求极高 / In Endless mode, Bosses continue cycling randomly — but skip tags no longer negate debuffs, and score requirements become astronomical.

| Boss名称 / Name | 效果 / Effect | 无尽模式中的影响 / Impact in Endless |
|---|---|---|
| **绿色植物 / The Verdant Leaf** | 每张牌计分-10筹码 / -10 Chips per scoring card | 如果你的倍率极高(×1e6+)——筹码影响相对较小 / If you have ×1e6+ mult, chip loss is negligible |
| **圣甲虫 / The Scarab** | 不能使用塔罗牌和幻灵牌 / Cannot use Tarot or Spectral | 无尽模式下致命——你无法改造牌组。提前准备好最终牌组 / Deadly in Endless — can't modify deck. Have your final deck ready |
| **镰刀 / The Scythe** | 每轮只能出1次牌 / One hand per round | 绝对致命——如果你不是单发Build就必死 / Absolutely lethal — if you're not a one-shot build, you die |

**无尽模式推荐**: 如果你打算玩无尽，推荐男爵+钢牌Build(不依赖多次出牌)或超高人品Build(一次出牌直接爆炸) / **Endless recommendation**: If aiming for Endless, Baron+Steel build (no multi-hand dependency) or massive one-shot build is your only option.

## Boss应对检查清单 / Boss Counter Checklist

| 准备 / Preparation | 应对的Boss / Counters |
|---|---|
| **保留2种花色** / Keep 2+ suits | 水印 / Watermark |
| **不依赖多次出牌** / Don't rely on multi-hand | 利爪、镰刀 / Flint, Scythe |
| **保留消耗牌** / Keep consumables | 圣杯 / Grail |
| **多种牌型可用** / Multiple hand types | 眼盲 / Eye |
| **保持牌组精简** / Keep deck thin | 轮盘——更多塔罗牌利用率 / Wheel — more Tarot efficiency |
| **观察Boss预告** / Watch Boss preview | 所有Boss——提前规划Build路线 / All bosses — plan ahead |

**Boss预告机制**: 在选关界面的右上角你可以看到下一个Boss是什么——提前看到"水印"或"利爪"就可以在商店里有针对性地买牌 / **Boss preview**: The top-right of the map screen shows the next Boss — seeing "Watermark" or "Flint" lets you shop specifically for countermeasures.

## 跳过标签与Boss / Skip Tags Against Bosses

跳过标签(Skip Tags)是金色盲注的奖励——有些标签可以直接抵消Boss效果 / Skip tags are rewards from skipping blinds — some can directly negate Boss effects.

| 标签 / Tag | 效果 / Effect | 针对Boss / Vs Boss |
|---|---|---|
| **Boss标签 / Boss Tag** | 直接跳过当前Boss——不触发效果 / Skip current Boss completely | 任何Boss——最强针对 / Any boss — best counter |
| **金钱标签 / Money Tag** | 得到大量金钱 / Get bonus money | 经济类Boss(如圣杯) / Economy bosses (Grail) |
| **手牌标签 / Hand Tag** | 增加出牌次数 / Bonus hands | 利爪 / Flint |
| **弃牌标签 / Discard Tag** | 增加弃牌次数 / Bonus discards | 圣杯——利于保留消耗牌 / Grail — keep consumables |

**策略**: 如果看到下一个Boss是你的Build天敌(利爪对多次出牌Build)，优先选择跳过Boss盲注的标签 / **Strategy**: If the upcoming boss hard-counters your build (Flint vs multi-hand), prioritize Boss Skip tags.

## 高阶技巧 / Advanced Boss Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **双Boss效应** | 在无尽模式中，多个Boss效果可以叠加——比如"圣杯+利爪"同时出现 = 地狱难度 / In Endless, multiple boss effects stack — Grail+Flint = nightmare |
| **斗牛士利用** | 斗牛士小丑在Boss战中每弃一次牌给$4——Boss战前买入，打完卖掉赚差价 / Matador gives $4 per Boss discard — buy before boss, sell after for profit |
| **Boss轮换策略** | 如果你有能力跳过当前Boss(用Boss标签)，下一个Boss从剩余池中抽取 / If you skip a Boss (via Boss Tag), the next Boss is drawn from the remaining pool |
| **牌组多样性** | 你的Build越依赖特定条件(如单一花色、单一牌型)，Boss越容易克制你——适当保持多样性 / The more conditional your build (single suit, single hand), the more vulnerable you are to Boss debuffs — maintain some flexibility |
