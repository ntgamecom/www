# 小丑牌图鉴 / Joker Catalog

## 核心小丑牌分类 / Joker Categories

| 分类 / Category | 说明 / Description | 代表小丑 / Example |
|:----:|------|------|
| **倍率型 / +Mult** | 提供加法倍率——基础分引擎 / Provide additive mult — score foundation | Abstract Joker, Scary Face |
| **乘算型 / ×Mult** | 提供乘法倍率——高分关键 / Provide multiplicative mult — key to huge scores | Barron, Vampire, Photograph |
| **筹码型 / +Chips** | 增加基础筹码——配合倍率用 / Increase base chips — pair with mult | Bull, Stone Joker |
| **经济型 / Economy** | 提供金钱或利息 / Provide money or interest | Golden Joker, Matador |
| **触发型 / Trigger** | 特定条件触发的额外效果 / Trigger effects on conditions | Hack, Sock and Buskin |
| **花色型 / Suit** | 基于花色的加成 / Suit-based bonuses | Ancient Joker, Rough Gem |

## 经典Build组合 / Classic Synergies

### 人头牌暴击流 / Face Card Crit Build

| 核心小丑 / Core Jokers | 配合 / Synergy |
|------|------|
| **照片(Photograph) × 2** | 第一张人头牌×2，第二张再×2 = ×4倍率 / First face ×2, second ×2 = ×4 Mult |
| **印花(Stamp)** | 每张人头牌+20筹码——基础分提升 / +20 Chips per face card |
| **笑脸(Smiley Face)** | 每张计分人头牌+5倍率 / +5 Mult per scored face card |
| **推荐牌型**: 对子(人头牌对子) / Pair (face card pair) | 简单稳定——每把都出人头对 / Simple — play face pairs every round |

### 同花爆炸流 / Flush Explosion Build

| 核心小丑 / Core Jokers | 配合 / Synergy |
|------|------|
| **血瓶(Bloodstone)** | ♥牌1/2概率×2——期望值×1.5x5张=巨大倍率 / 50% ×2 per ♥ — huge mult |
| **远古小丑(Ancient Joker)** | 每轮选一种花色——该花色牌给×2倍率 / Pick a suit — that suit gives ×2 Mult |

| **推荐牌型**: 同花(全♥或全♠) / Flush (all ♥ or all ♠) | 用塔罗牌把全牌组转成一种花色 / Use Tarot to convert deck to one suit |
