# 小丑牌图鉴 / Joker Catalog

小丑牌是Balatro的引擎——理解每个小丑牌的效果和配合方式是高分局的关键。小丑牌按稀有度分为普通(Common)、罕见(Uncommon)、稀有(Rare)和传奇(Legendary)四级 / Jokers are the engine of Balatro — understanding each Joker's effect and synergies is the key to high scores. Jokers come in four rarities: Common, Uncommon, Rare, and Legendary.

---

## 小丑牌稀有度 / Joker Rarity System

| 稀有度 / Rarity | 出率(约) / Approx Rate | 说明 / Description |
|:----:|:----:|------|
| **普通 / Common** | 70% | 基础效果——+倍率/+筹码。前期主力 / Basic effects — +Mult/+Chips. Early game backbone |
| **罕见 / Uncommon** | 22% | 条件触发的更强效果——中后期核心 / Conditional stronger effects — mid-to-late game core |
| **稀有 / Rare** | 7% | 构建Build的核心——通常提供×倍率 / Build enablers — often provide ×Mult |
| **传奇 / Legendary** | 1% | 游戏最强效果——一局可能见不到一个 / Game-breaking effects — may not see one per run |

**小丑牌出现机制**: 商店中每张小丑牌有独立稀有度判定。刷新(用金币)会重置商店所有卡牌——包括稀有度。所以想刷特定稀有牌，需要有足够金币反复刷新 / **Rarity mechanics**: Each Joker in the shop has an independent rarity roll. Rerolling (with gold) resets all cards — including rarities. To hunt specific rarities, save gold for rerolls.

## 核心小丑牌分类 / Joker Categories

| 分类 / Category | 说明 / Description | 代表小丑 / Example |
|:----:|------|------|
| **倍率型 / +Mult** | 提供加法倍率——基础分引擎。前期保底，后期容易被乘算型超越 / Provide additive mult — score foundation. Good early, outscaled by ×Mult late | Abstract Joker (+4 Mult), Scary Face (+5 Mult per face) |
| **乘算型 / ×Mult** | 提供乘法倍率——高分关键。效果通常是条件触发(特定卡牌/花色/牌型) / Provide multiplicative mult — key to insane scores. Usually conditional (specific cards/suits/hands) | Barron (×1.5 per King in hand), Photograph (×2 first face) |
| **筹码型 / +Chips** | 增加基础筹码——倍率再高也要筹码支撑。配合倍率型使用 / Increase base chips — even ×1000 mult means nothing with 0 chips. Pair with mult Jokers | Bull (+2 chips per $1), Stone Joker (+25 chips per stone card) |
| **经济型 / Economy** | 提供金钱或利息——间接战斗力。钱=更多刷新=更多小丑牌 / Provide money or interest — indirect power. Gold = more rerolls = more Jokers | Golden Joker ($4/round), Matador ($4 per Boss discard) |
| **触发型 / Trigger** | 卡牌被计分时触发额外效果——如再触发一次、复制、增强。增幅效果的核心 / Trigger extra effects when cards score — retrigger, copy, enhance. Core of amplification | Hack (retrigger 2-4 ranked), Sock and Buskin (retrigger face) |
| **花色型 / Suit** | 基于花色的加成——配合同花流 / Suit-based bonuses — pair with flush builds | Ancient Joker (×2 for one suit/round), Rough Gem (+7 chips per ♦) |

## 经典Build组合 / Classic Synergies

### 人头牌暴击流 / Face Card Crit Build (推荐新手 / Beginner-friendly)

| 核心小丑 / Core Jokers | 效果 / Effect | 获取难度 / Difficulty |
|------|------|:----:|
| **照片(Photograph)** | 第一张人头牌×2 | 罕见 / Uncommon |
| **笑脸(Smiley Face)** | 每张人头牌+5倍率 | 罕见 / Uncommon |
| **印花(Stamp)** | 每张人头牌+20筹码 | 普通 / Common |
| **部落(Tribe)** | 每张人头牌给+2倍率(替代笑脸) | 罕见 / Uncommon |
| **推荐牌型**: 对子(人头对)——因为只出2张牌也能触发照片，且出牌消耗少，每回合可以出更多次 / **Best hand**: Pair (face pair) — only 2 cards to trigger Photograph, fewer hand uses for more rounds |
| **Build策略**: 前中期用塔罗牌把牌组中非人头牌转成人头牌——优先转化♠♣(塔罗牌The Emperor)。后期配合染色、重触发小丑进一步放大 / **Strategy**: Convert non-face cards to face cards with Tarot. Focus on ♠♣ (The Emperor Tarot). Late game add retrigger effects |

### 同花爆炸流 / Flush Explosion Build

| 核心小丑 / Core Jokers | 效果 / Effect | 获取难度 / Difficulty |
|------|------|:----:|
| **血瓶(Bloodstone)** | ♥1/2概率×2——5张=期望值×32 / 50% ×2 per ♥ — 5♥ = ×32 expected | 稀有 / Rare |
| **远古小丑(Ancient Joker)** | 每轮一种花色×2——配合血瓶可叠加 / Rotating suit ×2 per round — stacks with Bloodstone | 稀有 / Rare |
| **粗糙宝石(Rough Gem)** | 每张♦给+7筹码——稳定垫高基础分 / +7 Chips per ♦ — stable base chip boost | 普通 / Common |
| **水印(Water)** | 第一张计分牌变所需花色——稳定同花 / First scored card changes to needed suit | 罕见 / Uncommon |
| **推荐牌型**: 5张同花(全♥或全♠)——用塔罗牌把牌组大量转成一种花色 / **Best hand**: 5-card Flush — use Tarot to convert deck to one suit |
| **Build策略**: 集中转单花色(推荐♥——因为血瓶只对♥生效)。The Hanged Man幻灵牌销毁废牌→浓缩牌组→用Death/Strength把低级牌变成同花色 / **Strategy**: Focus one suit (prefer ♥ for Bloodstone). Spectral Hanged Man destroys junk → concentrate deck → Death/Strength to convert cards to suit |

### 钢牌男爵流 / Steel Baron Build (高分局/高分挑战 / High score challenges)

| 核心小丑 / Core Jokers | 效果 / Effect |
|------|------|
| **男爵(Baron)** | 手牌中每张K×1.5——手牌满时7张K=×1.5^7≈×17 / Each King ×1.5, 7 Kings = ×17 |
| **钢× Steel Joker** | 牌组中每张钢牌×0.2——20张钢牌=×4.0 / Each Steel card ×0.2, 20 Steels = ×4.0 |
| **DNA** | 每回合复制第一张打出的牌——复制K然后转钢 / Each round copy first played card — copy King then make Steel |
| **筹谋(Blueprint)** | 复制男爵——×1.5变成×2.25 / Copy Baron — ×1.5 becomes ×2.25 |
| **推荐牌型**: 高牌(High Card)——因为不在乎手牌本身得分，只关注手牌中保留的K / **Best hand**: High Card — only care about Kings kept in hand |
| **Build策略**: 第一步拿DNA→复制K直到牌组大量K→用幻灵牌转钢牌→拿男爵和钢×→无限循环。这个Build是1e12成就的标准解法 / **Strategy**: Get DNA first → copy Kings → convert to Steel via Spectral → get Baron + Steel Joker → infinite loop. This is THE build for 1e12 achievement |

## 经济型小丑优先级 / Economy Joker Priority

| 小丑 / Joker | 效果 / Effect | 何时买 / When to Buy | 回本速度 / Payback |
|:----:|------|------|:----:|
| **金色小丑 / Golden Joker** | 每回合结束得$4 / $4 end of round | 前3关——越早越好 / Ante 1-3, ASAP | 2回合回本——最好经济小丑 / 2 rounds — best economy |
| **桶金 / Piggy Bank** | 利息上限提高到$25→$50 / Interest cap $25→$50 | 有钱之后——配合高利息 / When rich — with high interest | 长远投资——后期收益巨大 / Long-term investment |
| **赊账 / Credit Card** | 允许负钱购物——每回合-3利息 / Allow negative money, -$3 interest | 急需核心小丑时——比如在商店看到男爵但钱不够 / Only when you NEED that specific Joker | 应急工具——不是常规经济 / Emergency only |
| **节俭 / Thrift** | 每张保留的消费牌给$1 / $1 per held consumable | 中期——配合大量塔罗牌/星球牌 / Mid-game with many consumables | 每张消费牌$1——持有5张=$5/回合 / $1 per card |
| **斗牛士 / Matador** | Boss盲注每弃一张牌给$4 / $4 per Boss discard | 打Boss前购买——用完可以卖掉 / Buy before Boss blind, sell after | 一次性经济——不长期持有 / One-time use |

**经济管理核心原则**: Balatro的利息上限是$25(桶金后$50)——每保留$5给$1利息(上限$5/$10)。所以前中期目标是在每关结束时保持$25以上(最大化利息)，用多余的钱刷新商店找核心小丑。不要把钱全花光——留本吃利息 / **Core economy principle**: Interest = $1 per $5 held, cap $25 ($50 with Piggy Bank). Goal: maintain $25+ at end of each round for max interest. Don't spend everything — keep principal for interest income.
