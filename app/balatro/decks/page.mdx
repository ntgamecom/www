# 关卡与路线 / Biomes & Routes

小丑牌的关卡是盲注系统——每轮需要达到目标分数。但你也可以跳过盲注获得特殊标签奖励 / Balatro uses a blind system — each round needs a target score. You can also skip blinds for special tag rewards.

## 盲注策略 / Blind Strategy

| 盲注类型 / Type | 特点 / Feature | 策略 / Strategy |
|:----:|------|------|
| **小盲 / Small Blind** | 免费过关——不需要出牌 / Free pass — no hand needed | 默认跳过——拿标签奖励。除非没信心 / Default skip for tag bonus. Only play if underpowered |
| **大盲 / Big Blind** | 有金钱奖励 / $ reward | 打——除非标签奖励更好（如免费小丑牌标签）/ Play unless tag reward is better (free Joker tag etc) |
| **首领盲 / Boss Blind** | 有负面效果 / Has debuff | 必打——提前规划Build。如"禁止同花"就换牌型 / Always play — plan build around upcoming boss debuff |
