# 牌组与盲注 / Decks & Blinds

小丑牌的关进程围绕盲注系统——每轮需要达到目标分数。你可以选择挑战盲注获得金钱，也可以跳过盲注获得特殊标签奖励。牌组选择决定了你的起始资源和战术方向 / Balatro's progression revolves around the blind system — each round needs a target score. You can play blinds for cash or skip for special tag rewards. Deck choice determines your starting resources and tactical direction.

## 牌组策略详解 / Deck Strategy Guide

### 红牌组 / Red Deck

**适用Build**: 弃牌流——依赖弃牌触发效果的小丑牌(如"弃牌望远镜"、"第六感"/"兜售商")。打法：利用额外弃牌来刷出关键牌型。经济上中规中矩——+1弃牌帮你省下刷商店的钱 / **Best builds**: Discard-reliant builds — Jokers that trigger on discard (Telescope, Sixth Sense, Droll). Use extra discards to fish for key hands. Economy: average — free discards save shop reroll money.

**核心打法**: 前期多弃牌找高分牌型。中期配合"钢铁小丑"或"弃牌+倍率"类小丑形成combo。后期不缺弃牌——可以激进地弃掉所有低分组合 / **Core play**: Early: discard aggressively for high-scoring hands. Mid: pair with Steel Joker or discard-to-mult Jokers for synergy. Late: discard freely without worry.

### 蓝牌组 / Blue Deck

**适用Build**: 出牌触发类——如"血石"、"红牌"、"烤焦"等。+1出牌让每轮多一次得分机会——容错率最高。新手首选 / **Best builds**: Hand-trigger builds — Bloodstone, Red Card, Scorched. +1 hand means one extra scoring chance per round — highest error margin. Best for beginners.

**核心打法**: 出牌数的优势让蓝牌组可以走**保险路线**——先出小牌探路，再用大牌得分。经济上略吃亏(多出牌=少利息)。建议捡取所有+出牌数的星球牌 / **Core play**: Hand count advantage enables **safe play** — test with weak hands first, then score big. Economy slightly suffers (more hands = less interest). Grab all +hand Planet cards.

### 黑牌组 / Black Deck

**适用Build**: 小丑槽依赖流——每回合-$1是永久经济惩罚。额外小丑槽的优势要到中后期才显现。**新手不推荐** / **Best builds**: Joker slot-dependent builds. The -$1/round is a permanent economy penalty. The extra Joker slot only pays off mid-to-late. **Not for beginners**.

**核心打法**: 前期必须靠标签奖励补经济——跳过盲注拿"金钱标签"或"小丑标签"。中期6个小丑槽可以塞满combo——找"倍率×倍率"组合。后期: 7个小丑槽(基础5+黑牌1+Boss盲注奖励1)达到极致combo / **Core play**: Early: rely on tag rewards for economy — skip blinds for Money/Joker tags. Mid: 6 Joker slots enable maximum ×Mult combos. Late: up to 7 slots (base 5 + Black + boss reward) for ultimate synergy.

### 黄牌组 / Yellow Deck

**适用Build**: 经济流。$10开局让你在前3关可以买几乎所有见到的商店货物——快速建立小丑组合。新手友好 / **Best builds**: Economy builds. $10 head start lets you buy almost everything in shops for the first 3 rounds — fast Joker setup. Beginner-friendly.

**核心打法**: 前3关最大化利用资金优势——见小丑就买。利息策略从第4关开始执行。黄牌组的资金优势在5关后逐渐消失，但你的小丑组合应该已经成型了 / **Core play**: Spend aggressively first 3 rounds — buy every Joker. Start interest strategy from ante 4. Cash advantage fades after ante 5, but your Joker setup should be complete by then.

### 斑纹牌组 / Checkered Deck

**适用Build**: 同花流(Flush)——确定性极高。牌组的52张只有♠和♥，同花概率领先其他牌组10倍以上 / **Best builds**: Flush builds — extremely consistent. 52 cards are only ♠ and ♥, flush probability is 10x+ better than any other deck.

**核心打法**: 初期找同花相关小丑("同花之王"、"花色小丑"、"古朴小丑")。利用"世界"塔罗牌把其他牌转为♠或♥。斑纹牌组+蓝牌组风格=稳定通关 / **Core play**: Early: hunt flush-related Jokers (Flush King, Suit Joker, Rough Gem). Convert remaining cards via "World" tarot. Checkered Deck + Blue Deck style = consistent wins.

## 盲注策略 / Blind Strategy

| 盲注类型 / Type | 特点 / Feature | 策略 / Strategy |
|:----:|------|------|
| **小盲 / Small Blind** | 免费过关——不需要出牌 / Free pass | 默认跳过——拿标签奖励。除非你的Build太弱无法通过大盲 / Default skip for tag bonus. Only play if too weak for big blind |
| **大盲 / Big Blind** | 有金钱奖励 / $ reward | 打——除非标签奖励更好(免费小丑标签、负面标签等)/ Play unless tag is better (free Joker tag, Negative tag etc) |
| **首领盲 / Boss Blind** | 有负面效果 / Has debuff | 必打——提前2关规划Build应对Boss负面效果。如"禁止同花色"就转对子/"禁止对子"就转同花 / Always play — plan 2 rounds ahead for boss debuff. |

### 跳关决策 / Skip Decision Guide

**跳过小盲**通常是最优选择——标签奖励的价值通常超过小盲的金钱。但以下情况建议挑战小盲: 你当前一分钱都没有(连商店都买不起)；你的Build还没成型需要更多出牌机会；你正在存钱吃利息 / **Skipping small blind** is usually optimal — tag value exceeds small blind cash. Exceptions: you have $0 (can't even shop); your build needs more hands to stabilize; you're saving for interest.

## 标签奖励参考 / Tag Reference

| 标签 / Tag | 效果 / Effect | 跳过建议 / Skip Recommendation |
|:----:|------|------|
| **金钱标签 / Money Tag** | 下一关获得$25 | 前4关很值——25刀可以买2-3个小丑 / Great first 4 antes — $25 buys 2-3 Jokers |
| **星球标签 / Planet Tag** | 赠送随机星球牌×3 | 如果你的主牌型还没升级——必拿。星球牌只能通过商店或这种标签拿到 / Must-take if main hand not leveled. Planets only from shops or tags |
| **小丑标签 / Joker Tag** | 赠送随机小丑牌 | 初期必拿——免费小丑没有理由拒绝 / Must-take early — free Joker is always welcome |
| **小丑复制标签 / Joker Duplicate Tag** | 复制现有的一个小丑牌 | 有核心小丑牌时无敌——比如复制"Barron"或"血石" / OP if you have a core Joker — e.g., duplicate Barron or Bloodstone |
| **负面标签 / Negative Tag** | 赠送一个负面(不占槽)小丑牌 | 任何时候都值——不占槽的免费小丑。最值钱的标签之一 / Always worth — free slotless Joker. One of the best tags |
| **塔罗标签 / Tarot Tag** | 赠送塔罗牌×3 | 玩牌组改造流时值——可以改12张牌 + 找"死神"销毁低牌 / Good for deck-modifying builds — 12 cards modified + Death for deletions |
| **全息标签 / Holographic Tag** | 赠送全息(倍率+10)小丑牌 | 有价值——倍率总是好的。全息小丑是很好的过渡品 / Decent — free mult. Holographic Joker is a solid transitional piece |
| **巨大标签 / Mega Tag** | 免费最大化商店所有内容 / Max everything in next shop | 后期值——可以刷出传奇小丑牌。Ante 6+时必拿 / Valuable late — can spawn legendary Jokers. Must-take at Ante 6+ |
