# 隐藏要素 / Secrets & Unlocks

|---|---|---|

| **筹谋 / Blueprint** | 用高牌通关 / Win with High Card | 复制右边小丑效果 / Copy rightmost Joker effect |
| **男爵 / Baron** | 在无对子情况下通关游戏 / Win without ever having a Pair | 手牌每张K给×1.5 / Each King in hand ×1.5 |

## 特殊牌组策略 / Special Deck Strategies

### 等离子牌组详解 / Plasma Deck Guide

| 方面 / Aspect | 说明 / Description |
|---|---|
| **核心机制** | 筹码和倍率在计分时取平均——(筹码+倍率)/2作为新的筹码和倍率值。这使极端堆叠的效率翻倍 / Chips and mult are averaged at scoring — (chips+mult)/2 becomes both new chips and mult. This doubles the efficiency of extreme stacking |
| **推荐策略** | 放弃平衡发展——极端堆叠筹码或极端堆叠倍率。推荐堆筹码(因为筹码来源更稳定) / Don't balance — hyper-focus on either chips or mult. Chips are recommended (more consistent sources) |
| **最佳小丑** | Bull(+筹码/$)、Stone Joker(+筹码/石牌)、Steel Joker(×倍率/钢牌)。一个堆筹码一个堆倍率 / Bull(+Chips/$), Stone Joker(+Chips/stone), Steel Joker(×Mult/steel). One chips, one mult |
| **推荐牌型** | 高牌或对子——因为等离子牌组的分数计算方式让高牌/对子的低基础分不再是问题 / High Card or Pair — the averaging mechanic makes low base hands competitive |

### 幽灵牌组详解 / Ghost Deck Guide

| 方面 / Aspect | 说明 / Description |
|---|---|
| **核心优势** | 每关第一张幻灵牌免费——加上开局送的灵魂幻灵牌(必出传奇小丑)。前期爆发力极强 / First Spectral per round free + start with Soul Spectral (guaranteed Legendary). Massive early game spike |
| **推荐策略** | 第一关就用灵魂幻灵牌拿传奇小丑——然后围绕传奇小丑构建Build。中后期每关免费幻灵牌用来销毁废牌或复制核心牌 / Use Soul immediately for Legendary Joker — build around it. Later rounds use free Spectral for deck thinning or duplication |
| **最佳配合** | 幻灵牌"圣杯"(牌组中一张牌变钢牌)配合DNA无限复制 / Spectral Chalice (convert card to Steel) with DNA infinite duplication |
