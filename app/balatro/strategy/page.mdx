## 高分核心策略 / Core High-Score Strategy

### 倍率堆叠原理 / Mult Stacking

分数 = (基础筹码 + 筹码加成) × (基础倍率 + 加法倍率) × 乘法倍率连乘 / Score = (base chips + chip bonuses) × (base mult + add mult) × (mult × mult × mult...)

| 阶段 / Stage | 重点 / Focus | 目标 / Goal |
|:----:|------|------|
| **前4关 / Early (Ante 1-4)** | 加法倍率(+Mult) + 筹码 | 建立稳定分数引擎 / Build consistent scoring engine |
| **5-6关 / Mid (Ante 5-6)** | 寻找乘法倍率(×Mult) | 过渡到乘法阶段 / Transition to multiplicative scaling |
| **7-8关 / Late (Ante 7-8)** | 乘法倍率连乘 | 分数爆炸 / Score explosion |
| **无限模式 / Endless** | 纯乘法倍率 + 手牌中K/Q | 需要天文数字 / Need astronomical scores |

## 牌组改造进阶 / Advanced Deck Modification

| 操作 / Action | 方法 / Method | 效果 / Effect |
|:----:|------|------|
| **移除低牌 / Remove Low Cards** | 塔罗牌"死神"("The Death")——销毁2张低牌 / Tarot "Death" — destroy 2 low cards | 牌组变薄，更容易抽到高牌 / Thinner deck = more consistent |
| **增加同花色牌 / Add Same Suit** | 塔罗牌"战车"("The Chariot")——把1张牌变成指定花色 / Tarot "Chariot" — change 1 card to needed suit | 提高同花概率 / Higher flush consistency |
| **全牌组变一种点数 / Convert to One Rank** | 先用"魔术师"复制——再用"世界"转换 / Copy with "Magician" then convert with "World" | 稳定出四条或葫芦 / Consistent 4-of-a-kind or full house |
| **全钢牌化 / Steel Everything** | 幻灵牌"化钢"("Familiar") + 复制 / Spectral "Familiar" + duplication | 手牌中每张钢牌=×1.5倍率 / Each steel card in hand = ×1.5 Mult |

## Boss盲注应对 / Boss Blind Counters

| Boss负面效果 / Boss Debuff | 应对策略 / Counter Strategy |
|------|------|
| **禁用同花 / No Flush** | 提前准备第二种牌型——或者转对子流 / Have a backup hand type or pivot to pairs |
| **禁用顺子 / No Straight** | 同花流或对子流——不受影响 / Flush or pair builds — unaffected |
| **禁用花色 / Suit Disabled** | 用塔罗牌"世界"把牌转为可用花色 / Use "World" to convert cards to available suit |
| **弃牌无效 / Discards Disabled** | 增加出牌数——蓝牌组优势 / Stack hand count — Blue Deck advantage |
| **每轮出牌限制 / Hand Limit** | 集中出一次高分牌——出最强牌型 / Go all-in on one huge hand |
| **小丑牌禁用颜色 / Joker Color Ban** | 检查你的小丑牌颜色——备选方案 / Check Joker colors — have a backup plan |
| **翻倍盲注分数 / Score Doubled** | 确保你的分数引擎有富余量——不要踩线过关 / Overbuild your engine — don't cut it close |
