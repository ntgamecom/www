# 终局内容 / Endgame Content

通关Ante 8只是小丑牌的开始——真正的挑战在于无尽模式、高分冲击和挑战模式全通。本章覆盖所有终局玩法 / Clearing Ante 8 is just the beginning of Balatro — the real challenge lies in Endless Mode, high score runs, and completing all challenges. This chapter covers all endgame content.

---

## 通关后解锁 / Post-Win Unlocks

通关游戏一次后开启的内容 / Content unlocked after first win.

| 解锁内容 / Unlock | 说明 / Description |
|---|---|
| **挑战模式 / Challenge Mode** | 20个固定牌组的特殊挑战——通关后在下拉菜单中可以选择 / 20 fixed-deck special challenges — selectable from the dropdown menu after winning |
| **更高难度 / Higher Difficulty** | 无——但可以通过增加"赌注"(Stakes)提高难度。每个牌组有8个赌注等级 / None directly — but each deck has 8 Stake levels that increase difficulty |
| **无尽模式 / Endless Mode** | 通关Ante 8后继续游戏(一直打到分数不够为止)。在这个模式下你可以继续通关Boss直到分数要求超出你的上限 / After clearing Ante 8, the game continues (play until you can't score). Keep clearing bosses until score requirements exceed your output |

## 赌注系统 / Stake System

每个牌组有8个赌注——每个赌注增加游戏难度 / Each deck has 8 Stakes — each Stake increases game difficulty.

| 赌注 / Stake | 难度变化 / Difficulty Change | 说明 / Description |
|---|---|---|
| **白赌注 / White Stake** | 基础难度 / Base difficulty | 默认 / Default |
| **绿赌注 / Green Stake** | 目标分数+30% / Score requirement +30% | 增加了对Build效率的要求 / Requires more efficient builds |
| **黑赌注 / Black Stake** | +1个负面Boss标签 / Boss gets +1 extra debuff | Boss更棘手 / Bosses become trickier |
| **蓝赌注 / Blue Stake** | 目标分数+60% / Score requirement +60% | 进一步压缩存活空间 / Further squeeze on survival |
| **紫赌注 / Purple Stake** | 商店通常卡牌价格+25% / Shop card prices +25% | 经济更加紧张 / Economy becomes tighter |
| **橙赌注 / Orange Stake** | 目标分数+100% / Score requirement +100% | 非顶级Build难以通关 / Only top-tier builds can win |
| **金赌注 / Gold Stake** | 商店小丑牌价格+50% / Shop Joker prices +50% | 经济压力最大 / Maximum economic pressure |
| **红赌注 / Red Stake** | 以上全部+目标分数+150% / All above + score requirement +150% | 游戏最难难度——只有最强Build才能存活 / Hardest difficulty — only the strongest builds survive |

**金赌注通关奖励**: 每个牌组首次金赌注通关会解锁对应的金色牌组皮肤——共15个皮肤 / **Gold Stake reward**: Each deck's first Gold Stake win unlocks a golden version of that deck — 15 total skins.

### 无尽模式推荐Build / Endless Mode Builds

**Build 1: 男爵钢牌无限流 / Baron Steel Infinite (推荐 / Recommended)**

| 核心小丑 / Core Jokers | 作用 / Role |
|---|---|
| **男爵 / Baron** | 每张手牌K给×1.5倍率 / Each King in hand ×1.5 |
| **钢× / Steel Joker** | 每张牌组钢牌给×0.2 / Each Steel in deck ×0.2 |
| **DNA** | 每回合复制K进牌组 / Each round copy King into deck |
| **筹谋 / Blueprint** | 复制男爵或钢× / Copy Baron or Steel |

**操作流程**:
1. 先拿DNA → 每回合复制一张K到牌组
2. 用幻灵牌(圣杯)把K转成钢牌
3. 拿男爵和钢×小丑
4. 筹谋复制男爵(×1.5变×2.25)
5. 每回合用高牌出牌——保留手牌中的K

**优点**: 分数稳定增长——每多一张钢牌倍率就更高。理论可以无限增长 / Consistent growth — every extra Steel card increases mult. Theoretically infinite

**缺点**: 怕"利爪"Boss(限制出牌次数) / Weak vs Flint Boss (limited hands)

## 终局策略原则 / Endgame Strategy Principles

| 原则 / Principle | 说明 / Description |
|---|---|
| **倍率乘法叠加 / ×Mult Stacking** | 加法倍率终有极限——乘算倍率互相乘才是分数爆炸的关键。男爵(×1.5) + 筹谋(复制=×2.25) + 钢× = 指数级增长 / +Mult has limits — ×Mult stacking is how scores explode. Baron(×1.5) + Blueprint(copy=×2.25) + Steel× = exponential growth |
| **牌组压缩 / Deck Compression** | 牌组越小越好(30-45张)——确保抽到核心牌的稳定性。无尽模式中每回合多抽一张废牌都可能致命 / Smaller deck (30-45) = consistent draws. In Endless, one bad draw per round can kill a run |
| **跳过策略 / Skip Strategy** | 无尽模式中不要跳过Boss盲注——Boss标签在无尽模式中不生效。专注打怪+购物 / Don't skip bosses in Endless — Boss Tags don't work. Focus on fighting + shopping |
| **经济管理 / Economy Management** | 无尽模式中金钱越来越不重要(目标分数太高了)——但前期仍然是基础 / Money matters less in Endless (score requirements are too high) — but still important early |

## 常见终局问题 / Endgame FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **为什么我在Ante 12突然打不过？/ Why can't I pass Ante 12?** | 分数要求指数增长——到了Ante 12-14你需要的不是+Mult而是×Mult的堆叠。确保你至少有2-3个×Mult来源 / Score req grows exponentially — by Ante 12-14 you need ×Mult stacking, not +Mult. Ensure 2-3 ×Mult sources |
| **需要特定的Boss对策吗？/ Need specific boss counters?** | 无尽模式最怕"利爪"(限制出牌)和"圣杯"(限制消耗牌)。男爵流对利爪免疫——因此它是无尽首选 / Flint (limited hands) and Grail (limited consumables) are biggest threats. Baron build is immune to Flint — hence it's the top choice |
| **有没有最高分上限？/ Is there a score cap?** | 技术上没有——但分数的显示方式在1e308后变为"无限"(Infinity)/"NaN"。这是浮点数显示限制，不是游戏上限 / Technically no — but score display changes to "Infinity"/"NaN" after 1e308. This is a floating-point display limit, not a game cap |

## 挑战模式全攻略 / Challenge Mode Full Guide

### 困难挑战详解 / Hard Challenge Breakdown

| 挑战 / Challenge | 难点 / Difficulty | 推荐策略 / Strategy |
|---|---|---|
| **金手指 / The Golden Needle** | 每关只有1次出牌机会——必须一次出牌达标。没有任何容错 / One hand per round — must score enough in a single play. Zero room for error |
| **策略** | 选高基础分牌型(四条或同花顺)——用星球牌大幅升级。经济全部投资在倍率小丑上 / Pick high base hand (Four of a Kind or Straight Flush) — heavy Planet upgrades. All money into mult Jokers |
| **过载 / Overclock** | 初始三倍倍率但每轮递减。分数要求也提升——后期倍率归零后极度困难 / Start with ×3 mult, decays each round. Score req also increases — extremely hard once mult zeroes out |
| **策略** | 前期快速打出高分数——不要拖回合。用等离子牌组在倍率消失前冲过去 / Score fast early — don't drag rounds. Use Plasma Deck to rush through before mult vanishes |
| **暴发户 / The Upstart** | 利息上限$50——经济爆炸但商店价格也爆炸 / Interest cap $50 — massive economy but prices are also inflated |
| **策略** | 疯狂存钱吃利息——钱够了全刷商店找核心小丑。经济不是问题，关键是找到对的牌 / Save aggressively for interest — once rich, dump all into shop rerolls. Money not the issue, finding the right cards is |
