# BOSS攻略 / Boss Strategies

艾尔登法环有数十个Boss——本章收录所有追忆Boss和主要Boss的详细打法。按主线推进顺序排列 / Elden Ring features dozens of bosses — this chapter covers all remembrance bosses and major boss strategies, ordered by main story progression.

---

## 主线Boss / Main Story Bosses

### 恶兆妖鬼玛尔基特 / Margit, the Fell Omen

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 史东薇尔城门口 / Stormveil gate |
| **推荐等级 / Level** | 15-25 |
| **弱点 / Weakness** | 打击、出血 / Strike, Bleed |
| **抗性 / Resistances** | 圣 / Holy |

**打法要点 / Strategy:**
- 玛尔基特的攻击节奏多变——会延迟出招。不要看到抬手就滚——等武器快落下时再翻滚 / Margit has delayed attacks — don't roll on his wind-up, roll when the weapon is about to land
- 召唤涅斐丽·露的水母骨灰可以吸引仇恨 / Summon Nepheli Loux or spirit jellyfish to draw aggro
- 可以使用道具"玛尔基特的镣铐"（在雾林地下找到）两次定住他 / Use "Margit's Shackles" (found in Mistwood) to stun him twice
- 推荐使用**打击类武器**或**出血武器**（如初始打刀）/ Use **strike weapons** or **bleed weapons** (like starting katana)

> **建议**：如果打不过——先去啜泣半岛探索升级再来 / **Tip**: If you're stuck — explore the Weeping Peninsula and level up first

### 接肢葛瑞克 / Godrick the Grafted

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 史东薇尔城深处 / Stormveil depths |
| **推荐等级 / Level** | 25-35 |
| **弱点 / Weakness** | 出血 / Bleed |
| **抗性 / Resistances** | 圣 / Holy |

**打法要点 / Strategy:**
- **第一阶段**: 葛瑞克使用斧头和龙头攻击。保持中距离，等他出完大硬直后上前输出 / Phase 1: Uses axe and dragon head. Stay mid-range, punish after big attacks
- **第二阶段**: 他给自己装上龙头——范围火焰攻击。拉开距离等火焰结束后再打 / Phase 2: Attaches a dragon head — AoE fire attacks. Back off during fire, then engage
- **推荐近战贴身打**——他的远程龙头喷火很容易躲，近战反而安全 / **Stay close** — dragon breath is easier to dodge up close
- 可以召唤**涅斐丽·露**（金色召唤标记在门口）/ Summon **Nepheli Loux** (gold sign at the door)
- 葛瑞克的**断肢**可以拿去大赐福换葛瑞克的大卢恩 / Trade Godrick's Remembrance at Roundtable Hold

### 满月女王蕾娜菈 / Rennala, Queen of the Full Moon

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 雷亚卢卡利亚魔法学院 / Raya Lucaria Academy |
| **推荐等级 / Level** | 40-50 |
| **弱点 / Weakness** | 物理——高冲击力打断施法 / Physical — high poise damage interrupts casting |
| **抗性 / Resistances** | 魔力 / Magic |

**打法要点 / Strategy:**
- **第一阶段**: 蕾娜菈悬浮在空中，周围有唱歌的学生。你需要击杀发金光的学生来让她的护盾破碎——然后攻击本体。重复3次 / Phase 1: Rennala floats with singing students. Kill the golden-glowing student to break her shield — then attack her. Repeat 3x
- **第二阶段**: 蕾娜菈使用强力魔法——召唤巨球、月光剑和灵魂。保持移动，近身猛攻 / Phase 2: Uses powerful magic — giant orbs, moon blade, and spirits. Stay mobile, rush her down
- **第二阶段的关键**：蕾娜菈的近战攻击很弱——一旦贴身她就是靶子 / **Key**: Rennala is weak in melee — once you close the distance, she's a sitting duck
- 击败后可以用**泪滴幼体**重置属性 / Defeating her unlocks **Larval Tear** respec

### 碎星拉塔恩 / Starscourge Radahn

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 红狮子城 / Redmane Castle |
| **推荐等级 / Level** | 60-75 |
| **弱点 / Weakness** | 出血、腐败 / Bleed, Rot |
| **抗性 / Resistances** | 魔力、雷 / Magic, Lightning |

**打法要点 / Strategy:**
- **祭典之战**——这是游戏中最壮观的Boss战之一。你可以召唤大量NPC协助作战 / A festival battle — one of the most spectacular fights in the game. You can summon many NPC allies
- **第一阶段**: 拉塔恩使用大弓远程射击——骑马上左右跑动躲箭。接近后召唤NPC群殴。不要正面硬扛 / Phase 1: Uses greatbow — ride Torrent zigzag to dodge arrows. Approach and summon NPCs. Don't face-tank
- **第二阶段**: 拉塔恩变成**流星**砸向地面——看到消失后立刻骑马跑开躲陨石。第二阶段他使用引力魔法和重力波 / Phase 2: Transforms into a **meteor** — sprint away on Torrent when he vanishes. Uses gravity magic and shockwaves
- **关键策略**: 不断重新召唤被击杀的NPC——他们的标记会反复出现 / **Key**: Keep re-summoning fallen NPCs — their signs reappear
- 击败后获得**碎星的追忆**和**拉塔恩大卢恩** / Drops Remembrance + Great Rune

### 恶兆王蒙葛特 / Morgott, the Omen King

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 王城罗德尔——黄金树大教堂 / Leyndell — Erdtree Sanctuary |
| **推荐等级 / Level** | 80-100 |
| **弱点 / Weakness** | 出血、打击 / Bleed, Strike |
| **抗性 / Resistances** | 圣 / Holy |

**打法要点 / Strategy:**
- 蒙葛特是玛尔基特的升级版——招式更多更快。同样有延迟斩击 / Morgott is an upgraded Margit — more moves, faster. Same delayed attacks
- **第一阶段**: 使用大剑和锤子攻击——招式华丽但大开大合。等他出完连招后输出 / Phase 1: Uses sword and hammer — flashy but telegraphed. Punish after combos
- **第二阶段**: 增加圣属性武器和范围攻击。他会跳起来插地造成大范围AOE——跳起躲避 / Phase 2: Adds holy weapons and AoE. Jumping ground slam — jump to dodge
- **召唤骨灰**可以大幅降低难度——推荐仿身泪滴 / **Summon spirit ashes** to greatly reduce difficulty — Mimic Tear recommended
- 击败后获得**恶兆王的追忆** / Drops Remembrance

### 火焰巨人 / Fire Giant

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 火焰大锅 / Flame Peak (Mountaintops) |
| **推荐等级 / Level** | 110-130 |
| **弱点 / Weakness** | 打击、出血 / Strike, Bleed |
| **抗性 / Resistances** | 火 / Fire |

**打法要点 / Strategy:**
- **第一阶段**: 攻击他**左腿**——他的左腿有伤，打这里伤害最高。保持在他身后 / Phase 1: Attack his **left leg** — it's wounded, deals bonus damage. Stay behind him
- **第二阶段**: 火焰巨人砸断自己的腿——变成用双手爬行。他的眼睛和手掌是弱点 / Phase 2: Breaks his own leg — crawls on hands. Eyes and hands are weak points
- **骑马战斗**非常有效——托雷特可以快速绕后躲避火焰喷射 / **Mounted combat** is very effective — Torrent can quickly circle and dodge fire sprays
- 推荐使用**出血武器**——火焰巨人血极厚，出血可以大幅加速战斗 / Use **bleed weapons** — the Fire Giant has enormous HP, bleed speeds up the fight significantly

### 黑剑玛利喀斯 / Maliketh, the Black Blade

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 崩毁的法姆·亚兹拉 / Farum Azula |
| **推荐等级 / Level** | 120-150 |
| **弱点 / Weakness** | 打击 / Strike |
| **抗性 / Resistances** | 圣——极高 / Holy — very high |

**打法要点 / Strategy:**
- **第一阶段（野兽祭司）**: 人形野兽祭司——使用石块投掷和近战撕咬。保持中距离等他攻击后反击 / Phase 1 (Beast Clergyman): Humanoid form — stone throws and melee bites. Stay mid-range and punish after attacks
- **第二阶段（黑剑）**: 玛利喀斯极为敏捷——跳来跳去，使用黑焰攻击。**命中后降低你的HP上限** / Phase 2 (Maliketh): Extremely agile — leaps around with black flame attacks. **Reduces your max HP on hit**
- **关键**: 玛利喀斯的血不算厚——但机动性极高。用**狮子斩**或其他高冲击力战技可以快速打出处决 / **Key**: Maliketh has moderate HP — but very high mobility. Use **Lion's Claw** or high-poise-damage skills for quick staggers
- **柱子是你的朋友**——绕柱躲他的跳斩 / **Pillars are your friend** — use them to block his jump attacks
- 击败后获得**黑剑的追忆** / Drops Remembrance

### 初始之王葛孚雷 / Godfrey, First Elden Lord

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 黄金树底层 / Leyndell, after burning the Erdtree |
| **推荐等级 / Level** | 130-150 |
| **弱点 / Weakness** | 出血 / Bleed |
| **抗性 / Resistances** | 圣 / Holy |

**打法要点 / Strategy:**
- **第一阶段（战士葛孚雷）**: 使用战斧大开大合——他的攻击范围大但前摇明显。等他砸地后输出 / Phase 1 (Warrior Godfrey): Uses a greataxe — wide sweeps with clear wind-ups. Punish after ground slams
- **第二阶段（初始之王）**: 葛孚雷扔掉斧头——使用**徒手格斗**。他会抓住你投掷——看到抓取动作翻滚躲避 / Phase 2 (Hoarah Loux): Throws away the axe — **hand-to-hand combat**. Watch for grab attacks — roll when you see the grab animation
- **第二阶段的关键**: 他的范围咆哮冲击波需要**跳跃躲避**——而不是翻滚 / **Key**: His AoE roar shockwave requires a **jump dodge** — not a roll
- 这是纯拼操作的战斗——没有取巧方法。学习他的出招节奏 / This is a pure skill check — no shortcuts. Learn his attack rhythms

### 黄金律法拉达冈 / Radagon of the Golden Order + 艾尔登之兽 / Elden Beast

最终Boss连续战斗——击败拉达冈后立刻面对艾尔登之兽 / Final boss gauntlet — immediately face the Elden Beast after defeating Radagon.

| Boss | 弱点 / Weakness | 抗性 / Resistance | 策略 / Strategy |
|------|------|------|------|
| **拉达冈 / Radagon** | 出血、打击 / Bleed, Strike | 圣 / Holy | 猛攻型Boss——不要给他喘息机会。他的攻击前摇明显但范围很大。圣属性伤害减半 / Aggressive boss — don't give him breathing room. Telegraphed attacks but wide range. -50% holy damage |
| **艾尔登之兽 / Elden Beast** | 雷 / Lightning | 圣 / Holy | 巨大的宇宙生物——不断游走远离你。使用神圣光环和光剑攻击。追逐它并持续输出 / Giant cosmic being — constantly swims away. Uses holy rings and sword. Chase and keep up DPS |

**通用策略 / General Strategy:**
- 推荐**圣减伤护符**（神圣龙徽护符+2）/ Equip **holy damage negation talisman** (Haligdrake +2)
- 艾尔登之兽阶段——**骑马没用**，只能跑步追 / Elden Beast phase — **Torrent is disabled**, you must sprint to chase
- 拉达冈吃**出血**——出血武器可以快速击杀第一阶段 / Radagon is weak to **bleed** — bleed weapons melt phase 1
- 击败后选择结局——触碰艾尔登法环成为艾尔登之王 / Choose ending — touch the Elden Ring to become Elden Lord

## 可选Boss / Optional Bosses

### 圣树女武神玛莲妮亚 / Malenia, Blade of Miquella

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 米凯拉的圣树底层 / Haligtree Roots |
| **推荐等级 / Level** | 130-160 |
| **弱点 / Weakness** | 出血、冰冻 / Bleed, Frost |
| **抗性 / Resistances** | 圣、毒 / Holy, Poison |

**游戏公认最难Boss**——没有之一 / **Widely considered the hardest boss in the game**

**一阶段 / Phase 1:**
- 玛莲妮亚的**水鸟乱舞**（Waterfowl Dance）是游戏中最致命的招式——看到她跳起后：第一段向外跑→第二段向内滚→第三段向她的方向滚 / Her **Waterfowl Dance** is the deadliest move in the game — when she leaps: run away (1st) → roll into her (2nd) → roll toward her (3rd)
- 她每次攻击**都能回血**——这也是她最难的原因之一 / She **heals on every hit** — one reason she's so hard
- 推荐使用**冻霜踏地**或**尸山血海**——冰冻+出血可以快速压制 / Use **Hoarfrost Stomp** or **Rivers of Blood** — frost + bleed for quick pressure
- **召唤仿身泪滴**——最好给她装个盾，让仿身泪滴吸引仇恨 / **Summon Mimic Tear** — equip it with a shield for aggro management

**二阶段 / Phase 2:**
- 玛莲妮亚展开翅膀——增加**猩红腐败**攻击和水鸟乱舞的频率 / Grows wings — adds **Scarlet Rot** attacks and more frequent Waterfowl Dance
- 她的**分身攻击**需要仔细判断真身位置 / Her **clone attack** requires careful positioning to find the real one
- 推荐带**腐败苔药**解除腐败状态 / Bring **Preserving Boluses** for rot cleansing

### 鲜血君王蒙格 / Mohg, Lord of Blood

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 蒙格温王朝 / Mohgwyn Palace |
| **推荐等级 / Level** | 120-150 |
| **弱点 / Weakness** | 出血 / Bleed |
| **抗性 / Resistances** | 火、圣 / Fire, Holy |

**打法要点 / Strategy:**
- **关键道具**: 在**利亚尼亚的白金村**获取"纯血骑士勋章"可以传送到蒙格温王朝 / **Key item**: Get "Pureblood Knight's Medal" from the White-Faced Varre quest for teleport
- **第一阶段**: 蒙格使用三叉戟攻击——出招较慢。保持中距离等他出完硬直再输出 / Phase 1: Trident attacks — slow wind-ups. Stay mid-range and punish
- **妮弥艾（Nihil）**: 蒙格在第二阶段开始时会**三次施法**给自己加盾并烧你的血——这无法躲避。必须在战斗前获取**蒙格的囚具**（在利亚尼亚地下）或在身上装**纯血骑士勋章** / Nihil: Mohg casts three times to buff himself and bleed you — unavoidable. Get **Mohg's Shackle** (Liurnia underground) before the fight
- 推荐使用**蒙格的囚具**——战斗中可以使用两次定住他 / Use **Mohg's Shackle** twice in the fight to stun him
- 这是一场需要高DPS的战斗——**出血武器**效果极佳 / High DPS fight — **bleed weapons** excel here

### 死龙弗尔桑克斯 / Lichdragon Fortissax

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 深根底层 / Deeproot Depths |
| **推荐等级 / Level** | 100-130 |
| **弱点 / Weakness** | 打击 / Strike |
| **抗性 / Resistances** | 雷、圣 / Lightning, Holy |

**打法要点 / Strategy:**
- 死龙体型巨大——攻击范围极广但前摇明显。站在**龙头侧面**攻击 / Giant dragon — wide hitboxes but long wind-ups. Attack from the **side of the head**
- 它的**雷击**需要翻滚躲避——不要站着不动 / Lightning strikes require rolling — don't stand still
- 骑马战斗效果一般——建议步战 / Mounted combat is mediocre — fight on foot
- 推荐使用**物理伤害高的武器**——打头伤害最高 / Use **high physical damage weapons** — hitting the head deals bonus damage

### 祖灵之民 / Regal Ancestor Spirit

| 信息 / Info | 说明 / Description |
|------|------|
| **地点 / Location** | 希芙拉河 / Siofra River |
| **推荐等级 / Level** | 50-70 |
| **弱点 / Weakness** | 物理 / Physical |

**打法要点 / Strategy:**
- 这是游戏中最**优雅的Boss战**——祖灵行动迟缓，攻击模式优美但容易预判 / One of the most **elegant boss fights** — slow movement, beautiful but telegraphed attacks
- 点亮希芙拉河的所有火焰让灵庙出现 / Light all flames in Siofra River to access the Ancestor Spirit
- **最简单的Boss之一**——保持移动，躲开它的冲锋和光芒攻击，找机会输出 / **One of the easiest bosses** — stay mobile, dodge charges and light attacks, punish openings
