# 战斗系统 / Combat System

艾尔登法环的战斗在传统魂系基础上加入了跳跃、骑战和战灰系统，带来前所未有的战术深度 / Elden Ring's combat builds on traditional souls combat with jumping, mounted combat, and the Ashes of War system, delivering unprecedented tactical depth.

---

## 近战战斗基础 / Melee Combat Basics

### 攻击类型 / Attack Types

| 攻击 / Attack | 操作 / Input | 用途 / Use |
|---|---|---|
| **轻攻击 / Light Attack** | R1 / 鼠标左键 | 快速输出，连击压制 / Fast DPS, combo pressure |
| **重攻击 / Heavy Attack** | R2 / Shift+左键 | 高伤害+削韧，可蓄力 / High damage + stagger, chargeable |
| **跳跃攻击 / Jump Attack** | 跳跃中+R1/R2 | 削韧最强——跳劈流核心 / Best stagger — jump attack build core |
| **冲刺攻击 / Running Attack** | 奔跑中+R1/R2 | 突进起手技 / Gap-closing opener |
| **蹲伏攻击 / Crouch Attack** | 蹲伏中+R1/R2 | 出招隐蔽，适合PVP / Sneaky attack, great for PvP |
| **双持攻击 / Power Stance** | 同类武器双手各一，L1 | 双持DPS翻倍 / Dual-wield DPS doubles |

### 削韧与处决 / Poise & Critical Hits

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **削韧值 / Poise Damage** | 每次攻击都有削韧值——累计打满后敌人出大硬直 / Each attack deals poise damage — fills a hidden stagger meter |
| **处决 / Critical Hit** | 大硬直时走到正面按R1处决——造成巨额伤害 / When staggered, press R1 in front for massive crit damage |
| **背刺 / Backstab** | 在敌人背后轻攻击——潜行秒杀小兵 / Light attack from behind — stealth kills mobs |
| **跳劈优势 / Jump Attack Advantage** | 跳跃攻击削韧最高——双持巨剑跳劈可快速打崩多数Boss / Jump attacks deal highest poise damage — dual greatsword jump attacks break most bosses fast |

## 防御与闪避 / Defense & Evasion

### 翻滚 / Rolling

| 负重档位 / Equip Load | 翻滚速度 / Roll Speed | 无敌帧 / I-Frames |
|---|---|---|
| **轻负重 < 30% / Light Load** | 最远最快 / Farthest, fastest | 13帧 / 13 frames |
| **中负重 30-70% / Med Load** | 标准 / Standard | 13帧 / 13 frames |
| **重负重 70-100% / Heavy Load** | 极慢翻滚 / Fat roll | 12帧 / 12 frames |
| **特重 > 100% / Overload** | 不能翻滚 / Can't roll | 无 / None |

> **建议**：保持中负重以下——重负重翻滚几乎没用。大部分玩家选择中负重以穿更好装备 / **Tip**: Stay under 70% equip load — fat rolling is nearly useless. Most players choose medium load for better armor.

### 格挡 / Blocking

| 盾牌类型 / Shield Type | 减伤率 / Damage Negation | 说明 / Description |
|---|---|---|
| **小盾 / Small Shield** | 60-75% | 轻便，战技弹反优秀 / Light, great for parry |
| **中盾 / Medium Shield** | 75-90% | 最佳平衡——推荐100%物防中盾 / Best balance — recommended 100% phys block |
| **大盾 / Great Shield** | 90-100% | 减伤最高但重——可用盾牌战技 / Best negation but heavy — has shield skills |
| **指痕盾 / Fingerprint Shield** | 100%+ | 最强大盾——最高防御强度 / Best greatshield — highest guard boost |

### 弹反 / Parrying

弹反是将敌人攻击弹开使其露出大硬直的高阶技巧——成功后可接处决 / Parrying deflects an enemy attack and leaves them open for a critical hit.

| 弹反工具 / Parry Tool | 弹反帧 / Active Frames | 难度 / Difficulty |
|---|---|---|
| **小盾 / Small Shield** | 8-12帧 | 最容易 / Easiest |
| **中盾 / Medium Shield** | 6-10帧 | 中等 / Medium |
| **曲剑 / Curved Sword** | 8-12帧 | 较容易 / Easier |
| **格挡匕首 / Parrying Dagger** | 8-12帧 | 较容易 / Easier |
| **战灰: 黄金格挡 / Ash: Golden Parry** | 10-14帧 | 最容易——有弹反光环 / Easiest — has parry aura |
| **战灰: 风暴格挡 / Ash: Storm Wall** | 10-14帧 | 容易 / Easy |

## 战灰系统 / Ashes of War

战灰是艾尔登法环最核心的武器自定义系统——每个武器都可以更换战技 / Ashes of War are the core weapon customization system — every weapon can swap its skill.

| 战灰分类 / Category | 示例 / Example | 用途 / Use |
|---|---|---|
| **物理系 / Physical** | 狮子斩、鲜血斩击 / Lion's Claw, Bloody Slash | 高伤害物攻 / High physical damage |
| **魔法系 / Magic** | 魔法阵、卡利亚大剑 / Spinning Weapon, Carian Greatsword | 智力补正 / INT scaling |
| **圣律系 / Sacred** | 圣律剑刃、共享圣律 / Sacred Blade, Shared Order | 对不死特攻 / Anti-undead |
| **毒/血系 / Status** | 毒雾、切腹 / Poisonous Mist, Seppuku | 异常状态累积 / Status buildup |
| **位移系 / Mobility** | 猎犬步法、碎步 / Bloodhound's Step, Quickstep | 闪避无敌 / Dodge i-frames |
| **防御系 / Defensive** | 铁壁盾防、无战技 / Barricade Shield, No Skill | 盾牌强化 / Shield boost |

## 骑战系统 / Mounted Combat

骑乘灵马托雷特时可以进行骑战——对野外Boss和群战极其有效 / Mounted combat on Torrent is extremely effective against field bosses and groups.

| 骑战操作 / Mounted Action | 操作 / Input | 说明 / Description |
|---|---|---|
| **骑乘攻击 / Mounted Attack** | R1/R2 | 单手剑/枪横扫 / Sword/spear sweep |
| **骑乘冲刺攻击 / Mounted Charge** | 奔跑中R1/R2 | 高伤害冲锋 / High damage charge |
| **双持骑乘攻击 / Power Stance Mounted** | L1 | 双持武器骑砍 / Dual-wield mounted |
| **下马攻击 / Dismount Attack** | 下马中按攻击 | 下马跳劈 / Dismount jump attack |
| **骑乘闪避 / Mounted Dodge** | ○ / Space | 马匹加速闪避 / Horse speed burst |

| 优势 / Advantage | 说明 / Description |
|---|---|
| **机动性 / Mobility** | 马匹速度快——可以风筝敌人 / High speed — kite enemies |
| **跳跃 / Jump** | 马可二段跳——到达步行不可达之处 / Double jump — reach unreachable areas |
| **高度优势 / Height** | 居高临下——攻击可打到弱点 / Higher vantage point — hit weak points |
| **回避 / Evasion** | 骑马回避距离大——但吃伤害会硬直 / Long evade distance — but taking damage stuns |

## PvP对战 / PvP Combat

### 入侵与决斗 / Invasions & Duels

| 模式 / Mode | 触发方式 / Trigger | 说明 / Description |
|---|---|---|
| **入侵 / Invasion** | 使用唤勾指药 / Use Taunter's Tongue | 被入侵或入侵他人——2v1或1v1 / Get invaded or invade — 2v1 or 1v1 |
| **决斗 / Duel** | 画红色召唤符 / Red Summon Sign | 1v1公平决斗——通常在学院门前 / 1v1 fair duels — usually at Raya Lucaria gate |
| **竞技场 / Colosseum** | 圆桌厅堂或竞技场 / Roundtable or Colosseum | 官方PvP模式——死斗/组队/混战 / Official PvP — duel/team/ffa |

### PvP技巧 / PvP Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **延迟注意 / Latency Awareness** | 联机有延迟——提前预判翻滚和攻击 / Online has latency — pre-emptive rolls and attacks |
| **装备负重 / Equip Load** | PvP中保持中负重以下——翻滚距离重要 / Stay under med load — roll distance matters |
| **战技选择 / Skill Choice** | 猎犬步法过去过于强势但现在已削弱 / Bloodhound's Step was OP but now nerfed |
| **指头/魔法 / Spells** | 迅剑和魔法辉剑是PvP强势法术 / Swift Glintstone Shard and Glintblade are PvP meta |
| **忍耐 / Endurance** | 物理减伤战技——冲脸换血利器 / Phys damage reduction skill — great for trading |

## 战斗常见问题 / Combat FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **为什么我打不出处决？/ Can't crit?** | 需要打满削韧值——重武器和跳跃攻击削韧高 / Need to fill stagger meter — heavy weapons and jump attacks deal more poise damage |
| **弹反时机是什么？/ Parry timing?** | 在敌人武器即将碰到你的瞬间按L1——练习在圆桌厅堂的骑士处 / Press L1 right before the weapon hits — practice at Roundtable knight |
| **负重多少合适？/ Ideal equip load?** | 30%-70%是最佳区间——低于30%滚得更远但没有明显帧数优势 / 30-70% is ideal — below 30% rolls farther but same i-frames |
| **战灰可以更换吗？/ Can I change Ash of War?** | 可以——在任何赐福处更换。丢失的战灰可以在铁匠处重新获得 / Yes — at any Site of Grace. Lost ashes can be re-obtained at smiths |
| **魔法和祷告能同时用吗？/ Magic + Incantations?** | 需要同时装备法杖和圣印——战斗中切换 / Equip both staff and seal — switch during combat |
