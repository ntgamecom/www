# 技能与法术 / Skills & Magic

艾尔登法环拥有系列最丰富的技能和法术系统——战灰、魔法、祷告三大体系提供了数百种战术选择 / Elden Ring features the richest skill and magic system in the series — Ashes of War, sorceries, and incantations offer hundreds of tactical options.

## 魔法系统 / Sorcery System

魔法需要法杖和智力属性——主要分为辉石、重力、卡利亚和暗月流派 / Sorceries require a staff and INT — divided into Glintstone, Gravity, Carian, and Moon schools.

### 法师属性阈值 / Mage Stat Thresholds

| 智力 / INT | 解锁内容 / Unlocks |
|---|---|
| **10-20** | 基础辉石魔法——辉石魔砾、辉石弯弧 / Basic glintstone — Pebble, Arc |
| **20-30** | 卡利亚剑术——卡利亚大剑、卡利亚迅剑 / Carian sword spells — Greatsword, Piercer |
| **30-40** | 重力魔法——岩石球、碎星 / Gravity — Rock Sling, Starscourge |
| **40-50** | 亚杜拉月光剑、彗星亚兹勒 / Adula's Moonblade, Comet Azur |
| **50-60** | 暗月大剑、毁灭流星 / Dark Moon Greatsword, Stars of Ruin |
| **60-70** | 最强魔法——艾尔登流星 / Elden Stars (best at 68+) |
| **80+** | 法爷毕业——智力软上限 / Mage soft cap |

### 信仰属性阈值 / Faith Stat Thresholds

| **10-20** | 基础火焰+治愈祷告 / Basic fire + heal incants |

| **30-40** | 龙飨祷告——龙火、龙冰 / Dragon Communion — Dragonfire, Dragonice |

| **60-80** | 最强祷告——虫丝、腐败吐息 / Pest Threads, Ekzyke's Decay |
