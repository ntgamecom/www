# 艾尔登法环攻略 / Elden Ring Guide

Martin. Explore six major regions of the Lands Between, challenge demigod bosses, and become the Elden Lord.

## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **开放世界 / Open World** | 六大区域无缝连接——自由探索，任选路线 / Six seamless regions — explore freely, choose your own path |
| **坐骑系统 / Torrent** | 灵马托雷特——快速移动+骑战 / Spectral steed — fast travel + mounted combat |
| **战灰系统 / Ashes of War** | 可更换的武器战技——自由搭配 / Swappable weapon skills — freely customizable |
| **骨灰召唤 / Spirit Ashes** | 召唤灵魂协助战斗——单人玩家的福音 / Summon spirits to aid in battle — a boon for solo players |
| **联机系统 / Multiplayer** | 合作推图/PVP入侵——召唤符和唤勾指药 / Co-op and PVP invasion — summon signs and Taunter's Tongue |
| **跳跃 / Jump** | 首次在魂系游戏中加入跳跃键——彻底改变探索方式 / First souls game with a jump button — transforms exploration |
| **地图碎片 / Map Fragments** | 需收集地图碎片来解锁区域地图 / Collect map fragments to reveal regional maps |
| **赐福点 / Sites of Grace** | 类似篝火的存档点——引导之线指引主线路 / Bonfire-like checkpoints — guidance lines point to main path |

## 开局推荐 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **选武士或流浪骑士 / Start as Samurai or Vagabond** | 这两个职业前期装备好，属性分配合理，适合新手 / Best starting gear and stat distribution for beginners |
| **2** | **优先加生命力 / Prioritize Vigor** | 前期全部点数投入生命力到40——活着才有输出 / Invest all early levels into Vigor to 40 — alive = DPS |
| **3** | **跟着赐福指引 / Follow the Grace** | 赐福发出的金色粒子指引主线方向——迷路时跟着走 / Gold particles from Sites of Grace point to main path |
| **4** | **利用骨灰 / Use Spirit Ashes** | 遇到困难Boss时召唤骨灰——它们可以吸引仇恨帮你创造输出机会 / Summon spirits for tough bosses — they draw aggro and create openings |
| **5** | **别怕逃 / Don't Fear Running** | 打不过的Boss/区域可以绕过——先去别的地方升级变强再回来 / Can't beat a boss/area? Skip it — level up elsewhere and come back later |
| **6** | **马战很重要 / Master Mounted Combat** | 骑马上打野外Boss和精英有巨大优势——打不过就上马 / Mounted combat is OP for field bosses and elites — mount up when struggling |
| **7** | **地图碎片先拿 / Get Map Fragments First** | 每到一个新区域——先找地图碎片（地图上的火炬图标）再探索 / In a new region — find the map fragment (torch icon) before exploring |
| **8** | **升级武器优先于升级 / Upgrade Weapons First** | 武器升级带来的伤害提升远大于属性加点——锻造石是最高优先级 / Weapon upgrades > stat levels — smithing stones are top priority |
