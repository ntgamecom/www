## 核心机制 / Core Mechanics

### 赐福 / Sites of Grace

赐福是交界地的存档点——功能和黑魂的篝火类似 / Sites of Grace are the checkpoints of the Lands Between — similar to Dark Souls bonfires.

| 功能 / Function | 说明 / Description |
|------|------|
| **休息 / Rest** | 恢复HP/FP/精力，刷新敌人，重置消耗品 / Restores HP/FP/stamina, respawns enemies, refills consumables |
| **升级 / Level Up** | 在圆桌厅堂或梅琳娜出现后可在赐福升级 / Level up after reaching Roundtable Hold or meeting Melina |
| **传送 / Fast Travel** | 在任何赐福间自由传送 / Free fast travel between any discovered Sites of Grace |
| **记忆 / Memorize** | 在赐福中记忆/遗忘法术和祷告 / Memorize/forget sorceries and incantations |
| **调配灵药 / Mix Physick** | 使用圣杯瓶调配灵药——限时增益效果 / Mix the Wondrous Physick — temporary buff effect |
| **战灰 / Ash of War** | 在赐福中更换战灰 / Equip/change Ashes of War |
| **引导之线 / Guidance** | 赐福发出的金色粒子指向主线方向 / Gold particles point toward main story path |

### 圣杯瓶 / Flask of Crimson & Cerulean Tears

| 瓶 / Flask | 效果 / Effect | 升级方式 / Upgrade |
|------|------|------|
| **红露滴圣杯瓶 / Crimson Tears** | 恢复HP / Restore HP | 用圣杯露滴在赐福升级 / Upgrade with Sacred Tears at Sites of Grace |
| **蓝露滴圣杯瓶 / Cerulean Tears** | 恢复FP（蓝条）/ Restore FP | 同上 / Same |
| **分配次数 / Allocation** | 可在赐福中分配红蓝瓶数量 / Allocate between red/blue at Sites of Grace |

> **注意**：圣杯露滴在各地教堂中可以找到——全地图共有约12个，全部找齐后可达+12最大等级 / **Note**: Sacred Tears are found in churches across the map — ~12 total, max upgrade level +12

### 死亡机制 / Death Mechanics

| 死亡后果 / Death Consequence | 说明 / Description |
|------|------|
| **卢恩丢失 / Rune Loss** | 死亡时掉落在死亡地点积累的所有卢恩 / Drop all accumulated runes at death location |
| **捡回卢恩 / Rune Retrieval** | 跑到死亡地点触碰血痕即可捡回 / Run to death spot and touch bloodstain to recover |
| **再次死亡 / Second Death** | 在捡回卢恩前再次死亡——卢恩永久丢失 / Die again before retrieving — runes are permanently lost |
| **地图标记 / Map Marker** | 地图上会标记你的死亡位置——有金色边框 / Death location marked on map with gold border |

### 骨灰召唤 / Spirit Ashes

| 机制 / Mechanic | 说明 / Description |
|------|------|
| **获取 / Acquisition** | 在各地找到或从商人购买——可以升级 / Found throughout world or bought from merchants — can be upgraded |
| **使用 / Usage** | 在有墓碑图标的位置才能召唤 / Can only summon at locations with tombstone icon |
| **消耗 / Cost** | 每次召唤消耗FP——部分特殊骨灰消耗HP / Costs FP per summon — some special ashes cost HP |
| **升级 / Upgrade** | 找罗德莉卡在圆桌厅堂升级——使用墓地铃兰 / Upgrade with Roderika at Roundtable Hold — uses Grave Glovewort |
| **最强骨灰 / Best Ashes** | 仿身泪滴（最强）、黑刀狄希、古龙骑士克里斯托福 / Mimic Tear (strongest), Black Knife Tiche, Ancient Dragon Knight Kristoff |

## 开局路线建议 / Starting Path

```
引导之始 → 宁姆格福（南部）→ 探险宁姆格福全图
→ 史东薇尔城（Boss: 恶兆妖鬼 + 接肢葛瑞克）→ 利耶尼亚（北部）
→ 卡利亚城寨 → 盖利德（右下方红土区域）→ 亚坛高原
→ 王城罗德尔 → 巨人山顶 → 化圣雪原 → 逐渐进入结局路线
```

## 新手常见问题 / FAQ for Beginners

| 问题 / Question | 回答 / Answer |
|------|------|
| **打不过大树守卫怎么办？/ Tree Sentinel too hard?** | 绕过！他不是必打Boss——先去探索别的地方升级再来 / Skip him! He's optional — explore elsewhere and come back later |
| **卢恩掉了捡不回来怎么办？/ Lost runes?** | 别焦虑——卢恩可以再刷。后期有更好的刷卢恩地点 / Don't panic — runes are farmable. Better farming spots later |
| **武器怎么升级？/ How to upgrade?** | 找铁匠（圆桌厅堂或大赐福）使用锻造石 / Talk to smith (Roundtable or Great Forge) with smithing stones |
| **怎么双持武器？/ How to power stance?** | 装备两把同类武器后按R1/L1双持攻击——伤害翻倍 / Equip two same-type weapons, press R1/L1 for power-stance attacks |
| **联机需要什么？/ Online requirements?** | 使用唤勾指药（入侵）或绘制召唤标记（合作）/ Use Taunter's Tongue (invasion) or summon signs (co-op) |
| **DLC怎么进？/ How to enter DLC?** | 击败碎星拉塔恩+蒙格后，在鲜血王朝的茧前互动 / Defeat Starscourge Radahn + Mohg, then interact with cocoon in Mohgwyn Palace |
