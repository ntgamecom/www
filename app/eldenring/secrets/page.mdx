## DLC: 黄金树幽影 / Shadow of the Erdtree

### 进入方法 / How to Enter

| 步骤 / Step | 说明 / Description |
|---|---|
| **1** | 击败碎星拉塔恩 + 蒙格鲜血君王 / Defeat Starscourge Radahn + Mohg |
| **2** | 前往鲜血王朝——蒙格温的茧 / Go to Mohgwyn Palace — the cocoon |
| **3** | 互动进入幽影之地 / Interact to enter Land of Shadow |

### DLC新要素 / DLC New Features

| 要素 / Feature | 说明 / Description |
|---|---|
| **幽影之地 / Land of Shadow** | 全新开放世界区域——约等于本体2个区域的体量 / New open world zone — about 2 base-game regions in size |
| **幽影树碎片 / Scadutree Fragments** | 收集提升幽影树祝福——增加攻击力和减伤 / Collect to boost Scadutree Blessing — increases attack and defense in DLC |
| **灵灰 / Revered Spirit Ashes** | 强化骨灰召唤在DLC中的强度 / Strengthens spirit summons in DLC |
| **新武器类型 / New Weapon Types** | 格斗术、调香瓶、投掷剑、反手剑、兽爪、大太刀 / Hand-to-hand, perfume bottles, throwing blades, backhand blades, beast claws, great katanas |
| **新战灰 / New Ashes of War** | 约30+新战灰 / ~30+ new Ashes of War |
| **新魔法/祷告 / New Spells** | 约20+新法术 / ~20+ new spells |
| **新Boss / New Bosses** | 约10+个主要Boss——含穿刺者梅瑟莫等 / ~10+ main bosses — including Messmer the Impaler |

### DLC主要Boss / DLC Main Bosses

| Boss | 幽影祝福建议 / Scadutree Blessing | 策略 / Strategy |
|---|---|---|

| **癫火之王米德拉 / Midra** | 12-15 | 发狂抗性很重要 / Madness resist essential |
| **穿刺者梅瑟莫/第二形态 / Messmer Phase 2** | 15-18 | 蛇形攻击——用冰属性 / Serpent attacks — ice weakness |
| **拉塔恩/约定之王 / Promised Consort Radahn** | 18-20 | 最终Boss——极其困难 / Final boss — extremely difficult |
