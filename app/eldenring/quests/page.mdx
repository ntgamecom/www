# 任务与NPC / Quests & NPCs

艾尔登法环的NPC支线以隐晦著称——不提供任务日志，全靠玩家探索和记录对话 / Elden Ring's NPC quests are famously obscure — no quest log, relying entirely on player exploration and dialogue tracking.

## 任务常见问题 / Quests FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **不小心打了NPC怎么办？/ Hit an NPC?** | 去教会镇的乌龟那里赎罪——需要星星泪滴 / Go to Church of Vows turtle — need Celestial Dew |
| **NPC不见了？/ NPC disappeared?** | 检查他们下一个位置——大部分NPC会在多个地点出现 / Check their next location — most NPCs move between spots |
| **癫火怎么解除？/ Remove Frenzied Flame?** | 使用米莉森的金针——打败女武神后得到 / Use Millicent's needle — get after defeating Malenia |
| **怎么联机？/ How to co-op?** | 使用唤勾指放标记——对方使用唤勾指药 / Use Tarnished's Furled Finger — other uses Taunter's Tongue |
| **入侵等级范围？/ Invasion level range?** | 通常±20级——武器强化也影响匹配 / Usually ±20 levels — weapon upgrade also affects matchmaking |
| **金色召唤符和红色有什么区别？/ Gold vs Red sign?** | 金色=合作（被召唤当帮手），红色=决斗（PVP）/ Gold=co-op, Red=PvP duel |
| **怎么增加入侵次数？/ More invasion items?** | 溃烂血指可无限使用——但复仇之证是消耗品 / Festering Bloody Finger is infinite — Recusant is consumable |
| **白面具梵雷任务线？/ Varre quest?** | 在白面具处用血指入侵三次——获得纯血骑士勋章去鲜血王朝 / Invade 3 times — get Pureblood Knight's Medal to Mohgwyn |
| **火山官邸入侵任务？/ Volcano Manor invasions?** | 加入火山官邸后接受委托——入侵特定NPC / After joining — accept contracts to invade specific NPCs |
| **联机密码怎么设置？/ Password settings?** | 系统→网络→联机密码——和朋友联机必须设置相同密码 / System → Network → Password — must match with friends |
| **白面具任务能跳过吗？/ Skip Varre quest?** | 不能跳过——这是去鲜血王朝最快的方法。不做的话要从化圣雪原传送过去 / Can't skip — fastest way to Mohgwyn. Alternative is Snowfield teleporter |
