# 世界探索 / World Exploration

交界地由六大主要区域和多个地下区域组成——每个区域都有独特的地貌、Boss和秘密。按大致推图顺序排列 / The Lands Between consists of six main regions and several underground areas — each with unique landscapes, bosses, and secrets. Ordered by recommended progression.

---

## 宁姆格福 / Limgrave

新手区域——包含南部草原、雾林、半岛。推荐开局后先探索整个宁姆格福再去史东薇尔城 / Starting area — southern plains, Mistwood, and the Peninsula. Explore all of Limgrave before entering Stormveil.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **引导之始 / Stranded Graveyard** | 游戏起点——完成教程后出门 / Starting point — exit tutorial cave | 教程NPC、出门就是开阔世界 / Tutorial NPC, open world |
| **艾蕾教堂 / Church of Elleh** | 第一个商人——也是拿召唤铃的地方 / First merchant — also where you get Spirit Calling Bell | 商人咖列、晚上见菈妮拿铃铛 / Merchant Kale, Ranni at night for bell |
| **关卡前方 / Gatefront** | 史东薇尔城入口——有大量士兵 / Stormveil entrance — lots of soldiers | 关卡前方赐福、地图碎片 / Gatefront Site of Grace, map fragment |
| **雾林 / Mistwood** | 浓雾覆盖的森林——有传说级BOSS / Mist-covered forest — legendary boss | 半狼布莱泽、死之鸟 / Blaidd the Half-Wolf, Deathbird |
| **啜泣半岛 / Weeping Peninsula** | 宁姆格福南部的半岛——适合早期探索 / Peninsula south of Limgrave — good early-game area | 摩恩城（狮子混种Boss）/ Castle Morne (Leonine Misbegotten) |
| **龙息废墟 / Dragon-Burnt Ruins** | 有地下室的废墟——内含宝箱陷阱 / Underground cellar — trapped chest inside | 陷阱传送到盖利德的矿洞——前期高风险高回报 / Trap teleports to Caelid tunnel — high risk, high reward |

### 宁姆格福关键Boss / Limgrave Key Bosses

| Boss | 地点 / Location | 掉落 / Drop | 难度 / Difficulty |
|------|------|------|:----:|
| **大树守卫 / Tree Sentinel** | 引导之始出门 / Start area | 黄金戟 / Golden Halberd | ★★★★☆ |
| **恶兆妖鬼 / Margit** | 史东薇尔城门口 / Stormveil gate | 护符皮袋 + 大量卢恩 / Talisman pouch + many runes | ★★★☆☆ |
| **接肢葛瑞克 / Godrick** | 史东薇尔城深处 / Stormveil depths | 葛瑞克的大卢恩 + 接肢的追忆 / Godrick's Great Rune + Remembrance | ★★★☆☆ |
| **猎犬骑士达瑞威尔 / Darriwil** | 雾林废墟 / Mistwood ruins | 猎犬长牙 + 猎犬步伐战灰 / Bloodhound's Fang + Bloodhound's Step | ★★★☆☆ |

## 利耶尼亚 / Liurnia

湖泊区域——从史东薇尔城向北经过湖即可到达。魔法学院坐落于此 / Lake region — reachable by going north from Stormveil across the lake. Home to the Raya Lucaria Academy.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **湖区 / Lake Area** | 大片的浅水区——有巨人划船 / Vast shallow water — giants rowing boats | 多个地下墓穴、结晶洞窟 / Multiple catacombs, crystal caves |
| **雷亚卢卡利亚魔法学院 / Raya Lucaria** | 魔法学院——满月女王所在地 / Magic academy — home of Rennala | 满月女王的追忆、卡利亚法杖 / Remembrance of Rennala, Carian Regal Scepter |
| **卡利亚城寨 / Caria Manor** | 通往卡利亚王室区域 / Path to Carian royal area | 三姐妹塔——菈妮支线 / Three Sisters towers — Ranni's questline |
| **湖之利耶尼亚矿洞 / Liurnia Tunnel** | 重要锻造石来源 / Important smithing stone source | 锻造石1-4级 / Smithing stones [1]-[4] |

### 利耶尼亚关键Boss / Liurnia Key Bosses

| Boss | 地点 / Location | 掉落 / Drop |
|------|------|------|
| **满月女王蕾娜菈 / Rennala** | 魔法学院顶部 / Raya Lucaria summit | 满月女王的追忆 + 重置属性的泪滴幼体 / Remembrance + Larval Tear (respec) |
| **禁卫骑士罗蕾塔 / Loretta** | 卡利亚城寨顶部 / Caria Manor summit | 罗蕾塔的大弓战灰 / Loretta's Greatbow Ash of War |

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **艾奥尼亚沼泽 / Aeonia Swamp** | 腐败沼泽——需要高免疫力 / Rot swamp — needs high immunity | 艾奥尼亚蝶、腐烂树灵 / Aeonian Butterfly, Putrid Tree Spirit |
| **红狮子城 / Redmane Castle** | 拉塔恩祭典场所——碎星Boss战 / Radahn's festival grounds — Starscourge boss fight | 碎星拉塔恩（追忆+大卢恩）/ Starscourge Radahn |
| **盖利德神授塔 / Caelid Tower** | 激活拉塔恩大卢恩的地方 / Activate Radahn's Great Rune here | 神授塔 + 野兽神殿支线 / Divine Tower + Bestial Sanctum quest |
| **腐败教堂 / Church of Plague** | 重要道具位置 / Important item location | 磷火余烬——升级骨灰材料 / Ghost Glovewort — spirit ash upgrade material |

## 亚坛高原 / Altus Plateau

从利耶尼亚北部迪可达斯大升降梯或矿洞绕路到达——王城罗德尔就在前方 / Reached via the Grand Lift of Dectus or a tunnel route from Liurnia — Leyndell lies ahead.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|------|------|------|
| **迪可达斯大升降梯 / Grand Lift of Dectus** | 需要两块迪可达斯符节才能启动 / Requires two Dectus Medallions | 符节一半在海德要塞/另一半在法洛尼亚要塞 / One half in Fort Haight, other in Fort Faroth |
| **亚坛大道 / Altus Highway** | 通往王城的主路 / Main road to Leyndell | 多个地牢 + 日荫城 / Multiple dungeons + Shaded Castle |
| **日荫城 / Shaded Castle** | 火山官邸支线相关区域 / Volcanic Manor quest area | 玛雷家行刑剑 / Marais Executioner's Sword |
| **火山官邸 / Volcano Manor** | 支线区域——刺杀任务 / Optional area — assassination quests | 拉卡德的大卢恩 + 亵渎神躯 / Rykard's Great Rune + Blasphemous Blade |

### 亚坛高原关键Boss / Altus Key Bosses

## 隐藏区域 / Hidden Areas

### 米凯拉的圣树 / Miquella's Haligtree

游戏最难的隐藏区域——需要圣树秘密符节（一半在索尔城，一半在化圣雪原的隐蔽处） / The hardest hidden area in the game — requires the Haligtree Secret Medallion (half in Castle Sol, half hidden in Consecrated Snowfield).

| 区域子区 / Sub-area | 说明 / Description |
|------|------|
| **圣树镇 / Elphael** | 内部区域——极度危险，有大量精英 / Interior area — extremely dangerous, many elites |
| **圣树底层 / Haligtree Roots** | 玛莲妮亚Boss所在地 / Malenia boss arena |

> **关键Boss**：**圣树女武神玛莲妮亚 / Malenia, Blade of Miquella**——公认游戏最难Boss。掉落：女武神的追忆 + 大卢恩 / Considered the hardest boss in the game. Drops: Remembrance + Great Rune

### 崩毁的法姆·亚兹拉 / Crumbling Farum Azula

击败火焰巨人后被传送到此——游戏终点前最后一站 / Teleported here after defeating the Fire Giant — the final area before the endgame.

| 关键Boss | 掉落 / Drop |
|------|------|
| **神皮双人组 / Godskin Duo** | 神皮追忆 / Remembrance |
| **黑剑玛利喀斯 / Maliketh** | 黑剑的追忆 + 大卢恩 / Remembrance + Great Rune |
| **野兽祭司 / Beast Clergyman** | 玛利喀斯第一阶段 / Maliketh phase 1 |
