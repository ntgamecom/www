# 殿堂攻略 / Palace Walkthroughs

本章节详细解析游戏中全部8个殿堂的攻略路线、Will种子位置、Boss打法及秘宝夺取流程。每个殿堂都是一次独特的潜入体验，掌握正确的路线可以节省大量时间。

This chapter provides detailed walkthroughs for all 8 Palaces in the game, including Will Seed locations, boss strategies, and treasure extraction routes. Each Palace offers a unique infiltration experience, and knowing the correct route saves significant time.

## 殿堂系统概览 / Palace System Overview

殿堂是扭曲欲望的具象化空间，每个殿堂由一名腐败的成年人的心灵产生。攻略殿堂是游戏主线的核心内容。

Palaces are physical manifestations of distorted desires, each generated by a corrupt adult\'s heart. Infiltrating Palaces is the core of the main story.

### 共通机制 / Common Mechanics

- **安全等级 Security Level**：被敌人发现会提升安全等级，等级越高敌人越警觉、增援越多。降低等级可以通过躲在掩体后等待或使用特定道具
- **Will种子 Will Seeds**：每个殿堂有3颗，收集后获得饰品，为对应殿堂的Persona提供强化
- **秘宝路线 Treasure Route**：到达秘宝所在房间后自动解锁路线，之后可以发预告信
- **发预告信 Calling Card**：必须发预告信后才能夺取秘宝。发信后第二天殿堂Boss会变强
- **地图完成率 Map Completion**：探索全部区域可以提升地图完成率，获得奖励

- Security Level: Increases when spotted by enemies; higher levels make enemies more alert. Reduce it by hiding or using items.
- Will Seeds: 3 per Palace, collecting all rewards an accessory that boosts the associated Persona.
- Treasure Route: Unlocked upon reaching the Treasure room, allowing you to send the calling card.
- Calling Card: Must be sent before stealing the Treasure. The boss becomes stronger after the card is sent.
- Map Completion: Exploring all areas increases completion rate for rewards.

## 第一殿堂：城堡 / Palace 1 — Kamoshida\'s Castle

**主人**：鸭志田卓（Suguru Kamoshida）
**主题**：城堡 Castle
**目标**：夺取金牌 Treasure: Gold Medal
**截止日期**：5月2日 Deadline: May 2

### 攻略流程 / Walkthrough

1. **4月12日（强制）**：剧情进入殿堂，基础教学战斗，与坂本龙司汇合
2. **4月15日（强制）**：再次进入殿堂，遇到摩尔加纳，学习潜入基本操作
3. **自由攻略期**：4月18日之后可以自由进入殿堂推进
4. **推荐攻略日**：4月24日-28日之间完成全部探索
5. **发预告信**：最晚5月1日发信，5月2日夺取秘宝

### 关键区域 / Key Areas

- **地下牢房 Underground Jail**：教学区域，Will种子1
- **中庭 Courtyard**：宽阔区域，有巡逻敌人
- **城楼 Castle Keep**：Boss区域前的最上层

### Will种子位置 / Will Seed Locations

| 编号 | 区域 | 具体位置 |
|---|---|---|
| 1 | 地下牢房 | 第一个岔路左侧牢房内 |
| 2 | 中庭 | 铁索桥后右侧宝箱旁 |
| 3 | 城楼 | 二楼阳台尽头 |

**奖励饰品**：**金银奖牌 Gold Silver Medal**（物理攻击力小幅提升）

### Boss战：鸭志田卓 / Boss: Suguru Kamoshida

**阶段1**：鸭志田单体，弱点是冰属性（Ice）。建议使用Morgarna的冰属性技能。他会使用"英雄般的演技"召唤两个小兵。

**阶段2**：鸭志田骑上战马形态。弱点仍然是冰属性。他会使用"王者的嘶吼"清除我方Buff。注意回复HP。

**攻略要点**：
- 全员换上有冰属性技能的Persona
- 在鸭志田召唤小兵时优先击倒小兵获得One More
- Boss的物理攻击伤害较高，保持HP在一半以上

## 第二殿堂：美术馆 / Palace 2 — Madarame\'s Museum

**主人**：斑目一流斎（Ichiryusai Madarame）
**主题**：美术馆 Museum
**目标**：夺取画作 Treasure: Painting
**截止日期**：6月5日

### 攻略流程 / Walkthrough

1. **5月3日以后**：解锁进入殿堂，斑目美术馆
2. **关键机制**：需要找到四个钥匙卡（四色门禁卡）以打开不同区域的门
3. **推荐攻略日**：5月20日-25日之间完成探索
4. **发预告信**：最晚6月4日

### 四色门禁卡 / Four Color Keycards

- **红色门禁卡 Red Card**：一楼展厅左侧的宝箱中
- **蓝色门禁卡 Blue Card**：二楼东侧展室
- **黄色门禁卡 Yellow Card**：三楼南侧走廊
- **绿色门禁卡 Green Card**：地下水道区域

### Will种子位置

| 编号 | 区域 | 具体位置 |
|---|---|---|
| 1 | 一楼展厅 | 红色门禁卡所在宝箱附近 |
| 2 | 二楼西侧 | 需要蓝色门禁卡 |
| 3 | 屋顶花园 | 最终区域前 |

**奖励饰品**：**沙之书 Sand Book**（提升冰属性防御力）

### Boss战：斑目一流斎 / Boss: Ichiryusai Madarame

Boss战分为四个阶段，斑目会召唤不同属性的画作分身。

| 阶段 | 分身属性 | 弱点 |
|---|---|---|
| 初音 | 风属性 Wind | 无弱点，高回避 |
| 和音 | 回复 | 物理攻击 |
| 狂言 | 物理 | 冰属性 |
| 不羁 | 电属性 | 风属性 |

**攻略要点**：
- 使用属性攻击击破分身，每击破一个分身，Boss的防御力下降
- 优先击破"不羁"分身（弱风），减少电属性攻击威胁
- 斑目的本体在分身全部击破前不会受到全额伤害
- 使用Yusuke的冰属性技能覆盖多个目标

## 第三殿堂：银行 / Palace 3 — Kaneshiro\'s Bank

**主人**：金城润星（Junya Kaneshiro）
**主题**：银行 Bank
**目标**：夺取金库 Treasure: Safe
**截止日期**：7月9日

### 攻略流程

1. **6月5日以后**：解锁进入
2. **关键机制**：需要收集三个密码（金城身体上的纹身数字），分别为2226、5999、9831
3. **推荐攻略日**：6月20日-30日

### 三个密码位置 / Three Code Locations

密码隐藏在纹身中，需要在殿堂中探索得知。
- **密码1**：2226 — 办公区第一个房间的电脑
- **密码2**：5999 — 金库区走廊的备忘录
- **密码3**：9831 — 天台的通风管道旁

### Will种子位置

| 编号 | 区域 |
|---|---|
| 1 | 办公区保险柜 |
| 2 | 金库区走廊角落 |
| 3 | 天台区域 |

**奖励饰品**：**银行家胸针 Banker\'s Brooch**（提升电属性攻击力）

### Boss战：金城润星 / Boss: Junya Kaneshiro

Boss是巨大的金库形态（Kaneshiro\'s Piggy Bank）。他会召唤成堆的钱币小怪。

**阶段1**：金库形态，具有很高的防御力。需要在击倒小怪获得One More后攻击本体。

**阶段2**：金城真人形态，弱点是风属性（Wind）。

**攻略要点**：
- 使用范围攻击清理小怪
- 阶段1中使用物理攻击攻击钱币堆
- 阶段2中全员使用风属性技能
- 保持Buff和Debuff覆盖（攻击力下降Debuff对Boss有效）

## 第四殿堂：金字塔 / Palace 4 — Futaba\'s Pyramid

**主人**：佐仓双叶（Futaba Sakura）
**主题**：金字塔 Pyramid / Tomb
**目标**：夺取宝物 Treasure: Headset
**截止日期**：8月31日

### 攻略流程

1. **7月9日以后**：解锁进入
2. **关键机制**：双叶的殿堂是巨大的机关迷宫，需要使用开关和激光陷阱
3. **推荐攻略日**：7月20日-8月10日

### 激光陷阱机关 / Laser Trap Puzzles

- 使用操作杆调整激光方向
- 部分激光是绊线警报，触碰会提高安全等级
- 有些激光必须解除才能通过

### Will种子位置

| 编号 | 区域 |
|---|---|
| 1 | 入口大厅右侧通道 |
| 2 | 墓室走廊中间 |
| 3 | 王座厅前的房间 |

**奖励饰品**：**太阳之书 Sun Book**（祝福属性攻击强化）

### Boss战：双叶的认知 / Boss: Futaba\'s Cognition

这场战斗是特殊的守护者战。Boss是巨大的"认知双叶"，战斗机制独特。

**攻略要点**：
- Boss会使用"诅咒"和"绝望"状态攻击
- 准备好治疗绝望的药品
- Boss召唤的小怪需要优先清理
- 使用物理攻击攻击Boss本体效果较好

## 第五殿堂：宇宙空间站 / Palace 5 — Okumura\'s Space Station

**主人**：奥村邦和（Kunio Okumura）
**主题**：宇宙空间站 Space Station
**目标**：夺取宝物 Treasure: Luxury Products
**截止日期**：10月11日

### 攻略流程

1. **8月31日以后**：解锁进入
2. **关键机制**：需要乘坐传送带穿越不同区域
3. **重要提醒**：这是游戏中最容易卡关的Boss战之一

### 传送带区域 / Conveyor Belt Areas

传送带会将角色带向特定方向。部分传送带需要开启开关才能改变方向。注意在传送带上会遇到强制战斗。

### Will种子位置

| 编号 | 区域 |
|---|---|
| 1 | 第一传送带区域的角落 |
| 2 | 第三传送带的中段 |
| 3 | 重力控制室 |

**奖励饰品**：**宇宙徽章 Space Medal**（提升咒怨属性攻击力）

### Boss战：奥村邦和 / Boss: Kunio Okumura

这是游戏中最具挑战性的Boss战之一。Boss会不断召唤机器人小兵，必须在限定时间内全部击倒。

**阶段流程**：Boss会召唤4批机器人，每批4个（特定颜色组合），必须在2回合内消灭全部。如果没有在规定时间内消灭，Boss会使用"自爆"全灭我方。

**攻略要点**：
- **关键**：使用单体弱点攻击配合Baton Pass
- 第一波：弱风属性的机器人
- 第二波：弱冰属性
- 第三波：弱电属性
- 第四波：各种属性混合
- 提前准备充足的SP回复道具
- 使用攻击Buff（Charge/Concentrate）增大伤害
- Morgana的Magarudyne（高级风属性群攻）非常关键
- 如果Haru在队伍中，使用她的Psi技能
- 确保有充足的Baton Pass等级以最大化伤害链

## 第六殿堂：赌场 / Palace 6 — Niijima\'s Casino

**主人**：新岛冴（Sae Niijima）
**主题**：赌场 Casino
**目标**：夺取宝物 Treasure: Medal
**截止日期**：11月20日

### 攻略流程

1. **10月11日以后**：解锁进入
2. **关键机制**：伪装成荷官潜入，使用筹码系统
3. **推荐攻略日**：10月20日-11月5日

### 筹码机制 / Chip Mechanic

在该殿堂中，您的行动受筹码数量限制。通过完成小游戏和击败敌人获取筹码。部分门需要支付筹码才能通过。

### Will种子位置

| 编号 | 区域 |
|---|---|
| 1 | 一楼角子机区 |
| 2 | 二楼贵宾室 |
| 3 | 地下金库前 |

**奖励饰品**：**幸运骰子 Lucky Dice**（提升暴击率）

### Boss战：新岛冴 / Boss: Sae Niijima

Boss是骑摩托车的形态（Sae\'s Shadow）。她具有多种属性攻击和强力物理技能。

**攻略要点**：
- Boss会改变属性弱点，每回合使用扫描查看
- 准备好全属性覆盖的Persona
- 她会使用"加速"提升行动次数
- 使用Debuff降低她的命中率和防御力
- Makoto的核热属性攻击效果很好

## 第七殿堂：巡航舰 / Palace 7 — Shido\'s Cruiser

**主人**：狮童正义（Masayoshi Shido）
**主题**：豪华游轮 Luxury Cruiser
**目标**：夺取宝物 Treasure: Legislation
**截止日期**：12月18日

### 攻略流程

1. **11月20日以后**：解锁进入
2. **关键机制**：需要收集乘客的"认可票券"，通过击败精英敌人获得
3. **推荐攻略日**：11月25日-12月10日
4. **注意**：这是流程最长的殿堂，建议分配两天完成

### 认可票券 / Approval Tickets

需要收集到足够数量的认可票券才能进入最终区域。票券可以通过击败特定敌人和在隐藏房间中寻找获得。

### Will种子位置

| 编号 | 区域 |
|---|---|
| 1 | 甲板区域 |
| 2 | 宴会厅夹层 |
| 3 | 船长室前走廊 |

**奖励饰品**：**航海罗盘 Navigation Compass**（提升命中率）

### Boss战：狮童正义 / Boss: Masayoshi Shido

Boss战分为五个阶段（对应狮童的五个人格形态），是游戏中阶段最多的Boss战。

**阶段1**：政治家形态，弱点是风属性
**阶段2**：战士形态，弱点是电属性
**阶段3**：祭司形态，弱点是冰属性
**阶段4**：圣母形态，弱点是核热属性
**阶段5**：最终形态，全属性无效但有固定破绽

**攻略要点**：
- 每个阶段都要准备不同的属性攻击
- 这是测试玩家Persona覆盖面是否全面的战斗
- 保持全员等级在65以上
- 大量准备SP完全回复道具
- 使用Ariadne的Thread或类似的全回复技能

## 第八殿堂：实验室 / Palace 8 — Maruki\'s Laboratory

**主人**：丸喜拓人（Takuto Maruki）
**主题**：研究所 / 实验室 Laboratory
**目标**：夺取宝物 Treasure: Data
**解锁条件**：丸喜Coop达到Rank 9（必须在11月18日之前完成）

### 攻略流程

1. **1月2日以后**（第三学期）：解锁进入
2. **关键机制**：这是一个数据化空间的实验室，使用彩色门禁卡解锁
3. **推荐攻略日**：1月10日-1月20日
4. **截止日期**：2月3日

### Will种子位置

| 编号 | 区域 |
|---|---|
| 1 | B1层数据中心 |
| 2 | B2层培养室 |
| 3 | B3层核心区域前 |

**奖励饰品**：**真理之钥 Key of Truth**（全属性攻击力提升）

### Boss战：丸喜拓人 / Boss: Takuto Maruki

这是游戏的最终Boss，具有极其强大的战斗能力。

**阶段1**：丸喜的正常形态，使用全能性攻击
**阶段2**：丸喜的认知融合形态，可以使用多种属性
**阶段3**：亚当·卡德蒙（Adam Kadmon）形态，终极形态

**攻略要点**：
- 所有队员等级推荐75以上
- 准备全能属性耐性的Persona
- Boss会使用"绝望"和"洗脑"状态攻击，准备对应治疗物品
- 第三阶段会强制使用特定角色
- 使用"Charge"和"Concentrate"增幅伤害
- 最终一击使用Showtime会有特殊演出

## 殿堂攻略技巧总结 / Palace Strategy Summary

### 通用建议 / General Tips

- **一次通关 vs 多次攻略**：皇家版优化了SP恢复机制，尽可能在一次潜入中完成殿堂探索，但不必强求
- **安全屋利用**：到达新安全屋后返回入口再重新潜入可以刷新部分敌人
- **Will种子优先**：每到达一个新区域先探寻Will种子位置
- **SP管理**：携带SP回复物品，利用物理攻击节省SP
- **等级均衡**：确保所有队员等级接近，避免单人等级过低
- **备用装备**：每人准备两把武器（针对不同属性弱点）
