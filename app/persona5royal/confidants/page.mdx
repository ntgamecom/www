# Coop角色 / Confidants

Coop（协助关系）系统是《女神异闻录5 皇家版》中发展社交关系并获得战斗和日常增益的核心系统。全作共有23位Coop角色，每位对应一个塔罗Arcana。发展Coop关系不仅可以获得强力能力，还能解锁高阶Persona的合成权限。

The Confidant system is central to building social relationships in P5R, providing combat and daily life benefits. There are 23 Confidants total, each tied to a Tarot Arcana. Ranking up Confidants unlocks powerful abilities and high-level Persona fusion.

## Coop系统基础 / Confidant System Basics

### 解锁与提升 / Unlocking and Ranking

每位Coop角色有特定的解锁条件，通常是剧情推进到特定日期或达到特定五维属性要求。提升Rank需要与该角色共度时光（通常每次提升消耗一个白天或傍晚时间段）。

Each Confidant has specific unlock conditions tied to story progress or social stat requirements. Ranking up requires spending time with them, usually consuming a daytime or evening slot.

### 好感度机制 / Affinity Points

每次与Coop角色互动会增加好感度点数。好感度积累到一定程度后触发Rank升级事件。以下方式可以增加好感度：

- **共度时光**：进行Coop活动获得基础好感度
- **选择正确对话选项**：在剧情事件中选择角色喜欢的回答
- **携带对应Arcana的Persona**：携带与Coop角色同Arcana的Persona可提升好感度获取量（约1.2倍）
- **使用占卜**：新宿占卜师Chihaya的"好感度占卜"可提升效果
- **赠送礼品**：特定礼物可大幅提升好感度

### 好感度对话选项指南 / Dialogue Option Guide

每个Coop角色都有3-4个Rank升级事件，每个事件中有多个对话选择。正确选择可以大幅提升好感度。

通关时携带对应Arcana的Persona可以使对话加分加倍。以下是通用的选择策略：

- 认真回应角色的问题，选择支持、理解、鼓励的选项
- 避免冷漠、嘲讽或选择沉默
- 对于恋爱选项，在Rank 8-9时会有明确选择
