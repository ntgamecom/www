# 结局指南 / Endings Guide

《女神异闻录5 皇家版》拥有多个结局，包括多个Bad Ending（坏结局）、两个Normal Ending（普通结局）和一个True Ending（真结局）。本章节详细说明每个结局的达成条件和关键选择。

Persona 5 Royal features multiple endings, including several bad endings, two normal endings, and one true ending. This chapter details the conditions and key choices for each ending.

## 结局分类 / Ending Categories

| 结局类型 | 数量 | 说明 |
|---|---|---|
| Bad Ending 坏结局 | 5种 | 游戏提前结束，不进入后续章节 |
| Normal Ending 普通结局 | 1种 | 原版P5的结局（没有第三学期） |
| True Ending 真结局 | 1种 | 皇家版追加结局，包含第三学期 |

## Bad Ending 条件 / Bad Ending Conditions

Bad Ending会在游戏中多个节点出现，通常是在关键选择中做出了错误决定。

### Bad Ending 1 — 11月20日 / 出卖伙伴

**触发条件**：在狮童殿堂攻略中，当新岛冴（Sae Niijima）审讯主角时，选择"出卖同伴"的选项。

**具体场景**：在接受审讯时，会被问到关于怪盗团成员的信息。如果选择如实交代所有成员的真实身份，游戏会直接结束。

**避免方法**：在审讯中始终保持沉默，选择"我不会说的"、"我不记得了"等拒绝回答的选项。

### Bad Ending 2 — 12月18日 / 接受交易

**触发条件**：在狮童殿堂剧情结束后，被检察官询问是否同意一项秘密交易。如果选择"接受交易"，游戏立即结束。

**避免方法**：选择"拒绝交易"或"我不同意"。

### Bad Ending 3 — 12月24日 / 放弃抵抗

**触发条件**：在圣诞节前夕的剧情事件中，主角可以选择放弃作为怪盗的身份。如果选择放弃，游戏结束。

**避免方法**：选择"我不会放弃"或"继续作为怪盗"。

### Bad Ending 4 — 1月2日 / 接受虚假世界

**触发条件**（第三学期）：在第三学期开始时，丸喜拓人向主角展示了他的"理想世界"。如果选择"接受这个世界"，游戏直接进入工作人员表。

**避免方法**：选择"这不是我想要的"或"拒绝接受"。

### Bad Ending 5 — 2月3日 / 放弃斗争

**触发条件**（第三学期）：在丸喜殿堂最终决战前，主角可以选择放弃挑战。选择放弃则进入Bad Ending。

**避免方法**：选择"我不会放弃"或"我要战斗到底"。

|---|---|---|---|
| 1 | 11月20日 | 审讯中出卖同伴 | "说出实情" |
| 2 | 12月18日 | 接受检察官交易 | "接受交易" |

## Normal Ending 普通结局 / Original Ending

**条件**：完成狮童殿堂的攻略并击败最终Boss（狮童正义），但丸喜拓人的Coop等级未达到Rank 9（或未在11月18日前达到Rank 8以上）。

**结局内容**：
- 游戏在圣诞节后结束
- 回到原版的结局：怪盗团成功让狮童悔改，主角回国后回到秀尽学园
- 没有第三学期内容
- 芳泽霞的角色故事不完整
- 明智吾郎的故事线以开放方式结束

**区别提示**：Normal Ending的结局阶段在1月左右结束，而True Ending会持续到3月。

## True Ending 真结局 / Royal True Ending

**条件**：需要满足以下全部条件：

1. **丸喜拓人（顾问官 Councillor）Coop达到Rank 9**（必须在11月18日前完成）。注意：Rank 9和Rank 10的事件触发时间不同，Rank 9是必要条件
2. **在1月2日拒绝丸喜的虚假世界**（选择"我不会接受"或"拒绝"）
3. **在2月3日前完成丸喜殿堂的攻略**
4. **在最终决战中击败最终Boss丸喜拓人**

### 第三学期解锁详细流程 / Third Semester Unlock Process

```
11月18日前 → Maruki Coop Rank 9（最关键）
     ↓
12月24日 → 圣诞节剧情（正常进行）
     ↓
1月1日-3日 → 新年事件
     ↓
1月4日 → 第三学期开始，芳泽霞正式入队
     ↓
1月5日-9日 → 自由行动期间
     ↓
1月10日 → 丸喜殿堂解锁
     ↓
1月10日-2月2日 → 殿堂攻略期
     ↓
2月3日 → 最终决战（截止日期）
     ↓
2月4日-14日 → 结局事件、情人节
     ↓
2月15日-3月 → 真结局
```

### 真结局追加内容 / True Ending Additional Content

相比Normal Ending，真结局追加了以下内容：

- **丸喜拓人第三个学期**：包含8-10小时的全新剧情
- **新殿堂**：丸喜的实验室（研究所主题殿堂）
- **新Boss**：丸喜拓人 / 亚当·卡德蒙（Adam Kadmon）
- **新角色完成度**：芳泽霞的完整故事线和Coop结局
- **明智吾郎的完整结局**：其剧情在第三学期得到完整收束
- **隐藏Boss**：完成第三学期后可以挑战隐藏最强Boss
- **追加Persona**：伊邪那岐大神、撒旦耶尔等超强力Persona解锁
- **最终展示**：真结局后的额外CG和结尾

## Maruki Coop 截止日期 / Maruki Coop Deadline

**最晚日期**：11月17日（必须在11月18日的剧情事件之前）

**每周可用时间**：
- Maruki通常每周有2-3次可互动时间
- 学校午休和放学后可以在副校长室找到他
- 下雨天他也在学校

**升级建议**：
- 每次见面携带"顾问官"Arcana的Persona（加好感度）
- 早期就开始发展（5月即可开始）
- 不要等到最后一个月才集中提升
- 11月初确认Rank，如果未达标立即优先处理

**错过后的补救**：无法补救。如果不满足条件，第三学期直接无法进入，只能读档或二周目。

## Akechi Coop 与结局 / Akechi Confidant and Endings

明智吾郎的Coop虽然不直接影响第三学期的解锁，但影响他的个人结局：

- **Akechi Coop Rank 8以下**：在第三学期中，明智不会以真身份出现在故事中
- **Akechi Coop Rank 8及以上**：明智会在第三学期中以特定方式回归，并参与最终Boss战
- **Akechi Coop Rank 10**：在最终结局后，明智有一个特定的结局后彩蛋

**建议**：如果要追求完整的真结局体验，推荐将Akechi Coop提升到Rank 8以上。

## Kasumi Coop 与结局 / Kasumi Confidant and Endings

芳泽霞的Coop对结局的影响：

- **Kasumi Coop Rank 5以下**：在第三学期中芳泽霞的故事线不完整
- **Kasumi Coop Rank 5及以上**：触发第三学期的关键剧情
- **Kasumi Coop Max**：解锁她的Persona最终进化，获得核心技能

**建议**：在第三学期前尽量将芳泽霞的Coop提升到Rank 5。

## 最终Boss战 / Final Boss Battle

### 丸喜拓人 Boss战

**第一阶段：丸喜拓人**
- 属性弱点：无固定弱点，会随战斗阶段改变
- 攻击方式：使用全能属性攻击"意志之力"
- 特殊机制：回合中会随机强化一个属性

**第二阶段：亚当·卡德蒙（初始形态）**
- 攻击方式：全属性覆盖 + 状态异常技能
- 特殊机制：会召唤"认知"小兵
- 弱点：Boss使用过的属性中与其颜色相反的属性

**第三阶段：亚当·卡德蒙（最终形态）**
- 战斗模式：分为三个阶段子阶段
- 特殊机制：强制更换队员配置
- 核心攻击：Megidolaon级全能属性群攻

### 最终Boss攻略要点

| 准备事项 | 详细说明 |
|---|---|
| 推荐等级 | 全员75级以上，Joker推荐80级以上 |
| 推荐Persona | 全能耐性Persona（如Lucifer）、回复型Persona（如Kohryu） |
| 关键技能 | Debilitate、Heat Riser、Mediarahan、Amrita Shower |
| 道具准备 | SP完全回复品×10、万能药×10、复活药×5 |
| 装备建议 | Divine Pillar（伤害减半）、SP Adhesive 5 |
| 队员推荐 | Makoto（回复+核热）、Morgana（回复+风）、Kasumi（输出） |

### Boss战阶段详解

**Phase 1**：丸喜单人
- 他的攻击以魔法为主，使用Magarudyne、Maragidyne等大范围魔法
- 建议使用Makoto的Marakukaja提升全队防御
- 使用Debilitate降低Boss能力
- 防守为主，逼出他的真身

**Phase 2**：亚当·卡德蒙
- 颜色变换机制：Boss会每2回合切换一次属性状态
- 红色状态：使用火属性攻击，弱点是冰
- 蓝色状态：使用冰属性攻击，弱点是火
- 黄色状态：使用电属性攻击，弱点是风
- 绿色状态：使用风属性攻击，弱点是电
- 使用扫描确认当前弱点，针对性攻击

**Phase 3**：最终形态
- 分为三个子阶段，每次半血后转换
- 第一子阶段：全能属性攻击为主
- 第二子阶段：全能+物理混合攻击
- 第三子阶段：大范围全能攻击+召唤小兵
- 在第三子阶段需要快速清理小兵并集中攻击
- 使用Showtime技能可以造成大量伤害
- 最后阶段可以连续使用Showtime收尾

### 结局后要素 / Post-Game Content

完成真结局后（进入New Game+），您可以：

- **继承内容**：Persona全书、金钱、装备、社会属性、Coop好感度（部分）
- **新内容**：二周目新增对话选项、隐藏Boss挑战
- **挑战Boss**：双子/拉雯妲的复仇战、特别战斗挑战
- **全Persona收集**：可以开始收集所有Persona以解锁奖杯/成就
- **最强装备**：通过合成获得游戏中最强的装备
