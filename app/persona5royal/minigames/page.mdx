# 小游戏 / Minigames

《女神异闻录5 皇家版》中有丰富的小游戏可供游玩。这些小游戏不仅仅是休闲娱乐，还直接影响战斗能力的提升——例如飞镖提升Baton Pass等级，台球提升Technical伤害。本章节详细介绍所有小游戏的玩法、策略和奖励。

Persona 5 Royal features a variety of minigames that are more than just entertainment — darts improve Baton Pass ranks and billiards boost Technical damage. This chapter covers all minigames, strategies, and rewards.

## 棒球打击练习 / Batting Cages

**地点**：四轩茶屋 Yongen-Jaya（后巷）
**解锁时间**：6月后某天，与Ryuji对话后解锁
**奖励**：提升Charm（魅力）、获得金钱和道具

### 玩法说明 / How to Play

打击练习的目标是击中来球。球速有不同档次，从低速到高速。击球时机的把握是关键。

The goal is to hit incoming balls. Speed ranges from slow to fast. Timing is crucial.

| 难度 | 球速 | 击球提示 |
|---|---|---|
| Beginner 初级 | 80km/h | 球到击球区中央时挥棒 |
| Intermediate 中级 | 100km/h | 提前约0.3秒挥棒 |
| Advanced 高级 | 120km/h | 球投出后立即预判挥棒 |
| Expert 专家 | 140km/h | 需要预判球路 |

### 打击技巧 / Hitting Tips

- **观察投球姿势**：投手准备投球时可以大致判断球速
- **挥棒时机**：低速球等球进入打击区再挥棒，高速球需要在球到达前挥棒
- **连续击中奖励**：连续击中3球以上可获得额外奖金
- **全垒打**：击中球的白色中心区域可以打出全垒打，获得最高分

### 奖励一览 / Rewards

| 成绩 | 奖金 | 额外奖励 |
|---|---|---|
| 5球中0球 | 0 | 无 |
| 5球中1球 | 1000 | 无 |
| 5球中2球 | 2000 | 无 |
| 5球中3球 | 3000 | SP饮料 |
| 5球中4球 | 5000 | 技能卡 |
| 5球中5球 | 10000 | 稀有技能卡 |
| 全垒打 | 额外+5000 | 特别称号 |

### 升级策略 / Progression Tips

- 先从初级难度开始练习掌握节奏
- 连续击中3球后建议继续挑战同难度
- 全垒打需要精确瞄准球的白色区域
- 提高Charm可以选择在打击前后与Ryuji互动

## 飞镖 / Darts

**地点**：吉祥寺 Kichijoji（酒吧区）
**解锁时间**：7月后（吉祥寺开放后）
**奖励**：提升Baton Pass等级（最重要的小游戏）

### 玩法说明 / How to Play

飞镖游戏是皇家版中最重要的功能性小游戏。通过飞镖提升与队友的Baton Pass等级，可以极大增强战斗连携能力。

Darts is the most important functional minigame in Royal. Playing darts with teammates raises your Baton Pass rank, significantly enhancing combat synergy.

### 飞镖规则 / Dart Rules

游戏使用标准的501规则（从501分开始扣除，精确归零者胜）：

The game uses standard 501 rules (start at 501 points, first to exactly zero wins):

- **牛眼 Bullseye**：50分（红色内圈），25分（绿色外圈）
- **内环 Triple Ring**：基础分数的3倍
- **外环 Double Ring**：基础分数的2倍
- **单倍 Single**：基础分数

### 瞄准策略 / Aiming Strategy

| 目标区域 | 分数 | 技巧 |
|---|---|---|
| Triple 20 | 60分 | 最稳定的大分区域 |
| Triple 19 | 57分 | 备选高分目标 |
| Triple 18 | 54分 | 当20和19被挡住时 |
| Bullseye | 50分 | 适合收尾时使用 |
| Double 20 | 40分 | 收尾常用 |

### BP等级提升表 / Baton Pass Rank Requirements

与每个队员分别提升Baton Pass等级：

| 等级 | 需要的飞镖游戏次数 | 效果 |
|---|---|---|
| Rank 0 → 1 | 1次 | 开启Baton Pass功能 |
| Rank 1 → 2 | 2次（赢1次） | 伤害1.5倍，SP回复 |
| Rank 2 → 3 | 3次（赢2次） | 伤害1.8倍，可连续换手 |

### 推荐提升顺序 / Recommended Order

1. **Morgana**：作为队伍主要回复角色，BP等级影响早期战斗
2. **Ryuji**：物理主输出，高BP等级收益巨大
3. **Makoto**：全能型角色，提升BP等级性价比高
4. **Ann**：魔法输出核心，高BP增强魔法连携
5. **Yusuke**：冰属性主力
6. **Haru**：念动+物理

## 台球 / Billiards

**地点**：吉祥寺 Kichijoji（台球厅）
**解锁时间**：7月后（吉祥寺开放）
**奖励**：提升Technical攻击伤害（皇家版新内容）

### 玩法说明 / How to Play

台球小游戏提升的是Technical技术攻击的效果。Technical是皇家版新增的战斗机制，对处于异常状态的敌人造成额外属性伤害。

Billiards improves Technical attack damage, a new Royal mechanic that deals bonus elemental damage to enemies with status ailments.

### 台球规则 / Billiards Rules

台球分为两种模式：

- **9球 9-Ball**：按数字顺序将所有球打入袋中。适合快速练习
- **斯诺克打点 Snooker Challenge**：击球到指定位置，难度较高

### 技巧提示 / Billiards Tips

| 技巧 | 说明 |
|---|---|
| 力度控制 | 轻力度适合近距离击球，重力度适合长距离 |
| 角度调整 | 使用旋转可以改变母球走位 |
| 瞄准 | 目标球和袋口之间需要精确计算角度 |
| 顺序 | 在9球中必须从最小数字开始击球 |

### Technical升级效果 / Technical Upgrade Effects

| 升级次数 | 效果 |
|---|---|
| 1次 | Technical伤害+20% |
| 2次 | Technical伤害+40% |
| 3次 | Technical伤害+60% |
| 4次 | Technical必定击倒 |
| Max | Technical伤害翻倍+SP消耗减少 |

### 推荐练习频率

台球不需要像飞镖那样频繁练习。每个月玩1-2次即可逐步提升Technical到最高等级。

## 钓鱼 / Fishing

**地点**：四轩茶屋 神社 Pond at Shrine
**解锁时间**：获得钓竿后（可以从涉谷购买或从特定Coop获得）
**奖励**：提升Proficiency（灵巧度）、获得稀有道具

### 钓鱼玩法 / Fishing Mechanics

钓鱼需要在正确时机收竿。当浮标下沉时按下按钮即可。

Fishing requires timing. Press the button when the bobber sinks.

| 鱼种 | 出现条件 | 出售价格 |
|---|---|---|
| 鲫鱼 Crucian Carp | 任何时间，容易 | 200 |
| 鲤鱼 Carp | 任何时间，中等 | 500 |
| 黑鱼 Black Bass | 任何时间，中等 | 800 |
| 鲶鱼 Catfish | 雨天限定 | 1500 |
| 虹鳟 Rainbow Trout | 白天限定 | 1000 |
| 巨型鲶鱼 Giant Catfish | 特殊事件 | 5000 |

### 钓鱼技巧 / Fishing Tips

- 浮标完全沉入水中的瞬间收竿成功率最高
- 大型鱼需要反复收线（连打按钮）
- 不同鱼饵可以吸引不同种类的鱼
- 雨天适合钓鲶鱼
- 清晨和傍晚的钓鱼效果最好

### 钓鱼奖励 / Fishing Rewards

| 累计钓数 | 奖励 |
|---|---|
| 10条 | 鱼饵×10 |
| 20条 | 灵巧度提升书 |
| 30条 | 稀有鱼竿 |
| 50条 | 技能卡（水属性） |

### 吃法攻略 / Eating Strategy

速吃汉堡的技巧：

1. 把汉堡压扁再吃可以更快
2. 先将所有汉堡准备好再开始
3. 饮料可以帮助下咽（但会消耗时间）
4. 不要咬太大口以免噎住

**高分技巧**：
- 使用触摸板/鼠标快速点击
- 保持稳定节奏
- 挑战前先提升Guts等级
- 多次尝试熟悉节奏
