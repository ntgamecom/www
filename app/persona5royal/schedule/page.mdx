# 日程管理 / Schedule Guide

《女神异闻录5 皇家版》的时间管理系统要求玩家在有限的时间内做出最优选择。本章节提供全年的日程参考、活动日历和效率最大化的策略。

P5R\'s time management system requires optimal choices within limited time. This chapter provides a full calendar reference, event schedule, and efficiency-maximizing strategies.

## 时间系统概述 / Time System Overview

游戏时间从4月开始到次年3月结束。每天分为两个行动时间段，部分特殊日期只有一个时间段或无法行动。

The game runs from April to March of the following year. Each day has two action segments, with some special days having only one segment or no action at all.

### 时间段详解 / Time Segments

| 时间段 | 可用时间 | 可执行活动 |
|---|---|---|
| 上课日 白天 School Daytime | 放学后约14:00-18:00 | Coop活动、逛街、殿堂攻略 |
| 上课日 傍晚 School Evening | 约18:00-22:00 | 勒布朗活动、读书、制作道具 |
| 假日 白天 Holiday Daytime | 全天 | 同上（更多时间） |
| 假日 傍晚 Holiday Evening | 同上 | 同上 |
| 考试日 Exam Day | 仅傍晚 | 只能进行晚上活动 |
| 强制剧情日 Story Day | 无自由行动 | 推进主线 |
| 雨天 Rainy Day | 白天/傍晚 | 某些活动不可用（如殿堂） |

## 效率最大化策略 / Efficiency Maximization Strategy

### 一周目推荐策略 / Recommended 1st Playthrough Strategy

**4月-5月**：
- 优先推进Kamoshida和Madarame殿堂
- 开始发展Sojiro和Takemi的Coop
- 尽量在图书馆学习提升Knowledge到Rank 2

**6月-7月**：
- 解锁Chihaya（命运）Coop并使用好感度占卜节约时间
- 推进Kaneshiro和Futaba殿堂
- 开始川上（节制）Coop以获得夜间自由行动

**8月暑假**：
- 集中推进多个Coop关系
- 利用Chihaya占卜最大化每次Coop效率
- 完成Futaba殿堂

**9月-10月**：
- 重点推进Okumura殿堂
- 确保Maruki Coop在持续推进

**11月（最关键的月份）**：
- 11月18日前：Maruki Rank 9（绝对优先）
- 确保Akechi达到Rank 8
- 确保Kasumi达到Rank 5

**12月**：
- 完成Shido殿堂
- 完成所有未完成的Coop
- 准备第三学期

### Max Coop一周目清单 / Max Confidants Checklist

一周目可以达成全Coop Max，但需要严格的时间管理：

1. 每次出门携带对应Arcana的Persona（好感度1.2倍）
2. 使用Chihaya的好感度占卜
3. 优先发展有Coop加速效果的角色（川上、千早）
4. 雨天在图书馆提升Knowledge，晴天外出活动
5. 不要在非必要的活动上浪费时间
6. 利用神社祈福提升特定角色的好感度
