## 船只升级路线 / Ship Upgrade Path

| 等级 / Level | 推荐船型 / Recommended Ship | 如何获得 / How to Get |
|:----:|:----|:----|
| **1-3** | 单桅帆船 / Dhow | 完成序章自动获得 / Automatic from tutorial |
| **3-6** | 双桅帆船 / Brig | 恶名II解锁图纸→收集材料→在港口造船 / Infamy II unlock blueprint → gather mats → build at port |
| **6-8** | 双桅战船或纵帆船 / Brigantine or Schooner | 恶名IV解锁图纸→需要精炼木材+金属锭 / Infamy IV unlock → needs refined wood + metal ingots |
| **8-10** | 护卫舰 / Frigate | 恶名VI解锁图纸→需要大量高级材料 / Infamy VI unlock → needs many high-tier mats |
| **10** | 战列舰 / Ship of the Line | 恶名X解锁图纸→需要传说材料+大量银币 / Infamy X unlock → legendary mats + massive silver |

**升级建议**: 新手阶段不要跳过双桅帆船直接造战船——资源消耗大且驾驶难度高 / **Advice**: Don't skip from Dhow directly to Brigantine — resource cost is high and ship is hard to handle.
