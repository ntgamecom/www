## 任务系统概述 / Quest System Overview

碧海黑帆的任务分为**主线任务**（推进剧情）、**支线契约**（从各个港口接取的重复/非重复任务）、**世界事件**（定时刷新的全局活动）和**公会任务**（玩家公会专属）/ Skull and Bones quests are divided into **Main Quests** (story progression), **Side Contracts** (repeatable/non-repeatable from ports), **World Events** (timed global activities), and **Guild Missions** (player guild exclusive).

### 任务类型 / Quest Types

| 类型 / Type | 图标 / Icon | 可重复 / Repeatable | 推荐 / Recommended |
|---|---|---|---|
| 主线任务 / Main Story | 金色卷轴 / Gold Scroll | 否 / No | 优先完成——解锁新区域和功能 / Priority — unlock new areas and features |
| 悬赏 / Bounty | 骷髅头 / Skull | 是 / Yes | 刷金币和经验的最佳方式 / Best for gold + XP |
| 商队护卫 / Convoy Escort | 盾牌 / Shield | 是 / Yes | 安全——适合低等级 / Safe — good for low levels |
| 讨伐 / Hunt | 交叉刀 / Crossed Swords | 是 / Yes | 战斗经验+掉落 / Combat XP + loot |
| 采集 / Gathering | 斧头 / Axe | 是 / Yes | 获得制作材料 / Get crafting materials |
| 公会任务 / Guild Mission | 公会标志 / Guild Crest | 每天 / Daily | 公会声望 + 专属奖励 / Guild rep + exclusive rewards |

## 支线契约 / Side Contracts

### 悬赏任务 / Bounty Contracts

悬赏任务是最稳定的金币来源——打等级相似的敌人事半功倍 / Bounties are the most stable gold source — fight enemies at your level for best efficiency.

| 悬赏目标 / Bounty Target | 等级 / Level | 推荐船等级 / Rec Ship Level | 基础奖金 / Base Bounty |
|---|---|---|---|
| 小型海盗船 / Small Pirate Ship | 3-5 | 2+ | 1,500金 / 1,500 gold |
| 中型商船 / Medium Merchant | 5-8 | 4+ | 3,000金 / 3,000 gold |
| 海军护卫舰 / Navy Frigate | 8-10 | 7+ | 6,000金 / 6,000 gold |
| 精英海盗 / Elite Pirate | 10-12 | 10+ | 12,000金 / 12,000 gold |
| 传奇目标 / Legendary Target | 13-15 | 12+ | 25,000金 / 25,000 gold |

**悬赏技巧 / Bounty Tips**：
- 同一个港口可以接多个悬赏——省去来回跑路的时间 / Take multiple bounties from one port — save travel time
- 优先接"通缉海盗"——因为其他玩家也可能在打同一个目标——可以蹭 / Priority: "Wanted Pirates" — other players may already be fighting them — free assist
- 悬赏目标的掉落物可以卖给发布任务的港口获得额外奖金 / Loot from bounty targets can be sold at the issuing port for bonus gold

### 商队护卫 / Convoy Escort

**任务流程 / Quest Flow**：
1. 在港口接护卫任务 / Accept at port
2. 跟随商船前往目标港口 / Follow merchant ship to destination
3. 保护它免受海盗攻击 / Protect from pirate attacks
4. 到达后领取奖励 / Collect reward on arrival

**收益分析 / Profit Analysis**：

| 时长 / Duration | 基础金 / Base Gold | 额外掉落 / Extra Loot | 每小时收益 / Per Hour |
|---|---|---|---|
| 15分钟 / 15min | 2,000 | 随机材料 / Random materials | ~8,000 |
| 30分钟(长途) / 30min (long) | 5,000 | 稀有材料 / Rare materials | ~10,000 |

**护卫技巧 / Escort Tips**：
- 走在商船前面——不是后面——海盗从前面来 / Sail ahead of merchant — not behind — pirates come from the front
- 提前在航线上放水雷——海盗会走直线追商船 / Pre-place mines on route — pirates chase in straight line
- 如果商船血量低于30%——使用烟雾弹让海盗失去目标 / If merchant HP below 30% — use smoke screen

### 公会战 / Guild Wars

公会战每周开启一次——两个公会争夺特定海域的控制权 / Guild Wars start once a week — two guilds compete for control of specific sea zones.

**公会战流程 / Guild War Flow**：
1. 宣战：公会会长向敌对公会宣战 / Declaration: Guild Leader declares war on rival guild
2. 争夺：双方在争议海域击杀对方船只获得积分 / Skirmish: Kill enemy guild ships for points
3. 占领：积分达到阈值后——占领该海域的中立港口 / Occupy: Reach point threshold — capture neutral port in zone
4. 奖励：占领海域的公会获得该海域的税收（持续一周）/ Reward: Occupying guild gets zone tax (lasts 1 week)

**公会战奖励 / Guild War Rewards**：
- 胜利公会：海域税收的50% + 专属公会涂装 / Winning: 50% zone tax + exclusive guild paint
- 参与奖励：无论胜负——参与会员获得公会币 / Participation: guild coins for all participants

## 收益最大化 / Profit Maximization

### 金币获取路线 / Gold Acquisition Route

**新手阶段(1-10级) / New Player (Lv1-10)**：
```
1. 采集任务——收集铁木 + 帆布 / Gathering — Ironwood + Canvas
2. 在港口出售多余的采集材料 / Sell extra materials at port
3. 完成海上悬赏(小型海盗船) / Complete sea bounties (small pirates)
4. 用金币升级船体到2级 / Upgrade hull to level 2
```

**中期阶段(11-15级) / Mid Game (Lv11-15)**：
```
1. 护卫商船任务——稳定金币/ Escort merchants — stable gold
2. 讨伐虎鲨和乌贼——出售材料 / Hunt tiger sharks and squid — sell materials
3. 每天参加一次世界事件 / Join one world event per day
4. 投资建造第二艘船(商船) / Invest in second ship (merchant)
```

**后期阶段(16-20级) / Late Game (Lv16-20)**：
```
1. 刷传奇悬赏——每趟25,000+ / Farm legendary bounties — 25,000+ per run
2. 公会战——每次胜利几百万金币 / Guild Wars — millions per victory
3. 深海寻宝——高级宝藏价值极高 / Deep sea treasure hunting — high value
4. 倒卖禁运品——低价买入高价卖出 / Contraband trading — buy low sell high
```
