## 技能系统概述 / Skill System Overview

碧海黑帆的技能系统分为**船长技能**（角色天赋）、**船只技能**（船体特长）和**船员技能**（被动加成）。三者结合决定你的战斗力 / Skull and Bones' skill system is divided into **Captain Skills** (character talents), **Ship Abilities** (hull perks), and **Crew Skills** (passive bonuses). Together they determine your combat effectiveness.

### 技能分类 / Skill Categories

| 技能类型 / Type | 来源 / Source | 效果 / Effect | 升级方式 / Upgrade |
|---|---|---|---|
| 船长技能 / Captain Skills | 角色升级 / Character level | 战斗/采集/制造成长 / Combat/harvest/crafting | 使用金币和材料升级 / Gold + materials |
| 船只特长 / Ship Perks | 船体类型 / Hull type | 特定武器/移动加成 / Specific weapon/movement bonuses | 改造船坞 / Shipyard upgrades |
| 船员特长 / Crew Perks | 招募船员 / Crew recruitment | 被动增益——如视野/移速 / Passive buffs — vision, speed | 任务解锁 / Quest unlocks |
| 武器专精 / Weapon Spec | 使用特定武器 / Weapon usage | 提升该武器伤害和装填 / Damage + reload for that weapon | 持续使用升级 / Usage-based |

## Build推荐 / Build Recommendations

### PvE炮击流 / PvE Cannon Build

```
船体: 火力船 / Bombardier
主武器: 迫击炮 x2 / Mortar x2
副武器: 长管炮 x2 / Long Gun x2
船长技能: 炮术 > 爆炸专家 > 快速装填
船员: 炮手 + 工程兵
主动技能: 锁定射击 + 紧急维修
```

**点评 / Review**：标准PvE配装——远程用迫击炮清理港口——近战换长管炮输出 / Standard PvE — mortar clear harbors from range — Long Gun in mid range.

### PvP决斗流 / PvP Duel Build

```
船体: 侦察船 / Scout Ship
主武器: 长管炮 x3 / Long Gun x3
副武器: 鱼雷 x1 / Torpedo x1
船长技能: 炮术 > 快速装填 > 精准射击
船员: 航海长 + 炮手
主动技能: 加速冲刺 + 烟雾弹
```

**点评 / Review**：放风筝打——利用速度优势保持距离——长管炮打爆对手 / Kite build — use speed to maintain range — Long Gun wears them down.

### 冲锋肉搏流 / Ram & Brawl Build

```
船体: 冲锋船 / Ram Ship
主武器: 重炮 x2 / Heavy Cannon x2
副武器: 火焰喷射器 x2 / Flamethrower x2
船长技能: 炮术 > 快速装填 > 爆炸专家
船员: 工程兵 + 医生
主动技能: 加速冲刺 + 紧急维修
```

**点评 / Review**：贴近敌人——重炮+冲锋接火焰喷射——一波带走 / Get close — Heavy Cannon + Ram + Flamethrower — delete enemy.
