## Boss战概述 / Boss Overview

碧海黑帆的Boss分为三类：**世界Boss**（定时刷新的超强单体敌舰）、**精英舰队**（3-5艘船的护卫队）和**海怪**（巨大的海洋生物）。每种Boss都有独特的机制和弱點 / Skull and Bones bosses are divided into three types: **World Bosses** (timed spawn super-strong single ships), **Elite Fleets** (3-5 ship convoys), and **Sea Monsters** (giant oceanic creatures). Each has unique mechanics and weaknesses.

| 船员 / Crew | 全部最佳船员配置 / Full optimal crew setup |

## 世界Boss / World Bosses

### 幽灵船长 / The Ghost Captain

**出现位置 / Location**：红岛海域——每晚22:00（游戏时间）刷新 / Red Island waters — spawns 22:00 (in-game time) daily.

**Boss信息 / Boss Info**：
| 属性 / Stat | 数值 / Value |
|---|---|
| 等级 / Level | 12 |
| 船体 / Hull | 幽灵帆船——半透明——炮击命中率-20% / Ghost Galleon — translucent — -20% cannon accuracy |
| 武器 / Weapons | 幽灵炮——穿甲伤害 / Ghost Cannons — AP damage |
| 特殊能力 / Special | 幽灵形态——每30秒变成半透明——闪避+50% / Ghost Form — translucent every 30s — +50% dodge |

**击杀策略 / Kill Strategy**：

**阶段一：打破幽灵形态 / Phase 1: Break Ghost Form**
1. 使用火焰武器——火伤可以让幽灵形态失效 / Use fire weapons — fire damage cancels ghost form
2. 火箭炮是最佳选择——持续点火 / Rockets are best — sustained fire
3. 在幽灵形态期间不要浪费弹药——等形态结束再爆发 / Don't waste ammo during ghost form — wait for it to end

**阶段二：全力输出 / Phase 2: Full DPS**
1. 幽灵形态被打破后——他有10秒的暴露窗口 / After ghost form broken — 10s vulnerability window
2. 使用加农炮/重炮近距离输出 / Use Culverin/Heavy Cannon at close range
3. 打船尾和船头——船体中间护甲最厚 / Target stern and bow — mid-hull has thickest armor

**阶段三：逃跑阶段 / Phase 3: Escape Phase**
1. 血量低于20%时——幽灵船长会试图逃跑 / Below 20% HP — Ghost Captain tries to flee
2. 用鱼雷封锁他的逃跑路线 / Use torpedoes to block escape route
3. 加速冲刺追上去——撞停他 / Speed boost to ram — stop him

**掉落 / Loot**：
- 幽灵帆船蓝图(30%) / Ghost Galleon Blueprint (30%)
- 幽灵炮图纸(25%) / Ghost Cannon Schematic (25%)
- 黑珍珠(100%) / Black Pearl (100%)

### 海魔王戴维·琼斯 / Davy Jones

**出现位置 / Location**：魔鬼三角海域——天气恶劣时出现 / Devil's Triangle — appears during bad weather.

**Boss信息 / Boss Info**：
| 属性 / Stat | 数值 / Value |
|---|---|
| 等级 / Level | 15 |
| 船体 / Hull | 被诅咒的巨型盖伦——血量极高 / Cursed Mega-Galleon — extremely high HP |
| 武器 / Weapons | 电击炮——可瘫痪你的船只/  Lightning Cannon — can disable your ship |
| 特殊能力 / Special | 召唤风暴——全屏减速+降低视野 / Summon Storm — full map slow + vision reduction |

**击杀策略 / Kill Strategy**：

**唯一推荐**：组队击杀——4人/ Solo 需要满级+完美装备 / Group kill recommended — 4 players / Solo needs max level + perfect gear.

**阶段一：处理风暴 / Phase 1: Handle the Storm**
1. 风暴会降低你的移动速度和炮击精度 / Storm reduces movement speed and cannon accuracy
2. 使用导航员的"精准航道"技能减少风暴影响 / Use Navigator's "Precision Course" to reduce storm effect
3. 保持距离——风暴中心附近的大浪会砸船 / Keep distance — large waves near storm center damage ships

**阶段二：打断电击炮 / Phase 2: Disrupt Lightning Cannon**
1. 电击炮蓄力时有明显的蓝色光芒 / Lightning Cannon has visible blue glow when charging
2. 在蓄力期间重击船头——可以打断施法 / Heavy hit during charge — interrupts the cast
3. 被电击炮打中后——快速使用修理包 / After getting hit — use repair kit immediately

**阶段三：集火收尾 / Phase 3: Focus Fire**
1. 戴维·琼斯血量低于30%时会进入狂暴模式——伤害翻倍 / Below 30% HP — enrages — damage doubles
2. 全员使用爆发技能——争取30秒内击杀 / All use burst skills — kill within 30s
3. 工程兵技能全员开启——没有修理的容错 / Engineer skills all on — no error margin

**掉落 / Loot**：
- 戴维·琼斯的诅咒之剑(武器皮肤)(15%) / Davy Jones' Cursed Sword (weapon skin) (15%)
- 电击炮蓝图(20%) / Lightning Cannon Schematic (20%)
- 传奇航海图(10%) / Legendary Sea Chart (10%)

## 精英舰队 / Elite Fleets

### 皇家海军舰队 / Royal Navy Fleet

**出现位置 / Location**：非洲西海岸——东印度群岛通商路线 / West African coast — East Indies trade routes.

**编队组成 / Fleet Composition**：
| 船只 / Ship | 数量 / Count | 角色 / Role |
|---|---|---|
| 护卫舰 / Frigate | 2 | 主力输出——重炮 / Main DPS — heavy cannon |
| 侦察舰 / Scout | 1 | 视野+呼叫支援 / Vision + call reinforcements |
| 旗舰 / Flagship | 1 | 指挥官——领导力加成 / Commander — leadership buff |

**击杀策略 / Kill Strategy**：
1. 先打侦察舰——防止它呼叫更多的舰队增援 / Kill Scout first — prevents reinforcement calls
2. 然后集火旗舰——旗舰沉没后其他船会恐慌 / Then focus Flagship — other ships panic when flagship sinks
3. 最后处理两艘护卫舰——逐个击破 / Finally handle the 2 Frigates — one at a time
4. 使用鱼雷封锁护卫舰的逃跑路线 / Torpedo block Frigate escape routes

**掉落 / Loot**：
- 海军炮蓝图 / Naval Cannon Blueprint
- 皇家帆布(时装材料) / Royal Canvas (cosmetic material)
- 大量金币和经验 / Large gold + XP

### 海盗公会舰队 / Pirate Guild Fleet

**出现位置 / Location**：加勒比海域——海盗公会的巡逻路线 / Caribbean waters — Pirate Guild patrol routes.

**编队组成 / Fleet Composition**：
| 船只 / Ship | 数量 / Count | 角色 / Role |
|---|---|---|
| 战斗船 / Battle Ship | 2 | 冲锋+近战 / Ram + brawl |
| 火力船 / Bombardier | 1 | 远程迫击炮 / Long-range mortar |
| 旗舰 / Flagship | 1 | 疾速——走位灵活 / Speed — agile positioning |

**击杀策略 / Kill Strategy**：
1. 远程风筝打死火力船——迫击炮对你的威胁最大 / Kite and kill Bombardier from range — mortar is the biggest threat
2. 接着用冲锋撞停战斗船——不要和他们拼角力 / Ram to stop Battle Ships — don't trade blows
3. 旗舰速度太快——使用水雷封锁它的移动路径 / Flagship is too fast — use mines to block movement
4. 减速后集火旗舰 / Focus Flagship after slowing it down

**掉落 / Loot**：
- 公会战舰涂装(稀有) / Guild Warship Paint (rare)
- 禁运品(高价值售卖品) / Contraband (high-value sell item)
- 海盗声望+500 / +500 Pirate Reputation

## 海怪 / Sea Monsters

### 克拉肯 / Kraken

**出现位置 / Location**：深海区域——概率刷新 / Deep ocean — random spawn.

**Boss信息 / Boss Info**：
| 属性 / Stat | 数值 / Value |
|---|---|
| 等级 / Level | 18(全游戏最强) / 18 (strongest in game) |
| 血量 / HP | 极高 / Extreme |
| 攻击方式 / Attacks | 触手拍击(近战)、水炮(远程)、漩涡牵拉(控制) / Tentacle slam, water cannon, vortex pull |

**击杀策略 / Kill Strategy**：

**阶段一：触手阶段 / Phase 1: Tentacles**
1. 克拉肯周围有4条可攻击的触手——打断它们可以减少其攻击频率 / 4 attackable tentacles — breaking them reduces attack frequency
2. 优先打断远程水炮的触手 / Priority: break the water cannon tentacle
3. 触手被破坏后过60秒会再生 / Tentacles regenerate after 60s

**阶段二：主体阶段 / Phase 2: Main Body**
1. 所有触手都被打断后——克拉肯露出头部弱点 / All tentacles broken — Kraken head weak point exposed
2. 全员集火头部——使用重炮和迫击炮 / Team focus head — use Heavy Cannon and Mortar
3. 注意克拉肯会释放漩涡——被吸入后持续掉血——加速冲刺可以挣脱 / Attention: Kraken's vortex — continuous damage — Speed Boost to escape

**阶段三：狂暴阶段 / Phase 3: Enrage**
1. 血量低于20%——克拉肯进入狂暴——攻击速度翻倍 / Below 20% HP — enrages — double attack speed
2. 此时不要贪刀——确保血量安全再输出 / Don't greed — keep HP safe before DPSing
3. 如果团队血量不健康——轮流拉仇恨——其他人修理 / Rotate aggro if team HP is low — others repair

**掉落 / Loot**：
- 克拉肯之眼(传奇饰品) / Kraken's Eye (Legendary accessory)
- 深海宝藏(价值500,000金币) / Deep Sea Treasure (500,000 gold value)
- 传说级船体材料(100%) / Legendary Hull Material (100%)

### 利维坦 / Leviathan

**出现位置 / Location**：地图边缘——极深处Deep Ocean / Map edge — extreme deep ocean.

**特点 / Trait**：比克拉肯更稀有——但掉落也更好 / Rarer than Kraken — but better loot.

**攻击模式 / Attack Pattern**：
- 冲撞——直线高速冲刺——被撞到基本秒杀 / Charge — linear high-speed rush — basically instakill on hit
- 尾扫——范围AOE——击退/ / Tail sweep — AoE — knockback
- 深海潜水——潜入水下后从你的正下方出现 / Deep dive — submerges then appears directly below you

**击杀方法 / Kill Method**：
1. 永远保持移动——停下=被秒 / Never stop moving — standing still = death
2. 当利维坦潜入水中时——向反方向全速前进 / When it dives — full speed in opposite direction
3. 使用鱼雷——鱼雷在水中的速度比炮弹快——更容易命中 / Use torpedoes — faster in water than cannonballs — easier to hit
4. 需要至少4人组队——单人不可能击杀 / Need at least 4 players — impossible solo

## 团队配合策略 / Team Coordination

### 标准4人队阵容 / Standard 4-Player Team

| 位置 / Position | 船体 / Hull | 职责 / Role |
|---|---|---|
| 主坦 / Main Tank | 战斗船 / Battle Ship | 吸引Boss注意力 / Draw boss aggro |
| 输出 / DPS | 火力船 / Bombardier | 远程主要伤害 / Primary ranged damage |
| 副输出 / Sub-DPS | 侦察船 / Scout Ship | 侧翼攻击+补给 / Flank attack + supply |
| 支援 / Support | 商船(战斗改装) / Merchant (combat mod) | 修理+增益+水雷 / Repair + buff + mines |

### Boss战通用技巧 / General Boss Tips

1. **不要挤在一起**——Boss的范围攻击会全吃 / Don't cluster — boss AoE will hit everyone
2. **修理包不要省**——血量低于50%就用 / Don't hoard repair kits — use below 50% HP
3. **分配好仇恨顺序**——谁被打谁后退——而不是一起后退 / Assign aggro rotation — aggroed player retreats, not everyone
4. **提前看好风向和洋流**——顺风打Boss逃跑快 / Check wind and current — downwind = faster escape
5. **每场Boss战准备至少20分钟的战斗时间**——不要急着打 / Budget at least 20 minutes per boss fight — don't rush
