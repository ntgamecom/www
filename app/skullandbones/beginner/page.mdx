# 新手入门 / Beginner Guide

碧海黑帆是一款以海盗为主题的MMO航海游戏。你将驾驶自己的船只，在印度洋上航行、战斗、贸易。本文档带你从零开始成为合格的海盗船长 / Skull and Bones is a pirate-themed MMO naval game. You'll command your own ship, sailing the Indian Ocean, fighting, trading, and exploring. This guide takes you from zero to a competent pirate captain.

---

## 游戏核心循环 / Core Game Loop

| 阶段 / Phase | 内容 / Activities |
|:----:|------|
| **1. 接取合约 / Take Contracts** | 在港口接取任务——送货/悬赏/探索 / Accept quests at ports — delivery/bounty/exploration |
| **2. 出海航行 / Set Sail** | 根据合约目标前往指定海域 / Navigate to contract objectives |
| **3. 海上遭遇 / Encounters** | 遇敌(玩家/NPC)→战斗或逃跑 / Enemy encounters (player/NPC) — fight or flee |
| **4. 收集战利品 / Collect Loot** | 击败敌船后打捞残骸获取资源 / Salvage wreckage after defeating enemy ships |
| **5. 回港升级 / Return & Upgrade** | 回到港口交任务→获得银币和声望→升级船只/装备 / Return to port → earn silver & infamy → upgrade ship/gear |

## 基础操作 / Basic Controls

| 操作 / Action | 说明 / Description |
|:----:|------|
| **W/S** | 加速/减速 / Increase/Decrease speed |
| **A/D** | 转向 / Steer left/right |
| **Shift** | 爆发加速(消耗体力) / Sprint (consumes stamina) |
| **鼠标左键** | 左侧火炮齐射 / Left broadside fire |
| **鼠标右键** | 右侧火炮齐射 / Right broadside fire |
| **Q/E** | 特殊武器(迫击炮/鱼叉等) / Special weapons (mortar/harpoon etc.) |
| **F** | 互动/登船 / Interact/Board |
| **C** | 望远镜——标记敌人 / Spyglass — mark enemies |
| **M** | 地图 / Map |
| **Tab** | 切换武器类型 / Switch weapon type |

## 新手成长路线 / Beginner Progression Path

| 阶段 / Phase | 目标 / Goal | 推荐行动 / Recommended Actions |
|:----:|:----|:----|
| **1. 新手船阶段 / Starter Ship** (1-3小时) | 熟悉基础操作和战斗 / Learn basics + combat | 完成序章教程→接取送货合约→练习火炮瞄准 / Complete tutorial → delivery contracts → practice cannon aim |
| **2. 升级到中型船 / Medium Ship** (3-8小时) | 赚够银币升级船只 / Earn silver for ship upgrade | 做悬赏合约→收集木材和金属→购买中型船图纸→造船 / Bounty contracts → gather wood+metal → buy blueprint → build |
| **3. 中型船阶段 / Medium Ship** (8-20小时) | 解锁高级武器+参与PVP / Unlock advanced weapons + PvP | 探索新区域→制作迫击炮/鱼叉→参加世界事件→尝试PVP / Explore new zones → craft mortars/harpoons → world events → try PvP |
| **4. 大型船阶段 / Large Ship** (20-40小时) | 打造终极舰队 / Build ultimate fleet | 挑战传奇海盗→制作传说装备→组建舰队→占领据点 / Challenge legendary pirates → legendary gear → form fleet → capture outposts |

## 资源系统 / Resource System

| 资源 / Resource | 获取方式 / Source | 主要用途 / Main Use |
|:----:|:----|:----|
| **银币 / Silver** | 完成任务+出售物品 / Quests + sell items | 购买船只/装备/材料 / Buy ships/gear/materials |
| **木材 / Wood** | 打捞沉船+海上采集 / Salvage + sea gathering | 造船+船体升级 / Shipbuilding + hull upgrades |
| **金属 / Metal** | 打捞+港口购买 / Salvage + port purchase | 武器+护甲制作 / Weapon + armor crafting |
| **布料 / Cloth** | 打捞商船+制作 / Salvage merchant ships + craft | 帆+装饰/ Sail + cosmetics |
| **食物 / Food** | 港口购买+海上采集 / Port purchase + sea gathering | 航行消耗——缺食物会减速 / Sailing consumption — slows without food |
| **朗姆酒 / Rum** | 制作+掠夺 / Craft + plunder | 船员士气+特殊Buff / Crew morale + special buffs |

## 声望系统 / Infamy System

| 等级 / Rank | 解锁内容 / Unlocks |
|:----:|:----|
| **恶名1-3 / Infamy I-III** | 基础船只+武器蓝图 / Basic ship + weapon blueprints |
| **恶名4-6 / Infamy IV-VI** | 中型船+迫击炮+鱼叉 / Medium ships + mortars + harpoons |
| **恶名7-9 / Infamy VII-IX** | 大型船+传说装备图纸 / Large ships + legendary gear blueprints |
| **恶名10 / Infamy X** | 终极船只+最高级装备 / Ultimate ships + top-tier gear |

**快速提升声望**: 做高难度合约(红色骷髅标记)+参与世界事件(海上龙卷风/黄金舰队) / **Fast infamy**: Take high-difficulty contracts (red skull) + participate in world events (sea tornadoes/gold fleets).

## 新手常见错误 / Common Beginner Mistakes

| 错误 / Mistake | 正确做法 / Correct |
|:----:|------|
| **接了超过能力的合约 / Taking hard contracts** | 合约有推荐等级和船型——红色=别碰、黄色=挑战、绿色=安全 / Check recommended level — red=danger, yellow=tough, green=safe |
| **不做修理就出海 / Unsafe sailing** | 出海前在港口修船——船体血量低于50%时航行速度下降30% / Repair at port before sailing — below 50% hull = 30% speed penalty |
| **不升级船体 / Skipping hull upgrades** | 船体等级决定血量和负重——优先升级再考虑武器 / Hull level = HP + carry weight — upgrade hull before weapons |
| **看见PVP标志不跑 / Ignoring PvP flags** | PVP海域的海盗标志=该区域允许PVP——不想打架就远离 / PvP zones have a pirate icon — stay away if not looking for a fight |
| **不带修理工具 / No repair kits** | 战斗前确保带了修理工具——战斗中也可以修复船体 / Always carry repair kits — can repair hull mid-combat |
