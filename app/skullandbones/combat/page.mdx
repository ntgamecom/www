# 海战系统 / Naval Combat

碧海黑帆的海战是游戏核心——强调炮术、机动性和战术决策。了解不同武器类型的用法和战场环境利用是取胜关键 / Naval combat is the core of Skull and Bones — emphasizing cannon accuracy, maneuverability, and tactical decision-making. Understanding weapon types and environmental advantages is key to victory.

---

## 基础战斗机制 / Basic Combat Mechanics

| 机制 / Mechanic | 说明 / Description |
|:----:|------|
| **齐射 / Broadside** | 船只侧面火炮齐射——最大伤害输出方式 / Side cannon volley — maximum damage output |
| **炮门角度 / Firing Arc** | 火炮有固定的射击角度——需要调整船身来瞄准 / Cannons have fixed angles — adjust ship to aim |
| **装弹时间 / Reload** | 每次齐射后需要装弹——约10-15秒 / Reload after each broadside — ~10-15 seconds |
| **弱点攻击 / Weak Point Hits** | 攻击船尾(船舵)=减速、攻击帆(桅杆)=减速、攻击船体=伤害 / Stern hit=slow, Mast hit=slow sails, Hull hit=damage |
| **登船战 / Boarding** | 近距离登船→小游戏→获胜后获得额外战利品 / Close-range boarding → minigame → bonus loot on win |

| 武器 / Weapon | 弹药 / Ammo | 最佳射程 / Range | 效果 / Effect | 适用场景 / Best For |
|:----:|:----:|:----:|:------|:----:|
| **球弹 / Round Shot** | 炮弹 / Cannonball | 中距 / Medium | 基础伤害——均衡 / Base damage — balanced | 通用——所有战斗 / All-purpose |
| **链弹 / Chain Shot** | 链弹 / Chain Shot | 中距 / Medium | 对帆伤害x3——减速效果 / 3x sail damage — slow effect | 打断敌人移动 / Disable enemy movement |
| **葡萄弹 / Grapeshot** | 葡萄弹 / Grapeshot | 近距 / Short | 高伤害+高暴击率——射程短 / High dmg + crit — short range | 近身一击——搭配高机动船 / Close-range finisher |
| **迫击炮 / Mortar** | 迫击炮弹 / Mortar Shell | 超远程 / Extreme | 抛物线射击——可越岛/越船 / Arcing shot — over islands/ships | 远程轰炸——PVE刷怪 / Long-range AoE — PvE farming |
| **鱼叉 / Harpoon** | 鱼叉 / Harpoon | 中距 / Medium | 牵引敌人→拉近距离+眩晕 / Pull enemy → close gap + stun | 控制+配合葡萄弹 / Control + grapeshot setup |
| **火箭 / Rocket** | 火箭弹 / Rocket | 远程 / Long | 高伤害+燃烧DoT / High dmg + burn DoT | 风筝战术 / Kite tactics |
| **水雷 / Mine** | 水雷 / Mine | 近距(部署) / Short | 部署后敌人触爆——范围伤害 / Deploy field — enemy triggers AoE | 逃跑/伏击 / Escape/ambush |
| **修理光束 / Repair Beam** | 无消耗 / No ammo | 近距 / Short | 修理队友船体——冷却30秒 / Repair ally hull — 30s cooldown | 组队辅助角色 / Support build |

## 战斗技巧 / Combat Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **保持移动 / Keep Moving** | 停船=活靶子——永远保持最低航速来转弯 / Stopped = sitting duck — always maintain minimum speed for turning |
| **预判弹道 / Lead Your Shots** | 炮弹出膛有飞行时间——需要预瞄敌人前进方向 / Cannonballs have travel time — aim ahead of enemy course |
| **利用风向 / Use Wind** | 逆风速度慢——保持顺风/侧风航行。风向影响炮弹飞行 / Upwind = slow. Keep downwind/crosswind. Wind affects cannonball trajectory |
| **先打帆再打船体 / Sails First, Hull Second** | 先链弹打爆帆→敌人减速→然后围着转圈→球弹打船体 / Chain shot sails first → slow enemy → circle strafe → round shot hull |
| **不要进包围圈 / Don't Get Surrounded** | 1vN时保持在外围打游击——不要冲进敌群 / 1vN: stay on perimeter — don't charge into the group |
| **利用岛屿掩护 / Use Islands** | 绕岛航行可以阻挡敌人炮线——利用岛屿卡视角 / Islands block enemy cannon fire — use them for line-of-sight cover |

## PVP技巧 / PvP Tips

| 场景 / Scenario | 策略 / Strategy |
|:----:|------|
| **1v1对战 / Duel** | 优先保持侧面对敌——你的齐射角度越大越好。同时尽量不让敌人对你齐射 / Keep broadside aimed at enemy. Minimize their broadside angle on you |
| **组队战 / Group Fight** | 分工明确：1坦克(护卫舰吸引火力)+2输出(战船侧翼齐射)+1辅助(修理+控制) / Roles: 1 tank (Frigate aggro) + 2 DPS (flanking) + 1 support (repair+control) |
| **逃跑战 / Escape** | 丢水雷封路→逆风转顺风→修理船体→全速前进 / Drop mines → turn downwind → repair → full speed |
| **伏击战 / Ambush** | 岛屿背后等目标经过→出来就是贴脸葡萄弹→2轮齐射带走 / Hide behind island → ambush with grapeshot → 2 broadsides kill |
| **追击战 / Pursuit** | 用鱼叉拉近敌人→迫击炮预判前方弹道→链弹减速→追上后葡萄弹 / Harpoon pull → mortar predict → chain shot slow → grapeshot finish |
