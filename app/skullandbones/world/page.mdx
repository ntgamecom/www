# 海域探索 / World Exploration

碧海黑帆的世界以印度洋为舞台——从非洲东海岸到东南亚群岛。每个区域有不同的港口、资源、敌人等级和气候条件 / Skull and Bones' world is set in the Indian Ocean — from East Africa to Southeast Asia. Each region has different ports, resources, enemy levels, and weather conditions.

---

## 航行技巧 / Sailing Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **顺风航行 / Wind Sailing** | 观察旗帜/帆——顺风时速度+30%。学会利用风向可以大幅缩短航行时间 / Observe flags — downwind = +30% speed. Using wind cuts travel time significantly |
| **快速旅行 / Fast Travel** | 已解锁的港口之间可以快速旅行(消耗一些银币) / Unlocked ports can be fast traveled (costs some silver) |
| **危险海域标记 / Danger Zones** | 地图上的骷髅标记=PVP区域。不想打架就绕路 / Skull markers on map = PvP zones. Detour if avoiding PvP |
| **风暴规避 / Storm Avoidance** | 风暴减视野+减速度+持续掉血。地图上风暴云清晰可见——绕过去 / Storms reduce visibility + speed + chip damage. Clearly visible on map — go around |
| **夜间航行 / Night Sailing** | 夜晚能见度降低但商船防御降低——适合偷襲 / Reduced visibility but merchant ships are less defended — good for ambush |
