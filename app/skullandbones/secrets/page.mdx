## 藏宝图系统 / Treasure Map System

碧海黑帆中的藏宝图是最有价值的隐藏要素——每个藏宝图指向一个隐藏的宝藏位置——包含大量金币、稀有材料和独特装备 / Treasure maps in Skull and Bones are the most valuable hidden content — each points to a hidden treasure location containing large gold amounts, rare materials, and unique gear.

| 获取方式 / How to Get | 概率 / Chance | 地图品级 / Map Tier |

| 公会战胜利 / Guild war victory | 20% | 金色 / Gold |

### 已知藏宝图位置 / Known Treasure Locations

**红岛宝藏 / Red Island Treasure**：
1. 地图坐标：C7区域——沉船残骸东侧200米 / C7 zone — 200m east of shipwreck remains
2. 藏宝提示："在红色岩石和棕榈树之间——太阳升起时影子指向的方向" / "Between red rocks and palm tree — direction of shadow at sunrise"
3. 奖励：10,000金 + 红宝石x3 + 独特船帆涂装 / 10,000 gold + Ruby x3 + unique sail paint

**魔鬼三角宝藏 / Devil's Triangle Treasure**：
1. 地图坐标：E12区域——漩涡中心西北侧 / E12 zone — northwest of vortex center
2. 藏宝提示："三道闪电劈开的地方——海怪沉睡之下" / "Where three lightning strikes — beneath the sleeping sea monster"
3. 奖励：25,000金 + 黑珍珠x2 + 传说级火炮 / 25,000 gold + Black Pearl x2 + Legendary Cannon

**东印度宝藏 / East Indies Treasure**：
1. 地图坐标：H5区域——瀑布后面的洞穴 / H5 zone — cave behind waterfall
2. 藏宝提示："水声轰鸣处——走进水帘——宝藏就在暗处" / "Where water roars — walk through the veil — treasure waits in darkness"
3. 奖励：15,000金 + 香料包x5(高价值物品) + 独特罗盘 / 15,000 gold + Spice Pack x5 + unique compass

## 隐藏岛屿 / Hidden Islands

### 无名岛 / Unnamed Island

**位置 / Location**：地图西南角——E4区域——常规航线不会经过 / Southwest corner — E4 zone — off all standard trade routes.

**隐藏内容 / Hidden Content**：
- 岛上有废弃的船坞——可以解锁隐藏船体改造 / Abandoned shipyard — unlocks hidden hull modification
- 山洞内有一具骷髅和一个日记——关于失落的海盗舰队 / Cave has skeleton + journal — about the lost pirate fleet
- 在岛的最高点用望远镜观察——会发现远处另一处隐藏坐标 / Use spyglass at highest point — find another hidden coordinate

**如何发现 / How to Find**：
- 在夜晚靠近E4区域——你会看到岛上发出微弱的磷光 / Approach E4 at night — faint phosphorescent glow from island
- 或者——在圣安妮港的酒馆喝一杯朗姆酒后——喝醉的船长会提到"西南方那个发光的小岛" / Or — drink rum at Sainte-Anne's tavern — drunk captain mentions "the glowing island to the southwest"

### 骷髅礁 / Skull Reef

**位置 / Location**：B10区域——浅海礁石区——船体容易受损 / B10 zone — shallow reef zone — ship hull damage risk.

**隐藏内容 / Hidden Content**：
- 礁石中央的沉船中有——**骷髅船图鉴碎片**——集齐5个建造骷髅船 / Central shipwreck contains — **Skeleton Ship Blueprint Fragments** — collect 5 to build Skeleton Ship
- 暗礁中有隐藏通道——只能用小船通过 / Hidden passage in reef — only passable by small boat
- 最深处有一个水下洞穴——需要潜水装备 / Deepest part has underwater cave — needs diving gear

### 幽灵岛 / Ghost Island

**位置 / Location**：只在特定天气条件下出现（暴风雨中）——随机位置 / Appears only in specific weather (storm) — random location.

**隐藏内容 / Hidden Content**：
- 岛上全是幽灵海盗——击杀他们获得幽灵装备 / Island full of ghost pirates — kill them for ghost gear
- 在岛的中间有一艘完全完好的鬼船——可以夺取 / Center of island has a fully intact ghost ship — capturable
- 鬼船的船长室有唯一物品——**永恒之火**（武器皮肤）/ Captain's quarters contains unique item — **Eternal Flame** (weapon skin)

## 彩蛋 / Easter Eggs

### 黑旗致敬 / Black Flag Tribute

碧海黑帆作为《刺客信条4：黑旗》的精神续作——包含多处对黑旗的致敬 / As a spiritual successor to Assassin's Creed IV: Black Flag — contains multiple tributes.

**爱德华·肯维的帽子 / Edward Kenway's Hat**：在D12区域的一个沉船中——你可以找到爱德华·肯维标志性的三角帽——作为装饰品 / In a shipwreck at D12 zone — find Edward Kenway's iconic tricorn hat — as a cosmetic item.

**"船歌"彩蛋 / "Sea Shanty" Easter Egg**：在特定港口停靠时——有极低概率你的船员会唱起黑旗中的经典船歌《Leave Her Johnny》/ When docking at certain ports — very low chance your crew sings the classic shanty "Leave Her Johnny" from Black Flag.

### 现代海盗文化彩蛋 / Modern Pirate Culture Easter Eggs

**杰克·史派罗 / Jack Sparrow**：在酒馆中——有几率看到一个和杰克船长极为相似的海盗在喝酒——他会说一句："Why is the rum always gone?" / In taverns — chance to see a Jack-Sparrow-like pirate drinking — he says: "Why is the rum always gone?"

**海盗法典 / Pirate Code**：在游戏中的某本书中——写着"海盗法典不是实际的法律——而是...指南"——引用《加勒比海盗》的台词 / A book in game reads: "The Pirate Code is not a legal code — it's more... guidelines" — Pirates of the Caribbean reference.

**"你见过这个蜗牛吗？" / "Have You Seen This Snail?"**：在一个隐藏洞穴的墙上——画着一只蜗牛——旁边写着"SpongeBob's house"——海绵宝宝的海星屋彩蛋 / On hidden cave wall — a painted snail — "SpongeBob's house" written next to it.

### 开发者彩蛋 / Developer Easter Eggs

**开发者房间 / Dev Room**：在游戏地图的最边缘——有一个非常小且难以发现的岛屿——岛上有开发者的名字刻在岩石上 / At very edge of map — a tiny barely-discoverable island — developer names carved on rocks.

**独角兽岛的彩虹 / Rainbow Over Unicorn Island**：在特定命名岛屿"独角兽岛"（坐标J7）——在彩虹出现时——岛上的宝箱会多出双倍奖励 / At "Unicorn Island" (J7) — during rainbow weather — chest rewards are doubled.
