# 世界与探索 / World & Exploration

## 区域总览 / Region Overview

霍格沃茨之遗的地图分为 **6大区域**——总面积约 5平方公里 / 6 major regions — ~5 sq km.

| 区域 / Region | 等级范围 / Level | 特点 / Traits |
|---|---|---|
| **霍格沃茨城堡 / Hogwarts Castle** | 1+ | 巨大的可探索城堡——7层+塔楼+地下 + 大量秘密 / Vast explorable castle — 7 floors + towers + dungeons + secrets |
| **活米村 / Hogsmeade** | 1+ | 巫师村庄——商店+旅店+支线 / Wizarding village — shops + inn + quests |
| **霍格沃茨谷地 / Hogwarts Valley** | 5-20 | 城堡外的初始区域——各种地牢+强盗营地 / Starting area outside castle |
| **禁林 / Forbidden Forest** | 10-25 | 密林+蜘蛛+巨怪——危险但奖励丰厚 / Dense forest + spiders + trolls — dangerous but rewarding |
| **北福特沼泽 / North Ford Bog** | 15-25 | 沼泽地——大量水生物和有毒植物 / Swamp — aquatic enemies + toxic plants |
| **马伦韦姆湖 / Marunweem Lake** | 20-30 | 湖边区域——巨人+海妖 / Lakeside region — giants + sirens |
| **费尔德克罗夫特 / Feldcroft** | 15-25 | 塞巴斯蒂安的家乡——村庄+矿洞 / Sebastian's hometown — village + mine |
| **波德萨尔海岸 / Poidsear Coast** | 25-35 | 海边+悬崖——高级区域 / Coast + cliffs — high-level area |
| **克拉格玛尔海岸 / Cragcroftshire** | 25-35 | 最西部——最难的区域 / Farthest west — hardest region |

## 霍格沃茨城堡详解 / Hogwarts Castle Guide

城堡共有 **7层 + 地下室 + 多个塔楼**——超过100个可探索房间 / 7 floors + basement + towers — 100+ explorable rooms.

| 楼层 / Floor | 关键地点 / Key Locations |
|---|---|
| **地下室 / Dungeon** | 斯莱特林公共休息室、魔药课教室、厨房 / Slytherin common room, Potions class, Kitchens |
| **一楼 / Ground Floor** | 礼堂、大理石楼梯、院长办公室 / Great Hall, Marble Staircase, Headmaster's Office |
| **二楼 / First Floor** | 图书馆、变形课教室、走廊 / Library, Transfiguration, Corridors |
| **三楼 / Second Floor** | 黑魔法防御课、天文塔 / Defence Against the Dark Arts, Astronomy Tower |
| **四楼 / Third Floor** | 草药学教室(通往温室) / Herbology class (to greenhouses) |
| **五楼 / Fourth Floor** | 魔咒课教室、算术占卜 / Charms, Arithmancy |
| **六楼 / Fifth Floor** | 占卜课教室、有求必应屋入口 / Divination, Room of Requirement entrance |
| **塔楼 / Towers** | 格兰芬多塔(+公共休息室)、拉文克劳塔、天文塔 / Gryffindor Tower (+common room), Ravenclaw Tower, Astronomy Tower |

## 梅林试炼 / Merlin Trials

全地图共 **95个梅林试炼**——完成一定数量后扩展装备容量 / 95 Merlin Trials — expand gear inventory capacity.

| 等级 / Tier | 需要完成 / Required | 奖励 / Reward |
|---|---|---|
| I | 2个 / 2 | +4装备容量 / +4 gear slots |
| II | 6个 / 6 | +4装备容量 / +4 gear slots |
| III | 10个 / 10 | +4装备容量 / +4 gear slots |
| IV | 14个 / 14 | +4装备容量 / +4 gear slots |
| V | 18个 / 18 | +4装备容量 / +4 gear slots |
| 最终 / Final | 全部95个 / All 95 | 成就/奖杯 + 最大容量 / Achievement + max capacity |

**梅林试炼类型 / Trial Types**:
- 滚石入洞 / Roll stone into hole
- 点燃火盆 / Light braziers
- 破坏石柱 / Destroy pillars
- 植物攀爬阵 / Plant puzzle
- 气球射击 / Balloon shooting

## 快速旅行 / Fast Travel

**飞路之焰网络**——霍格沃茨之遗的快速旅行系统 / Floo Flame Network — fast travel system.

| 类型 / Type | 用途 / Use |
|---|---|
| 城堡飞路点 / Castle Floo | 在城堡内部快速移动——数十个点 / Fast travel within the castle — dozens of points |
| 世界飞路点 / World Floo | 在地图上的各个地点——解锁后可以快速旅行 / Fast travel across the map |
| 扫帚飞行 / Broom Flight | 连接各区域——比跑步快得多 / Connect regions — much faster than running |

**建议**: 每到一个新区域——先解锁附近的飞路点 / Unlock all Floo Flames when entering a new area
