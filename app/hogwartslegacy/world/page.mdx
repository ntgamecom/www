# 世界与探索 / World & Exploration

## 区域总览 / Region Overview

霍格沃茨之遗的地图分为 **6大区域**——总面积约 5平方公里，充满地牢、营地、宝藏和谜题 / 6 major regions — ~5 sq km, filled with dungeons, camps, treasures, and puzzles.

| 区域 / Region | 等级范围 / Level | 特点 / Traits | 关键地点 / Key Locations |
|---|---|---|---|
| **霍格沃茨城堡 / Hogwarts Castle** | 1+ | 巨大的可探索城堡——7层+塔楼+地下室，超过100个房间 / Vast explorable castle — 7 floors + towers + dungeons, 100+ rooms | 大礼堂、图书馆、天文塔、各学院公共休息室 / Great Hall, Library, Astronomy Tower, common rooms |
| **活米村 / Hogsmeade** | 1+ | 巫师村庄——商店+旅店+多个支线任务起点 / Wizarding village — shops + inn + multiple quest starters | 三把扫帚、奥利凡德、蜜蜂公爵 / Three Broomsticks, Ollivander's, Honeydukes |
| **霍格沃茨谷地 / Hogwarts Valley** | 5-20 | 城堡外的初始区域——各种地牢+强盗营地+梅林试炼 / Starting area outside castle — dungeons + bandit camps + Merlin Trials | 霍格莫德车站、禁林边缘、地图密室 / Hogsmeade Station, Forest edge, Map Chamber |
| **禁林 / Forbidden Forest** | 10-25 | 密林+蜘蛛+巨怪+独角兽——危险但奖励丰厚 / Dense forest + spiders + trolls + unicorns — dangerous but rewarding | 禁林深处、蜘蛛巢穴、独角兽栖息地 / Deep Forest, Acromantula nest, Unicorn habitat |
| **北福特沼泽 / North Ford Bog** | 15-25 | 沼泽地——大量水生物和有毒植物，能见度低 / Swamp — aquatic enemies + toxic plants, low visibility | 废弃城堡、沼泽洞穴 / Ruined castle, swamp caves |
| **马伦韦姆湖 / Marunweem Lake** | 20-30 | 湖边区域——巨人+海妖+渔村 / Lakeside region — giants + sirens + fishing village | 马伦韦姆村、巨人洞穴 / Marunweem village, giant caves |
| **费尔德克罗夫特 / Feldcroft** | 15-25 | 塞巴斯蒂安的家乡——村庄+矿洞+悬崖 / Sebastian's hometown — village + mine + cliffs | 费尔德克罗夫特村、莫兰矿洞 / Feldcroft village, Morlan Mine |
| **波德萨尔海岸 / Poidsear Coast** | 25-35 | 海边+悬崖——高级区域，有大量飞行敌人 / Coast + cliffs — high-level area, many flying enemies | 灯塔、海岸洞穴 / Lighthouse, coastal caves |
| **克拉格玛尔海岸 / Cragcroftshire** | 25-35 | 最西部——最难的区域，推荐30级后再来 / Farthest west — hardest region, recommended level 30+ | 克拉格玛尔村、古老遗迹 / Cragcroft village, ancient ruins |

**区域探索建议 / Region Exploration Tips**:
- 每个区有飞路之焰快速旅行点——先解锁再探索 / Each region has Floo Flames — unlock before exploring
- 等级差距过大的区域敌人会一击秒杀你——不要跨30级挑战 / Large level gaps result in one-shots — don't challenge enemies 30+ levels above you
- 扫帚飞行可以快速跨区域——但低空飞行可能引到敌人 / Broom flight enables fast cross-region travel — low flight may aggro enemies

## 霍格沃茨城堡详解 / Hogwarts Castle Guide

城堡共有 **7层 + 地下室 + 多个塔楼**——超过100个可探索房间，数十个飞路之焰快速旅行点 / 7 floors + basement + towers — 100+ explorable rooms, dozens of Floo Flames.

| 楼层 / Floor | 关键地点 / Key Locations | 主要功能 / Main Use |
|---|---|---|

| **一楼 / Ground Floor** | 大礼堂、大理石楼梯、院长办公室、庭院 / Great Hall, Marble Staircase, Headmaster's Office, Courtyard | 主线+传送中心 / Main quest + travel hub |
| **二楼 / First Floor** | 图书馆、变形课教室、走廊、冥想厅 / Library, Transfiguration, Corridors, Meditation Hall | 任务+收集 / Quests + collectibles |

### 城堡秘密 / Castle Secrets

| 秘密 / Secret | 位置 / Location | 奖励 / Reward | 提示 / Hint |
|---|---|---|---|
| **飞路之焰 / Floo Flames** | 全城堡数十个 / Dozens across the castle | 快速旅行点 / Fast Travel point | 每层至少3-5个，全部解锁 / 3-5 per floor, unlock all |
| **画中门 / Painted Doors** | 各楼层墙壁上的画——可以进入隐藏房间 / Paintings on walls — enter hidden rooms | 装备+收集品 / Gear + collectibles | 走近按E互动 / Approach + press E |
| **算术占卜门 / Arithmancy Doors** | 各楼层——门上三角形符号谜题 / Various floors — triangle symbol puzzles | 装备/收集品 / Gear/Collectibles | 用对应的咒语按门上的符号顺序施法 / Cast spells matching the symbol sequence |
| **天文台 / Astronomy Tables** | 5个在城堡外的地图上 / 5 outside castle | 装备+经验 / Gear + XP | 必须在夜晚使用 / Must use at night |
| **飞贼雕像 / Demiguise Statues** | 全部30个——城堡14+活米村6+高地10 / All 30 — castle 14 + Hogsmeade 6 + Highlands 10 | 撬锁咒升级 / Alohomora upgrade | 只能在夜晚收集 ——按I等待 / Only at night — press I to wait |
| **魔法历史页 / Field Guide Pages** | 城堡内150+张 / 150+ inside castle | 经验+收集进度 / XP + completion | 使用显形(Revelio)寻找——每间房至少1张 / Use Revelio — at least 1 per room |
| **隐藏宝箱 / Hidden Chests** | 各角落——有些在画后/地板下 / Everywhere — some behind paintings/under floors | 装备+金钱 / Gear + gold | 显形会高亮 / Revelio highlights them |
| **秘密通道 / Secret Passages** | 城堡各处——连接不同楼层 / Throughout castle — connect different floors | 捷径+收集品 / Shortcuts + collectibles | 有些需要特定咒语开门 / Some require specific spells |

**城堡探索技巧 / Castle Exploration Tips**:
1. 每进一个新房间——按C(显形)扫描——宝藏和收集品一目了然 / Every new room — press C (Revelio) — reveals all treasures and collectibles
2. 算术占卜门是三解谜三角——需要在门附近找到对应的滚动符号 / Arithmancy doors have 3 triangles — find the rolling symbols nearby
3. 城堡飞路点激活后可以瞬间移动——充分利用传送节省时间 / Castle Floo Flames enable instant travel — use them to save time
4. 有些区域要在主线后期才能进入——记下来日再来 / Some areas are locked until later in the main quest — note them and return
5. 钟楼有隐藏的机关——在特定时间会触发特殊事件 / The Clock Tower has hidden mechanisms — special events at specific times

## 梅林试炼 / Merlin Trials

全地图共 **95个梅林试炼**——完成一定数量后扩展装备容量，同时解锁成就/奖杯 / 95 Merlin Trials — expand gear inventory capacity, plus achievement/trophy.

| 等级 / Tier | 需要完成 / Required | 奖励 / Reward | 累计容量 / Total Slots |
|---|---|---|---|
| I | 2个 / 2 | +4装备容量 / +4 gear slots | 24 |
| II | 6个 / 6 | +4装备容量 / +4 gear slots | 28 |
| III | 10个 / 10 | +4装备容量 / +4 gear slots | 32 |
| IV | 14个 / 14 | +4装备容量 / +4 gear slots | 36 |
| V | 18个 / 18 | +4装备容量 / +4 gear slots | 40 |
| 最终 / Final | 全部95个 / All 95 | 成就/奖杯 + 最大容量 / Achievement + max capacity | 48 |

**梅林试炼类型 / Trial Types**:

| 类型 / Type | 解法 / Solution | 分布 / Distribution |
|---|---|---|
| **滚石入洞 / Roll Stone into Hole** | 用退敌三尺(Depulso)把石头推入洞中 / Use Depulso to push stone into hole | 常见 / Common |
| **点燃火盆 / Light Braziers** | 用火焰熊熊(Incendio)或雷霆霹雳(Confringo)点燃所有火盆 / Light all braziers with Incendio or Confringo | 常见 / Common |
| **破坏石柱 / Destroy Pillars** | 用吹飞咒(Bombarda)或古代魔法破坏石柱 / Destroy pillars with Bombarda or Ancient Magic | 中等 / Medium |
| **植物攀爬阵 / Plant Puzzle** | 用统统加护(Glacius)冻结植物后攀爬至顶部 / Freeze plants with Glacius then climb to top | 少见 / Uncommon |
| **气球射击 / Balloon Shooting** | 用基础施法或任意咒语击破所有气球 / Pop all balloons with Basic Cast or any spell | 少见 / Uncommon |
| **石头堆叠 / Stone Stacking** | 用昏昏倒地(Accio)把石头拉至指定位置 / Use Accio to pull stones to marked positions | 中等 / Medium |

**梅林试炼搜索技巧 / Merlin Trial Search Tips**:
- 在扫帚上飞行时更容易从空中发现——它们通常有蓝色光芒 / Easier to spot from broom — they usually emit a blue shimmer
- 完成所有95个试炼耗时约8-12小时 / Completing all 95 takes ~8-12 hours
- 不需要一次性全部完成——随探索逐步完成即可 / No need to do all at once — complete them gradually as you explore

## 快速旅行 / Fast Travel

**飞路之焰网络**——霍格沃茨之遗的快速旅行系统，覆盖全地图数百个点 / Floo Flame Network — fast travel system covering hundreds of points across the map.

| 类型 / Type | 用途 / Use | 数量 / Count |
|---|---|---|
| 城堡飞路点 / Castle Floo | 在城堡内部快速移动——每层多个 / Fast travel within the castle — multiple per floor | 30+ |
| 世界飞路点 / World Floo | 在地图上的各个地点——解锁后可快速旅行 / Fast travel across the map — unlock to use | 100+ |
| 扫帚飞行 / Broom Flight | 连接各区域——比跑步快得多，但不能传送 / Connect regions — much faster than running, but not teleport | 持续可用 / Always available |

**建议**: 每到一个新区域——先解锁附近的飞路点，形成传送网络后再深入探索 / Unlock all Floo Flames when entering a new area, build the network before deep exploration
