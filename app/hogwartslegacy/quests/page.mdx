## 挑战类别 / Challenge Categories

| 类别 / Challenge | 说明 / Description |
|---|---|
| **战斗 / Combat** | 击败特定类型的敌人——蜘蛛、巨怪、强盗等 / Defeat specific enemy types |
| **探索 / Exploration** | 发现飞路火——解锁区域 / Discover Floo Flames |
| **收集 / Collection** | 收集外观+魔杖/ Collect appearances + wands |
| **挑战 / Challenge** | 完成梅林试炼+天文台等 / Merlin Trials + Astronomy Tables |

**挑战奖励 / Challenge Rewards**:
- 每次升级获得装备+外观奖励 / Each rank-up = gear + appearance reward
- 全部完成解锁成就/奖杯 / All complete = achievement
