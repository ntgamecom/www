### 塞巴斯蒂安·萨洛 / Sebastian Sallow (斯莱特林 / Slytherin)

**关系 / Relationship**: 同学→朋友→战友（最终可能决裂）/ Classmate → Friend → Ally (may eventually fall out)

| 任务 / Quest | 等级 / Level | 奖励 / Reward | 说明 / Description |
|---|---|---|---|
| 塞巴斯蒂安的请求 / Sebastian's Request | 8 | 雷霆霹雳 / Confringo | 在活米村触发——学习火焰弹射咒 / Triggered at Hogsmeade — learn Confringo |
| 影子的书 / The Shadow of the Study | 10 | 黑魔法知识 / Dark Arts lore | 探索禁林中的隐藏书房——接触黑魔法 / Explore hidden study in Forest — encounter Dark Arts |
| 地牢 / Under the Castle | 12 | 剧情推进 / Story progress | 霍格沃茨地牢下——发现黑魔法物品 / Beneath Hogwarts dungeon — discover dark artifacts |
| 在迷雾中 / In the Shadow of the Bloodline | 14 | 钻心剜骨 / Crucio | **关键选择**——是否学习不可饶恕咒 / **Key choice** — learn Crucio or not |
| 在莫兰矿 / In the Shadow of the Mine | 18 | 剧情推进 / Story progress | 费尔德克罗夫特矿洞——塞巴斯蒂安的过去 / Feldcroft mine — Sebastian's past |
| 在时间中 / In the Shadow of Time | 20 | 咒咒虐(可选) / Imperio (optional) | 另一个不可饶恕咒选择——控制敌人 / Another Unforgivable — control enemies |
| 在希望中 / In the Shadow of Hope | 22 | 剧情高潮 / Story climax | 塞巴斯蒂安姐姐的故事——情感转折 / Anne's story — emotional turning point |
| **塞巴斯蒂安的结局** / Sebastian's Fate | 24 | **杀戮咒 / Avada Kedavra** | 最终决择——决定塞巴斯蒂安的命运 / Final choice — decide Sebastian's fate |

**关键选择 / Key Choice**: 

是否将塞巴斯蒂安交给魔法部——决定他是否留在霍格沃茨 / Turn Sebastian in or not — determines if he stays at Hogwarts

| 选择 / Choice | 后果 / Consequence |
|---|---|
| **举报 / Report** | 塞巴斯蒂安被开除，离开霍格沃茨——不再可在世界中遇到 / Sebastian expelled, leaves Hogwarts — no longer found in the world |
| **不举报 / Don't Report** | 塞巴斯蒂安继续上学——但不再有更多同伴任务 / Sebastian stays at Hogwarts — no more companion quests |

**关系影响 / Relationship Impact**:
- 如果学习所有不可饶恕咒——塞巴斯蒂安会更信任你 / Learning all Unforgivables — Sebastian trusts you more
- 拒绝学习不会影响任务线——但会改变对话 / Refusing to learn doesn't block the questline — but changes dialogue
- 不举报塞巴斯蒂安是大多数玩家的选择——保留他在霍格沃茨 / Not reporting is the popular choice — keeps him at Hogwarts

## 作业任务 / Assignment Quests

作业任务是解锁新咒语和配方的关键——建议一接到就做 / Assignments are key to unlocking new spells and recipes — do them as soon as received.

| 任务 / Assignment | 授课教授 / Professor | 完成条件 / Condition | 奖励 / Reward |
|---|---|---|---|
| 魔药课作业I / Potions Assignment I | 夏普 / Sharp | 制作2瓶药水 / Brew 2 potions | 马德拉配方 / Maxima recipe |
| 草药学作业I / Herbology I | 加维兰 / Garlick | 种植2株植物 / Plant 2 plants | 牵裙草配方 / Chinese Chomping Cabbage recipe |
| 魔咒课作业I / Charms I | 罗南 / Ronan | 格挡5次攻击 / Block 5 attacks | 阿拉霍洞开I / Alohomora I |
| 变形课作业I / Transfiguration I | 维格 / Weasley | 使用变形法术3次 / Use transfiguration 3x | 吹飞咒 / Bombarda |
| 魔药课作业II / Potions II | 夏普 / Sharp | 制作1瓶强化魔药 / Brew 1 enhanced potion | 减伤配方 / Fortification recipe |
| 草药学作业II / Herbology II | 加维兰 / Garlick | 种植5株植物 / Plant 5 plants | 冰刺植物配方 / Mandrake recipe |
| 魔咒课作业II / Charms II | 罗南 / Ronan | 完成连招 / Complete combo | 降敌陷阱 / Arresto Momentum |
| 变形课作业II / Transfiguration II | 维格 / Weasley | 变形8个目标 / Transfigure 8 targets | 退敌三尺 / Depulso |

**作业任务技巧 / Assignment Tips**:
- 作业任务没有时间限制——随时可以做 / No time limit — can be done anytime
- 有些作业需要特定等级才能解锁 / Some assignments require specific character levels
- 建议优先完成魔咒课和变形课作业——它们奖励战斗咒语 / Prioritize Charms and Transfiguration — they reward combat spells
- 魔药课和草药学作业解锁配方——长期收益高 / Potions and Herbology unlock recipes — high long-term value
- 接到新作业后不要拖延——早完成早获得奖励 / Don't procrastinate on new assignments — complete early for rewards

## 任务顺序推荐 / Recommended Quest Order

为了最佳体验和等级曲线，建议按以下顺序推进 / For the best experience and level curve, follow this order:

| 阶段 / Phase | 推荐任务 / Recommended Quests | 原因 / Reason |

| **第1阶段(等级1-8)** | 主线1-8 + 活米村支线 / Main 1-8 + Hogsmeade side | 解锁核心法术和机制 / Unlock core spells and mechanics |

| **第3阶段(等级14-20)** | 作业任务优先 + 主线试炼 / Assignments first + main trials | 作业奖励关键咒语 / Assignments reward key spells |

| **第5阶段(等级28-30)** | 最终决战 + 收集品扫尾 / Final battle + collectible cleanup | 完成主线 + 白金收尾 / Complete story + cleanup for platinum |

|---|---|---|
