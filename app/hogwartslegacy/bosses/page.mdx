# BOSS攻略 / Boss Fights Guide

霍格沃茨之遗中有多种类型的Boss战——从巨型Boss精英到剧情重要对决 / Hogwarts Legacy features multiple boss types — from giant elite enemies to critical story showdowns.

---

## 巨怪 / Trolls

### 巨怪类型 / Troll Types

| 巨怪 / Troll | 等级 / Level | 生命 / HP | 弱点 / Weakness | 免疫 / Resistant |
|---|---|---|---|---|
| **普通巨怪 / Common Troll** | 10-20 | 2000+ | 火系咒语 / Fire spells | 控制类 / Control spells |
| **盔甲巨怪 / Armored Troll** | 20-30 | 3500+ | 需要先破甲——火系 / Break armor first — fire | 物理伤害 / Physical dmg |
| **山地巨怪 / Mountain Troll** | 30-40 | 5000+ | 冰系冻住后火系 / Freeze then fire | 普通施法 / Basic casts |
| **黑暗巨怪 / Dark Troll** | 40-50 | 8000+ | 古代魔法 / Ancient Magic | 大部分咒语 / Most spells |

**巨怪攻击模式 / Troll Attack Patterns**:

| 攻击 / Attack | 前摇 / Wind-up | 闪避时机 / Dodge Timing | 反击窗口 / Counter Window |
|---|---|---|---|
| 重拳砸地 / Slam | 明显——抬起手臂 / Clear — arm lift | 砸地瞬间翻滚 / Roll at slam | 较长——3-4次施法 / Long — 3-4 casts |
| 横扫 / Sweep | 转身蓄力 / Turn and wind-up | 横向翻滚 / Side dodge | 中等——2次施法 / Medium — 2 casts |
| 投石 / Throw Rock | 弯腰捡石头 / Bend to pick | 向侧方跑 / Run sideways | 短——1-2次施法 / Short |
| 冲锋 / Charge | 低头蓄力 / Head down | 侧向闪避 / Side dodge | 短——1次施法 / Short |

**巨怪打法策略 / Troll Strategy**:
1. 保持中距离——巨怪的近战连招伤害极高 / Keep mid-range — melee combo hits hard
2. 使用火焰咒语打弱点——火焰熊熊+爆破咒 / Use fire spells — Incendio + Confringo
3. 盔甲巨怪先用喷喷障或飞来咒破掉盔甲 / Armored: use Depulso/Accio to break armor
4. 巨怪攻击后摇很长——是输出窗口 / Long recovery after attacks — damage window
5. 古代魔法投掷对巨怪有奇效——造成大量伤害 + 眩晕 / AM throw is very effective — big dmg + stun

## 剧情Boss / Story Bosses

### 兰洛克 / Ranrok

| 阶段 / Phase | 生命 / HP | 攻击模式 / Attack Pattern | 策略 / Strategy |
|---|---|---|---|
| 第1阶段 / Phase 1 | 70% | 远程火球+召唤黑暗精灵 / Ranged fire + dark elf summons | 躲避火球——清理小兵——输出窗口打本体 / Dodge fireballs |
| 第2阶段 / Phase 2 | 40% | 火焰龙卷风+地火 / Fire tornado + ground fire | 场地变小——AOE频繁——保持移动 / Arena shrinks |
| 第3阶段 / Phase 3 | 10% | 狂暴+大型AOE / Enrage + big AoE | 古代魔法终结技收尾 / AM finisher to end |

**兰洛克关键技巧 / Ranlock Key Tips**:
1. 第一阶段不要急于输出——优先清理召唤的黑暗精灵 / Phase 1: clear adds before focusing boss
2. 第二阶段地面火焰范围大——利用护身咒抵挡 / Phase 2: ground fire is wide — use Protego
3. 第三阶段他施法速度加快——完美闪避更重要 / Phase 3: faster casts — perfect dodge is critical
4. 古代魔法终结技消耗2格——可以造成15%+伤害 / AM finisher (2 bars) deals 15%+ HP damage

### 维克多·卢克伍德 / Victor Rookwood

| 阶段 / Phase | 生命 / HP | 攻击模式 / Attack Pattern | 策略 / Strategy |
|---|---|---|---|
| 第1阶段 / Phase 1 | 60% | 普通咒语+瞬移 / Standard spells + teleport | 追踪瞬移方向——预判输出 / Track teleport direction |
| 第2阶段 / Phase 2 | 30% | 分身术——多个幻影 / Creates clones | 找到本体——分身被击中消失 / Find real one |
| 第3阶段 / Phase 3 | 10% | 强化状态——攻击频率翻倍 / Enhanced state | 防御为主——等大招后反击 / Defend first — counter after big move |

**卢克伍德关键技巧 / Rookwood Key Tips**:
1. 分身阶段——所有分身会一起攻击——使用AOE同时清理 / Clone phase: use AoE to clear clones
2. 他的护盾很快消失——用对应颜色咒语立即破盾 / His shield is brief — break immediately
3. 瞬移后他的位置有绿色光点提示 / Green indicator shows teleport destination

### 傲慢的教师 / Crucio Teacher (塞巴斯蒂安任务线)

| 项目 / Aspect | 说明 / Description |
|---|---|
| 位置 / Location | 地下墓穴 / Catacombs |
| 难度 / Difficulty | 高——适合25+级 / High — recommended 25+ |
| 机制 / Mechanic | 他使用钻心剜骨——被击中后会持续掉血 / Uses Crucio — DoT on hit |
| 奖励 / Reward | 学会钻心剜骨 / Learn Crucio spell |

**对抗策略 / Counter Strategy**:
1. 带上大量疗伤药——钻心的持续伤害很猛 / Carry many Wiggenweld potions
2. 护身咒可以挡下钻心咒——但完美护身更安全 / Protego blocks Crucio — Perfect Protego ideal
3. 当他施法时全力破盾——打断施法 / Break shield while he casts
4. 使用冰冻咒后接爆破咒——伤害翻倍 / Freeze + Confringo combo
