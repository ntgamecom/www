# 新手入门 / Beginner Guide

## 操作基础 / Basic Controls

### PC版默认键位 / PC Default Controls

| 按键 / Key | 功能 / Action |
|---|---|
| W/A/S/D | 移动 / Move |
| 鼠标左键 / Mouse 1 | 基础施法 / Basic Cast |
| 鼠标右键 / Mouse 2 | 格挡(护身护卫) / Block (Protego) |
| 鼠标右键+左键 | 古代魔法投掷 / Ancient Magic Throw |
| Shift | 冲刺(地面)/加速(扫帚) / Sprint/Accelerate (broom) |
| Space | 跳跃(地面)/上升(扫帚)/翻滚(战斗中) / Jump/Climb/Dodge Roll |
| Ctrl | 潜行 / Sneak |
| Q | 施放当前法术 / Cast Current Spell |
| E | 使用/互动 / Use/Interact |
| R | 锁定目标 / Lock Target |
| 1/2/3/4 | 法术轮盘切换(4套) / Spell Set Switch (4 sets) |
| Tab | 魔法界面 / Magic Menu |
| M | 地图 / Map |
| F | 古代魔法攻击 / Ancient Magic Attack |
| C | 扫描(显形) / Revelio |
| V | 切换视角 / Camera Toggle |

| 设施 / Facility | 功能 / Function | 解锁条件 / Unlock |

**有求必应屋核心技巧 / Room Tips**:
1. 种**土豆叶(Edurus)**和**马德拉卵石(Maxima)**——战斗中最有用的两种药水 / Grow Edurus + Maxima — most useful combat potions

3. 附魔台升级装备——比拾取更高等级的装备更有效 / Enchantment table upgrades are more effective than finding higher-level gear

## 战斗基础 / Combat Basics

### 法术颜色系统 / Spell Color System

敌人的护盾有颜色——必须用对应颜色法术打破 / Enemy shields have colors — must use matching color to break.

| 颜色 / Color | 法术类型 / Type | 代表法术 / Examples |
|---|---|---|
| **紫色 / Purple** | 控制 / Control | 昏昏倒地(Accio)、统统加护(Glacius) |
| **红色 / Red** | 伤害 / Damage | 除你武器(Expelliarmus)、火焰熊熊(Incendio) |
| **黄色 / Yellow** | 力量 / Force | 退敌三尺(Depulso)、飞沙走石(Flipendo) |

### 核心战斗循环 / Core Combat Loop

```
护身护卫(右键格挡) → 完美弹反→ 法术(1/2/3/4切换轮盘破盾) → 古代魔法攻击(F) → 重复
Protego (right-click) → Perfect block → Spell (break shield) → Ancient Magic (F) → Repeat
```

| 技巧 / Technique | 操作 / Input | 效果 / Effect |
|---|---|---|
| 完美护身 / Perfect Protego | 敌人攻击瞬间右键 / Right-click right before hit | 击晕附近敌人 + 反制 / Stun + counter |
| 翻滚闪避 / Dodge Roll | Space + 方向 / + direction | 无敌帧——用于无法格挡的红光攻击 / I-frames — for unblockable red attacks |
| 古代魔法投掷 / Ancient Magic Throw | 右键+左键 / Right+Left click | 抓起场景物体扔向敌人——高伤害 / Grab objects and throw — high damage |
| 石化 / Petrificus Totalus | 潜行到背后 / Sneak behind | 秒杀大部分敌人——无声 / Instant kill most enemies — silent |
| 终结技 / Finisher | 敌人眩晕时靠近 / Near stunned enemy | 酷炫处决动画 / Stylish execution |

## 天赋系统 / Talent System

| 天赋树 / Talent Tree | 说明 / Description |
|---|---|
| **咒语 / Spells** | 增强各法术的效果——推荐优先加满 / Boost spell effects — recommended priority |
| **黑暗艺术 / Dark Arts** | 增强不可饶恕咒——控制+dps / Boost Unforgivable Curses — control + DPS |
| **核心 / Core** | 基础能力——生命值+古代魔法+防御 / Core abilities — HP + Ancient Magic + defense |
| **潜行 / Stealth** | 增强潜行击杀——石化+隐身 / Boost stealth kills — Petrificus + Disillusionment |

**推荐加点顺序 / Recommended Talent Order**:
1. **咒语(核心输出)**— 第5级开始点 / Spells (main DPS) — start at level 5
2. **核心(生存)**— 10级左右点几点活力 / Core (survival) — some vitality around level 10
3. **黑暗艺术(可选)**— 如果你打算走不可饶恕咒路线 / Dark Arts (optional) — if using Unforgivable Curses
4. **潜行(最后)**— 中后期 / Stealth (late)

## 新手常见错误 / Common Beginner Mistakes

| # | 错误 / Mistake | 正确做法 / Correct Way |
|---|---|---|
| 1 | 不用护身护卫 / Forget Protego | 格挡是战斗中最重要的操作——完美格挡致晕敌人 / Blocking is the most important combat mechanic |
| 2 | 只用一套法术 / Use only 1 spell set | 随时切换4套法术轮盘——应对不同敌人 / Switch between 4 sets constantly — counter different enemies |
| 3 | 不做作业任务 / Skip assignments | 作业任务奖励新咒语——咒语是战斗核心 / Assignments reward spells — spells = combat power |
| 4 | 不升级装备 / Don't upgrade gear | 天降神兵附魔台可以大幅提升装备属性 / Enchantment table massively boosts gear stats |
| 5 | 随便乱加点天赋 / Waste talent points | 天赋不能重置——看到不喜欢的Build要花50小时重玩 / Talents can't be reset — bad build = 50 hour replay |
| 6 | 不买扫帚 / Don't buy broom | 扫帚是跑图的必备工具——越早买越好 / Broom is essential for exploration — buy ASAP |
