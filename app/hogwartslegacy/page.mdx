# 霍格沃茨之遗攻略 / Hogwarts Legacy Guide

《霍格沃茨之遗》是Avalanche Software开发的开放世界动作RPG——19世纪的霍格沃茨，你作为一名拥有古老血脉的五年级新生，发现了古老的魔法秘密。从霍格沃茨城堡到禁林，从活米村到高地——整个魔法世界等你探索。本攻略涵盖全部咒语与战斗技巧、霍格沃茨全地图指引、同伴关系任务以及全收集品指南。

Hogwarts Legacy is Avalanche Software's open-world action RPG set in the 19th-century Wizarding World — as a fifth-year student with an ancient bloodline, you uncover a long-forgotten magical secret. From Hogwarts Castle to the Forbidden Forest, Hogsmeade to the Highlands — the entire magical world awaits. This guide covers all spells and combat, Hogwarts map guide, companion relationship quests, and complete collectibles.

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **学院区别不大 / Houses Are Similar** | 只有第一个学院专属支线不同——其他内容完全一样 / Only the first house-specific quest differs — rest is identical |
| **2** | **早点开飞路之焰 / Unlock Floo Flames** | 飞路网快速旅行——开得越多跑图越方便 / Floo Network fast travel — unlock as many as possible |
| **3** | **能开锁的咒语优先 / Get Alohomora ASAP** | 解锁咒语——很多宝箱在锁后面 / Unlock spell — many chests behind locks |
| **4** | **装备属性比外观重要 / Stats > Looks** | 装备外观可以幻化——始终穿数值最高的 / Gear appearance can be transmogged — always equip highest stats |
| **5** | **多做作业任务 / Do Assignment Quests** | 作业任务奖励新咒语——咒语越多战斗越强 / Assignments reward new spells — more spells = stronger combat |
| **6** | **扫帚买了就永远能用 / Buy Broom Once** | 扫帚是一次性投资——尽早买 / Broom is a one-time purchase — buy early |
| **7** | **天赋不能重置 / Talents Cannot Be Reset** | 天赋点数有限——加点前想好 / Talent points are finite — plan before spending |
| **8** | **土豆叶治百病 / Edurus Potion** | 石化药水让你无敌几秒——战斗中极度实用 / Makes you invincible for seconds — extremely useful in combat |

## 难度说明 / Difficulty Guide

| 难度 / Difficulty | 说明 / Description |
|---|---|
| 故事 / Story | 适合想体验剧情的玩家 / For story-focused players |
| 普通 / Normal | 标准体验——推荐首次游玩 / Standard — recommended first play |
| 困难 / Hard | 敌人更肉+伤害更高——需要熟悉战斗连招 / Tankier enemies + more damage — requires combat mastery |

**困难模式核心提示 / Hard Mode Tips**:
1. 一定要频繁使用**护身护卫(Protego)**——完美格挡眩晕敌人 / Use Protego frequently — perfect block stuns enemies
2. 翻滚(闪避)是无敌帧——比格挡更安全 / Dodge roll has i-frames — safer than blocking
3. 药水是关键——战斗中吃马德拉卵石+土豆叶 / Potions are key — use Maxima + Edurus in combat
4. 养成切换法术轮盘的习惯——不要只用一套 / Practice switching spell sets — don't stick to one loadout
