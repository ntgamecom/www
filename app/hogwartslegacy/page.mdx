# 霍格沃茨之遗攻略 / Hogwarts Legacy Guide

《霍格沃茨之遗》是Avalanche Software开发的开放世界动作RPG——19世纪的霍格沃茨，你作为一名拥有古老血脉的五年级新生，发现了古老的魔法秘密。从霍格沃茨城堡到禁林，从活米村到高地——整个魔法世界等你探索。本攻略涵盖全部咒语与战斗技巧、霍格沃茨全地图指引、同伴关系任务以及全收集品指南。

Hogwarts Legacy is Avalanche Software's open-world action RPG set in the 19th-century Wizarding World — as a fifth-year student with an ancient bloodline, you uncover a long-forgotten magical secret. From Hogwarts Castle to the Forbidden Forest, Hogsmeade to the Highlands — the entire magical world awaits. This guide covers all spells and combat, Hogwarts map guide, companion relationship quests, and complete collectibles.

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **学院区别不大 / Houses Are Similar** | 只有第一个学院专属支线不同——其他内容完全一样 / Only the first house-specific quest differs — rest is identical |
| **2** | **早点开飞路之焰 / Unlock Floo Flames** | 飞路网快速旅行——开得越多跑图越方便，每到一个新区先开飞路点 / Floo Network fast travel — unlock as many as possible, prioritize in new areas |
| **3** | **能开锁的咒语优先 / Get Alohomora ASAP** | 解锁咒语——很多宝箱在锁后面，II级和III级需要收集飞贼雕像 / Unlock spell — many chests behind locks, levels II and III require Demiguise statues |
| **4** | **装备属性比外观重要 / Stats &gt; Looks** | 装备外观可以幻化——始终穿数值最高的，分解或出售旧装备 / Gear appearance can be transmogged — always equip highest stats, sell or dismantle old gear |
| **5** | **多做作业任务 / Do Assignment Quests** | 作业任务奖励新咒语——咒语越多战斗越强，还能解锁新配方 / Assignments reward new spells — more spells = stronger combat, also unlocks new recipes |
| **6** | **扫帚买了就永远能用 / Buy Broom Once** | 扫帚是一次性投资——尽早买，最快的光轮2000需要4500金币 / Broom is a one-time purchase — buy early, fastest (Yew Broom) costs 4500 gold |
| **7** | **天赋不能重置 / Talents Cannot Be Reset** | 天赋点数有限(共50点左右)——加点前想好，建议先规划Build / Talent points are finite (~50 total) — plan before spending, map out your build early |
| **8** | **土豆叶治百病 / Edurus Potion** | 石化药水让你无敌几秒——战斗中极度实用，配合马德拉卵石更强 / Makes you invincible for seconds — extremely useful in combat, combo with Maxima |
| **9** | **善用显形咒 / Use Revelio Constantly** | 按C扫描周围——高亮所有可互动物品、隐藏宝箱和收集品——CD很短 / Press C to scan — highlights all interactables, hidden chests, and collectibles — short cooldown |
| **10** | **药水材料自己种 / Grow Your Own Mats** | 在草药台种植药水材料——比购买便宜得多，建议种3-5个大种植盆 / Grow potion ingredients at planting stations — much cheaper than buying, use 3-5 large pots |

**学院选择对游戏的影响 / Impact of House Choice**: 
- 主线任务完全相同——不影响故事结局 / Main quest is identical — no impact on ending

- 部分对话有细微差异——NPC会根据你的学院有不同反应 / Some dialogue differences — NPCs react based on your house
- 学院专属支线只在第1-2小时——不影响后续流程 / Exclusive quest only in the first 1-2 hours — no impact on later game

## 难度说明 / Difficulty Guide

| 难度 / Difficulty | 说明 / Description | 推荐人群 / Recommended For |
|---|---|---|
| 故事 / Story | 极低战斗难度——敌人伤害很低 / Very low combat difficulty — minimal enemy damage | 只想体验剧情/不擅长动作游戏的玩家 / Story-only players or action game newcomers |
| 普通 / Normal | 标准体验——推荐首次游玩 / Standard — recommended first play | 大多数玩家 / Most players |
| 困难 / Hard | 敌人更肉+伤害更高——需要熟悉战斗连招 / Tankier enemies + more damage — requires combat mastery | 有动作游戏经验的玩家 / Experienced action gamers |
| 极难 / Hardcore | 最高难度——敌人伤害极高+大量生命 / Maximum difficulty — extreme enemy damage and HP | 寻求挑战的老玩家 / Veterans seeking a challenge |

**困难模式核心提示 / Hard Mode Tips**:
1. 一定要频繁使用**护身护卫(Protego)**——完美格挡眩晕敌人 / Use Protego frequently — perfect block stuns enemies
2. 翻滚(闪避)是无敌帧——比格挡更安全，红光攻击必须翻滚 / Dodge roll has i-frames — safer than blocking, red attacks must be dodged
3. 药水是关键——战斗中吃马德拉卵石+土豆叶+专注药水 / Potions are key — use Maxima + Edurus + Focus in combat
4. 养成切换法术轮盘的习惯——不要只用一套，至少切换2-3套 / Practice switching spell sets — use at least 2-3 different loadouts
5. 优先升级核心天赋——生命值和古代魔法充能是生存基础 / Prioritize Core talents — HP and Ancient Magic charge are survival basics
6. 利用环境——场景中的爆炸桶和悬吊物可以造成大量伤害 / Use environment — explosive barrels and hanging objects deal massive damage

### 战斗系统核心机制 / Combat System Core

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **护盾颜色 / Shield Colors** | 紫(控制)/红(伤害)/黄(力量)——必须用同色法术破盾 / Purple(control)/Red(damage)/Yellow(force) — must match color |
| **古代魔法 / Ancient Magic** | 攻击填充紫色条——满后可释放高伤终结技或投掷物体 / Fill purple bar via attacks — use for high-dmg finisher or object throw |
| **完美格挡 / Perfect Protego** | 攻击瞬间格挡——眩晕周围敌人并触发反击 / Block at the moment of impact — stuns nearby enemies and triggers counter |
| **翻滚闪避 / Dodge Roll** | 有短暂无敌帧——红光攻击无法格挡必须翻滚 / Brief i-frames — red attacks cannot be blocked, must dodge |
| **连击系统 / Combo System** | 连续攻击叠加连击数——某些天赋和装备词条依赖连击 / Consecutive hits build combo count — some talents and traits scale with combo |
| **终结技 / Finisher** | 眩晕敌人后靠近触发——酷炫处决动画，回少量生命 / Approach stunned enemy — stylish execution, small HP restore |
