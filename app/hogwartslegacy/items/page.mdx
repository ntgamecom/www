# 收集与装备 / Items & Collectibles

## 装备系统 / Gear System

霍格沃茨之遗的装备分为 **5个槽位**——属性随机生成，品质和稀有度决定词条数量 / 5 gear slots — stats are randomly generated, quality and rarity determine trait slots.

## 外观幻化 / Transmog System

**装备外观和属性分离**——你可以保留外观的同时更换装备，无需牺牲颜值 / Appearance and stats are separate — keep looks while swapping gear, no need to sacrifice style.

| 操作 / Action | 方法 / How |
|---|---|
| 更改外观 / Change Appearance | 在装备界面按F(PC)/三角(PS)/Y(Xbox)——选择外观 / In gear menu, press appropriate key — select appearance |
| 外观预览 / Preview | 悬停在任意已获得的装备外观上——可预览效果 / Hover over any unlocked appearance — preview effect |
| 外观解锁 / Unlock | 每次拾取新装备都会解锁对应外观——分解/出售不影响 / Picking up new gear unlocks its appearance — selling/dismantling doesn't remove it |
| 外观收集进度 / Collection Progress | 在魔法界面的"外观"选项卡查看收集进度 / Check progress in "Appearances" tab of Magic Menu |

**重要**: 只穿最高等级的装备——不好的外观直接在装备界面换成喜欢的 / Always equip highest stat gear — just change the look in the gear menu
