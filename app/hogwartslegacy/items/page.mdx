## 外观幻化 / Transmog

**装备外观和属性分离**——你可以保留外观的同时更换装备 / Appearance and stats are separate — keep looks while swapping gear.

| 操作 / Action | 方法 / How |
|---|---|
| 更改外观 / Change Appearance | 在装备界面按F(PC)/三角(PS)/Y(Xbox) / In gear menu, press appropriate key |
| 外观预览 / Preview | 悬停在任意已获得的装备外观上 / Hover over any unlocked appearance |
| 外观解锁 / Unlock | 每次拾取新装备都会解锁对应外观 / Picking up new gear unlocks its appearance |

**重要**: 只穿最高等级的装备——不好的外观直接在装备界面换成喜欢的 / Always equip highest stat gear — just change the look
