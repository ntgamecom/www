# 战斗系统 / Combat System

霍格沃茨之遗的战斗系统围绕咒语组合展开——四种颜色类别对应不同功能——通过切换咒语组实现连击 / Hogwarts Legacy combat revolves around spell combinations — four color categories with different functions — switching spell sets enables combos.

## 断裂机制 / Stun & Break System

### 护盾与断裂 / Shield & Breaking

敌人有不同颜色的护盾——需要使用对应颜色的咒语才能打破 / Enemies have colored shields — matching color spell breaks them.

| 护盾颜色 / Shield Color | 破解咒语颜色 / Counter Color | 破解后效果 / Effect After Break |
|---|---|---|
| **紫色 / Purple** | 紫色咒语 / Purple spells | 敌人眩晕——可追加攻击 / Stunned |
| **黄色 / Yellow** | 黄色咒语 / Yellow spells | 敌人击退——可能撞到其他敌人 / Knocked back |
| **红色 / Red** | 红色咒语 / Red spells | 大量伤害+击倒 / Heavy damage + knockdown |

**断裂技巧 / Breaking Tips**:
| 技巧 / Tip | 说明 / Description |
|---|---|
| 不要硬打护盾 / Don't Attack Shield | 不用对应颜色咒语——护盾不会减 / Matching color required to break |
| 提前准备 / Prepare Loadout | 一场战斗中带齐三种颜色咒语 / Carry all 3 colors in battle |
| 护盾恢复 / Shield Regen | 敌人护盾一段时间后会再生 / Shields regenerate after a while |
| 多人场景 / Multi-Enemy | 优先破解正在施法的敌人的护盾 / Priority on casting enemies |

## 古代魔法系统 / Ancient Magic

### 古代魔法计量表 / Ancient Magic Meter

| 计量格 / Meter Segment | 填充方式 / Fill Method | 消耗 / Cost |
|---|---|---|
| 第1格 / Slot 1 | 基础施法50次 / 50 Basic Casts | 古代魔法投掷 / AM Throw |
| 第2格 / Slot 2 | 完美闪避10次 / 10 Perfect Dodges | 古代魔法终结技 / AM Finisher |
| 第3格 / Slot 3 | 断裂咒语击中15次 / 15 Shield Breaks | 古代魔法终结技 / AM Finisher |
| 第4格 / Slot 4 | 任意组合 / Any Combo | 古代魔法终结技 / AM Finisher |
| 第5格 / Slot 5 | 任意组合(需要天赋) / Any Combo (talent) | 古代魔法终结技 / AM Finisher |

**古代魔法投掷 / Ancient Magic Throw**:
- 按住R1/RB瞄准附近物体——然后投掷 / Aim nearby objects — then throw
- 物体越大——伤害越高 / Bigger object = more damage
- 场景中发光物体是古代魔法投掷物 / Glowing objects are AM throwables

**古代魔法终结技 / Ancient Magic Finisher**:
- 消耗2或3格计量表 / Cost: 2 or 3 meter segments
- 造成大量伤害——并且产生华丽过场动画 / Massive damage + cinematic kill
- 可以对Boss使用——造成约10-15%最大生命伤害 / ~10-15% max HP damage on bosses

## 防御系统 / Defense System

### 防护与闪避 / Protego & Dodge

| 防御动作 / Action | 操作 / Input | 效果 / Effect |
|---|---|---|
| **护身咒 / Protego** | 按住防御键 / Hold defend | 生成护盾——挡下基础攻击 / Shield blocks basic attacks |
| **完美护身 / Perfect Protego** | 攻击前瞬间按防御 / Press at last moment | 护盾碎裂——敌人眩晕——反击窗口 / Shield breaks — enemy stunned |
| **闪避 / Dodge** | 按翻滚键 / Dodge button | 向前后左右翻滚 / Roll in any direction |
| **完美闪避 / Perfect Dodge** | 攻击前瞬间翻滚 / Dodge at last moment | 时间减慢——反击机会 / Slow motion — counter window |
| **幻身咒潜行 / Disillusionment** | 按潜行键 / Stealth button | 部分隐身——背后暗杀 / Partial invisibility |

**完美护身后连招 / Perfect Protego Combo**:
1. 敌人攻击瞬间按防御 / Block at the last moment
2. 护身咒碎裂——敌人眩晕 / Protego breaks — enemy stunned
3. 立刻使用飞来咒拉过来 + 火焰熊熊 / Accio pull + Incendio
4. 或使用爆破咒远程追击 / Or Confringo ranged follow-up
