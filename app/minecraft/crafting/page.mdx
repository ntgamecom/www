## 合成系统 / Crafting System

我的世界拥有丰富的合成系统。从基础的工作台到复杂的酿造台，每个工作站都有独特的用途。

Minecraft has a rich crafting system. From the basic crafting table to the complex brewing stand, each workstation has unique uses.

### 工作台 / Crafting Table

工作台（Crafting Table）是最重要的工作站，提供 3x3 的合成网格。

The crafting table is the most important workstation, providing a 3x3 crafting grid.

| 重要配方 / Key Recipe | 材料 / Materials | 产物 / Result |
|---|---|---|
| 木棍 / Stick | 木板 x2 竖直 | 木棍 x4 |
| 火把 / Torch | 木棍 + 煤炭或木炭 | 火把 x4 |
| 箱子 / Chest | 木板 x8 围圈 | 箱子 x1 |
| 床 / Bed | 羊毛 x3 + 木板 x3 | 床 x1 |
| 梯子 / Ladder | 木棍 x7 H 形 | 梯子 x3 |
| 铁砧 / Anvil | 铁块 x3 + 铁锭 x4 | 铁砧 x1 |
| 附魔台 / Enchanting Table | 钻石 x2 + 黑曜石 x4 + 书 x1 | 附魔台 x1 |
| 末影箱 / Ender Chest | 黑曜石 x8 + 末影之眼 x1 | 末影箱 x1 |

熔炉（Furnace）用于烧制食物和冶炼矿石。

The furnace is used for cooking food and smelting ores.

### 高炉 / Blast Furnace

高炉（Blast Furnace）专门用于冶炼矿石，速度是熔炉的两倍。

The blast furnace smelts ores twice as fast as a regular furnace.

- **合成材料**：铁锭 x5 + 熔炉 x1 + 平滑石头 x3
- **注意**：只能冶炼矿石和金属装备，不能烧制食物

### 烟熏炉 / Smoker

烟熏炉（Smoker）专门用于烧制食物，速度是熔炉的两倍。

The smoker cooks food twice as fast as a regular furnace.

- **合成材料**：原木 x4 + 熔炉 x1
- **注意**：只能烧制食物，不能冶炼矿石

### 锻造台 / Smithing Table

The smithing table upgrades diamond tools to netherite.

| 下界合金剑 / Netherite Sword | 钻石剑 + 下界合金锭 | 下界合金剑 |
| 下界合金盔甲 / Netherite Armor | 钻石盔甲 + 下界合金锭 | 下界合金盔甲 |
| 下界合金斧 / Netherite Axe | 钻石斧 + 下界合金锭 | 下界合金斧 |

### 合成基本原则 / Crafting Principles

- **形状重要 / Shape Matters** — 大多数配方要求特定形状排列
- **无序配方 / Shapeless Recipes** — 部分配方不要求形状，如染料合成
- **数量决定产出 / Quantity Determines Output** — 某些配方产出多个物品
- **标签系统 / Tag System** — 某些配方接受同类型任意变体（如任意木板、任意原木）
