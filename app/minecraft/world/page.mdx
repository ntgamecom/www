## 世界探索 / World Exploration

我的世界的世界是程序化生成的，包含多个维度和丰富的建筑结构。本章是探险者的全面指南。

Minecraft's world is procedurally generated with multiple dimensions and structures. This is your explorer's guide.

### 世界生成 / World Generation

#### 种子系统 / Seed System

- **世界种子 / World Seed** — 一个数字或文本字符串，决定整个世界的地形
- **相同的种子产生完全相同的世界**（相同版本下）
- 使用 `/seed` 命令查看当前世界的种子
- 可以在创建世界时输入特定种子

#### 地形生成 / Terrain Generation

1.21 版本中，世界生成分为以下几个层次：

| 高度层 / Y-level | 描述 / Description |
|---|---|
| 320 (最高) | 山脉顶峰，云层之上 |
| 256 | 云层高度（基岩版为 128） |
| 128 | 传统最高高度 |
| 63 | 海平面 / Sea Level |
| 0 | 基岩层起始 |
| -32 | 深层钻石生成区 |
| -64 | 世界最低点，基岩层 |

#### 生物群系分布 / Biome Distribution

- **温度决定 / Temperature-based** — 温暖群系靠近赤道（世界中心），寒冷群系靠近两极
- **地形高度 / Elevation** — 低洼处倾向于海洋和沼泽，高处倾向于山地
- **随机变异 / Random Variation** — 同一温度带内可能出现多种群系

### 主世界建筑结构 / Overworld Structures

#### 村庄 / Village

- **生物群系 / Biomes**: 平原、沙漠、针叶林、雪原、热带草原
- **居民 / Inhabitants**: 村民、铁傀儡、猫
- **战利品 / Loot**: 箱子内的食物、工具、绿宝石
- **探索价值**: 交易、免费床、初始工具

#### 沙漠神殿 / Desert Pyramid

- **生物群系**: 沙漠
- **宝藏 / Treasure**: 隐藏密室中的 TNT 和宝箱
- **陷阱 / Trap**: 压力板连接 TNT，破坏压力板即可安全取宝
- **资源**: 陶瓦、砂岩

#### 丛林神殿 / Jungle Pyramid

- **生物群系**: 丛林
- **陷阱 / Trap**: 发射器射出箭矢
- **宝藏 / Treasure**: 两个箱子，内含钻石、铁锭等
- **谜题 / Puzzle**: 拉杆组合打开隐藏房间

#### 沼泽小屋 / Swamp Hut

- **生物群系**: 沼泽
- **居民**: 女巫和一只黑猫
- **战利品**: 酿造台、红石粉、药水材料

#### 林地府邸 / Woodland Mansion

- **生物群系**: 黑森林
- **规模**: 大型多层建筑
- **居民**: 卫道士、唤魔者、恼鬼
- **战利品**: 钻石、金苹果、附魔书、不死图腾

#### 海底神殿 / Ocean Monument

- **生物群系**: 深海
- **居民**: 守卫者、远古守卫者
- **战利品**: 海绵、海晶灯、金块
- **挑战**: 挖掘疲劳效果，需要牛奶解除

#### 沉船 / Shipwreck

- **生物群系**: 所有海洋
- **战利品**: 藏宝图、食物、铁锭
- **结构**: 船头、船身、船尾各有箱子

#### 埋藏的宝藏 / Buried Treasure

- **如何找到**: 使用沉船中的藏宝图
- **标记**: 地图上的红叉
- **宝藏**: 海洋之心、铁锭、钻石、心形旗帜图案

#### 试炼密室 / Trial Chambers

- **生物群系**: 地下（Y -40 到 -20）
- **新增**: 1.21 版本
- **特色**: 试炼刷怪笼（Trial Spawner）、试炼钥匙、重锤（Mace）
- **战利品**: 1.21 新武器和装备

### 下界 / The Nether

下界是地狱般的维度，包含独特的资源和生物。

The Nether is a hell-like dimension with unique resources and mobs.

#### 如何进入 / How to Enter

- 建造下界传送门（Obsidian 框架，最低 4x5）
- 用打火石激活传送门
- 至少需要 10 个黑曜石

#### 下界资源 / Nether Resources

| 资源 / Resource | 位置 / Location | 用途 / Use |
|---|---|---|
| 下界石英 / Nether Quartz | 下界荒地 | 比较器、探测铁轨 |
| 下界金矿 / Nether Gold Ore | 下界荒地 | 金粒 |
| 远古残骸 / Ancient Debris | Y 8-22 | 下界合金碎片 |
| 岩浆块 / Magma Block | 下界各群系 | 岩浆膏、气泡柱 |
| 发光墨囊 / Glow Ink Sac | 发光鱿鱼 | 发光物品展示框 |
| 下界疣 / Nether Wart | 下界要塞 | 酿造基础材料 |

#### 下界要塞 / Nether Fortress

- **居民**: 烈焰人、凋零骷髅、骷髅
- **战利品**: 烈焰棒、凋零骷髅头颅、下界疣
- **重要性**: 获得烈焰棒才能去末地

#### 堡垒遗迹 / Bastion Remnant

- **居民**: 猪灵、猪灵蛮兵
- **战利品**: 金块、钻石、下界合金装备
- **注意**: 不要打开金箱子，会激怒猪灵

#### 下界交通 / Nether Travel

- **1 格下界 = 8 格主世界**
- 下界快速旅行是高效的长距离移动方式
- 搭建冰道或蓝冰道可以极速旅行

### 末地 / The End

末地是游戏的最终维度，击败末影龙即可通关。

The End is the final dimension. Defeat the Ender Dragon to beat the game.

#### 如何进入 / How to Enter

1. 击杀烈焰人获得烈焰棒
2. 用烈焰棒合成烈焰粉
3. 烈焰粉 + 末影珍珠 = 末影之眼
4. 投掷末影之眼寻找要塞
5. 在要塞中激活末地传送门

#### 击败末影龙 / Defeating the Ender Dragon

**准备物品 / Recommended Items**:

- 钻石或下界合金剑
- 弓（力量附魔）和大量箭矢
- 大量金苹果
- 抗火药水
- 缓落药水或水桶（防止摔落）
- 床（在末地会爆炸，用于快速炸水晶和龙）

**战斗策略 / Battle Strategy**:

1. 先摧毁所有末影水晶（阻止龙回血）
2. 用水晶塔上的铁栏杆爬上去
3. 用水桶或梯子安全返回地面
4. 箭射末影龙或等待它俯冲时用剑攻击
5. 龙死亡后获得龙头和经验值

#### 末地外岛 / Outer End Islands

- **如何到达**: 通过击败末影龙后生成的末地折跃门
- **主要建筑**: 末地船、末地城
- **战利品**: 鞘翅、龙头、钻石装备、附魔书
- **生物**: 潜影贝（可掉落潜影壳，制作潜影盒）

#### 末地城与末地船 / End City & End Ship

| 结构 / Structure | 战利品 / Loot | 特点 / Feature |
|---|---|---|
| 末地城 / End City | 钻石装备、附魔书、铁块 | 多层塔楼，有潜影贝守卫 |
| 末地船 / End Ship | 鞘翅、龙头、钻石块 | 漂浮在末地城旁，需要搭桥过去 |

### 探索技巧 / Exploration Tips

- **标记路径 / Mark Your Path** — 用火把、旗帜或不同颜色的方块标记已探索区域
- **带上床 / Bring a Bed** — 在未探索区域设置重生点
- **准备足够食物 / Stock Food** — 长途旅行消耗大量饱食度
- **使用地图 / Use Maps** — 制图台可放大和复制地图
- **鞘翅飞行 / Elytra Flight** — 使用烟花火箭推进鞘翅，是最快的长距离移动方式
- **下界公路 / Nether Highway** — 在下界搭建高效交通网络

### 常用探险指令 / Useful Exploration Commands

| 指令 / Command | 用途 / Use |
|---|---|
| `/locate structure minecraft:village` | 定位最近的村庄 |
| `/locate biome minecraft:desert` | 定位最近的沙漠群系 |
| `/seed` | 查看当前世界种子 |
| `/gamerule keepInventory true` | 死亡不掉落物品 |
| `/time set day` | 设为白天 |
