## 欢迎来到我的世界 / Welcome to Minecraft

我的世界（Minecraft）是一款沙盒建造游戏，由 Mojang Studios 开发。玩家可以在三维世界中自由建造、探索、战斗和生存。本指南涵盖 Java 版和基岩版（Bedrock Edition）的核心玩法，适合所有水平的玩家。

Minecraft is a sandbox game developed by Mojang Studios. Players can freely build, explore, fight, and survive in a 3D world. This guide covers both Java Edition and Bedrock Edition, suitable for all skill levels.

### 游戏版本 / Game Editions

| 版本 / Edition | 平台 / Platform | 特点 / Features |
|---|---|---|
| Java 版 / Java Edition | Windows, Mac, Linux | 模组支持丰富，抢先体验新内容 |
| 基岩版 / Bedrock Edition | 移动端、主机、Win10 | 跨平台联机，性能优化更好 |
| 教育版 / Education Edition | 多平台 | 用于教学，包含化学等特殊功能 |

### 游戏模式 / Game Modes

- **生存模式 / Survival** — 收集资源、维持生命、对抗怪物
- **创造模式 / Creative** — 无限资源、飞行能力、自由建造
- **冒险模式 / Adventure** — 自定义地图，受限交互
- **旁观模式 / Spectator** — 穿墙飞行，无法交互

### 游戏特色 / Key Features

- **无限世界 / Infinite World** — 程序化生成，每次开档都独一无二
- **红石电路 / Redstone Circuitry** — 模拟真实电路，实现自动化
- **多元维度 / Multiple Dimensions** — 主世界、下界（Nether）、末地（The End）
- **跨平台联机 / Cross-platform Play** — 基岩版支持手机/平板/主机/Win10 互通
- **模组生态 / Mod Ecosystem** — Java 版拥有全球最大的模组社区
