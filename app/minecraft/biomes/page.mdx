## 生物群系 / Biomes

我的世界拥有超过 60 种生物群系，每个群系都有独特的地形、生物和资源。

Minecraft has over 60 biomes, each with unique terrain, mobs, and resources.

### 温暖群系 / Warm Biomes

#### 平原 / Plains

- **地形 / Terrain**: 平坦开阔，草色鲜绿
- **常见生物 / Mobs**: 牛、羊、猪、鸡、马
- **资源 / Resources**: 橡树、草、花、村庄
- **变种 / Variants**: 向日葵平原（Sunflower Plains）

#### 沙漠 / Desert

- **地形 / Terrain**: 沙地广阔，有仙人掌和枯死灌木
- **常见生物 / Mobs**: 兔子、僵尸村民（村庄）
- **资源 / Resources**: 沙子、砂岩、仙人掌、沙漠水井、沙漠神殿
- **特点 / Feature**: 从来不下雨

#### 热带草原 / Savanna

- **地形 / Terrain**: 台地起伏，金合欢树散布
- **常见生物 / Mobs**: 牛、羊、马、鸡
- **资源 / Resources**: 金合欢木、村庄
- **变种 / Variants**: 高原热带草原（Savanna Plateau）、被风蚀的热带草原（Windswept Savanna）

#### 恶地 / Badlands

- **地形 / Terrain**: 红色和橙色陶瓦层叠，地表裸露
- **常见生物 / Mobs**: 金色兔子
- **资源 / Resources**: 陶瓦、红沙、金矿（露天）、废弃矿井
- **变种 / Variants**: 被风蚀的恶地（Eroded Badlands）、恶地高原（Badlands Plateau）

#### 丛林 / Jungle

- **地形 / Terrain**: 茂密高树，地形崎岖
- **常见生物 / Mobs**: 鹦鹉、豹猫、熊猫、鸡
- **资源 / Resources**: 可可豆、丛林原木、竹子、西瓜、丛林神殿
- **变种 / Variants**: 丛林边缘（Jungle Edge）、竹林（Bamboo Jungle）

#### 樱花群系 / Cherry Grove

- **地形 / Terrain**: 粉色樱花树散布于山丘
- **常见生物 / Mobs**: 牛、羊
- **资源 / Resources**: 樱花原木、粉色花瓣
- **特点 / Feature**: 1.20 新增，地面覆盖粉红色粒子

### 温带群系 / Temperate Biomes

#### 森林 / Forest

- **地形 / Terrain**: 密集的橡树和桦树
- **常见生物 / Mobs**: 狼、牛、羊、猪、鸡
- **资源 / Resources**: 橡木、桦木、花、蘑菇
- **变种 / Variants**: 繁花森林（Flower Forest）、桦木森林（Birch Forest）

#### 针叶林 / Taiga

- **地形 / Terrain**: 云杉林覆盖，常有狼群
- **常见生物 / Mobs**: 狼、狐狸、牛、羊、鸡
- **资源 / Resources**: 云杉木、甜浆果、村庄
- **变种 / Variants**: 巨型针叶林（Giant Tree Taiga）、雪针叶林（Snowy Taiga）

#### 黑森林 / Dark Forest

- **地形 / Terrain**: 浓密黑暗橡树，光照低
- **常见生物 / Mobs**: 敌对生物生成率高
- **资源 / Resources**: 深色橡木、蘑菇、林地府邸
- **特点 / Feature**: 白天也有怪物生成

#### 沼泽 / Swamp

- **地形 / Terrain**: 浅水区、藤蔓垂挂、黏土块
- **常见生物 / Mobs**: 史莱姆（夜间）、女巫小屋
- **资源 / Resources**: 橡木、藤蔓、蓝莓、睡莲、黏土
- **特点 / Feature**: 水位低，能见度差

### 寒冷群系 / Cold Biomes

#### 雪原 / Snowy Plains

- **地形 / Terrain**: 冰雪覆盖的荒原
- **常见生物 / Mobs**: 兔子、流浪者（骷髅变种）
- **资源 / Resources**: 雪、冰、北极熊（冰刺平原）
- **变种 / Variants**: 冰刺平原（Ice Spikes）、冰洋（Frozen Ocean）

#### 积雪针叶林 / Snowy Taiga

- **地形 / Terrain**: 雪覆针叶林
- **常见生物 / Mobs**: 狼、狐狸、北极熊
- **资源 / Resources**: 云杉木、甜浆果
- **特点 / Feature**: 降雪覆盖所有方块

#### 冰洋 / Frozen Ocean

- **地形 / Terrain**: 冰封海面
- **常见生物 / Mobs**: 北极熊、流浪者
- **资源 / Resources**: 冰、蓝冰

### 山地群系 / Mountain Biomes

#### 风袭丘陵 / Windswept Hills

- **地形 / Terrain**: 陡峭山壁、暴露岩石
- **常见生物 / Mobs**: 羊、山羊
- **资源 / Resources**: 煤矿、铁矿、绿宝石矿
- **变种 / Variants**: 风袭沙砾丘陵（Windswept Gravelly Hills）

#### 草甸 / Meadow

- **地形 / Terrain**: 鲜花盛开的绿色山地
- **常见生物 / Mobs**: 牛、羊、兔子、驴
- **资源 / Resources**: 多种花卉

#### 樱花山丘 / Cherry Hills

- **地形 / Terrain**: 1.20 新增，樱花树覆盖的缓坡
- **常见生物 / Mobs**: 被动生物

### 生物群系搜索技巧 / Biome Finding Tips

- **使用 /locate** — 输入 `/locate biome minecraft:desert` 快速定位
- **地图制作 / Mapping** — 用制图台和纸张制作探险地图
- **从高处观察 / High Ground** — 爬到山顶或高塔上俯瞰周围
- **沿河流走 / Follow Rivers** — 河流是自然通道，通向不同群系
- **鞘翅飞行 / Elytra Flight** — 获得鞘翅后从高空快速探索
