## 生存模式 / Survival Mode

生存模式是我的世界的核心玩法。你需要管理生命值与饱食度，建造庇护所，采集资源，并击败怪物。

Survival mode is the core of Minecraft. You must manage health and hunger, build shelters, gather resources, and fight monsters.

### 生命与饱食度 / Health and Hunger

#### 生命值 / Health

- 玩家有 20 点生命值（10 颗心）
- 自然恢复需要饱食度达到 18 以上（9 个鸡腿）
- 恢复速度为每 4 秒 1 点生命值
- 伤害来源包括：怪物攻击、摔落、岩浆、火焰、溺水、饥饿、仙人掌等

#### 饱食度 / Hunger

- 显示为 20 点饱食度（10 个鸡腿）
- 疾跑、跳跃、挖掘都会消耗饱食度
- 不同食物恢复的饱食度不同
- 饱食度归零后会开始消耗饥饿值，并持续受到伤害

| 食物 / Food | 饱食度恢复 / Hunger Restored | 饱食饱和度 / Saturation |
|---|---|---|
| 牛排 / Steak | 8 | 12.8 |
| 熟猪排 / Cooked Porkchop | 8 | 12.8 |
| 面包 / Bread | 5 | 6 |
| 金苹果 / Golden Apple | 4 | 9.6 |
| 烤土豆 / Baked Potato | 5 | 6 |
| 胡萝卜 / Carrot | 3 | 4.8 |
| 甜浆果 / Sweet Berries | 2 | 1.2 |
| 腐肉 / Rotten Flesh | 4 | 0.8 |

### 庇护所建造 / Shelter Building

#### 临时庇护所 / Temporary Shelter

- 挖山挖洞是最快的方式
- 用泥土或圆石搭建 5x4x3 的小屋
- 安装门（Door）防止怪物进入
- 内部放置工作台、熔炉、床和箱子

#### 永久基地 / Permanent Base

选址要点 / Key factors for base location:

- **靠近水源 / Near Water** — 方便农业和水源无限使用
- **平坦地形 / Flat Terrain** — 减少平整土地的工作量
- **资源丰富 / Resource-rich** — 附近有森林、山脉和洞穴
- **生物群系 / Biome** — 平原或森林最为理想

#### 基地必备设施 / Essential Base Facilities

1. **工作区 / Crafting Area** — 多个工作台、熔炉、箱子
2. **农场 / Farm** — 小麦、胡萝卜、土豆、甜菜
3. **动物围栏 / Animal Pen** — 圈养牛、羊、鸡、猪
4. **仓库 / Storage Room** — 分类存放物品的大箱子
5. **矿道 / Mine Shaft** — 通向地下的安全通道
6. **床铺 / Bedroom** — 设置重生点

### 采矿指南 / Mining Guide

采矿是生存模式中最重要的活动之一。不同矿石分布在不同的高度层。

Mining is one of the most important activities. Different ores are found at different Y-levels.

| 矿石 / Ore | 最佳 Y 层 / Best Y-level | 用途 / Use |
|---|---|---|
| 煤矿 / Coal Ore | 95-136 | 燃料、火把 |
| 铁矿 / Iron Ore | 15-232 | 工具、盔甲、铁砧 |
| 金矿 / Gold Ore | -64-32 | 下界合金、金苹果、动力铁轨 |
| 青金石 / Lapis Lazuli | 0-32 | 附魔、染料 |
| 红石 / Redstone Ore | -64-32 | 红石电路 |
| 钻石 / Diamond Ore | -64-16 | 最强工具和盔甲（仅次于下界合金） |
| 绿宝石 / Emerald Ore | 0-32 | 与村民交易 |
| 铜矿 / Copper Ore | 0-96 | 铜块、避雷针、望远镜 |

#### 采矿技巧 / Mining Tips

- **阶梯式采矿 / Staircase Mining** — 挖 1x2 的阶梯向下，安全且方便返回
- **鱼骨式采矿 / Branch Mining** — 主通道两侧每隔 2 格挖分支，覆盖率最高
- **带足火把 / Bring Torches** — 防止怪物在黑暗中生成
- **带水桶 / Bring Water Bucket** — 掉入岩浆时用水自救
- **标记路线 / Mark Your Path** — 用火把或方块标记已探索区域

#### 自动农田 / Automatic Farm

使用水和活塞可以实现自动收割。水流将作物冲到收集点，再用活塞控制水源开关。

Using water and pistons creates automatic harvesting. Water flows push crops to a collection point.

### 村民交易 / Villager Trading

村民（Villagers）是生存模式中获取稀有物品的重要途径。

Villagers are essential for obtaining rare items in survival.

#### 村民职业 / Villager Professions

| 职业 / Profession | 工作站点 / Workstation | 主要交易物 / Main Trades |
|---|---|---|
| 盔甲匠 / Armorer | 高炉 / Blast Furnace | 钻石盔甲 |
| 工具匠 / Toolsmith | 锻造台 / Smithing Table | 钻石工具 |
| 武器匠 / Weaponsmith | 磨石 / Grindstone | 钻石剑 |
| 图书管理员 / Librarian | 讲台 / Lectern | 附魔书 |
| 牧师 / Cleric | 酿造台 / Brewing Stand | 红石粉、荧石粉 |
| 农民 / Farmer | 堆肥桶 / Composter | 食物、绿宝石 |
| 渔夫 / Fisherman | 木桶 / Barrel | 鱼类、附魔钓鱼竿 |

#### 交易技巧 / Trading Tips

- **治愈僵尸村民 / Cure Zombie Villager** — 用虚弱药水加金苹果可以治愈，获得大幅折扣
- **刷新交易 / Refresh Trades** — 破坏并重新放置工作站可重置村民职业
- **绿宝石农场 / Emerald Farm** — 与农民交易农作物获取绿宝石，是最稳定的经济来源
