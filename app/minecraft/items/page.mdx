### 信标效果 / Beacon Effects

信标（Beacon）是一个提供区域增益效果的金字塔结构。

A beacon provides area buff effects using a pyramid structure.

#### 信标等级 / Beacon Tiers

| 层数 / Layers | 方块数量 / Blocks | 范围 / Range | 可用效果 / Available Effects |
|---|---|---|---|
| 1 层 | 9 | 20 格 | 速度、急迫 |
| 2 层 | 34 | 30 格 | + 抗性提升、跳跃提升 |
| 3 层 | 83 | 40 格 | + 力量 |
| 4 层 | 164 | 50 格 | + 二级效果 |

#### 信标效果列表 / Beacon Effects

| 效果 / Effect | 效果描述 / Description |
|---|---|
| 速度 I-II / Speed I-II | 增加移动速度 |
| 急迫 I-II / Haste I-II | 增加挖掘速度 |
| 抗性提升 I-II / Resistance I-II | 减少受到的伤害 |
| 跳跃提升 I-II / Jump Boost I-II | 增加跳跃高度 |
| 力量 I-II / Strength I-II | 增加近战攻击伤害 |
| 再生 I-II / Regeneration I-II | 持续恢复生命 |

#### 信标材料 / Beacon Materials

- **铁锭 / Iron Ingot** — 最常见且易获取
- **金锭 / Gold Ingot** — 效率高，中量获取
- **绿宝石 / Emerald** — 可与村民交易获得
- **钻石 / Diamond** — 最稀有，装饰价值高
- **下界合金锭 / Netherite Ingot** — 最高级但非常昂贵
