## 战斗与装备 / Combat & Gear

我的世界中的战斗系统涉及多种武器、盔甲和状态效果。掌握这些知识将帮助你在冒险中生存。

Minecraft's combat system involves various weapons, armor, and status effects. Mastering these will help you survive.

### 武器系统 / Weapons

#### 剑 / Sword

剑是近战主力武器，攻击范围较近但伤害稳定。

| 材质 / Material | 伤害 / Damage | 耐久度 / Durability | 附魔能力 / Enchantability |
|---|---|---|---|
| 木剑 / Wooden | 4 | 59 | 15 |
| 石剑 / Stone | 5 | 131 | 5 |
| 铁剑 / Iron | 6 | 250 | 14 |
| 金剑 / Golden | 4 | 32 | 22 |
| 钻石剑 / Diamond | 7 | 1561 | 10 |
| 下界合金剑 / Netherite | 8 | 2031 | 15 |

#### 斧 / Axe

斧头伤害高于剑，但攻速较慢，适合一击脱离战术。

Axes deal more damage than swords but attack slower, suitable for hit-and-run tactics.

| 材质 / Material | 伤害 / Damage | 攻速 / Attack Speed |
|---|---|---|
| 木斧 / Wooden | 7 | 0.8 |
| 石斧 / Stone | 9 | 0.8 |
| 铁斧 / Iron | 9 | 0.9 |
| 钻石斧 / Diamond | 9 | 1.0 |
| 下界合金斧 / Netherite | 10 | 1.0 |

#### 弓 / Bow

弓是远程武器，需要箭矢。拉满弦时伤害最高。

The bow is a ranged weapon requiring arrows. Full draw deals maximum damage.

- **伤害 / Damage**: 1-10（拉满时 / At full draw）
- **射速 / Fire Rate**: 约 0.5 秒拉满 / About 0.5s to full draw
- **射程 / Range**: 约 50 格 / About 50 blocks

| 附魔 / Enchantment | 效果 / Effect | 最高等级 / Max Level |
|---|---|---|
| 力量 / Power | 增加 25% 伤害 | V |
| 冲击 / Punch | 击退敌人 | II |
| 火矢 / Flame | 点燃目标 | I |
| 无限 / Infinity | 射箭不消耗箭矢 | I |

#### 弩 / Crossbow

弩是蓄力型远程武器，可装填烟花火箭或多种箭矢。

The crossbow is a charged ranged weapon that can load fireworks or multiple arrows.

- **蓄力时间 / Charge Time**: 1.5 秒
- **伤害 / Damage**: 6-11（取决于附魔）
- **特殊附魔 / Special Enchantments**: 快速装填（Quick Charge）、多重射击（Multishot）、穿透（Piercing）

#### 三叉戟 / Trident

三叉戟可从溺尸（Drowned）身上获得，是唯一同时支持近战和远程的武器。

The trident is obtained from drowned zombies, the only weapon supporting both melee and ranged.

| 附魔 / Enchantment | 效果 / Effect |
|---|---|
| 忠诚 / Loyalty | 投掷后自动返回 |
| 引雷 / Channeling | 雷暴时召唤闪电 |
| 激流 / Riptide | 雨天或水中时向前冲刺 |
| 穿刺 / Impaling | 对水生生物造成额外伤害 |

### 附魔系统 / Enchanting

附魔台（Enchanting Table）需要书架来提高附魔等级。

The enchanting table needs bookshelves to increase enchantment levels.

#### 附魔要求 / Requirements

- **附魔台 + 15 个书架** 达到最高 30 级附魔
- 书架离附魔台一格，中间可空一格
- 使用青金石（Lapis Lazuli）作为附魔材料

### 战斗技巧 / Combat Tips

- **盾牌格挡 / Shield Block** — 右键举起盾牌，可挡住 100% 正面近战和远程伤害
- **暴击跳跃 / Critical Hit** — 下落时攻击造成 150% 伤害
- **走位 / Strafe** — 左右移动躲避骷髅的箭矢
- **利用地形 / Use Terrain** — 高地优势增加攻击距离和视野
- **冷却机制 / Cooldown (Java)** — Java 版 1.9 后武器有攻击冷却，满条攻击伤害最高
