## 新手入门 / Beginner Guide

欢迎来到我的世界！本章将帮助你从零开始，掌握游戏的基础知识与生存技巧。

Welcome to Minecraft. This chapter will help you start from scratch and master the basics.

### 首次进入世界 / First 10 Minutes

当你创建一个新世界后，你会随机出生在某个生物群系中。首先确认周围环境：

When you create a new world, you spawn in a random biome. First, assess your surroundings:

1. **查看树木 / Look for Trees** — 木头是初期最重要的资源。用手或斧头砍伐至少 4 个原木（Log）。
2. **制作木板 / Craft Planks** — 打开物品栏（E 键），将原木合成为木板（Planks）。
3. **制作工作台 / Craft a Crafting Table** — 用 4 个木板合成工作台，放在地上。
4. **制作木镐 / Craft a Wooden Pickaxe** — 用 3 个木板和 2 个木棍合成木镐。
5. **采集石头 / Mine Stone** — 用木镐挖掘石头，获得圆石（Cobblestone）。
6. **制作石制工具 / Upgrade to Stone Tools** — 用圆石和木棍制作石镐、石斧、石剑。

### 第一天生存 / Day 1 Survival

你的第一个白天只有 10 分钟。按优先级完成以下任务：

Your first day is only 10 minutes. Prioritize these tasks:

- **收集木头 / Gather Wood** — 至少 8-12 个原木
- **制作工具 / Craft Tools** — 工作台、木镐、石镐、石剑
- **寻找动物 / Find Animals** — 杀羊（获得羊毛做床）、牛（获得牛肉和皮革）
- **收集食物 / Collect Food** — 杀动物获得生肉，在熔炉中烤熟
- **寻找庇护所 / Find Shelter** — 小山丘、洞穴入口，或自己搭建一个临时小屋

#### 如何制作床 / How to Make a Bed

床（Bed）是你跳过夜晚的关键。需要 3 个羊毛和 3 个木板：

- 找到至少 3 只羊（Sheep），用剑击杀获得羊毛
- 在工作台上将羊毛与木板合成床
- 夜晚右键点击床即可睡觉，跳过危险的黑夜

### 第一天夜间 / Surviving the First Night

如果没来得及做床，你需要找一个安全的地方躲起来：

If you cannot make a bed, find a safe place to hide:

- **挖洞 / Dig a Hole** — 在地上挖一个 1x2 的坑，跳进去后用方块封住顶部
- **封闭洞穴 / Block Cave Entrance** — 用泥土或圆石堵住洞口
- **不要乱跑 / Do Not Wander** — 夜间野外有僵尸（Zombie）、骷髅（Skeleton）、爬行者（Creeper）等怪物

### 第一个工具升级 / First Tool Upgrades

| 工具 / Tool | 材料 / Material | 用途 / Use |
|---|---|---|
| 石剑 / Stone Sword | 圆石 x2, 木棍 x1 | 近战攻击，比木剑伤害更高 |
| 石镐 / Stone Pickaxe | 圆石 x3, 木棍 x2 | 挖掘石头和铁矿石 |
| 石斧 / Stone Axe | 圆石 x3, 木棍 x2 | 快速砍树 |
| 石锹 / Stone Shovel | 圆石 x1, 木棍 x2 | 快速挖掘泥土、沙子、沙砾 |

### 基础合成表 / Basic Recipes

#### 工作台 / Crafting Table

4 个木板围成方形，放在工作台格子的 2x2 区域。

Place 4 planks in a 2x2 square in the inventory crafting grid.

#### 木棍 / Sticks

2 个木板垂直放置，产出 4 个木棍。

Place 2 planks vertically, yields 4 sticks.

#### 熔炉 / Furnace

8 个圆石围成一圈（留空中间），在工作台上合成。

Place 8 cobblestone in a ring (leave center empty) on the crafting table.

### 实用技巧 / Useful Tips

- **按住左键 / Hold Left-click** — 连续挖掘或攻击，比单次点击效率更高
- **Shift + 点击 / Shift-click** — 在箱子和工作台界面中快速移动物品
- **F3 键 / F3 Key** — Java 版调试画面，显示坐标 FPS 和生物群系信息
- **床的爆炸 / Bed Explosion** — 床只能在下界和末地以外使用，否则会剧烈爆炸
- **初始地图 / Starter Map** — 基岩版出生时自带一张空地地图，手持即可填充

### 常见错误 / Common Mistakes

- **用手挖石头 / Mining Stone with Fists** — 手挖石头不会掉落任何东西，必须用镐
- **不建庇护所 / Skipping Shelter** — 第一夜不找地方躲藏是新手最常见的死因
- **饿死 / Starving** — 饱食度低时无法疾跑和恢复生命，一定要随身携带食物
- **挖脚下 / Digging Straight Down** — 正下方可能是岩浆或洞穴，极其危险
- **在木屋点火 / Lighting Wooden House on Fire** — 打火石或岩浆会点燃木质建筑
