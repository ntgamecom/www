# 装备与道具 / Items

装备和道具系统为漫威争锋增添了额外的战略深度。本章详细介绍所有装备效果、团队技能以及搭配策略。

The item and equipment system adds extra strategic depth to Marvel Rivals. This chapter covers all item effects, team-up abilities, and loadout strategies.

## 装备系统概述 / Item System Overview

装备可以在比赛开始前配置，影响英雄的战斗表现。每场比赛最多携带2件装备。

Items can be configured before a match starts and affect hero combat performance. Each match allows up to 2 items.

|:--|:--|
| 装备1 / Item 1 | 主要装备 / Primary Item |
| 装备2 / Item 2 | 辅助装备 / Secondary Item |

## 装备搭配策略 / Itemization Strategy

### 新手推荐配置 / Beginner Builds

| 定位 / Role | 装备组合 / Item Combo |
|:--|:--|
| 坦克 / Vanguard | 生命护符 + 钢铁皮肤 / Life Talisman + Iron Skin |

| 辅助 / Strategist | 治疗增幅器 + 疾跑靴 / Healing Amp + Swift Boots |

### 进阶配置思路 / Advanced Build Concepts

- **对抗物理输出**：钢铁皮肤 + 生命护符 / vs Physical: Iron Skin + Life Talisman
- **对抗技能爆发**：护盾发生器 + 生命护符 / vs Burst: Shield Generator + Life Talisman
- **最大化治疗**：治疗增幅器 + 能量强化器 / Max Healing: Healing Amp + Energy Amplifier
- **刺客打法**：力量宝石 + 精准瞄准镜 / Assassin Build: Power Gem + Precision Scope
