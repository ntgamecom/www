# 排位竞技 / Ranked

排位赛是漫威争锋最具挑战性的游戏模式。本章详细介绍排位系统、段位分布、冲分技巧和终结技禁用策略。

Ranked play is the most challenging game mode in Marvel Rivals. This chapter covers the ranked system, tier distribution, climbing tips, and ban phase strategies.

## 排位系统 / Ranked System

### 段位体系 / Tier System

漫威争锋采用标准的段位体系，从低到高依次为：

Marvel Rivals uses a standard tier system, from lowest to highest:

| 段位 / Tier | 中文 / Chinese | 说明 / Description |
|:--|:--|:--|
| Bronze | 青铜 | 初始段位 / Starting tier |
| Silver | 白银 | 初级玩家 / Beginner players |
| Gold | 黄金 | 中等水平 / Intermediate |
| Platinum | 铂金 | 高级玩家 / Advanced |
| Diamond | 钻石 | 精英玩家 / Elite |
| Master | 大师 | 顶尖玩家 / Top players |
| Grandmaster | 宗师 | 最高段位 / Highest tier |

每个段位分为5个小段（Bronze I-V, Silver I-V等）。

Each tier is divided into 5 sub-tiers (Bronze I-V, Silver I-V, etc.).

### 排位点数 / Rank Points (RP)

| 行为 / Action | RP变化 / RP Change |
|:--|:--|
| 胜利 / Win | +20 ~ +25 RP |
| 失败 / Loss | -15 ~ -20 RP |
| 平局 / Draw | 0 ~ +5 RP |
| 连胜奖励 / Win Streak Bonus | 额外 +3 ~ +5 RP |
| 表现优异 / Performance Bonus | 额外 +1 ~ +3 RP |

### 定级赛 / Placement Matches

首次进入排位需要完成10场定级赛。定级赛结果决定初始段位。

First-time ranked players must complete 10 placement matches. Results determine initial tier.

## 冲分技巧 / Climbing Tips

### 心态管理 / Mindset

- **保持积极 / Stay Positive**: 消极心态会直接影响发挥 / Negative attitude directly impacts performance
- **接受失败 / Accept Losses**: 连败时暂停一下，避免连跪心态 / Take breaks during loss streaks
- **专注自身 / Focus on Yourself**: 不要责怪队友，想想自己能做什么 / Do not blame teammates, focus on self-improvement
- **目标导向 / Goal Oriented**: 每天设定小目标，稳步提升 / Set small daily goals for steady improvement

### 英雄池 / Hero Pool

| 建议 / Suggestion | 说明 / Explanation |
|:--|:--|
| 主玩2-3位英雄 / Main 2-3 heroes | 专精比广度更重要 / Depth beats breadth |
| 每类定位至少会1位 / At least 1 per role | 适应阵容需要 / Adapt to team needs |
| 关注版本强势英雄 / Play meta heroes | 版本之子胜率更高 / Meta heroes have higher win rates |

### 游戏内决策 / In-Game Decisions

**进攻时机 / When to Engage**:

- 等待全队集合完毕 / Wait for full team regroup
- 确认所有关键技能不在冷却 / Confirm key abilities are off cooldown
- 利用人数优势（多打少）/ Exploit numbers advantage (outnumber enemy)
- 在敌方终极技能用完时进攻 / Attack when enemy ultimates are down

**撤退时机 / When to Retreat**:

- 己方有人被击杀 / A teammate is eliminated
- 关键技能进入冷却 / Key abilities are on cooldown
- 敌方使用终结技 / Enemy uses ultimate
- 血量低于安全线 / HP is critically low

### 数据分析 / Data Analysis

- 关注自己的比赛数据统计 / Review your match statistics
- 找出胜率和败局的原因 / Identify reasons for wins and losses
- 对比自己和顶尖玩家的差异 / Compare with top players on leaderboards
- 使用第三方统计工具辅助分析 / Use third-party tools for analysis

## 禁用阶段指南 / Ban Phase Guide

### 禁用阶段说明 / Ban Phase Overview

在钻石及以上段位，比赛开始前有禁用阶段：

In Diamond and above tiers, there is a ban phase before each match:

1. 双方轮流禁用英雄 / Teams alternate banning heroes
2. 每方有2个禁用名额 / Each team has 2 ban slots
3. 禁用的英雄本场比赛不可使用 / Banned heroes are unavailable for the match

### 禁用策略 / Ban Strategy

**优先禁用的英雄 / Priority Bans**:

| 理由 / Reason | 建议禁用 / Suggested Bans |
|:--|:--|
| 版本最强 / Meta OP | 当前版本胜率最高的英雄 / Highest win rate hero |
| 克制我方阵容 / Counters Our Comp | 对我方策略威胁最大的英雄 / Biggest threat to our strategy |
| 对手专精英雄 / Opponent One-Trick | 对手擅长的主玩英雄 / Opponent's main hero |
| 团队不擅应对 / Team Struggles Against | 团队不擅长对付的英雄 / Hero your team cannot handle |

**禁用注意事项 / Ban Considerations**:

- 不要浪费禁用名额在没人玩的英雄上 / Do not waste bans on unpopular heroes
- 与队友沟通禁用计划 / Communicate ban plan with team
- 考虑预留强势英雄为己方所用 / Consider leaving strong picks for your team
- 观察敌方过往对局禁用模式 / Observe enemy past ban patterns

## 排位常见问题 / Ranked FAQ

**Q**: 什么段位可以开始打排位？/ What level do I need for ranked?
**A**: 完成新手教程并达到指定等级即可 / Complete tutorial and reach required level.

**Q**: 排位赛季持续多久？/ How long is a ranked season?
**A**: 通常为2-3个月 / Typically 2-3 months.

**Q**: 赛季末段位会重置吗？/ Does rank reset at season end?
**A**: 会，但新赛季的定级赛会参考上赛季段位 / Yes, but placements consider previous season rank.
