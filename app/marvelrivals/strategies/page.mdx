# 战术策略 / Strategies

高级战术是区分高手和普通玩家的关键。本章涵盖阵容策略、进攻防守技巧、沟通方法和英雄克制关系。

Advanced tactics separate great players from good ones. This chapter covers composition strategies, attack/defense techniques, communication methods, and counter-picking.

## 阵容体系 / Team Systems

### 推进体系 / Brawl Comp

以坦克为核心缓慢推进，依靠持续伤害和回复消耗对手。

Slow push centered around tanks, relying on sustained damage and sustain.

| 优势 / Pros | 劣势 / Cons |
|:--|:--|
| 正面战斗力强 / Strong frontline | 机动性差 / Low mobility |
| 容错率高 / High error tolerance | 容易被绕后 / Vulnerable to flanks |
| 控制点多 / Lots of CC | 依赖团队配合 / Relies on team coordination |

**推荐英雄 / Recommended Heroes**:
- 坦克 / Tank: 绿巨人 Hulk, 美国队长 Captain America
- 输出 / DPS: 钢铁侠 Iron Man, 雷神索尔 Thor
- 辅助 / Support: 火箭浣熊 Rocket, 奇异博士 Dr Strange

### 放血体系 / Dive Comp

利用高机动英雄快速切入敌方后排，击杀关键目标后撤离。

Use high-mobility heroes to quickly dive the enemy backline, eliminate key targets, then disengage.

| 优势 / Pros | 劣势 / Cons |
|:--|:--|
| 爆发伤害高 / High burst | 容错率低 / Low error tolerance |
| 打乱敌方阵型 / Disrupts enemy formation | 害怕控制技能 / Vulnerable to CC |
| 快速击杀后排 / Fast backline kills | 推进能力弱 / Weak sustained push |

**推荐英雄 / Recommended Heroes**:
- 坦克 / Tank: 雷神索尔 Thor（进攻型坦克 / Offensive tank）
- 输出 / DPS: 蜘蛛侠 Spider-Man, 黑豹 Black Panther
- 辅助 / Support: 风暴女 Storm（加速增益 / Speed buffs）

### 双辅体系 / Double Support Comp

两个辅助互相配合，提供极强的持续作战能力。

Two supports working together to provide maximum sustain.

| 优势 / Pros | 劣势 / Cons |
|:--|:--|
| 治疗量极高 / Massive healing | 输出不足 / Lacks damage |
| 持续作战强 / Strong sustain | 推进速度慢 / Slow pushes |
| 反制爆发阵容 / Counters burst comps | 需要坦克非常坚韧 / Requires very durable tanks |

## 进攻策略 / Attack Strategies

### 正面突破 / Frontal Assault

集中全部力量从一个方向推进。

Concentrate all force and push from one direction.

适用于 / Best for:
- 当己方阵容正面更强时 / When your comp has stronger frontline
- 狭窄地图通道 / Narrow map corridors
- 时间紧迫需要快速推进 / Time pressure requires fast push

### 分兵包抄 / Split Push

将队伍分成两组，从不同方向进攻。

Split the team into two groups attacking from different directions.

适用于 / Best for:
- 敌方阵容机动性差 / Enemy comp has low mobility
- 开阔地图 / Open maps
- 牵制敌方资源 / Splitting enemy attention

### 声东击西 / Fake Rotation

佯攻一个方向，实际从另一侧发起真正进攻。

Fake an attack in one direction, then strike from another side.

执行方式 / Execution:
1. 部分队员在一个方向施压 / Some players apply pressure in one direction
2. 主力悄悄转移到另一侧 / Main force rotates to the other side
3. 同时发起进攻 / Attack simultaneously

## 防守策略 / Defense Strategies

### 分层防守 / Layered Defense

在不同位置设置多条防线，延缓敌方推进。

Set up multiple defensive lines at different positions to delay enemy push.

| 防线 / Line | 位置 / Position | 目标 / Goal |
|:--|:--|:--|
| 第一线 / First | 前线接触点 / Front contact | 消耗对手资源 / Drain enemy resources |
| 第二线 / Second | 关键拐角 / Key corner | 利用掩体防守 / Defend from cover |
| 第三线 / Third | 目标点附近 / Near objective | 死守目标 / Hold objective |

### 拖延战术 / Stall Tactics

在不利情况下通过拖延时间等待队友复活。

Delay enemy progress while waiting for teammates to respawn.

技巧 / Tips:
- 利用地图掩体周旋 / Use map cover to stall
- 不要主动求战 / Do not actively engage
- 保留逃生技能 / Keep escape abilities ready
- 呼叫队友尽快赶来 / Call for teammates to regroup

### 反制绕后 / Anti-Flank

针对敌方刺客和侧翼进攻的防守策略。

Defensive strategy against enemy flankers and assassins.

- 后排治疗者互相照看 / Supports watch each other's backs
- 输出英雄时刻留意侧面 / DPS heroes watch flanks
- 坦克适时回防保护后排 / Tanks rotate back to peel
- 利用视野装备侦察侧翼 / Use vision items to scout flanks

## 团队沟通 / Team Communication

### 信息通报 / Information Calls

| 信息类型 / Type | 示例 / Example |
|:--|:--|
| 位置 / Location | "蜘蛛侠在左侧屋顶" / Spider-Man on left roof |
| 技能状态 / Ability Status | "雷神没有大招" / Thor no ultimate |
| 目标 / Objective | "先去占点" / Go capture point first |
| 请求支援 / Request Help | "我需要治疗" / I need healing |
| 集火标记 / Focus Target | "先杀火箭浣熊" / Kill Rocket first |

### 有效沟通原则 / Effective Communication

- **简洁明了 / Keep It Short**: 一句话传达关键信息 / One sentence for key info
- **保持冷静 / Stay Calm**: 情绪化沟通影响团队士气 / Emotional comms hurt team morale
- **正面反馈 / Positive Reinforcement**: 称赞队友的好操作 / Acknowledge good plays
- **建设性建议 / Constructive Feedback**: 不要指责，给出解决方案 / Do not blame, offer solutions

### 信号系统 / Ping System

漫威争锋内置了标记系统用于快速沟通：

Marvel Rivals has a built-in ping system for quick communication:

- 标记位置 / Ping Location: 按住C + 鼠标指向位置 / Hold C + aim at location
- 标记敌人 / Ping Enemy: 瞄准敌人按C / Aim at enemy and press C
- 标记目标 / Ping Objective: 瞄准目标点按C / Aim at objective and press C
- 请求治疗 / Request Heal: 快速按C / Quick press C

### 阵容调整策略 / Comp Adjustment Strategy

**开局阶段 / Early Game**:
1. 选择适合地图的英雄 / Pick map-friendly heroes
2. 确保阵容平衡 / Ensure balanced composition
3. 预留调整空间 / Leave room for adjustment

**中期调整 / Mid-Game Adjustments**:
- 如果被压制，考虑切换英雄 / If being countered, consider swapping
- 观察敌方薄弱环节 / Identify enemy weak points
- 配合队友的阵容调整 / Coordinate swaps with teammates

**后期优化 / Late Game Optimization**:
- 针对敌方核心玩家 / Target the enemy carry player
- 保存关键技能反制敌方大招 / Save key abilities for enemy ults
- 调整站位应对敌方策略 / Adjust positioning against enemy strategy
