# 战斗系统 / Combat

理解漫威争锋的战斗系统是提升实力的关键。本章详细介绍伤害类型、治疗机制、机动性选择和团队配合策略。

Understanding the combat system is key to improving in Marvel Rivals. This chapter covers damage types, healing mechanics, mobility options, and team synergy.

## 伤害类型 / Damage Types

### 基础伤害 / Base Damage

每位英雄的普攻和技能都有特定的伤害数值。伤害受距离、命中部位和护甲影响。

Each hero's basic attacks and abilities have specific damage values. Damage is affected by distance, hit location, and armor.

| 伤害类型 / Type | 说明 / Description |
|:--|:--|
| 物理伤害 / Physical | 受护甲减免 / Reduced by armor |
| 能量伤害 / Energy | 穿透部分护甲 / Partially pierces armor |
| 真实伤害 / True | 无视所有减免 / Ignores all mitigation |
| 持续伤害 / DoT | 随时间造成伤害 / Deals damage over time |

### 暴击机制 / Critical Hits

攻击头部（爆头）可以造成额外伤害。不同英雄的爆头倍率不同：

Attacking the head (headshot) deals bonus damage. Different heroes have different headshot multipliers:

- 狙击型英雄 / Sniper-type heroes: 2.5x
- 步枪型英雄 / Rifle-type heroes: 2.0x
- 近战英雄 / Melee heroes: 1.5x
- 范围技能 / AoE abilities: 无爆头 / No headshot

### 伤害衰减 / Damage Falloff

远程攻击在超过一定距离后伤害会衰减。熟悉英雄的有效射程至关重要。

Ranged attacks suffer damage falloff beyond a certain distance. Knowing your hero's effective range is crucial.

- 近战 / Melee: 无衰减 / No falloff
- 中程 / Mid-range: 20m后开始衰减 / Falloff begins after 20m
- 远程 / Long-range: 40m后衰减至60% / Falls to 60% after 40m

## 治疗机制 / Healing Mechanics

### 治疗来源 / Healing Sources

| 来源 / Source | 类型 / Type | 说明 / Description |
|:--|:--|:--|
| 英雄技能 / Hero Abilities | 主动 / Active | 策略家英雄的治疗技能 / Strategist healing abilities |
| 装备道具 / Items | 主动&被动 / Active & Passive | 治疗装备和回复道具 / Healing items and regen |
| 场景补给 / Map Pickups | 被动 / Passive | 地图上的生命球 / Health orbs on the map |
| 自然回复 / Natural Regen | 被动 / Passive | 脱离战斗后缓慢回血 / Slow regen out of combat |

### 治疗优先级 / Healing Priority

治疗者在团战中应优先治疗：

Healers should prioritize in team fights:

1. 己方坦克 / Your Vanguard (Tank)
2. 核心输出 / Key Duelist (DPS)
3. 另一位治疗 / Other Strategist (Support)
4. 满血队友需要增益 / Full HP allies needing buffs

### 反治疗 / Anti-Heal

部分英雄或装备具有降低治疗效果的能力，称为反治疗或禁疗效果。

Some heroes or items have abilities that reduce healing effectiveness, known as anti-heal.

### 位移技巧 / Positioning Tips

- **高打低 / High Ground Advantage**: 占据高地可以获得更好的视野和掩体 / High ground provides better sightlines and cover
- **侧翼进攻 / Flanking**: 从侧面攻击打乱敌方阵型 / Attack from the side to disrupt enemy formation
- **撤退路线 / Escape Route**: 始终保留一个逃生技能 / Always keep one escape ability ready

## 团队配合 / Team Synergy

### 技能连携 / Ability Combos

有效的技能组合可以产生强大的协同效果：

Effective ability combinations create powerful synergies:

- **控制+输出 / CC + Damage**: 雷神眩晕敌人后，钢铁侠释放终极技能 / Thor stuns, then Iron Man ults
- **护盾+治疗 / Shield + Heal**: 美国队长举盾保护，火箭浣熊在后方治疗 / Cap shields, Rocket heals behind
- **加速+突进 / Speed + Engage**: 风暴女加速全队，绿巨人跳入敌阵 / Storm speeds up team, Hulk jumps in

### 团队搭配原则 / Team Composition Principles

- 至少1位坦克 / At least 1 Vanguard
- 至少2位输出 / At least 2 Duelists
- 至少1位辅助 / At least 1 Strategist
- 理想阵容 / Ideal: 2 Vanguards, 2 Duelists, 2 Strategists

## 终极技能管理 / Ultimate Tracking

### 终极技能使用时机 / Ultimate Usage Timing

- **进攻型 / Offensive**: 配合队友控制链使用 / Use with teammate CC chains
- **防守型 / Defensive**: 反制敌方大招 / Counter enemy ultimates
- **占据点 / Objective**: 在争夺目标关键点时释放 / Use during critical objective contests

### 终极技能充能速度 / Ultimate Charge Rates

不同英雄的终极技能充能速度不同。一般来说：

Different heroes have different ultimate charge rates. Generally:

- 坦克英雄 / Tanks: 中速充能 / Medium charge rate
- 输出英雄 / DPS: 快速充能（通过伤害）/ Fast charge (via damage)
- 辅助英雄 / Supports: 中高速充能（通过治疗）/ Medium-fast charge (via healing)

### 追踪敌方大招 / Tracking Enemy Ultimates

- 注意敌方英雄的击杀和助攻提示 / Watch kill feed and assist notifications
- 估算敌方最后一次使用大招的时间 / Estimate when enemy last ulted
- 保留防御技能应对关键大招 / Save defensive abilities for key ults
- 沟通大招状态给队友 / Communicate ultimate status to team
