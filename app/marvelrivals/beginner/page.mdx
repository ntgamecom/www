## 英雄选择指南 / Hero Selection Tips

### 如何选择第一位英雄 / How to Pick Your First Hero

新手可以从以下几位上手难度较低的英雄开始：

Beginners can start with these low-difficulty heroes:

- **钢铁侠 / Iron Man (Duelist)**: 远程输出，操作直观 / Ranged damage, straightforward gameplay
- **火箭浣熊 / Rocket Raccoon (Strategist)**: 治疗支援，生存能力强 / Healing support, strong survivability
- **绿巨人 / Hulk (Vanguard)**: 近战坦克，容错率高 / Melee tank, high error tolerance

### 英雄分类简介 / Hero Class Overview

- **Vanguard（先锋 / 坦克）**: 负责吸收伤害、保护队友 / Absorb damage, protect teammates
- **Duelist（决斗家 / 输出）**: 负责造成伤害、击杀敌人 / Deal damage, eliminate enemies
- **Strategist（策略家 / 辅助）**: 负责治疗和提供增益 / Heal and provide buffs

## 基础机制 / Basic Mechanics

### 生命值与护盾 / Health & Shields

每位英雄拥有基础生命值（Health）。部分英雄可以生成护盾（Shields），护盾可以吸收伤害但会随时间衰减。

Each hero has base Health. Some heroes can generate Shields, which absorb damage but decay over time.

### 终极技能充能 / Ultimate Charge

终极技能通过以下方式充能：

Ultimate abilities charge through:

- 造成伤害 / Dealing damage
- 治疗队友 / Healing teammates
- 承受伤害 / Taking damage
- 占据目标 / Capturing objectives

充能满后按 R（或 Y）释放。Ultimate abilities can be used once fully charged.

### 重生机制 / Respawn Mechanic

被击杀后进入重生倒计时。重生时间随游戏进程变化。

After elimination, enter a respawn countdown. Respawn time varies as the match progresses.

### 场景互动 / Environment Interaction

部分地图包含可破坏墙壁、移动平台和危险区域。

Some maps feature destructible walls, moving platforms, and hazard zones.

## 新手第一场比赛指南 / First Match Guide

### 赛前准备 / Pre-Match Preparation

1. 选择一位你感兴趣的英雄 / Pick a hero that interests you
2. 在训练场（Practice Range）熟悉技能 / Familiarize with abilities in Practice Range
3. 检查团队阵容是否平衡 / Check if team composition is balanced

### 比赛中的注意事项 / During the Match

- **跟随团队 / Stick with the Team**: 不要单独行动 / Do not wander alone
- **关注目标 / Focus on Objectives**: 击杀敌人重要，但占领目标更重要 / Kills matter, but objectives matter more
- **注意技能冷却 / Track Cooldowns**: 不要浪费核心技能 / Do not waste key abilities
- **及时回撤 / Know When to Retreat**: 血量低时后退找治疗 / Fall back for healing when low

### 比赛结束之后 / After the Match

- 查看比赛数据统计 / Review match statistics
- 思考可以改进的地方 / Think about what can be improved
- 尝试不同的英雄 / Try different heroes

## 训练场指南 / Practice Range Guide

训练场是练习英雄技能的最佳场所。你可以：

The Practice Range is the best place to practice hero abilities. You can:

- 测试技能连招 / Test ability combos
- 练习瞄准 / Practice aim
- 熟悉地图布局 / Learn map layouts
- 尝试不同装备配置 / Try different loadout configurations

建议每位新英雄至少花10分钟在训练场熟悉技能。

It is recommended to spend at least 10 minutes in the Practice Range for each new hero.
