# 英雄详解 / Heroes

漫威争锋拥有丰富多样的英雄阵容，每位英雄都有独特的战斗风格和技能组合。本章将详细介绍所有英雄的分类、能力和使用技巧。

Marvel Rivals features a diverse roster of heroes, each with unique combat styles and ability kits. This chapter covers all heroes by class, abilities, and usage tips.
