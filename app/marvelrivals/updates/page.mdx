## 英雄平衡性调整 / Hero Balance Changes

### 增强 / Buffs

近期补丁中增强的英雄：

Heroes that received buffs in recent patches:

**雷神索尔 / Thor**
- 闪电打击伤害提升10% / Lightning strike damage increased by 10%
- 终极技能充能速度提升 / Ultimate charge rate increased
- 护盾持续时间延长 / Shield duration extended

**火箭浣熊 / Rocket Raccoon**
- 治疗射线范围扩大 / Healing beam range increased
- 烟雾弹冷却时间缩短 / Smoke bomb cooldown reduced
- 基础生命值提升25点 / Base HP increased by 25

**黑豹 / Black Panther**
- 飞镖投掷伤害提升 / Dart throw damage increased
- 隐身突进距离延长 / Cloak dash distance increased
- 被动标记持续时间延长 / Passive mark duration extended

### 削弱 / Nerfs

近期补丁中削弱的英雄：

Heroes that received nerfs in recent patches:

**蜘蛛侠 / Spider-Man**
- 基础生命值降低25点 / Base HP reduced by 25
- 蛛网射击冷却时间增加 / Web shot cooldown increased
- 终极技能伤害降低15% / Ultimate damage reduced by 15%

**钢铁侠 / Iron Man**
- 导弹齐射范围缩小 / Missile barrage radius reduced
- 飞行状态下移速加成降低 / Flight speed boost reduced
- 终极技能充能需求增加 / Ultimate charge requirement increased

**绿巨人 / Hulk**
- 被动变身后的生命值加成降低 / Passive transformation HP bonus reduced
- 怒吼减伤效果降低 / Roar damage reduction lowered
- 跳跃技能冷却增加 / Leap ability cooldown increased

### 机制调整 / Mechanic Changes

- **爆头判定优化 / Headshot Hitbox Refinement**: 部分英雄的爆头判定范围调整 / Adjusted headshot hitboxes for certain heroes
- **治疗衰减机制 / Healing Falloff**: 远距离治疗效果衰减 / Healing effectiveness falls off over distance
- **重生时间调整 / Respawn Timer Adjustment**: 后期重生时间略微增加 / Late-game respawn time slightly increased
- **地图碰撞修复 / Map Collision Fixes**: 修复了多个地图的碰撞体积问题 / Fixed collision issues on multiple maps
