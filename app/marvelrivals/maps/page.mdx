# 地图模式 / Maps & Modes

熟悉地图和游戏模式是取胜的关键。本章详细介绍所有地图、模式特点以及各场景的策略要点。

Knowing the maps and game modes is essential for victory. This chapter covers all maps, mode features, and map-specific strategies.

## 游戏模式 / Game Modes

### 据点争夺 / Domination

双方争夺一个固定据点，率先达到100%占领度的队伍获胜。

Both teams fight over a fixed capture point. The first team to reach 100% capture progress wins.

| 模式特性 / Feature | 说明 / Description |
|:--|:--|
| 目标 / Objective | 占领并守住据点 / Capture and hold the point |
| 获胜条件 / Win Condition | 达到100%占领进度 / Reach 100% capture progress |
| 回合数 / Rounds | 三局两胜 / Best of 3 |
| 重生方式 / Respawn | 定时重生 / Timed respawn |

**策略要点 / Strategy Tips**:

- 开局快速占领据点 / Rush the point at match start
- 坦克在点内抗压，输出在外围打 / Tank in point, DPS from perimeter
- 不要全部挤在点内 / Do not all crowd inside the point
- 控制据点周边的关键位置 / Control key positions around the point

### 运载目标 / Convoy

攻击方护送运载目标到达终点，防守方阻止推进。

Attackers escort a payload to the destination, defenders stop its progress.

| 模式特性 / Feature | 说明 / Description |
|:--|:--|
| 目标 / Objective | 护送运载目标 / Escort the payload |
| 获胜条件 / Win Condition | 到达终点或时间耗尽 / Reach destination or time expires |
| 检查点 / Checkpoints | 中间检查点延长加时 / Mid-checkpoints extend overtime |
| 重生方式 / Respawn | 定时重生 / Timed respawn |

**策略要点 / Strategy Tips**:

- 攻击方：至少一名队友在车附近 / Attackers: keep at least one near payload
- 防守方：在关键拐角处建立防线 / Defenders: set up at key corners
- 运载目标可以提供掩体 / Payload provides mobile cover
- 加时阶段全力保车或拆车 / Overtime: all-in on payload

### 汇聚模式 / Convergence

融合了占领据点和运载目标的混合模式。

A hybrid mode combining Domination and Convoy elements.

| 模式特性 / Feature | 说明 / Description |
|:--|:--|
| 目标 / Objective | 先占领据点，后护送运载目标 / Capture point, then escort payload |
| 获胜条件 / Win Condition | 运载目标到达终点 / Payload reaches destination |
| 阶段 / Phases | 占领阶段 + 护送阶段 / Capture phase + Escort phase |
| 重生方式 / Respawn | 定时重生 / Timed respawn |

## 地图详解 / Map Guide

### 东京2099 / Tokyo 2099

未来风格的日本都市地图，包含多层战斗区域。

A futuristic Japanese city map with multi-layered combat zones.

**关键位置 / Key Locations**:
- **中央广场 / Central Plaza**: 主要交火区域 / Main engagement zone
- **高架列车 / Elevated Train**: 侧翼进攻路线 / Flanking route
- **地下通道 / Underground Passage**: 绕后路线 / Flanking path
- **神社顶部 / Shrine Top**: 狙击位置 / Sniper perch

**策略 / Strategy**:
- 控制高架列车可获得高度优势 / Control the train for high ground
- 地下通道适合刺客英雄绕后 / Underground is great for flankers
- 中央广场视野开阔，远程英雄有利 / Central Plaza has open sightlines

### 阿斯加德 / Asgard

以北欧神话为主题的宫殿地图，拥有宏伟的建筑结构。

A Norse mythology-themed palace map with grand architecture.

**关键位置 / Key Locations**:
- **彩虹桥 / Bifrost Bridge**: 主要通道 / Main corridor
- **王座大厅 / Throne Room**: 关键占领点 / Key capture point
- **瞭望塔 / Watchtower**: 高点视野 / High ground vision
- **武器库 / Armory**: 侧翼入口 / Side entrance

**策略 / Strategy**:
- 彩虹桥狭窄适合近战坦克 / Bifrost is narrow, good for melee tanks
- 瞭望塔提供全图视野 / Watchtower provides map-wide vision
- 王座大厅的柱子可以当作掩体 / Throne pillars provide cover

### 瓦坎达 / Wakanda

黑豹的故乡，融合了高科技与自然景观的地图。

Black Panther's homeland, blending high-tech and natural landscapes.

**关键位置 / Key Locations**:
- **皇宫入口 / Palace Entrance**: 正面战场 / Frontline
- **花园迷道 / Garden Maze**: 伏击位置 / Ambush spot
- **科技实验室 / Vibranium Lab**: 室内战斗区 / Indoor combat zone
- **瀑布平台 / Waterfall Platform**: 高地控制 / High ground control

**策略 / Strategy**:
- 花园迷道适合机动型英雄 / Garden maze favors mobile heroes
- 实验室室内战斗适合近战 / Lab indoor fights favor melee
- 瀑布平台控制权决定胜负关键 / Waterfall platform control is crucial

### 克林特区 / Kyln

星际监狱主题，狭窄通道和开阔区域并存的空间站地图。

A space station map with both narrow corridors and open areas.

**关键位置 / Key Locations**:
- **监管中心 / Control Center**: 中央据点 / Central point
- **囚犯区 / Prison Block**: 多层战斗区 / Multi-level combat
- **气锁走廊 / Airlock Corridor**: 狭窄通道 / Narrow passage
- **观测平台 / Observation Deck**: 外部视野 / External view

**策略 / Strategy**:
- 监管中心是主要争夺点 / Control Center is the main contest area
- 囚犯区上下层需要团队协同 / Prison Block requires team coordination
- 气锁走廊适合范围伤害英雄 / Airlock favors AoE heroes

### 地狱厨房 / Hell's Kitchen

夜魔侠的家乡，黑暗狭窄的城市街道地图。

Daredevil's hometown, a dark and narrow urban street map.

**关键位置 / Key Locations**:
- **街头路口 / Street Intersection**: 主要战场 / Main battlefield
- **屋顶 / Rooftops**: 狙击和侦察位置 / Sniper and recon positions
- **地下铁 / Subway Station**: 快速转移路线 / Fast rotation route
- **仓库内部 / Warehouse Interior**: 室内交火区 / Indoor engagement zone

**策略 / Strategy**:
- 屋顶是远程英雄的理想位置 / Rooftops are ideal for ranged heroes
- 地下铁可以快速绕到敌方后方 / Subway allows quick flanks
- 街道狭窄适合近战缠斗 / Streets favor close-quarters combat

## 地图通用技巧 / General Map Tips

- **熟悉重生路线 / Know Spawn Routes**: 了解从重生点到目标的最短路径 / Learn fastest paths from spawn to objective
- **记忆血包位置 / Memorize Health Pack Locations**: 每个地图的血包位置都是关键资源 / Health pack locations are critical resources
- **利用高低差 / Use Verticality**: 高地优势在漫威争锋中非常重要 / High ground advantage is crucial in Marvel Rivals
- **破坏环境 / Environmental Destruction**: 部分墙壁和平台可以破坏，创造新的路径 / Some walls and platforms are destructible, creating new routes
- **适应不同模式 / Adapt Per Mode**: 同一地图在不同模式下的策略可能完全不同 / Same map may have completely different strategies per mode
