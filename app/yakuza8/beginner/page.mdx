## 职业系统基础 / Job System Basics

### 职业切换 / Job Switching

在"Hello Work"或手机APP上切换 / Change at Hello Work or via phone app.

| 要素 / Element | 说明 / Description |
|---|---|
| 角色等级 / Character Level | 共99级——影响基础属性 / Max 99 — affects base stats |
| 职业等级 / Job Level | 每个职业独立等级——共99级/ Each job has its own level — max 99 |
| **被动技能继承** / Passive Carryover | 升级职业时解锁的被动——切换到其他职业仍然有效 / Passives unlocked from one job apply to all jobs |
| 职业外观 / Job Appearance | 每个职业有独立外观——装备该职业时显示 / Each job has unique appearance |

**职业练级策略 / Leveling Strategy**:
- 前期：每个角色先练1-2个职业到30级解锁核心被动 / Early: level 1-2 jobs to 30 for core passives
- 中期：解锁更多职业后——给每个角色配一个主职业+一个副职被动 / Mid: main job + sub-job passives
- 后期：全职业练满——全被动叠加 = 最强 / Late: all jobs maxed = ultimate power

### 职业属性继承 / Stat Inheritance

| 属性 / Stat | 职业影响 / Job Impact |
|---|---|
| 攻击力 / Attack | 部分继承——主职业贡献60%/其他职业40% / Partial — main job 60%, others 40% |
| 防御力 / Defense | 同上 / Same as above |
| 生命 / HP | 部分继承 / Partial |
| 魔法 / MP | 部分继承 / Partial |
| 速度 / Speed | 完全继承——全职业累加 / Full — all jobs stack |

## 羁绊系统 / Bond System

| 等级 / Level | 效果 / Effect | 提升方式 / Method |
|---|---|---|
| 1-10 | 基本属性加成 / Base stat bonus | 一起战斗、送礼物 / Fight together, give gifts |
| 10-20 | 解锁对话事件 / Dialogue events | 特定任务 / Specific quests |
| 20-30 | **合体技解锁** / **Tag Team Unlock** | 羁绊达到20后可用 / Available after bond 20 |
| 30-40 | 合体技强化 / Tag Team upgrade | 羁绊继续提升 / Continue raising bond |
| 40-50 | 最强合体技 / Best tag team | 羁绊满级 / Max bond |

**提升羁绊最快方法 / Fastest Bond Raising**:
1. 在餐厅请客吃饭——特别是寿司/怀石料理 / Treat to meals — especially sushi/kaiseki
2. 送角色喜欢的礼物 / Give loved gifts
3. 带他们完成支线任务 / Complete substories with them
4. 在战斗中多让他们完成最后一击 / Let them land finishing blows in combat

## 装备系统 / Gear System

| 部位 / Slot | 装备类型 / Types | 获取方式 / Source |
|---|---|---|
| 武器 / Weapon | 各职业专属武器 / Job-specific | 商店购买+收集材料锻造 / Buy + craft from materials |
| 头部 / Head | 帽子、头饰 / Hats, headbands | 商店+掉落 / Shop + drops |
| 身体 / Body | 服装、防弹衣 / Clothes, vests | 商店+掉落 / Shop + drops |
| 饰品 / Accessory | 戒指、护符 / Rings, charms | 商店+支线奖励 / Shop + substory rewards |

**装备核心提示 / Gear Tips**:
1. 每个角色要有一把好武器——商店买+自己强化 > 纯掉落 / A good weapon is essential — buy + upgrade > pure drops
2. 防具和饰品比武器便宜但同样重要 / Armor and accessories are as important as weapons
3. 装备的特殊效果(火抗/麻痹抗性)在特定Boss战中有奇效 / Special effects (fire/paralyze resist) are crucial for specific bosses

## 新手常见错误 / Common Beginner Mistakes

| # | 错误 / Mistake | 正确做法 / Correct Way |
|---|---|---|
| 1 | 只用一个职业 / Stick to one job | 每个职业练到解锁核心被动就换——后期全能才是王道 / Level each job for passives — versatility is king late-game |
| 2 | 不加点体力 / Skip HP upgrades | 角色等级提升时注意分配——血量永远是第一属性 / Allocate stats carefully — HP is the most important stat |
| 3 | 忽略羁绊 / Ignore bonds | 合体技的伤害极高——必须练 / Tag team moves deal massive damage — must level |
| 4 | 不买防具 / Skip armor | 武器升级很贵——但防具的性价比更高 / Weapon upgrades are expensive — armor is better value |
| 5 | 不同敌人类别 / Ignore enemy types | 敌人有弱点和抗性——用对类型伤害翻倍 / Enemies have weaknesses and resistances — correct type = double dmg |
