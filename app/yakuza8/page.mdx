# 如龙8攻略 / Like a Dragon: Infinite Wealth Guide

《如龙8》是SEGA/RGG Studio开发的日式角色扮演游戏——春日一番与桐生一马双主角的宏大篇章。从横滨到夏威夷，从黑道到普通市民——一段关于过去、现在与未来的感人故事。本攻略涵盖全职业解析与Build、双地图探索指南、主线支线攻略以及全收集品指南。

Like a Dragon: Infinite Wealth is SEGA/RGG Studio's Japanese RPG — the epic saga of Ichiban Kasuga and Kazuma Kiryu as dual protagonists. From Yokohama to Hawaii, from yakuza to ordinary citizens — a touching story about past, present, and future. This guide covers all jobs and builds, dual-map exploration, story and substory guides, and complete collectibles.

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **多切换职业练级 / Switch Jobs Frequently** | 职业等级共享被动技能——每个职业练到一定等级后解锁永久的被动/ Switch often — each job's passives are permanent once unlocked |
| **2** | **横滨先探索再夏威夷 / Explore Yokohama First** | 前期在横滨多逛逛——有大量支线和收集品 / Explore Yokohama thoroughly early — many substories and collectibles |
| **3** | **存钱买好装备 / Save Money for Gear** | 装备是战斗力的核心——比等级更重要 / Gear is core to combat power — more important than levels |
| **4** | **桐生线不着急 / Don't Rush Kiryu's Story** | 桐生的章节是固定的——他的等级在第四章之后才开始成长 / Kiryu's chapters are fixed — his leveling starts after Chapter 4 |
| **5** | **苏生药多备 / Stock Revival Items** | 战斗中队友死亡如果没有苏生药——只能用MP少的复活技能/ Stock revival items — limited if you run out |
| **6** | **QTE不要按错 / Don't Miss QTEs** | 技能有追加QTE——完美按到可以追加伤害/加效果 / Skills have bonus QTEs — perfect timing adds damage/effects |
| **7** | **野外的垃圾箱翻一翻 / Check Trash Bins** | 横滨和夏威夷的垃圾箱里经常有不错的材料 / Trash bins often contain useful materials |
| **8** | **外卖APP点餐 / Use Food Delivery App** | 手机外卖可以随时补充HP/MP——战斗中不能 / Food delivery restores HP/MP — but not in combat |

## 战斗系统 / Combat System

| 要素 / Element | 说明 / Description |
|---|---|
| **回合制行动 / Turn-Based** | 按速度顺序行动——移动+选择技能 / Act in speed order — move + select skill |
| **QTE追加 / QTE Bonuses** | 攻击时按提示按△/Y——完美追加额外伤害 / Press △/Y at prompt — perfect = extra damage |
| **移动系统 / Movement** | 回合中可以自由移动——放技能时可以走到最佳位置 / Free movement in turns — position for optimal skill placement |
| **环境互动 / Environment** | 拿起地上的自行车/招牌/路障砸人——超高伤害 / Pick up bikes/signs/barriers — huge damage |
| **全体攻击范围 / AoE Markers** | 技能有范围标记——可以一次打多个敌人 / Skills have AoE markers — hit multiple enemies |
| **斗志 / Will** | 满斗志可以发动"斗志爆发"——大幅度强化下一击 / Full Will = "Will Burst" — massively boost next attack |
| **合体技 / Tag Team** | 羁绊等级提升后解锁双人合体技——超高伤害 / Bond level unlocks tag team moves — huge damage |

## 难度说明 / Difficulty Guide

| 难度 / Difficulty | 说明 / Description |
|---|---|
| **普通 / Normal** | 标准体验——推荐首次游玩 / Standard — recommended first play |
| **困难 / Hard** | 敌人更肉+更聪明——建议至少10级后再考虑 / Tankier + smarter — recommended after level 10 |
| **终极 / Legend** | **通关后解锁**——最适合二周目的挑战 / **Post-game unlock** — best for NG+ |

**困难模式建议 / Hard Mode Tips**:
1. 第一优先：存钱买防具——防御就是生命 / Priority 1: buy armor — defense = life
2. 职业等级>角色等级——多练不同职业 / Job level > character level — level multiple jobs
3. 药水不要省——打不过就吃药 / Use items freely — no shame in healing mid-boss
4. QTE一定要完美按——困难模式下那点追加伤害很关键 / Perfect QTEs are critical on Hard
