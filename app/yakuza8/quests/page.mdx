## 支线任务系统 / Substory System

如龙8有 **50+个支线任务**——分布在横滨和夏威夷 / 50+ substories across both maps.

| 类型 / Type | 数量 / Count | 触发方式 / Trigger |
|---|---|---|
| **普通支线 / Normal Substories** | ~30个 | 在地图上看到"!"标记——靠近触发 / "!" marker on map — approach to trigger |
| **羁绊剧场 / Bond Cinema** | 每个队友5-8个 | 提升羁绊等级解锁 / Raise bond level |
| **Sujimon相关 / Sujimon Quests** | ~10个 | 在夏威夷Sujimon道馆触发 / Trigger at Sujimon gyms |
| **摄影任务 / Photography** | ~15个 | 夏威夷拍照点 / Photo spots in Hawaii |
| **外卖任务 / Delivery Missions** | ~10个 | 疯狂送达小游戏触发 / Crazy Delivery minigame |
