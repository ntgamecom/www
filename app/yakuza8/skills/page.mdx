## 技能系统概览 / Skill System Overview

如龙8的技能分为**职业技能**(当前职业独有的技能)和**觉醒技能**(通过羁绊剧场和主线解锁)。每个角色可以装备4个继承技能(从其他职业习得的技能)。技能通过**极技槽**释放——极技槽上限默认4格——通过装备和羁绊提升 / Yakuza 8 skills are divided into **Job Skills** (current job exclusive) and **Awakening Skills** (unlocked via Bond Drama and story). Each character equips 4 inherited skills (from other jobs). Skills use **Heat Gauge** — default 4 bar cap — increased via gear and bonds.

### 技能获取方式 / Skill Acquisition

| 方式 / Method | 技能类型 / Type | 数量 / Count | 说明 / Notes |
|---|---|---|---|
| 职业升级 / Job Level | 职业技能 / Job skills | 每个职业5-8个 | 职业等级1-30级解锁 / Job level 1-30 |
| 羁绊剧场 / Bond Drama | 觉醒技能 / Awakening | 每个角色1-2个 | 完成羁绊最终章 / Complete Bond finale |
| 主线剧情 / Main Story | 觉醒技能 / Awakening | 全队共3-4个 | 关键剧情节点 / Key story moments |
| 支线任务 / Side Quests | 共通技能 / Shared | 10+ | 特定支线奖励 / Specific side quests |
| 隐藏Boss / Hidden Boss | 特殊技能 / Special | 3-5个 | 击败隐藏Boss掉落 / Hidden boss drops |

## 技能传承推荐 / Skill Inheritance Recommendations

**春日**: 继承刑警的"正义之盾"(群体减伤)、女招待的"终极款待"(全队满血)。**纱荣子**: 继承流浪汉的"冷气"(低成本群奶)和舞者的"加速"(全队速度+30%)。**足立**: 继承勇者的"勇气呐喊"(全队攻击Buff)和杀手的"瞄准"(暴击Buff)。**韩俊基**: 继承勇者的"英雄之盾"(无敌保命)和流浪汉的"冰墙"(物理盾)。**难波**: 继承女招待的"魅惑微笑"(控制)和刑警的"搜查令"(防御Buff)。**终局目标**: 全角色4个继承槽全部装满——打造完美的技能组合 / **Ichiban**: Inherit Detective's "Justice Shield" (party DR) + Hostess's "Ultimate Hospitality" (full heal). **Saeko**: Inherit Homeless's "Cool Breeze" (cheap party heal) + Dancer's "Speed Up" (SPD buff). **Adachi**: Inherit Hero's "Courage Shout" (ATK buff) + Hitman's "Aim" (crit buff). **Joon-gi**: Inherit Hero's "Hero Shield" (invincible) + Homeless's "Ice Wall" (phys block). **Nanba**: Inherit Hostess's "Charming Smile" (CC) + Detective's "Search Warrant" (DEF buff). **Endgame Goal**: All 4 inheritance slots filled on all characters.
