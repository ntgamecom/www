# 世界与探索 / World & Exploration

## 双地图总览 / Dual Map Overview

如龙8有 **2个主要城市**——横滨和夏威夷 / 2 major cities — Yokohama and Hawaii.

| 地图 / Map | 面积 / Size | 风格 / Style | 可探索时期 / Available |
|---|---|---|---|
| **横滨伊势佐木异人町 / Yokohama Ijincho** | 中 / Medium | 日本城市——狭窄街道+商业区 / Japanese city — narrow streets + commercial | 第1章起 / From Chapter 1 |
| **夏威夷檀香山市 / Honolulu** | 大 / Large | 热带城市——海滩+购物街+山区 / Tropical city — beaches + malls + mountains | 第3章起 / From Chapter 3 |

### 两地往返方式 / Travel Between Maps

| 方法 / Method | 可用时期 / Available | 费用 / Cost |
|---|---|---|
| **主线剧情**——第4章后自动开放往返 / Main story — automatic after Chapter 4 | 第4章+ / Ch4+ | 免费 / Free |
| **夏威夷机场**——随时可以飞回横滨 / Hawaii airport — fly back anytime | 第5章+ / Ch5+ | 1万 / 10k |
| **横滨车站**——坐列车去夏威夷(机场) / Yokohama station — train to Hawaii airport | 第5章+ / Ch5+ | 1万 / 10k |

### 小游戏攻略 / Minigame Tips

| 小游戏 / Minigame | 攻略要点 / Tips |
|---|---|
| **卡拉OK / Karaoke** | 注意节奏——和7代一样分Easy/Normal/Hard三个难度。全连有成就 / Same as Y7 — three difficulties. Full combo = achievement |
| **棒球 / Baseball** | 先练挥棒时机——投球有固定模式。挑战模式奖励好装备 / Practice timing — pitches have patterns. Challenge mode rewards good gear |
| **麻将 / Mahjong** | 如龙麻将比普通麻将简单——有"红宝牌"加成。不会打的可以去设定调成"简单" / Easier than real mahjong — has red dora. Set to "Easy" in options |
| **疯狂送达 / Crazy Delivery** | 控制好滑板车的平衡——撞到东西会减速。等级越高解锁更远订单+更多收入 / Balance scooter — collisions slow you down. Higher level = longer routes + more money |
| **钓鱼 / Fishing** | 不同地点有不同鱼种——稀有鱼可以卖高价。钓到的鱼也能做料理材料 / Different spots have different fish — rare fish sell high. Fish can be used in cooking |
| **跳蚤市场 / Flea Market** | 摆摊时卖高价值低数量的物品——比如稀有材料和装备。定价偏高但不要太离谱 / Sell high-value low-quantity items — rare mats and gear. Price high but not ridiculous |

## Sujimon攻略 / Sujimon Battle Guide

**如龙版的宝可梦**——全游戏中可以收集超过200种Sujimon / Yakuza's Pokemon — collect 200+ Sujimon.

| 要素 / Element | 说明 / Description |
|---|---|
| **收集 / Collection** | 击败特定敌人后有一定几率变成Sujimon / Defeat specific enemies — chance to become Sujimon |
| **属性 / Types** | 火/水/雷/风/土/光/暗——互相克制 / Fire/Water/Lightning/Wind/Earth/Light/Dark — rock-paper-scissors |
| **稀有度 / Rarity** | N→R→SR→SSR——SSR最难获得 / N→R→SR→SSR — SSR rarest |
| **战斗 / Battle** | Sujimon之间有自动对战——谁的属性占优谁赢 / Auto-battle between Sujimon — type advantage wins |
| **道馆 / Gym** | 夏威夷各地有Sujimon道馆——收集徽章 / Gyms across Hawaii — collect badges |

### Sujimon对战策略 / Sujimon Battle Strategy

| 策略 / Strategy | 说明 / Description |
|---|---|
| **属性优势优先 / Type Advantage First** | 对战时属性克制是最大的胜负因素——尽量用克制属性的Sujimon / Type advantage is the biggest factor — always use super-effective types |
| **收集平衡队伍 / Build a Balanced Team** | 至少准备3种不同属性的强力Sujimon——应对不同对手 / Have at least 3 strong Sujimon of different types for different opponents |
