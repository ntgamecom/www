# 职业与技能 / Jobs & Skills

如龙8共有20+个职业——分为基础职业和高级职业。职业不仅决定角色的战斗技能，还影响属性成长和被动能力。合理搭配主副职业是通关高难度的关键 / Yakuza 8 has 20+ jobs — base and advanced. Jobs determine combat skills, stat growth, and passives. Smart main/sub job combos are key to high-difficulty clears.

---

## 职业系统详解 / Job System Overview

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **主职业 / Main Job** | 决定角色的战斗技能和外观。主职业的等级决定技能解锁进度 / Determines combat skills and appearance. Level = skill unlock rate |
| **副职业被动 / Sub Job Passive** | 转职为其他职业后，已习得的被动技能仍然生效。通过切换职业学习不同被动来强化主职业 / Learned passives remain active after switching jobs. Rotate jobs to learn different passives that strengthen your main job |
| **职业等级 / Job Level** | 每个职业独立升级——使用该职业战斗获得经验。等级越高解锁越多技能 / Each job levels independently — gain XP by fighting in that job. Higher level = more skills unlocked |
| **职业转换 / Job Change** | 在职业介绍所随时可以转职——但转职后战斗技能会变为主职业技能，等级从1开始 / Change jobs anytime at the job center — but combat skills change to the new job's skills starting from level 1 |
| **职业契合度 / Job Affinity** | 每个角色对不同职业有天生契合度——S级契合的角色使用该职业时属性加成更高 / Each character has natural affinity for different jobs — S-rank affinity gives higher stat bonuses |
| **技能点数 / Skill Points** | 职业升级获得SP——用于解锁技能树上的技能 / Gain SP from job levels — spend to unlock skills on the skill tree |

## 职业策略详解 / Job Strategy Guide

### 春日一番 — 勇者为核心 / Ichiban — Hero Core

春日的勇者职业是游戏中最全面的物理职业——有护盾(勇者之盾)、有单伤(勇者之剑)、有群伤(勇者之吼)。**副职业被动搭配**:
- **橄榄球**: HP+25% —— 提升坦度
- **厨师**: 火属性伤害+25% —— 应对弱火敌人
- **舞者**: 速度+15% —— 提升行动顺序

**战斗节奏**: 开场→勇者之盾(全队安全)→勇者之剑输出(单体/群伤看情况)→盾没了补盾→循环 / **Combat rhythm**: Start with Hero's Shield → Hero's Sword DPS → re-shield when needed → repeat.

**进阶Build**: 勇者(主) + 厨师(被动)+ 橄榄球(被动) + 保镖(被动) = 全方位物理战神 / **Advanced Build**: Hero (main) + Chef (passive) + Football (passive) + Bodyguard (passive) = all-round physical god.

### 桐生一马 — 堂岛之龙 / Kiryu — Dragon of Dojima

桐生的专属职业"堂岛之龙"包含全游戏最强技能组合——虎落(反击技)和怒龙(全屏AOE)。**副职业被动搭配**:
- **保镖**: 单体伤害+20% —— 虎落反击秒Boss
- **武士**: 暴击率+15% —— 提升爆发
- **格斗家**: 反击概率+10% —— 更多虎落触发

**战斗节奏**: 等敌人攻击→虎落反击→敌人硬直→平A输出→等下一次攻击→虎落。桐生对Boss战时几乎可以无限虎落循环 / **Combat rhythm**: Wait for enemy attack → Tiger Drop counter → enemy stunned → normal attacks → wait → repeat. Kiryu can almost infinitely Tiger Drop loop on bosses.

**进阶Build**: 堂岛之龙(主) + 保镖(被动)+ 武士(被动) + 格斗家(被动) = 反击战神 / **Advanced Build**: Dragon of Dojima (main) + Bodyguard (passive) + Samurai (passive) + Martial Artist (passive) = counter god.

### 纱荣子 — 最强治疗 / Saeko — Best Healer

纱荣子的偶像职业是全游戏最强的治疗职业。**副职业被动搭配**:
- **女仆**: 回复量+25%
- **街头音乐**: 冰抗+30%
- **管家**: 防御+15%

**战斗节奏**: 保持全队满血→治疗压力不大时用"燃爆舞台"加全队Buff→压力大时全力治疗 / **Combat rhythm**: Keep party topped off → use Stage Explosion for buff when safe → full heal when pressured.

**进阶Build**: 偶像(主) + 女仆(被动)+ 街头音乐(被动) + 管家(被动) = 终极奶妈 / **Advanced Build**: Idol (main) + Maid (passive) + Street Musician (passive) + Butler (passive) = ultimate healer.

### 韩俊基 — 忍者输出 / Joon-gi — Ninja DPS

忍者是游戏中最灵活的DPS职业——影分身+毒雾+手里剑三重输出方式。**副职业被动搭配**:
- **占卜师**: 雷属性+20%
- **舞者**: 速度+15%
- **格斗家**: 暴击+10%

**战斗节奏**: 开场毒雾(对手大Boss时超高效)→影分身输出→毒没了补毒→循环 / **Combat rhythm**: Poison Mist (great vs bosses) → Shadow Clone DPS → re-poison when expired → repeat.

**进阶Build**: 忍者(主) + 占卜师(被动)+ 舞者(被动) + 格斗家(被动) = 极速毒王 / **Advanced Build**: Ninja (main) + Fortuneteller (passive) + Dancer (passive) + Martial Artist (passive) = speed poison king.

### 赵天佑 — 保镖爆发 / Zhao — Bodyguard Burst

赵的物理攻击成长是最高的——配合保镖的"一闪"和"居合"能打出毁天灭地的单体伤害 / Zhao has the highest physical ATK growth — with Bodyguard's Flash and Iaijutsu, he deals devastating single-target damage.

| 副职业被动 / Sub Job Passive | 效果 / Effect |
|---|---|
| **勇者** | 物理伤害+10% / Physical dmg +10% |
| **武士** | 暴击伤害+20% / Crit dmg +20% |
| **格斗家** | 攻击力+15% / ATK +15% |

**战斗策略**: 赵的定位是纯单体物理核弹——开场先让辅助上Buff, 然后赵用居合斩一发入魂 / **Strategy**: Zhao is a pure single-target physical nuke — let supports buff first, then Zhao deletes with Iaijutsu.

### 足立宏一 — 武士坦克 / Adachi — Samurai Tank

足立的高防御适合做坦克——但他用武士职业时嘲讽+高伤的配合也非常出色 / Adachi's high defense suits tanking — but as Samurai, taunt + high damage is excellent.

| 副职业被动 / Sub Job Passive | 效果 / Effect |
|---|---|
| **侦探** | 防御力+20% / DEF +20% |
| **管家** | 全异常抗性+15% / All status resist +15% |
| **橄榄球** | HP+25% / HP +25% |

## 高级技巧 / Advanced Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **职业被动叠加 / Passive Stacking** | 同类型被动可以叠加——例如两个"攻击+20%"的被动=实际+44%。全游戏最多叠加3个同类型 / Same-type passives stack multiplicatively. Cap at 3 same-type passives |
| **装备职业武器 / Equip Job Weapons** | 每个职业有专属武器——装备后提升对应职业的技能伤害。转职后也要换武器 / Each job has exclusive weapons — equipping them boosts skill damage. Remember to swap weapons when changing jobs |
| **QTE练习 / Practice QTEs** | 如龙8的技能QTE(按键提示)影响伤害倍率——完美QTE=1.5倍伤害。每个职业的QTE节奏不同——花5分钟练习 / Skill QTEs affect damage multiplier — perfect QTE = 1.5x damage. Each job has different rhythm — spend 5 minutes practicing |
| **敌人弱点利用 / Exploit Weaknesses** | 每个敌人有1-2个属性弱点(物理/火/冰/雷/光/暗)——用克制属性技能打伤害翻倍。战斗中按R2查看敌人信息 / Each enemy has 1-2 elemental weaknesses — hitting them doubles damage. Press R2 during combat to check enemy info |
| **职业契合度最大化 / Maximize Affinity** | 每个角色对职业有S/A/B/C契合度——S级契合提供额外20%属性加成。尽量给角色配S级职业 / Each character has S/A/B/C affinity for jobs — S-rank gives +20% stats. Match characters to S-rank jobs |
| **SP分配策略 / SP Allocation** | 优先解锁核心技能再点被动——不要平均分配。每个职业的核心技能一般在5-10级解锁 / Prioritize core skills before passives — don't spread evenly. Core skills typically unlock at job level 5-10 |
| **双队伍职业搭配 / Two-Party Job Balance** | 第4章后你有春日队和桐生队两个队伍——确保两队都有治疗和坦克 / After Ch4 you have two parties — ensure both have a healer and tank |
