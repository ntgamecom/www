{/* ===== 世界概述 ===== */}

## 世界概述 / World Overview

《双影奇境》最突出的特点是 **双世界交替结构**。游戏共分 6 个主要章节，奇数章为科幻世界 (Mio 的故事)，偶数章为奇幻世界 (Zoe 的故事)，最终章为两个世界的融合。

The most prominent feature of Split Fiction is its **dual-world alternating structure**. The game has 6 main chapters: odd-numbered chapters are sci-fi worlds (Mio's story), even-numbered chapters are fantasy worlds (Zoe's story), and the final chapter merges both worlds.

## 世界对比 / World Comparison

| 特征 / Feature | 科幻世界 / Sci-Fi World | 奇幻世界 / Fantasy World |
|---|---|---|
| 视觉风格 / Visual Style | 冷色调、金属质感、霓虹灯光 | 暖色调、自然材质、魔法光芒 |
| 音乐 / Music | 电子合成器、工业节奏 | 管弦乐、民谣旋律 |
| 敌人 / Enemies | 机器、无人机、数据体 | 怪物、精灵、龙类 |
| 解谜风格 / Puzzle Style | 技术操作、数据破解 | 自然互动、元素操控 |
| 色调 / Color Palette | 蓝色、紫色、白色 | 绿色、金色、红色 |

## 第一章：数据之城 / Chapter 1: Data City

## 世界背景 / World Setting

Mio 笔下的近未来都市，人类与人工智能共存。城市由巨大的中央 AI 核心控制，但核心出现异常，导致城市系统紊乱。

A near-future metropolis from Mio's writing where humans and AI coexist. The city is controlled by a giant central AI core that has malfunctioned, causing systemic chaos.

## 关卡流程 / Level Flow

1. **城市外围** / City Outskirts — 学习基础移动和互动操作
2. **中央枢纽** / Central Hub — 首次遭遇无人机敌人，学习战斗基础
3. **数据塔楼** / Data Tower — 垂直攀爬关卡，引入双人协作机关
4. **AI 核心** / AI Core — 第一章 BOSS 战

## 环境解谜 / Environmental Puzzles

- **终端配对** / Terminal Pairing — 两人同时激活两个终端开启通道
- **数据流跳跃** / Data Stream Jump — 在流动的数据流上跳跃前进
- **防火墙破解** / Firewall Bypass — Mio 扫描弱点，Zoe 远程打击

## 探索奖励 / Exploration Rewards

- 3 份 **丢失的手稿** / Lost Manuscripts
- 2 个 **能力升级芯片** / Ability Upgrade Chips
- 1 个 **隐藏服装** / Hidden Costume

## 第六章：创意之源 / Chapter 6: Source of Creation

## 世界背景 / World Setting

两个世界的融合点——创意的本源。Mio 和 Zoe 发现她们的故事世界正在崩塌，必须合作抵达核心才能逃脱。

The merging point of both worlds — the source of creativity itself. Mio and Zoe discover their story worlds are collapsing and must cooperate to reach the core and escape.

## 关卡流程 / Level Flow

1. **世界裂缝** / World Rift — 科幻和奇幻元素开始混合

3. **创意风暴** / Creative Storm — 不断变化的场景，需要同时使用两种世界的能力
4. **最终核心** / Final Core — 终极 BOSS 战

## 环境解谜 / Environmental Puzzles

- **混合机关** / Hybrid Mechanisms — 同时需要技术和魔法才能激活的机关
- **创造与毁灭** / Create & Destroy — 一人创造平台，另一人摧毁障碍
- **镜像世界** / Mirror Worlds — 两人在不同世界中执行对称操作

## 探索奖励 / Exploration Rewards

- 6 份 **丢失的手稿** / Lost Manuscripts (全部收集后解锁隐藏结局)

{/* ===== 探索技巧 ===== */}

## 探索技巧 / Exploration Tips

1. **发光物体** / Glowing objects — 任何发光的物体都可以互动或收集
2. **岔路探索** / Branching paths — 看到岔路时，两人分头探索效率最高
3. **回溯发现** / Backtrack discoveries — 获得新能力后回到旧关卡，会发现新的可互动点
4. **声音提示** / Audio cues — 特定区域有独特的音效，提示隐藏物品的存在
5. **手稿收集** / Manuscript collection — 每个世界有固定数量的手稿，收集会影响结局
