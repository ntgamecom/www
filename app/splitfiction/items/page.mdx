## 收集品概述 / Collectibles Overview

《双影奇境》的收集品系统比前作更加丰富。收集品不仅用于解锁成就，还能揭示 Mio 和 Zoe 的创作背景故事，甚至影响最终结局。

Split Fiction's collectible system is more extensive than the previous title. Collectibles not only unlock achievements but also reveal Mio and Zoe's creative backgrounds and can even affect the final ending.

| 类型 / Type | 数量 / Count | 功能 / Function |
|---|---|---|

{/* ===== 收集小技巧 ===== */}

## 收集小技巧 / Collection Tips

1. 使用 **章节选择** 回溯时，已收集的物品不会重置，不会重复收集
2. 屏幕右上角会显示当前章节已收集/总数，方便追踪
3. **两人中任意一人收集** 都计入总进度，但存档绑定各自的账号
4. 部分收集品需要特定能力才能获取，先在后续章节获得能力后再回溯
5. 手稿收集满 36 份后可以解锁的隐藏结局非常值得一看
