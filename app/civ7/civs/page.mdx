## 文明系统概述 / Civilization Overview

文明7延续了系列的传统——每个文明拥有独特的**能力**、**单位**和**建筑**。新作加入了**文明切换**机制——在进入新时代时可以选择新文明 / Civ7 continues the tradition — each civ has unique **abilities**, **units**, and **buildings**. New feature: **civilization switching** — choose a new civ when entering a new age.

### 文明切换机制 / Civ Switching

| 时代 / Age | 文明选择 / Civ Selection | 保留内容 / Retained |
|---|---|---|
| 古代 / Ancient | 初始选择——固定 / Initial pick — fixed | - |
| 探索 / Exploration | 可切换至一个关联文明 / Switch to an affiliated civ | 领袖 + 部分能力 / Leader + some abilities |
| 现代 / Modern | 可再次切换 / Switch again | 领袖 + 累积的能力 / Leader + accumulated abilities |

**切换规则 / Switching Rules**：
- 每个文明有一组可切换的"进化路径" / Each civ has a set of "evolution paths"
- 你可以选择历史上或文化上相关联的文明 / Choose historically or culturally related civilizations
- 不切换也可以保留原文明——但会错过新能力 / You can stay — but miss new abilities

### 时代过渡策略 / Age Transition Strategy

**古代→探索时代 / Ancient to Exploration**：
- 保持初始文明的军队核心——切换文明后保留单位 / Keep core army from initial civ — units retain after switching
- 选择与初始文明相邻的文明（历史路线）/ Pick historically adjacent civs
- 如果不确定——选择希腊→罗马→拜占庭的标准路线 / When unsure: Greece → Rome → Byzantine standard path

**探索→现代时代 / Exploration to Modern**：
- 根据你的胜利目标选择文明 / Pick based on your win condition
- 如果已经确立了军事优势——转德国/苏联扩大优势 / If already militarily ahead — pick Germany/Soviet
- 如果是科技领先——转德国/英国用工业区加速 / If tech leading — pick Germany/UK for industrial boost
