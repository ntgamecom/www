# 新手入门 / Beginner Guide

欢迎来到文明7！本指南将帮助你从零开始掌握游戏的基础机制 / Welcome to Civilization VII! This guide helps you master the basics from scratch.

---

## 游戏模式 / Game Modes

| 模式 / Mode | 说明 / Description |
|---|---|
| **单人游戏 / Single Player** | 标准对局——对抗AI文明 / Standard game — vs AI civs |
| **多人游戏 / Multiplayer** | 在线对局——对抗其他玩家 / Online game — vs other players |
| **高级模式 / Advanced** | 自定义规则——调整胜利条件/时代长度等 / Custom rules — adjust victory conditions, age length |
| **教程 / Tutorial** | 内置教学——推荐首次游玩 / Built-in tutorial — recommended for first play |

## 民生与满意度 / Amenities & Happiness

| 满意度等级 / Level | 效果 / Effect | 解决方法 / Solution |
|---|---|---|
| **狂欢 / Ecstatic** | 全产出+20% / All yields +20% | 保持多奢侈资源 / Maintain diverse luxuries |
| **高兴 / Happy** | 全产出+10% / All yields +10% | 每城3-4种奢侈资源 / 3-4 luxury types per city |
| **稳定 / Content** | 标准产出 / Standard yields | 正常状态 / Normal state |
| **不满 / Unhappy** | 全产出-10% / All yields -10% | 进口奢侈资源或建娱乐区 / Import luxuries or build entertainment |
| **暴怒 / Revolting** | 全产出-20% + 叛乱风险 / All yields -20% + revolt risk | 立即解决——派驻军+买奢侈 / Immediate — garrison + buy luxuries |

**满意度提升方法 / Ways to Boost Amenities**:
1. 开发不同种类的奢侈资源——每种提供4城的满意度 / Improve diverse luxury resources — each provides amenities to 4 cities
2. 建娱乐区——竞技场/动物园等 / Build entertainment district — arena, zoo, etc.
3. 与AI交易多余的奢侈资源——换你没有的 / Trade excess luxuries with AI for ones you lack
4. 政策卡——有专门提升满意度的政策 / Policy cards — specific cards boost amenities
5. 伟人——部分大商人提供额外满意度 / Great People — some Great Merchants provide extra amenities

## 新手常见错误 / Common Beginner Mistakes

| # | 错误 / Mistake | 正确做法 / Correct Way |
|---|---|---|
| 1 | 不探索——窝在首都周围 / Don't explore — stay near capital | 开局就派侦察兵出去——地图信息就是力量 / Scout immediately — map knowledge is power |
| 2 | 不建开拓者——只有一城 / Don't build settlers — one city only | 开局尽快铺3-4城——越多城越快发展 / Expand to 3-4 cities ASAP — more cities = faster development |
| 3 | 忽视外交关系 / Ignore diplomacy | 保持良好外交——避免被宣战包围 / Maintain good relations — avoid getting dogpiled |
| 4 | 不关注传承路线 / Skip Legacy Paths | 每个时代先做完传承路线——否则下个时代落后 / Complete Legacy Paths each age — or fall behind next age |
| 5 | 军队全是远古单位 / Army full of ancient units | 每个时代更新军队——科技落后=被碾压 / Upgrade army each age — outdated units get crushed |
| 6 | 不管城邦 / Ignore city-states | 城邦给大量加成——派使者拿宗主国地位 / City-states give huge bonuses — send envoys for suzerain status |
