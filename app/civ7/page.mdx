# 文明7攻略 / Civilization VII Guide

《席德·梅尔的文明7》是Firaxis Games开发的经典4X策略游戏最新作——带领你的文明从远古时代走向未来，在动态变化的游戏中做出关键决策。本攻略涵盖新手入门指引、全领袖与文明解析、时代演进策略、胜利路线规划以及科技市政树详解。

Civilization VII is Firaxis Games' latest entry in the classic 4X strategy series — lead your civilization from antiquity to the future, making critical decisions in a dynamically evolving world. This guide covers beginner tips, all leaders and civilizations, age progression strategies, victory routes, and the full tech & civics tree.

## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|---|---|
| **时代演进 / Age System** | 每次进入新时代——文明会演进为新的文明——保留部分特性 / Each age transition — your civ evolves into a new one — retains some traits |
| **领袖自由搭配 / Leader-Civ Separation** | 领袖和文明可以自由搭配——创造无限组合 / Leaders and civs are mix-and-match — infinite combos |

## 时代演进 / Age System

| 时代 / Age | 时代范围 / Era | 文明演变 / Civ Evolution | 核心玩法 / Core Gameplay |
|---|---|---|---|
| **古典时代 / Antiquity** | 远古~古典时期 | 初始文明——如罗马、埃及、中国 / Starting civs — Rome, Egypt, China | 建立城市、探索周边、早期战争 / Found cities, explore, early warfare |
| **探索时代 / Exploration** | 中世纪~大航海 | 过渡文明——如诺曼、蒙古、阿拔斯 / Transition civs — Norman, Mongol, Abbasid | 跨海殖民、宗教传播、贸易路线 / Overseas colonies, religion, trade |
| **近代 / Modern** | 工业时代~信息时代 | 最终文明——如美国、德国、日本 / Final civs — America, Germany, Japan | 工业化、意识形态、全面战争 / Industrialization, ideology, total war |

**时代过渡机制 / Age Transition**:
- 每个时代结束时有**危机事件**——需要应对挑战 / Each age ends with a **crisis event** — must respond to challenges
- 完成传承路线获得**传承点**——可用于解锁下一时代的加成 / Complete Legacy Paths for **Legacy Points** — unlock bonuses for next age
- 进入新时代时选择**新文明**——保留部分旧文明特性 / Pick a **new civilization** each age — retain some old civ traits

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **开局先探路 / Scout First** | 用侦察兵探索周围——找到好位置建第二城 / Scout surroundings with your warrior/scout — find good spots for 2nd city |
| **2** | **纪念碑优先建 / Monument First** | 纪念碑加速文化——尽早解锁市政 / Monument speeds up culture — unlocks civics faster |
| **3** | **不要爆铺太多城 / Don't Overexpand** | 初期3-4城足够——铺太快会拖累科技和满意度 / 3-4 cities early is enough — overexpansion hurts science and amenities |
| **4** | **传承路线必须做 / Do Legacy Paths** | 每个时代的传承路线是必做任务——不做会落下 / Legacy Paths are essential each age — skipping them cripples you |
| **5** | **贸易路线是命脉 / Trade Routes Are Vital** | 贸易路线给金币、科技、文化——越早建越好 / Trade routes give gold, science, culture — build them early |
| **6** | **指挥官不要死 / Protect Commanders** | 指挥官死了会掉级——军队战斗力大降 / Dead commanders lose levels — army effectiveness plummets |
| **7** | **城邦要抢 / Prioritize City-States** | 第一个发现城邦的文明获得使者+加成——抢首发 / First to meet a city-state gets envoy + bonus — race for first contact |
| **8** | **适应性思维 / Stay Flexible** | 不要死磕一个胜利路线——根据地图和局势调整 / Don't tunnel on one victory type — adapt to map and situation |

## 资源与地块 / Resources & Terrain

| 资源类型 / Type | 说明 / Description | 优先级 / Priority |
|---|---|---|
| **奢侈资源 / Luxury** | 提供满意度——多余的可交易 / Provide amenities — surplus can be traded | 中 / Medium |
| **战略资源 / Strategic** | 用于生产和军事单位——铁、马、石油 / Used for production and military — iron, horses, oil | 高 / High |
| **加成资源 / Bonus** | 提供地块产出加成——小麦、石头、铜 / Provide tile yield bonuses — wheat, stone, copper | 低 / Low |
| **自然奇观 / Natural Wonders** | 独特地块——提供特殊加成 / Unique tiles — provide special bonuses | 高 / High |

**地块产出说明 / Tile Yield Guide**:
- **食物(Food)**→ 城市人口增长 / City population growth
- **生产力(Production)** → 建造单位/建筑速度 / Build unit/building speed
- **科技(Science)** → 科技研究速度 / Tech research speed
- **文化(Culture)** → 市政研究速度 / Civics research speed
- **金币(Gold)** → 购买单位/建筑 / Purchase units/buildings
- **信仰(Faith)** → 购买宗教单位/伟人 / Purchase religious units/great people
