## 城市布局 / City Layout

文明7的城市布局继续采用区域系统——但进行了大幅简化。城市中心 + 专业区域 + 特色建筑构成你的城市核心 / Civ7 city layout keeps the district system but simplifies it. City Center + Specialized Districts + Unique Buildings form your city core.

### 区域规划原则 / District Planning Principles

| 原则 / Principle | 说明 / Explanation |
|---|---|
| 相邻加成 / Adjacency | 区域放在一起获得的加成 > 分开放 / Districts grouped together get more bonuses than spread out |
| 地形利用 / Terrain | 河流/山脉/海洋为特定区域提供额外加成 / Rivers, mountains, oceans give bonuses to specific districts |
| 文明特色 / Civ Unique | 特色建筑需要特定的相邻条件——提前规划 / Unique buildings need specific adjacency — plan ahead |
| 奇观选址 / Wonder Placement | 奇观有地理要求（如靠近海岸、在平原等）/ Wonders have terrain requirements (coast, flatland, etc.) |
| 城市数量 / City Count | 3-4个专业城市 + 若干次级城市是最优解 / 3-4 specialized cities + several feeder cities is optimal |

### 经典城市布局 / Classic City Layout

```
示例布局 / Example Layout (T形 / T-shape)：

       [山脉 / Mountain]
           |
     [图书馆]—[学院]—[天文台]
           |
   [纪念碑]—[市中心]—[军营]
           |       |
   [神殿]—[广场]    [商业中心]
           |
     [浴场]—[水渠]
```

**相邻加成分析 / Adjacency Analysis**：
- 学院+图书馆+天文台：基础+4科技 / Academy + Library + Observatory: base +4 science
- 市中心+广场+商业：基础+3金币+2文化 / City Center + Plaza + Commercial: base +3 gold +2 culture
- 水渠+浴场+市中心：+2食物+2生产力 / Aqueduct + Bath + City Center: +2 food +2 production

## 经济配装 / Economic Builds

### 金币流 / Gold Focus

**核心建筑 / Core Buildings**：
1. 商业中心 + 市场 + 银行 / Commercial Hub + Market + Bank
2. 港口 + 灯塔 + 造船厂 / Harbor + Lighthouse + Dockyard
3. 大集市（奇观）/ Grand Bazaar (Wonder)

**政策卡 / Policy Cards**：
- 自由贸易 / Free Trade：每条贸易路线+3金币 / +3 gold per trade route
- 海运霸主 / Naval Supremacy：港口+5金币 / +5 gold on harbors
- 市场调节 / Market Regulation：商业中心+4金币 / +4 gold on commercial hubs

**推荐文明 / Recommended Civ**：葡萄牙 / Portugal（海上贸易翻倍）

### 生产力流 / Production Focus

**核心建筑 / Core Buildings**：
1. 工业区 + 工坊 + 工厂 / Industrial Zone + Workshop + Factory
2. 水渠 + 浴场 / Aqueduct + Bath
3. 鲁尔山谷（奇观）/ Ruhr Valley (Wonder)

**政策卡 / Policy Cards**：
- 劳动密集 / Labor Intensive：每个农场+1生产力 / +1 production per farm
- 工业革命 / Industrial Revolution：工厂+6生产力 / +6 production on factories
- 军工复合体 / Military-Industrial：军营+2生产力 / +2 production on barracks

**推荐文明 / Recommended Civ**：德国 / Germany（工业区相邻加成翻倍）

### 食物/人口流 / Food Focus

**核心建筑 / Core Buildings**：
1. 水渠 / Aqueduct
2. 浴场 / Bath
3. 粮仓 / Granary
4. 皇家园艺（奇观）/ Hanging Gardens (Wonder)

**推荐文明 / Recommended Civ**：埃及 / Egypt（泛滥平原+食物）

### 资源交易策略 / Resource Trading

1. **你过剩的资源**——卖给AI换金币——不要囤积 / Sell excess resources to AI for gold — don't hoard
2. **如果你缺少某个战略资源**——优先从AI手中购买——不要让军队因缺资源而废了 / If lacking a strategic resource — buy from AI before army becomes obsolete
3. **奢侈品资源的收益递减**——第二个同样奢侈品只有一半收益 / Luxury resources have diminishing returns — second copy gives half benefit

## 文明专属配装 / Civ-Specific Builds

### 中国长城流 / Chinese Great Wall Build

**核心 / Core**：建造长城段——每段+4文化+2金币——连接在一起还有额外加成 / Build Great Wall segments — each +4 culture +2 gold — connected segments get bonuses.

**策略 / Strategy**：在边境上一字排开建造长城——防守+文化双重收益 / Line the border with Great Wall — defense + culture dual benefit.

### 罗马扩张流 / Roman Expansion Build

**核心 / Core**：浴场+水渠的组合提供超高食物和生产力 / Bath + Aqueduct combo provides high food and production.

**策略 / Strategy**：优先建造浴场——然后铺城——每条道路自动连接给金币 / Priority build Bath — then expand — roads auto-connect for gold.

### 英国海上帝国 / British Maritime Empire

**核心 / Core**：港口+造船厂+工厂——三件套提供的金币翻倍 / Harbor + Dockyard + Factory — triple gold yield.

**策略 / Strategy**：在所有沿海城市建设港口——工厂给城市提供生产力加成 / Build harbors in all coastal cities — factories give city-wide production.
