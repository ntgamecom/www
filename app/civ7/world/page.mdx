# 世界与时代 / World & Ages

文明7的世界跨越三个时代——每个时代都有独特的规则和挑战 / The world of Civ VII spans three ages — each with unique rules and challenges.

---

## 地图类型 / Map Types

| 地图类型 / Map Type | 说明 / Description | 推荐玩家 / Best For |
|---|---|---|
| **大陆 / Continents** | 2-3块大陆——中间有海洋——标准体验 / 2-3 continents with oceans — standard | 所有玩家 / All players |
| **盘古大陆 / Pangaea** | 所有文明在同一块大陆上——战争频繁 / All civs on one landmass — war-heavy | 军事胜利玩家 / Military victory |
| **群岛 / Archipelago** | 大量岛屿——海军至关重要 / Many islands — navy critical | 海军文明 / Naval civs |

| **分形 / Shuffle** | 随机生成——未知挑战 / Random generation — unknown challenge | 寻求挑战 / Seeking challenge |

## 时代详解 / Age Deep Dive

### 古典时代 / Antiquity Age

| 时代特征 / Features | 说明 / Description |
|---|---|
| **长度 / Length** | 约80-120回合 / ~80-120 turns |
| **核心玩法 / Core** | 定居——探索——建立基础城市 / Settle — explore — establish core cities |
| **传承路线 / Legacy** | 文化传承——建造7个特色建筑 / Culture Legacy — build 7 unique buildings |
| **危机 / Crisis** | 外族入侵或内部叛乱 / Barbarian invasion or internal rebellion |
| **时代目标 / Goals** | 铺3-4城、建立贸易路线、完成古典传承路线 / Found 3-4 cities, establish trade, complete Antiquity Legacy Path |

**古典关键科技 / Key Antiquity Techs**:
- **陶器 / Pottery** → 粮仓——人口增长 / Granary — population growth
- **采矿 / Mining** → 矿场——生产力 / Mine — production
- **写作 / Writing** → 图书馆——科技 / Library — science
- **制图 / Cartography** → 查看全地图边界 / Reveal map borders

### 探索时代 / Exploration Age

| 时代特征 / Features | 说明 / Description |
|---|---|
| **长度 / Length** | 约80-120回合 / ~80-120 turns |
| **核心玩法 / Core** | 跨海殖民——发现新大陆——建立海外城市 / Overseas colonization — discover new lands — establish overseas cities |
| **传承路线 / Legacy** | 经济传承——建立7个贸易路线 / Economic Legacy — establish 7 trade routes |
| **危机 / Crisis** | 宗教分裂或殖民叛乱 / Religious schism or colonial rebellion |
| **时代目标 / Goals** | 建立海外殖民地、控制城邦、发展宗教 / Establish colonies, control city-states, develop religion |

**探索关键科技 / Key Exploration Techs**:
- **天文导航 / Celestial Navigation** → 跨海航行能力 / Ocean crossing ability
- **印刷术 / Printing Press** → 宗教传播加速 / Religion spread boost
- **火药 / Gunpowder** → 火器单位 / Gunpowder units
- **银行 / Banking** → 金币产出大幅增加 / Gold output boost

### 近代 / Modern Age

| 时代特征 / Features | 说明 / Description |
|---|---|
| **长度 / Length** | 约80-120回合 / ~80-120 turns |
| **核心玩法 / Core** | 工业化——意识形态——冲刺最终胜利 / Industrialization — ideology — rush final victory |
| **传承路线 / Legacy** | 科技或经济传承——决定胜利方式 / Science or Economic Legacy — determines victory path |
| **危机 / Crisis** | 世界大战或意识形态冲突 / World war or ideology conflict |
| **时代目标 / Goals** | 选择胜利路线并全力冲刺 / Pick victory path and go all-in |

**近代关键科技 / Key Modern Techs**:
- **蒸汽动力 / Steam Power** → 工厂——工业产能 / Factory — industrial production
- **电力 / Electricity** → 城市建筑大幅升级 / City building upgrades
- **飞行 / Flight** → 空军单位 / Air units
- **原子能 / Nuclear Fission** → 核武器——最终科技 / Nuclear weapons — endgame tech

## 城邦指南 / City-States Guide

| 城邦类型 / Type | 宗主加成 / Suzerain Bonus | 推荐程度 / Rating |
|---|---|---|
| **军事城邦 / Militaristic** | 战斗力+5或军队维护费-50% / +5 combat strength or -50% army maintenance | ⭐⭐⭐⭐ |
| **科技城邦 / Scientific** | 科技+10%或大科学家点数+2 / Science +10% or +2 great scientist points | ⭐⭐⭐⭐⭐ |
| **文化城邦 / Cultural** | 文化+10%或大艺术家点数+2 / Culture +10% or +2 great artist points | ⭐⭐⭐⭐⭐ |
| **商业城邦 / Commercial** | 金币+15%或贸易路线容量+1 / Gold +15% or +1 trade route cap | ⭐⭐⭐⭐ |
| **工业城邦 / Industrial** | 生产力+10%或建筑速度+20% / Production +10% or building speed +20% | ⭐⭐⭐⭐⭐ |
| **宗教城邦 / Religious** | 信仰+10%或传教士力量+2 / Faith +10% or missionary strength +2 | ⭐⭐⭐ |

**城邦攻略策略 / City-State Strategy**:
1. **抢首发 / First Meet Bonus**: 第一个发现城邦自动获得1使者——抢首发很重要 / First to meet gets 1 free envoy — race for first contact
2. **派使者 / Send Envoys**: 3个使者获得宗主国地位——每轮派一个免费使者 / 3 envoys = suzerain — send 1 free envoy per turn
3. **做任务 / Complete Quests**: 城邦有时会发布任务——完成获得额外使者 / City-states sometimes give quests — complete for bonus envoys
4. **保护城邦 / Protect Them**: 被宣战后宗主国会失去宗主地位——要及时保护 / If declared war on, suzerain loses status — defend them

## 外交系统 / Diplomacy System

| 外交行动 / Diplomatic Action | 说明 / Description | 影响力消耗 / Influence Cost |
|---|---|---|
| **派遣使者 / Send Envoy** | 向城邦派遣使者 / Send envoy to city-state | 15 |
| **缔结贸易 / Establish Trade** | 与AI文明建立贸易路线 / Establish trade route with AI | 免费 / Free |
| **声明友谊 / Declare Friendship** | 提升关系——防止突袭 / Improve relations — prevent surprise war | 20 |
| **缔结同盟 / Alliance** | 军事/科技/文化同盟——共享部分收益 / Military/Science/Culture alliance — share benefits | 50 |
| **谴责 / Denounce** | 降低对方声望——为宣战做准备 / Reduce their reputation — prepare for war | 10 |
| **宣战 / Declare War** | 不谴责直接宣战会有外交惩罚 / Declaring without denounce has diplomatic penalty | — |
| **提案制裁 / Propose Sanctions** | 联合国（近代）中提出制裁 / Propose sanctions in UN (Modern age) | — |

**外交技巧 / Diplomatic Tips**:
- 不要随便谴责——AI会记住你的行为 / Don't denounce casually — AI remembers your actions
- 同盟好处巨大——优先找文化/科技相近的AI结盟 / Alliances are powerful — prioritize civs with similar culture/science
- 贸易路线不仅给金币——还提升双方关系 / Trade routes boost both gold and relations
- 中立城邦派使者——战争状态下也可以 / Send envoys to neutral city-states — even during war
