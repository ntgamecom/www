# 胜利之路 / Victory Strategies

文明7有四种胜利路线——本指南为你详解每条路线的关键步骤 / Civ VII has 4 victory types — this guide details the key steps for each path.

---

## 科技胜利 / Science Victory

**目标**: 完成所有太空项目——率先殖民系外行星 / Goal: Complete all space projects — first to colonize an exoplanet

### 关键步骤 / Key Steps

| 阶段 / Phase | 行动 / Actions | 时代 / Age |
|---|---|---|
| **科技基础 / Science Foundation** | 多建学院和图书馆——每城必建 / Build campuses and libraries in every city | 古典 / Antiquity |
| **大科学家 / Great Scientists** | 招募大科学家——给大量科技加成 / Recruit Great Scientists — huge science bonuses | 全时代 / All ages |
| **工业产能 / Industrial Power** | 建工厂、发电站——提升生产力用于太空项目 / Build factories, power plants — boost prod for space projects | 近代 / Modern |
| **太空项目 / Space Projects** | 建太空港→发射卫星→载人航天→殖民飞船 / Build Spaceport → Launch Satellite → Manned Spaceflight → Colony Ship | 近代 / Modern |

### 最佳文明组合 / Best Civ Combos

| 文明 / Civ | 理由 / Reason |
|---|---|
| **玛雅→阿拔斯→德国** | 科技加成逐代叠加——最终科技碾压 / Science bonuses stack each age — eventual science domination |
| **罗马→诺曼→美国** | 适应性强——铺城多=科技快 / Flexible — more cities = faster science |

### 科技胜利技巧 / Science Victory Tips

| # | 技巧 / Tip |
|---|---|
| 1 | **科技城邦必抢**——一个科技城邦加10%科技，三个就加30% / Prioritize scientific city-states — each adds 10% science |
| 2 | **图书馆/大学/研究所每个城都要建**——科技建筑是科技产出的基石 / Build library/university/lab in every city — science buildings are the foundation |
| 3 | **屯大科学家**——他们的即时科技提升在最后冲刺阶段极其关键 / Save Great Scientists — their instant tech boost is critical in the final sprint |
| 4 | **太空港所在城要有高生产力**——要么建在工业区旁边，要么用工程师加速 / The spaceport city needs high production — build near industrial zone or use engineers |
| 5 | **理性主义政策卡**——科技胜利路线必装 / Rationalism policy card — essential for science victory |

## 文化胜利 / Culture Victory

**目标**: 文化影响力覆盖所有其他文明 / Goal: Cultural influence reaches all other civilizations

### 关键步骤 / Key Steps

| 阶段 / Phase | 行动 / Actions | 时代 / Age |
|---|---|---|
| **文化基础 / Culture Foundation** | 建纪念碑、剧院广场——每城产出文化 / Build monuments, theater squares — culture per city | 古典 / Antiquity |
| **文物与杰作 / Artifacts & Masterpieces** | 建博物馆放杰作——文化主题化给额外加成 / Build museums for masterpieces — theming bonuses | 探索 / Exploration |
| **旅游收益 / Tourism** | 建海滩度假区、国家公园——旅游是文化胜利的核心 / Build resorts, national parks — tourism is the core of culture victory | 近代 / Modern |
| **开放边界 / Open Borders** | 与所有文明签开放边界——游客量翻倍 / Open borders with all civs — doubles tourism output | 全时代 / All ages |

### 最佳文明组合 / Best Civ Combos

| 文明 / Civ | 理由 / Reason |
|---|---|
| **希腊→法国→美国** | 文化加成逐代叠加——大艺术家+博物馆优势 / Culture stacking — great artist + museum bonuses |
| **埃及→诺曼→法国** | 早期文化建筑+后期杰作加成 / Early culture buildings + late masterpiece bonuses |

### 文化胜利技巧 / Culture Victory Tips

| # | 技巧 / Tip |
|---|---|
| 1 | **开边界协议**——与每个AI签开放边界，你的旅游输出翻倍 / Sign open borders with every AI — doubles tourism output |
| 2 | **不同文明的游客算法**——你的旅游高于对方的文化产出时开始获得游客 / Tourism > their culture = tourists start flowing |
| 3 | **主题博物馆**——同一个博物馆里放同文明同时代的艺术品给额外加成 / Themed museums — same civ/era art in one museum gives bonus |
| 4 | **国家公园**——自然学家的关键作用——后期旅游主要来源之一 / National parks — naturalist's key role — major late-game tourism source |
| 5 | **电脑不用赢文化**——AI不爱走文化胜利，所以时间上相对充裕 / AI rarely goes for culture victory — you have more time |

## 军事胜利 / Military Victory

**目标**: 占领或征服所有其他文明的初始首都 / Goal: Capture or conquer all other civs' original capitals

### 关键步骤 / Key Steps

| 阶段 / Phase | 行动 / Actions | 时代 / Age |
|---|---|---|
| **早期军队 / Early Army** | 建兵营——生产足够军队——至少4-6个单位 / Build barracks — produce enough army — at least 4-6 units | 古典 / Antiquity |
| **指挥官升级 / Commander Leveling** | 指挥官带队作战升级——解锁技能树 / Commander leads battles to level up — unlock skill tree | 全时代 / All ages |
| **时代军队更新 / Age Army Upgrade** | 每个时代升级军队——远古单位打不过火枪手 / Upgrade army each age — ancient units can't beat muskets | 每个时代 / Each age transition |
| **首都收割 / Capital Hunt** | 找到并占领所有AI的初始首都——不需要灭国 / Find and capture all AI original capitals — don't need to eliminate them | 近代 / Modern |

### 最佳文明组合 / Best Civ Combos

| 文明 / Civ | 理由 / Reason |
|---|---|
| **阿兹特克→蒙古→日本** | 全程战争加成——奢侈越多战斗力越高 / Permanent war bonuses — more luxuries = stronger |
| **罗马→诺曼→德国** | 早期扩张+中期攻城+后期工业军事产能 / Early expansion + mid siege + late industrial military prod |

### 军事胜利技巧 / Military Victory Tips

| # | 技巧 / Tip |
|---|---|
| 1 | **指挥官不要送死**——升级到高等级的指挥官比3个单位还重要 / Don't feed commanders — a high-level commander is worth 3 units |
| 2 | **主力部队配合攻城武器**——剑士+投石机的组合 / Main army + siege weapons — swordsmen + catapults combo |
| 3 | **后方城专注产能**——征服的城专注生产军队 / Core cities focus on production — conquered cities spam units |
| 4 | **别被多线战争拖累**——各个击破，签好停战再打下一个 / Don't fight multi-front wars — defeat one by one, then sign peace |
| 5 | **城邦盟友很重要**——军事城邦给你白送军队 / City-state allies matter — militaristic ones give free units |

## 经济胜利 / Economic Victory

**目标**: 建立全球贸易网络——积累足够财富 / Goal: Build global trade network — accumulate enough wealth

### 关键步骤 / Key Steps

| 阶段 / Phase | 行动 / Actions | 时代 / Age |
|---|---|---|
| **贸易基础 / Trade Foundation** | 早期建商业区——尽快建立贸易路线 / Build commercial hubs early — establish trade routes ASAP | 古典 / Antiquity |
| **贸易路线扩张 / Trade Expansion** | 多建商队/商船——每条路线带来金币+其他收益 / Build more traders — each route gives gold + other yields | 全时代 / All ages |
| **伟人招募 / Great People** | 抢大商人——给大量金币和贸易相关加成 / Recruit Great Merchants — massive gold and trade bonuses | 全时代 / All ages |
| **全球贸易 / Global Network** | 与每个AI建立贸易路线——控制所有城邦 / Trade with every AI — control all city-states | 近代 / Modern |

### 最佳文明组合 / Best Civ Combos

| 文明 / Civ | 理由 / Reason |
|---|---|
| **埃及→葡萄牙→美国** | 贸易路线收益翻倍——金币爆炸 / Trade route bonuses doubled — gold explosion |
| **腓尼基→葡萄牙→美国** | 海军贸易路线优势——跨大陆贸易翻倍 / Naval trade advantage — cross-continent trade doubled |

### 经济胜利技巧 / Economic Victory Tips

| # | 技巧 / Tip |
|---|---|
| 1 | **金币就是万能**——用金币买建筑、买单位、买伟人 / Gold is versatile — buy buildings, units, great people |
| 2 | **商队/商船不要停**——始终保持所有贸易路线在运行 / Keep all trade routes active — never let one idle |
| 3 | **商业城邦必抢**——商业城邦给贸易路线容量+金币加成 / Prioritize commercial city-states — they give trade route cap + gold bonus |
| 4 | **灯塔+港口**——海滨城市的贸易路线给额外金币 / Lighthouse + Harbor — coastal city trade routes give bonus gold |
| 5 | **后期用金币买太空项目**——科技胜利也可以用金币加速 / Late game, buy space projects with gold — accelerates science victory too |

## 传承路线全攻略 / Legacy Path Guide

| 时代 / Age | 传承路线 / Legacy Path | 要求 / Requirements | 奖励 / Reward |
|---|---|---|---|
| **古典 / Antiquity** | 文化传承 / Culture | 建造7个特色建筑 / Build 7 unique buildings | 下个时代开局+2文化每城 / +2 culture per city next age |
| **古典 / Antiquity** | 军事传承 / Military | 消灭15个敌军单位 / Kill 15 enemy units | 下个时代开局+5战斗力 / +5 combat strength next age |
| **探索 / Exploration** | 经济传承 / Economic | 建立7个贸易路线 / Establish 7 trade routes | 下个时代开局+2贸易路线容量 / +2 trade route cap next age |
| **探索 / Exploration** | 宗教传承 / Religious | 让3个文明信仰你的宗教 / Convert 3 civs to your religion | 下个时代开局+5信仰每城 / +5 faith per city next age |
| **近代 / Modern** | 科技传承 / Science | 完成3个太空项目 / Complete 3 space projects | 最终科技胜利所需项目-1 / -1 space project for science victory |
| **近代 / Modern** | 经济传承 / Economic | 积累10000金币 / Accumulate 10000 gold | 最终经济胜利所需财富-20% / -20% wealth needed for economic victory |

**核心建议 / Core Advice**: 每个时代必须完成至少1条传承路线——最好完成2条。不做传承路线的后果是下个时代开局落后AI一大截 / Complete at least 1 Legacy Path per age — ideally 2. Skipping Legacy Paths puts you far behind AI next age.
