## 新手入门指南 / Beginner Guide

欢迎来到泰拉瑞亚的世界。本攻略将帮助你从零开始，逐步掌握游戏的基础知识和生存技巧。

Welcome to the world of Terraria. This guide will help you start from zero and gradually master the basics and survival skills of the game.

### 世界创建 / World Creation

在开始游戏之前，你需要创建一个世界。世界的大小和难度将直接影响你的游戏体验。

Before starting the game, you need to create a world. The world size and difficulty will directly affect your gaming experience.

**世界大小 / World Size**:

| 大小 / Size | 优点 / Pros | 缺点 / Cons |
|---|---|---|
| 小 / Small | 资源集中、快速探索 / Resource dense, fast exploration | 空间有限、后期资源可能不足 / Limited space, may lack late-game resources |
| 中 / Medium | 平衡体验 / Balanced experience | 无显著缺点 / No significant downsides |
| 大 / Large | 资源丰富、探索时间长 / Abundant resources, long exploration | 跑图耗时、不方便 / Time-consuming travel, inconvenient |

**推荐 / Recommendation**: 新手选择 **中世界 (Medium World)**，经典模式 (Classic Mode) 最为合适。不建议一上来就玩专家或大师模式。

### 创建角色 / Character Creation

创建角色时你可以自定义外观，但这不影响游戏属性。角色可以进入任何世界，也可以在多个世界之间切换。

When creating a character, you can customize the appearance, but this does not affect game stats. Characters can enter any world and switch between multiple worlds.

### 第一天 / The First Day

游戏的第一天至关重要。以下是你的行动清单：

The first day of the game is crucial. Here is your action checklist:

1. **砍树 / Chop Trees**: 用斧头砍树获取木材 (Wood)，这是最重要的基础材料。
2. **建造房屋 / Build a Shelter**: 在日落之前至少建造一个简易房屋。NPC会在你建造合适的房间后入住。
3. **收集资源 / Gather Resources**: 采集石头 (Stone)、土块 (Dirt)、仙人掌 (Cactus) 等材料。
4. **探索地表 / Explore the Surface**: 寻找宝箱 (Chests) 和生命水晶 (Life Crystals)。
5. **制作工作台 / Craft a Work Bench**: 用木材制作工作台，这是大部分合成的基础。

**第一天房屋最小要求 / Minimum House Requirements**:
- 大小至少 4x8 格（内部空间）/ Minimum 4x8 tiles (interior space)
- 墙壁完整覆盖 / Fully enclosed with walls
- 至少有一张桌子/工作台和一把椅子 / At least one table/work bench and one chair
- 光源（火把）/ Light source (torches)
- 有一个入口（门）/ An entrance (door)

### 第一个夜晚 / The First Night

夜晚是泰拉瑞亚最危险的时刻。僵尸 (Zombies) 和恶魔眼 (Demon Eyes) 会在地表生成。

Night time is the most dangerous time in Terraria. Zombies and Demon Eyes will spawn on the surface.

**夜间生存技巧 / Night Survival Tips**:
- 躲在房屋内 / Stay inside your house
- 在门口放置物块挡住僵尸 / Place blocks at the door to block zombies
- 使用弓箭或手里剑远程攻击 / Use bows or shurikens for ranged attacks
- 如果血量低，吃蘑菇 (Mushrooms) 回血 / Eat mushrooms to heal if low on health
- 可以挖矿度过夜晚 / You can mine through the night

### 第一个NPC / The First NPC

**向导 (Guide)** 是游戏开始时就已经存在的NPC。他可以为你提供合成提示，只需将材料拿给他即可看到所有可能的合成配方。

The Guide is the NPC that already exists at the start of the game. He can provide crafting tips — just show him a material to see all possible crafting recipes.

其他早期入住的NPC / Other early-moving NPCs:

| NPC | 入住条件 / Move-in Condition | 作用 / Function |
|---|---|---|
| 商人 / Merchant | 玩家拥有超过50银币 / Player has over 50 silver coins | 出售基础工具和物品 / Sells basic tools and items |
| 护士 / Nurse | 玩家已有超过100血量 / Player has over 100 HP | 治疗玩家、移除debuff / Heals player, removes debuffs |
| 爆破专家 / Demolitionist | 玩家拥有炸药 / Player has explosives | 出售炸弹和手雷 / Sells bombs and grenades |
| 染料商 / Dye Trader | 玩家拥有任何染料材料 / Player has any dye material | 染料相关 / Dye-related |
| 动物学家 / Zoologist | 图鉴完成10% / Bestiary 10% complete | 出售坐骑和宠物 / Sells mounts and pets |

### 早期工具与武器 / Early Tools and Weapons

**必备工具 / Essential Tools**:

| 工具 / Tool | 材料 / Material | 作用 / Function |
|---|---|---|
| 铜镐 / Copper Pickaxe | 铜锭 x12 + 木材 x4 | 挖掘基本方块 / Basic mining |
| 铜斧 / Copper Axe | 铜锭 x9 + 木材 x3 | 砍树 / Chopping trees |
| 铜剑 / Copper Broadsword | 铜锭 x8 + 木材 x2 | 基础近战武器 / Basic melee weapon |
| 木弓 / Wooden Bow | 木材 x10 | 远程攻击 / Ranged attack |
| 木箭 / Wooden Arrow | 木材 x1 + 石块 x1 (x25) | 弹药 / Ammunition |

尽快升级到 **铁/铅工具 (Iron/Lead Tools)**，这能显著提高效率。

Upgrade to Iron/Lead tools as soon as possible to significantly improve efficiency.

### 矿石升级路径 / Ore Upgrade Path

泰拉瑞亚有两种矿石系列，每个世界只会生成其中一种：

Terraria has two ore series, and each world generates only one of them:

| 系列一 / Tier 1 | 系列二 / Tier 2 | 制作工具 / Tool Type |
|---|---|---|
| 铜 / Copper | 锡 / Tin | 最基础 / Most basic |
| 铁 / Iron | 铅 / Lead | 基础 / Basic |
| 银 / Silver | 钨 / Tungsten | 中级 / Intermediate |
| 金 / Gold | 铂金 / Platinum | 高级 / Advanced |
| 魔矿 / Demonite | 猩红矿 / Crimtane | 前期顶级 / Pre-Hardmode top |

### 城镇建造 / Town Building

NPC城镇的布局会影响你的游戏效率：

The layout of NPC towns affects your game efficiency:

- 每个NPC需要独立的房屋 / Each NPC needs a separate house
- 房屋之间不要太远 / Houses should not be too far apart
- 建议建造多层房屋 / Building multi-story houses is recommended
- 使用平台 (Platforms) 做天花板可以节省材料 / Using platforms as ceilings saves materials
- 建造NPC监狱（最小合格房间）可以最大化空间效率 / Building NPC prisons (minimum valid rooms) maximizes space efficiency

**NPC居住偏好 / NPC Biome Preferences**:
| 生物群系 / Biome | 喜欢居住的NPC / NPC Preferences |
|---|---|
| 森林 / Forest | 向导、商人 / Guide, Merchant |
| 沙漠 / Desert | 护士、军火商 / Nurse, Arms Dealer |
| 丛林 / Jungle | 树妖、巫医 / Dryad, Witch Doctor |
| 雪地 / Snow | 机械师、机器侠 / Mechanic, Cyborg |
| 海洋 / Ocean | 海盗、渔夫 / Pirate, Angler |

### 职业选择建议 / Class Selection Advice

对于新手玩家，以下是各个职业的入门建议：

For new players, here are the entry-level suggestions for each class:

**近战 (Melee)**: 最推荐新手。高防御意味着更高的容错率。前期使用木剑或仙人掌剑过渡，尽快制作附魔剑 (Enchanted Sword) 或获取村正 (Muramasa)。

**远程 (Ranged)**: 较为安全，但需要注意弹药管理。前期使用木弓加火焰箭 (Flaming Arrows) 火力不错。

**魔法 (Magic)**: 前期较为困难，魔力消耗大。建议先玩其他职业积累经验。可以使用宝石法杖 (Gem Staves) 作为起步魔法武器。

**召唤 (Summoner)**: 对新手来说最具挑战性。召唤师防御低、操作复杂。不过如果熟悉了游戏，召唤师非常强力。

**混合 (Mixed)**: 新手不需要严格限定职业，灵活使用各种武器更容易生存。

### 探索要点 / Exploration Tips

- 多准备火把 (Torches) 照亮洞穴 / Prepare plenty of torches to light up caves
- 使用回忆药水 (Recall Potion) 或魔法镜 (Magic Mirror) 快速回城 / Use Recall Potion or Magic Mirror to return home quickly
- 注意聆听宝箱和生命水晶的叮当声 / Listen for the jingle of chests and life crystals
- 探索时随身携带平台和绳子 / Carry platforms and ropes while exploring
- 在地面上放置床 (Bed) 可以设置重生点 / Placing a bed sets your spawn point
- 食物增益很重要，制作碗装汤 (Bowl of Soup) 提供基础增益 / Food buffs are important, craft Bowl of Soup for a basic buff

### 常见陷阱 / Common Pitfalls

- 不要在天黑后还在地表游荡 / Do not wander on the surface after dark
- 不要随意打开洞穴中的宝箱，可能会触发陷阱 / Do not open chests in caves randomly, they may trigger traps
- 不要在血量低时挑战BOSS / Do not challenge bosses with low health
- 不要把NPC房屋建在腐化/猩红之地附近，NPC可能搬出 / Do not build NPC houses near Corruption/Crimson, NPCs may move out
- 保存金币去商人处购买重要物品 / Save gold coins to buy important items from the Merchant
