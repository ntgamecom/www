## 事件攻略指南 / Events Guide

泰拉瑞亚拥有多种游戏事件，每个事件都有独特的敌人和奖励。掌握事件的触发条件和最佳攻略方法可以让你获得稀有物品。

Terraria features multiple game events, each with unique enemies and rewards. Mastering event triggers and optimal strategies helps you obtain rare items.

### 血月 / Blood Moon

血月是一个基础事件，让夜晚变得更加危险。

Blood Moon is a basic event that makes the night more dangerous.

**触发条件 / Trigger Condition**:
- 当玩家拥有至少120血量时，每晚有约11.1%概率触发 / When player has 120+ HP, about 11.1% chance each night
- 可以使用血泪 (Bloody Tear) 手动触发 / Can be manually triggered with Bloody Tear

**事件特点 / Event Features**:
- 月亮变成红色 / Moon turns red
- 僵尸移动速度加快，开门能力增强 / Zombies move faster, can open doors
- 池塘中的鱼会攻击玩家 / Fish in ponds attack players
- 新增多种敌人 / New enemy types spawned

**新增敌人 / New Enemies**:

| 敌人 / Enemy | HP | 掉落物 / Drops |
|---|---|---|
| 血腥僵尸 / Blood Zombie | 100 | 破蝙蝠翼 (Broken Bat Wing) |
| 滴落者 / Drippler | 140 | 血泪 (Bloody Tear)、鲨塔 / Bloody Tear, Shark Tooth |
| 恐怖狗鱼 / Goblin Shark | 200 | 仅水中 / Water only |

**策略 / Strategy**:
- 建议在家门口搭建封闭战斗平台 / Build an enclosed fighting platform near home
- 利用此事件刷取血泪用于将来手动触发 / Use this event to farm Bloody Tears for future use
- 注意保护NPC，僵尸会进屋攻击 / Protect NPCs, zombies can enter houses

### 哥布林军团 / Goblin Army

哥布林军团是游戏前期的入侵事件，击败后可以获得哥布林工匠NPC。

Goblin Army is a pre-Hardmode invasion event. Defeating it unlocks the Goblin Tinkerer NPC.

**触发条件 / Trigger Condition**:
- 玩家拥有至少200血量，并已破坏至少一个暗影之珠/猩红之心 / Player has 200+ HP and has broken at least one Shadow Orb/Crimson Heart
- 使用哥布林战旗 (Goblin Battle Standard) 手动触发 / Use Goblin Battle Standard to manually trigger

**事件特点 / Event Features**:
- 哥布林会从地图两侧入侵 / Goblins invade from both sides of the map
- 击败足够数量后事件结束 / Event ends after defeating enough enemies
- 击败后哥布林工匠会出现在地下洞穴中 / Goblin Tinkerer appears in underground caves after defeat

**敌人列表 / Enemy List**:

| 敌人 / Enemy | 特点 / Characteristics |
|---|---|
| 哥布林战士 / Goblin Warrior | 基础近战 / Basic melee |
| 哥布林弓手 / Goblin Archer | 远程攻击 / Ranged attack |
| 哥布林魔法师 / Goblin Sorcerer | 传送 + 魔法攻击 / Teleports + magic |
| 哥布林掷弹兵 / Goblin Peon | 投掷手雷 / Throws grenades |
| 哥布林召唤师 / Goblin Summoner | 召唤暗影仆从 / Summons shadow minions |

**策略 / Strategy**:
- 在基地两侧建造战斗平台 / Build battle platforms on both sides of base
- 使用范围攻击武器清理成群的哥布林 / Use AoE weapons to clear groups
- 优先击杀魔法师和掷弹兵 / Prioritize Sorcerer and Peon

### 霜月军团 / Frost Legion

霜月军团是圣诞节相关的事件。

Frost Legion is a Christmas-themed event.

**触发条件 / Trigger Condition**:
- 困难模式后使用雪球 (Snow Globe) 触发 / Use Snow Globe in Hardmode
- 雪球在圣诞节期间可以从礼物中获得 / Snow Globe obtained from presents during Christmas

**敌人列表 / Enemy List**:

| 敌人 / Enemy | HP | 掉落物 / Drops |
|---|---|---|
| 雪人 / Snowman | 150 | 无 / None |
| 雪人暴徒 / Snowman Gangsta | 200 | 无 / None |

**策略 / Strategy**: 与哥布林军团类似，使用范围武器清理。

### 海盗入侵 / Pirate Invasion

海盗入侵是困难模式事件，击败后可以获得海盗NPC。

Pirate Invasion is a Hardmode event. Defeating it unlocks the Pirate NPC.

**触发条件 / Trigger Condition**:
- 困难模式中玩家拥有200+血量 / Hardmode, player has 200+ HP
- 使用海盗地图 (Pirate Map) 触发 / Use Pirate Map to trigger
- 海盗地图可以从海洋中的敌人身上掉落 / Pirate Map drops from ocean biome enemies

**敌人列表 / Enemy List**:

| 敌人 / Enemy | HP | 特点 / Characteristics |
|---|---|---|
| 海盗水手 / Pirate Deckhand | 200 | 基础近战 / Basic melee |
| 海盗射手 / Pirate Corsair | 300 | 使用火枪 / Uses musket |
| 海盗船长 / Pirate Captain | 800 | 发射炮弹 / Fires cannonballs |
| 鹦鹉 / Parrot | 150 | 飞行敌人 / Flying enemy |
| 海盗船 / Pirate Ship | 2000 | 大型飞行BOSS / Large flying boss |

**策略 / Strategy**:
- 准备宽阔的场地应对大量敌人 / Prepare a wide area for many enemies
- 海盗船掉落价值极高的金币 / Pirate Ship drops very valuable gold coins
- 击败足够敌人后获得幸运金币 (Lucky Coin) 掉落机会 / Chance to drop Lucky Coin after defeating enough enemies

### 日食 / Solar Eclipse

日食是困难模式事件，拥有游戏中最难的普通敌人。

Solar Eclipse is a Hardmode event featuring some of the hardest normal enemies.

**触发条件 / Trigger Condition**:
- 困难模式中击杀任意机械BOSS后，每天有5%概率触发 / After killing any mech boss in Hardmode, 5% chance each day
- 使用日食石板 (Solar Tablet) 手动触发 / Use Solar Tablet to manually trigger
- 日食石板由丛林神庙掉落 / Solar Tablet drops from Jungle Temple

**敌人列表 / Enemy List**:

| 敌人 / Enemy | HP | 掉落物 / Drops |
|---|---|---|
| 沼泽怪 / Swamp Thing | 800 | 无 / None |
| 弗兰肯斯坦 / Frankenstein | 800 | 破蝙蝠翼 / Broken Bat Wing |
| 吸血鬼 / Vampire | 1500 | 破蝙蝠翼 (困难模式) / Broken Bat Wing (Hardmode) |
| 眼魔 / Eyezor | 2000 | 无 / None |
| 死神 / Reaper | 2000 | 死神兜帽 / Reaper Hood |
| 弗里茨博士 / Dr. Man Fly | 1200 | 无 / None |
| 螺旋转 / Nailhead | 2400 | 投掷钉枪 / Nail Gun |
| 蛾子 / Mothron | 3000 | 破碎英雄之剑、蛾子之翼 / Broken Hero Sword, Mothron Wings |
| 穿白衣的女性 / The Bride | 900 | 无 / None |

**策略 / Strategy**:
- 日食是获取破碎英雄之剑合成泰拉之刃的最佳途径 / Best way to get Broken Hero Sword for Terra Blade
- 在封闭空间内战斗可以限制吸血鬼的行动 / Fight in enclosed spaces to limit Vampire movement
- 使用水蜡烛增加刷新率以更快累积击杀 / Use Water Candle to increase spawn rate

### 南瓜月 / Pumpkin Moon

南瓜月是困难模式后的季节性事件，有多个波次。

Pumpkin Moon is a post-Hardmode seasonal event with multiple waves.

**触发条件 / Trigger Condition**:
- 夜晚使用南瓜月勋章 (Pumpkin Moon Medallion) / Use Pumpkin Moon Medallion at night
- 合成材料：30个南瓜 + 5个精灵尘 + 5个魂 / 30 Pumpkins + 5 Pixie Dust + 5 Souls

**波次敌怪 / Wave Enemies**:

| 敌人 / Enemy | 掉落物 / Drops |
|---|---|
| 稻草人 / Scarecrow | 无 / None |
| 乌鸦 / Splinterling | 无 / None |
| 地狱犬 / Hellhound | 无 / None |
| 胡狼 / Giant Flying Fox | 蝙蝠翼 / Bat Wing |
| 波兰人 / Poltergeist | 无 / None |
| 南瓜王 / Pumpking | 死亡镰刀、蝙蝠权杖 / The Horseman's Blade, Bat Scepter |
| 哀木 / Mourning Wood | 尖刺弓、吸血鬼法杖 / Stake Launcher, Raven Staff |

**策略 / Strategy**:
- 南瓜月是获取大量金币的好方法 / Great source of gold coins
- 使用高伤害范围武器快速清怪 / Use high-damage AoE weapons to clear quickly
- 波次越高奖励越好，保持生存到最后一波 / Higher waves = better rewards, survive to the final wave

### 霜月 / Frost Moon

霜月是南瓜月的升级版，难度更高。

Frost Moon is an upgraded version of Pumpkin Moon, with higher difficulty.

**触发条件 / Trigger Condition**:
- 夜晚使用霜月勋章 (Frost Moon Medallion) / Use Frost Moon Medallion at night
- 合成材料：20个丝绸 + 5个魂核 + 5个寒冬之心 / 20 Silk + 5 Souls of Fright + 5 Frost Cores

**敌人列表 / Enemy List**:

| 敌人 / Enemy | 难度 / Difficulty | 掉落物 / Drops |
|---|---|---|
| 姜饼人 / Gingerbread Man | 低 / Low | 无 / None |
| 精灵 / Elf | 中 / Medium | 无 / None |
| 雪人 / Yeti | 中 / Medium | 无 / None |
| 驯鹿 / Reindeer | 中 / Medium | 驯鹿坐骑 / Reindeer Mount |
| 冰女王 / Ice Queen | 高 / High | 暴雪法杖、冰霜鳞甲 / Blizzard Staff, Frost Queen Wings |
| 圣诞老人坦克 / Santa-NK1 | 高 / High | 链式机枪、糖果弓 / Chain Gun, Elf Fusiliers |
| 坦克 / Everscream | 中 / Medium | 松针法杖 / Razor Pine |

**策略 / Strategy**:
- 霜月是最赚钱的事件之一 / Frost Moon is one of the most profitable events
- 准备充分的药水和增益 / Prepare ample potions and buffs
- 链式机枪 (Chain Gun) 是霜月的珍贵掉落物 / Chain Gun is a prized drop from Frost Moon

### 旧日军团 / Old One's Army

旧日军团是泰拉瑞亚与地牢守护者2 (Dungeon Defenders 2) 的联动事件。

Old One's Army is a crossover event between Terraria and Dungeon Defenders 2.

**触发条件 / Trigger Condition**:
- 在酒馆老板处购买滴到召唤物品 / Purchase summoning items from the Tavernkeep
- 滴到分三个难度等级，对应不同阶段的奖励 / Three difficulty tiers with different rewards

**敌人列表 / Enemy List**:

| 敌人 / Enemy | 特点 / Characteristics |
|---|---|
| 地精 / Etherian Goblin | 基础近战 / Basic melee |
| 独角兽 / Etherian Unicorn | 冲撞攻击 / Charge attack |
| 飞龙 / Etherian Wyvern | 飞行敌人 / Flying enemy |
| 双足飞龙 / Betsy | 最终BOSS / Final boss |
| 哥布林术士 / Etherian Goblin Sorcerer | 魔法攻击 / Magic attack |

**策略 / Strategy**:
- 使用酒馆老板出售的哨兵武器（手推车/弩炮）/ Use Tavernkeep's sentry weapons (ballista, flameburst)
- 注意保护水晶不被破坏 / Protect the crystal from being destroyed
- 击败贝琪可以获得强大的贝琪之翼 / Defeat Betsy for powerful Betsy Wings

### 火星暴乱 / Martian Madness

火星暴乱是困难模式后期的外星人入侵事件。

Martian Madness is a post-Hardmode alien invasion event.

**触发条件 / Trigger Condition**:
- 在击杀世纪之花后在世界的1/3高度以上击杀火星探测器 / Kill a Martian Probe above 1/3 world altitude after defeating Plantera
- 让火星探测器逃走来触发事件 / Let the probe escape to trigger the event

**敌人列表 / Enemy List**:

| 敌人 / Enemy | 特点 / Characteristics |
|---|---|
| 火星工程师 / Martian Engineer | 放置炮塔 / Places turrets |
| 火星军官 / Martian Officer | 发射追踪射线 / Fires homing rays |
| 灰色外星人 / Gray Martian | 快速近战 / Fast melee |
| 火星飞碟 / Martian Saucer | 最终BOSS，分三个阶段 / Final boss, three phases |

**策略 / Strategy**:
- 火星飞碟掉落宇宙车钥匙，可以无限飞行 / Martian Saucer drops Cosmic Car Key for infinite flight
- 使用高伤害远程武器攻击飞碟 / Use high-damage ranged weapons on the saucer
- 优先清理炮塔 / Prioritize clearing turrets
