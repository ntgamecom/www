## 建造指南 / Building Guide

建造是泰拉瑞亚的核心玩法之一。合理的建筑布局可以极大提升游戏体验。本指南将涵盖NPC房屋、竞技场、农场和电路系统。

Building is a core gameplay element in Terraria. Proper building layouts can greatly enhance the gaming experience. This guide covers NPC housing, arenas, farms, and wiring systems.

### NPC房屋要求 / NPC Housing Requirements

每个NPC都需要独立的房屋才能入住。以下是合格房屋的具体要求：

Each NPC needs an independent house to move in. Here are the specific requirements for a valid house:

**房屋合格标准 / House Validation Criteria**:

| 要求 / Requirement | 说明 / Description |
|---|---|
| 大小 / Size | 内部空间至少4x8格，最大不超过10x14格 / Interior at least 4x8, at most 10x14 |
| 墙壁 / Walls | 必须完全封闭，使用背景墙填充 / Fully enclosed with background walls |
| 光源 / Light Source | 至少一个光源（火把、蜡烛等）/ At least one light source (torch, candle, etc.) |
| 家具 / Furniture | 至少一张平坦表面物品（桌子/工作台）+ 一把椅子 / At least one flat surface item + one chair |
| 入口 / Entrance | 门、平台或机关门 / Door, platform, or actuated door |
| 环境 / Environment | 不能有腐化/猩红背景，方块数不能超过一定比例 / No Corruption/Crimson background, limited evil blocks |

**NPC房屋布局示例 / NPC Housing Layout Example**:

```
[ 墙墙墙墙墙墙墙墙墙墙 ]    [ = 墙 / Wall
[ 桌                         ]    桌 = 桌子 / Table
[             椅            ]    椅 = 椅子 / Chair
[ T      火把               ]    T = 火把 / Torch
[ 门                        ]    门 = Door
[ 墙墙墙墙墙墙墙墙墙墙 ]
```

**NPC监狱 / NPC Prison**:
- 最小有效房屋：4格宽 x 6格高（内部）/ Minimum valid house: 4 wide x 6 high (interior)
- 使用平台作为地板和天花板 / Use platforms for floor and ceiling
- 适合大量NPC入驻 / Suitable for mass NPC housing

### NPC房屋布局建议 / NPC Housing Layout Tips

**集中式布局 / Centralized Layout**:
- 将NPC房屋建在基地中心，方便快速找到 / Build NPC houses in the center of your base for easy access
- 使用多层设计节省空间 / Use multi-story design to save space
- 每个房间可以放入相同的装饰套装 / Each room can have the same furniture set

**生物群系偏好 / Biome Preferences**:
- 某些NPC喜欢特定生物群系，可以在相应生物群系建造房屋以获得好感度加成 / Some NPCs prefer specific biomes, building houses there gives happiness bonuses
- 好感度影响售卖价格 / Happiness affects selling prices
- 商人喜欢森林 (Forest)、军火商喜欢沙漠 (Desert)、树妖喜欢丛林 (Jungle) / Merchant likes Forest, Arms Dealer likes Desert, Dryad likes Jungle

**混居设计 / Mixed Housing Design**:
- 可以将多个NPC房屋建在同一个建筑内 / Multiple NPC houses can be in the same building
- 每个房间独立但共享走廊 / Each room is independent but shares a hallway
- 这样可以集中防御和装饰 / This centralizes defense and decoration

### BOSS竞技场 / Boss Arenas

优秀的竞技场设计可以大幅提高BOSS战的胜率。

A well-designed arena can significantly increase your chances of winning boss fights.

**通用竞技场 / General Arena**:

| 要素 / Element | 建议 / Suggestion |
|---|---|
| 层数 / Layers | 3-5层平台，间距约6格 / 3-5 platform layers, spacing about 6 tiles |
| 宽度 / Width | 约200-300格 / About 200-300 tiles |
| 材料 / Material | 木平台 (Wood Platforms) 或任意平台 / Wood Platforms or any platform |
| 篝火 / Campfire | 每层放置多个，提供生命恢复 / Multiple per layer for life regen |
| 巢穴心 / Heart Lantern | 悬挂在平台上方 / Hang above platforms |
| 星星瓶 / Star in a Bottle | 魔法职业需要 / Needed for magic classes |
| 补给箱 / Supply | 平台两端放置弹药和药水 / Ammo and potions at platform ends |

**血肉墙竞技场 / Wall of Flesh Arena**:
- 在地狱搭建一条跨越整个地图的桥 / Build a bridge spanning the entire Underworld
- 使用实体方块或者平台 / Use solid blocks or platforms
- 桥面距离岩浆约10格 / Bridge about 10 tiles above lava
- 每隔一段放置篝火和巢穴心 / Place campfires and heart lanterns periodically

**月亮领主竞技场 / Moon Lord Arena**:
- 超长平台（约400-500格）/ Very long platforms (about 400-500 tiles)
- 使用星云灯笼 (Nebula Lamp) 和星星瓶 / Use Nebula Lamp and Star in a Bottle
- 放置蜂蜜池 (Honey Pools) 提供恢复 / Place Honey Pools for healing
- 使用心形雕像 (Heart Statues) 连接电路 / Use Heart Statues wired to switches

### 农场设计 / Farm Designs

**凝胶农场 / Gel Farm**:
- 建造一个简单的史莱姆房间 / Build a simple slime room
- 使用水蜡烛增加刷新率 / Use Water Candle to increase spawn rate
- 站在附近挂机即可 / Stand nearby and AFK
- 也可以使用史莱姆王冠召唤史莱姆王掉落更多凝胶 / Use Slime Crown to summon King Slime for more gel

**灵魂农场 / Soul Farms**:
- 在困难模式的地下腐化/神圣/猩红区域建造 / Build in Underground Corruption/Hallow/Crimson areas
- 设计刷怪平台，确保玩家在刷怪范围内 / Design spawn platforms ensuring player is within spawn range
- 使用战斗药水和水蜡烛 / Use Battle Potion and Water Candle
- 不同灵魂对应不同生物群系 / Different souls correspond to different biomes
  - 暗影之魂 (Soul of Night): 地下腐化/猩红 / Underground Corruption/Crimson
  - 光明之魂 (Soul of Light): 地下神圣 / Underground Hallow
  - 飞行之魂 (Soul of Flight): 天空飞龙 / Sky Wyverns

**生物群系钥匙农场 / Biome Key Farm**:
- 在海洋建造一个巨大的人工生物群系 / Build a massive artificial biome in the Ocean
- 使用1500+个对应生物群系的物块 / Use 1500+ blocks of the corresponding biome
- 水蜡烛加战斗药水增加刷怪效率 / Water Candle + Battle Potion for spawn efficiency
- 用于刷取生物群系钥匙 (Biome Keys)打 / Used for farming Biome Keys

**松露虫农场 / Truffle Worm Farm**:
- 在海洋中建造人工蘑菇生物群系 / Build an artificial Mushroom biome in the Ocean
- 使用200+个蘑菇泥块 (Mud + Glowing Mushroom) / Use 200+ mushroom mud blocks
- 松露虫是召唤猪鲨的关键物品 / Truffle Worms are key to summoning Duke Fishron

### 电路基础 / Wiring Basics

泰拉瑞亚的电路系统非常强大，可以实现自动化建筑。

Terraria's wiring system is very powerful and can create automated structures.

**基础材料 / Basic Materials**:

| 材料 / Material | 作用 / Function | 获取 / Acquisition |
|---|---|---|
| 电线 / Wire | 连接机关 / Connects mechanisms | 机械师购买 / Mechanic purchase |
| 开关 / Switch | 手动触发 / Manual trigger | 机械师购买 / Mechanic purchase |
| 压力板 / Pressure Plate | 自动触发 / Auto trigger | 机械师购买 / Mechanic purchase |
| 计时器 / Timer | 定时触发 / Timed trigger | 机械师购买 / Mechanic purchase |
| 致动器 / Actuator | 切换方块状态 / Toggle block state | 机械师购买 / Mechanic purchase |

**常用电路设计 / Common Circuit Designs**:

**自动门 / Automatic Door**:
```
[ 压力板 ] → [ 电线 ] → [ 门 ]
```
踩上压力板自动开门 / Step on pressure plate to auto-open door

**刷怪场 / Mob Farm**:
```
[ 计时器 ] → [ 电线 ] → [ 机关（长矛/火焰箭）]
```
定时触发攻击机关 / Timed triggered attack mechanisms

**心形雕像农场 / Heart Statue Farm**:
```
[ 计时器 ] → [ 电线 ] → [ 心形雕像 ]
```
每个雕像最多连接3个，使用1秒计时器 / Each statue can be connected to up to 3, use 1-second timer

**自动熔炉 / Auto Furnace**:
```
[ 箱子 ] → [ 漏斗 ] → [ 熔炉 ] → [ 漏斗 ] → [ 箱子 ]
```
使用漏斗 (Hopper) 传输物品 / Use Hoppers to transfer items

### 实用建筑技巧 / Useful Building Tips

- **挖掘隔离带 / Digging Isolation Trenches**: 在基地周围挖3格宽的空沟，防止腐化蔓延 / Dig 3-tile-wide empty trenches around base to prevent Corruption spread
- **直升机平台 / Helicopter Platform**: 在基地上空建造大型平台，方便空中移动和战斗 / Build large platforms above base for aerial movement and combat
- **水下基地 / Underwater Base**: 使用气泡机 (Bubble Machine) 制造空气泡 / Use Bubble Machine to create air pockets
- **空中基地 / Sky Base**: 在高空建造基地，远离地面敌人 / Build base at high altitude, away from ground enemies
- **蘑菇农场 / Mushroom Farm**: 在地下建造泥块层，种植发光蘑菇 / Build mud layers underground, plant Glowing Mushrooms
- **树场 / Tree Farm**: 种植大面积的树木，使用橡子 (Acorns) 种树 / Plant large tree areas using Acorns
- **蜂蜜池 / Honey Pool**: 在竞技场中放置蜂蜜池，提供生命恢复 / Place honey pools in arenas for life regen
- **NPC商店直达 / NPC Shop Access**: 使用平台和机关门创建快速通道 / Use platforms and actuated doors for fast access

### 装饰建议 / Decoration Tips

- **对应生物群系的主题装饰 / Themed Decorations**: 森林主题使用木家具，沙漠主题使用棕榈木 / Forest theme with wood furniture, desert theme with palm wood
- **使用油漆 / Using Paint**: 油漆可以改变方块颜色，不改变功能 / Paint changes block color without changing function
- **背景墙选择 / Background Wall Selection**: 不同背景墙营造不同氛围 / Different background walls create different atmospheres
- **家具套装 / Furniture Sets**: 每种木材都有对应的全套家具 / Each wood type has a complete furniture set
- **博物馆展柜 / Museum Displays**: 使用物品框 (Item Frame) 展示收集品 / Use Item Frames to display collectibles
