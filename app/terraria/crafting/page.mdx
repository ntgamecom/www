## 合成系统指南 / Crafting Guide

合成是泰拉瑞亚中获取装备和物品的核心方式。游戏拥有数十种合成站和数千种配方。本指南将帮助你掌握主要的合成路线。

Crafting is the core way to obtain equipment and items in Terraria. The game has dozens of crafting stations and thousands of recipes. This guide will help you master the main crafting routes.

### 重要配方链 / Important Recipe Chains

**泰拉之刃配方链 / Terra Blade Recipe Chain**:

泰拉之刃是困难模式最强的剑类武器之一，合成链非常复杂：

Terra Blade is one of the strongest swords in Hardmode, with a complex crafting chain:

1. 附魔剑 (Enchanted Sword) 或 村正 (Muramasa) / Enchanted Sword or Muramasa
2. 附魔剑 + 村正 + 草剑 + 菲亚林战剑 + 火山 → 永夜刃 / Enchanted Sword + Muramasa + Blade of Grass + Fiery Greatsword + Volcano → Night's Edge
3. 永夜刃 + 破碎英雄之剑 → 真永夜刃 / Night's Edge + Broken Hero Sword → True Night's Edge
4. 附魔剑 + 破碎英雄之剑 → 真附魔剑 / Enchanted Sword + Broken Hero Sword → True Excalibur
5. 真永夜刃 + 真附魔剑 → 泰拉之刃 / True Night's Edge + True Excalibur → Terra Blade

所需材料 / Required Materials:
- 永夜刃 / Night's Edge: 附魔剑 + 村正 + 草剑 + 菲亚林战剑 + 火山
- 破碎英雄之剑 / Broken Hero Sword: 怪物掉落（雕像兵团/日食）/ Mothron drop (Solar Eclipse)
- 真永夜刃 / True Night's Edge: 永夜刃 + 破碎英雄之剑
- 真附魔剑 / True Excalibur: 附魔剑 + 破碎英雄之剑
- 泰拉之刃 / Terra Blade: 真永夜刃 + 真附魔剑

**十字章护盾配方链 / Ankh Shield Recipe Chain**:

十字章护盾是最强大的饰品之一，提供多种免疫：

Ankh Shield is one of the strongest accessories, providing multiple immunities:

1. 钴蓝盾 (Cobalt Shield) → 黑曜石盾 (Obsidian Shield) 添加黑曜石 / Add Obsidian
2. 黑曜石盾 + 十字章吊坠 → 十字章护盾 / Obsidian Shield + Ankh Charm → Ankh Shield

十字章吊坠所需配件 / Ankh Charm components:
- 盔甲贝亮 (Armor Polish): 免疫破甲 / Immune to Broken Armor
- 药用绷带 (Medicated Bandage): 免疫流血 / Immune to Bleeding
- 维生素 (Vitamins): 免疫虚弱 / Immune to Weakness
- 粘性绷带 (Adhesive Bandage): 免疫伤口 / Immune to Wounds
- 面膜 (Bezoar): 免疫中毒 / Immune to Poison
- 扩音器 (Countercurse Mantra): 免疫沉默 / Immune to Silence
- 快走时钟 (Fast Clock): 免疫缓慢 / Immune to Slow
- 三桶 (Trifold Map): 免疫困惑 / Immune to Confused
- 盲眼 (Blindfold): 免疫黑暗 / Immune to Darkness
- 手机 (Megaphone): 免疫沉默 / Immune to Silence
- 纳扎尔 (Nazar): 免疫诅咒 / Immune to Cursed

**天界壳配方链 / Celestial Shell Recipe Chain**:

天界壳提供强大的全属性提升：

Celestial Shell provides powerful all-stat boosts:

1. 月亮石 (Moon Stone) + 太阳石 (Sun Stone) → 天界石 (Celestial Stone)
2. 天界石 + 月亮贝壳 (Moon Shell) → 天界壳 (Celestial Shell)

月亮贝壳合成 / Moon Shell crafting:
- 月光护身符 (Moon Charm): 狼人变身 / Werewolf transformation
- 海神贝壳 (Neptune's Shell): 水下呼吸 / Underwater breathing

### 材料获取 / Material Gathering

**矿石获取 / Ore Acquisition**:
- 前期矿石通过洞穴探索开采 / Early ores mined from cave exploration
- 困难模式矿石需要打破恶魔祭坛后才会生成 / Hardmode ores spawn after breaking Demon Altars
- 打破祭坛后依次生成钴蓝/秘银/精金（或钯金/山铜/钛金）/ Breaking altars spawns Cobalt/Mythril/Adamantite (or Palladium/Orichalcum/Titanium) in order
