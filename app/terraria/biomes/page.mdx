## 生物群系指南 / Biomes Guide

泰拉瑞亚的世界由多种生物群系 (Biomes) 组成，每个生物群系都有独特的敌人、资源和环境特点。了解这些生物群系对于高效探索和生存至关重要。

The world of Terraria consists of multiple biomes, each with unique enemies, resources, and environmental features. Understanding these biomes is essential for efficient exploration and survival.

### 森林 / Forest

森林是游戏中最基础的生物群系，也是玩家开始游戏的地方。这里相对安全，适合建立基地。

Forest is the most basic biome in the game and where players start the game. It is relatively safe and suitable for building a base.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 史莱姆 (Slimes)、僵尸 (Zombies)、恶魔眼 (Demon Eyes) |
| 独特资源 / Unique Resources | 木材 (Wood)、石料 (Stone)、橡子 (Acorns) |
| 植物 / Plants | 太阳花 (Daybloom)、树 (Trees) |
| 危险等级 / Danger Level | 低 / Low |

**夜间变化 / Nighttime Changes**: 夜晚僵尸会变得更多，还会出现恶魔眼。满月时僵尸能力增强。

### 沙漠 / Desert

沙漠位于世界的两侧，由沙块组成。地下沙漠 (Underground Desert) 在困难模式后变得更加危险且资源丰富。

The Desert is located on both sides of the world, composed of sand blocks. The Underground Desert becomes more dangerous and resource-rich after Hardmode.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 蚁狮 (Antlion)、秃鹫 (Vulture)、沙史莱姆 (Sand Slime) |
| 独特资源 / Unique Resources | 沙块 (Sand Block)、仙人掌 (Cactus)、化石 (Fossil) |
| 地下沙漠敌人 / Underground Desert Enemies | 蚁狮蜂 (Antlion Swarmer)、蚁狮战士 (Antlion Warrior) |
| 危险等级 / Danger Level | 中 / Medium |

**注意 / Note**: 困难模式后地下沙漠会生成强大的敌人如沙鲨 (Sand Shark)，挖掘时要格外小心。

### 雪地 / Snow Biome

雪地生物群系覆盖着冰雪，地表和地下都有独特的生态环境。

The Snow biome is covered with ice and snow, with unique ecosystems on the surface and underground.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 冰冻史莱姆 (Ice Slime)、雪怪 (Zombie Eskimo)、寒冰捕手 (Ice Elemental) |
| 独特资源 / Unique Resources | 冰雪块 (Ice Block)、雪块 (Snow Block)、寒铁矿 (Crimtane替代矿物) |
| 地下冰层 / Underground Ice | 冰霜核心 (Frost Core)、寒冰箱 (Frozen Chest) |
| 危险等级 / Danger Level | 中 / Medium |

**冰层陷阱 / Ice Traps**: 地下冰层有薄冰 (Thin Ice)，踩上去会碎掉掉落。使用荧光棒 (Glowsticks) 可以照亮深坑。

### 丛林 / Jungle

丛林是游戏中最危险但资源最丰富的生物群系之一。地下丛林 (Underground Jungle) 是游戏中期的重要地点。

The Jungle is one of the most dangerous yet resource-rich biomes. The Underground Jungle is a crucial location in the mid-game.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 丛林史莱姆 (Jungle Slime)、食人花 (Man Eater)、黄蜂 (Hornet) |
| 独特资源 / Unique Resources | 丛林孢子 (Jungle Spores)、毒刺 (Stinger)、藤蔓 (Vine) |
| 地下丛林 / Underground Jungle | 蜜蜂巢穴 (Bee Hive)、蜂后 (Queen Bee)战斗场地 |
| 危险等级 / Danger Level | 高 / High |

**重要物品 / Important Items**: 丛林是获得丛林盔甲 (Jungle Armor)、毒法杖 (Poison Staff)、荆棘旋刃 (Thorn Chakram) 的地方。地下丛林可以找到蜂蜡 (Bee Wax) 制作蜜蜂武器。

### 腐化之地 / Corruption

腐化之地是世界的邪恶生物群系之一（与猩红之地互斥），呈现出紫色的荒芜景象。

The Corruption is one of the world's evil biomes (mutually exclusive with Crimson), presenting a purple wasteland.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 吞噬怪 (Eater of Souls)、噬魂怪 (Devourer) |
| 独特资源 / Unique Resources | 魔矿 (Demonite Ore)、暗影之珠 (Shadow Orb)、腐化种子 (Corrupt Seeds) |
| 暗影之珠 / Shadow Orbs | 打破后掉落物品，第三次必掉火枪 (Musket) |
| 危险等级 / Danger Level | 中到高 / Medium to High |

**腐化扩展 / Corruption Spread**: 在困难模式后，腐化之地会快速蔓延到其他生物群系。使用圣种 (Hallowed Seeds) 可以创造神圣之地来对抗腐化。

### 猩红之地 / Crimson

猩红之地的猩红对应腐化之地，呈现出红色的血肉主题。

The Crimson is the counterpart to the Corruption, featuring a red fleshy theme.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 血腥僵尸 (Blood Zombie)、脸怪 (Face Monster)、猩红喀迈拉 (Crimera) |
| 独特资源 / Unique Resources | 猩红矿 (Crimtane Ore)、猩红之心 (Crimson Heart)、组织样本 (Tissue Sample) |
| 猩红之心 / Crimson Hearts | 打破后掉落物品，第三次必掉送葬者 (The Undertaker) |
| 危险等级 / Danger Level | 中到高 / Medium to High |

**腐化 vs 猩红 / Corruption vs Crimson**: 猩红之地的武器倾向于生命窃取和持续伤害，而腐化之地偏向召唤和范围攻击。

### 地牢 / Dungeon

地牢位于地图的一侧，入口处有地牢守护者 (Dungeon Guardian) 守卫。击败骷髅王 (Skeletron) 后才能安全进入。

The Dungeon is located on one side of the map, guarded by the Dungeon Guardian at the entrance. It can only be safely entered after defeating Skeletron.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 愤怒骷髅 (Angry Bones)、诅咒骷髅头 (Cursed Skull)、暗黑法师 (Dark Caster) |
| 独特资源 / Unique Resources | 骨块 (Bone Block)、水蜡烛 (Water Candle)、各种宝箱 |
| 困难模式 / Hardmode | 新增圣骑士 (Paladin)、战术骷髅 (Tactical Skeleton)等敌人 |
| 危险等级 / Danger Level | 高 / High |

**地牢宝箱 / Dungeon Chests**: 地牢中有锁住的金箱 (Gold Chest)，需要对应的钥匙 (Dungeon Key) 打开。

### 地狱 / Underworld

地狱位于地图的最底部，由灰烬块组成，充斥着岩浆。

The Underworld is located at the bottom of the map, composed of Ash Blocks and filled with lava.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 恶魔 (Demon)、火焰小鬼 (Fire Imp)、骨蛇 (Bone Serpent) |
| 独特资源 / Unique Resources | 狱石 (Hellstone)、黑曜石 (Obsidian)、火焰之花 (Fireblossom) |
| 恶魔祭坛 / Demon Altars | 用于合成重要物品 |
| 危险等级 / Danger Level | 非常高 / Very High |

**狱石采集 / Hellstone Mining**: 狱石需要使用梦魇镐 (Nightmare Pickaxe) 或更好的镐来挖掘。挖掘时会释放岩浆，需要小心。建议使用黑曜石皮药水 (Obsidian Skin Potion)。

### 神圣之地 / Hallow

神圣之地是困难模式后生成的生物群系，是腐化/猩红的对立面。这里色彩缤纷，宛如童话世界。

The Hallow is a biome generated after Hardmode, the counterpart to Corruption/Crimson. It is colorful, like a fairy tale world.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 独角兽 (Unicorn)、精灵 (Pixie)、闪光史莱姆 (Illuminant Slime) |
| 独特资源 / Unique Resources | 水晶碎块 (Crystal Shard)、精灵尘 (Pixie Dust)、独角兽角 (Unicorn Horn) |
| 地下神圣 / Underground Hallow | 混沌精 (Chaos Elemental)、光明碎片 |
| 危险等级 / Danger Level | 中 / Medium |

**神圣之地的用途 / Uses of Hallow**: 神圣之地可以阻止腐化/猩红的蔓延。可以用圣水 (Holy Water) 或购买神圣种子来人工创造。

### 天空 / Sky

天空位于地图的高处，有漂浮岛 (Floating Islands) 和天空湖 (Sky Lakes)。

The Sky biome is located high above the ground, featuring Floating Islands and Sky Lakes.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 鹰身女妖 (Harpy)、困难模式后出现飞龙 (Wyvern) |
| 独特资源 / Unique Resources | 星怒 (Star Fury)、闪亮石 (Shiny Red Balloon) |
| 天空湖房屋 / Sky Lake House | 可以找到空岛宝箱 |
| 危险等级 / Danger Level | 中 / Medium |

**探索方法 / Exploration Methods**: 使用重力药水 (Gravitation Potion) 或搭建高塔来寻找空岛。后期可以用翅膀 (Wings) 飞行探索。

### 海洋 / Ocean

海洋位于地图的最两侧，是大片的水域。

The Ocean is located at the farthest ends of the map, consisting of large bodies of water.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 鲨鱼 (Shark)、海蜗牛 (Sea Snail)、螃蟹 (Crab) |
| 独特资源 / Unique Resources | 珊瑚 (Coral)、海水 (Water)、棕榈木 (Palm Wood) |
| 海洋宝箱 / Ocean Chest | 需要海洋钥匙 (Ocean Key) 打开 |
| 危险等级 / Danger Level | 中 / Medium |

**钓鱼 / Fishing**: 海洋是钓鱼的好地方，可以钓到剑鱼 (Swordfish)、锯齿鲨 (Sawtooth Shark) 等特殊鱼种。

### 蘑菇生物群系 / Mushroom Biome

发光蘑菇生物群系 (Glowing Mushroom Biome) 可以人工创建或自然生成在地下。地表版本需要人工创建。

The Glowing Mushroom biome can be created artificially or found naturally underground. The surface version must be created artificially.

| 项目 / Item | 详情 / Details |
|---|---|
| 常见敌人 / Common Enemies | 蘑菇鱼 (Mushroom Fish)、巨型发光蘑菇 (Giant Glowing Mushroom) |
| 独特资源 / Unique Resources | 发光蘑菇 (Glowing Mushroom)、蘑菇矿 (Shroomite) |
| NPC / NPC | 松露人 (Truflle) 入住 |
| 危险等级 / Danger Level | 低 / Low |

### 生物群系蔓延 / Biome Spread

在困难模式中，腐化/猩红和神圣会快速蔓延。了解蔓延机制有助于保护你的基地。

In Hardmode, Corruption/Crimson and Hallow spread quickly. Understanding the spread mechanism helps protect your base.

- 腐化/猩红可以通过草、石块、沙块蔓延 / Corruption/Crimson spreads through grass, stone, and sand
- 神圣同样可以蔓延 / Hallow can also spread
- 使用向日葵 (Sunflowers) 可以阻止腐化蔓延到普通草地 / Sunflowers prevent Corruption from spreading to normal grass
- 在基地周围挖3格宽的隔离带可以有效防止蔓延 / A 3-tile-wide isolation trench around your base effectively prevents spread
- 困难模式后尽快购买克莱门汀 (Clentaminator) 来治理 / Buy the Clentaminator as soon as possible in Hardmode to manage spread
