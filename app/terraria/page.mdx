## 游戏概述 / Game Overview

**泰拉瑞亚 (Terraria)** 是一款由 **Re-Logic** 开发的2D沙盒冒险游戏 (2D sandbox adventure game)，于2011年5月16日首次发布。游戏融合了探索、建造、战斗和生存等多种元素，被玩家誉为"2D版我的世界"，但其内容深度和战斗系统远超简单的沙盒游戏。

Terraria is a 2D sandbox adventure game developed by Re-Logic, first released on May 16, 2011. It combines exploration, building, combat, and survival elements, and is often called a "2D Minecraft," though its content depth and combat system far exceed a simple sandbox game.

### 游戏特色 / Key Features

- **随机生成的世界 / Procedurally Generated Worlds**: 每次创建世界都会生成独特的地形、洞穴和生物群落。
- **丰富的BOSS战 / Rich Boss Fights**: 超过15个独特的BOSS等待挑战，每个都有独特的攻击模式和掉落物。
- **数千种物品 / Thousands of Items**: 包括武器、盔甲、饰品、工具、坐骑、宠物等。
- **多人联机 / Multiplayer**: 支持最多8人联机合作或PvP。
- **跨平台 / Cross-Platform**: 支持PC、主机和移动设备。

### 职业系统介绍 / Class System Introduction

泰拉瑞亚拥有独特的职业系统，虽然游戏没有强制选择职业，但根据使用的武器类型可以分为以下五大职业：

Terraria features a unique class system. Although the game does not force you to choose a class, based on weapon types there are five main classes:

| 职业 / Class | 特点 / Characteristics | 代表武器 / Typical Weapons |
|---|---|---|
| 近战 / Melee | 高防御、近距离输出 / High defense, close-range DPS | 剑 (Swords), 悠悠球 (Yoyos), 连枷 (Flails) |
| 远程 / Ranged | 安全距离输出、弹药系统 / Safe distance, ammo system | 弓 (Bows), 枪 (Guns), 发射器 (Launchers) |
| 魔法 / Magic | 高爆发、魔力消耗 / High burst, mana consumption | 法杖 (Staves), 魔法书 (Spell Books), 魔法枪 (Magic Guns) |
| 召唤 / Summoner | 召唤仆从战斗、操作要求高 / Minion-based, high skill ceiling | 召唤杖 (Summon Staves), 鞭子 (Whips) |
| 投手 / Thrower | 仅限前期、投掷武器 / Pre-Hardmode only, throwing weapons | 手里剑 (Shurikens), 刀 (Knives), 投斧 (Throwing Axes) |

### 游戏阶段 / Game Progression Phases

泰拉瑞亚的游戏进程可以分为以下几个主要阶段：

Terraria's game progression can be divided into the following main phases:

1. **前期 / Pre-Hardmode (Early)**: 从创建角色到击败骷髅王 (Skeletron)。探索世界、建造第一个房子、收集基础资源。
2. **中期 / Pre-Hardmode (Late)**: 从击败骷髅王到击败血肉墙 (Wall of Flesh)。下地狱、准备BOSS战、收集暗影之魂。
3. **困难模式前期 / Hardmode (Early)**: 敲碎恶魔祭坛、挖取新矿石、应对世界变化。击败三个机械BOSS。
4. **困难模式中期 / Hardmode (Mid)**: 击败世纪之花 (Plantera) 和石巨人 (Golem)。探索神庙、获取强大武器。
5. **困难模式后期 / Hardmode (Late)**: 击败教徒 (Lunatic Cultist) 和月亮领主 (Moon Lord)。挑战事件、获取毕业装备。

### 游戏模式 / Game Modes

创建角色和世界时可以选择不同的难度模式：

When creating a character and world, different difficulty modes are available:

- **旅行模式 / Journey Mode**: 可以调整时间、天气、无敌，适合新手探索。
- **经典模式 / Classic Mode**: 标准游戏体验，死亡掉落一半金币。
- **专家模式 / Expert Mode**: BOSS有新的攻击模式，掉落专属物品。
- **大师模式 / Master Mode**: 更难，BOSS掉落宠物和装饰品。
