## 战斗系统指南 / Combat Guide

泰拉瑞亚的战斗系统丰富且复杂，理解伤害机制、修饰语和增益效果会让你在战斗中更加游刃有余。

Terraria's combat system is rich and complex. Understanding damage mechanics, modifiers, and buff effects will make you more effective in battle.

### 伤害类型 / Damage Types

游戏中有四种主要伤害类型，以及一种仅限前期的类型：

There are four main damage types in the game, plus one that is only available in the early game:

| 伤害类型 / Damage Type | 加成装备 / Scaling | 特点 / Characteristics |
|---|---|---|
| 近战 / Melee | 近战伤害、暴击、攻速 / Melee damage, crit, speed | 高防御、近距离 / High defense, close range |
| 远程 / Ranged | 远程伤害、暴击 / Ranged damage, crit | 安全距离、需要弹药 / Safe distance, needs ammo |
| 魔法 / Magic | 魔法伤害、暴击、魔力 / Magic damage, crit, mana | 高爆发、魔力管理 / High burst, mana management |
| 召唤 / Summoner | 召唤伤害、仆从数量 / Summon damage, minion count | 仆从自动攻击、低防御 / Minion auto-attack, low defense |
| 投手 / Throwing | 投掷伤害 / Throwing damage | 仅前期可用 / Pre-Hardmode only |

### 伤害计算 / Damage Calculation

基础伤害计算遵循以下公式：

Basic damage calculation follows this formula:

- 武器基础伤害 (Base Damage) x 伤害加成 (Damage Bonus) - 敌人防御力 (Enemy Defense) x 0.5
- 暴击造成双倍伤害 / Critical hits deal double damage
- 基础暴击率为4% / Base critical strike chance is 4%

**防御与减伤 / Defense and Damage Reduction**:
- 每点防御力减少0.5点伤害 / Each defense point reduces damage by 0.5
- 减伤效果来自特定饰品和增益 / Damage reduction comes from specific accessories and buffs
- 减伤在防御计算之后应用 / Damage reduction is applied after defense calculation

### 职业战术 / Class-Specific Tactics

**近战战术 / Melee Tactics**:
- 使用十字章护盾 (Ankh Shield) 防止击退 / Use Ankh Shield to prevent knockback
- 注意走位，利用剑的范围攻击 / Pay attention to positioning, use sword swing range
- 悠悠球 (Yoyos) 在前期非常强力 / Yoyos are very powerful in the early game
- 困难模式使用日耀喷发 (Solar Eruption) 穿透攻击 / Use Solar Eruption in Hardmode for piercing attacks

**远程战术 / Ranged Tactics**:
- 使用弹药箱 (Ammo Box) 减少弹药消耗 / Use Ammo Box to reduce ammo consumption
- 灵液箭 (Ichor Arrows) 降低敌人防御 / Ichor Arrows reduce enemy defense
- 子弹类型选择非常重要，水晶子弹 (Crystal Bullets) 提供弹射 / Bullet type selection is important, Crystal Bullets provide bounces
- 使用蘑菇矿甲 (Shroomite Armor) 的隐身效果增加伤害 / Use Shroomite Armor stealth effect to increase damage

**魔法战术 / Magic Tactics**:
- 提升魔力上限最大到200 / Increase mana to max 200
- 使用魔力花 (Mana Flower) 自动喝魔力药水 / Use Mana Flower for auto mana potion
- 星云甲 (Nebula Armor) 是魔法师终极装备 / Nebula Armor is the ultimate mage gear
- 魔力再生手环 (Mana Regeneration Band) 加快回蓝 / Mana Regeneration Band speeds up mana recovery

**召唤战术 / Summoner Tactics**:
- 鞭子 (Whips) 是召唤师的核心 / Whips are the core of summoner gameplay
- 使用召唤药水 (Summoning Potion) 增加仆从上限 / Use Summoning Potion to increase minion limit
- 优先保证走位安全 / Prioritize safe positioning
- 甲虫莎草纸 (Papyrus Scarab) 增加仆从数量和伤害 / Papyrus Scarab increases minion count and damage

### 战斗小技巧 / Combat Tips

- 使用水蜡烛 (Water Candle) 和战斗药水 (Battle Potion) 可以增加刷怪率 / Use Water Candle and Battle Potion to increase spawn rate
- 和平蜡烛 (Peace Candle) 可以减少刷怪率 / Peace Candle reduces spawn rate
- 高处的平台可以让你安全地攻击地面敌人 / Platforms at height allow safe attack on ground enemies
- 保持移动比站着不动更难被击中 / Staying mobile is better than standing still
- 利用钩爪 (Grappling Hook) 快速脱离危险 / Use Grappling Hook to quickly escape danger
- 巢穴心 (Heart Lantern) 和篝火 (Campfire) 提供生命恢复 / Heart Lantern and Campfire provide life regen
