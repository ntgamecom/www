# 生存与制造 / Survival & Crafting

在深海迷航2中，制造是进步的核心驱动力。从基础工具到深海载具，从食物处理器到大型基地，一切都从资源收集开始 / In Subnautica 2, crafting is the core driver of progression. From basic tools to deep-sea vehicles, from food processors to massive bases, everything starts with resource gathering.

---

## 基础资源 / Basic Resources

### 常见资源 / Common Resources

| 资源 / Resource | 获取方式 / How to Get | 主要用途 / Main Uses |
|------|------|------|
| **钛 / Titanium** | 金属碎片、石灰岩 / Metal scrap, limestone | 几乎所有建筑和工具的基底材料 / Base material for nearly everything |
| **石英 / Quartz** | 浅水区石灰岩 / Limestone in shallow water | 玻璃、窗户、电子元件 / Glass, windows, electronics |
| **铜 / Copper** | 石灰岩、玄武岩 / Limestone, basalt | 电线、电池、电子设备 / Wiring, batteries, electronics |
| **海藻块 / Creepvine** | 海藻林采集 / Harvest from Kelp Forest | 纤维织物、润滑油 / Fiber mesh, lubricant |
| **酸蘑菇 / Acid Mushroom** | 浅水洞穴 / Shallow caves | 电池、化学品 / Batteries, chemicals |
| **金 / Gold** | 砂岩、页岩 / Sandstone, shale | 高级电子设备 / Advanced electronics |
| **银 / Silver** | 砂岩 / Sandstone | 电线、医疗包制造 / Wiring, medkit fabrication |

| **锂 / Lithium** | 150m+ | 矿石开采、页岩 / Ore deposits, shale | 强化潜水服、载具升级模块 / Reinforced suit, vehicle upgrade modules |

### 基地布局建议 / Base Layout Tips

| 建议 / Tip | 说明 / Description |
|------|------|
| **选址靠近资源 / Build Near Resources** | 在浅水区与深海区交界处建立基地——方便往返补给 / Build at the boundary between shallow and deep zones for easy resupply |
| **多层设计 / Multi-Level Design** | 垂直建造比水平更省材料，也更安全 / Vertical building saves materials and is safer than horizontal |
| **备用电源 / Backup Power** | 生物反应器+太阳能板组合——夜间/阴天有保障 / Bioreactor + solar panels — guarantees power at night/cloudy weather |
| **外部照明 / External Lighting** | 在基地周围放置照明灯，防止夜间被大型生物接近 / Place exterior lights to deter large creatures approaching at night |
| **改造窗口位置 / Window Placement** | 窗口会增加基地结构弱点——只在观景位置放置 / Windows add structural weakness — only place at scenic spots |

## 制造优先级 / Crafting Priority

| 阶段 / Phase | 优先制造 / Priority Craft | 目的 / Purpose |
|------|------|------|
| **开局 / Start** | 生存刀 → 扫描仪 → 氧气瓶 → 焊接器 | 生存和探索基础 / Survival and exploration basics |
| **早期 / Early** | 推进器 → 基础基地 → 电池充电桩 → 种植箱 | 提升效率 / Efficiency upgrades |
| **中期 / Mid** | 探索者号 → 生物反应器 → 水净化器 → 高级氧气瓶 | 区域探索 / Zone exploration |
| **中后期 / Mid-Late** | 深潜者号 → 深度升级 → 强化潜水服 → 核反应堆 | 深海探索 / Deep exploration |
| **后期 / Late** | 独眼巨人号2.0 → 防御系统 → 终极装备 | 最终区域 / Endgame zones |

> **重要提示**：扫描仪是你最好的朋友——看到任何未知碎片/技术都先扫描！/ **Tip**: The Scanner is your best friend — scan every unknown fragment and piece of technology you find!
