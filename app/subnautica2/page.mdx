# 深海迷航2 攻略 / Subnautica 2 Guide

深海迷航2（Subnautica 2）是Unknown Worlds Entertainment开发的开放世界水下生存冒险游戏，是《深海迷航》系列的最新正统续作。你将在全新外星海洋世界中求生，探索未知生态，建造水下基地，解开星球的秘密 / Subnautica 2 is the latest open-world underwater survival adventure game from Unknown Worlds Entertainment, the official sequel to the beloved Subnautica series. Survive in an alien ocean, explore unknown ecosystems, build underwater bases, and unravel the planet's mysteries.

**当前状态 / Status**: 抢先体验（Early Access）| **平台 / Platforms**: PC (Steam/Epic), Xbox Series X\|S | **类型 / Genre**: 开放世界生存冒险 / Open-world Survival Adventure

## 与一代的关键差异 / Key Differences from Subnautica 1

| 对比项 / Aspect | 深海迷航 / Subnautica 1 | 深海迷航2 / Subnautica 2 |
|------|------|------|
| **故事背景 / Setting** | 行星4546B / Planet 4546B | **全新星球** / **All-new planet** |
| **主角 / Protagonist** | 极光号幸存者 / Aurora survivor | 全新角色 / New protagonist |
| **多人合作 / Co-op** | 单人 / Single-player | **最多4人联机** / **Up to 4-player co-op** |
| **世界地图 / World Map** | 固定地图 / Fixed map | **程序化+手工混合** / **Procedural + handcrafted hybrid** |
| **载具系统 / Vehicles** | 海蛾号、独眼巨人号、海虾号 / Seamoth, Cyclops, PRAWN | **全新载具系列** / **All-new vehicle line-up** |
| **建造系统 / Building** | 基础模块建造 / Modular bases | **增强建造系统，更多自定义** / **Enhanced building with more customization** |
| **生物种类 / Creatures** | 固定生物群 / Fixed creatures | 新生物群 + 部分熟悉面孔回归 / New creatures + some returning favorites |
| **剧情深度 / Story** | PDA日志驱动的线性叙事 / PDA-log driven | **更丰富的叙事方式** / **Richer storytelling** |

## 游戏特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **全新外星海洋 / All-New Alien Ocean** | 探索一个完全不同的星球，充满未知的生态和危险 / Explore an entirely different planet filled with unknown ecosystems and dangers |
| **4人联机合作 / 4-Player Co-op** | 与好友一起建造基地、驾驶载具、探索深海——这是系列首次完整支持联机 / Build bases, pilot vehicles, and explore the deep sea with friends — first in the series with full co-op support |
| **动态生态系统 / Dynamic Ecosystem** | 生物之间有完整的食物链和行为互动，你在其中也是生态的一环 / Creatures have a full food chain and behavioral interactions — you're part of the ecosystem |
| **增强建造系统 / Enhanced Building** | 更多模块、更多装饰、更自由的布局——打造你的梦想深海基地 / More modules, more decorations, freer layout — build your dream deep-sea base |
| **全新载具 / New Vehicles** | 速度型、深潜型、货运型等多种全新载具，各有特色 / Speed, deep-dive, cargo, and more — each with unique capabilities |
| **昼夜与天气 / Day/Night & Weather** | 动态昼夜循环 + 水下天气效果影响能见度和探索 / Dynamic day/night cycle + underwater weather effects affecting visibility and exploration |
