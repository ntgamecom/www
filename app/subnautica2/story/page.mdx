# 剧情与流程 / Story & Progression

深海迷航2的剧情通过PDA日志、先驱者设施和环境叙事展开。本章提供主线流程指南和关键节点的详细说明 / Subnautica 2's story unfolds through PDA logs, Precursor facilities, and environmental storytelling. This chapter provides a main story walkthrough and detailed guidance for key milestones.

---

## 故事背景 / Story Background

在一代事件之后多年，另一艘探索飞船——**信天翁号（Albatross）**——坠落在另一个被海洋覆盖的星球上。你是少数幸存者之一，必须在这片陌生的水域中生存、探索、并最终找到逃离的方法 / Years after the events of Subnautica 1, another exploration vessel — the **Albatross** — has crash-landed on a different ocean planet. You are one of the few survivors, and you must survive, explore, and find a way to escape this alien world.

> **与一代的联系 / Connection to Subnautica 1**: 新星球与4546B有某些关联，具体由你在探索中发现 / The new planet has some connection to 4546B — discover how as you explore.

## 主线流程 / Main Story Walkthrough

### 第一章：坠落 / Chapter 1: Crash

| 步骤 / Step | 行动 / Action | 关键地点 / Key Location | 提示 / Tips |
|:----:|------|------|------|
| **1** | 从坠毁的信天翁号残骸中逃生 / Escape the crashed Albatross wreck | 坠毁点 / Crash Site | 沿着残骸中的气泡向上浮——跟随光线 / Follow air bubbles up through the wreck — swim toward the light |
| **2** | 抵达救生舱 / Reach your Lifepod | 救生舱（出生点）/ Lifepod (spawn point) | 修复救生舱的通讯系统和制造器 / Repair the lifepod's radio and fabricator |
| **3** | 接收首个求救信号 / Receive first distress signal | 救生舱通讯台 / Lifepod radio | 信号指向附近的**小型残骸**——前往调查 / Signal points to a nearby **small wreck** — investigate |

**完成第一章的标志**：建造了第一个工具（生存刀）并修复了救生舱通讯器 / Milestone: Craft your first tool (Survival Knife) and repair the lifepod radio.

### 第二章：探索 / Chapter 2: Exploration

| 步骤 / Step | 行动 / Action | 关键地点 / Key Location | 提示 / Tips |
|:----:|------|------|------|
| **1** | 调查求救信号来源 / Investigate distress signal source | 珊瑚浅滩-小型残骸 / Coral Shallows - Small Wreck | 扫描残骸碎片获得新蓝图 / Scan wreckage fragments for new blueprints |
| **2** | 接收更多信号 / Receive more signals | 随机刷新 / Random spawns | 信号来自其他幸存者的救生舱——找到它们可以获得更多资源 / Signals from other lifepods — find them for resources |
| **3** | 在海藻森林找到关键扫描数据 / Find key scan data in Kelp Forest | 海藻森林-数据缓存 / Kelp Forest - Data Cache | 扫描数据指向**先驱者技术**的位置 / Scan data reveals **Precursor technology** locations |
| **4** | 收到"神秘信号"——一个未知来源的通讯 / Receive "Mysterious Signal" — unknown origin | 任意位置 / Any location | 剧情关键转折点——信号来自先驱者设施 / Story turning point — signal originates from a Precursor facility |

**完成第二章的标志**：找到海藻森林的数据缓存，收到神秘信号 / Milestone: Find the Kelp Forest data cache and receive the Mysterious Signal.

### 第三章：真相 / Chapter 3: The Truth

| 步骤 / Step | 行动 / Action | 关键地点 / Key Location | 提示 / Tips |
|:----:|------|------|------|
| **1** | 定位并进入第一个**先驱者设施** / Locate and enter the first **Precursor facility** | 红草高原-地下入口 / Red Grass Plateau - Underground Entrance | 设施入口被密码锁住——需要在周边找到两个密码碎片 / Entrance is code-locked — find 2 code fragments nearby |
| **2** | 解锁先驱者数据终端 / Unlock Precursor data terminal | 先驱者设施内部 / Inside Precursor facility | 揭示了星球的历史和**疾病**的真相 / Reveals the planet's history and the truth about the **Disease** |
| **3** | 获取**治愈酶**蓝图 / Acquire **Enzyme Cure** blueprint | 先驱者设施-生物实验室 / Precursor facility - Bio Lab | 需要稀有材料来制造解药 / Requires rare materials to synthesize |
| **4** | 制造治愈酶——这让你可以安全进入被感染的更深区域 / Craft the Enzyme Cure — allows safe entry into infected deeper zones | 你的基地制造器 / Your base fabricator | 所需材料：蘑菇孢子×3 + 蓝晶石×1 + 血藻提取物×2 / Materials: Fungal Spores×3 + Kyanite×1 + Blood Kelp Extract×2 |

**完成第三章的标志**：制造出治愈酶，获得进入深海的通行证 / Milestone: Craft the Enzyme Cure, gaining access to deep infected zones.

## 结局分支 / Ending Branches

### 结局A：逃离星球 / Ending A: Escape the Planet

| 条件 / Condition | 过程 / Process | 结果 / Result |
|------|------|------|
| 收集全部逃生火箭组件，不激活行星发动机 / Collect all Escape Rocket parts, don't activate Planetary Engine | 建造并发射逃生火箭 / Build and launch the Escape Rocket | 你成功逃离星球，但留下了行星被小行星摧毁的命运 / You escape the planet, but leave it to be destroyed by the asteroid |

### 结局B：拯救星球 / Ending B: Save the Planet

| 条件 / Condition | 过程 / Process | 结果 / Result |
|------|------|------|
| 收集全部逃生火箭组件 + 激活行星发动机密钥 / Collect all Escape Rocket parts + Planetary Engine Key | 激活行星发动机 → 小行星被改变轨道 → 使用逃生火箭离开 / Activate Planetary Engine → asteroid redirected → escape via rocket | 你同时拯救了星球和自己——完美的结局 / You save both the planet and yourself — the perfect ending |

### 隐藏结局C？/ Hidden Ending C?

有传言说游戏中存在第三个结局——需要满足非常特定的条件。这可能需要你在联机模式下完成特殊的配合操作 / Rumors of a third ending exist — requiring very specific conditions, possibly involving special co-op cooperation.

## 联机提示 / Co-op Tips

| 场景 / Scenario | 建议 / Suggestion |
|------|------|
| **资源收集 / Resource Gathering** | 四人分工：一人开物流者号、三人分别采集不同资源 / One drives Hauler, three collect different resource types |
| **探索 / Exploration** | 两人一队分头探索，随时沟通发现 / Pair up and explore separately, communicate findings |
| **基地建造 / Base Building** | 指定一人为主要建筑师，其他人为材料运输者 / Designate one as primary builder, others as material runners |
| **BOSS战 / Boss Encounters** | 一人引怪、其他人从侧面攻击。联机时利维坦更难对付 / One lures, others attack from the side. Leviathans scale with player count |
| **最终区域 / Endgame Zone** | 全员必须携带足够补给和备用氧气瓶——失去任何一人都可能导致团灭 / Everyone must bring ample supplies and backup O2 tanks — losing any player can wipe the team |
