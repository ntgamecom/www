# 新手入门 / Beginner Guide

深海迷航2延续了一代的核心理念——坠落在外星海洋中，从零开始求生，最终找到离开的方法。但游戏加入了大量新系统和机制，无论新老玩家都有许多需要学习的内容 / Subnautica 2 continues the core premise — crash-land on an alien ocean, survive from scratch, and find a way to leave. But with many new systems and mechanics, both newcomers and veterans have plenty to learn.

---

## 游戏目标 / Core Objective

你坠毁在一颗被海洋覆盖的未知星球上。首要目标是**生存**，然后是**探索**和**建造**，最终目标是**找到离开的方法** / You crash-land on an unknown ocean planet. Your primary goal is **survival**, followed by **exploration** and **building**, with the ultimate goal of **finding a way off the planet**.

### 核心循环 / Core Loop

```
探索 → 收集资源 → 建造工具/基地 → 探索更深区域 → 收集更稀有资源 → 建造更高级装备 → 推进剧情 → 最终逃离
Explore → Gather resources → Build tools/base → Explore deeper → Collect rare resources → Build advanced gear → Advance story → Escape
```

## 生存基础 / Survival Basics

### 五大生存指标 / Five Vital Stats

| 指标 / Stat | 图标 / Icon | 说明 / Description | 如何维持 / How to Maintain |
|------|:----:|------|------|
| **氧气 / Oxygen** | 氧气瓶图标 | 水下行动消耗氧气，归零开始掉血 / Consumed underwater, zero = damage | 浮出水面换气、建造管道、制造高容量氧气瓶 / Surface, build pipes, craft high-capacity tanks |
| **生命值 / Health** | 心形图标 | 受到攻击或环境伤害会降低，归零则死亡 / Reduced by attacks or environmental damage, zero = death | 使用医疗包、在基地医疗站恢复 / Use medkits, base med station |
| **饱食度 / Food** | 叉子图标 | 随时间下降，归零掉血 / Decreases over time, zero = damage | 捕鱼烹饪、种植食用植物 / Catch fish to cook, grow edible plants |
| **水分 / Water** | 水滴图标 | 随时间下降，归零掉血 / Decreases over time, zero = damage | 过滤海水、收集淡水植物、建造净水设备 / Filter seawater, collect fresh water plants, build water purifiers |
| **体温 / Body Temp** | 温度计图标 | 在极寒/极热区域快速变化，极端温度持续掉血 / Changes rapidly in extreme cold/hot zones, extreme temps = damage | 潜水服升级、携带发热/冷却装备 / Upgrade dive suit, carry thermal/cooling equipment |

> **重要**：游戏开始时你只有**60秒氧气**和一个**小氧气瓶**——不要游太远！/ **IMPORTANT**: You start with only **60 seconds of oxygen** and a **small oxygen tank** — don't swim too far!

## 开局指南 / First Steps

### 第一阶段：求生 / Phase 1: Survival (Hours 0-2)

| 步骤 / Step | 行动 / Action | 目标 / Goal |
|:----:|------|------|
| **1** | 从救生舱中搜刮基础物资 / Scavenge your lifepod for basic supplies | 获得制造器、扫描仪、焊接器 / Get Fabricator, Scanner, Welder |
| **2** | 在救生舱附近浅水区收集资源 / Gather resources in shallow waters near lifepod | 钛、石英、铜、海藻块 / Titanium, Quartz, Copper, Creepvine |
| **3** | 制造基础工具 / Craft basic tools | 生存刀、手电筒、氧气瓶 / Survival Knife, Flashlight, O2 Tank |
| **4** | 探索周边生态区 / Explore nearby biomes | 了解地图，找到海藻林入口 / Learn the map, find Kelp Forest entrance |
| **5** | 建造第一个小型基地 / Build your first small base | 管道+生物反应器供电 / Tubes + Bioreactor for power |

### 第二阶段：发展 / Phase 2: Development (Hours 2-8)

| 步骤 / Step | 行动 / Action | 目标 / Goal |
|:----:|------|------|
| **1** | 制造基础载具 / Build first vehicle | 探索更远区域 / Explore farther areas |
| **2** | 建立完整基地 / Establish full base | 种植、净水、储存、医疗 / Farming, water filtration, storage, medical |
| **3** | 探索深度100-200米区域 / Explore 100-200m depths | 收集更高级材料 / Collect advanced materials |
| **4** | 制造升级设备 / Craft upgraded equipment | 高级氧气瓶、强化潜水服 / Advanced O2 tank, reinforced dive suit |
| **5** | 跟随主线信号/求救信号 / Follow story signals / distress calls | 推进剧情 / Advance the story |

### 第三阶段：深入 / Phase 3: Deep Exploration (Hours 8+)

| 步骤 / Step | 行动 / Action | 目标 / Goal |
|:----:|------|------|
| **1** | 制造深潜载具 / Build deep-dive vehicle | 探索200-500米深度 / Explore 200-500m depths |
| **2** | 在深海建立前哨基地 / Build deep-sea outpost | 为更深的探索提供补给 / Supply depot for deeper exploration |
| **3** | 探索主要剧情地标 / Explore main story landmarks | 收集关键数据 / Collect key data |
| **4** | 制造最终逃生所需装备 / Craft endgame equipment | 准备最终旅程 / Prepare for the final journey |

## 新玩家常见误区 / Common Beginner Mistakes

| 误区 / Mistake | 正确做法 / Correct Approach |
|------|------|
| **完全不建基地 / Never Build a Base** | 救生舱容量有限且无法升级——尽早建造基地存储物资和充电 / Lifepod storage is limited and can't be upgraded — build a base early for storage and charging |
| **忽略食物和水 / Ignore Food & Water** | 这两项是持续消耗的——建立一个种植区+净水设备可以实现自给自足 / These drain constantly — set up a farm + water filtration for self-sufficiency |
| **携带过多贵重资源 / Overloading on Valuables** | 死亡会掉落背包——不要带全部家当出门，尤其是深潜时 / Death drops your inventory — don't bring everything, especially on deep dives |
| **唯一只靠救生舱导航 / Only Use Lifepod Navigation** | 在基地和关键位置建造信标——午夜深潜时救生舱的光很容易跟丢 / Build beacons at base and key locations — the lifepod light is easy to lose |
| **游向大型生物 / Swimming at Large Creatures** | 不要主动招惹利维坦级生物——它们通常不会主动追击到小区域 / Don't provoke leviathan-class creatures — they usually won't chase into tight spaces |

## 给一代玩家的提示 / Tips for Subnautica Veterans

| 提示 / Tip | 说明 / Description |
|------|------|
| **别按旧地图思维走 / Unlearn the Map** | 星球不同，地形完全不同。扔弃4546B的记忆 / Different planet, entirely different terrain. Forget 4546B |
| **联机时合理分工 / Divide Labor in Co-op** | 4人联机时一人专门收集、一人专注于建造、两人探索——效率倍增 / In 4-player co-op, assign roles: gatherer, builder, 2 explorers |
| **新生物有新威胁 / New Threats** | 部分新生物的行为模式与一代完全不同——不要用老经验判断 / Some new creatures have entirely different behavior patterns — don't assume |
| **建造系统更灵活 / Building Is More Flexible** | 新作中可在更多地形上建造，支撑结构要求更低 / More terrain is buildable, fewer support structure requirements |
| **体温是新挑战 / Body Temp Is New** | 极寒深海和热液喷口区需要专门的温度管理装备 / Deep cold zones and thermal vents require specialized temperature gear |
