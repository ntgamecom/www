### 死神利维坦攻略 / Reaper Leviathan Strategy

**位置 / Location**:
- 极光号船头/船尾 / Aurora bow/stern
- 沙丘地 / Dunes
- 交界区外围 / Crash Zone edges

**属性 / Stats**:

| 属性 / Attribute | 数值 / Value |
|---|---|
| 生命值 / HP | 5000+ |
| 攻击伤害 / Attack DMG | 80 (擒抱) / 80 (grab) |
| 游泳速度 / Speed | 快 / Fast |
| 探测范围 / Detection | 100米 / 100m |
| 攻击范围 / Attack Range | 15米 / 15m |

**躲避策略 / Evasion Strategy**:

| 方法 / Method | 说明 / Description |
|---|---|
| 静音引擎 / Silent Running | 海蛾号静音模式——减少被察觉 / Seamoth silent mode — reduce detection |
| 声呐干扰 / Sonar Decoy | 独眼巨人号可发射诱饵弹 / Cyclops fire decoy |
| 垂直规避 / Vertical Dodge | 死神攻击是水平向的——垂直移动有效 / Attacks horizontal — dive/rise |
| 利用地形 / Terrain | 在岩石缝隙中躲避——死神体型大进不来 / Hide in crevices |
| 警戒声音 / Warning Sound | 死神的咆哮声距离很远——提前预警 / Roar heard from far away |

**如果被抓住 / If Grabbed**:
- 海蛾号被抓住时——弹射座椅逃生 / Seamoth grabbed — eject
- 独眼巨人号被攻击——关闭引擎——等待 / Cyclops attacked — shut down engine
- 游泳时被抓住——用生存刀猛刺它的眼睛 / Swim-grabbed — stab its eye

### 幽灵利维坦攻略 / Ghost Leviathan Strategy

**成年形态位置 / Adult Locations**:
- 失落之河巨洞 / Lost River Big Cave
- 北部血藻区 / North Blood Kelp
- 虚空边界(幼体)/ Void Border (Juvenile)

**攻击模式 / Attack Pattern**:

| 攻击 / Attack | 前兆 / Tell | 应对 / Counter |
|---|---|---|
| 冲锋撞击 / Charge Ram | 后退蓄力——身体发光 / Back up — body glows | 侧向闪避 / Side dodge |
| 酸液喷吐 / Acid Spit | 张开大嘴——聚集绿色能量 / Open mouth — green energy | 横向移动 / Strafe |
| 尾扫 / Tail Sweep | 转身甩尾 / Turn and sweep | 靠近头部——尾巴范围大 / Stay near head |

**击杀策略 / Kill Strategy**:
- 使用海虾号钻臂+加农炮——近距输出 / Prawn Suit drill arm + cannon — close combat
- 毒气 torpedo——造成持续的伤害 / Gas torpedoes — sustained damage
- 热刀(热武器刀)——攻击头部造成最大伤害 / Heat knife — max damage to head
- 注意它被激怒后会召唤幼体幽灵鳐 / Enrage summons juvenile ghosts

## 海龙实战击杀 / Sea Dragon Practical Kill

**推荐装备 / Recommended**:
海虾号(深度III) + 钻臂 + 格斗臂 / Prawn Suit (Depth III) + Drill Arm + Grapple Arm

**步骤 / Steps**:
1. 钩爪勾住海龙头部——贴在它身上 / Grapple onto its head — stay attached
2. 另一只手臂换钻头——持续钻头攻击 / Switch other arm to drill — continuous damage
3. 它无法甩掉你——钻头持续造成大量伤害 / It cannot shake you off
4. 注意它的火球攻击——但你贴在头上不会被火球打中 / Fireball can't hit you at close range
