# 战斗与防御 / Combat & Defense

深海迷航2并非传统战斗游戏——但面对危险生物时的防御和反击是生存关键 / Subnautica 2 is not a traditional combat game — but defense and counterattack against dangerous creatures are key to survival.

---

## 基础防御 / Basic Defense

### 徒手防御 / Unarmed Defense

| 生物 / Creature | 最佳应对 / Best Response | 说明 / Description |
|---|---|---|
| **潜伏者 / Stalker** | 用生存刀驱赶——或给予金属转移注意 / Knife scare — or give metal | 给它金属它就咬金属去了 / Gives metal to distract |
| **沙鲨 / Sand Shark** | 游到它头顶上方——它从沙里跳出时闪避 / Swim above — dodge when it leaps | 它的攻击轨迹是直线的 / Linear attack pattern |
| **洞穴爬行者 / Cave Crawler** | 用灭火器喷射——它们会逃 / Fire extinguisher — they flee | 或直接冲过去不管 / Or just rush past |
| **毒刺 / Bone Shark** | 关闭灯光+静默通过 / Lights off + silent pass | 它们被灯光和声音吸引 / Attracted to light + sound |
| **小型鱼群 / Small Fish** | 不构成威胁——尽管游过去 / No threat — swim through | 只有鲍勃鱼咬人但伤害低 / Only Biters deal low dmg |

### 生存刀战斗 / Knife Combat

| 刀类型 / Knife Type | 伤害 / DMG | 攻速 / Speed | 特效 / Special |
|---|---|---|---|
| 标准生存刀 / Standard Knife | 20 | 快 / Fast | 无 / None |
| 金刚石刀 / Diamond Blade | 30 | 快 / Fast | 高耐久 / High durability |
| 热刀 / Thermoblade | 25 | 中等 / Medium | 额外灼烧伤害+烹饪食物 / Extra burn + cooks food |

**刀战技巧 / Knife Fighting Tips**:
1. 横向移动——不要直线冲向大型生物 / Strafe — don't charge large creatures
2. 攻击生物的弱点(眼睛、鳃、腹部) / Target weak points (eyes, gills, belly)
3. 边打边退——利用生物攻击后摇 / Hit and run — exploit recovery frames
4. 热刀可以杀死小鱼后直接吃——野外生存利器 / Thermoblade cooks fish on kill — wilderness survival

### 独眼巨人号防御 / Cyclops Defense

| 系统 / System | 功能 / Function | 能量消耗 / Power Cost |
|---|---|---|
| 护盾发生器 / Shield | 无敌5秒——无法攻击 / Invincible 5s — can't attack | 大量 / High |
| 静默运行 / Silent Running | 移动速度降50%——侦测范围降80% / -50% speed — -80% detection | 中等 / Medium |
| 诱饵发射器 / Decoy Launcher | 发射诱饵吸引生物 / Fire decoy to distract | 消耗诱饵 / Devours decoy |
| 后方摄像头 / Rear Camera | 倒车时观察身后 / Rear view when reversing | 极低 / Minimal |
| 灭火系统 / Fire System | 自动扑灭内部火灾 / Auto-extinguish fires | 中等 / Medium |

**独眼巨人逃生流程 / Cyclops Escape Protocol**:
1. 发现利维坦后——立即切换到静默运行 / Spot leviathan — silent running
2. 发动机关闭——只依靠惯性滑行 / Engine off — coast on momentum
3. 如果被发现——发射诱饵后全速反向逃跑 / If detected — fire decoy + full reverse
4. 船体受损超过50%时——立即弃船逃生 / Hull below 50% — abandon ship

**海虾号击杀利维坦技巧 / Prawn Leviathan Killing**:
1. 格斗臂勾住利维坦头部 / Grapple onto leviathan head
2. 另一侧切换钻臂——持续钻伤害 / Switch to drill arm — constant damage
3. 利维坦无法甩掉你——持续输出至死 / It cannot shake you — DPS to death
4. 注意海虾号血量——保持在50%以上 / Monitor HP — keep above 50%
