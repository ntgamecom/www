# 生物图鉴 / Creatures Guide

深海迷航2的海洋生态系统中栖息着数十种生物，从温顺的小鱼到恐怖的利维坦——了解它们的习性和行为是生存的关键 / Subnautica 2's ocean ecosystem is home to dozens of species, from docile small fish to terrifying leviathans. Understanding their behavior is key to survival.

---

## 生物分类 / Creature Classification

| 分类 / Class | 说明 / Description | 示例 / Examples |
|------|------|------|
| **小型鱼类 / Small Fish** | 无威胁，可捕捉食用 / Harmless, can be caught for food | 孔鱼、蹦跳鱼、灯笼鱼 / Peeper, Boomerang, Lantern Fish |
| **草食生物 / Herbivores** | 不主动攻击，受惊会逃跑 / Non-aggressive, flees when startled | 兔鱼、天翼鱼 / Rabbit Fish, Skyray |
| **中型掠食者 / Medium Predators** | 领地性，靠近会攻击 / Territorial, attacks when approached | 海藻龙、洞穴蟹、冲击鱼 / Kelp Dragon, Cave Crab, Crashfish |
| **大型掠食者 / Large Predators** | 主动猎食，看见玩家就会追击 / Active hunter, chases player on sight | 血藻龙、深渊潜伏者 / Blood Kelp Dragon, Abyss Lurker |
| **利维坦级 / Leviathan Class** | 顶级掠食者——全游戏最危险的生物 / Apex predator — the most dangerous in the game | 幽灵利维坦、熔岩利维坦 / Ghost Leviathan, Lava Leviathan |
| **被动巨兽 / Passive Giants** | 体型巨大但无攻击性，可近距离观察 / Massive but harmless, can be observed up close | 礁背鲸 / Reefback Whale |

|------|:----:|------|------|
| **孔鱼 / Peeper** | 极小 / Tiny | 所有浅层区域 / All shallow zones | 烤孔鱼——前期最佳食物来源 +5食物 / Cooked Peeper — best early food +5 food |
| **蹦跳鱼 / Boomerang** | 小 / Small | 红草高原、珊瑚浅滩 / Red Grass Plateau, Coral Shallows | 烤蹦跳鱼 +7食物 / Cooked Boomerang +7 food |

## 中型掠食者 / Medium Predators

### 海藻龙 / Kelp Dragon

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **体型 / Size** | 中大型 / Medium-Large (~5m) |
| **所在区域 / Location** | 海藻森林 / Kelp Forest |
| **攻击性 / Aggression** | 中——被接近后会发出警告，持续接近才攻击 / Medium — warns first, attacks on continued approach |
| **攻击方式 / Attack** | 尾鞭抽击（25伤害）/ Tail whip (25 damage) |

**应对策略 / Tactics**:
- 保持距离观察，不要主动挑衅 / Keep distance and observe, don't provoke
- 被攻击后直线游走——海藻龙转向较慢 / Swim in straight lines when fleeing — turns slowly
- 杀死掉落**海藻龙鳞片**——可用于制作中级护甲 / Drops **Kelp Dragon Scale** — used for mid-tier armor

### 洞穴蟹 / Cave Crab

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **体型 / Size** | 中型 / Medium (~1.5m) |
| **所在区域 / Location** | 水母洞穴 / Jellyshroom Caves |
| **攻击性 / Aggression** | 高——伏击型，玩家靠近即攻击 / High — ambush predator, attacks when close |
| **攻击方式 / Attack** | 钳击（35伤害）+ 拖拽 / Pincer (35 damage) + grab |

**应对策略 / Tactics**:
- 洞穴蟹通常附着在天花板或墙面上——注意观察上方 / Usually clings to ceilings/walls — look up!
- 被抓住时快速摇晃方向键挣脱 / Shake the movement stick rapidly when grabbed
- 推进器的速度可以轻松甩开它们 / Seaglide speed easily outruns them

### 冲击鱼 / Crashfish

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **体型 / Size** | 极小 / Tiny |
| **所在区域 / Location** | 红草高原砂岩、珊瑚洞穴 / Red Grass Plateau sandstone, coral caves |
| **攻击性 / Aggression** | 特高——被惊动后冲向玩家并自爆 / Extreme — charges and explodes when disturbed |
| **攻击方式 / Attack** | 自爆（55伤害——对早期致命！）/ Self-destruct (55 damage — fatal early game!) |

**应对策略 / Tactics**:
- **听到嘶嘶声立刻后退**——这是冲击鱼被惊动的警告 / **Back away on hearing the hiss** — that's the warning
- 收集砂岩时先扫描周围是否有冲击鱼再靠近 / Scan for Crashfish before approaching sandstone outcrops
- 被追击时利用地形阻挡——冲击鱼撞到障碍物会提前爆炸 / Use terrain — Crashfish explodes on impact with obstacles

## 大型掠食者 / Large Predators

### 血藻龙 / Blood Kelp Dragon

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **体型 / Size** | 大型 / Large (~15m) |
| **所在区域 / Location** | 血藻区 / Blood Kelp Zone |
| **攻击性 / Aggression** | 极高——看到玩家立即追击 / Extreme — chases on sight |
| **攻击方式 / Attack** | 撕咬（55伤害）+ 毒液（每秒8伤害持续5秒）/ Bite (55) + Venom (8/s for 5s) |

**应对策略 / Tactics**:
- **不要进入血藻区深处**直到你有了深潜者号和防御模块 / **Don't enter deep Blood Kelp Zone** until you have Deep Diver + Defense Module
- 利用海藻林作掩护——血藻龙体型大，难以在茂密区域穿行 / Use kelp for cover — too large to navigate dense areas
- 被咬中后的毒液需要用医疗包快速解除 / Treat venom with medkit immediately after being bitten
- 击败后掉落**血藻龙毒腺**——可用于制作强力武器 / Drops **Venom Gland** — used for powerful weapon crafting

### 深渊潜伏者 / Abyss Lurker

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **体型 / Size** | 大型 / Large (~12m) |
| **所在区域 / Location** | 深海大裂谷 / Deep Grand Ravine |
| **攻击性 / Aggression** | 中——伏击型，关灯时活跃 / Medium — ambush predator, active in darkness |
| **攻击方式 / Attack** | 突袭撕咬（45伤害）/ Ambush bite (45 damage) |

**应对策略 / Tactics**:
- 深渊潜伏者会被光源吸引——关掉手电筒可降低遭遇概率 / Attracted to light — turning off your flashlight reduces encounters
- 它在攻击前会发出低频轰鸣声——听到后果断闪避 / Emits a low-frequency rumble before attacking — dodge on hearing it
- 弱点：攻击后有3秒硬直——是反击窗口 / Weakness: 3-second recovery after attack — counter window
- 推进器的快速移动可以甩开追击 / Seaglide's speed outruns it

## 利维坦级生物 / Leviathan Class

### 幽灵利维坦 / Ghost Leviathan

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **体型 / Size** | 巨型 / Gigantic (~50m) |
| **所在区域 / Location** | 失落之河 / Lost River |
| **攻击性 / Aggression** | 极高——进入视线范围即追击 / Extreme — attacks on sight |
| **攻击方式 / Attack** | 冲撞（80伤害）+ 灵魂吸取（每秒15伤害持续6秒）/ Charge (80) + Soul Drain (15/s for 6s) |

**应对策略 / Tactics**:
- **这是全游戏第二危险的生物**——不要正面对抗 / **Second most dangerous in the game** — do not engage directly
- 幽灵利维坦有固定的巡逻路线——观察并规划绕行路线 / Has fixed patrol routes — observe and plan an avoidance path
- 独眼巨人号的静音模式可以有效降低被发现的概率 / Cyclops silent running mode significantly reduces detection
- 如果被追击：潜入狭窄岩缝——利维坦体型太大无法进入 / If chased: dive into tight crevices — too large to follow
- 击杀需要高级武器+大量弹药——通常不值得 / Killing requires advanced weapons + lots of ammo — usually not worth it

### 熔岩利维坦 / Lava Leviathan

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **体型 / Size** | 巨型 / Gigantic (~60m) |
| **所在区域 / Location** | 熔岩区 / Lava Zone |
| **攻击性 / Aggression** | 极限——全游戏最具攻击性的生物 / Maximum — most aggressive creature in the game |
| **攻击方式 / Attack** | 熔岩吐息（100伤害+灼烧5秒）+ 冲撞（65伤害）/ Lava breath (100 + burn 5s) + charge (65) |

**应对策略 / Tactics**:
- **全游戏最危险的生物**——遭遇时保持冷静 / **Most dangerous creature in the game** — stay calm when encountered
- 熔岩吐息伤害极高且有灼烧持续效果——被击中优先清理灼烧状态 / Lava breath deals massive damage with burn DoT — prioritize extinguishing
- 它在熔岩中移动极快——不要在开阔区域和它周旋 / Moves extremely fast in lava — don't engage in open areas
- 利用熔岩区的岩石隧道和狭窄通道规避 / Use rock tunnels and narrow passages for cover
- 独眼巨人号的**护盾发生器**是穿越熔岩区的最佳保障 / Cyclops **Shield Generator** is the best tool for navigating this zone
- **不建议尝试击杀**——耗时极长且收益有限 / **Not recommended to kill** — takes extremely long with limited reward
