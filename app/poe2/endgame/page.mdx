# 终局内容 / Endgame

当你完成第 7 章战役后，真正的冒险才刚刚开始。终局内容是 PoE2 最核心的玩法，
包括异界地图探索、首领挑战、装备打造和角色深度定制。

Once you complete Act 7, the real adventure begins. The endgame is the core of PoE2,
featuring atlas exploration, boss challenges, gear crafting, and deep character customization.

---

## 异界地图系统 / Atlas Mapping System

### 基本概念 / Core Concepts

异界地图系统是 PoE2 终局的主要刷图模式：

| 术语 / Term | 说明 / Description |
|---|---|
| 地图 / Map | 可进入的随机区域，由地图物品开启 |
| 地图等级 / Map Tier | 1-16 级，等级越高怪物越强掉落越好 |
| 异界天赋树 / Atlas Passive Tree | 影响地图产出和词缀的全局天赋 |
| 守望石 / Watchstone | 提升地图等级和奖励的道具 |
| 词缀 / Modifiers | 地图上的词缀增加难度与奖励 |
| 觉醒等级 / Awakening Level | 影响地图掉落和首领难度 |

### 地图流程 / Map Progression

1. 在 **藏身处 (Hideout)** 的地图仪中插入地图物品
2. 打开传送门进入地图区域
3. 清理地图内的怪物，找到地图首领
4. 击败首领获得掉落和下一等级的地图
5. 积累守望石提升异界等级

### 地图词缀说明 / Map Modifiers

| 词缀 / Modifier | 效果 / Effect | 难度 / Difficulty |
|---|---|---|
| 增加怪物数量 | 更多怪物 = 更多掉落 | 低 |
| 增加物品数量 | 直接提升掉落数量 | 中 |
| 怪物增伤 | 怪物伤害 +X% | 中 |
| 元素反射 | 攻击反射元素伤害 | 高 |
| 物理反射 | 攻击反射物理伤害 | 高 |
| 无法回复 | 生命/魔力无法自然回复 | 高 |
| 时空锁链 | 玩家减速 | 中 |

**新手提示：** 反射词缀对特定流派非常危险，建议使用与你的伤害类型
不匹配的地图词缀，或使用 **反射免疫** 天赋点。

## 首领战 / Boss Fights

### 终局首领类型 / Endgame Boss Types

| 首领 / Boss | 位置 / Location | 难度 / Difficulty | 奖励 / Rewards |
|---|---|---|---|
| 地图首领 / Map Boss | 每个地图最终区域 | 中 | 地图、通货、装备 |
| 征服者 / Conquerors | 异界特定区域 | 高 | 守望石、觉醒装备 |
| 裂界守卫 / Elderslayers | 裂界领域 | 很高 | 裂界装备、碎片 |
| 塑界者 / Shaper | 塑界领域 | 极高 | 塑界装备、独特物品 |
| 岳父 / Elder | 虚空领域 | 极高 | 岳父装备、独特物品 |
| 最终首领 / Uber Boss | 特殊条件解锁 | 传奇 | 最强独特物品 |

### 首领战准备 / Boss Preparation

| 准备项 / Item | 推荐数值 / Recommended Value |
|---|---|
| 生命 / Life | 3500+ |
| 元素抗性 / Elemental Resist | 75% |
| 混沌抗性 / Chaos Resist | 30%+ |
| 护甲 / Armour | 5000+（近战） |
| 能量护盾 / Energy Shield | 3000+（法系） |
| DPS | 50,000+ |
| 移动速度 / Move Speed | 25%+ |

### 首领战通用策略 / General Boss Strategy

1. **学习攻击模式** — 每个首领都有固定的技能循环
2. **不要贪刀** — 优先保证生存，在安全窗口输出
3. **准备抗性药水** — 特定首领需要针对性抗性
4. **使用位移技能** — 躲避大范围攻击的关键
5. **开启护甲/护盾光环** — 提高容错率

## 终极内容 / Pinnacle Content

### 裂隙 / Breach

裂隙是异界中随机出现的次元裂缝：

| 特性 / Feature | 说明 / Description |
|---|---|
| 开启方式 | 地图中随机出现裂隙手 |
| 奖励 | 大量怪物密度，裂隙专属独特物品 |
| 挑战 | 高难度裂隙首领：裂隙之王 |

### 深渊 / Abyss

深渊裂缝产出大量经验和专属装备：

| 特性 / Feature | 说明 / Description |
|---|---|
| 开启方式 | 地图中随机出现深渊裂缝 |
| 奖励 | 深渊装备、升级材料 |
| 挑战 | 深渊首领：深渊主教 |

### 穿越 / Incursion

通过阿尔瓦的穿越遇到过去的神庙：

| 特性 / Feature | 说明 / Description |
|---|---|
| 开启方式 | 与 NPC 阿尔瓦对话进入 |
| 奖励 | 神庙改造装备、双瓦房 |
| 挑战 | 神庙首领：建筑师 |

### 枯疫 / Blight

塔防式玩法，保护核心不被怪物侵蚀：

| 特性 / Feature | 说明 / Description |
|---|---|
| 开启方式 | 地图中随机出现枯疫感染 |
| 奖励 | 油瓶、独特戒指、大量通货 |
| 挑战 | 枯疫首领：枯疫之王 |

## 异界天赋树 / Atlas Passive Tree

与角色天赋树类似，异界天赋树影响你的地图体验。

### 关键天赋区域 / Key Atlas Regions

| 区域 / Region | 专注方向 / Focus | 推荐人群 / Recommended For |
|---|---|---|
| 裂隙区域 / Breach Region | 裂隙出现率 + 奖励 | 喜欢高密度刷图的玩家 |
| 深渊区域 / Abyss Region | 深渊产出 + 经验 | 快速升级的玩家 |
| 地图区域 / Map Region | 地图掉落 + 等级 | 推进异界进度的玩家 |
| 首领区域 / Boss Region | 首领掉落 + 地图充能 | 专注首领战的玩家 |

### 新手推荐天赋 / Recommended Early Passives

1. **地图掉落概率** — 确保地图不断档
2. **物品数量加成** — 提升每次刷图的收益
3. **怪群数量** — 更多怪物 = 更多经验 + 掉落
4. **工艺材料掉落** — 积累打造资源

## 藏身处系统 / Hideout System

藏身处是你的私人基地，可自由装饰和放置 NPC：

| 功能 / Function | 描述 / Description |
|---|---|
| 地图仪 / Map Device | 进入地图的核心设备 |
| 工艺台 / Crafting Bench | 装备打造 |
| 仓库 / Stash | 存放物品，可购买额外标签页 |
| NPC 进驻 | 可将商人、铁匠等 NPC 放入藏身处 |
| 装饰系统 | 自由布置藏处场景 |
