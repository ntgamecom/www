# 技能宝石 / Skill Gems

技能宝石是流放之路2 的核心战斗系统。每个技能宝石对应一个主动技能或辅助效果，
通过装备上的插槽系统组合使用。

Skill gems are the core combat system of Path of Exile 2. Each gem corresponds to an
active skill or support effect, combined through the socket system on gear.

---

## 技能宝石基础 / Skill Gem Basics

### 宝石类型 / Gem Types

| 类型 / Type | 颜色 / Color | 说明 / Description |
|---|---|---|
| 主动技能宝石 / Active Skill Gem | 红色/绿色/蓝色 | 提供可使用的主动技能 |
| 辅助宝石 / Support Gem | 对应颜色 | 增强或改变主动技能的效果 |
| 光环宝石 / Aura Gem | 多种颜色 | 提供持续性光环效果 |
| 诅咒宝石 / Curse Gem | 蓝色/绿色 | 对敌人施加减益效果 |

### 插槽系统 / Socket System

不同于初代，PoE2 的装备直接内置插槽：

| 装备 / Equipment | 插槽数量 / Sockets | 说明 / Notes |
|---|---|---|
| 主手武器 / Main Hand | 2-4 | 主要技能插槽 |
| 副手武器 / Off Hand | 1-2 | 辅助技能 |
| 头盔 / Helmet | 1-2 | 光环/诅咒 |
| 胸甲 / Body Armour | 2-4 | 大型技能组 |
| 手套 / Gloves | 1-2 | 移动/功能性技能 |
| 鞋子 / Boots | 1-2 | 光环/辅助 |

### 技能连接 / Skill Linking

插槽之间的连线决定了宝石之间的连接关系：

- **直接连接** 的插槽：主动宝石 + 辅助宝石组合生效
- **未连接** 的插槽：技能独立运行，不受辅助宝石影响
- 连线数量决定了你可以串联多少个辅助宝石

## 辅助宝石详解 / Support Gems Guide

辅助宝石是改变技能行为的关键。每个辅助宝石为连接的主动技能提供特定加成。

### 红色辅助宝石 / Red Support Gems

| 宝石 / Gem | 效果 / Effect |
|---|---|
| 近战物理伤害 / Melee Physical Damage | 近战物理伤害 +30% |
| 扩大範圍 / Increased AoE | 范围技能半径 +20% |
| 护体 / Fortify | 击中敌人时获得护体减伤 |
| 几率点燃 / Chance to Ignite | 攻击附加 25% 概率点燃 |

## 技能宝石升级 / Gem Leveling

技能宝石通过击杀怪物获得经验值并自动升级。

### 升级收益 / Leveling Benefits

| 宝石等级 / Gem Level | 效果提升 / Effect Increase |
|---|---|
| 1-5 | 基础伤害提升 10-20%，需求等级增长 |
| 6-10 | 伤害显著提升，可能增加额外投射物或范围 |
| 11-15 | 伤害大幅提升，解锁附加效果 |
| 16-20 | 接近满级，伤害达到最大化 |

### 升级技巧 / Leveling Tips

1. **优先升级主力技能** — 集中经验到你的主要输出技能
2. **辅助宝石可稍后升级** — 辅助宝石的等级不如主动技能关键
3. **不要忽视光环宝石** — 光环效果随等级提升显著
4. **使用宝石匠的棱镜** — 特定通货可提升宝石品质

### 品质提升 / Gem Quality

通过 **宝石匠的棱镜 (Gemcutter's Prism)** 提升技能宝石品质：

| 品质 / Quality | 每 1% 效果 / Effect per 1% |
|---|---|
| 主动技能宝石 | +0.5% 技能效果 |
| 辅助宝石 | +0.5% 辅助效果 |
| 光环宝石 | +0.5% 光环效果 |
| 诅咒宝石 | +0.5% 诅咒效果 |

**最大品质：20%**

## 实用技巧 / Pro Tips

1. **武器切换** 允许你设置两套技能组合，应对不同场景
2. **辅助宝石可以随时更换** — 不要害怕尝试新组合
3. **技能特效透明度** 可在设置中调整，提高高密度战斗的可视性
4. **查看技能面板 (G)** 可以详细查看每个技能的 DPS、范围和冷却
5. **部分辅助宝石有属性需求** — 确保你的属性达标
