## 流派构建基础 / Build Fundamentals

流放之路2的Build构建围绕**核心技能**展开——选择一个主技能，围绕它选择对应的**辅助宝石**、**装备词缀**和**天赋**。所有Build都需要平衡伤害、生存和清图效率 / PoE2 builds center around a **core skill** — choose one main skill, then select supporting **support gems**, **gear affixes**, and **passives**. Every build needs to balance damage, survivability, and clear speed.

### Build选择要素 / Build Selection Factors

| 要素 / Factor | 重要性 / Importance | 说明 / Notes |
|---|---|---|
| 伤害类型 / Damage Type | 核心 / Core | 元素/物理/混沌——决定天赋走向 / Elemental/Physical/Chaos — determines passive direction |
| 生存机制 / Survival | 必要 / Essential | 护盾/护甲/闪避/格挡——至少选两个 / ES/Armour/Evasion/Block — pick at least two |
| 清图能力 / Clear Speed | 重要 / Important | 范围伤害+移速决定刷图效率 / AoE + movespeed determines mapping efficiency |
| 单体伤害 / Single Target | 关键 / Critical | 对Boss和终局内容的输出能力 / Boss and endgame damage output |
| 造价 / Budget | 实际 / Practical | 入门Build vs 终局Build的装备成本差距 / Entry vs endgame gear cost difference |

### 开荒Build原则 / League Starter Principles

开荒Build需要低装备门槛、平滑升级曲线和良好清图能力。推荐选择**元素伤害**为主的Build——因为元素伤害在前期没有抗性穿透压力时输出最高。**混沌伤害Build**开荒较慢——前期装备不好时伤害偏低 / League starters need low gear requirement, smooth leveling curve, and good clear speed. Prioritize **elemental damage** builds — highest early damage without penetration pressure. **Chaos builds** are slower to league start — low damage without proper gear.

## 全职业顶流Build / Top Meta Builds

### 法师(元素使) / Sorceress (Elementalist)

**冰霜新星流 / Frost Nova Build**

核心技能: **冰霜新星** + 辅助: **多重投射** + **冰冷穿透** + **施法回响** + **元素集中** + **冰冷伤害**。天赋走向: 元素伤害→冰冷伤害→施法速度→法力回复。升华选择**元素使**——元素伤害+25%——元素穿透+10% / Core: **Frost Nova** + supports: **Multiple Projectiles** + **Cold Penetration** + **Spell Echo** + **Elemental Focus** + **Cold Damage**. Passives: Elemental → Cold → Cast speed → Mana. Ascendancy: **Elementalist** — +25% elemental damage, +10% penetration.

装备需求: 法杖词缀——+1全法术等级、法术伤害、施法速度。头盔——+1冰冷法术等级。手套——法术附加冰冷伤害。戒指——法力回复速度。独特装备推荐: **冰冷之眼**——+1冰冷技能等级的头盔 / Gear: Wand with +1 all spell skills, spell damage, cast speed. Helmet with +1 cold skills. Gloves with added cold. Rings with mana regen. Unique: **Frost Gaze** helm with +1 cold skills.

**强度评价: S级**——清图速度极快，单体输出优秀，装备门槛适中。弱点: 移速慢、防御薄。推荐开荒指数: ★★★★★ / **Tier: S** — excellent clear speed, strong single target, moderate gear. Weakness: slow, fragile. League start: ★★★★★

**火球术流 / Fireball Build**

核心技能: **火球术** + 辅助: **多重投射** + **火焰穿透** + **点燃扩散** + **法术暴击** + **火焰伤害**。天赋走向: 元素→火焰→点燃几率→暴击。配合**元素超载**——非暴击的暴击时伤害x1.4 / Core: **Fireball** + supports: **Multiple Projectiles** + **Fire Penetration** + **Ignite Spread** + **Spell Crit** + **Fire Damage**. Passives: Elemental → Fire → Ignite chance → Crit. Sync with **Elemental Overload** — non-crit crits do x1.4 damage.

**强度评价: A+级**——清图能力超强、点燃机制优秀、清图视觉效果好。弱点: 单体伤害不如冰系。推荐开荒指数: ★★★★ / **Tier: A+** — excellent clear, strong ignite mechanic, great visual. Weakness: single target weaker than cold. League start: ★★★★

### 游侠(锐眼) / Ranger (Deadeye)

**闪电箭流 / Lightning Arrow Build**

核心技能: **闪电箭** + 辅助: **弹幕** + **闪电穿透** + **暴击伤害** + **快速攻击** + **元素集中**。天赋走向: 弓伤害→闪电伤害→暴击率→暴击伤害→攻速。升华**锐眼**——+1投射物——远距离伤害+20%——穿透+2目标 / Core: **Lightning Arrow** + supports: **Barrage** + **Lightning Pen** + **Crit Damage** + **Faster Attacks** + **Elemental Focus**. Passives: Bow → Lightning → Crit → Attack speed. Ascendancy: **Deadeye** — +1 projectile, +20% far damage, +2 pierce.

装备需求: 弓——高物伤基底+闪电点伤+暴击率+攻速。箭袋——+1箭矢、攻速、暴击加成。胸甲——高闪避+生命。鞋子——移动速度。独特推荐: **死亡之作**——低血时攻速翻倍的弓 / Gear: Bow with high phys + added lightning + crit + attack speed. Quiver with +1 arrow, attack speed, crit multi. Chest: high evasion + life. Boots: movespeed. Unique: **Death's Harp** — double attack speed on low life.

**强度评价: S级**——速刷天花板，移速极快，清图效率顶级。弱点: 生存较弱、需要操作。推荐开荒指数: ★★★★ / **Tier: S** — fastest mapper, excellent clear. Weakness: weaker survival, requires skill. League start: ★★★★

### 决斗者(冠军) / Duelist (Champion)

**旋风斩流 / Cyclone Build**

核心技能: **旋风斩** + 辅助: **粉碎** + **近战物理伤害** + **暴击伤害** + **护体** + **增加范围**。天赋走向: 物理→近战→攻速→暴击→生命回复。升华**冠军**——护体效果+100%——护体时免疫流血/中毒 / Core: **Cyclone** + supports: **Pulverize** + **Melee Phys** + **Crit Damage** + **Fortify** + **Increased AoE**. Passives: Phys → Melee → Attack speed → Crit → Life regen. Ascendancy: **Champion** — +100% fortify effect — immune to bleed/poison while fortified.

装备需求: 双手剑/斧——高物伤、高攻速、暴击率。头盔——生命+护甲。胸甲——高护甲+生命。戒指——物理点伤+生命。腰带——生命+元素抗性 / Gear: 2H sword/axe with high phys, attack speed, crit. Helmet: life + armour. Chest: high armour + life. Rings: phys + life. Belt: life + resists.

**强度评价: A级**——经典近战Build，生存强、稳定输出。弱点: 清图速度中等、装备造价较高。推荐开荒指数: ★★★ / **Tier: A** — classic melee, strong survival, consistent DPS. Weakness: average clear, higher gear cost. League start: ★★★

### 女巫(死灵法师) / Witch (Necromancer)

**召唤骷髅流 / Summon Skeleton Build**

核心技能: **召唤骷髅** + 辅助: **召唤生物伤害** + **多重投射** + **召唤生物速度** + **元素大军** + **召唤生物生命**。天赋走向: 召唤生物伤害→数量→生命→抗性→光环效果。升华**死灵法师**——+2召唤骷髅上限——召唤生物+30%伤害 / Core: **Summon Skeleton** + supports: **Minion Damage** + **Multiple Projectiles** + **Minion Speed** + **Elemental Army** + **Minion Life**. Passives: Minion damage → count → life → resists → aura effect. Ascendancy: **Necromancer** — +2 skeleton cap, +30% minion damage.

装备需求: 法杖——+1召唤技能、召唤生物伤害。胸甲——+1召唤技能、光环效果。头盔——+1召唤技能。珠宝——召唤生物攻速+伤害。独特推荐: **亡灵之音**——召唤生物+1等级的头盔 / Gear: Wand with +1 minion, minion damage. Chest: +1 minion, aura effect. Helmet: +1 minion. Jewels: minion attack speed + damage. Unique: **Death's Whisper** helm with +1 minion level.

**强度评价: A+级**——最适合新手的Build之一——操作简单安全。弱点: 召唤物AI不稳定、清Boss慢。推荐开荒指数: ★★★★★ / **Tier: A+** — best for beginners — simple, safe. Weakness: minion AI inconsistency, slower boss kill. League start: ★★★★★

### 暗影刺客(刺客) / Shadow (Assassin)

**毒性旋涡流 / Toxic Vortex Build**

核心技能: **毒性旋涡** + 辅助: **毒性扩散** + **持续伤害加成** + **减速** + **范围扩大** + **持续时间延长**。天赋走向: 混沌伤害→持续性伤害→范围→闪避→毒混伤。升华**刺客**——中毒有5%几率造成双倍伤害——对中毒敌人+30%暴击率 / Core: **Toxic Vortex** + supports: **Toxin Spread** + **Damage Over Time** + **Slow** + **Increased AoE** + **Duration**. Passives: Chaos → DoT → AoE → Evasion → Chaos hybrid. Ascendancy: **Assassin** — poison 5% double damage, +30% crit vs poisoned.

**强度评价: B+级**——持续伤害高、安全(放毒后跑)。弱点: 爆发不足、被免疫中毒地图完克。推荐开荒指数: ★★ / **Tier: B+** — high DoT, safe (kite gameplay). Weakness: low burst, countered by poison immunity maps. League start: ★★

## 装备打造基础 / Gear Crafting Basics

### 通货使用优先级 / Currency Use Priority

| 通货 / Currency | 用途 / Use | 优先级 / Priority |
|---|---|---|
| 混沌石 / Chaos Orb | 重置稀有装备词缀 / Reroll rare affixes | 高(终局) / High (endgame) |
| 神圣石 / Divine Orb | 重置数值/制作强力装备 / Reroll values/craft | 极高(终局) / Very High (endgame) |
| 点金石 / Alchemy Orb | 白装变金装 / Normal to rare | 中(升级期) / Medium (leveling) |
| 改造石 / Alteration Orb | 重铸魔法装备 / Reroll magic affixes | 中(制作底材) / Medium (base crafting) |
| 富豪石 / Regal Orb | 魔法变稀有 / Magic to rare | 低-中 / Low-Medium |
| 剥离石 / Annulment Orb | 移除随机词缀 / Remove random affix | 低(高风险) / Low (risky) |
| 瓦尔宝珠 / Vaal Orb | 瓦尔腐化——随机效果 / Vaal corruption — random outcome | 低(终局拼运气) / Low (endgame gamble) |

### 装备制作流程 / Gear Crafting Process

**底材选择**: 找对应Build的高物等基底——优先84+物等(可洗出T1词缀)。**魔法阶段**: 用改造石洗出需要的2个前缀或后缀——目标是有1个T1核心词缀。**升级稀有**: 使用富豪石变成稀有——如果获得的是有用词缀则可以继续。**追加词缀**: 用混沌石继续重铸直到满意——或使用工艺台加一个可选词缀 / **Base selection**: Find high-ilvl base for your build — prioritize 84+ (unlocks T1 affixes). **Magic stage**: Use Alteration to get 2 desired prefixes or suffixes — aim for 1 T1 core affix. **Upgrade**: Use Regal Orb — if useful affix gained, continue. **Add affixes**: Use Chaos to reroll until satisfied — or use crafting bench for one chosen affix.

## 天赋树规划 / Passive Tree Planning

**升级期(1-70级)**: 优先拿伤害天赋——直奔核心伤害大点。生存只拿路过时的生命/护盾小点。70级前不要洗点——留到最后再优化。推荐使用**天赋模拟器**如PoB提前规划路线 / **Leveling (1-70)**: Prioritize damage passives — rush to core damage nodes. Only take life/ES small nodes along the way. Don't respec before 70 — optimize later. Use **passive tree planner** like PoB for route planning.

**终局优化**: 用**后悔之书**洗点——重新分配为终局Build模板。比例分配: 伤害:生存:功能 = 6:3:1。**珠宝插槽**优先开——一颗好的珠宝相当于5-8个天赋点 / **Endgame Optimization**: Use **Orb of Regret** to respec — adjust for final build. Ratio: damage:survival:utility = 6:3:1. **Jewel sockets** priority — 1 good jewel = 5-8 passive points.

**异界天赋树**: 击杀T14+地图Boss获得异界天赋点。优先级: 地图掉落率→高阶地图掉率→Boss遭遇x2→赛季内容x2→地图品质 / **Atlas Passive Tree**: Kill T14+ map bosses for Atlas points. Priority: Map drop rate → Higher tier → Boss encounter x2 → League content x2 → Map quality.

## 升华试炼攻略 / Ascendancy Trial Guide

升华试炼分为三个阶段: **普通试炼**(第三章)、**残酷试炼**(第四章)、**终极试炼**(第六章)。每完成一个获得2点升华点数——共6点。试炼内容为迷宫——需要应对机关陷阱和最终Boss。**提示**: 带足够的生命药水——迷宫中的机关造成固定百分比伤害。使用位移技能大幅降低难度。推荐等级比试炼要求高5级再去 / Ascendancy Trials: **Normal** (Act 3), **Cruel** (Act 4), **Ultimate** (Act 6). Each grants 2 Ascendancy points — 6 total. Trials are labyrinths with traps and final boss. **Tip**: Bring ample life flasks — traps do %HP damage. Mobility skills greatly reduce difficulty. Recommended 5 levels above trial requirement.
