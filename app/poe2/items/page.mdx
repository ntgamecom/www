# 装备与交易 / Items & Trading

装备是你在瓦尔克拉斯生存和战斗的根本。本指南涵盖装备类型、通货系统、
交易市场以及基础打造方法。

Equipment is your lifeline in Wraeclast. This guide covers gear types, currency system,
the trading marketplace, and basic crafting methods.

## 交易系统 / Trading System

PoE2 使用官方的交易网站进行玩家间交易。

### 交易流程 / Trading Process

1. **访问官方交易网站** / Visit the official trade site
2. **搜索你需要的装备** / Search for the item you need
3. **使用过滤条件缩小范围** / Use filters to narrow results
4. **点击 "Whisper" 按钮** / Click the Whisper button
5. **游戏内自动发送私信** / Automatic in-game whisper is sent
6. **卖方邀请你组队** / Seller invites you to party
7. **前往卖方的藏身处** / Travel to seller's hideout
8. **将通货放入交易窗口** / Place currency in trade window
9. **确认交易** / Confirm the trade

### 交易技巧 / Trading Tips

| 技巧 / Tip | 说明 / Explanation |
|---|---|
| 设置价格过滤器 | 排除超出预算的物品 |
| 使用批量搜索 | 一次搜索多个词缀组合 |
| 关注在线状态 | 只搜索 "在线" 的卖家 |
| 学习缩写 | 如：6L = 6 连，T1 = 顶级词缀 |
| 还价礼仪 | 礼貌询问是否可议价 |
| 注意骗局 | 检查物品确认后再点确认 |
| 使用公开仓库 | 设置公开仓库标签页自动上架 |

## 基础打造入门 / Basic Crafting Guide

### 四个核心打造策略 / 4 Core Crafting Strategies

#### 1. 蜕变 → 增幅 (Transmute → Augment)

最快最基础的方法，适合制作前期过渡装备：

```
普通装备 → 蜕变石 → 魔法装备(1词缀) → 增幅石 → 魔法装备(2词缀)
```

#### 2. 蜕变 → 增幅 → 富豪 (T-A-Regal)

制作稀有前期装备：

```
普通 → 蜕变(1词缀) → 增幅(2词缀) → 富豪(变稀有,3词缀)
```

#### 3. 混沌石重铸 (Chaos Spam)

使用混沌石反复重置稀有装备的词缀：

```
稀有装备 → 混沌石 → 新的稀有装备（重复直到满意）
```

**注意：** 成本较高，适合中后期制作

#### 4. 工艺台补足 (Bench Craft)

使用工艺台为装备添加缺失的关键词缀：

```
稀有装备（缺抗性） → 工艺台 → 附加抗性词缀
```

## 装备管理建议 / Gear Management Tips

1. **使用过滤规则** — 高亮有价值的基底和词缀组合
2. **整理仓库** — 按用途分类：打造材料、地图、装备、通货
3. **定期鉴定** — 不要错过潜在的极品装备
4. **学习词缀池** — 了解哪些词缀可以同时存在
5. **不要执着于完美** — 够用就好，逐步替换
