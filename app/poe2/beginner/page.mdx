# 新手入门 / Beginner Guide

欢迎来到瓦尔克拉斯！本指南将帮助新玩家从零开始，顺利度过游戏前期，避开常见误区。

Welcome to Wraeclast! This guide will help new players start from zero, progress smoothly
through the early game, and avoid common pitfalls.

---

## 第一章：游戏开始 / Getting Started

### 职业选择建议 / Class Selection Advice

对于第一次游玩的新手，以下职业较为友好：

For first-time players, the following classes are more beginner-friendly:

| 职业 / Class | 上手难度 / Difficulty | 推荐理由 / Reason |
|---|---|---|
| 战士 / Warrior | 低 / Easy | 近战高血量，容错率高 / High HP pool, forgiving melee |
| 游侠 / Ranger | 中 / Medium | 远程安全距离，操作直观 / Safe ranged gameplay |
| 女巫 / Witch | 中 / Medium | 召唤流自动战斗，适合懒人 / Minion playstyle, relaxed pace |
| 佣兵 / Mercenary | 低 / Easy | 远程物理，装备平滑 / Ranged物理， gear progression |
| 僧侣 / Monk | 高 / Hard | 近战敏捷，操作要求高 / Melee evasion, high skill ceiling |
| 女术士 / Sorceress | 高 / Hard | 元素法师，需走位 / Elemental caster, positioning required |

### 起始操作 / First Steps

1. 完成序章区域（河岸断崖 / Riverbank）并进入小镇 **克里格尔村 / Krigel Village**
2. 接受第一个任务并与 NPC **乌娜 / Una** 对话获取初始技能宝石
3. 装备武器并与 **技能宝石插槽** 互动，装入你的第一个技能
4. 按 `I` 打开背包，按 `P` 打开天赋树，熟悉界面布局
5. 完成任务 **"通往灾祸之路 / Road to Ruin"** 解锁第一个辅助宝石插槽

## 第三章：基础机制 / Core Mechanics

### 生命与能量护盾 / Life & Energy Shield

- **生命 (Life)**：基础生存资源，可通过装备和天赋提升。新手建议至少 1500-2000 生命进入终局
- **Energy Shield**：法力护盾，优先吸收元素伤害，适合女术士和僧侣
- **护甲 (Armour)**：减少物理伤害，战士专精
- **闪避 (Evasion)**：概率躲避攻击，游侠和僧侣优先

### 抗性 / Resistances

抗性是游戏中最关键的防御属性之一：

Resistance is one of the most critical defensive stats:

- 火焰抗性 / Fire Resistance — 减少火焰伤害
- 冰霜抗性 / Cold Resistance — 减少冰霜伤害
- 闪电抗性 / Lightning Resistance — 减少闪电伤害
- 混沌抗性 / Chaos Resistance — 减少混沌伤害（后期重要）

**目标：** 在第 4-5 章时将三元素抗性堆满至 75%，混沌抗性在终局前达到 30%+。

**Target:** Cap elemental resistances at 75% by Acts 4-5, chaos resistance at 30%+ before endgame.

### 属性 / Attributes

| 属性 / Stat | 关联职业 / Classes | 作用 / Function |
|---|---|---|
| 力量 / Strength | 战士、佣兵 | 增加生命和近战物理伤害 |
| 敏捷 / Dexterity | 游侠、僧侣 | 增加闪避和命中 |
| 智力 / Intelligence | 女术士、女巫 | 增加能量护盾和法术伤害 |

### 技能宝石系统 / Skill Gem System

技能宝石是 PoE2 的核心战斗系统。不同于初代的插槽连接机制，PoE2 将技能宝石
直接插入武器或防具上的内置插槽中。辅助宝石则放置在相连的插槽中。

Skill gems are the core combat system. Unlike PoE1's linked socket system, PoE2 has
skill gems inserted directly into weapon or armor sockets. Support gems are placed
in connected sockets.

详细说明请参阅 **技能宝石 / Skill Gems** 页面。

See the **Skill Gems** page for details.

## 第四章：前期升级指南 / Early Leveling Guide

### 1-20 级快速升级 / Fast Leveling 1-20

1. **选择范围技能** — 优先选择能击中多个敌人的技能清图
2. **保持装备更新** — 每 5 级检查武器伤害，每 10 级检查防具
3. **不做支线任务** — 除非奖励技能点或辅助宝石
4. **使用移动技能** — 尽早获取冲刺或传送技能提高跑图速度
5. **收集鉴定卷轴** — 用于鉴定掉落的高基底装备

### 常用升级技能推荐 / Recommended Leveling Skills

| 职业 / Class | 推荐技能 / Recommended Skills |
|---|---|
| 战士 / Warrior | 地震打击 / Earthquake Strike, 猛击 / Slam |
| 游侠 / Ranger | 分裂箭 / Split Arrow, 毒蛇打击 / Viper Strike |
| 女术士 / Sorceress | 火球 / Fireball, 冰霜新星 / Frost Nova |
| 僧侣 / Monk | 风暴之拳 / Storm Fist, 疾风击 / Wind Strike |
| 佣兵 / Mercenary | 弹幕 / Barrage, 爆炸箭 / Explosive Arrow |
| 女巫 / Witch | 召唤骷髅 / Summon Skeleton, 混沌之箭 / Chaos Bolt |

## 第五章：常见误区 / Common Mistakes

### 新手最容易犯的 10 个错误 / Top 10 Beginner Mistakes

| 编号 | 错误 / Mistake | 正确做法 / Correct Approach |
|---|---|---|
| 1 | 忽视抗性 / Ignoring resistances | 优先堆满元素抗性至 75% |
| 2 | 混合伤害类型 / Mixing damage types | 专精一种伤害类型（物理/元素/混沌） |
| 3 | 忽略生命词缀 / Skipping life mods | 每个装备位都要有生命词缀 |
| 4 | 过度囤积通货 / Hoarding currency | 适当使用通货提升装备 |
| 5 | 不使用工艺台 / Not using crafting bench | 工艺台可补足短板词缀 |
| 6 | 天赋点分布过于分散 / Spreading passives | 集中投资关键天赋 cluster |
| 7 | 技能宝石等级过低 / Low gem levels | 及时升级技能宝石 |
| 8 | 不关注装备基底 / Ignoring base types | 基底决定了装备的防御类型 |
| 9 | 拒绝交易 / Refusing to trade | 合理利用交易提升效率 |
| 10 | 不查资料 / Not researching | 善用 Wiki 和社区 Build 指南 |

### 生存贴士 / Survival Tips

- 遇到困难区域时，先检查抗性 → 再检查生命总值 → 最后考虑更换技能
- 不要害怕使用 **复活币**，它们在战役中会不断掉落
- 每个村庄都有 **传送门**，可以快速返回补充药水
- 与 NPC **埃布 / Ebb** 交易时，优先购买移动速度加成的鞋子
