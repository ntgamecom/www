# 职业详解 / Classes & Ascendancies

流放之路2 提供了 **6 个基础职业**，每个职业可以选择 **2 个升华职业（Ascendancy）**，
共计 12 种进阶方向。职业决定了你的起始属性、天赋树起点和可用技能类型。

Path of Exile 2 offers **6 base classes**, each with **2 ascendancies**, for a total of
12 advanced specializations. Your class determines starting attributes, passive tree
position, and available skill types.

---

## 战士 / Warrior

战士是力量的化身，擅长近战物理攻击，拥有最高的基础生命和护甲。
适合喜欢正面战斗、操作稳健的玩家。

The Warrior embodies strength, excelling at melee physical attacks with the highest
base life and armour. Ideal for players who enjoy straightforward, durable combat.

### 基础属性 / Base Stats

| 属性 / Stat | 初始值 / Starting Value |
|---|---|
| 力量 / Strength | 32 |
| 敏捷 / Dexterity | 14 |
| 智力 / Intelligence | 14 |
| 生命 / Life | 85 |
| 护甲 / Armour | 12 |

### 升华职业 / Ascendancies

#### 战争使者 / Warbringer

专注于增益效果和团队支援的坦克型升华：

| 能力 / Ability | 效果 / Effect |
|---|---|
| 战吼强化 / War Cry Mastery | 战吼效果提升 50%，范围扩大 |
| 铁壁 / Iron Wall | 护甲额外提供 5% 物理减免 |
| 军旗 / Banner | 放置军旗提供团队光环 |
| 毁灭冲锋 / Destruction Charge | 冲锋技能造成双倍伤害 |

#### 图腾使者 / Totem Master

召唤图腾辅助战斗的升华，适合喜欢炮台玩法的玩家：

| 能力 / Ability | 效果 / Effect |
|---|---|
| 图腾增殖 / Totemic Proliferation | 可同时放置 3 个图腾 |

| 坚韧图腾 / Stalwart Totem | 图腾提供防御光环 |
| 图腾链接 / Totem Link | 图腾被击毁时爆炸造成 AoE |

### 推荐 Build / Recommended Builds

- **地震战士 / Earthquake Warrior** — 高范围物理伤害，清图效率高
- **图腾坦克 / Totem Tank** — 图腾输出 + 自身防御，适合新手
- **流血猛击 / Bleed Slam** — 高额流血 DoT，针对首领战

## 进阶职业获取 / Unlocking Ascendancies

升华职业需要在战役中完成特定任务解锁：

| 步骤 / Step | 要求 / Requirement |
|---|---|
| 1 | 完成第 5 章首领战 / Complete Act 5 boss |
| 2 | 进入升华迷宫 / Enter the Labyrinth |
| 3 | 完成第一次升华试炼 / Complete first Ascendancy trial |
| 4 | 获得前 2 个升华点 / 获得 2 Ascendancy points |
| 5 | 在第 6-7 章继续完成更多试炼 / Complete further trials for additional points |

每个升华职业最多可获得 **8 个升华天赋点**，建议优先投资核心能力。

Each Ascendancy can earn up to **8 Ascendancy points**; prioritize core abilities first.
