## 主线任务流程 / Main Quest Line

流放之路2的主线分为六章——每章包含多个区域和Boss战。全程约8-12小时(熟悉后)——新手第一次可能需要15-20小时 / PoE2's main story spans six acts — each containing multiple zones and boss fights. Full run ~8-12 hours (experienced) — 15-20 hours for first time.

| 章节 / Act | 区域 / Zones | 主线Boss / Boss | 推荐等级 / Level Range | 关键奖励 / Key Reward |
|---|---|---|---|---|
| 第一章 / Act 1 | 溺亡海岸、海穴、僧院、监狱 / Drowned Coast, Sea Caves, Monastery, Prison | 莫薇儿 / Merveil | 1-16 | 海妖之泪、辅助宝石解锁 |
| 第二章 / Act 2 | 森林营地、瓦尔遗迹、沙漠 / Forest Camp, Vaal Ruins, Desert | 瓦尔超灵 / Vaal Oversoul | 16-28 | 瓦尔宝石、铁匠解锁 |
| 第三章 / Act 3 | 乌旗兵营、机械工厂、圣域 / Banner Barracks, Factory, Sanctum | 乌旗守卫 / Risen Flagellant | 28-45 | 附魔师解锁、升华点 |
| 第四章 / Act 4 | 地狱前厅、熔炉、混沌神殿 / Hell's Anteroom, Forge, Chaos Temple | 玛拉凯 / Malachai | 45-55 | 混沌核心、升华点 |

**主线速推技巧**: 每个区域只做主线任务和相关的重要支线。跳过重复刷怪。在等级达到区域推荐等级+3时推进下一个区域。**天赋之书**任务给的天赋点优先拿——提高效率。组队速推比单人快30-40%——但需要协调好 / **Fast push tips**: Do only main quest + critical side quests per zone. Skip grinding. Move to next zone when level is zone+3. **Book of Skill** quests granting passive points are priority. Party play is 30-40% faster than solo.

### 功能解锁支线 / Function Unlock Quests

**图书馆任务(第一章)**: 在僧院书库找到3本失落的法术书——交给NPC解锁**辅助宝石商人**——以后可以购买所有基础辅助宝石。**重要性: 极高**——没有辅助宝石你的技能没有连携效果 / **Library Quest (Act 1)**: Find 3 lost spellbooks in Monastery library — unlocks **Support Gem Vendor** — buy all basic support gems thereafter. **Importance: Critical** — without support gems, skills lack linking effects.

**铁匠任务(第二章)**: 在沙漠中找到铁匠阿祖里的3把传奇锤子碎片。完成后解锁**6孔装备制作**和**品质升级**功能——终局装备的基础。解锁后可以在铁匠处为武器增加品质(最多20%)——每1%品质=1%物理伤害 / **Blacksmith Quest (Act 2)**: Find 3 pieces of Azurite's hammer in desert. Unlocks **6-socket crafting** and **Quality upgrade** — foundation of endgame gearing. Each 1% quality = 1% phys damage on weapons (max 20%).

**附魔师任务(第三章)**: 在兵营找到附魔师的3本附魔书。解锁**附魔系统**——可以为装备添加额外词缀。**附魔类型**: 攻击附加元素伤害、移动速度、抗性上限提升等 / **Enchanter Quest (Act 3)**: Find 3 enchantment tomes in barracks. Unlocks **Enchantment System** — add extra affixes to gear. **Enchant types**: Added elemental damage, movement speed, max resist boost, etc.

**地图设备任务(第五章)**: 击杀深渊之主后获得**地图设备**——在藏身处放置后可以开始异界地图之旅。完成深渊之主的任务线还将解锁**地图升级**功能——三张同阶合成升一阶 / **Map Device Quest (Act 5)**: Get **Map Device** after killing Lord of the Abyss — place in hideout to start Atlas mapping. Completing Lord's quest also unlocks **Map Upgrade** recipe — 3 same-tier maps upgrade to 1 next-tier.

### 重要NPC任务线 / Important NPC Quest Chains

**航海家伊娜**: 在第二章和第五章出现——完成她的任务链获得**藏身处装饰品**和各种**地图碎片**。最终奖励是**航海家罗盘**——增加异界地图中遭遇赛季内容的几率 / **Navigator Ina**: Appears in Act 2 and 5 — her quest chain rewards **Hideout Decorations** and **Map Fragments**. Final reward: **Navigator's Compass** — increases league content encounter rate in Atlas maps.

**神秘学者奥里亚斯**: 在第三章出现——完成他的3个连锁任务可以解锁**瓦尔宝石**的高级使用方法——包括**双瓦**(装备获得2个瓦尔词缀)的配方 / **Occultist Orias**: Appears in Act 3 — completing 3 linked quests unlocks advanced **Vaal Gem** usage — including **Double Vaal** recipe (gear gains 2 Vaal affixes).

## 终局异界任务 / Endgame Atlas Quests

### 觉醒者任务链 / Awakener Quest Chain

**前置条件**: 完成全部六章主线并在异界地图中达到等级80+ / **Prerequisite**: Complete all 6 acts, reach level 80+ in Atlas maps

| 步骤 / Step | 内容 / Content | 奖励 / Reward |
|---|---|---|
| 1. 收集四块觉醒碎片 | 从T14+地图Boss掉落 / Drops from T14+ map bosses | 每碎片解锁异界天赋点 / Unlock Atlas passive points |
| 2. 合成觉醒者之玉 | 在觉醒祭坛合成4碎片 / Fuse 4 fragments at Awakening Altar | 开启塑界者领域 / Opens Shaper's Realm |
| 3. 击败塑界者 / Defeat Shaper | 在塑界者领域战斗 / Fight in Shaper's Realm | 塑界者碎片 + 异界天赋点 |
| 4. 裂界者入侵 / Eradicator Invasion | 在异界地图中触发裂界者事件 / Trigger Eradicator in maps | 高阶地图 + 裂界者精华 |
| 5. 最终觉醒 / Final Awakening | 觉醒者终局战 / Awakener final battle | 觉醒者之冠(独特头盔) / Awakener's Crown (Unique) |

### 异界天赋树任务 / Atlas Passive Tree

异界天赋树是终局的战略选择系统。击杀T14+地图Boss获得异界天赋点——总共可获得约120点。天赋影响地图掉落、Boss遭遇率和赛季内容出现率。**推荐开局**: 先点出地图掉落率→地图数量→高阶地图掉落率。10-15点后转向Boss遭遇率和赛季内容 / The Atlas Passive Tree is an endgame strategic system. Kill T14+ map bosses for Atlas points — ~120 total. Passives affect map drops, boss encounters, and league content rate. **Recommended start**: Map drop rate → Map quantity → Higher tier map drop. After 10-15 points, pivot to boss and league content.

## 赛季挑战 / Seasonal Challenges

### 挑战类型 / Challenge Types

| 挑战类型 / Type | 举例 / Example | 难度 / Difficulty |
|---|---|---|
| 击杀挑战 / Kill Challenges | 击杀1000个稀有怪物 / Kill 1000 rare monsters | 简单 / Easy |
| BOSS击杀挑战 / Boss Challenges | 击败所有地图Boss / Defeat all map bosses | 中等 / Medium |
| 异界挑战 / Atlas Challenges | 完成所有异界天赋目标 / Complete all Atlas objectives | 困难 / Hard |
| 赛季机制挑战 / League Mechanic | 完成赛季玩法25次 / Complete league mechanic 25 times | 中等 / Medium |
| 终局挑战 / Endgame Challenges | 击败觉醒者/终局Boss / Defeat Awakener/Endgame boss | 极难 / Very Hard |

完成36/40个赛季挑战获得赛季奖励套装——包括武器特效、角色特效和赛季称号。奖励会继承到标准模式。**挑战完成建议**: 先从简单的击杀和收集类开始——最后再攻坚最难的那几个 / Completing 36/40 seasonal challenges grants the seasonal reward set — including weapon effect, character effect, and seasonal title. Rewards carry to Standard. **Strategy**: Start with easy kill/collect challenges — tackle hard ones last.

## 任务奖励选择建议 / Quest Reward Choices

**技能宝石选择**: 优先选和你主Build匹配的宝石。不确定的话选防御向(如**护体**、**钢铁之肤**)——通用性最强。**装备奖励选择**: 优先选腰带和戒指——这两个部位的稀有底材最难获得。武器奖励过渡期有用但终局替代快 / **Skill Gem Choice**: Prioritize gems matching your build. If unsure, pick defensive (**Fortify**, **Steelskin**) — most versatile. **Gear Reward Choice**: Prioritize belts and rings — hardest slots to find good bases. Weapon rewards are useful for leveling but quickly replaced.

**珠宝奖励**: 珠宝奖励优先拿——珠宝槽在任何Build中都重要。注意选和你伤害类型匹配的珠宝词缀。**通货奖励**: 混沌石>神圣石>点金石>改造石——按这个优先级选通货奖励 / **Jewel Rewards**: Take jewel rewards first — jewel sockets are important in any build. Pick jewel affixes matching your damage type. **Currency Rewards**: Chaos Orb > Divine Orb > Alchemy Orb > Alteration Orb — follow this priority for currency choices.

## 成就任务 / Achievement Quests

**全区域探索成就**: 完成所有区域的探索目标——包括隐藏区域。解锁成就"庇护之地的探索者"——奖励藏身处装饰品"世界地图"。**全Boss击杀成就**: 击杀所有剧情Boss和终局Boss——奖励称号"Boss猎人"。**全支线成就**: 完成全部支线任务——奖励天赋重置书x5 / **Full Zone Exploration**: Complete all zone exploration objectives including hidden zones. Unlocks "Sanctuary Explorer" achievement — rewards hideout decoration "World Map". **All Boss Kills**: Kill all story and endgame bosses — rewards title "Boss Hunter". **All Side Quests**: Complete every side quest — rewards Orb of Regret x5.
