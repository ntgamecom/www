## 技能系统概述 / Skill System Overview

**中文：** 龙之信条2的技能系统分为职业技能（主动技能）、核心技能（被动/基础能力）和武器技能（特定武器动作）。每个职业有4个主动技能槽位和多个被动核心技能。技能通过职业等级解锁——职业等级越高，可用的强力技能越多。大师技能是每个职业的终极技能，需要完成特定任务才能获得。

**English:** Dragon's Dogma 2's skill system is divided into job skills (active abilities), core skills (passives/basics), and weapon skills (weapon-specific moves). Each job has 4 active skill slots and multiple passive core skills. Skills unlock through job rank — higher rank means more powerful skills. Master skills are ultimate abilities requiring specific quests to unlock.

| 技能类型 / Type | 说明 / Description |
|---|---|
| 职业技能 / Job Skills | 主动技能，占用技能槽 / Active skills, use skill slots |
| 核心技能 / Core Skills | 基础攻击和被动能力 / Basic attacks and passives |
| 大师技能 / Master Skills | 职业终极技能，需任务解锁 / Ultimate job skills, quest-locked |
| 武器技能 / Weapon Skills | 特定武器的特殊动作 / Special weapon-specific moves |

## 技能搭配建议 / Skill Builds

**中文：** 以下是为不同玩法推荐的技能搭配方案。

**English:** Recommended skill loadouts for different playstyles.

**坦克战士 / Tank Fighter**:
- 主动：盾牌反击、挑衅斩、盾牌猛击、圣壁 / Active: Shield Counter, Provoke Slash, Shield Bash, Holy Wall
- 核心：防御姿态、盾牌掌握 / Core: Defense Stance, Shield Mastery
- 玩法：接怪→挑衅→格挡→反击 / Engage → taunt → block → counter

**输出法师 / DPS Mage**:
- 主动：雷电风暴、火焰墙、冰冻术、陨石召唤 / Active: Thunderstorm, Fire Wall, Frigor, Meteor Call
- 核心：法术速咏、魔法精通 / Core: Fast Cast, Magic Mastery
- 玩法：大范围元素魔法覆盖，优先元素克制 / Wide AoE element coverage, prioritize elemental weaknesses

**暗杀盗贼 / Assassin Thief**:
- 主动：背刺、百裂斩、烟雾弹、伏击 / Active: Backstab, Hundred Kisses, Smoke Bomb, Ambush
- 核心：暴击提升、背后攻击强化 / Core: Crit Up, Back Attack Boost
- 玩法：烟雾弹隐身→绕后→伏击背刺秒杀 / Smoke → stealth → flank → backstab oneshot

**远程魔弓 / Magic Archer Ranged**:
- 主动：穿越魔弹、追踪火焰、猎龙箭 / Active: Seeking Shot, Flame Torrent, Dragon Hunt Arrow
- 核心：追踪强化、魔法箭精通 / Core: Tracking Boost, Magic Arrow Mastery
- 玩法：保持距离，追踪弹自动攻击弱点 / Keep distance, homing shots auto-target weak points
