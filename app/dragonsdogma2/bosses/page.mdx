## 狮鹫 / Griffin

**位置 / Location**: 维蒙德野外——常在平原和废墟区域 / Vermund wilds — plains and ruins
**等级建议 / Level**: 8-15

**Boss特性 / Boss Traits**:
- 飞行——大部分时间在空中 / Flies — mostly airborne
- 可被拉回地面——攻击腿部 / Can be grounded — attack legs
- 弱火 / Weak to fire

**打法 / Strategy**:
1. **弓箭手射翅膀**：射翅膀可以把它打下来 / Shoot wings to ground it
2. **火属性武器/魔法**：伊格尼/火球术造成额外伤害 / Fire weapons/magic deal bonus damage
3. **攀爬打头**：打下来后跳到背上攻击头部——弱点 / Climb and attack head when grounded
4. **法师的圣光球**：圣光球自动追踪飞行目标——好用 / Holy orb auto-tracks flying targets

**掉落 / Loot**: 狮鹫羽翼（制作材料）/ Griffin wing (crafting material)

---

## 独眼巨人 / Cyclops

**位置 / Location**: 全区域——各种桥梁和洞穴前 / All regions — bridges and cave entrances
**等级建议 / Level**: 10-30（视种类而定）/ Varies by type

**Boss特性 / Boss Traits**:
- 只有**眼睛**是弱点 / Only **eye** is weak point
- 会挥舞大棒范围攻击 / Club swing AoE
- 部分装备盔甲——需要破坏盔甲才能伤到 / Some wear armor — break armor first

**打法 / Strategy**:
1. **射眼睛**：弓箭手/巫师瞄准眼睛——造成巨大硬直 / Shoot the eye — massive stagger
2. **攀爬打眼**：盗贼/战士爬上去戳眼睛 / Climb onto it and stab the eye
3. **先拆盔甲**：盔甲独眼巨人先破坏头盔暴露眼睛 / Armored cyclops — destroy helmet first
4. **阿尔德打断**：它蓄力重击时可以打断 / Interrupt its charge attack

**掉落 / Loot**: 独眼巨人之眼（制作材料）/ Cyclops eye (crafting material), 大量金币 / Lots of gold

## 奇美拉 / Chimera

**位置 / Location**: 维蒙德北部森林 / Vermund northern forests
**等级建议 / Level**: 15-25

**Boss特性 / Boss Traits**: 狮子头+羊头+蛇尾——三部分各有不同弱点 / Lion head + goat head + snake tail — each segment has different weaknesses.

| 部位 / Part | 弱点 / Weakness | 攻击模式 / Attack Pattern |
|---|---|---|
| 狮子头 / Lion Head | 火 / Fire | 咬、扑 / Bite, pounce |
| 羊头 / Goat Head | 雷 / Lightning | 远程闪电 / Ranged lightning |
| 蛇尾 / Snake Tail | 冰 / Ice | 毒液喷射 / Poison spray |

**打法 / Strategy**:
1. **优先杀羊头**：羊头会远程闪电——最烦人 / Kill goat head first — ranged lightning is annoying
2. **分部位击破**：每个部位独立血量——逐个击破 / Each part has independent HP — destroy one at a time
3. **蛇尾可以提前打断**：蛇尾在没有头时会自己攻击——先打掉 / Snake tail attacks independently — remove it
4. **火属性武器全程有效** / Fire weapons are effective throughout

**掉落 / Loot**: 奇美拉部位材料（制作防具）/ Chimera parts (armor crafting)

## 龙 / Dragons (Drake, Wyrm, Wyvern)

游戏中有三种龙——但战斗逻辑类似 / Three dragon types with similar combat logic.

| 类型 / Type | 出现地点 / Location | 属性弱点 / Element Weakness | 特殊 / Special |
|---|---|---|---|
| 德拉寇 / Drake | 维蒙德野外 / Vermund wilds | 冰 / Ice | 吐息最频繁 / Most breath attacks |
| 法夫尼尔 / Wyrm | 巴塔尔沙漠 / Battahl desert | 雷 / Lightning | 召唤魔法 / Summon magic |
| 塔塔内尔 / Wyvern | 山顶、螺旋之塔 / Mountain tops | 圣 / Holy | 飞行时间最长 / Most airborne |

**等级建议 / Level**: 30-45

**核心打法 / Core Strategy**:
1. **攀爬打心脏**：龙胸口有发光的心脏——最高伤害点 / Climb to the glowing heart on its chest — highest damage spot
2. **打断吐息**：龙吐息时有明显前摇——看到后闪避或打断 / Dragon breath has clear wind-up — dodge or interrupt
3. **尾部可断**：攻击尾部可以断尾——获得特殊制作材料 / Tail can be severed — special crafting material
4. **属性克制必须到位**：带对属性的武器/魔法伤害翻倍 / Correct element = 2x damage
5. **耐力药必备**：打龙需要大量攀爬——带足蘑菇/耐力药 / Climbing burns stamina — bring plenty of mushrooms

**掉落 / Loot**: 龙血晶（WLC——装备升级核心材料）/ Wyrmlife Crystal (WLC — core upgrade material), 龙鳞 / Dragon scale, 龙心 / Dragon heart

## 泰坦 / Talos

**位置 / Location**: 螺旋之塔 / Spiral Tower
**等级建议 / Level**: 35-45

**Boss特性 / Boss Traits**: 巨型石像——全游戏最大的敌人 / Giant stone statue — largest enemy in the game.

**打法 / Strategy**:
1. **分阶段战斗**：泰坦身体各部位可逐个击破 / Multi-phase — destroy body parts one by one
2. **先打腿部**：破坏腿部让它跪下——露出身体弱点 / Destroy legs first to make it kneel — exposes body weak points
3. **攀爬打胸口核心**：核心是最大的弱点 / Climb to chest core — biggest weak point
4. **魔法攻击有效**：巫师陨石术伤害极高 / Sorcerer Meteoron deals massive damage
5. **注意掉落**：被摔下来会掉很多血——抓稳了 / Falling off deals heavy damage — hold tight

**掉落 / Loot**: 泰坦核心 / Talos Core（传说级制作材料 / Legendary crafting material）

## 无头骑士 / Dullahan

**位置 / Location**: 夜间野外随机刷新 / Random night spawn in wilds
**等级建议 / Level**: 25-35

**Boss特性 / Boss Traits**:
- 只在**夜晚**出现 / Only spawns at **night**
- 免疫物理伤害——必须用魔法/圣属性 / Immune to physical — must use magic/holy
- 移动速度极快 / Extremely fast

**打法 / Strategy**:
1. **圣光球**：法师的圣光球是最大克星 / Mage Holy Orb is its biggest counter
2. **魔弓手追踪箭**：追踪箭自动锁定——不怕它快 / Magick Archer homing shot auto-tracks
3. **如果全物理队伍——直接逃跑** / If full physical party — just run
4. **掉落独特装备**——值得一刷 / Drops unique gear — worth farming

**掉落 / Loot**: 无头骑士铠甲（传说级）/ Dullahan armor (legendary)

## 通用Boss战技巧 / General Boss Tips

1. **带多种属性武器**：至少携带2种不同属性的武器/魔法应对不同Boss / Carry 2+ element types for different bosses
2. **随从指令"Help"**：被Boss抓住时按Help让随从救你 / Press "Help" when grabbed — pawns will rescue you
3. **露营Buff**：战前露营吃料理——获得攻击/防御Buff / Camp and cook before boss fights for buffs
4. **弱点多段判定**：攻击弱点不仅伤害高——还能延长硬直时间 / Weak point attacks extend stagger duration
5. **复活道具**：带足"唤醒石"——可在战斗中复活随从 / Bring "Wakestones" — revive pawns mid-combat
6. **风筝战术**：大型Boss移动慢——远程职业可以风筝 / Large bosses are slow — ranged can kite
7. **跳劈开局**：战斗开始跳劈可以骑上大型Boss / Jump attack at fight start to mount large bosses
