## 任务系统概述 / Quest System Overview

**中文：** 龙之信条2的任务系统采用开放世界自由探索模式。主线任务推动剧情发展，支线任务提供经验、装备和职业技能。部分支线有严格的时间限制——错过特定时间点后任务会失败。游戏中有几个关键选择点会永久改变NPC的命运和可用任务。

**English:** Dragon's Dogma 2 uses an open-world free exploration quest system. Main quests advance the story. Side quests provide XP, gear, and job skills. Some side quests have strict time limits — miss a window and they fail permanently. Several key choice points permanently change NPC fates and available quests.

**任务类型 / Quest Types**:

| 类型 / Type | 说明 / Description |
|---|---|
| 主线 / Main Quests | 推进剧情，解锁新区域 / Advance story, unlock regions |
| 支线 / Side Quests | 角色故事、装备奖励、技能解锁 / Character stories, gear, skills |
| 职业任务 / Job Quests | 解锁各职业大师技能 / Unlock master skills |
| 护送任务 / Escort Quests | 护送NPC到指定地点 / Escort NPCs to destinations |
| 告示板任务 / Notice Board | 重复性杀怪/收集任务 / Repeatable kill/gather quests |

## 主线流程 / Main Quest Flow

**中文：** 主线分为两个主要章节。第一章围绕维蒙德王国，第二章前往巴达尔地区。

**English:** The main story has two acts. Act 1 revolves around Vermond Kingdom, Act 2 moves to Balder region.

**第一章 / Act 1 — 维蒙德王国**:

1. **觉醒者诞生 / Birth of the Arisen** — 从海德拉城出发，在要塞遇到乌尔里克 / Depart from Hydra City, meet Ulrika at the fort
2. **王城之路 / Road to the Capital** — 到达维蒙德王城，谒见国王 / Arrive at Vermond capital, meet the King
3. **蛇之道路 / Serpent's Path** — 探索地下墓穴寻找蛇形浮雕 / Explore catacombs for serpent reliefs
4. **伪王 / False King** — 揭穿冒充国王的阴谋 / Expose the false king conspiracy
5. **雾之森 / Misty Forest** — 穿越迷雾森林追踪线索 / Cross Misty Forest tracking clues

**关键选择 / Key Choice**:
- 在"伪王"任务中选择：揭穿假国王或和他合作 / In "False King": expose the imposter or cooperate with him

**第二章 / Act 2 — 巴达尔**:

1. **沙漠之门的另一边 / Beyond the Desert Gate** — 穿越巴达尔沙漠 / Cross the Balder desert
2. **火山之心 / Heart of the Volcano** — 探索火山区，寻找龙之泪 / Explore volcanic zone, find Dragon's Tear
3. **暗影编织者 / Shadow Weaver** — 对抗暗影教派 / Fight the Shadow Cult
4. **命运之龙 / Dragon of Fate** — 最终决战，面对宿命之龙 / Final battle against the Dragon of Fate

**注意 / Note**: 在第二章开始前，务必完成所有维蒙德地区的未完结支线，因为进入巴达尔后部分任务会变为不可用 / Complete all Vermond side quests before Act 2 — some become unavailable after crossing the desert.

## 大师技能任务详情 / Master Skill Quest Details

**中文：** 全职业大师技能的任务触发和完成方法。

**English:** Quest triggers and completion methods for all master skills.

**战士大师技能 / Fighter Master Skill — 圣战之盾**:
- 触发：在维蒙德王城与战士大师布罗迪尔对话 / Talk to Fighter Master Brodir in Vermond capital
- 任务：击败竞技场中的三个挑战者 / Defeat three challengers in the arena
- 要求：战士职业等级7+ / Fighter job rank 7+

**弓箭手大师技能 / Archer Master Skill — 猎龙箭**:
- 触发：在精灵之森找到弓箭手大师艾琳 / Find Archer Master Eilin in Elven Forest
- 任务：用弓箭射落5只飞行怪物 / Shoot down 5 flying monsters with bow
- 要求：弓箭手职业等级7+ / Archer job rank 7+

**法师大师技能 / Mage Master Skill — 陨石召唤**:
- 触发：在巴达尔巫祭神殿找到法师大师索尔 / Find Mage Master Sol in Balder's Temple of Sorcery
- 任务：收集3个陨石碎片（在火山区域）/ Collect 3 meteor fragments (volcanic zone)
- 要求：法师职业等级7+ / Mage job rank 7+

**斗士大师技能 / Warrior Master Skill — 崩山斩**:
- 触发：在挖掘现场找到斗士大师冈纳 / Find Warrior Master Gunnar at excavation site
- 任务：用大剑击败一只巨型独眼巨人 / Defeat a giant cyclops with greatsword
- 要求：斗士职业等级7+ / Warrior job rank 7+

**其他大师技能 / Other Master Skills**: 类似条件，需要在各自职业达到7级后，找到对应的职业大师触发任务 / Similar conditions: reach job rank 7+, find the corresponding master, complete their challenge.

## 护送任务指南 / Escort Quest Guide

**中文：** 护送任务需要保护NPC穿越危险区域到达目的地。NPC的生命值很低，被攻击几次就会死亡导致任务失败。

**English:** Escort quests require protecting NPCs through dangerous areas. NPCs have very low HP and die after a few hits.

**护送技巧 / Escort Tips**:
- 在接护送任务前先清空路径上的敌人 / Clear the path of enemies before accepting the escort
- 使用战士的挑衅技能拉住敌人仇恨 / Use Warrior's provoke skill to hold aggro
- 法师的护盾和治疗可以保护NPC / Mage shields and heals protect the NPC
- 携带足够的回复道具 / Bring plenty of healing items
- 如果NPC卡住了，尝试推动或快速旅行后重试 / If NPC gets stuck, try pushing or fast travel and retry

| 任务 / Quest | NPC | 目的地 / Destination | 奖励 / Reward |
|---|---|---|---|
| 学者的请求 / Scholar's Request | 学者 / Scholar | 维蒙德图书馆 / Vermond Library | 技能书 / Skill Book |
| 商人的旅途 / Merchant's Journey | 商人 / Merchant | 巴达尔集市 / Balder Market | 金币+稀有材料 / Gold + rare materials |
| 朝圣之路 / Pilgrim's Path | 朝圣者 / Pilgrim | 火山岛神殿 / Volcanic Temple | 传说护符 / Legendary Charm |
