# 职业与Build / Vocations & Builds

## 总览 / Overview

龙之信条2有 **10种职业** 分三个层级：4基础 + 2高级 + 4混合 / 10 vocations: 4 basic + 2 advanced + 4 hybrid.

---

## ⚔️ 战士 / Fighter

**武器 / Weapon**: 剑 + 盾 / Sword + Shield
**定位 / Role**: 坦克 / 物理近战 / Tank / Physical melee

**核心技能 / Core Skills**:

| 技能 / Skill | 效果 / Effect | 推荐 / Recommendation |
|---|---|---|
| 盾牌反击 / Shield Bash | 格挡后反击——破防 + 眩晕 / Block then counter — stagger + stun | 必带 / Essential |
| 剑气斩 / Sword Beam | 远程剑气——弥补手短 / Ranged energy slash | 推荐 / Recommended |
| 圆月斩 / Full Moon Slash | 360度范围攻击 / 360-degree AoE slash | 对群 / Anti-group |
| 防御姿态 / Defensive Stance | 减伤+仇恨 / Damage reduction + aggro | 坦克必带 / Tank essential |

**被动专长 / Augments**:
- **坚韧 / Fortitude**: +物理防御 / +Physical defense
- **盾卫 / Shield Defense**: 格挡消耗耐力-30% / Block stamina cost -30%

**评价 / Tier**: A — 新手友好、坦克定位明确、生存能力强 / Beginner-friendly, clear tank role, great survivability

## 🌟 斗士 / Warfarer

**武器 / Weapon**: 全武器 / All weapons
**定位 / Role**: 全能 / 武器大师 / Versatile / Weapon master

**解锁条件 / Unlock**: 两个混合职业解锁后 / After unlocking two hybrid vocations

**核心特色 / Core Feature**: 可以装备**所有类型的武器**——在不同武器之间切换战斗 / Can equip **all weapon types** — switch between them in combat.

| 推荐组合 / Recommended Loadout | 说明 / Description |
|---|---|
| 魔弓 + 剑盾 + 法杖 | 远程+防御+治疗的全面组合 / Ranged + defense + healing |
| 大剑 + 双匕首 + 魔弓 | 近距离爆发+攀爬+远程 / Burst + climbing + ranged |
| 法杖 + 大法杖 + 香炉 | 纯法系+控场 / Pure magick + CC |

**被动专长 / Augments**: 可装备最多6个被动——从所有已学职业中选 / Can equip up to 6 augments from any learned vocation.

**评价 / Tier**: S — 后期成型最强，但需要大量职业等级积累 / Best endgame vocation but requires extensive leveling
