## 隐藏区域 / Secret Areas

**中文：** 以下是不在地图上显示的隐藏区域，通常需要通过特殊方式进入。

**English:** These secret areas are not shown on the map and usually require special entry methods.

**维蒙德王国隐藏区域 / Vermond Kingdom Secrets**:

1. **王城地下密室 / Capital Underground Chamber**: 王城城堡的水井下方有一条密道通往隐藏武器库 / Well in the castle courtyard leads to a hidden armory
2. **海德拉沉船 / Hydra Shipwreck**: 海德拉城东海岸的海底有一艘沉船，内含传说级戒指 / Sunken ship off Hydra's east coast, contains a legendary ring
3. **精灵之森瀑布洞穴 / Elven Forest Waterfall Cave**: 精灵之森最大的瀑布后面有一个隐藏洞穴 / Behind the largest waterfall in Elven Forest

**巴达尔隐藏区域 / Balder Region Secrets**:

1. **沙漠地下神殿 / Desert Underground Temple**: 巴达尔沙漠地图中央的下方有一个埋在沙下的古代神殿入口 / Buried ancient temple entrance in central Balder desert
2. **龙之墓地 / Dragon Graveyard**: 火山西侧的隐藏峡谷，有大量龙骨和稀有矿石 / Hidden canyon west of volcano, contains dragon bones and rare ores
3. **雾之湖 / Lake of Mist**: 地图东南角常年被浓雾笼罩的湖泊，中心岛有传说装备 / Southeast corner lake, perma-fog, center island has legendary gear

## 隐藏职业解锁 / Hidden Job Unlocks

**中文：** 游戏中除了基础职业外，还有几个高级职业需要特定条件才能解锁。

**English:** Beyond the basic jobs, several advanced jobs require specific conditions to unlock.

**魔弓手 / Magic Archer**:

- 解锁条件：完成精灵之森的"精灵的请求"任务 / Complete "The Elven Request" quest in Elven Forest
- 职业描述：使用魔法箭矢，攻击会自动追踪敌人 / Uses magic arrows that auto-track enemies
- 推荐理由：远程输出天花板，追踪弹无视敌人闪避 / Best ranged DPS, homing shots ignore enemy dodges

**幻术师 / Trickster**:

- 解锁条件：在雾之森找到隐藏的幻术师营地并完成试炼 / Find hidden Trickster camp in Misty Forest and complete trial
- 职业描述：制造分身迷惑敌人，使用烟雾和幻术 / Creates decoys, uses smoke and illusions
- 推荐理由：唯一可以制造分身的职业，控场能力极强 / Only job with decoy creation, best crowd control

**魔剑士 / Mystic Spearhead**:

- 解锁条件：在火山岛的古代遗迹中找到魔剑士石碑 / Find the Mystic Spearhead tablet in Volcanic Island ruins
- 职业描述：使用魔法长枪，可以瞬移和元素附魔 / Uses magic spear, teleport and element enchantment
- 推荐理由：机动性最高，输出和生存兼备 / Highest mobility, balanced offense and defense

## 隐藏Boss攻略 / Secret Boss Strategies

**中文：** 以下隐藏Boss需要特定条件才能遇到。

**English:** These secret bosses require specific conditions to encounter.

**远古之龙 / Ancient Dragon**:

- 触发条件：在龙之墓地使用龙之号角 / Use Dragon Horn at Dragon Graveyard
- 等级建议：40+ / Recommended level: 40+
- 弱点：胸口发光的心脏 / Weak point: glowing chest heart
- 掉落：龙之泪（传说强化材料）/ Drops: Dragon's Tear (legendary upgrade material)
- 打法：登龙到胸口攻击心脏，注意龙息的扇形范围 / Climb to chest, attack heart, watch for breath cone

**暗影领主 / Shadow Lord**:

- 触发条件：完成暗影教派全部支线后回到暗影神殿 / Return to Shadow Temple after completing all Shadow Cult quests
- 等级建议：35+ / Recommended level: 35+
- 弱点：背后的魔法核心 / Weak point: magic core on its back
- 掉落：暗影斗篷（唯一防具）/ Drops: Shadow Cloak (unique armor)
- 打法：需要高机动性职业绕后攻击 / High mobility job needed to flank

**九头蛇王 / Hydra King**:

- 触发条件：在海德拉城东海岸的夜晚出现 / Appears at night on Hydra City's east coast
- 等级建议：30+ / Recommended level: 30+
- 弱点：每个头的脖子，所有头必须全部砍断 / Weak point: each neck, all heads must be severed
- 掉落：九头蛇王冠 / Drops: Hydra King Crown
- 打法：逐个砍头，注意毒雾和头撞攻击 / Sever heads one by one, watch for poison mist and headbutt
