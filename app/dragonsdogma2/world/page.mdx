## 传送系统 / Fast Travel System

龙之信条2的传送系统独特——需要手动放置传送基石并使用传送石 / DD2 has a unique fast travel system — place portcrystals manually and use ferrystones.

### 传送基石 / Portcrystals

游戏中有 **10枚固定传送基石**（放置后不可移动但永久存在）+ 部分可重复拾取的 / 10 fixed portcrystals + some reusable ones.

**推荐放置位置 / Recommended Placements**:
1. 梅尔维——开局村庄 / Melve — starting village
2. 古战场遗迹入口 / Ancestral Battlegrounds entrance
3. 螺旋之塔前 / Spiral Tower entrance
4. 水神殿入口 / Water Temple entrance
5. 边境检查站 / Border checkpoint
6. 巴特尔——巴塔尔首都 / Bakbattahl — Battahl capital
7. 火山营地 / Volcanic Camp
8. 圣殿入口 / Sacred Temple entrance
9. 龙之雷前 / Dragon's Thunder entrance
10. 最终Boss区域前 / Before final boss zone

### 传送石 / Ferrystones

| 类型 / Type | 数量 / Quantity | 获取 / Source |
|---|---|---|
| 普通传送石 / Ferrystone | 无限（购买）/ Infinite (buy) | 旅店老板/商人——约2000G/个 / Innkeepers/merchants — ~2000G each |

## 探索技巧 / Exploration Tips

1. **高处跳下不摔死**：从高处跳下时按攻击键可以安全落地（翻滚着陆）/ Press attack while falling to safe-land (combat roll)
2. **动物追踪**：跟随野鹿可以找到水源和水边采集点 / Follow deer to find water sources and gathering spots
3. **攀爬捷径**：很多峭壁可以攀爬上去——看起来是死路其实有路 / Many cliffs can be climbed — dead ends may be paths
4. **夜间篝火**：在篝火露营等待到天亮——避免夜战 / Camp at bonfires to wait until dawn — avoid night combat
5. **标记地图**：用标记功能标注未探索的洞穴入口 / Use map markers for unexplored cave entrances
6. **水上行走**：特定技能和装备可以在水上行走（魔弓手职业）/ Specific skills enable water walking (Magick Archer)
