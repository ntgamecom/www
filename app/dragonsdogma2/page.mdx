# 龙之信条2攻略 / Dragon's Dogma 2 Guide

《龙之信条2》是Capcom开发的开放世界动作RPG，延续了前作的精髓——宏大的奇幻世界、壮观的Boss战、创新的随从系统。你将扮演"觉醒者"，在两大王国维蒙德和巴塔尔的广阔世界中冒险，挑战传说巨龙。本攻略涵盖全职业解析、世界探索、Boss战策略和装备收集。

Dragon's Dogma 2 is an open-world action RPG by Capcom, continuing the legacy of the original — a vast fantasy world, epic boss battles, and the innovative Pawn system. Play as the Arisen, adventuring across the two kingdoms of Vermund and Battahl to challenge the legendary Dragon. This guide covers all vocations, world exploration, boss strategies, and gear collection.

## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|---|---|
| **觉醒者 / Arisen** | 被龙夺走心脏的不死者——无法真正死亡 / Dragon heart-taken immortal — cannot truly die |
| **随从 / Pawns** | AI控制的队友——可自定义职业、技能、性格、外观 / AI companions — customizable vocation, skills, inclination, appearance |
| **职业系统 / Vocations** | 可在旅店随时切换——基础/高级/混合职业，技能独立升级 / Switch anytime at inn — basic/advanced/hybrid, skills level independently |
| **攀爬战斗 / Climbing Combat** | 跳上大型魔物身体攻击弱点——核心战斗机制 / Jump on large monsters to attack weak points — core mechanic |
| **昼夜循环 / Day-Night Cycle** | 夜晚野外会刷出强力亡灵——危险但奖励丰厚 / Night spawns dangerous undead — risky but rewarding |
| **传送基石 / Portcrystals** | 可放置的传送点——探索核心机制 / Placeable fast travel points — core exploration mechanic |
| **Ferrystone / 传送石** | 消耗品——传送到任意放置的传送基石或主要城市 / Consumable — teleport to any placed portcrystal or major city |

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **多切换职业 / Try All Vocations** | 职业等级独立——每个职业练到一定等级解锁被动技能和专长，可跨职业装备 / Vocation rank is per-class — level each to unlock augments and core skills usable across classes |
| **2** | **随从搭配要合理 / Balance Your Party** | 至少带1个法师（治疗）+ 1个近战（坦克）。三个输出没有治疗会很难受 / Bring at least 1 Mage (healer) + 1 melee tank. Triple DPS is painful |
| **3** | **探索比推主线重要 / Explore Over Main Quest** | 主线流程很短（约15-20小时），大量内容在野外探索——洞穴、塔楼、隐藏Boss / Main quest is short (~15-20h) — most content is exploration |
| **4** | **放置传送基石 / Place Portcrystals** | 前期获得所有传送基石后，放在Boss区域和探索关键点 / Place all portcrystals at boss zones and key exploration points |
| **5** | **夜间慎行 / Be Careful at Night** | 夜晚野外亡灵成群——新手建议在篝火等到天亮 / Night spawns undead hordes — camp at bonfires until dawn early game |
| **6** | **多用露营 / Camp Often** | 露营可以存档、料理Buff、升级职业经验加成 / Camping saves, cooking buffs, vocation XP bonus |
| **7** | **提升好感度 / Raise Affinity** | 和NPC对话送礼物提升好感——影响任务结局和特殊奖励 / Gift NPCs to raise affinity — affects quest endings and rewards |
| **8** | **随从性格很重要 / Pawn Inclinations Matter** | 主随从的性格决定AI行为——主随从性格会受你的行为影响 / Pawn inclination dictates AI behavior — your actions shape it |
| **9** | **体力管理 / Manage Stamina** | 跑步、攀爬、重击都耗体力——携带大量蘑菇和耐力药 / Running, climbing, heavy attacks consume stamina — carry plenty of mushrooms |
| **10** | **看告示板任务 / Check Notice Boards** | 告示板有大量赏金和探索任务——经验+金钱来源 / Notice boards have bounty and exploration quests — XP + gold source |
