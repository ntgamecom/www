## 战斗系统概述 / Combat Overview

**中文：** 龙之信条2的战斗系统是游戏最核心的亮点。战斗围绕"登龙"（爬怪物身体攻击弱点）、职业切换、随从配合和物理破坏四大机制展开。大型怪物有不同的身体部位可破坏，攻击正确的部位可以造成巨量伤害。战斗的重点不在于等级碾压，而在于对怪物行为和弱点的理解。

**English:** Dragon's Dogma 2's combat is the game's standout feature. It revolves around four core mechanics: climbing monsters to strike weak points, job switching, pawn cooperation, and physical destruction. Large monsters have destructible body parts — hitting the right spot deals massive damage. Combat is about understanding monster behavior and weaknesses rather than level grinding.

## 登龙系统 / Climbing System

**中文：** 登龙是龙之信条最具特色的战斗机制。面对大型怪物时，玩家可以抓住怪物的身体进行攀爬，找到弱点部位进行攻击。登龙消耗耐力——耐力耗尽时会被甩下来。部分职业（如斗士）的登龙攻击有额外伤害加成。

**English:** Climbing is Dragon's Dogma's signature mechanic. Against large monsters, players can grab and climb the monster's body to reach weak points. Climbing consumes stamina — the monster will shake you off when stamina runs out. Certain jobs (like Warrior) have bonus climbing attack damage.

**登龙技巧 / Climbing Tips**:

1. 优先攀爬到怪物背部或头部——这些通常是弱点部位 / Prioritize climbing to the back or head — these are often weak points
2. 在怪物攻击间隙登龙，否则会被甩掉 / Climb during attack windows or you'll be shaken off
3. 耐力管理至关重要——留下一半耐力用于格挡或逃生 / Stamina management is critical — keep half for blocking or escape
4. 斗士的登龙重击可以对弱点造成巨额伤害 / Warrior's climbing heavy attack deals massive weak point damage
5. 幻术师可以制造分身吸引仇恨，让本体安心登龙 / Trickster can create decoys to draw aggro while you climb

**怪物甩人机制 / Shake-off Mechanics**:

| 怪物动作 / Monster Action | 应对 / Counter |
|---|---|
| 甩动身体（Sway）/ Body Shake | 快速移动到另一侧 / Move to opposite side quickly |
| 翻滚 / Roll | 立刻跳下，否则被压 / Jump off immediately to avoid crush |
| 撞墙 / Wall Smash | 在撞击前松手跳下 / Release and jump off before impact |
| 飞行冲刺 / Flying Charge | 抓稳龙鳞 / Hold onto dragon scales tightly |

## 弱点系统 / Weak Point System

**中文：** 每个怪物都有至少一个弱点部位，攻击弱点可以造成3-5倍的伤害。弱点通常包括眼睛、心脏、翅膀关节和腹部。部分弱点需要用特定武器或技能才能有效攻击到。

**English:** Every monster has at least one weak point. Hitting weak points deals 3-5x damage. Common weak points include eyes, hearts, wing joints, and underbellies. Some weak points require specific weapons or skills to reach effectively.

**常见怪物弱点 / Common Monster Weak Points**:

| 怪物 / Monster | 弱点 / Weak Point | 推荐攻击方式 / Best Attack Method |
|---|---|---|
| 狮鹫 / Griffin | 翅膀关节、头部 / Wing joints, head | 登龙攻击翅膀 / Climb and hit wings |
| 独眼巨人 / Cyclops | 眼睛、膝盖后侧 / Eye, back of knee | 弓箭射眼 / Bow to the eye |
| 奇美拉 / Chimera | 羊头（魔法）、蛇尾 / Goat head (magic), snake tail | 先杀羊头再打蛇尾 / Kill goat head first then tail |
| 龙 / Dragon | 心脏（胸口发光处）/ Heart (glowing chest) | 登龙到胸口攻击心脏 / Climb to chest, attack heart |
| 九头蛇 / Hydra | 每个头的脖子 / Each head's neck | 逐个砍断所有头 / Sever all heads one by one |
| 戈伦 / Golem | 背部魔法核心 / Back magic core | 绕后登龙打核心 / Climb around back to hit core |

## 部位破坏系统 / Part Break System

**中文：** 部分破坏是龙之信条2的重要机制。破坏怪物特定部位可以获得额外掉落物品，同时也会改变怪物的行为模式。例如，破坏狮鹫的翅膀可以阻止它飞行，破坏独眼巨人的膝盖可以让它跪地。

**English:** Part breaking is a key mechanic. Destroying specific monster parts yields bonus loot and alters monster behavior. For example, breaking a griffin's wing grounds it, and breaking a cyclops' knee forces it to kneel.

**部位破坏收益 / Part Break Rewards**:

| 怪物 / Monster | 可破坏部位 / Breakable Parts | 掉落 / Loot |
|---|---|---|
| 狮鹫 / Griffin | 翅膀×2、喙 / Wings x2, Beak | 狮鹫羽毛、爪子 / Griffin Feather, Talon |
| 独眼巨人 / Cyclops | 膝盖护甲×2、眼罩 / Knee armor x2, Eyepatch | 独眼巨人角 / Cyclops Horn |
| 龙 / Dragon | 心脏、翅膀×2、角×2 / Heart, Wings x2, Horns x2 | 龙鳞、龙骨、龙血晶 / Dragon Scale, Bone, Blood Crystal |
| 九头蛇 / Hydra | 每个头的脖子×8+ / Each head's neck x8+ | 九头蛇牙 / Hydra Fang |

## 职业战斗风格 / Job Combat Styles

**中文：** 每个职业都有独特的战斗机制。了解各职业的优劣势有助于组建平衡的队伍。

**English:** Each job has a unique combat mechanic. Understanding each job's strengths and weaknesses helps build a balanced party.

**战士 / Fighter**: 高防御、盾牌格挡、仇恨控制 / High defense, shield block, aggro control。核心技能：盾牌反击、挑衅斩 / Core skills: Shield Counter, Provoke Slash

**弓箭手 / Archer**: 远程输出、弱点攻击、支援 / Ranged DPS, weak point shots, support。核心技能：致命射击、锁定箭雨 / Core skills: Deathly Shot, Lock-on Barrage

**法师 / Mage**: 元素魔法、治疗、增益 / Elemental magic, healing, buffs。核心技能：治疗之环、雷电风暴 / Core skills: Healing Circle, Thunderstorm

**斗士 / Warrior**: 高伤害大剑、登龙攻击增强 / High damage greatsword, climbing attack boost。核心技能：蓄力斩、登龙重击 / Core skills: Charge Slash, Climbing Heavy

**盗贼 / Thief**: 高速连击、背刺、陷阱 / Fast combos, backstab, traps。核心技能：背刺、百裂斩 / Core skills: Backstab, Hundred Kisses

**魔弓手 / Magic Archer**: 魔法+弓箭混合、追踪弹 / Magic + bow hybrid, homing shots。核心技能：穿越魔弹、追踪火焰 / Core skills: Seeking Shot, Flame Torrent

**幻术师 / Trickster**: 制造分身、烟雾弹、迷惑 / Decoy creation, smoke bombs, confusion。核心技能：分身、烟雾帷幕 / Core skills: Simulacrum, Smoke Screen

**魔剑士 / Mystic Spearhead**: 魔法剑攻击、瞬移、元素附魔 / Magic blade attacks, teleport, element enchant。核心技能：瞬移斩、元素附魔 / Core skills: Teleport Slash, Element Enchant

## 随从战斗配合 / Pawn Combat Synergy

**中文：** 随从AI在龙之信条2中大幅改进。他们会根据自身职业和特性和平战斗。正确的队伍配置可以大幅降低战斗难度。

**English:** Pawn AI has been greatly improved in Dragon's Dogma 2. They fight according to their job and inclinations. The right party setup dramatically reduces combat difficulty.

**理想队伍配置 / Ideal Party Composition**:

1. **主角（主随从）/ Arisen (Main Pawn)**: 根据喜好选择职业，推荐战士或斗士 / Choose as preferred, recommended Fighter or Warrior
2. **坦克随从 / Tank Pawn**: 战士，带盾牌挑衅技能 / Fighter with shield provoke skills
3. **治疗随从 / Healer Pawn**: 法师，带治疗之环和清洁术 / Mage with Healing Circle and Cleanse
4. **远程随从 / Ranged Pawn**: 弓箭手或魔弓手 / Archer or Magic Archer

**随从指令 / Pawn Commands**:
- "过来"：随从聚集到玩家身边保护 / "Come" — pawns gather to protect the player
- "前进"：随从主动攻击 / "Go" — pawns attack aggressively
- "帮助"：随从优先使用治疗技能 / "Help" — pawns prioritize healing skills
- "撤退"：随从停止攻击回避 / "Back" — pawns stop attacking and evade

## 大型怪物战斗范例 / Boss Fight Examples

**中文：** 以下是几个典型大型怪物的战斗策略。

**English:** Combat strategies for several typical large monsters.

**狮鹫 / Griffin**:
1. 优先破坏翅膀使其无法起飞 / Prioritize breaking wings to ground it
2. 登龙到头部攻击眼睛 / Climb to head, attack eyes
3. 狮鹫飞行时用弓箭射翼膜迫降 / Use bow on wing membrane when flying
4. 注意狮鹫的抓取攻击——被抓到会被带到高空扔下 / Watch for grab attack — it will fly high and drop you

**龙 / Dragon**:
1. 登龙到胸口攻击发光的心脏 / Climb to chest, attack glowing heart
2. 龙翼可破坏以限制其机动性 / Dragon wings can be broken to limit mobility
3. 注意龙的吐息攻击——扇形范围广伤害高 / Watch for breath attack — wide cone, high damage
4. 龙尾扫击范围大，不要站在后方 / Tail sweep has wide range, don't stand behind

**九头蛇 / Hydra**:
1. 逐个砍断每个头的脖子——不砍完所有头不会死 / Sever each head's neck — must cut all to kill
2. 被砍断的脖子会喷毒雾 / Severed necks spray poison mist
3. 优先处理靠近玩家的头 / Prioritize heads closest to the player
4. 准备好解毒药应对毒雾 / Stock antidotes for poison mist
