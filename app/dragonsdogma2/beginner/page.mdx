# 新手入门 / Beginner Guide

## 觉醒者 / The Arisen

你是被龙夺走心脏的"觉醒者"——拥有不死之身，死亡后会在最近的旅店/住所复活，但会损失部分经验值（DC）/ You are the Arisen, whose heart was taken by the Dragon — immortal, resurrect at the last inn on death, losing some XP (Death Currency, DC).

| 机制 / Mechanic | 说明 / Description |
|---|---|
| 死亡惩罚 / Death Penalty | 损失当前升级所需经验的一部分——DC会累积在死亡地点 / Lose a portion of XP toward next level — DC (Death Currency) accumulates at death site |
| 回收DC / Recover DC | 回到死亡地点触摸光柱可回收损失的经验 / Return to death spot and touch the light to recover lost XP |
| 永久死亡 / Permadeath? | 随从才会真死——觉醒者无限复活 / Only pawns can permanently die — Arisen resurrects infinitely |

## 职业系统 / Vocation System

职业分 **3 个层级**——在旅店随时切换，切换后会自动重新分配属性点 / 3 tiers — switch anytime at inn, stats auto-reallocate.

| 层级 / Tier | 说明 / Description | 列表 / List |
|---|---|---|
| 基础 / Basic | 初始可选——武器单一 / Available from start — single weapon | 战士 / 弓箭手 / 法师 / 盗贼 |
| 高级 / Advanced | 基础职业练到一定等级后解锁 / Unlock by leveling basic vocations | 大剑战士 / 巫师 |
| 混合 / Hybrid | 两种基础职业组合解锁 / Unlock by combining two basic vocations | 魔弓手 / 魔剑士 / 幻术师 / 斗士 |

### 职业等级 / Vocation Rank

每个职业独立计算等级（1-9级），战斗获得职业经验提升 / Each vocation ranks independently (1-9), gains XP from combat.

| 职业等级 / Rank | 解锁 / Unlocks |
|---|---|
| 1-3 | 基础技能——武器核心技能 / Core weapon skills |
| 4-6 | 额外技能槽——武器技能 / Additional skill slots — weapon arts |
| 7-9 | 高级技能 + 被动专长（可跨职业装备）/ Advanced skills + augments (equippable across vocations) |

**建议**：每个职业至少练到 **职业等级4** 解锁所有武器技能槽，再决定是否深入 / Level each vocation to at least rank 4 for all skill slots

## 随从系统 / Pawn System

龙之信条的核心特色——你有一个专属主随从 + 可雇佣2个其他玩家的随从 / Core feature — 1 main pawn + 2 hired pawns from other players.

### 主随从 / Main Pawn

- **完全自定义**：外貌、职业、技能、性格 / Full customization: appearance, vocation, skills, inclinations
- **永久存在**：不会真正死亡——可复活 / Permanent — cannot truly die, can revive
- **可被召唤**：其他玩家可以在裂痕石雇佣你的主随从——获得RC奖励 / Other players can hire your pawn — earn RC rewards
- **性格受你影响**：你的战斗方式会影响主随从的性格倾向 / Your playstyle influences pawn inclinations

### 雇佣随从 / Hired Pawns

| 来源 / Source | 等级 / Level | 费用 / Cost |
|---|---|---|
| 裂痕石 / Riftstone | 与你相近 / Near your level | RC点数 / RC points |
| 其他玩家 / Other Players | 任意等级 / Any level | RC（高等级更贵）/ RC (higher level = more expensive) |
| 路上遇到的流浪随从 / Wandering Pawns | 与你相近 / Near your level | 免费——可直接邀请 / Free — directly invite |

### 队伍组成建议 / Party Composition Tips

| 阵容 / Setup | 打法 / Playstyle |
|---|---|
| 战士+法师+弓箭手+法师 | 均衡——适合新手 / Balanced — best for beginners |
| 盗贼+巫师+巫师+法师 | 高输出——法师负责Buff+治疗 / High DPS — Mage heals + buffs |
| 大剑战士+斗士+巫师+法师 | Boss特化——大剑打弱点 / Boss specialist |
| 全法师/巫师 / All Mages | 智力碾压——但脆皮 / Intellect check — but squishy |

## 核心机制 / Core Mechanics

### 攀爬战斗 / Climbing Combat

跳上大型魔物身体——攻击弱点（发光部位）/ Jump onto large monsters — attack glowing weak points.

| 操作 / Action | 按键 / Button | 说明 / Description |
|---|---|---|
| 跳跃攀爬 / Jump Climb | 跳向魔物 / Jump toward monster | 自动抓住 / Auto-grab |
| 移动 / Move | 摇杆 / Stick | 在怪物身上攀爬移动 / Move across monster body |
| 轻攻击 / Light Attack | 攻击键 | 对弱点造成伤害 / Damage weak points |
| 重攻击 / Heavy Attack | 蓄力攻击 | 高伤害但消耗耐力 / High damage, costs stamina |
| 特殊动作 / Special | 特定技能 | 某些技能有攀爬特化效果 / Some skills have climbing-specific effects |

**攀爬提示 / Climbing Tips**：
- 注意耐力槽——耐力耗尽会被甩下来 / Watch stamina — exhausted = thrown off
- 部分魔物会甩身/翻滚——及时跳下躲避 / Some monsters shake/roll — jump off to avoid
- 盗贼的攀爬速度最快 / Thief climbs fastest
- 大剑战士可以直接把魔物拽下来 / Warrior can pull monsters down

### 露营系统 / Camping

| 功能 / Function | 说明 / Description |
|---|---|
| 存档 / Save | 露营时自动存档 / Auto-save on camping |
| 料理 / Cooking | 用材料做料理——提供战斗Buff（持续到下次露营）/ Cook food for combat buffs (until next camp) |
| 休息 / Rest | 恢复HP/耐力——消除负面状态 / Restore HP/stamina — cure negative statuses |
| 篝火 / Campfire Kit | 需要携带篝火工具包才能露营 / Need a camping kit to set up camp |

### 昼夜循环 / Day-Night Cycle

| 时段 / Time | 危险等级 / Danger | 说明 / Description |
|---|---|---|
| 白天 / Day | ★ | 标准探索——大部分魔物较弱 / Standard exploration — most monsters are weaker |
| 黄昏 / Dusk | ★★ | 亡灵开始出现 / Undead start appearing |
| 夜晚 / Night | ★★★★ | 亡灵成群——新魔物刷新（夜行种）/ Undead hordes — new spawns (nocturnal variants) |
| 黎明 / Dawn | ★★ | 亡灵撤退 / Undead retreat |

## 实用技巧 / Practical Tips

1. **采集一切 / Gather Everything**：花草、矿石、水果——炼金和料理都需要 / Herbs, ores, fruits — needed for alchemy and cooking
2. **多存档 / Save Often**：在旅店休息时会存档——野外露营也能存档 / Inn rest = save; camping = save
3. **升级装备优先 / Upgrade Gear First**：在铁匠铺升级装备比买新的划算——龙血晶是升级材料 / Upgrading gear is better than buying new — WLC is upgrade material
4. **随从会背叛 / Pawns Can Betray**：某些任务中随从可能背叛——注意任务描述 / Some quests may cause pawn betrayal — read quest text
5. **窃取物品 / Stealing**：在NPC背后按互动键可以偷窃——但被抓住会降低好感 / Press interact behind NPC to steal — caught = affinity penalty
6. **技能槽有限 / Limited Skill Slots**：每个职业只有4个技能槽——谨慎选择 / Only 4 skill slots per vocation — choose carefully
7. **负重影响速度 / Encumbrance**：超重时移动变慢、耐力消耗加倍 / Overweight = slow movement + double stamina cost
8. **随从指令 / Pawn Commands**：用指令"Help"可以叫随从治疗你 / "Help" command calls pawns to heal you
