## 装备升级 / Gear Upgrades

在铁匠铺使用材料和金币升级装备——最高 **3星** / Upgrade gear at smithy — max **3 stars**.

| 星级 / Stars | 加成 / Bonus | 材料 / Materials |
|---|---|---|
| 1星 | 基础属性+30% / +30% base stats | 普通矿石+金币 / Common ore + gold |
| 2星 | 基础属性+60% / +60% base stats | 高级矿石+龙血晶 / Rare ore + WLC |
| 3星 | 基础属性+100% / +100% base stats | 传说矿石+龙血晶x3 / Legendary ore + 3x WLC |

**关键材料 / Key Materials**:
- **龙血晶 / Wyrmlife Crystal (WLC)**: 击杀龙获得——后期升级必备 / Dragon drop — essential for late upgrades
- **精炼矿石 / Refined Ore**: 矿脉采集+铁匠加工 / Mining + smithy processing
- **魔法晶石 / Magick Crystal**: 击杀元素生物获得 / Elemental enemy drops

## 铁匠铺与强化 / Smithy & Enhancement

每个主要城市有铁匠铺——提供以下服务 / Every major city has a smithy:

| 服务 / Service | 费用 / Cost | 说明 / Description |
|---|---|---|
| 购买武器/防具 / Buy gear | 金币 / Gold | 基础装备——等级越高刷新越好 / Basic gear |
| 装备升级 / Upgrade | 金币+材料 / Gold+mats | 最高3星——大幅提升属性 / Up to 3 stars |
| 装备分解 / Dismantle | 免费 / Free | 获得升级材料 / Get upgrade materials |
