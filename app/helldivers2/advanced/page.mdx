## 高级移动技巧 / Advanced Movement

### 滑铲 / Slide

滑铲是游戏中最被低估的机动技巧——操作：跑动中按下蹲伏键（C→C长按）/ Slide is the most underrated movement tech — how-to: sprint + crouch (C → hold C).

| 技巧 / Tech | 操作 / Input | 效果 / Effect | 用途 / Use |
|------|------|------|------|
| **滑铲 / Basic Slide** | 奔跑→按蹲伏 / Sprint → Crouch | 快速滑行下坡/平地 / Slide on slopes/flat ground | 快速穿越开阔地被 / Cross open areas quickly |
| **滑铲射击 / Slide Shooting** | 滑铲中开枪 / Shoot while sliding | 移动中维持火力 / Maintain fire while moving | 边滑边打冲锋虫 / Dodge Chargers while shooting |
| **滑铲转身 / Slide Turn** | 滑铲中转向 / Turn while sliding | 改变滑行方向 / Change slide direction | 绕柱子/掩体 / Corner peeking |
| **连续滑铲 / Chain Slide** | 滑铲结束→立刻再按 / Slide → immediately slide again | 保持移动速度 / Maintain momentum | 长途快速移动 / Long-distance speed |

## 团队配合 / Team Coordination

### 标记系统 / Ping System

| 操作 / Action | PS5 | PC | 说明 / Description |
|------|:----:|:----:|------|
| **标记/标点 / Ping** | 方向键↑ / D-pad up | Q / 鼠标中键 / Middle mouse | 标记敌人/物品/位置 / Mark enemies/items/locations |
| **警告 / Warning** | 快速双击↑ / Double-tap up | 双击Q / Double-tap Q | 危险警告——敌人/空袭/地雷 / Danger warning |
| **确认 / Affirmative** | 按住↑+选确认 / Hold up + select | 按住Q+选确认 / Hold Q + select | 确认指令/回复队友 / Confirm orders/reply |
| **请求 / Request** | 按住↑+选请求 / Hold up + select | 按住Q+选请求 / Hold Q + select | 请求弹药/支援/治疗 / Request ammo/support/heals |

> **关键**：标记是最高效的沟通方式——比打字快，比开麦易懂。优秀团队每秒都在标记 / **Key**: Ping is the most efficient communication — faster than typing, clearer than voice. Good teams ping constantly

|:----:|------|------|

## 单刷指南 / Solo Play Guide

虽然绝地潜兵2是合作游戏，但有时你需要或想要一个人玩 / While Helldivers 2 is a co-op game, sometimes you need or want to play solo.

### 单刷基本原则 / Solo Fundamentals

| 原则 / Principle | 说明 / Description |
|------|------|
| **潜行优先 / Stealth First** | 你没有队友吸引火力——不要主动开战。蹲伏移动，绕开巡逻队 / No one to draw aggro — don't start fights. Crouch-walk, avoid patrols |
| **完成目标 = 胜利 / Objectives = Victory** | 不需要清光全图——只做任务目标，然后撤离 / Don't need to clear the map — do objectives, then extract |
| **高机动配置 / High Mobility Loadout** | 轻型护甲+跳跃背包+体力增强——跑得快就是活得久 / Light armor + Jump Pack + Stamina Enhancement — speed = survival |
| **呼叫增援不要省 / Call Reinforcements Freely** | 死了就呼叫自己——单人时增援次数够用 / Die → reinforce yourself — enough reinforcements for solo |

### 单刷推荐配置 / Solo Loadout

| 槽位 / Slot | 推荐 / Recommendation | 理由 / Why |
|------|------|------|
| **护甲 / Armor** | 轻甲+马拉松 / Light + Marathon | 跑得快 + 体力多 / Fast + endless stamina |
| **增益 / Booster** | 体力增强 / Stamina Enhancement | 跑得更快 / Even faster |
| **主武器 / Primary** | SG-225破坏者 / Breaker | 近距离高爆发——应对遭遇战 / Close-range burst for unexpected fights |
| **支援武器 / Support** | 类星体炮 / Quasar Cannon | 无限弹药——远程反甲 / Infinite ammo — long-range AT |
| **战略配备1 / Stratagem 1** | 跳跃背包 / Jump Pack | 快速脱离战场 / Quick escape |
| **战略配备2 / Stratagem 2** | 500kg炸弹 / 500kg Bomb | 秒杀精英+摧毁目标 / One-shot elites + objectives |
| **战略配备3 / Stratagem 3** | 机枪炮台 / MG Sentry | 吸引火力+辅助输出 / Distraction + DPS |

### 单刷任务流程 / Solo Mission Flow

```
空降 → 蹲伏移动 → 避战 → 到达目标点 → 快速完成目标
→ 立刻远离 → 绕路到下一个目标 → 重复
→ 全部完成 → 呼叫撤离 → 跑向撤离点 → 防御 → 登船
```

> **核心思想**：你不是兰博——你是特工。快速进，快速出，不要恋战 / **Mantra**: You're not Rambo — you're a spy. Get in, get out, don't linger

### 摧毁任务 / Destroy Missions

|------|------|------|
| **虫族巢穴 / Bug Nest** | 手榴弹/空袭 / Grenade/Airstrike | 每个洞口扔一个手榴弹——确认火焰熄灭才有效 / Toss a grenade in each hole — confirm fire goes out |

### 激活任务 / Activate Missions

| 技巧 / Tip | 说明 / Description |
|------|------|------|
| **一人操作，三人防守 / One works, three defend** | 标准配合——一个人操作终端，其他人四周清怪 / Standard tactic — one operates, others clear around |
| **先清后开 / Clear First, Then Activate** | 在激活前先清理周围的敌人和刷兵点 / Clear nearby enemies and spawn points before activating |
| **炮台保护 / Sentry Protection** | 在终端两侧放机枪炮台——提供火力掩护 / Place MG Sentries on both sides of terminal for cover |

### 撤离技巧 / Extraction Tips

| 技巧 / Tip | 说明 / Description |
|------|------|------|
| **不要在撤离点等 / Don't Wait at Extract** | 呼叫撤离后不要立刻到撤离点——运输船需要90秒到达，提前到只会被围攻 / After calling extract, don't rush to the LZ — the shuttle takes 90s, arriving early means getting swarmed |
| **提前布防 / Pre-set Defenses** | 在撤离点周围放炮台/地雷——运输船抵达时敌人也会蜂拥而至 / Place sentries/mines around the LZ — enemies will swarm when the shuttle arrives |
| **最后一人 / Last Man Standing** | 如果队友都死了——你必须活着登上运输船。所有人的样本都在你身上 / If squad is dead — you MUST board. Everyone's samples are on you |
| **烟雾掩护 / Smoke Cover** | 在撤离点扔烟雾弹——减少被敌人命中的概率 / Smoke the LZ to reduce enemy accuracy |
