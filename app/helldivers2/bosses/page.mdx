### 击杀方法 / Kill Methods

**方法一：类星体炮 / Quasar Cannon Method**

1. 保持30米以上距离 / Maintain 30m+ distance
2. 瞄准口部（张开时）/ Aim for mouth (when it opens)
3. 类星体打口部——5发击杀 / 5 Quasar shots to mouth = kill
4. 充能间隙横向移动 / Strafe during charge cooldown

**方法二：轨道激光 / Orbital Laser Method**

1. 呼叫轨道激光——瞄准阔步虫位置 / Call orbital laser — target the Titan
2. 激光会自动跟踪——全程照射约8秒 / Laser auto-tracks — ~8s continuous burn
3. 一个激光可击杀1-2只阔步虫 / One laser kills 1-2 Titans
4. 注意激光也会烧到队友 / Laser also burns teammates

**方法三：500kg炸弹 / 500kg Bomb Method**

1. 精确投放在阔步虫脚下 / Precision drop at Titan's feet
2. 500kg爆炸范围15米——确保直接命中 / 500kg has 15m blast — ensure direct hit
3. 一发击杀——但投掷时机是关键 / One-shot kill — timing is critical
4. 在阔步虫停止移动时投掷 / Drop when Titan pauses movement

**方法四：团队集火 / Team Focus Fire**

1. 全队使用机炮/类星体——同时开火 / Team uses Autocannon/Quasar — sync fire
2. 瞄准同一弱点（口部或腹部）/ Target same weak point (mouth or belly)
3. 2-3人同时打——10秒内击杀 / 2-3 players firing together — kill in 10s
4. 一人负责清除刷新的小虫——其他人集火 / One player clears adds — others focus fire

### 击杀方法 / Kill Methods

**方法一：绕后打发动机 / Rear Engine Method**

1. 绕到坦克后方 / Flank to tank rear
2. 打发动机舱——弱点是红色散热格栅 / Shoot engine compartment — red vent grille
3. 类星体炮——3发可摧毁 / Quasar Cannon — 3 shots destroy
4. RPG——4发锁定击杀 / Spear — 4 lock-on shots kill

**方法二：轨道精准打击 / Precision Strike Method**

1. 瞄准坦克正下方 / Aim directly below tank
2. 一发轨道精准打击——直接命中可秒杀 / One Precision Strike — direct hit kills
3. 需要精确瞄准——坦克移动时很难打中 / Needs precise aim — hard vs. moving tank

**方法三：热成像+火箭筒 / Thermal + Recoilless**

1. 使用望远镜标记坦克 / Binoculars to mark tank
2. 导弹发射器锁定标记 / Spear locks on marked target
3. 4发全部命中后部——摧毁 / 4 rear hits = destroy
4. 队友牵引机器人注意力 / Teammate draws bot aggro

### 坦克巡逻队 / Tank Patrols

在难度8以上的机器人任务中——坦克会以2-3辆编队巡逻。应对方法： / At difficulty 8+ bot missions — tanks patrol in 2-3 vehicle squads.

- **不要正面接战**——先撤到掩体后 / Don't engage head-on — retreat to cover first
- **用地雷封锁道路**——坦克会碾过地雷 / Mine the road — tanks drive over mines
- **逐个击破**——先打最后一辆 / Pick them off one by one — kill the rear tank first

## 任务事件Boss / Mission Event Bosses

### 虫后 / Bug Queen

**出现方式 / Spawn Method**：特定"消灭虫巢"任务中——摧毁全部小巢后虫后会从主巢中出现 / Specific "Destroy Nest" missions — destroy all small nests, Queen emerges from main nest.

**攻击模式 / Attack Pattern**：
| 攻击 / Attack | 伤害 / Damage | 应对 / Counter |
|---|---|---|
| 产卵 / Spawn | 每3秒产5只小虫 / 5 bugs every 3s | 优先清理小虫——不要让它们堆积 / Clear adds first |
| 酸液波 / Acid Wave | 前方锥形AOE——秒杀 / Cone AOE — one-shot | 侧向跑出锥形范围 / Run laterally out of cone |
| 尾部刺击 / Tail Stab | 高伤害 / High dmg | 不要站在虫后正后方 / Never stand directly behind |

**击杀方法 / Kill Method**：
1. 全队在虫后召唤小虫前就位 / Position before Queen spawns adds
2. 2人吸引虫后注意力 / 2 players draw aggro
3. 2人用类星体/轨道激光打虫后腹部 / 2 players hit belly with Quasar/Laser
4. 小虫刷出后立刻AOE清理——集束炸弹最佳 / AOE the adds immediately — cluster bomb ideal

### 巨型指挥官 / Hulk Commander

**出现方式 / Spawn Method**：机器人基地的指挥官——击败所有小兵后出现 / Bot base commander — appears after clearing all soldiers.

**特殊机制 / Special Mechanic**：Hulk Commander会强化周围的机器人——让他们伤害翻倍 / Buffs nearby bots — doubles their damage.

**击杀方法 / Kill Method**：
1. 先清掉周围所有小兵——不要让指挥官Buff生效 / Clear all surrounding soldiers first
2. 打头部——但正面有装甲盾 / Headshot — but front has armor shield
3. 绕后——打背部的燃料罐 / Flank — hit fuel canister on back
4. 燃料罐着火后——全队后撤——5秒后爆炸 / Fuel canister lit — retreat — 5s to explosion

## Boss战综合技巧 / Boss Fight General Tips

### 战前准备 / Pre-Battle Preparation

1. **确认每个人的战略配备**——至少2人有反重甲能力 / Everyone needs anti-heavy — minimum 2 players
2. **弹药满——体力满——激励剂满** / Full ammo, full stamina, full stims
3. **确认撤离点位置**——打不过可以跑 / Know extract location — retreat is always an option
4. **不要在一个位置待太久**——Boss会呼叫增援 / Don't stay in one spot — Boss calls reinforcements

### 战斗中的走位 / Mid-Battle Positioning

- **三角阵型**：3人站三角——Boss只能面对一个方向 / Triangle formation — Boss faces one direction at a time
- **轮流吸引**：谁被Boss盯上——谁就后退——其他人输出 / Draw aggro rotation — target retreats, others DPS
- **不要站在一起**：AOE攻击会把抱团的队伍全灭 / Never group up — AOE kills clustered teams
- **利用地形**：高地打低地有射程优势 / High ground advantage — range bonus

### Boss战常见错误 / Common Boss Fight Mistakes

**错误1：过于分散** / Mistake 1: Too Spread Out
队友之间距离超过50米——被Boss逐个击破 / Teammates 50m+ apart — picked off one by one.

**错误2：不清理小兵** / Mistake 2: Ignoring Adds
全队集火Boss但不处理小兵——最后被小兵包围打死 / All focus Boss but ignore adds — overwhelmed by small enemies.

**错误3：浪费战略配备** / Mistake 3: Wasting Stratagems
在Boss还没出现之前就把轨道炮用了——Boss出现时没技能 / Using orbital on patrols before Boss appears — no skills for Boss.

**错误4：站桩输出** / Mistake 4: Stationary DPS
站在原地打Boss——被Boss的AOE技能直接秒杀 / Standing still while DPSing — killed by Boss AOE.
