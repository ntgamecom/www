# 护甲与增益 / Armor & Boosters

选择合适的护甲和增益可以大幅提升你的生存能力和战斗效率。本章全面解析护甲系统、被动能力和增益效果 / Choosing the right armor and boosters can dramatically improve your survival and combat effectiveness. This chapter covers armor systems, passive abilities, and booster effects.

---

## 护甲重量等级 / Armor Weight Classes

| 类型 / Type | 护甲值 / Armor | 速度 / Speed | 耐力恢复 / Stamina Regen | 特点 / Characteristics |
|:----:|:----:|:----:|:----:|------|
| **轻型 / Light** | 50 | +10% | +50% | 最高机动性——适合侦查和游击战术 / Maximum mobility — for scouts and hit-and-run |
| **中型 / Medium** | 100 | 0% | 0% | 均衡选择——适合大多数玩家和场景 / Balanced — for most players and situations |
| **重型 / Heavy** | 150 | -10% | -50% | 最高防御——适合正面交火和防御任务 / Maximum protection — for front-line and defense |

### 重量选择建议 / Weight Selection Guide

| 场景 / Scenario | 推荐重量 / Recommended Weight | 理由 / Why |
|------|:----:|------|
| **新手熟悉游戏 / Learning the game** | 中型 / Medium | 平衡——容错率适中 / Balanced — forgiving enough |
| **低难度清任务 / Low-difficulty farming** | 轻型 / Light | 跑得快=完成任务快 / Faster movement = faster missions |
| **高难度虫族 / High-difficulty bugs** | 重型 / Heavy | 虫族近战多——护甲减少大量伤害 / Bugs are melee-heavy — armor reduces significant damage |
| **高难度机器人 / High-difficulty bots** | 中型+护盾包 / Medium + Shield Pack | 机器人用精准射击——护盾比护甲更有效 / Bots use precision fire — shields > armor |
| **侦查跑图 / Scouting** | 轻型 / Light | 速度决定一切 / Speed is everything |

## 增益 / Boosters

增益是任务出发前选择的全局效果——每队可选一个，效果全队共享 / Boosters are global effects selected before launch — one per squad, affects all members.

| 增益 / Booster | 效果 / Effect | 评级 / Tier | 推荐场景 / Best For |
|------|------|:----:|------|
| **体力增强 / Stamina Enhancement** | 体力消耗-30% / Stamina drain -30% | **S级** | **全场通用——逃跑/追击/跑图全受益** / **Best in slot — running, chasing, traversing** |
| **医疗针升级 / Stim Upgrade** | 医疗针效果+50% / Stim effectiveness +50% | **A级** | 高难度生存必备 / Essential for high-difficulty survival |
| **护甲强化 / Armor Enhancement** | 全队护甲值+25 / Team armor +25 | A级 | 额外坦度——适合新手 / Extra survivability — good for beginners |
| **增援预算 / Reinforcement Budget** | 初始增援+2次 / +2 starting reinforcements | B级 | 新手队——容易死人 / Beginner squads — more deaths expected |
| **雷达增强 / Radar Enhancement** | 雷达扫描范围+50% / Radar range +50% | B级 | 侦察任务——找样本/隐藏点 / Scout missions — find samples/hidden items |
| **体力恢复 / Stamina Recovery** | 体力恢复速度+50% / Stamina regen +50% | B级 | 配合马拉松被动——无限跑图 / With Marathon passive — infinite sprint |
| **毒气抵抗 / Gas Resistance** | 毒气伤害-100% / Gas damage -100% | C级 | 仅限毒气相关任务 / Only for gas-related objectives |
| **电网抵抗 / Electrical Resist** | 电网伤害-50% / Electrical damage -50% | C级 | 特定地图有用 / Situational use |

> **建议**：体力增强是目前版本最强增益——任何配置任何任务都推荐携带。其次推荐医疗针升级 / **Tip**: Stamina Enhancement is the current meta — useful in every loadout and every mission. Stim Upgrade is a strong second
