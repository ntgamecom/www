# 新手入门 / Beginner Guide

欢迎加入超级地球的精英部队！绝地潜兵2与常规射击游戏有很大不同——友军伤害、战略配备、团队配合——本章带你快速上手 / Welcome to Super Earth's elite forces! Helldivers 2 is very different from typical shooters — friendly fire, stratagems, and teamwork — this chapter gets you up to speed quickly.

## 游戏核心循环 / Core Gameplay Loop

```
登上飞船 → 选择星球/任务 → 装备战略配备 → 空降星球 → 完成任务目标
→ 收集样本/奖章 → 呼叫撤离 → 抵达撤离点 → 登上运输船 → 回到飞船
→ 升级飞船/解锁装备 → 选择下一个任务
```

### 任务阶段详解 / Mission Phase Breakdown

| 阶段 / Phase | 说明 / Description |
|------|------|
| **空降 / Drop** | 从飞船以单人舱或全体舱降落。选择降落点很重要——避开开阔地和密集敌人 / Drop in a single or squad pod. Landing zone selection matters — avoid open areas and dense enemies |
| **执行 / Execute** | 完成主要目标（摧毁/占领/激活等），同时处理次要目标和遭遇战 / Complete main objectives (destroy/capture/activate) while handling side objectives and patrols |
| **收集 / Collect** | 收集地图上的经验样本、超级积分和奖章 / Gather XP samples, Super Credits, and Medals on the map |
| **撤离 / Extract** | 完成所有目标后呼叫运输船——抵达撤离点→击退敌人→等待登船 / Call in the shuttle → reach extraction point → fight off enemies → board |

## 战略配备系统 / Stratagem System

战略配备是游戏的核心机制——通过输入方向键代码呼叫各种支援 / Stratagems are the core mechanic — input directional codes to call in various support.

### 输入方法 / How to Input

1. 按住 **L1** (PS5) / **Ctrl** (PC) 打开战略配备界面 / Open the stratagem interface
2. 用**方向键** (PS5) / **WASD** (PC) 输入对应代码 / Input the code with the D-pad / WASD
3. 放开按键投掷信标——空投将在落地位置抵达 / Release to throw the beacon — the drop arrives at the landing spot

> **关键技巧**：在跑动中或跳跃时可以输入战略配备码——不要站着不动！/ **Key Tip**: You can input stratagem codes while sprinting or jumping — don't stand still!

| **机枪 / Machine Gun** | ↓,←,↓,↑,← | 万金油支援武器——清理虫群和机器人步兵 / Versatile support weapon — clears bugs and bot infantry |
| **空袭 / Airstrike** | ↑,→,↓,→ | 简单好用的范围攻击——清场利器 / Simple, effective AoE — great for clearing areas |

## 友军伤害须知 / Friendly Fire Guide

**这是最重要的机制——友军伤害永远开着** / **This is the most important mechanic — friendly fire is ALWAYS on**

| 场景 / Situation | 建议 / Tip |
|------|------|
| **射击时 / When Shooting** | 开火前确认弹道上没有队友——绝地潜兵2的子弹可以穿透友军再击中敌人 / Check for teammates in your line of fire — bullets pass through friendlies before hitting enemies |
| **投掷战略配备 / Throwing Stratagems** | 战略配备信标会粘在队友身上——如果看到信标贴在队友背后，立刻远离 / Beacons stick to teammates — if you see one stuck to a teammate, RUN |
| **手榴弹 / Grenades** | 手榴弹反弹不明——注意墙壁反弹方向 / Grenade bounces are unpredictable — watch for wall rebounds |
| **空袭/轨道炮 / Airstrikes/Orbitals** | 扔完战略配备後立刻远离落点——至少跑15米 / After throwing, immediately move away from the impact zone — at least 15 meters |
| **误杀队友后 / After TK** | 按互动键道歉——但队友可能还是会踢你 / Press interact to apologize — but your team might still kick you |

> **金色规则**：如果你看到队友在和敌人近战——**不要开枪打那个敌人**——等你打中敌人的时候已经打中队友了 / **Golden Rule**: If you see a teammate meleeing an enemy — **do not shoot** — you'll hit the teammate before the enemy.

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **先做教程 / Do the Tutorial** | 游戏有完整的操作教程——不要跳过，战略配备输入需要肌肉记忆 / Full tutorial available — don't skip it, stratagem inputs need muscle memory |
| **2** | **从虫族低难度开始 / Start with Low-Difficulty Bugs** | 虫族在三派系中最简单。难度1-3适合新手熟悉基本操作 / Terminids are the easiest faction. Difficulty 1-3 is perfect for learning |
| **3** | **跟随队友 / Stick with Your Team** | 单走=死得快。和队友保持目视距离，互相支援 / Going solo = quick death. Stay within visual range of your team |
| **4** | **完成任务后呼叫撤离 / Call Extract After Objectives** | 完成所有目标后立刻呼叫撤离——运输船需要90秒到达 / After completing objectives, call extraction immediately — the shuttle takes 90 seconds |
| **5** | **收集样本升级飞船 / Collect Samples** | 绿色/橙色/紫色样本是升级飞船的关键资源——看到就捡 / Green/orange/purple samples are key for ship upgrades — pick them up whenever you see them |
| **6** | **不要浪费增援 / Don't Waste Reinforcements** | 全队共有共享的增援次数——死亡过多的队友会消耗团队资源 / The squad shares reinforcement count — dying repeatedly drains team resources |
| **7** | **语音沟通 / Use Voice Chat** | 虽然ping系统很好用，但开麦交流战术配合效率远超标记 / Ping system is good, but voice chat for tactics is far more effective |
| **8** | **弹药管理 / Manage Ammo** | 地图上有弹药盒，队友可以带补给包——不要随便浪费子弹 / Ammo boxes are on the map, teammates can bring supply packs — don't waste bullets |
| **9** | **了解撤离机制 / Understand Extraction** | 撤离时所有玩家必须进入运输船。如果有人在登船时阵亡——等于没撤离 / All players must board the shuttle. If someone dies while boarding — they don't extract |
| **10** | **享受混乱 / Embrace the Chaos** | Helldivers 2 是一场美丽的混乱——就算全队爆炸、空袭砸自己人——也是乐趣的一部分 / Helldivers 2 is beautiful chaos — even when the whole squad explodes or airstrikes hit friendlies — it's part of the fun |
