## 战斗基础 / Combat Basics

绝地潜兵2的战斗以俯视角射击为基础，结合空袭配合和策略性走位。最大特色是**友军伤害始终开启**——你的每一发子弹都可能杀死队友 / Helldivers 2 combat is top-down shooter with stratagem support and tactical positioning. The core feature is **friendly fire always on** — every bullet can kill your teammates.

### 核心机制 / Core Mechanics

| 机制 / Mechanic | 说明 / Description | 使用技巧 / Tips |
|---|---|---|
| 射击 / Shooting | 全方向自由射击——有后座力和散布 / Free-aim with recoil and spread | 蹲下减少后座50% / Crouch reduces recoil 50% |
| 疾跑 / Sprint | 按Shift冲刺——回复体力 / Sprint with Shift — stamina regen | 被追杀时蛇形跑——虫族近战有前摇 / Zigzag when chased — bugs have attack windup |
| 翻滚 / Dive | 按Alt+方向键翻滚——躲避攻击 / Alt+direction to dive | 朝侧面或朝着来袭方向滚 / Dive sideways or toward attacker |
| 近战 / Melee | 按F近战——可击退小型敌人 / F to melee — knocks back small enemies | 被虫族包夹时打出一条路 / Create escape route when surrounded |
| 蹲下/卧倒 / Crouch/Prone | Ctrl切换姿态 / Ctrl to toggle | 卧倒减少被弹面80%——面对重甲单位必备 / Prone reduces hitbox 80% — essential vs. heavy armor |

### 火力团队配合 / Fire Team Coordination

**4人队标准分工 / Standard 4-Player Roles**：
- **指挥 / Commander**：使用望远镜标记目标和呼叫轨道打击 / Binoculars mark targets, call orbital strikes
- **重火力 / Heavy**：携带机炮/类星体炮——处理重甲单位 / Autocannon/Quasar Cannon — anti-heavy
- **支援 / Support**：携带补给背包和弹药——维持团队续航 / Supply pack + ammo — sustain the team
- **侦察 / Scout**：潜行标记巡逻队——优先消灭警报单位 / Stealth mark patrols — eliminate alarm units first

## 团队配合进阶 / Advanced Teamwork

### 救援技巧 / Rescue Techniques

1. **倒地标志**：队友倒地后——先清场再救援——不要直接冲 / Downed teammate — clear area before res — don't rush in
2. **吸引火力**：一人翻滚吸引敌人注意——另一人救援 / One player dodge to draw aggro — other revives
3. **烟雾掩护**：救援前先丢烟雾弹——让敌人失去视野 / Smoke before revive — block enemy vision
4. **激励剂接力**：用激励剂可以让倒地队友的恢复速度翻倍 / Stim pistol doubles downed teammate's recovery speed

### 弹药管理 / Ammo Management

- 主武器弹药用完前就呼叫补给——不要等到0发 / Call resupply before primary runs out — not at 0
- 补给背包每次用完4个格子可补充全队4份 / Supply pack fully stocks when empty — shares 4 supplies
- 毒刺/导弹等一次性武器——打完再丢——不要带着空筒子跑 / One-shot weapons — empty before dropping — don't carry spent tubes

### 撤离战术 / Extraction Tactics

1. **提前3分钟开始向撤离点移动**——不要等到30秒 / Start moving to extract 3 minutes early — not 30s
2. **撤离点布防**：在撤离点周围放置哨戒炮和地雷 / Set sentries and mines around extract zone
3. **烟雾包围**：在撤离点周围的一圈都丢上烟雾弹——让敌人失去目标 / Ring the LZ with smoke — blind the enemies
4. **最后30秒**：全员收缩到撤离点中心——互相掩护背部 / Final 30s — collapse to center — cover each other's back

## 特殊战斗场景 / Special Combat Scenarios

### 虫洞防守 / Bug Nest Defense

**战术 / Tactic**：在虫洞口放置地雷和泰瑟炮——让虫子出来就被电死 / Place mines and Tesla at nest entrance — bugs die on spawn.

### 机器人基地突袭 / Bot Base Assault

**战术 / Tactic**：不要从正面进攻——绕到基地后方的高地——先打防空炮 / Never assault front-on — flank to high ground — kill AA first.

### 火车护送 / Train Escort

**战术 / Tactic**：2人守前——2人守后——类星体炮对付刷新的坦克虫 / 2 front guard — 2 rear — Quasar Cannon for charger spawns.
