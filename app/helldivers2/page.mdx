# 绝地潜兵2 攻略 / Helldivers 2 Guide

绝地潜兵2（Helldivers 2）是Arrowhead Game Studios开发的第三人称合作射击游戏。作为超级地球的精英士兵，你将与其他三名玩家组队，在 procedurally 生成的战场上对抗外星威胁，传播管理式民主 / Helldivers 2 is a third-person cooperative shooter by Arrowhead Game Studios. As an elite soldier of Super Earth, you team up with up to three players on procedurally generated battlefields to fight alien threats and spread managed democracy.

**平台 / Platforms**: PS5, PC (Steam) | **发售日 / Release**: 2024年2月8日 / Feb 8, 2024 | **类型 / Genre**: 第三人称合作射击 / Third-person Co-op Shooter | **跨平台联机 / Crossplay**: 支持PS5 ↔ PC完全跨平台 / Full PS5 ↔ PC crossplay

## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **战略配备系统 / Stratagem System** | 输入方向指令呼叫空投——武器、攻击、支援全部通过战略配备码呼叫 / Input directional codes to call airstrikes, support weapons, and supplies |
| **友军伤害 / Friendly Fire** | **队友伤害全程开启**——你的每一发子弹都可能杀死队友——谨慎开火！/ **Friendly fire is always on** — every bullet can kill your teammates — fire with caution! |
| **动态星球战争 / Galactic War** | 所有玩家共同参与的持久战——每个星球都有社区进度条 / Persistent community-driven war — every planet has a community progress bar |
| **程序化战场 / Procedural Maps** | 每次任务的地图布局、敌人位置、目标分布都不同 / Map layout, enemy positions, and objective locations change every mission |
| **ragdoll 物理 / Ragdoll Physics** | 爆炸会让敌人和队友飞出去——效果夸张又搞笑 / Explosions send enemies and teammates flying — over-the-top and hilarious |
| **飞船为家 / Ship as Hub** | 你的飞船是行动基地——在其中选择任务、配置装备、升级船舰 / Your ship is the hub — select missions, equip gear, upgrade the ship |

## 战争债券系统 / Warbond System

| 概念 / Concept | 说明 / Description |
|------|------|
| **免费债券 / Free Warbond** | 基础进度奖励——使用经验值解锁 / Base progression rewards — unlock with XP |
| **高级债券 / Premium Warbond** | 使用"超级积分"解锁——包含独有武器、装甲和外观 / Unlock with Super Credits — exclusive weapons, armor, and cosmetics |
| **超级积分 / Super Credits** | 付费货币，但也可在地图上拾取（每局约10-30个）/ Premium currency, but can also be found on the map (~10-30 per mission) |
| **当前债券 / Current Warbond** | 每个债券约持续6-8周——过期后可能回归 / Each bond lasts ~6-8 weeks — may return later |
