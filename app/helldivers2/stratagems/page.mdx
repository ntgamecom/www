# 战略配备 / Stratagems Guide

战略配备是绝地潜兵2的灵魂——从机枪到轨道炮，从盾牌到炮台。本章详解所有战略配备类型及推荐选择。关于武器和护甲的详细信息请参考[武器装备](/helldivers2/weapons)和[护甲与增益](/helldivers2/armor)章节 / Stratagems are the soul of Helldivers 2 — from machine guns to orbital cannons, shields to sentries. This chapter covers all stratagem types and recommendations. For weapons and armor details, see the [Weapons](/helldivers2/weapons) and [Armor & Boosters](/helldivers2/armor) chapters.

---

## 战略配备类型 / Stratagem Types

| 类型 / Type | 说明 / Description | 示例 / Example |
|------|------|------|
| **支援武器 / Support Weapons** | 可拾取的强力武器——替代主武器槽 / Powerful pickup weapons — replace primary slot | 机枪、无后坐力炮、火焰喷射器 / MG, Recoilless, Flamethrower |
| **空袭 / Airstrikes** | 一次性范围轰炸 / One-time AoE bombardment | 集束炸弹、500kg炸弹 / Cluster Bomb, 500kg Bomb |
| **轨道攻击 / Orbital Attacks** | 从飞船发射的轨道打击 / Ship-to-ground orbital bombardments | 轨道炮打击、轨道激光 / Orbital Precision Strike, Orbital Laser |
| **防御装备 / Defensive Gear** | 护盾、炮台、地雷 / Shields, sentries, mines | 护盾背包、机枪炮台、反人员地雷 / Shield Pack, MG Sentry, AP Mine |
| **补给类 / Supplies** | 弹药、增援、补给 / Ammo, reinforcements, supplies | 补给包、增援信标 / Supply Pack, Reinforce Beacon |
| **侦察类 / Scout** | 标记、探路、信息 / Marking, pathfinding, intel | 无人机、标记枪 / Drone, Marker Gun |
