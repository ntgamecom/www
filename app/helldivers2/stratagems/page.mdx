# 战略配备 / Stratagems Guide

战略配备是绝地潜兵2的灵魂——从机枪到轨道炮，从盾牌到炮台。合理的战略配备搭配决定了你能打到什么难度 / Stratagems are the soul of Helldivers 2 — from machine guns to orbital cannons, shields to sentries. The right loadout determines how far you can push difficulty.

---

## 战略配备类型 / Stratagem Types

| 类型 / Type | 说明 / Description | 示例 / Example |
|------|------|------|
| **支援武器 / Support Weapons** | 可拾取的强力武器——占用背包位或武器槽。通常需要空投呼叫 / Powerful pickup weapons — take backpack or weapon slot. Usually airdropped | 机枪、无后坐力炮、火焰喷射器 / MG, Recoilless, Flamethrower |
| **空袭 / Airstrikes** | 一次性范围轰炸——快速、暴力、冷却短。适合清场 / One-time AoE bombardment — fast, violent, short cooldown. Great for clearing | 集束炸弹、500kg炸弹 / Cluster Bomb, 500kg Bomb |
| **轨道攻击 / Orbital** | 从飞船发射——威力大但延迟高，需要预判 / Ship-based — high power but long delay, needs prediction | 轨道炮打击、轨道激光 / Precision Strike, Orbital Laser |
| **防御装备 / Defensive** | 护盾、炮台、地雷——控制战场 / Shields, sentries, mines — battlefield control | 护盾背包、机枪炮台 / Shield Pack, MG Sentry |
| **补给类 / Supplies** | 弹药、增援、补给——团队生存关键 / Ammo, reinforcements, supplies — team survival | 补给包、增援信标 / Supply Pack, Reinforce |
| **侦察类 / Scout** | 标记、探路——高难度必备 / Marking, pathfinding — essential on higher difficulties | 无人机、标记枪 / Drone, Marker Gun |

## 战略配备进阶技巧 / Advanced Stratagem Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **战略配备代码记忆 / Memorize Codes** | 常用配备记住代码——紧急情况下不用看菜单直接按。500kg: ↓↑←↓↑→, 轨道炮: ↑→←↓ / Memorize codes for常用 strats — 500kg: ↓↑←↓↑→, Precision: ↑→←↓ |
| **战略配备组合 / Stratagem Combos** | 500kg+轨道激光：先用500kg炸开护甲，轨道激光烧裸露弱点。或者空袭清场→再丢500kg收尾 / 500kg → Orbital Laser: 500kg breaks armor, laser burns weak points |
| **战略配备插队 / Queue Priority** | 紧急时刻可以中断当前输入切换战略配备。但要小心——按错可能召来空袭炸自己 / In emergencies, cancel current input to call priority stratagem. But mis-input = team wipe |
| **呼叫位置选择 / Call-in Positioning** | 支援武器叫在掩体后面——不要在开阔地等空投。增援信号弹丢在安全区域——不要在敌人脚下召队友 / Call support weapons behind cover. Don't reinforce in the middle of a firefight |
| **炮台放置 / Sentry Placement** | 炮台架在高处、侧翼——不要和炮台站一起，不然敌人AOE连你一起炸。炮台之间留间距，防止一个AOE清掉全部 / Place sentries on high ground, flanking positions. Don't stand near them. Space them apart |
| **补给管理 / Ammo Management** | 补给包每90秒补充4个针+弹药——可以用"补给"战略额外呼叫。机器人任务弹药消耗快——多带补给 / Supply pack replenishes every 90s. Bot missions burn through ammo fast — bring extra supplies |
