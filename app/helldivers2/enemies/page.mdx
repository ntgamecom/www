# 敌人与派系 / Enemies & Factions

绝地潜兵2目前有三个敌对派系，各有完全不同的战斗风格和单位类型。了解每个派系的弱点和最佳应对策略是在高难度生存的关键 / Helldivers 2 currently has three enemy factions, each with completely different combat styles and unit types. Knowing each faction's weaknesses and optimal counter-strategies is key to surviving higher difficulties.

---

## 派系对比 / Faction Comparison

| 对比项 / Aspect | 虫族 / Terminids | 机器人 / Automatons | 光能者 / Illuminate |
|------|------|------|------|
| **主要威胁 / Main Threat** | 近战海量冲锋 / Melee swarm charge | 远程精准火力 / Ranged precision fire | 高速机动+能量护盾 / Fast mobility + energy shields |
| **弱點 / Weakness** | 火焰、爆炸、范围伤害 / Fire, explosives, AoE | 精准射击、穿甲武器、EMP / Precision, AP weapons, EMP | 高射速武器、破盾+爆发 / High fire rate, break shield + burst |
| **推荐护甲 / Armor** | 中型/重型 / Medium/Heavy | 推荐携带护盾包 / Shield pack recommended | 高速轻型/中型 / Fast light/medium |
| **最难对付 / Hardest** | 虫潮泰坦 + 冲锋虫群 / Bile Titan + Charger swarms | 坦克群 + 火箭兵 / Tank squads + Rocket Devastators | 大型幻影 + 升腾者 / Large Illusionists + Elevators |

| 策略 / Strategy | 针对派系 / For Faction | 说明 / Description |

## 派系巡逻与行为模式 / Faction Patrol & Behavior

了解每个派系的巡逻和行为模式可以帮助你更好地避战或设伏 / Understanding each faction's patrol and behavior patterns helps you avoid or ambush effectively.

### 虫族行为 / Terminid Behavior

| 行为 / Behavior | 说明 / Description | 应对 / Counter |
|------|------|------|
| **巡逻队 / Patrols** | 3-8只虫一组，沿固定路线巡逻——战士虫在前，工兵在后 / 3-8 bugs on fixed patrol routes — Warriors front, Scavengers rear | 蹲伏绕开——不要惊动。一旦战斗打响，附近巡逻队全部涌来 / Crouch and bypass — once engaged, nearby patrols all converge |
| **增援呼叫 / Reinforcements** | 工兵/指挥虫会释放信息素呼叫增援——地面会冒出新的虫洞 / Scavengers/Commanders release pheromones — new bug holes open in ground | **看到信息素立刻击杀释放者——阻止增援** / **Kill the caller immediately to prevent reinforcements** |
| **虫巢警戒 / Nest Alert** | 当你靠近虫巢时，所有巢穴内的虫会激活并涌出 / All bugs in nest activate and swarm when you approach | 远程用空袭/手榴弹清理巢穴——不要靠近 / Clear nests from range with airstrikes/grenades |
| **追击距离 / Chase Range** | 虫族追击距离极远——会追你到地图另一端 / Bugs chase for very long distances — will follow across the map | 跑到开阔地用空袭清掉追击者 / Lead them into open area and airstrike |

### 机器人行为 / Automaton Behavior

| 行为 / Behavior | 说明 / Description | 应对 / Counter |
|------|------|------|
| **巡逻队 / Patrols** | 3-5台机器人——步兵+侦察兵组合。侦察兵会扫描周围 / 3-5 bots — Troopers + Scouts. Scouts scan the area | **优先击杀侦察兵**——它会呼叫增援 / **Kill Scouts first** — they call reinforcements |
| **射击精度 / Accuracy** | 机器人射击精度极高——中远距离也能命中 / Bots have very high accuracy — can hit at medium-long range | 永远利用掩体——不要暴露在开阔地超过3秒 / Always use cover — don't expose >3 seconds in open |
| **增援运输 / Dropship Reinforcements** | 一旦被侦查兵发现或警报响起——运输船会在30秒内抵达空投增援 / Once alerted by Scout or alarm — dropship arrives in 30s with reinforcements | **在运输船着陆前击落它**——需要大量火力 / **Shoot the dropship before landing** — needs significant firepower |
| **炮塔网络 / Turret Network** | 机器人基地周围有多个固定炮塔——有警戒范围 / Factory compounds have multiple fixed turrets with detection cones | 从远距离用轨道打击摧毁炮塔后推进 / Destroy turrets with orbital strikes from range before advancing |
| **EMP攻击 / EMP Attacks** | 某些机器人可发射EMP弹——暂时瘫痪HUD和战略配备 / Certain bots fire EMP rounds — temporarily disables HUD and stratagems | 被EMP击中后立刻找掩体躲避 / Take cover immediately when hit by EMP |

### 光能者行为 / Illuminate Behavior

| 行为 / Behavior | 说明 / Description | 应对 / Counter |
|------|------|------|
| **巡逻队 / Patrols** | 2-4个信徒+1个巡察者——巡察者负责指挥 / 2-4 Voteless + 1 Overseer — Overseer commands the group | 先杀巡察者——信徒没有巡察者会变得无序 / Kill Overseer first — Voteless become disorganized without leadership |
| **护盾机制 / Shield Mechanism** | 所有光能者单位都有能量护盾——护盾恢复极快 / All Illuminate units have energy shields — shields regenerate quickly | 持续火力压制——不要让护盾有机会恢复 / Sustained fire — don't let shields regenerate |
| **传送 / Teleportation** | 光能者可以短距离传送——尤其是被集火时 / Illuminate can short-range teleport — especially when focus-fired | 传送后立刻重新定位目标——不要追 / Re-acquire target after teleport — don't chase |
| **召唤传送门 / Portal Summon** | 大型光能者单位可以召唤增援传送门——不断刷小兵 / Large units can summon reinforcement portals — continuously spawn small units | **优先摧毁传送门**——用500kg或轨道激光 / **Destroy portal first** — with 500kg or Orbital Laser |

| 场景 / Scenario | 单位组合 / Composition | 应对策略 / Strategy |
|------|------|------|
| **虫族冲锋队 / Bug Rush** | 8-15战士 + 2-3冲锋虫 + 跳跃虫侧翼 / 8-15 Warriors + 2-3 Chargers + Hunters flanking | 用集束炸弹清小兵→无后坐力炮杀冲锋虫→霰弹清理残余 / Cluster bomb the swarm → Recoilless the Chargers → shotgun clean-up |
