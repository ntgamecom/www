# 任务与成长 / Missions & Progression

从低级到高级，从新手到老兵——本章涵盖所有任务类型、难度等级、飞船升级和进度系统 / From green recruit to grizzled veteran — this chapter covers all mission types, difficulty levels, ship upgrades, and progression systems.

---

## 任务类型 / Mission Types

### 主要任务目标 / Primary Objectives

| 任务类型 / Mission Type | 说明 / Description | 典型操作 / Typical Actions |
|------|------|------|
| **摧毁 / Destroy** | 摧毁指定目标——巢穴、工厂、传送门 / Destroy specified targets — nests, factories, portals | 用手榴弹/空袭/战略配备摧毁目标点 / Use grenades/airstrikes/stratagems on marked points |
| **激活 / Activate** | 启动终端、激活开关、上传数据 / Activate terminals, toggle switches, upload data | 在目标点操作终端→防御直到上传完成 / Operate terminal → defend until upload completes |
| **护卫 / Escort** | 护送重要NPC或物资至撤离点 / Escort VIP or cargo to extraction | 清理沿途敌人→保护目标到达目的地 / Clear path → protect target until destination |
| **清除 / Eliminate** | 击杀特定精英单位或指定数量的敌人 / Kill specific elite units or X number of enemies | 定位目标→全队集火→确认击杀 / Locate → focus fire → confirm kill |
| **收集 / Gather** | 在地图上收集指定数量的物品 / Collect X number of items across the map | 扫描标记位置→拾取→集齐后呼叫撤离 / Scan marked locations → pick up → extract when collected |

### 次要目标 / Side Objectives

| 目标 / Objective | 奖励 / Reward | 说明 / Description |
|------|------|------|
| **雷达站 / Radar Station** | 全图标记所有兴趣点 / Reveal all points of interest on map | 激活后可看到全图未探索位置 / Reveals unexplored locations when activated |
| **空中优势 / Air Superiority** | 击落运输船/飞行单位奖励 / Bonus for shooting down dropships/flyers | 在机器人地图上价值最高 / Highest value on Automaton maps |
| **样本容器 / Sample Container** | 额外经验样本 / Bonus XP samples | 藏在特定位置——仔细搜索 / Hidden in specific locations — search carefully |
| **寄生虫/侦查兵 / Parasite/Scout** | 经验奖励 / XP bonus | 在虫族/机器人地图上击杀特定寄生单位 / Kill specific parasitic units on bug/bot maps |

## 难度等级 / Difficulty Levels

| 难度 / Difficulty | 等级 / Tier | 推荐等级 / Player Level | 特点 / Features |
|:----:|:----:|:----:|------|
| **平凡 / Trivial** | 1 | 1+ | 仅小型敌人，无大型单位 / Small enemies only, no large units |
| **简单 / Easy** | 2 | 1+ | 少量中型单位 / Few medium units |
| **中等 / Medium** | 3 | 5+ | 开始出现精英单位 / Elites begin appearing |
| **挑战 / Challenging** | 4 | 10+ | 精英单位增多，任务目标增加 / More elites, more objectives |
| **困难 / Hard** | 5 | 15+ | 首次出现大型单位 / Large units first appear |
| **极限 / Extreme** | 6 | 20+ | 大型单位+增援频率大幅提高 / Large units + significantly increased reinforcement rate |
| **自杀 / Suicide** | 7 | 25+ | **友军伤害增加**+敌人血量提升 / **Increased friendly fire** + enemy HP buff |
| **不可能 / Impossible** | 8 | 30+ | 敌人密度极高，建议满配团队 / Extreme enemy density, full squad recommended |
| **地狱 / Hell** | 9 | 40+ | 全图都是精英和大型单位——配合与策略的极限考验 / Map-wide elites and large units — ultimate test of teamwork and strategy |
| **绝地 / Helldive** | 10 | 50+ | **最高难度**——仅推荐有默契的满编团队挑战 / **Maximum difficulty** — for coordinated full squads only |

### 难度提升的影响 / Difficulty Scaling

| 随难度提升 / Scales With Difficulty | 详细变化 / Details |
|------|------|
| **敌人数量 / Enemy Count** | 巡逻队更大更频繁，增援更多 / Patrols are larger and more frequent, more reinforcements |
| **单位类型 / Unit Types** | 新单位类型在高难度解锁 / New unit types unlock at higher difficulties |
| **敌人HP / Enemy HP** | 高难度敌人有血量加成 / Higher HP modifiers at higher difficulties |
| **任务目标 / Objectives** | 更多主要目标和次要目标 / More main and side objectives |
| **样本数量 / Sample Count** | 更多高等级样本出现 / Higher-tier samples appear more frequently |
| **友军伤害 / Friendly Fire** | 高难度下友军伤害增加（?）/ Friendly fire damage increases (?) |

> **建议**：不要跨太多级挑战——在你能稳定通关的难度上反复打，积攒样本升级飞船后再尝试更高难度 / **Tip**: Don't skip too many difficulty levels — farm at a comfortable difficulty, upgrade your ship, then push higher.

## 飞船升级 / Ship Upgrades

在飞船中枢使用样本和资金升级舰船模块——每个升级提供战术优势 / Use samples and funds at the Ship Management terminal to upgrade ship modules — each upgrade provides tactical advantages.

### 升级模块 / Upgrade Modules

| 模块 / Module | 升级效果 / Effect | 所需材料 / Materials |
|------|------|------|
| **爱国者管理中心 / Patriot Administration Center** | 减少战略配备冷却时间 / Reduce stratagem cooldown | 绿/橙/紫样本 + 资金 / Green/Orange/Purple samples + funds |
| **轨道炮 / Orbital Cannons** | 增加轨道攻击的威力和精度 / Increase orbital strike damage and accuracy | 绿/橙/紫样本 + 资金 / Green/Orange/Purple samples + funds |
| **空袭系统 / Airstrike System** | 减少空袭冷却，增加覆盖范围 / Reduce airstrike cooldown, increase coverage | 绿/橙/紫样本 + 资金 / Green/Orange/Purple samples + funds |
| **增援系统 / Reinforce System** | 增加初始增援次数，减少增援冷却 / Increase starting reinforcements, reduce reinforce cooldown | 绿/橙/紫样本 + 资金 / Green/Orange/Purple samples + funds |
| **实验室 / Lab** | 解锁新战略配备和高等级装备 / Unlock new stratagems and high-tier gear | 绿/橙/紫样本 + 资金 / Green/Orange/Purple samples + funds |
| **工程部 / Engineering Bay** | 减少支援武器冷却，提升装备性能 / Reduce support weapon cooldown, improve gear performance | 绿/橙/紫样本 + 资金 / Green/Orange/Purple samples + funds |

### 样本类型 / Sample Types

| 样本 / Sample | 颜色 / Color | 主要出现难度 / Main Difficulty | 用途 / Use |
|------|:----:|:----:|------|
| **普通样本 / Common Sample** | 绿色 / Green | 1-4 | 基础升级 / Basic upgrades |
| **稀有样本 / Rare Sample** | 橙色 / Orange | 3-6 | 中级升级 / Mid-tier upgrades |
| **超级样本 / Super Sample** | 紫色 / Purple | 7+ | **高级升级——高难度专属** / **High-tier upgrades — high difficulty only** |

> **样本策略**：每个队员都捡样本。撤离时只要有一个活着的队友登船——全队收集的样本都会保留 / **Sample Strategy**: Every squad member should pick up samples. As long as one teammate extracts alive — ALL samples collected by the squad are kept.

## 星球战争 / Galactic War

| 系统 / System | 说明 / Description |
|------|------|
| **社区进度 / Community Progress** | 所有玩家在同一张银河地图上推进战线——每个星球的解放进度是所有玩家共享的 / All players advance the same galactic front — every planet's liberation progress is shared by the entire community |
| **主要命令 / Major Orders** | 每数天发布一次全局任务——全服玩家共同完成目标领取奖励 / Global objectives released every few days — the entire server works together for rewards |
| **防御战 / Defense Campaigns** | 当派系反攻时触发防御事件——需要在限定时间内守住星球 / Triggered when factions counterattack — defend the planet within a time limit |
| **解放率 / Liberation Rate** | 玩家完成的每局任务贡献解放率——玩家越多解放越快 / Each mission completed contributes to the liberation rate — more players = faster liberation |

> **参与星球战争**：不需要额外操作——只要在任何星球完成任何任务，都自动为解放进度做贡献 / **Participating**: No extra effort needed — completing any mission on any planet automatically contributes to the liberation progress.
