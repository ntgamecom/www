## 欢迎漂泊者 / Welcome Rover

鸣潮是一款内容丰富的开放世界动作RPG。本章将帮助你从零开始，理解核心系统、资源管理和每日流程，快速踏上旅途。

Wuthering Waves is a content-rich open-world action RPG. This chapter helps you start from zero, understand core systems, resource management, and daily routines.

## 开局指引 / Starting Guide

游戏开始后，你将控制漂泊者在实验室中苏醒。前期跟随主线任务推进，解锁以下核心功能：

After starting, you control the Rover awakening in a lab. Follow the main story quests early on to unlock these core features:

- 主线第一章：解锁战斗基础、角色升级和回声系统
- 主线第二章：解锁今州城区域和部分支线任务
- Union等级5级：解锁抽卡系统（唤取）
- Union等级10级：解锁周本首领
- Union等级15级：解锁联机系统
- Union等级20级：解锁深渊之塔（逆境深塔）部分层数

- Act 1: Unlock basic combat, character upgrades, and Echo system
- Act 2: Unlock Jinzhou area and some side quests
- Union Level 5: Unlock gacha system (Convene)
- Union Level 10: Unlock weekly bosses
- Union Level 15: Unlock co-op system
- Union Level 20: Unlock Tower of Adversity (partial)

建议优先完成主线至解锁所有日常功能，再回头探索世界和完成支线任务。

Priority: finish the main story until all daily features are unlocked, then explore the world and complete side quests.

## Union等级系统 / Union Level System

Union等级相当于玩家的冒险等级，通过消耗体力获取经验，以及完成任务和活动来提升。Union等级决定了世界等级、副本难度解锁和诸多功能的开启。

Union Level is the player's adventure rank, increased by spending Waveplates, completing quests, and participating in events. It determines world level, domain difficulty unlocks, and feature access.

| Union等级 / Level | 解锁内容 / Unlocks |
|------|------|
| 1-10 | 主线剧情、唤取系统、每日任务 / Main story, Convene, Daily quests |
| 10-20 | 周本首领、模拟训练高级难度 / Weekly bosses, Simulation Training higher tiers |
| 20-30 | 逆境深塔、无音区最高难度 / Tower of Adversity, highest difficulty Forgery Challenge |
| 30-40 | 全副本最高难度解锁、世界等级提升 / Max domain difficulty, world level increase |
| 40-50+ | 终局玩法全解锁、材料掉落最大化 / All endgame content, max material drops |

Union等级达到45级后，世界等级提升至最高，所有副本奖励达到上限。

After Union Level 45, the world level reaches its maximum and all domain rewards are capped.

## 体力系统 / Waveplate System

鸣潮的体力名为「结晶波片」，简称体力或波片。波片上限为240，每6分钟恢复1点（24小时恢复满）。波片用于领取副本奖励和野外Boss材料：

The stamina system is called Crystal Waveplate, capped at 240, regenerating 1 point every 6 minutes. Waveplate is used to claim domain rewards and field boss materials:

- **模拟训练 / Simulation Training**：角色突破材料和技能升级材料（每场消耗40-60波片）
- **无音区 / Forgery Challenge**：武器突破材料（每场消耗40-60波片）
- **角斗场 / Boss Challenge**：角色突破Boss材料（每场消耗60波片）
- **周本首领 / Weekly Bosses**：技能升级高阶材料（每场消耗60波片，每周有奖励次数上限）

- Simulation Training: Ascension and skill materials (40-60 Waveplates per run)
- Forgery Challenge: Weapon ascension materials (40-60 per run)
- Boss Challenge: Character ascension boss materials (60 per run)
- Weekly Bosses: Advanced skill materials (60 per run, limited weekly)

每天建议清空波片，优先刷取角色突破材料和技能材料。

Daily recommendation: clear your Waveplate, prioritizing character ascension and skill materials.

## 唤取系统 / Convene System

唤取即抽卡系统。鸣潮有三种主要卡池：新手唤取、常驻唤取和限时唤取。

The Convene is the gacha system. There are three main banner types: Beginner's Convene, Standard Convene, and Limited Convene.

**新手唤取 / Beginner's Convene**：消耗8折唤取券（8个道具而非10个），50抽内必出五星角色。抽完50次后卡池关闭。

Uses discounted Convene tickets (8 instead of 10), guarantees a 5-star character within 50 pulls. Banner closes after 50 pulls.

**常驻唤取 / Standard Convene**：使用常驻唤取券。五星角色包括安可、维里奈、卡卡罗、凌阳和雪豹。没有时间限制。

Uses standard Convene tickets. Available 5-star characters include Encore, Verina, Calcharo, Lingyang, and the Snow Leopard. No time limit.

**限时角色唤取 / Limited Character Convene**：使用限时唤取券。80抽保底五星，五星出现时50%概率为当期UP角色。若歪掉，下一次五星必为UP角色。保底继承至下一期限时池。

Uses limited Convene tickets. 80-pull pity for 5-star. On 5-star, 50% chance is the featured character. If lost, next 5-star is guaranteed featured. Pity carries over to next banner.

**限时武器唤取 / Limited Weapon Convene**：80抽保底，不会歪，必定获得当期UP武器。

80-pull pity, no 50/50, guarantees the featured weapon.

**唤取资源获取 / Obtaining Convene Currency**：

- 完成每日活跃度任务获得星声
- 完成主线、支线任务和世界探索
- 参与版本活动
- 深渊之塔（逆境深塔）每周结算
- 月卡（珍珠纪行）每日领取
- 直接充值购买月相

- Daily activity tasks for Astrite
- Complete main quests, side quests, and world exploration
- Participate in version events
- Tower of Adversity weekly reset
- Monthly pass (Pearl Pass) daily login
- Direct top-up for Lunite

星声是唤取的主要货币，160星声兑换1个限时唤取券。建议零氪玩家优先抽取限时角色池，每月约可获取50-70抽。

Astrite is the main gacha currency, 160 Astrite per limited Convene ticket. F2P players should prioritize limited character banners, earning approximately 50-70 pulls per month.

## 角色培养优先级 / Character Building Priority

前期资源有限，建议集中培养一支主力队伍（1-2名输出+1名辅助）：

Early resources are limited, focus on one main team (1-2 DPS + 1 support):

1. **优先提升角色等级 / Priority: Character Level**：角色等级直接影响基础属性和技能倍率。突破后提升巨大。
2. **武器等级 / Weapon Level**：武器提供基础攻击力和副属性，升级收益稳定。
3. **技能等级 / Skill Levels**：优先升级核心输出技能和共鸣解放。
4. **回声装配 / Echo Equipping**：前期主词缀正确即可，不必追求完美副词缀。

1. Character level directly affects base stats and skill multipliers.
2. Weapon level provides flat ATK and secondary stats.
3. Skill levels: prioritize core DPS skills and Resonance Liberation.
4. Echo equipping: correct main stats are sufficient early on, don't chase perfect substats.

## 便捷功能 / Convenience Features

**传送锚点 / Teleport Waypoints**：地图上有大量传送点，激活后可快速旅行。Teleport points are abundant, activate for fast travel.

**自动寻路 / Auto-Navigate**：追踪任务时，地图上显示清晰路径指引。Quest tracking shows clear paths on the map.

**材料追踪 / Material Tracking**：在角色突破界面可以直接追踪所需材料的位置。From the character ascension screen, directly track material locations.

**食材自动烹饪 / Auto-Cooking**：解锁食谱后，可一键批量制作料理。After unlocking recipes, batch-cook meals with one click.

**装备预设 / Loadout Presets**：可为角色保存多套回声和武器配置，方便切换。Save multiple Echo and weapon loadouts for quick switching.

## 日常流程 / Daily Routine

建议的每日流程（约15-20分钟）：

Recommended daily routine (about 15-20 minutes):

1. 登录领取登录奖励和月卡奖励 / Login for daily and monthly pass rewards
2. 完成每日活跃度任务（4个任务共40活跃度） / Complete 4 daily activity tasks for 40 activity points
3. 清空波片（优先技能材料本和Boss材料） / Clear Waveplate (priority: skill materials and boss materials)
4. 完成体力外的限时活动任务 / Complete limited-time event tasks
5. 探索收集素材（沿途采集） / Explore and collect materials along the way
6. 每周：周本首领3次、深渊之塔结算 / Weekly: 3 weekly boss runs, Tower of Adversity reset

坚持日常是稳定积累资源、提升战斗力的关键。

Consistent daily play is key to steadily accumulating resources and improving combat power.
