## 技能系统概述 / Skill System Overview

每位共鸣者拥有六种技能，分别对应不同的按键和触发条件。技能等级提升需要消耗特定材料，材料可通过模拟训练、周本Boss和活动获取。

Each Resonator has six skills mapped to different inputs and conditions. Skill leveling requires specific materials from Simulation Training, weekly bosses, and events.

## 六种技能 / Six Skills

**普通攻击 / Basic Attack**：角色的标准攻击连段。提升等级增加各段伤害倍率。不同武器类型的普攻模组差异巨大——迅刀攻速快、长刃范围大、佩枪可远程。The character's standard attack combo. Leveling increases damage multipliers for each hit. Different weapon types have vastly different attack patterns.

**共鸣技能 / Resonance Skill**：每个角色的标志性技能，按E键释放。冷却时间5-15秒不等，是角色的主要输出和功能来源。部分角色拥有可充能的共鸣技能（最多3层充能）。Each character's signature skill, activated with E. Cooldown ranges 5-15 seconds. Main source of damage and utility. Some characters have stackable skills (up to 3 charges).

技能示例：卡卡罗的共鸣技能是突进斩击后接后续连段；维里奈的技能是部署一个回复领域，持续为队友回血。Examples: Calcharo's skill is a dash slash with follow-up combos; Verina's skill deploys a healing field.

**共鸣解放 / Resonance Liberation**：角色大招，按R键释放。需要积攒共鸣能量（通过攻击和受击积累）。共鸣解放拥有极高的伤害倍率或强大的辅助效果，是战斗中的核心爆发手段。Character ultimate, activated with R. Requires Resonance Energy (built through attacks and taking damage). High damage multiplier or powerful support effect. Core burst tool.

共鸣解放通常附带华丽的演出动画，并在释放期间提供短暂的无敌帧。建议在敌人瘫痪或获得增伤buff时使用。

Resonance Liberation usually comes with flashy cutscene animations and brief invincibility frames. Use during enemy瘫痪 or when damage buffs are active.

**共鸣回路 / Forte Circuit**：每个角色的核心被动机制，是区别于其他角色的独特玩法。Forte Circuit通常以特殊资源条的形式出现，需要通过特定操作积累资源，消耗资源后触发强化技能或特殊状态。Each character's unique core passive that defines their playstyle. Usually appears as a special resource bar built through specific actions, consumed for enhanced skills or special states.

Forte Circuit示例 / Examples：

- 部分角色：普通攻击和共鸣技能积累特殊能量，满能量后强化下一次技能
- 部分角色：特定连击后进入特殊状态（如附魔、攻速提升、伤害增加）
- 部分角色：施放共鸣技能后获得强化普攻，持续数秒

- Some characters build special energy through basic attacks and skills, enhancing the next skill when full
- Some characters enter special states after specific combos (infusion, attack speed up, damage increase)
- Some characters gain enhanced basic attacks after using Resonance Skill, lasting several seconds

Forte Circuit是玩好一个角色的关键——理解每个角色的特殊机制，才能打出完整伤害。

Mastering Forte Circuit is key to playing a character well. Understanding each character's special mechanic is essential for full damage output.

**变奏技能 / Intro Skill**：当协奏能量充满时切换角色入场，新角色会自动释放变奏技能。变奏技能通常具有高伤害或强控制效果，且释放期间角色无敌。Activated when swapping in with full Concerto Energy. Usually has high damage or strong control, with invincibility frames during execution.

**延奏技能 / Outro Skill**：协奏能量充满时切换角色离场，退场角色自动释放延奏技能。延奏技能通常为下一个入场的角色提供增益效果（增伤、属性附魔、能量回复、冷却缩减等）。Activated when swapping out with full Concerto Energy. Typically provides buffs to the next character (damage bonus, elemental infusion, energy regen, cooldown reduction, etc.).

延奏技能是队伍连招循环的核心，合理安排切换顺序可以最大化全队输出。

Outro Skills are the core of team rotation. Planning swap order maximizes team damage output.

## 技能升级材料 / Skill Upgrade Materials

技能升级需要三类材料：

Skill leveling requires three types of materials:

**基础材料 — 模拟训练掉落 / Basic Materials — Simulation Training Drops**：

- 燃素系列 / Combust Series：用于热熔和导电角色技能
- 蒸馏系列 / Distill Series：用于冷凝和衍射角色技能
- 脉动系列 / Pulse Series：用于湮灭和气动角色技能

每个角色升级技能需要特定系列的材料，取决于角色的元素属性。

Each character requires a specific material series based on their element.

**突破材料 / Boss Materials**：角色突破所需的Boss专属材料，在技能升级的后期需求量也很大。

Boss-specific ascension materials are also needed in large quantities for later skill levels.

**周本材料 / Weekly Boss Materials**：高阶技能升级所需，每周只有3次奖励机会。需要合理规划每周使用哪个周本的奖励次数。

Required for high-level skill upgrades, limited to 3 weekly rewards. Plan which weekly boss to use each week.

**技能升级消耗 / Skill Upgrade Costs (per skill)**：

| 技能等级 / Level | 材料需求 / Material Cost | 金币 / Shell Credits |
|:----:|:----:|:----:|
| 1-6 | 基础材料 + 少量金币 | Low |
| 6-8 | 进阶材料 + Boss材料 | Medium |
| 8-10 | 高阶材料 + 周本材料 | High |

所有技能升满需要大量资源，建议优先提升核心技能。

Maxing all skills requires大量 resources. Prioritize core skills first.

## 天赋优先级 / Talent Priority

一般天赋升级顺序（因角色而异）：

General talent upgrade order (varies by character):

1. **共鸣解放 / Resonance Liberation**：通常伤害占比最高，优先升满
2. **共鸣技能 / Resonance Skill**：冷却短、使用频率高，第二优先
3. **共鸣回路 / Forte Circuit**：核心机制，根据角色特性决定优先级
4. **普通攻击 / Basic Attack**：站场主C优先，辅助角色可不升级
5. **变奏/延奏技能 / Intro/Outro Skill**：通常优先级最低，但特定角色除外

1. Resonance Liberation: highest damage contribution, priority max
2. Resonance Skill: short cooldown, high usage, second priority
3. Forte Circuit: core mechanic, priority depends on character
4. Basic Attack: priority for on-field DPS, supports can skip
5. Intro/Outro Skill: usually lowest priority except for specific characters

**示例：卡卡罗技能优先级 / Calcharo Skill Priority**：

共鸣解放 → 共鸣回路 → 共鸣技能 → 普通攻击 → 变奏/延奏

**示例：维里奈技能优先级 / Verina Skill Priority**：

共鸣技能 → 共鸣解放 → 共鸣回路 → 变奏/延奏 → 普通攻击

## 技能小贴士 / Skill Tips

**冷却管理 / Cooldown Management**：注意每个角色的技能冷却时间，合理安排输出循环避免空窗期。Track each character's cooldowns to avoid downtime in the rotation.

**能量管理 / Energy Management**：共鸣解放需要能量，部分角色可以通过技能和武器被动更快充能。Some characters can快速 charge energy through skills and weapon passives.

**取消后摇 / Animation Cancel**：部分技能的收招动画可以通过闪避或切换角色取消，提高输出效率。Some skills' recovery animations can be canceled with dodge or swap for better DPS.

**空中释放 / Aerial Use**：部分角色可在空中释放共鸣技能和共鸣解放，利用跳跃增加操作空间。Some characters can use skills and ultimates mid-air for vertical play.

**协奏能量规划 / Concerto Energy Planning**：观察队伍中每个角色的协奏能量积累速度，安排合理的切换节奏。Track each character's Concerto Energy gain rate for optimal swap rhythm.

## 技能优先级深度分析 / Deep Dive Skill Priority

**主C技能占比分析 / Main DPS Skill Damage Distribution**：

不同类型主C的技能伤害占比存在显著差异。了解主力角色的伤害分布有助于优化技能升级顺序。

Different main DPS types have notably different skill damage distributions. Understanding these helps optimize skill priority.

**共鸣解放依赖型 / Liberation-Reliant**：伤害主要集中在共鸣解放，占比可达40%-50%。此类角色优先升满共鸣解放。Example characters' damage is 40-50% from Liberation, so prioritize it first.

**普攻依赖型 / Basic Attack-Reliant**：站场时间长的持续输出角色，普攻伤害占比高，需要优先升级普通攻击。On-field sustained DPS characters have higher basic attack damage share.

**技能循环型 / Skill Rotation-Reliant**：依赖共鸣技能+共鸣回路连段的角色，需要均衡升级多个技能。Characters relying on skill combos need balanced upgrades across multiple skills.

## 技能特殊机制 / Special Skill Mechanics

**强化状态 / Enhanced State**：部分角色在满足条件后进入强化状态，期间技能效果和伤害大幅提升。Some characters enter enhanced states under conditions, greatly boosting skill effects and damage.

**连携攻击 / Linked Attacks**：特定角色组合在协奏切换时触发额外连携攻击，造成独立于技能倍率的额外伤害。Specific character combos trigger bonus linked attacks on Concerto swaps.

**蓄力机制 / Charging Mechanic**：部分技能可以长按蓄力，根据蓄力时间增加伤害、范围或附加效果。Some skills can be held to charge, increasing damage, range, or附加 effects.

**追击机制 / Pursuit Mechanic**：技能命中后自动追击敌人，确保伤害不落空。适用于机动性高的敌人。Skills that automatically pursue enemies after hitting, ensuring damage connects against mobile foes.
