## 回声系统概述 / Echo System Overview

回声（Echo）是鸣潮独有的装备系统，融合了圣遗物和怪物收集两种机制。击败幻象（TD）后有概率掉落其对应的回声。每个回声不仅提供属性加成，还可以作为主动技能在战斗中释放——召唤怪物进行攻击或施加效果。

Echoes are Wuthering Waves' unique equipment system, blending relic and monster collection mechanics. Defeating TDs has a chance to drop their corresponding Echo. Each Echo provides stat bonuses and can also be used as an active skill in combat, summoning the monster for an attack or buff effect.

## 回声基本概念 / Echo Basics

**Cost值 / Cost System**：每个回声有Cost值（1-4）。队伍中所有回声的Cost总和不能超过上限（初期12，后期可提升）。高Cost回声属性更高，但占用更多Cost。Each Echo has a Cost value (1-4). Total Cost across the team cannot exceed the cap (12 early, higher later). Higher Cost Echoes have better stats but consume more Cost.

**主属性 / Main Stat**：每个回声的主属性固定，取决于回声的类型和位置。主属性在+0时确定，随强化等级提升而增加。Fixed per Echo type and slot. Determined at +0, increases with enhancement level.

**副属性 / Substats**：回声有多个副属性词条，+0时初始0-3条，强化时每5级增加一条新副属性或强化已有属性（最多5条）。Multiple substat lines, starting at 0-3 at +0, gaining a new line or enhancing an existing one every 5 levels (max 5 lines).

**套装效果 / Sonata Effects**：装配同一套装的多件回声可以激活Sonata套装效果（2件套和5件套）。套装效果提供强大的属性加成和被动能力。Equipping multiple Echoes from the same set activates Sonata set bonuses (2-piece and 5-piece), providing powerful stat bonuses and passives.

**声骸等级 / Echo Level**：回声最高15级，消耗其他回声作为经验材料升级。每升5级增加一个随机副属性词条。Max level 15, use other Echoes as EXP materials. Every 5 levels adds a random substat.

## Cost配置指南 / Cost Allocation

标准配置 / Standard Setup：4-3-3-1-1 或 4-4-1-1-1

- **Cost 4**：2件，主属性固定为暴击率、暴击伤害或治疗加成。从世界Boss和特定精英掉落。
- **Cost 3**：2件，主属性为攻击力%、防御力%、生命值%、元素伤害加成或效率属性。从精英幻象掉落。
- **Cost 1**：2件，主属性固定为攻击力、防御力或生命值（固定数值）。从小型幻象掉落。

- Cost 4: 2 pieces, main stat: Crit Rate, Crit DMG, or Healing Bonus. From world bosses and specific elites.
- Cost 3: 2 pieces, main stat: ATK%, DEF%, HP%, Elemental DMG%, or efficiency. From elite TDs.
- Cost 1: 2 pieces, main stat: flat ATK, DEF, or HP. From small TDs.

**推荐配置 / Recommended Setup**：

主C优先暴击/暴伤头 + 元素伤害增伤杯 + 攻击沙。辅助优先治疗加成或生命/防御主属性。

DPS: Crit/Crit DMG head + Elemental DMG cup + ATK% sand. Support: Healing Bonus or HP/DEF main stats.

## 主属性推荐 / Main Stat Recommendations

**主C / Main DPS**：

| 部位 / Slot | 主属性 / Main Stat |
|:----:|:----:|
| Cost 4 | 暴击率 / Crit Rate 或 暴击伤害 / Crit DMG |
| Cost 3 | 对应元素伤害加成 / Elemental DMG% |
| Cost 1 | 攻击力% / ATK% |

暴击率与暴击伤害建议保持1:2的比例（如75%暴击率配150%暴击伤害）。

Aim for a 1:2 crit rate to crit damage ratio (e.g., 75% rate to 150% damage).

**副C / Sub-DPS**：攻击力%或元素伤害优先，配合队伍需求补充效率属性。

**辅助/治疗 / Support/Healer**：治疗加成（回血角色）、生命值%或防御力%（取决于角色的加成机制）。效率属性保证共鸣解放循环。

## 副属性优先级 / Substats Priority

输出角色副属性优先级 / DPS Substats Priority：

1. 暴击率 / Crit Rate
2. 暴击伤害 / Crit DMG
3. 攻击力% / ATK%
4. 共鸣效率 / Energy Recharge（如果循环需要）
5. 攻击力固定值 / Flat ATK
6. 其他 / Others

辅助角色副属性则优先考虑共鸣效率、生命%/防御%。

Support substats: Energy Recharge, HP%, DEF% as priority.

## 回声获取途径 / Echo Farming Routes

**世界刷取 / Field Farming**：前往该回声对应怪物的密集区域反复击杀。注意部分精英怪有固定刷新点，在传送锚点附近规划循环路线。Go to the monster's密集 spawn area and farm repeatedly. Some elites have fixed spawn points; plan a loop route near teleport waypoints.

推荐刷取地点 / Recommended Farming Locations：

- 浮星祛暗：今州城周边衍射幻象密集区
- 沉日劫明：哀声崖附近湮灭幻象
- 彻空冥雷：归墟港市导电幻象区域
- 啸谷长风：虎口山脉气动幻象点
- 熔山裂谷：虎口山脉熔岩区域
- 冻结极寒：云陵冰域幻象

**副本获取 / Domain Farming**：部分套装可通过无音区副本集中获取，效率高于野外刷取。Some sets can be farmed in Forgery Challenge domains, more efficient than field farming.

**活动获取 / Event Rewards**：版本活动常赠送指定套装的自选回声箱，是获取高品质回声的重要途径。Version events often give selectable Echo boxes for specific sets.

## 回声强化策略 / Echo Enhancement Strategy

**前期（Union 1-30）**：主属性正确即可使用，不必追求完美副词缀。优先强化主C的Cost 4和经验，Cost 3和Cost 1 +5到+10即可。

**中期（Union 30-40）**：开始筛选副属性，将资源投入好底子的回声（3-4条有效初始词条）。

**后期（Union 40+）**：追求完美毕业——全套5件正确套装，主属性正确，副属性2-3条以上有效。

**强化小贴士 / Enhancement Tips**：

- 使用+0的回声作为强化材料效率最低，先用垃圾回声强化到+5再喂给目标
- 不要一次把所有资源投入一个回声，先强化到+5/+10看看副属性成长
- 保留所有暴击率/暴击伤害主属性的Cost 4，无论套装

- Using +0 Echoes as EXP is inefficient. Level junk Echoes to +5 first before feeding.
- Don't dump all resources into one Echo. Level to +5/+10 first to check substat rolls.
- Keep all Cost 4 Echoes with Crit Rate/Crit DMG main stats regardless of set.

## 回声主动技能 / Echo Active Skill

装备回声后，在战斗中按特定键位（默认T键）可释放回声技能。高Cost回声的技能通常有更长的冷却时间和更强的效果。

Equipping an Echo gives an active skill in combat (default T key). Higher Cost Echoes generally have longer cooldowns and stronger effects.

回声技能可以灵活运用：部分召唤物可吸引仇恨，部分范围攻击可清杂，部分增益技能可辅助爆发。

Echo skills are versatile: some summons draw aggro, some AoEs clear mobs, some buff skills assist burst windows.

## 回声选择策略 / Echo Selection Strategy

**新手过渡期 / Early Game Transition (UL 1-30)**：

任意2件套Sonata效果即可，优先Cost 4暴击主属性。不需要追求5件套装效果，2+2+1混搭即可。

Any 2-piece Sonata effect is fine. Prioritize Cost 4 with Crit main stat. No need for 5-piece sets; 2+2+1 mixed works.

**中期目标 / Mid Game Goal (UL 30-45)**：

为目标角色刷齐5件正确套装。主属性正确，Cost 4和Cost 3强化到+10以上。

Farm 5-piece correct set for target characters. Correct main stats, enhance Cost 4 and Cost 3 to +10 or higher.

**后期毕业 / Late Game (UL 45+)**：

追求毕业级回声——5件完美套装，主属性全部正确，副属性2-3条以上有效词条，强化全部+15。

Chase endgame Echoes: 5-piece perfect set, all correct main stats, 2-3+ useful substats, all enhanced to +15.

## 回声存储与管理 / Echo Storage & Management

回声背包有容量上限。达到上限后无法继续拾取新回声。建议定期清理：

Echo inventory has a cap. Once full, new Echoes cannot be picked up. Regular cleanup recommended:

**保留建议 / Keep These**：
- Cost 4：暴击率、暴击伤害、治疗加成主属性
- Cost 3：元素伤害加成、共鸣效率主属性
- Cost 1：攻击力%、生命值%、防御力%主属性（仅限套装内）
- 3条以上有效初始副属性的任何套装回声

**建议分解 / Discard These**：
- 3星及以下所有回声
- 主属性不适用的高Cost回声
- 0条有效初始副属性的回声
- 长期不使用的套装回声

分解回声获得强化经验材料，用于升级高品质回声。

Dismantling Echoes grants EXP materials for upgrading higher quality Echoes.
