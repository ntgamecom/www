# 职业与技能 / Jobs & Skills

歧路旅人2中有8个基础职业和4个隐藏职业——每个角色可装备一个主职和一个副职 / Octopath Traveler II has 8 base jobs and 4 secret jobs — each character equips one main and one subjob.

---

## 基础职业 / Base Jobs

| 职业 / Job | 武器 / Weapons | 定位 / Role | 核心技能 / Key Skills |
|---|---|---|---|
| **剑士 / Warrior** | 剑/长柄 / Sword/Polearm | 物理输出 / Physical DPS | 十文字斩(单体伤害)、千裂斩(全体)/ Cross Slash, Thousand Spears |
| **舞者 / Dancer** | 短剑/扇 / Dagger/Fan | Buff+Debuff | 狮子之舞(物攻up)、孔雀之舞(魔攻up)/ Lion Dance, Peacock Dance |
| **药师 / Apothecary** | 剑/斧 / Sword/Axe | 治疗/辅助 / Heal/Support | 调和(万能制造)、治疗药剂 / Concoct, Heal |
| **商人 / Merchant** | 枪/弓 / Polearm/Bow | 金币攻击/辅助 / Gold Attack/Support | 雇佣、集资(全体金币伤害)/ Hired Help, Collect |
| **学者 / Scholar** | 剑/杖 / Sword/Staff | 魔法输出 / Magic DPS | 洞察(弱点查看)、大火球术、暴风雪 / Analyze, Fire Storm, Blizzard |
| **盗贼 / Thief** | 短剑/剑 / Dagger/Sword | 速度物理 / Fast Physical | 偷窃、暗杀、弱点突刺 / Steal, Assassinate, Weakness Stab |
| **猎人 / Hunter** | 弓/斧 / Bow/Axe | 物理+宠物 / Physical+Pet | 捕获、雨箭齐发(全体弓)/ Capture, Rain of Arrows |
| **神官 / Cleric** | 杖/枪 / Staff/Polearm | 治疗+光魔法 / Heal + Light | 群体治疗、圣光(光属性多段)/ Heal More, Holy Light |

## 隐藏职业 / Secret Jobs

4个隐藏职业需要特定条件解锁——每个都极其强力 / 4 secret jobs require specific conditions to unlock — each is extremely powerful.

### 1. 魔剑士 / Armsmaster

| 项目 / Aspect | 说明 / Description |
|---|---|
| **获取地点 / Location** | 温特兰多斯——外海洞穴 / Winterlands — Sea Cave |
| **解锁条件 / Unlock** | 收集全部6种传说武器 / Collect all 6 legendary weapons |
| **核心机制 / Mechanic** | 可装备全部6种武器——技能根据装备武器变化 / Can equip all 6 weapon types — skills change based on equipped weapon |
| **推荐角色 / Best For** | 光之战士或卡丝蒂 / Hikari or Castti |
| **强力技能 / Key Skill** | 六重粉碎(6种武器连续攻击——每下破防)/ Sixfold Strike (6 weapon hits — each breaks shield) |

### 2. 占星师 / Arcanist

| 项目 / Aspect | 说明 / Description |
|---|---|
| **获取地点 / Location** | 克里斯特兰多斯——冰封洞窟 / Crestlandos — Frozen Cave |
| **解锁条件 / Unlock** | 解谜——点亮5个魔法阵 / Puzzle — light 5 magic circles |
| **核心机制 / Mechanic** | 扩散魔法——可以给队友附魔武器属性 / Diffuse magic — can enchant allies' weapons with elements |
| **推荐角色 / Best For** | 特米诺斯或奥丝瓦尔德 / Temenos or Osvald |
| **强力技能 / Key Skill** | 元素扩散(全队武器附加元素属性)/ Elemental Diffusion (all allies' weapons gain element) |

### 3. 发明家 / Inventor

| 项目 / Aspect | 说明 / Description |
|---|---|
| **获取地点 / Location** | 布里克兰多斯——东侧 / Brightlands — East |
| **解锁条件 / Unlock** | 收集4个发明材料 / Collect 4 invention materials |
| **核心机制 / Mechanic** | 独特的"发明"技能——不消耗SP——每场战斗限用一次 / Unique "Invention" skills — no SP cost — once per battle each |
| **推荐角色 / Best For** | 帕特提欧或托罗 / Partitio or Throné |
| **强力技能 / Key Skill** | 弹簧靴(全体物理攻击+提速)/ Spring Boots (AoE physical + speed boost) |

### 4. 祈祷师 / Conjurer

| 项目 / Aspect | 说明 / Description |
|---|---|
| **获取地点 / Location** | 托托哈哈岛——遗迹 / Toto'haha — Ruins |
| **解锁条件 / Unlock** | 击败守护Boss / Defeat guardian boss |
| **核心机制 / Mechanic** | 召唤精灵——给全队附加元素效果 / Summon spirits — add elemental effects to party |
| **推荐角色 / Best For** | 阿格妮亚或卡丝蒂 / Agnea or Castti |
| **强力技能 / Key Skill** | 精灵附魔(全队武器附加指定属性)/ Spirit Enchantment (all allies' weapons gain element) |

**JP使用建议 / JP Usage Tips**:
- 每位角色先学满主职的核心技能——再练副职 / Each character should max their main job's core skills first — then level subjob
- 被动技能非常重要——学会后永久生效——即使切换职业也保留 / Passive skills are very important — permanent even after switching jobs
- 建议把全队8人的副职都练到至少学会被动技能为止 / Level all 8 characters' subjobs to at least unlock the passive
