## 隐藏职业 / Hidden Jobs

歧路旅人2有**12种职业**——8个基础+4个隐藏职业。隐藏职业需要找到**职业祠堂**——进入后击败守门Boss即可解锁该职业。隐藏职业可以给任意角色装备——作为主职业或副职业 / OT2 has **12 jobs** — 8 base + 4 hidden. Hidden jobs require finding **Job Shrines** — defeat the guardian boss to unlock. Hidden jobs can be equipped by any character as primary or secondary.

## 副职业系统 / Secondary Job System

每个角色可以装备一个**副职业**——获得副职业的所有技能(但属性和HP使用主职业的)。这是游戏最大深度所在 / Each character can equip a **Secondary Job** — gains all skills from that job (but stats/HP use primary job). This is the game's deepest system.

### 副职业技能继承规则

**基础技能**: 副职业的所有技能可用——等级继承主职业的等级(副职业等级不会单独提升)。**EX技能**: 需要完成特定角色章节后才能解锁——EX技能是角色的核心技。**辅助技能**: 副职业的辅助技能可以装备最多4个——来源不限主/副职业——这是Build深度所在 / **Basic Skills**: All skills usable — level inherited from primary job. **EX Skills**: Unlock after completing character chapters — core skills. **Support Skills**: Can equip 4 — regardless of primary/secondary source — this is build depth.

## 隐藏Boss / Secret Bosses

### 噬船之蛇 / Ship-Devouring Serpent

**位置**: 大洋中央漩涡 / Location: Ocean central whirlpool
**触发条件**: 完成商人第四章后——乘船到大洋中央——与漩涡互动 / Trigger: Complete Merchant Ch.4 — sail to ocean center — interact with whirlpool
**推荐等级**: 45+ / Recommended Level: 45+

**核心机制**: Boss有11点护盾——弱点: 剑、弓、火、冰、雷。**阶段一**: 蛇身攻击——物理AOE+毒液喷射(全体中毒)。**阶段二**: 召唤水蛇杂兵——不断补充护盾。**阶段三**: 全屏大漩涡——蓄力3回合——必须在此期间打满Break否则即死 / Core Mechanics: 11 SP — weak to Sword, Bow, Fire, Ice, Lightning. **Phase 1**: Body attacks — phys AoE + poison spit (party poison). **Phase 2**: Summon water serpents — replenish SP. **Phase 3**: Full-screen Maelstrom — 3-turn charge — must Break during or instant death.

**掉落**: **海神之剑**——剑系最强武器——水属性伤害+30%, 全属性+10 / **Drops**: **Sea God Sword** — best sword weapon — +30% water damage, +10 all stats.

### 隐藏终局Boss：虚空神 / Void God

**位置**: 所有角色第四章通关后——终局迷宫最深处 / Location: After all 8 characters' Ch.4 — deepest final dungeon
**推荐等级**: 55+ / Recommended Level: 55+

**核心机制**: 游戏的最强Boss——拥有13点护盾——弱点是全属性——但每次被击破后弱点会变换。**阶段一**: 虚空形态——使用各种属性AOE攻击。**阶段二**: 召唤分身——分身有全部技能但血量低。**阶段三**: 虚空崩坏——全屏随机属性连击——需要持续回复+无敌技能轮换。**打法**: 全队55+——全员装备隐藏职业——带满BP回复药——牧师的无敌和舞娘的全体延迟是MVP / Core Mechanics: Strongest boss — 13 SP — weak to all elements — changes weakness after each break. **Phase 1**: Void form — various elemental AoEs. **Phase 2**: Summon clones — full skill set but low HP. **Phase 3**: Void Collapse — random elemental barrage — constant heals + invincible rotation needed. **Strategy**: All lv55+ with hidden jobs — full BP recovery items — Cleric invincible + Dancer delay are MVP.

**掉落**: **虚空之书**——解锁隐藏成就"超越神"——全游戏唯一金色称号 / **Drops**: **Void Tome** — unlocks hidden achievement "Beyond God" — only golden title in game.
