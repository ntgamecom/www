Octopath Traveler II is Square Enix's HD-2D JRPG developed by Team Asano — eight travelers pursue their own destinies across the continent of Solistia. ## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **推荐首发光之战士或卡丝蒂 / Start with Hikari or Castti** | 剑士有稳定单体伤害——药师有治疗——最适合前期 / Warrior has reliable damage — Apothecary has healing — best for early game |
| **2** | **尽快收集8位角色 / Recruit All 8 ASAP** | 不用急着推主线——先去世界各地把8位主角都拉入队伍 / Don't rush main story — travel the world and recruit all 8 first |
| **3** | **活用破防系统 / Master the Break System** | 每个敌人有弱点和护盾——打弱点可以破防——破防后疯狂输出 / Every enemy has weaknesses and shields — hit weaknesses to break — then unload |
| **4** | **积攒BP合理使用 / Use Boost Wisely** | 不要每回合都把BP用光——留BP用于破防后爆发输出 / Don't dump all BP every turn — save for burst after breaking |
| **5** | **副职尽早解锁 / Unlock Subjobs Early** | 每个角色到第2章时可以接副职——大幅提升战术灵活性 / Subjobs unlock around Chapter 2 — massively increases tactical flexibility |
| **6** | **昼夜切换很重要 / Switch Day/Night Often** | 部分NPC白天和晚上的对话/互动不同——部分隐藏道具只能在特定时间获取 / Some NPCs have different interactions day vs night — some items only available at specific times |
| **7** | **存钱买好装备 / Save Money for Gear** | 装备比等级提升更重要——每个城镇的武器店值得升级 / Gear matters more than levels — weapon shops in each town are worth upgrading |
| **8** | **不要跳过支线 / Don't Skip Side Quests** | 支线任务奖励大量JP（职业点数）和稀有装备 / Side quests reward lots of JP (job points) and rare gear |

## 战斗系统详解 / Combat System

| 要素 / Element | 说明 / Description |
|---|---|
| **回合制 / Turn-Based** | 按速度顺序行动——每个角色每回合一次行动 / Act in speed order — one action per character per turn |
| **护盾点 / Shield Points (SH)** | 用弱点攻击可以减少SH——SH归零时敌人进入"破防"状态 / Hit weaknesses to reduce SH — at 0, enemy is "broken" |
| **破防状态 / Break** | 破防的敌人**跳过一回合**+**受到的伤害增加** / Broken enemies **skip a turn** + **take increased damage** |
| **增幅点 / BP** | 每回合获得1BP——最大可存到5——消耗BP增加攻击次数或技能效果 / Gain 1 BP per turn — max 5 — spend to boost attacks or skills |
| **潜力技能 / Latent Power** | 每个角色有专属的潜力条——满了可以释放强力技能 / Each character has a unique Latent gauge — unleash a powerful skill when full |
| **弱点分析 / Analyze** | 学者奥丝瓦尔德的初始技能——可以查看一个敌人的全部弱点 / Osvald's starting skill — reveals all weaknesses of one enemy |

**战斗策略核心 / Core Battle Strategy**:
```
第一回合: 学者分析弱点 + 其他角色普攻打破弱点
第二回合: 所有角色攻击弱点 —— 迅速破防
第三回合: 全队增幅爆发 —— 最大伤害输出
```
