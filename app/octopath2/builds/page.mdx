## Build构建基础 / Build Fundamentals

歧路旅人2的Build核心在**主职业+副职业+辅助技能+装备**四个维度。每个角色可以有完全不同的Build方向——取决于你如何搭配这四个要素。终局Build的伤害差距可以高达10倍——合理的Build配置是通关隐藏Boss的必要条件 / OT2's build core is **Primary Job + Secondary Job + Support Skills + Gear**. Each character can have vastly different builds depending on how you combine these four elements. Endgame build damage variance can be 10x — proper build is mandatory for hidden boss.

### 辅助技能系统 / Support Skill System

辅助技能是Build的灵魂——每个职业有4个辅助技能——角色可以装备最多4个(不限来源)。辅助技能提供被动加成——是提升战力的核心手段。**重要辅助技能推荐**: 剑士的"突破极限"(伤害上限突破9999——物法双修核心)、学者的"增强属攻"(属性攻击+30%——法系必备)、商人的"经验翻倍"(获得经验x2——刷级必备——建议全员装备)、盗贼的"偷窃几率+"(偷窃成功率+50%——刷装备用)、猎人的"先制攻击+"(战斗先制率+50%——探索安全)/ Support skills are the soul of builds — each job has 4 — characters equip up to 4 (any source). ## 强力Build推荐 / Strong Build Recommendations

### 物理核爆流 / Physical Nuke

| 角色 / Character | 主职业 / Primary | 副职业 / Secondary | 定位 / Role |
|---|---|---|---|
| 剑士 / Warrior | 剑士 / Warrior | 豪武匠 / Armsmaster | 主输出 / Main DPS |

**辅助技能**: 突破极限(BP Eater——伤害上限9999→99999)、物理攻击+50%、暴击+20%、BP回复加速(每回合BP+1)/ **Support Skills**: BP Eater, Phys ATK +50%, Crit +20%, BP Regen Up.
**装备**: 海神之剑(+30%属性伤害)、源氏之铠(全属性+15)、加速靴(速度+60——确保先手)、BP回复饰品(每回合额外BP)。**输出循环**: 前三回合积累BP(防御或普攻)→第四回合增幅普攻(3BP共4hit)破防→增幅技能(3BP)豪武匠六种武器连击——每击约15000→总计约90000伤害——单回合爆发。**推荐队友**: 舞娘(全属性Buff)、学者(破防加速+探弱点)、商人(供应BP)。**伤害上限**: 配合舞娘的Buff+商人的BP支援——单回合可突破99999——到约150000 / **Gear**: Sea God Sword (+30% element), Genji Armor (+15 all), Speed Boots (+60 SPD), BP Regen Acc. **Rotation**: First 3 turns build BP (guard/normal) → Boost normal (3BP 4 hits) break → Boost skill (3BP) Armsmaster 6-weapon combo — ~15k per hit → ~90k total. **Teammates**: Dancer (all stat buff) + Scholar (break + weakness) + Merchant (BP supply). **Max damage**: With Dancer buff + Merchant BP — single round over 99999 — up to ~150000.

### 魔法核爆流 / Magic Nuke

| 角色 / Character | 主职业 / Primary | 副职业 / Secondary | 定位 / Role |
|---|---|---|---|
| 学者 / Scholar | 学者 / Scholar | 魔导师 / Arcanist | 主输出 / Main DPS |

**辅助技能**: 突破极限(BP Eater)、增强属攻(Elemental Edge——+30%属性攻)、MP消耗减半(Half MP Cost——续航核心)、属性连击强化(Elemental Barrage——连击数翻倍)/ **Support Skills**: BP Eater, Elemental Edge (+30%), Half MP Cost, Element Barrage Up.
**装备**: 魔剑士之剑(+100ATK+100MATK——双攻)、时间之盾(全抗+25%)、智力+30饰品(法系核心)、属性暴击饰品(属性暴击率+15%)。**输出循环**: 学者用洞察探弱点→魔导师属性连击攻击2-3个弱点→破防后增幅学者大招"陨石风暴"(Meteor Storm)——约120000全属性伤害。**推荐队友**: 祈祷师(Conjurer——给武器附魔元素——增加弱点打击面)、舞娘(属性攻击Buff——额外+30%属性伤害)。**优势**: 这个Build不受敌人物理抗性影响——所有敌人都适用——而且AOE清场能力极强 / **Gear**: Magic Swordsman Blade (+100 ATK/MATK), Time Shield (+25% all resist), INT +30 acc, Element Crit acc (+15% element crit). **Rotation**: Scholar Scrutiny → Arcanist Element Barrage (2-3 weaknesses) → Boost Scholar's "Meteor Storm" — ~120000 all-element damage. **Teammates**: Conjurer (element enchant weapon — more weakness coverage) + Dancer (element ATK buff — +30% element dmg). **Advantage**: Works vs all enemies — no phys resist issues — excellent AoE.

### 无限BP循环流 / Infinite BP Loop

| 角色 / Character | 主职业 / Primary | 副职业 / Secondary | 定位 / Role |
|---|---|---|---|
| 商人 / Merchant | 商人 / Merchant | 发明家 / Inventor | BP管理 / BP battery |

**辅助技能**: BP回复加速、每回合BP+1、HP/MP自动回复、道具效果翻倍 / **Support Skills**: BP Regen Up, +1 BP/turn, Auto Heal MP, Item Effect 2x.
**装备**: BP回复饰品(每回合额外BP)、速度靴(先手给BP)、最大SP+100饰品(续航)、防御饰品(生存)。**核心玩法**: 商人技能"捐赠BP"(给队友1BP,消耗自己1BP)配合被动每回合+2BP=每回合净赚1BP+可以一直给队友供BP。发明家道具强化(Duo)可以让消耗品效果加倍——比如全体回复道具从2000变为4000。这个Build让全队每回合都有BP用——破防和爆发频率翻倍——是全队战斗力的放大器 / **Gear**: BP Regen acc, Speed Boots, +100 SP acc, DEF acc. **Core**: Merchant "Donate BP" (give 1 BP, cost 1 BP) + passive +2 BP/turn = net +1 BP/turn + infinite supply. Inventor "Duo" doubles consumable effects — party heal 2000→4000. This build enables full party BP every turn — doubles break and burst frequency — team force multiplier.

### 高速破防流 / Fast Break

| 角色 / Character | 主职业 / Primary | 副职业 / Secondary | 定位 / Role |
|---|---|---|---|
| 猎人 / Hunter | 猎人 / Hunter | 豪武匠 / Armsmaster | 破防专家 / Break specialist |

**辅助技能**: 行动速度+30%、先制攻击几率+50%、破防时额外伤害(+50%破防后伤害)、BP回复加速 / **Support Skills**: SPD +30%, Preemptive Rate +50%, Break Bonus DMG (+50% after break), BP Regen Up.
**装备**: 加速靴(速度+60)、速度+40饰品、先制攻击饰品(战斗必先手)、破防伤害饰品(破防后+30%伤)。**核心玩法**: 猎人速度最快——第一回合几乎必先动。潜力技能"无限弓矢"(Latent: Endless Arrow)——对单目标连续射击直到破防——配合豪武匠的全武器克制可以1回合破防任何敌人。破防后给物理核爆流剑士制造输出窗口——是全队进攻节奏的起爆器 / **Gear**: Speed Boots (+60), SPD+40 acc, Preemptive acc (guaranteed first strike), Break DMG acc (+30% after break). **Core gameplay**: Hunter is fastest — almost guaranteed first move. Latent "Endless Arrow" — shoot until break — combined with Armsmaster's all-weapon coverage — can break any enemy in 1 turn. After break → Phys Nuke Warrior kills. This build is the team's tempo starter.

## 终局Build策略 / Endgame Build Strategy

**隐藏Boss战推荐阵容**: 物理核爆(剑士→主输出)+魔法核爆(学者→副输出+探弱点)+无限BP(商人→BP电池)+高速破防(猎人→破防+辅助)。这个组合可以应对隐藏Boss虚空神的全部8个阶段。**隐藏Boss虚空神攻略**: 1-4阶段是属性阶段(各弱一种属性)——魔法核爆学者应对——5-6阶段物理阶段——物理核爆剑士应对——7阶段混合——需要两个DPS协作——8阶段真身——全力爆发。**通用策略**: 破防后不要急着爆发——先用舞娘或学者上满Buff——一回合破防+一回合Buff+一回合爆发——虽然多一回合但伤害翻倍。**装备强化**: 终局前把主力武器强化到最高等级——伤害提升约50%。辅助技能全解锁: 8个角色40个辅助技能中选择最优的4个——需要仔细搭配。**道具储备**: BP回复药x10(打隐藏Boss至少需要)、全回复药x5、Buff药x5、复活药x3——这是打隐藏Boss的最低消耗量 / **Hidden Boss Party**: Phys Nuke + Magic Nuke + Infinite BP + Fast Break. **Void God strategy**: Phases 1-4 element (Magic Nuke), 5-6 phys (Phys Nuke), 7 hybrid (both), 8 true form (all out). **General rule**: Don't burst immediately after break — use Buff turn first — 1 break + 1 buff + 1 burst = double damage. **Max gear**: Upgrade weapons to max — ~50% damage boost. **Items**: BP recovery x10, Full heal x5, Buff item x5, Revive x3 — minimum for hidden boss.
