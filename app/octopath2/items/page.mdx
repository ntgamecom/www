## 隐藏Boss / Hidden Superboss

游戏最终隐藏Boss需要完成以下前置条件 / The hidden superboss requires these prerequisites:

| 条件 / Requirement | 说明 / Description |
|---|---|
| **完成全部8位角色第4章 / All 8 Ch4 completed** | 8人故事全部通关 / All 8 stories cleared |
| **收集4个隐藏职业 / All 4 secret jobs** | 魔剑士+占星师+发明家+祈祷师 / Armsmaster + Arcanist + Inventor + Conjurer |
| **收集古代石板 / Ancient Tablets** | 每个角色章节结束后去特定地点收集 / Collect specific locations after each character's story |
| **推荐等级 / Recommended Level** | 50+——全队最佳装备 / 50+ — full BiS gear |

**隐藏Boss战斗建议 / Superboss Tips**:
- 带卡丝蒂(调和MVP)和特米诺斯(光魔法破盾)/ Bring Castti (Concoct MVP) and Temenos (light break)
- BP恢复石榴多带——多准备几个 / Stock plenty of BP recovery石榴
- 配好装备——尤其是异常状态抵抗 / Gear properly — especially status resistance
- 准备打30分钟以上的持久战 / Prepare for a 30+ minute battle
