# 角色故事 / Character Stories

八位主角各有4章独立故事——本指南提供每一章的要点和攻略建议 / Each of the 8 characters has 4 independent chapters — this guide provides key points and tips for each.

---

## 光 / Ochette (猎人)

| 章节 / Ch | 推荐等级 / Lv | 地点 / Location | BOSS / Boss | 攻略要点 / Tips |
|---|---|---|---|---|
| **第1章** | 1-4 | 托托哈哈岛 / Toto'haha | 大型螳螂 / Giant Mantis | 教程关卡——学习抓怪物能力 / Tutorial — learn Capture mechanic |
| **第2章** | 15-20 | 克里斯特兰多斯 / Crestlandos | 雪狼王 / Snow Wolf King | 注意用火属性攻击——弱点火 / Use fire attacks — weak to fire |
| **第3章** | 25-30 | 温特兰多斯 / Winterlands | 冰霜巨人 / Frost Giant | 推荐带神官治疗——冰抗装备 / Bring Cleric for heals — ice resist gear |
| **第4章** | 35-40 | 托托哈哈岛深处 / Deep Toto'haha | 暗黑之兽 / Dark Beast | 最终Boss——多带BP恢复道具 / Final boss — bring BP recovery items |

**光的能力 / Ochette's Skills**:
- 白天可以**捕获**怪物作为宠物——战斗中召唤 / Capture monsters by day — summon in battle
- 捕获的怪物有使用次数限制——用完了需要重新抓 / Captured monsters have limited uses — recapture when depleted
- 某些强力怪物需要在特定区域/特定天气抓 / Some powerful monsters require specific areas/weather

## 特米诺斯 / Temenos (神官)

| 章节 / Ch | 推荐等级 / Lv | 地点 / Location | BOSS / Boss | 攻略要点 / Tips |
|---|---|---|---|---|
| **第1章** | 1-4 | 克里斯特兰多斯 / Crestlandos | 异端教徒 / Heretic | 光属性魔法是核心 / Light magic is key |
| **第2章** | 15-20 | 布里克兰多斯 / Brightlands | 审判官 / Inquisitor | 暗属性抗性低——特米诺斯的"圣光"三连发 / Weak to dark — use Temenos's "Holy Light" x3 |
| **第3章** | 25-30 | 克里斯特兰多斯(圣堂)/ Crestlandos | 大主教 / Archbishop | BOSS会放群体沉默——准备好解沉默药 / Boss uses AoE silence — bring cure items |
| **第4章** | 35-40 | 圣堂深处 / Deep Sanctum | 最终Boss | 特米诺斯的故事是最精彩的推理剧之一——注意细节 / Temenos's story is one of the best murder mysteries — pay attention to details |

**特米诺斯的特点 / Temenos Notes**:
- 潜力技能: 他的魔法可以额外多打一次(3连发→6连发)/ Latent: his magic hits an extra time (3 hits → 6 hits)
- 非常适合破盾——光属性破大量敌人的弱点 / Excellent for shield breaking — light element breaks many enemies
- 装备副职学者后——魔法输出极高 / Subjob Scholar = massive magic DPS
