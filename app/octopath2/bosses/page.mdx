## 章节Boss攻略 / Chapter Boss Guides

### 剑士第一章Boss：戈弗雷 / Warrior Ch.1 Boss: Godfrey

**位置**: 剑士的城堡——训练场 / Location: Warrior's Castle — Training Grounds
**推荐等级**: 6-8 / Recommended Level: 6-8
**护盾**: 4 / Shield Points: 4
**弱点**: 枪、短剑、弓、风 / Weaknesses: Lance, Dagger, Bow, Wind

**阶段一——训练模式**: 戈弗雷使用**斩击**(单体物理)+**防御姿态**(自身防御上升)。打法: 用弱点武器削盾——3BP增幅普攻4连击可以1回合破防。破防后用增幅剑技输出——重复即可 / Phase 1 — Training: Uses **Slash** (single phys) + **Defense Stance** (DEF up). Strategy: Use weak weapons to reduce SP — 3BP boost normal attack breaks in 1 turn. Boost sword skill during break — repeat.

**阶段二——认真模式**: 血量低于50%时戈弗雷的**全力斩**——蓄力1回合后的高伤害单体——被破防可以打断。增加**反击姿态**——反击物理攻击——用属性攻击或短剑规避 / Phase 2 — Serious: Below 50% HP — **Full Power Slash** — 1-turn charge high damage — interrupt with Break. Adds **Counter Stance** — counters phys attacks — use element or dagger to avoid.

### 学者第一章Boss：守护者 / Scholar Ch.1 Boss: The Guardian

**位置**: 学院地下遗迹 / Location: Academy Underground Ruins
**推荐等级**: 7-9 / Level: 7-9
**护盾**: 5 / Shield: 5
**弱点**: 剑、斧、火、光 / Weaknesses: Sword, Axe, Fire, Light

**核心机制**: 守护者使用**属性反射盾**——随机反射一种属性——对该属性的攻击完全免疫并治愈Boss。破防可以消除反射盾。**召唤晶石**——召唤3颗元素晶石——每颗晶石提供Boss抗性——优先击破。打法: 先清理晶石→破防→全力输出——重复 / Core Mechanics: **Element Reflect** — randomly reflects one element — immune to that element + heals boss. Break removes reflect. **Summon Crystals** — 3 element crystals — each gives boss resistance — destroy first. Strategy: Clear crystals → Break → burst — repeat.

### 终章Boss：最终之影 / Final Chapter Boss: Final Shadow

**位置**: 终章迷宫最深处 / Location: Deepest Final Dungeon
**推荐等级**: 45-50 / Level: 45-50
**护盾**: 8 / Shield: 8
**弱点**: 剑、短剑、斧、火、冰、雷(第一阶段) / Sword, Dagger, Axe, Fire, Ice, Lightning (Phase 1)

**阶段一**: 最终之影使用**暗影连击**(单体3连物理)+**暗影爆发**(全体暗属性)+**召唤暗影**(召唤2个杂兵——必须清理否则Boss会吞噬回复大量HP)。核心: 优先清理召唤物——暗影爆发前全员满血 / Phase 1: **Shadow Combo** (3-hit phys) + **Shadow Burst** (party dark) + **Summon Shadow** (2 adds — must clear or boss consumes for HP). Priority: Clear adds first — full HP before Shadow Burst.

**阶段二**: 血量低于40%——进入**绝望形态**——每回合末释放**绝望波动**(全体暗属性+随机减益)——增加**即死凝视**(单体即死——牧师的无敌可挡)。核心: 破防节奏至关重要——绝望波动可以通过破防跳过。即死凝视出现前确保牧师有潜力值——无敌可以完全免疫 / Phase 2: Below 40% HP — **Despair Form** — **Despair Wave** (party dark + random debuff) at end of each turn — adds **Death Gaze** (single instant death — Cleric invincible blocks). Core: Break rhythm critical — Break skips Despair Wave. Keep Cleric latent ready before Death Gaze — invincible completely blocks.

## 隐藏职业Boss / Hidden Job Bosses

### 豪武匠的试炼 / Armsmaster Trial

**位置**: 终章桥前左侧岔路 / Location: Left path before final bridge
**推荐等级**: 35-40 / Level: 35-40
**护盾**: 7 / Shield: 7
**弱点**: 全部6种武器——每回合变换一种弱点——不在当前弱点上的攻击不削盾 / Weak: All 6 weapons — changes each turn — only current weakness's attacks reduce SP

**核心机制**: Boss每回合切换一种武器弱点——使用其他武器完全不削盾。**策略**: 使用豪武匠(6种武器全克制)或者多个不同武器的角色——覆盖所有武器类型。**攻击模式**: Boss用你当前装备武器的对应技能攻击——如果你装备剑Boss就用剑技。裸装打他不会用技能只会普攻——但极脆皮——也可以直接强度碾压 / Core: Changes weapon weakness each turn — other weapons don't reduce SP. **Strategy**: Use Armsmaster (all 6 weapons) or multiple characters covering all weapons. **Attack pattern**: Boss uses skill matching your equipped weapon — unequipped = basic attacks only but easy to overlevel.

### 魔导师的试炼 / Arcanist Trial

**位置**: 流沙沙漠隐藏祠堂 / Location: Sandflow Desert hidden shrine
**推荐等级**: 38-43 / Level: 38-43
**护盾**: 6 / Shield: 6
**弱点**: 剑、短剑、斧、火、冰、雷、风、光、暗——全部弱点——但每破防一次弱点减少2个 / All elements — but each break removes 2 weaknesses permanently

**核心机制**: Boss每次被破防后永久失去2个弱点——护盾也增加2点——战斗会越来越难。**策略**: 前几次破防全力爆发——争取尽早结束战斗。随着弱点变少、护盾变厚——战斗压力递增——这是典型的"时间限制"Boss。/ Core: Each Break permanently removes 2 weaknesses AND increases SP by 2 — fight gets harder over time. **Strategy**: Max burst during early breaks — finish fight early. Pressure increases as weaknesses shrink and SP grows — classic "time limit" boss.

## Boss战通用策略 / General Boss Strategy

1. **探弱点优先**——学者第一回合用洞察——确定全部弱点再进攻 / Reveal first — Scholar Scrutiny turn 1 — know all weaknesses before attacking
2. **Boss蓄力时破防**——大部分Boss的蓄力大招被Break打断——这是最重要的战术 / Break during boss charge — most boss ults are interrupted by Break — key tactic
3. **BP药水不要省**——Boss战至少带5个BP回复药——爆发期连续增幅输出 / Don't save BP items — carry 5+ BP recovery — chain Boost during burst windows
4. **装备速度调整**——让牧师和辅助最先行动、输出之后行动——保证治疗先出手 / Speed tune — healer/support fastest, DPS after — ensures heals before damage
5. **增益减益全上**——舞娘的全属性Buff、学者的防御Debuff——叠加后伤害翻倍 / Full buff/debuff — Dancer all-stat buff + Scholar DEF debuff — doubles damage
6. **保存潜力值给关键技能**——牧师的无敌留给Boss大招回合、剑士的破留给蓄力打断 / Save Latent for key moments — Cleric invincible for boss ult, Warrior break for charge interrupt
7. **全队等级差不超过5**——等级低的角色容易在AOE中阵亡 / Keep party level spread within 5 — underleveled characters die to AoE easily
8. **属性装备对应Boss弱点属性**——Boss弱火就堆火伤装备——可以翻倍输出 / Gear element to boss weakness — boss weak to Fire = stack Fire gear for 2x damage
