# 新手入门 / Beginner Guide

欢迎来到索里斯希亚大陆！本指南带你从零开始探索歧路旅人2的世界 / Welcome to Solistia! This guide takes you from zero to hero.

---

## 角色推荐 / Starter Character Advice

| 推荐度 / Rating | 角色 / Character | 理由 / Reason |
|---|---|---|
| **⭐⭐⭐⭐⭐** | **光之战士 / Hikari (剑士/Warrior)** | 最强的物理输出——单体伤害高——学习"十文字斩"后可以轻松处理前期所有战斗 / Best physical DPS — high single-target damage — "Cross Slash" handles early game |
| **⭐⭐⭐⭐⭐** | **卡丝蒂 / Castti (药师/Apothecary)** | 唯一开局有治疗的角色——调和(Concoct)技能极其灵活——可以群体治疗+解除异常+破盾 / Only starting character with healing — Concoct is extremely versatile — AoE heal + status cure + shield break |
| **⭐⭐⭐⭐** | **特米诺斯 / Temenos (神官/Cleric)** | 光属性魔法可以破大量敌人的弱点——群体治疗——非常适合辅助 / Light magic breaks many enemies' weaknesses — AoE heal — great support |
| **⭐⭐⭐⭐** | **托罗 / Throné (盗贼/Thief)** | 速度快——偷窃技能可以获取大量道具——开局省很多钱 / Fast — Steal gets lots of items — saves tons of money early |

> 提示：开局选谁都可以——8位主角很快就能全部入队——不用太纠结 / Tip: Pick anyone — all 8 join quickly — don't overthink it

## 开局路线建议 / Starting Path

| 阶段 / Phase | 目标 / Goal | 建议等级 / Lv |
|---|---|---|
| **第1阶段 / Phase 1** | 选一位主角——完成她的第一章 / Pick a starter — complete their Chapter 1 | 1-5 |
| **第2阶段 / Phase 2** | 走遍世界——收集全部8位主角 / Travel the world — recruit all 8 characters | 5-10 |
| **第3阶段 / Phase 3** | 做支线攒JP和装备——每个主城看看 / Do side quests for JP and gear — visit every town | 10-15 |
| **第4阶段 / Phase 4** | 选择一位主角推第2章——以此类推 / Pick one character and do Chapter 2 — rinse and repeat | 15-25 |
| **第5阶段 / Phase 5** | 第3章+第4章——装备隐藏职业——挑战隐藏Boss / Chapters 3+4 — get secret jobs — challenge hidden boss | 25-50 |

**收集所有角色的最佳路线 / Best Route to Recruit All**:
```
光(初始/赫巴兰多斯东) → 卡丝蒂(赫巴兰多斯西)
→ 托罗(利夫兰多斯东) → 帕特提欧(温特兰多斯西)
→ 奥丝瓦尔德(利夫兰多斯西) → 阿格妮亚(利夫兰多斯)
→ 特米诺斯(克里斯特兰多斯) → 光(Ochette)(托托哈哈岛)
```

## 升级与职业点 / Leveling & JP

| 项目 / Item | 说明 / Description |
|---|---|
| **经验值 / XP** | 战斗获得——提升角色等级——增加HP/SP/属性 / Gain from battles — raises character level — increases HP/SP/stats |
| **职业点 / JP** | 战斗获得——用于学习职业技能——每个职业的技能树独立 / Gain from battles — used to learn job skills — each job has its own skill tree |
| **副职经验 / Subjob XP** | 副职也获得经验——除非已经学会该职业的所有技能 / Subjobs also earn JP — unless all skills are already learned |
| **等级上限 / Level Cap** | 角色等级最高99——职业等级不影响角色等级 / Character level max 99 — job level is separate from character level |

**刷JP技巧 / JP Farming Tips**:
- 前期多做支线任务——每个支线奖励大量JP / Do side quests early — each gives substantial JP
- 装备"JP增幅"饰品——某些隐藏装备可以提升JP获取 / Equip "JP Boost" accessories — some hidden gear increases JP gain
- 遇到高经验怪物群——用增幅BP最大化每场战斗收益 / When facing tough groups — boost to maximize per-battle gains

## 地图能力详解 / Path Actions

每位主角有2种地图能力——白天和晚上不同 / Each character has 2 path actions — different for day and night.

| 角色 / Character | 白天 / Day | 晚上 / Night | 说明 / Description |
|---|---|---|---|
| **光(Ochette) / Hunter** | 抓怪物 / Capture | 挑衅 / Provoke | 白天抓怪物当宠物——晚上和NPC单挑 / Capture monsters as pets by day — duel NPCs at night |
| **卡丝蒂 / Apothecary** | 询问 / Inquire | 调查 / Soothe | 白天打听情报——晚上检查NPC状态 / Gather info by day — check NPC status at night |
| **特米诺斯 / Cleric** | 引导 / Guide | 打探 / Coerce | 白天让NPC跟随你——晚上用武力获取信息 / Have NPC follow you by day — force info at night |
| **奥丝瓦尔德 / Scholar** | 调查 / Scrutinize | 窃取 / Steal | 白天调查NPC获得信息——晚上偷东西 / Investigate NPCs for info by day — steal items at night |
| **帕特提欧 / Merchant** | 雇佣 / Hire | 购买 / Purchase | 白天雇佣NPC帮你战斗——晚上从NPC处购买物品 / Hire NPCs to fight by day — buy items from NPCs at night |
| **阿格妮亚 / Dancer** | 诱惑 / Allure | 引导 / Guide | 白天让NPC跟随(与引导不同效果)——晚上带NPC走 / Have NPC follow (diff effect from Guide) by day — lead NPCs at night |
| **托罗 / Thief** | 偷窃 / Steal | 暗杀 / Ambush | 白天偷NPC身上的东西——晚上攻击NPC进入战斗 / Steal items from NPCs by day — ambush NPCs for battle at night |
| **光之战士 / Warrior** | 比试 / Bribe | 挑衅 / Challenge | 白天用钱买通NPC——晚上单挑NPC / Bribe NPCs with money by day — duel NPCs at night |

## 新手常见错误 / Common Beginner Mistakes

| # | 错误 / Mistake | 正确做法 / Correct Way |
|---|---|---|
| 1 | **只练一个角色 / Only level one character** | 注意给队伍里所有角色换装备——不上的角色也要给好武器 / Rotate gear among all characters — bench members need good weapons too |
| 2 | **不分析弱点 / Don't analyze weaknesses** | 学者奥丝瓦尔德的"洞察"技能是破防的核心——没有学者就要试错打弱点 / Osvald's Analyze is core to Break system — without Scholar, guess weaknesses by trial and error |
| 3 | **不舍得用BP / Hoard BP** | BP就是用来花的——留着不会生利息——破防后全队增幅输出 / BP is meant to be spent — break the enemy, then boost everyone for maximum damage |
| 4 | **不看NPC装备 / Ignore NPC gear** | 盗贼偷NPC经常会出极品装备——每个城镇的新NPC都试试 / Thief's Steal often gives rare gear — try every new NPC in each town |
| 5 | **不做支线 / Skip side quests** | 支线给JP+装备+解锁副职业——不做后面会很难打 / Side quests give JP + gear + unlock subjobs — skipping them makes the game harder later |
| 6 | **昼夜不分 / Ignore day/night** | 同样的NPC白天和晚上能交互的内容不同——多切换试试 / Same NPCs have different interactions day vs night — toggle and check |
