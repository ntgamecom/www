## 探索技巧 / Exploration Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **地图能力收集 / Gather Path Actions** | 每个角色都有独特的地图能力——收集全部8人后可以在任何场景灵活使用 / Each character has unique path actions — with all 8, you can handle any situation |
| **盗贼偷全城 / Steal Everything** | 托罗的白天偷窃技能——每个城镇的每个NPC都偷一遍——经常出好东西 / Throné's day Steal — try every NPC in every town — often gets rare items |
| **昼夜切换 / Day/Night Cycling** | 部分商店/宝箱/对话只在特定时间可用——养成到新地方就切昼夜的习惯 / Some shops/chests/dialogs are time-specific — toggle day/night when entering new areas |
| **船探索 / Ship Exploration** | 拿到船后可以自由探索海上——有大量隐藏地点和宝箱 / After getting the ship, explore the seas — many hidden locations and chests |
| **支线标记 / Quest Markers** | 支线任务不会自动标记位置——记下NPC位置或看攻略 / Side quests don't auto-mark — note NPC locations or check guide |
| **快速旅行 / Fast Travel** | 每个被发现的城镇都可以快速传送——多开传送点 / Every discovered town can be fast-traveled — unlock as many as possible |

- 进入新迷宫前**存盘点**(如果离城镇远的话)/ Save before entering new dungeons (if far from town)

- 遇到分支路线先走一边——宝箱拿完再回来走另一边 / On branching paths, clear one side first then return for the other

## 昼夜影响 / Day/Night Effects

| 系统 / System | 白天 / Day | 晚上 / Night |
|---|---|---|
| **NPC互动 / NPC Interaction** | 用白天地图能力 / Use day path actions | 用晚地图能力 / Use night path actions |
| **商店 / Shops** | 所有商店营业 / All shops open | 部分商店关门——黑市开门 / Some closed — black market open |
| **遇敌率 / Encounter Rate** | 标准 / Standard | **提高 / Increased** |
| **部分谜题 / Some Puzzles** | 特定条件需要白天触发 / Some require day | 特定条件需要晚上触发 / Some require night |
| **NPC位置 / NPC Positions** | 在固定位置 / At fixed positions | 可能在睡觉或移动 / May be sleeping or moved |

**建议**: 每到一个新区域——先白天逛一遍再晚上逛一遍——不要漏东西 / Recommendation: Visit each new area both day and night — don't miss anything
