## 技能系统概述 / Skill System Overview

**中文：** 怪物猎人：荒野的技能系统通过防具技能和装饰品实现。每件防具自带若干技能点数，装饰品可插入防具的装饰品槽来补充技能。技能分为进攻型、防御型、辅助型和通用型四大类。每个技能有多个等级，等级越高效果越强。合理搭配技能是配装的核心。

**English:** Monster Hunter Wilds' skill system uses armor skills and decorations. Each armor piece has innate skill points, and decorations slot into armor to supplement skills. Skills are categorized into offensive, defensive, support, and general types. Each skill has multiple levels for increased effect. Proper skill combination is the core of build crafting.

| 技能分类 / Category | 说明 / Description |
|---|---|
| 进攻型 / Offensive | 提升伤害输出 / Increase damage output |
| 防御型 / Defensive | 提升生存能力 / Increase survivability |
| 辅助型 / Support | 提升狩猎效率 / Increase hunting efficiency |
| 通用型 / General | 多功能增益 / Multi-purpose buffs |

## 配装思路 / Build Philosophy

**中文：** 配装的核心是围绕武器和怪物选择技能组合。以下是几种通用配装框架。

**English:** Build crafting revolves around selecting skill combinations for your weapon and target monster. Below are general build frameworks.

**物理会心流 / Physical Affinity Build**:
- 核心技能：攻击7 + 看破7 + 弱特3 + 超心3 + 挑战者5 / Core: Atk7 + CE7 + WE3 + CB3 + Agi5
- 适合武器：太刀、大剑、大锤、双刀 / Best for: LS, GS, Hammer, DB
- 说明：最大化会心率和会心伤害，通用性最高的配装 / Maximize affinity and crit damage, most versatile

**属性强化流 / Elemental Build**:
- 核心技能：属强5 + 属会1 + 攻击4 + 看破4 + 弱特3 / Core: Element Atk5 + CE1 + Atk4 + CE4 + WE3
- 适合武器：双刀、弓、盾斧（属性瓶）/ Best for: DB, Bow, CB (element phials)
- 说明：针对怪物弱点属性配装，伤害上限更高但需要多套 / Match element to monster weakness, higher ceiling but multiple sets needed

**防御生存流 / Defensive Survival Build**:
- 核心技能：体增3 + 防御7 + 精灵加护3 + 回避性能3 + 回避距离2 / Core: HB3 + Def7 + DB3 + EW3 + EE2
- 适合武器：长枪、铳枪、新手开荒 / Best for: Lance, GL, beginner progression
- 说明：生存优先，适合开荒高难度怪物 / Survival priority, great for progression

## 装饰品系统 / Decorations

**中文：** 装饰品是镶嵌在防具上的技能珠，每个装饰品提供1-2级特定技能。高级装饰品提供稀有技能组合。

**English:** Decorations are skill gems slotted into armor. Each decoration provides 1-2 levels of a specific skill. High-tier decorations offer rare skill combinations.

| 装饰品等级 / Tier | 稀有度 / Rarity | 效果 / Effect |
|---|---|---|
| 一级 / Tier 1 | ★ | 1级技能 ×1 / Single skill level 1 |
| 二级 / Tier 2 | ★★ | 2级技能 ×1 / Single skill level 2 |
| 三级 / Tier 3 | ★★★ | 1级技能 ×2 或 2级技能×1 / Double skill or single level 2 |
| 四级 / Tier 4 | ★★★★ | 2级技能 ×2 / Double skill level 2 |

**刷取推荐 / Farming Recommendations**:
- 攻击珠/痛击珠：历战古龙任务高概率掉落 / Atk/WE jewels: high rate from Tempered Elder quests
- 匠珠：HR 50+历战任务 / Handicraft: HR 50+ Tempered quests
- 挑战珠：HR 100+历战王任务 / Agitator: HR 100+ Arch-Tempered

## 技能模拟器用法 / Skill Simulator

**中文：** 使用配装器可以模拟技能搭配，确保在技能上限内实现最优配置。

**English:** Using a build simulator helps optimize skill configurations within skill caps.

**技能上限 / Skill Caps**:
- 攻击/看破等核心技能：7级上限 / Core skills like Atk/CE: max 7
- 弱特/超心等：3级上限 / WE/CB: max 3
- 属性强化：5级上限 / Element Atk: max 5
- 防御技能：7级上限 / Def skills: max 7

**配装技巧 / Build Tips**:
- 优先达到会心率100%——弱特3(50%)+看破7(40%)+攻击7(5%)+挑战者(20%)自由组合 / Prioritize 100% affinity: WE3(50%)+CE7(40%)+Atk7(5%)+Agi5(20%) mix
- 超会心3是提升会心收益的关键 / CB3 is key to maximizing affinity value
- 不要为了单个技能过度牺牲核心技能 / Don't oversacrifice core skills for a single niche skill
