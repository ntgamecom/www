## 战斗系统概述 / Combat Overview

**中文：** 怪物猎人：荒野的战斗系统在系列基础上引入了多项革新。核心战斗围绕武器攻击、伤口破坏、集中模式和骑乘系统展开。新加入的"伤口系统"允许猎人在怪物特定部位积累伤害后触发伤口，使用集中攻击造成巨额伤害。骑乘系统可以让猎人骑乘怪物并控制其移动方向。

**English:** Monster Hunter Wilds' combat builds on the series foundation with several innovations. Core combat revolves around weapon attacks, wound breaking, Focus Mode, and the mount system. ## 伤口系统详解 / Wound System

**中文：** 伤口系统是荒野新增的核心战斗机制。当猎人对怪物同一部位进行连续攻击时，该部位会先产生"伤痕"，继续攻击后伤痕会变成"伤口"。此时可以使用集中攻击（按R1/RB）造成额外伤害和大硬直。

**English:** The Wound system is Wilds' new core mechanic. Consecutive attacks on the same monster part create "scratches", which develop into "wounds" with further damage. Use Focus Attack (R1/RB) on wounds for bonus damage and a large stun.

**伤口系统机制 / Wound Mechanics**:

- 伤痕阶段：部位表面出现白色痕迹，表示该部位已积累伤害 / Scratch phase: white marks appear on the part
- 伤口阶段：部位出现红色裂口，可触发集中攻击 / Wound phase: red gash appears, Focus Attack ready
- 集中攻击：对伤口使用后造成高额伤害+大硬直+额外掉落 / Focus Attack: high damage + big stun + extra carves
- 伤口愈合：一段时间后伤口未被打爆会自然愈合 / Wounds heal over time if left unbroken

**高效制造伤口的技巧 / Wound Generation Tips**:

1. 多段攻击武器（双刀、盾斧、太刀）制造伤口最快 / Multi-hit weapons (Dual Blades, CB, LS) generate wounds fastest
2. 优先攻击弹性值低的部位（头部、尾巴）/ Prioritize low-elasticity parts (head, tail)
3. 集中模式按住L2/LT可以直接瞄准伤口位置 / Hold L2/LT in Focus Mode to aim at wound spots
4. 队友也可以攻击你制造的伤口 / Teammates can also trigger your wounds

## 集中模式详解 / Focus Mode

**中文：** 集中模式是荒野新增的瞄准系统。按住L2进入集中模式，可以自由调整攻击方向瞄准怪物特定部位。在集中模式下，伤口位置会高亮显示，方便快速定位。

**English:** Focus Mode is a new aiming system in Wilds. Hold L2 to enter Focus Mode, allowing free aim adjustment to target specific monster parts. Wound locations are highlighted in Focus Mode for quick targeting.

**集中模式核心用途 / Focus Mode Uses**:
1. 精准攻击特定部位（头、尾、翅膀）/ Precision attacking specific parts
2. 快速定位和触发伤口 / Quick wound targeting and triggering
3. 调整攻击方向——某些武器攻击可360度转向 / Adjust attack direction — some weapons can turn 360 degrees
4. 集中模式下翻滚改为蹲伏步，距离更短但硬直更小 / Dodge becomes a crouching step in Focus Mode — shorter but less recovery

## 骑乘系统 / Mount System

**中文：** 荒野的骑乘系统允许猎人骑乘小型怪物作为移动工具，也可以骑乘大型怪物进行控制。骑乘大型怪物时，可以控制其移动方向并撞墙造成伤害。

**English:** Wilds' mount system lets hunters ride small monsters for transportation and large monsters for control. When riding a large monster, you can steer it into walls for damage.

**骑乘操作 / Mount Controls**:
- 骑乘小型怪物：在野外靠近小型怪物按△/Y上骑 / Approach small monster in field, press △/Y to mount
- 骑乘大型怪物：使用骑乘攻击积累骑乘值，成功后骑上 / Use mounting attacks to build mount value, mount when full
- 控制方向：左摇杆控制怪物移动方向 / Left stick controls monster movement
- 撞墙：按R2/RT让怪物撞向墙壁，造成伤害+倒地 / Press R2/RT to slam monster into walls for damage + knockdown

## 通用战斗技巧 / General Combat Tips

**中文：** 以下技巧适用于所有武器。

**English:** These tips apply to all weapon types.

1. **伤口优先**：战斗中优先制造并触发伤口，造成大量硬直输出窗口 / Prioritize wound generation and triggering for stun windows
2. **部位破坏**：破坏部位可改变怪物行为和获得额外掉落 / Break parts to alter behavior and get bonus loot
3. **环境陷阱**：利用落石陷阱、藤蔓陷阱和环境生物辅助战斗 / Use environmental traps, vines, and endemic life
4. **道具使用**：闪光弹、陷阱、捕获用麻醉球是猎人的必备工具 / Flash pods, traps, tranq bombs are essential tools
5. **磨刀时机**：利用怪物换区或大硬直时磨刀 / Sharpen during zone transitions or big stun windows
6. **进食**：任务开始前一定吃猫饭，提供关键Buff / Always eat before quests for essential buffs
7. **回避性能**：根据怪物配置调整回避性能技能 / Adjust Evade Window skill based on monster matchup
