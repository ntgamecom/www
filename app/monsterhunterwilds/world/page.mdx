# 地图探索 / World Exploration

怪物猎人荒野的地图"天堑沙原"是系列首个真正意义上的无缝开放世界——多个生物群落无缝连接，怪物自由迁徙，环境动态变化。理解地图结构能大幅提升狩猎效率 / The Forbidden Lands is the series' first true seamless open world — multiple biomes connected without loading, monsters roam freely, dynamic environments. Understanding the map structure dramatically improves hunting efficiency.

## 沙原 / Windward Plains

新手练级区域——广阔的沙漠+绿洲地形。白天炎热，夜晚降温。沙尘暴期间能见度降低但可以偷袭怪物 / Starter leveling zone — vast desert + oasis terrain. Hot during day, cooler at night. Sandstorms reduce visibility but allow sneak attacks on monsters.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **绿洲营地 / Oasis Camp** | 1区——初始营地 / Base camp | 道具箱、猫饭、任务板 / Item box, canteen, quest board |
| **地下营地 / Underground Camp** | 8区——绿洲下方洞穴 / Below oasis cave | 第二营地——解锁支线任务 / 2nd camp — unlock via side quest |
| **龙骨采集点 / Bone Piles** | 2、4区——大量龙骨 / Zones 2, 4 — many bones | 防具制作素材来源 / Armor crafting materials |
| **沙原矿脉 / Plains Ore Veins** | 6、9区——矿石 / Zones 6, 9 — ores | 武器升级基础素材 / Basic weapon upgrade mats |

## 密林 / Scarlet Forest

热带雨林区域——多层垂直结构。地面、树干和树顶平台相通。夜间有独特的环境生物 / Tropical rainforest — multi-layer vertical structure. Ground, tree trunks, and canopy platforms connect. Unique endemic life at night.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **中央营地 / Central Camp** | 5区——地图正中央 / Zone 5 — map center | 解锁后到各区域都近 / Central access to all zones |
| **蜂巢区域 / Beehive Area** | 3、7区——蜂蜜采集点 / Zones 3, 7 — honey gathering | 合成回复药G必备 / Required for Mega Potions |
| **毒沼 / Poison Swamp** | 4区——毒地形/ Zone 4 — poison terrain | 毒怪鸟栖息地 / Gypceros habitat |
| **树顶平台 / Canopy Platform** | 8区——树顶——骑乘攻击好位置 / Zone 8 — tree top — mount attack spot | 跳跃攻击积累骑乘值 / Jump attacks build mount |

## 油涌谷 / Oilwell Basin

火山与油沼混合区域——高温地形需要耐热衣装。油沼表面可以点燃造成范围伤害 / Volcano + oil marsh mix — need Heat Guard mantle. Oil marsh surface can be ignited for AoE damage.

| 地点 / Location | 说明 / Description | 重要内容 / Key Content |
|:----:|------|------|
| **火山营地 / Volcano Camp** | 2区——安全区 / Zone 2 — safe zone | 唯一安全营地区域 / Only safe camp in zone |
| **油沼区域 / Oil Marsh** | 5、7区——油沼表面可点燃 / Zones 5, 7 — ignitable oil | 引诱怪物到油沼→点燃→额外伤害 / Lure monster to oil → ignite → bonus dmg |
| **矿脉 / Ore Veins** | 3、6、9区——大量高级矿石 / Zones 3, 6, 9 — many high-grade ores | 上位武器升级素材 / HR weapon upgrade mats |
| **熔岩洞穴 / Lava Cave** | 8区——炎王龙出没 / Zone 8 — Teostra territory | 古龙级战斗区域 / Elder dragon fight zone |

## 营地位置 / Camp Locations

每个区域有多个可解锁的营地——营地越多，狩猎时换区越方便。营地需要完成支线任务解锁 / Each zone has multiple unlockable camps — more camps = faster zone changes during hunts. Camps are unlocked via side quests.

| 区域 / Zone | 营地数 / Camps | 推荐优先解锁 / Priority |
|:----:|:----:|:----|
| **沙原 / Plains** | 4 | 1区营地(初始) → 8区营地(绿洲附近——狩猎常用) / Zone 1 (default) → Zone 8 (near oasis — frequently used) |
| **密林 / Forest** | 3 | 5区营地(地图中央——到各区域都近) / Zone 5 (center — convenient access) |
| **油涌谷 / Basin** | 3 | 2区营地(安全区——火山区域少) / Zone 2 (safe — fewer camps in volcano) |

## 探索技巧 / Exploration Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **骑乘自动导航 / Auto-Navigate** | 打开地图→标记目标→赛克特自动跑到目标点——跑图时刷手机吧 / Mark target on map → Seikret auto-runs — browse your phone while traveling |
| **导虫追踪 / Scoutfly Tracking** | 狩猎开始后按L2/R3导虫会自动追踪目标怪物足迹——不用自己找 / Press L2/R3 after starting a quest — Scoutflies auto-track monster footprints |
| **调查任务 / Investigations** | 捡怪物脚印/痕迹→获得调查点数→解锁调查任务(额外奖励掉落) / Pick up tracks → research points → unlock investigations (bonus reward boxes) |
| **环境陷阱利用 / Use Environment** | 落石、毒沼、蜂巢、藤蔓陷阱——把怪物引过去打 / Falling rocks, poison swamps, beehives, vine traps — lure monsters into them |
| **捕获环境生物 / Catch Endemic Life** | 靠近小生物时→打开道具轮盘→选捕获网→捕捉。可以养在据点当宠物 / Equip Capture Net from item wheel → catch small creatures. Raise them as pets in base |
| **随从探索队 / Palico Expedition** | 在据点派随从出去探索——回来时带回素材 / Send Palico on expeditions from base — they return with materials |
