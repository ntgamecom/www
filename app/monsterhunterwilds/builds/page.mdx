# 配装Build / Armor Builds

怪物猎人荒野的配装围绕武器类型和技能组合展开。技能通过防具自带、装饰品(珠子)和护石获得。本章提供各武器的主流配装方案，从开荒到毕业循序渐进 / Builds in Monster Hunter Wilds revolve around weapon type and skill combos. Skills come from armor innate skills, decorations, and talismans. This chapter provides mainstream builds per weapon, from early game to endgame.

---

## 开荒通用配装 / Early Game All-Purpose Build

| 部位 / Slot | 防具 / Armor | 核心技能 / Key Skill |
|:----:|:----:|:----:|
| **头 / Head** | 怪鸟头盔 / Chatacabra Helm | 攻击Lv1 / Attack Boost Lv1 |
| **胸 / Chest** | 怪鸟铠甲 / Chatacabra Mail | 体力增强Lv1 / Health Boost Lv1 |
| **手 / Arms** | 雌火龙手 / Rathian Vambraces | 精灵加护Lv1 / Divine Blessing Lv1 |
| **腰 / Waist** | 雌火龙腰 / Rathian Coil | 回复速度Lv1 / Recovery Speed Lv1 |
| **腿 / Legs** | 怪鸟腿 / Chatacabra Greaves | 跑者Lv1 / Marathon Runner Lv1 |

**开荒思路**: 不要纠结完美技能——选防御力高的穿上。珠子(装饰品)上位才开始大量掉落。优先技能: **体力增强**(+HP上限)、**攻击**(+攻击)、**精灵加护**(概率减伤) / **Early game tip**: Don't chase perfect skills — prioritize defense. Decorations drop from High Rank. Priority skills: Health Boost (+HP), Attack Boost (+ATK), Divine Blessing (DR proc).

## 近战通用毕业配装 / Melee Endgame General Build

| 技能 / Skill | 推荐等级 / Lv | 说明 / Description |
|:----:|:----:|------|
| **攻击 / Attack Boost** | Lv7 | 基础攻击力+21 + 会心+5% / +21 base ATK + 5% affinity |
| **看破 / Critical Eye** | Lv7 | 会心+40% / +40% affinity |
| **超会心 / Critical Boost** | Lv3 | 会心伤害从1.25x → 1.40x / Crit dmg 1.25x → 1.40x |
| **弱点特效 / Weakness Exploit** | Lv3 | 打弱点+50%会心 / +50% affinity on weak points |
| **体力增强 / Health Boost** | Lv3 | +50 HP上限 / +50 max HP |
| **匠 / Handicraft** | 视武器 | 延长斩味 / Extends sharpness |

**通用物理配装核心**: 看破7 + 弱特3 + 超会心3 + 攻击4 = 满会心+高爆伤。适合所有物理武器 / **Core physical build**: Critical Eye 7 + Weakness Exploit 3 + Critical Boost 3 + Attack 4 = 100% affinity + high crit damage. Works for all physical weapons.

## 生存技能优先级 / Survival Skill Priority

| 优先级 / Priority | 技能 / Skill | 效果 / Effect |
|:----:|:----|:----:|
| ★★★★★ | **体力增强 / Health Boost** | +50最大HP——**全流派必备** / +50 max HP — **mandatory for all builds** |
| ★★★★ | **精灵加护 / Divine Blessing** | 50%概率减伤50%——等效25%常驻减伤 / 50% chance to reduce damage by 50% — ~25% effective DR |
| ★★★★ | **回避性能 / Evade Window** | 翻滚无敌帧增加——熟悉怪物后收益极高 / Increases dodge i-frames — huge value once you know monster patterns |
| ★★★ | **回复速度 / Recovery Speed** | 红血回复加速——配合耐回(Recovery Up)更佳 / Faster recoverable HP regen — pair with Recovery Up |
| ★★★ | **耐火/雷/冰/龙 / Elemental Resist** | 对抗特定属性怪物——降低被秒概率 / Counter specific elemental monsters — reduces one-shot risk |

## 历战王配装思路 / Tempered Build Strategy

| 阶段 / Phase | 配装重点 / Focus | 说明 / Description |
|:----:|:----|------|
| **历战初期 / Early Tempered** | 生存优先——体质+精灵加护+回复速度 / Survival first | 历战怪物攻击翻倍——不要堆纯输出 / Tempered monster attack is 2x — don't stack pure offense |
| **历战中期 / Mid Tempered** | 看破+弱特+超会心 / Crit build | 熟悉怪物后开始补输出 / Start adding offense after learning patterns |
| **历战王 / Arch-Tempered** | 满会心+超会心3+精灵加护5 / Max crit + Divine Blessing 5 | 怪物有新增招式——需要同时堆生存和输出 / New moves — need both survival and offense |
