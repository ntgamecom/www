## 任务系统概述 / Quest System Overview

**中文：** 怪物猎人：荒野的任务系统分为主线任务（关键任务）、集会所任务、自由任务和调查任务四种。主线任务推动剧情和HR等级解锁，集会所任务提供额外挑战，自由任务是刷素材的主要方式，调查任务提供更高奖励。

**English:** Monster Hunter Wilds' quest system has four types: Key Quests (main story), Hub Quests, Optional Quests, and Investigations. Key Quests advance the story and HR level. Hub Quests provide additional challenges. Optional Quests are the main material farming method. Investigations offer higher rewards.

| 任务类型 / Type | 说明 / Description |
|---|---|
| 主线关键任务 / Key Quests | 推动剧情，解锁新区域和怪物 / Advance story, unlock areas and monsters |
| 集会所任务 / Hub Quests | 多人联机任务，更高难度 / Multiplayer quests, higher difficulty |
| 自由任务 / Optional Quests | 可重复接取，刷素材 / Repeatable, material farming |
| 调查任务 / Investigations | 随机生成，高奖励 / Randomly generated, high rewards |

## 自由任务与调查任务 / Optional & Investigation

**中文：** 自由任务可以反复接取，是刷素材的最佳方式。调查任务随机生成，提供更高的金钱和素材奖励。

**English:** Optional quests are repeatable and are the best way to farm materials. Investigations are randomly generated with higher zenny and material rewards.

**自由任务刷新机制 / Optional Quest Generation**:
- 完成任务后，自由任务列表会刷新 / Quest list refreshes after each completion
- 与NPC对话可以解锁新的自由任务 / Talk to NPCs to unlock new optional quests
- 自由任务也会出现在野外地图中发现的"调查点" / Optional quests also appear at "survey points" found in the field

**调查任务系统 / Investigation System**:
1. 在野外地图中收集怪物痕迹（脚印、毛发、黏液）解锁调查任务 / Collect monster tracks (footprints, fur, mucus) to unlock investigations
2. 调查任务有次数限制（通常1-5次）/ Investigations have limited uses (usually 1-5)
3. 调查任务有额外奖励条件（捕获、部位破坏、时间限制）/ Bonus conditions (capture, part break, time limit)
4. 金色/银色箱子提供稀有素材的高几率掉落 / Gold/silver boxes offer higher rare material drop rates

## 调查任务类型 / Investigation Types

**中文：** 调查任务根据奖励类型分为金箱、银箱和铜箱三种。金箱提供最高稀有度的素材掉落。

**English:** Investigations are divided by reward box type: Gold, Silver, and Bronze. Gold boxes provide the highest rarity material drops.

| 箱子类型 / Box Type | 稀有素材概率 / Rare Mat Rate | 奖励金 / Zenny |
|---|---|---|
| 金箱 / Gold Box | 高（15%-20%）/ High | 大量 / Large |
| 银箱 / Silver Box | 中（8%-12%）/ Medium | 中等 / Medium |
| 铜箱 / Copper Box | 低（3%-5%）/ Low | 少量 / Small |

**刷素材策略 / Farming Strategy**:
- 稀有素材（宝玉/红玉）：优先金箱+断尾+破头奖励 / Rare materials (gems/rubies): prioritize gold boxes + tail cut + head break
- 普通素材：银箱和捕获奖励足够 / Common materials: silver boxes + capture rewards suffice
- 调查任务次数耗尽后可以刷新：完成任意任务或睡觉 / Refresh investigations by completing any quest or sleeping

{/* ===== 大小金冠 / Gold Crowns ===== */}

## 大小金冠 / Gold Crowns

**中文：** 大小金冠是怪物尺寸的成就系统。击败极大（大金）或极小（小金）的个体怪物可以解锁对应的奖杯/成就。

**English:** Gold Crowns are an achievement system for monster sizes. Defeating extra large (Large Gold) or extra small (Small Gold) individual monsters unlocks corresponding trophies.

**刷金冠技巧 / Crown Farming Tips**:
- 调查任务和活动任务出现金冠个体的概率更高 / Investigations and event quests have higher crown rates
- 特定活动任务专门提高了金冠概率（如"最盛大的狩猎"）/ Specific event quests boost crown rates
- 目测判断：比普通个体明显大或小就是金冠个体 / Visual judgment: clearly larger or smaller than normal
- 大金冠和/or小金冠全收集解锁白金奖杯 / All large/small crowns unlock platinum trophy
