# 武器指南 / Weapon Guide

怪物猎人荒野有14种武器——每种玩法完全不同。选择一把适合你的武器是狩猎的第一步。本章详解每种武器的核心机制、操作要点和推荐场景 / Monster Hunter Wilds has 14 weapon types — each plays completely differently. Choosing the right weapon is the first step in your hunting journey. This chapter covers core mechanics, key controls, and recommended scenarios for each.

---

## 近战武器 / Melee Weapons

### 大剑 / Great Sword

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★★★☆☆ |
| **核心玩法 / Core** | 蓄力斩——一刀入魂。真·蓄力斩是最高单发伤害技 / Charged slash — one big hit. True Charged Slash = highest single-hit damage |
| **新动作 / New** | 集中模式：瞄准弱点后真蓄力斩自动修正方向 / Focus Mode: TCS auto-aims at weak points |
| **推荐场景 / Best for** | 攻击大开大合的怪物——趁怪物大硬直时蓄力输出 / Monsters with big attack windows — punish with charged slashes |

**操作要点**: 按住△蓄力→第一阶段(强蓄)→第二阶段(真蓄)。集中模式下真蓄力的方向可以自由调整。不要贪蓄力——判断怪物攻击前摇提前翻滚取消 / **Tips**: Hold △ to charge → Stage 1 (Strong) → Stage 2 (True). Focus Mode lets you aim TCS direction. Don't over-commit — roll cancel if monster is about to attack.

### 太刀 / Long Sword

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★★★★☆ |
| **核心玩法 / Core** | 气刃斩连段→练气槽满→气刃大回旋→开刃(提升攻击力)。见切斩(反击技)是核心位移手段 / Spirit Blade combo → fill gauge → Spirit Roundslash → level up. Foresight Slash (counter) is core movement |
| **新动作 / New** | 集中模式：气刃兜割自动追踪弱点 / Focus Mode: Spirit Helm Breaker auto-tracks weak points |
| **推荐场景 / Best for** | 熟悉怪物动作的老手——见切反击是太刀的核心乐趣 / Experienced players who know monster patterns — counters are the core joy |

**操作要点**: 练气槽不满时不要用气刃斩(会掉刃)。见切斩(R2+○)在怪物攻击命中瞬间输入→成功后接气刃大回旋直接开刃。太刀的复杂度和人气一样高 / **Tips**: Don't use Spirit Blade without gauge (loses level). Foresight Slash (R2+○) at attack impact → follow with Roundslash for instant level-up. Most popular weapon in MH — and hardest to master.

### 片手剑 / Sword & Shield

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★☆☆☆☆ |
| **核心玩法 / Core** | 攻守兼备——连段快、可防御、可喝药不收刀。可骑乘攻击 / Jack of all trades — fast combos, can block, use items unsheathed. Mounting attacks |
| **新动作 / New** | 集中模式：精准连击——连续命中弱点后追加高伤终结技 / Focus Mode: Precision combo — bonus finishing move after consecutive weak point hits |
| **推荐场景 / Best for** | **新手首选**——任何场景都能打 / **Best for beginners** — works in any scenario |

**操作要点**: 片手剑的核心优势是"不收刀喝药"——怪物出大招时你跑开直接喝药，比其他武器快2秒。JR(后跳蓄力斩)有很长的无敌帧 / **Tips**: Core advantage = use items without sheathing — heal 2 seconds faster than other weapons. Backstep has generous i-frames.

### 双剑 / Dual Blades

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★★☆☆☆ |
| **核心玩法 / Core** | 鬼人化(消耗耐力)→攻速极大提升→鬼人连段→鬼人强化(脱离后短暂维持) / Demon Mode (consumes stamina) → max attack speed → demon combo → Archdemon Mode (lingers after exit) |
| **新动作 / New** | 集中模式：鬼人乱舞自动追踪弱点——乱舞中可微调方向 / Focus Mode: Demon Flurry auto-tracks weak points — slight direction control during flurry |
| **推荐场景 / Best for** | 属性/异常状态输出——双剑的属性补正高、异常积累快 / Element/status DPS — high elemental modifiers, fast status buildup |

### 大锤 / Hammer

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★☆☆☆☆ |
| **核心玩法 / Core** | 打击武器——打头=晕怪。蓄力→大地的回应(转圈升龙)/ 蓄力→上撩 / Blunt weapon — hit head = stun. Charge → Spinning Bludgeon / Charge → Upswing |
| **新动作 / New** | 集中模式：蓄力中可移动调整——终结技强制眩晕 / Focus Mode: move while charging — finisher guarantees stun |
| **推荐场景 / Best for** | 新手+老手都喜欢——打头晕怪后全队疯狂输出 / All players — stun gives entire team massive DPS window |

### 狩猎笛 / Hunting Horn

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★★★☆☆ |
| **核心玩法 / Core** | 演奏旋律给全队加Buff——攻击也可积累旋律。荒野中笛子的输出大幅提升 / Perform melodies for team buffs — attacks also build melodies. Significantly improved damage in Wilds |
| **推荐场景 / Best for** | 联机——一个会玩的笛子让全队狩猎时间缩短30% / Multiplayer — a skilled HH user reduces hunt time by 30% |

### 长枪 / Lance

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★★☆☆☆ |
| **核心玩法 / Core** | 最强防御——防反→反击突刺→精准防御→防性不需堆太高 / Best defense — Power Guard → Counter Thrust → perfect block → low Guard skill investment needed |
| **推荐场景 / Best for** | 新手(安全) / 面对攻击频率高的怪物 / Beginners (safety) / Monsters with fast, relentless attacks |

### 铳枪 / Gunlance

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★★★☆☆ |
| **核心玩法 / Core** | 长枪+炮击——炮击无视肉质固定伤害。全弹发射(装填所有子弹→一发打出)是核心爆发技 / Lance + gun — shelling ignores hitzones for fixed damage. Full Burst (load all shells → fire at once) = core burst |
| **推荐场景 / Best for** | 肉质差/弹刀厉害的怪物——炮击不看你肉质 / Monsters with bad hitzones — shelling ignores them |

### 斩斧 / Switch Axe

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★★★☆☆ |
| **核心玩法 / Core** | 剑模式(高伤+属性瓶) ↔ 斧模式(长距离+机动)。剑模式攻击积攒觉醒槽→觉醒后剑攻击附带额外爆炸 / Sword mode (high dmg + phial) ↔ Axe mode (range + mobility). Sword attacks build Awakened gauge → extra explosions |
| **推荐场景 / Best for** | 喜欢两种形态切换的玩家——操作自由度极高 / Players who enjoy weapon mode switching — high skill expression |

### 盾斧 / Charge Blade

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★★★★★ |
| **核心玩法 / Core** | 最复杂的武器——剑模式攒能量→充入瓶(红盾)→斧模式消耗瓶爆发(超解/高解)。荒野中红盾和红斧可同时维持 / Most complex weapon — sword mode builds energy → charge phials (red shield) → axe mode spends phials (SAED/AMP). Both red shield and red axe can be active simultaneously |
| **推荐场景 / Best for** | 追求操作上限的玩家——精通后秒天秒地 / Players who love high skill ceilings — devastating when mastered |

### 操虫棍 / Insect Glaive

| 特性 / Trait | 说明 / Description |
|:----:|------|
| **难度 / Difficulty** | ★★★☆☆ |
| **核心玩法 / Core** | 跳跃攻击+操控猎虫。发射猎虫到怪物部位→吸取精华(红/白/橙)→获得强化Buff。三色精华集齐=全强化 / Jump attacks + Kinsect control. Mark monster part → kinsect extracts essence (red/white/orange) → buffs. All 3 colors = full buff |
| **推荐场景 / Best for** | 喜欢空中战斗和高机动性的玩家 / Players who enjoy aerial combat and high mobility |
