# 新手入门 / Beginner Guide

怪物猎人：荒野是系列最新作，引入大量新手友好机制。即使你从未接触过怪物猎人系列，也能快速上手 / Monster Hunter Wilds is the most beginner-friendly entry in the series. Even if you've never played a Monster Hunter game before, you'll get the hang of it quickly.

---

## 狩猎基础流程 / Basic Hunting Flow

| 步骤 / Step | 说明 / Description |
|:----:|------|
| **1. 接任务 / Accept Quest** | 在据点公会柜台或阿尔玛处接取狩猎任务——指定目标怪物 / Accept a hunting quest from the Guild counter or Alma — target monster specified |
| **2. 准备出发 / Prepare** | 在营地补给箱拿取免费道具——吃猫饭加血耐上限——检查装备是否满耐久 / Grab free items from supply box — eat at canteen for HP/stamina — check weapon sharpness |
| **3. 追踪怪物 / Track Monster** | 骑乘赛克特(Seikret)自动导航到怪物位置——按L2用导虫追踪脚印/痕迹 / Ride Seikret for auto-navigation — use L2 to track footprints and traces with Scoutflies |
| **4. 战斗 / Combat** | 攻击怪物→躲避攻击→破坏部位→怪物换区→继续追击 / Attack → dodge → break parts → monster flees → pursue |
| **5. 捕获或击杀 / Capture or Kill** | 怪物瘸腿(逃跑睡觉)时设陷阱+捕获玉=捕获(奖励更多)。或者继续打到死 / Trap + tranq bombs when limping/sleeping = capture (more rewards). Or kill by depleting HP |
| **6. 剥取/奖励 / Carve/Reward** | 击杀后剥取素材(3-4次) + 任务奖励结算——回村造装备 / Carve 3-4 times + quest rewards — return to craft gear |

## 关键操作 / Key Controls

| 操作 / Action | PS5 | Xbox | 说明 / Description |
|:----:|:----:|:----:|------|
| **攻击 / Attack** | △ | Y | 主攻击键——连按派生连段 / Main attack — chain for combos |
| **特殊攻击 / Special** | ○ | B | 武器专属动作(如太刀气刃斩、盾斧装瓶) / Weapon-specific actions |
| **闪避 / Dodge** | ✕ | A | 翻滚闪避——有无敌帧 / Roll dodge — has i-frames |
| **防御 / Guard** | R2 | RT | 带盾武器可防御——其他武器此键为特殊动作 / Block with shield weapons — special actions for others |
| **道具轮盘 / Item Wheel** | L1 | LB | 选择和使用道具——调整顺序可以提高效率 / Select and use items — customize for efficiency |
| **集中模式 / Focus Mode** | R1 + L2 | RB + LT | 瞄准弱点部位——进入此模式后攻击精准锁定弱点 / Aim at weak points — attacks lock onto wounds in this mode |
| **骑乘 / Mount** | ✕(靠近) | A(near) | 上/下赛克特。骑乘时可磨刀、喝药、自动导航 / Mount/dismount Seikret. Sharpen, heal, auto-navigate while mounted |
| **投射器 / Slinger** | L2 | LT | 发射环境道具(闪光弹、音爆弹、捕获网等) / Fire environmental items (flash pods, sonic bombs, capture net) |

| **片手剑 / Sword & Shield** | ★☆☆☆☆ | 攻守兼备——可防御、可喝药不收刀、连段简单 / Jack of all trades — block, use items without sheathing, simple combos |
| **大锤 / Hammer** | ★☆☆☆☆ | 高伤害+机动性——可以晕怪(打击头部)。操作直观 / High damage + mobility — can stun monsters (hit head). Intuitive controls |
| **长枪 / Lance** | ★★☆☆☆ | 最强防御——防反后稳定输出。节奏偏慢但极其安全 / Best defense — counter thrust after block. Slow but extremely safe |
| **双剑 / Dual Blades** | ★★☆☆☆ | 攻速极快——鬼人化后DPS爆炸。需要管理耐力 / Fastest attack speed — demon mode for DPS burst. Stamina management required |
| **轻弩 / Light Bowgun** | ★★☆☆☆ | 远程安全输出——可以打属性弹/异常弹。适合不喜欢贴脸的玩家 / Safe ranged DPS — elemental and status ammo. For players who prefer range |

## 开荒装备路线 / Early Gear Progression

| 阶段 / Phase | 推荐防具 / Armor | 推荐武器 / Weapon | 目标 / Goal |
|:----:|:----:|:----:|------|
| **低级(LR 1-3)** | 初始防具→怪鸟套 | 铁制武器→怪鸟武器 | 熟悉操作、学怪物行为 / Learn controls and monster patterns |
| **低级(LR 4-6)** | 怪鸟套→雌火龙套 | 雌火龙武器 | 解锁更多怪物、攒材料 / Unlock more monsters, gather materials |
| **低级(LR 7-8)** | 雌火龙套→火龙/雷狼龙混搭 | 火龙/雷狼龙武器 | 准备进入上位 / Prepare for High Rank |
| **上位(HR 1-3)** | 上位怪鸟套→上位雌火套 | 上位铁制→上位雌火武器 | 防御力断层提升——优先换防具 / Defense gap — upgrade armor first |
| **上位(HR 4+)** | 目标怪物对应弱点套 | 对应弱点属性武器 | 开始针对性配装 / Start optimizing builds |

## 新手常见问题 / Common Newbie Mistakes

| 错误 / Mistake | 正确做法 / Correct |
|:----:|------|
| **不强化装备 / Skip Upgrading** | 每打2-3种新怪就回村升级一次装备——防御力跟不上会被秒 / Upgrade every 2-3 new monsters — outdated defense = one-shots |
| **只会站桩输出 / Greedy DPS** | 怪物猎人的核心是"等机会→输出→收手"循环。贪刀是猫车第一原因 / Core rhythm is "wait → punish → reset". Greed is #1 cart cause |
| **不捕获 / Never Capture** | 捕获=额外素材奖励+任务时间更短。瘸腿的怪物一定要抓 / Capture = bonus materials + shorter quest. Always capture when limping |
| **不带满道具 / Empty Item Pouch** | 出发前检查道具袋——回复药G x10、陷阱x2、捕获玉x8、砥石是必须的 / Check pouch before every quest — Mega Potions x10, traps x2, tranqs x8, whetstone are mandatory |
| **不用投射器 / Ignore Slinger** | 闪光弹闪飞龙、音爆弹炸钻地怪、捕获网抓小生物——投射器功能远比你想象的多 / Flash pods for flyers, sonic bombs for burrowers, capture net for endemic life |
