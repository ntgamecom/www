# 怪物图鉴 / Monster List

怪物猎人荒野的怪物分为多个等级：低级(LR)、上位(HR)和历战个体。了解每只怪物的弱点属性、可破坏部位和攻击模式是高效狩猎的关键 / Monsters in Monster Hunter Wilds are divided into Low Rank, High Rank, and Tempered variants. Knowing each monster's weakness, breakable parts, and attack patterns is key to efficient hunting.

## 怪物狩猎策略 / Key Monster Strategies

| 怪物 / Monster | 核心策略 / Core Strategy | 推荐武器 / Weapon | 注意 / Watch Out |
|:----:|:----|:----:|:----|
| **雌火龙 / Rathian** | 断尾(减少毒范围)→破坏头→捕获 / Sever tail → break head → capture | 大剑、太刀 | 毒尾横扫范围大 / Wide poison tail sweep |
| **火龙 / Rathalos** | 闪光弹打下来→断尾→破头 / Flash pod → sever tail → break head | 弓、轻弩 | 飞天时间久——闪光弹必带 / Long flight time — always bring flash pods |
| **雷狼龙 / Zinogre** | 蓄电时打头打断→倒地输出 / Hit head during charge → interrupt → DPS | 大锤、狩猎笛 | 蓄电完成后威胁翻倍 / Full charge = double threat |
| **角龙 / Diablos** | 音爆弹(钻地时)→撞墙→破角 / Sonic bomb (burrowed) → wall bang → break horns | 大剑、长枪 | 钻地→音爆弹出地后有大硬直 / Sonic bomb = big stagger |
| **碎龙 / Brachydios** | 打头/前爪→注意黏菌→翻滚消黏菌 / Hit head/paws → watch slime → roll 3x | 双剑、片手剑 | 黏菌爆炸秒人——翻滚3次消除 / Slime explosion = one-shot — roll 3x |
| **灭尽龙 / Nergigante** | 破坏白刺→倒地输出→循环 / Break white spikes → knockdown → repeat | 大剑、操虫棍 | 白刺再生速度快——优先破坏 / Spikes regrow fast — prioritize breaking |
| **钢龙 / Kushala Daora** | 毒飞刀取消风压→打头倒地 / Poison knife cancels wind → head stagger | 弓、轻弩 | 风压是最大威胁——带毒武器或风压无效技能 / Wind is biggest threat — bring poison or Windproof |
| **炎王龙 / Teostra** | 打头打断粉尘爆炸→超新星时跑远 / Hit head to interrupt dust → run from supernova | 水属性武器 | 粉尘爆炸范围大——看到粉尘就翻滚 / Dust has wide AoE — roll on sight |

## 战斗通用技巧 / Universal Combat Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **断尾 / Sever Tail** | 切断系武器(太刀/大剑/片手等)专注打尾巴→断尾后额外剥取一次 / Severing weapons focus tail → one extra carve |
| **撞墙 / Wall Bang** | 怪物非怒状态时→投射器装种火石→瞄准墙→射→怪物撞墙倒地 / Non-enraged monster → slinger pod → aim at wall → fire → monster wall-slams |
| **骑乘攻击 / Mounting** | 跳跃攻击累积骑乘值→骑乘成功时按△攻击→怪物倒地后疯狂输出头部 / Jump attacks build mount value → press △ during mount → monster topples → massive head DPS |
| **陷阱捕获 / Trap Capture** | 怪物瘸腿睡觉→脚边放陷阱→捕获玉x2→捕获成功。捕获不消耗陷阱耐久 / Limping/sleeping → trap at feet → tranq bomb x2 → capture success. Trap isn't consumed |
| **历战脚印 / Tempered Tracks** | 捡历战怪物脚印→调查点数→解锁更多调查任务 / Pick up tempered tracks → research points → unlock more investigations |
