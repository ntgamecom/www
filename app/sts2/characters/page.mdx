# 角色攻略 / Character Strategy Guide

每位角色都有独特的核心机制、卡牌池和玩法风格。本章提供深度角色分析，包含构筑思路、关键卡牌推荐、遗物优先级和实战技巧 / Each character has a unique core mechanic, card pool, and playstyle. This chapter provides in-depth analysis including deck-building advice, key card picks, relic priorities, and combat tips.

## 通用构筑原则 / Universal Deck-Building Principles

### 卡组平衡 / Deck Balance

| 原则 / Principle | 说明 / Description |
|------|------|
| **攻击:防御比 / Offense:Defense Ratio** | 第1章 60:40，第2章 50:50，第3章+ 40:60——后期敌人攻击更高，需要更多防御 / Act 1: 60:40, Act 2: 50:50, Act 3+: 40:60 — later enemies hit harder, need more defense |
| **费用曲线 / Energy Curve** | 平均费用1.5-2.0最佳——太多高费牌会导致卡手 / Average cost 1.5-2.0 is optimal — too many expensive cards causes dead hands |
| **过牌量 / Draw Power** | 每回合至少能过3-5张额外牌才算合格的运转 / At least 3-5 extra cards drawn per turn for a functional engine |
| **缩放机制 / Scaling** | 至少要有1-2张缩放牌，否则后期BOSS无法击败 / At least 1-2 scaling cards, otherwise late bosses are unbeatable |
| **AOE需求 / AoE Needs** | 至少1-2张AOE——第2章的群怪战斗极多 / At least 1-2 AoE cards — Act 2 has many multi-enemy fights |

### 删牌优先级 / Card Removal Priority

1. **初始打击 / Basic Strikes** — 最弱最无用的卡，优先删除 / Weakest and most useless, remove first
2. **初始防御 / Basic Defends** — 在拿到更好的防御牌后删除 / Remove after acquiring better block cards
3. **诅咒牌 / Curses** — 拿到后尽快通过事件或商店删除 / Remove ASAP via events or shops
4. **冗余高费牌 / Redundant Expensive Cards** — 费用曲线太高时删高费低效牌 / Remove expensive inefficient cards when curve is too high

## 织影者 / Shadowweaver (推荐新手 / Beginner Friendly)

**核心机制：暗影印记 (Shadow Marks) + 终结技 (Finishers)**

织影者通过在敌人身上叠加暗影印记来放大伤害，每层印记使目标受到伤害+5%。叠满10层时可消耗印记触发强力终结技。节奏感强，操作直观 / Shadowweaver stacks Shadow Marks on enemies (+5% damage taken per stack). At 10 stacks, consume for powerful Finishers. Strong rhythm, intuitive gameplay.

### 打法技巧 / Playstyle Tips

- **前期不急终结**：印记的增伤效果是全战受益，不要为了终结技过早消耗10层印记 / **Don't rush Finishers**: the +5% dmg amp per mark benefits your entire team's damage
- **优先保持全局印记**：尽量让每个敌人都挂着至少4-5层印记 / **Maintain universal marks**: keep 4-5 stacks on every enemy
- **群战用暗影爆发起手**：先叠全队印记再逐个收割 / **Open with Shadow Burst** in group fights: apply marks first, then pick off targets
- **夜幕降临是关键缩放牌**：如果能拿到，你的叠层效率翻倍 / **Nightfall is your key scaling card**: doubles your stacking speed
- **与铸甲师不同**：织影者需要主动叠层和爆发节奏，不是被动挨打 / **Unlike Ironwarden**: Shadowweaver needs active stacking and burst timing, not passive tanking

## 铸甲师 / Ironwarden (强烈推荐新手 / Highly Recommended)

**核心机制：护甲 (Armor) + 反击 (Counter)**

铸甲师累积独立于格挡的护甲值，受到攻击时护甲吸收伤害并触发反击。玩法简单直接：叠甲 → 挨打 → 反伤 → 循环。容错率最高的角色 / Ironwarden accumulates Armor (separate from Block). When hit, Armor absorbs damage and triggers Counters. Simple loop: stack armor → get hit → counter → repeat. Most forgiving character.

- **开局三回合叠甲**：前3回合不要急着输出，把护甲叠到15+ / **First 3 turns: stack armor**: don't rush damage, get 15+ Armor first
- **注意护甲和格挡的区别**：护甲吸收伤害后触发反击，格挡不会——先叠甲，再补格挡 / **Armor vs Block**: Armor triggers counter on absorption, Block doesn't — prioritize Armor
- **铁砧牌是关键时刻用的**：不要在护甲低于10时使用铁砧 / **Anvil cards are for key moments**: don't use them when Armor is below 10
- **铁棘是必拿牌**：没有铁棘的反击流伤害不足 / **Iron Thorns is mandatory**: Counter build lacks damage without it
- **简单但不要无脑**：虽然铸甲师容错率高，但无脑叠甲不输出也会被磨死 / **Simple but not mindless**: high forgiveness but pure armor with no offense will lose to attrition

**核心机制：计时器 (Timer) + 回溯 (Rewind)**

计时器每回合计时前进一格，在特定刻度(3/6/9/12)触发不同效果。回溯牌可以重置计时器或撤销本回合操作。需要精准规划回合节奏——上限极高但下限极低 / Timer advances each turn, triggering effects at specific ticks (3/6/9/12). Rewind cards reset the timer or undo actions. Requires precise turn planning — extremely high ceiling but very low floor.

### 打法技巧 / Playstyle Tips

- **了解刻度效果**：刻度3=单体伤害，刻度6=AOE+减益，刻度9=强力增益，刻度12=终极效果。每个角色的刻度效果不同，先检查你的角色 / **Know your ticks**: tick 3 = single dmg, tick 6 = AoE + debuff, tick 9 = strong buff, tick 12 = ultimate. Effects vary per character — check yours first
- **回溯是双刃剑**：使用回溯后你本回合获得的计时器推进会重置——注意节奏 / **Rewind is a double-edged sword**: it resets Timer progress gained this turn — watch your pacing
- **不要浪费过牌**：时计师有很多过牌手段，保持手牌流畅运转 / **Don't neglect draw**: Chronomancer has many draw tools — keep your hand flowing
- **控制场面优于爆发**：时计师的爆发集中在刻度9-12，在此之前以控制为主 / **Control over burst**: Chronomancer's burst is at ticks 9-12, focus on control before that
- **高难度角色**：不建议新手第一个尝试时计师——建议先玩铸甲师和织影者 / **High difficulty**: not recommended as first character — try Ironwarden or Shadowweaver first

核心思路：均衡积累三种充能，释放三元素终极融合。爆发力最强 / Core idea: balance all three charges for triple-element ultimate Fusion. Highest burst potential.

| 阶段 / Phase | 选卡重点 / Card Focus | 策略 / Strategy |
|------|------|------|
| 第1章 / Act 1 | 火花, 冰晶, 闪电 / Spark, Ice Crystal, Lightning Bolt | 均衡获取三种基础充能牌 / Balance all three basic charge cards |
| 第2章 / Act 2 | 三种光环中拿到至少1-2种 / Get 1-2 Aura cards | 自动产能降低手牌依赖 / Auto-generation reduces hand reliance |
| 第3章 / Act 3 | 元素风暴, 元素王冠遗物 / Elemental Storm, Elemental Crown relic | 瞬间满充能爆发体系 / Instant full-charge burst system |

**火焰灼烧流 / Fire Burn**

核心思路：专注火+冰充能，利用灼烧持续伤害磨死敌人。适合高血量BOSS战 / Core idea: focus Fire+Ice, use burn DoT to wear down enemies. Good for high-HP boss fights.

| 阶段 / Phase | 选卡重点 / Card Focus | 策略 / Strategy |
|------|------|------|
| 第1章 / Act 1 | 火花x2, 冰晶 / Spark x2, Ice Crystal | 火冰基础 / Fire-Ice foundation |
| 第2章 / Act 2 | 火焰光环, 冰霜光环, 烈焰风暴(高伤火系) / Fire Aura, Frost Aura, Flame Storm (high fire dmg) | 自动产能+高伤 / Auto-generation + high damage |
| 第3章 / Act 3 | 永恒火焰遗物, 灼烧增效牌 / Eternal Flame relic, burn amp cards | 灼烧叠加体系 / Burn stack system |

### 打法技巧 / Playstyle Tips

- **灵活切换融合组合**：不要固定一种模式。群怪用火+雷(AOE爆发)，BOSS用火+冰(持续伤害)，需要控制用冰+雷(减速) / **Adapt Fusion combos**: don't lock into one pattern. Mobs = Fire+Lightning (AoE), Boss = Fire+Ice (DoT), Control = Ice+Lightning (slow)
- **光环牌是中后期核心**：自动产能让你每回合稳定获得充能，解放出手空间打伤害 / **Aura cards are mid-late core**: auto-generation frees your hands for damage cards
- **注意充能上限**：默认充能上限5层，超过的部分会浪费——及时释放融合技能 / **Watch charge cap**: default cap is 5 — excess is wasted, release Fusion skills in time
- **转换核心遗物非常强力**：可以根据当前需要灵活转换元素 / **Conversion Core relic is very strong**: freely convert elements based on current need
- **元素风暴是游戏最强爆发卡之一**：瞬间获得2层全元素，配合融合术打出毁灭性combo / **Elemental Storm is among the game's best burst cards**: instantly 2 of each charge, pair with Fusion for devastating combo

## 常见错误与纠正 / Common Mistakes & Fixes

| 错误 / Mistake | 纠正 / Fix |
|------|------|
| **卡组太厚 / Deck too thick** | 控制在20-25张以内，多去商店删牌 / Keep under 20-25 cards, prioritize removal at shops |
| **忽视AOE / Ignoring AoE** | 每个角色都需要至少1-2张AOE，第2章没有AOE等于自杀 / Every character needs 1-2 AoE — no AoE in Act 2 is suicide |
| **缩放不足 / Insufficient scaling** | 如果第2章末尾还没有缩放机制，这局基本走远了 / If no scaling by end of Act 2, the run is likely doomed |
| **过度追求完美Build / Forcing a specific build** | 根据实际拿到的牌灵活调整，不要死磕一个流派 / Adapt to what you're offered — don't force one archetype |
| **消耗品囤积 / Hoarding consumables** | 药水是消耗品，不用就是浪费——该用就用 / Potions are consumable — not using them is the waste |
| **路线选择错误 / Bad pathing** | 了解自己Build的强度——强就走精英多拿遗物，弱就走小怪苟活 / Know your build's strength — strong = take elites, weak = avoid them |
