# 流派构建 / Builds & Archetypes

杀戮尖塔2在初代基础上引入了新角色和新机制——但Build构建的核心逻辑仍然相同。本章覆盖新版本中的主流Build和角色构建思路 / Slay the Spire 2 builds upon the original with new characters and mechanics — but the core logic of build construction remains the same. This chapter covers the meta builds and character construction strategies in the new version.

---

## 新角色Build / New Character Builds

### 死灵法师 / Necromancer (新角色 / New Character)

死灵法师的机制围绕"亡灵"和"献祭"——召唤亡灵随从并通过献祭它们获得强力效果 / Necromancer's mechanic revolves around "Undead" and "Sacrifice" — summon undead minions and sacrifice them for powerful effects.

| Build方向 / Build | 核心卡 / Core Cards | 策略 / Strategy |
|---|---|---|
| **亡灵大军 / Undead Army** | 骷髅召唤(2费-召唤2骷髅)+ 亡灵强化(所有亡灵+2力量)+ 墓穴(每回合自动召唤) / Skeleton Summon (2-cost, summon 2 skeletons) + Undead Strengthen (+2 STR to all undead) + Tomb (auto-summon each turn) | 堆亡灵数量——配合群体强化，亡灵越多输出越高 / Stack undead count — with group buff, more undead = more damage |
| **献祭爆发 / Sacrifice Burst** | 黑暗献祭(摧毁亡灵-获得力量/能量)+ 亡灵再生(亡灵死亡抽牌)+ 灵魂风暴(献祭时触发AOE) / Dark Sacrifice (destroy undead for STR/energy) + Undead Rebirth (draw on undead death) + Soul Storm (AOE on sacrifice) | 低费亡灵快速召唤→献祭循环→每次献祭触发多层效果 / Low-cost undead fast summon → sacrifice loop → multi-trigger per sacrifice |
| **亡语流 / Deathrattle** | 缚灵(亡灵死亡时叠甲)+ 碎骨(亡灵死亡时AOE伤害)+ 灵魂链接(亡灵死亡时回血) / Wraith (block on undead death) + Bone Shard (AOE on undead death) + Soul Link (heal on undead death) | 亡灵什么时候死不重要——只要死了就触发效果。配合群体AOE / Don't care when undead die — effects trigger on death. Pair with mass AOE |

**死灵法师核心机制**: 亡灵随从占用场上位置(最多5个)——每个亡灵有独立血量和攻击力。召唤亡灵消耗能量——但献祭亡灵可以回收能量 / **Necromancer core mechanic**: Undead minions occupy board slots (max 5) — each has independent HP and attack. Summoning costs energy — sacrificing undead refunds energy.

### 游侠 / Ranger (新角色 / New Character)

游侠是高速高机动角色——专注于"连击"和"标记"系统 / Ranger is a fast, high-mobility character — focused on "Combo" and "Mark" systems.

| Build方向 / Build | 核心卡 / Core Cards | 策略 / Strategy |
|---|---|---|
| **连击流 / Combo Build** | 快速射击(0费-造成3点伤害-连击+1)+ 终结技(连击>5时造成双倍伤害)+ 冲刺(连击>3时额外抽牌) / Quick Shot (0-cost, 3 dmg, combo+1) + Finisher (double damage if combo>5) + Dash (extra draw if combo>3) | 用0费卡快速叠连击→连击数高了后用终结技爆发 / Stack combo with 0-cost cards → burst with Finisher at high combo |
| **标记流 / Mark Build** | 猎人标记(1费-敌人获得"标记")+ 狙击(对标记敌人造成3倍伤害)+ 陷阱(对标记敌人触发额外效果) / Hunter's Mark (1-cost, apply Mark) + Snipe (3x damage vs Marked) + Trap (extra effect vs Marked) | 上标记→高倍率伤害/效果触发。适合控制和爆发 / Apply Mark → high multiplier damage/effect trigger. Control + burst |
| **毒刃流 / Poison Blade** | 淬毒(武器附加毒)+ 毒刃(造成伤害+叠毒)+ 剧毒爆发(毒层数造成伤害) / Poison Coating (weapon poison) + Poison Blade (damage + poison) + Toxin Burst (deal poison stack damage) | 叠毒+物理输出双线——类似STS初代的毒+小刀混合 / Poison + physical dual-scaling — similar to STS1's poison + blade hybrid |

## Build构建核心原则 / Core Build Principles

| 原则 / Principle | 说明 / Description |
|---|---|
| **早期明确方向** | Act 1前半段就要确定Build方向——不要等到Act 2才决定走什么流派 / Determine build direction in early Act 1 — don't wait until Act 2 |
| **1个核心+2个辅助** | 每个Build需要1个核心输出思路(如毒/力量)+1个防御手段+1个过牌引擎 / Each build needs 1 core DPS (poison/STR) + 1 block plan + 1 draw engine |
| **牌组不要偏科** | 全攻击牌=死得早。全防御牌=打不死人。攻击:防御:过牌≈ 1:1:1 / All attack = early death. All block = can't kill. Ratio: attack:block:draw ≈ 1:1:1 |
| **灵活转型** | 如果拿不到核心卡——及时放弃当前Build转其他 / If core cards don't appear — pivot to another build early |
