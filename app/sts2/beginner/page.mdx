# 新手入门 / Beginner Guide

《杀戮尖塔2》继承了前作的卡牌构建Roguelike核心玩法，但引入了大量新系统和新角色。无论你是老玩家还是新玩家，都有很多需要重新学习的内容 / Slay the Spire 2 inherits the deck-building roguelike core of its predecessor but introduces many new systems and characters. Whether you're a veteran or a newcomer, there's plenty to learn.

---

## 快速入门 / Quick Start

### 游戏目标 / Goal

选择一名角色，从高塔底部开始攀登，每通过一个章节(Act)击败一名BOSS，最终击败高塔顶端的最终BOSS / Choose a character, start at the bottom of the Spire, clear each Act by defeating its boss, and ultimately conquer the final boss at the top.

### 基本流程 / Basic Flow

1. **选择角色 / Choose Character** — 4名全新角色，各有独特的初始卡组和初始遗物 / 4 brand-new characters, each with unique starting deck and relic
2. **规划路线 / Plan Path** — 在地图上选择前进路线：精英怪(强力遗物)、篝火(升级/回复)、商店(购物)、问号(随机事件) / Plan your route: Elites (relics), Campfires (upgrade/heal), Shop (buy/sell), ? (events)
3. **构筑牌组 / Build Deck** — 战后选卡，删减初始牌，打造强力牌组 / Choose cards post-combat, remove starter cards, build a powerful deck
4. **击败BOSS / Defeat Boss** — 每章末尾有BOSS战，击败后进入下一章 / Each Act ends with a boss — defeat it to proceed
5. **最终决战 / Final Battle** — 击败第4章最终BOSS即为通关 / Defeat the Act 4 final boss to win the run

## 第2-4章路线策略 / Act 2-4 Path Strategy

| 章节 / Act | 重点 / Focus | 策略 / Strategy |
|:----:|------|------|
| **1** | 建立Build基础 / Establish foundation | 2精英+1-2篝火升级+1商店删牌——尽量多打精英拿遗物 / 2 elites + 1-2 campfires upgrade + 1 shop removal — fight as many elites as possible for relics |
| **2** | Build定型 / Solidify build | 第2章精英是最危险的——评估你的Build强度再决定打不打。Build不够强就走小怪和问号。第2章至少有1次篝火留给精英前的回复 / Act 2 elites are the most dangerous — evaluate your build before challenging. Weak build = avoid elites. Save at least 1 campfire for healing before elite fights |
| **3** | 最终完善 / Final polish | 所有资源用来找关键卡/遗物。第3章的精英战是最终装备检查。如果打不过精英，你可能也打不过BOSS / Invest everything in key cards/relics. Act 3 elites are the final gear check. If you can't beat elites, you likely can't beat the boss |
| **4** | 决战准备 / Final prep | 只有1个篝火+1个商店+1个精英+最终BOSS。全力输出。如果有回城卷轴，可以考虑提前规划路线 / 1 campfire + 1 shop + 1 elite + final boss. Go all-in on damage. If you have a Town Portal scroll, consider strategic pathing |

## 进阶路线规划 / Advanced Pathing Strategy

### 如何评估精英强度 / How to Evaluate Elite Difficulty

| 因素 / Factor | 说明 / Description |
|------|------|
| **卡组攻击力 / Deck Damage** | 能否在3-4回合内击杀精英？如果能，大胆打 / Can you kill the elite in 3-4 turns? If yes, go for it |
| **防御能力 / Defense** | 能否承受精英2轮攻击？如果不能，先找篝火回血或升级防御牌 / Can you survive 2 elite attacks? If not, campfire heal or upgrade defense first |
| **AOE能力 / AoE Capability** | 某些精英带小兵——没有AOE会很痛苦 / Some elites have adds — no AoE = painful |
| **当前血量 / Current HP** | 血量低于60%时，打精英风险极高 / Below 60% HP, elite fighting is extremely risky |
| **消耗品存量 / Consumables** | 有强力药水或卷轴时，精英胜率大幅提升 / Strong potions/scrolls significantly improve elite win rate |

### 篝火使用进阶 / Advanced Campfire Use

| 场景 / Scenario | 推荐操作 / Action |
|------|------|
| **血量80%+且核心卡未升级 / 80%+ HP with unupgraded key cards** | 升级卡牌 / Upgrade |
| **血量50-80% / HP 50-80%** | 如果是精英或BOSS前——休息；如果是小怪前——升级 / Rest if before elite/boss; upgrade if before normal fights |
| **血量低于50% / Below 50% HP** | 必须休息——不要冒险 / Must rest — don't gamble |
| **所有核心卡已升级 / All key cards upgraded** | 考虑休息。多余篝火可以用来回血以应对意外 / Consider resting. Extra campfire HP is insurance |
| **有回城卷轴 / Have Town Portal** | 可以冒险先走精英路线，打不过再用卷轴回篝火 / Can risk a harder path and retreat if needed |

## 与一代的关键差异 / Key Differences from STS1

| 差异 / Difference | 说明 / Description |
|------|------|
| **全新角色 / New Characters** | 4名全新角色，任何老角色都不会回归。每个角色有全新的机制和卡牌 / All 4 characters are brand-new. None from STS1 return. Each has unique mechanics and card pools |
| **视觉大升级 / Visual Overhaul** | 从像素风升级为精美手绘风格，角色立绘、卡牌插图、场景全部重制 / Upgraded from pixel art to beautiful hand-drawn style — characters, card art, and environments are all remade |
| **全新资源系统 / New Resource System** | 除能量外，部分角色拥有专属资源(如狂热值、专注力、印记等) / Beyond energy, some characters have unique resources (Fury, Focus, Marks, etc.) |
| **卡牌强化系统 / Card Upgrade 2.0** | 篝火升级不再是唯一方式，特定事件、商店和战斗奖励也可升级卡牌 / Campfire upgrades aren't the only way — events, shops, and combat rewards can also upgrade cards |
| **遗物交互升级 / Relic Interactions** | 遗物之间的组合效果更多，部分遗物可主动激活 / More relic synergies, some relics have active abilities |
| **药水→消耗品 / Potions → Consumables** | 药水系统扩展为更广泛的消耗品系统，包含药水、卷轴、符咒等 / Potions expanded into a broader consumable system including potions, scrolls, and charms |
| **路线选择更多 / More Path Variety** | 每章地图更大，分支更多，隐藏房间和秘密路径 / Larger maps per Act, more branches, hidden rooms and secret paths |
| **新增传说卡牌 / Legendary Cards** | 极其稀有的卡牌等级，效果改变游戏规则——只有特定事件可获得 / Extremely rare card tier with game-changing effects — only from specific events |
| **可激活遗物 / Active Relics** | 部分遗物在战斗中可以主动点击使用，消耗一次使用次数 / Some relics can be clicked during combat for an active effect, limited uses |

## 新玩家入门建议 / Tips for New Players

| 建议 / Tip | 说明 / Description |
|------|------|
| **从推荐角色开始 / Start with Recommended** | 选择界面会标注推荐新手使用的角色，他们的机制更直接 / The character select screen marks beginner-friendly characters with more straightforward mechanics |
| **前期攻击优先 / Prioritize Attacks Early** | Act 1前期卡组伤害不足，优先拿攻击牌确保能打过精英 / Early Act 1, your deck lacks damage — prioritize attacks to survive elite encounters |
| **不要忽略防御 / Don't Ignore Defense** | 纯攻击卡组在Act 2以后会暴毙，注意平衡格挡和回复 / Pure offense dies in Act 2+ — balance your deck with block and sustain |
| **利用篝火升级 / Use Campfires Wisely** | 前期多升级核心输出卡，中后期根据情况选择休息或升级 / Upgrade key damage cards early; mid-to-late game, choose between rest and upgrade based on your HP |
| **认识敌人意图 / Learn Enemy Intents** | 敌人头顶的意图图标指示下一回合行动，据此规划出牌 / Enemy intent icons show their next action — plan your plays accordingly |
| **精简卡组 / Keep Deck Lean** | 15-25张是最佳区间。在商店优先删除初始卡 / 15-25 cards is ideal. Prioritize removing starter cards at shops |
| **适应而非强求流派 / Adapt, Don't Force** | 根据拿到的卡和遗物决定流派，固定思维是新手最大的敌人 / Build around what you're offered — forcing a specific archetype is the #1 beginner mistake |
| **观察敌人行为模式 / Learn Attack Patterns** | 多数敌人有固定行为循环——观察几回合后就能预测行动 / Most enemies follow fixed loops — observe for a few turns and you can predict their actions |
| **不要害怕重开 / Don't Fear Restarts** | 有时候开局不顺，重新开始是正常的——每个失败的Run都让你更了解游戏 / Sometimes runs go poorly — restarting is normal. Every failed run teaches you something |
| **善用游戏暂停 / Use Pause Feature** | 需要思考时按暂停——这允许你慢慢分析局势，尤其适合新手 / Pause when you need to think — it lets you analyze the situation calmly |

### 卡组节奏论 / Deck Tempo Theory

| 节奏类型 / Tempo Type | 说明 / Description | 适用 / Applies To |
|------|------|------|

| 检查项 / Check | 合格标准 / Passing Grade | 危险信号 / Red Flag |
|------|------|------|
| **第1章精英 / Act 1 Elite** | 3-4回合内击杀 / Kill in 3-4 turns | 需要6+回合 / Takes 6+ turns |

## 消耗品系统详解 / Consumable System

STS2将药水系统扩展为三类消耗品——合理使用是高难度通关的关键 / STS2 expands potions into three consumable types — smart use is key to high-difficulty clears:

| 类型 / Type | 特点 / Trait | 示例 / Example |
|:----:|------|------|
| **药水 / Potion** | 战斗中一次性效果 / One-time combat effect | 烈焰药水(20伤害)、铁壁药水(15护甲) / Fire Potion, Iron Potion |
| **卷轴 / Scroll** | 非战斗用途，任意界面使用 / Non-combat, any screen | 回城卷轴(回到上个篝火)、升级卷轴(升1张牌) / Town Portal, Upgrade Scroll |
| **符咒 / Charm** | 持续多场战斗的Buff / Multi-combat buff | 力量符咒(下3场+2力量) / Strength Charm (+2 Str 3 combats) |

**消耗品管理**: 药水不要舍不得用——带着3瓶满的药水进Boss战时如果没用过，说明之前省太过了。卷轴要灵活使用——回城卷轴在走错路线时特别有用 / **Consumable management**: Don't hoard potions — entering boss with 3 full potions means you conserved too much. Use scrolls flexibly — Town Portal is great for pathing mistakes.

**消耗品优先级 / Consumable Priority**:
- **精英战 / Elite fights**: 准备1瓶战斗药水(伤害/格挡) / Prepare 1 combat potion (damage/block)
- **BOSS战 / Boss fights**: 准备至少2瓶战斗药水+1个卷轴(如升级卷轴)/ Prepare at least 2 combat potions + 1 scroll
- **紧急情况 / Emergency**: 血量极低时，回复类药水比升级更重要 / When critically low HP, heal potions beat upgrades
- **回城卷轴妙用 / Town Portal tricks**: 可以用来"重置"路线——走一条高风险路线，拿到奖励后用回城卷轴回到篝火休息 / Use TP to "reset" your path — take a high-risk route, grab rewards, then TP back to campfire

## 常见术语 / Common Terminology

| 术语 / Term | 意思 / Meaning |
|------|------|
| **AOE / AOE** | 群体伤害——一次攻击所有敌人 / Area of Effect — hits all enemies at once |
| **过牌 / Draw** | 抽取更多卡牌的能力 / Ability to draw more cards |
| **运转 / Engine** | 让卡组自动高效循环的核心配合 / Core synergy that makes your deck cycle efficiently |
| **费用曲线 / Curve** | 卡组的能量消耗分布，平衡低费和高费卡 / Energy cost distribution of your deck |
| **缩放 / Scaling** | 战斗中越打越强的能力(叠力量、叠球体等) / Ability to grow stronger as combat progresses |
| **OTK / OTK** | 一回合击杀 / One Turn Kill |
| **进阶 / Ascension** | 通关后解锁的难度系统，增加游戏挑战 / Post-win difficulty system that adds challenge |
| **DPS / DPS** | 每回合输出能力——决定你能否在敌人杀死你之前杀死它 / Damage per turn — determines if you kill enemies before they kill you |
| **致胜路线 / Win Condition** | 你击败BOSS的核心策略——围绕这个构建卡组 / Your core strategy to defeat the boss — build around it |
| **消耗品 / Consumable** | 药水、卷轴、符咒的总称——一次性使用物品 / Collective term for potions, scrolls, and charms — single-use items |

## 第4章解锁条件 / Act 4 Unlock Conditions

要进入第4章(虚空之心)，你需要在第3章完成以下步骤 / To reach Act 4 (Heart of Void), complete these steps in Act 3:

1. **获得3个宝石碎片 / Collect 3 Gem Shards** — 从第3章的特定精英和BOSS获得特殊遗物 / Special relics from specific Act 3 elites and bosses
2. **在末尾篝火选择"进入虚空" / Choose "Enter the Void" at the final campfire** — 必须持有全部3个碎片 / Must have all 3 shards
3. **击败虚空守卫者精英 / Defeat the Void Guardian elite** — 第4章唯一的精英战 / The only elite fight in Act 4
4. **击败虚空之心 / Defeat the Heart of Void** — 最终BOSS / The final boss

**注意**: 第一次通关不需要进第4章。建议新手先专注于击败第3章BOSS解锁进阶系统 / **Note**: Act 4 is not required for first victory. Beginners should focus on beating the Act 3 boss to unlock Ascension.
