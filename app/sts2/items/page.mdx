# 物品与装备 / Items & Equipment

杀戮尖塔2在初代遗物/药水/卡牌系统上做了大量扩展——增加了新的稀有度等级、新类型遗物和全新的"装备"系统 / Slay the Spire 2 expands significantly on the original relic/potion/card system — adding new rarity tiers, new relic types, and a brand new "Equipment" system.

---

## 装备系统 / Equipment System

杀戮尖塔2新增了"装备"系统——与遗物不同，装备占用"装备槽"(共3个)且可以在战斗中更换 / STS2 adds an "Equipment" system — unlike relics, equipment uses "Equipment Slots" (3 total) and can be swapped during combat.

| 装备类型 / Type | 效果 / Effect | 获取方式 / Source |
|---|---|---|
| **武器 / Weapon** | 增加攻击牌伤害+可能附加效果 / Increases attack card damage + possible bonus effects | 商店购买、精英掉落、Boss掉落 / Shop, elite drop, boss drop |
| **护甲 / Armor** | 增加回合起始护甲+减伤 / Increases starting block + damage reduction | 商店购买、宝箱 / Shop, chest |
| **饰品 / Accessory** | 提供特殊效果——如每回合额外抽牌 / Special effects — extra draw per turn, etc. | 事件奖励、Boss掉落 / Event reward, boss drop |
| **戒指 / Ring** | 角色专属——每个角色有独特的戒指效果 / Character exclusive — unique effect per character | 角色初始获得 / Character starter |

## 装备系统策略 / Equipment Strategy

| 策略 / Strategy | 说明 / Description |
|---|---|
| **武器优先** | 装备槽有限——武器通常是最优先的。增加攻击牌伤害直接影响通关速度 / Limited slots — weapons are usually top priority. Attack damage increase directly impacts clear speed |
| **护甲与防御平衡** | 如果牌组的防御不够——护甲类比武器更重要 / If deck lacks block — armor > weapons |
| **饰品看Build** | 饰品效果根据Build变化极大——如连击流Build中+连击的饰品价值极高 / Accessory value varies hugely by build — combo+ accessory is invaluable for combo builds |
| **装备升级** | 在休息处可以用$100升级一件装备——永久提升效果 / Upgrade at rest sites for $100 — permanent effect improvement |
| **战斗中更换** | 装备可以在战斗中更换——但消耗本回合1点能量 / Swappable during combat — costs 1 energy for the turn |
