# BOSS攻略 / Boss Strategies

本章详解杀戮尖塔2所有BOSS的机制和打法策略，包含站位建议、前期准备、分阶段对策等 / Comprehensive boss strategies for Slay the Spire 2, including mechanics, preparation tips, and phase-by-phase tactics.

---

## 通用BOSS准备 / General Boss Preparation

| 准备事项 / Preparation | 说明 / Description |
|------|------|
| **了解BOSS机制 / Know the Boss** | 每章开局确认BOSS，围绕其机制构建卡组 / Check the boss at the start of each Act and build your deck around countering it |
| **药水/消耗品 / Consumables** | 在BOSS门前确保携带合适的消耗品，不要浪费在小怪战 / Save your best consumables for the boss |
| **血量管理 / HP Management** | BOSS门前尽量满血。如果血量低，选择篝火休息而不是升级 / Enter boss fights at or near full HP — rest instead of upgrade if needed |
| **卡组聚焦 / Deck Focus** | BOSS战不需要全面——专注于你最擅长的赢法 / Don't try to do everything — focus on your win condition |
| **记住缩放 / Remember Scaling** | 大部分BOSS有多个阶段，你需要能越打越强的缩放机制 / Most bosses have multiple phases — you need scaling that grows stronger as the fight progresses |
| **删到关键牌 / Trim to Essentials** | BOSS前最后商店：如果能删掉一张诅咒或冗余初始牌，优先做这个 / Last shop before boss: if you can remove a curse or redundant starter, prioritize it |
| **检查遗物组合 / Check Relic Combos** | 确认你的遗物之间有良好的协同——没有冲突或无效遗物 / Ensure your relics have good synergy — no conflicting or useless relics |

### 按角色的BOSS准备 / Character-Specific Boss Prep

| 角色 / Character | 战前重点 / Pre-fight Focus |
|------|------|
| **织影者 / Shadowweaver** | 确保有足够的印记叠层手段和终结技——需要高缩放伤害 / Ensure enough Mark stacking + Finishers — need high scaling damage |
| **铸甲师 / Ironwarden** | 护甲叠层+铁砧爆发组合——注意BOSS是否有无视护甲的攻击 / Armor stacking + Anvil burst — watch for armor-bypass attacks |
| **时计师 / Chronomancer** | 计时器节奏对齐BOSS阶段变化——回溯留给关键回合 / Align Timer rhythm with boss phases — save Rewind for key turns |
| **元素使 / Elementalist** | 充能积累+融合爆发——根据BOSS弱点选择元素组合 / Charge accumulation + Fusion burst — choose element combo based on boss weakness |

## 第一章BOSS：废王 / The Rust King

### 基本信息 / Basic Info

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **HP / HP** | 120-140 |
| **基础攻击 / Base Attack** | 12-15 |
| **特殊机制 / Special** | 召唤史莱姆、全屏AOE / Summon slimes, full-screen AoE |
| **核心弱点 / Core Weakness** | 虚弱效果、AOE攻击 / Weak effect, AoE attacks |

### 推荐卡牌 / Recommended Cards

| 卡牌类型 / Card Type | 推荐 / Recommended | 理由 / Reason |
|------|------|------|
| **AOE攻击 / AoE Attack** | 旋风斩、火球术 / Whirlwind, Fireball | 清理召唤的史莱姆 / Clean up summoned slimes |
| **虚弱 / Weak** | 虚弱诅咒、缴械 / Weak Curse, Disarm | 减少BOSS输出——尤其是第二阶段 / Reduce boss damage — especially Phase 2 |
| **单体高伤 / Single-target** | 终结一击、铁砧猛击 / Finishing Strike, Anvil Strike | 快速击杀本体 / Fast boss kill |
| **格挡 / Block** | 铁壁、冰晶 / Iron Wall, Ice Crystal | 保证生存 / Survival guarantee |

### 分阶段打法 / Phase-by-Phase

**第一阶段（100%-50%）/ Phase 1 (100%-50%)**

| 行动 / Action | 频率 / Frequency | 应对 / Counter |
|------|:----:|------|
| 普通攻击12-15 / Normal attack 12-15 | 每回合 / Every turn | 常规格挡 / Standard block |
| 召唤2个废墟史莱姆 / Summon 2 Ruin Slimes | 每3回合 / Every 3 turns | 带AOE清理 / AoE to clear |

**策略 / Strategy**:
- 用AOE卡清理召唤的史莱姆，同时用单体攻击打BOSS / Use AoE for slimes while single-target damaging the boss
- 不要浪费高费AOE只打BOSS——史莱姆虽然血少但累积伤害 / Don't waste high-cost AoE on boss only — slimes add up
- **虚弱**效果可以减少BOSS的大量输出 / **Weak** significantly reduces the boss's damage output
- 推荐角色：**铸甲师**（护甲反击流可以轻松应对） / Recommended: **Ironwarden** (Armor Counter handles this easily)

**第二阶段（50%-0%）/ Phase 2 (50%-0%)**

| 行动 / Action | 说明 / Description | 应对 / Counter |
|------|------|------|
| 攻击提升至18-22 / Attack up to 18-22 | 不再召唤 / No more summons | 提升格挡量 + 虚弱 / Increase block + Weak |
| 全屏AOE 8点 / Full AoE 8 damage | 每回合 / Every turn | 护甲和格挡并重，AOE无法被闪避 / Both armor and block matter |

**策略 / Strategy**:
- 这是DPS检测——第二阶段你会被持续AOE消耗 / This is a DPS check — sustained AoE drains you
- 铸甲师的护甲在AOE面前比格挡更有优势 / Ironwarden's Armor is better than block against this AoE
- 尽量在2-3回合内结束第二阶段 / Aim to finish Phase 2 in 2-3 turns
- 保留爆发性消耗品（如力量药水、伤害卷轴）用于此阶段 / Save burst consumables for this phase

**角色特定技巧 / Character-Specific Tips**:
- **织影者**: 第1阶段用毒刃叠中毒+印记，10层终结技直接打第二阶段的爆发 / Apply poison + marks in Phase 1, Finisher at 10 stacks for Phase 2 burst
- **铸甲师**: 第1阶段叠满护甲，第二阶段用铁砧猛击爆发 / Max armor in Phase 1, Anvil Strike burst in Phase 2
- **时计师**: 计时器刻度6对齐进第二阶段的时间点 / Align Timer tick 6 with Phase 2 transition
- **元素使**: 火+雷融合清史莱姆，火+冰融合打BOSS本体 / Fire+Lightning Fusion for slimes, Fire+Ice for boss

## 第二章BOSS：深渊之主 / The Abyss Lord

### 基本信息 / Basic Info

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **HP / HP** | 200-240 |
| **基础攻击 / Base Attack** | 12-15 → 25-30 |
| **特殊机制 / Special** | 脆弱+虚弱debuff、召唤蠕虫、卡牌转化虚空 / Brittle + Weak debuffs, summon worms, card-to-Void conversion |
| **核心弱点 / Core Weakness** | debuff净化、高缩放输出 / Debuff cleanse, high scaling damage |

### 推荐卡牌 / Recommended Cards

| 卡牌类型 / Card Type | 推荐 / Recommended | 理由 / Reason |
|------|------|------|
| **净化 / Cleanse** | 净化药剂、神圣庇护 / Purify Potion, Divine Protection | **最关键**——移除脆弱和虚弱 / **Most critical** — remove Brittle and Weak |
| **AOE / AoE** | 暗影爆发、旋风斩 / Shadow Burst, Whirlwind | 清理召唤的蠕虫 / Clear summoned worms |
| **缩放 / Scaling** | 夜幕降临、计时加速 / Nightfall, Temporal Haste | 第三阶段需要高爆发 / Need high burst for Phase 3 |
| **高单体伤害 / High ST Dmg** | 终结一击、元素风暴 / Finishing Strike, Elemental Storm | 快速击杀BOSS / Fast boss kill |

### 分阶段打法 / Phase-by-Phase

**第一阶段（100%-60%）/ Phase 1 (100%-60%)**

| 行动 / Action | 应对 / Counter |
|------|------|
| 攻击12-15 + 脆弱+虚弱 / Attack 12-15 + Brittle + Weak | 携带净化牌（如**净化药剂**或角色自带的净化技能）/ Bring cleanse (e.g., **Purify Potion** or character cleanse skills) |
| 每3回合召唤1个深渊蠕虫 / Summon 1 Abyss Worm every 3 turns | 优先击杀蠕虫，不要让场上堆叠过多 / Kill worms first to prevent accumulation |

**策略 / Strategy**:
- **净化debuff是这场战斗的关键**——脆弱让你受到翻倍伤害，虚弱减少你一半输出 / **Cleanse is critical** — Brittle doubles incoming damage, Weak halves your output
- 蠕虫必须尽快处理——它们叠加脆弱会让你被BOSS一击秒杀 / Kill worms fast — stacked Brittle from worms + boss = oneshot
- 尝试在进入第二阶段前建立缩放机制 / Try to establish your scaling before Phase 2

**第二阶段（60%-30%）/ Phase 2 (60%-30%)**

| 行动 / Action | 应对 / Counter |
|------|------|
| 攻击18-22 / Attack 18-22 | 格挡量需要提升 / Increase block output |
| 召唤2个蠕虫 / Summon 2 worms | AOE清理效果更好 / AoE is more efficient |
| 黑暗领域（每回合全体5伤害）/ Dark Aura (5 AoE/turn) | 持续回复手段很重要 / Sustain becomes important |

**策略 / Strategy**:
- 这是过渡阶段——场面压力增加但还在可控范围 / Transition phase — pressure increases but manageable
- 如果角色有AOE能力，这是发挥的好时机 / AoE shines here
- 尽量在黑暗领域造成太多伤害前进入第三阶段 / Try to push to Phase 3 before Dark Aura chips you too much

**第三阶段（30%-0%）/ Phase 3 (30%-0%)**

| 行动 / Action | 应对 / Counter |
|------|------|
| 攻击25-30 / Attack 25-30 | 高额格挡或护甲 / High block or armor needed |
| 黑暗领域10伤害 / Dark Aura 10 dmg | 回复药水或自身回复 / Heal potions or self-heal |
| 每回合一张牌变虚空 / Each turn 1 card → Void | **全力输出——拖越久你失去的卡牌越多** / **Go all out — the longer it goes, the more cards you lose** |

**策略 / Strategy**:
- 这里没有战术可言——只有纯粹的DPS / No tactics here — pure DPS
- 所有消耗品全部用上 / Burn all consumables
- 如果你有OTK潜力，现在就是发挥的时候 / If you have OTK potential, this is the time
- 推荐角色：**织影者**（暗影印记的缩放机制在这里表现出色）/ Recommended: **Shadowweaver** (Shadow Mark scaling excels here)

**角色特定技巧 / Character-Specific Tips**:
- **织影者**: 暗影印记叠满10层后用终结技——配合夜幕降临自动叠层 / Stack to 10 Marks then Finisher — pair with Nightfall for auto-stacking
- **铸甲师**: 用不灭熔炉维持护甲，钢铁风暴在第三阶段收尾 / Maintain armor with Unending Forge, finish with Steel Storm in Phase 3
- **时计师**: 回溯用来应对BOSS的虚空转化——如果关键牌被转化，回溯撤销 / Use Rewind to counter Void conversion — undo if key card is Voided
- **元素使**: 火+冰融合灼烧是最适合持续伤害的策略 / Fire+Ice Fusion burn is the best sustained damage approach

## 第三章BOSS：天穹守望者 / The Celestial Watcher

### 基本信息 / Basic Info

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **HP / HP** | 280-320 |
| **基础攻击 / Base Attack** | 14-18 → 25-30 |
| **特殊机制 / Special** | 星辰召唤（自爆）、星光AOE、易伤叠加 / Star summons (explode), Starlight AoE, Vulnerable stacking |
| **核心弱点 / Core Weakness** | 高单体爆发、易伤净化 / High single-target burst, Vulnerable cleanse |

### 推荐卡牌 / Recommended Cards

| 卡牌类型 / Card Type | 推荐 / Recommended | 理由 / Reason |
|------|------|------|
| **低费攻击 / Low-cost Attack** | 火花、时光穿刺 / Spark, Time Pierce | 高效清理星辰（10-15HP）/ Efficient Star clearing (10-15 HP) |
| **单体爆发 / Single Burst** | 铁砧猛击、终结一击 / Anvil Strike, Finishing Strike | 快速过渡阶段 / Fast phase transitions |
| **易伤净化 / Vuln Cleanse** | 净化药剂、神圣庇护 / Purify Potion, Divine Protection | 易伤&gt;2层=濒死 / Vuln&gt;2 stacks = near death |
| **格挡 / Block** | 铁壁、冰晶 / Iron Wall, Ice Crystal | 硬吃星辰爆炸 / Tank Star explosions |

### 分阶段打法 / Phase-by-Phase

**第一阶段（100%-66%）/ Phase 1 (100%-66%)**

| 行动 / Action | 应对 / Counter |
|------|------|
| 攻击14-18 / Attack 14-18 | 标准格挡 / Standard block |
| 每回合召唤1颗星辰（2回合后自爆12-15伤害）/ Summon 1 Star (explodes 12-15 after 2 turns) | 在星辰自爆前击杀它们——优先目标 / Kill Stars before they explode — top priority |

**策略 / Strategy**:
- 星辰是这场战斗的核心机制——让星辰自爆=吃额外伤害 / Stars are the core mechanic — letting them explode = free damage on yourself
- 每个星辰大约10-15HP，低费攻击牌可以轻松解决 / Stars have ~10-15 HP, cheap attacks handle them easily
- 尽量保持场上星辰数量为0 / Try to keep Stars at 0 on the field

**第二阶段（66%-33%）/ Phase 2 (66%-33%)**

| 行动 / Action | 应对 / Counter |
|------|------|
| 攻击18-22 / Attack 18-22 | 格挡提升 / More block |
| 星光AOE 10 + 1易伤 / Starlight AoE 10 + 1 Vulnerable | 净化易伤或承受伤害后回复 / Cleanse Vulnerable or heal through |
| 召唤2颗星辰 / Summon 2 Stars | AOE清理更高效 / AoE for efficiency |

**策略 / Strategy**:
- 易伤叠加超过2层非常危险——BOSS的攻击会变成36-44 / 2+ Vulnerable stacks are dangerous — boss attacks become 36-44
- 如果你有净化手段，在易伤&gt;2层时使用 / If you have cleanse, use it when Vulnerable &gt; 2 stacks
- 没有净化的话，在易伤回合打满格挡 / Without cleanse, max out block on Vulnerable turns

**第三阶段（33%-0%）/ Phase 3 (33%-0%)**

| 行动 / Action | 应对 / Counter |
|------|------|
| 攻击25-30 / Attack 25-30 | 最大格挡 / Max block |
| 星辰坠落（全体15-20）/ Starfall (all 15-20) | 护甲/格挡并重 + 回复 / Armor + block + heal |
| 星光升级（15伤+2易伤）/ Starlight upgraded (15 dmg + 2 Vulnerable) | 必须净化易伤——不净化=被秒 / MUST cleanse Vulnerable — otherwise oneshot |

**策略 / Strategy**:
- **最危险的阶段**——配合易伤，BOSS的伤害可以轻松达到60+/turn / **Most dangerous phase** — with Vulnerable, boss damage easily exceeds 60+/turn
- 如果你在2-3回合内不能击杀，必败 / If you can't kill in 2-3 turns, you're dead
- 推荐角色：**时计师**（回溯能力可以完美应对星辰爆炸的时间差）/ Recommended: **Chronomancer** (Rewind handles Star explosion timing perfectly)

**角色特定技巧 / Character-Specific Tips**:
- **织影者**: 用暗影风暴快速叠全队印记——星辰的HP很低，终结技溢出浪费 / Use Shadow Storm for fast AoE marks — Stars have low HP, avoid overkill with Finishers
- **铸甲师**: 护甲可以硬吃星辰爆炸——你是最不需要担心自爆的角色 / Armor tanks Star explosions — you care least about the explosion damage
- **时计师**: 回溯配合时间锁——让星辰在自爆前"停滞" / Rewind + Time Lock — "freeze" Stars before they explode
- **元素使**: 火+雷融合可以一击清掉场上所有星辰和BOSS / Fire+Lightning Fusion can clear all Stars and damage boss in one shot

## 第四章隐藏BOSS：虚空之心 / The Heart of Void

### 基本信息 / Basic Info

| 属性 / Stat | 数值 / Value |
|------|:----:|
| **HP / HP** | 400-500 |
| **基础攻击 / Base Attack** | 15（无视格挡）→ 30（无视格挡）/ 15 (unblockable) → 30 (unblockable) |

| **核心弱点 / Core Weakness** | 极高缩放输出、低费高效卡组 / Extremely high scaling output, low-cost efficient deck |

### 推荐卡牌 / Recommended Cards

| 卡牌类型 / Card Type | 推荐 / Recommended | 理由 / Reason |
|------|------|------|
| **高缩放 / High Scaling** | 计时加速、夜幕降临、不灭熔炉 / Temporal Haste, Nightfall, Unending Forge | 唯一的取胜方式——越打越强 / Only way to win — grow stronger over time |
| **低费高效 / Low-cost Efficient** | 火花、时光穿刺 / Spark, Time Pierce | 每费都要输出——没有多余能量给低效牌 / Every energy must deal damage |
| **过牌 / Draw** | 暗影爆发、回溯 / Shadow Burst, Rewind | 保持手牌不枯竭 / Keep hand from emptying |
| **高费慎拿 / Avoid High Cost** | 不要拿5+费的牌 / Avoid 5+ cost cards | 暗影复制会把高费牌用于攻击你 / Shadow Copy uses high-cost cards against you |

### 战前准备 / Pre-fight Preparation

| 准备 / Prep | 说明 / Description |
|------|------|
| **确保高缩放 / Ensure High Scaling** | 你需要能在5-8回合内打出400+伤害的能力——力量、暗影印记、元素融合等高缩放机制至关重要 / You need 400+ damage in 5-8 turns — Strength, Shadow Marks, Elemental Fusion, etc. are essential |

| **遗物选择 / Relic Choice** | 优先选择增加输出的遗物，格挡遗物价值有限 / Prioritize damage relics — block relics have limited value here |
| **删除高费牌 / Remove Expensive Cards** | 虚拟复制会针对你手中费用最高的牌——高费牌=自残 / Shadow Copy targets your highest-cost card — expensive cards = self-harm |

### 分阶段打法 / Phase-by-Phase

**第一阶段（100%-75%）/ Phase 1 (100%-75%)**

| 行动 / Action | 说明 / Description | 应对 / Counter |
|------|------|------|
| 虚空攻击15（无视格挡）/ Void Attack 15 (unblockable) | 无法格挡，只能靠回复承受 / Unblockable — must heal through | 带回复手段，或者在这一阶段快速推进 / Bring sustain or push through fast |
| 虚空脉冲——移除2张牌 / Void Pulse — removes 2 cards | 从牌组中永久移除（本场战斗）/ Permanently removed for this combat | 尽快推进到下一阶段以减少损失 / Push to next phase ASAP to minimize losses |

**策略 / Strategy**:
- 这一阶段相对温和——你的缩放机制应该在这期间建立起来 / This phase is relatively mild — establish your scaling here
- 不要使用消耗品——留着后面用 / Don't use consumables yet — save them
- 理想情况下2-3回合通过第一阶段 / Ideally push through in 2-3 turns

**第二阶段（75%-40%）/ Phase 2 (75%-40%)**

| 行动 / Action | 应对 / Counter |
|------|------|
| 虚空攻击20（无视格挡）/ Void Attack 20 (unblockable) | 回复/吸血手段很重要 / Heal/lifesteal becomes important |
| 虚空脉冲——移除3张牌 / Void Pulse — removes 3 cards | 如果你的卡组小于15张，这已经非常危险 / If your deck &lt; 15 cards, this gets dangerous |
| 暗影复制——复制你费用最高的牌 / Shadow Copy — copies your highest-cost card | 不要留高费AOE在手上——它会被用来打你自己 / Don't hold expensive AoE in hand — it'll be used against you |

**策略 / Strategy**:
- 注意你的手牌管理——如果你拿了**破灭**这样的高费牌，暗影复制会很痛苦 / Watch hand management — if you drafted a high-cost card, Shadow Copy hurts
- 开始使用部分消耗品，但至少留1-2个给第三阶段 / Start using some consumables but save 1-2 for Phase 3
- 如果卡组被移除太多，考虑牺牲部分血量快速推进 / If too many cards are removed, consider saccing HP to push through

**第三阶段（40%-0%）/ Phase 3 (40%-0%)**

| 行动 / Action | 应对 / Counter |
|------|------|
| 虚空攻击30（无视格挡）/ Void Attack 30 (unblockable) | 必须靠吸血或大量回复硬扛 / Must tank through lifesteal or massive heals |
| 每回合虚空脉冲（移除2张）/ Void Pulse every turn (removes 2) | **全力输出——你的卡组在快速消失** / **Go all out — your deck is vanishing** |
| 末日倒计时——5回合后999伤害 / Doom Countdown — 999 damage after 5 turns | **你最多有5个回合，通常在3-4回合内需结束战斗** / **You have at most 5 turns, realistically 3-4** |

**策略 / Strategy**:
- **这是杀戮尖塔2最极端的DPS检测** / **This is STS2's most extreme DPS check**
- 所有消耗品全部使用 / Use ALL remaining consumables
