# 隐藏要素 / Secrets & Unlockables

杀戮尖塔2在初代基础上增加了大量新解锁内容和隐藏要素——包括新角色解锁、新遗物、新事件和终局挑战 / Slay the Spire 2 adds a wealth of new unlockable content and secrets on top of the original — including new character unlocks, new relics, new events, and endgame challenges.

## 新角色解锁 / New Character Unlocks

| 角色 / Character | 解锁条件 / Unlock Condition |
|---|---|
| **铁甲战士 / Ironclad** | 默认解锁 / Default |
| **静默猎人 / Silent** | 使用铁甲战士通关一次 / Win one run with Ironclad |
| **故障机器人 / Defect** | 使用静默猎人通关一次 / Win one run with Silent |
| **观者 / Watcher** | 使用故障机器人通关一次 / Win one run with Defect |
| **死灵法师 / Necromancer** | 使用观者通关一次 / Win one run with Watcher |
| **游侠 / Ranger** | 使用死灵法师通关一次 / Win one run with Necromancer |

**解锁顺序**: 铁甲战士→静默猎人→故障机器人→观者→死灵法师→游侠。每个角色首次通关解锁下一个 / **Unlock order**: Ironclad → Silent → Defect → Watcher → Necromancer → Ranger. Each first clear unlocks the next.
