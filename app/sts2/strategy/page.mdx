# 策略技巧 / Strategy & Tips

杀戮尖塔2在保留初代核心玩法的基础上增加了全新机制——装备系统、新角色和新游戏+模式。本章覆盖新版本特有的策略考量 / Slay the Spire 2 retains the core gameplay of the original while adding brand new mechanics — equipment system, new characters, and New Game+ mode. This chapter covers the unique strategic considerations of the new version.

---

## 新机制策略 / New Mechanics Strategy

### 装备系统策略 / Equipment System Strategy

| 阶段 / Phase | 装备策略 / Equipment Strategy |
|---|---|
| **Act 1** | 优先武器——缺伤害。随便什么武器都行，哪怕是普通铁剑 / Prioritize weapons — need damage. Any weapon works, even Common Iron Sword |
| **Act 2** | 开始考虑护甲——如果牌组防御不够，买/拿护甲 / Start considering armor — if deck lacks block, acquire armor |
| **Act 3** | 找饰品——特殊效果在这个阶段价值最高 / Hunt for accessories — special effects are most valuable at this stage |
| **装备升级优先** | 武器升级>饰品升级>护甲升级。武器升级直接增加输出 / Weapon upgrade > Accessory upgrade > Armor upgrade. Weapon upgrade directly increases DPS |

**装备槽管理**: 不要拿同类型的装备——武器x1+护甲x1+饰品x1是最优配置。拿3个武器等于浪费了护甲和饰品槽 / **Slot management**: Don't stack same type — 1 weapon + 1 armor + 1 accessory is optimal. 3 weapons wastes armor and accessory slots.

### 死灵法师操作 / Necromancer Playstyle

| 要点 / Key Point | 说明 / Description |
|---|---|
| **亡灵数量管理** | 最多5个亡灵——满了就不能再召唤。需要定期献祭来腾出空间 / Max 5 undead — can't summon when full. Sacrifice regularly to free space |
| **献祭=资源** | 献祭不是损失——是资源转化。献祭亡灵=抽牌/能量/AOE / Sacrifice isn't loss — it's resource conversion. Sacrifice = draw/energy/AOE |
| **亡灵质量>数量** | 一个强化过的亡灵(4+力量) > 5个普通骷髅 / One buffed undead (4+ STR) > 5 basic skeletons |
| **防御依赖亡灵** | 死灵法师缺少直接叠甲手段——靠"缚灵"(亡灵死亡叠甲)和亡灵铠甲 / Necromancer lacks direct block — relies on Wraith (block on undead death) and Undead Armor |

### 游侠操作 / Ranger Playstyle

| 要点 / Key Point | 说明 / Description |
|---|---|
| **连击是资源** | 连击在回合结束时清零——所以要在当回合用完。连击>5时打终结技 / Combo resets at end of turn — spend it. Finisher at combo>5 |
| **连击叠加** | 0费"快速射击"是叠连击的核心——配合"身手敏捷"可以无限叠 / 0-cost Quick Shot is the combo core — with Nimble, infinite stacking |
| **标记管理** | 标记只对单体生效——打群架时优先给精英或Boss上标记 / Mark is single-target — prioritize marking elites and bosses in group fights |
| **隐身利用** | 影身药水和暗影步提供隐身——隐身期间全力输出 / Shadow Potion and Shadow Step grant stealth — go all-out DPS while stealthed |

## NG+策略 / NG+ Strategy

| NG+等级 / NG+ Level | 主要变化 / Main Change | 应对策略 / Strategy |
|---|---|---|
| **NG+1** | 敌人HP+25%。解锁"远古遗物"池 / +25% enemy HP. Unlock Ancient Relic pool | 需要更高的输出——优先拿伤害型遗物和装备。远古遗物很强但概率低 / Need higher DPS — prioritize damage relics and equipment. Ancient Relics are strong but rare |
| **NG+2** | 精英出现率+50%。更多精英=更多遗物但更危险 / +50% elite spawn rate. More elites = more relics but more danger | 确保你的Build在Act 1已经有AOE能力——精英群战更多 / Ensure AOE by Act 1 — more group elite fights |
| **NG+3** | 商店价格+50%。经济压力增大 / +50% shop prices. Economy pressure | Act 1多打怪攒钱。删牌优先级降低——太贵了 / Fight more in Act 1 to save gold. Card removal lower priority — too expensive |
| **NG+4** | 药水掉落减半。容错率降低 / Halved potion drops. Lower margin for error | 更加依赖卡牌和遗物——药水不可靠。需要更好的Build / Rely more on cards and relics — potions unreliable. Need stronger builds |
| **NG+5** | 最大生命-20%。越来越容易暴毙 / -20% max HP. Easier to die | 防御牌/护甲的重要度大幅提升——活着才有输出 / Block and armor importance skyrockets — survive first |

## 新敌人策略 / New Enemy Strategies

### Act 1新敌人 / New Act 1 Enemies

| 敌人 / Enemy | 攻击模式 / Pattern | 应对 / Counter |
|---|---|---|
| **亡灵骷髅 / Undead Skeleton** | 每回合攻击2次——每次3-5伤害。死后分裂成2个小骷髅(1血/1攻) / Attacks 2x per turn, 3-5 dmg each. Splits into 2 small skeletons (1HP/1ATK) on death | 集中火力逐个击破——不要让多个骷髅同时触发分裂。用AOE一次清完 / Focus fire one at a time — don't let multiple trigger split simultaneously. AOE to clear all at once |
| **游侠刺客 / Ranger Assassin** | 每2回合进入隐身(不可选中)。隐身解除后攻击造成高额伤害 / Stealth every 2 turns (untargetable). Deals high damage on stealth exit | 用AOE或多段攻击可以破隐——或在它隐身时叠甲防御 / AOE or multi-hit breaks stealth — or just block while stealthed |

### Act 2新精英 / New Act 2 Elites

| 精英 / Elite | 特殊能力 / Ability | 应对 / Counter |
|---|---|---|
| **亡灵法师 / Necromancer Elite** | 每回合召唤亡灵——如果亡灵存活则自己受伤减少50% / Summons undead each turn — 50% damage reduction while undead alive | 优先AOE清亡灵再打本体——否则伤害减半 / AOE undead first — otherwise damage halved |
| **游侠队长 / Ranger Captain** | 连击系统——每回合连击增加，连击越高攻击越强 / Combo system — gains combo each turn, combo = more damage | 用虚弱降低它的攻击。快速击杀——拖越久连击越高越危险 / Apply Weak. Kill fast — longer = higher combo = more dangerous |

### Act 3新Boss / New Act 3 Bosses

| Boss / Boss | 机制 / Mechanic | 策略 / Strategy |
|---|---|---|
| **亡灵龙 / Undead Dragon** | 每回合召唤亡灵+自身每回合获得力量。亡灵被击杀时给自己回血 / Summons undead + gains STR each turn. Heals when undead die | 不打亡灵直接打本体——但需要能承受亡灵的攻击 / Ignore undead, hit boss directly — but must survive undead attacks |
