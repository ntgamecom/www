# 杀戮尖塔2 攻略 / Slay the Spire 2 Guide

杀戮尖塔2（Slay the Spire 2）是Mega Crit开发的经典卡牌Roguelike正统续作。在继承前作核心玩法的基础上，带来了全新的角色阵容、机制体系和视觉风格。本章攻略将带你从零开始征服这座全新的高塔 / Slay the Spire 2 is the official sequel by Mega Crit, building on the beloved deck-building roguelike formula with an all-new cast of characters, mechanics, and a fresh visual identity. This guide covers everything from beginner tips to advanced boss strategies.

**重要提醒**：杀戮尖塔2是完全独立的游戏，拥有全新的角色、卡牌和系统，不要与一代混淆 / Important: Slay the Spire 2 is a completely separate game with new characters, cards, and systems — do not confuse it with the original.

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|---|---|---|
| **1** | **不要用一代思维玩 / Don't Play Like STS1** | 虽然是续作但系统变化很大——很多老经验和直觉在这里不适用。空杯心态 / The systems are very different — many STS1 intuitions don't apply. Come with an open mind |
| **2** | **尝试所有角色 / Try All Characters** | 每个角色的手感差异很大——4个角色至少各玩3-5局再定主玩角色 / Each feels very different — play each character 3-5 runs before picking a main |
| **3** | **关注新资源系统 / Learn the New Resources** | 新资源系统是核心机制——理解如何使用/管理新资源直接影响你的胜负 / New resource system is core — understanding how to use/manage new resources directly affects your win rate |
| **4** | **前期多试错 / Experiment Early** | 游戏刚发售不久——很多强力构筑还未被发现。多尝试不同的卡牌组合 / Game is still new — many OP builds are undiscovered. Experiment with different card combos |
| **5** | **注意遗物联动 / Watch Relic Synergies** | 新遗物系统有更多联动——某些遗物的组合可以产生毁灭性效果 / New relic system has more synergies — certain combos can produce devastating effects |
| **6** | **利用每日挑战学习 / Use Daily Climb to Learn** | 每日挑战可以看其他玩家用什么构筑——学习思路的最好方式 / Daily Climb lets you see what others are using — best way to learn |

## 核心机制详解 / Core Mechanics

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **能量系统 / Energy System** | 继承自STS1的基础系统——每回合有3点能量——卡牌消耗能量——能量管理是构筑核心 / Inherited from STS1 — 3 energy per turn — cards consume energy — energy management is build core |
| **全新资源 / New Resources** | 2代引入的新资源类型——角色专属资源(如"怒气""灵感")+全局资源(如"知识")——极大扩展策略深度 / New resource types introduced — character-specific (Rage, Inspiration) + global (Knowledge) — massively expands strategic depth |
| **角色机制 / Character Mechanics** | 每个角色有独特的资源机制——类似机器人(Defect)的球体但全新设计——每个角色的打法完全不同 / Each character has unique resource mechanics — similar to Defect's orbs but completely redesigned — each character plays completely differently |
| **遗物交互 / Relic Interactions** | 某些遗物可以升级、组合、交易——不再是简单的"拿到就永久生效" / Some relics can be upgraded, combined, traded — no longer simple "get and forget" |
| **事件系统 / Event System** | 更多分支选择——某些事件的结果取决于你的资源/卡牌/遗物——决策深度超过一代 / More branching choices — some event outcomes depend on your resources/cards/relics — deeper decisions than STS1 |
