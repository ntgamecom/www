## 技能系统概览 / Skill System Overview

**中文：** 战神：诸神黄昏的技能系统相比前作大幅扩展。每个武器拥有独立的技能树，外加克雷多斯的基础技能和阿特柔斯的独立技能树。技能点（XP）通过战斗、完成任务和发现收集品获得。技能可以随时免费重置，因此可以大胆尝试不同的流派组合。

**技能树分类（共6大技能树）：**
1. 利维坦之斧技能树 — 冰霜系战斗技能
2. 混沌之刃技能树 — 火焰系战斗技能
3. 德劳普尼尔长枪技能树 — 风系战斗技能
4. 克雷多斯基础技能 — 通用战斗能力
5. 盾牌技能树 — 格挡与反击
6. 阿特柔斯技能树 — 弓箭与符文召唤

**English:** The skill system is vastly expanded from the 2018 game. Each weapon has its own skill tree, plus Kratos' core skills and Atreus' independent tree. Skill points (XP) are earned through combat, quests, and collectibles. Respeccing is free and unlimited — experiment freely.

**Six skill trees:**
1. Leviathan Axe — Frost combat skills
2. Blades of Chaos — Fire combat skills
3. Draupnir Spear — Wind combat skills
4. Kratos Core Skills — Universal combat abilities
5. Shield Skills — Block and counter
6. Atreus Skills — Bow and runic summons

{/* ===== 利维坦之斧技能树 / Leviathan Axe Skills ===== */}

## 利维坦之斧技能树 / Leviathan Axe Skills

**中文：** 利维坦之斧技能树专注于冰霜控制和精准打击。技能分为三大类：近战连击、投掷技能、冰霜强化。

**必须优先解锁的技能：**

1. **冰霜爆破（Frost Rush）** — R2冲刺重击。这是斧头最重要的起手技能，可以快速接近敌人并造成冻结效果。消耗2技能点。

2. **利维坦之怒（Leviathan's Fury）** — 长按R2蓄力的重击，造成大量冰霜伤害。对大型敌人尤为有效。消耗3技能点。

3. **冰川之盾（Glacial Shield）** — 盾反后接R1释放冰霜冲击。这是斧头与盾牌联动的最佳技能。消耗2技能点。

4. **返回风暴（Return Storm）** — 按三角键召回斧头时释放冰霜AoE。在群体战斗中非常实用。消耗3技能点。

5. **冰霜之力（Frost Power）** — 被动技能，增加斧头冰霜积累速度。优先级很高，建议尽快点满。消耗2技能点。

**English:** The Leviathan Axe tree focuses on frost control and precision strikes. Three branches: melee combos, throw skills, frost enhancement.

**Must-unlock skills:**

1. **Frost Rush** — R2 sprinting heavy. Best gap closer, applies freeze. Cost: 2 SP.
2. **Leviathan's Fury** — Hold R2 charged heavy. Massive frost damage vs large enemies. Cost: 3 SP.
3. **Glacial Shield** — R1 after parry, frost blast. Best axe-shield synergy. Cost: 2 SP.
4. **Return Storm** — Frost AoE on axe recall (Triangle). Essential in group fights. Cost: 3 SP.
5. **Frost Power** — Passive, faster frost buildup. Max ASAP. Cost: 2 SP.

{/* ===== 混沌之刃技能树 / Blades of Chaos Skills ===== */}

## 混沌之刃技能树 / Blades of Chaos Skills

**中文：** 混沌之刃技能树强调范围控制和持续灼烧。适合清理杂兵和对群战斗。

**必须优先解锁的技能：**

1. **焰刃风暴（Flame Whiplash）** — 按住R2蓄力后释放旋转攻击，是大范围清场神技。面对5+敌人时一招扭转战局。消耗2技能点。

2. **阿瑞斯之力（Strength of Ares）** — 被动技能，提升灼烧伤害30%。混沌之刃的核心被动，优先级最高。消耗3技能点。

3. **混沌之跃（Chaos Leap）** — 空中按R2发动俯冲攻击，落地造成范围火焰伤害。适合从高处突袭。消耗2技能点。

4. **塔塔罗斯之怒（Tartarus Rage）** — R2重型符文打击的终结技，造成大范围击退和燃烧。消耗3技能点。

5. **火焰之鞭（Whip of Fire）** — 连击中按R1可以延伸攻击距离，面对远程敌人时非常有用。消耗1技能点。

**English:** The Blades tree excels at area control and burn damage. Best for crowd clearing.

**Must-unlock skills:**

1. **Flame Whiplash** — Hold R2 spin attack. Best AoE in the game. Cost: 2 SP.
2. **Strength of Ares** — Passive +30% burn damage. Core passive. Cost: 3 SP.
3. **Chaos Leap** — Aerial R2 slam, AoE fire. Great for high-ground ambushes. Cost: 2 SP.
4. **Tartarus Rage** — Heavy Runic finisher, massive knockback + burn. Cost: 3 SP.
5. **Whip of Fire** — Extended range on R1 combo. Useful vs ranged enemies. Cost: 1 SP.

{/* ===== 德劳普尼尔长枪技能树 / Draupnir Spear Skills ===== */}

## 德劳普尼尔长枪技能树 / Draupnir Spear Skills

**中文：** 长枪是本作的新武器，技能树强调高速连击和元素引爆。适合单挑精英和快速破防。

**必须优先解锁的技能：**

1. **长枪突刺（Spear Rush）** — R1连续突刺，每段命中都会叠加风元素眩晕值。长枪的核心输出技能。消耗2技能点。

2. **风爆（Wind Burst）** — 按三角键引爆所有投掷的长枪，造成范围风元素伤害。这是长枪最独特的机制，也是最高伤害来源。消耗3技能点。

3. **动势无限（Momentum Unbound）** — 被动技能，每次命中提升攻击速度，可叠加3层。持久战的神技。消耗2技能点。

4. **精准之掷（Precision Throw）** — R2投掷长枪，可以插在敌人身上后引爆。远程消耗的利器。消耗2技能点。

5. **大地粉碎（Earth Shatter）** — 跳跃中R2砸地，将周围的敌人击飞。适合在被包围时突围。消耗2技能点。

**English:** The spear is new to Ragnarök. The tree favors fast combos and elemental detonation. Best for dueling elites.

**Must-unlock skills:**

1. **Spear Rush** — R1 rapid thrusts, wind stun stacking. Core DPS skill. Cost: 2 SP.
2. **Wind Burst** — Triangle detonates planted spears. Highest burst damage. Cost: 3 SP.
3. **Momentum Unbound** — Passive attack speed per hit, 3 stacks. Excellent in long fights. Cost: 2 SP.
4. **Precision Throw** — R2 spear throw, detonate later. Ranged poke tool. Cost: 2 SP.
5. **Earth Shatter** — Aerial R2 slam, AoE knockback. Escape from surround. Cost: 2 SP.

{/* ===== 克雷多斯基础技能 / Kratos Core Skills ===== */}

## 克雷多斯基础技能 / Kratos Core Skills

**中文：** 基础技能树包含不限于特定武器的通用能力。这些技能对所有武器都有增益效果。

**推荐优先解锁：**

1. **斯巴达突刺（Spartan Lunge）** — L2+R1投掷武器后的追击技能，可以在投掷后快速拉近距离。消耗1技能点。
2. **终结技强化（Finisher Enhancement）** — 增加眩晕条满后的处决伤害。消耗2技能点。
3. **翻滚强化（Roll Enhancement）** — 增加翻滚距离和无敌帧。生存能力的核心。消耗2技能点。
4. **复仇之力（Power of Vengeance）** — 低血量时增加所有伤害20%的被动。适合极限打法。消耗3技能点。
5. **路标（Wayfarer）** — 增加探索时获得的XP。开荒时非常有用的被动。消耗1技能点。

**English:** Core skills benefit all weapons equally:

1. **Spartan Lunge** — L2+R1 gap closer after throw. Cost: 1 SP.
2. **Finisher Enhancement** — Bonus execute damage. Cost: 2 SP.
3. **Roll Enhancement** — Longer roll, more i-frames. Essential survival. Cost: 2 SP.
4. **Power of Vengeance** — +20% all damage at low HP. Cost: 3 SP.
5. **Wayfarer** — Bonus XP from exploration. Great early on. Cost: 1 SP.

{/* ===== 盾牌技能树 / Shield Skills ===== */}

## 盾牌技能树 / Shield Skills

**中文：** 盾牌技能独立于主武器技能树，专注于防御和反击：

- **守护者之盾（Guardian Shield）** — 格挡后短时间内提升防御力。基础的防御技能。
- **坚定反击（Firm Counter）** — 盾反成功后恢复少量生命值。推荐所有玩家优先解锁。
- **盾牌猛击（Shield Bash）** — R2攻击后的盾牌追击，可以打断敌人动作。
- **元素护盾（Elemental Shield）** — 格挡时积累元素抗性，减少对应元素伤害。

**English:** Shield skills are independent and focus on defense + counter:

- **Guardian Shield** — Post-block defense buff.
- **Firm Counter** — HP recovery on successful parry. Unlock first.
- **Shield Bash** — R2 follow-up, interrupts enemy actions.
- **Elemental Shield** — Elemental resistance on block.

## 阿特柔斯技能树 / Atreus Skills

**中文：** 阿特柔斯拥有独立的技能树，包括弓箭射击和符文召唤。阿特柔斯在战斗中是重要的辅助角色：

**箭矢类型：**
- **电击箭（Shock Arrows）**：造成眩晕，打断敌人攻击。最常用的箭矢类型。
- **音波箭（Sonic Arrows）**：标记敌人，标记结束时造成额外伤害。
- **符文箭（Runic Arrows）**：造成属性伤害，配合武器元素使用。

**符文召唤（Runic Summons）：**
阿特柔斯可以召唤不同的生物协助战斗：
- **狼群（狼）**：最早获得的召唤，可以吸引敌人火力。
- **鹰**：范围侦察和干扰。
- **熊**：高伤害召唤，适合BOSS战。

**推荐早期解锁：**
- 电击箭强化 — 增加眩晕积累速度。
- 符文召唤冷却缩减 — 让阿特柔斯更频繁地使用召唤技能。
- 自动射击 — 阿特柔斯会自动瞄准附近的敌人射击。

**English:** Atreus has his own skill tree with bow attacks and runic summons. He's a crucial combat support:

**Arrow types:**
- **Shock Arrows** — Stun buildup, interrupt enemies. Most versatile.
- **Sonic Arrows** — Mark enemies, bonus pop damage.
- **Runic Arrows** — Elemental damage, weapon synergy.

**Runic Summons:**
- **Wolves** — Early summon, aggro draw.
- **Hawk** — AoE reconnaissance and disrupt.
- **Bear** — High damage, best for boss fights.

**Early unlocks:** Shock Arrow upgrade, summon cooldown reduction, auto-aim.

## 技能加点优先级 / Skill Priority Order

**中文：** 以下是最推荐的技能加点顺序（按游戏进程）：

**前期（1-10小时）：**
1. 冰霜爆破（斧） — 核心战斗技能
2. 焰刃风暴（混沌刃） — 清杂兵
3. 坚定反击（盾） — 盾反回血
4. 翻滚强化（基础） — 生存
5. 路标（基础） — XP增益

**中期（10-25小时）：**
1. 风爆（长枪） — 长枪核心
2. 斯巴达突刺（基础） — 连招衔接
3. 阿瑞斯之力（混沌刃） — 灼烧增伤
4. 阿特柔斯符文召唤 — 团队辅助
5. 复仇之力（基础） — 残血爆发

**后期（25+小时）：**
1. 所有武器最终技能 — 解锁完整技能树
2. 阿特柔斯全技能 — 最大化辅助能力
3. 终结技强化 — 处决增伤

**English:** Recommended skill unlock order:

**Early (1-10 hrs):**
1. Frost Rush (Axe) — Core combat
2. Flame Whiplash (Blades) — AoE clear
3. Firm Counter (Shield) — Parry heal
4. Roll Enhancement (Core) — Survival
5. Wayfarer (Core) — XP boost

**Mid (10-25 hrs):**
1. Wind Burst (Spear) — Spear core
2. Spartan Lunge (Core) — Combo link
3. Strength of Ares (Blades) — Burn damage
4. Atreus Runic Summon — Team support
5. Power of Vengeance (Core) — Low HP burst

**Late (25+ hrs):**
1. All weapon finishers — Max skill trees
2. Full Atreus skills — Max support
3. Finisher Enhancement — Execute damage

## 终局Build技能搭配 / Endgame Builds

**中文：** 以下是通关后的最强技能搭配方案：

**元素循环流（Element Rotation Build）：**
- 三点全满以下被动：冰霜之力、阿瑞斯之力、动势无限
- 核心思路：冰冻结 → 灼烧 → 引爆 → 循环
- 技能搭配：冰霜爆裂（起手）+ 焰刃风暴（清场）+ 风爆（终结）

**盾反暴击流（Parry Crit Build）：**
- 核心技能：坚定反击、冰川之盾、塔塔罗斯之怒
- 核心思路：等待敌人攻击 → 盾反 → 重型符文反击
- 需要配合无畏盾使用，最大化盾反窗口

**English:** Best endgame skill loadouts:

**Element Rotation Build:**
- Max passives: Frost Power, Strength of Ares, Momentum Unbound
- Loop: Freeze → Burn → Detonate → Repeat
- Skills: Frost Burst (opener) + Flame Whiplash (clear) + Wind Burst (finisher)

**Parry Crit Build:**
- Core: Firm Counter, Glacial Shield, Tartarus Rage
- Flow: Wait → Parry → Heavy Runic punish
- Requires Dauntless Shield for max parry window.
