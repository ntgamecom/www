## 技能与魔法 / Skills & Magic

### 魔法体系总览 / Magic System Overview

《黑夜君临》的魔法系统分为**法术**（Sorceries）和**祷告**（Incantations）两大类别。法术主要由智力属性驱动，祷告则由信仰属性驱动。

Nightreign's magic system is divided into two main categories: **Sorceries** (driven by Intelligence) and **Incantations** (driven by Faith).

### 法术 / Sorceries

法术是智力系魔法，主要造成魔法属性伤害，部分法术有特殊效果。

Sorceries are Intelligence-based magic, primarily dealing magic damage, with some having special effects.

### 记忆格位 / Memory Slots

魔法需要占用**记忆格位**（Memory Slots）。通过在篝火处休息可以配置装备的法术和祷告。初始拥有2个格位，可通过收集记忆石增加。

Spells require **Memory Slots** to equip. Rest at Sites of Grace to configure equipped sorceries and incantations. Start with 2 slots, increase by collecting Memory Stones.
