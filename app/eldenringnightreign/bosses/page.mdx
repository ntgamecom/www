## BOSS攻略 / Boss Guide

### BOSS分类 / Boss Categories

《黑夜君临》的BOSS分为**白天BOSS**（Day Bosses）和**夜晚BOSS**（Night Bosses）两大类。白天BOSS是可选挑战，夜晚BOSS是强制战斗。

Nightreign bosses are divided into **Day Bosses** (optional) and **Night Bosses** (mandatory). Day bosses are optional challenges while night bosses are forced encounters.

### 白天BOSS / Day Bosses

白天BOSS在地图上以金色标记显示。击败它们可以获得大量卢恩和稀有装备。

Day bosses appear as gold markers on the map. Defeating them grants substantial runes and rare equipment.

#### 黄金骑士戈德里克 / Golden Knight Godrick

**位置** / **Location**: 风暴前线 / Storm Front
**难度** / **Difficulty**: 中等 / Medium

**攻击模式** / **Attack Patterns**:

1. 横扫斩击 / Horizontal Slash — 大范围横扫，向后闪避 / Wide horizontal sweep, dodge backward
2. 跳跃下劈 / Jump Slam — 跳起后下砸地面 / Leaps and slams the ground
3. 黄金吐息 / Golden Breath — 喷出黄金火焰 / Breathes golden fire
4. 二阶段旋风 / Phase 2 Whirlwind — 旋转连续攻击 / Spinning multi-hit combo

**推荐打法** / **Strategy**:

- 保持在BOSS身后输出 / Stay behind the boss for damage
- 使用冰霜或出血武器 / Use frost or bleed weapons
- 注意吐息的前摇 / Watch for breath attack wind-up
- 二阶段保持距离观察 / Keep distance and observe in phase 2

#### 腐败树灵 / Putrid Tree Spirit

**位置** / **Location**: 毒腐洞穴 / Rot Caves
**难度** / **Difficulty**: 困难 / Hard

**攻击模式** / **Attack Patterns**:

1. 撕咬 / Bite — 快速前咬 / Quick forward bite
2. 尾鞭 / Tail Whip — 转身甩尾 / Spin tail whip
3. 腐败吐息 / Rot Breath — 扇形腐败喷吐 / Cone-shaped rot spray
4. 钻地突袭 / Burrow Strike — 钻入地下后从脚下钻出 / Burrows underground then erupts below

**推荐打法** / **Strategy**:

- 装备火焰武器或火焰祷告 / Equip fire weapons or fire incantations
- 保持移动避免腐败累积 / Keep moving to avoid rot buildup
- 使用火焰壶清除自身腐败 / Use Fire Pot to cleanse self-rot
- 钻地时连续奔跑躲避 / Run continuously when it burrows

### 夜晚BOSS / Night Bosses

每个夜晚结束时必须面对的强力BOSS。难度逐夜提升。

Powerful bosses faced at the end of each night. Difficulty increases each night.

#### 第一夜BOSS / Night 1 Boss: 暗影守卫 / Shadow Warden

**血量** / **HP**: 中等 / Medium
**弱點** / **Weakness**: 神圣伤害 / Holy damage

**阶段** / **Phases**: 2个阶段 / 2 Phases

**阶段1技能** / **Phase 1 Skills**:

| 技能 / Skill | 应对 / Counter |
|---|---|
| 暗影突刺 / Shadow Thrust | 侧向闪避 / Side dodge |
| 黑暗波 / Dark Wave | 跳跃躲避 / Jump over |
| 召唤分身 / Clone Summon | 快速击杀分身 / Kill clones quickly |

**阶段2技能** / **Phase 2 Skills**:

| 技能 / Skill | 应对 / Counter |
|---|---|
| 巨型暗影剑 / Giant Shadow Blade | 连续翻滚 / Chain roll |
| 暗影爆发 / Shadow Burst | 远离中心 / Move away from center |
| 黑暗降世 / Darkness Fall | 寻找光柱区域 / Find light pillar zones |

**推荐Build** / **Recommended Build**: 神圣武器 / Holy weapon, 远程攻击 / Ranged attacks

#### 第二夜BOSS / Night 2 Boss: 火焰巨人残骸 / Flame Giant Remnant

**血量** / **HP**: 高 / High
**弱點** / **Weakness**: 冰霜伤害 / Frost damage

**阶段** / **Phases**: 2个阶段 / 2 Phases

**阶段1技能** / **Phase 1 Skills**:

| 技能 / Skill | 应对 / Counter |
|---|---|
| 火焰拳砸 / Flame Fist Slam | 闪避到侧面 / Dodge to the side |
| 岩浆喷发 / Magma Eruption | 远离地面裂缝 / Stay away from ground cracks |
| 火焰轮盘 / Fire Wheel | 向BOSS方向翻滚 / Roll toward the boss |

**阶段2技能** / **Phase 2 Skills**:

| 技能 / Skill | 应对 / Counter |
|---|---|
| 陨石火雨 / Meteor Fire Rain | 找岩石遮挡 / Find rock cover |
| 熔岩波浪 / Lava Wave | 跳跃或站高地 / Jump or find high ground |
| 自爆冲刺 / Self-Destruct Charge | 拉开距离 / Create distance |

**推荐打法** / **Strategy**:

- 使用冰霜武器或冰系魔法 / Use frost weapons or ice magic
- 分配玩家吸引BOSS注意力 / Assign one player to tank aggro
- 阶段2时注意地面岩浆 / Watch ground lava in phase 2
- 牧师准备群体恢复 / Cleric prepare group heal

#### 第三夜最终BOSS / Night 3 Final Boss: 夜之王 / Night Lord

**血量** / **HP**: 极高 / Very High
**弱點** / **Weakness**: 圣律伤害 / Sacred damage

**阶段** / **Phases**: 3个阶段 / 3 Phases

**阶段1技能** / **Phase 1 Skills**:

1. 漆黑之刃 / Void Blade — 横向暗影斩击 / Horizontal shadow slash
2. 虚空裂隙 / Void Rift — 地面出现裂隙造成伤害 / Rifts appear on ground dealing damage
3. 夜之枷锁 / Night's Shackle — 束缚一名玩家 / Binds one player in place

**阶段2技能** / **Phase 2 Skills**:

1. 暗影分身 / Shadow Doppelganger — 生成自身分身 / Creates a clone of itself
2. 虚空风暴 / Void Storm — 全场范围攻击 / Full arena AoE attack
3. 重力场 / Gravity Field — 吸引所有玩家到中心 / Pulls all players to center

**阶段3技能** / **Phase 3 Skills**:

1. 永恒黑暗 / Eternal Darkness — 全屏黑暗降低视野 / Full screen darkness reduces vision
2. 夜之终焉 / End of Night — 全屏大伤害技能 / Full screen massive damage skill
3. 虚空化身 / Void Avatar — 巨大化并强化所有攻击 / Grows larger and enhances all attacks

**推荐打法** / **Strategy**:

- 全队至少40生命力 / Entire party minimum 40 Vigor
- 准备圣律系武器和祷告 / Prepare sacred weapons and incantations
- 阶段2优先击杀分身 / Phase 2 priority kill the clone
- 阶段3共享回复资源 / Phase 3 share healing resources
- 保留夜之力技能到阶段3 / Save Night Power skills for phase 3

### 世界BOSS / World Bosses

在地图上随机出现的超强力BOSS，它们会在固定位置刷新并在被击杀或隔夜后消失。

Ultra-powerful bosses that appear randomly on the map. They spawn at fixed locations and disappear when killed or when the day changes.

| BOSS名称 / Name | 位置 / Location | 推荐等级 / Recommended Level | 奖励 / Reward |
|---|---|---|---|
| 远古巨龙 / Ancient Dragon | 龙息山丘 / Dragon's Hill | 建议第二天 / Day 2+ | 龙心脏 + 传说武器 / Dragon Heart + Legendary Weapon |
| 死之鸟 / Deathbird | 各大桥梁 / Major Bridges | 当天即可 / Same day | 死亡钩爪 / Death Talisman |
| 行走墓地 / Walking Mausoleum | 地图边缘 / Map Edge | 建议第三天 / Day 3+ | 追忆 / Remembrance |
| 戈玛 / Goma the Many | 废弃要塞 / Ruined Fort | 第二天 / Day 2 | 腐败武器 / Rot Weapon |

### 地牢BOSS / Dungeon Bosses

各个地牢深处的BOSS，击杀后可以获得特殊技能或道具。

Bosses found in the depths of various dungeons. Defeating them grants special skills or items.

- **墓穴守卫** / **Catacomb Guardian**: 精通陷阱和伏击 / Expert in traps and ambushes
- **矿洞巨人** / **Mine Giant**: 弱雷属性 / Weak to lightning
- **塔楼魔女** / **Tower Witch**: 使用重力魔法 / Uses gravity magic
- **下水道恶鬼** / **Sewer Ogre**: 毒属性攻击 / Poison attacks

### BOSS通用技巧 / General Boss Tips

1. **观察死亡** / **Learn from death** — 每次失败都是学习机会 / Each death is a learning opportunity
2. **调整装备** / **Adjust equipment** — 根据BOSS属性弱点换装备 / Swap gear based on boss elemental weakness
3. **分配角色** / **Assign roles** — 坦克、输出、治疗分工明确 / Clear tank-dps-healer role division
4. **保留资源** / **Save resources** — 不要在小怪身上浪费强力道具 / Don't waste powerful items on minor enemies
5. **环境利用** / **Use environment** — 利用地形和场景物件 / Use terrain and scene objects
6. **沟通** / **Communicate** — 实时语音沟通BOSS动作 / Real-time voice communication for boss actions
