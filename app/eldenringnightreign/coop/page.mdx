## 联机合作 / Co-op Multiplayer

### 合作概述 / Co-op Overview

《黑夜君临》的核心是合作体验。游戏支持最多三名玩家在线联机，所有内容都围绕团队配合设计。联机方式灵活，可以匹配路人，也可以邀请好友组队。

The core of Nightreign is the co-op experience. The game supports up to three players online, with all content designed around team coordination. Flexible matchmaking allows queueing with randoms or inviting friends.

### 匹配系统 / Matchmaking System

#### 公开匹配 / Public Matchmaking

系统会自动将你匹配到合适的队伍中。匹配时可以考虑以下因素：

The system automatically matches you to a suitable party. Matchmaking considers:

- **地区** / **Region** — 优先同地区玩家 / Priority for same-region players
- **通关进度** / **Progress** — 尽量匹配进度相近的玩家 / Try to match players with similar progress
- **网络质量** / **Connection Quality** — 选择延迟低的玩家 / Selects low-latency connections
- **职业搭配** / **Class Composition** — 尽量避免全队同职业 / Avoids party of all same class

#### 好友联机 / Friend Invite

通过游戏内好友系统可以邀请好友组队。支持跨平台联机（需要平台账号绑定）。

Invite friends via the in-game friend system. Supports cross-platform play (requires platform account linking).

#### 单人模式 / Solo Mode

也可以选择单人游玩。单人模式下敌人的血量和伤害会相应降低，但某些BOSS战会非常困难。

You can also play solo. Enemy HP and damage are reduced in solo mode, but certain boss fights become very difficult.

### 三人队伍配合 / 3-Player Team Synergy

#### 理想队伍构成 / Ideal Team Composition

| 角色 / Role | 职业 / Class | 职责 / Responsibility |
|---|---|---|
| 坦克 / Tank | 骑士 / Knight | 拉仇恨、吸收伤害 / Draw aggro, absorb damage |
| 输出 / DPS | 战士/武士/游侠 | 主要伤害输出 / Primary damage dealer |
| 治疗 / Healer | 牧师 / Cleric | 回复、增益、辅助 / Healing, buffing, support |

#### 备选方案 / Alternative Compositions

| 组合 / Composition | 优势 / Advantage | 劣势 / Disadvantage |
|---|---|---|
| 双输出+治疗 / 2 DPS + Healer | 伤害高，续航稳 / High dmg, stable sustain | 坦克压力大 / Tank stressed |
| 坦克+双治疗 / Tank + 2 Healers | 极其安全但慢 / Very safe but slow | 伤害不足 / Lacks damage |
| 三输出 / 3 DPS | 爆发极高 / Extremely high burst | 没有容错率 / No margin for error |
| 混合法系 / Hybrid Magic | 属性伤害多样 / Diverse elemental dmg | 前期较弱 / Weak early game |

### 沟通系统 / Communication System

#### 语音聊天 / Voice Chat

游戏内置语音聊天功能，建议所有玩家开启。沟通以下信息：

Built-in voice chat is recommended for all players. Communicate:

- **敌人位置** / **Enemy location** — 报告发现的敌人 / Report spotted enemies
- **资源分享** / **Resource sharing** — 谁需要什么材料 / Who needs what materials
- **BOSS技能** / **Boss abilities** — 提醒队友BOSS即将释放的技能 / Alert teammates of incoming boss skills
- **撤退信号** / **Retreat signal** — 打不过时及时撤退 / Timely retreat when overwhelmed
- **死亡求救** / **Death求救** — 倒地时告知位置 / Report location when downed

#### 快捷标记 / Quick Pings

使用标记系统可以快速提示队友：

Use the ping system to quickly alert teammates:

- 按住标记键打开轮盘 / Hold ping key to open the wheel
- **攻击标记** / **Attack Ping** — 标记要攻击的目标 / Mark a target to attack
- **拾取标记** / **Loot Ping** — 标记有价值的物品 / Mark valuable items
- **危险标记** / **Danger Ping** — 标记危险区域 / Mark dangerous areas
- **前进标记** / **Move Ping** — 指示前进方向 / Indicate direction to move
- **请求帮助** / **Help Request** — 请求队友支援 / Request teammate assistance

### 复活系统 / Revive System

#### 倒地机制 / Downed Mechanic

当玩家HP归零时会进入**倒地**（Downed）状态。在此状态下：

When a player's HP reaches zero they enter a **Downed** state. In this state:

- 可以缓慢爬行 / Can slowly crawl
- 可以使用部分道具 / Can use some items
- 等待队友救援 / Waiting for teammate revival
- 倒计时30秒，超时则死亡 / 30-second countdown, death if expired

#### 复活流程 / Revival Process

队友走到倒地玩家身边按下互动键即可开始救援。救援需要约3秒钟，期间不能移动或防御。

Teammates walk to the downed player and press the interact button to begin revival. Revival takes approximately 3 seconds and cannot be interrupted by movement or blocking.

#### 完全死亡 / Full Death

如果倒计时结束或队友全灭，游戏结束。完全死亡的玩家只能观战直到队伍重新开始。

If the timer expires or the entire party is wiped, the run ends. Fully dead players can only spectate until the party restarts.

**复活策略** / **Revive Strategy**:

1. 优先复活治疗职业 / Priority revive the healer class
2. 在安全区域进行复活 / Only revive in safe areas
3. 坦克吸引BOSS注意力让治疗复活 / Tank distracts boss while healer revives
4. 使用烟雾弹或隐身道具掩护复活 / Use smoke or invisibility items to cover revival

### 资源分配 / Resource Distribution

团队中资源管理非常重要：

Resource management within the team is crucial:

| 资源 / Resource | 分配建议 / Distribution Suggestion |
|---|---|
| 卢恩 / Runes | 优先给输出升级武器 / Priority to DPS for weapon upgrades |
| 锻造石 / Smithing Stones | 平均分配给全队 / Even distribution to entire party |
| 治疗道具 / Healing Items | 大部分给坦克和治疗 / Mostly to tank and healer |
| 稀有材料 / Rare Materials | 根据当前任务需求分配 / Assign based on current mission needs |
| 技能书 / Skill Books | 优先给核心输出技能 / Priority to core DPS skills |

### 联机技巧 / Co-op Tips

1. **不要分散太远** / **Don't spread too far** — 最多保持一个屏幕的距离 / Keep at most one screen apart
2. **共享战利品** / **Share loot** — 不需要的装备可以扔给队友 / Drop unwanted gear for teammates
3. **技能连携** / **Skill combos** — 规划技能释放顺序 / Plan skill cast order
4. **坦克拉怪** / **Tank pulls** — 由坦克先手开怪 / Let the tank initiate combat
5. **治疗站位** / **Healer positioning** — 治疗站在队伍最后方 / Healer stays in the backline
6. **互相掩护** / **Cover each other** — 喝药时队友负责掩护 / Teammates cover during healing
7. **同步撤退** / **Synchronized retreat** — 撤退时一起行动 / Retreat together as a group
8. **检查队友装备** / **Check teammate gear** — 开局时确认大家的装备配置 / Confirm everyone's gear setup at start

Co-op play provides additional rewards:

- **羁绊系统** / **Bond System** — 与好友共同游戏提升羁绊等级，解锁特殊称号 / Level up bond with friends to unlock special titles
