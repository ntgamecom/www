## 职业与角色 / Classes

### 职业总览 / Class Overview

《黑夜君临》提供了多种职业选择，每种职业都有独特的**初始属性**、**装备**和**专属技能**。与《艾尔登法环》不同，本作更强调职业之间的团队配合。

Nightreign offers a variety of classes, each with unique **starting stats**, **equipment**, and **exclusive abilities**. Unlike Elden Ring, this title emphasizes team synergy between classes.

### 战士 / Warrior

**类型** / **Type**: 近战输出 / Melee DPS
**难度** / **Difficulty**: 简单 / Easy

战士是全能型近战职业，精通各种近战武器。拥有均衡的攻防能力和较高的容错率。

The Warrior is an all-round melee class proficient in various melee weapons. It has balanced offense and defense with high forgiveness.

| 属性 / Stat | 初始值 / Starting Value |
|---|---|
| 生命力 / Vigor | 15 |
| 耐力 / Endurance | 13 |
| 力量 / Strength | 16 |
| 灵巧 / Dexterity | 14 |
| 智力 / Intelligence | 9 |
| 信仰 / Faith | 9 |
| 感应 / Arcane | 8 |

**初始装备** / **Starting Gear**: 长剑 (Longsword)、黄铜盾 (Brass Shield)、锁子甲 (Chainmail Set)

**专属技能** / **Unique Ability**: 战意高昂 (Battle Ardor) — 连续攻击时攻击力逐渐提升 / Attack power gradually increases during consecutive attacks

**推荐Build** / **Recommended Build**: 力量40 / 灵巧30 / 生命力35 / 耐力30

### 骑士 / Knight

**类型** / **Type**: 防御型坦克 / Defensive Tank
**难度** / **Difficulty**: 简单 / Easy

骑士拥有最高的初始生命值和防御力，擅长吸引敌人仇恨和保护队友。使用大盾时格挡能力极强。

The Knight has the highest starting HP and defense, excels at drawing aggro and protecting teammates. Extremely strong blocking with greatshields.

| 属性 / Stat | 初始值 / Starting Value |
|---|---|
| 生命力 / Vigor | 18 |
| 耐力 / Endurance | 15 |
| 力量 / Strength | 14 |
| 灵巧 / Dexterity | 12 |
| 智力 / Intelligence | 9 |
| 信仰 / Faith | 13 |
| 感应 / Arcane | 7 |

**初始装备** / **Starting Gear**: 阔剑 (Broadsword)、大盾 (Greatshield)、骑士甲 (Knight Armor Set)

**专属技能** / **Unique Ability**: 钢铁守护 (Iron Guardian) — 盾牌格挡时减少全队受到的伤害 / Reduces party damage taken while blocking with shield

**推荐Build** / **Recommended Build**: 生命力45 / 耐力35 / 力量25 / 信仰20

### 游侠 / Rogue

**类型** / **Type**: 敏捷刺客 / Agile Assassin
**难度** / **Difficulty**: 中等 / Medium

游侠擅长快速近战和远程攻击，背刺伤害极高。拥有全职业最高的灵巧属性和移动速度。

The Rogue excels at fast melee and ranged attacks with extremely high backstab damage. Has the highest Dexterity and movement speed of all classes.

| 感应 / Arcane | 14 |

**初始装备** / **Starting Gear**: 匕首 (Dagger)、短弓 (Shortbow)、皮甲 (Leather Armor Set)

**专属技能** / **Unique Ability**: 暗影步 (Shadowstep) — 短距离瞬移到敌人背后 / Short-range teleport behind an enemy

**推荐Build** / **Recommended Build**: 灵巧45 / 感应30 / 生命力25 / 耐力25

### 法师 / Mage

**类型** / **Type**: 远程法系 / Ranged Magic DPS
**难度** / **Difficulty**: 中等 / Medium

法师是纯粹的魔法使用者，拥有强大的远程攻击能力和范围伤害。但生命值较低，需要队友保护。

The Mage is a pure magic user with powerful ranged attacks and area damage. Low HP requires team protection.

**推荐Build** / **Recommended Build**: 智力50 / 灵巧20 / 生命力20 / 耐力15

### 牧师 / Cleric

**类型** / **Type**: 治疗辅助 / Healer Support
**难度** / **Difficulty**: 中等 / Medium

牧师是唯一的治疗职业，能够回复队友生命并提供增益效果。在三人队伍中不可或缺。

The Cleric is the only healing class, capable of restoring teammate HP and providing buffs. Indispensable in a three-player party.

**专属技能** / **Unique Ability**: 神圣恩典 (Divine Grace) — 范围内持续恢复队友HP / Continuous HP regeneration in an area around the caster

**推荐Build** / **Recommended Build**: 信仰45 / 生命力30 / 耐力20 / 力量20

### 囚犯 / Prisoner

**难度** / **Difficulty**: 困难 / Hard

囚犯是近战与魔法的混合职业，使用刺剑和魔法进行战斗。操作难度较高但上限也高。

The Prisoner is a hybrid melee-magic class, fighting with rapier and magic. High skill ceiling but rewarding.

### 武士 / Samurai

**类型** / **Type**: 强力近战 / Powerful Melee
**难度** / **Difficulty**: 简单 / Easy

武士来自苇名地区，擅长太刀和长弓。拥有独特的居合战技，爆发伤害极高。

The Samurai hails from the Land of Reeds, proficient with katanas and longbows. Has a unique Iaijutsu skill with extremely high burst damage.

### 团队配合建议 / Team Synergy

| 队伍组合 / Party Comp | 说明 / Description |
|---|---|
| 战士 + 骑士 + 牧师 | 最稳健的搭配，适合新手 / Most stable, best for beginners |
| 武士 + 游侠 + 法师 | 高输出组合，需要默契 / High damage, needs coordination |
| 囚犯 + 法师 + 牧师 | 混合伤害与治疗 / Hybrid damage with healing |
| 骑士 + 游侠 + 武士 | 全物理菜刀队 / Full physical damage team |

### 职业解锁 / Unlockable Classes

完成特定挑战后可以解锁隐藏职业：

Hidden classes can be unlocked after completing specific challenges:

- **狂战士** / **Berserker** — 通关Night 3后解锁 / Unlocked after clearing Night 3
- **死灵法师** / **Necromancer** — 在特定地牢中找到古老卷轴 / Found an ancient scroll in a specific dungeon
- **龙骑士** / **Dragoon** — 击杀隐藏世界BOSS龙神 / Defeat hidden world boss Dragon God
