## 终局与挑战 / Endgame & Challenges

### 终局概览 / Endgame Overview

完成首次第三夜BOSS战只是开始。《黑夜君临》提供了丰富的终局内容，鼓励玩家反复挑战。

Completing your first Night 3 boss fight is just the beginning. Nightreign offers extensive endgame content encouraging repeated playthroughs.

### 遗物系统 / Relic System

#### 遗物介绍 / Relic Introduction

每次通关后可以获得**遗物**（Relics），这是本作的核心roguelite元素。遗物提供强大的被动效果，可以在后续游戏中使用。

Each clear awards **Relics**, the core roguelite element of this title. Relics provide powerful passive effects usable in subsequent runs.

| 遗物名称 / Relic Name | 效果 / Effect | 品质 / Rarity |
|---|---|---|
| 远古龙心 / Ancient Dragon Heart | 火焰伤害+25% / Fire damage +25% | 稀有 / Rare |
| 暗月碎片 / Dark Moon Shard | 魔法伤害+30% / Magic damage +30% | 传说 / Legendary |
| 黄金树嫩芽 / Golden Seedling | 开局额外圣杯瓶 / Bonus flask charge at start | 普通 / Common |
| 白银脚环 / Silver Anklet | 移动速度+10% / Movement speed +10% | 稀有 / Rare |
| 死灵之书 / Necronomicon | 击杀召唤亡灵战士 / Summon undead warrior on kill | 传说 / Legendary |
| 混沌之卵 / Chaos Egg | 所有属性+3但受伤增加 / All stats +3 but damage taken up | 诅咒 / Cursed |
| 时之沙漏 / Time Hourglass | 白天持续时间+3分钟 / Daytime +3 minutes | 稀有 / Rare |
| 友谊徽章 / Friendship Badge | 团队治疗效果+20% / Team heal effect +20% | 普通 / Common |

#### 遗物搭配 / Relic Synergy

部分遗物同时装备会产生协同效果：

Some relics create synergy effects when equipped together:

- 龙心 + 龙鳞碎片 / Dragon Heart + Dragon Scale Fragment: 变为龙形态变身 / Transform into dragon form
- 暗月碎片 + 星之核 / Dark Moon Shard + Star Core: 魔法附带追踪效果 / Magic gains homing effect
- 黄金树嫩芽 + 永恒露滴 / Golden Seedling + Eternal Dew: 圣杯瓶可无限使用一次 / Flask can be used infinitely once per run

#### 遗物解锁 / Relic Unlocks

不同难度和挑战模式会解锁专属遗物：

Different difficulties and challenge modes unlock exclusive relics:

- **普通通关** / **Normal Clear**: 解锁基础遗物池 / Unlocks basic relic pool
- **困难通关** / **Hard Clear**: 解锁稀有遗物 / Unlocks rare relics
- **噩梦通关** / **Nightmare Clear**: 解锁传说遗物 / Unlocks legendary relics
- **全成就** / **All Achievements**: 解锁唯一外观遗物 / Unlocks unique cosmetic relic

### 难度选项 / Difficulty Options

通关一次后可以解锁更高难度：

Higher difficulties are unlocked after clearing the game once:

#### 困难模式 / Hard Mode

- 敌人伤害+50% / Enemy damage +50%
- 敌人血量+75% / Enemy HP +75%
- 更少的篝火点 / Fewer Sites of Grace
- 更短的白天时间 / Shorter daytime
- 额外奖励: 稀有遗物掉落率提升 / Bonus: Rare relic drop rate increased

#### 噩梦模式 / Nightmare Mode

- 敌人伤害+100% / Enemy damage +100%
- 敌人血量+150% / Enemy HP +150%
- 只有1个初始圣杯瓶 / Only 1 starting flask charge
- 夜晚出现额外精英怪 / Extra elite enemies at night
- 友军伤害开启 / Friendly fire enabled
- 额外奖励: 传说遗物必定掉落 / Bonus: Legendary relic guaranteed drop

#### 自定义难度 / Custom Difficulty

玩家可以自定义以下参数调整难度：

Players can customize the following parameters:

| 参数 / Parameter | 范围 / Range | 默认 / Default |
|---|---|---|
| 敌人伤害倍率 / Enemy Damage | 0.5x - 3.0x | 1.0x |
| 敌人血量倍率 / Enemy HP | 0.5x - 3.0x | 1.0x |
| 白天时长 / Day Length | 5-30分钟 / 5-30 min | 15分钟 / 15 min |
| 初始等级 / Starting Level | 1-50 | 1 |
| 初始装备 / Starting Gear | 无/基础/高级 / None/Basic/Advanced | 基础 / Basic |
| 重生次数 / Revive Count | 0-无限 / 0-Infinite | 3 |

### NG+ / New Game Plus

通关后可以选择进入NG+，继承部分进度重新开始：

After clearing, you can enter NG+ carrying over some progress:

**继承内容** / **Carried Over**:
- 已解锁的职业 / Unlocked classes
- 已获得的遗物 / Collected relics
- 外观奖励 / Cosmetic rewards
- 合作点数 / Co-op points

**重置内容** / **Reset**:
- 角色等级 / Character level
- 武器装备 / Equipment and weapons
- 地图进度 / Map progress
- 剧情道具 / Story items

### 挑战模式 / Challenge Modes

#### 无尽黑夜 / Endless Night

一种特殊模式，没有三天限制，夜晚BOSS无限轮替出现。每击败一个BOSS会获得强化，直到队伍团灭。

A special mode with no three-day limit, night bosses rotate infinitely. Each boss defeat grants power-ups until the party wipes.

**奖励** / **Rewards**:
- 每5波获得一个遗物 / Relic every 5 waves
- 第10波解锁隐藏外观 / Wave 10 unlocks hidden cosmetic
- 最高波数排行榜 / Highest wave leaderboard

#### 速通模式 / Speedrun Mode

限制时间完成游戏。在最短的游戏时间内通关第三夜。

Complete the game within a time limit. Clear Night 3 in the shortest possible game time.

| 评价 / Rank | 时间要求 / Time Requirement | 奖励 / Reward |
|---|---|---|
| S级 / S Rank | 低于30分钟 / Under 30 min | 金色称号 / Gold title |
| A级 / A Rank | 30-45分钟 / 30-45 min | 银色称号 / Silver title |
| B级 / B Rank | 45-60分钟 / 45-60 min | 铜色称号 / Bronze title |
| C级 / C Rank | 超过60分钟 / Over 60 min | 参与奖 / Participation |

#### 禁咒模式 / Cursed Run

开局随机获得三个负面遗物，但通关奖励大幅提升。

Start with three random cursed relics, but greatly increased clear rewards.

| 诅咒 / Curse | 效果 / Effect |
|---|---|
| 虚弱之咒 / Curse of Frailty | HP上限减半 / Max HP halved |
| 饥渴之咒 / Curse of Thirst | 圣杯瓶数量减半 / Flask count halved |
| 迟缓之咒 / Curse of Slowness | 移动速度-30% / Movement speed -30% |
| 脆弱之咒 / Curse of Brittleness | 受到的物理伤害翻倍 / Physical damage taken doubled |
| 沉默之咒 / Curse of Silence | 无法使用魔法 / Cannot use magic |

### 重复游玩价值 / Replay Value

为什么值得反复游玩：

Why Nightreign is worth repeated playthroughs:

1. **随机化要素** / **Randomized elements** — 每次游戏的物品和敌人位置不同 / Item and enemy positions change each run
2. **职业多样性** / **Class variety** — 每种职业玩法差异巨大 / Each class plays dramatically differently
3. **难度梯度** / **Difficulty scaling** — 从普通到噩梦逐步挑战 / Gradual challenge from Normal to Nightmare
4. **社交体验** / **Social experience** — 和不同朋友组队的乐趣 / Fun of playing with different friends
5. **Build多样性** / **Build diversity** — 无数种Build组合 / Countless build combinations
6. **排行榜竞争** / **Leaderboard competition** — 与全球玩家比拼 / Compete with global players
