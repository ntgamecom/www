# 艾尔登法环 黑夜君临
## Elden Ring Nightreign

### 游戏概览 / Game Overview

**艾尔登法环 黑夜君临**（Elden Ring Nightreign）是 FromSoftware 开发的独立合作生存动作RPG。本作并非《艾尔登法环》的DLC，而是以同名世界观为基础打造的**全新独立作品**，支持最多三名玩家联机合作。

Elden Ring Nightreign is a standalone co-op survival action RPG developed by FromSoftware. Built on the same world mythology as the original Elden Ring, this is a **brand-new independent title** (not DLC) supporting up to three-player online co-op.

### 关键特色 / Key Features

| 特色 / Feature | 说明 / Description |
|---|---|
| 游戏类型 / Genre | 合作生存动作RPG / Co-op Survival Action RPG |
| 玩家数量 / Players | 1-3人联机 / 1-3 Player Co-op |
| 核心循环 / Core Loop | 三日昼夜循环生存 / 3-Day Night Cycle Survival |
| 战斗系统 / Combat | 基于艾尔登法环强化版 / Enhanced Elden Ring combat |
| 地图 / Map | 类似宁姆格福的开放世界 / Limgrave-inspired open world |

### 昼夜循环系统 / Day-Night Cycle

游戏的核心机制是**三天的昼夜循环**。白天是探索、收集和升级的时间；夜晚则要面对不断增强的敌人和BOSS战。每一天结束时会迎来更强大的夜晚BOSS，第三天则是最终决战。

The core mechanic is a **three-day day-night cycle**. Days are for exploration, gathering, and upgrading. Nights bring increasingly difficult enemies and boss fights. Each day ends with a more powerful night boss, culminating in the final battle on Day 3.

### 与《艾尔登法环》的区别 / Differences from Elden Ring

- **独立作品** / Standalone title — 无需原版游戏 / No base game required
- **专注合作** / Co-op focused — 而非单人RPG / Not a single-player RPG
- **生存要素** / Survival elements — 时间管理与资源分配 / Time management and resource allocation
- **没有PVP** / No PVP — 纯合作体验 / Pure co-op experience
- **roguelite元素** / Roguelite elements — 每次游戏不同 / Each run is different
