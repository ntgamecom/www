## 地图与探索 / World & Exploration

### 地图结构 / Map Structure

《黑夜君临》的地图灵感来源于《艾尔登法环》的**宁姆格福**（Limgrave）地区，但进行了大幅扩展和重新设计。地图分为多个子区域，每天会根据游戏进程解锁新区域。

Nightreign's map is inspired by Elden Ring's **Limgrave** region but greatly expanded and redesigned. The map is divided into multiple sub-zones, with new areas unlocking each day based on progression.

### 区域总览 / Zone Overview

#### 起始区域 / Starting Zone: 晨曦平原 / Dawn Plains

**难度** / **Difficulty**: 简单 / Easy
**推荐探索时间** / **Recommended Time**: 第一天白天 / Day 1 Daytime

晨曦平原是玩家初始降落的区域。这里有温和的野生动物、分散的废墟和基础资源点。适合熟悉操作和收集初期装备。

Dawn Plains is where players initially land. It features docile wildlife, scattered ruins, and basic resource nodes. Ideal for learning controls and gathering early gear.

**主要地点** / **Points of Interest**:

- **晨风教堂** / **Morning Wind Church** — 初始篝火点，NPC商人 / Starting grace site, NPC merchant
- **破碎集市** / **Ruined Market** — 大量可拾取的制作材料 / Many craftable material pickups
- **风车村** / **Windmill Village** — 有支线任务的小村庄 / Small village with side quest
- **暗水洞窟** / **Darkwater Cave** — 简单的初期地牢 / Simple early dungeon

#### 第一天解锁 / Day 1 Unlock: 薄暮森林 / Twilight Forest

**难度** / **Difficulty**: 中等 / Medium
**推荐探索时间** / **Recommended Time**: 第一天 / Day 1

穿过晨曦平原东边的桥梁即可到达薄暮森林。森林中视野受限，容易遭遇伏击。

Cross the bridge east of Dawn Plains to reach Twilight Forest. Vision is limited in the forest, making ambushes common.

**主要地点** / **Points of Interest**:

- **枯木教堂** / **Deadwood Church** — 技能教师所在地 / Skill teacher location
- **狼穴** / **Wolf Den** — 获得初期骨灰 / Early spirit ash location
- **铁匠小屋** / **Smithy Shack** — 初级武器升级 / Basic weapon upgrade
- **森林瞭望塔** / **Forest Watchtower** — 解锁地图碎片 / Unlocks map fragment

#### 第二天解锁 / Day 2 Unlock: 灰烬高地 / Ash Highlands

**难度** / **Difficulty**: 困难 / Hard
**推荐探索时间** / **Recommended Time**: 第二天 / Day 2

灰烬高地是一个火山活动频繁的区域，地形崎岖，敌人强大。这里可以找到高级升级材料。

Ash Highlands is a volcanically active region with rugged terrain and powerful enemies. Advanced upgrade materials can be found here.

**主要地点** / **Points of Interest**:

- **流火要塞** / **Flowfire Fortress** — 中获得传说武器 / Mid-game legendary weapon
- **岩浆洞穴** / **Magma Cave** — 火焰系祷告教学 / Fire incantation teacher
- **龙骨遗迹** / **Dragonbone Ruins** — 古代龙族相关剧情 / Ancient dragon lore
- **燃烧之桥** / **Burning Bridge** — 连接第三区域 / Connects to zone 3

#### 第三天解锁 / Day 3 Unlock: 终末之地 / Endlands

**难度** / **Difficulty**: 极难 / Very Hard
**推荐探索时间** / **Recommended Time**: 第三天白天 / Day 3 Daytime

终末之地是最终BOSS前的最后区域。这里有最强大的敌人和最好的战利品。

Endlands is the final area before the final boss. It houses the strongest enemies and the best loot.

**主要地点** / **Points of Interest**:

- **虚空遗迹** / **Void Ruins** — 最终商人 / Final merchant
- **黑塔** / **Black Tower** — 隐藏BOSS挑战 / Hidden boss challenge
- **永恒大门** / **Eternal Gate** — 最终BOSS入口 / Final boss entrance
- **遗物圣殿** / **Relic Sanctum** — 解锁终局遗物系统 / Unlocks endgame relic system

### 地图机制 / Map Mechanics

#### 篝火传送 / Grace Teleportation

已激活的篝火点可以作为传送点。篝火也会完全恢复HP和FP，并刷新区域内的小怪。

Activated Sites of Grace serve as teleport points. They fully restore HP and FP and respawn minor enemies in the area.

#### 地图碎片 / Map Fragments

每个主要区域都有一个**地图碎片**（Map Fragment）需要收集。拾取后才能显示该区域的完整地形和标记。

Each major zone has a **Map Fragment** to collect. Picking it up reveals the full terrain and markers for that zone.

| 区域 / Zone | 地图碎片位置 / Fragment Location |
|---|---|
| 晨曦平原 / Dawn Plains | 初始教学后自动获得 / Auto-obtained after tutorial |
| 薄暮森林 / Twilight Forest | 森林瞭望塔顶层 / Top of Forest Watchtower |
| 灰烬高地 / Ash Highlands | 燃烧之桥中间 / Middle of Burning Bridge |
| 终末之地 / Endlands | 虚空遗迹深处 / Deep within Void Ruins |

#### 快速旅行 / Fast Travel

在篝火处打开地图可以选择传送到任何已激活的篝火点。但不能在战斗中使用。

Open the map at a Site of Grace to teleport to any activated Grace. Cannot be used during combat.

#### 日夜切换 / Day-Night Transition

- **白天持续时间** / **Daytime Duration**: 约15分钟 / Approximately 15 minutes
- **夜晚持续时间** / **Nighttime Duration**: 约5分钟（BOSS战延长）/ Approx 5 minutes (extended during boss fight)
- **时间预警** / **Time Warning**: 白天剩余2分钟时屏幕会闪烁红光 / Screen flashes red when 2 minutes of daytime remain

### 探索技巧 / Exploration Tips

1. **高处优先** / **High Ground First** — 到达新区域先去制高点观察 / Reach high ground first to scout a new area
2. **跟着光线** / **Follow the Light** — 金色光芒指引重要地点 / Golden light guides important locations
3. **搜索角落** / **Search Corners** — 许多隐藏物品在死角 / Many hidden items are in dead ends
4. **留意发光植物** / **Watch for Glowing Plants** — 稀有制作材料的标志 / Markers for rare crafting materials
5. **破坏可碎墙壁** / **Break Illusory Walls** — 攻击可疑的墙壁 / Attack suspicious walls
6. **夜晚更危险** / **Night is More Dangerous** — 敌人属性在夜晚大幅提升 / Enemy stats greatly increase at night
7. **标记资源点** / **Mark Resource Nodes** — 在篝火处重置后可再次拾取 / Respawns after resting at Grace

### 区域进度影响 / Zone Progression Effects

第二天开始时，第一天的区域会发生变化：

At the start of Day 2, Day 1 areas change:

- 部分篝火失效 / Some graces become inactive
- 出现新敌人类型 / New enemy types appear
- 可拾取物品刷新 / Pickable items respawn
- 部分道路封锁或开放 / Some paths close or open
