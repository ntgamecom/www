## 新手入门 / Beginner's Guide

### 游戏开始 / Getting Started

当你首次进入《黑夜君临》，系统会引导你完成基础教学区域。这个**教程区域**（Tutorial Area）会教授移动、攻击、闪避和基本互动。完成教程大约需要15-20分钟。

When you first start Nightreign, you will be guided through a basic tutorial area. This tutorial teaches movement, attacking, dodging, and basic interactions. Completing the tutorial takes approximately 15-20 minutes.

### 角色创建 / Character Creation

创建角色时你需要选择**初始职业**（Starting Class）。每种职业有独特的初始装备、属性和技能。对于新手，推荐选择：

When creating a character you need to choose a starting class. Each class has unique starting equipment, stats, and abilities. For beginners, we recommend:

- **战士** / **Warrior** — 最平衡的选择，近战输出稳健 / The most balanced choice, solid melee damage
- **骑士** / **Knight** — 高防御高生命，容错率最高 / High defense and HP, most forgiving
- **牧师** / **Cleric** — 能回血能输出，团队作用大 / Can heal and deal damage, great for teams

建议第一次游戏选择**战士**或**骑士**，熟悉后再尝试法师或盗贼等高风险高回报职业。

We suggest picking **Warrior** or **Knight** for your first run, then trying higher-risk-high-reward classes like Mage or Rogue later.

### 第一天生存指南 / Day 1 Survival Guide

#### 白天阶段 / Day Phase

第一天白天是你最安全的探索时间。建议优先做以下事情：

Day 1 daytime is your safest exploration period. Prioritize the following:

1. **搜索物资** / **Scavenge supplies** — 拾取所有可见的素材和消耗品 / Pick up all visible materials and consumables
2. **升级武器** / **Upgrade weapons** — 找到铁匠NPC或工作台升级初始武器 / Find the blacksmith NPC or workbench to upgrade your starting weapon
3. **学习技能** / **Learn skills** — 找到技能教师购买或学习新技能 / Find skill teachers to buy or learn new skills
4. **标注地图** / **Mark the map** — 标记感兴趣的地点以便之后返回 / Mark points of interest for later return
5. **击杀小怪** / **Kill minor enemies** — 积累经验和卢恩 / Accumulate experience and runes

#### 夜晚阶段 / Night Phase

第一次夜晚降临会让你感到压力。重要提示：

The first nightfall will feel stressful. Important tips:

- **寻找篝火** / **Find Sites of Grace** — 篝火是安全区，可以休息和传送 / Sites of Grace are safe zones for resting and teleporting
- **不要单独行动** / **Don't wander alone** — 始终与队友保持在一起 / Always stay with your teammates
- **使用远程攻击** / **Use ranged attacks** — 如果武器允许，远程消耗敌人 / If your weapon allows it, poke enemies from range
- **保存回复道具** / **Save healing items** — 不要浪费治疗，BOSS战前保留 / Don't waste heals, save them for the boss fight
- **注意时间** / **Watch the clock** — 夜晚BOSS出现前有预警提示 / There is a warning before the Night Boss appears

### 常见错误 / Common Mistakes

新手常犯的错误以及如何避免：

Common mistakes beginners make and how to avoid them:

1. **忽略升级** / **Ignoring upgrades** — 每到一个新区域先找铁匠 / Always find the blacksmith first in a new area
2. **浪费圣杯瓶** / **Wasting flask charges** — 半血时不要立即喝药，等脱离战斗再回复 / Don't heal immediately at half HP, wait until disengaged
3. **深入敌阵** / **Overextending** — 不要被敌人包围，退后重整 / Don't get surrounded, retreat and regroup
4. **不探索** / **Not exploring** — 每个角落都可能有宝箱或捷径 / Every corner may have a chest or shortcut
5. **忽视负重** / **Ignoring equip load** — 保持中等负重以下以保证灵活翻滚 / Stay at medium equip load or below for agile rolling

### 推荐新手Build / Recommended Beginner Build

**战士Build** / **Warrior Build**

- **属性加点** / **Stat Priority**: 生命力 Vigor 30, 耐力 Endurance 25, 力量 Strength 25, 灵巧 Dexterity 18
- **推荐武器** / **Recommended Weapon**: 直剑或阔剑 / Longsword or Broadsword
- **推荐盾牌** / **Recommended Shield**: 黄铜盾或熨斗盾 / Brass Shield or Iron Roundshield
- **推荐战技** / **Recommended Ash of War**: 准备架势或鲜血斩击 / Square Off or Bloody Slash

### 游戏目标 / Game Objectives

每次游戏的核心目标是在**三天内**尽可能地提升实力，在第三天夜晚面对最终BOSS。如果全队阵亡，游戏结束并返回主菜单。

The core objective of each run is to power up as much as possible over **three days** and face the final boss on Night 3. If the entire party dies, the run ends and returns to the main menu.
