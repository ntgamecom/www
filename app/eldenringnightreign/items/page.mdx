#### 传说武器 / Legendary Weapons

游戏中存在10把传说武器，获取它们可以获得成就：

There are 10 legendary weapons in the game that grant achievements:

1. **夜陨** / **Nightfall** — 大剑，夜晚攻击力翻倍 / Greatsword, double damage at night
2. **暗月大剑** / **Dark Moon Greatsword** — 智力近战终极武器 / Ultimate int melee weapon
3. **古龙岩槌** / **Ancient Dragon Hammer** — 雷属性大槌 / Lightning great hammer
4. **圣律大剑** / **Sacred Relic Sword** — 神圣属性特大剑 / Holy colossal sword
5. **腥红匕首** / **Crimson Dagger** — 击杀回血 / Heals on kill
6. **虚空之弓** / **Void Bow** — 无限弹药 / Infinite ammo
7. **永夜法杖** / **Eternal Night Staff** — 夜间魔法威力提升40% / 40% night magic power
8. **太阳圣印** / **Sun Seal** — 白天祷告效果翻倍 / Double day incantation effect
9. **龙牙拳套** / **Dragon Tooth Fist** — 拳套高削韧 / High poise fist weapon
10. **时之镰** / **Time Scythe** — 击杀延长白天时间 / Kills extend daytime

防具分为头部、胸部、手部和腿部四个部位，提供物理防御和属性防御。

Armor has four slots (head, chest, arms, legs) providing physical and elemental defense.

### 制作系统 / Crafting System

在地图上收集制作材料后，可以在篝火处制作道具。制作配方通过**料理书**（Cookbooks）解锁。

After gathering materials on the map, craft items at Sites of Grace. Recipes are unlocked via **Cookbooks**.

| 料理书 / Cookbook | 解锁配方 / Recipes Unlocked | 位置 / Location |
|---|---|---|
| 流浪汉料理书 / Nomadic Cookbook 1 | 基础壶类 / Basic pots | 新手区域 / Starting area |
| 流浪汉料理书 / Nomadic Cookbook 2 | 进阶壶类 / Advanced pots | 地下墓穴 / Catacombs |
| 传教士料理书 / Missionary Cookbook | 腌肝类 / Livers | 教堂 / Church |
| 装甲兵料理书 / Armorer's Cookbook | 箭矢类 / Arrows | 矿洞 / Mine |
| 古龙料理书 / Ancient Dragon Cookbook | 高级药剂 / Advanced medicine | 龙息山丘 / Dragon's Hill |

Weapon upgrades require specific Smithing Stones. Different upgrade levels need different materials.

| +1 至 +3 | 锻造石 1 / Smithing Stone 1 | 整个地图 / Entire map |
| +4 至 +6 | 锻造石 2 / Smithing Stone 2 | 矿洞 / Mines |
| +7 至 +9 | 锻造石 3 / Smithing Stone 3 | 后期区域 / Late areas |
| +10 至 +12 | 锻造石 4 / Smithing Stone 4 | 隐藏区域 / Hidden areas |
| +13 至 +15 | 锻造石 5 / Smithing Stone 5 | 世界BOSS / World Bosses |
