# 物品收集 / Items & Collectibles

卧龙的收集品主要包括：龙脉精华(提升HP)、龙脉晶石(提升药水回复)、旌旗(提升不屈等级)和武器/装备。全收集是通关高周目的基础 / Wo Long's main collectibles: Dragon Essence (HP), Dragon Crystals (heal potency), Battle Flags (Fortitude), and weapons/gear. Full collection is essential for higher NG cycles.

---

## 龙脉精华 / Dragon Essence

龙脉精华永久提升HP上限——共25个，每个+约2% HP。全收集后HP约为初始状态的2.5倍——对高周目生存至关重要 / Dragon Essence permanently raises max HP — 25 total, each ~+2% HP. Full collection roughly 2.5x base HP — critical for NG+ survival.

| 数量 / Count | 来源 / Source | 说明 / Note |
|:----:|------|------|
| **主线** | 12个主线关卡 | 20个——部分关卡有多个 / 20 total, some missions have multiple |
| **支线** | 特定支线奖励 | 2个——通关对应支线获得 / 2 from specific side missions |
| **DLC1** | 逐鹿中原 / Battle of Zhongyuan | 1个 |
| **DLC2** | 称霸江东 / Conqueror of Jiangdong | 1个 |
| **DLC3** | 风起荆襄 / Rise of Jingxiang | 1个 |
| **总计 / Total** | **全收集** | **25个** |

**收集提示**: 每关拿到龙脉精华后左上角会有"龙脉精华+"提示。如果通关时没看到这个提示，说明漏掉了。回关卡补收时看地图上的黄色标记——精华和晶石在地图上会闪烁 / **Tip**: You'll see "Dragon Essence+" in the top-left when you pick one up. If you don't see it before the boss, you missed it. Yellow markers on the map show essence/crystal locations when revisiting.

## 龙脉晶石 / Dragon Crystal

龙脉晶石提升药水(龙脉之气)的治疗量——共13个。每收集一个，药水回复量增加约5-8%。满级时一瓶药水可回复80%+ HP / Dragon Crystals increase healing from Dragon's Pulse — 13 total. Each crystal adds ~5-8% heal potency. At max, one potion heals 80%+ HP.

| 关卡 / Mission | 位置 / Location | 获取条件 / Condition |
|------|------|------|
| 1-2 黄天凶妖 / Demon Fort | Boss战后场景——左侧悬崖边 / After boss — left cliff edge | 击败Boss后回头即可 |
| 1-3 顿悟 / Enlightenment | 洞穴深处水池——潜入水下 / Deep cave pool — dive underwater | 需要找到水池入口 |
| 2-1 虎牢关 / Hulao Pass | 关底右侧岔路——击败精英守门 / Right fork before boss — defeat elite guard | 精英怪较强，建议清完小怪再打 |
| 2-2 荣耀 / Glory | 城寨顶层宝箱 / Fortress top floor chest | 需要绕路到顶层 |
| 3-1 天柱山 / Mount Tianzhu | 山顶佛像背后 / Behind mountain-top Buddha statue | 容易被忽略——打完精英后绕到佛像后面 |
| 3-2 天柱山之战 / Battle of Mt. Tianzhu | 战场右侧小屋角落 / Right shack corner in battlefield | - |
| 4-1 终局 / Finale | 王座下方暗门——进入地下密室 / Hidden door under throne — enter underground chamber | 击败最终Boss前也可以去 |

**为什么龙脉晶石重要**: 一周目可能感受不到，但到了二周目敌人伤害翻倍，药水不够用是卡关主因。建议一周目就把所有晶石收齐——不要留到二周目再去补 / **Why it matters**: NG+ enemy damage doubles. Insufficient healing is a common wall. Collect all 13 in NG — don't leave them for NG+.

## 旌旗收集 / Battle Flag Checklist

旌旗决定"不屈等级"上限——不屈等级越高，Boss战中你的士气下限越高。士气每比Boss高1级=攻防各+3% / Battle Flags determine your Fortitude cap. Higher Fortitude means higher minimum morale vs bosses. Each morale level above = ~3% ATK + DEF.

| 关卡 / Mission | 旌旗数 / Flags | 牙旗数 / Marking Flags | 不屈等级上限 / Max Fortitude |
|:----:|:----:|:----:|:----:|
| 山村遇袭 / Village of Calamity | 3 | 5 | 3 |
| 黄天凶妖 / Demon Fort | 4 | 6 | 7 |
| 虎牢关 / Hulao Pass | 3 | 4 | 10 |
| 顿悟 / Enlightenment | 3 | 4 | 13 |
| 荣耀 / Glory | 3 | 5 | 16 |
| 天柱山 / Mount Tianzhu | 4 | 6 | 20 |
| 荣耀之路 / Way of Glory | 3 | 4 | 23 |
| 天柱山之战 / Battle of Mt. Tianzhu | 4 | 6 | 27 |
| 终局 / Finale | 4 | 6 | 31 |
| 各DLC关卡 / DLC Missions | 各3面 / 3 each | - | 最终可达40 |

**旌旗全收集=不屈25**：全收集后进Boss战士气不低于25——Boss通常士气为25，你和Boss同级=公平对决。漏掉1面旗=不屈24=Boss比你高1级=他多3%攻防 / **All flags = Fortitude 25**: With full flags, your morale never drops below 25 vs bosses — equal playing field. Missing 1 flag = Fortitude 24 = boss has +3% advantage.

## 装备刷取 / Gear Farming

| 目标 / Target | 推荐关卡 / Mission | 说明 / Note |
|------|------|------|
| **等级提升** | 最新解锁的主线 / Latest story mission | 装备掉落等级永远取决于你当前打的关卡等级 |
| **特定套装 / Set Farming** | 对应Boss关卡反复刷 / Re-fight specific boss | 每个Boss有专属套装掉落——例如吕布掉落"霸王将军"套 |
| **四星装备 / 4-Star Gear** | 一周目后期主线 / Late NG story | 四星比三星多一个词缀位——过渡期目标 |
| **五星装备 / 5-Star Gear** | 二周目及以上 / NG+ and beyond | 五星只能在NG+掉落。五星有独特的"恩宠"词缀——决定后期Build方向 |
| **宝石镶嵌 / Gem Embedding** | 分解多余装备获得宝石 / Disassemble spare gear | 宝石分五行(金木水火土)——对应提升各属性攻击/防御。后期强化核心手段 |
| **武器强化 / Weapon Upgrade** | 铁匠铺使用强化材料 / Blacksmith upgrade | 强化等级上限随周目增加——NG+可+9，NG++可+12 |

**刷装备技巧**: 穿"装备掉落率提升"词缀的装备(饰品或头盔词缀)→选择"幸运"高的守护灵→复刷Boss关卡。最快的刷法：打完Boss→回村→再进。不要退出游戏，直接循环 / **Farming tips**: Equip "Item Drop Rate UP" affixes (accessory or helmet) → use high-luck Guardian Spirit → repeat boss missions. Fastest: kill boss → return to village → re-enter. Don't quit game, just loop.

**全收集建议流程**: 一周目主线通关→传送回每个关卡补收集(用地图黄色标记导航)→刷武器/装备成就→进入二周目。不需要在一周目全收集五星装备——那是二周目的内容 / **Recommended completion flow**: Clear NG story → teleport back to each mission for collectibles (use yellow map markers) → farm weapon/gear achievements → enter NG+.
