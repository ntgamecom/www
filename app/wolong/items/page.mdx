# 物品收集 / Items & Collectibles

卧龙的收集品主要包括：龙脉精华(提升HP)、龙脉晶石(提升药水回复)、旌旗(提升不屈等级) / Wo Long collectibles: Dragon Essence (HP up), Dragon Crystal (potion heal up), Battle Flags (Fortitude up).

---

## 龙脉精华 / Dragon Essence

龙脉精华永久提升HP上限——共25个，每个+约2% HP / Dragon Essence permanently raises max HP — 25 total, each ~+2% HP.

| 数量 / Count | 说明 / Note |
|:----:|------|
| **主线可收集** | 20个——分布在12个主线关卡中 / 20 in 12 main missions |
| **支线可收集** | 2个——特定支线奖励 / 2 from specific side missions |
| **DLC** | 3个——每个DLC各1个 / 3 total, 1 per DLC |
| **总计 / Total** | **25个**——全收集后HP翻倍不止 / **25** — HP more than doubles at max |

---

## 龙脉晶石 / Dragon Crystal

龙脉晶石提升药水(龙脉精华)的治疗量——共13个 / Dragon Crystals increase healing amount of Dragon's Pulse — 13 total.

| 关卡 / Mission | 位置 / Location |
|------|------|
| 1-2 | Boss战后场景——悬崖边 / After boss — cliff edge |
| 1-3 | 塔楼内部宝箱 / Inside tower chest |
| 2-1 | 村落后方水井底部 / Well bottom in village rear |
| 2-2 | 山洞内岔路尽头 / End of cave fork |
| ... | ... |

## 装备刷取 / Gear Farming

| 目标 / Target | 推荐关卡 / Mission | 说明 / Note |
|------|------|------|
| **高等级装备** | 后期主线——尤其是"天柱山" / Late main missions — esp. "Mount Tianzhu" | 等级最高的关卡掉率最高 |
| **特定套装** | 对应Boss的关卡反复刷 / Re-fight specific boss missions | 每个Boss掉落特定套装部件 |
| **五星装备** | 二周目以上 / NG+ and above | 五星只能在NG+掉落 / 5-star is NG+ only |
| **宝石** | 分解多余装备 / Disassemble spare gear | 宝石镶嵌是后期提升关键 |
