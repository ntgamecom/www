# 战斗系统 / Combat System

卧龙的战斗系统以"化解"为核心——类似只狼的弹反但更具攻击性 / Wo Long's combat system centers on "Deflecting" — similar to Sekiro's parry but more aggressive in nature.

---

## 化解系统 / Deflect System

### 基本化解 / Basic Deflect

| 操作 / Input | 效果 / Effect |
|---|---|
| **按B/○格挡 / Press B/○ to block** | 普通格挡——消耗气势 / Normal block — consumes spirit |
| **攻击命中瞬间按B/○ / Press B/○ at impact** | **化解**——不扣气势 + 增加敌人气势 / **Deflect** — no spirit cost + builds enemy spirit |
| **化解成功 / Successful Deflect** | 敌人进入大硬直——可以接绝脉 / Enemy staggered — can follow with Fatal Strike |
| **完美化解 / Perfect Deflect** | 蓝色闪光效果——比普通化解更多气势伤害 / Blue flash — more spirit damage than normal deflect |

### 化解转击 / Deflect Counter

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **化解后攻击 / Deflect then Attack** | 化解成功后立刻按攻击键——反击连击 / After deflect, press attack for counter combo |
| **气势伤害 / Spirit Damage** | 每次化解增加敌人的气势槽——满了可以绝脉 / Each deflect builds enemy's spirit bar — when full, fatal strike |
| **化解优先级 / Deflect Priority** | 化解优于格挡——尽量多化解少格挡 / Deflect is better than block — deflect more, block less |

## 气势系统 / Spirit System

气势是卧龙的精力系统——但比魂系的精力复杂得多 / Spirit is Wo Long's stamina system — but much more complex than souls stamina.

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **气势槽 / Spirit Bar** | 屏幕下方——蓝色为正（你优势），红色为负（你劣势） / Bottom of screen — blue positive, red negative |
| **正气势 / Positive Spirit** | 攻击命中+化解成功——增加正气势 / Hitting attacks + deflecting — builds positive spirit |
| **负气势 / Negative Spirit** | 被攻击+格挡+使用奇术——增加负气势 / Taking hits + blocking + using wizardry — builds negative spirit |
| **气势归零 / Spirit Zero** | 负气势满了=气势崩坏——进入大硬直，非常危险 / Full negative = spirit break — huge stagger, extremely dangerous |
| **绝脉 / Fatal Strike** | 敌人气势槽崩坏后——按R1+△处决 / After enemy spirit break — press for massive damage |

### 气势管理技巧 / Spirit Management Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **保持正气势 / Stay Positive** | 积极进攻和化解——不要被动防守 / Stay aggressive with attacks and deflects — don't be passive |
| **负气势时撤退 / Retreat When Negative** | 负气势时防御力下降+移动变慢——先拉开距离 / Negative spirit = lower defense + slower moves — disengage |
| **绝脉是核心输出 / Fatal Strikes = Core DPS** | 尽快打崩敌人气势然后绝脉——这是最高效的伤害方式 / Break spirit fast then fatal strike — most efficient damage |

## 武器系统 / Weapon System

### 武器类型 / Weapon Types

| 武器 / Weapon | 特点 / Features |
|---|---|
| **直剑 / Straight Sword** | 均衡——攻速适中 / Balanced — medium speed |
| **弯刀 / Curved Sword** | 快速连击——适合累积气势 / Fast combos — good for spirit building |

| **长枪 / Spear** | 突刺距离远+安全 / Long thrust range + safe |

| **大刀 / Greatsword** | 重武器代表 / Heavy weapon representative |
| **拳套 / Fists** | 最快攻速——武术技专用 / Fastest attack speed — for martial arts |

## 绝脉与处决 / Fatal Strikes

| 触发方式 / Trigger | 说明 / Description |
|---|---|
| **气势崩坏 / Spirit Break** | 敌人气势槽变红后按R1+△ / Enemy spirit bar turns red then R1+△ |
| **背刺 / Backstab** | 从背后接近按R1+△——对小兵秒杀 / From behind press R1+△ — instant kill on mobs |
| **处决动画 / Execution Animation** | 绝脉有固定动画——动画中无敌 / Fixed animation — invincible during animation |
| **绝脉伤害 / Fatal Strike Damage** | 基于武器攻击力和气势伤害加成 / Based on weapon attack and spirit damage bonus |

## 奇术系统 / Wizardry

奇术相当于卧龙的魔法——消耗气势释放 / Wizardry is Wo Long's magic — consumes spirit to cast.

| 奇术属性 / Element | 效果 / Effect | 代表法术 / Representative |
|---|---|---|
| **火 / Fire** | 火焰伤害+灼烧DOT / Fire damage + burn DOT | 炎咒、火球术 / Flame spell, Fireball |
| **水 / Water** | 冰伤害+冻伤减速 / Ice damage + freeze slow | 冰枪、寒冰术 / Ice spear, Freeze |
| **木 / Wood** | 雷电伤害+麻痹 / Lightning damage + paralysis | 雷击术、闪电链 / Lightning, Chain |
| **金 / Metal** | 毒伤害+中毒DOT / Poison damage + poison DOT | 毒咒、腐蚀术 / Poison, Corrosion |
| **土 / Earth** | 物理伤害+防御增益 / Physical damage + defense buff | 岩枪、大地护盾 / Rock spear, Earth shield |

### 奇术使用 / Wizardry Use

| 操作 / Input | 说明 / Description |
|---|---|
| **装备奇术 / Equip Wizardry** | 在牙旗处装备——最多4个 / Equip at Battle Flag — max 4 |
| **释放奇术 / Cast Wizardry** | L1+△/○/□/✕ / L1+key |
| **消耗 / Cost** | 消耗正气势——放太多会让你进入负气势 / Consumes positive spirit — overuse puts you in negative |
| **五行相克 / Elemental Cycle** | 木>土>水>火>金>木——克制的元素造成额外伤害 / Wood>Earth>Water>Fire>Metal>Wood — countered element deals bonus damage |

## 战斗常见问题 / Combat FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **为什么我格挡也扣血？/ Block still hurts?** | 普通格挡扣气势——气势变负后扣血。尽量用化解 / Normal block costs spirit — negative spirit = HP loss. Deflect instead |
| **化解时机怎么把握？/ Deflect timing?** | 跟只狼类似——在攻击命中瞬间按格挡键 / Similar to Sekiro — press block at moment of impact |
| **气势怎么快速恢复？/ Recover spirit fast?** | 成功化解+攻击命中——不进攻时自动缓慢恢复 / Successful deflects + attacks — slow auto recovery when idle |
| **绝脉怎么放不出来？/ Can't fatal strike?** | 敌人气势槽必须完全变红（崩坏状态）/ Enemy spirit bar must be completely red (broken) |
| **五行相克有用吗？/ Elemental cycle useful?** | 非常重要——用克制的属性打Boss伤害翻倍 / Very important — countered element deals 2x damage |
| **双武器怎么切换？/ Weapon switch?** | 按方向键左右切换两把武器——战斗中随时换 / D-pad left/right to swap between two weapons mid-combat |
| **绝脉伤害受什么影响？/ Fatal strike scaling?** | 受武器攻击力+气势伤害加成影响。使用气势伤害高的武器绝脉更强 / Scales with weapon ATK + spirit damage bonus |

## 实战技巧 / Combat Tips

| 技巧 / Tip | 说明 / Description |
|---|---|
| **架式切换 / Stance Switching** | 上段高伤低攻速/中段均衡/下段快速——根据战况灵活切换 / High=high dmg slow / Mid=balanced / Low=fast — switch based on situation |
| **化解后追击 / Deflect Follow-up** | 化解成功后轻攻击——反击连击。完美化解后反击伤害增加 / Light attack after deflect — counter combo. Perfect deflect increases counter damage |
| **绝脉优先级 / Fatal Strike Priority** | 气势崩坏后优先绝脉而不是继续攻击——绝脉伤害远超普通连招 / Prioritize fatal strike over normal attacks on spirit break — much higher damage |
| **气势崩坏应对 / Spirit Break Recovery** | 你的气势崩坏后——快速按闪避键脱离。负气势时不要硬拼 / When your spirit breaks — quick dodge away. Don't fight while negative spirit |
| **背刺技巧 / Backstab Tips** | 从背后接近未发现你的敌人——可以背刺秒杀小兵 / Approach unaware enemies from behind — instant kill on soldiers |
| **奇术打断 / Wizardry Interrupt** | 快速奇术可以打断敌人的蓄力攻击——特别是火系和小型雷系法术 / Fast wizardry can interrupt enemy charge attacks — fire and small lightning work best |
| **气势攻击和普通攻击区别？/ Spirit vs normal attack?** | 气势攻击（R2）造成更多气势伤害——适合打崩敌人。普通攻击（R1）速度快但气势伤害低 / Spirit attacks (R2) deal more spirit damage — good for breaking. Normal attacks (R1) are faster but deal less spirit damage |
| **吕布怎么打？/ Lu Bu strategy?** | 骑马阶段用远程或树枝打下来——下马后化解他的红色攻击。他出红招时是最佳输出窗口 / Mounted phase use ranged to dismount — deflect his red attacks on ground. Red moves = best damage window |
