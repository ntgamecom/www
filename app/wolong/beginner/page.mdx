# 新手入门 / Beginner Guide

卧龙的核心是化解——忘掉魂系的翻滚。本章详解所有基础机制 / Wo Long's core is deflection — forget souls-style rolling. This chapter covers all fundamental mechanics.

---

## 核心操作 / Core Controls (PC/PS)

| 操作 / Action | PS | PC | 说明 / Description |
|------|:----:|:----:|------|
| **攻击 / Attack** | R1 | 鼠标左键 / Left click | 普通攻击——轻攻击连段 / Normal attack — light attack chain |
| **重攻击 / Heavy Attack** | R2 | Shift+左键 / Shift+Left click | 蓄力重击——高伤害+高崩解 / Charged heavy — high damage + high spirit damage |
| **化解 / Deflect** | ○ | Space | **核心操作**——攻击命中瞬间按化解，零伤害+敌人硬直。比闪避重要一百倍 / **Core mechanic** — press at impact for 0 damage + enemy stagger. 100x more important than dodge |
| **闪避 / Dodge** | ✕ + 方向 / ✕ + direction | Ctrl | 几乎没有无敌帧——用于走位而非躲攻击 / Almost no i-frames — for positioning, not dodging |
| **绝脉 / Fatal Strike** | ○（架势崩后）/ ○ (broken posture) | Space | 处决——架势崩溃后近身按化解键。造成固定比例伤害 / Execute — press deflect on broken posture. Deals fixed % damage |
| **防御 / Block** | L1 | Q | 格挡——会消耗气势但仍然扣血。不建议一直防守 / Block — consumes spirit still takes chip damage. Not for sustained defense |
| **奇术 / Wizardry** | R1+□/△/○/✕ | 数字键 1-4 | 使用装备的五行法术 / Use equipped elemental spells |
| **使用道具 / Use Item** | 方向键 / D-pad | E / R | 使用快捷道具栏中的物品 / Use quick slot items |
| **锁定 / Lock On** | R3 | 鼠标中键 / Middle click | 锁定当前敌人 / Lock onto current target |
| **神兽召唤 / Divine Beast** | L2+R2 | V | 释放神兽技能——需要满神兽条 / Unleash Divine Beast ability — requires full gauge |

## 化解系统详解 / Deflection System

化解是卧龙的灵魂——熟练掌握就是高手 / Deflection is the soul of Wo Long — master it and you master the game.

| 概念 / Concept | 说明 / Description |
|------|------|
| **化解时机 / Timing** | 在敌人攻击命中的瞬间按○——有约8帧（水属性加点可增加窗口）/ Press ○ at the moment of impact — ~8 frame window (Water stat widens it) |
| **化解成功效果 / Success Effect** | 不扣血 + 敌人硬直 + 大量削敌人气势。你的气势恢复 / Zero damage + enemy stagger + massive enemy spirit damage. You gain spirit |
| **化解失败效果 / Fail Effect** | 正常被击中——扣血。不是灾难，但不要依赖 / Take full damage. Not a disaster, but don't rely on luck |
| **化解红光技 / Red-Attack Deflection** | 红光技（Boss大招）同样可以化解——成功后敌人长时间硬直，给你大量输出窗口 / Red-glow attacks can also be deflected — success = long enemy stun for massive punish |
| **绝脉追击 / Fatal Strike Follow-Up** | 化解成功后紧接着攻击可以快速打崩敌人架势 / Follow up deflection with attacks to quickly break enemy posture |
| **连续化解 / Chain Deflection** | 部分Boss有高速连击——需要连续化解。节奏感很重要 / Some bosses have rapid combos — chain deflect. Rhythm is key |
| **水属性 / Water Stat** | 增加化解判定窗口——新手最推荐首加的属性 / Widens deflect window — most recommended stat for beginners |

> **训练方法**：前期张梁(Boss)是最好的练习对象——他的攻击前摇清晰，节奏固定，适合练化解 / **Practice**: Early boss Zhang Liang has clear telegraphed attacks and fixed rhythm — perfect deflection practice

## 士气系统 / Morale System

士气是卧龙独有的等级系统——直接决定你的战斗能力 / Morale is Wo Long's unique leveling system — directly determines combat effectiveness.

| 概念 / Concept | 说明 / Description |
|------|------|
| **士气等级 / Morale Level** | 1-25级——直接影响造成的伤害（乘数）和受到的伤害（除数） / 1-25 — directly multiplies damage dealt and divides damage taken |
| **不屈等级 / Fortitude** | 你的基础士气——死亡后士气的下限（不会低于此等级）。通过插旌旗提升 / Your base morale — the floor after death. Raised by planting Battle Flags |
| **提升士气 / Raising Morale** | 击杀敌人 +1，绝脉精英 +2-3，插旌旗固定不屈等级 / Kill enemy +1, fatal strike elite +2-3, flags raise Fortitude |
| **死亡惩罚 / Death Penalty** | 士气降低，至少降到不屈等级。连续死亡会越来越惨 / Morale drops, at least to Fortitude level. Consecutive deaths snowball |
| **士气差 / Morale Gap** | 如果Boss士气比你高——你打它像挠痒，它打你像切菜。务必找齐旌旗再打Boss / If boss morale > yours — you tickle them, they obliterate you. Always max flags before boss |
| **旌旗类型 / Flag Types** | **旌旗(大)**: 战场上的主要旗点，提升不屈等级 + 回满血。**牙旗(小)**: 小旗点，存档+补给 / **Battle Flag (large)**: Raises Fortitude + full heal. **Marking Flag (small)**: Checkpoint + resupply |

## 五德属性系统 / Five Phase Stats

| 属性 / Stat | 中文 | 影响 / Affects | 推荐武器 / Recommended Weapons |
|:----:|:----:|------|------|
| **Wood** | 木 | HP + 雷属性伤害 / HP + Lightning damage | 棍杖 / Staff |
| **Fire** | 火 | 攻击力 + 火属性伤害 / Attack + Fire damage | 大刀 / Glaive, 双剑 / Dual Swords |
| **Earth** | 土 | 负重 + 土属性伤害 / Weight limit + Earth damage | 锤 / Hammer, 大斧 / Great Axe |
| **Metal** | 金 | 气势伤害 + 毒属性 / Spirit damage + Poison | 拳套 / Fists, 弯刀 / Curved Sword |
| **Water** | 水 | 化解窗口 + 冰属性 / Deflect window + Ice damage | 双刀 / Dual Blades, 剑 / Sword |

> **加点建议**：优先加主武器的对应属性到需求值，然后补**水**（增加化解窗口）。其他属性视需要加 / **Tip**: Meet weapon stat requirements first, then invest in **Water** (deflect window). Add others as needed

## 开局职业选择 / Starting Class

| 职业 / Class | 初始属性 / Stats | 推荐理由 / Why Pick |
|------|------|------|
| **木行 / Wood Phase** | 木+7 | HP最高——新手最容易生存。初始武技实用 / Highest HP — best survivability for beginners |
| **火行 / Fire Phase** | 火+7 | 攻击最高——适合高玩速杀 / Highest attack — for aggressive players |
| **土行 / Earth Phase** | 土+7 | 负重高——可以穿重甲+大武器 / High weight limit — heavy armor + big weapons |
| **金行 / Metal Phase** | 金+7 | 气势控制优秀——适合技巧型玩家 / Best spirit management — for technical players |
| **水行 / Water Phase** | 水+7 | 化解最易——增加化解窗口，新手推荐 / Easiest deflection — widest window, recommended for beginners |

> **推荐**：新手选**水行**或**木行**。水行化解易，木行血量高 / **Recommendation**: Beginners pick **Water** or **Wood**. Water for deflection, Wood for HP

## 新手常见问题 / FAQ

| 问题 / Question | 回答 / Answer |
|------|------|
| **打张梁打不过？/ Stuck on Zhang Liang?** | 第一阶段躲他三连后攻击；第二阶段他的红光技可化解——成功后处决。保持中距离引诱他出招 / Phase 1: dodge his 3-hit then attack; Phase 2: deflect his red attack → fatal strike. Mid-range bait |
| **士气不够打Boss？/ Morale too low for boss?** | 回头找旌旗——地图上红色旗点标记是你漏掉的。每个小兵杀一遍提升士气 / Go back for flags — red markers on map show missed ones. Kill every mob for morale |
| **装备太重了？/ Over-encumbered?** | 土属性增加负重上限。或者换轻甲。负重超过100%时翻滚变慢 / Earth stat increases weight limit. Or equip lighter armor. Over 100% = fat roll |
| **奇术怎么装备？/ How to equip wizardry?** | 在牙旗处选择"奇术装备"——需要对应五德点数达到要求 / At flag: "Equip Wizardry" — requires minimum phase stat |
| **怎么刷装备？/ How to farm gear?** | 后期在"天柱山"刷特定Boss。前期随便用掉落——不用刻意刷 / Late game: farm specific bosses at "Mount Tianzhu." Early game: just use drops |
