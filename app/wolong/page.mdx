## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|------|------|
| **化解 / Deflection** | 核心战斗机制——与只狼类似：按B/○在攻击命中瞬间格挡，零伤害+崩解敌人架势。比闪避更重要 / Core combat — similar to Sekiro: press B/○ at the moment of impact for zero damage + posture break. Better than dodging |
| **士气系统 / Morale System** | 士气等级（1-25）直接决定攻防倍率。击杀敌人提升士气，死亡大幅降低。旌旗提高基础士气（不屈等级）/ Morale level (1-25) directly affects damage dealt/taken. Killing raises morale, dying drops it. Battle Flags raise base morale (Fortitude) |
| **五德 / Five Phases** | 木(木)、火(火)、土(土)、金(金)、水(水)——属性加点影响武器需求、奇术解锁、神兽搭配 / Wood, Fire, Earth, Metal, Water — stat allocation affects weapon requirements, wizardry unlocking, divine beast synergy |
| **奇术 / Wizardry Spells** | 各属性对应的法术——火攻、冰刺、雷击、土盾、毒雾等。消耗气势条释放 / Elemental spells — fire, ice, lightning, earth, poison. Consumes spirit gauge |
| **神兽 / Divine Beasts** | 守护灵——提供被动加成和主动技能。可切换，每个神兽对应一种五德属性 / Guardian spirits — passive bonuses + active skills. Swappable, each aligned to a phase |
| **气势 / Spirit Gauge** | 类似精力条——攻击消耗气势，化解和攻击命中恢复气势。气势为正时伤害提升 / Similar to stamina — attacks consume spirit, deflections & hits restore spirit. Positive spirit = bonus damage |
| **绝脉 / Fatal Strike** | 打崩敌人架势后按B/○处决——造成巨额伤害。对精英/首领可使用两次 / Execute on broken posture — massive damage. Works twice on elites/bosses |

## 开局建议 / Starting Tips

| # | 建议 / Tip | 说明 / Description |
|:----:|------|------|
| **1** | **学会化解，不要翻滚 / Master Deflection, Not Dodging** | 卧龙的翻滚判定严格且无敌帧少——化解（B/○）才是生存关键。从张梁开始练习 / Dodging has tight i-frames — deflection (B/○) is your survival tool. Practice on Zhang Liang |
| **2** | **士气是王道 / Morale is Everything** | 如果Boss士气比你高很多——你几乎打不动。先去清小兵提升士气再挑战Boss。找齐旌旗 / If boss morale >> yours, you deal 0 damage. Clear mobs for morale first. Find all flags |
| **3** | **重视绝脉 / Prioritize Fatal Strikes** | 处决造成固定比例伤害——不管打多少血，一招绝脉直接削掉一大截。对Boss可以用两次 / Fatal strike deals fixed % damage — massive chunk regardless of remaining HP. Can be used twice on bosses |
| **4** | **五行相克 / Exploit Elemental Weakness** | 火克金、金克木、木克土、土克水、水克火——用克制属性攻击额外削气势。武器附魔对应克制属性 / Fire > Metal > Wood > Earth > Water > Fire — counter-element attacks deal bonus spirit damage. Buff weapon with counter element |
| **5** | **红旗必全拿 / Collect All Battle Flags** | 关卡中的每一面旌旗（Battle Flag）都提升你的不屈等级（最低士气）。不拿旗的话死一次掉到1级——根本没法打 / Every Battle Flag raises Fortitude (minimum morale). Miss flags = die once to morale 1 — impossible |
| **6** | **不要贪刀 / Don't Get Greedy** | 卧龙的Boss攻击频率高、硬直短——贪一刀可能被反打一套。化解后打1-2刀就收 / Bosses attack fast with short windows — one greedy hit = combo punish. 1-2 hits after deflection then reset |
| **7** | **前期加水/土 / Prioritize Water/Earth Early** | 水属性增加化解判定窗口长度；土属性增加负重上限和防御。前期最实用的两种加点 / Water increases deflection window; Earth increases weight limit and defense. Most useful early stats |
| **8** | **装备制作 / Craft Gear Early** | 在牙旗处可以用素材升级和镶嵌宝石——不要只靠掉落。强化武器优先级最高 / Upgrade and embed gems at flags — don't rely on drops. Prioritize weapon upgrades |
