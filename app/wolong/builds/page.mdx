# 流派构建 / Builds & Equipment

卧龙的Build围绕**主武器+主属性(五德)+神兽**三角展开——没有最强Build只有最适合你的 / Wo Long builds revolve around the **weapon + main stat (phase) + divine beast** triangle — there's no single best build, only what fits your playstyle.

---

## 武器推荐 / Weapon Recommendations

| 武器 / Weapon | 推荐属性 / Stat | 特点 / Characteristics |
|:----:|:----:|------|
| **直剑 / Straight Sword** | 水/木 | 均衡型——速度快、耗精低 / Balanced — fast, low spirit cost |
| **大刀 / Glaive** | 火/土 | 大范围横扫+高崩解——适合压制流 / Wide sweeps + high spirit damage — for aggressive players |
| **双剑 / Dual Swords** | 火/水 | 攻速之王——连击快但范围短 / Fastest attack speed — quick combos but short range |
| **棍杖 / Staff** | 木/土 | 中距离+可蓄力——安全输出 / Mid-range + chargeable — safe damage |
| **大斧 / Great Axe** | 土/金 | 最慢但最高伤害+最高崩解——一刀流 / Slowest but highest damage + spirit break — one-shot style |
| **拳套 / Fists** | 金/火 | 连段丰富——操作上限高 / Most versatile combos — high skill ceiling |
| **弯刀 / Curved Sword** | 金/水 | 流血+毒属性——DOT流 / Bleed + poison — damage-over-time style |
| **锤 / Hammer** | 土 | 钝器——高崩解+容易打晕 / Blunt — high spirit damage + easy stuns |

## 主流Build推荐 / Popular Builds

### 1. 火德猛攻流 / Fire Aggression Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主属性 / Main Stat** | 火(Fire) 99 |
| **副属性 / Secondary** | 土(Earth) 20+（穿重甲）/ for heavy armor |
| **武器 / Weapon** | 大刀 / Glaive |
| **神兽 / Divine Beast** | 朱雀 / Suzaku (火) |
| **战斗风格 / Playstyle** | 极限输出——火属性附魔+神兽技能爆发。适合熟练化解的玩家 / Max damage — fire buff + divine beast burst. For players comfortable with deflection |

**核心思路**: 火属性增加攻击力——堆到99火+朱雀神兽，附火符后一套连招可以秒大部分精英 / Fire stat increases attack — 99 Fire + Suzaku, fire buff = combo kill most elites

### 2. 水德化解流 / Water Deflection Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主属性 / Main Stat** | 水(Water) 99 |
| **副属性 / Secondary** | 木(Wood) 20+（加HP）/ for HP |
| **武器 / Weapon** | 双剑 / Dual Swords |
| **神兽 / Divine Beast** | 玄武 / Xuanwu (水) |
| **战斗风格 / Playstyle** | 防御反击——最大化化解窗口+冰属性控制。适合新手 / Defensive counter — max deflect window + ice control. Good for beginners |

**核心思路**: 水增加化解判定窗口——配合玄武的冰属性，化解后冰冻敌人，安全输出 / Water widens deflect window — with Xuanwu's ice, deflect = freeze enemy = safe damage

### 3. 土德重甲流 / Earth Tank Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主属性 / Main Stat** | 土(Earth) 80+ |
| **副属性 / Secondary** | 金(Metal) 20+ |
| **武器 / Weapon** | 大斧或锤 / Great Axe or Hammer |
| **神兽 / Divine Beast** | 应龙 / Yinglong (土) |
| **战斗风格 / Playstyle** | 坦克——重甲高防+高崩解武器，和Boss换血 / Tank — heavy armor + high spirit damage weapon, trade blows with bosses |

## 装备制作与强化 / Gear Crafting

| 操作 / Action | 说明 / Description |
|------|------|
| **升级武器 / Upgrade Weapon** | 在牙旗处使用铁匠——消耗"钢铁"素材升级。+9为最高 / Use blacksmith at flag — consumes "Steel" to upgrade. Max +9 |
| **镶嵌宝石 / Embed Gems** | 武器/防具有空槽——镶嵌对应五德宝石增加属性伤害/抗性 / Gear has slots — embed phase gems for elemental damage/resistance |
| **装备套装 / Set Bonuses** | 穿同一套的多件触发套装效果——后期开始收集 / Wearing multiple pieces of same set triggers set bonuses — start collecting late game |
| **优先升级武器 / Priority: Weapon** | 攻击力是最重要的——素材优先升级武器，防具后补 / Attack is king — upgrade weapon first, armor second |
