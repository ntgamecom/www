# 流派构建 / Builds & Equipment

卧龙的Build围绕**主武器+主属性(五德)+神兽**三角展开——没有最强Build只有最适合你的。关键在于找到你的战斗风格然后针对性加点和配装 / Wo Long builds revolve around the **weapon + main stat (phase) + divine beast** triangle — there's no single best build, only what fits your playstyle. The key is finding your combat style and building around it.

---

## 属性系统 / Attribute System

卧龙有五德属性：火、水、木、金、土。每种影响不同的战斗能力：

| 属性 / Phase | 影响 / Affects | 适合武器 / Weapons | 推荐神兽 / Divine Beast |
|:----:|------|:----:|:----:|
| **火 / Fire** | 攻击力 / Attack Power | 大刀、双剑、拳套 / Glaive, Dual Swords, Fists | 朱雀 / Suzaku |
| **水 / Water** | 化解窗口 / Deflect Window | 直剑、双剑、弯刀 / Straight Sword, Dual Swords, Curved Sword | 玄武 / Xuanwu |
| **木 / Wood** | 血量 / HP | 棍杖、直剑 / Staff, Straight Sword | 青龙 / Qinglong |
| **金 / Metal** | 气势伤害 / Spirit Damage | 弯刀、拳套 / Curved Sword, Fists | 白虎 / Baihu |
| **土 / Earth** | 防御+负重 / Defense + Equip Load | 大斧、锤、大刀 / Great Axe, Hammer, Glaive | 应龙 / Yinglong |

| 武器 / Weapon | 推荐属性 / Stat | 特点 / Characteristics |
|:----:|:----:|------|
| **直剑 / Straight Sword** | 水/木 | 均衡型——速度快、耗精低 / Balanced — fast, low spirit cost |

## 主流Build推荐 / Popular Builds

### 1. 火德猛攻流 / Fire Aggression Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主属性 / Main Stat** | 火(Fire) 99 |
| **副属性 / Secondary** | 土(Earth) 20+（穿重甲）/ for heavy armor |
| **武器 / Weapon** | 大刀 / Glaive |
| **神兽 / Divine Beast** | 朱雀 / Suzaku (火) |
| **战斗风格 / Playstyle** | 极限输出——火属性附魔+神兽技能爆发。适合熟练化解的玩家 / Max damage — fire buff + divine beast burst. For players comfortable with deflection |

**核心思路**: 火属性增加攻击力——堆到99火+朱雀神兽，附火符后一套连招可以秒大部分精英 / Fire stat increases attack — 99 Fire + Suzaku, fire buff = combo kill most elites

**实战打法**: 开局上火附魔(火符或奇术)→冲脸一套轻击+重击连段→气势条打满→气势攻击(重击终结技)→再次循环。朱雀神兽技能在Boss倒地时释放——伤害最大化。**副属性20土**: 满足重甲负重需求——否则翻滚变慢 / **Combat flow**: Fire buff → aggressive combo → max spirit → spirit attack → repeat. Use Suzaku burst when boss is staggered. **20 Earth**: required for heavy armor — keeps dodge roll speed normal.

### 2. 水德化解流 / Water Deflection Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主属性 / Main Stat** | 水(Water) 99 |
| **副属性 / Secondary** | 木(Wood) 20+（加HP）/ for HP |
| **武器 / Weapon** | 双剑 / Dual Swords |
| **神兽 / Divine Beast** | 玄武 / Xuanwu (水) |
| **战斗风格 / Playstyle** | 防御反击——最大化化解窗口+冰属性控制。适合新手 / Defensive counter — max deflect window + ice control. Good for beginners |

**核心思路**: 水增加化解判定窗口——配合玄武的冰属性，化解后冰冻敌人，安全输出 / Water widens deflect window — with Xuanwu's ice, deflect = freeze enemy = safe damage

**实战打法**: 等敌人攻击→化解(按B/○)→敌人硬直(冰冻)→双剑轻击连段→气势攻击→收手→等下一次攻击循环。水德流的精髓在于"化解→冰冻→输出"三步循环——化解决定生存，冰冻决定控制，输出决定击杀速度 / **Combat flow**: Wait for enemy attack → deflect (B/○) → enemy stunned (frozen) → dual sword combo → spirit attack → reset. The essence is "deflect → freeze → damage" three-step loop.

### 3. 土德重甲流 / Earth Tank Build

| 配置 / Setup | 推荐 / Recommendation |
|------|------|
| **主属性 / Main Stat** | 土(Earth) 80+ |
| **副属性 / Secondary** | 金(Metal) 20+ |
| **武器 / Weapon** | 大斧或锤 / Great Axe or Hammer |
| **神兽 / Divine Beast** | 应龙 / Yinglong (土) |
| **战斗风格 / Playstyle** | 坦克——重甲高防+高崩解武器，和Boss换血 / Tank — heavy armor + high spirit damage weapon, trade blows with bosses |

**实战打法**: 土德流的容错率最高——穿最重的甲吃Boss一下也不疼。大斧蓄力攻击(按住重击)打崩解效果极好——2-3下就能清空Boss气势条。气势条空了立刻接气势攻击(打倒地)→然后继续蓄力循环 / **Combat flow**: Highest margin for error — heavy armor absorbs boss hits. Great Axe charged attacks (hold heavy) deal massive spirit damage — 2-3 hits empty boss spirit bar. Follow with spirit attack (knockdown) → repeat.

## 装备制作与强化 / Gear Crafting

| 操作 / Action | 说明 / Description |
|------|------|
| **升级武器 / Upgrade Weapon** | 在牙旗处使用铁匠——消耗"钢铁"素材升级。+9为最高 / Use blacksmith at flag — consumes "Steel" to upgrade. Max +9 |
| **镶嵌宝石 / Embed Gems** | 武器/防具有空槽——镶嵌对应五德宝石增加属性伤害/抗性 / Gear has slots — embed phase gems for elemental damage/resistance |
| **装备套装 / Set Bonuses** | 穿同一套的多件触发套装效果——后期开始收集 / Wearing multiple pieces of same set triggers set bonuses — start collecting late game |
| **优先升级武器 / Priority: Weapon** | 攻击力是最重要的——素材优先升级武器，防具后补 / Attack is king — upgrade weapon first, armor second |

**装备升级节奏**: 每关至少升级一次武器。素材不够时可以重复刷之前通过的关卡。+3前素材用不完——+6以后开始稀缺。建议锁定2-3把喜欢的武器集中升级，不要分散投资 / **Upgrade rhythm**: Upgrade weapon at least once per mission. Replay earlier missions if low on materials. Pre-+3: abundant. +6+: scarce. Focus on 2-3 favorite weapons — don't spread upgrades thin.
