# 世界探索 / World Exploration

卧龙的关卡是线性的——但每关都有大量旌旗、隐藏路径和收集品。本章列出所有关键位置 / Wo Long's missions are linear — but each has many flags, hidden paths, and collectibles. This chapter covers all key locations.

---

## 关卡一：山村遇袭 / Village of Calamity

| 项目 / Item | 位置 / Location | 说明 / Note |
|------|------|------|
| **旌旗(x3)** | 1. 起始区域右转；2. 穿过山洞后左侧；3. Boss 房前石阶 / 1. Right from start; 2. Left after cave; 3. Stairs before boss | 全找到不屈+3 / All found = Fortitude +3 |
| **龙脉精华** | 山洞内的分岔路——跳下右侧悬崖 / Cave fork — jump off right cliff | 升级龙脉 +1 HP |
| **隐藏路径** | 巨大野猪前面——左侧小道绕过不打 / Before big boar — left path to skip it | 推荐绕过——前期打不过 / Recommended skip — too strong early |

## 探索技巧 / Exploration Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **标记看全图 / Use the Map** | 按触摸板打开地图——红色旗图标=你漏掉的旌旗。旗帜是攻略核心 / Open map — red flag icons = missed Battle Flags. Flags are key to progression |
| **梯子捷径 / Unlock Shortcuts** | 每关有很多梯子/门需要从另一侧打开——打开后从牙旗直通Boss房 / Many ladders/gates open from one side only — unlocking gives a direct route from flag to boss |
| **高低差 / Verticality** | 多上屋顶/高台——旌旗和龙脉精华经常在屋顶上 / Explore rooftops and ledges — flags and essence are often elevated |
| **刷士气 / Farm Morale Before Boss** | 打Boss前在附近低风险区域刷几个小兵——把士气刷到比Boss高再进雾门 / Before boss, farm nearby mobs until your morale exceeds the boss's |
| **DLC关卡 / DLC Missions** | 逐鹿中原(1)、称霸江东(2)、风起荆襄(3)——通关主线后解锁，建议二周目再打 / DLC 1-3 unlock after main story — recommended for NG+ |
