# 世界探索 / World Exploration

卧龙的关卡是线性的——但每关都有大量旌旗、隐藏路径和收集品。本攻略按主线关卡顺序列出所有关键位置 / Wo Long's missions are linear — but each has many flags, hidden paths, and collectibles. This guide covers all key locations in story order.

---

## 关卡系统说明 / Mission System Overview

卧龙共有 **13个主线关卡** + **大量支线关卡**。每个关卡有：
- **旌旗（大旗）**：增加不屈等级上限，大幅降低难度。全收集达到不屈25后，Boss战中士气不会低于25
- **牙旗（小旗）**：存档点和快速移动点，激活后解锁传送
- **龙脉精华**：永久提升HP上限，共25个
- **龙脉晶石**：提升药水回复量，共13个

Wo Long has **13 main missions + many side missions**. Each has Battle Flags (increase Fortitude cap), Marking Flags (checkpoints), Dragon Essence (HP up), and Dragon Crystals (heal power up).

### 山村遇袭 / Village of Calamity

| 项目 / Item | 位置 / Location | 说明 / Note |
|------|------|------|
| **旌旗(x3)** | 1. 起始区域右转小巷 / Alley right of start；2. 穿过山洞后左侧平台 / Platform left after cave；3. Boss 房前石阶 / Stairs before boss | 全找到不屈+3 |
| **牙旗(x5)** | 流程中自然发现 / Found along path | 激活即可传送 |

| **Boss: 张梁 / Zhang Liang** | 关卡终点 / Mission end | 新手劝退Boss——重点学习化解 / First major wall — learn deflection |

**攻略重点**: 张梁是卧龙的第一个Boss也是教学Boss。他有两个阶段：第一阶段缓慢的斧击容易化解，第二阶段会变身妖魔，攻击更快更狠。关键在于注意化解红光杀招，不要贪刀。如果实在打不过，可以先去支线练级再回来 / Zhang Liang is both tutorial and first wall. Phase 1 has slow axe swings (easy deflects), Phase 2 transforms into a demon with faster combos. Key: deflect his red critical attacks and don't get greedy.

Bring full heals, Earth defense and Water damage reduction. This fight tests your complete mastery of deflection.

## 探索技巧 / Exploration Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **标记看全图 / Use the Map** | 按触摸板打开地图——红色旗帜图标=你漏掉的旌旗。每次看到红旗都要去拿——1点不屈等级就是约5%的减伤 / Open map — red flag icons = missed Battle Flags. Every missing flag means ~5% more damage taken |
| **梯子捷径 / Unlock Shortcuts** | 每关有很多梯子/门需要从另一侧打开——打开后从牙旗直通Boss房。探索时遇到锁着的门/收起的梯子要先记住位置 / Many ladders/gates open from one side only — unlocking gives a direct route from last flag to boss |
| **高低差探索 / Verticality** | 多上屋顶/高台——旌旗和龙脉精华经常在屋顶或悬崖下。部分隐藏路径需要从高处跳下到达 / Explore rooftops and ledges — flags and essence are often elevated or hidden below cliffs |
| **刷士气再打Boss / Farm Before Boss** | 打Boss前在附近低风险区域刷几个小兵——把士气刷到比Boss高再进雾门。士气每高1级，你的攻防+约3% / Before boss, farm nearby mobs until your morale exceeds the boss's. Each morale level above the enemy gives ~3% ATK/DEF |
| **注意化解时机 / Deflection Timing** | 化解(格挡)是卧龙的核心机制——成功化解红光杀招不仅无伤还能让Boss大硬直。练习在攻击命中前的一瞬间按化解键 / Deflection (parry) is Wo Long's core mechanic. Successfully deflecting red critical attacks leaves bosses wide open. Practice timing — press deflect just before impact |
| **五行相克 / Five Phases** | 土克水→水克火→火克木→木克土→金克土。使用克Boss属性的武器和法术能大幅提升伤害。查看Boss身上的光环颜色判断属性 / Earth beats Water beats Fire beats Wood beats Earth beats Metal beats Wood. Use phase-advantage weapons for big damage. Check boss aura color |
| **支线关卡 / Side Missions** | 支线也是刷装备和练级的好地方。红色支线（高难度）掉率更高但也更危险。建议主线卡关时先去刷支线 / Side missions are great for farming. Red side missions (hard) have better drops. If stuck on main, farm sides first |
| **DLC关卡 / DLC Missions** | 逐鹿中原(1)、称霸江东(2)、风起荆襄(3)——通关主线后解锁，建议二周目后再打。DLC新增Boss如典韦、太史慈难度极高 / DLC 1-3 unlock after main story — recommended for NG+. DLC bosses like Dian Wei and Taishi Ci are extremely hard |
| **多周目变化 / NG+ Changes** | 二周目新增五星装备掉落、新词缀系统、Boss加强版。建议一周目全收集，二周目开始认真配装 / NG+ adds 5-star gear drops, new affix systems, and enhanced bosses. Do 100% in NG, start build-crafting in NG+ |

## 推荐探索顺序 / Recommended Progress Order

1. **一周目**：主线通关为主——尽量搜全旌旗但不强求。卡Boss就去刷士气
2. **全收集**：通关后用传送回每个关卡补全旌旗和龙脉精华
3. **二周目**：装备收集和Build构建——开始挑战DLC和更高难度

1. **NG**: Focus on clearing main story, collect flags when convenient. Farm morale if stuck on bosses.
2. **Completion**: After clearing, teleport back to collect all flags, essences, and crystals.
3. **NG+**: Gear collection and build crafting — tackle DLC and higher difficulties.
