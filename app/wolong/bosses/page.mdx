# BOSS攻略 / Boss Strategies

卧龙有约40个Boss——但所有Boss的核心攻略都一样：**化解红光技，绝脉收尾** / Wo Long has ~40 bosses — but the core strategy is the same for all: **deflect red attacks, fatal strike to finish**.

---

## 主线Boss / Story Bosses

### 张梁 / Zhang Liang (第一关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 山村遇袭 / Village of Calamity |
| **士气 / Morale** | 推荐 5+ |
| **弱点 / Weakness** | 雷属性 / Lightning |

**打法:**
- **第一阶段**: 人形态——使用大铁锤。攻击节奏慢但范围大 / Human form — large iron hammer. Slow but wide arcs
- 他的三连锤砸地——侧闪→等他砸完→打2刀→后退 / His 3-hit slam — side dodge → wait for finish → 2 hits → back off
- 跳起重砸——向侧面闪避然后反击 / Jump slam — side dodge → counter
- **第二阶段**: 妖魔化——攻击更快+红光技 / Demon form — faster + red attacks
- **红光技**: 他仰头吼叫后飞扑——看准时机**化解**。成功后大硬直，接一套连招 / Red attack: roar then lunge — **deflect** at right timing. Big stun window
- 核心节奏：化解红光→打2-3刀→等下一次红光 / Core rhythm: deflect red → 2-3 hits → wait for next red

### 封豨 / Fengxi (第二关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 黄天凶妖 / The Demon Fort |
| **士气 / Morale** | 推荐 8+ |
| **弱点 / Weakness** | 火属性 / Fire |

**打法:**
- 巨大的野猪型妖怪——头撞+冲撞为主 / Giant boar demon — head slam + charge
- 它冲锋时侧向闪避→然后追着它的**屁股打**——这是唯一的安全输出位置 / When it charges, side dodge → chase its **rear** — only safe damage zone
- **红光技**: 蓄力后全身发光冲刺——化解后它倒地很长时间，全力输出 / Red attack: glowing charge — deflect = long knockdown, go all out
- 它的正面和侧面都有伤害判定——不要在正前方 / Front and sides have hitboxes — never stand in front

### 张宝 / Zhang Bao (第三关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 黄天凶妖·终 / The Demon Fort Final |
| **士气 / Morale** | 推荐 12+ |
| **弱点 / Weakness** | 土属性（克雷）/ Earth (counters Lightning) |

**打法:**
- 雷法道士——远程雷球+近身雷杖 / Lightning mage — ranged lightning orbs + close-range staff
- 保持移动——他的雷球有追踪性，走位躲开 / Keep moving — his orbs track, strafe to dodge
- **红光技**: 召唤落雷——地面上有红圈预警。走出红圈 / Red attack: calls lightning — red circles on ground. Walk out of them
- 近身时他的杖击速度中等——节奏化解后打2刀 / Mid-speed staff strikes — deflect then 2 hits
- 第二阶段他召唤两个分身——先杀分身再打本体 / Phase 2 summons 2 clones — kill clones first then boss

## Boss通用技巧 / General Boss Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **士气检查 / Morale Check** | 进Boss房前确认士气 ≥ Boss士气。如果不 >=，先去刷级 / Before fog gate, check morale ≥ boss morale. If not, go farm |
| **红光化解优先 / Prioritize Red Deflections** | 所有Boss的红光技化解后都有大硬直——这是最佳输出窗口。等红光比主动进攻安全 / Every red attack deflect = big punish window. Waiting for reds is safer than aggressive offense |
| **绝脉两次 / Two Fatal Strikes** | 大部分Boss需要两次绝脉才会死。第一次打崩架势→绝脉→继续打→第二次绝脉收尾 / Most bosses need 2 fatal strikes. Break posture → fatal → continue → 2nd fatal = dead |
| **奇术辅助 / Wizardry Support** | Boss弱什么属性就用对应附魔——攻击+附魔消耗气势，但大幅提升伤害 / Use counter-element buff on weapon — costs spirit but massively boosts damage |
| **神兽时机 / Divine Beast Timing** | 满神兽条不要急着放——等Boss大硬直时（化解红光后）再放，确保全中 / Save divine beast for big stun windows (post-red deflection) — guarantee full hit |
| **防御优于闪避 / Block > Dodge** | L1格挡可以挡住大部分攻击——虽然扣气势但比被砍要好。不确定时就按L1 / Block (L1) absorbs most attacks — costs spirit but beats getting hit. When in doubt, hold L1 |
