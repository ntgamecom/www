# BOSS攻略 / Boss Strategies

卧龙有约40个Boss——但所有Boss的核心攻略都一样：**化解红光技，绝脉收尾**。红光技化解后Boss大硬直+削减大量气势槽——这是主要的输出窗口 / Wo Long has ~40 bosses — but the core strategy is universal: **deflect red attacks, fatal strike to finish**. Red attack deflection = massive stun + spirit damage — your primary damage window.

---

## 核心Boss战法则 / Core Boss Rules

| 法则 / Rule | 说明 / Description |
|:----:|------|
| **士气优先 / Morale First** | 进Boss房前确认士气 ≥ Boss士气(高1-2级最佳)。士气差1级≈攻防差3%——差5级就是15% / Check morale before boss. 1 level diff = ~3% ATK/DEF gap |
| **等红光 > 主动进攻 / Wait for Red** | 主动去打容易被反杀——等Boss出红光技→化解→输出。安全又高效 / Attacking first gets you killed. Wait for red → deflect → punish. Safe AND efficient |
| **两次绝脉 / Two Fatal Strikes** | 绝大多数Boss需要两次绝脉(处决)才能击杀。第一次打崩气势槽→绝脉→继续→第二次绝脉 / Most bosses need 2 fatal strikes. Break posture → fatal → repeat |
| **属性克制 / Element Counter** | 查看Boss身上光环颜色——用克制的属性附魔。克制=伤害+约30% / Check boss aura color for element. Counter-element buff = ~+30% damage |
| **防御保底 / Block is Safe** | 不确定时按L1格挡——比被打到好。格挡只扣气势(H被打中扣血量 + 被打飞 / When in doubt, hold L1 to block. Beats eating the hit |

## 主线Boss详解 / Story Bosses

### 张梁 / Zhang Liang (第一关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 山村遇袭 / Village of Calamity |
| **推荐士气 / Recommended Morale** | 5+ |
| **弱点 / Weakness** | 雷属性 / Lightning |
| **难度 / Difficulty** | ★★★ 新手第一个坎——教学Boss / First wall — tutorial boss |

张梁是卧龙的教学Boss——两个阶段检验你对化解和气势系统的掌握 / Zhang Liang tests your grasp of deflection and spirit mechanics across two phases.

**第一阶段(人形态)**: 使用大铁锤，攻击慢但范围大。关键攻击模式：
- **三连锤砸地**: 左→右→中连续砸三下。侧向闪避→等他砸完→打2刀→后退
- **跳起重砸**: 跳起后延迟约1秒砸下。向侧面闪避然后输出2刀
- **横扫**: 大范围横抡——格挡可接反击

**Phase 1 (Human)**: Large iron hammer, slow with wide arcs. Key attacks: 3-hit slam (left→right→center — side dodge each), jump slam (1s delay — side dodge), sweep (blockable).

**第二阶段(妖魔形态)**: 攻击更快+新增红光技。核心变招：
- **红光技 - 飞扑**: 仰头吼叫→飞扑。看到吼叫→等0.5秒→化解。成功后大硬直→连续输出4-5刀
- **红光技 - 乱舞**: 连续挥舞→最后一下是红光。前几下格挡→最后一下化解
- **普通攻击速度翻倍**——不要贪刀，打2刀就后退

**Phase 2 (Demon)**: Faster + red attacks. Core red attacks: Lunge (roar → pounce — deflect at 0.5s after roar, massive punish), Flurry (block hits → deflect final red hit, great punish). Double attack speed.

**新手建议**: 如果打不过，先回支线关卡练级到士气8+再来。张梁的节奏是"等→化解→打2刀→等"。不要试图主动连招——这是新手最常见的死因 / **Beginner tip**: Farm morale to 8+ if stuck. Rhythm is "wait → deflect → 2 hits → wait". Never initiate combos — most common death cause.

### 封豨 / Fengxi (第二关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 黄天凶妖 / The Demon Fort |
| **推荐士气 / Recommended Morale** | 8+ |
| **弱点 / Weakness** | 火属性 / Fire |
| **难度 / Difficulty** | ★★☆ 大型Boss——可预测但伤害高 / Large boss — predictable but punishing |

巨大的野猪型妖怪——头撞+冲撞为主。**核心策略**: 永远站在它**屁股后面**。它的正面和侧面都有大范围伤害判定——只有后方是安全的。**红光技**：蓄力→全身发光→冲刺。化解时机是发光后约0.5秒——成功后它会倒地很长时间(约3-4秒)，全力输出 / Giant boar demon — head slam + charge. **Core strategy**: Always stay behind it. Front and sides have massive hitboxes. Red attack: charge up → glow → rush. Deflect ~0.5s after glow — long knockdown (~3-4s), go all out.

**阶段变化**: 
- **P1(100%-50%)**: 头撞+冲锋为主。冲锋会撞墙→短暂眩晕→追着屁股打
- **P2(50%-0%)**: 新增甩尾(大范围)+毒液喷吐(地面毒池)。甩尾有前摇——看到尾巴抬起就闪避。毒池持续约10秒

**P1 (100%-50%)**: Head slam + charge. Charge into wall → stun → chase rear. **P2 (50%-0%)**: Tail sweep (wide range, raise tell → dodge) + poison spit (ground pool, 10s duration).

### 张宝 / Zhang Bao (第三关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 黄天凶妖·终 / Demon Fort Final |
| **推荐士气 / Recommended Morale** | 12+ |
| **弱点 / Weakness** | 土属性——克雷 / Earth — counters Lightning |
| **难度 / Difficulty** | ★★★ 法师型Boss——需要贴身压制 / Mage boss — needs aggressive close range |

雷法道士——远程雷球+近身雷杖。**核心策略**: 迅速贴近——法师Boss在远距离最危险。近身后他的近战攻击速度中等，容易化解。**注意**: 他的雷球有追踪性——不要直线走位，S型逼近 / Lightning mage — ranged orbs + staff. **Core strategy**: Close distance fast. He's most dangerous at range. In melee, his staff attacks are mid-speed and easy to deflect. **Note**: Orbs track — strafe in S-pattern, don't run straight.

**红光技**: 召唤落雷——地面上出现红圈预警(约1.5秒后落雷)。走出红圈即可。有时候他会连续召唤5-6个落雷——持续移动不要停 / Red attack: Summon lightning — red circles on ground (1.5s delay). Walk out of circles. May chain 5-6 strikes in sequence — keep moving.

**P2(50%-0%)**: 召唤两个雷属性分身(约70%HP)。分身比本体弱很多——优先击杀。分身死后本体会有短暂硬直→全力输出。注意分身的雷球和本体的雷球同时出现→弹幕密度翻倍 / P2 (50%-0%): Summons 2 lightning clones (~70% HP). Clones are much weaker — kill first. Boss is briefly stunned after clones die → full damage. Watch for double projectile density.

### 吕布 / Lu Bu (虎牢关)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 虎牢关 / Hulao Pass |
| **推荐士气 / Recommended Morale** | **15+** (强烈推荐刷满) / **15+** (strongly recommended) |
| **弱点 / Weakness** | 土属性——所有Boss中雷属性最弱的 / Earth — his elemental weakness |
| **难度 / Difficulty** | ★★★★★ 卧龙最难Boss之一 / One of Wo Long's hardest |

吕布是卧龙最具挑战性的Boss——极长的连招和快速突进。**核心策略**: 不要试图和他换血——你换不过。等他的连招打完→在间歇期打2刀→后退。他的连招可以长达8-10下，中间有假动作 / Lu Bu is Wo Long's most challenging boss — extremely long combos and fast charges. **Core strategy**: Don't trade — you will lose. Wait for combo to finish → 2 hits → retreat. His combos can be 8-10 hits with feints.

**关键攻击**:
- **跳劈(红光)**: 跳起→空中停留约1秒→劈下。化解时机：即将落地时。成功后吕布倒地→打3-4刀
- **赤兔冲锋(红光)**: 骑马冲锋→速度快。侧向闪避比化解容错高
- **连舞**: 方天画戟连续挥舞4-6下——每次挥舞节奏略有不同。建议全部格挡→最后一击后反击
- **特殊技 - 无双**: HP低于30%时触发——全屏AOE攻击。看到吕布蓄力→立刻拉开距离→等AOE结束→再接近

**Key attacks**: Jump slam (red) — 1s hang time, deflect at landing. Horse charge (red) — side dodge is safer. Dance combo — 4-6 hits with varying rhythm, block all then counter. Ultimate — below 30% HP, screen-wide AOE, create distance immediately.

**推荐装备**: 土属性附魔武器(+30%伤害)。防御外脊或HP核心(提高容错)。神兽带应龙(回血)——这场战斗是持久战 / **Recommended gear**: Earth-element weapon. Defense or HP gear. Divine Beast: Qinglong (heal) — this is an endurance fight.

### 于吉 / Yu Ji (最终Boss)

| 信息 / Info | 说明 / Description |
|------|------|
| **关卡 / Mission** | 终局 / Finale |
| **推荐士气 / Recommended Morale** | 25 (满) / 25 (max) |
| **弱点 / Weakness** | 无固定弱点——建议土属性+水属性混合 / No fixed weakness — Earth + Water mix |
| **难度 / Difficulty** | ★★★★★★ 三阶段超长战斗——纯粹对化解的终极大考 / 3-phase marathon — ultimate deflection exam |

于吉是三个阶段的漫长战斗，考验你对化解系统的全面掌握 / Yu Ji is a 3-phase marathon testing every aspect of your deflection mastery.

**P1 - 人形态(100%-70%)**: 
- 人形道士——使用法术+近战混合
- **关键**: 他的法术有很明显的前摇(手上发光)。看到光→等0.5秒→闪避
- 近战攻击节奏中等——2-3连击→硬直→可以打3刀
- **红光技**: 突进抓取——前摇短(约0.3秒)，需要快速反应。反应不够可以格挡(格挡红光也会扣很多气势但保命)

Human form — magic + melee hybrid. **Key**: Magic has clear hand-glow windup. See glow → 0.5s delay → dodge. Melee is mid-tempo — 2-3 hits → stun → 3-hit punish. Red attack: Rush grab — very short tell (0.3s). Block if you can't deflect in time.

**P2 - 妖魔形态(70%-35%)**:
- 化为妖魔——攻击方式大变，新增大量AOE
- **核心变化**: 地面会出现持续AOE区域(黑色旋涡)——站在上面持续扣血。移动战斗
- **红光技**: 两种——1) 全屏AOE(难度一般——跳起躲或拉远) 2) 追踪光束(持续约4秒——绕圈跑)
- **P2关键**: 不要贪刀——打1-2刀就换位。这个阶段的AOE密集度很高

Demon form — completely new moveset, heavy AOE. **Key change**: Black vortex zones on ground — moving DoT. Red attacks: 1) Screen-wide AOE (jump or create distance) 2) Tracking beam (4s duration — circle strafe). **P2 key**: 1-2 hits then reposition. AOE density is high.

**P3 - 最终形态(35%-0%)**:
- 最快最狠的阶段——所有攻击都带属性伤害
- **技能密度极高**：几乎是连续的AOE+追踪+近战组合
- **核心策略**: 保持移动——站定超过2秒就会吃到AOE
- **红光技**: 连续红光——2-3个红光连续释放。需要连续化解，节奏是快-快-慢
- **决胜时刻**: 最后一次绝脉(处决)会触发剧情动画——确保气势槽打崩后就触发绝脉

Fastest and deadliest phase — all attacks have element damage. **Extreme density**: Nearly continuous AOE + tracking + melee combos. **Core strategy**: Never stop moving — standing still for 2s = death. Red combo: 2-3 consecutive reds — rhythm is fast-fast-slow. **Final moment**: Last fatal strike triggers ending cutscene.

**推荐装备**: 药水满级(至少10瓶)。土属性加防+水属性减伤。神兽带麒麟(输出)或应龙(续航)。士气必须满25 / **Recommended gear**: Max potions (10+). Earth DEF + Water DMG reduction. Divine Beast: Qilin (DPS) or Qinglong (sustain). Morale must be 25.

## Boss战进阶技巧 / Advanced Boss Tips

| 技巧 / Tip | 说明 / Description |
|:----:|------|
| **化解红光时机 / Red Deflection Timing** | 所有Boss的红光技都有前摇——记住每个的节奏。一般规律：蓄力类(慢)——看到蓄力后等0.5-1秒化解；突进类(快)——看到动作立刻化解。不熟悉的红光先格挡试一次——格挡红光虽然大扣气势但可以观察节奏 / Learn each boss's red attack rhythm. Charge-type: wait 0.5-1s after windup. Rush-type: deflect on visual. Block unfamiliar reds first to learn timing |
| **奇术辅助 / Wizardry Assist** | 打Boss前上好Buff，比临时加Buff安全。核心Buff：克金符(减防)+ 冰杖(减速)+ 雷附魔(属性伤害)。不要在Boss战中尝试补Buff——等安全窗口 / Pre-buff before boss, not during. Core: Weakness (DEF down) + Ice (slow) + element weapon (damage). Don't rebuff mid-fight — wait for safe window |
| **绝脉节奏 / Fatal Strike Rhythm** | 第一次打崩气势后→绝脉→Boss起身时气势恢复约30%→继续压制→第二次崩气势→绝脉击杀。两次绝脉之间的间隔越短越好——Boss从绝脉起身后有一小段无敌动画，结束后立刻继续攻击 / After first fatal → boss recovers ~30% spirit → re-apply pressure → second fatal = kill. Minimize gap between fatals — boss has brief invincibility after standing up, then vulnerable immediately |
| **神兽时机 / Divine Beast Timing** | 满神兽条不要急着放——最佳时机：化解红光后(Boss倒地)或绝脉动画时。放早了Boss可能闪开/打断释放动画。应龙的加血在绝脉动画期间释放也可以回血 / Best divine beast timing: during red deflect stun or fatal strike animation. Early release risks boss dodging. Qinglong heal can be used during fatal animation |
| **多周目Boss变化 / NG+ Boss Changes** | NG+开始Boss新增攻击+更高士气。NG++Boss有完全新技能。每个周目都要重新适应——不要以为"我知道这个Boss" / NG+: new attacks + higher morale. NG++: entirely new skills. Relearn every boss each NG cycle |
| **奇术取消 / Spell Cancel** | 部分奇术释放后可用化解/闪避取消后摇——用于骗Boss出红光。例如放雷附魔后立刻闪避——Boss有时会以为你在硬直而出红光→你躲开了 / Some spells can be animation-canceled with deflect/dodge — use to bait boss red attacks |
