## 任务常见问题 / Quests FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **怎么解锁副战场？/ Unlock sub-battles?** | 在基地的竹简处领取 / Collect from bamboo scrolls at base |
| **同伴好感怎么提升？/ Increase bond?** | 多次一起出战——每次关卡带同一个同伴 / Bring same companion on multiple missions |
| **牙旗和旌旗区别？/ Flag types difference?** | 牙旗=存档点+奇术学习 / 旌旗=提升士气上限 / Battle Flag = save + wizardry / Standard Flag = morale cap |
| **龙脉结晶怎么用？/ Use Dragon Pulse?** | 在牙旗处用来强化奇术槽 / At Battle Flag for wizardry slot upgrade |
| **最强同行推荐？/ Best companion?** | 赵云（均衡）或曹操（辅助）/ Zhao Yun (balanced) or Cao Cao (support) |
| **主线通关后做什么？/ After main story?** | 清完所有副战场——解锁终极试炼。之后进入NG+刷5星装备 / Clear all sub-battles — unlock ultimate trial. Then NG+ for 5-star gear |
| **龙脉精华和龙脉结晶区别？/ Pulse essence vs crystal?** | 精华=增加奇术伤害 / 结晶=解锁奇术槽位 / Essence=increase wizardry damage / Crystal=unlock slots |
| **装备耐久怎么恢复？/ Repair gear?** | 在牙旗处休息自动恢复——不需要找铁匠 / Auto-repairs on resting at Battle Flag — no need to visit blacksmith |
| **士气等级继承吗？/ Morale carries over?** | 每关独立计算——新关卡从1级开始。但旌旗插好后的上限是永久的 / Per-mission — starts at 1 each mission. But flag caps are permanent |
