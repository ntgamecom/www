# 技能与奇术 / Skills & Wizardry

卧龙的技能体系分为两大部分——奇术（五行法术）和武技（武器技能）/ Wo Long's skill system is divided into two parts — wizardry (elemental spells) and martial arts (weapon skills).

## 士气系统 / Morale System

士气是卧龙的独特系统——影响你的攻防数值 / Morale is Wo Long's unique system — affects both attack and defense.

| 机制 / Mechanic | 说明 / Description |
|---|---|
| **士气等级 / Morale Rank** | 1-25级——击杀敌人增加 / 1-25 — increases on kills |
| **士气影响 / Effect** | 每级士气+攻击力和防御力 / Each rank increases ATK and DEF |
| **最大士气 / Max Morale** | 每个区域有士气上限——由旌旗决定 / Per-region cap — determined by battle flags |
| **士气下降 / Morale Drop** | 死亡扣士气——但不会低于区域下限 / Death decreases morale — capped by region minimum |
| **旌旗 / Battle Flags** | 地图上的旗帜——插旗提升士气上限 / Flags on map — planting raises morale cap |

## 技能常见问题 / Skills FAQ

| 问题 / Question | 回答 / Answer |
|---|---|
| **奇术怎么学？/ How to learn wizardry?** | 在牙旗处——消耗技能点学习 / At Battle Flag — spend skill points |
| **士气一直掉怎么办？/ Morale keeps dropping?** | 多插旌旗——建立士气上限。死亡扣士气但不会低于旗子等级 / Plant more flags — raises cap. Death reduces but not below flag level |
| **五行全修还是专精？/ All elements or one?** | 推荐主修一系+副修一系——全修太分散 / Recommend one main + one sub — spreading too thin |
| **最强奇术系？/ Best element?** | 火系前期强——木/水系后期强。具体看玩法 / Fire early — Wood/Water late. Depends on playstyle |
| **武技可以换吗？/ Can I change martial arts?** | 可以——在牙旗处或铁匠处更换武器上的武技 / Yes — at Battle Flag or Forge |
| **龙脉结晶有什么用？/ Dragon Pulse use?** | 在牙旗处使用——解锁新的奇术槽位，最多4个 / Use at Battle Flag — unlock new wizardry slots, max 4 |
| **武技和奇术区别？/ Martial Arts vs Wizardry?** | 武技=武器技能（不消耗气势）/ 奇术=五行法术（消耗气势）/ Martial Arts=weapon skills (no spirit) / Wizardry=element spells (costs spirit) |
