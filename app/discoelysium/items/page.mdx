# 物品与装备 / Items & Equipment

《极乐迪斯科》的物品系统独特而重要。服装提供技能加成，药物和酒精提供临时增益，而书籍提供永久提升。本页面全面介绍所有重要物品。

The item system in Disco Elysium is unique and important. Clothing provides skill bonuses, drugs and alcohol offer temporary boosts, and books grant permanent improvements. This page covers all important items.

## 服装系统 / Clothing System

### 服装基础 / Clothing Basics

服装是游戏中最重要的装备类型。每件衣服都提供特定技能的加成或减益。你可以同时穿戴：帽子、眼镜、上衣、下装、鞋子、配饰（手套、领带等）。

Clothing is the most important equipment type. Each piece provides bonuses or penalties to specific skills. You can wear simultaneously: hat, glasses, shirt, pants, shoes, accessories (gloves, ties, etc.).

**装备槽位 / Equipment Slots**:
- 头部（Head）: 帽子、头盔等 / Hats, helmets, etc.
- 面部（Face）: 眼镜、面具等 / Glasses, masks, etc.
- 上身（Torso）: 衬衫、外套等 / Shirts, jackets, etc.
- 下身（Legs）: 裤子、裙子等 / Pants, skirts, etc.
- 足部（Feet）: 鞋子、靴子等 / Shoes, boots, etc.
- 手部（Hands）: 手套、戒指等 / Gloves, rings, etc.
- 颈部（Neck）: 领带、围巾等 / Ties, scarves, etc.

### 最佳服装推荐 / Best Clothing by Slot

**头部 / Head**:
- **「对不起警官」头盔 / Sorry Cop Helmet** — 坚忍不拔 +1 / Endurance +1。初期易得 / Easy to get early game
- **迪斯科眼镜 / Disco Glasses** — 标新立异 (Savoir Faire) +2，食髓知味 (Composure) -1。适合风格流 / For style builds

**面部 / Face**:

- **学者的眼镜 / Scholar Glasses** — 逻辑思维 (Logic) +1。推理必备 / Essential for deduction

**上身 / Torso**:
- **糟糕的迪斯科上衣 / Awful Disco Shirt** — 内陆帝国 (Inland Empire) +2，博学多闻 (Encyclopedia) -1。超自然流派首选 / Top for supernatural builds
- **防弹背心 / Bulletproof Vest** — 坚忍不拔 (Endurance) +2。高难度推荐 / Recommended for hard difficulty
- **工作夹克 / Work Jacket** — 体格类技能加成 / Physique skill bonuses。实用选择 / Practical choice

**下身 / Pants**:
- **工装裤 / Cargo Pants** — 能工巧匠 (Perception) +1。探索必备 / Essential for exploration
- **旧牛仔裤 / Old Jeans** — 手脚麻利 (Hand-Eye Coordination) +1。均衡选择 / Balanced choice

**足部 / Feet**:
- **军用靴 / Military Boots** — 手脚麻利 (Hand-Eye Coordination) +1，坚韧 +1。全能有军靴 / Versatile and solid
- **跑鞋 / Running Shoes** — 鬼祟玲珑 (Reaction Speed) +1。敏捷流派 / For agility builds

**手部 / Hands**:
- **皮手套 / Leather Gloves** — 四肢发达 (Interfacing) +1。维修检定专用 / For repair checks
- **紫色手套 / Purple Gloves** — 能工巧匠 (Perception) +1。细节发现者 / For detail finding

**颈部 / Neck**:
- **领带 / Tie** — 可从冷藏库获得。各种颜色提供不同加成 / Obtainable from freezer, different colours give different bonuses
- **经典红色领带 / Classic Red Tie** — 内陆帝国 (Inland Empire) +1。超自然系 / For supernatural builds

**辩论家套装 / Debaters Set**: 学者的眼镜 + 工作夹克 + 紫色手套 — 高精神属性流派。High Psyche builds.

## 药物与酒精 / Drugs & Alcohol

### 药物系统 / Drug System

药物提供强大的临时技能加成，但有成瘾风险。过度使用会导致负面状态和剧情影响。

Drugs provide powerful temporary skill bonuses but carry addiction risks. Overuse leads to negative states and story consequences.

**重要药物 / Important Drugs**:
- **吡啶甲酯 / Pyro**: 提升精神类技能，降低体格类。对话类任务的神器。Boosts Psyche skills, reduces Physique. Essential for dialogue tasks.
- **magnesium / 镁**: 提升体格类技能。体力任务必备。Boosts Physique skills. Essential for physical tasks.
- **酒精 / Alcohol**: 按类型提供不同加成。社交润滑剂但也损害健康。Various bonuses by type. Social lubricant but harms health.
- **香烟 / Cigarettes**: 轻度提升精神集中。尼古丁成瘾需要管理。Mild Psyche boost. Nicotine addiction needs management.

### 酒精种类 / Alcohol Types

**啤酒 / Beer**: 提升痛苦耐受力 (Pain Threshold) +1。适合战斗判定。Good for combat checks.

**烈酒 / Spirits**: 提升能说会道 (Empathy) +1。对话专用。Dialogue专用.

**葡萄酒 / Wine**: 提升故弄玄虚 (Drama) +1。表演系。For performance builds.

**鸡尾酒 / Cocktails**: 多重效果，取决于配方。Multiple effects depending on recipe.

### 成瘾管理 / Addiction Management

**成瘾症状 / Addiction Symptoms**:
- 戒断反应降低相关技能 / Withdrawal reduces related skills
- 影响日常对话选项 / Affects daily dialogue options
- 长期使用改变金的看法 / Long-term use affects Kim opinion
- 某些结局受成瘾影响 / Certain endings affected by addiction

**安全使用原则 / Safe Use Principles**:
- 仅在重要判定前使用 / Use only before important checks
- 交替使用不同药物以降低风险 / Alternate different drugs to reduce risk
- 利用思维阁治疗成瘾 / Use Thought Cabinet to treat addiction
- 不要让药物使用率超过50%的游戏天数 / Do not use drugs on more than 50% of game days

## 工具与书籍 / Tools & Books

### 工具 / Tools

- **撬棍 / Crowbar**: 打开上锁的箱子和门 / Open locked boxes and doors
- **手电筒 / Flashlight**: 照亮黑暗区域 / Light up dark areas
- **测量工具 / Measuring Tape**: 测量现场尺寸 / Measure scene dimensions
- **录音机 / Tape Recorder**: 录制对话作为证据 / Record conversations as evidence

### 书籍 / Books

书籍阅读后提供永久技能加成。每本书需要消耗游戏内时间阅读。

Books grant permanent skill bonuses after reading. Each book consumes in-game time to read.

**必读书籍 / Must-Read Books**:
- **《瑞瓦肖史》/ History of Revachol**: 博学多闻 (Encyclopedia) +1 永久加成 / Permanent bonus
- **《政治哲学导论》/ Intro to Political Philosophy**: 标新立异 (Rhetoric) +1 永久加成 / Permanent bonus
- **《共情的力量》/ The Power of Empathy**: 能说会道 (Empathy) +1 永久加成 / Permanent bonus
- **《逻辑谬误大全》/ Logical Fallacies**: 逻辑思维 (Logic) +1 永久加成 / Permanent bonus

**书籍阅读技巧 / Book Reading Tips**:
- 在睡前阅读以节省时间 / Read before bed to save time
- 同时携带多本书 / Carry multiple books at once
- 优先阅读与你流派相关的书籍 / Prioritise books relevant to your build
- 某些书籍需要特定技能等级才能理解 / Some books require specific skill levels to understand
