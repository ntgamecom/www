# 技能与必杀 / Skills & Specials

《街霸6》的技能系统包含必杀技、超必杀技、独特攻击和通用动作四大类。熟练掌握这些技能是战胜对手的基础。

SF6's skill system includes special moves, super arts, unique attacks and universal actions. Mastering these tools is the foundation of victory.

## 必杀技系统 / Special Move System

必杀技是每个角色的标志性招式，通过特定指令输入发动。不同版本（轻/中/重）的必杀技在伤害、速度、帧数上各有差异。

Special moves are each character's signature techniques, executed with specific command inputs. Different versions (Light/Medium/Heavy) vary in damage, speed and frame data.

| 版本 / Version | 特点 / Characteristics |
|---|---|
| 轻 / Light (LK/LP) | 发招快，伤害低，有利帧好 Fast startup, low damage, good frame advantage |
| 中 / Medium (MK/MP) | 平衡型，综合性能中等 Balanced, moderate overall performance |
| 重 / Heavy (HK/HP) | 发招慢，伤害高，通常有确反风险 Slow startup, high damage, often punishable |
| 超驱动 / OD (Overdrive) | 消耗2格驱动槽，性能全面提升 Costs 2 Drive stocks, all properties enhanced |

### 必杀技类型 / Special Move Types

| 类型 / Type | 说明 / Description |
|---|---|
| 波动系 / Projectile | 飞行道具类，如波动拳、音速手刀 Ranged attacks like Hadouken, Sonic Boom |
| 升龙系 / Dragon Punch (DP) | 对空无敌技，如升龙拳 Anti-air invincible moves like Shoryuken |
| 突进系 / Rush | 快速接近技，如龙卷脚、闪避强击 Fast approach moves like Tatsumaki, Sand Blast |
| 指令投 / Command Grab | 接近后进行投技，无法拆投 Close-range throw that cannot be teched |
| 蓄力系 / Charge | 需要蓄力输入的特殊技，如脚刀 Requires charge input like Flash Kick |
| 特殊架势 / Stance | 进入特殊姿态解锁额外招式 Enter special stance unlocking additional moves |

## 超必杀技 / Super Arts

超必杀技消耗超必杀槽（Super Gauge），槽上限为3格。升级版的Critical Art（CA）在体力低于25%时自动激活。

Super Arts consume the Super Gauge (max 3 stocks). The upgraded Critical Art (CA) activates automatically when HP is below 25%.

### 等级1超必杀 / Level 1 Super Art

消耗1格槽，发招快但伤害较低。适合连段收尾或确反。

Costs 1 stock, fast startup but lower damage. Ideal for combo enders and punishes.

Costs 2 stocks, provides unique tactical functions like self-buff or special pressure.

### 等级3超必杀 / Level 3 Super Art

消耗3格槽，高伤害，通常带有霸体或无敌帧。命中后插入高伤害演出。

Costs 3 stocks, high damage, typically with armor or invincibility. Cinematic high-damage sequence on hit.

Critical Art (CA)版本在低血量时自动升级，伤害进一步提升约20%。

The Critical Art (CA) version auto-upgrades at low HP, increasing damage by approximately 20%.

### 投技系统 / Throw System

投技是防御对手打击压制的重要手段。

Throws are essential for breaking through an opponent's defensive pressure.

| 动作 / Action | 输入 / Input | 说明 / Description |
|---|---|---|
| 前投 / Forward Throw | 轻拳+轻脚（前方向）LP+LK (forward) | 将对手投向前方，可版边壁弹 Throw opponent forward, corner bounce possible |
| 后投 / Back Throw | 轻拳+轻脚（后方向）LP+LK (back) | 将对手投至另一侧，切换位置 Throw opponent to other side, swap positions |
| 拆投 / Throw Tech | 被投时同输入 At throw, same input | 在投发生帧内输入拆投 Input within throw startup frames to tech |
| 指令投 / Command Grab | 特定必杀技指令 Specific command | 不可拆投的特殊投技 Special throw that cannot be teched |

## Meaty设置 / Meaty Setups

Meaty是指在对手起身时精确地让攻击判定覆盖对手的无敌帧结束瞬间。正确执行meaty后，即使对手防御也会陷入不利帧。

A meaty setup means timing an attack so its active frames cover the exact moment the opponent's wake-up invincibility ends. Correctly executed, the opponent is at disadvantage even on block.

| 设置方法 / Setup Method | 说明 / Description |
|---|---|
| 时间差攻击 / Timed Attack | 击倒后走一小段出招 Walk forward slightly after knockdown then attack |
| 连段后追击 / Combo Follow-up | 特定连段后自动建立meaty Certain combos set up automatic meaty |
| DR取消压制 / DR Cancel Pressure | DR接近后出招建立meaty DR approach into meaty attack |
| 逆向meaty / Cross-up Meaty | 跳跃穿至另一侧攻击 Jump cross-up for ambiguous meaty |

### 安全跳 / Safe Jump

安全跳是一种特殊的meaty，攻击时机精确到即使被对手用升龙技确反，你也可以在落地后防御。安全跳需要精确的跳入时机和攻击发生帧知识。

Safe Jump is a specific meaty timed so that even if the opponent uses a reversal DP, you can block after landing. Safe Jumps require precise jump timing and knowledge of attack startup frames.

熟练掌握技能与必杀系统，理解每个招式的性能和用法，是成为高手的重要一步。

Mastering the skills and specials system, understanding each move's properties and applications, is a crucial step toward becoming a high-level player.
