# 排位竞技 / Ranked Mode

排位赛是《街霸6》的核心竞技模式，玩家通过在线对战提升段位。本页详细介绍排位系统、段位结构、评分机制和竞技技巧。

Ranked mode is the core competitive mode of SF6 where players advance through leagues via online battles. This page details the ranking system, league structure, rating mechanics and competitive tips.

## 段位系统 / League System

排位赛共分为多个段位，从新手到大师层级分明。每个大段位内分为5个小段位（Lega）。

Ranked mode has multiple leagues from Rookie to Master, each divided into 5 sub-ranks (Legas) within major leagues.

| 段位 / League | 说明 / Description |
|---|---|
| 新手 / Rookie | 起始段位，适合初学者 Starting league, suitable for beginners |
| 铁牌 / Iron | 基础段位之一，玩家正在学习基础技能 Basic league, players learning fundamentals |
| 铜牌 / Bronze | 玩家开始掌握角色连段 Players beginning to master character combos |
| 银牌 / Silver | 具备基本立回和压制能力 Basic neutral and pressure ability |
| 金牌 / Gold | 能够执行标准连段和基础对策 Can execute standard combos and basic counterplay |
| 白金 / Platinum | 熟练DA系统应用，立回稳定 Proficient Drive System use, stable neutral |
| 钻石 / Diamond | 高水平玩家，角色精通度高 High-level players with high character mastery |
| 大师 / Master | 顶级玩家段位，启用MR评分系统 Top player league, MR rating system enabled |
| 宗师 / Grand Master | 大师段位中的最高MR阶层 Highest MR tier within Master league |
| 传奇 / Legend | 全平台MR排名前500的玩家称号 Title for top 500 MR players across platform |

### 段位升级条件 / League Promotion Conditions

普通段位中，击败段位相近的对手获得联赛经验值（League Points, LP）。积累足够的LP后通过晋级赛（Promotion Match）升至下一段位。

In normal leagues, defeating opponents of similar rank earns League Points (LP). Accumulate enough LP, then pass Promotion Matches to advance to the next league.

| 段位范围 / League Range | 晋级赛要求 / Promotion Requirement |
|---|---|
| Rookie - Diamond 1 | LP达到当前段位上限 + 2/3场晋级赛胜率 LP cap + win 2 out of 3 promotion matches |
| Diamond 2 - Diamond 5 | LP达到当前段位上限 + 3/5场晋级赛胜率 LP cap + win 3 out of 5 promotion matches |
| Diamond 5 - Master | 特殊晋级条件，可能需要连胜 Special conditions, may require win streak |

## 大师段位与MR系统 / Master Rank & MR System

达到大师段位后，LP系统切换为MR（Master Rating）评分系统，反映大师段位玩家的相对实力。

Upon reaching Master rank, the LP system switches to MR (Master Rating), reflecting relative skill among Master-tier players.

### MR系统特点 / MR System Features

| 特性 / Feature | 说明 / Description |
|---|---|
| 起始MR / Starting MR | 新晋大师从1300MR开始 New Masters start at 1300 MR |
| 胜败加减 / Win/Loss Change | 约+/-5 MR每局 Approximately +/-5 MR per match |
| MR重置 / MR Reset | 每个赛季MR会部分重置，根据前赛季最终MR MR is partially reset each season based on final MR |
| 排名追踪 / Rank Tracking | 可在游戏内外查看全球MR排名 View global MR ranking in and out of game |
| 降段保护 / Demotion Protection | 大师段位不会掉回钻石，但MR可降至0 Master rank does not demote to Diamond, but MR can drop to 0 |

### MR Tips

| 建议 / Tip | 详情 / Details |
|---|---|
| 稳定发挥 / Play Consistently | MR系统中每局积分权重相似，避免大起大落 Play each match consistently, avoid extreme swings |
| 连胜加分 / Win Streak Bonus | 连续击败高MR对手可获得额外MR分数 Defeating higher MR opponents in a streak earns bonus MR |
| 高MR对战 / High MR Matches | 击败高MR对手加分更多，值得主动挑战 Defeating higher MR opponents yields more points, worth seeking out |

## 定级赛 / Placement Matches

首次进入排位赛或赛季重置后，需要完成定级赛（Placement Matches）来确定初始段位。

First-time ranked entry or after season reset requires Placement Matches to determine starting league.

| 定级场数 / Matches | 10场 10 matches |
|---|---|
| 初始等级范围 / Initial Range | 从新手到钻石1，取决于10场胜负和对手强度 From Rookie to Diamond 1, depends on win/loss and opponent quality |
| 影响因素 / Factors | 胜场数、连胜次数、对手段位平均值 Wins, win streaks, average opponent rank |

### 定级赛策略 / Placement Strategies

| 策略 / Strategy | 说明 / Description |
|---|---|
| 全力以赴 / Go All Out | 定级赛决定起始段位，尽量使用主力角色 Placement determines starting league, use your best character |
| 记录对比 / Record Opponents | 注意对手段位和胜率，系统根据对手强度调整分数 Note opponent rank and win rate, system adjusts based on opponent quality |
| 心态准备 / Mental Prep | 定级赛可能遇到各种水平对手，保持冷静 Placements may match various skill levels, stay calm |

## 防恶意行为系统 / Anti-Griefing System

Capcom为排位赛设计了多重防护机制，确保公平竞技环境。

Capcom has designed multiple protection mechanisms for ranked mode to ensure a fair competitive environment.

| 防护机制 / Protection | 说明 / Description |
|---|---|
| 断线处罚 / Disconnect Penalty | 频繁断线将被暂时封禁排位 Frequent disconnections result in temporary ranked ban |
| 降级保护 / Demotion Protection | 刚升段后数局内不会立即降级 No immediate demotion for several matches after promotion |
| 段位保护 / League Protection | 大段位之间不会降级（大师除外） No demotion between major leagues (except Master) |
| 对局质量 / Match Quality | 系统优先匹配网络质量和段位相近的对手 System prioritizes matching by connection quality and rank proximity |
| 举报系统 / Report System | 可以对不良行为玩家进行举报 Can report players for misconduct |

## 锦标赛模式 / Tournament Mode

游戏内置的锦标赛模式允许玩家创建和参加线上比赛。

The in-game tournament mode allows players to create and participate in online competitions.

| 类型 / Type | 说明 / Description |
|---|---|
| 官方锦标赛 / Official Tournament | 由Capcom举办，有奖品 Official events by Capcom with prizes |
| 玩家锦标赛 / Player Tournament | 玩家自行创建，自定义规则 Player-created events with custom rules |
| 限时赛事 / Limited Events | 特定活动期间的特殊赛 Special events during specific periods |

### 锦标赛格式 / Tournament Format

| 格式 / Format | 说明 / Description |
|---|---|
| 双败淘汰 / Double Elimination | 主流比赛赛制，败者组复活机会 Standard competition format with losers bracket |
| 单败淘汰 / Single Elimination | 简洁快速，一场定胜负 Simple and fast, single match determines winner |
| 小组赛+淘汰 / Group + Bracket | 先小组循环后淘汰制 Group stage round-robin followed by bracket |

## 斗士擂台 / Fight Hub

斗士擂台是《街霸6》的线上社交大厅。玩家可以在这里进行非排名对战、街机游戏、赛事参与和社交互动。

Battle Hub is SF6's online social lobby. Players can engage in casual matches, arcade games, events and social interactions here.

| 功能 / Feature | 说明 / Description |
|---|---|
| 街机柜 / Arcade Cabinets | 进行非排名对战 Play casual matches |
| 经典游戏 / Classic Games | 使用驱动票游玩街霸经典旧作 Play classic SF titles with Drive Tickets |
| 锦标赛 / Tournaments | 参加或创建赛事 Participate in or create tournaments |
| 头像战斗 / Avatar Battles | 使用自定义头像进行特殊对战 Special battles with custom avatars |
| 商店 / Shop | 购买装扮道具 Purchase cosmetic items |
| 社交 / Social | 与在线玩家互动的交流空间 Socialize and interact with online players |

排位赛是检验实力的最佳场所，但也要记住游戏的根本目的是娱乐。合理安排排位和休闲时间，享受格斗的乐趣。

Ranked mode is the best place to test your skills, but remember the ultimate purpose of gaming is fun. Balance ranked and casual play, and enjoy the fight.
