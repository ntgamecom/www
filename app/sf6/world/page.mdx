# 世界巡回 / World Tour

世界巡回是《街霸6》的单人角色扮演模式。你可以创建自己的角色，在Metro City（都会市）和其他世界中探索、学习和战斗。本页为你提供世界巡回模式的全面攻略。

World Tour is SF6's single-player RPG mode. Create your custom avatar, explore Metro City and other worlds, learn from masters and battle through the open world. This page provides a comprehensive guide for World Tour mode.

## 模式概述 / Mode Overview

世界巡回模式将格斗游戏与角色扮演元素相结合。你从都会市开始，逐步解锁新区域、学习新技能、收集装备并挑战更强的对手。

World Tour mode combines fighting game mechanics with RPG elements. Start in Metro City, gradually unlock new areas, learn new skills, collect equipment and challenge stronger opponents.

| 要素 / Element | 说明 / Description |
|---|---|
| 角色创建 / Character Creation | 自定义角色的外观、体型、服装和语音 Customize avatar appearance, body type, costume and voice |
| 开放世界 / Open World | 在都会市和其他地图中自由探索 Free exploration in Metro City and other maps |
| 师匠系统 / Mentor System | 拜师学习各角色的招牌技能 Learn signature moves from character masters |
| 任务系统 / Quest System | 主线任务和大量支线任务 Main story quests and numerous side quests |
| 等级提升 / Leveling | 通过战斗获得经验值提升等级 Gain XP through battles to increase level |
| 技能树 / Skill Tree | 通过技能点解锁被动和主动能力 Unlock passive and active abilities with skill points |

## 角色创建 / Character Creation

游戏开始时你将被引导进入角色创建系统。你的选择主要影响外观，所有角色在初始属性上是一致的。

At game start you are guided into the character creation system. Your choices primarily affect appearance; all characters have identical base stats.

| 自定义项 / Customization | 可选项 / Options |
|---|---|
| 性别 / Gender | 男性、女性 Male, Female |
| 体型 / Body Type | 多种体态类型可选 Multiple body type options available |
| 面部 / Face | 面部特征的详细调整 Detailed facial feature adjustments |
| 发型 / Hairstyle | 多种发型和颜色 Multiple hairstyles and colors |
| 服装 / Costume | 初始服装选择，后续可在商店购买 Initial costume choice, more available for purchase later |
| 语音 / Voice | 多种声线选择 Multiple voice type options |
| 名字 / Name | 自定义角色名称 Custom character name |

## 师匠系统 / Mentor System

师匠系统是世界巡回的核心机制。你可以拜师各个《街霸》角色，学习他们的必杀技和战斗风格。

The Mentor System is World Tour's core mechanic. You can train under various SF characters to learn their special moves and fighting styles.

### 可拜师的师匠 / Available Mentors

| 师匠 / Mentor | 地点 / Location | 可学技能 / Learnable Skills |
|---|---|---|
| Luke（卢克） | 都会市 Metro City | 闪避强击、毁灭之锤 Sand Blast, Eraser |
| Ryu（隆） | 都会市 Metro City | 波动拳、升龙拳、龙卷旋风腿 Hadouken, Shoryuken, Tatsumaki |
| Chun-Li（春丽） | 都会市 Metro City | 气功拳、百裂脚、鹤脚落 Kikoken, Hyakuretsukyaku, Crane Kick |
| Jamie（杰米） | 唐人街 Chinatown | 醉拳系招式 Drunken fist techniques |
| Marisa（玛丽莎） | 斗技场 Colosseum | 角斗士系招式 Gladiator techniques |
| Dhalsim（达尔西姆） | 瑜伽圣地 Yoga Temple | 瑜伽招式 Yoga techniques |
| Blanka（布兰卡） | 丛林 Jungle | 野性攻击 Wild attacks |
| Kimberly（金柏莉） | 都会市地下 Metro City Underground | 忍术 Ninjutsu techniques |

### 师匠等级提升 / Mentor Bond Level

每个师匠有独立的羁绊等级（Bond Level），通过完成任务、赠送礼物和一起战斗来提升。

Each mentor has an independent Bond Level, raised by completing quests, gifting items and fighting together.

| 等级 / Level | 解锁内容 / Unlocks |
|---|---|
| 1级 / Level 1 | 基础招式学习 Available for basic move training |
| 5级 / Level 5 | 第二招式 Second move unlock |
| 10级 / Level 10 | 第三招式 Third move unlock |
| 15级 / Level 15 | 师匠专属装扮 Mentor exclusive costume |
| 20级 / Level 20 | 所有招式可学 All moves available for learning |

## 开放世界活动 / Open World Activities

### 主线任务 / Main Story Quests

主线任务引导你探索都会市及周边区域，逐步揭示世界巡回的剧情。完成后可获得大量经验值、金钱和特殊装备。

Main story quests guide you through Metro City and surrounding areas, gradually revealing World Tour's storyline. Completion rewards large amounts of XP, money and special equipment.

### 支线任务 / Side Quests

| 类型 / Type | 说明 / Description |
|---|---|
| 路人挑战 / Bystander Battles | 与路上行人战斗 Fight pedestrians on the street |
| 送货任务 / Delivery Quests | 将物品送至指定地点 Deliver items to designated locations |
| 收集任务 / Collection Quests | 收集特定数量的物品或击败特定敌人 Collect specific items or defeat specific enemies |
| 擂台挑战 / Arena Challenges | 在特定地点参加擂台赛 Participate in arena battles at specific locations |
| 师匠任务 / Mentor Quests | 师匠发布的个人任务 Personal quests given by mentors |

## 技能树系统 / Skill Tree System

通过战斗获得技能点（Skill Points），在技能树中解锁各种能力。

Earn Skill Points through battles to unlock various abilities in the Skill Tree.

### 技能树分支 / Skill Tree Branches

| 分支 / Branch | 说明 / Description |
|---|---|
| 攻击 / Attack | 提升基础攻击力、暴击率和连段伤害 Basic attack power, critical rate, combo damage |
| 防御 / Defense | 提升防御力、体力上限和回复速度 Defense, HP cap, recovery speed |
| 技巧 / Technique | 提升指令投伤害、超必杀槽增长速度 Command grab damage, super gauge gain rate |
| 移动 / Movement | 提升移动速度、冲刺速度和跳跃距离 Move speed, dash speed, jump distance |
| 特殊 / Special | 解锁独特能力，如额外必杀技槽位 Unlock unique abilities like extra special move slots |

### 技能点获取 / Skill Point Acquisition

| 方式 / Method | 获得量 / Amount |
|---|---|
| 击败敌人 / Defeat Enemies | 少量 Small amount |
| 完成任务 / Complete Quests | 中量 Moderate amount |
| 升级 / Level Up | 大量 Large amount |
| 师匠等级提升 / Mentor Bond Up | 中量 Moderate amount |
| 收集品 / Collectibles | 少量 Small amount |

## 世界巡回小贴士 / World Tour Tips

| 建议 / Tip | 详情 / Details |
|---|---|
| 尽早拜师 / Find Mentors Early | 第一时间找到所有师匠，开启技能学习通道 Locate all mentors as early as possible to unlock skill learning |
| 多样化技能 / Diversify Skills | 不要只学一个角色的技能，组合不同师匠的招式最有效 Do not focus on one character; mixing moves from different mentors is most effective |
| 做支线任务 / Do Side Quests | 支线任务提供大量经验和金钱，不要跳过 Side quests provide lots of XP and money, do not skip them |
| 更换装备 / Update Gear | 定期检查并更换更好的装备 Regularly check and upgrade to better equipment |
| 探索每个角落 / Explore Everywhere | 隐藏奖励和秘密师匠可能在任何角落 Hidden rewards and secret mentors could be anywhere |

世界巡回模式是体验街霸6世界的最佳方式，也是解锁装扮和道具的主要途径。

World Tour mode is the best way to experience the world of SF6 and the primary method of unlocking costumes and items.
