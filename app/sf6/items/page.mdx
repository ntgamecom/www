# 装备与道具 / Items

《街霸6》拥有丰富的自定义装扮系统，包括角色服装、头像装备、表情动作和各种收集品。本页详细介绍各类道具的获取途径和使用方式。

SF6 features an extensive customization system including character costumes, avatar gear, emotes and various collectibles. This page details all item types and their acquisition methods.

## 头像装备 / Avatar Gear

Avatar gear covers head, upper body, lower body, feet, accessories and weapons. Equipment affects both appearance and stats in World Tour mode.

## 斗士通行证 / Fighting Pass

斗士通行证是定期更新的奖励系统，每个赛季持续约60天。通过完成每日和每周挑战积累通行证点数（Pass Points）来解锁奖励。

Fighting Pass is a regularly updated reward system, each season lasting approximately 60 days. Earn Pass Points through daily and weekly challenges to unlock rewards.

| 奖励类型 / Reward Type | 免费路线 / Free Track | 高级路线 / Premium Track |
|---|---|---|
| 装扮 / Cosmetics | 有限 Limited | 全套 Full set |
| 驱动票 / Drive Tickets | 少量 Small amount | 大量 Large amount |
| 表情动作 / Emotes | 基础款 Basic | 限定款 Exclusive |
| BGM | 无 None | 专属 BGM Exclusive BGM |
| 头像装备 / Avatar Gear | 少量 Some | 全套 Full set |
| 相框 / Frame | 基础款 Basic | 限定款 Exclusive |

### 通行证等级系统 / Pass Level System

通行证共30级，每级需要约1000点通行证点数。每日挑战提供约500点，每周挑战约3000点。购买高级通行证后可立即解锁所有已达成等级的 premium 奖励。

The pass has 30 levels, each requiring approximately 1000 Pass Points. Daily challenges provide about 500 points, weekly challenges about 3000 points. Purchasing the premium pass immediately unlocks all earned premium rewards.

## 驱动票 / Drive Tickets

驱动票是游戏内的免费货币，主要用于购买斗士擂台中出现的经典游戏和部分装饰品。

Drive Tickets are the in-game free currency, primarily used for classic games in Battle Hub and some cosmetic items.

| 获取方式 / Acquisition Method | 驱动票数量 / Amount |
|---|---|
| 每周挑战 / Weekly Challenges | 约 3000 |
| 斗士通行证 / Fighting Pass | 约 5000-10000（视等级）Depending on level |
| 世界巡回 / World Tour | 一次性奖励若干 One-time reward batches |
| 限时活动 / Limited Events | 不定 Variable |

## 斗士金币 / Fighter Coins

斗士金币是付费货币，1美元约等于100斗士金币。用于购买DLC角色、服装、通行证等。

Fighter Coins are the premium currency, approximately 100 coins per 1 USD. Used to purchase DLC characters, costumes, passes and more.

### 免费获取装扮建议 / Free Cosmetic Tips

- 每周完成所有挑战以获得驱动票 Complete all weekly challenges for Drive Tickets
- 活跃参与限时活动获取限定装扮 Participate actively in limited events for exclusive cosmetics
- 探索世界巡回全部区域寻找隐藏宝箱 Explore all World Tour areas for hidden treasure chests
- 提升所有师匠等级解锁专属装扮 Raise all mentor bond levels for exclusive costumes

The cosmetic system is an important personalization element in SF6. Proper planning maximizes the number of items you can obtain for free.
