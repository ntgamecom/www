# 连段指南 / Combo Guide

连段是《街霸6》中将伤害最大化的核心手段。本页从基础连段结构到高级连段技巧进行全面讲解。

Combos are the core method of maximizing damage in SF6. This page covers everything from basic combo structure to advanced combo techniques.

## 连段基础知识 / Combo Fundamentals

所有连段都建立在特定的连接方式之上：

All combos are built on specific connection methods:

| 连接方式 / Connection | 说明 / Description |
|---|---|
| 取消 / Cancel | 前一招未结束即可输入下一招，如必杀技取消 Input next move before current ends, e.g. special cancel |
| 目押 / Link | 前一招收招后特定帧数内输入下一招 Input next move within specific frames after current recovery |
| 驱动突进取消 / DR Cancel | 消耗驱动槽取消普通技收招，连接后续攻击 Cancel normal recovery with Drive Rush for follow-up |
| 超必杀取消 / Super Cancel | 从必杀技取消出超必杀技 Cancel from special move into super art |
| 连段破坏 / Juggle | 对手浮空状态下的追加攻击 Follow-up attacks while opponent is airborne |

### 连段构成要素 / Combo Components

一个标准的连段通常包含以下要素：

A standard combo typically includes:

| 环节 / Phase | 说明 / Description |
|---|---|
| 起始技 / Starter | 确认命中的第一下攻击 The first attack that confirms hit |
| 连接段 / Link | 中间连接部分，保持连段不断 Mid-combo linking to maintain the chain |
| 伤害搬运 / Carry | 将对手向版边搬运 Carry opponent toward corner |
| 收尾技 / Ender | 高伤害收尾，通常用必杀技或超必杀技 High-damage finish, typically with special or super |
| 起攻设置 / Oki Setup | 击倒后设置后续压制 Set up follow-up pressure after knockdown |

### Drive Rush连段 / Drive Rush Combos

DR取消是SF6中最核心的连段进阶技巧：

DR cancel is the most essential advanced combo technique in SF6:

```
标准DR连段结构 / Standard DR Combo Structure:

起始击中确认 - 特殊取消DR - 前冲 - 中攻击 - 重攻击 - 必杀技收尾
Hit confirm - Special cancel DR - Dash forward - MP - HP - Special ender

示例 RURyu：
中拳（确认）- 重波动拳 - DR取消 - 前前 - 重拳 - 升龙拳
MP (confirm) - HP Hadouken - DR cancel - Dash forward - HP - Shoryuken
```

DR连段消耗3格驱动槽，在版边可以造成极高伤害。

DR combos cost 3 Drive Gauge stocks, dealing massive damage near the corner.

### 超必杀取消路线 / Super Cancel Routes

超必杀取消是从必杀技中取消出超必杀技，是重要的伤害提升手段：

Super Cancel is canceling a special move into a super art, a key damage optimization tool:

| 取消路线 / Cancel Route | 说明 / Description |
|---|---|
| 必杀技 - LV1超必杀技 Special - SA1 | 消耗1格超必杀槽，最小伤害提升 Costs 1 super stock, minimal damage boost |
| 必杀技 - LV2超必杀技 Special - SA2 | 消耗2格，中伤害提升 Costs 2 stocks, moderate damage boost |
| 必杀技 - LV3超必杀技 Special - SA3 | 消耗3格，高伤害提升，通常有霸体 Costs 3 stocks, high damage, typically armored |
| CA超必杀技 / Critical Art | 体力低于25%时LV3超必杀升级为CA，伤害大幅提升 SA3 upgrades to CA below 25% HP, massive damage boost |

## 连段练习建议 / Combo Practice Tips

| 建议 / Tip | 详情 / Details |
|---|---|
| 从简到繁 / Simple to Complex | 先掌握3下以内的基础连段，再逐步增加长度 Master 3-hit combos first, then gradually extend |
| 分步练习 / Step by Step | 将连段拆分为起始、连接、收尾三部分单独练习 Break combos into starter, linker, ender and practice each separately |
| 两侧练习 / Both Sides | 在屏幕左右两侧都要能稳定执行 Practice execution on both left and right sides |
| 实战应用 / Real Match Practice | 在实战中故意寻找连段机会，不要只打训练模式 Seek combo opportunities in real matches, not just training mode |

连段是手段而非目的。掌握2-3个可靠的连段后，更应该关注立回、确认和决策。

Combos are a means, not the end goal. After mastering 2-3 reliable combos, focus more on neutral, confirms and decision-making.
