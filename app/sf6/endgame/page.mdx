# 终局内容 / Endgame Guide

当你在排位赛中达到大师段位、完成了世界巡回模式后，游戏仍然提供了丰富的终局内容等待你探索。本页介绍大师段位后的高深玩法和持续可玩内容。

After reaching Master rank in ranked mode and completing World Tour, the game still offers plenty of endgame content to explore. This page covers high-level play and replayable content post-Master.

## 大师段位深度内容 / Master Rank Depth

达到大师段位只是真正战斗的开始。大师段位中你需要持续提升MR评分，与全球顶尖玩家同台竞技。

Reaching Master rank is just the beginning of the real fight. In Master rank you must continuously improve your MR rating and compete against the world's top players.

### MR赛季系统 / MR Season System

| 赛季特特性 / Season Feature | 说明 / Description |
|---|---|
| 赛季时长 / Season Duration | 约3-4个月 Approximately 3-4 months |
| MR重置/MR Reset | 新赛季从前赛季最终MR的80%开始 New season starts at 80% of previous season's final MR |
| 赛季奖励 / Season Rewards | 根据最终段位发放专属装扮和称号 Exclusive cosmetics and titles based on final rank |
| 排行榜 / Leaderboards | 每个平台独立排名，传奇段位为全平台前500 Each platform has separate ranking, Legend is top 500 across all platforms |

### 大师进阶目标 / Master Advancement Goals

| 目标 / Goal | 说明 / Description |
|---|---|
| 1500 MR | 大师平均水平线 Average Master level |
| 1600 MR | 地区级强者水平 Regional strong player level |
| 1700 MR | 国家级选手水平 National competitor level |
| 1800+ MR | 世界级玩家水平 World-class player level |
| 传奇 / Legend | 全平台MR前500 Top 500 MR across all platforms |

## 斗士擂台终局 / Battle Hub Endgame

斗士擂台不仅仅是休闲场所，也是终局内容的重要组成部分。

Battle Hub is not just a casual space but an important part of the endgame content.

### 头像战斗 / Avatar Battles

在世界巡回模式中培养的自定义头像可以在斗士擂台的特殊竞技场中对战。头像的属性来自世界巡回的装备和技能组合，形成独特的战斗体验。

Your custom avatar trained in World Tour can battle in special arenas in Battle Hub. Avatar stats come from World Tour equipment and skill combinations, creating a unique battle experience.

| 头像竞技类型 / Type | 说明 / Description |
|---|---|
| 休闲赛 / Casual | 不计排名，纯粹娱乐 No rank tracking, pure fun |
| 排位赛 / Ranked | 头像排位积分系统，有专属段位 Avatar ranking point system with dedicated leagues |
| 组队战 / Team Battle | 3v3组队头像对战 3v3 team avatar battles |
| 极限赛 / Extreme Battle | 特殊规则对战，如道具出现或屏幕反转 Special rule matches like item drops or screen flip |

### 街机柜对战 / Arcade Cabinet Battles

在斗士擂台的街机柜上可以进行标准格斗对战。这是练习新角色、测试新策略的理想场所。

Standard fighting battles can be played at Battle Hub arcade cabinets. This is the ideal place to practice new characters and test new strategies.

| 功能 / Feature | 说明 / Description |
|---|---|
| 随机匹配 / Random Match | 系统自动匹配对手 System automatically matches opponents |
| 好友对战 / Friend Match | 邀请指定好友对战 Invite specific friends to battle |
| 观战 / Spectate | 观看其他玩家的对战 Watch other players' matches |
| 录像保存 / Replay Save | 对战录像自动保存，可回顾 Match replays automatically saved for review |

### 经典街机游戏 / Classic Arcade Games

使用驱动票在斗士擂台玩到街霸系列的经典旧作，包括初代《街头霸王》、《街头霸王II》系列和《街头霸王III》等。

Use Drive Tickets to play classic SF series titles in Battle Hub, including the original Street Fighter, the SFII series and Street Fighter III.

| 游戏 / Game | 驱动票消耗 / Drive Ticket Cost |
|---|---|
| 街头霸王II / Street Fighter II | 1票/次 1 ticket per play |
| 超级街头霸王II Turbo / Super SFII Turbo | 1票/次 1 ticket per play |
| 街头霸王Alpha系列 / SF Alpha Series | 2票/次 2 tickets per play |
| 街头霸王III 3rd Strike / SFIII 3rd Strike | 2票/次 2 tickets per play |

## 角色精通 / Character Mastery

每个角色都有角色精通系统，使用该角色进行战斗可以提升精通等级。

Each character has a Mastery system. Playing as a character raises their Mastery level.

### 精通奖励 / Mastery Rewards

| 精通等级 / Mastery Level | 奖励 / Reward |
|---|---|
| 1-5级 / Level 1-5 | 角色颜色解锁 Character color unlocks |
| 6-10级 / Level 6-10 | 称号和相框 Titles and frames |
| 11-15级 / Level 11-15 | 专属表情动作 Exclusive emotes |
| 16-20级 / Level 16-20 | 特殊装扮和BGM Special costume and BGM |
| 21-25级 / Level 21-25 | 金色颜色和终极称号 Gold colors and ultimate titles |

### 精通提升建议 / Mastery Tips

| 建议 / Tip | 说明 / Description |
|---|---|
| 多角色轮换 / Rotate Characters | 将练习分散到多个角色以全面提高 Spread practice across multiple characters for all-around improvement |
| 排位赛与休闲结合 / Ranked + Casual Mix | 排位赛提升最快，休闲赛可降低压力 Ranked is fastest improvement, casual reduces pressure |
| 利用训练模式 / Use Training Mode | 针对弱点科目进行专项训练 Targeted training for specific weaknesses |

## 锦标赛与赛事 / Tournaments & Events

### 游戏内锦标赛 / In-Game Tournaments

斗士擂台定期举办官方和玩家自建的锦标赛。参加锦标赛是终局的主要内容之一。

Battle Hub regularly hosts official and player-created tournaments. Participating in tournaments is a key endgame activity.

| 赛事级别 / Level | 频率 / Frequency | 奖励 / Rewards |
|---|---|---|
| 每日锦标赛 / Daily | 每日开放 Daily available | 少量驱动票 Few Drive Tickets |
| 每周锦标赛 / Weekly | 每周末 Weekend event | 装扮和大量驱动票 Cosmetics and many Drive Tickets |
| 赛季锦标赛 / Seasonal | 每赛季一次 Once per season | 限定装扮和称号 Exclusive cosmetics and titles |
| 官方赛事 / Official | 不定期 Irregular | 实物奖品和现金奖励 Physical prizes and cash rewards |

### 观赛与学习 / Spectating & Learning

| 平台 / Platform | 说明 / Description |
|---|---|
| 游戏内观战 / In-Game Spectate | 直接观看高手对局 Watch high-level matches directly in game |
| CFN（斗士网络）/ Capcom Fighter Network | 查看顶级玩家的战斗录像 View top player match replays |
| 回放模式 / Replay Mode | 下载并分析高端玩家对局 Download and analyze high-level matches |

## 持续挑战 / Ongoing Challenges

终局阶段可以为自己设定以下目标来保持游戏动力：

In the endgame phase, set yourself the following goals to maintain motivation:

| 挑战 / Challenge | 说明 / Description |
|---|---|
| 全角色大师 / All Characters Master | 将所有角色打上大师段位 Bring all characters to Master rank |
| 角色精通100% / 100% Character Mastery | 所有角色达到最高精通等级 Max mastery level on all characters |
| 全成就/奖杯 / All Achievements/Trophies | 达成游戏全部成就/奖杯 Unlock all game achievements/trophies |
| 全球排名 / Global Ranking | 冲击全球MR排行榜 Top 全球 MR leaderboard |
| 锦标赛优胜 / Tournament Win | 在官方或大型赛事中夺冠 Win an official or major tournament |
| 教学相长 / Teach Others | 将自己的知识分享给社区，帮助新玩家 Share knowledge with the community, help new players |

终局内容的核心在于持续的自我提升和社区参与。无论你的目标是什么，享受游戏过程最重要。

The core of endgame content lies in continuous self-improvement and community engagement. Whatever your goals, enjoying the game matters most.
