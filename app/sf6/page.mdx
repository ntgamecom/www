## 游戏概述 / Game Overview

《街头霸王6》是由Capcom（卡普空）开发的格斗游戏系列最新作，采用RE引擎开发，于2023年6月2日全球发售。本作支持PlayStation 5、PlayStation 4、Xbox Series X|S和PC（Steam）平台，并全面支持跨平台联机对战（Cross-Play）。游戏秉承系列经典的格斗系统，同时引入全新的Drive System（驱动系统）和Modern Control（现代操作）模式，大幅降低了新手入门门槛。

Street Fighter 6 is the latest entry in Capcom's legendary fighting game series, developed on the RE Engine and released globally on June 2, 2023. It supports PlayStation 5, PlayStation 4, Xbox Series X|S and PC (Steam) with full cross-platform play. The game carries forward the classic fighting system while introducing the全新的 Drive System and Modern Control scheme, significantly lowering the barrier for newcomers.

## 核心特色 / Key Features

| 特色 / Feature | 说明 / Description |
|---|---|
| 驱动系统 / Drive System | 五种驱动技能整合于单一资源槽，涵盖突击、格挡、反击等多种战术选项 Five drive skills unified under one resource gauge for diverse tactical options |
| 现代操作 / Modern Controls | 一键必杀与自动连段，适合新手和休闲玩家 One-button specials and auto combos for newcomers and casual players |
| 经典操作 / Classic Controls | 六键传统布局，追求最大操作精度和自由度 Six-button legacy layout for maximum precision and freedom |
| 跨平台联机 / Cross-Play | PS5、PS4、Xbox Series X|S、PC全平台互通对战 Seamless online battles across all platforms |
| 世界巡回 / World Tour | 单人角色扮演模式，自定义角色探索 Metro City Single-player RPG mode with custom avatar exploration in Metro City |
| 斗士擂台 / Battle Hub | 线上社交大厅，街机对战与锦标赛 Online social hub with arcade cabinets and tournament events |
| 格斗中心 / Fighting Ground | 经典街机、训练模式和本地对战合集 Classic arcade, training mode and local versus collection |

PC minimum requirements: Windows 10 (64-bit), Intel Core i5-7500 or AMD Ryzen 3 1200, 8GB RAM, GTX 1060 or RX 580. Recommended: Windows 11, i7-8700 or Ryzen 5 3600, 16GB RAM, RTX 2070 or RX 6700 XT.
